# 配对交易策略图

本图提出了一种基于两种证券相对价值的配对交易策略，采用独特方法识别并利用两种相关资产之间的价格差异。

## 概述

配对交易是一种市场中性策略，当两种资产的价格比率偏离历史常态时，同时买入一种资产并卖出另一种资产。本图以两种具体证券为例：SBER@TQBR 和 GAZP@TQBR。

## 策略逻辑

- **指数计算**：策略根据公式 `SBER@TQBR / GAZP@TQBR` 计算指数。该指数有助于判断一只股票相对于另一只股票的相对强弱。
- **买入条件**：若指数上升，表明 SBER@TQBR 相对于 GAZP@TQBR 变贵，策略买入较便宜的资产（GAZP@TQBR）并卖出较贵的资产（SBER@TQBR）。
- **卖出条件**：若指数下降，表明 SBER@TQBR 相对于 GAZP@TQBR 变便宜，策略买入较贵的资产（SBER@TQBR）并卖出较便宜的资产（GAZP@TQBR）。

## 主要特点

- **四舍五入值**：使用 `round` 运算符将计算出的指数值转换为整数。这种简化通过提供更清晰、更具操作性的信号来辅助决策。
- **市场中性**：旨在从价格比率向历史平均值收敛中获利，不受市场整体方向影响。

## 应用与收益

- **风险规避**：通过交易历史相关的配对品种，策略将市场风险降至最低，因为一方的收益通常可以抵消另一方的损失。
- **利用价格低效**：策略利用配对证券价格的临时低效，这些低效预计最终将回归均值。

## 执行

- **设置条件**：在实施策略之前，确保对两种证券进行密切监控，以识别可能触发交易的重大背离。
- **运营动态**：根据历史数据和市场状况，持续监控和重新校准买入和卖出的阈值水平，对于策略的成功至关重要。

本图不仅概述了配对交易的稳健框架，还强调了数学工具（如四舍五入）在简化复杂交易决策中的重要性。
