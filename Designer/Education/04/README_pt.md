# Diagrama de Estratégia Simples para Designer
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Este documento acompanha um arquivo esquemático que descreve uma estratégia de trading simples usando a ferramenta Designer do StockSharp. O diagrama fornece uma representação visual do fluxo de trabalho e dos componentes da estratégia, permitindo que os usuários entendam e repliquem sua funcionalidade em seu próprio ambiente de trading.

## Visão Geral da Estratégia

A estratégia simples demonstrada neste diagrama concentra-se em sinais de trading básicos baseados em critérios predefinidos, como médias móveis, limites de preço ou indicadores de volume. O objetivo é fornecer uma compreensão fundamental de como construir e executar uma estratégia dentro do framework do Designer.

## Componentes do Diagrama

O diagrama inclui os seguintes componentes principais:

- **Fontes de Dados**: Define as entradas de dados de mercado usadas pela estratégia, incluindo dados de preço e volume.
- **Indicadores**: Utiliza indicadores técnicos comuns que acionam sinais de trading quando certas condições são atendidas.
- **Nós de Decisão**: Mostra os caminhos lógicos que determinam quando e como as negociações são executadas com base nos dados e indicadores.
- **Comandos de Execução**: Detalha as ações tomadas quando um sinal de trading é confirmado, como ordens de compra ou venda.

## Usando o Diagrama

Para usar este diagrama efetivamente:

1. **Abra o Arquivo no Designer**: Carregue o arquivo esquemático na ferramenta Designer para visualizar e modificar a estratégia.
2. **Estude o Fluxo de Trabalho**: Entenda como cada componente interage e como os dados fluem pela estratégia.
3. **Personalize a Estratégia**: Faça ajustes nos componentes da estratégia para atender às suas preferências de trading e critérios de gestão de risco.
4. **Simule e Implante**: Teste a estratégia usando os recursos de simulação de dados históricos no Designer e ajuste com base nos resultados antes de operar ao vivo.

Este diagrama serve como ferramenta educacional para aqueles que são novos na automação de trading e no design de estratégias usando o software Designer. Ele fornece um exemplo prático de como uma estratégia simples pode ser visualmente mapeada e executada dentro de uma poderosa plataforma de trading.
