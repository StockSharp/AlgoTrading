# Designer 简单策略图解
[English](README.md) | [Русский](README_ru.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

本文档随附一个图解文件，概述了使用 StockSharp Designer 工具的简单交易策略。该图解以可视化方式呈现了策略的工作流程和组件，帮助用户理解并在自己的交易环境中复现其功能。

## 策略概述

此图解中演示的简单策略基于预定义标准（如移动平均线、价格阈值或成交量指标）生成基本交易信号。其目标是提供关于如何在 Designer 框架内构建和执行策略的基础理解。

## 图解组件

该图解包含以下关键组件：

- **数据源**：定义策略使用的市场数据输入，包括价格和成交量数据。
- **指标**：使用在满足特定条件时触发交易信号的常见技术指标。
- **决策节点**：展示根据数据和指标确定何时以及如何执行交易的逻辑路径。
- **执行命令**：详细说明交易信号确认后所采取的操作，例如买入或卖出订单。

## 使用图解

要有效使用此图解：

1. **在 Designer 中打开文件**：将图解文件加载到 Designer 工具中，以查看和修改策略。
2. **研究工作流程**：了解每个组件如何交互以及数据如何流经策略。
3. **自定义策略**：调整策略组件以符合您的特定交易偏好和风险管理标准。
4. **模拟并部署**：使用 Designer 中的历史数据模拟功能测试策略，并根据结果进行调整，然后再进行实盘交易。

此图解是 Designer 软件交易自动化和策略设计新手的教育工具。它提供了一个实际示例，展示如何以可视化方式规划和执行简单策略，使其在强大的交易平台中运行。
