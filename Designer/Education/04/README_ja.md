# Designer のシンプル戦略ダイアグラム
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このドキュメントは、StockSharp の Designer ツールを使用したシンプルなトレーディング戦略の概要を示すスキーマファイルに付随するものです。このダイアグラムは戦略のワークフローとコンポーネントを視覚的に表現しており、ユーザーが自分のトレーディング環境でその機能を理解し、再現できるようにします。

## 戦略の概要

このダイアグラムで示されるシンプルな戦略は、移動平均線、価格しきい値、出来高指標などの事前定義された基準に基づく基本的なトレーディングシグナルに焦点を当てています。目的は、Designer フレームワーク内で戦略を構築し実行する方法の基本的な理解を提供することです。

## ダイアグラムのコンポーネント

ダイアグラムには以下の主要コンポーネントが含まれています：

- **データソース**：価格データや出来高データを含む、戦略が使用する市場データの入力を定義します。
- **インジケーター**：特定の条件が満たされたときにトレーディングシグナルをトリガーする一般的なテクニカルインジケーターを使用します。
- **決定ノード**：データとインジケーターに基づいて、いつどのように取引を実行するかを決定するロジックパスを示します。
- **実行コマンド**：買い注文や売り注文など、トレーディングシグナルが確認されたときに取られるアクションを詳述します。

## ダイアグラムの使い方

このダイアグラムを効果的に使用するには：

1. **Designer でファイルを開く**：スキーマファイルを Designer ツールに読み込んで、戦略を表示・修正します。
2. **ワークフローを理解する**：各コンポーネントがどのように連携し、データが戦略をどのように流れるかを理解します。
3. **戦略をカスタマイズする**：特定のトレーディング設定やリスク管理基準に合わせて、戦略コンポーネントを調整します。
4. **シミュレーションして展開する**：Designer の履歴データシミュレーション機能を使用して戦略をテストし、実際のトレーディングを開始する前に結果に基づいて調整します。

このダイアグラムは、Designer ソフトウェアを使用したトレーディング自動化と戦略設計の初心者向けの教育ツールとして機能します。シンプルな戦略がどのように視覚的にマッピングされ、強力なトレーディングプラットフォーム内で実行できるかの実践的な例を提供します。
