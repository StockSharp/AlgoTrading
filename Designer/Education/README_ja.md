Designer ビデオチュートリアルディレクトリへようこそ。このフォルダには、StockSharp の YouTube チャンネルで公開されているビデオチュートリアルに関連するリンクとリソースが収録されています。これらのチュートリアルは、基本的なセットアップから高度な戦略設計まで、Designer プログラムを効果的に使用するためのユーザー教育に焦点を当てています。

## 概要

StockSharp の YouTube チャンネルでは、StockSharp プラットフォーム内の Designer ツールを最大限に活用できるよう支援することを目的としたビデオチュートリアルシリーズ（[今すぐ視聴](https://www.youtube.com/playlist?list=PLpwmSw294KYrfM1ALrONIz9MLpRQ1Uwfe)）を提供しています。このディレクトリは、これらの動画のキュレーションリストとして機能し、教育コンテンツへの簡単なアクセスを提供します。

## このディレクトリの使い方

これらのビデオチュートリアルを最大限に活用するには：

1. **動画を選択する**：現在の学習ニーズや関心に合ったチュートリアルを選んでください。
2. **視聴して学ぶ**：提供されたリンクをクリックして YouTube でチュートリアルを視聴します。メモを取りながら、または自分の Designer インストール環境で手順を追いながら視聴することをお勧めします。
3. **学んだことを実践する**：動画で紹介された技術や戦略を自分のトレーディング環境で実装してみてください。
4. **コミュニティへの参加**：いいねやコメント、質問を通じて動画に参加しましょう。このやり取りにより、コミュニティや制作者からさらなる洞察と助けが得られます。

## 貢献

ここに記載されていない Designer に関連する有用なビデオチュートリアルを見つけた場合、または自分でチュートリアルを作成した場合は、ぜひ貢献を検討してください：

- **新しいリンクを提出する**：他の Designer ユーザーに役立つと思われる追加のビデオチュートリアルへのリンクを提供してください。
- **チュートリアルへのフィードバック**：掲載されているチュートリアルの改善提案や誤りを見つけた場合は、お知らせください。
