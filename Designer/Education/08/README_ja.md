# 高度なマルチタイムフレーム戦略ダイアグラム
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このファイルは、StockSharp の Designer プラットフォーム専用に設計された、異なるタイムフレームのローソク足を使用する複雑な戦略ダイアグラムを示しています。この例では、複数のブランチにわたってさまざまな条件を活用し、過去の価格データに基づいて取引を実行します。

## 戦略の詳細

ダイアグラムは2つの主要なブランチに分かれており、それぞれが5分足ローソク足を過去の価格の極値と比較して取引判断を行います。

### 第1ブランチ — 過去の極値
- **買い条件**：5分足ローソク足の終値が過去20日間の最高値を上回る場合、戦略は買い注文を開始します。
- **売り条件**：5分足ローソク足の終値が過去10日間の最安値を下回る場合、売り注文が執行されます。

### 第2ブランチ — 逆方向の条件
- **売り条件**：5分足ローソク足の終値が過去20日間の最安値を下回る場合、売り注文が執行されます。
- **買い条件**：5分足ローソク足の終値が過去10日間の最高値を上回る場合、買いが開始されます。

## バージョン固有の機能と変更点
- **フラグブロックの外観**：Designer バージョン5では、フラグブロックの外観が更新されました。
- **戦略の適応**：バージョン5では、売りと買いの両シグナルに対して2つのブロックを使用するよう戦略が変更されました。この調整は、新しいバージョンの Designer においてシグナルがアクションをトリガーする方法の変化によるものです。

このダイアグラムは、短期の価格アクションを長期の価格記録と比較することで、重要な価格変動に反応する戦略を実装およびテストするためのフレームワークを提供します。マルチブランチのアプローチにより、トレーダーは同じ基礎データを使用して、さまざまな戦略的対応を試みることができます。
