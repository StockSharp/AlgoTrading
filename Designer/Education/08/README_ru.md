# Схема стратегии с несколькими таймфреймами
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Этот файл иллюстрирует сложную схему стратегии, использующей свечи из разных таймфреймов, специально разработанную на платформе Designer от StockSharp. В данном примере применяются различные условия в нескольких ветках для совершения сделок на основе исторических ценовых данных.

## Детали стратегии

Схема разделена на две основные ветки, каждая из которых использует пятиминутные свечи, сравниваемые с историческими ценовыми экстремумами для принятия торговых решений:

### Первая ветка — исторические экстремумы
- **Условие покупки**: стратегия инициирует ордер на покупку, если цена закрытия пятиминутной свечи превышает наибольшую цену за последние 20 дней.
- **Условие продажи**: ордер на продажу выставляется, если цена закрытия пятиминутной свечи ниже наименьшей цены за последние 10 дней.

### Вторая ветка — обратные условия
- **Условие продажи**: ордер на продажу выставляется, если цена закрытия пятиминутной свечи ниже наименьшей цены за последние 20 дней.
- **Условие покупки**: инициируется покупка, если цена закрытия пятиминутной свечи превышает наибольшую цену за последние 10 дней.

## Особенности версий и изменения
- **Внешний вид куба-флага**: в пятой версии Designer внешний вид куба-флага был обновлён.
- **Адаптации стратегии**: также в версии 5 стратегия была изменена — для сигналов продажи и покупки теперь используются по два куба. Это изменение связано с тем, как сигналы запускают действия в новой версии Designer.

Данная схема предоставляет основу для реализации и тестирования стратегий, реагирующих на значительные ценовые движения путём сравнения краткосрочных ценовых действий с долгосрочными ценовыми рекордами. Подход с несколькими ветками позволяет трейдерам экспериментировать с различными стратегическими реакциями на основе одних и тех же исходных данных.
