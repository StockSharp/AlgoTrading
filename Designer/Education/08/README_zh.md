# 高级多时间框架策略图

本文件展示了一个复杂的策略图，使用来自不同时间框架的蜡烛图，专为 StockSharp Designer 平台设计。该示例利用多个分支中的不同条件，基于历史价格数据执行交易。

## 策略详情

该图分为两个主要分支，每个分支使用五分钟蜡烛图与历史价格极值进行比较，以做出交易决策：

### 第一分支——历史极值
- **买入条件**：若五分钟蜡烛图的收盘价高于过去20天的最高价，策略发出买入订单。
- **卖出条件**：若五分钟蜡烛图的收盘价低于过去10天的最低价，执行卖出订单。

### 第二分支——反向条件
- **卖出条件**：若五分钟蜡烛图的收盘价低于过去20天的最低价，执行卖出订单。
- **买入条件**：若五分钟蜡烛图的收盘价高于过去10天的最高价，发出买入信号。

## 版本特性与变更
- **标志方块外观**：在 Designer 第5版中，标志方块的外观已更新。
- **策略调整**：同样在第5版中，策略已修改为在卖出和买入信号中各使用两个方块。此调整源于新版 Designer 中信号触发动作方式的变化。

本图为实现和测试对重大价格波动做出反应的策略提供了框架，通过将短期价格行为与长期价格记录进行比较来作出决策。多分支方法允许交易者基于相同的基础数据，尝试不同的策略应对方式。
