# Схема стратегии с рабочим временем и таймером
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Этот файл схемы демонстрирует применение куба «Рабочее время» вместе с другими соответствующими кубами на платформе Designer для реализации торговых стратегий на основе времени.

## Обзор

Схема изучает различные конфигурации с использованием куба «Рабочее время», который позволяет трейдерам исполнять стратегии, опираясь на конкретные временные условия.

## Ключевые компоненты

- **Куб «Рабочее время»**: используется для определения активных торговых часов или конкретного времени исполнения сделок.
- **Куб «Переменная»**: с именем «Стратегия» — хранит и обрабатывает переменные, специфичные для стратегии.
- **Куб «Конвертер»**: применяется для преобразования и получения данных, связанных со временем, для принятия временных решений.

## Детали стратегии

### Стратегия с условием рабочего времени
- **Покупка перед закрытием**: стратегия инициирует ордер на покупку за одну минуту до завершения установленного рабочего времени, стремясь использовать возможные ценовые движения в конце торговой сессии.

### Срабатывание в конкретное время
- **Покупка в фиксированное время**: реализует покупку ровно в 18:00, синхронизируя исполнение сделки со значимыми рыночными событиями или типичным временем закрытия.

### Расширенное закрытие по времени из урока 7
- **Закрытие позиций**: закрывает все открытые позиции за пять минут до конца рабочего времени — стратегия, направленная на исключение удержания позиций на ночь или реакции на колебания цен в конце дня.

## Примечание об изменениях в версии 5

В пятой версии программного обеспечения Designer были улучшены расчёты времени и совместная работа куба «Рабочее время». После импорта стратегий, использующих эти функции, рекомендуется воссоздать их в платформе для обеспечения корректной работы и использования обновлённых формул расчёта времени.

Данная схема предоставляет комплексную основу для разработки и тестирования стратегий, в значительной мере опирающихся на точное время исполнения сделок. Она является незаменимым инструментом для трейдеров, ориентированных на внутридневные стратегии или придерживающихся строгих торговых часов.
