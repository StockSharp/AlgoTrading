# ポジション保護キューブ：テイクプロフィットとストップロス
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このファイルには、StockSharp Designer で設計されたビジュアル戦略が含まれており、特にテイクプロフィットとストップロスの操作に対してポジション保護キューブの使用を実演しています。この戦略は、Designer 環境内でこれらの重要なトレーディングツールを設定して活用し、トレーディングポジションを効果的に管理・保護する方法の明確な例を提供します。

## 概要

このファイルで示されているスキームは、テイクプロフィットとストップロスキューブのシンプルな実装を示しており、ユーザーがこれらの機能を実際のトレーディングシナリオにどのように適用できるかを確認できます。これは、Designer のドラッグ＆ドロップインターフェースを使用したポジション保護テクニックの理解と適用を助けることを目的とした教育リソースです。

### 内容

- **テイクプロフィットキューブ**：利益を確定するために、あらかじめ設定された価格水準で自動的に売り注文を実行します。
- **ストップロスキューブ**：潜在的な損失を限定するために、あらかじめ設定された価格水準で自動的に売り注文を実行します。

このスキームは、自動化されたリスク管理ツールでトレーディング戦略を強化したいトレーダーに特に有用です。Designer のビジュアルレイアウトとインタラクティブ要素により、トレーディング保護の学習と実装に優れた手段となっています。

### 使用方法

このファイルを StockSharp Designer に読み込んで、戦略を表示し操作します。ユーザーは Designer 内でパラメータを直接変更して、特定のトレーディング要件とリスク管理戦略に合わせることができます。このファイルは、自動化されたセーフティメカニズムを持つ堅牢なトレーディング戦略を構築したい人向けの実践的な例として機能します。
