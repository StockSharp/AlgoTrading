# Кубы защиты позиции: тейк-профит и стоп-лосс
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Этот файл содержит визуальную стратегию, созданную в StockSharp Designer, демонстрирующую использование кубов защиты позиции — конкретно для операций тейк-профита и стоп-лосса. Стратегия наглядно показывает, как настроить и применять эти важнейшие торговые инструменты в среде Designer для эффективного управления и защиты торговых позиций.

## Обзор

Схема, представленная в этом файле, демонстрирует простую реализацию кубов тейк-профита и стоп-лосса, позволяя пользователям видеть, как эти функции могут применяться в реальных торговых сценариях. Это образовательный ресурс, помогающий трейдерам понять и применить методы защиты позиций с помощью интерфейса перетаскивания Designer.

### Содержимое

- **Куб Take Profit**: Автоматически выполняет ордер на продажу на заранее определённом ценовом уровне для фиксации прибыли.
- **Куб Stop Loss**: Автоматически выполняет ордер на продажу на заранее определённом ценовом уровне для ограничения потенциальных убытков.

Эта схема особенно полезна для трейдеров, желающих дополнить свои стратегии автоматизированными инструментами управления рисками. Визуальный макет и интерактивные элементы Designer делают её отличным способом изучения и реализации защиты торговых позиций.

### Использование

Загрузите этот файл в StockSharp Designer для просмотра и работы со стратегией. Пользователи могут изменять параметры непосредственно в Designer в соответствии со своими торговыми требованиями и стратегиями управления рисками. Этот файл служит практическим примером для тех, кто хочет строить надёжные торговые стратегии с автоматизированными механизмами защиты.
