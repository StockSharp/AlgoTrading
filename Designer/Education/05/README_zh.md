# 持仓保护积木：止盈与止损
[English](README.md) | [Русский](README_ru.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

该文件包含在 StockSharp Designer 中设计的可视化策略，演示了持仓保护积木的使用，特别是止盈和止损操作。该策略清晰地展示了如何在 Designer 环境中设置和使用这些关键交易工具，以有效管理和保护交易持仓。

## 概述

本文件中展示的方案提供了止盈和止损积木的直观实现，让用户了解这些功能如何应用于真实的交易场景。这是一个教育资源，旨在帮助交易者使用 Designer 的拖放界面理解和应用持仓保护技术。

### 内容

- **止盈积木**：在预定价格水平自动执行卖出订单以锁定利润。
- **止损积木**：在预定价格水平自动执行卖出订单以限制潜在损失。

该方案对于希望通过自动化风险管理工具增强交易策略的交易者尤为有用。Designer 的可视化布局和交互元素使其成为学习和实施交易保护的绝佳方式。

### 使用方法

将此文件加载到 StockSharp Designer 中以查看策略并与之互动。用户可以直接在 Designer 中修改参数，以满足其特定的交易需求和风险管理策略。该文件为有兴趣构建具有自动化安全机制的稳健交易策略的用户提供了实践参考。
