# Cubos de Proteção de Posição: Take Profit e Stop Loss
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Este arquivo contém uma estratégia visual projetada no StockSharp Designer, demonstrando o uso de cubos de proteção de posição especificamente para operações de take profit e stop loss. A estratégia fornece um exemplo claro de como configurar e utilizar essas ferramentas de trading críticas dentro do ambiente Designer para gerenciar e proteger posições de trading de forma eficaz.

## Visão Geral

O esquema ilustrado neste arquivo mostra uma implementação direta de cubos de take profit e stop loss, permitindo que os usuários vejam como esses recursos podem ser aplicados em cenários de trading reais. É um recurso educacional voltado a ajudar os traders a entender e aplicar técnicas de proteção de posição usando a interface de arrastar e soltar do Designer.

### Conteúdo

- **Cubo Take Profit**: Executa automaticamente uma ordem de venda em um nível de preço predeterminado para garantir lucros.
- **Cubo Stop Loss**: Executa automaticamente uma ordem de venda em um nível de preço predeterminado para limitar perdas potenciais.

Este esquema é particularmente útil para traders que desejam aprimorar suas estratégias de trading com ferramentas automatizadas de gestão de risco. O layout visual e os elementos interativos do Designer tornam-no uma excelente forma de aprender e implementar proteções de trading.

### Uso

Carregue este arquivo no StockSharp Designer para visualizar e interagir com a estratégia. Os usuários podem modificar os parâmetros diretamente no Designer para atender aos seus requisitos específicos de trading e estratégias de gestão de risco. Este arquivo serve como exemplo prático para aqueles interessados em construir estratégias de trading robustas com mecanismos de segurança automatizados.
