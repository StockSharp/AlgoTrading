# Cubos de Protección de Posición: Take Profit y Stop Loss
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Este archivo contiene una estrategia visual diseñada en StockSharp Designer, que demuestra el uso de cubos de protección de posición específicamente para operaciones de take profit y stop loss. La estrategia proporciona un ejemplo claro de cómo configurar y utilizar estas herramientas de trading críticas dentro del entorno Designer para gestionar y proteger posiciones de trading de manera efectiva.

## Descripción General

El esquema ilustrado en este archivo muestra una implementación sencilla de cubos de take profit y stop loss, permitiendo a los usuarios ver cómo estas características pueden aplicarse en escenarios de trading reales. Es un recurso educativo dirigido a ayudar a los traders a entender y aplicar técnicas de protección de posición usando la interfaz de arrastrar y soltar de Designer.

### Contenido

- **Cubo Take Profit**: Ejecuta automáticamente una orden de venta a un nivel de precio predeterminado para asegurar ganancias.
- **Cubo Stop Loss**: Ejecuta automáticamente una orden de venta a un nivel de precio predeterminado para limitar pérdidas potenciales.

Este esquema es particularmente útil para los traders que buscan mejorar sus estrategias con herramientas automatizadas de gestión de riesgo. El diseño visual y los elementos interactivos de Designer lo convierten en una excelente manera de aprender e implementar protecciones de trading.

### Uso

Cargue este archivo en StockSharp Designer para ver e interactuar con la estrategia. Los usuarios pueden modificar los parámetros directamente dentro de Designer para adaptarlos a sus requisitos específicos de trading y estrategias de gestión de riesgo. Este archivo sirve como ejemplo práctico para quienes están interesados en construir estrategias de trading robustas con mecanismos de seguridad automatizados.
