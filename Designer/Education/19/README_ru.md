# Схема стратегии пирамидирования
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Данная схема рассматривает стратегию пирамидирования, основанную на принципах торговой стратегии из предыдущего урока. Её суть — наращивание позиции в прибыльной сделке для максимизации потенциальной прибыли.

## Обзор

Пирамидирование — метод, при котором трейдеры увеличивают размер позиции, используя нереализованную прибыль от успешных сделок. Данная стратегия повышает потенциальную доходность без привлечения дополнительного капитала.

## Логика стратегии

- **Начальная позиция**: стратегия открывается с базовым размером позиции, определённым начальными торговыми условиями из предыдущей стратегии.
- **Поэтапное наращивание**: по мере движения сделки в нужном направлении и накопления нереализованной прибыли стратегия поэтапно увеличивает размер позиции при достижении заданных уровней прибыли.
- **Управление рисками**: каждая новая добавляемая позиция меньше предыдущей, что предотвращает чрезмерное накопление риска.

## Ключевые особенности

- **Масштабирование на основе прибыли**: стратегия увеличивает размер позиции соразмерно получаемой прибыли, согласовывая потенциальное вознаграждение с результативностью актива.
- **Контролируемая экспозиция**: ограничивая размер каждой дополнительной позиции, стратегия поддерживает контролируемую рыночную экспозицию и способствует общему управлению рисками.

## Применение и преимущества

- **Эффективность капитала**: пирамидирование позволяет трейдерам использовать успешные сделки для увеличения доходности без привлечения дополнительного капитала.
- **Динамическое управление размером позиции**: размер позиции динамически корректируется в зависимости от рыночных результатов, что позволяет трейдерам извлекать выгоду из благоприятной рыночной конъюнктуры.

## Исполнение

- **Критерии добавления позиций**: стратегия определяет чёткие критерии на основе результатов первоначальной сделки, прежде чем добавлять новые позиции.
- **Механизм фиксации прибыли**: предусмотрены механизмы фиксации прибыли — такие как скользящий стоп или целевой уровень прибыли — для защиты накопленной прибыли по мере роста размера позиции.

Схема стратегии пирамидирования, описанная здесь, использует базовую стратегию из предыдущего урока в качестве отправной точки для введения более продвинутых техник управления сделкой. Данный подход нацелен не только на максимизацию доходности, но и на интеграцию строгих средств контроля рисков, обеспечивая удержание потенциальных потерь под контролем по мере наращивания позиций.
