# 使用数学公式和表达式的策略示例
[English](README.md) | [Русский](README_ru.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

该文件包含使用 StockSharp Designer 平台设计的交易策略详细示例。该策略整合了数学表达式和公式，根据技术指标满足的特定条件执行交易。

## 策略概述

该图解演示了两种流行技术指标在交易决策中的应用：

### 布林带策略
- **买入条件**：当价格K线向上穿越布林带指标的上轨时，触发买入订单。
- **卖出条件**：当价格K线向下穿越布林带指标的下轨时，执行卖出订单。

### MACD指标策略
- **买入条件**：当MACD曲线从负值变为正值时，发起买入订单。
- **卖出条件**：当MACD曲线从正值变为负值时，触发卖出订单。

## 附加功能

- **可视化比较**：该图解支持两种策略结果的并排可视化比较。
- **结果导出**：包含将测试结果导出到文件以供进一步分析的功能。

该图解为理解和应用交易策略中的数学工具提供了实用框架，充分发挥了 Designer 平台的能力。
