# Пример стратегии с математическими формулами и выражениями
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Этот файл содержит подробный пример торговой стратегии, разработанной с использованием платформы Designer от StockSharp. Стратегия интегрирует математические выражения и формулы для совершения сделок на основе конкретных условий, выполняемых техническими индикаторами.

## Обзор стратегии

Эта схема демонстрирует применение двух популярных технических индикаторов для принятия торговых решений:

### Стратегия на основе полос Боллинджера (Bollinger Bands)
- **Условие покупки**: Ордер на покупку срабатывает, когда ценовая свеча пересекает верхнюю кривую индикатора Bollinger Bands снизу вверх.
- **Условие продажи**: Ордер на продажу исполняется, когда ценовая свеча пересекает нижнюю кривую индикатора Bollinger Bands сверху вниз.

### Стратегия на основе индикатора MACD
- **Условие покупки**: Инициирует ордер на покупку, когда кривая MACD меняет знак с отрицательного на положительный.
- **Условие продажи**: Генерирует ордер на продажу, когда кривая MACD меняет знак с положительного на отрицательный.

## Дополнительные функции

- **Визуальное сравнение**: Схема позволяет сравнивать результаты обеих стратегий в режиме «рядом».
- **Экспорт результатов**: Предусмотрена функция экспорта результатов тестирования в файл для дальнейшего анализа.

Эта схема предоставляет практическую основу для понимания и применения математических инструментов в торговых стратегиях с использованием возможностей платформы Designer.
