# 数学的な公式と式を使った戦略サンプル
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このファイルには、StockSharp の Designer プラットフォームを使用して設計されたトレーディング戦略の詳細なサンプルが含まれています。この戦略は、テクニカルインジケーターが満たす特定の条件に基づいて取引を実行するために、数学的な式と公式を組み合わせています。

## 戦略の概要

このスキーマは、トレーディング判断を行うための2つの人気のテクニカルインジケーターの適用を実演しています：

### ボリンジャーバンド戦略
- **買い条件**：価格のロウソク足が Bollinger Bands インジケーターの上限バンドを上抜けた場合に買い注文がトリガーされます。
- **売り条件**：価格のロウソク足が Bollinger Bands インジケーターの下限バンドを下抜けた場合に売り注文が実行されます。

### MACDインジケーター戦略
- **買い条件**：MACDカーブがマイナスからプラスへ符号が変わったときに買い注文を発注します。
- **売り条件**：MACDカーブがプラスからマイナスへ符号が変わったときに売り注文をトリガーします。

## 追加機能

- **ビジュアル比較**：スキーマにより、両戦略の結果を並べて視覚的に比較することができます。
- **結果のエクスポート**：テスト結果をファイルにエクスポートしてさらなる分析を行う機能が含まれています。

このスキーマは、Designer プラットフォームの機能を活用して、トレーディング戦略における数学的ツールを理解し適用するための実践的な枠組みを提供します。
