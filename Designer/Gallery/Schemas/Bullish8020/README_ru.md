## Обзор стратегии

Стратегия «Bullish8020» разработана для [StockSharp Designer](https://doc.stocksharp.com/topics/designer.html) с целью высокоточного использования специфических бычьих свечных паттернов. Стратегия направлена на выявление рыночных возможностей с сильным бычьим настроением, используя уникальный анализ паттернов в сочетании с объёмом и ценовым действием.

## Детали стратегии

### Обнаружение паттерна: Bullish8020

- **Описание**: Стратегия обнаруживает бычий сценарий, при котором [цена открытия](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/data_sources/candles.html) ниже цены закрытия, а размер тела свечи в четыре раза превышает сумму обеих теней, что указывает на сильное давление покупателей.
- **Свечной паттерн**: «Bullish8020» проверяет условие `(O < C) && (B >= 4*(BS+TS))`, где `O` — открытие, `C` — закрытие, `B` — размер тела, `BS` — нижняя тень, `TS` — верхняя тень.

### Исполнение сделок

- **Тип ордера**: Рыночный [ордер](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/positions/modify.html)
- **Вход**: Покупка при подтверждении паттерна «Bullish8020» ([паттерн](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/common/indicator.html)), сигнализирующего о потенциальном росте.
- **Стратегия выхода**:
  - **Стоп-лосс**: Установлен на 0.5% ниже точки входа для ограничения потенциальных убытков.
  - **Рыночные условия**: Сделки исполняются по текущим рыночным ценам для быстрого реагирования на распознавание паттерна.

### Управление рисками

- **Размер позиции**: Стратегия использует динамическое определение размера позиции на основе текущих рыночных условий и профиля риска трейдера.
- **Стратегия стоп-лосса**: Строгий [стоп-лосс](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/common/protect_position.html) применяется для защиты от непредвиденных разворотов рынка.

## Детали реализации

- **Платформа**: Реализована на платформе StockSharp с использованием её мощного API для обработки данных в режиме реального времени и исполнения ордеров.
- **Используемые индикаторы**: Сочетает распознавание свечных паттернов с анализом объёма для повышения точности торговых сигналов.

## Заключение

Стратегия «Bullish8020» предоставляет трейдерам надёжный инструмент для использования специфических бычьих паттернов на рынке. Она разработана для максимизации прибыли от сильных бычьих формаций при соблюдении строгих протоколов управления рисками для защиты инвестиций.
