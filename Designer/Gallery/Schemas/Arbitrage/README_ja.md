# BTC と ETH のペアトレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

BTC と ETH のペアトレーディング戦略は、2つの人気の高い暗号通貨、Bitcoin（BTC）と Ethereum（ETH）を取引するために設計されています。この暗号通貨アービトラージ戦略は、これら2つの資産間のアービトラージ機会を識別することに基づいており、BTC と ETH の価格差が一定のしきい値に達した瞬間を活用できるようにトレーダーを支援します。

![schema](schema.png)

この戦略は、一方の暗号通貨を買い入れながら同時に他方を売却するメカニズムを実装し、両者の価値の一時的な乖離から利益を得ることを目指しています。これにより、全体的な市場トレンドに縛られることなく、最小限の市場変動から収益機会を求める人々にとって魅力的な戦略となっています。

## インストール

この戦略を有効化して使用するには、StockSharp Designer がインストールされている必要があります。戦略は[ストラテジーギャラリー](https://doc.stocksharp.com/topics/designer/strategy_gallery.html)からダウンロードおよびインストールできます。これにより、個別のトレーダーの要件に応じた戦略の統合とカスタマイズが容易になります。

## パラメーター

- **資産 1**：ETHUSDT@BNB
- **資産 2**：BTCUSDT@BNB
- **しきい値**：0.02（絶対値）
- **取引量**：5000（絶対値）
- **スリッページ**：1.0（絶対値）
- **最大注文数**：3（絶対値）

## 仕組み

1. **価格データ収集**：戦略は Binance 取引所から BTC と ETH の価格データを収集します。
2. **価格計算**：BTC と ETH の価格比率を計算します。
3. **シグナル生成**：価格比率が定義されたしきい値を超えると、戦略は買いと売りのシグナルを生成します。
4. **注文実行**：戦略は割安な資産を買い、割高な資産を売る成行注文を実行します。
5. **利益計算**：実行された取引に基づいて利益を計算し、さらなる機会を求めて市場を監視します。

## テスト

実際の市場に適用する前に、歴史的データで戦略をテストして有効性と潜在的リスクを評価することが重要です。これにより、価格乖離のしきい値と資金管理の最適なパラメーターを特定できます。
