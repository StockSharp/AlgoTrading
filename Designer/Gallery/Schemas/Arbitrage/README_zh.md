# BTC 与 ETH 配对交易策略
[English](README.md) | [Русский](README_ru.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

## 概述

BTC 与 ETH 配对交易策略专为交易两种热门加密货币——比特币（BTC）和以太坊（ETH）而设计。该加密货币套利策略基于识别这两种资产之间的套利机会，使交易者能够在 BTC 与 ETH 之间的价差达到一定阈值时加以利用。

![schema](schema.png)

该策略实现了买入一种加密货币同时卖出另一种的机制，旨在从两者价值的临时差异中获利。这使该策略对于那些希望从细微市场波动中获利、而不依赖整体市场趋势的人颇具吸引力。

## 安装

要激活和使用该策略，必须安装 StockSharp Designer。该策略可从[策略库](https://doc.stocksharp.com/topics/designer/strategy_gallery.html)下载和安装，便于根据个人交易者需求进行集成和定制。

## 参数

- **资产 1**：ETHUSDT@BNB
- **资产 2**：BTCUSDT@BNB
- **阈值**：0.02（绝对值）
- **交易量**：5000（绝对值）
- **滑点**：1.0（绝对值）
- **最大订单数**：3（绝对值）

## 工作原理

1. **价格数据收集**：策略从 Binance 交易所收集 BTC 和 ETH 的价格数据。
2. **价格计算**：计算 BTC 与 ETH 之间的价格比率。
3. **信号生成**：当价格比率超过设定阈值时，策略生成买入和卖出信号。
4. **订单执行**：策略执行市价订单，买入被低估的资产，卖出被高估的资产。
5. **利润计算**：根据已执行的交易计算利润，并持续监控市场以寻找进一步机会。

## 测试
