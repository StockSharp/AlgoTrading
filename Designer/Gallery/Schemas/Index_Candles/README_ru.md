# Диаграмма создания составного индекса из нескольких серий свечей
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Данный файл диаграммы демонстрирует стратегию создания составного индекса из серий свечей различных финансовых инструментов с использованием галереи стратегий платформы Designer. Стратегия агрегирует данные нескольких ценных бумаг для формирования единого индекса, который можно применять для оценки общих рыночных настроений или производительности отдельного сектора.

![schema](schema.png)

## Обзор стратегии

Стратегия предполагает объединение ценовых данных нескольких ценных бумаг в единый [индекс](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/data_sources/index.html). Как правило, при этом используются методы нормализации или взвешивания, обеспечивающие пропорциональный вклад каждой ценной бумаги в итоговое значение индекса.

## Компоненты диаграммы

- **Узлы сбора данных**: отвечают за получение [данных свечей](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/data_sources/candles.html) по каждой выбранной ценной бумаге.
- **Узлы нормализации**: применяют нормализацию к данным свечей, обеспечивая равномерный вклад в [расчёт итогового индекса](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/data_sources/index.html) и нивелируя влияние различных ценовых масштабов.
- **Узлы взвешивания**: назначают веса каждой ценной бумаге на основе заранее определённых критериев, таких как рыночная капитализация или историческая волатильность.
- **Узел расчёта индекса**: агрегирует нормализованные и взвешенные ценовые данные для вычисления итогового значения индекса.

## Точки входа и выхода

- **Точки входа**: как правило, традиционных точек входа нет, поскольку данная стратегия не предполагает непосредственного принятия торговых решений.
- **Выходные данные**: основным результатом является значение индекса в реальном времени, отражающее совокупное движение включённых в него ценных бумаг.

## Применение

Трейдеры и аналитики могут использовать эту диаграмму для:
- мониторинга общей динамики конкретного сектора или рынка путём создания пользовательского индекса;
- сравнения отдельных ценных бумаг с более широким рыночным индексом для выявления опережающей или отстающей динамики;
- использования пользовательского индекса в качестве эталона для оценки эффективности портфеля.

## Образовательная ценность

Данная диаграмма стратегии особенно ценна в образовательных целях и даёт представление о:
- механике расчёта индексов, а также о значимости нормализации и взвешивания данных в финансовом анализе;
- применении объединённых данных из нескольких источников для создания значимых финансовых показателей.

Пользователи могут импортировать эту диаграмму в платформу Designer, чтобы изучить и изменить подход, адаптировать его к другим наборам ценных бумаг или усложнить методологию расчёта индекса.

Этот файл является частью обширной коллекции стратегий, доступных в платформе Designer, направленной на углубление понимания пользователями агрегирования финансовых данных и построения индексов.
