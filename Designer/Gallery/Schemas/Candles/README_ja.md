# データソースとチャートキューブの基本的な使い方ダイアグラム
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このダイアグラムは、Designer プラットフォーム内で「Candles」データソースと「Chart」キューブを使用する方法を簡単に示しています。市場データを取得してグラフ形式で視覚化する基本的な方法をユーザーが理解できるよう設計されています。

![schema](schema.png)

## 概要

このダイアグラムは、特定の金融商品のローソク足データを取得してチャートに表示するために必要な基本的なセットアップを示しています。Designer を初めて使用するユーザーや、シンプルなデータ視覚化手法から始めたいユーザーの基礎的な例として機能します。

## ダイアグラムのコンポーネント

- **Candles データソース**：選択した金融商品から[ローソク足データ](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/data_sources/candles.html)を取得する主要ノードです。ユーザーは銘柄、データ範囲、ローソク足の時間足（例：1分足、5分足）を指定できます。
- **チャートキューブ**：取得したデータをグラフィカルインターフェースに[プロット](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer/elements/common/chart.html)するために使用されるノードです。始値、高値、安値、終値などのローソク足の様々な属性を表示できます。

## 機能

- **データ取得**：ダイアグラムはまず、Candles データソースキューブで指定されたパラメーターを使用してローソク足データを取得します。
- **データ視覚化**：取得したデータはチャートキューブに渡され、Designer 環境内のチャートにローソク足がプロットされます。

## ユースケース

このダイアグラムは以下に特に有用です：
- Designer でのデータ取得と視覚化の設定方法を学ぶ新規ユーザー。
- 分析のために市場データを素早く視覚化したいトレーダーやアナリスト。
- 教育目的で、プラットフォーム内のデータソースノードと視覚化ツール間の基本的な相互作用を示す場合。

## 実際の応用

この基本的なセットアップを理解して活用することで、ユーザーは以下のことができます：
- リアルタイムまたは過去データの分析のために市場データの視覚的表現を素早く設定する。
- Designer で利用可能な追加の分析ツールやインジケーターを組み込むことで基本ダイアグラムを拡張する。
- より複雑な取引戦略やデータ研究の構成要素としてチャートを活用する。

このダイアグラムは、データの取り扱いと視覚化におけるユーザーの習熟度を高めることを目的とした、Designer プラットフォームで利用可能な広範な教育リソースの一部です。
