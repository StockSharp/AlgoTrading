# Каталог исходного кода стратегий
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Этот каталог содержит схематические представления торговых стратегий, доступных в разделе [Галерея стратегий](https://doc.stocksharp.com/topics/designer/strategy_gallery.html) платформы [Designer](https://stocksharp.com/store/strategy-designer/). Файлы с исходным кодом стратегий хранятся в каталоге [API](../../API), откуда их можно загрузить как самостоятельные программы через API StockSharp или импортировать в Designer.

## Обзор

Галерея стратегий в [Designer](https://stocksharp.com/store/strategy-designer/) предлагает разнообразную коллекцию торговых стратегий, разработанных сообществом и экспертами StockSharp. Этот каталог служит репозиторием схематических версий данных стратегий, позволяя пользователям исследовать, изменять и совершенствовать их в соответствии со своими торговыми потребностями. Реализации исходного кода доступны в каталоге [API](../../API).

## Содержание

- **Схематические стратегии**: Диаграммные представления торговых стратегий, которые [наглядно отображают](https://doc.stocksharp.com/topics/designer/strategies/using_visual_designer.html) поток и условия торговых операций. Они идеально подходят для пользователей, предпочитающих визуальный подход к разработке стратегий.
- **Стратегии на C#**: Файлы исходного кода стратегий расположены в каталоге [API](../../API). Их можно загружать как самостоятельные программы через API StockSharp или импортировать в Designer для дальнейшей настройки.

## Использование

Пользователи могут скачивать эти стратегии для изучения их внутренней логики, [тестирования](https://doc.stocksharp.com/topics/designer/backtesting/getting_started.html) в симуляционных средах или применения в реальной торговле. Каждую стратегию можно импортировать в платформу Designer, где её можно дополнительно настроить или использовать в качестве основы для разработки новых стратегий.

## Участие в разработке

Хотя в этом каталоге в основном размещены стратегии, разработанные для платформы Designer, вклад сообщества приветствуется. Пользователи, разработавшие собственные стратегии и желающие поделиться ими с сообществом StockSharp, могут предложить свои схематические файлы для включения сюда, тогда как вклад в виде исходного кода следует добавлять в каталог [API](../../API).

## Образовательная ценность

- **Инструмент обучения**: Начинающие трейдеры и разработчики могут изучать структуру и методы программирования, использованные в этих стратегиях.
- **База для инноваций**: Опытные пользователи могут использовать эти стратегии как основу для создания более сложных алгоритмов, адаптированных к конкретным рыночным условиям.

Этот каталог служит не только ресурсом торговых стратегий, но и центром сообщества для обмена знаниями и содействия инновациям в области алгоритмической торговли.
