# Стратегия Renko Trend Reversal V2
[English](README.md) | [中文](README_cn.md)

Renko Trend Reversal V2 входит, когда открытие ренко пересекает закрытие. Использует кирпичи ренко на основе ATR и выставляет стоп-лосс и тейк-профит. Шорты можно отключить.

## Детали

- **Критерий входа**: пересечение открытия и закрытия ренко с учётом времени
- **Длин/Шорт**: обе стороны (шорты опциональны)
- **Критерий выхода**: стоп-лосс или тейк-профит
- **Стопы**: да
- **Значения по умолчанию**:
  - `RenkoAtrLength` = 10
  - `StopLossPct` = 3
  - `TakeProfitPct` = 20
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Renko
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
