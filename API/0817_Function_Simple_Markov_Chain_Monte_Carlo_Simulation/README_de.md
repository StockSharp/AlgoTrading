# Einfache Markov-Kette Monte-Carlo-Simulation
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demonstriert eine einfache Markov-Kette-Monte-Carlo-Simulation (MCMC), die einen zufälligen Pfad erzeugt.

## Details
- **Zweck**: Zufallspfaderzeugung über einfache MCMC.
- **Indikatoren**: keine.
- **Long/Short**: nicht anwendbar.
- **Stops**: keine.
