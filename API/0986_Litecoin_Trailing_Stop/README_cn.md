# Litecoin 追踪止损策略
[English](README.md) | [Русский](README_ru.md)

**Litecoin 追踪止损策略** 使用考夫曼自适应移动平均线 (KAMA) 识别多空趋势。KAMA 上升时做多，下降时做空。经过可配置的延迟后，基于百分比的追踪止损用于保护利润。

## 细节
- **入场条件**：KAMA 方向并在入场之间设置冷却时间。
- **多/空**：双向。
- **离场条件**：追踪止损。
- **止损**：延迟后的百分比追踪止损。
- **默认值**：
  - `KamaLength = 50`
  - `BarsBetweenEntries = 30`
  - `TrailingStopPercent = 12`
  - `DelayBars = 50`
- **过滤器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：KAMA
  - 止损：追踪止损
  - 复杂度：基础
  - 时间框架：中期
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险水平：中等
