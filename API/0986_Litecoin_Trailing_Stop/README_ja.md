# Litecoin トレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Litecoin トレーリングストップ戦略**は、カウフマン適応移動平均 (KAMA) を使用して強気・弱気トレンドを検出します。KAMAが上昇しているときはロングポジション、下落しているときはショートポジションをオープンします。設定可能な遅延後、パーセンテージベースのトレーリングストップが利益を保護します。

## 詳細
- **エントリー条件**: エントリー間のクールダウンを伴うKAMAの傾き。
- **ロング/ショート**: 両方向。
- **エグジット条件**: トレーリングストップ。
- **ストップ**: 遅延後のトレーリングストップ。
- **デフォルト値**:
  - `KamaLength = 50`
  - `BarsBetweenEntries = 30`
  - `TrailingStopPercent = 12`
  - `DelayBars = 50`
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: KAMA
  - ストップ: トレーリング
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
