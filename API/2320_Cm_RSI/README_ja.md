# cm RSI戦略

## 概要

この戦略はMetaTrader 4エキスパート「cm_RSI」の直接ポートです。相対力指数（RSI）インジケーターを使用してモメンタムリバーサルを捉えます。

アルゴリズムはローソク足の始値から計算されたRSI値を監視します。RSIが設定可能な*買いレベル*を下回った後に上回ったときにロングポジションを開きます。RSIが設定可能な*売りレベル*を上回った後に下回ったときにショートポジションを開きます。各取引は価格ポイントで表された固定のテイクプロフィットとストップロス値で保護されます。

## 戦略ロジック

1. ユーザーが定義した期間でローソク足の始値を使用してRSIを計算する。
2. 前のRSI値が買いレベルを下回り、現在の値が上にクロスした場合、ロング成行ポジションを開く。
3. 前のRSI値が売りレベルを上回り、現在の値が下にクロスした場合、ショート成行ポジションを開く。
4. 各取引は同じ設定可能なボリュームを使用し、ストップロスとテイクプロフィット注文で保護される。

## パラメーター

| 名前 | 説明 |
|------|------|
| `RsiPeriod` | RSI計算期間。 |
| `BuyLevel` | ロングエントリーを発動するRSIレベル。 |
| `SellLevel` | ショートエントリーを発動するRSIレベル。 |
| `TakeProfit` | 絶対価格ポイントでのテイクプロフィット。 |
| `StopLoss` | 絶対価格ポイントでのストップロス。 |
| `OrderVolume` | 各取引に適用されるボリューム。 |
| `CandleType` | 計算に使用するローソク足タイプ。 |

## 注意事項

- 戦略は確定したローソク足のみを処理する。
- 常に1つのオープンポジションのみを保持する。
- `StartProtection`はストップロスとテイクプロフィット注文を自動的に管理するために使用される。
