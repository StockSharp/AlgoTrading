# Estratégia Double Vegas SuperTrend Aprimorada
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina dois indicadores SuperTrend cujos multiplicadores se adaptam à volatilidade do canal Vegas. Uma posição comprada é aberta quando ambas as tendências são altistas e uma posição vendida quando ambas são baixistas. Períodos de manutenção opcionais e proteções de take-profit ou stop-loss são suportados.

## Parâmetros
- Tipo de candle
- Período ATR 1
- Janela Vegas 1
- Multiplicador 1
- Ajuste de volatilidade 1
- Período ATR 2
- Janela Vegas 2
- Multiplicador 2
- Ajuste de volatilidade 2
- Direção de operação
- Usar dias de manutenção
- Dias de manutenção
- Condição TP/SL
- Take Profit (%)
- Stop Loss (%)
