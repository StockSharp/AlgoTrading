# Verbesserte Doppel-Vegas-SuperTrend-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert zwei SuperTrend-Indikatoren, deren Multiplikatoren sich an die Volatilität des Vegas-Kanals anpassen. Eine Long-Position wird eröffnet, wenn beide Trends bullish sind, und eine Short-Position, wenn beide bearish sind. Optionale Haltedauern sowie Take-Profit- oder Stop-Loss-Schutz werden unterstützt.

## Parameter
- Kerzentyp
- ATR-Periode 1
- Vegas-Fenster 1
- Multiplikator 1
- Volatilitätsanpassung 1
- ATR-Periode 2
- Vegas-Fenster 2
- Multiplikator 2
- Volatilitätsanpassung 2
- Handelsrichtung
- Haltetage verwenden
- Haltetage
- TP/SL-Bedingung
- Take Profit (%)
- Stop Loss (%)
