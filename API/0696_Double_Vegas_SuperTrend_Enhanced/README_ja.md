# ダブル Vegas SuperTrend 強化戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Vegasチャンネルのボラティリティに適応する乗数を持つ2つのSuperTrendインジケーターを組み合わせます。両方のトレンドが強気の時にロングポジションを開き、両方が弱気の時にショートポジションを開きます。オプションの保有期間とテイクプロフィットまたはストップロスの保護がサポートされています。

## パラメーター
- ローソク足タイプ
- ATR期間 1
- Vegasウィンドウ 1
- 乗数 1
- ボラティリティ調整 1
- ATR期間 2
- Vegasウィンドウ 2
- 乗数 2
- ボラティリティ調整 2
- 取引方向
- 保有日数を使用
- 保有日数
- TP/SL条件
- テイクプロフィット (%)
- ストップロス (%)
