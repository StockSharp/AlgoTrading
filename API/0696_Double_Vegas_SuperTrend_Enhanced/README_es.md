# Estrategia Double Vegas SuperTrend Mejorada
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia combina dos indicadores SuperTrend cuyos multiplicadores se adaptan a la volatilidad del canal Vegas. Se abre una posición larga cuando ambas tendencias son alcistas y una posición corta cuando ambas son bajistas. Se admiten períodos de mantenimiento opcionales y protecciones de take-profit o stop-loss.

## Parámetros
- Tipo de vela
- Período ATR 1
- Ventana Vegas 1
- Multiplicador 1
- Ajuste de volatilidad 1
- Período ATR 2
- Ventana Vegas 2
- Multiplicador 2
- Ajuste de volatilidad 2
- Dirección de operación
- Usar días de mantenimiento
- Días de mantenimiento
- Condición TP/SL
- Take Profit (%)
- Stop Loss (%)
