# Стратегия Gold Trade Setup
[English](README.md) | [中文](README_cn.md)

Стратегия основана на адаптивной скользящей средней Кауфмана и индикаторе SuperTrend.
Продаёт, когда AMA растёт и SuperTrend переключается в восходящий тренд.
Покупает, когда AMA падает и SuperTrend переключается в нисходящий тренд.

## Детали

- **Критерии входа**: направление AMA с переключением SuperTrend.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: фиксированные уровни цели и стопа.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `AmaLength` = 14
  - `FastLength` = 2
  - `SlowLength` = 30
  - `AtrPeriod` = 10
  - `Factor` = 3.0
  - `TargetMultiplier` = 3.0
  - `RiskMultiplier` = 1.0
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: KAMA, SuperTrend
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
