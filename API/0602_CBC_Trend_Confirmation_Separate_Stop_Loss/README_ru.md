# Стратегия CBC с подтверждением тренда и отдельным стоп-лоссом
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует состояние смены цвета свечи (CBC) для фиксации пробоя предыдущего максимума или минимума. Вход выполняется при подтверждении тренда через EMA и VWAP и ограничен заданным временным окном. Выход использует тейк-профит на основе ATR и стоп-лосс по экстремумам предыдущей свечи.

## Подробности

- **Условия входа**: смена CBC, опциональный фильтр сильных сигналов, сравнение медленной EMA с VWAP, торговля в заданные часы.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: тейк-профит по множителю ATR, стоп-лосс на пробой прошлого экстремума.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `AtrLength` = 14
  - `ProfitTargetMultiplier` = 1.0m
  - `StrongFlipsOnly` = true
  - `EntryStartHour` = 10
  - `EntryEndHour` = 15
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA, VWAP, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
