# Exp 3XMA Ichimoku 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MQLエキスパート `exp_3xma_ishimoku` の変換版です。短縮された期間のIchimokuインジケーターを使用し、クラウドブレイクアウトに逆らって行動します。

Kijun線がIchimokuクラウドの境界と比較されます。Kijunがクラウドの上から中へ落ちた場合、戦略はショートポジションをクローズし、買いが許可されていればロングポジションを開きます。Kijunがクラウドの下から中へ上昇した場合、ロングポジションがクローズされ、ショートポジションが開かれる場合があります。

分析のデフォルト時間軸は4時間ローソク足です。

## パラメーター
- **Tenkan Period** – Tenkan-sen線の長さ。
- **Kijun Period** – Kijun-sen線の長さ。
- **Senkou Span B Period** – 第2 Senkou スパンの期間。
- **Allow Buy** – ロングポジションの開設を有効にする。
- **Allow Sell** – ショートポジションの開設を有効にする。
- **Candle Type** – インジケーター計算に使用するローソク足シリーズ。

## 仕組み
1. 選択したローソク足シリーズをサブスクライブし、Ichimokuインジケーターをバインドします。
2. 完成済みのローソク足のみを処理します。
3. Kijun線がクラウドの境界をクロスするタイミングを検出します。
4. 反対ポジションをクローズし、許可されていればシグナル方向に新しいポジションを開きます。
