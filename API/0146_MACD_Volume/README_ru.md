# Стратегия MACD Volume
[English](README.md) | [中文](README_cn.md)

Стратегия совмещает индикатор MACD (Moving Average Convergence Divergence) с подтверждением объёмом. Вход выполняется, когда линия MACD пересекает сигнальную линию и это подтверждается ростом объёма.

Пересечения MACD фильтруются увеличением объёма для подтверждения импульса. Покупки происходят на бычьем пересечении с растущим объёмом, продажи — наоборот.

Стратегия может быть полезна трейдерам, отслеживающим всплески объёма. Риск ограничивается стопом, основанным на ATR.

## Подробности

- **Условия входа**:
  - Лонг: `MACD crosses above Signal && Volume > AvgVolume * VolumeMultiplier`
  - Шорт: `MACD crosses below Signal && Volume > AvgVolume * VolumeMultiplier`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Пересечение MACD в обратную сторону
- **Стопы**: процентные по параметру `StopLossPercent`
- **Значения по умолчанию**:
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `VolumePeriod` = 20
  - `VolumeMultiplier` = 1.5m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: MACD, Объём
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
