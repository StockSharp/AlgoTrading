# Milestone トレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMilestone 22.5エキスパートアドバイザーのStockSharpポートです。2本の平滑化移動平均線とボラティリティおよびスパイクフィルターを組み合わせ、トレンド内の押し目を取引します。ローソク足が前のバーの極値を突破し、速い移動平均線がその動きを確認すると、主要トレンドの方向にポジションが開かれます。ATRは静かな市場での取引を防ぎ、大きなローソク足の実体はスパイクとして扱われます。

オリジナルのMQL版のバックテストは主要な通貨ペアで良好なパフォーマンスを示しています。C#への翻訳は明確さを重視し、エントリーとイグジットには成行注文のみを使用します。

## 詳細

- **エントリー条件**:
  - トレンド強度が`MinTrend`と`MaxTrend`の間にある。
  - ローソク足が前の高値または安値を突破し、速いSMAが確認する。
  - ATRが`MinRange`より上で、ローソク足の実体が`CandleSpike`より小さい。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対のシグナルがポジションを閉じる。
- **ストップ**: 実装されていない；反対のシグナルがストップとして機能する。
- **デフォルト値**:
  - `SlowMaPeriod` = 120
  - `FastMaPeriod` = 30
  - `AtrPeriod` = 14
  - `MinTrend` = 10
  - `MaxTrend` = 100
  - `MinRange` = 5
  - `CandleSpike` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: SMA、ATR
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
