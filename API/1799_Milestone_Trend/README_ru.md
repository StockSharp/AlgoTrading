# Стратегия Milestone Trend
[English](README.md) | [中文](README_cn.md)

Стратегия представляет собой порт на StockSharp эксперта Milestone 22.5. Она торгует откаты в направлении тренда, комбинируя две сглаженные скользящие средние с фильтрами волатильности и свечных "шпилек". Когда свеча пробивает экстремум предыдущего бара и быстрая средняя подтверждает движение, открывается позиция по тренду. ATR предотвращает работу на тихом рынке, а большие тела свечей рассматриваются как всплески и игнорируются.

Тесты оригинальной версии MQL показывают хорошую эффективность на основных валютных парах. Перевод на C# ориентирован на простоту и использует только рыночные ордера для входа и выхода.

## Детали

- **Критерии входа**:
  - Сила тренда между `MinTrend` и `MaxTrend`.
  - Свеча пробивает предыдущий максимум или минимум, а быстрая SMA подтверждает сигнал.
  - ATR выше `MinRange`, тело свечи меньше `CandleSpike`.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: противоположный сигнал закрывает позицию.
- **Стопы**: не реализованы, противоположный сигнал служит стопом.
- **Значения по умолчанию**:
  - `SlowMaPeriod` = 120
  - `FastMaPeriod` = 30
  - `AtrPeriod` = 14
  - `MinTrend` = 10
  - `MaxTrend` = 100
  - `MinRange` = 5
  - `CandleSpike` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA, ATR
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
