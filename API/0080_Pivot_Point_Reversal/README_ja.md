# Pivot Pointリバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

日次Pivot Pointとそのサポートおよびレジスタンスレベルは、イントラデイの価格動向の転換点として機能することが多いです。この戦略は前日の高値、安値、終値からクラシックなフロアトレーダーのピボットを計算し、その後S1またはR1からバウンスするローソク足を探します。

テストでは年平均リターンが約127%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

価格がサポートレベルS1に近づき強気のローソク足を形成すると、ロングポジションが取られます。価格がレジスタンスレベルR1をテストし弱気のローソク足を印刷すると、ショートが開かれます。中央ピボットに達するか保護ストップが発動すると取引が決済されます。

この手法は各取引日の開始時に新しいピボット計算でリセットされ、明確なイントラデイレンジのあるセッションに適しています。

## 詳細

- **エントリー条件**: S1付近の強気のローソク足またはR1付近の弱気のローソク足。
- **ロング/ショート**: 両方。
- **エグジット条件**: 価格が中央ピボットを越えるか、ストップロス。
- **ストップ**: はい、パーセントベース。
- **デフォルト値**:
  - `CandleType` = 5 minute
  - `StopLossPercent` = 2
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: Pivot Points
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
