# Nunchucks Indicator Strategy
[Русский](README_ru.md) | [中文](README_cn.md)

The Nunchucks indicator highlights new highs and lows by drawing extra candles extending beyond the previous extremums. It is a visual tool that helps identify breakout tails without producing trade signals.

## Details

- **Entry Criteria**: None. The strategy is purely visual.
- **Long/Short**: N/A
- **Exit Criteria**: None.
- **Stops**: No.
- **Default Values**:
  - `Show Zero Tails` = false
  - `Candle Type` = 1-minute
- **Filters**:

  - Complexity: Simple
