# Nunchucks 指标策略
[English](README.md) | [Русский](README_ru.md)

Nunchucks 指标通过在前高或前低之外绘制额外的蜡烛来突出新的高点和低点。它是一种用于识别突破尾部的可视化工具，不会生成交易信号。

## 详情

- **入场条件**：无，此策略仅用于可视化。
- **多/空方向**：无
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `Show Zero Tails` = false
  - `Candle Type` = 1 分钟
- **筛选条件**：
  - 类型：指标
  - 方向：无
  - 指标：价格行为
  - 止损：无
  - 复杂度：简单
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：无
