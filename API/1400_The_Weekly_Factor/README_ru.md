# Стратегия The Weekly Factor
[English](README.md) | [中文](README_cn.md)

Реализует паттерн Weekly Factor, описанный Андреа Унгером. Стратегия торгует пробои максимума или минимума сессии при сжатии пятидневного диапазона.

## Подробности
- **Условия входа**: После начала сессии при выполнении Weekly Factor пробой хая -> лонг, пробой лоя -> шорт.
- **Направление**: Оба.
- **Условия выхода**: Закрытие в начале новой сессии или через два дня при прибыли.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `RangeFilter` = 0.5
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Weekly factor
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: 15m
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
