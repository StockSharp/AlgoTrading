# 週間ファクター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Andrea Ungerが説明した週間ファクターパターンを実装します。5日間のレンジに圧縮が見られるとき、セッション高値または安値のブレイクアウトを取引する戦略です。

## 詳細
- **エントリー条件**: セッション開始後、週間ファクター条件が真で価格がセッション高値を上抜け -> ロング、セッション安値を下抜け -> ショート。
- **ロング/ショート**: 両方。
- **エグジット条件**: 新しいセッション開始時または2日間の利益ポジション後にクローズ。
- **ストップ**: なし。
- **デフォルト値**:
  - `RangeFilter` = 0.5
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Weekly factor
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 15m
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
