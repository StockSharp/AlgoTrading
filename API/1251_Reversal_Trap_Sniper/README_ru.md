# Стратегия Reversal Trap Sniper
[English](README.md) | [中文](README_cn.md)

Reversal Trap Sniper ищет RSI-ловушки, когда импульс разворачивается, а цена продолжает движение.
Покупает после краткого выхода в зону перекупленности с закрытием выше, продаёт после выхода из перепроданности с закрытием ниже.

## Детали

- **Критерий входа**: RSI был перекуплен/перепродан три бара назад, текущий RSI пересекает уровень обратно, а цена движется в том же направлении
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: ATR-стоп или тейк-профит либо ограничение по барам
- **Стопы**: на основе ATR
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `RsiOverbought` = 70
  - `RsiOversold` = 30
  - `RiskReward` = 2
  - `MaxBars` = 30
  - `AtrLength` = 14
- **Фильтры**:
  - Категория: Разворот
  - Направление: Обе стороны
  - Индикаторы: RSI, ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
