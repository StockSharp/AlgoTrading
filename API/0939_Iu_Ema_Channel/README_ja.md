# IU EMAチャネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradingViewスクリプト「IU EMA Channel Strategy」から変換されました。この戦略は、高値と安値から構築されたEMAチャネルを価格が突破したときに取引します。ストップロスは前のローソク足の極値に設定され、テイクプロフィットはリスク・リワード比を使用して計算されます。

## 詳細

- **エントリー条件**: 終値が高値EMAを上抜けでロング、安値EMAを下抜けでショート。
- **ロング/ショート**: 両方。
- **エグジット条件**: 前のローソク足極値でのストップロスまたはリスク・リワード比によるテイクプロフィット。
- **ストップ**: はい、固定ストップと目標。
- **デフォルト値**:
  - `EmaLength` = 100
  - `RiskToReward` = 2
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 可変
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
