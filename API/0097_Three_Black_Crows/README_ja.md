# Three Black Crows Strategy
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

三羽烏（さんばがらす）は三白兵の弱気版であり、上昇後に3本の長い陰線で構成されます。このパターンは各終値がセッションの安値付近に落ち、売り手が支配権を握ったことを示唆します。

テストでは平均年間リターンが約178%であることが示されています。株式市場で最もパフォーマンスが優れています。

この戦略は3本目のカラスが現れたらショートポジションを開始し、モメンタムが下方向に継続することを期待します。パターンが抵抗線で形成された場合、他のシステムで開かれたロングを終了するためにも使用できます。

リスクはパターン高値の上に置かれたタイトなパーセントストップで管理され、価格がそのレベルを上回って引けた場合に取引が終了します。

## 詳細

- **エントリー条件**: パターン一致
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15分
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick

  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
