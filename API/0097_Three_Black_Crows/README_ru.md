# Стратегия «Три чёрных ворона»
[English](README.md) | [中文](README_zh.md)

«Три чёрных ворона» — медвежий аналог паттерна «Три белых солдата», состоящий из трёх длинных красных свеч после роста. Паттерн указывает, что продавцы захватили контроль, поскольку каждая свеча закрывается возле минимума сессии.

Стратегия открывает короткую позицию после появления третьего ворона, ожидая продолжения движения вниз. Её также можно использовать для выхода из лонгов, открытых другими системами, если паттерн образуется у сопротивления.

Риск ограничивается жёстким процентным стопом над максимумом паттерна, сделки закрываются, если цена закрывается выше этого уровня.

## Детали

- **Условия входа**: совпадение паттерна
- **Лонг/шорт**: оба направления
- **Условия выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, на процентной основе
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 178%. Стратегию лучше запускать на фондовом рынке.
