# ADXブレイクアウト
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ADXブレイクアウト戦略はADXの強い拡大を監視します。読み取り値が典型的な範囲を超えてジャンプすると、価格は新しい動きを始めることが多いです。

テストでは平均年間リターンが約97%であることが示されています。暗号資産市場で最も良いパフォーマンスを発揮します。

最近のデータと偏差乗数から派生したバンドをインジケーターが突破したときにポジションが開かれます。ストップ付きのロングとショートの取引が可能です。

このシステムは初期のブレイクアウトを求めるモメンタムトレーダーに適しています。ADXが平均値に戻ったら取引を決済します。デフォルト値は`ADXPeriod` = 14から始まります。

## 詳細

- **エントリー条件**: インジケーターが偏差乗数分だけ平均を超える。
- **ロング/ショート**: 両方向。
- **エグジット条件**: インジケーターが平均に回帰する。
- **ストップ**: あり。
- **デフォルト値**:
  - `ADXPeriod` = 14
  - `AvgPeriod` = 20
  - `Multiplier` = 0.1m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `StopLoss` = 2.0m
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: ADX
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
