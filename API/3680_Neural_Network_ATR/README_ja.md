# ニューラル ネットワーク ATR 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略は、軽量ニューラル ネットワークを組み合わせることにより、「Neurotest」エキスパート アドバイザを複製します。
StockSharp 内の ATR ベースの資金管理を備えたネットワーク層。モデルが消費するのは、
最新の完成した M15 キャンドルを取得し、それを 5 つの正規化された特徴に変換します。
終値の勢い、日中のレンジ、ローソク足の実体、出来高の拡大とボラティリティ（ATRから
価格比）。シグモイド出力を持つ単一の隠れ層が確率スコアを生成します
これは動的学習率によってスケールされます。スコアはユーザー定義と比較されます
ポジションをオープンまたは反転するためのしきい値を売買します。

## 取引ルール

1. 15 分のローソク足 (構成可能) をサブスクライブし、同じ期間の ATR を計算します。
2. 以前のローソク足と現在の完成したローソク足から 5 つの正規化された特徴を構築します
キャンドルを作成し、ニューラル ネットワークを評価します。
3. 調整された予測が買い閾値を上回っており、現在のポジションが
長くない場合は、ロング取引を開始します（必要に応じてショートエクスポージャーを終了します）。
4. 調整された予測が売り閾値を下回っており、現在のポジションが
ショートではない場合は、ショートトレードを入力してください。
5. 各エントリには、ATR ベースのストップロス注文とテイクプロフィット注文が添付されます。 ATR が形成されない場合、
ポイント単位のフォールバック距離が使用されます。
6. 現在のスプレッドが設定された制限を超える場合、ローソク足は無視されます。

## リスク管理

- ポジション サイズはポートフォリオの資本と ATR のストップ距離から計算されます。
ストップでの損失は資本の `Max Risk %` に相当します。
- 保護注文では、設定可能なリスク対報酬の乗数が使用されます。
- 毎日または合計ドローダウンが制限を超えると、取引は自動的に停止します。
- ペナルティ システムは、毎日
利益目標は達成されず、次の取引日まで将来のシグナルが弱まります。

## パラメーター

| パラメータ | 説明 |
|-----------|-------------|
| **最大リスク %** | 株式に対する取引ごとの最大リスクの割合。 |
| **1 日あたりの損失 %** | 取引を停止する毎日のドローダウンしきい値。 |
| **総損失率** | 取引を停止するグローバルドローダウンしきい値。 |
| **1 日あたりの利益 %** | ペナルティがスキップされる前の毎日の利益目標。 |
| **学習率** | ニューラル出力に適用されるスケーリング係数。 |
| **隠しレイヤー** | 隠れ層内のニューロンの数。 |
| **買いのしきい値/売りのしきい値** | ロングエントリーとショートエントリーのトリガーレベル。 |
| **キャンドルタイプ** | シグナルに使用されるローソクの種類と時間枠。 |
| **ATR 期間** | ATR インジケーターの期間。 |
| **最大スプレッド** | 価格ステップにおける最大許容スプレッド。 |
| **リスクリワード** | 停止距離に対するテイクプロフィット乗数。 |
| **フォールバック停止** | ATR が利用できない場合の停止距離 (ポイント単位)。 |

## 注意事項

- 各決定の前に買値/売値スプレッドを監視するには、レベル 1 サブスクリプションが必要です。
- ニューラル ネットワークの重みはランダムに初期化されますが、決定的です (シード 42)。の
学習率変調は、元の MQL エキスパートの適応動作をエミュレートします。
- 接続されたポートフォリオが `CurrentValue`、`StepPrice`、およびボリューム制限を提供していることを確認してください
ポジションのサイジングと保護注文が正しく機能するようにします。
