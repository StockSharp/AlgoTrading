# RedK Slow Smooth WMA 策略
[English](README.md) | [Русский](README_ru.md)

使用三次加权移动平均来过滤噪声。当平滑均线改变方向时开仓：向上转为做多，向下转为做空。

## 细节

- **入场条件**:
  - **多头**: 三重 WMA 斜率转向上。
  - **空头**: 三重 WMA 斜率转向下。
- **多空**: 双向。
- **出场条件**: 反向信号。
- **止损**: 无。
- **默认值**:
  - `CombinedSmoothness` = 15
  - `CandleType` = 1 分钟
- **过滤器**:
  - 类别: 趋势跟随
  - 方向: 双向
  - 指标: WeightedMovingAverage
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 中期
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中等
