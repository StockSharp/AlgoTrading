# Стратегия RedK Slow Smooth WMA
[English](README.md) | [中文](README_cn.md)

Использует тройное сглаживание взвешенным средним для фильтрации шума. Позиция открывается, когда сглаженная средняя меняет направление: long при развороте вверх, short при развороте вниз.

## Детали

- **Условия входа**:
  - **Long**: наклон тройного WMA меняется вверх.
  - **Short**: наклон тройного WMA меняется вниз.
- **Long/Short**: обе стороны.
- **Условия выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `CombinedSmoothness` = 15
  - `CandleType` = 1 минута
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: WeightedMovingAverage
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
