# Stoch Komposter戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQL5エキスパート**Exp_iStochKomposter**のポートです。ストキャスティックオシレーターを使用してモメンタムの反転を検出し、%Kラインが事前定義された閾値をクロスしたときに取引します。

## 仕組み

- 選択した時間軸でストキャスティックオシレーターを計算。
- %Kが下部レベル（デフォルト30）を上抜けすると**買い**シグナルを生成。
- %Kが上部レベル（デフォルト70）を下抜けすると**売り**シグナルを生成。
- 各シグナルで、戦略は逆方向のポジションをクローズし、シグナル方向に成行注文で新しいポジションを開きます。
- オプションのストップロスとテイクプロフィットのレベルは`StartProtection`で適用されます。

## パラメーター

| 名前 | 説明 | デフォルト |
|------|------|------------|
| `KPeriod` | %Kラインの計算期間 | 5 |
| `DPeriod` | %Dラインの平滑化期間 | 3 |
| `UpLevel` | 売りをトリガーする買われすぎ閾値 | 70 |
| `DownLevel` | 買いをトリガーする売られすぎ閾値 | 30 |
| `StopLoss` | 価格単位での絶対ストップロス | 1000 |
| `TakeProfit` | 価格単位での絶対テイクプロフィット | 2000 |
| `CandleType` | 計算用の時間軸 | 1時間 |

## 注意事項

- 戦略は完成したローソク足にのみ作動します。
- オリジナルインジケーターのATRレベルは計算しません。MQL版では矢印配置にのみ使用されていました。
- ポジションサイズは戦略の`Volume`プロパティで定義されます。
