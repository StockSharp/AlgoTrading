# RSI ロングのみ確認済みクロスバック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、RSIが閾値を下回った後に再びその閾値を上抜けるのを待ちます。クロスバックにより売られすぎ状態を確認してからロングポジションに入ります。RSIが出口レベルを上抜けたときにポジションをクローズします。パラメーターではショート取引も可能ですが、デフォルト値では実質的にショートは無効化されています。

## 詳細

- **エントリー条件**: RSIが売られすぎレベルを下回った後、そのレベルを上抜ける。
- **ロング/ショート**: デフォルトではロングのみ。
- **エグジット条件**: RSIがロング出口レベルを上抜けるか、オプションのショートルールが発動する。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `CandleType` = 5 minute
  - `RsiLength` = 14
  - `Oversold` = 44
  - `LongExitLevel` = 70
  - `ShortEntryLevel` = 100
  - `ShortExitLevel` = 0
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: ロング
  - インジケーター: RSI
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
