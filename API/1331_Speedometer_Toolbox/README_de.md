# Speedometer Toolbox
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Zeichnet eine RSI-basierte Tachometer-Anzeige ausschließlich zur Visualisierung.

## Details

- **Einstiegskriterien**: Keine, nur Visualisierung.
- **Long/Short**: Keine.
- **Ausstiegskriterien**: Nicht anwendbar.
- **Stops**: Nein.
- **Standardwerte**:
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
  - `RsiLength` = 14
  - `Radius` = 20
- **Filter**:
  - Kategorie: Visualisierung
  - Richtung: Keine
  - Indikatoren: RSI
  - Stops: Nein
  - Komplexität: Grundlegend
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Keine
