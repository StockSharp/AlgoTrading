# Speedometer Toolbox
[English](README.md) | [中文](README_cn.md)

Визуализационная стратегия, рисующая на графике спидометр на основе RSI.

## Детали

- **Критерии входа**: отсутствуют, только визуализация.
- **Длинные/короткие**: нет.
- **Критерии выхода**: не применимо.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
  - `RsiLength` = 14
  - `Radius` = 20
- **Фильтры**:
  - Категория: Визуализация
  - Направление: нет
  - Индикаторы: RSI
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: отсутствует
