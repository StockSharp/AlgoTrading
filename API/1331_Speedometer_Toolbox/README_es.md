# Speedometer Toolbox
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Dibuja un indicador visual de velocímetro basado en RSI solo para visualización.

## Detalles

- **Criterios de entrada**: Ninguno, solo visualización.
- **Largo/Corto**: Ninguno.
- **Criterios de salida**: No aplicable.
- **Stops**: No.
- **Valores predeterminados**:
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
  - `RsiLength` = 14
  - `Radius` = 20
- **Filtros**:
  - Categoría: Visualización
  - Dirección: Ninguno
  - Indicadores: RSI
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Intradía
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Ninguno
