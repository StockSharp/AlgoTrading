# Faithインジケーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格がより高い高値またはより低い安値を作るときの出来高拡大を測定することで、市場の「信頼度（faith）」を評価します。正のスコアは買い手の優位を示し、負のスコアは売り手の優勢を示します。戦略は正スコアと負スコアの間の遷移時にトレードします。

## 詳細

- **エントリー条件:** Faithスコアがゼロを上抜け → 買い；ゼロを下抜け → 売り。
- **ロング/ショート:** 両方。
- **エグジット条件:** 逆シグナル。
- **インジケーター:** Highest、SMA。
