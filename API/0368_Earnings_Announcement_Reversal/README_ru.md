# Earnings Announcement Reversal
[English](README.md) | [中文](README_zh.md)

Стратегия **Earnings Announcement Reversal** продаёт недавних победителей и покупает проигравших в день объявления отчёта.

## Детали
- **Условия входа**: в день отчёта шорт акций с положительной доходностью и покупка с отрицательной.
- **Направление**: обе стороны.
- **Выход**: позиции корректируются после сигнала, явного правила удержания нет.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `LookbackDays = 5`
  - `HoldingDays = 3`
  - `CandleType = TimeSpan.FromDays(1).TimeFrame()`
- **Фильтры**:
  - Категория: событийная
  - Направление: обе
  - Индикаторы: доходность
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: дневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
