# 資産クラス・モメンタム・ローテーション戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このローテーションモデルは、最近のモメンタムが最も強い資産クラスに資本を配分します。各期間、システムは資産ETFをランク付けし、上位をホールドしながら下位は回避します。

モメンタムがプラスの資産がない場合はキャッシュを防御的資産として使いながら、毎月リバランスを実施します。

## 詳細

- **データ**: 資産クラスETFの月次トータルリターン。
- **エントリー**: ポジティブモメンタムを持つ上位N資産を保有。
- **エグジット**: 資産が上位ランキングから外れたときに入れ替え。
- **銘柄**: 幅広い資産クラスのETF。
- **リスク**: キャッシュプロキシとポジション上限を使用。
