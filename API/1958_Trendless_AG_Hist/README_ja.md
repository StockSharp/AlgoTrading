# Trendless AG Histogram戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は**Trendless AG Histogram**インジケーターによって検出された反転を取引します。インジケーターは価格と平滑化された移動平均との距離を測定し、その結果を再度平滑化してゼロ周辺のヒストグラムを形成します。局所的な最小値は潜在的な上昇反転を示し、局所的な最大値は下降反転を示します。

ヒストグラムが方向を変えるとポジションが開かれます。インジケーターが前の値より低い後に上昇すると、ロングポジションが開かれます。前の値より高い後に下落すると、ショートポジションが開かれます。オプションのストップロスとテイクプロフィットのレベルでリスクを管理します。

## 詳細

- **エントリー条件**:
  - **ロング**: 前の値がその前より低かった状態でヒストグラム値が上昇している。
  - **ショート**: 前の値がその前より高かった状態でヒストグラム値が下落している。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 反対シグナルまたはストップロス/テイクプロフィットのレベル。
- **ストップ**: 価格単位での固定ストップロスとテイクプロフィット。
- **デフォルト値**:
  - `Fast Length` = 7。
  - `Slow Length` = 5。
  - `Stop Loss` = 1000。
  - `Take Profit` = 2000。
  - `Candle Type` = 12時間ローソク足。
- **フィルター**:
  - カテゴリ: トレンドフォロー。
  - 方向: 両方。
  - インジケーター: 移動平均に基づくカスタムインジケーター。
  - ストップ: はい。
  - 複雑さ: 中程度。
  - 時間軸: 中期。
  - 季節性: いいえ。
  - ニューラルネットワーク: いいえ。
  - ダイバージェンス: はい。
  - リスクレベル: 中。
