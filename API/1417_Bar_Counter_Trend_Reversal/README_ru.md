# Стратегия Bar Counter Trend Reversal
[English](README.md) | [中文](README_cn.md)

Ищет несколько подряд растущих или падающих баров и открывает контртрендовые сделки, когда цена достигает границ канала.

## Детали

- **Условия входа**: серии ростов или падений с дополнительным подтверждением объёмом и каналом
- **Лонг/Шорт**: Оба
- **Условия выхода**: противоположный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `NoOfRises` = 3
  - `NoOfFalls` = 3
  - `ChannelLength` = 20
  - `ChannelMultiplier` = 2
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: Keltner Channel или Bollinger Bands
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
