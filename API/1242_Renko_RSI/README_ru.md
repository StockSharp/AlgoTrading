# Renko RSI
[English](README.md) | [中文](README_cn.md)

Стратегия торгует по ренко и сигналам RSI.

Тесты показывают среднюю доходность. Лучше работает на рынках с выраженными ренко трендами.

Renko RSI использует ренко-бриксы на основе ATR и короткий RSI. При пересечении уровня перепроданности открывается покупка, при падении ниже уровня перекупленности открывается продажа.

## Подробности

- **Критерии входа**: RSI пересекает уровни перепроданности/перекупленности.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `RenkoAtrLength` = 14
  - `RsiLength` = 2
  - `RsiOverbought` = 80
  - `RsiOversold` = 20
  - `CandleType` = Renko ATR(14)
- **Фильтры**:
  - Категория: Momentum
  - Направление: оба
  - Индикаторы: RSI, Renko
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: Renko
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
