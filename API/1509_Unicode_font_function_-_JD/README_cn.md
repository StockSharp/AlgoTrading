# Unicode font function - JD
[English](README.md) | [Русский](README_ru.md)

演示如何将普通文本转换为带有加粗、斜体或大写样式的Unicode字符的工具策略。

## 详情

- **用途**：展示字符替换为Unicode变体的方法。
- **交易**：无。
- **参数**：
  - `InputString` – 待转换文本
  - `UpperCase` – 转换为大写样式
  - `Bold` – 使用粗体字符
  - `Italic` – 使用斜体字符
- **筛选**：
  - 类型：工具
  - 方向：无
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：无
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险级别：无
