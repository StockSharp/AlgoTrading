# PowerZone 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、弱気のローソク足に続く連続した強気のローソク足（またはその逆）によって形成される"PowerZone"オーダーブロックを識別します。強気ゾーンを上回るブレイクアウトがロングトレードを発動し、弱気ゾーンを下回る崩落がショートポジションを開きます。ターゲットとストップはゾーンのレンジに基づきます。

## 詳細

- **エントリー条件**:
  - **ロング**: `Periods+1`本前の弱気ローソク足に続く`Periods`本の強気ローソク足とゾーン高値を上抜ける価格。
  - **ショート**: `Periods+1`本前の強気ローソク足に続く`Periods`本の弱気ローソク足とゾーン安値を下抜ける価格。
- **ロング/ショート**: 両方。
- **エグジット条件**: ゾーンレンジの倍数によるテイクプロフィットとストップロス。
- **インジケーター**: なし。
- **デフォルト値**:
  - `Periods` = 5
  - `Threshold` = 0
  - `UseWicks` = false
  - `Take Profit Factor` = 1.5
  - `Stop Loss Factor` = 1
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - Stops: はい
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
