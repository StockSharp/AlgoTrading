# Gold-Freitag-Anomalie-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kauft bei der ersten Freitagskerze innerhalb eines bestimmten Datumsbereichs und schließt die Position nach einer festen Anzahl von Kerzen.

## Details

- **Einstiegskriterien:** Kauf bei der ersten Freitagskerze innerhalb der Start-/Enddaten.
- **Ausstiegskriterien:** Position nach N Kerzen schließen (Standard: 4).
- **Long/Short:** Nur Long.
- **Indikatoren:** Keine.
- **Stops:** Keine.
- **Zeitrahmen:** Konfigurierbarer Kerzentyp (standardmäßig 1 Stunde).
- **Kategorie:** Zeitbasiert.
- **Komplexität:** Niedrig.
