# ゴールド金曜日アノマリー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、指定された日付範囲内の最初の金曜日ローソク足で買いエントリーし、固定本数のローソク足を保有した後にポジションを決済します。

## 詳細

- **エントリー条件:** 開始日/終了日の範囲内の最初の金曜日ローソク足で買い。
- **エグジット条件:** N本のローソク足保有後にポジションを決済（デフォルト4本）。
- **ロング/ショート:** ロングのみ。
- **インジケーター:** なし。
- **ストップ:** なし。
- **時間軸:** 設定可能なローソク足タイプ（デフォルト1時間）。
- **カテゴリ:** 時間ベース。
- **複雑さ:** 低。
