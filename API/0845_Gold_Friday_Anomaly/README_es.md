# Estrategia de Anomalía del Oro del Viernes
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia compra en la primera vela del viernes dentro de un rango de fechas especificado y sale después de mantener un número fijo de velas.

## Detalles

- **Criterios de entrada:** Comprar en la primera vela del viernes dentro de las fechas de inicio/fin.
- **Criterios de salida:** Cerrar la posición después de mantener N velas (por defecto 4).
- **Largo/Corto:** Solo largos.
- **Indicadores:** Ninguno.
- **Stops:** Ninguno.
- **Marco temporal:** Tipo de vela configurable (1 hora por defecto).
- **Categoría:** Basado en tiempo.
- **Complejidad:** Bajo.
