# ローソク足トレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略は連続するローソク足の方向に基づいてポジションを建てます。
指定された数の連続した陽線が現れた後にロングポジションを建て、同じ数の陰線が現れた後にショートポジションを建てます。
逆のシグナルが発生したとき、既存のポジションをクローズすることができます。

## パラメーター

- **Candle Type**: 分析に使用するローソク足の時間軸。
- **Trend Candles**: アクションを発動するために必要な一方向への連続ローソク足の数。
- **Take Profit %**: エントリー価格のパーセンテージで表すオプションのテイクプロフィット。
- **Stop Loss %**: エントリー価格のパーセンテージで表すオプションのストップロス。
- **Enable Long Entry**: ロングポジションの建て入れを許可する。
- **Enable Short Entry**: ショートポジションの建て入れを許可する。
- **Enable Long Exit**: 逆シグナルでロングポジションのクローズを許可する。
- **Enable Short Exit**: 逆シグナルでショートポジションのクローズを許可する。

## ロジック

1. 選択した時間軸のローソク足データを購読します。
2. 連続した陽線と陰線の数を追跡します。
3. 陽線カウンターが必要数に達したとき：
   - 許可されていればショートポジションをクローズします。
   - 許可されていればロングポジションを建てます。
4. 陰線カウンターが必要数に達したとき：
   - 許可されていればロングポジションをクローズします。
   - 許可されていればショートポジションを建てます。
5. オプションの保護注文は `StartProtection` を使って設定されます。

## 注意事項

- シグナルは完成したローソク足のみで処理されます。
- 戦略はエントリーとエグジットに `BuyMarket` と `SellMarket` を使用します。
- コード内のすべてのコメントは要件通り英語で書かれています。
