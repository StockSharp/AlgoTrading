# Estrategia Dinámica del Panel de Puntos
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia combina la nube Ichimoku con dos medias móviles exponenciales de cero retraso. Cuenta cuántos componentes están por encima del precio y va largo cuando las señales alcistas dominan o corto cuando prevalecen las señales bajistas.
