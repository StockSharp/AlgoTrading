# Dynamisches Punkte-Dashboard-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert die Ichimoku-Wolke mit zwei Zero-Lag exponentiellen gleitenden Durchschnitten. Sie zählt, wie viele Komponenten über dem Preis liegen, und geht long, wenn bullische Signale dominieren, oder short, wenn bärische Signale überwiegen.
