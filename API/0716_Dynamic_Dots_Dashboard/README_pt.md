# Estratégia Dinâmica do Painel de Pontos
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina a nuvem Ichimoku com duas médias móveis exponenciais de zero atraso. Conta quantos componentes estão acima do preço e vai comprado quando os sinais de alta dominam ou vendido quando os sinais de baixa prevalecem.
