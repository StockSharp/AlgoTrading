# Стратегия Keltner Rsi
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает каналы Келтнера и индикатор RSI. Ищет возможности возврата к среднему, когда цена касается границ канала и RSI подтверждает перепроданность или перекупленность.

Каналы Келтнера отображают недавнюю волатильность, а RSI измеряет экстремумы импульса. Вход выполняется, когда RSI поддерживает выход за пределы канала.

Подходит трейдерам, торгующим отскоки от границ волатильности. Стопы основаны на множителе ATR.

## Подробности

- **Условия входа**:
  - Длинная: `Close < LowerBand && RSI < RsiOversoldLevel`
  - Короткая: `Close > UpperBand && RSI > RsiOverboughtLevel`
- **Long/Short**: Оба
- **Условия выхода**:
  - Цена возвращается к EMA
- **Стопы**: процентный уровень через `StopLossPercent`
- **Параметры по умолчанию**:
  - `EmaPeriod` = 20
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2.0m
  - `RsiPeriod` = 14
  - `RsiOverboughtLevel` = 70m
  - `RsiOversoldLevel` = 30m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: Оба
  - Индикаторы: Keltner Channel, RSI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
