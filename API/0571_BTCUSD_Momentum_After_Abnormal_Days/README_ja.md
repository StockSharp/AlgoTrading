# BTCUSD 異常日後のモメンタム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は1日のリターンを `(close - open) / open` として計算し、設定可能な期間の移動平均と標準偏差と比較します。リターンが上限閾値を超えるとロングポジションを建て、下限閾値を下回るとショートポジションを建てます。すべてのポジションは翌日の終値で決済されます。

## 詳細

- **エントリー条件**:
  - リターン > 平均 + k × 標準偏差 → ロング。
  - リターン < 平均 - k × 標準偏差 → ショート。
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 翌日の終値ですべてのポジションを決済。
- **ストップ**: なし
- **デフォルト値**:
  - ルックバック期間 = 5
  - 異常リターン閾値 (k) = 1.6
  - 1取引あたりの資本 = 1000
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: SMA, StandardDeviation
  - ストップ: なし
  - 複雑さ: 低
  - 時間軸: 長期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
