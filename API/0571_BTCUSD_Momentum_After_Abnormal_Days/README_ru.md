# Стратегия BTCUSD Momentum After Abnormal Days
[English](README.md) | [中文](README_cn.md)

Стратегия рассчитывает дневную доходность как `(close - open) / open` и сравнивает её со скользящим средним и стандартным отклонением за настраиваемый период. Если доходность превышает верхний порог, открывается длинная позиция; если падает ниже нижнего порога — короткая. Все позиции закрываются на закрытии следующего дня.

## Детали

- **Условия входа**:
  - Доходность > среднее + k × std → покупка.
  - Доходность < среднее - k × std → продажа.
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Закрытие всех позиций на закрытии следующего дня.
- **Стопы**: отсутствуют
- **Значения по умолчанию**:
  - Период расчёта = 5
  - Порог аномальной доходности (k) = 1.6
  - Капитал на сделку = 1000
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Долгий
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
