# RSI失败摆动策略
[English](README.md) | [Русский](README_ru.md)

RSI失败摆动是一种经典的反转方法，当RSI在超卖区形成更高的低点或在超买区形成更低的高点时出现。
未能触及新的极值通常预示趋势可能反转。
当RSI守住前低并回升至30上方时买入，或未能超过前高并跌破70时卖出。
设置百分比止损控制风险，当RSI穿越相反水平时离场。

## 细节

- **入场条件**：指标信号
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：反转
  - 方向：双向
  - 指标：RSI
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
