# EMA 道氏理论策略
[English](README.md) | [Русский](README_ru.md)

该策略将快慢指数移动平均线（EMA）交叉与简化的道氏理论趋势过滤结合。趋势依据最近的波峰与波谷确定。当EMA方向与趋势一致且价格突破相应的波动极值时开仓。

## 细节

- **入场条件**：
  - **多头**：快EMA ≥ 慢EMA 且价格突破最近的波峰。
  - **空头**：快EMA < 慢EMA 且价格跌破最近的波谷。
- **出场条件**：相反信号。
- **止损/止盈**：无。
- **默认参数**：
  - 快EMA周期 = 47
  - 慢EMA周期 = 50
  - 波动周期 = 6 根K线
- **筛选**：
  - 类型：趋势跟随
  - 方向：双向
  - 指标：EMA、波峰/波谷
  - 止损：无
  - 复杂度：中等
  - 时间框架：中期
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
