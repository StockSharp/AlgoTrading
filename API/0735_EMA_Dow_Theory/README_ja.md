# EMA ダウ理論戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、速い指数移動平均線（EMA）と遅い指数移動平均線のクロスオーバーに、基本的なダウ理論のトレンドフィルターを組み合わせます。トレンドは直近のスウィング高値と安値によって判断されます。EMAがトレンド方向と一致したときにポジションを取ります。

## 詳細

- **エントリー条件**:
  - **ロング**: 速いEMA ≥ 遅いEMA で、価格が直近のスウィング高値を上抜けする。
  - **ショート**: 速いEMA < 遅いEMA で、価格が直近のスウィング安値を下抜けする。
- **エグジット条件**: 逆シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - 速いEMAの期間 = 47
  - 遅いEMAの期間 = 50
  - スウィング期間 = 6バー
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA、スウィング高値/安値
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
