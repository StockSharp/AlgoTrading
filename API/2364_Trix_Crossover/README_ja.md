# TRIXクロスオーバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、潜在的な反転を検出するために異なる期間を持つ2つのTRIX（三重指数移動平均オシレーター）インジケーターを使用します。低速TRIXが上昇している間に高速TRIXがローカルボトムを形成したときにロングポジションを建てます。低速TRIXが下落している間に高速TRIXがローカルトップを形成したときにショートポジションを建てます。

## パラメーター

- **Fast TRIX Period** – 高速TRIXインジケーターの期間。
- **Slow TRIX Period** – 低速TRIXインジケーターの期間。
- **Take Profit** – 絶対価格単位での利益目標。
- **Stop Loss** – 絶対価格単位での最大損失。
- **Candle Type** – ローソク足の時間軸またはデータ型。

## 取引ロジック

1. 選択したローソク足タイプを購読する。
2. 完成した各ローソク足で高速および低速TRIXの値を計算する。
3. 高速TRIXの値が前の値より高く、前の値がその前の値より低く、低速TRIXが上昇しているときにロングエントリー。
4. 高速TRIXの値が前の値より低く、前の値がその前の値より高く、低速TRIXが下落しているときにショートエントリー。
5. 同時に1つのポジションのみ保有する。
6. ストップロスとテイクプロフィットの保護が自動的に適用される。

## 注意事項

この戦略はMQL5スクリプトの適応版であり、StockSharp内でTRIXインジケーターを操作する方法を示しています。
