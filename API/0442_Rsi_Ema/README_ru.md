# Стратегия RSI + EMA
[English](README.md) | [中文](README_cn.md)

Данная система объединяет осциллятор относительной силы (RSI) с фильтром тренда из двух экспоненциальных скользящих средних. RSI показывает краткосрочные зоны перепроданности и перекупленности, а пара EMA определяет направление основного тренда. Сделки открываются только по тренду, что позволяет избежать входов против устойчивого движения рынка.

Когда импульс цены опускает RSI ниже уровня перепроданности и при этом быстрая EMA располагается выше медленной, предполагается восходящий тренд и открывается длинная позиция. Если же RSI поднимается выше отметки перекупленности, а быстрая EMA всё ещё выше медленной, стратегия открывает короткую сделку, ожидая кратковременную коррекцию в пределах глобального канала.

Позиции закрываются, когда RSI выходит из экстремальной зоны в противоположную сторону, что означает завершение отката. Метод прост, но эффективен для ловли краткосрочных колебаний в трендовых рынках. Он лучше всего работает на ликвидных инструментах, где экстремумы RSI возникают часто, а направление тренда сохраняется.

## Детали

- **Условия входа**:
  - **Long**: `RSI < oversold` и `EMA1 > EMA2`.
  - **Short**: `RSI > overbought` и `EMA1 > EMA2`.
- **Направление торговли**: обе стороны.
- **Условия выхода**:
  - **Long**: `RSI > overbought`.
  - **Short**: `RSI < oversold`.
- **Стопы**: не предусмотрены.
- **Значения по умолчанию**:
  - `RSI Length` = 14.
  - `Overbought/Oversold` = 70 / 30.
  - `EMA Lengths` = 150 / 600.
- **Фильтры**:
  - Категория: Моментум
  - Направление: Обе стороны
  - Индикаторы: Несколько
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
