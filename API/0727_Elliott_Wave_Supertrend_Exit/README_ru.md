# Стратегия Elliott Wave Supertrend Exit
[English](README.md) | [中文](README_cn.md)

Стратегия открывает позиции при локальных разворотах и закрывает их при смене направления Supertrend с фиксированным стоп-лоссом в процентах.

## Детали

- **Условия входа**:
  - Long: цена формирует локальный минимум
  - Short: цена формирует локальный максимум
- **Long/Short**: обе стороны
- **Условия выхода**:
  - смена направления Supertrend или уровень стоп-лосса
- **Стопы**: фиксированный процент от цены входа
- **Значения по умолчанию**:
  - `WaveLength` = 4
  - `SupertrendLength` = 10
  - `SupertrendMultiplier` = 3
  - `StopLossPercent` = 10
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: обе
  - Индикаторы: Highest, Lowest, SuperTrend
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: краткосрочный
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
