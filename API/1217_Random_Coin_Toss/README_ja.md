# ランダムコイントス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この実験的な戦略は、N本のバーごとにコインを投げ、その結果に基づいてロングまたはショートエントリーします。リスクはATRベースのストップロスとテイクプロフィットレベルで管理されます。

テストでは年平均リターン約8%が示されています。暗号資産市場で最もよく機能します。

アイデアは、規律ある出口を維持しながらランダムエントリーの基準値を提供することです。

## 詳細

- **エントリー条件**: `EntryFrequency`本のバーごとにコインを投げる；表ならロング、裏ならショート。
- **ロング/ショート**: 両方向。
- **エグジット条件**: ストップロスまたはテイクプロフィットに到達。
- **ストップ**: はい。
- **デフォルト値**:
  - `AtrLength` = 14
  - `SlMultiplier` = 1m
  - `TpMultiplier` = 2m
  - `EntryFrequency` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: 実験的
  - 方向: 両方
  - インジケーター: ATR
  - ストップ: はい
  - 複雑さ: シンプル
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
