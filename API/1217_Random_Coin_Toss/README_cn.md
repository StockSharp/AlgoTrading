# 随机抛硬币策略
[English](README.md) | [Русский](README_ru.md)

该实验性策略每隔 N 根K线抛一次硬币，根据结果做多或做空。风险通过基于ATR的止损和止盈控制。

回测显示年化收益约为 8%，在加密货币市场表现最佳。

其目的是在保持纪律性退出的同时，为随机入场提供基准。

## 细节

- **入场条件**：每隔 `EntryFrequency` 根K线抛硬币，正面做多，反面做空。
- **多/空**：双向。
- **出场条件**：达到止损或止盈。
- **止损**：是。
- **默认值**:
  - `AtrLength` = 14
  - `SlMultiplier` = 1m
  - `TpMultiplier` = 2m
  - `EntryFrequency` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **过滤器**:
  - 类别: 实验
  - 方向: 双向
  - 指标: ATR
  - 止损: 是
  - 复杂度: 简单
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 高
