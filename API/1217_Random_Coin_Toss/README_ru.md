# Стратегия Random Coin Toss
[English](README.md) | [中文](README_cn.md)

Экспериментальная стратегия, которая каждые N баров подбрасывает монету и открывает длинную или короткую позицию в зависимости от результата. Риск контролируется стоп-лоссом и тейк-профитом на основе ATR.

Тестирование показывает среднюю годовую доходность около 8%. Лучше всего работает на рынке криптовалют.

Идея состоит в том, чтобы дать базовый ориентир для случайных входов при строгом управлении выходами.

## Подробности

- **Условия входа**: каждые `EntryFrequency` баров подбрасывается монета; орёл — покупка, решка — продажа.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**: достижение стоп-лосса или тейк-профита.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `AtrLength` = 14
  - `SlMultiplier` = 1m
  - `TpMultiplier` = 2m
  - `EntryFrequency` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Экспериментальная
  - Направление: Обе
  - Индикаторы: ATR
  - Стопы: Да
  - Сложность: Простая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
