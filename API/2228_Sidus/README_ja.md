# Sidus 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はSIDUS移動平均システムを実装しています。2本の線形加重移動平均と確認用指数平均のクロスオーバーを使用して取引します。短期LWMAが長期LWMAをクロスするとき、またはロングLWMAがスローEMAをクロスするときにポジションをオープンします。逆方向のクロスオーバーはポジションをクローズまたはリバースします。パーセントベースのストップロスとテイクプロフィットがリスクを管理します。

テストでは平均年間収益約25%が示されています。為替ペアで最もパフォーマンスを発揮します。

核心的なアイデアは、速い移動平均と遅い移動平均が再整列したときのトレンド転換を捉えることです。LWMAペアは価格変動に素早く反応し、遅いEMAがノイズをフィルタリングします。強気または弱気の整列が発生すると、戦略はその方向でエントリーし、不利な動きの際に保護レベルに頼って出口を見つけます。

## 詳細

- **エントリー条件**:
  - **ロング**: 速いLWMAが遅いLWMAを上回るクロス *または* 遅いLWMAが遅いEMAを上回るクロス。
  - **ショート**: 速いLWMAが遅いLWMAを下回るクロス *または* 遅いLWMAが遅いEMAを下回るクロス。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆方向クロスオーバーまたは保護ストップレベル。
- **ストップ**: はい、`StartProtection`を介したパーセントベースのテイクプロフィットとストップロスを使用。
- **デフォルト値**:
  - 速いEMAの長さ = 18。
  - 遅いEMAの長さ = 28。
  - 速いLWMAの長さ = 5。
  - 遅いLWMAの長さ = 8。
  - テイクプロフィット = 2%。
  - ストップロス = 1%。
- **フィルター**: なし。
