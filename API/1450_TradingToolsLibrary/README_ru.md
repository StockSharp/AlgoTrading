# Trading Tools Library Strategy
[English](README.md) | [中文](README_cn.md)

Простая стратегия пересечения SMA с фильтром RSI и задержкой между входами.

## Детали
- **Критерии входа**:
  - **Лонг**: быстрая SMA пересекает медленную снизу вверх и RSI ниже `RsiUpper`
  - **Шорт**: быстрая SMA пересекает медленную сверху вниз и RSI выше `RsiLower`
- **Длинные/короткие**: обе стороны
- **Критерии выхода**:
  - Обратный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `ShortLength` = 10
  - `LongLength` = 30
  - `RsiLength` = 14
  - `CooldownBars` = 3
  - `RsiUpper` = 60
  - `RsiLower` = 40
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: обе стороны
  - Индикаторы: SMA, RSI
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
