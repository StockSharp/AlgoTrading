# Supertrend Cross RSI-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Kombiniert einen SuperTrend auf höherem Zeitrahmen mit RSI-basierten Ausstiegen.

- **Long**: Preis kreuzt die SuperTrend-Linie des höheren Zeitrahmens von unten nach oben.
- **Short**: Preis kreuzt die SuperTrend-Linie des höheren Zeitrahmens von oben nach unten.
- **Long-Ausstieg**: Take-Profit-Prozentsatz oder RSI über dem oberen Limit.
- **Short-Ausstieg**: Take-Profit-Prozentsatz oder RSI unter dem unteren Limit.
- **Indikatoren**: SuperTrend, RSI.
