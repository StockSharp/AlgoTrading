# Supertrend Cross RSI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

上位時間軸のSuperTrendとRSIベースのエグジットを組み合わせます。

- **ロング**: 価格が上位時間軸のSuperTrendラインを上抜け。
- **ショート**: 価格が上位時間軸のSuperTrendラインを下抜け。
- **ロング・エグジット**: テイクプロフィット・パーセンテージ、またはRSIが上限を上回る。
- **ショート・エグジット**: テイクプロフィット・パーセンテージ、またはRSIが下限を下回る。
- **インジケーター**: SuperTrend, RSI。
