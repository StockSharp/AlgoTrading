# Estrategia de Mapa de Calor de Matriz de Correlación
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Muestra una matriz de correlación entre hasta siete valores seleccionados. La estrategia se suscribe a cada instrumento, suaviza los precios de cierre con una media móvil y registra las correlaciones con el tiempo y entre instrumentos. No se realizan operaciones de trading.
