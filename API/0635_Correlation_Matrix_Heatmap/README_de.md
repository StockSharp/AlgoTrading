# Korrelationsmatrix-Heatmap-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Zeigt eine Korrelationsmatrix zwischen bis zu sieben ausgewählten Wertpapieren an. Die Strategie abonniert jedes Instrument, glättet Schlusskurse mit einem gleitenden Durchschnitt und protokolliert Korrelationen mit der Zeit und zwischen den Instrumenten. Es werden keine Handelsaktionen durchgeführt.
