# Estratégia de Mapa de Calor de Matriz de Correlação
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Exibe uma matriz de correlação entre até sete ativos selecionados. A estratégia assina cada instrumento, suaviza os preços de fechamento com uma média móvel e registra as correlações com o tempo e entre os instrumentos. Nenhuma ação de trading é executada.
