# マルチ時間軸 RSI グリッド戦略（矢印付き）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、現在の時間軸と 2 つの上位時間軸の RSI が買われすぎまたは売られすぎのレベルに達したときにトレードします。すべての RSI が一致したときに最初のポジションを建て、その後 ATR ベースのグリッドとロット乗数の増加を使って追加ポジションを積み増します。戦略は日次利益率を目標とし、毎日リセットし、逆シグナルまたはドローダウンで決済します。

## パラメーター
- ローソク足タイプ
- RSI 長さ
- 売られすぎレベル
- 買われすぎレベル
- 上位時間軸 1
- 上位時間軸 2
- グリッド乗数ファクター
- ロット乗数ファクター
- 最大グリッドレベル数
- 日次利益目標 %
- ATR 長さ
