# LANZ戦略 4.0 バックテスト
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

LANZ戦略4.0バックテストは、スウィングピボットを使ってトレンド転換を検出するブレイクアウト戦略です。価格が最後のピボット高値を上抜けるとロングエントリー、最後のピボット安値を下抜けるとショートエントリーします。ポジションサイズはリスクパーセントとpips値から計算され、ストップロスは最後のスウィングにバッファを加えた位置、テイクプロフィットはリスクリワード比で設定されます。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 価格が最後のピボット高値を上抜ける。
  - **ショート**: 価格が最後のピボット安値を下抜ける。
- **エグジット条件**: ストップロスまたはテイクプロフィット。
- **ストップ**: 直近スウィング高値/安値にバッファを追加。
- **デフォルト値**:
  - `SwingLength` = 180
  - `SlBufferPoints` = 50
  - `RiskReward` = 1
  - `RiskPercent` = 1
  - `PipValueUsd` = 10
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ロング＆ショート
  - インジケーター: Highest, Lowest
  - 複雑さ: 中程度
  - リスクレベル: 中
