# Smart Fib Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия использует пробой простой скользящей средней для входа и ATR-основанные полосы Фибоначчи для выхода.

## Детали

- **Критерий входа**: Пересечение цены закрытия и SMA.
- **Лонг/шорт**: Оба направления.
- **Критерий выхода**: Достижение ценой полосы Фибоначчи ATR.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `SmaLength` = 50
  - `FibSmaLength` = 8
  - `AtrLength` = 6
  - `FirstFactor` = 1.618
  - `SecondFactor` = 2.618
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: SMA, ATR
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
