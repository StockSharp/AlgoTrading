# 利益ラベル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

**利益ラベル戦略**は、MetaTrader 5 エキスパート アドバイザー *利益ラベル (54352)* を StockSharp の高レベル API に変換します。この戦略は、オープンポジションへのトリプル指数移動平均 (TEMA) クロスオーバーを監視し、ポジションがクローズされた後にチャート上に利益ラベルを描画します。トレンドが上向きに反転すると、アルゴリズムはロング ポジションをオープンし、トレンドが下向きに反転するとショート ポジションをオープンします。反対のポジションがまだ有効な場合、ストラテジーはまずそれを決済し、実現利益ラベルを印刷します。

ローソク足は `SubscribeCandles` サブスクリプションを通じて処理され、インジケーターは実装を完全に高レベルに保つために `Bind` を通じてバインドされます。終了したローソク足は TEMA 値を更新し、取引の決定をトリガーします。

## 取引ルール

1. **強気クロスオーバー**: 現在の TEMA 値が以前の値を上回り、古い測定値が下降勾配を示している場合、現在アクティブなロングポジションがない場合、ストラテジーはロングポジションをオープンします。
2. **弱気クロスオーバー**: TEMA が同様に下落すると、アクティブなショートがない場合はショート ポジションがオープンします。
3. **ポジション反転**: 新しいシグナルの瞬間に反対のポジションが存在する場合、ストラテジーは新しい注文を出す前にオープンポジションをクローズします。
4. **利益ラベル**: ポジションが完全にクローズされると、実現損益が計算され、`DrawText` を使用してチャートに表示されます。

## パラメーター

| 名前 | デフォルト | 説明 |
| ---- | ------- | ----------- |
| `CandleType` | `TimeSpan.FromMinutes(1).TimeFrame()` | キャンドルのサブスクリプションに使用される時間枠。 |
| `TemaPeriod` | `6` | 三重指数移動平均の期間。 |
| `TradeVolume` | `0.1` | 各成行注文で発注された数量。 |
| `PlacingTrade` | `false` | ライブ注文の発注を有効または無効にします。 |
| `LabelOffset` | `0` | 取引価格の上の利益ラベルに適用される垂直オフセット。 |

## 注意事項

- この戦略は完成したローソク足のみに依存しており、インジケーター バッファーには直接アクセスしません。
- MQL バージョンの保護的なストップロスとテイクプロフィットのレベルは複製されません。逆の信号が到着すると位置が逆転します。
- 利益ラベルは、セキュリティ通貨が利用可能な場合は常に使用し、それ以外の場合は生の値に戻ります。
