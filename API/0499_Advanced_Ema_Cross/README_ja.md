# 高度なEMAクロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、短期EMAが長期EMAを上回るときにロングポジションを取り、正規化ATR、ADXトレンド強度、およびSuperTrendの方向確認でシグナルをフィルタリングします。ストップロスとテイクプロフィットのレベルは、50期間EMAから推定されるUSD強度に基づいて適応します。

## 詳細

- **エントリー条件**:
  - 短期EMAが長期EMAを上抜ける。
  - トレンド方向に応じた閾値を超える正規化ATR。
  - SuperTrendが強気または弱気市場を確認する。
- **エグジット条件**:
  - 最小保有期間後の逆方向EMAクロスまたはADXが閾値を超える。
  - ストップロスまたはテイクプロフィットに到達。
- **インジケーター**: EMA, ATR, ADX, SuperTrend, SMA（出来高）。
- **ストップ**: 動的パーセンテージのストップロスとテイクプロフィット。
- **タイプ**: トレンドフォロー。
- **時間軸**: 30分（デフォルト）。
