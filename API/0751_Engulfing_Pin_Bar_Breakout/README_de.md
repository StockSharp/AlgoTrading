# Engulfing & Pin-Bar-Ausbruch-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie wartet auf eine Hammer- oder bullische Engulfing-Kerze und geht beim Ausbruch über das Signalhoch long. Bei bärischen Setups wird ein Shooting Star oder eine bärische Engulfing-Kerze verwendet und beim Ausbruch unter das Signaltief verkauft. Der Stop-Loss wird auf der entgegengesetzten Seite der Signalkerze platziert und der Take-Profit nutzt ein Risiko-Ertrags-Verhältnis.

## Details

- **Einstiegskriterien:** Hammer oder bullisches Engulfing gefolgt von Ausbruch über das Hoch; Shooting Star oder bärisches Engulfing gefolgt von Ausbruch unter das Tief.
- **Long/Short:** Beide.
- **Ausstiegskriterien:** Stop auf der gegenüberliegenden Seite der Signalkerze; Take-Profit als Risiko-Vielfaches.
- **Stops:** Ja.
- **Standardwerte:**
  - Long-Gewinnverhältnis = 5
  - Short-Gewinnverhältnis = 4
  - Risikoprozentsatz = 0.02
  - Kerzen-Zeitrahmen = 1 Minute
- **Filter:**
  - Kategorie: Ausbruch
  - Richtung: Beide
  - Indikatoren: Keine
  - Stops: Ja
  - Komplexität: Mittel
  - Zeitrahmen: Beliebig
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
