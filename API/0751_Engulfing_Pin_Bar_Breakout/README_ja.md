# 包み線とピンバー・ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はハンマーまたは強気の包み線を待ち、シグナルの高値を上抜けたブレイクアウトでロングエントリーします。弱気のセットアップでは流れ星または弱気の包み線を使用し、シグナルの安値を下抜けたところで売ります。ストップロスはシグナルローソク足の反対側に置かれ、テイクプロフィットはリスク/リワード比率を使用します。

## 詳細

- **エントリー条件:** ハンマーまたは強気の包み線の後に高値をブレイクアウト；流れ星または弱気の包み線の後に安値をブレイクアウト。
- **ロング/ショート:** 両方。
- **エグジット条件:** シグナルローソク足の反対側でストップ；リスクの倍数でテイクプロフィット。
- **ストップ:** はい。
- **デフォルト値:**
  - ロング利益比率 = 5
  - ショート利益比率 = 4
  - リスクパーセント = 0.02
  - ローソク足の時間軸 = 1分
- **フィルター:**
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 中
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
