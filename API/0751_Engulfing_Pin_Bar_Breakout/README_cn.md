# 吞没与针形线突破策略
[English](README.md) | [Русский](README_ru.md)

该策略等待锤子线或看涨吞没形态出现，并在向上突破信号蜡烛高点时做多。对于看跌情况，使用流星线或看跌吞没，在跌破信号蜡烛低点时做空。止损放在信号蜡烛另一端，止盈按风险收益比设置。

## 细节

- **入场条件：** 锤子线或看涨吞没后向上突破；流星线或看跌吞没后向下突破。
- **多空方向：** 双向。
- **出场条件：** 止损在信号蜡烛另一端；按风险倍数止盈。
- **止损：** 有。
- **默认值：**
  - 多头收益倍数 = 5
  - 空头收益倍数 = 4
  - 风险百分比 = 0.02
  - 蜡烛周期 = 1 分钟
- **筛选：**
  - 类别：突破
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：中等
  - 周期：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险水平：中等
