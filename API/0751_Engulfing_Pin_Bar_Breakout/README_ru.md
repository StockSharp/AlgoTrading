# Стратегия «Пробой Поглощения и Пин-бара»
[English](README.md) | [中文](README_cn.md)

Стратегия ожидает появления молота или бычьего поглощения и открывает лонг при пробое максимума сигнальной свечи. Для медвежьих сценариев используется падающая звезда или медвежье поглощение с входом в шорт при пробое минимума сигнальной свечи. Стоп ставится за противоположную границу сигнальной свечи, тейк-профит задается через соотношение риск/прибыль.

## Детали

- **Критерии входа:** молот или бычье поглощение с последующим пробоем максимума; падающая звезда или медвежье поглощение с пробоем минимума.
- **Длинные/короткие:** обе стороны.
- **Критерии выхода:** стоп за противоположной границей сигнальной свечи; тейк-профит по мультипликатору риска.
- **Стопы:** да.
- **Значения по умолчанию:**
  - Соотношение прибыли лонг = 5
  - Соотношение прибыли шорт = 4
  - Процент риска = 0.02
  - Таймфрейм свечей = 1 минута
- **Фильтры:**
  - Категория: Пробой
  - Направление: Обе
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
