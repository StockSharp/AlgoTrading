# Chaikin Momentum Scalper
[English](README.md) | [中文](README_cn.md)

Скальперская стратегия использует осциллятор Chaikin для фиксации смены импульса. Лонг открывается при пересечении выше нуля и цене выше SMA(200). Шорт — при пересечении ниже нуля и цене ниже SMA. Выход осуществляется по стоп-лоссу и тейк-профиту на основе ATR.

## Детали

- **Условия входа**: Осциллятор Chaikin пересекает ноль вверх/вниз при цене выше/ниже SMA.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: Стоп-лосс и тейк-профит по ATR.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `FastLength` = 3
  - `SlowLength` = 10
  - `SmaLength` = 200
  - `AtrLength` = 14
  - `AtrMultiplierSL` = 1.5m
  - `AtrMultiplierTP` = 2.0m
  - `CandleType` = TimeSpan.FromHours(1)
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: Chaikin Oscillator, SMA, ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
