# Стратегия Ilan Dynamic HT
[English](README.md) | [中文](README_cn.md)

Сеточная мартингейловая стратегия, открывающая позиции по сигналам RSI и расширяющая позицию с помощью динамического ценового диапазона. Каждый дополнительный ордер увеличивает объём по мультипликатору и использует общий тейк-профит и стоп-лосс.

## Детали

- **Условия входа**:
  - Лонг: RSI ниже `RsiMinimum`
  - Шорт: RSI выше `RsiMaximum`
- **Лонг/Шорт**: Лонг и Шорт
- **Условия выхода**:
  - Достижение общего тейк-профита или стоп-лосса
- **Стопы**:
  - `TakeProfit` в пунктах
  - `StopLoss` в пунктах
- **Значения по умолчанию**:
  - `LotExponent` = 1.4
  - `MaxTrades` = 10
  - `DynamicPips` = true
  - `DefaultPips` = 120
  - `Depth` = 24
  - `Del` = 3
  - `BaseVolume` = 0.1
  - `RsiPeriod` = 14
  - `RsiMinimum` = 30
  - `RsiMaximum` = 70
  - `TakeProfit` = 100
  - `StopLoss` = 500
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Сетка / Мартингейл
  - Направление: Лонг и Шорт
  - Индикаторы: RSI, Highest, Lowest
  - Стопы: Тейк-профит, Стоп-лосс
  - Сложность: Продвинутая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
