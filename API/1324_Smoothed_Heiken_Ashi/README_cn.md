# 平滑 Heiken-Ashi 策略
[English](README.md) | [Русский](README_ru.md)

该策略使用 EMA 平滑后的 Heiken-Ashi 蜡烛图来突出价格加速。当一根看涨蜡烛的实体大于前一根蜡烛时做多；当看跌实体扩大时平仓。

## 细节

- **入场条件**: 看涨平滑 Heiken-Ashi 蜡烛的实体大于前一根
- **多空方向**: 多头
- **出场条件**: 看跌实体扩大
- **止损**: 无
- **默认值**:
  - `EmaLength` = 40
- **过滤器**:
  - 分类: 形态
  - 方向: 多头
  - 指标: EMA, Heikin-Ashi
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
