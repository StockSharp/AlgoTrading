# 平滑化 Heiken-Ashi 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

EMAで平滑化されたHeiken-Ashiローソク足は価格動向の加速を強調します。平滑化された上昇ローソク足が前のローソク足よりも大きな実体を持つとき、ロングポジションを建てます。下降の実体が拡大するとポジションを決済します。

## 詳細

- **エントリー条件**: 平滑化されたHeiken-Ashiの上昇ローソク足の実体が前のローソク足より大きい
- **ロング/ショート**: ロング
- **エグジット条件**: 下降の実体が拡大する
- **ストップ**: いいえ
- **デフォルト値**:
  - `EmaLength` = 40
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ロング
  - インジケーター: EMA, Heikin-Ashi
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
