# Стратегия Smoothed Heiken-Ashi
[English](README.md) | [中文](README_cn.md)

Стратегия использует сглаженные EMA свечи Heiken-Ashi для выявления ускорения движения цены. Лонг открывается, когда бычья свеча имеет тело больше предыдущего. Позиция закрывается, когда медвежье тело расширяется.

## Детали

- **Критерий входа**: бычья сглаженная свеча с телом больше предыдущего
- **Длин/Шорт**: Только лонг
- **Критерий выхода**: расширение медвежьего тела
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `EmaLength` = 40
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Лонг
  - Индикаторы: EMA, Heikin-Ashi
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
