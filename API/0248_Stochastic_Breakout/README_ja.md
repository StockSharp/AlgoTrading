# Stochasticブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このブレイクアウトアプローチは、Stochasticオシレーターが最近の平均から急激に動くことを監視します。%Kラインがボラティリティ調整された閾値を上下にブレイクすると、トレンドを開始する可能性のあるモメンタムの爆発を示します。

テストでは平均年間リターンが約181%であることが示されています。暗号資産市場で最も良いパフォーマンスを発揮します。

%Kが収縮期間後に上部閾値を上回ったときにロングポジションが発動されます。%Kが下部閾値を下回ったときにショートポジションが取られます。オシレーターが平均値に向かって戻るか保護的なストップに達したら取引を決済します。

この戦略はモメンタムスイングへの早期エントリーを望むイントラデイトレーダー向けに設計されています。ボラティリティベースのバンドを使用することでノイズを排除し、決定的な動きのみがシグナルを生成します。

## 詳細
- **エントリー条件**:
  - **ロング**: %K > Avg + DeviationMultiplier * StdDev
  - **ショート**: %K < Avg - DeviationMultiplier * StdDev
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: %K < Avg のときに決済
  - **ショート**: %K > Avg のときに決済
- **ストップ**: あり、パーセントストップロス。
- **デフォルト値**:
  - `StochasticPeriod` = 14
  - `KPeriod` = 3
  - `DPeriod` = 3
  - `LookbackPeriod` = 20
  - `DeviationMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Stochastic Oscillator
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
