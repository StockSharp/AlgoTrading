# MSL EA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

MSL EAは、最近の局所的な極値から動的なサポートラインとレジスタンスラインを構築するブレイクアウト戦略です。短期のフラクタル高値と安値を検出し、指定されたtick分の距離で調整し、価格がこれらのレベルを超えてクローズしたときにポジションを開きます。元のMQL4実装から変換されました。

## 仕組み

1. アルゴリズムはローソク足の高値と安値を追跡して局所的な極値を決定します。
2. 最後に検出された *Level* 個の極値の中の最高の高値と最低の安値が、レジスタンスとサポートラインとして格納されます。
3. 各ラインは市場ノイズを考慮して *Distance* tick分オフセットされます。
4. 終値が上限ラインを上抜けたとき、ロングポジションが開かれます。下限ラインを下抜けたとき、ショートポジションが開かれます。
5. 同時取引数は *Max Trades* によって制限されます。

## パラメーター

- **Max Trades** – 許可される最大オープンポジション数。
- **Level** – レベル構築に使用される局所的な極値の数。
- **Distance** – ラインを配置する際の極値からのオフセット（tick単位）。
- **Candle Type** – 戦略が処理するローソク足の時間軸。

## 注意事項

このC#バージョンは高レベルのStockSharp APIを使用し、英語のコメントが含まれています。元のMQL4ヘルパーライブラリからのリスク管理機能は、基本的なポジションチェックに簡略化されています。
