# Gann Laplace スムーズドハイブリッド VSA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はGannスタイルのトレンドフィルターとLaplaceスムーズドのボリュームスプレッド分析（VSA）を組み合わせます。VSA値は価格スプレッドをローソク足レンジで割り出来高を掛けて計算し、EMAでスムージングします。スムーズドVSAがトレンド移動平均に対する価格と整合したときに取引を行います。

## 詳細

- **エントリー条件**:
  - **ロング**: スムーズドVSA > 0 かつ終値 > トレンドMA。
  - **ショート**: スムーズドVSA < 0 かつ終値 < トレンドMA。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: スムーズドVSAが負になる。
  - **ショート**: スムーズドVSAが正になる。
- **ストップ**: StartProtectionを使用。
- **デフォルト値**:
  - `Trend Period` = 20
  - `VSA Smoothing` = 14
  - `Candle Type` = 15m
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: MA, 出来高
  - ストップ: はい
  - 複雑さ: 中
  - 時間軸: 中期
