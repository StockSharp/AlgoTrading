# Intelle city World Cycle Ath Atl Logarithmic Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия использует масштабированные скользящие средние для определения сигналов на исторических максимумах (ATH) и минимумах (ATL) по концепции Pi Cycle.

Система продаёт, когда масштабированная длинная ATH SMA пересекает короткую SMA сверху вниз, и покупает, когда масштабированная длинная ATL SMA пересекает короткую EMA снизу вверх.

## Детали

- **Условия входа**: Масштабированная длинная ATH SMA пересекает короткую SMA сверху вниз для продажи. Масштабированная длинная ATL SMA пересекает короткую EMA снизу вверх для покупки.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `AthLongLength` = 350
  - `AthShortLength` = 111
  - `AtlLongLength` = 471
  - `AtlShortLength` = 150
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: SMA, EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
