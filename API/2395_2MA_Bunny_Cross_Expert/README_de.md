# 2MA Bunny Cross Expert
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie **2MA Bunny Cross Expert** handelt den Crossover zweier einfacher gleitender Durchschnitte. Ein Long-Trade wird eröffnet, wenn der schnelle Durchschnitt über den langsamen steigt, während ein Short-Trade eröffnet wird, wenn der schnelle Durchschnitt unter den langsamen fällt. Jede entgegengesetzte Position wird geschlossen, bevor eine neue eröffnet wird.

## Details

- **Zweck**: Trendfolge durch gleitenden Durchschnitt-Crossover
- **Handel**: Long und Short
- **Indikatoren**: schneller und langsamer einfacher gleitender Durchschnitt
- **Stops**: keine
- **Standardwerte**:
  - `CandleType` = 1 minute
  - `FastLength` = 5
  - `SlowLength` = 20
