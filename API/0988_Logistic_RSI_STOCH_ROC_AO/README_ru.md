# Logistic RSI STOCH ROC AO
[English](README.md) | [中文](README_cn.md)

Стратегия применяет логистическое отображение к выбранному индикатору (AO, ROC, RSI, Stochastic) и торгует при пересечении нуля знаковой стандартной девиации.

## Детали

- **Критерий входа**: Знаковая стандартная девиация пересекает ноль снизу вверх.
- **Лонг/Шорт**: Обе стороны.
- **Критерий выхода**: Знаковая стандартная девиация пересекает ноль сверху вниз.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Indicator` = LogisticDominance
  - `Length` = 13
  - `LenLd` = 5
  - `LenRoc` = 9
  - `LenRsi` = 14
  - `LenSto` = 14
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Oscillator
  - Направление: Обе
  - Индикаторы: AwesomeOscillator, RateOfChange, RelativeStrengthIndex, StochasticOscillator, Highest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
