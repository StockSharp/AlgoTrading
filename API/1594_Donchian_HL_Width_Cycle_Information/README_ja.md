Donchianチャネルの幅とサイクル変化に基づいて取引する戦略。

この戦略は、ローソク足の極値とDonchianチャネルとの関係を監視する。下降サイクル後に上限バンドに達するとロングポジションを開く。上昇サイクル後に下限バンドに触れるとショートポジションを開く。

## 詳細

- **エントリー条件**: Donchianチャネルのサイクルトレンド変化。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対のサイクルシグナル。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `Length` = 28
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: Donchian
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
