# A Informer
[English](README.md) | [Русский](README_ru.md)

A Informer 是一个实用型策略，可自动为已有持仓添加止损和止盈，并以美元和点数显示当前收益。该策略不生成交易信号，旨在管理手动开立的交易。

策略监控新的成交，依据品种的价格步长将已实现收益转换为点数，并将信息记录到日志中。策略启动时即应用仓位保护。

## 详情

- **入场条件**: 手动开仓，策略本身不下单。
- **多空方向**: 双向。
- **出场条件**: 止损或止盈被触发。
- **止损**: 是（通过 `StartProtection`）。
- **默认值**:
  - `StopLoss` = 300 点
  - `TakeProfit` = 1000 点
- **过滤器**:
  - 类别: 实用工具
  - 方向: 双向
  - 指标: 无
  - 止损: 是
  - 复杂度: 基础
  - 时间框架: 任意
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
