# A Informer
[English](README.md) | [中文](README_cn.md)

A Informer — утилитарная стратегия, автоматически добавляющая к открытым позициям стоп-лосс и тейк-профит и отображающая текущую прибыль в долларах и пунктах. Она не генерирует торговых сигналов и предназначена для сопровождения сделок, открытых вручную.

Стратегия отслеживает новые сделки, переводит полученную прибыль в пункты с использованием шага цены инструмента и выводит эту информацию в журнал. Защита позиции активируется при запуске стратегии.

## Подробности

- **Критерий входа**: позиция открывается вручную, стратегия сама не торгует.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: срабатывание стоп-лосса или тейк-профита.
- **Стопы**: да (через `StartProtection`).
- **Значения по умолчанию**:
  - `StopLoss` = 300 пунктов
  - `TakeProfit` = 1000 пунктов
- **Фильтры**:
  - Категория: Утилита
  - Направление: Обе стороны
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
