# Momo Trades戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

移動平均とMACD構造でフィルタリングされたモメンタムブレイクアウトを取引する、MetaTraderの元のエキスパートアドバイザー「Momo_trades」の変換版です。

## 戦略のロジック
- 設定された時間軸の完成した足で動作し、一度に一つのネットポジションのみを処理します。
- 設定可能なバーシフトを持つ単純移動平均を使用して、価格が平均からどれだけ離れてクローズしたかを測定します。ロング取引では、シフトされたクローズが価格シフト閾値以上SMAを上回ることを要求し、ショートでは逆を要求します。
- MQLルールを反映したカスケードMACDモメンタムパターンを評価します：ロングの場合、いくつかの過去のMACDメインライン値がゼロを通じて増加し、ショートの場合はゼロを通じて減少する必要があります。これによりモメンタムが弱まっている間の取引を防ぎます。
- SMA距離フィルターとMACDパターンが同じ方向に揃ったとき、戦略ボリュームでマーケット注文を開きます。

## リスク管理
- ストップロス、テイクプロフィット、トレーリングストップ、トレーリングステップ、ブレイクイーブン、価格シフトのインプットはピップスで定義され、インストゥルメントのステップを使用して価格単位に自動変換されます。
- テイクプロフィットとトレーリング値が提供されている場合、価格がトレーリング距離にトレーリングステップを加えた分だけ前進した後にのみストップが追跡され、MQLの動作を再現します。
- テイクプロフィットが設定されていないがブレイクイーブン距離が設定されている場合、ブレイクイーブントリガーに達するとストップがエントリー価格に移動されます。
- すべてのストップとテイクレベルは完成した足ごとに再計算され、足の極値によって交差されるとマーケット注文でクローズされます。

## セッション管理
- `CloseEndDay`フラグは元のエキスパートアドバイザーと一致し、プラットフォーム時刻23:00（金曜日は21:00）にアクティブなポジションをクローズします。カットオフ後、戦略は翌日まで新しいエントリーをスキップします。

## パラメーター
- **SMA Period / MA Bar Shift** – 移動平均の長さと、SMAと価格の値を取得するために使用するバーインデックス。
- **MACD Fast / Slow / Signal / Bar Shift** – MACDの設定と、パターンチェックのために保存された値に適用されるオフセット。
- **Stop Loss / Take Profit / Trailing Stop / Trailing Step / Breakeven / Price Shift** – エグジット、トレーリング、SMAフィルターを制御するピップ距離。
- **Close End Of Day** – 設定されたセッション終了後にポジションをクローズします。
- **Candle Type** – 足とインジケーター計算に使用される時間軸。
