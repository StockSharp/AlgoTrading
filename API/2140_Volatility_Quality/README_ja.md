# ボラティリティ品質戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

平滑化された中央値価格の方向変化を使って取引する方法を示すサンプル戦略。元のMQLエキスパートは *Volatility Quality* インジケーターを使用していました；この実装は中央値価格の単純移動平均でそれを近似しています。

## 戦略ロジック
- 各ローソク足の中央値価格 `(High + Low) / 2` を計算する。
- 単純移動平均（SMA）で中央値価格を平滑化する。
- インジケーターの色を決定する：上昇値は **上**（色0）、下落値は **下**（色1）として扱う。
- 色が上から下に変わったとき、戦略はショートポジションを閉じてロングポジションを開く。
- 色が下から上に変わったとき、戦略はロングポジションを閉じてショートポジションを開く。
- 固定ストップロスとテイクプロフィットレベルによる基本的なリスク管理が適用される。

## パラメーター
| 名前 | 説明 |
|------|------|
| `Length` | 中央値価格に適用するSMAの平滑化期間。 |
| `Candle Type` | 計算に使用するローソク足の時間軸。 |
