# MovingUp MA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はオプションのリスク管理を備えた移動平均クロスオーバーシステムを実装しています。
速い移動平均が遅い移動平均を上抜けするとロングポジションを開き、反対のクロスでショートポジションを開きます。

## パラメーター
- **Fast MA** (`FastLength`): 速い単純移動平均の期間。
- **Slow MA** (`SlowLength`): 遅い単純移動平均の期間。
- **Use TP** (`UseTakeProfit`): テイクプロフィットルールを有効にする。
- **TP** (`TakeProfit`): 利益確定の価格距離。
- **Use SL** (`UseStopLoss`): ストップロスルールを有効にする。
- **SL** (`StopLoss`): ストップロスの価格距離。
- **Use TS** (`UseTrailingStop`): トレーリングストップロジックを有効にする。
- **TS** (`TrailingStop`): トレーリングストップの価格距離。
- **Candle** (`CandleType`): 計算に使用するローソク足のタイプ。

## トレードロジック
1. ローソク足データを購読し、2本のSMAインジケーターを計算する。
2. 速いMAと遅いMAのクロスオーバーを検出する。
3. ロングポジションがない場合、速いMAが遅いMAを上抜けしたときにロングに入る。
4. ショートポジションがない場合、速いMAが遅いMAを下抜けしたときにショートに入る。
5. 新しいローソク足ごとにリスク管理を適用する：
   - 価格が指定距離前進したときに利益確定する。
   - 価格が指定距離ポジションに対して逆行したときにストップロス。
   - 価格が有利な方向に動いたらトレーリングストップで利益を保護する。

## オリジナルMQL戦略
オリジナルのMQL4スクリプト`ma_v_1_3_3.mq4`には、ステップアップ/ダウンロジックや複雑なポジションサイジングなど多数の追加機能が含まれています。このC#バージョンはコアの移動平均クロスオーバーと基本的なリスク管理に焦点を当てています。
