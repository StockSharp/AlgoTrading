# Bollinger と Stochastic トレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格がボリンジャーバンド下限を下回って終値をつけ、Stochastic %Kが20未満のときにロングで参入します。価格が上限バンドを上回って終値をつけ、%Kが80を超えるときにショートで参入します。ATRベースのトレーリングストップがオープンポジションを保護します。

## 詳細
- **エントリー条件:**
  - **ロング:** close < ボリンジャーバンド下限 かつ %K < 20。
  - **ショート:** close > ボリンジャーバンド上限 かつ %K > 80。
- **ロング/ショート:** 両方。
- **エグジット条件:** ATRベースのトレーリングストップ。
- **ストップ:** ATR * 乗数に基づくトレーリングストップ。
- **デフォルト値:** Bollinger長 = 20、偏差 = 2、Stochastic長 = 14、平滑化 = 3、ATR期間 = 14、ATR乗数 = 1.5。
