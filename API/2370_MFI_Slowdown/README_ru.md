# Замедление MFI
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает индекс денежного потока (MFI) на старшем таймфрейме и реагирует при достижении экстремальных зон. Если включён `SeekSlowdown`, сигнал подтверждается только когда изменение MFI между двумя соседними барами меньше одной единицы. При восходящем сигнале закрываются короткие позиции и при необходимости открывается длинная, при нисходящем сигнале закрываются длинные и может открываться короткая. Управление рисками осуществляется через StartProtection.

## Подробности

- **Условия входа**:
  - Вверх: `MFI >= UpperThreshold` и (проверка замедления выключена или замедление обнаружено).
  - Вниз: `MFI <= LowerThreshold` и (проверка замедления выключена или замедление обнаружено).
- **Покупка/Продажа**: Обе стороны в зависимости от параметров.
- **Условия выхода**:
  - Обратный сигнал закрывает позицию.
  - Стоп‑лосс и тейк‑профит через `StopLossPercent` и `TakeProfitPercent`.
- **Стопы**: Да, через StartProtection.
- **Значения по умолчанию**:
  - `MfiPeriod` = 2
  - `UpperThreshold` = 90
  - `LowerThreshold` = 10
  - `SeekSlowdown` = true
  - `TakeProfitPercent` = 2
  - `StopLossPercent` = 1
  - `CandleType` = таймфрейм 6 часов
  - `BuyPosOpen` = `BuyPosClose` = `SellPosOpen` = `SellPosClose` = true
- **Фильтры**:
  - Категория: Моментум
  - Направление: Оба
  - Индикаторы: MFI
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Опционально (проверка замедления)
  - Уровень риска: Средний
