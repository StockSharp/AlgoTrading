# Стратегия Volatility Momentum Breakout
[English](README.md) | [中文](README_cn.md)

Стратегия объединяет уровни пробоя на основе ATR с фильтром тренда EMA и импульсом RSI для захвата сильных движений.

## Детали

- **Критерий входа**: закрытие цены выше/ниже уровней пробоя ATR при подтверждении EMA и RSI
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: стоп-лосс по ATR и тейк-профит с соотношением риск/прибыль 1:2
- **Стопы**: ATR
- **Значения по умолчанию**:
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 1.5
  - `Lookback` = 20
  - `EmaPeriod` = 50
  - `RsiPeriod` = 14
  - `RsiLongThreshold` = 50
  - `RsiShortThreshold` = 50
  - `RiskReward` = 2
  - `AtrStopMultiplier` = 1
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе стороны
  - Индикаторы: ATR, EMA, RSI, Highest, Lowest
  - Стопы: ATR
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
