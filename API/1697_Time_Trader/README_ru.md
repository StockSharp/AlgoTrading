# Стратегия Time Trader
[English](README.md) | [中文](README_cn.md)

Временная стратегия, которая входит в длинную и/или короткую позицию точно в заданный момент времени и защищает позицию стоп-лоссом и тейк-профитом.

## Детали

- **Критерий входа**: в момент `TradeHour:TradeMinute:TradeSecond` открывается покупка, если `AllowBuy`, и продажа, если `AllowSell`
- **Длин/Шорт**: обе стороны в зависимости от настроек
- **Критерий выхода**: стоп-лосс или тейк-профит
- **Стопы**: да, оба
- **Значения по умолчанию**:
  - `Volume` = 1
  - `TakeProfit` = 0.2
  - `StopLoss` = 0.2
  - `TradeHour` = 0
  - `TradeMinute` = 0
  - `TradeSecond` = 0
  - `AllowBuy` = true
  - `AllowSell` = true
  - `CandleType` = TimeSpan.FromSeconds(1).TimeFrame()
- **Фильтры**:
  - Категория: Time
  - Направление: обе
  - Индикаторы: нет
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
