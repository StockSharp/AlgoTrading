# 出来高MAクロス (Volume MA Cross)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は出来高を高速・低速の移動平均で処理します。高速出来高MAが低速出来高MAを上抜けると、参加者の増加を示してロングエントリーが発動します。下抜けは弱さのシグナルとなりショートを開始します。

テストでは年平均リターンが約46%となっています。株式市場での運用に最も適しています。

逆クロスが発生したときにポジションを決済します。価格は独自の移動平均で監視され、取引のフィルタリングに役立てられます。

出来高ベースのシグナルは価格動向に先行することが多く、早期エントリーが可能です。

## 詳細

- **エントリー条件**: 高速出来高MAが低速出来高MAをクロス。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆クロスまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `FastVolumeMALength` = 10
  - `SlowVolumeMALength` = 50
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: Volume MA
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
