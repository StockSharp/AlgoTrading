# マーチンゲールMACD戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、StockSharpフレームワークでオリジナルのMQLエキスパートアドバイザー「MartGreg_1」を再現します。2つのMACD（移動平均収束拡散）インジケーターを使用してリバーサルを探し、ポジションサイジングにマーチンゲールルールを適用します。

## 仕組み

- 最初のMACDは直近3本の完成ローソク足でローカルな極値を探します。
- 2番目のMACDは直近2つの値を比較してモメンタムの方向を決定します。
- 最初のMACDが谷を形成し、2番目のMACDが減少するとロングポジションが建てられます。
- 最初のMACDが山を形成し、2番目のMACDが増加するとショートポジションが建てられます。
- 負けトレードの後、次の注文サイズは設定された上限まで倍増されます。
- ストップロスとテイクプロフィットは絶対価格ポイントで設定されます。

## パラメーター

- `Shape` – 口座残高から初期ボリュームを計算するための除数。
- `Doubling Count` – 損失後の最大連続倍増回数。
- `Stop Loss` – ポイント単位のプロテクションストップ。
- `Take Profit` – ポイント単位の利益目標。
- `MACD1 Fast/Slow` – 最初のMACDの期間。
- `MACD2 Fast/Slow` – 2番目のMACDの期間。
- `Candle Type` – 分析用の時間軸。
