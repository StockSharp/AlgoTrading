# Стратегия Double Trading
[English](README.md) | [中文](README_cn.md)

Парная стратегия, открывающая противоположные позиции по двум коррелированным инструментам и закрывающая их при достижении суммарной прибыли.

## Детали

- **Критерий входа**: одновременное открытие первой и второй позиции в противоположных направлениях
- **Длин/Шорт**: одновременно лонг и шорт
- **Критерий выхода**: суммарная прибыль >= ProfitTarget
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Volume1` = 1
  - `Volume2` = 1.3
  - `ProfitTarget` = 20
  - `SecondSecurity` = требуется
- **Фильтры**:
  - Категория: Pair Trading
  - Направление: Хеджированное
  - Индикаторы: нет
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
