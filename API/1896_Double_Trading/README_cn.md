# Double Trading 策略
[English](README.md) | [Русский](README_ru.md)

成对交易策略，同时在两个相关品种上开立相反方向的头寸，当总利润达到目标值时全部平仓。

## 细节

- **入场条件**: 同时在两个品种上按相反方向开仓
- **多空方向**: 多与空同时持有
- **出场条件**: 总利润 >= ProfitTarget
- **止损**: 无
- **默认值**:
  - `Volume1` = 1
  - `Volume2` = 1.3
  - `ProfitTarget` = 20
  - `SecondSecurity` = 必填
- **过滤器**:
  - 分类: Pair Trading
  - 方向: 对冲
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
