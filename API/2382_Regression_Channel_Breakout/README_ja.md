# 回帰チャネルブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MQLスクリプト`e-Regr`に基づいた回帰チャネル取引システムを実装します。
設定可能な数の直近の足に対して線形回帰ラインを構築し、
指定された標準偏差の距離に上下のバンドを追加します。取引ルール：

- **ロングエントリー:** 足の安値が下限バンドに触れるか、下方に突き抜けたとき。
- **ショートエントリー:** 足の高値が上限バンドに触れるか、上方に突き抜けたとき。
- **エグジット:** 終値が逆方向に回帰ラインをクロスしたとき。
- **トレーリングストップ:** オプションのトレーリングロジックにより、取引が設定した利益に
  達した後にストップレベルが移動します。

## パラメーター

| 名前            | 説明                                                            |
|-----------------|-----------------------------------------------------------------|
| `CandleType`    | 計算に使用する足のタイプ。                                      |
| `Length`        | 回帰と標準偏差のための足の本数。                                |
| `Deviation`     | チャネル幅の標準偏差乗数。                                      |
| `UseTrailing`   | トレーリングストップロジックを有効にする。                      |
| `TrailingStart` | トレーリングが開始する前に必要な利益。                          |
| `TrailingStep`  | 価格とトレーリングストップの間の距離。                          |

この戦略は`SubscribeCandles`と`Bind`メソッドを通じてStockSharpの高水準APIを使用し、
足データとインジケーター値を受け取ります。
