# Multi-Factor Strategy
[English](README.md) | [中文](README_cn.md)

Многофакторная стратегия сочетает MACD, RSI и две скользящие средние для торговли по тренду. Лонг открывается, когда линия MACD выше сигнальной, RSI ниже 70, цена выше SMA50, а SMA50 выше SMA200. Шорт выполняется при обратных условиях.

Стоп и тейк рассчитываются как множители ATR.

## Подробности

- **Условия входа**:
  - **Лонг**: `MACD > Signal` && `RSI < 70` && `Close > SMA50` && `SMA50 > SMA200`.
  - **Шорт**: `MACD < Signal` && `RSI > 30` && `Close < SMA50` && `SMA50 < SMA200`.
- **Направление**: обе стороны.
- **Условия выхода**: стоп и тейк по ATR.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `FastLength` = 12
  - `SlowLength` = 26
  - `SignalLength` = 9
  - `RsiLength` = 14
  - `AtrLength` = 14
  - `StopAtrMultiplier` = 2
  - `ProfitAtrMultiplier` = 3
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: MACD, RSI, SMA, ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
