# 动量百分比
[English](README.md) | [Русский](README_ru.md)

策略根据价格的百分比变化捕捉动量。当动量超过正或负阈值时入场，出现反向信号或触及波动性止损时离场。通过固定回溯周期计算收益率，可适应不同市场，波动性止损帮助在剧烈不利走势时迅速退出。

## 详情
- **入场条件**: 基于 MA、Momentum 的信号
- **多空方向**: 双向
- **退出条件**: 反向信号或止损
- **止损**: 是
- **默认值**:
  - `MomentumPeriod` = 10
  - `ThresholdPercent` = 5m
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类型: 趋势
  - 方向: 双向
  - 指标: MA, Momentum
  - 止损: 是
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中

测试表明年均收益约为 97%，该策略在加密市场表现最佳。
