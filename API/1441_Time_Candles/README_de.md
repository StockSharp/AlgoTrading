# Time-Candles-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Zeigt RSI-Kerzen an, die aus einem benutzerdefinierten Zeitrahmen und seinem gleitenden Durchschnitt aufgebaut sind. Nur zur Visualisierung gedacht.

## Details

- **Einstiegskriterien**: Keine
- **Long/Short**: Keine
- **Ausstiegskriterien**: Keine
- **Stops**: Nein
- **Standardwerte**:
  - `CandleType` = 5-Minuten-Kerzen
  - `RsiLength` = 14
  - `SmaLength` = 14
- **Filter**:
  - Kategorie: Visualisierung
  - Richtung: Keine
  - Indikatoren: RSI, SMA
  - Stops: Nein
  - Komplexität: Grundlegend
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Niedrig
