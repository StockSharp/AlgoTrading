# Time Candles Strategy
[Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Displays RSI candles built from a custom time frame and its moving average. Intended for visualization only.

## Details

- **Entry Criteria**: None
- **Long/Short**: None
- **Exit Criteria**: None
- **Stops**: No
- **Default Values**:
  - `CandleType` = 5-minute candles
  - `RsiLength` = 14
  - `SmaLength` = 14
- **Filters**:
  - Category: Visualization
  - Direction: None
  - Indicators: RSI, SMA
  - Stops: No
  - Complexity: Basic
  - Timeframe: Intraday
  - Seasonality: No
  - Neural networks: No
  - Divergence: No
  - Risk level: Low
