# Стратегия Time Candles
[English](README.md) | [中文](README_cn.md)

Отображает свечи RSI, построенные по пользовательскому таймфрейму, и их скользящее среднее. Предназначена только для визуализации.

## Детали

- **Критерий входа**: нет
- **Длин/Шорт**: нет
- **Критерий выхода**: нет
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = свечи 5 минут
  - `RsiLength` = 14
  - `SmaLength` = 14
- **Фильтры**:
  - Категория: Визуализация
  - Направление: нет
  - Индикаторы: RSI, SMA
  - Стопы: нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
