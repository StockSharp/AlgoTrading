# Time Candles 策略
[English](README.md) | [Русский](README_ru.md)

显示基于自定义时间框架的 RSI 蜡烛及其移动平均线。仅用于可视化。

## 详情

- **入场条件**：无
- **多空方向**：无
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `CandleType` = 5 分钟蜡烛
  - `RsiLength` = 14
  - `SmaLength` = 14
- **筛选**：
  - 类别: 可视化
  - 方向: 无
  - 指标: RSI, SMA
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
