# Estrategia de Velas Temporales
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Muestra velas RSI construidas desde un marco temporal personalizado y su media móvil. Destinada únicamente a visualización.

## Detalles

- **Criterios de entrada**: Ninguno
- **Largo/Corto**: Ninguno
- **Criterios de salida**: Ninguno
- **Stops**: No
- **Valores predeterminados**:
  - `CandleType` = velas de 5 minutos
  - `RsiLength` = 14
  - `SmaLength` = 14
- **Filtros**:
  - Categoría: Visualización
  - Dirección: Ninguno
  - Indicadores: RSI, SMA
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Intradía
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Bajo
