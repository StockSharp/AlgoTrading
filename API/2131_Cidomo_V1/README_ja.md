# Cidomo V1 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

価格が直近のレンジを突破したときにトレードを行うデイリー・ブレイクアウト戦略。

## 概要

- **タイプ**: ブレイクアウト
- **エントリー**: 価格がルックバック期間の最高値を上回った場合に買い、最安値を下回った場合に売り。
- **エグジット**: ストップロス、テイクプロフィット、オプションのブレイクイーブンおよびトレーリングストップ。
- **インジケーター**: Highest, Lowest

## パラメーター

| 名前 | 説明 |
|------|------|
| `Lookback` | レンジ計算に使用するローソク足の本数。 |
| `Delta` | ブレイクアウトレベルに加算される価格オフセット。 |
| `StopLoss` | 価格ポイントでのストップロス。 |
| `TakeProfit` | 価格ポイントでのテイクプロフィット。 |
| `NoLoss` | この利益（ポイント）の後にストップをエントリーに移動。 |
| `Trailing` | ポイントでのトレーリング距離。 |
| `UseTimeFilter` | trueの場合、指定時刻以降にレベルが計算される。 |
| `TradeTime` | ブレイクアウトレベルを計算する時刻。 |
| `CandleType` | 計算に使用するローソク足の種類。 |

## 備考

この戦略は完了したローソク足のみを監視します。レベルは `TradeTime` 以降、1日1回再計算されます。
