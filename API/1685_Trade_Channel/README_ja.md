# トレーディングチャネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**トレーディングチャネル戦略**は、ドンチャン価格チャネルを中心にブレイクアウトとプルバックを取引します。上部バンドが変化せず、価格がそれに触れるかその下で引けるがピボットより上にある場合、ロングポジションが開かれます。ショートエントリーには逆のロジックが使用されます。ストップロスはATR値分だけ反対側のバンドの外側に設置されます。オプションのトレーリングストップにより、取引が利益方向に進むにつれてストップを引き上げることができます。

## パラメーター

- `ChannelPeriod` — ドンチャンチャネルの長さ。
- `AtrPeriod` — ストップロス計算用のATR期間。
- `Trailing` — トレーリングストップ距離（価格単位、0でトレーリング無効）。
- `CandleType` — 計算に使用するローソク足タイプ。
