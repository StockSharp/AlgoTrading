# Nova戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
- 固定秒数にわたって価格モメンタムを監視するMetaTrader 5の「Nova」エキスパートアドバイザーの変換版です。
- `CandleType`パラメーターで選択した任意のローソク足タイプで動作し、完成したローソク足でのみロジックを評価します。
- Level1データを使用して最良の買値と売値を追跡し、`SecondsAgo`秒前の値を保存します。
- 前のローソク足が強気で現在のアスクが保存されたアスクより少なくとも`StepPips`高い場合に**ロング**ポジションに入ります。
- 前のローソク足が弱気で現在のビッドが保存されたアスクより少なくとも`StepPips`低い場合に**ショート**ポジションに入ります。
- 対応するパラメーターがゼロより大きい場合、StockSharpの保護を使用して自動的なストップロスとテイクプロフィットレベルを適用します。
- 損失（ストップロスの発動）後、次の取引のボリュームは`LossCoefficient`で乗算されます。利益が出て決済した後はボリュームが`BaseVolume`にリセットされます。

## パラメーター
- `SecondsAgo` – 参照価格スナップショットと現在の評価時点の間の秒数。
- `StepPips` – pips単位のブレイクアウトフィルター。銘柄の価格ステップを使用して価格単位に変換されます（3/5桁の小数の銘柄は×10で調整されます）。
- `BaseVolume` – 初期の取引サイズ。取引所のボリュームステップと最小/最大制限に正規化されます。
- `StopLossPips` – 保護的ストップロスのpips単位の距離（0で無効化）。
- `TakeProfitPips` – 保護的テイクプロフィットのpips単位の距離（0で無効化）。
- `LossCoefficient` – 負けトレード後に最後に実行されたボリュームに適用される乗数。
- `CandleType` – シグナルに使用するローソク足ソース（時間軸、ティック、レンジなど）。

## 追加注記
- 戦略はオリジナルのMT5動作を再現するためにLevel1データ（最良ビッド/アスク）を必要とします。Level1が利用できない場合、ローソク足の終値をフォールバックとして提供します。
- ボリューム再計算は無効な注文を避けるために`Security.VolumeStep`、`Security.MinVolume`、`Security.MaxVolume`を尊重します。
- 価格変換は`Security.PriceStep`と`Security.Decimals`に依存し、戦略が4/5桁のFXシンボルと他の銘柄の両方に適応できるようにします。
- この戦略にはPythonバージョンは提供されていません。
