# IMACD Снайпер
[English](README.md) | [中文](README_cn.md)

Стратегия IMACD Sniper сочетает пересечения MACD с фильтром EMA, подтверждением объёма и сильными свечами. Динамические тейк-профит и стоп-лосс основаны на среднем диапазоне.

## Детали
- **Данные**: ценовые свечи.
- **Условия входа**:
  - **Лонг**: линия MACD пересекает сигнальную вверх, цена выше EMA, разница MACD больше минимума, обе линии далеко от нуля, объём выше среднего, сильная бычья свеча.
  - **Шорт**: линия MACD пересекает сигнальную вниз, цена ниже EMA, разница MACD больше минимума, обе линии далеко от нуля, объём выше среднего, сильная медвежья свеча.
- **Условия выхода**: противоположный крест MACD или достижение тейк-профита/стоп-лосса.
- **Стопы**: динамические тейк-профит и стоп-лосс на основе среднего диапазона.
- **Параметры по умолчанию**:
  - `FastLength` = 12
  - `SlowLength` = 26
  - `SignalLength` = 9
  - `MacdDeltaMin` = 0.03
  - `MacdZeroLimit` = 0.05
  - `RangeLength` = 14
  - `RangeMultiplierTp` = 4.0
  - `RangeMultiplierSl` = 1.5
  - `EmaLength` = 20
  - `CandleType` = tf(1m)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг и Шорт
  - Индикаторы: MACD, EMA, Объём
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
