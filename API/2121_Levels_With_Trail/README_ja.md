# レベルとトレーリング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MQLスクリプト `levels_with_trail.mq4` から変換されました。この戦略は、価格が指定されたレベルを超えたときにトレードを開き、ストップロスをトレーリングできます。

## 仕組み
- 選択した時間軸のロウソク足を購読します。
- オープンポジションがなく、終値が `Level Price` を上回っている場合は買い、下回っている場合は売りを行います。
- `Trail Stop` が有効な場合、ポジションが利益を出しているときストップロスが価格に追随します。
- ストップロス、テイクプロフィット、または反対のブレイクアウトシグナルがトリガーされたときにポジションが閉じられます。

## パラメーター
- `Stop Loss` – ストップロスのサイズ（価格単位）。
- `Take Profit` – テイクプロフィットのサイズ（価格単位）。
- `Level Price` – 監視するブレイクアウトレベル。
- `Trail Stop` – トレーリングストップロスを有効または無効にする。
- `Candle Type` – 分析に使用するロウソク足の時間軸。
