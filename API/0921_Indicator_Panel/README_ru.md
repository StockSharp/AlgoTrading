# Стратегия Indicator Panel
[English](README.md) | [中文](README_cn.md)

Отображает значения RSI, MACD, DMI, CCI, MFI, Momentum и двух скользящих средних для выбранного инструмента. Стратегия только выводит значения индикаторов и не совершает сделок.

## Подробности

- **Условия входа**: Нет
- **Направление**: Нет
- **Условия выхода**: Нет
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `RsiLength` = 14
  - `MacdFastLength` = 12
  - `MacdSlowLength` = 26
  - `MacdSignalLength` = 9
  - `DiLength` = 14
  - `AdxLength` = 14
  - `CciLength` = 20
  - `MfiLength` = 20
  - `MomentumLength` = 10
  - `Ma1IsEma` = false
  - `Ma1Length` = 50
  - `Ma2IsEma` = false
  - `Ma2Length` = 200
- **Фильтры**:
  - Категория: Информационная
  - Направление: Нет
  - Индикаторы: RSI, MACD, DMI, CCI, MFI, Momentum, MA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
