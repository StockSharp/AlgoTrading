# Сильное движение по DMI
[English](README.md) | [中文](README_cn.md)

Стратегия основана на мощных движениях индикатора DMI (Directional Movement Index).

DMI Power Move объединяет разницу между +DI и -DI с ADX, чтобы ловить сильные тренды. Сделки открываются, когда +DI заметно превышает -DI (или наоборот) и ADX высок. Выход происходит, когда ADX снижается или разница между DI сужается.

Такой подход отсеивает слабые сигналы, требуя как сильного направленного движения, так и роста ADX. В итоге получается меньше сделок, но потенциально более качественных трендовых входов.

## Подробности

- **Критерии входа**: сигналы на основе ADX, ATR, DMI.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал или стоп.
- **Стопы**: да.
- **Параметры по умолчанию**:
  - `DmiPeriod` = 14
  - `DiDifferenceThreshold` = 5м
  - `AdxThreshold` = 30м
  - `AdxExitThreshold` = 25м
  - `AtrMultiplier` = 2м
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ADX, ATR, DMI
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (15м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
