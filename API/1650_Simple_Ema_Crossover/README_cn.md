# 简单 EMA 交叉策略
[English](README.md) | [Русский](README_ru.md)

该策略使用两条指数移动平均线的交叉，并带有固定止损和止盈。

当快速 EMA 自下向上穿越慢速 EMA 时买入；当快速 EMA 自上向下穿越慢速 EMA 时卖出。

## 详情
- **入场条件**: 快速 EMA 与慢速 EMA 的交叉。
- **多空方向**: 双向。
- **退出条件**: 反向交叉或止损/止盈。
- **止损**: 有。
- **默认值**:
  - `Periods` = 17
  - `StopLoss` = 31 (绝对值)
  - `TakeProfit` = 69 (绝对值)
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类型: 趋势
  - 方向: 双向
  - 指标: EMA
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
