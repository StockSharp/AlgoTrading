# スイング高値安値ピボット戦略 [LV]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

確認されたスイング高値と安値の周辺で取引します。ピボット安値が現れると、戦略はピボットバーの価格でロング指値注文を出し、固定のストップと目標利益を設定します。ピボット高値はショートのセットアップを引き起こします。オプションの移動平均フィルターにより、取引をトレンド方向に限定できます。

## 詳細

- **入力**:
  - ピボットの長さ。
  - ストップロス距離（ティック）。
  - 目標利益距離（ティック）。
  - 第2目標利益とダブルエントリーのスイッチ。
  - 移動平均フィルターのタイプと長さ。
- **ロング/ショート**: 両方。
- **エグジット**: 固定ストップと最大2つの利益目標。
- **フィルター**:
  - カテゴリ: パターン認識
  - 方向: 両方
  - インジケーター: 移動平均
  - ストップ: 固定
  - 複雑さ: 高
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
