# Стратегия Fibonacci Bands
[English](README.md) | [中文](README_cn.md)

Расширяет каналы Кельтнера коэффициентами Фибоначчи и открывает сделки при пробое внешних границ с подтверждением RSI.

## Детали

- **Условия входа**: цена пересекает `fbUpper3` и RSI > 60 для лонга; пересекает `fbLower3` и RSI < 40 для шорта.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: возврат цены к скользящей средней.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `MaType` = WMA
  - `MaLength` = 233
  - `Fib1` = 1.618
  - `Fib2` = 2.618
  - `Fib3` = 4.236
  - `KcMultiplier` = 2
  - `KcLength` = 89
  - `RsiLength` = 14
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Volatility
  - Направление: Обе
  - Индикаторы: MA, ATR, RSI
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
