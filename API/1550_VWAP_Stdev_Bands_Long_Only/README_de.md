# VWAP Stdev Bänder-Strategie (Nur Long)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Kauft, wenn der Preis das untere VWAP-Standardabweichungsband nach unten durchbricht, und schließt beim Erreichen des Gewinnziels.

## Parameter

- **DevUp**: Standardabweichungsmultiplikator oberhalb des VWAP.
- **DevDown**: Standardabweichungsmultiplikator unterhalb des VWAP.
- **ProfitTarget**: Gewinnziel in Preiseinheiten.
- **GapMinutes**: Pause vor neuer Order in Minuten.
- **CandleType**: Kerzentyp.
