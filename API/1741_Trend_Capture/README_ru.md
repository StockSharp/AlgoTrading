# Trend Capture
[English](README.md) | [中文](README_cn.md)

Стратегия следования за трендом, объединяющая Parabolic SAR с фильтром ADX. Длинные позиции открываются, когда цена закрытия выше значения SAR и ADX ниже порога, что указывает на зарождение тренда. Короткие позиции открываются при противоположных условиях.

## Подробности

- **Условия входа**: Цена выше/ниже линии Parabolic SAR при ADX ниже `AdxLevel`.
- **Длинные/короткие**: В обе стороны.
- **Условия выхода**: Стоп-лосс, тейк-профит или противоположный сигнал.
- **Стопы**: Фиксированный стоп-лосс, тейк-профит и перевод в безубыток.
- **Значения по умолчанию**:
  - `SarStep` = 0.02
  - `SarMax` = 0.2
  - `AdxPeriod` = 14
  - `AdxLevel` = 20
  - `StopLoss` = 1800 пунктов
  - `TakeProfit` = 500 пунктов
  - `BreakEven` = 50 пунктов
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Parabolic SAR, ADX
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
