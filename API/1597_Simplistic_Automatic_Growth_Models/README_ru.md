# Стратегия Simplistic Automatic Growth Models
[English](README.md) | [中文](README_cn.md)

Стратегия строит полосы из накопительных средних максимумов и минимумов и входит в сделки при их пробое.

## Подробности

- **Условия входа**:
  - Цена закрытия выше верхней полосы — открывается лонг.
  - Цена закрытия ниже нижней полосы — открывается шорт.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Противоположный сигнал закрывает позицию.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `Length` = 10
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Highest, Lowest
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
