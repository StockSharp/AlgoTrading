# TMA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TMA戦略は複数の平滑化移動平均とローソク足パターンを使用して、200期間トレンドの方向にトレードします。3ラインストライクと包み足シグナルをセッションフィルターと組み合わせます。

## 詳細

- **エントリー条件**: EMA(2)がSMA(200)より上にある強気の包み足または3ラインストライク / EMA(2)がSMA(200)より下にある弱気の包み足または3ラインストライク、オプションのセッションフィルター付き
- **ロング/ショート**: 両方
- **エグジット条件**: EMA(2)がSMA(200)を交差するとき
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = 5分足ローソク
  - `FastLength` = 21
  - `MidLength` = 50
  - `Mid2Length` = 100
  - `SlowLength` = 200
  - `UseSession` = false
  - `SessionStart` = 08:30
  - `SessionEnd` = 12:00
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: SMA, EMA
  - ストップ: なし
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
