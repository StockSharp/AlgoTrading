# TMA 策略
[English](README.md) | [Русский](README_ru.md)

TMA 策略利用多条平滑移动平均线和 K 线形态，在 200 周期趋势方向上交易。它结合 3 线打击和吞没信号，并可选地使用交易时段过滤。

## 详情

- **入场条件**：在 EMA(2) 高于 SMA(200) 时出现看涨吞没或三连阴转阳；在 EMA(2) 低于 SMA(200) 时出现看跌吞没或三连阳转阴，并通过可选的时段过滤
- **多空方向**：双向
- **出场条件**：EMA(2) 与 SMA(200) 交叉
- **止损**：无
- **默认值**：
  - `CandleType` = 5 分钟蜡烛
  - `FastLength` = 21
  - `MidLength` = 50
  - `Mid2Length` = 100
  - `SlowLength` = 200
  - `UseSession` = false
  - `SessionStart` = 08:30
  - `SessionEnd` = 12:00
- **筛选**：
  - 类别: 趋势
  - 方向: 双向
  - 指标: SMA, EMA
  - 止损: 无
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中
