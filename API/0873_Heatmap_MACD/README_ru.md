# Тепловая карта MACD
[English](README.md) | [中文](README_cn.md)

Стратегия Heatmap MACD торгует, когда гистограммы MACD на пяти таймфреймах совпадают. Лонг открывается, когда все гистограммы становятся положительными, шорт — когда все становятся отрицательными. При необходимости позиция закрывается, если какая-либо гистограмма меняет знак против сделки.

## Детали
- **Данные**: свечи цен.
- **Условия входа**:
  - **Лонг**: гистограмма MACD > 0 на всех пяти таймфреймах и ранее не все были положительными.
  - **Шорт**: гистограмма MACD < 0 на всех пяти таймфреймах и ранее не все были отрицательными.
- **Условия выхода**: противоположный сигнал или опциональное закрытие при смене направления.
- **Стопы**: по умолчанию отсутствуют.
- **Параметры по умолчанию**:
  - `FastLength` = 9
  - `SlowLength` = 26
  - `SignalLength` = 9
  - `TimeFrame1` = tf(60)
  - `TimeFrame2` = tf(120)
  - `TimeFrame3` = tf(240)
  - `TimeFrame4` = tf(240)
  - `TimeFrame5` = tf(480)
  - `CloseOnOpposite` = false
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг и Шорт
  - Индикаторы: MACD
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Мульти-таймфрейм
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
