# Система Uhl MA Crossover
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Система Uhl MA Crossover строит две адаптивные линии (CTS и CMA), используя дисперсию для регулировки сглаживания. Длинная позиция открывается при пересечении CTS выше CMA, короткая — при пересечении ниже.

## Детали

- **Условие входа**: CTS пересекает CMA сверху.
- **Длинные/короткие**: Обе стороны.
- **Условие выхода**: CTS пересекает CMA снизу.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Length` = 100
  - `Multiplier` = 1
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Trend
  - Направление: Both
  - Индикаторы: SMA, Variance
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
