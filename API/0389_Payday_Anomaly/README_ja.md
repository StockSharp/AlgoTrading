# 給料日アノマリー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、典型的な給料支払日の前後に広範な市場ETFを保有することで「給料日」効果を利用します。ETFは月末の2営業日前から新しい月の3営業日目まで保有し、給与拠出による資金流入を捉えます。

それ以外の月の期間はポートフォリオを現金で保持します。日足ローソク足がウィンドウを決定し、成行注文でポジションを調整します。

## 詳細

- **銘柄**: 広範な市場ETF。
- **ウィンドウ**: 月末2日前から翌月の3営業日目まで。
- **ポジショニング**: ウィンドウ中はロング、それ以外はフラット。
- **データ**: 日足ローソク足。
- **リスク管理**: 注文価値が `MinTradeUsd` を下回る場合は取引をスキップ。
