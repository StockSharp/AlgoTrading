# EMA50 クロスオーバー 月次 DCA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

EMA50 クロスオーバー 月次 DCA は、価格が50期間EMAを上回って終値をつけた際に買い、毎月追加ポジションを積み立てます。未投資のDCA金額は現金として保存され、トレンドが再開した時点で投入されます。

価格がEMAを下回るとポジションを手仕舞いし、戦略は売りに転じます。

## 詳細

- **エントリー条件**: 終値 > EMA(50)
- **ロング/ショート**: ロングのみ
- **エグジット条件**: 価格がEMA(50)を下抜け
- **ストップ**: いいえ
- **デフォルト値**:
  - `CandleType` = 1週間
  - `DcaAmount` = 100000
  - `StartDate` = 1980-01-01
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング
  - インジケーター: EMA
  - ストップ: いいえ
  - 複雑さ: 初心者
  - 時間軸: 長期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
