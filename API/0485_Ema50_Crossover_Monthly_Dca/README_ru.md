# Стратегия EMA50 Crossover Monthly DCA
[English](README.md) | [中文](README_cn.md)

Стратегия покупает, когда цена закрывается выше 50-периодной EMA, и ежемесячно добавляет позицию. Неиспользованные DCA суммы копятся в виде наличных и инвестируются при возобновлении тренда.

Выход происходит при закрытии цены ниже EMA.

## Подробности

- **Условия входа**: close > EMA(50)
- **Длинные/короткие**: только лонг
- **Условия выхода**: цена ниже EMA(50)
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = 1 неделя
  - `DcaAmount` = 100000
  - `StartDate` = 1980-01-01
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Лонг
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Начальная
  - Таймфрейм: Долгосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
