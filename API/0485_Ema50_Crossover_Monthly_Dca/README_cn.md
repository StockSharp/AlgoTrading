# EMA50 Crossover Monthly DCA 策略
[English](README.md) | [Русский](README_ru.md)

当收盘价高于50期EMA时买入，每月追加投资。未使用的DCA金额会累积为现金并在趋势恢复时投入。

当价格跌破EMA时平仓。

## 细节

- **入场条件**：收盘价 > EMA(50)
- **多空方向**：仅做多
- **出场条件**：价格跌破 EMA(50)
- **止损**：无
- **默认值**：
  - `CandleType` = 1 周
  - `DcaAmount` = 100000
  - `StartDate` = 1980-01-01
- **过滤器**：
  - 类别：趋势跟随
  - 方向：多头
  - 指标：EMA
  - 止损：否
  - 复杂度：初级
  - 时间框架：长期
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
