# Стратегия One Click Close All
[English](README.md) | [中文](README_cn.md)

Утилита, которая закрывает все открытые позиции и при необходимости отменяет отложенные ордера. Полезна для быстрого закрытия портфеля одной командой.

## Подробности

- **Назначение**: Закрытие позиций и отмена ордеров
- **Условия входа**: отсутствуют — действия выполняются сразу после запуска
- **Длинные/Короткие**: обе стороны, в зависимости от текущего портфеля
- **Условия выхода**: не применимо (стратегия завершает работу после выполнения)
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RunOnCurrentSecurity` = true
  - `CloseOnlyManualTrades` = true
  - `DeletePendingOrders` = false
  - `MaxSlippage` = 5
- **Фильтры**:
  - Категория: Утилита
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Переменный
