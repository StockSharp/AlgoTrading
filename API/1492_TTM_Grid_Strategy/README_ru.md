# TTM Grid Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия строит сетку заявок на основе состояния TTM, вычисленного по EMA максимумов и минимумов. Сетка перезапускается при смене состояния, ордера размещаются при касании уровня сетки.

## Детали

- **Условия входа**: цена достигает уровня сетки согласно состоянию TTM.
- **Длинные/короткие**: в обе стороны.
- **Условия выхода**: нет (позиции накапливаются).
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `TtmPeriod` = 6
  - `GridLevels` = 5
  - `GridSpacing` = 0.01m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Сетка
  - Направление: Оба
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
