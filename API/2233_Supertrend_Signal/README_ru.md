# Стратегия Supertrend Signal
[English](README.md) | [中文](README_cn.md)

Стратегия открывает позиции при пересечении цены закрытия и линии SuperTrend. Покупка осуществляется, когда цена оказывается выше линии, продажа — когда цена падает ниже. Противоположные сигналы закрывают и разворачивают текущие позиции.

Индикатор SuperTrend основан на Average True Range (ATR) и следует за ценой, определяя преобладающее направление. Параметры позволяют настраивать период ATR, множитель и временной интервал свечей.

## Детали

- **Условия входа**:
  - Лонг: цена закрытия пересекает линию SuperTrend снизу вверх
  - Шорт: цена закрытия пересекает линию SuperTrend сверху вниз
- **Лонг/Шорт**: обе стороны
- **Условия выхода**:
  - Обратное пересечение SuperTrend
- **Стопы**: отсутствуют
- **Значения по умолчанию**:
  - `AtrPeriod` = 5
  - `Multiplier` = 3
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе стороны
  - Индикаторы: SuperTrend (на основе ATR)
  - Стопы: Нет
  - Сложность: Начальный уровень
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
