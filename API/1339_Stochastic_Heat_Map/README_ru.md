# Стратегия тепловой карты Stochastic
[English](README.md) | [中文](README_zh.md)

Стратегия Stochastic Heat Map усредняет ряд стохастиков с увеличивающимися периодами.
Полученный показатель сглаживается ещё раз, образуя быструю и медленную линии.
Лонг открывается при пересечении быстрой линии выше медленной, шорт — при обратном пересечении.

## Детали

- **Условия входа**: пересечение fast/slow
- **Лонг/шорт**: оба направления
- **Условия выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `Increment` = 10
  - `SmoothFast` = 2
  - `SmoothSlow` = 21
  - `PlotNumber` = 28
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Стохастик
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
