# 随机指标热力图策略
[English](README.md) | [Русский](README_ru.md)

该策略对多个递增周期的随机指标取平均值，并再次平滑得到快线和慢线。
当快线向上穿越慢线时做多，反向穿越时做空。

## 细节

- **入场条件**：快线与慢线交叉
- **多/空**：双向
- **退出条件**：相反信号
- **止损**：无
- **默认值**：
  - `CandleType` = 15 分钟
  - `Increment` = 10
  - `SmoothFast` = 2
  - `SmoothSlow` = 21
  - `PlotNumber` = 28
- **过滤条件**：
  - 类别: 趋势
  - 方向: 双向
  - 指标: 随机指标
  - 止损: 无
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
