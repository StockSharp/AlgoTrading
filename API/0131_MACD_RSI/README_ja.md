# MACD RSI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
MACD RSIは、MACDのモメンタムとRSIの買われすぎ・売られすぎのシグナルを組み合わせます。
両方のインジケーターが一致したとき、持続的な動きの確率が高まります。

テストでは年平均リターンが約130%であることが示されています。株式市場で最も効果的です。

この戦略は、MACDが上クロスしてRSIが売られすぎ圏から上昇したときにロングで入り、MACDが下クロスしてRSIが買われすぎ圏から下落したときにショートで売ります。

エントリー後にインジケーターが乖離した場合の損失を抑えるため、価格のパーセンテージベースのストップを使用します。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: MACD, RSI
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
