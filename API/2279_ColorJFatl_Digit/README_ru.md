# Стратегия ColorJFatl Digit
[English](README.md) | [中文](README_cn.md)

Стратегия использует направление наклона скользящей средней Jurik (JMA) для генерации сигналов. JMA приближает индикатор "ColorJFatl_Digit" из исходного эксперта MQL5. Длинная позиция открывается при развороте JMA вверх, короткая — при развороте вниз. Противоположные позиции закрываются при смене наклона.

Система торгует в обе стороны и по умолчанию не использует жёстких стопов. Подходит для инструментов, где развороты тренда можно отследить сглаженной адаптивной средней.

## Детали

- **Критерии входа**:
  - **Лонг**: наклон JMA меняется с отрицательного на положительный.
  - **Шорт**: наклон JMA меняется с положительного на отрицательный.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - **Лонг**: наклон JMA становится отрицательным.
  - **Шорт**: наклон JMA становится положительным.
- **Стопы**: нет по умолчанию.
- **Значения по умолчанию**:
  - `JMA Length` = 5
  - `Timeframe` = 4 часа
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Простая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
