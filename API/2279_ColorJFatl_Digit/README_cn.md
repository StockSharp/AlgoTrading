# ColorJFatl Digit 策略
[English](README.md) | [Русский](README_ru.md)

该策略利用 Jurik 移动平均线 (JMA) 的斜率方向来生成交易信号。JMA 近似于原始 MQL5 专家中的 "ColorJFatl_Digit" 指标。当 JMA 向上转折时开多单，向下转折时开空单。斜率反向时平掉相反头寸。

系统可双向交易，默认不使用硬性止损。适用于利用平滑自适应均线捕捉趋势变化的工具。

## 细节

- **入场条件**:
  - **多头**: JMA 斜率从负转正。
  - **空头**: JMA 斜率从正转负。
- **多空方向**: 双向。
- **出场条件**:
  - **多头**: JMA 斜率转为负值。
  - **空头**: JMA 斜率转为正值。
- **止损**: 默认无。
- **默认参数**:
  - `JMA Length` = 5
  - `Timeframe` = 4 小时
- **筛选器**:
  - 类别: 趋势跟随
  - 方向: 双向
  - 指标: 单一
  - 止损: 否
  - 复杂度: 简单
  - 时间框架: 中期
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
