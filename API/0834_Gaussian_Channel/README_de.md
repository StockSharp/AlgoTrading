# Gaussian Channel-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie baut einen Kanal mithilfe eines exponentiellen gleitenden Durchschnitts und der Average True Range auf. Trades werden eröffnet, wenn der Preis die ausgewählte Linie innerhalb eines Rückblickfensters kreuzt. Positionen können bei entgegengesetzten Signalen umgekehrt werden.
