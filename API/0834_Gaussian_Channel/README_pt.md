# Estratégia de Canal Gaussian
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia constrói um canal usando uma média móvel exponencial e o intervalo verdadeiro médio. As operações são abertas quando o preço cruza a linha selecionada dentro de uma janela de retrocesso. As posições podem ser revertidas em sinais opostos.
