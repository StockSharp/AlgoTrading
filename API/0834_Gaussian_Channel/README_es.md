# Estrategia de Canal Gaussian
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia construye un canal utilizando una media móvil exponencial y el rango verdadero promedio. Las operaciones se abren cuando el precio cruza la línea seleccionada dentro de una ventana de retroceso. Las posiciones pueden revertirse ante señales opuestas.
