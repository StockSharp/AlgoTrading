# Стратегия Bober XM
[English](README.md) | [中文](README_cn.md)

Стратегия Bober XM использует двойной канал на основе вычислений Кельтнера. Вход осуществляется при пробое границы канала с подтверждением от взвешенной средней и силы тренда через ADX. Выход выполняется, когда OBV пересекает свою среднюю при сохраняющемся сильном ADX.

Подходит трейдерам, которым нужны подтверждения импульса и выходы на основе объема.

## Детали

- **Условия входа**:
  - **Лонг**: `Close > UpperBand && Close > WMA && ADX > Threshold`
  - **Шорт**: `Close < LowerBand && Close < WMA && ADX > Threshold`
- **Направление**: оба
- **Условия выхода**:
  - **Лонг**: `OBV < OBV_MA && ADX > Threshold`
  - **Шорт**: `OBV > OBV_MA && ADX > Threshold`
- **Стопы**: процентный стоп по `StopLossPercent`
- **Значения по умолчанию**:
  - `EmaPeriod` = 20
  - `AtrPeriod` = 10
  - `KeltnerMultiplier` = 1.5m
  - `WmaPeriod` = 15
  - `ObvMaPeriod` = 22
  - `AdxPeriod` = 60
  - `AdxThreshold` = 35m
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Канал Кельтнера, WMA, OBV, ADX
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
