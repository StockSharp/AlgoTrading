# Billy Expert プルバックバイヤー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
Billy Expert はMetaTrader 5エキスパートアドバイザー「Billy expert」から変換されたロングオンリーのプルバック戦略です。下降する高値の連続を待ち、ベース時間軸でオープンし、その後異なる上位時間軸で計算された2つのストキャスティクスオシレーターからの強気の確認を確認します。両方のオシレーターが上昇モメンタムが存在することに同意すると、システムは設定可能な制限まで新しいロングポジションを追加します。

変換はStockSharp高レベルAPIガイドラインに従います。取引ボリューム、最大同時エントリー数、保護ストップとテイクプロフィットは戦略パラメーターを通じて制御され、元のMQLロジックに合わせた動作を実現します。

## 仕組み
1. 一次ローソク足シリーズ（デフォルト1分）とストキャスティクスオシレーター用の2つの上位時間軸（デフォルト5分と6分）をサブスクライブします。
2. ベース時間軸での最新4本の完成したローソク足を追跡します。有効なプルバックには、それら4本のバーにわたって厳密に下降する高値*と*始値が必要です。
3. 高速と低速のストキャスティクスオシレーターを評価します。戦略は、各オシレーターの最新と前回の%K値が%Dを上回り続けることを要求し、両方の時間軸でモメンタムがすでに上昇に転換したことを示します。
4. プルバックとモメンタムフィルターが確認され、オープンロングトレード数が`MaxPositions`未満の場合、`TradeVolume`のサイズで市場買い注文を送信します。
5. オプションのストップロスとテイクプロフィットレベルはpipsで表現され、インスツルメントの`PriceStep`を使用して絶対価格距離に変換されます。いずれかの距離がゼロに設定されると、対応する保護注文は省略されます。
6. ポジションはこれらの保護レベルを通じてのみ閉じられ、元のエキスパートアドバイザーの動作を模倣します。

## パラメーター
- `TradeVolume` – 各エントリーの注文サイズ（デフォルト `0.01`）。
- `StopLossPips` – pipsでのストップ距離（デフォルト `0`、無効）。
- `TakeProfitPips` – pipsでの利益目標（デフォルト `32`）。
- `MaxPositions` – 最大同時ロングトレード数（デフォルト `6`）。
- `Signal Candle` – 価格パターン用のベース時間軸（デフォルト `1` 分）。
- `Fast Stochastic TF` – 高速オシレーター用の時間軸（デフォルト `5` 分）。
- `Slow Stochastic TF` – 低速オシレーター用の時間軸（デフォルト `6` 分）。高速時間軸より長くなければならない。

## フィルターと動作
- **方向**：ロングのみ。
- **エントリートリガー**：下降する始値と高値を持つ4バーのプルバック。
- **モメンタムフィルター**：現在と前回の読みで%Kが%Dより上にあるデュアルストキャスティクスオシレーター。
- **リスク管理**：オプションのpipsベースのストップロスとテイクプロフィット。トレーリングロジックなし。
- **ポジションサイジング**：エントリーごとに固定`TradeVolume`、`MaxPositions`で上限。
- **マーケット**：小数pipで引用されるFXペア向けに設計されていますが、有効な`PriceStep`を提供する任意のインスツルメントで動作します。

## 使用上の注意
- `Fast Stochastic TF`が`Slow Stochastic TF`よりも厳密に短いことを確認してください、そうでないと戦略は起動時に停止します。
- 決済が完全に保護注文に依存しているため、インスツルメントのボラティリティに合わせて`StopLossPips`と`TakeProfitPips`を調整します。
- 戦略は弱気シグナルを無視し、スケールアウトしません；追加の保護にはポートフォリオレベルのリスクコントロールを使用してください。
- バックテストでは、最初のトレードの前に両方のストキャスティクスオシレーターが形成できるよう十分なウォームアップローソク足を提供してください。
