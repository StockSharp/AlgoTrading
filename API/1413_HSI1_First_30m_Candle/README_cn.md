# HSI1 First 30m Candle Strategy
[Русский](README_ru.md) | [English](README.md)

在15分钟图上交易首个30分钟区间的突破，每天只允许一次交易。

## 细节
- **入场条件**: 在交易时段内价格突破首个30分钟的高/低点。
- **多空方向**: 双向，可选择。
- **出场条件**: 基于区间的止盈或止损。
- **止损**: 有。
- **默认值**:
  - `RiskReward` = 1
  - `CandleType` = TimeSpan.FromMinutes(15)
- **过滤**:
  - 类别: 突破
  - 方向: 双向
  - 指标: 价格
  - 止损: 有
  - 复杂度: 中等
  - 周期: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险: 中等
