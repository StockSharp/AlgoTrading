# ADX for BTC
[English](README.md) | [中文](README_cn.md)

Стратегия использует индекс среднего направленного движения (ADX) с необязательным фильтром SMA для поиска сильных движений в Bitcoin.

Тестирование показывает среднегодичную доходность около 80%\. Стратегия лучше всего работает на рынке криптовалют.

Система покупает, когда ADX пересекает уровень входа снизу и фильтр тренда подтверждает рост. Позиция закрывается, когда ADX опускается ниже уровня выхода.

## Детали

- **Условия входа**: ADX пересекает `EntryLevel` снизу и (если включено) быстрая SMA > медленной SMA.
- **Длинные/короткие**: Только длинные позиции.
- **Условия выхода**: ADX пересекает `ExitLevel` сверху вниз.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `EntryLevel` = 14m
  - `ExitLevel` = 45m
  - `SmaFilter` = true
  - `SmaLength` = 200
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: ADX, SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
