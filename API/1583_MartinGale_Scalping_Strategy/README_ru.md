# Стратегия MartinGale Scalping
[English](README.md) | [中文](README_cn.md)

Пересечение SMA(3) и SMA(8) запускает входы с мартингейловым пирамидингом. Дополнительные заявки добавляются на каждом баре до срабатывания стопа или тейка.

## Детали

- **Условия входа**: `SMA3` выше `SMA8` для лонга, ниже для шорта; новые входы добавляются пока сигнал сохраняется.
- **Длинные/Короткие**: регулируется параметром `TradingMode`.
- **Условия выхода**: достижение `TakeProfit` или `StopLoss` и противоположное положение SMA.
- **Стопы**: да, на основе медленной SMA.
- **Значения по умолчанию**:
  - `FastLength` = 3
  - `SlowLength` = 8
  - `TakeProfit` = 1.03
  - `StopLoss` = 0.95
  - `TradingMode` = Long
  - `CandleType` = 5 минут
  - `MaxPyramids` = 5
- **Фильтры**:
  - Категория: Trend
  - Направление: Задаётся
  - Индикаторы: SMA
  - Стопы: Да
  - Сложность: Новичок
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Высокий
