# Не заставляй меня пересекаться
[English](README.md) | [中文](README_cn.md)

Стратегия пересечения EMA с вертикальным смещением.

## Подробности

- **Критерии входа**:
  - **Long**: Сдвинутая короткая EMA пересекает снизу сдвинутую длинную EMA.
  - **Short**: Сдвинутая короткая EMA пересекает сверху сдвинутую длинную EMA.
- **Long/Short**: Обе стороны.
- **Критерии выхода**: Обратное пересечение.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `ShortEmaLength` = 9
  - `LongEmaLength` = 21
  - `ShiftAmount` = -50
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Трендовая
  - Направление: Обе
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
