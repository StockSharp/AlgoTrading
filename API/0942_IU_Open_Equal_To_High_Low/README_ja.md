# IU 始値が高値・安値と等しい戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

その日の最初のローソク足の始値が安値と等しい場合にロングエントリーし、始値が高値と等しい場合にショートエントリーします。ストップロスは前のローソク足を使用し、テイクプロフィットは `RiskReward` 比に基づいて計算されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 最初のローソク足の始値がその安値と等しい。
  - **ショート**: 最初のローソク足の始値がその高値と等しい。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - ロングは前のローソク足の安値でストップロス、ショートは前のローソク足の高値でストップロス。
  - テイクプロフィットはエントリー価格から `RiskReward` を使用して計算。
- **ストップ**: はい。
- **デフォルト値**:
  - `RiskReward` = 2.
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: 価格アクション
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
