# Estrategia de Renombrado Final
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Estrategia utilitaria que renombra todos los archivos `.txt` de un directorio añadiendo prefijos secuenciales de cuatro dígitos. Expone parámetros para el número inicial y el directorio a procesar.

## Detalles

- **Criterios de entrada**: ninguno
- **Largo/Corto**: ninguno
- **Criterios de salida**: completa el renombrado
- **Stops**: no
- **Valores predeterminados**:
  - `StartNumber` = 2888
  - `DirectoryPath` = "TradingView"
- **Filtros**: ninguno
