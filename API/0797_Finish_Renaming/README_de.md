# Strategie zur abschließenden Umbenennung
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Hilfsstrategie, die alle `.txt`-Dateien in einem Verzeichnis umbenennt, indem sequenzielle vierstellige Präfixe hinzugefügt werden. Sie bietet Parameter für die Startnummer und das zu verarbeitende Verzeichnis.

## Details

- **Einstiegskriterien**: keine
- **Long/Short**: keine
- **Ausstiegskriterien**: Umbenennung abgeschlossen
- **Stops**: nein
- **Standardwerte**:
  - `StartNumber` = 2888
  - `DirectoryPath` = "TradingView"
- **Filter**: keine
