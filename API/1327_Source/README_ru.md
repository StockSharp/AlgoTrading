# Стратегия Source
[English](README.md) | [中文](README_cn.md)

Source открывает длинные позиции, когда свеча закрывается выше открытия, и короткие — когда закрывается ниже. Риск управляется параметрами стоп-лосса, тейк-профита и трейлинг-стопа.

## Детали

- **Критерий входа**: long при закрытии выше открытия, short при закрытии ниже открытия
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал или срабатывание стопов
- **Стопы**: стоп-лосс, тейк-профит и трейлинг-стоп по выбору
- **Значения по умолчанию**:
  - `SL %` = 1
  - `TP %` = 3
  - `Trail Points %` = 3
  - `Trail Offset %` = 1
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе стороны
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
