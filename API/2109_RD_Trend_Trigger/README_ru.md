# Стратегия RD Trend Trigger
[English](README.md) | [中文](README_cn.md)

Стратегия RD Trend Trigger использует осциллятор RD-TrendTrigger для фиксации разворотов тренда или пробоя уровней в зависимости от выбранного режима. В режиме twist сделки следуют за изменением направления осциллятора; в режиме disposition сделки открываются при пересечении осциллятором заданных уровней.

## Детали

- **Условия входа**:
  - **Режим twist**: открывать длинную позицию при развороте осциллятора вверх, короткую — при развороте вниз.
  - **Режим disposition**: открывать длинную позицию при росте осциллятора выше `HighLevel`; короткую — при падении ниже `LowLevel`.
- **Лонг/Шорт**: оба.
- **Условия выхода**:
  - Противоположные сигналы или явные условия выхода в режиме disposition, когда осциллятор поднимается выше `LowLevel`.
- **Стопы**: отсутствуют по умолчанию; защиту можно включить отдельно.
- **Значения по умолчанию**:
  - `Regress` = 15
  - `T3Length` = 5
  - `T3VolumeFactor` = 0.7
  - `HighLevel` = 50
  - `LowLevel` = -50
  - `Mode` = Twist
  - `CandleType` = 4-hour candles
- **Фильтры**:
  - Категория: следование тренду
  - Направление: Лонг & Шорт
  - Индикаторы: пользовательский RD-TrendTrigger (основан на high/low и Tillson T3)
  - Стопы: по желанию
  - Сложность: средняя
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
