# Laguerre ADX戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はAverage Directional Index (ADX)インジケーターの+DIと-DIコンポーネントにLaguerreフィルターを適用します。平滑化により方向性の動きのノイズが低減され、買い手と売り手の優位性の急激な変化が強調されます。Laguerreで平滑化された+DIが平滑化された-DIを下回るクロスをした場合、システムは強気の反転を期待してロングポジションに入ります。逆に、平滑化された+DIが平滑化された-DIを上回るクロスをした場合、システムはショートポジションを開きます。

現在の平滑化された値が反対側が主導権を握ったことを示すと、ポジションはクローズされます。この手法は逆張りアプローチとして設計されており、方向性インデックスの短期的な極値を利用します。

## 詳細

- **エントリー条件**:
  - **ロング**: Laguerre +DIがLaguerre –DIを下回るクロス。
  - **ショート**: Laguerre +DIがLaguerre –DIを上回るクロス。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - **ロング**: Laguerre –DIがLaguerre +DIを上回る。
  - **ショート**: Laguerre +DIがLaguerre –DIを上回る。
- **ストップ**: 固定ストップなし、デフォルトのポジション保護のみ。
- **デフォルト値**:
  - `ADX Period` = 14.
  - `Gamma` = 0.764 (Laguerre平滑化係数).
  - `Candle Type` = 4時間の時間軸.
- **フィルター**:
  - カテゴリ: 逆張り
  - 方向: 両方
  - インジケーター: ADX
  - ストップ: いいえ
  - 複雑さ: 中
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
