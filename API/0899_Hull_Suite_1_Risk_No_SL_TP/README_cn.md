# Hull Suite策略 – 1%风险，无SL/TP
[English](README.md) | [Русский](README_ru.md)

Hull Suite策略当所选Hull移动平均值相比两根K线前上涨时开多，当其下降时开空。策略不使用止损和止盈。

## 细节

- **入场条件**：
  - **多头**：Hull数值高于两根K线前。
  - **空头**：Hull数值低于两根K线前。
- **多空方向**：双向。
- **离场条件**：出现反向信号时反转仓位。
- **止损**：无。
- **默认值**：
  - `HullLength` = 55
  - `Mode` = Hma
- **过滤器**：
  - 分类：趋势跟随
  - 方向：双向
  - 指标：HMA、EHMA、THMA
  - 止损：无
  - 复杂度：低
  - 时间框架：5m
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：低
