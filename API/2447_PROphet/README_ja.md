# PROphet 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

PROphet戦略は、指定されたトレード時間中にシグナルを生成するために、最後の3本の完成ローソク足の価格レンジを評価します。カスタム関数がレンジをユーザー定義の係数と組み合わせます。関数が正であれば、戦略は対応する方向でポジションを開きます。

ロングトレードは係数`X1..X4`と`BuyStopPoints`で定義されるトレーリングストップを使用します。ショートトレードは係数`Y1..Y4`と`SellStopPoints`を使用します。ストップは、価格がスプレッドにストップ距離の2倍を加えた分だけポジションの有利な方向に動くとトレーリングされます。ポジションは18:00以降またはトレーリングストップに達したときにクローズされます。

## 詳細

- **エントリー条件**
  - **ロング**: `Qu(X1,X2,X3,X4) > 0` かつ現在時刻が10時から18時の間。
  - **ショート**: `Qu(Y1,Y2,Y3,Y4) > 0` かつ現在時刻が10時から18時の間。
- **エグジット条件**
  - **ロング**: 時刻 > 18 または最良売り気配値がトレーリングストップを下回る。
  - **ショート**: 時刻 > 18 または最良買い気配値がトレーリングストップを上回る。
- **パラメーター**
  - `EnableBuy` – ロングポジションの開設を許可。
  - `EnableSell` – ショートポジションの開設を許可。
  - `X1, X2, X3, X4` – ロングシグナル関数の係数。
  - `Y1, Y2, Y3, Y4` – ショートシグナル関数の係数。
  - `BuyStopPoints` – ロングトレードのトレーリングストップ距離（ポイント）。
  - `SellStopPoints` – ショートトレードのトレーリングストップ距離（ポイント）。
  - `CandleType` – 計算に使用するローソク足タイプ（デフォルト5分）。
- **フィルター**
  - カテゴリ: イントラデイ
  - 方向: 両方
  - インジケーター: なし
  - ストップ: トレーリング
  - 複雑さ: 中程度
  - 時間軸: 短期
