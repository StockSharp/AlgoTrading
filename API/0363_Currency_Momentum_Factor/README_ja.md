# 通貨モメンタム・ファクター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このファクター戦略は、中期モメンタムによって通貨をランク付けし、ロング/ショートポートフォリオを構築します。ルックバック期間中に最も強いパフォーマンスを示した通貨が買われ、最も弱い通貨が均等サイズでショートされます。

モメンタムは日足ローソク足を使って評価され、毎月最初の取引日にポートフォリオがリバランシングされます。最低USD金額を下回る注文はノイズを減らすために無視されます。

## 詳細

- **ユニバース**: 通貨ペアまたはETFのリスト。
- **シグナル**: モメンタムが最も高い `K` 通貨をロング、最も低い `K` 通貨をショート。
- **ルックバック**: `Lookback` 本の日足ローソク足で計算されるリターン（デフォルト252）。
- **リバランス**: 月次。
- **ポジショニング**: ロング/ショート、ドル中立。
- **パラメーター**:
  - `Universe` – 取引可能な通貨シンボル。
  - `Lookback` – モメンタム用のローソク足の本数。
  - `K` – ロングおよびショートするアセット数。
  - `MinTradeUsd` – 最低取引サイズ。
  - `CandleType` – ローソク足の時間軸（デフォルト: 1日）。
- **注意**: このサンプルはデモ目的のため、実際のモメンタム計算を省略しています。
