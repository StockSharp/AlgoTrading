# リスクに基づくポジションサイズ計算戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

口座リスクとストップロスのパーセンテージに基づいてトレードのサイズを決定する方法を示します。ポジションサイズの計算ロジックを示すため、エントリーはランダムです。

## 詳細

- **エントリー条件**:
  - **ロング**: 333本ごとのバー。
  - **ショート**: 444本ごとのバー。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - ストップロスのみ。
- **ストップ**: ストップロス。
- **デフォルト値**:
  - `Stop Loss %` = 10
  - `Risk Value` = 2
  - `Risk Is Percent` = true
  - `Long Period` = 333
  - `Short Period` = 444
- **フィルター**:
  - カテゴリ: Risk Management
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
