# Стратегия Calculation Position Size Based on Risk
[English](README.md) | [中文](README_cn.md)

Демонстрирует расчет объема позиции исходя из риска счёта и процента стоп-лосса. Входы случайные, чтобы показать логику расчёта.

## Детали

- **Условия входа**:
  - **Long**: каждый 333-й бар.
  - **Short**: каждый 444-й бар.
- **Long/Short**: Оба направления.
- **Условия выхода**:
  - Только стоп-лосс.
- **Стопы**: Стоп-лосс.
- **Значения по умолчанию**:
  - `Stop Loss %` = 10
  - `Risk Value` = 2
  - `Risk Is Percent` = true
  - `Long Period` = 333
  - `Short Period` = 444
- **Фильтры**:
  - Категория: Risk Management
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
