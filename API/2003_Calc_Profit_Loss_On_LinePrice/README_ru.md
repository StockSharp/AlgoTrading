# Стратегия Calc Profit Loss On LinePrice
[English](README.md) | [中文](README_cn.md)

Утилитарная стратегия, рассчитывающая потенциальную прибыль или убыток
по текущей позиции на заданном ценовом уровне. Стратегия отслеживает
закрытые свечи и при наличии открытой позиции вычисляет результат,
который был бы получен при закрытии по указанной цене. Сделки не совершаются,
значение выводится в журнал стратегии.

## Подробности

- **Критерий входа**: Нет
- **Длинная/Короткая**: Н/Д
- **Критерий выхода**: Н/Д
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `LinePrice` = 0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Утилита
  - Направление: Н/Д
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Минимальный
