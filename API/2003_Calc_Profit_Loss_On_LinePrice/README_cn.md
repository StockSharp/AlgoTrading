# Calc Profit Loss On LinePrice 策略
[English](README.md) | [Русский](README_ru.md)

该策略用于在指定价格水平评估当前持仓的潜在盈亏。
当存在持仓并且收到完成的K线时，它计算如果在设定价格平仓，
将得到的理论结果。策略本身不发出任何订单，只在日志中记录结果。

## 详情

- **入场条件**：无（工具型策略）
- **多空方向**：不适用
- **退出条件**：不适用
- **止损**：无
- **默认值**：
  - `LinePrice` = 0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **过滤器**：
  - 分类：工具
  - 方向：不适用
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：最低
