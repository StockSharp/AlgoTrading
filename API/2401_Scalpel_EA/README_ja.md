# Scalpel EA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTrader向けに書かれたオリジナルの*Scalpel EA*の簡略化された変換です。
商品チャンネル指数（CCI）フィルターとマルチタイムフレームブレイクアウト分析を組み合わせます。複数の上位タイムフレームが動きを確認した際に、短期モメンタムの方向で取引することを目的としています。

## ロジック

1. **インジケーター** – CCIはプライマリタイムフレームで計算されます。CCI値がゼロ周辺の設定可能なバンド内に留まっている場合のみ取引が許可されます。
2. **トレンド確認** – 30分、1時間、4時間のローソク足で、最新の高値と安値が以前のものと比較されます。
   - ロング取引には3つのタイムフレームすべてで上昇する安値が必要です。
   - ショート取引には3つのタイムフレームすべてで下落する高値が必要です。
3. **ブレイクアウト** – プライマリローソク足の終値が前のローソク足の高値（ロング用）または安値（ショート用）を突破したときにエントリーが発動します。
4. **リスク管理** – `StartProtection`は価格単位で計測された固定のテイクプロフィットとストップロスを配置します。

## パラメーター

| 名前 | 説明 |
| ---- | ----------- |
| `CciPeriod` | 商品チャンネル指数の期間。 |
| `CciLimit` | CCIの絶対閾値。エントリーは±リミット内でのみ許可されます。 |
| `TakeProfit` | 価格単位でのテイクプロフィット値。 |
| `StopLoss` | 価格単位でのストップロス値。 |
| `CandleType` | 取引用のプライマリタイムフレーム（デフォルト1分）。 |

## 注記

- 戦略は上位タイムフレームのトレンドを評価するために追加の30分、1時間、4時間のローソク足を購読します。
- ボリュームはベースクラスの`Strategy.Volume`プロパティから取得されます。
- 一度に1つのポジションのみがオープンされます。反対のシグナルは既存のポジションをクローズし、新しいポジションをオープンします。
