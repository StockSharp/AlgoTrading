# CCI 钩形反转策略
[English](README.md) | [Русский](README_ru.md)

当商品通道指数（CCI）在极端读数后钩回时，该策略用作入场触发。指标超过+100或低于-100后动能常迅速减弱并回撤。

当 CCI 在超卖区转头向上且价格仍略创新低时做多；在超买区向下弯而价格再创新高时做空。

每笔交易设置较小的固定止损，当 CCI 向反方向钩回或触发止损时离场。

## 细节

- **入场条件**：指标信号。
- **多/空**：双向。
- **退出条件**：止损或相反信号。
- **止损**：是，按百分比。
- **默认值**：
  - `CandleType` = 15 分钟
  - `StopLoss` = 2%
- **过滤条件**：
  - 类别: 反转
  - 方向: 双向
  - 指标: CCI
  - 止损: 有
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
