# 条件テーブル付きインジケーターテスト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は最新の終値をユーザー定義のレベルと比較し、条件が満たされた場合に成行注文を執行します。ロングとショートそれぞれにパラメーターで制御される個別のエントリーおよびエグジットルールがあります。

## 詳細

- **エントリー条件**:
  - **ロング**: 有効なロング条件が真である。
  - **ショート**: 有効なショート条件が真である。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - **ロング**: 有効なロング決済条件が真である。
  - **ショート**: 有効なショート決済条件が真である。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `LongOperator` = `>`
  - `CloseLongOperator` = `<`
  - `ShortOperator` = `<`
  - `CloseShortOperator` = `>`
- **フィルター**:
  - カテゴリ: その他
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
