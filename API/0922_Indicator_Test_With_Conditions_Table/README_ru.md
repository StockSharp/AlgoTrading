# Стратегия проверки условий с таблицей
[English](README.md) | [中文](README_cn.md)

Эта стратегия сравнивает цену закрытия с заданными уровнями и выполняет рыночные заявки при выполнении условий. Для длинной и короткой стороны предусмотрены отдельные параметры входа и выхода.

## Детали

- **Критерии входа**:
  - **Длинная**: включённое длинное условие истинно.
  - **Короткая**: включённое короткое условие истинно.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - **Длинная**: включённое условие закрытия длинной позиции истинно.
  - **Короткая**: включённое условие закрытия короткой позиции истинно.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `LongOperator` = `>`
  - `CloseLongOperator` = `<`
  - `ShortOperator` = `<`
  - `CloseShortOperator` = `>`
- **Фильтры**:
  - Категория: Другое
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Простая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
