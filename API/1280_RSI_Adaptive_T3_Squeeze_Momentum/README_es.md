# Estrategia RSI-Adaptativo T3 + Squeeze Momentum
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Combina un filtro de tendencia T3 de Tillson adaptado al RSI con la técnica de Squeeze Momentum. La longitud del T3 se ajusta entre los valores mínimo y máximo en función del RSI. Una posición larga se abre cuando el T3 adaptativo gira al alza durante una liberación del squeeze con momentum positivo. Las entradas cortas ocurren cuando el T3 gira a la baja con momentum negativo después de que termina el squeeze.
