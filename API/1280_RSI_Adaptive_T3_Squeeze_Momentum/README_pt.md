# Estratégia RSI-Adaptativo T3 + Squeeze Momentum
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Combina um filtro de tendência T3 de Tillson adaptado ao RSI com a técnica de Squeeze Momentum. O comprimento do T3 se ajusta entre os valores mínimo e máximo com base no RSI. Uma posição comprada é aberta quando o T3 adaptativo vira para cima durante uma liberação do squeeze com momentum positivo. Entradas vendidas ocorrem quando o T3 vira para baixo com momentum negativo após o término do squeeze.
