# RSI適応型T3 + スクイーズモメンタム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSI適応型のTillson T3トレンドフィルターとスクイーズモメンタム手法を組み合わせます。T3の長さはRSIに基づいて最小値と最大値の間で調整されます。スクイーズ解放中にモメンタムが正の状態で適応型T3が上向きに転換したときにロングポジションを開きます。スクイーズ終了後にモメンタムが負の状態でT3が下向きに転換したときにショートエントリーを行います。
