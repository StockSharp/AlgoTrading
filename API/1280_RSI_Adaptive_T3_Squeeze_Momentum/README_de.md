# RSI-Adaptiver T3 + Squeeze-Momentum-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Kombiniert einen RSI-adaptiven Tillson-T3-Trendfilter mit der Squeeze-Momentum-Technik. Die T3-Länge passt sich basierend auf dem RSI zwischen dem Mindest- und Höchstwert an. Eine Long-Position wird eröffnet, wenn der adaptive T3 während einer Squeeze-Auflösung mit positivem Momentum nach oben dreht. Short-Einstiege erfolgen, wenn der T3 nach dem Ende des Squeeze mit negativem Momentum nach unten dreht.
