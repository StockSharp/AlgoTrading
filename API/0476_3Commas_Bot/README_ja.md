# 3Commas Bot 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

3Commas Bot戦略の簡略版です。高速EMAが低速EMAをクロスした際に取引し、ATRベースのストップでリスクを管理します。固定報酬ターゲットとオプションのATRトレーリングストップをサポートします。

## 詳細

- **エントリー条件**:
  - **ロング**: 高速EMAが低速EMAを上抜け。
  - **ショート**: 高速EMAが低速EMAを下抜け。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - ATRストップ、オプションのテイクプロフィット、報酬閾値達成後のオプションATRトレーリングストップ。
- **ストップ**: ATRベース。
- **デフォルト値**:
  - `MaLength1` = 21
  - `MaLength2` = 50
  - `AtrLength` = 14
  - `RnR` = 1
  - `RiskM` = 1
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA, ATR
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
