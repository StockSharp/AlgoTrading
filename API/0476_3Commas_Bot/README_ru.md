# Стратегия 3Commas Bot
[English](README.md) | [中文](README_cn.md)

Упрощённая версия стратегии 3Commas Bot. Открывает сделки при пересечении быстрой EMA и медленной EMA. Управление рисками реализовано через стоп по ATR, поддерживаются фиксированная цель прибыли и опциональный трейлинг-стоп на основе ATR.

## Подробности

- **Условия входа**:
  - **Лонг**: быстрая EMA пересекает медленную снизу вверх.
  - **Шорт**: быстрая EMA пересекает медленную сверху вниз.
- **Направление**: обе стороны.
- **Условия выхода**:
  - Стоп по ATR, опциональный тейк-профит, опциональный ATR-трейлинг после достижения порога прибыли.
- **Стопы**: по ATR.
- **Параметры по умолчанию**:
  - `MaLength1` = 21
  - `MaLength2` = 50
  - `AtrLength` = 14
  - `RnR` = 1
  - `RiskM` = 1
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: EMA, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
