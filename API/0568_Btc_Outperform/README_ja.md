# BTCアウトパフォーム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

週次と四半期の終値を比較します。週次価格が四半期価格を上回る場合にロング、四半期価格が上回る場合にショートポジションを建てます。

## 詳細
- **エントリー条件:**
  - **ロング:** 週次終値 > 四半期終値。
  - **ショート:** 四半期終値 > 週次終値。
- **ロング/ショート:** 両方。
- **エグジット条件:** 逆シグナル。
- **ストップ:** なし。
- **デフォルト値:** 週次 = 7日、四半期 = 90日。
