# Estratégia de Superação de Desempenho BTC
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Compara os preços de fechamento semanais e trimestrais. Fica comprado quando o preço semanal é maior do que o preço trimestral e fica vendido quando o preço trimestral é maior.

## Detalhes
- **Critérios de entrada:**
  - **Comprado:** fechamento semanal > fechamento trimestral.
  - **Vendido:** fechamento trimestral > fechamento semanal.
- **Comprado/Vendido:** Ambos.
- **Critérios de saída:** Sinal inverso.
- **Stops:** Nenhum.
- **Valores padrão:** Semanal = 7 dias, Trimestral = 90 dias.
