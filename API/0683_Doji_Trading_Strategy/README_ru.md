# Стратегия Doji Trading
[English](README.md) | [中文](README_cn.md)

Стратегия ищет свечи Doji выше экспоненциальной скользящей средней. При обнаружении такого паттерна открывается длинная позиция. Стоп-лосс ставится на минимальный минимум последних баров, а трейлинг-стоп защищает прибыль после достаточного движения цены.

## Детали

- **Критерий входа**: Свеча Doji с закрытием выше EMA.
- **Длинные/Короткие**: Только длинные.
- **Критерий выхода**: Стоп на минимуме и трейлинг-стоп.
- **Стопы**: Да, фиксированный и трейлинг.
- **Значения по умолчанию**:
  - `CandleType` = 5 минут
  - `EmaLength` = 60
  - `Tolerance` = 0.05
  - `StopBars` = 450
  - `TrailTriggerPercent` = 1
  - `TrailOffsetPercent` = 0.5
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Лонг
  - Индикаторы: EMA, свечные модели
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
