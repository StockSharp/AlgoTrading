# Doji Trading 策略
[English](README.md) | [Русский](README_ru.md)

该策略寻找位于指数移动平均线上方的十字星形态。当出现这种形态时开多仓。止损设在最近若干根K线的最低点，并在价格充分向有利方向移动后启用追踪止损。

## 细节

- **入场条件**：收盘价在EMA之上的十字星。
- **多空方向**：仅做多。
- **出场条件**：最低点止损和追踪止损。
- **止损**：是，固定和追踪。
- **默认值**：
  - `CandleType` = 5 分钟
  - `EmaLength` = 60
  - `Tolerance` = 0.05
  - `StopBars` = 450
  - `TrailTriggerPercent` = 1
  - `TrailOffsetPercent` = 0.5
- **筛选**：
  - 分类：形态
  - 方向：多头
  - 指标：EMA, K线形态
  - 止损：是
  - 复杂度：入门
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
