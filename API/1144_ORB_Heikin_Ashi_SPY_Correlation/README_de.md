# ORB Heikin Ashi SPY-Korrelations-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Verwendet SPY als korrelierten Basiswert, um eine Opening Range mit optionalen Heikin Ashi Kerzen aufzubauen. Ausbrüche zwischen 10:00 und 12:00 werden nur gehandelt, wenn das Volumen den jüngsten Durchschnitt übersteigt, und alle Positionen werden zwischen 15:50 und 15:55 geschlossen.
