# Estratégia ORB Heikin Ashi com Correlação SPY
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Utiliza o SPY como ativo correlacionado para construir um intervalo de abertura com velas Heikin Ashi opcionais. Rompimentos entre 10:00 e 12:00 são operados somente quando o volume supera a média recente, e todas as posições são encerradas entre 15:50 e 15:55.
