# Estrategia ORB Heikin Ashi con Correlación SPY
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Utiliza SPY como activo correlacionado para construir un rango de apertura con velas Heikin Ashi opcionales. Los rompimientos entre las 10:00 y las 12:00 se operan únicamente cuando el volumen supera el promedio reciente, y todas las posiciones se cierran entre las 15:50 y las 15:55.
