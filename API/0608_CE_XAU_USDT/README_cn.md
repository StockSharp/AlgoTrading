# CE XAU/USDT 策略
[English](README.md) | [Русский](README_ru.md)

当收盘价向上穿越其SMA时做多，向下穿越时做空。

## 细节

- **入场条件：** 收盘价上穿SMA做多，下穿做空。
- **多空方向：** 双向。
- **出场条件：** 反向信号。
- **止损：** 无。
- **默认值：**
  - SMA周期 = 14
  - K线周期 = 1分钟
- **过滤器：**
  - 类别：趋势
  - 方向：多空
  - 指标：SMA
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
