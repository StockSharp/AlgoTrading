# CE XAU/USDT-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie geht long, wenn der Schlusskurs seinen SMA von unten kreuzt, und short, wenn der Schlusskurs den SMA von oben kreuzt.

## Details

- **Einstiegskriterien:** Schluss kreuzt SMA von unten für Long, von oben für Short.
- **Long/Short:** Beide.
- **Ausstiegskriterien:** Umgekehrtes Signal.
- **Stops:** Keine.
- **Standardwerte:**
  - SMA-Periode = 14
  - Kerzen-Zeitrahmen = 1 Minute
- **Filter:**
  - Kategorie: Trend
  - Richtung: Long & Short
  - Indikatoren: SMA
  - Stops: Nein
  - Komplexität: Niedrig
  - Zeitrahmen: Beliebig
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
