# Стратегия CE XAU/USDT
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда цена закрытия пересекает свою SMA снизу вверх, и короткую позицию при пересечении сверху вниз.

## Детали

- **Критерии входа:** пересечение цены закрытия и SMA вверх для лонга, вниз для шорта.
- **Длинные/короткие:** обе стороны.
- **Критерии выхода:** противоположный сигнал.
- **Стопы:** нет.
- **Значения по умолчанию:**
  - Период SMA = 14
  - Таймфрейм свечей = 1 минута
- **Фильтры:**
  - Категория: Тренд
  - Направление: Лонг и Шорт
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
