# Индикатор тренда SwingArm ATR
[English](README.md) | [中文](README_cn.md)

Реализует трейлинг-стоп на основе сглаженного ATR по Уайлдеру. Линия стопа меняет направление при пробое ценой и отображаются уровни Фибоначчи между стопом и недавним экстремумом. Инструмент служит визуальным ориентиром тренда и возможных зон входа.

## Детали

- **Параметры**:
  - Период ATR.
  - Множитель ATR.
  - Тип трейла (модифицированный или классический).
  - Отображение уровней Фибоначчи.
- **Сигналы**: отсутствуют, только индикатор.
- **Назначение**: определение тренда и зон поддержки/сопротивления.
- **Фильтры**:
  - Категория: на основе волатильности
  - Направление: нет
  - Индикаторы: ATR
  - Стопы: трейлинг
  - Сложность: средняя
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
