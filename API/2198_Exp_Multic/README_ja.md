# Exp Multic戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

テクニカル指標を使用せず、主要なFX通貨ペアの固定セットを取引するマルチ通貨戦略。
各ペアについてアルゴリズムは方向とボリュームを管理します。利益を生む動きがあった後はボリュームが増加し、損失の後は方向が反転します。全体の利益または損失が指定された閾値を超えると、取引が停止され全ポジションが決済されます。

## 詳細

- **エントリー条件**:
  - ポジションがなく口座残高が `Margin` を上回っている場合、事前に定義された方向で `MinVolume` のポジションを開く。
- **ロング/ショート**: 両方、各ペアの内部方向に依存。
- **エグジット条件**:
  - 利益が `KClose * MinVolume` を超えた場合にポジションを閉じる。
  - 損失が `KChange * 現在のボリューム` を超えた場合に方向を反転して決済する。
- **ストップ**: 明示的なストップなし。リスクは利益/損失の閾値で管理。
- **デフォルト値**:
  - `Loss` = 1900
  - `Profit` = 4000
  - `Margin` = 5000
  - `MinVolume` = 0.01
  - `KChange` = 2100
  - `KClose` = 4600
- **フィルター**:
  - カテゴリ: 資金管理
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: ティックベース
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
