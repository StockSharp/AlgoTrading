# Fractals & Alligator戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Bill WilliamsのAlligatorアラインメントとフラクタルブレイクアウト、モメンタム確認レイヤー、レンジフィルターを組み合わせてMetaTraderの「Fractals & Alligator」エキスパートを複製します。元のマルチタイムフレームロジックをエミュレートするために、上位時間軸で完成したローソク足を処理します。

## 詳細
- **エントリー条件**: Alligatorのリップ、ティース、ジョーが同方向に広がり、マウスの外側に新鮮なフラクタルが形成されるのを待つ。ロングセットアップでは、クローズが最新の強気フラクタルをティースの上で突破し、直近3回のモメンタム読み取りのいずれかが買いしきい値を超える必要がある。ショートはダウンサイドのルールを反映する。
- **ロング/ショート**: ロングとショートの両取引を開く。1つのネットポジションのみが維持される；新しいシグナルは既存の露出を反転させる。
- **エグジット条件**: 反対のフラクタルが貫通されるかAlligatorのアラインメントが崩壊したとき、ポジションが閉じられる。保護注文が残りの出口を処理する。
- **ストップ**: 元の資金管理の考え方に合わせて、ストップロス、テイクプロフィット、価格ステップでのオプショナルトレーリングストップのためにStockSharpの保護注文を使用。
- **デフォルト値**: Alligatorの長さ13/8/5、シフト8/5/3、14期間モメンタム、10バーレンジルックバック、20ステップ固定ボックス（ATRフィルター無効時）、テイクプロフィット50ステップ、ストップロス20ステップ、トレーリングストップ40ステップ。
- **フィルター**: オプションのATR乗数で価格が最近のレンジから少なくとも1ATR離れていることを確認；そうでない場合は価格ステップで表現された固定ボックスが使用される。モメンタムしきい値（0.3%）は低エネルギーのブレイクアウトを抑制する。
