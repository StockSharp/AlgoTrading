# ダーククラウドピアシングCCI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、MetaTrader Expert_ADC_PL_CCI アドバイザーの StockSharp 移植です。ピアシング ラインとダーク クラウド カバーのローソク足反転の価格アクションをスキャンし、確認としてコモディティ チャネル インデックス (CCI) を使用します。有効なパターンと極端な CCI の読み取り値が検出されると、この戦略は反転の方向に市場ポジションを開き、その後、CCI が極端なゾーンから出たときに終了します。

## インジケーター
- **商品チャネル指数 (CCI):** モメンタムの極値を確認し、出口条件を生成します。
- **平均実体長 (SMA):** パターン定義内の「長い」ローソク足を検証するために、ローソク体のサイズを測定します。
- **平均終値 (SMA):** は、元の MQL ロジックで使用される移動平均を反映する単純なトレンド フィルターとして機能します。

## 取引ルール
### Entry
- **強気シグナル (ピアシングライン):**
  1. 前のローソク足は終値を超えて始まる長い弱気のローソク足でなければなりません。
  2. Latest candle must be a long bullish candle that opens below the previous low and closes within the previous body, above its midpoint but below the prior open.
  3. 短期的な下降トレンドを確認するには、古いローソク足の中間点が移動平均を下回る必要があります。
  4. 最後に完了した CCI 値は、`-EntryConfirmationLevel` (デフォルトは `50`) 以下である必要があります。
  5. ショート ポジションが存在する場合は、ロングに入る前に完全にクローズされます。
- **弱気シグナル (暗雲カバー):** 強気シグナルのミラーロジックで、長い強気ローソク足の後に長い弱気ローソク足が続き、CCI が `EntryConfirmationLevel` 以上である間にギャップアップし、前の実体を貫通し、中点以下で閉じる。

### 終了
- **ロングポジション:** CCI が上から `ExitLevel` を下回るか、`-ExitLevel` を下にクロスするとクローズされ、勢いが正常化したことが示されます。
- **ショート ポジション:** CCI が下から `-ExitLevel` または `ExitLevel` を上抜けたときにクローズされます。

### ポジションサイズ
- 基本の `Volume` プロパティを使用します。シグナルが既存のポジションを反転する必要がある場合、ストラテジーは現在のポジションの絶対サイズを注文量に自動的に追加し、完全な反転を保証します。

## パラメーター
| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| `CandleType` | 検出に使用されるローソクの種類と時間枠。 | `1H` 時間枠 |
| `CciPeriod` | 商品チャネルインデックスのルックバックの長さ。 | `49` |
| `AverageBodyPeriod` | 本体サイズ移動平均のローソク足の数。 | `11` |
| `EntryConfirmationLevel` | パターンエントリを検証する絶対的な CCI レベル。 | `50` |
| `ExitLevel` | ポジション出口をトリガーする絶対的な CCI レベル。 | `80` |

## 注意事項
- この戦略は完了したキャンドルのみを処理し、部分的な更新を無視します。
- ストップロス注文や利食い注文は自動的に設定されません。 exit は、元の Expert Advisor と同様に純粋にシグナルベースです。
- ローソク足ロジックの等値許容差はセキュリティ設定に依存するため、商品に価格ステップが設定されていることを確認してください。
