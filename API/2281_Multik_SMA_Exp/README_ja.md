# Multik SMA Exp戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略はシンプル移動平均（SMA）の傾きに基づいた逆張りアプローチを実装しています。MetaTrader 5のエキスパートアドバイザー「Multik_SMA_Exp」からポートされました。

この戦略は直近の3つのSMA値を監視します。SMAが直近の2つの完成したセグメントで下落している場合、戦略はロングポジションに入ります。SMAが2つのセグメントで上昇している場合、ショートポジションを開きます。SMAの傾きが反転するとポジションは決済されます。

## パラメーター
- **MA Period** – 単純移動平均の長さ。デフォルト：50。
- **Candle Type** – 計算に使用する足の種類。デフォルト：1分足の時間軸。

## 取引ルール
1. 各確定足でSMAを計算します。
2. 傾きを判定します：
   - `dsma1 = SMA[n-1] - SMA[n-2]`
   - `dsma2 = SMA[n-2] - SMA[n-3]`
3. エントリー：
   - `dsma1 < 0` かつ `dsma2 < 0` かつロングポジションがない場合、買い。
   - `dsma1 > 0` かつ `dsma2 > 0` かつショートポジションがない場合、売り。
4. エグジット：
   - ロングポジション保有中に `dsma1 > 0` の場合、ロングを決済。
   - ショートポジション保有中に `dsma1 < 0` の場合、ショートを決済。

新規注文のボリュームは、戦略の `Volume` に現在のポジションの絶対値を加えた値を使用し、必要に応じて完全に反転します。
