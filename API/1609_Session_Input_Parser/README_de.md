# Session-Eingabe-Parser
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Der Session-Eingabe-Parser demonstriert, wie TradingView-Session-Strings in Start-/Endzeiten und Wochentage geparst werden.

## Details

- **Zweck**: Session-Strings parsen
- **Handel**: keiner
- **Indikatoren**: keine
- **Stops**: nein
- **Standardwerte**:
  - `SessionSimple` = "0800-1530"
  - `SessionWithWeekdays` = "0800-1530:1234567"
