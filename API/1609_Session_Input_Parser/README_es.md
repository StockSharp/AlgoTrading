# Analizador de Entrada de Sesión
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

El Analizador de Entrada de Sesión demuestra cómo analizar cadenas de sesión de TradingView en horas de inicio/fin y días de la semana.

## Detalles

- **Propósito**: analizar cadenas de sesión
- **Operativa**: ninguna
- **Indicadores**: ninguno
- **Stops**: no
- **Valores predeterminados**:
  - `SessionSimple` = "0800-1530"
  - `SessionWithWeekdays` = "0800-1530:1234567"
