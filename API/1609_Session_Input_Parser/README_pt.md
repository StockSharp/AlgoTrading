# Analisador de Entrada de Sessão
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

O Analisador de Entrada de Sessão demonstra como analisar strings de sessão do TradingView em horários de início/fim e dias da semana.

## Detalhes

- **Finalidade**: analisar strings de sessão
- **Negociação**: nenhuma
- **Indicadores**: nenhum
- **Stops**: não
- **Valores padrão**:
  - `SessionSimple` = "0800-1530"
  - `SessionWithWeekdays` = "0800-1530:1234567"
