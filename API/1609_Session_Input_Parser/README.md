# Session Input Parser
[Русский](README_ru.md) | [中文](README_cn.md)

Session Input Parser demonstrates how to parse TradingView session strings into start/end times and weekdays.

## Details

- **Purpose**: parse session strings
- **Trading**: none
- **Indicators**: none
- **Stops**: no
- **Default Values**:
  - `SessionSimple` = "0800-1530"
  - `SessionWithWeekdays` = "0800-1530:1234567"
