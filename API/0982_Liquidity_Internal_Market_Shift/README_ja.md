# 流動性内部マーケットシフト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、最近の高値または安値での流動性スウィープと一致する内部市場構造のシフトを検出します。価格が流動性ラインに触れ、その後反対方向に構造をシフトしたときにトレードがオープンされます。強気・弱気のセットアップのみ、または両方に制限できます。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格が直前の弱気構造を上抜けてクローズし、最近の安値の流動性ラインに触れている。
  - **ショート**: 価格が直前の強気構造を下抜けてクローズし、最近の高値の流動性ラインに触れている。
- **ロング/ショート**: 両方向または強気のみ / 弱気のみを選択可能。
- **エグジット条件**:
  - エントリー後の反対シグナル。
  - `StopLossPips` pipsのストップロス。
  - `TakeProfitPips` pipsのオプションのテイクプロフィット。
- **ストップ**: はい、設定可能なストップロスとオプションのテイクプロフィット。
- **フィルター**:
  - 指定された時間範囲内のみで取引。
  - シグナルロックにより数バー以内の重複エントリーを防止。
