# Стратегия Panel Joke
[English](README.md) | [中文](README_cn.md)

Эта стратегия является конверсией оригинальной системы *panel-joke* из MetaTrader в StockSharp. Она сравнивает текущую свечу с предыдущей по семи ценовым параметрам (open, high, low, среднее high и low, close, среднее high/low/close и взвешенное среднее high/low/close). Каждый параметр, который вырос, увеличивает счетчик покупок; падение увеличивает счетчик продаж.

Когда параметр `Enable Autopilot` включен, стратегия автоматически открывает или разворачивает позиции в сторону лидирующего счетчика. Дополнительные индикаторы или стопы не применяются.

## Детали

- **Условия входа**:
  - **Long**: Buy counter > Sell counter.
  - **Short**: Sell counter > Buy counter.
- **Условия выхода**: Разворот при появлении обратного сигнала.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Enable Autopilot` = `true`.
  - `Candle Type` = таймфрейм 5 минут.
- **Фильтры**:
  - Категория: Price action
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
