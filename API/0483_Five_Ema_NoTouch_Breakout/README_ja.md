# 5 EMA ノータッチ・ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

5 EMA ノータッチ・ブレイクアウト戦略は、5期間EMAの片側に完全に留まるローソク足を待ちます。その後、価格がそのセットアップローソク足の極値をブレイクした際に、ブレイクアウト方向にエントリーします。ストップロスは反対側の極値に置かれ、テイクプロフィットはリスクの倍数に設定されます。

## 詳細

- **エントリー条件**:
  - ローソク足の高値がEMAより下 → ロング準備；そのローソク足の高値をブレイクしたらエントリー。
  - ローソク足の安値がEMAより上 → ショート準備；そのローソク足の安値をブレイクしたらエントリー。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - セットアップローソク足の極値にストップ。
  - `RewardRisk` × リスクのターゲット。
- **ストップ**: はい。
- **デフォルト値**:
  - `EmaPeriod` = 5
  - `RewardRisk` = 3.0
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ロング/ショート
  - インジケーター: EMA
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: 5分
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
