# Стратегия 5 EMA No-Touch Breakout
[English](README.md) | [中文](README_cn.md)

Стратегия 5 EMA No-Touch Breakout ожидает свечу, которая полностью располагается по одну сторону от экспоненциальной средней с периодом 5. Когда цена впоследствии пробивает экстремум этой свечи, стратегия входит в сторону пробоя. Стоп-лосс ставится на противоположном экстремуме, а цель рассчитывается как несколько рисков.

## Подробности

- **Условия входа**:
  - Максимум свечи ниже EMA → готовим лонг; вход при пробое максимума.
  - Минимум свечи выше EMA → готовим шорт; вход при пробое минимума.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Стоп на экстремуме сигнальной свечи.
  - Цель на расстоянии `RewardRisk` × риск.
- **Стопы**: Да.
- **Параметры по умолчанию**:
  - `EmaPeriod` = 5
  - `RewardRisk` = 3.0
- **Фильтры**:
  - Категория: Пробой
  - Направление: Лонг/шорт
  - Индикаторы: EMA
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: 5 минут
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
