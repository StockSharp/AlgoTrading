# Estrategia de Trading de Criptomonedas por Calendario Lunar
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Opera basándose en el calendario lunar. La estrategia abre una posición larga el día 12 de cada mes lunar y la cierra el día 26.

Los datos del calendario lunar para 2020–2026 están precargados usando la zona horaria Asia/Seoul.
