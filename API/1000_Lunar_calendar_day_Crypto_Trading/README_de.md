# Krypto-Handelsstrategie nach Mondkalender

Handelt auf Basis des Mondkalenders. Die Strategie eröffnet am 12. Tag eines jeden Mondmonats eine Long-Position und schließt sie am 26. Tag.

Die Mondkalenderdaten für 2020–2026 sind vorgeladen und verwenden die Zeitzone Asia/Seoul.
