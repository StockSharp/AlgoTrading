# Motion Strategy
[Русский](README_ru.md) | [中文](README_cn.md)

This example demonstrates text rotation on each candle without any trading logic. It is a direct conversion of the TradingView `Motion` script for educational purposes.

## Details

- **Entry Criteria**: None
- **Long/Short**: None
- **Exit Criteria**: Not applicable
- **Stops**: No
- **Default Values**:
  - `Text` = `ABCDEFGHIJKLMNOPQRSTUVWXYZ 0123456789`
  - `Step` = 1
- **Filters**:
  - Category: Utility
  - Direction: None
  - Indicators: None
  - Stops: No
  - Complexity: Basic
  - Timeframe: Any
  - Seasonality: No
  - Neural networks: No
  - Divergence: No
  - Risk level: None
