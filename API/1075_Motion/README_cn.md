# Motion 策略
[English](README.md) | [Русский](README_ru.md)

此示例在每根K线轮换文本，没有任何交易逻辑。它是 TradingView `Motion` 脚本的演示性移植。

## 细节

- **入场条件**：无
- **多空方向**：无
- **出场条件**：不适用
- **止损**：无
- **默认值**：
  - `Text` = `ABCDEFGHIJKLMNOPQRSTUVWXYZ 0123456789`
  - `Step` = 1
- **过滤器**：
  - 类别：实用工具
  - 方向：无
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险水平：无
