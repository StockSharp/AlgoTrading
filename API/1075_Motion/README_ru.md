# Стратегия Motion
[English](README.md) | [中文](README_cn.md)

Пример показывает вращение текста на каждой свече без торговой логики. Это прямой перенос скрипта TradingView `Motion` в демонстрационных целях.

## Детали

- **Условия входа**: нет
- **Лонг/шорт**: нет
- **Условия выхода**: не применимо
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Text` = `ABCDEFGHIJKLMNOPQRSTUVWXYZ 0123456789`
  - `Step` = 1
- **Фильтры**:
  - Категория: утилита
  - Направление: отсутствует
  - Индикаторы: нет
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: отсутствует
