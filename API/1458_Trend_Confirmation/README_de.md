# Trendbestätigungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Strategie, die SuperTrend, MACD und VWAP kombiniert, um Trends zu bestätigen.

## Details
- **Einstiegskriterien**: SuperTrend-Richtung mit MACD-Bestätigung und Preis relativ zum VWAP.
- **Long/Short**: Beide Richtungen.
- **Ausstiegskriterien**: MACD kreuzt seine Signallinie gegen die Position.
- **Stops**: Keine.
- **Standardwerte**: ATR-Länge 10, Faktor 3, MACD schnell 12, langsam 26, Signal 9.
- **Filter**: SuperTrend und VWAP.
