# Heiken Ashi Simplified EA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Heikin Ashiローソク足に基づいたパターンベースのシステムです。この戦略は直前のHeikin Ashiの始値と終値のシーケンスを監視します。3本連続の終値がそれぞれの始値を上回り（または下回り）、始値が減速するプルバックを形成する場合、最後のHeikin Ashi始値から最小距離だけ価格が離れると次のローソク足がブレイクアウトトレードを発動できます。アルゴリズムは定義された上限までポジションをスケールアップします。

## 詳細

- **エントリー条件**:
  - **ロング**: 前3本のHA終値がそれぞれ以前の始値を上回り、始値が差を縮小しながら下降する系列を形成する。
  - **ショート**: 前3本のHA終値がそれぞれ以前の始値を下回り、始値が差を拡大しながら上昇する系列を形成する。
- **ロング/ショート**: 両側
- **エグジット条件**:
  - 反対シグナル
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = 1時間
  - `MaxPositions` = 3
  - `DistancePoints` = 300
  - `Volume` = 1
- **フィルター**:
  - カテゴリ: パターン・ブレイクアウト
  - 方向: 両方
  - インジケーター: Heikin Ashi
  - ストップ: なし
  - 複雑さ: 中程度
  - 時間軸: 時間足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
