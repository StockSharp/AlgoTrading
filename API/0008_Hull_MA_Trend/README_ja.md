# Hull MAトレンド
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Hull Moving Averageのトレンドに基づく戦略。

テスト結果では、年間平均リターンが約61%であることが示されています。暗号資産市場で最も効果を発揮します。

Hull MAトレンド戦略はHull Moving Averageの傾きを監視します。上昇する傾きはロングを促し、下降する傾きはショートを促します。各取引はATRトレーリングストップで保護されます。

その応答性の高い計算は従来の移動平均と比べて遅延を軽減し、システムが新しいモメンタムに素早く反応できるようにします。傾きが急に変化した場合、ATRストップが大きなドローダウンを避けるのに役立ちます。


## 詳細

- **エントリー条件**: MA、ATRに基づくシグナル。
- **ロング/ショート**: 両方の方向。
- **エグジット条件**: 逆シグナルまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `HmaPeriod` = 9
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: MA, ATR
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
