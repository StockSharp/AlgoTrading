# Hull均线趋势
[English](README.md) | [Русский](README_ru.md)

本策略关注Hull移动平均线的斜率。斜率上升时做多，下降时做空，并以ATR跟踪止损保护头寸。Hull均线响应更快，可迅速捕捉新动量，ATR止损则在趋势反转时限制回撤。

## 详情
- **入场条件**: 基于 MA、ATR 的信号
- **多空方向**: 双向
- **退出条件**: 反向信号或止损
- **止损**: 是
- **默认值**:
  - `HmaPeriod` = 9
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类型: 趋势
  - 方向: 双向
  - 指标: MA, ATR
  - 止损: 是
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中

测试表明年均收益约为 61%，该策略在加密市场表现最佳。
