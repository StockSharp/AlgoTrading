# Lineare Regressions-Steigung V1
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Verwendet die Steigung einer linearen Regression und ihre verschobene Kopie, um Trendänderungen zu erkennen.

## Details
- **Daten**: Preiskerzen.
- **Einstieg**:
  - Kaufen, wenn die Steigung unter ihren verschobenen Wert kreuzt.
  - Verkaufen, wenn die Steigung über ihren verschobenen Wert kreuzt.
- **Ausstieg**: Das entgegengesetzte Signal schließt die Position.
- **Instrumente**: Alle Instrumente.
- **Risiko**: Kein eingebauter Stop oder Ziel.
