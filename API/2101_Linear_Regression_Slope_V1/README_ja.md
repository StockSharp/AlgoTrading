# 線形回帰傾き V1
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

線形回帰の傾きとそのシフトされたコピーを使用してトレンドの変化を検出します。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー**:
  - 傾きがシフトされた値を下回るクロスで買い。
  - 傾きがシフトされた値を上回るクロスで売り。
- **エグジット**: 反対のシグナルでポジションをクローズ。
- **銘柄**: 任意の銘柄。
- **リスク**: 組み込みのストップや目標なし。
