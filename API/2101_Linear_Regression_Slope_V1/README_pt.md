# Inclinação de Regressão Linear V1
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Usa a inclinação de uma regressão linear e sua cópia deslocada para detectar mudanças de tendência.

## Detalhes
- **Dados**: Velas de preço.
- **Entrada**:
  - Comprar quando a inclinação cruza abaixo do seu valor deslocado.
  - Vender quando a inclinação cruza acima do seu valor deslocado.
- **Saída**: O sinal oposto fecha a posição.
- **Instrumentos**: Qualquer instrumento.
- **Risco**: Sem stop ou alvo integrado.
