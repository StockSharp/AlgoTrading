# Security-freies MTF-Beispielstrategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Zeigt, wie OHLC-Werte des Vortages ohne separate Security-Anfragen berechnet werden. Die Strategie eröffnet eine Long-Position, wenn der Kurs das Tageshoch des Vortages übersteigt, und eine Short-Position, wenn er unter das Tagestief des Vortages fällt.
