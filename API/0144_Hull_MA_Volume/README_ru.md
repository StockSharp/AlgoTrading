# Стратегия Hull MA Volume
[English](README.md) | [中文](README_cn.md)

Стратегия использует сглаженную скользящую среднюю Hull для определения направления тренда и подтверждение объёмом для входа в сделки.

Hull MA устраняет рыночный шум, а растущий объём подтверждает силу движения. Вход осуществляется, когда цена движется по направлению склона Hull при всплеске объёма.

Метод предназначен для трейдеров, отслеживающих участие в пробоях. Стопы на основе ATR защищают от резких разворотов.

## Подробности

- **Условия входа**:
  - Лонг: `HullMA(t) > HullMA(t-1) && Volume > AvgVolume * VolumeMultiplier`
  - Шорт: `HullMA(t) < HullMA(t-1) && Volume > AvgVolume * VolumeMultiplier`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Лонг: `HullMA(t) < HullMA(t-1)`
  - Шорт: `HullMA(t) > HullMA(t-1)`
- **Стопы**: `StopLossAtr` ATR от точки входа
- **Значения по умолчанию**:
  - `HullPeriod` = 9
  - `VolumePeriod` = 20
  - `VolumeMultiplier` = 1.5m
  - `StopLossAtr` = 2.0m
  - `AtrPeriod` = 14
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Hull MA, Скользящая средняя, Объём
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 169%. Стратегию лучше запускать на крипторынке.
