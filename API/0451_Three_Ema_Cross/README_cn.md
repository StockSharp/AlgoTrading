# 三重EMA交叉策略
[English](README.md) | [Русский](README_ru.md)

三重EMA交叉策略将传统的快/慢指数移动平均线交叉与更长周期的趋势过滤器结合。在快速EMA自下而上穿越慢速EMA后，策略等待价格回踩快速均线，只要收盘价仍位于长期趋势EMA之上即可进场。这种设置旨在捕捉主趋势中的短暂回调后的延续走势。

当动能减弱且快速EMA再次跌破慢速EMA时平仓。如果价格逆向移动，基于百分比的止损会保护持仓。该方法在趋势明显的市场表现较好，并能避免在震荡区间频繁交易。

## 细节

- **入场条件**：
  - 最近 *N* 根K线内出现快速EMA上穿慢速EMA。
  - 当前收盘价 ≥ 快速EMA 且当日最低价 ≤ 快速EMA。
  - 趋势EMA ≤ 当前收盘价。
- **方向**：仅做多。
- **出场条件**：
  - 快速EMA跌破慢速EMA。
- **止损**：按入场价的 `stop_loss_percent` 百分比设置。
- **默认参数**：
  - `FastEmaLength` = 10
  - `SlowEmaLength` = 20
  - `TrendEmaLength` = 100
  - `StopLossPercent` = 2.0
  - `CrossBackBars` = 10
- **过滤器**：
  - 类型：趋势跟随
  - 方向：多头
  - 指标：EMA
  - 止损：是
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
