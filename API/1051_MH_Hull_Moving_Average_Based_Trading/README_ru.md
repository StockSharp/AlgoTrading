# MH Hull Moving Average Based Trading
[English](README.md) | [中文](README_cn.md)

Стратегия прорыва на основе Hull Moving Average.

Стратегия сравнивает цену открытия со значениями уровней, рассчитанных по Hull Moving Average. Открывает длинные позиции при пробое верхнего уровня и короткие при пробое нижнего. Текущие позиции закрываются при противоположном пробое.

## Детали

- **Условия входа**: Положение цены относительно уровней Hull MA.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Противоположный пробой.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `HullPeriod` = 210
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: MA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
