# デュアル戦略セレクター V2 - Cryptogyani 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、SMA ベースの 2 つのロングオンリーアプローチを切り替えます。

- **戦略 1**: SMA クロスオーバーを取引し、オプションのトレーリングテイクプロフィットまたは固定ターゲットを使用します。
- **戦略 2**: 上位時間軸のトレンドで確認された SMA クロスオーバーを取引し、ATR ストップと部分的なテイクプロフィットを使用します。

## 詳細

- **エントリー条件**:
  - 戦略 1: 高速 SMA が低速 SMA を上抜ける。
  - 戦略 2: 高速 SMA が低速 SMA を上抜け、かつ価格が上位時間軸の SMA より高い。
- **エグジット条件**:
  - 戦略 1: テイクプロフィット目標またはトレーリングストップ。
  - 戦略 2: 部分的なテイクプロフィット後に ATR ベースのストップ。
- **インジケーター**: SMA, ATR。
- **方向**: ロングのみ。
- **ストップ**: あり。
