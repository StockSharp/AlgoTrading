# 30分足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このアプローチは、現在の 30 分足の始値を前のローソク足の終値と比較します。
新しいローソク足が前の終値より高く始まった場合、ロングポジションを建てます。
すでにロングを保有している状態で次のローソク足が前の終値より低く始まった場合、戦略はショートポジションに転換します。
すべての建玉は現在のローソク足が終わる 1 分前に決済されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 現在のローソク足の始値 > 前のローソク足の終値。
  - **ショート**: ロングを保有している状態で現在のローソク足の始値 < 前のローソク足の終値。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - ローソク足が閉じる 1 分前にすべてのポジションを決済する。
- **ストップ**: なし。
- **デフォルト値**:
  - `CandleType` = TimeSpan.FromMinutes(30).TimeFrame().
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: Price action
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
