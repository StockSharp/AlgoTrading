# Стратегия Pixel Art
[English](README.md) | [中文](README_cn.md)

Стратегия выводит в журнал заранее подготовленные логотипы пиксельного арта.

## Детали

- **Критерии входа**: нет
- **Лонг/Шорт**: нет
- **Критерии выхода**: нет
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Logo` = "pAulse"
- **Фильтры**:
  - Категория: Визуализация
  - Направление: Нет
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Нет
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Нет
