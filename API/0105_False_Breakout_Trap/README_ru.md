# Стратегия False Breakout Trap

False Breakout Trap стремится заработать на пробоях, которые не удерживаются за ключевыми уровнями поддержки или сопротивления.
Трейдеры нередко входят на пробое, но цена быстро разворачивается, оставляя их в ловушке.

Данная стратегия ждёт такого сбоя и входит в противоположном направлении, когда цена закрывается обратно внутри диапазона.

Стоп ставится близко, сразу за уровнем ложного пробоя, чтобы убытки оставались небольшими, если разворот не состоится.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Разворот
  - Направление: Обе
  - Индикаторы: Price Action
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
