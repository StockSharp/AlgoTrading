# フォールス・ブレイクアウト・トラップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
フォールス・ブレイクアウト・トラップは、主要なサポートやレジスタンスを超えたブレイクが維持できない場面を活用することを目的とします。
トレーダーはしばしばブレイクアウトに飛びつきますが、価格はすぐに反転し、彼らを罠にかけます。

テストでは年平均リターンが約52%であることが示されています。暗号資産市場で最も良いパフォーマンスを発揮します。

この戦略はそのような失敗を待ち、価格がレンジ内に戻って引けた時点で逆方向にエントリーします。

ストップはブレイクアウト失敗レベルのすぐ外側に置かれ、リバーサルが実現しない場合に損失を小さく抑えます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: Price Action
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
