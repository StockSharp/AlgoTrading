# DCA デュアル・トレーリング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、速いEMAが遅いEMAを上抜けたときにロングでエントリーします。価格がATRベースまたはパーセントの閾値だけ下落したとき、最大2つの安全オーダーが発注されます。ポジションは標準のトレーリングストップと、利益閾値到達後に有効になる二次ロックイン・トレーリングストップで保護されます。

## パラメーター
- ローソク足タイプ
- 速いEMA期間
- 遅いEMA期間
- 日付フィルターを使用
- 開始日
- ATRスペーシングを使用
- ATR期間
- ATR SO1乗数
- ATR SO2乗数
- フォールバック SO1 パーセント
- フォールバック SO2 パーセント
- クールダウンバー数
- ベースオーダーサイズ USD
- 安全オーダー1サイズ USD
- 安全オーダー2サイズ USD
- トレーリングストップパーセント
- ロックイントリガーパーセント
- ロックイントレイルパーセント
