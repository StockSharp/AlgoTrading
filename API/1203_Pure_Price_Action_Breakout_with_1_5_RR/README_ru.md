# Стратегия Pure Price Action Breakout with 1:5 RR
[English](README.md) | [中文](README_cn.md)

Стратегия Pure Price Action Breakout with 1:5 RR использует пересечение двух EMA с подтверждением RSI и объёма. Стоп‑лосс рассчитывается по ATR, а тейк‑профит в пять раз больше риска.

## Подробности

- **Условия входа**:
  - **Лонг**: быстрая EMA пересекает медленную снизу вверх, RSI > 50, объём выше SMA(20).
  - **Шорт**: быстрая EMA пересекает медленную сверху вниз, RSI < 50, объём выше SMA(20).
- **Направление**: обе стороны.
- **Условия выхода**:
  - Стоп‑лосс по ATR и тейк‑профит с соотношением 1:5.
- **Стопы**: стоп‑лосс = 1.5 × ATR, тейк‑профит = 5 × риск.
- **Параметры по умолчанию**:
  - `FastPeriod` = 9
  - `SlowPeriod` = 21
  - `RsiPeriod` = 14
  - `AtrPeriod` = 14
  - `VolumePeriod` = 20
  - `StopLossFactor` = 1.5
  - `RiskRewardRatio` = 5
  - `MaxTradesPerDay` = 5
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе
  - Индикаторы: EMA, RSI, ATR, SMA объёма
  - Стопы: ATR стоп‑лосс, тейк‑профит 1:5
  - Сложность: Низкая
  - Таймфрейм: 5m или 15m
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
