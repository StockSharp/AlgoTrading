# P-Square N-tes Perzentil-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Schätzt das ausgewählte Perzentil der Quellreihe mithilfe des P-Square-Algorithmus. Öffnet eine Long-Position, wenn der Wert das obere Perzentil überschreitet, und eine Short-Position, wenn der Wert unter das untere Perzentil fällt.

## Parameter
- `Percentile` – zu schätzendes Perzentil.
- `UseReturns` – Renditen statt Preise verarbeiten.
- `CandleType` – Kerzen-Datentyp.
