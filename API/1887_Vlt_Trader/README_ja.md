# Vlt トレーダー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は非常に低いボラティリティの期間を検出し、ブレイクアウト注文を準備します。現在のローソク足のレンジが指定されたルックバック期間にわたって最小になると、戦略は前のローソク足の周辺に買いストップ注文と売りストップ注文を配置します。

## パラメーター
- **Period** – 最小レンジ計算のルックバック期間。
- **Pending level** – ストップ注文を配置するための前のローソク足の高値/安値からのティック距離。
- **Stop loss** – ティック単位の保護ストップ。
- **Take profit** – ティック単位の利益目標。
- **Candle type** – 分析に使用する時間軸。

## ロジック
1. 完成した各ローソク足について、そのレンジ（`High - Low`）を計算します。
2. 直近 *Period* 本のローソク足にわたって最小レンジを追跡します。
3. 現在のレンジが新しい最小値を更新すると、既存の注文をキャンセルし、指定されたオフセットで前のローソク足の上下にストップ注文を配置します。
4. ポジションが開かれると、`StartProtection` がストップロスとテイクプロフィットを管理します。
