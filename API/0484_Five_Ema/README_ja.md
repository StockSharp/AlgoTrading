# 5 EMA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

5 EMA戦略は、5期間EMAの完全に下または上でクローズするローソク足をマークします。価格がブロックウィンドウ外で3バー以内にそのシグナルローソク足の極値をブレイクした場合、ブレイクアウト方向にエントリーします。ターゲットはユーザー定義のリスク・リワード比に基づき、特定の時刻に強制的に取引を閉じることができます。

## 詳細

- **エントリー条件**:
  - ローソク足の終値と高値がEMAより下 → ロング準備；3バー以内にシグナルの高値を上抜けたら買い。
  - ローソク足の終値と安値がEMAより上 → ショート準備；3バー以内にシグナルの安値を下抜けたら売り。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - シグナルローソク足の反対側の極値にストップ。
  - `TargetRR` × リスクのターゲット。
  - カスタム時刻でのオプション出口 (`ExitHour`, `ExitMinute`)。
- **ストップ**: はい。
- **デフォルト値**:
  - `EmaLength` = 5
  - `TargetRR` = 3.0
  - `ExitHour` = 15, `ExitMinute` = 30
  - `BlockStartHour` = 15, `BlockStartMinute` = 0
  - `BlockEndHour` = 15, `BlockEndMinute` = 30
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ロング/ショート
  - インジケーター: EMA
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: 5分
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
