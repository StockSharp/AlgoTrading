# Bj ローソク足パターン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はドラゴンフライ同時線とグレーブストーン同時線のローソク足パターンを探します。長い下ヒゲのドラゴンフライ同時線は強気リバーサルを示す可能性があり、長い上ヒゲのグレーブストーン同時線は弱気リバーサルを示す可能性があります。ドラゴンフライ同時線の後に買い、グレーブストーン同時線の後に売ります。

## 詳細

- **エントリー条件**: ドラゴンフライ同時線 → ロング；グレーブストーン同時線 → ショート。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆シグナルまたは裁量。
- **ストップ**: なし。
- **デフォルト値**:
  - `CandleType` = 15分
  - `DojiThreshold` = 0.1
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: いいえ
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
