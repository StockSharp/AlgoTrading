# Bj Candle Patterns 策略
[English](README.md) | [Русский](README_ru.md)

该策略识别“龙蜻十字星”和“墓碑十字星”形态。长下影的龙蜻十字星常预示多头反转，而长上影的墓碑十字星可能提示空头反转。检测到龙蜻十字星时做多，检测到墓碑十字星时做空。

## 细节

- **入场条件**：龙蜻十字星 → 做多；墓碑十字星 → 做空。
- **多/空**：双向。
- **退出条件**：反向信号或人工。
- **止损**：无。
- **默认值**：
  - `CandleType` = 15 分钟
  - `DojiThreshold` = 0.1
- **过滤条件**：
  - 类别: 形态
  - 方向: 双向
  - 指标: K线形态
  - 止损: 无
  - 复杂度: 初级
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
