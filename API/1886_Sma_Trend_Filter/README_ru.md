# Стратегия SMA Trend Filter
[English](README.md) | [中文](README_cn.md)

Многофреймовая стратегия, анализирующая наклон пяти простых скользящих средних (периоды 5, 8, 13, 21, 34) на трёх таймфреймах (15м, 1ч, 4ч). Рассчитывает бычьи и медвежьи оценки для каждого таймфрейма и торгует, когда все они совпадают по направлению.

## Подробности

- **Условия входа**:
  - Длинная: все три таймфрейма показывают рост не менее 50% SMA
  - Короткая: все три таймфрейма показывают падение не менее 50% SMA
- **Long/Short**: Оба
- **Условия выхода**: противоположный сигнал по порогу закрытия
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `OpenLevel` = 0
  - `CloseLevel` = 0
  - `CandleType1` = TimeSpan.FromMinutes(15).TimeFrame()
  - `CandleType2` = TimeSpan.FromHours(1).TimeFrame()
  - `CandleType3` = TimeSpan.FromHours(4).TimeFrame()
- **Фильтры**:
  - Категория: Trend-following
  - Направление: Оба
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Многофреймовый
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
