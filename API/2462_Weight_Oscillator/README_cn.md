# 权重振荡器策略
[English](README.md) | [Русский](README_ru.md)

该策略将 RSI、资金流量指数、Williams %R 和 DeMarker 组合成一个加权振荡器，并通过简单移动平均线进行平滑。当振荡器穿越设置的高位或低位时开仓或反向。趋势模式决定是顺势交易还是逆势交易。

## 细节

- **入场条件**：
  - **Trend = Direct**：
    - **做多**：振荡器跌破下限。
    - **做空**：振荡器升破上限。
  - **Trend = Against**：
    - **做多**：振荡器升破上限。
    - **做空**：振荡器跌破下限。
- **方向**：双向。
- **出场条件**：反向穿越反转仓位。
- **止损**：无明确止损。
- **过滤器**：加权振荡器与 SMA 平滑。
