# 加重オシレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、RSI、Money Flow Index、Williams %R、DeMarkerを単純移動平均で平滑化した加重オシレーターに組み合わせます。オシレーターが設定可能な高値または安値レベルをクロスしたときにポジションを開くか反転させます。トレンドモードにより、オシレーターシグナルに従うか逆張りするかが決まります。

## 詳細

- **エントリー条件**:
  - **Trend = Direct**:
    - **ロング**: オシレーターが低レベルを下回る。
    - **ショート**: オシレーターが高レベルを上回る。
  - **Trend = Against**:
    - **ロング**: オシレーターが高レベルを上回る。
    - **ショート**: オシレーターが低レベルを下回る。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆方向のクロスによりポジションが反転。
- **ストップ**: 明示的なストップなし。
- **フィルター**: SMA平滑化付き加重オシレーター。
