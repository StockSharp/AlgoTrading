# フラクタルブレイクアウト・トレンドフォロー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

フラクタルブレイクアウト・トレンドフォローは、ボラティリティが低いときに活性化した強気フラクタルの上方に買いストップ注文でエントリーします。

## 詳細

- **エントリー条件**: アリゲーターの歯ラインより上の上昇フラクタルかつATRパーセンタイル平均が閾値を下回る場合、フラクタルレベルに買いストップを設置。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: パーセンテージストップまたは下降フラクタル活性化のうち高い方でストップロス。
- **ストップ**: あり。
- **デフォルト値**:
  - `StopLossPercent` = 0.03
  - `AtrThreshold` = 50
  - `AtrPeriod` = 5
  - `CandleType` = TimeSpan.FromHours(1)
  - `TradeStart` = 2023-01-01
  - `TradeStop` = 2025-01-01
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロングのみ
  - インジケーター: Fractal, SMMA, ATR
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
