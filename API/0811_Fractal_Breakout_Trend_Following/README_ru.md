# Фрактальный прорыв и следование тренду
[English](README.md) | [中文](README_cn.md)

Стратегия фрактального прорыва входит по стоп-заявке выше активированного бычьего фрактала при низкой волатильности.

## Детали

- **Условия входа**: Восходящий фрактал выше линии зубов Аллигатора и средний процентиль ATR ниже порога; выставляется стоп-покупка на уровне фрактала.
- **Направление**: Только лонг.
- **Условия выхода**: Стоп-лосс по максимуму из процентного стопа и активированного нисходящего фрактала.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `StopLossPercent` = 0.03
  - `AtrThreshold` = 50
  - `AtrPeriod` = 5
  - `CandleType` = TimeSpan.FromHours(1)
  - `TradeStart` = 2023-01-01
  - `TradeStop` = 2025-01-01
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: Фрактал, SMMA, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
