# 分形突破趋势跟随
[Русский](README_ru.md) | [English](README.md)

分形突破趋势跟随在波动性较低时，当激活的看涨分形上方触发买入止损。

## 细节

- **入场条件**：上方分形位于鳄鱼指标牙线之上且 ATR 百分位平均值低于阈值；在分形位上挂买入止损。
- **多/空方向**：仅做多。
- **出场条件**：止损价取百分比止损与向下分形激活价的较高值。
- **止损**：有。
- **默认值**：
  - `StopLossPercent` = 0.03
  - `AtrThreshold` = 50
  - `AtrPeriod` = 5
  - `CandleType` = TimeSpan.FromHours(1)
  - `TradeStart` = 2023-01-01
  - `TradeStop` = 2025-01-01
- **筛选**：
  - 分类：趋势跟随
  - 方向：多头
  - 指标：分形、SMMA、ATR
  - 止损：有
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
