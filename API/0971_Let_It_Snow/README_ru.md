# Пусть идет снег
[English](README.md) | [中文](README_cn.md)

Декоративная стратегия, которая рисует на графике падающие снежинки и не совершает сделок.

## Детали

- **Критерии входа**: отсутствуют, только визуализация.
- **Длинные/короткие**: нет.
- **Критерии выхода**: не применимо.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Визуализация
  - Направление: нет
  - Индикаторы: нет
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: отсутствует
