# Let It Snow
[Русский](README_ru.md) | [中文](README_cn.md)

A decorative strategy that draws falling snowflakes on the chart without executing trades. It subscribes to a candle series and plots a snowflake symbol at a moving position based on the current time.

## Details

- **Entry Criteria**: None, visualization only.
- **Long/Short**: None.
- **Exit Criteria**: Not applicable.
- **Stops**: No.
- **Default Values**:
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Filters**:
  - Category: Visualization
  - Direction: None
  - Indicators: None
  - Stops: No
  - Complexity: Basic
  - Timeframe: Intraday
  - Seasonality: No
  - Neural Networks: No
  - Divergence: No
  - Risk Level: None
