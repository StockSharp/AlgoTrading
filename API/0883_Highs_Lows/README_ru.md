# Стратегия Highs Lows
[English](README.md) | [中文](README_cn.md)

Стратегия торгует на основе средних точек свечей относительно диапазона максимумов и минимумов.

Покупает, когда средняя текущей свечи ниже среднего между максимумом и минимумом, и нормализованное расстояние меньше LowThreshold. Закрывает длинную позицию, когда средняя выше этого значения и нормализованное расстояние превышает HighThreshold.

## Детали

- **Критерии входа**: Средняя свечи ниже диапазона и нормализованное расстояние ниже LowThreshold.
- **Длинные/короткие**: только длинные.
- **Критерии выхода**: Средняя выше диапазона и нормализованное расстояние выше HighThreshold.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Range` = 100
  - `LowThreshold` = 15m
  - `HighThreshold` = 85m
  - `CandleType` = TimeSpan.FromMinutes(240)
- **Фильтры**:
  - Категория: Диапазон
  - Направление: Длинные
  - Индикаторы: Highest, Lowest
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (240m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
