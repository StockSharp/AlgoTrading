# Стратегия SMA RSI Volume ATR
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает простое скользящее среднее (SMA), индекс относительной силы (RSI), подтверждение объёмом и фильтр волатильности на базе ATR.
Покупает, когда цена выше SMA, RSI в зоне перепроданности, объём превышает своё среднее в заданное число раз и волатильность растёт. Продаёт при обратных условиях.

Стопы управляются фиксированными процентами тейк‑профита и стоп‑лосса.

## Детали

- **Условия входа**:
  - **Лонг**: `Close > SMA` && `RSI < RsiOversold` && `Volume > AvgVolume * VolumeThreshold` && `ATR > ATR_{prev}`
  - **Шорт**: `Close < SMA` && `RSI > RsiOverbought` && `Volume > AvgVolume * VolumeThreshold` && `ATR > ATR_{prev}`
- **Лонг/шорт**: обе стороны
- **Условия выхода**: стоп‑лосс или тейк‑профит
- **Стопы**: да, в процентах
- **Значения по умолчанию**:
  - `SmaLength` = 50
  - `RsiLength` = 14
  - `RsiOverbought` = 70
  - `RsiOversold` = 30
  - `VolumeThreshold` = 1.5
  - `AtrLength` = 14
  - `TakeProfitPerc` = 1.5
  - `StopLossPerc` = 0.5
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе стороны
  - Индикаторы: SMA, RSI, Объём, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
