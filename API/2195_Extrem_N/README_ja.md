# Extrem N戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Extrem N戦略は、ローリングウィンドウで検出された新高値と新安値に基づいてリバーサルを取引します。

この戦略はドンチャンチャネルインジケーターを使用して価格の極値をマークします。バーがルックバック期間に対して新高値を付け、次のバーが新安値を付けると、ロングポジションが開かれます。新安値に続いて新高値が付いた場合にはショートポジションが開かれます。反対のシグナルは既存のポジションをクローズします。

- **エントリー条件**:
  - ロング: 前のバーが新高値を付け、現在のバーが新安値を付けた。
  - ショート: 前のバーが新安値を付け、現在のバーが新高値を付けた。
- **エグジット条件**:
  - ロングポジションはショートエントリーシグナルでクローズ。
  - ショートポジションはロングエントリーシグナルでクローズ。
- **パラメーター**:
  - `Period` – Donchianのルックバック期間（デフォルト: 9）。
  - `CandleType` – 処理の時間軸（デフォルト: 4時間）。
  - `BuyPosOpen` – ロングポジションの開設を許可する（デフォルト: true）。
  - `SellPosOpen` – ショートポジションの開設を許可する（デフォルト: true）。
  - `BuyPosClose` – ロングポジションのクローズを許可する（デフォルト: true）。
  - `SellPosClose` – ショートポジションのクローズを許可する（デフォルト: true）。
- **インジケーター**: Donchian Channel。
