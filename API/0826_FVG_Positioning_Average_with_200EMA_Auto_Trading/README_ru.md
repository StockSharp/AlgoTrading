# Стратегия FVG Positioning Average with 200EMA Auto Trading
[English](README.md) | [中文](README_cn.md)

Стратегия усредняет уровни бычьих и медвежьих разрывов справедливой стоимости (FVG) и сочетает их с EMA периодом 200. Сделка открывается, когда цена пересекает эти средние в направлении тренда.

## Детали

- **Условия входа**:
  - **Лонг**: цена пересекает сверху среднее медвежьих FVG, и все средние выше EMA.
  - **Шорт**: цена пересекает снизу среднее бычьих FVG, и все средние ниже EMA.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - Стоп-лосс на недавнем минимуме/максимуме.
  - Тейк-профит по соотношению риск/прибыль.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `FvgLookback` = 30
  - `AtrMultiplier` = 0.25
  - `LookbackPeriod` = 20
  - `EmaPeriod` = 200
  - `RiskReward` = 1.5
- **Фильтры**:
  - Категория: Прайс-экшен
  - Направление: Оба
  - Индикаторы: ATR, EMA, SMA, Highest, Lowest
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
