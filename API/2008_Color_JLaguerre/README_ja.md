# Color JLaguerre
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

色分けされたLaguerreオシレーターに基づく戦略です。

このインジケーターはJurikフィルターで価格動向を平滑化し、事前定義されたレベル内の位置に応じてラインを着色します。色の変化はトレンドの転換の可能性を示します。

戦略はオシレーターが中間レベルを上向きに越えたときにロングに入り、下向きに越えたときにショートに入ります。オシレーターが極端なレベルに達するか反対のシグナルが現れるとポジションがクローズされます。

## 詳細

- **エントリー条件**: 中間レベル付近でのLaguerreオシレーターの色の変化。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対シグナルまたは極端なレベルへの到達。
- **ストップ**: はい。
- **デフォルト値**:
  - `RsiLength` = 14
  - `HighLevel` = 85
  - `MiddleLevel` = 50
  - `LowLevel` = 15
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromHours(1)
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 時間足 (1h)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
