# Color JLaguerre
[English](README.md) | [中文](README_cn.md)

Стратегия на основе цветного осциллятора Laguerre.

Индикатор сглаживает движение цены фильтром Jurik и раскрашивает свою линию в зависимости от положения относительно уровней. Смена цвета указывает на возможный разворот тренда.

Стратегия открывает длинную позицию при пересечении средней линии снизу вверх и короткую при пересечении сверху вниз. Выход осуществляется при достижении экстремальных уровней или появлении противоположного сигнала.

## Детали

- **Условия входа**: смена цвета осциллятора Laguerre около среднего уровня.
- **Длинные/короткие**: в обе стороны.
- **Условия выхода**: противоположный сигнал или достижение экстремума.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `HighLevel` = 85
  - `MiddleLevel` = 50
  - `LowLevel` = 15
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromHours(1)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Часовой (1ч)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
