# 修正最適楕円フィルター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はJohn F. Ehlersが説明した*Modified Optimum Elliptic Filter*インジケーターを適用して方向転換を検出します。このインジケーターは次の再帰的な公式を使って高値と安値の平均を平滑化する2極デジタルフィルターです：

```
F(t) = 0.13785*(2*HL2(t) - HL2(t-1))
     + 0.0007 *(2*HL2(t-1) - HL2(t-2))
     + 0.13785*(2*HL2(t-2) - HL2(t-3))
     + 1.2103 * F(t-1) - 0.4867 * F(t-2)
```

ここで`HL2`は各ローソク足の中点`(High + Low)/2`です。

戦略はモメンタムを判断するために最新の3つのフィルター値を読み取ります。インジケーターが上昇しており最新値が前の値を超えている場合、ロングポジションをオープンします。インジケーターが下降しており現在値が前の値を下回っている場合、ショートポジションをオープンします。逆の条件が発生するとポジションが反転します。

## 詳細

- **エントリー条件**:
  - **ロング**: `F(t-1) < F(t-2)` かつ `F(t) > F(t-1)`.
  - **ショート**: `F(t-1) > F(t-2)` かつ `F(t) < F(t-1)`.
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆シグナルでポジションを反転。
- **ストップ**: 明示的なストップなし。
- **デフォルト値**:
  - `Candle Type` = 4時間の時間軸。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: なし
  - 複雑さ: 中程度
  - 時間軸: 中期
