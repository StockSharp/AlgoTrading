# トレンドライン・バウンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

市場は過去のスイング高値または安値を通じて描かれたトレンドラインを尊重することが多いです。この戦略は自動的に直近の価格動向に回帰線をフィットさせ、支配的なトレンドの方向にそれらのラインからバウンスするローソク足を探します。

テストでは年平均リターンが約124%であることが示されています。外国為替市場で最も良いパフォーマンスを発揮します。

直近のローソク足は上昇または下降傾斜のサポートラインとレジスタンスラインを計算するために保存されます。価格がトレンドラインに近づき、ローソク足が移動平均の正しい側にとどまりながらバウンスを確認する場合、システムはトレードに入ります。ストップは価格のパーセントを使って設定され、移動平均のクロスで決済されます。

優勢な方向にのみ取引し、サポートまたはレジスタンスでの明確な反応を待つことで、この手法はブレイクアウトを追わずに継続的な動きを捉えることを試みます。

## 詳細

- **エントリー条件**: 価格が計算されたトレンドラインに触れ、ローソク足がMAの上/下でトレンド方向に終値を付ける。
- **ロング/ショート**: 両方。
- **エグジット条件**: 価格が移動平均を越えるか、ストップロス。
- **ストップ**: はい、パーセントベース。
- **デフォルト値**:
  - `TrendlinePeriod` = 20
  - `MAPeriod` = 20
  - `BounceThresholdPercent` = 0.5
  - `CandleType` = 5 minute
  - `StopLossPercent` = 2
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: MA, Trendlines
  - ストップ: はい
  - 複雑さ: 上級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
