# Стратегия Ichimoku ADX

Стратегия основана на облаке Ишимоку и индикаторе ADX. Длинная позиция открывается, когда цена выше облака, Tenkan-сен выше Kijun-сен и ADX больше 25, что указывает на сильный восходящий тренд. Короткая позиция открывается, когда цена находится под облаком, Tenkan-сен ниже Kijun-сен и ADX также выше 25, что говорит о сильном нисходящем движении. Выход из позиции происходит, если цена пересекает облако в обратном направлении.

Метод объединяет сигналы Ишимоку с ADX для отбора мощных трендов. Сделки совершаются при пробое облака при подтверждении силы через ADX.

Подходит трейдерам, предпочитающим структурированные трендовые сетапы. Стопы по ATR защищают от резких колебаний.

## Подробности
- **Условия входа**:
  - Long: `Price > Cloud && Tenkan > Kijun && ADX > AdxThreshold`
  - Short: `Price < Cloud && Tenkan < Kijun && ADX > AdxThreshold`
- **Long/Short**: оба
- **Условия выхода**:
  - Цена пересекает облако в противоположную сторону
- **Стопы**: плавающий стоп по облаку Ишимоку
- **Значения по умолчанию**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanBPeriod` = 52
  - `AdxPeriod` = 14
  - `AdxThreshold` = 25m
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Trend
  - Направление: оба
  - Индикаторы: Ichimoku Cloud, ADX
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
