# Ichimoku Adx Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторах облака Ишимоку и ADX. Критерии входа: Long: Price > Kumo (облако) && Tenkan > Kijun && ADX > 25 (восходящий тренд с сильным движением) Short: Price < Kumo (облако) && Tenkan < Kijun && ADX > 25 (нисходящий тренд с сильным движением) Критерии выхода: Long: Price < Kumo (цена падает ниже облака) Short: Price > Kumo (цена поднимается выше облака).

Эта стратегия сочетает сигналы облака Ишимоку с ADX, чтобы фильтровать мощные тренды. Сделки совершаются при пробое облака, подтверждённом ADX.

Она подойдёт трейдерам, предпочитающим структурированные трендовые установки. Стопы по ATR защищают от неблагоприятных движений.

## Детали

- **Критерии входа**:
  - Long: `Price > Cloud && Tenkan > Kijun && ADX > AdxThreshold`
  - Short: `Price < Cloud && Tenkan < Kijun && ADX > AdxThreshold`
- **Long/Short**: Оба направления
- **Критерии выхода**:
  - Цена пересекает облако в обратную сторону
- **Стопы**: используется облако Ишимоку как трейлинг-стоп
- **Значения по умолчанию**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanBPeriod` = 52
  - `AdxPeriod` = 14
  - `AdxThreshold` = 25m
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Trend
  - Направление: Оба
  - Индикаторы: Ichimoku Cloud, ADX
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 187%\. Стратегию лучше запускать на фондовом рынке.
