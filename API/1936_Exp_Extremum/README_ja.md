# Exp Extremum戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ルックバックウィンドウ内の価格の極値を比較することで検出されるリバーサルを取引します。現在のローソク足が価格を以前の高値や安値を超えて動かすかどうかを観察し、この比較の符号が変化したときに反応します。

## 仕組み

1. 完成した各ローソク足について、戦略は以下を求めます：
   - 直近 *N* バーにわたる最安値の高値。
   - 直近 *N* バーにわたる最高値の安値。
2. 現在の高値/安値とこれらのレベルの差が合計されます。
3. 正の合計は上昇圧力を示し、負の合計は下降圧力を示します。
4. 2バー前の符号が1バー前の符号と逆の場合、リバーサルシグナルが現れます：
   - 上昇から下降 → ロングポジションを開く。
   - 下降から上昇 → ショートポジションを開く。
5. オプションのパーミッションにより、ロング/ショートポジションの開閉を独立して無効にできます。

## パラメーター

- `Length` – 極値計算のインジケーター期間。
- `CandleType` – 受信するローソク足の時間軸。
- `BuyPosOpen` / `SellPosOpen` – ロングまたはショートポジションを開く許可。
- `BuyPosClose` / `SellPosClose` – ロングまたはショートポジションを閉じる許可。

## 注意事項

この戦略は、ローソク足サブスクリプションと組み込みの `Highest`/`Lowest` インジケーターを持つ高レベルAPIを使用します。ポジションは成行注文で開かれ、反対のシグナルが現れると `ClosePosition()` で決済されます。
