# Модель импульса с гамма-взвешенной средней ценой для фьючерса BTC
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия вычисляет гамма-взвешенную среднюю цену (GWAP) для поиска импульса на фьючерсе BTC. Длинные позиции открываются, когда цена выше GWAP и три последних закрытия последовательно растут. Короткие позиции открываются, когда цена ниже GWAP и три последних закрытия последовательно падают.

## Детали

- **Критерии входа**:
  - **Long**: закрытие выше GWAP и последние три закрытия растут.
  - **Short**: закрытие ниже GWAP и последние три закрытия снижаются.
- **Long/Short**: обе стороны.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `Length` = 60
  - `GammaFactor` = 0.75
- **Фильтры**:
  - Категория: Импульс
  - Направление: Обе
  - Индикаторы: GWAP
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: 1 мин
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
