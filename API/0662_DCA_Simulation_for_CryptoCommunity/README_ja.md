# DCA Simulation CryptoCommunity戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、オプションのセーフティオーダーとトレーリングテイクプロフィットを備えたドルコスト平均法をシミュレートします。ベースオーダーから始まり、定期的に追加資本を投資したり、価格下落後に平均コストを下げることができます。

## 詳細

- **エントリー条件**:
  - ポジションが未保有で、日付が設定された範囲内にある場合、ベース数量を購入。
  - Nバーごとのオプションの定期DCAオーダー。
  - 直近高値から指定パーセント価格が下落した場合のオプションのセーフティオーダー。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - 目標パーセントでのテイクプロフィット、オプションでトレーリングストップあり。
- **ストップ**: テイクプロフィット / トレーリングストップ。
- **デフォルト値**:
  - ベースオーダー = 100 USD。
  - DCA金額 = 30バーごとに10 USD。
  - セーフティオーダー金額 = 価格偏差15%で100 USD。
  - テイクプロフィット = 1000%、トレーリング = 25%。
  - 開始日 = 2021-11-01、終了日 = 9999-01-01。
- **フィルター**:
  - カテゴリ: 積み立て。
  - 方向: ロング。
  - インジケーター: なし。
  - ストップ: はい。
  - 複雑さ: 中程度。
  - 時間軸: 任意。
  - 季節性: いいえ。
  - ニューラルネットワーク: いいえ。
  - ダイバージェンス: いいえ。
  - リスクレベル: 中。
