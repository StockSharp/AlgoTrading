# フィボナッチ・リトレースメント戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ZigZagピボットから導出されたフィボナッチ・リトレースメント・ブレイクアウトを取引します。

## アイデア

1. ZigZagアプローチを使用してスウィングの高値と安値を検出する。
2. 最後の2つのピボット間にフィボナッチ・リトレースメントレベル（23.6%、38.2%、61.8%、76.4%）を構築する。
3. 上昇トレンドでは、価格がいずれかのフィボナッチレベルを上回って終値をつけると買い。
4. 下降トレンドでは、価格がいずれかのフィボナッチレベルを下回って終値をつけると売り。
5. 各注文は固定ストップロスとスウィング範囲に基づくテイクプロフィットで保護される。
6. ポジションのクローズ後、再取引前に指定された本数のバーを待機する。

## パラメーター

- `ZigzagDepth` – 新しいピボットを検索するための深度。
- `SafetyBuffer` – 価格がレベルを超えて動く必要のある点数での距離。
- `TrendPrecision` – トレンド方向を検出するためのピボット間の最小差。
- `CloseBarPause` – 取引クローズ後に待機するバーの本数。
- `TakeProfitFactor` – テイクプロフィット拡張として使用するスウィング範囲の割合。
- `StopLossPoints` – エントリー価格からのストップロス距離（点数）。
- `CandleType` – 計算に使用するローソク足の種類。
