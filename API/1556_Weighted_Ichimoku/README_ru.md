# Стратегия Weighted Ichimoku
[English](README.md) | [中文](README_cn.md)

Стратегия объединяет сигналы индикатора Ишимоку в взвешенный счёт.
Покупает, когда счёт превышает порог входа, и выходит, когда счёт опускается ниже порога выхода.

## Детали

- **Критерий входа**: счёт >= BuyThreshold
- **Лонг/Шорт**: только лонг
- **Критерий выхода**: счёт <= SellThreshold или ниже нуля, если порог отключён
- **Стопы**: нет
- **Значения по умолчанию**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanBPeriod` = 52
  - `Offset` = 26
  - `BuyThreshold` = 60
  - `SellThreshold` = -49
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: Ichimoku
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
