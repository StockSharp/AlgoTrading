# CyberiaTrader戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、元の**CyberiaTrader.mq5**システムをStockSharp向けに簡略化してポーティングしたものです。複数の古典的なテクニカル指標を組み合わせて市場の方向性を評価し、ほとんどのフィルターが一致したときに取引をオープンします。

## インジケーター

- **MACD** – 高速/低速EMAとシグナルラインを使用してモメンタムの変化を検出します。
- **単純移動平均** – 主要なトレンドを判断します。
- **Commodity Channel Index** – 買われすぎ/売られすぎの状態をフィルタリングします。
- **Average Directional Index** – +DIと-DIコンポーネントを通じて方向性の強さを確認します。

## パラメーター

| 名前 | 説明 |
| --- | --- |
| `MacdFast` | MACDの高速EMA期間。 |
| `MacdSlow` | MACDの低速EMA期間。 |
| `MacdSignal` | MACDのシグナルライン期間。 |
| `MaPeriod` | 移動平均トレンドフィルターの期間。 |
| `CciPeriod` | Commodity Channel Indexの期間。 |
| `AdxPeriod` | Average Directional Indexの期間。 |
| `EnableMacd` | MACDフィルターの有効/無効。 |
| `EnableMa` | 移動平均フィルターの有効/無効。 |
| `EnableCci` | CCIフィルターの有効/無効。 |
| `EnableAdx` | ADXフィルターの有効/無効。 |
| `CandleType` | 入力足の時間軸。 |

## 取引ロジック

1. 有効化されたすべてのインジケーターの値が各確定足で計算されます。
2. フィルターはそれぞれのルールに基づいて買いまたは売りをブロックできます：
   - MACDがシグナルラインより上にある場合はショートエントリーをブロック、下にある場合はロングエントリーをブロック。
   - 価格が移動平均より上にある場合はショートをブロック、下にある場合はロングをブロック。
   - CCIが+100より上の場合はロングをブロック、-100より下の場合はショートをブロック。
   - +DIが-DIより大きい場合はショートをブロック、-DIが+DIより大きい場合はロングをブロック。
3. 一方が許可されており、もう一方がブロックされている場合にのみ取引がオープンされます。
4. 基本的なポジション保護は、テイクプロフィット2%とストップロス1%を使用します。

## 注意事項

この変換は元のアルゴリズムの主要な方向性フィルターに焦点を当てています。MQL5バージョンの広範な確率分析と補助モジュールは、わかりやすさのために意図的に省略されています。
