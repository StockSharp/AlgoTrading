# 宵の明星パターン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
宵の明星は明けの明星の鏡像ですが、潜在的な天井を示します。強い陽線で始まり、小さな迷いロウソクが続き、最初のバーの中間点を下回って引ける陰線で終わります。

テストでは年平均リターン約100%を示しています。外国為替市場で最もよく機能します。

アルゴリズムは3本のロウソクのシーケンスを監視します。パターンが形成されると、小さな中央ロウソクの高値の上にストップを置いてショートで入ります。価格が確認ロウソクの安値を下回るか、ストップが発動するとポジションがエグジットします。

このセットアップは買われすぎ状態からの急速なリバーサルを予測するため、トレードは通常、短い下方向のモメンタム駆動の動きを目標とします。

## 詳細

- **エントリー条件**: 三本足の宵の明星パターン。
- **ロング/ショート**: ショートのみ。
- **エグジット条件**: 価格が確認バーの安値を下回るかストップロス。
- **ストップ**: はい、中央ロウソクの高値の上。
- **デフォルト値**:
  - `CandleType` = 5 minute
  - `StopLossPercent` = 1
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ショート
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
