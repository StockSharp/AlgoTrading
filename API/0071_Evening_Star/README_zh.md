# 暮星形态策略

暮星形态与晨星相反，通常表示顶部形成。它由一根大阳线开始，随后是一根小型犹豫蜡烛，最后以一根收盘价跌破第一根中点的阴线结束。

算法监控连续三根K线。当出现该形态时，建立空仓，止损设在中间蜡烛高点上方。若价格跌破确认蜡烛低点或触发止损则平仓。

因该形态预示超买后的快速反转，交易通常针对短期向下的动量。

## 细节

- **入场条件**：完整的三根暮星形态。
- **多/空**：仅做空。
- **退出条件**：价格跌破确认蜡烛低点或止损。
- **止损**：有，位于中间蜡烛高点上方。
- **默认值**：
  - `CandleType` = 5 分钟
  - `StopLossPercent` = 1
- **过滤条件**：
  - 类别: 形态
  - 方向: 空头
  - 指标: K线形态
  - 止损: 有
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
