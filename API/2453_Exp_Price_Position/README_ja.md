# Exp 価格ポジション
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Exp Price Position**戦略は、価格の位置とステップトレンドフィルターを組み合わせたオリジナルのMetaTraderエキスパートアドバイザーを適応させたものです。
2つの中央移動平均の関係を評価して直近のスイングレベルを特定し、次に高速・低速の平滑化移動平均ペアを確認してトレンドの方向を決定します。
価格の位置とステップトレンドの両方が現在のローソク足の構造と一致した場合にのみ注文が出されます。

この戦略は、価格が動的な中央レベルに引き戻された後にトレンド転換が起きる市場向けに設計されています。リスク管理にはトレーリングストップとテイクプロフィット比率が適用されます。

## 詳細

- **エントリー条件**: 直近スイングレベルより上で強気のステップトレンドの場合ロング取引、下で弱気のステップトレンドの場合ショート取引。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 逆シグナルまたは保護ストップ。
- **ストップ**: あり、テイクプロフィット比率を持つトレーリングストップ。
- **デフォルト値**:
  - `FastPeriod` = 2
  - `SlowPeriod` = 30
  - `MedianFastPeriod` = 26
  - `MedianSlowPeriod` = 20
  - `TpSlRatio` = 3m
  - `TrailingStopPips` = 10m
  - `CandleType` = TimeSpan.FromHours(1)
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Smoothed Moving Average, Simple Moving Average
  - ストップ: トレーリング
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
