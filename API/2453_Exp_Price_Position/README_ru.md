# Exp Price Position
[English](README.md) | [中文](README_cn.md)

Стратегия **Exp Price Position** адаптирует оригинальный эксперт MetaTrader, который сочетает положение цены и фильтр направления.
Она оценивает соотношение двух медианных скользящих средних, чтобы определить последний уровень разворота, и затем проверяет пару сглаженных скользящих средних для определения тренда.
Сделки открываются только при согласии положения цены и тренда со структурой текущей свечи.

Стратегия предназначена для рынков, где разворот тренда происходит после отката цены к динамическому медианному уровню. Для управления риском используется трейлинг-стоп и коэффициент тейк-профита.

## Детали

- **Условия входа**: Цена выше последнего уровня разворота при бычьем тренде; ниже при медвежьем.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Обратный сигнал или защитный стоп.
- **Стопы**: Да, трейлинг-стоп с коэффициентом тейк-профита.
- **Значения по умолчанию**:
  - `FastPeriod` = 2
  - `SlowPeriod` = 30
  - `MedianFastPeriod` = 26
  - `MedianSlowPeriod` = 20
  - `TpSlRatio` = 3m
  - `TrailingStopPips` = 10m
  - `CandleType` = TimeSpan.FromHours(1)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Smoothed Moving Average, Simple Moving Average
  - Стопы: Трейлинг
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
