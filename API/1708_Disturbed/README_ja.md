# Disturbed戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このヘッジ戦略は、ロングとショートの成行注文を同時に発注し、現在のスプレッドに基づいて管理します。どちらかの側に対してスプレッド1つ分価格が動くと、その仏置が決済されます。残りのポジションは、スプレッドの設定可能な倍数に等しい利益または損失を目標とします。

## 詳細

- **エントリー条件**:
  - 開始時に、買いと売りの成行注文を同時に発注します。
- **ロング/ショート**: 両方同時。
- **エグジット条件**:
  - スプレッド1つ分の損失を出した側を決済する。
  - 残りの側を `gainMultiplier * spread` の利益または損失で決済する。
- **ストップ**: スプレッドベースのレベルによる暗黙的なもの。
- **フィルター**: なし。
