# RSI ボックス（疑似グリッドボット）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSI の買われすぎ・売られすぎシグナルから価格レベルを導くグリッドベースの戦略です。RSI が極値を越えると、直近の高値と安値から動的なグリッドラインが再計算されます。価格が次のグリッドレベルを上下に突破したときに取引が発生します。ショートはオプションで対応しています。

## 詳細

- **エントリー条件**: RSI の極値の後、価格が次のグリッドラインを越える。
- **ロング/ショート**: デフォルトはロング、ショートはオプション。
- **エグジット条件**: 価格が反対方向のグリッドラインを越える。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `RsiPeriod` = 14
  - `Overbought` = 70
  - `Oversold` = 30
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `UseShorts` = false
- **フィルター**:
  - カテゴリ: グリッド
  - 方向: 両方
  - インジケーター: RSI, Highest, Lowest
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
