# RSI Box（伪网格机器人）
[English](README.md) | [Русский](README_ru.md)

该策略根据RSI的超买和超卖信号构建价格网格。当RSI越过极值时，网格线会根据最近的高点和低点重新计算。当价格突破下一条网格线时进行交易，可选择做空。

## 详情

- **入场条件**：RSI出现极值后，价格突破下一条网格线。
- **多空方向**：默认多头，可选空头。
- **出场条件**：价格突破相反方向的网格线。
- **止损**：无。
- **默认参数**：
  - `RsiPeriod` = 14
  - `Overbought` = 70
  - `Oversold` = 30
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `UseShorts` = false
- **过滤条件**：
  - 类别：网格
  - 方向：双向
  - 指标：RSI、Highest、Lowest
  - 止损：无
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
