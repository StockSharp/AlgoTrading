# RSI Box (Псевдо Grid Bot)
[English](README.md) | [中文](README_cn.md)

Сетка цен строится на основе уровней перекупленности и перепроданности RSI. После пересечения экстремума RSI линии сетки пересчитываются по недавним максимумам и минимумам. Сделки совершаются при пробое следующей линии. Шорты по желанию.

## Детали

- **Вход**: Цена пересекает следующую линию после экстремума RSI.
- **Лонг/Шорт**: Лонг по умолчанию, шорт опционально.
- **Выход**: Цена пересекает противоположную линию.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `RsiPeriod` = 14
  - `Overbought` = 70
  - `Oversold` = 30
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `UseShorts` = false
- **Фильтры**:
  - Категория: Сетка
  - Направление: Оба
  - Индикаторы: RSI, Highest, Lowest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
