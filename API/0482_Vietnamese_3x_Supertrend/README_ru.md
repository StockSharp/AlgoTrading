# Стратегия Vietnamese 3x Supertrend
[English](README.md) | [中文](README_cn.md)

Стратегия использует три индикатора SuperTrend с разными параметрами ATR. Входы в длинную позицию выполняются на откатах, когда медленный тренд остается нисходящим. При благоприятном движении можно включить стоп в точке безубытка.

## Детали

- **Условия входа**:
  - Медленный SuperTrend указывает на нисходящий тренд.
  - **Long 1**: Средний тренд вверх, быстрый тренд вниз.
  - **Long 2**: Средний тренд вниз и цена выше линии быстрого SuperTrend.
  - **Long 3**: Быстрый тренд вниз и пробой выше максимума во время этого тренда.
- **Длинные/Короткие**: Только long.
- **Условия выхода**:
  - Все SuperTrend переходят вверх и свеча закрывается медвежьей.
  - Средняя цена входа выше текущего закрытия.
  - При включении стоп безубытка.
- **Стопы**: Стоп безубытка по желанию.
- **Значения по умолчанию**:
  - `FastAtrLength` = 10
  - `FastMultiplier` = 1
  - `MediumAtrLength` = 11
  - `MediumMultiplier` = 2
  - `SlowAtrLength` = 12
  - `SlowMultiplier` = 3
  - `UseHighestOfTwoRedCandles` = false
  - `UseEntryStopLoss` = true
  - `UseAllDowntrendExit` = true
  - `UseAvgPriceInLoss` = true
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Long
  - Индикаторы: SuperTrend
  - Стопы: Опционально
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
