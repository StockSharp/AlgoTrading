# PS ジャニュアリー・バロメーター・バックテスター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

2月から6月の終値が1月の高値を超えた場合にロングポジションを取るジャニュアリー・バロメーターを実装します。オプションのフィルターとして、サンタクロース・ラリーと最初の5日間のリターンが正であることを要件とします。

## 詳細

- **エントリー条件**: 2月から6月の終値が1月の高値を上回り、オプションの季節性フィルターを満たす
- **ロング/ショート**: ロングのみ
- **エグジット条件**: 12月にポジションをクローズ
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = 1 month
  - `UseSantaClausRally` = false
  - `UseFirstFiveDays` = false
- **フィルター**:
  - カテゴリ: 季節性
  - 方向: ロングのみ
  - インジケーター: 季節性
  - ストップ: なし
  - 複雑さ: 初心者
  - 時間軸: 月足
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
