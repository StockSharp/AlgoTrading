# Стратегия PS January Barometer Backtester
[English](README.md) | [中文](README_cn.md)

Реализует индикатор January Barometer: длинная позиция открывается, когда закрытие в феврале–июне превышает максимум января. Дополнительные фильтры требуют положительного результата Santa Claus Rally и/или первых пяти дней года.

## Детали

- **Критерий входа**: закрытие февраля–июня выше январского максимума с учётом сезонных фильтров
- **Длин/Шорт**: Длинная
- **Критерий выхода**: закрытие позиции в декабре
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `CandleType` = 1 месяц
  - `UseSantaClausRally` = false
  - `UseFirstFiveDays` = false
- **Фильтры**:
  - Категория: Seasonality
  - Направление: Long
  - Индикаторы: Seasonality
  - Стопы: Нет
  - Сложность: Beginner
  - Таймфрейм: Monthly
  - Сезонность: Yes
  - Нейросети: No
  - Дивергенция: No
  - Уровень риска: Medium
