# スイング・ブレイクアウト戦略 PRO
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

価格が直近の確認されたスイング高値または安値を超えてクローズしたときに取引するブレイクアウト戦略。直近のスイングポイント間の距離がストップロスと目標レベルを定義します。

## 詳細

- **ロング**: 前回終値が直近スイング高値を上回り、かつ現在の高値が前回高値を上回る。
- **ショート**: 前回終値が直近スイング安値を下回り、かつ現在の安値が前回安値を下回る。
- **ストップ**: 反対側のスイングレベル。
- **ターゲット**: 直近スイング高値と安値の間のレンジ。
- **インジケーター**: 内部ピボット計算。
