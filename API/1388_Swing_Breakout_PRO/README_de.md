# Swing-Ausbruch-Strategie PRO
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Ausbruchsstrategie, die handelt, wenn der Kurs über das letzte bestätigte Swing-Hoch oder unter das letzte Swing-Tief schließt. Der Abstand zwischen den letzten Swing-Punkten definiert Stop-Loss- und Zielniveaus.

## Details

- **Long**: vorheriger Schlusskurs über letztem Swing-Hoch und aktuelles Hoch über vorherigem Hoch.
- **Short**: vorheriger Schlusskurs unter letztem Swing-Tief und aktuelles Tief unter vorherigem Tief.
- **Stops**: gegenüberliegendes Swing-Niveau.
- **Ziele**: Spanne zwischen letztem Swing-Hoch und -Tief.
- **Indikatoren**: interne Pivot-Berechnung.
