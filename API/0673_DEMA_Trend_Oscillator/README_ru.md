# Стратегия DEMA Trend Oscillator
[English](README.md) | [中文](README_cn.md)

Стратегия нормализует двойную экспоненциальную среднюю (DEMA) с помощью скользящей средней и стандартного отклонения. Открывает лонг, когда нормализованное значение превышает порог и цена находится выше верхней полосы; открывает шорт, когда значение ниже порога и цена ниже нижней полосы. Использует ATR-трейлинг, стоп по полосе и тейк-профит по соотношению риск/прибыль.

## Детали

- **Условия входа**:
  - Long: нормализованное значение > `LongThreshold` и минимум > верхней полосы
  - Short: нормализованное значение < `ShortThreshold` и максимум < нижней полосы
- **Направление**: Оба
- **Условия выхода**:
  - Long: достижение тейк-профита, стопа по полосе или трейлинг-стопа
  - Short: достижение тейк-профита, стопа по полосе или трейлинг-стопа
- **Стопы**: стоп по полосе, ATR-трейлинг, тейк-профит
- **Значения по умолчанию**:
  - `DemaPeriod` = 40
  - `BaseLength` = 20
  - `LongThreshold` = 55m
  - `ShortThreshold` = 45m
  - `RiskReward` = 1.5m
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: DEMA, SMA, StandardDeviation, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
