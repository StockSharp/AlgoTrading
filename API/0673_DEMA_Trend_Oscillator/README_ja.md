# DEMAトレンドオシレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は二重指数移動平均（DEMA）を移動平均と標準偏差で正規化します。正規化された値がロング閾値を超え、価格が上部バンドを上回っているときにロングポジションを取ります。正規化された値がショート閾値を下回り、価格が下部バンドを下回っているときにショートポジションを取ります。ATRベースのトレーリングストップ、バンドストップロス、リスクリワードベースのテイクプロフィットを使用します。

## 詳細

- **エントリー条件**:
  - ロング: 正規化値 > `LongThreshold` かつ 安値 > 上部バンド
  - ショート: 正規化値 < `ShortThreshold` かつ 高値 < 下部バンド
- **ロング/ショート**: 両方
- **エグジット条件**:
  - ロング: 価格がテイクプロフィット、バンドストップロス、またはトレーリングストップに到達
  - ショート: 価格がテイクプロフィット、バンドストップロス、またはトレーリングストップに到達
- **ストップ**: バンドストップロス、ATRトレーリング、リスクリワードテイクプロフィット
- **デフォルト値**:
  - `DemaPeriod` = 40
  - `BaseLength` = 20
  - `LongThreshold` = 55m
  - `ShortThreshold` = 45m
  - `RiskReward` = 1.5m
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: DEMA, SMA, StandardDeviation, ATR
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
