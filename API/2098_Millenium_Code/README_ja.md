# Millenium Code戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Millenium Code**戦略は、1日に最大1回の取引を開く位置取引システムです。方向は、最近の高値と安値でフィルタリングされた移動平均クロスオーバーによって決定されます。取引はユーザーが定義した時間に置かれ、時間、ストップロス、テイクプロフィット、または最大期間によってクローズされます。

## 取引ロジック

1. 指定された開始時間に、戦略は現在の曜日に取引が許可されているかどうかを確認します。
2. 速い単純移動平均と遅い単純移動平均が比較されます。速いMAが遅いMAを上回り、価格がブレイクアウトを確認した場合、ロングポジションが開かれます。逆の条件でショートポジションが開かれます。
3. 1日に1回の取引のみ許可されます。次の取引日まで、後続のシグナルは無視されます。
4. 以下の場合にポジションがクローズされます:
   - ストップロスまたはテイクプロフィットのレベルに達した場合。
   - 設定されたクローズ時間が来た場合。
   - 最大取引期間を超えた場合。

## パラメーター

- **Candle Type** – 入力ローソク足の時間軸。
- **Fast MA** – 速い移動平均の期間。
- **Slow MA** – 遅い移動平均の期間。
- **HighLow Bars** – 最近の高値と安値を検索するために使用するローソク足の数。
- **Reverse** – 買い/売りシグナルを反転します。
- **Stop Loss** – 価格ステップ単位のストップロスまでの距離。
- **Take Profit** – 価格ステップ単位のテイクプロフィットまでの距離。
- **Open Hour/Minute** – エントリーを探し始める時間（-1で無効）。
- **Close Hour/Minute** – ポジションをクローズする時間（-1で無効）。
- **Duration** – 最大取引期間（時間単位、0で無効）。
- **Sunday ... Friday** – 各曜日の取引を有効にします。

## 注意事項

この戦略は高レベルAPIの機能のみを使用し、インジケーターの履歴に直接アクセスすることを避けています。教育的な例として意図されており、投資アドバイスではありません。
