# ダイバージェンストレーダー（クラシックコンバージョン）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、StockSharp の高レベル API 内の MetaTrader 4 エキスパート アドバイザー **Divergence Trader** の動作を再現します。選択したローソク足価格に基づいて 2 つの単純な移動平均が計算されます (デフォルトではオープン)。システムは、高速平均と低速平均の間の距離がバーごとにどのように変化するかを監視します。

* スプレッドが上向きに拡大し、乖離値が *買い閾値* と *ステイアウト閾値 * の間に留まる場合、ロングポジションがオープンされるか、既存のショートポジションがカバーされます。
* スプレッドがミラーリングされたしきい値内で下側に拡大すると、ショートポジションが入力されるか、既存のロング取引がクローズされます。

元のエキスパートアドバイザーのバーごとの処理と一致する、完成したローソク足のみが使用されます。すべての管理ルールは、イベント駆動型の高レベル呼び出し (`BuyMarket` / `SellMarket`) で実装されます。

## 取引ルール

1. 設定されたローソク足タイプをサブスクライブし、*Fast SMA* と *Slow SMA* の期間で 2 つの SMA を計算します。
2. 現在のスプレッド (`fast - slow`) を計算し、前のスプレッドと比較して発散値を取得します。
3. 乖離がプラスで、*買い閾値*以上、*ステイアウト閾値*以下の場合はロングと入力します。
4. 発散が負の場合、`-Buy Threshold` 以下、かつ `-Stay Out Threshold` 以上の場合は short を入力します。
5. 反対のシグナルが現れるたびに、既存のポジションを反転させます。
6. 新しいエントリを *開始時間* と *終了時間* の間のローカル タイム ウィンドウに制限します (午前 0 時を超えるラップはサポートされています)。

## リスク管理

* オプションの固定 *テイクプロフィット (pips)* および *ストップロス (pips)* レベルは、ローソク足の高値/安値で監視されます。
* *損益分岐点トリガー (ピップ)* は、ポジションが指定されたピップ数を獲得すると、ストップを `entry ± Break-Even Buffer` に移動します。
* *トレーリングストップ (pips)* は、トレードで利益が得られると最も有利な価格に従います。 9999 を設定すると、トレーリング ストップが無効になり、元の EA のデフォルトが反映されます。
* バスケット管理は、未実現損益が *バスケット利益* に達するか、口座通貨で `-Basket Loss` を下回ると、未決済のエクスポージャーをすべてクローズします。

## パラメーター

| パラメータ | 説明 |
|-----------|-------------|
| `Order Volume` | 新しいポジションをオープンするときに使用されるボリューム。 |
| `Fast SMA` / `Slow SMA` | 2 つの単純移動平均の期間。 |
| `Applied Price` | ローソク足コンポーネントは両方の移動平均に転送されます。 |
| `Buy Threshold` | より低いダイバージェンス境界によりロングトレードが可能になります。 |
| `Stay Out Threshold` | それを超えると新たな取引が行われない上限の発散境界。 |
| `Take Profit (pips)` / `Stop Loss (pips)` | ピップ単位で測定されるオプションのハードエグジット。 |
| `Trailing Stop (pips)` | トレードで利益が出た後に適用されるトレーリング距離。 |
| `Break-Even Trigger (pips)` | 損益分岐点にストップを移動する前に必要な利益 (pips)。 |
| `Break-Even Buffer (pips)` | 損益分岐点ストップに追加のバッファーが追加されました。 |
| `Basket Profit` / `Basket Loss` | アカウント通貨でのグローバル資本制限。 |
| `Start Hour` / `Stop Hour` | ローカル取引セッションウィンドウ。 |
| `Candle Type` | キャンドルのサブスクリプションと計算に使用される時間枠。 |

## 使用上の注意

* ストラテジーを証券にアタッチし、元のチャートの時間枠と一致するローソク足のタイプを設定します。
* pip ベースのコントロールが正しく機能するように、楽器の `PriceStep`/`StepPrice` プロパティが設定されていることを確認してください。
* トレーリング ストップや損益分岐点シフトなどの機能を無効にするには、それらのパラメーターを従来のセンチネル値 (9999) またはゼロのままにします。
