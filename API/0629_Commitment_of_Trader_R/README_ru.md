# Стратегия Commitment of Trader R
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует индикатор Williams %R и необязательный фильтр тренда на основе SMA. Длинная позиция открывается, когда %R превышает верхний порог и цена выше SMA. Короткая позиция открывается, когда %R падает ниже нижнего порога и цена ниже SMA. Позиции закрываются при выходе индикатора из зоны.

## Детали
- **Условия входа**:
  - **Long**: %R > верхнего порога и (цена > SMA, если включено)
  - **Short**: %R < нижнего порога и (цена < SMA, если включено)
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - **Long**: %R < верхнего порога
  - **Short**: %R > нижнего порога
- **Стопы**: нет
- **Значения по умолчанию**:
  - `WilliamsPeriod` = 252
  - `UpperThreshold` = -10
  - `LowerThreshold` = -90
  - `SmaEnabled` = true
  - `SmaLength` = 200
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Обе
  - Индикаторы: Williams %R, SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
