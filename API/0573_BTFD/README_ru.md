# BTFD
[English](README.md) | [中文](README_cn.md)

Стратегия покупки на падении с использованием объёма и RSI, включает пять уровней фиксации прибыли и защитный стоп.

## Детали

- **Условия входа**: Скачок объёма выше SMA и RSI ниже зоны перепроданности.
- **Лонг/Шорт**: Только лонг.
- **Условия выхода**: Пять уровней тейк-профита или стоп-лосс.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `VolumeLength` = 70
  - `VolumeMultiplier` = 2.5
  - `RsiLength` = 20
  - `RsiOversold` = 30
  - `Tp1` = 0.4
  - `Tp2` = 0.6
  - `Tp3` = 0.8
  - `Tp4` = 1.0
  - `Tp5` = 1.2
  - `Q1` = 20
  - `Q2` = 40
  - `Q3` = 60
  - `Q4` = 80
  - `Q5` = 100
  - `StopLossPercent` = 5
  - `CandleType` = TimeSpan.FromMinutes(3)
- **Фильтры**:
  - Категория: Разворот
  - Направление: Лонг
  - Индикаторы: RSI, SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (3м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
