# Стратегия Color Schaff DeMarker Trend Cycle
[English](README.md) | [中文](README_cn.md)

**Стратегия Color Schaff DeMarker Trend Cycle** использует осциллятор, построенный на быстрых и медленных значениях DeMarker. Индикатор выполняет два шага стохастика, формируя циклическое значение в диапазоне от -100 до +100. В зависимости от уровня и наклона осциллятора задаются цвета, которые применяются для генерации торговых сигналов.

Стратегия открывает длинные позиции, когда осциллятор выходит из верхней зоны, и закрывает короткие. Короткие позиции открываются, когда осциллятор покидает нижнюю зону, а длинные позиции закрываются. Идея состоит в том, чтобы реагировать на изменения импульса на экстремальных уровнях.

## Детали

- **Критерии входа**:
  - **Длинная позиция**: предыдущий цвет > 5 и текущий цвет < 6.
  - **Короткая позиция**: предыдущий цвет < 2 и текущий цвет > 1.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - **Длинная позиция**: цвет < 2 при открытой длинной позиции.
  - **Короткая позиция**: цвет > 5 при открытой короткой позиции.
- **Стопы**: без явного стоп-лосса и тейк-профита.
- **Значения по умолчанию**:
  - `FastDeMarker` = 23
  - `SlowDeMarker` = 50
  - `Cycle` = 10
  - `HighLevel` = 60
  - `LowLevel` = -60
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: DeMarker, Highest, Lowest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: 4H
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
