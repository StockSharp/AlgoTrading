# Color Schaff DeMarkerトレンドサイクル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Color Schaff DeMarkerトレンドサイクル戦略**は、速いDeMarker値と遅いDeMarker値から派生したカスタムオシレーターを使用します。インジケーターは2つのストキャスティクスステップを適用して、-100から+100の間を振動するサイクル値を作成します。オシレーターのレベルと傾きに基づいて色が割り当てられ、それが取引シグナルの生成に使用されます。

オシレーターが上部ゾーンを出ると、戦略はロングポジションに入り、ショートポジションを離れます。オシレーターが下部ゾーンを離れると、ショートポジションを開き、ロングポジションを離れます。アイデアは、極端なレベルでのモメンタムの変化に反応することです。

## 詳細

- **エントリー条件**:
  - **ロング**: 前の色 > 5かつ現在の色 < 6。
  - **ショート**: 前の色 < 2かつ現在の色 > 1。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: ロングポジションがオープンの時、色 < 2。
  - **ショート**: ショートポジションがオープンの時、色 > 5。
- **ストップ**: 明示的なストップロスやテイクプロフィットなし。
- **デフォルト値**:
  - `FastDeMarker` = 23
  - `SlowDeMarker` = 50
  - `Cycle` = 10
  - `HighLevel` = 60
  - `LowLevel` = -60
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: DeMarker, Highest, Lowest
  - ストップ: いいえ
  - 複雑さ: 中
  - 時間軸: 4H
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
