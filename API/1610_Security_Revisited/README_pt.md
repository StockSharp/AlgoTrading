# `security()` Revisitado
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

`security()` Revisitado demonstra como acessar dados de períodos superiores no StockSharp e controlar o repintado usando valores anteriores ou atuais.

## Detalhes

- **Finalidade**: acesso a dados de período superior com controle de repintado
- **Negociação**: demonstração
- **Indicadores**: nenhum
- **Stops**: não
- **Valores padrão**:
  - `CandleType` = 1 minute
  - `HigherTimeframe` = 5 minutes
  - `Repaint` = false
