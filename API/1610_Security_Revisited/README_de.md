# `security()` Neu betrachtet
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

`security()` Neu betrachtet zeigt, wie man in StockSharp auf Daten eines höheren Zeitrahmens zugreift und das Neuzeichnen durch Verwendung vorheriger oder aktueller Werte steuert.

## Details

- **Zweck**: Zugriff auf Daten höherer Zeitrahmen mit Neuzeichnungskontrolle
- **Handel**: Demonstration
- **Indikatoren**: keine
- **Stops**: nein
- **Standardwerte**:
  - `CandleType` = 1 minute
  - `HigherTimeframe` = 5 minutes
  - `Repaint` = false
