# `security()` Revisitado
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

`security()` Revisitado muestra cómo acceder a datos de marcos temporales superiores en StockSharp y controlar el repintado usando valores anteriores o actuales.

## Detalles

- **Propósito**: acceso a datos de marco temporal superior con control de repintado
- **Operativa**: demostración
- **Indicadores**: ninguno
- **Stops**: no
- **Valores predeterminados**:
  - `CandleType` = 1 minute
  - `HigherTimeframe` = 5 minutes
  - `Repaint` = false
