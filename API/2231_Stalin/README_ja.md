# Stalin インジケーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQL5の「Stalin」インジケーターのロジックを再現します。
一対の指数移動平均（EMA）とオプションのRSIフィルターを使用します。
高速EMAが低速EMAを下から上にクロスし、RSIが50を超えるとロングシグナルが発生します。
高速EMAが低速EMAを上から下にクロスし、RSIが50を下回るとショートシグナルが発生します。

シグナルは必要な価格移動によって確認され、最後のシグナルからの距離によってフィルタリングされます。
ポジションは成行注文で建てられ、反対シグナルで転換されます。

## 詳細

- **エントリー条件**:
  - **ロング**: `FastEMA(t-1) < SlowEMA(t-1)` && `FastEMA(t) > SlowEMA(t)` && `RSI(t) > 50`.
  - **ショート**: `FastEMA(t-1) > SlowEMA(t-1)` && `FastEMA(t) < SlowEMA(t)` && `RSI(t) < 50`.
- **確認**: 価格がブレイクアウトレベルから `Confirm` ポイント動くまで取引が遅延されます。
- **Flatフィルター**: 前回のシグナル価格から `Flat` ポイント以内の新しいシグナルは無視されます。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `FastLength` = 14.
  - `SlowLength` = 21.
  - `RsiLength` = 17.
  - `Confirm` = 0ポイント（無効）。
  - `Flat` = 0ポイント（無効）。
  - `CandleType` = 1時間ローソク足。
- **フィルター**:
  - カテゴリ: トレンドフォロー。
  - 方向: 両方。
  - インジケーター: 複数。
  - ストップ: いいえ。
  - 複雑さ: 中程度。
  - 時間軸: 中期。
  - 季節性: いいえ。
  - ニューラルネットワーク: いいえ。
  - ダイバージェンス: いいえ。
  - リスクレベル: 中。
