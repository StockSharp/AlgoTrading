# Стратегия Dekidaka-Ashi Candles Volume
[English](README.md) | [中文](README_cn.md)

Стратегия использует подход Dekidaka-Ashi, где диапазон свечи зависит от сглаженного объёма. Бычьи сигналы открывают лонг, медвежьи — шорт. Свечи, пересекающие обе границы, закрывают позиции.

## Детали

- **Условия входа**:
  - Сильный или слабый бычий сигнал: максимум выше верхней границы и минимум выше нижней.
  - Сильный или слабый медвежий сигнал: максимум ниже верхней границы и минимум ниже нижней.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Противоположный сигнал или свеча, пересекающая обе границы (неопределённость).
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `BodySize` = 1
  - `VolumeSmooth` = 1
  - `CandleType` = таймфрейм 5 минут
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Лонг и шорт
  - Индикаторы: EMA, объём
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
