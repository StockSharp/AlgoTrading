# Dekidaka-Ashi 蜡烛与成交量策略
[English](README.md) | [Русский](README_ru.md)

该策略使用 Dekidaka-Ashi 方法，根据成交量平滑调整蜡烛范围。出现看涨信号时做多，出现看跌信号时做空。上下界同时被突破的蜡烛将平掉已有仓位。

## 细节

- **入场条件**：
  - 强或弱看涨信号：最高价高于上界且最低价高于下界。
  - 强或弱看跌信号：最高价低于上界且最低价低于下界。
- **多空方向**：双向。
- **出场条件**：
  - 反向信号或同时穿越上下界的蜡烛（不确定）。
- **止损**：无。
- **默认值**：
  - `BodySize` = 1
  - `VolumeSmooth` = 1
  - `CandleType` = 5 分钟
- **筛选**：
  - 类型：形态
  - 方向：多头与空头
  - 指标：EMA、成交量
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
