# Relative Strength RSMK Plus Perk
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторе относительной силы Маркоса Кацаноса (RSMK).

Стратегия сравнивает актив с рыночным индексом. Длинная позиция открывается, когда RSMK пересекает сигнальную линию сверху вниз, короткая — когда пересекает снизу вверх. Обратное пересечение закрывает позицию.

## Детали

- **Критерии входа**: пересечение RSMK и сигнальной линии.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Period` = 90
  - `Smooth` = 3
  - `SignalPeriod` = 20
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: RSMK, EMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
