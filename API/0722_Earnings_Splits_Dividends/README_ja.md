ニュースフィードから収益、株式分割、配当情報を受信する方法を示します。

この戦略は市場ニュースを購読し、収益・株式分割・配当に関連する見出しが現れた際にログエントリを記録します。
取引操作は行わず、コーポレートアクションデータの処理例として機能します。

- **エグジット条件**: 該当なし。
- **ストップ**: なし。
- **フィルター**:
  - カテゴリ: Data
  - 方向: なし
  - インジケーター: なし
  - ストップ: なし
  - 複雑さ: 基本
