# Lucros, Splits e Dividendos
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Demonstra como receber informações sobre lucros, splits e dividendos a partir do feed de notícias.

A estratégia assina as notícias do mercado e registra uma entrada no diário quando aparece uma manchete relacionada a lucros, splits de ações ou dividendos.
Não realiza operações de trading, mas serve como exemplo de tratamento de dados de ações corporativas.

## Detalhes

- **Critérios de entrada**: Nenhum.
- **Comprado/Vendido**: Sem trading.
- **Critérios de saída**: Não aplicável.
- **Stops**: Não.
