# Ganancias, Splits y Dividendos
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demuestra cómo recibir información sobre ganancias, splits y dividendos a través del feed de noticias.

La estrategia se suscribe a las noticias del mercado y registra una entrada en el diario cuando aparece un titular relacionado con ganancias, splits de acciones o dividendos.
No realiza operaciones de trading, sino que sirve como ejemplo del manejo de datos de acciones corporativas.

## Detalles

- **Criterios de entrada**: Ninguno.
- **Largo/Corto**: Sin trading.
- **Criterios de salida**: No aplicable.
- **Stops**: No.
- **Filtros**:
  - Categoría: Data
  - Dirección: Ninguno
  - Indicadores: Ninguno
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Diario
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Bajo
