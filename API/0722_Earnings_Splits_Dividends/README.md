# Earnings, Splits, Dividends
[Русский](README_ru.md) | [中文](README_cn.md)

Demonstrates how to receive earnings, split and dividend information from the news feed.

The strategy subscribes to market news and writes a log entry when a headline related to earnings, stock splits or dividends appears.
It does not perform trading operations but serves as an example of handling corporate action data.
