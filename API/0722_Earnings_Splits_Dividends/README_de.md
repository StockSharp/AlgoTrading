Demonstriert, wie Informationen zu Gewinnen, Splits und Dividenden aus dem Nachrichten-Feed empfangen werden.

Die Strategie abonniert Marktnachrichten und schreibt einen Protokolleintrag, wenn eine Schlagzeile zu Gewinnen, Aktiensplits oder Dividenden erscheint.
Sie führt keine Handelsoperationen durch, sondern dient als Beispiel für die Verarbeitung von Unternehmensereignisdaten.
