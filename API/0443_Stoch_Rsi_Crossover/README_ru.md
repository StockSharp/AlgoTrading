# Стратегия пересечения Stochastic RSI
[English](README.md) | [中文](README_cn.md)

Метод преобразует классический индекс относительной силы в Stochastic RSI и сглаживает его, получая линии %K и %D. Когда %K пересекает %D внутри заранее заданных зон, это подразумевает краткосрочный разворот импульса. Торговля разрешается только тогда, когда тройная структура EMA подтверждает направление глобального тренда, что позволяет отсекать ложные сигналы.

После появления пересечения цена закрытия должна находиться выше или ниже быстрой EMA в зависимости от направления сигнала. Это защищает от входа против тенденции и концентрирует внимание на моментах, когда импульс совпадает с направлением. Параметры сглаживания и длину RSI можно настраивать, изменяя чувствительность системы к всплескам волатильности.

Для оценки риска используется показатель среднего истинного диапазона. Множители текущего ATR задают ориентиры стоп‑лосса и тейк‑профита, динамически расширяясь в периоды высокой активности и сужаясь при спокойном рынке. Скрипт не отправляет защитные ордера автоматически, но рассчитанные уровни помогают вручную управлять позицией или могут подключаться к дополнительным модулям риска.

## Детали

- **Условия входа**:
  - **Покупка**: `%K` пересекает `%D` снизу вверх, `%K` в диапазоне `[10,60]`, EMA указывают бычий тренд, цена выше EMA1.
  - **Продажа**: `%K` пересекает `%D` сверху вниз, `%K` в диапазоне `[40,95]`, EMA указывают медвежий тренд, цена ниже EMA1.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Не реализованы.
- **Стопы**: Предлагаются множители ATR, но не ставятся автоматически.
- **Значения по умолчанию**:
  - `SmoothK` = 3, `SmoothD` = 3.
  - `RsiLength` = 14, `StochLength` = 14.
  - `Ema1Length` = 20, `Ema2Length` = 50, `Ema3Length` = 100.
  - `AtrLength` = 14, `AtrLossMultiplier` = 1.5, `AtrProfitMultiplier` = 2.0.
- **Фильтры**:
  - Категория: Импульсная
  - Направление: Оба
  - Индикаторы: Несколько
  - Стопы: Опционально
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
