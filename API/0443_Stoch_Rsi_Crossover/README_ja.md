# Stochastic RSIクロスオーバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この手法はクラシックなRelative Strength IndexをStochastic RSIに変換し、結果を%Kと%Dラインに平滑化します。%Kが慎重に選ばれたゾーン内で%Dをクロスすると、その動きは短期的なモメンタムの変化を意味します。アルゴリズムは3層のEMA構造が広義のトレンドの方向を確認した時のみ取引し、ダマシをフィルタリングします。

クロスオーバーが現れると、終値もシグナルに応じて速いEMAの上または下に位置しなければなりません。これは優勢なトレンドに逆らって起こるオシレーションで行動することを防ぎ、モメンタムが方向と一致する瞬間に注意を向けます。トレーダーはスムージング期間とRSIの長さを調整して、システムがボラティリティスパイクにどれほど敏感に反応するかを調整できます。

リスクはAverage True Rangeの読みを通じて参照されます。現在のATRの乗数がストップロスと利益目標を提案し、ボラティリティの高い市場では拡大し活動が落ち着くと縮小する動的なレベルを提供します。スクリプトは保護的な注文を自動的に送信しませんが、これらの計算されたレベルは手動管理を補助したり、追加のリスクモジュールに組み込むことができます。

## 詳細

- **エントリー条件**:
  - **ロング**: `%K`が`%D`を上抜け、`%K`が`[10,60]`の範囲、EMAsが強気に整列、価格がEMA1の上。
  - **ショート**: `%K`が`%D`を下抜け、`%K`が`[40,95]`の範囲、EMAsが弱気に整列、価格がEMA1の下。
- **ロング/ショート**: 両方。
- **エグジット条件**: 組み込みなし。
- **ストップ**: ATRの倍数が提案されるが自動的には配置されない。
- **デフォルト値**:
  - `SmoothK` = 3, `SmoothD` = 3.
  - `RsiLength` = 14, `StochLength` = 14.
  - `Ema1Length` = 20, `Ema2Length` = 50, `Ema3Length` = 100.
  - `AtrLength` = 14, `AtrLossMultiplier` = 1.5, `AtrProfitMultiplier` = 2.0.
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: オプション
  - 複雑さ: 中程度
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
