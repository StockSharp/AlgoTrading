# ハイイールドスプレッドSMAフィルター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はハイイールドスプレッドまたはVIX指数に基づいて取引します。選択したスプレッドが閾値を越え、オプションの価格フィルターが確認した時にポジションを開きます。価格フィルターはロングの場合は終値が単純移動平均より上であること、ショートの場合は下であることを要求します。ポジションは固定本数のバー後に決済されます。

## 詳細

- **エントリー条件**:
  - **ロング**: スプレッド > 閾値 かつ 終値 > SMA（有効時）。
  - **ショート**: スプレッド < 閾値 かつ 終値 < SMA（有効時）。
- **ロング/ショート**: 両方、パラメーターで選択。
- **エグジット条件**:
  - 保有期間バー後にポジションを決済。
- **ストップ**: なし。
- **デフォルト値**:
  - `Threshold` = 5
  - `HoldingPeriod` = 5
  - `SmaLength` = 50
- **フィルター**:
  - カテゴリ: マクロ
  - 方向: 両方
  - インジケーター: High Yield Spread/VIX, SMA
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 1d（デフォルト）
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
