# Парный трейдинг акций

[English](README.md) | [中文](README_zh.md)

Упрощённая стратегия парного трейдинга для нескольких пар акций. Для каждой пары отслеживается ценовое отношение и его z-оценка в скользящем окне. При превышении порога открывается длинная и короткая позиция, а при возврате z-оценки позиция закрывается.

Алгоритм позволяет одновременно торговать несколькими независимыми парами.

## Детали

- **Вселенная**: список пар акций.
- **Сигнал**: z-оценка ценового отношения выше `EntryZ`.
- **Выход**: закрытие при достижении `ExitZ`.
- **Данные**: дневные свечи, окно по умолчанию 60 дней.
- **Контроль риска**: сделка пропускается, если сумма меньше `MinTradeUsd`.
