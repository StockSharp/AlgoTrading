# 株式ペアトレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この簡略化されたペアトレーディング戦略は、複数の株式ペアで運用されます。各ペアについて、価格比率をローリングウィンドウで追跡し、そのzスコアを計算します。zスコアがエントリー閾値を超えるとロング/ショート取引が開始され、zスコアが戻るとポジションがクローズされます。

このアルゴリズムは複数の独立したペアの同時取引をサポートします。

## 詳細

- **ユニバース**: 株式ペアのリスト。
- **シグナル**: 価格比率のzスコアが `EntryZ` を超える。
- **エグジット**: zスコアが `ExitZ` に達したらクローズ。
- **データ**: 日足ローソク足、デフォルトは60日のルックバック。
- **リスク管理**: 注文価値が `MinTradeUsd` を下回る場合は取引をスキップ。
