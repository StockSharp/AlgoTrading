# Стратегия AutoFib Breakout
[English](README.md) | [中文](README_cn.md)

Стратегия строит динамическое расширение Фибоначчи от последнего локального максимума и минимума и открывает длинную позицию, когда цена пробивает уровень 1.618 в восходящем тренде, определяемом EMA(200). Риск контролируется стопом и целью на основе ATR.

## Детали

- **Условия входа**: закрытие выше уровня 1.618 и выше EMA200.
- **Длинная/Короткая**: только длинные.
- **Условия выхода**: стоп-лосс по ATR или тейк-профит 3×ATR.
- **Стопы**: да, на основе ATR.
- **Значения по умолчанию**:
  - `EmaLength` = 200
  - `AtrLength` = 14
  - `FibLevel` = 1.618
  - `PivotPeriod` = 10
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Пробой
  - Направление: Лонг
  - Индикаторы: EMA, ATR, Highest, Lowest
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
