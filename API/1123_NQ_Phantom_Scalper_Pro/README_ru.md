# Стратегия NQ Phantom Scalper Pro
[English](README.md) | [中文](README_cn.md)

Стратегия прорыва полос VWAP с дополнительными фильтрами по объёму и тренду.

## Детали

- **Условия входа**:
  - **Лонг**: Цена закрывается выше верхней полосы VWAP при подтверждающем объёме.
  - **Шорт**: Цена закрывается ниже нижней полосы VWAP при подтверждающем объёме.
- **Лонг/Шорт**: Оба направления
- **Условия выхода**:
  - Цена возвращается к VWAP или срабатывает ATR-стоп.
- **Стопы**: на основе ATR
- **Значения по умолчанию**:
  - `Band #1 Mult` = 1.0
  - `Band #2 Mult` = 2.0
  - `ATR Length` = 14
  - `ATR Stop Mult` = 1.0
  - `Volume SMA Period` = 20
  - `Volume Spike Mult` = 1.5
  - `Trend EMA Length` = 50
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: VWAP, ATR, EMA, SMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
