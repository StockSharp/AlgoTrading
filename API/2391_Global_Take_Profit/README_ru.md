# Global Take Profit Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает общий прибыль/убыток и закрывает все позиции при достижении заданной цели прибыли.

## Детали

- **Критерии входа**: отсутствуют. Стратегия только контролирует уже открытые позиции.
- **Длинные/короткие**: работает с обоими направлениями.
- **Критерии выхода**: закрытие всех позиций при достижении заданного профита.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Mode` = Percent
  - `TakeProfit` = 100
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: 1 минута
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий

## Как это работает

1. При запуске стратегия запоминает текущую стоимость портфеля.
2. После завершения каждой свечи вычисляется реализованный профит стратегии.
3. Если профит отрицательный и цель ещё не достигнута, проверка пропускается.
4. Когда профит превышает заданный порог (в процентах от начальной стоимости или в валюте), стратегия закрывает все позиции рыночными заявками.
5. После закрытия позиций мониторинг начинается заново.
