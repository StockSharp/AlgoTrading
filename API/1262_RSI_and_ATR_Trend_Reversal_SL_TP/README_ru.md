# RSI и ATR Trend Reversal SL TP
[English](README.md) | [中文](README_cn.md)

Стратегия использует RSI и ATR для определения разворотов тренда с динамическими уровнями стоп-лосса и тейк-профита.

## Детали

- **Критерий входа**: Пересечение цены адаптивного порога RSI/ATR.
- **Длинные/Короткие**: Оба.
- **Критерий выхода**: Обратное пересечение.
- **Стопы**: Реализованы через динамический порог.
- **Значения по умолчанию**:
  - `RsiLength` = 8
  - `RsiMultiplier` = 1.5
  - `Lookback` = 1
  - `MinDifference` = 10
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Trend
  - Направление: Both
  - Индикаторы: RSI, ATR
  - Стопы: Да
  - Сложность: Intermediate
  - Таймфрейм: Intraday (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
