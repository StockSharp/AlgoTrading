# Renko Strategy
[English](README.md) | [中文](README_cn.md)

Эта стратегия открывает длинную позицию, когда бычий кирпич ренко появляется после медвежьего, и короткую позицию при обратном изменении.

## Подробности

- **Условия входа**:
  - **Лонг**: бычий кирпич после медвежьего.
  - **Шорт**: медвежий кирпич после бычьего.
- **Лонг/Шорт**: Оба.
- **Условия выхода**:
  - Обратный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `BoxSize` = 10m.
  - `Volume` = 1m.
  - `CandleType` = Renko.
- **Фильтры**:
  - Категория: Trend Following
  - Направление: Оба
  - Индикаторы: Renko
  - Стопы: Нет
  - Сложность: Basic
  - Таймфрейм: Renko
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Medium
