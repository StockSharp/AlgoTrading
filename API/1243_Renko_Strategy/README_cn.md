# Renko Strategy
[English](README.md) | [Русский](README_ru.md)

该策略在看跌砖后出现看涨砖时做多，在相反变化时做空。

## 细节

- **入场条件**:
  - **多头**: 看跌砖后出现看涨砖。
  - **空头**: 看涨砖后出现看跌砖。
- **多头/空头**: 两者.
- **出场条件**:
  - 反向信号.
- **止损**: 无.
- **默认值**:
  - `BoxSize` = 10m.
  - `Volume` = 1m.
  - `CandleType` = Renko.
- **筛选**:
  - 类别: Trend Following
  - 方向: 双向
  - 指标: Renko
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: Renko
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
