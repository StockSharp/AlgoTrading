# 道氏理论趋势策略
[English](README.md) | [Русский](README_ru.md)

该策略通过枢轴高点和低点判断趋势方向。当出现更高的高点和更高的低点时做多，当出现更低的高点和更低的低点时做空。

## 细节

- **入场条件**：更高的高点和低点做多；更低的高点和低点做空。
- **多空方向**：双向。
- **离场条件**：反向信号。
- **止损**：无。
- **默认值**：
  - `PivotLookback` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **过滤条件**：
  - 分类：趋势跟随
  - 方向：双向
  - 指标：价格行为
  - 止损：无
  - 复杂度：基础
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：是
  - 风险等级：中等
