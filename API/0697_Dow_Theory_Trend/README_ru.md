# Стратегия тренда по теории Доу
[English](README.md) | [中文](README_cn.md)

Стратегия определяет направление тренда по пивотам. При появлении последовательности повышающихся максимумов и минимумов открывается длинная позиция, при понижении максимумов и минимумов открывается короткая позиция.

## Детали

- **Условия входа**: рост пивотных максимумов и минимумов для лонга; падение максимумов и минимумов для шорта.
- **Long/Short**: обе стороны.
- **Условия выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `PivotLookback` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: Ценовое действие
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
