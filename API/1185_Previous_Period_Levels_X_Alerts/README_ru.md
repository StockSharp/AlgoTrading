# Предыдущие уровни периода - X Alerts
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает уровни открытия, максимума, минимума и закрытия предыдущего периода старшего таймфрейма. На основном таймфрейме рассчитывается скользящая средняя и при её пересечении этих уровней формируются сообщения в журнал, аналогично индикатору TradingView "Previous Period Levels - X Alerts".

## Детали

- **Условия входа**: нет, только логирование пересечений SMA с уровнями.
- **Длинные/короткие позиции**: Н/Д.
- **Условия выхода**: Н/Д.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `ReferenceCandleType` = TimeSpan.FromHours(1)
  - `SmaLength` = 3
  - `UseOpen` = true
  - `UseHigh` = true
  - `UseLow` = true
  - `UseClose` = true
- **Фильтры**:
  - Категория: Levels
  - Направление: Neutral
  - Индикаторы: SMA
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
