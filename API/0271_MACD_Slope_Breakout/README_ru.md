# Прорыв наклона MACD

Стратегия MACD Slope Breakout отслеживает скорость изменения MACD. Чрезмерно крутой наклон сигнализирует о зарождении нового тренда.

Вход выполняется, когда наклон превышает типичный уровень на несколько стандартных отклонений. Сделка открывается в направлении ускорения и сопровождается стопом.

Стратегия интересна активным трейдерам, желающим рано войти в тренд. Позиции закрываются, когда наклон возвращается к нормальным значениям. Значение по умолчанию `FastEma` = 12.

## Подробности

- **Условие входа**: Индикатор превышает среднее значение на величину коэффициента отклонения.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `FastEma` = 12
  - `SlowEma` = 26
  - `SignalMa` = 9
  - `SlopePeriod` = 20
  - `BreakoutMultiplier` = 2.0m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
