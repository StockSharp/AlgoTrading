# Стратегия Fibonacci-Only V2
[English](README.md) | [中文](README_cn.md)

Стратегия работает с уровнями Фибоначчи 19% и 82.56%, рассчитанными по 93 свечам. Вход выполняется при касании или пробое уровня с подтверждением свечой. Риск контролируется стоп-лоссом по ATR и опциональным трейлинг-стопом.

## Детали

- **Критерий входа**: касание или пробой уровней 19% / 82.56% с подтверждением свечой
- **Длинная/короткая сторона**: обе
- **Критерий выхода**: стоп-лосс или трейлинг-стоп
- **Стопы**: да
- **Значения по умолчанию:**
  - `CandleType` = 15 минут
- **Фильтры:**
  - Категория: пробой Фибоначчи
  - Направление: обе
  - Индикаторы: ATR, Highest, Lowest
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
