# Estratégia de Padrão E9 Shark-32
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia identifica o padrão Shark-32 em três velas consecutivas e opera rompimentos acima da máxima do padrão ou abaixo da mínima do padrão. As posições são fechadas em níveis de alvo espelhados ou stops fixos.

## Parâmetros
- Tipo de vela
- Stop loss comprado (%)
- Stop loss vendido (%)
