# Estrategia de Patrón E9 Shark-32
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia identifica el patrón Shark-32 en tres velas consecutivas y opera rupturas por encima del máximo del patrón o por debajo del mínimo del patrón. Las posiciones se cierran en niveles objetivo espejados o stops fijos.

## Parámetros
- Tipo de vela
- Stop loss largo (%)
- Stop loss corto (%)
