# E9 Shark-32 パターン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、3本の連続したローソク足上で Shark-32 パターンを識別し、パターン高値を上抜けるブレイクアウトまたはパターン安値を下抜けるブレイクアウトで取引します。ポジションは鏡像ターゲットレベルまたは固定ストップロスで閉じます。

## パラメーター
- ローソク足タイプ
- ロング・ストップロス (%)
- ショート・ストップロス (%)
