# E9 Shark-32 Muster-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie erkennt das Shark-32 Muster auf drei aufeinanderfolgenden Kerzen und handelt Ausbrüche über dem Musterhoch oder unter dem Mustertief. Positionen werden bei gespiegelten Zielleveln oder festen Stop-Losses geschlossen.

## Parameter
- Kerzentyp
- Long Stop-Loss (%)
- Short Stop-Loss (%)
