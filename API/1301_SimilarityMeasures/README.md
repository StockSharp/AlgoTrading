# Similarity Measures Strategy
[Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

A sample strategy demonstrating the use of similarity measures.
It calculates the Euclidean distance between price and its simple moving average (SMA).
When the distance falls below a threshold the strategy buys, and when it rises above the threshold it sells.
