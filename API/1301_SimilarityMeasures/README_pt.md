# Estratégia de Medidas de Similaridade
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Uma estratégia de exemplo que demonstra o uso de medidas de similaridade.
Calcula a distância euclidiana entre o preço e sua média móvel simples (SMA).
Quando a distância cai abaixo de um limiar, a estratégia compra; quando sobe acima do limiar, vende.
