# Ähnlichkeitsmaße-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Eine Beispielstrategie zur Demonstration der Verwendung von Ähnlichkeitsmaßen.
Sie berechnet den euklidischen Abstand zwischen dem Preis und seinem einfachen gleitenden Durchschnitt (SMA).
Wenn der Abstand unter einen Schwellenwert fällt, kauft die Strategie; wenn er über den Schwellenwert steigt, verkauft sie.
