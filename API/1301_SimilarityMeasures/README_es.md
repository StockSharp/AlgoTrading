# Estrategia de Medidas de Similitud
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Una estrategia de ejemplo que demuestra el uso de medidas de similitud.
Calcula la distancia euclidiana entre el precio y su media móvil simple (SMA).
Cuando la distancia cae por debajo de un umbral, la estrategia compra; cuando sube por encima del umbral, vende.
