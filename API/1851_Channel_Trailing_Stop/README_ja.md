# チャネル・トレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Donchianチャネルのブレイクアウトエントリーとトレーリングストップ管理を使用した戦略。

価格がチャネル外でクローズしたときにトレードを開きます。トレーリングストップはチャネルの反対側にオフセットを加えた位置を追跡します。オプションの「ノーズ」トレーリングは、現在価格とテイクプロフィットからの距離が等しくなるようストップロスを維持します。約定後にペンディングオーダーを削除することができます。

## 詳細

- **エントリー条件**: チャネル範囲外でのクローズ。
- **ロング/ショート**: 両方。
- **エグジット条件**: トレーリングストップまたは反対シグナル。
- **ストップ**: トレーリングストップ、オプションのノーズ。
- **デフォルト値**:
  - `TrailPeriod` = 5
  - `TrailStop` = 50
  - `UseNooseTrailing` = true
  - `UseChannelTrailing` = true
  - `DeletePendingOrders` = true
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: Donchian Channel
  - ストップ: トレーリング
  - 複雑さ: 中級
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
