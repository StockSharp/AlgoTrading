# Канальный трейлинг-стоп
[English](README.md) | [中文](README_cn.md)

Стратегия открывает сделки при пробое канала Дончиана и управляет риском с помощью трейлинг-стопа.

Когда цена закрывается вне канала, открывается позиция. Трейлинг-стоп тянется по противоположной стороне канала с добавленным смещением. Опциональный "узел" поддерживает симметрию между текущей ценой и тейк-профитом. После исполнения сделок возможно удаление отложенных ордеров.

## Детали

- **Условия входа**: закрытие за пределами канала.
- **Направление**: обе стороны.
- **Условия выхода**: трейлинг-стоп или встречный сигнал.
- **Стопы**: трейлинг-стоп, опционально узел.
- **Значения по умолчанию**:
  - `TrailPeriod` = 5
  - `TrailStop` = 50
  - `UseNooseTrailing` = true
  - `UseChannelTrailing` = true
  - `DeletePendingOrders` = true
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе
  - Индикаторы: Канал Дончиана
  - Стопы: Трейлинг
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
