# Стратегия Hancock RSI Volume
[English](README.md) | [中文](README_cn.md)

Стратегия рассчитывает индекс относительной силы (RSI), взвешенный по объёму, по мотивам скрипта Hancock из TradingView. RSI учитывает объёмы покупок и продаж для оценки силы тренда. Длинная позиция открывается при переходе RSI в восходящий тренд, короткая — при переходе в нисходящий.

## Детали

- **Критерии входа**:
  - **Long**: тренд RSI меняется на рост.
  - **Short**: тренд RSI меняется на падение.
- **Long/Short**: обе стороны.
- **Критерии выхода**:
  - Сигнал противоположного тренда.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `RSI Length` = 14.
  - `Threshold` = 0.1.
  - `Use Wicks` = true.
- **Фильтры**:
  - Категория: Momentum
  - Направление: Обе
  - Индикаторы: RSI, Volume
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
