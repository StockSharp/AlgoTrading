# Hancock RSI出来高戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はTradingViewのHancockスクリプトに触発された出来高加重RSIを計算します。RSIは強気・弱気の出来高を使ってトレンドの強さを測定します。RSIのトレンドが上向きに転じるとロングポジションを建て、下向きに転じるとショートポジションを建てます。

## 詳細

- **エントリー条件**:
  - **ロング**: RSIトレンドが上向きに転換。
  - **ショート**: RSIトレンドが下向きに転換。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 反対のトレンドシグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `RSI Length` = 14.
  - `Threshold` = 0.1.
  - `Use Wicks` = true.
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: RSI, 出来高
  - ストップ: いいえ
  - 複雑さ: 中
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
