# マルチレイヤー Acceleration/Deceleration 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は Acceleration/Deceleration オシレーターを使って最大 5 つのロングエントリーを積み重ねます。フラクタルとアリゲーターの歯で識別されたトレンドの方向にモメンタムが強まるたびに、バーの高値より上に買いストップ注文を置きます。オシレーターが弱まるかトレンドが反転すると、すべての未決注文がキャンセルされポジションが決済されます。

## 詳細

- **エントリー条件**:
  - アリゲーターの歯より上にある上方フラクタルを価格が突破したときに上昇トレンドを確認。
  - AC オシレーターが緑のバーパターンを表示し、終値が EMA フィルターを上回る。
  - 活性化レベルに最大 5 つのストップ注文を発注。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - トレンドが下降に転換する。
  - オシレーターがマイナスになる。
- **ストップ**: フラクタルベースのストップロスを使用。
- **デフォルト値**:
  - `EMA Length` = 100.
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロングのみ
  - インジケーター: 複数
  - ストップ: はい
  - 複雑さ: 複雑
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
