# Парный трейдинг с фильтром волатильности

Стратегия Pairs Trading Volatility Filter использует пары вместе с фильтрами волатильности. Вход осуществляется только при выполнении заданных условий.

Сигналы появляются, когда индикатор превышает порог, а волатильность соответствует установленным критериям. Позиции могут быть как длинными, так и короткими с встроенными стопами.

Подходит трейдерам, для которых важен контроль риска: стратегия выходит из позиции, как только индикатор возвращается к среднему или волатильность меняется. Начальное значение `LookbackPeriod` = 20.

## Детали

- **Условия входа**: Индикатор разворачивается в сторону среднего значения.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `LookbackPeriod` = 20
  - `EntryThreshold` = 2.0m
  - `ExitThreshold` = 0.0m
  - `StopLossPercent` = 2.0m
- **Фильтры**:
  - Категория: Средневозвратная
  - Направление: Оба
  - Индикаторы: Pairs
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
