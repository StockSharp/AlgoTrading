# Fibonacci Auto Trend Scouter 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はFibonacci数に基づく2つのローリング極値を使用して新興トレンドを探索します。短期ウィンドウ（8）は直近の高値・安値を追跡し、長期ウィンドウ（21）はコンテキストを提供します。短期高値が長期高値を上回ったときにロングポジションを建て、短期安値が長期安値を下回ったときにショートポジションを建てます。

## 詳細

- **エントリー条件**:
  - **ロング**: 短期高値 > 長期高値。
  - **ショート**: 短期安値 < 長期安値。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 反対シグナルでポジションを反転。
- **ストップ**: なし。
- **デフォルト値**:
  - `Short period` = 8
  - `Long period` = 21
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: なし
  - 複雑さ: シンプル
  - 時間軸: 中期
