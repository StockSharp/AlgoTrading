# Source 策略
[English](README.md) | [Русский](README_ru.md)

Source 策略在收盘价高于开盘价时做多，在收盘价低于开盘价时做空。可选的止损、止盈和追踪止损参数用于管理持仓风险。

## 详情

- **入场条件**：收盘价高于开盘价做多，收盘价低于开盘价做空
- **多空**：双向
- **出场条件**：反向信号或止损机制触发
- **止损**：可选止损、止盈、追踪止损
- **默认值**：
  - `SL %` = 1
  - `TP %` = 3
  - `Trail Points %` = 3
  - `Trail Offset %` = 1
- **过滤器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
