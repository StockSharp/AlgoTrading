# Double Bottom and Top Hunter Strategy
[English](README.md) | [中文](README_cn.md)

Эта стратегия ищет паттерны двойного дна и двойной вершины, сравнивая недавние минимумы и максимумы. Двойное дно фиксируется, когда минимальное значение достигается дважды в рамках расширенного периода, двойная вершина использует максимальное значение. Длинные и короткие позиции открываются соответственно и закрываются при пробое противоположного экстремума после обновления текущего.

## Подробности

- **Критерии входа**:
  - **Long**: Обнаружено двойное дно.
  - **Short**: Обнаружена двойная вершина.
- **Long/Short**: Оба направления.
- **Критерии выхода**:
  - Long: Новый максимум выше предыдущего максимума при падении цены ниже предыдущего минимума.
  - Short: Новый минимум ниже предыдущего минимума при росте цены выше предыдущего максимума.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Length` = 100
  - `Lookback` = 100
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Highest, Lowest
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
