# Double Bottom and Top Hunter Strategy
[English](README.md) | [Русский](README_ru.md)

该策略通过比较近期的高点和低点来寻找双底和双顶形态。当最低点在更长的回溯期内被触及两次时，形成双底；双顶则使用最高点。策略相应开立多头和空头头寸，并在出现新的极值后价格突破相反极值时平仓。

## 详情

- **入场条件**:
  - **多头**: 检测到双底。
  - **空头**: 检测到双顶。
- **多/空**: 双向。
- **出场条件**:
  - 多头: 新高高于先前高点且价格跌破先前低点。
  - 空头: 新低低于先前低点且价格突破先前高点。
- **止损**: 无。
- **默认值**:
  - `Length` = 100
  - `Lookback` = 100
- **筛选器**:
  - 类别: 模式
  - 方向: 双向
  - 指标: Highest, Lowest
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 低
