# ダブルボトム・ダブルトップ・ハンター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、直近の安値と高値を比較することでダブルボトムとダブルトップのパターンを探します。ダブルボトムは最安値が広いルックバック期間内で2回到達した時に発生し、ダブルトップは最高値を使用します。ロングとショートのポジションはそれぞれ開設され、新たな極値が形成された後に価格が反対の極値を突破した時に決済されます。

## 詳細

- **エントリー条件**:
  - **ロング**: ダブルボトム検出。
  - **ショート**: ダブルトップ検出。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - ロング: 前回高値を超える新高値で価格が前回安値を下回る。
  - ショート: 前回安値を下回る新安値で価格が前回高値を上回る。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 100
  - `Lookback` = 100
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Highest, Lowest
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
