# Стратегия Three Signal Directional Trend
[English](README.md) | [中文](README_cn.md)

Стратегия **Three Signal Directional Trend** объединяет три индикатора: MACD, стохастик и скорость скользящей средней (ROC). Каждый индикатор подает сигнал на покупку или продажу. Позиция открывается, когда хотя бы два индикатора совпадают по направлению. Такой подход помогает отфильтровать шум и следовать основному тренду.

## Подробности

- **Условия входа:**
  - Совпадение минимум двух сигналов.
  - **Long**: MACD растет, стохастик в зоне перепроданности, MA ROC положительный.
  - **Short**: MACD падает, стохастик в зоне перекупленности, MA ROC отрицательный.
- **Long/Short**: обе стороны.
- **Условия выхода:**
  - Противоположный сигнал.
- **Стопы**: отсутствуют.
- **Значения по умолчанию:**
  - `AvgLength` = 50
  - `RocLength` = 1
  - `AvgRocLength` = 10
  - `StochLength` = 14
  - `SmoothK` = 3
  - `Overbought` = 80
  - `Oversold` = 20
  - `MacdFastLength` = 12
  - `MacdSlowLength` = 26
  - `MacdAvgLength` = 9
- **Фильтры:**
  - Категория: следование тренду
  - Направление: обе стороны
  - Индикаторы: MACD, Stochastic, SMA, ROC
  - Стопы: нет
  - Сложность: низкая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
