# ハイブリッド・スキャルピング・ボット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSIシグナルとEMAトレンドフィルター、オプションのボリューム確認を組み合わせたハイブリッド・スキャルピングシステムです。シグナルの感度を非常に緩やかから強力まで調整でき、クイック・エグジットとトレーリングストップ機能を備えています。

テストでは年間平均リターン約35%を示しています。流動性の高い暗号通貨ペアで最も効果を発揮します。

戦略はRSIのしきい値とローソク足の強さに基づいてロングまたはショートにエントリーし、オプションでトレンドとボリュームによるフィルタリングを行います。ポジションは設定可能なテイクプロフィット、ストップロス、トレーリングロジックで保護され、日次トレード制限は各セッション開始時にリセットされます。

## 詳細

- **エントリー条件**:
  - **買い**: RSIが30未満で強気のローソク足、感度に応じてオプションのトレンド/ボリュームフィルター。
  - **売り**: RSIが70超で弱気のローソク足、オプションのトレンド/ボリュームフィルター。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - テイクプロフィット、ストップロス、トレーリングストップ、またはRSI/EMAの急速反転。
- **ストップ**: あり。パーセンテージベースのSL/TPとオプションのトレーリングストップ。
- **フィルター**:
  - 設定に応じたトレンドおよびボリュームフィルター。
