# Arpit Bollinger Band戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Bollinger Bandのブレイクアウト戦略で、2本前のロウソク足がバンド外でクローズするのを待ち、現在の価格がその足の高値/安値を突破した時にエントリーします。

- **インジケーター**: Bollinger Bands (EMA 20、偏差 1.5)
- **エントリー**: 2本前のロウソク足が下部バンド以下でクローズし、現在の高値がその足の高値を超えた場合にロング。2本前のロウソク足が上部バンド以上でクローズし、現在の安値がその足の安値を下回った場合にショート。
- **ストップ**: 現在のロウソク足レンジを超えた位置に5%バッファを加えてストップを設定し、リスク‑リワード比に基づいてテイクプロフィットを設定。
