# EMAグリッド・マーチンゲール・クールダウン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

EMAベースのロングオンリーグリッドシステムを実装し、オプションのマーチンゲールサイジングとグリッド間のクールダウンを備えています。両方の短期EMAが長期EMAを上抜けた際に新しいグリッドが開始されます。追加の買いは固定ピップ間隔で配置され、ポジションは加重平均価格にバッファを加えた価格でクローズされます。
