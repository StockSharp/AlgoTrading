# Fibo Candles トレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、トレンド方向を判断するためにカスタムの **Fibo Candles** テクニックを使用します。
インジケーターは、現在の終値と直近の高値/安値レンジのフィボナッチ比率比較に基づいて、
各ローソク足を2色のうちの1色で塗ります。色の変化は潜在的なリバーサルを示します。
色が強気になると、戦略はショートポジションをクローズしてロングポジションをオープンします。
色が弱気になると、ロングポジションをクローズしてショートポジションをオープンします。

この手法は、ルックバック期間と選択可能なフィボナッチレベルを通じて市場のボラティリティに適応します。
絶対ポイントでのストップロスとテイクプロフィットがすべてのトレードを保護します。

## 詳細

- **エントリー条件**:
  - **ロング**: 現在のローソク足の色が弱気から強気に変化する。
  - **ショート**: 現在のローソク足の色が強気から弱気に変化する。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 反対の色が現れたとき、既存のポジションはクローズされる。
- **ストップ**: `StartProtection` によるポイント単位の固定ストップロスとテイクプロフィット。
- **デフォルト値**:
  - `Period` = 10（高値/安値レンジの測定に使用するローソク足数）。
  - `Fibo Level` = 0.236（トレンド判断に使用する比率）。
  - `Stop Loss` = 1000ポイント。
  - `Take Profit` = 2000ポイント。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Highest, Lowest
  - ストップ: はい
  - 複雑さ: 中
  - 時間軸: デフォルトは時間足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
