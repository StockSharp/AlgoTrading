# PriceChannel Stop
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Price Channel Stopインジケーターに基づいた戦略です。

インジケーターは指定された期間内の最高値と最安値を計算してドンチャンチャネルを形成します。ストップレベルは`Risk`ファクターを使用してチャネル内に構築されます。価格が上部ストップを上回って終値をつけると、トレンドは強気に切り替わります。下部ストップを下回って終値をつけると、トレンドは弱気に切り替わります。戦略はこれらのリバーサルでポジションをオープンし、オプションで逆方向のポジションをクローズします。

## 詳細

- **エントリー条件**: 価格がストップレベルを突破する。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 逆シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `ChannelPeriod` = 5
  - `Risk` = 0.10
  - `CandleType` = TimeSpan.FromHours(1)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: ドンチャンチャネル
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ (1h)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
