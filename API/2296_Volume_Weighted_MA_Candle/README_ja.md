# Volume Weighted MA ローソク足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ローソク足の始値と終値に対して出来高加重移動平均（VWMA）を構築します。これらのVWMAの相対的な位置がローソク足の「色」を定義します。

## トレードロジック
1. VWMA(始値)がVWMA(終値)を下回るとき、ローソク足は**強気**です。
2. VWMA(始値)がVWMA(終値)を上回るとき、ローソク足は**弱気**です。
3. 前のローソク足が強気で現在のものが中立または弱気になった場合、戦略はロングポジションを開き、ショートポジションをすべて閉じます。
4. 前のローソク足が弱気で現在のものが中立または強気になった場合、戦略はショートポジションを開き、ロングポジションをすべて閉じます。

## パラメーター
- `VWMA Period` – 両方の出来高加重移動平均を計算するための期間。
- `Candle Type` – 計算に使用するローソク足の時間軸。

デフォルトで保護ブロックが有効：テイクプロフィット2%、ストップロス1%。
