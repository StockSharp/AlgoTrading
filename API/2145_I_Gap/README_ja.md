# I-Gap戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**I-Gap戦略**はMetaTraderの「i-GAP」エキスパートアドバイザーを再現したものです。前のローソク足の終値と現在のローソク足の始値の間の価格ギャップを監視します。指定された価格ステップ数を超える下方向の始値ギャップは、ロングエントリーを発動し、必要に応じて既存のショートポジションを決済します。上方向のギャップはショートポジションに対して同様に機能します。

## 詳細
- **エントリー条件**: 連続するローソク足間の始値ギャップが設定サイズを超える。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対方向のギャップシグナル。
- **ストップ**: 固定ストップロスまたはテイクプロフィットなし。
- **デフォルト値**:
  - `CandleType` = 1 hour
  - `GapSize` = 5
  - `BuyPosOpen` = true
  - `SellPosOpen` = true
  - `BuyPosClose` = true
  - `SellPosClose` = true
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
