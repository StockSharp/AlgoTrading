# Spearman順位相関係数戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このペアトレーディング戦略は、2つの証券間のSpearman順位相関を測定します。相関が正の閾値を超えると、最初の証券をショート、2番目の証券をロングします。負の閾値を下回ると逆のポジションを取ります。相関がゼロに戻るとポジションを閉じます。

## 詳細

- **エントリー条件:**
  - **最初をロング / 2番目をショート**: 相関 < -Threshold。
  - **最初をショート / 2番目をロング**: 相関 > Threshold。
- **ロング/ショート**: ペアトレーディング。
- **エグジット条件:**
  - 相関の絶対値 < Threshold / 2。
- **ストップ**: いいえ。
- **デフォルト値:**
  - `CorrelationPeriod` = 10
  - `Threshold` = 0.8
- **フィルター:**
  - カテゴリ: 相関
  - 方向: 両方
  - インジケーター: Spearman Rank Correlation
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
