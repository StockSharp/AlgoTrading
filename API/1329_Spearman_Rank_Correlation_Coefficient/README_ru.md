# Стратегия Spearman Rank Correlation Coefficient
[English](README.md) | [中文](README_cn.md)

Стратегия парной торговли измеряет коэффициент ранговой корреляции Спирмена между двумя инструментами. При превышении положительного порога стратегия открывает короткую позицию по первому инструменту и длинную по второму. При падении ниже отрицательного порога позиции меняются местами. Закрытие происходит, когда корреляция возвращается к нулю.

## Детали

- **Критерии входа:**
  - **Long первый / Short второй**: корреляция < -Threshold.
  - **Short первый / Long второй**: корреляция > Threshold.
- **Long/Short**: Парная торговля.
- **Критерии выхода:**
  - Абсолютное значение корреляции < Threshold / 2.
- **Стопы**: Нет.
- **Значения по умолчанию:**
  - `CorrelationPeriod` = 10
  - `Threshold` = 0.8
- **Фильтры:**
  - Категория: Correlation
  - Направление: Both
  - Индикаторы: Spearman Rank Correlation
  - Стопы: No
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
