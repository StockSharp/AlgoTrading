# 月末強気戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
月末強気は、ポートフォリオマネージャーが保有を調整する際に株式が最後の数取引日に上昇することが多いという観察に基づいています。
ウィンドウドレッシングに伴う買い圧力が月次クローズ前に信頼性の高い上昇バイアスをもたらす可能性があります。

テストでは年平均リターン約 94% を示しています。株式市場で最も良いパフォーマンスを発揮します。

戦略は月の最後の数日に近いところで買い、新しい月の最初の取引日に決済してその傾向を捉えます。

予期しない弱さに備えて、直近のサポートの下にストップを置きます。

## 詳細

- **エントリー条件**: カレンダー効果トリガー
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: 季節性
  - 方向: 両方
  - インジケーター: 季節性
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
