# Стратегия Pin Bar Reversal
[English](README.md) | [中文](README_cn.md)

Использует свечи pin bar с фильтром тренда и целями по ATR. Бычий pin bar выше SMA открывает длинную позицию, медвежий ниже SMA — короткую. Вход пропускается при низкой волатильности.

## Детали

- **Условия входа**: pin bar по направлению тренда с длинной тенью, маленьким телом и ATR выше `MinAtr`.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: стоп‑лосс или тейк‑профит на основе ATR.
- **Стопы**: да, множители ATR.
- **Значения по умолчанию**:
  - `TrendLength` = 50
  - `MaxBodyPct` = 0.30
  - `MinWickPct` = 0.66
  - `AtrLength` = 14
  - `StopMultiplier` = 1
  - `TakeMultiplier` = 1.5
  - `MinAtr` = 0.0015
  - `CandleType` = 1 час
- **Фильтры**:
  - Категория: Pattern
  - Направление: Обе
  - Индикаторы: SMA, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
