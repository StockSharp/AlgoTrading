# Super Take戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はロングとショートのポジションを交互に切り替え、負け取引のたびにマーチンゲール乗数を使ってテイクプロフィットを増加させます。ストップロスは固定で、テイクプロフィットは勝ち取引の後に基本値にリセットされます。常に方向を切り替えて損失後の目標を調整することで、この戦略は以前のドローダウンを回復しようとします。

新しいポジションは、アクティブなポジションがない場合にのみ開かれます。最初の取引はデフォルトでロングです。それ以降の各取引は、最後に決済されたポジションの反対方向で開かれます。

## 詳細

- **エントリー条件**:
  - **ロング**: アクティブなポジションなし、かつ最後に決済されたポジションがショートまたは存在しない。
  - **ショート**: アクティブなポジションなし、かつ最後に決済されたポジションがロング。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 価格が動的テイクプロフィットまたは固定ストップロスに達したときにポジションを決済。
- **ストップ**: 固定ストップロス、負け取引後のマーチンゲールによる動的テイクプロフィット。
- **デフォルト値**:
  - `TakeProfit` = 10
  - `StopLoss` = 15
  - `MartinFactor` = 1.8
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: シンプル
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
