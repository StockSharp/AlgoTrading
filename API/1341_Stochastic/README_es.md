# Estrategia Stochastic
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Estrategia simple que utiliza el Oscilador Stochastic. Se abre una posición larga cuando %K cruza por encima del umbral de sobreventa. Se abre una posición corta cuando %K cruza por debajo del umbral de sobrecompra. Los niveles predeterminados son 50 para ambas direcciones.
