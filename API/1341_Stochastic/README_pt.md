# Estratégia Stochastic
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia simples usando o Oscilador Stochastic. Uma posição comprada é aberta quando %K cruza acima do limiar de sobrevenda. Uma posição vendida é aberta quando %K cruza abaixo do limiar de sobrecompra. Os níveis padrão são 50 para ambas as direções.
