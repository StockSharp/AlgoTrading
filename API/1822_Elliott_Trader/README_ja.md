# Elliott Trader戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

4時間足でストキャスティクス・オシレーターが極値に達したときに段階的なポジションを開く戦略です。最初の成行注文に続いてリミット注文のグリッドを設置します。利益目標に達し、移動平均線とボリンジャーバンドでトレンドが確認されるとポジションを決済します。

## エントリールール
- H4ローソク足でストキャスティクス・オシレーター（%K長さ21、平滑化3）を使用。
- %K ≥ **買われ過ぎ**レベルのとき:
  - 成行で売り。
  - 設定されたpips距離で現在価格より上に最大8本の追加`SellLimit`注文を設置。
- %K ≤ **売られ過ぎ**レベルのとき:
  - 成行で買い。
  - 設定されたpips距離で現在価格より下に最大8本の追加`BuyLimit`注文を設置。

## エグジットルール
- 実現利益が**ProfitTarget**に達し、価格がトレンドを確認したとき:
  - ロングポジションは価格が下部ボリンジャーバンドより上にあり、200期間SMAが55期間SMAより上にあるときに決済。
  - ショートポジションは価格が上部ボリンジャーバンドより下にあり、200期間SMAが55期間SMAより下にあるときに決済。
- %K ≥ 90かつ200期間SMA ≤ 55期間SMAのとき、保留中の買いリミット注文をキャンセル。
- %K ≤ 10かつ200期間SMA ≥ 55期間SMAのとき、保留中の売りリミット注文をキャンセル。

## パラメーター
- `StochLength` – ストキャスティクスの%K期間。
- `OverboughtLevel` – 売り始めるレベル。
- `OversoldLevel` – 買い始めるレベル。
- `ProfitTarget` – オープンポジションを決済するために必要な実現利益。
- `Order2Offset` … `Order9Offset` – 追加リミット注文のpips距離。
- `CandleType` – ローソク足の時間軸、デフォルト4時間。

## インジケーター
- StochasticOscillator
- BollingerBands
- SMA（200と55）
