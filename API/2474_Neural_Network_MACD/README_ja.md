# ニューラルネットワーク MACD 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、4 つの重みを持つシンプルなパーセプトロンフィルターと古典的な MACD クロスオーバーを組み合わせています。MACD とニューラルネットワークの両方が方向性に同意した場合にのみポジションを建てます。

## 仕組み

1. **パーセプトロンフィルター**  
   3 つのパーセプトロンが、現在の終値と過去の始値の差を使ってプライスモメンタムを評価します。各パーセプトロンには 4 つの整数重み（`X11`…`X34`）があり、`0` は影響なしを意味します。パーセプトロンの出力は価格差の加重和です。  
   `Pass` パラメーターに応じて、1 つ、2 つ、または 3 つすべてのパーセプトロンが意思決定に参加します。フィルターはストップロスとテイクプロフィットの距離（`Sl1`, `Tp1`, `Sl2`, `Tp2`）も定義します。
2. **MACD 確認**  
   標準的な MACD (12, 26, 9) が計算されます。MACD ラインがゼロ以下でシグナルラインを上抜けたときに買いシグナルが発生します。MACD ラインがゼロ以上でシグナルラインを下抜けたときに売りシグナルが発生します。
3. **トレード実行**  
   - MACD とパーセプトロンフィルターの両方が正の場合、ロングポジションを建てます。  
   - 両方が負の場合、ショートポジションを建てます。  
   ストップロスまたはテイクプロフィットレベルに到達するとポジションを決済します。

## パラメーター

| 名前 | 説明 |
| ---- | ---- |
| `X11…X34` | パーセプトロン入力の重み。 |
| `Tp1`, `Sl1` | 最初のパーセプトロンのテイクプロフィットとストップロス。 |
| `Tp2`, `Sl2` | 2 番目のパーセプトロンのテイクプロフィットとストップロス。 |
| `P1`, `P2`, `P3` | パーセプトロン入力の計算に使うバー単位のシフト。 |
| `Pass` | 使用するパーセプトロンの数 (1-3)。 |
| `CandleType` | 計算に使用するローソク足シリーズ。 |
