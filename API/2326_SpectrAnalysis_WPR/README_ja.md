# SpectrAnalysis WPR戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MQL5エキスパート *Exp_i-SpectrAnalysis_WPR* から変換されました。
Williams %Rインジケーターの方向を分析し、インジケーターの転換に従ってポジションをオープン/クローズします。

## ロジック

1. 選択した時間軸のローソク足を購読する。
2. 設定した期間でWilliams %Rを計算する。
3. 最新の2つのインジケーター値を保持して上向きまたは下向きの方向を検出する。
4. インジケーターが上向きに転換しロングエントリーが許可されている場合:
   - 有効な場合はショートポジションをクローズ。
   - 新しいロングポジションをオープン。
5. インジケーターが下向きに転換しショートエントリーが許可されている場合:
   - 有効な場合はロングポジションをクローズ。
   - 新しいショートポジションをオープン。

完成したローソク足のみが処理されます。戦略は複雑な履歴クエリを使用せず、高レベルAPIバインディングに依存します。

## パラメーター

| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| `Candle Type` | 計算に使用するローソク足の時間軸 | `4h` |
| `WPR Period` | Williams %Rインジケーターの期間 | `13` |
| `Allow Long Entry` | ロングポジションのオープンを許可 | `true` |
| `Allow Short Entry` | ショートポジションのオープンを許可 | `true` |
| `Allow Long Exit` | ロングポジションのクローズを許可 | `true` |
| `Allow Short Exit` | ショートポジションのクローズを許可 | `true` |

## 注意事項

元のMQLバージョンはWilliams %Rの出力にスペクトル分析を適用していました。
このC#変換では標準のWilliams %Rインジケーターを使用し、最近のインジケーター値を追跡することでシグナルロジックを再現しています。
