# Стратегия Stop Level Counter
[English](README.md) | [中文](README_cn.md)

Вспомогательная стратегия, оценивающая потенциальную прибыль для сделок на покупку и продажу по указанному уровню цены.
Подписывается на котировки Level1, вычисляет прибыль для обоих направлений на основе разницы между выбранным уровнем и текущими Bid/Ask и выводит результаты в журнал.
На графике рисуется горизонтальная линия, отображающая уровень.

## Подробности

- **Условия входа**: Нет
- **Long/Short**: Оба (прибыль оценивается для лонга и шорта)
- **Условия выхода**: Нет
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `Level` = 0 (инициализируется текущим Bid при старте)
  - `Volume` = 1
- **Фильтры**:
  - Категория: Utility
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
