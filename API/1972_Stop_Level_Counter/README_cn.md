# Stop Level Counter Strategy 策略 (中文)
[English](README.md) | [Русский](README_ru.md)

该工具型策略用于评估在用户设定的价格水平买入或卖出的潜在收益。
策略订阅 Level1 报价，根据所选价格与当前买价/卖价的差异计算多空方向的收益，并把结果记录在日志中。
图表上会绘制一条水平线表示该价格水平。

## 详情

- **入场条件**：无
- **多/空**：双向（同时估算多头和空头收益）
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `Level` = 0（启动时初始化为当前买价）
  - `Volume` = 1
- **筛选**：
  - 类别：Utility
  - 方向：双向
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险水平：低
