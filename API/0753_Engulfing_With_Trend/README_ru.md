# Стратегия поглощения по тренду
[English](README.md) | [中文](README_zh.md)

Стратегия сочетает фильтр SuperTrend с бычьими и медвежьими моделями поглощения. Сделка открывается, когда свеча поглощает предыдущую в направлении текущего тренда. Уровни стоп‑лосса и тейк‑профита рассчитываются по диапазону паттерна.

## Детали

- **Условия входа**: Поглощение в сторону направления SuperTrend.
- **Направление**: Лонг и шорт.
- **Условия выхода**: Стоп‑лосс или тейк‑профит.
- **Стопы**: Да, по экстремумам свечи и ATR‑смещению.
- **Значения по умолчанию**:
  - `CandleType` = 5 минут
  - `AtrPeriod` = 10
  - `AtrMultiplier` = 3
  - `BoringThreshold` = 25
  - `EngulfingThreshold` = 50
  - `StopLevel` = 200
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Лонг и шорт
  - Индикаторы: SuperTrend, свечной анализ
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
