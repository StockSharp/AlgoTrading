# Стратегия IMA Expert
[English](README.md) | [中文](README_cn.md)

Стратегия торгует по относительной скорости цены относительно её скользящей средней.
Показатель `Close / SMA - 1` сравнивается между двумя последними свечами. Сильный рост вызывает открытие длинной позиции, сильное падение — короткой.

## Детали

- **Условия входа**:
  - Long: `(IMA_now - IMA_prev) / abs(IMA_prev) >= SignalLevel`
  - Short: `(IMA_now - IMA_prev) / abs(IMA_prev) <= -SignalLevel`
- **Условия выхода**: Обратный сигнал
- **Размер позиции**: `RiskLevel` и `StopLossTicks` определяют объём сделки, ограниченный `MaxVolume`
- **Направление**: Оба
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `SmaPeriod` = 5
  - `TakeProfitTicks` = 50
  - `StopLossTicks` = 1000
  - `SignalLevel` = 0.5
  - `RiskLevel` = 0.01
  - `MaxVolume` = 1
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
