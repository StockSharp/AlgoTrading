# スムージング平均戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、追加のスムージングオフセットを伴う単純移動平均（SMA）を中心にトレードします。終値が平均からオフセット距離を超えてクロスしたときにポジションを開くことで、移動平均からの価格乖離を活用しようとします。

## 仕組み
- 選択したローソク足タイプのSMAを計算する。
- オープンポジションがない場合：
  - 終値が `SMA + Smoothing` を下回ったときにショートポジションを開く。
  - 終値が `SMA - Smoothing` を上回ったときにロングポジションを開く。
- ショートポジションがオープンの場合：
  - 終値が `SMA + Smoothing` を上回ったときにポジションをクローズする。
- ロングポジションがオープンの場合：
  - 終値が `SMA - Smoothing` を下回ったときにポジションをクローズする。

この戦略は成行注文を使用し、完成済みのローソク足のみで動作します。

## パラメーター
- **MA Period** – SMAのルックバック期間。
- **Smoothing** – シグナル生成時にSMAに加算または減算する価格オフセット。
- **Candle Type** – 計算に使用するローソク足の時間軸。

## 注意事項
この変換は、元のMQL4スクリプト `smoothingaverage.mq4` に基づいています。
