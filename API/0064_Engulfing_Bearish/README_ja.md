# 弱気の陰線包み足パターン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
このパターンはラリー後の弱気な動きの始まりを捉えることを目指します。弱気の陰線包み足は、赤いロウソクが直前の強気ボディを完全に飲み込んだときに発生します。パターン前にいくつかの連続した上昇バーを数えることで、市場が以前に上昇していたことを確認します。

テストでは年平均リターン約79%を示しています。株式市場で最もよく機能します。

アルゴリズムは各ロウソクを順に保存します。新しいバーが始値より低く引け、そのボディが直前の強気バーを包む場合、空売りが実行されます。ストップロスはリスクを制限するためパターンの高値の上に置かれます。

ポジションは通常、保護的なストップで管理されますが、状況が変わった場合はトレーダーが手動で決済することもできます。上昇トレンドを要求することで、乱高下する市場での偽シグナルを避けるのに役立ちます。

## 詳細

- **エントリー条件**: 弱気ロウソクが直前の強気バーを包む、オプションで上昇トレンド存在。
- **ロング/ショート**: ショートのみ。
- **エグジット条件**: ストップロスまたは裁量。
- **ストップ**: はい、パターンの高値の上。
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLossPercent` = 1
  - `RequireUptrend` = true
  - `UptrendBars` = 3
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ショート
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
