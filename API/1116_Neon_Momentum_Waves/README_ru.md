# Стратегия Neon Momentum Waves
[English](README.md) | [中文](README_cn.md)

Стратегия Neon Momentum Waves использует пересечения гистограммы MACD для торговли в обе стороны. Лонг открывается при пересечении гистограммой уровня входа сверху (по умолчанию ноль), шорт — при пересечении снизу. Позиции закрываются, когда гистограмма достигает заданных уровней выхода.

## Детали

- **Условия входа**: Гистограмма MACD пересекает уровень входа.
- **Направление**: Лонг и шорт.
- **Условия выхода**: Гистограмма пересекает уровни выхода для лонга/шорта.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `FastLength` = 12
  - `SlowLength` = 26
  - `SignalLength` = 20
  - `EntryLevel` = 0
  - `LongExitLevel` = 11
  - `ShortExitLevel` = -9
  - `CandleType` = 1 минута
- **Фильтры**:
  - Категория: Моментум
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
