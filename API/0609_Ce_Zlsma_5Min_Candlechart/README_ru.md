# Стратегия CE ZLSMA 5MIN Candlechart
[English](README.md) | [中文](README_cn.md)

Трендовая система на основе Zero Lag LSMA и свечей Heikin Ashi с фильтром Chandelier Exit. Покупает при смене направления вверх и закрытии свечи выше ZLSMA.

## Детали

- **Критерии входа**:
  - Лонг: направление сменяется вверх, закрытие Heikin Ashi выше ZLSMA и открытия
- **Лонг/Шорт**: Лонг
- **Критерии выхода**:
  - Лонг: закрытие ниже ZLSMA
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `ZlsmaLength` = 50
  - `AtrPeriod` = 1
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Трендовая
  - Направление: Лонг
  - Индикаторы: ZLSMA, ATR, Heikin Ashi
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
