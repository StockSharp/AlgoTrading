# RSI 30-70 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このシンプルなモメンタム戦略は、相対力指数（RSI）を使って売られすぎおよび買われすぎゾーンを特定します。RSI が売られすぎのレベルを下回るとロングポジションを建てます。RSI が買われすぎの閾値を上回るとトレードを決済します。このシステムは単一の時間軸で動作し、ショートトレードは行いません。

## 詳細

- **エントリー条件**:
  - **ロング**: `RSI < oversold`.
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - **ロング**: `RSI > overbought`.
- **ストップ**: なし。
- **デフォルト値**:
  - `RSI Length` = 14.
  - `Overbought/Oversold` = 70 / 30.
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: Long
  - インジケーター: 単一
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
