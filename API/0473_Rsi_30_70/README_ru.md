# Стратегия RSI 30-70
[English](README.md) | [中文](README_cn.md)

Эта простая стратегия моментума использует индекс относительной силы (RSI) для определения зон перепроданности и перекупленности. Когда RSI опускается ниже уровня перепроданности, открывается длинная позиция. Сделка закрывается, когда RSI поднимается выше уровня перекупленности. Система работает на одном таймфрейме и не открывает короткие позиции.

## Детали

- **Условия входа**:
  - **Long**: `RSI < oversold`.
- **Направление торговли**: только Long.
- **Условия выхода**:
  - **Long**: `RSI > overbought`.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `RSI Length` = 14.
  - `Overbought/Oversold` = 70 / 30.
- **Фильтры**:
  - Категория: Моментум
  - Направление: Long
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
