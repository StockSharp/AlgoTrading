# Eugene ローソク足パターン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

「Eugene」が説明するローソク足パターンを取引する戦略です。アルゴリズムは直近4本のローソク足を分析し、インサイドバーと特殊な「バード」フォーメーションを確認し、ブレイクアウトレベルを計算します。追加の確認条件が満たされた場合、前のローソク足の極値のブレイクアウトでポジションを開きます。オプションのストップロスとテイクプロフィットレベルは価格ステップで表されます。

## 詳細

- **エントリー条件**:
  - ロング: 現在の高値が前の高値を上回り、前の安値がさらに前の高値を下回り、現在の安値が前の安値を上回り、さらにジグレベルまたは時間フィルターによる確認。
  - ショート: 現在の安値が前の安値を下回り、前の高値がさらに前の安値を上回り、現在の高値が前の高値を下回り、さらにジグレベルまたは時間フィルターによる確認。
- **ロング/ショート**: 両方
- **エグジット条件**:
  - ロング: 反対シグナルが現れるかストップロス/テイクプロフィットに達したときに売る。
  - ショート: 反対シグナルが現れるかストップロス/テイクプロフィットに達したときに買う。
- **ストップ**: 価格ステップでの固定距離
- **デフォルト値**:
  - `Volume` = 1m
  - `StopLossPoints` = 0
  - `TakeProfitPoints` = 0
  - `InvertSignals` = false
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: なし
  - ストップ: オプション
  - 複雑さ: 中級
  - 時間軸: 短期
  - 季節性: イントラデイ (時間 >= 8 フィルター)
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
