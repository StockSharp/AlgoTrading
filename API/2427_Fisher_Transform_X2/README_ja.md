# Fisher Transform X2戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はFisher Transformインジケーターを2つの異なる時間軸で使用します。上位時間軸が全体的なトレンドを定義し、下位時間軸はFisherがそのトレンドに逆らって前の値をクロスする際にエントリーを生成します。オプションのパラメーターにより、トレンド変化時またはクロスシグナル時にポジションを閉じることができます。

## 詳細

- **エントリー条件**:
  - **ロング**: `トレンドFisherが上昇中` && `シグナルFisherが前の値を下方クロス`
  - **ショート**: `トレンドFisherが下降中` && `シグナルFisherが前の値を上方クロス`
- **ロング/ショート**: 両方
- **エグジット条件**:
  - オプションのトレンド反転時クローズ
  - オプションのシグナル時間軸での逆方向Fisherクロス時クローズ
- **ストップ**: ポイント単位のテイクプロフィットとストップロス
- **デフォルト値**:
  - `Trend Length` = 10
  - `Signal Length` = 10
  - `Trend Timeframe` = 6時間
  - `Signal Timeframe` = 30分
  - `Take Profit` = 2000ポイント
  - `Stop Loss` = 1000ポイント
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Fisher Transform
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: マルチ時間軸
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
