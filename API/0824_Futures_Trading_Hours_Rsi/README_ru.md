# Стратегия Futures Trading Hours RSI
[English](README.md) | [中文](README_cn.md)

Стратегия работает только в часы торгов американских фьючерсов (08:30–15:00 по центральному времени). Осциллятор RSI используется для входа в длинную позицию при пересечении уровня перепроданности снизу и для входа в короткую позицию при пересечении уровня перекупленности сверху. После 15:00 CT все открытые позиции закрываются.

## Детали

- **Критерии входа**:
  - **Длинная позиция**: RSI пересекает уровень перепроданности снизу в пределах сессии
  - **Короткая позиция**: RSI пересекает уровень перекупленности сверху в пределах сессии
- **Длинные/короткие**: обе стороны
- **Критерии выхода**:
  - Закрытие всех позиций в конце сессии (15:00 CT)
- **Стопы**: отсутствуют
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `OverSoldLevel` = 30
  - `OverBoughtLevel` = 70
  - `SessionStart` = 08:30
  - `SessionEnd` = 15:00
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
