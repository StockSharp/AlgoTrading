# 先物取引時間帯RSI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は米国先物セッション時間（08:30〜15:00 CT）のみで取引します。RSI（相対力指数）を使用し、オシレーターが売られすぎレベルを上抜けするとロングエントリー、買われすぎレベルを下抜けするとショートエントリーします。15:00 CT以降、すべての建玉が決済されます。

## 詳細

- **エントリー条件**:
  - **ロング**: セッション中にRSIが売られすぎレベルを上抜け
  - **ショート**: セッション中にRSIが買われすぎレベルを下抜け
- **ロング/ショート**: 両方向
- **エグジット条件**:
  - セッション終了時（15:00 CT）にすべてのポジションを決済
- **ストップ**: なし
- **デフォルト値**:
  - `RsiLength` = 14
  - `OverSoldLevel` = 30
  - `OverBoughtLevel` = 70
  - `SessionStart` = 08:30
  - `SessionEnd` = 15:00
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: RSI
  - Stops: いいえ
  - 複雑さ: 低
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
