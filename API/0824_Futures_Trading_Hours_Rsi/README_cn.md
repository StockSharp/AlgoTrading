# 期货交易时段 RSI 策略
[English](README.md) | [Русский](README_ru.md)

该策略仅在美国期货交易时段（中央时间 08:30–15:00）运行。使用 RSI 指标，当振荡器向上穿越超卖水平时做多，当向下穿越超买水平时做空。15:00 CT 后自动平掉所有持仓。

## 详情

- **入场条件**:
  - **多头**: RSI 在交易时段内上穿超卖水平
  - **空头**: RSI 在交易时段内下穿超买水平
- **多空方向**: 双向
- **退出条件**:
  - 交易时段结束（15:00 CT）时平仓
- **止损**: 无
- **默认值**:
  - `RsiLength` = 14
  - `OverSoldLevel` = 30
  - `OverBoughtLevel` = 70
  - `SessionStart` = 08:30
  - `SessionEnd` = 15:00
- **过滤器**:
  - 类型: 震荡指标
  - 方向: 双向
  - 指标: RSI
  - 止损: 无
  - 复杂度: 低
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
