# VWAP ブレイクアウト ATR戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

価格がVWAPを上抜けしたときにロング、下抜けしたときにショートでエントリーします。ストップロスとテイクプロフィットはATR乗数に基づきます。

## パラメーター

- **AtrLength**: ATR期間。
- **StopAtrMultiplier**: ストップロス用ATR乗数。
- **TakeAtrMultiplier**: テイクプロフィット用ATR乗数。
- **CandleType**: ローソク足の種類。
