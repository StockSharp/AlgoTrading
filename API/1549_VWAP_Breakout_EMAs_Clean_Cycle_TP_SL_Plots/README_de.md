# VWAP Ausbruch ATR-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Geht Long, wenn der Preis über den VWAP steigt, und Short, wenn er darunter fällt. Stop-Loss und Take-Profit basieren auf ATR-Vielfachen.

## Parameter

- **AtrLength**: ATR-Zeitraum.
- **StopAtrMultiplier**: ATR-Multiplikator für den Stop-Loss.
- **TakeAtrMultiplier**: ATR-Multiplikator für den Take-Profit.
- **CandleType**: Kerzentyp.
