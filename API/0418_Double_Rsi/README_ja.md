# Double RSI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Double RSI は、2つのRelative Strength Index計算を使用します：1つはトレードチャート上、
もう1つは上位時間軸上です。両方のRSI読みが同じ方向を支持する場合のみ取引が行われ、
短期エントリーを長期のモメンタムと合わせます。

メインの時間軸は、RSIが買われすぎゾーンや売られすぎゾーンからクロスするのを探します。
上位時間軸のRSIがその動きを確認すると、戦略はポジションを開きます。オプションの
利確設定で、事前に定義された動き後に利益を確定することができます。

## 詳細
- **データ**: 2つの時間軸の価格ローソク足。
- **エントリー条件**:
  - **ロング**: 下位時間軸のRSIが売られすぎゾーンを抜け、かつ上位時間軸のRSIが強気。
  - **ショート**: 下位時間軸のRSIが買われすぎゾーンを抜け、かつ上位時間軸のRSIが弱気。
- **エグジット条件**: 反対方向のRSIシグナル、または `UseTP` が真の場合は利確。
- **ストップ**: デフォルトではなし。
- **デフォルト値**:
  - `CandleType` = tf(5)
  - `RSILength` = 14
  - `MTFTimeframe` = tf(15)
  - `UseTP` = False
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: ロング/ショート
  - インジケーター: RSI (multi‑timeframe)
  - 複雑さ: 中程度
  - リスクレベル: 中
