# デリバティブ・ゼロクロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は価格デリバティブの符号変化に基づいて取引します。デリバティブは価格モメンタムを期間で割り100を掛けて計算されます。デリバティブがゼロラインを越えると、現在のポジションがクローズされ、反対方向のポジションがオープンされます。

## パラメーター

- `DerivativePeriod` - デリバティブ計算のスムージング期間。
- `PriceType` - デリバティブに使用するソース価格。
- `BuyEntry` - ロングポジションの建玉を許可。
- `SellEntry` - ショートポジションの建玉を許可。
- `BuyExit` - ロングポジションのクローズを許可。
- `SellExit` - ショートポジションのクローズを許可。
- `StopLoss` - ストップロス（ポイント単位）。
- `TakeProfit` - テイクプロフィット（ポイント単位）。
- `CandleType` - ローソク足の時間軸。

## ロジック

1. ローソク足を購読し、選択した価格のモメンタムを計算する。
2. モメンタムを期間で割り100を掛けてデリバティブを算出する。
3. デリバティブが正から非正に変わると、ロングポジションをオープンしショートポジションをクローズする。
4. デリバティブが負から非負に変わると、ショートポジションをオープンしロングポジションをクローズする。
5. リスク管理のためストップロスとテイクプロフィットを適用する。
