# Bitcoin 指数的利益戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は高速EMAが低速EMAを上抜けるとロングに入ります。ポジションサイズは口座資産のリスクパーセンテージから計算されます。エグジットはEMAの下抜け、ストップロス、テイクプロフィット、またはトレーリングストップで発生します。

## 詳細

- **エントリー条件**:
  - 高速EMAが低速EMAを上抜け → ロング。
- **ロング/ショート**: ロングのみ
- **エグジット条件**:
  - 高速EMAが低速EMAを下抜け。
  - リスクパーセンテージでのストップロス。
  - テイクプロフィット = リスク × 報酬倍率。
  - 最高値からのトレーリングストップ。
- **ストップ**: SL, TP, トレーリングストップ
- **デフォルト値**:
  - 高速EMA期間 = 9
  - 低速EMA期間 = 21
  - リスクパーセント = 1
  - 報酬倍率 = 2
  - トレーリングストップオフセット率 = 0.5
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: ロングのみ
  - インジケーター: EMA
  - ストップ: SL & TP & Trailing
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
