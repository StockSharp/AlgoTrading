# Golden Ratio Cubes戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Golden Ratio Cubes戦略はフィボナッチ数学を使ってブレイクアウトを検出します。参照
ウィンドウ内の最高値と最安値を追跡し、黄金比（φ ≈ 1.618）に基づく拡張を計算します。
価格がこれらの拡張を超えて終値を付けると、戦略はブレイクアウトの方向にエントリーします。

## 詳細

- **エントリー条件**:
  - 最近のレンジの黄金比拡張を上回る終値 → 買い。
  - 最近のレンジの黄金比拡張を下回る終値 → 売り。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 反対のブレイクアウトシグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `Lookback` = 34
  - `Phi` = 1.618
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ロング & ショート
  - インジケーター: Highest、Lowest
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
