# Стратегия Yesterday's High
[English](README.md) | [中文](README_cn.md)

Лонг-стратегия пробоя, которая размещает buy stop выше максимума предыдущего дня.
Дополнительные фильтры ROC, трейлинг-стоп и закрытие по EMA улучшают контроль риска.

## Детали

- **Условия входа**: Закрытие ниже вчерашнего максимума, затем buy stop на уровне high + gap
- **Лонг/Шорт**: Только лонг
- **Условия выхода**: Стоп-лосс, тейк-профит, опциональный трейлинг-стоп или пересечение EMA
- **Стопы**: Да, в процентах
- **Значения по умолчанию**:
  - `Gap` = 1
  - `StopLoss` = 3
  - `TakeProfit` = 9
  - `UseRocFilter` = false
  - `RocThreshold` = 1
  - `UseTrailing` = true
  - `TrailEnter` = 2
  - `TrailOffset` = 1
  - `CloseOnEma` = false
  - `EmaLength` = 10
  - `CandleType` = 1 минута
- **Фильтры**:
  - Категория: Пробой
  - Направление: Лонг
  - Индикаторы: Цена, ROC, EMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
