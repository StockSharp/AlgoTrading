# Hedge Average戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTraderの「Hedge Average」エキスパートを再現します。2つの期間にわたる始値と終値の単純移動平均を比較します。

## トレードロジック

- `Period1` と `Period2` の始値と終値のSMAを計算します。
- 長期間の始値平均がその終値平均を上回り、**かつ**短期間の始値平均がその終値平均を下回る場合、ロングポジションを開きます。
- 長期間の始値平均がその終値平均を下回り、**かつ**短期間の始値平均がその終値平均を上回る場合、ショートポジションを開きます。
- 取引は `StartHour` から `EndHour` の間のみ許可されます。
- オプションのストップロスとテイクプロフィットは絶対価格単位で設定されます。トレーリングストップを有効にすると、保護ストップが価格とともに移動します。

## パラメーター

- `Period1` – 高速平均の期間。
- `Period2` – 低速平均の期間。
- `StartHour` – 取引が開始される時刻。
- `EndHour` – 取引が停止される時刻。
- `CandleType` – 計算に使用するローソク足の時間軸。
- `TakeProfit` – 価格単位でのテイクプロフィット距離。
- `StopLoss` – 価格単位でのストップロス距離。
- `UseTrailing` – ストップロス距離に基づくトレーリングストップを有効化。

## 注記

この戦略は単一ポジションアプローチを使用し、元のMQLバージョンの金額ベースの利益目標は再現しません。
