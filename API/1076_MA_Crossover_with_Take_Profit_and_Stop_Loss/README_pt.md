# Estratégia de Cruzamento de MA com Take Profit e Stop Loss
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia entra comprada quando uma SMA rápida cruza acima de uma SMA lenta. Os níveis de take profit e stop loss são baseados em percentuais e se movem quando o preço sobe acima do preço de entrada.

## Parâmetros
- Fast MA Length
- Slow MA Length
- Take Profit %
- Stop Loss %
- Dynamic Take Profit %
- Dynamic Stop Loss %
- Candle Type
