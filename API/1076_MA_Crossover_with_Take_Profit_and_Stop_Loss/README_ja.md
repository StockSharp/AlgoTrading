# テイクプロフィットとストップロス付き MA クロスオーバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

速いSMAが遅いSMAを上抜けたときにロングポジションをエントリーします。テイクプロフィットとストップロスのレベルはパーセンテージベースで、価格がエントリー価格を上回ると移動します。

## パラメーター
- Fast MA Length
- Slow MA Length
- Take Profit %
- Stop Loss %
- Dynamic Take Profit %
- Dynamic Stop Loss %
- Candle Type
