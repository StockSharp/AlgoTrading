# MA-Crossover-Strategie mit Take Profit und Stop Loss
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie geht Long, wenn ein schneller SMA einen langsamen SMA von unten kreuzt. Take-Profit- und Stop-Loss-Niveaus sind prozentbasiert und verschieben sich, wenn der Preis über den Einstiegspreis steigt.

## Parameter
- Fast MA Length
- Slow MA Length
- Take Profit %
- Stop Loss %
- Dynamic Take Profit %
- Dynamic Stop Loss %
- Candle Type
