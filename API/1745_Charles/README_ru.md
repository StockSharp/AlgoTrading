# Стратегия Charles EMA RSI
[English](README.md) | [中文](README_cn.md)

Эта стратегия повторяет идею советника Charles, комбинируя экспоненциальные скользящие средние (EMA) с фильтром RSI и трейлинг-стопом. Она торгует в обе стороны и динамически защищает позиции.

Система следит за быстрой и медленной EMA на выбранном таймфрейме. Когда быстрая EMA пересекает медленную снизу вверх и RSI превышает 55, стратегия открывает длинную позицию. Когда быстрая EMA пересекает медленную сверху вниз и RSI опускается ниже 45, стратегия открывает короткую позицию. После входа используется трейлинг-стоп, а фиксированные тейк-профит и стоп-лосс управляются встроенной защитой позиции.

## Детали

- **Критерии входа**:
  - **Длинная позиция**: `быстрая EMA` пересекает `медленную EMA` снизу вверх и `RSI > 55`.
  - **Короткая позиция**: `быстрая EMA` пересекает `медленную EMA` сверху вниз и `RSI < 45`.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - трейлинг-стоп;
  - стоп-лосс или тейк-профит.
- **Стопы**: используется встроенная защита с трейлингом.
- **Значения по умолчанию**:
  - `FastPeriod` = 18
  - `SlowPeriod` = 60
  - `RsiPeriod` = 14
  - `TakeProfit` = 0.02
  - `StopLoss` = 0.008
  - `TrailStart` = 0.006
  - `TrailOffset` = 0.003
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Несколько
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: 1 час по умолчанию
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
