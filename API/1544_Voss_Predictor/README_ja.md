# Voss Predictor戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はジョン・エーラーズのVoss予測フィルターをバンドパスフィルターと組み合わせて、価格の動きを事前に見越します。予測フィルターがバンドパス出力を上回るとロングポジションを建て、下回るとショートポジションを建てます。

## 詳細

- **エントリー**: Voss予測フィルターがバンドパスフィルターを上抜けする。
- **エグジット**: Voss予測フィルターがバンドパスフィルターを下抜けする。
- **タイプ**: トレンドフォロー。
- **ストップ**: なし。
