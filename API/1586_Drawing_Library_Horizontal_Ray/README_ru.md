# Стратегия Horizontal Ray
[English](README.md) | [中文](README_cn.md)

Рисует горизонтальные лучи в точках пересечения SMA и торгует по направлению кросса.

## Детали

- **Условия входа**: пересечение `SMA20` и `SMA50` вверх — лонг, вниз — шорт.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: противоположное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `FastLength` = 20
  - `SlowLength` = 50
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Drawing
  - Направление: Обе
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Новичок
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Низкий
