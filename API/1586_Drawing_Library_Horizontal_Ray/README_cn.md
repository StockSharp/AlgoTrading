# Horizontal Ray 策略
[English](README.md) | [Русский](README_ru.md)

在 SMA 交叉点绘制水平射线，并按交叉方向交易。

## 细节

- **入场条件**：`SMA20` 上穿 `SMA50` 做多，下穿做空。
- **多空**：双向。
- **出场条件**：相反交叉。
- **止损**：否。
- **默认值**：
  - `FastLength` = 20
  - `SlowLength` = 50
  - `CandleType` = 5 分钟
- **过滤器**：
  - 类别: Drawing
  - 方向: 双向
  - 指标: SMA
  - 止损: 否
  - 复杂度: 初级
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
