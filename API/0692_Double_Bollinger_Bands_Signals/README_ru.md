# Стратегия Double Bollinger Bands Signals
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует две группы полос Боллинджера. Покупка происходит, когда цена пересекает нижнюю полосу с отклонением 3 сигмы, а продажа — когда цена пересекает верхнюю полосу с отклонением 3 сигмы. Позиции закрываются на противоположных полосах с отклонением 2 сигмы.

## Детали

- **Условия входа**:
  - Long: закрытие пересекает снизу вверх нижнюю полосу 3 SD
  - Short: закрытие пересекает сверху вниз верхнюю полосу 3 SD
- **Направление**: Оба
- **Условия выхода**:
  - Long: закрытие пересекает сверху вверх верхнюю полосу 2 SD
  - Short: закрытие пересекает снизу вниз нижнюю полосу 2 SD
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Length` = 20
  - `Width1` = 2m
  - `Width2` = 3m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: Оба
  - Индикаторы: Bollinger Bands
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
