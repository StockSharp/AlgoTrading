# Separated Moving Average
[English](README.md) | [中文](README_cn.md)

Стратегия строит отдельные скользящие средние для бычьих и медвежьих закрытий. Лонг открывается, когда бычья средняя поднимается выше медвежьей, а шорт — при обратном пересечении. Поддерживает SMA, EMA или HMA и может работать по ценам Heikin Ashi.

## Детали

- **Условия входа**: пересечение бычьей средней выше медвежьей.
- **Длинные/короткие**: обе стороны.
- **Условия выхода**: обратное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `MaType` = MaType.SMA
  - `Length` = 20
  - `UseHeikinAshi` = true
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: SMA, EMA, HMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
