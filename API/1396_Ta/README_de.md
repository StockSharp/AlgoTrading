# Ta-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Strategie basierend auf MACD-Crossover mit Unterstützungs- und Widerstandspivots, RSI- und ADX-Bestätigung. Zwei Gewinnziele mit Teilausstieg werden verwendet.

## Details

- **Einstieg**
  - **Long**: MACD kreuzt über Signal, Kurs über Widerstand, RSI > 50, +DI > -DI, ADX > 20.
  - **Short**: MACD kreuzt unter Signal, Kurs unter Unterstützung, RSI < 50, -DI > +DI, ADX > 20.
- **Ausstieg**: zwei Take-Profit-Niveaus und ein Stop-Loss.
