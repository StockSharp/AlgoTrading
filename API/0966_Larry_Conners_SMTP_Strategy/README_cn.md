# Larry Conners SMTP 策略
[English](README.md) | [Русский](README_ru.md)

该策略在出现10根K线最低点且当前K线拥有过去10根中最大的波动区间并且收盘价处于区间上部25%时做多。买入止损单下达在最高价上方一个最小跳动值，止损沿新的低点上移。

## 细节

- **入场条件**：
  - **多头**：当前低点等于过去10根K线的最低点，今日区间为最近10根中最大，且收盘处于区间上部25%；在`High + TickSize`处放置买入止损单。
- **多空方向**：仅做多。
- **出场条件**：
  - 止损跟随自入场以来的最高最低价。
- **止损**：有。
- **默认值**：
  - `TickSize` = 0.01
  - `CandleType` = TimeSpan.FromDays(1).TimeFrame().
- **筛选**：
  - 类型: 反转
  - 方向: 多头
  - 指标: Highest, Lowest
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 日线
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
