# Turn of the Month Honestcowboy 策略
[English](README.md) | [Русский](README_ru.md)

一种基于日历的策略，在每月末附近建立多头，并在新月开始后的几天内平仓，同时考虑周末影响。

## 详情

- **入场条件**：接近月末的日期并调整周末
- **多空方向**：多头
- **出场条件**：月初几天后
- **止损**：无
- **默认参数**：
  - `DaysBeforeEnd` = 2
  - `DaysAfterStart` = 3
- **过滤器**：
  - 分类：季节性
  - 方向：多头
  - 指标：无
  - 止损：否
  - 复杂度：初级
  - 时间框架：日线
  - 季节性：是
  - 神经网络：否
  - 背离：否
  - 风险等级：低
