# Estratégia Vortex MTF
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia calcula o indicador Vortex em um período configurável.
Posições compradas são abertas quando a linha Vortex positiva cruza acima da negativa.
Posições vendidas são abertas no cruzamento oposto.

## Detalhes

- **Indicadores**: Vortex
- **Direção**: Comprado e vendido
- **Período**: Configurável
