# Vortex MTF-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie berechnet den Vortex-Indikator auf einem konfigurierbaren Zeitrahmen.
Long-Positionen werden eröffnet, wenn die positive Vortex-Linie die negative nach oben kreuzt.
Short-Positionen werden beim umgekehrten Kreuzungssignal eröffnet.

## Details

- **Indikatoren**: Vortex
- **Richtung**: Long und Short
- **Zeitrahmen**: Konfigurierbar
