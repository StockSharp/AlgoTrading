# Estrategia Vortex MTF
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia calcula el indicador Vortex en un marco temporal configurable.
Las posiciones largas se abren cuando la línea Vortex positiva cruza por encima de la negativa.
Las posiciones cortas se abren en el cruce opuesto.

## Detalles

- **Indicadores**: Vortex
- **Dirección**: Largo y corto
- **Marco temporal**: Configurable
