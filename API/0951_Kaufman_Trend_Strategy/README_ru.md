# Стратегия Kaufman Trend
[English](README.md) | [中文](README_cn.md)

**Kaufman Trend Strategy** использует фильтр Калмана для оценки цены и её скорости. Сила тренда вычисляется по скорости и нормализуется на основе последних значений. Вход осуществляется, когда сила тренда превышает порог и цена находится выше или ниже фильтрованного значения. Стопы строятся по последним экстремумам с поправкой на ATR, а фиксация прибыли происходит по этапам при ослаблении импульса.

## Детали
- **Условия входа**: сила тренда и положение цены относительно фильтра.
- **Лонг/Шорт**: оба направления.
- **Условия выхода**: поэтапное закрытие и ослабление тренда или срабатывание стопа.
- **Стопы**: да, минимум/максимум свечей ± ATR.
- **Значения по умолчанию**:
  - `TakeProfit1Percent = 50`
  - `TakeProfit2Percent = 25`
  - `TakeProfit3Percent = 25`
  - `SwingLookback = 10`
  - `AtrPeriod = 14`
  - `TrendStrengthEntry = 60`
  - `TrendStrengthExit = 40`
  - `CandleType = TimeSpan.FromMinutes(15).TimeFrame()`
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Калман
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (15м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
