# ADX Expert 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
**ADX Expert 戦略**は、元の MetaTrader 4 エキスパートアドバイザー「ADX Expert」（MQL スクリプト 20315）の直接変換です。このエキスパートは、Average Directional Index（ADX）が指定された閾値を下回っている間、正の Directional Index（+DI）と負の Directional Index（-DI）のラインのクロスを探し、市場がレンジ相場であることを示します。ソースエキスパートと同様に、一度に開けるポジションは 1 つだけです。

## トレードロジック
1. 戦略は選択されたローソク足シリーズ（デフォルトは 15 分ローソク足）を購読し、設定された期間で Average Directional Index を計算します。
2. 以下の場合に買い注文が出されます：
   - +DI ラインが -DI ラインを上回るようにクロスする。
   - ADX 値が定義された閾値（デフォルト 20）を下回り、弱いトレンドを示している。
   - 現在のスプレッドが `MaxSpreadPoints` フィルターを下回っている。
   - 現在ポジションがない。
3. 以下の場合に売り注文が出されます：
   - +DI ラインが -DI ラインを下回るようにクロスする。
   - ADX 値がまだ許容閾値より低い。
   - スプレッド要件とフラットポジション条件が満たされている。
4. 保護的なストップロスとテイクプロフィットレベルは `StartProtection` を通じて割り当てられ、MQL バージョンの固定ストップとターゲットを反映しています。価格ポイント（価格ステップ）で表現され、値をゼロに設定することで無効にできます。

この戦略は単一ポジションのワークフローに基づいています：現在のポジションがその保護注文によって閉じられるまで、新しいシグナルは無視されます。

## パラメーター
| パラメーター | 説明 | デフォルト |
| --- | --- | --- |
| `TradeVolume` | すべての成行注文に使用される注文サイズ。 | 0.1 |
| `AdxPeriod` | ADX 計算の期間。 | 14 |
| `AdxThreshold` | まだトレードを許可する最大 ADX 値。 | 20 |
| `MaxSpreadPoints` | 価格ポイントでの最大許容スプレッド。フィルターを無効にするには 0 に設定。 | 20 |
| `StopLossPoints` | 価格ポイントでのストップロス距離。 | 200 |
| `TakeProfitPoints` | 価格ポイントでのテイクプロフィット距離。 | 400 |
| `CandleType` | 指標計算のためのローソク足タイプ（デフォルトは 15 分ローソク足）。 | 15 分タイムフレーム |
