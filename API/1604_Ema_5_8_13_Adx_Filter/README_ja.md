# EMA 5-8-13 ADXフィルター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は5期間と8期間のEMAクロスをトレードし、13期間EMAを決済に使用します。オプションのADXフィルターでトレンドの強さを確認します。EMA5がEMA8を上抜けしADXが閾値を超えたときにロングポジションを建て、逆シグナルでショートポジションを建てます。

## 詳細

- **エントリー条件**:
  - **ロング**: EMA5がEMA8を上抜け かつ ADX > 閾値。
  - **ショート**: EMA5がEMA8を下抜け かつ ADX > 閾値。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: 終値 < EMA13
  - **ショート**: 終値 > EMA13
- **ストップ**: なし。
- **デフォルト値**:
  - `ADX threshold` = 20
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: なし
  - 複雑さ: シンプル
  - 時間軸: 短期
