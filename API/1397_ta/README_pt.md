# Estratégia de Biblioteca Ta
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Calcula a taxa de crescimento anual composta (CAGR) entre velas consecutivas e opera na direção da taxa de crescimento.

## Detalhes

- **Entrada**: comprar quando CAGR > 0, vender quando CAGR < 0.
- **Saída**: sinal oposto.
