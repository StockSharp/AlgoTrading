# Ta-Bibliothek-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Berechnet die zusammengesetzte jährliche Wachstumsrate (CAGR) zwischen aufeinanderfolgenden Kerzen und handelt in Richtung der Wachstumsrate.

## Details

- **Einstieg**: kaufen wenn CAGR > 0, verkaufen wenn CAGR < 0.
- **Ausstieg**: entgegengesetztes Signal.
