# Estrategia de Biblioteca Ta
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Calcula la tasa de crecimiento anual compuesta (CAGR) entre velas consecutivas y opera en la dirección de la tasa de crecimiento.

## Detalles

- **Entrada**: comprar cuando CAGR > 0, vender cuando CAGR < 0.
- **Salida**: señal opuesta.
