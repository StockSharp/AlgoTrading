# SupportResistTrade 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MetaTraderから移植されたブレイクアウト戦略で、長期EMAトレンドフィルターと動的なサポート・レジスタンスレベルを組み合わせています。最近のスイング範囲を振り返り、価格がトレンド方向に前の天井または床を突破するのを待ち、段階的なpipベースのトレーリングストップでポジションを管理します。

## 詳細

- **エントリー条件**: 価格が前の `Lookback` 期間の高値（ロング）または安値（ショート）を超えて終値を付け、バーがEMA `MaPeriod` より上/下で始まる
- **ロング/ショート**: 両方
- **エグジット条件**: トレーリングストップが発動するか、利益のあるポジションが更新されたサポート/レジスタンスバンドを逆方向に越える
- **ストップ**: 反対側のバンドで初期ストップ、+20/+40/+60 pipの動きの後にトレール（それぞれ10/20/30 pipを確保）
- **デフォルト値**:
  - `Lookback` = 55
  - `MaPeriod` = 500
  - `CandleType` = 1分
  - `OrderVolume` = 0.1
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: EMA, Highest, Lowest
  - ストップ: トレーリング
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
