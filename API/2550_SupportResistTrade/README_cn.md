# Support Resist Trade 策略
[English](README.md) | [Русский](README_ru.md)

从 MetaTrader 移植的突破策略，结合长周期 EMA 趋势过滤和动态支撑/阻力区间。系统回看最近 `Lookback` 根 K 线，等待价格顺着趋势突破前高或前低，并使用分阶段的点值追踪止损管理仓位。

## 细节

- **入场条件**: K 线收盘突破 `Lookback` 区间最高点（做多）或最低点（做空），且当前开盘价位于 EMA `MaPeriod` 上方/下方
- **多空方向**: 双向
- **出场条件**: 追踪止损被触发，或盈利仓位重新穿越更新后的支撑/阻力水平
- **止损**: 初始止损放在相反的区间边界，价格获利 +20/+40/+60 点时依次上调（锁定 10/20/30 点）
- **默认值**:
  - `Lookback` = 55
  - `MaPeriod` = 500
  - `CandleType` = 1 分钟
  - `OrderVolume` = 0.1
- **过滤器**:
  - 分类: Breakout
  - 方向: 双向
  - 指标: EMA, Highest, Lowest
  - 止损: 追踪
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
