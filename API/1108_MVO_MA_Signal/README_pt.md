# MVO - Estratégia de Sinal MA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia baseada em Heikin Ashi que gera sinais a partir de cruzamentos de médias móveis ou valores extremos do Money Flow Index.
Emprega gestão de stop loss e alvo de lucro baseados em ATR, com break-even e stop trailing opcionais.
