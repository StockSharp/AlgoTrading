# MVO - Estrategia de Señal MA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Estrategia basada en Heikin Ashi que genera señales a partir de cruces de medias móviles o valores extremos del Money Flow Index.
Emplea gestión de stop loss y toma de ganancias basados en ATR, con break-even y stop trailing opcionales.
