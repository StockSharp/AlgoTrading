# MVO - MA-Signal-Strategie

Heikin-Ashi-basierte Strategie, die Signale aus gleitenden Durchschnittskreuzungen oder extremen Money-Flow-Index-Werten generiert.
Verwendet ATR-basiertes Stop-Loss- und Take-Profit-Management mit optionalem Break-Even und Trailing-Stop.
