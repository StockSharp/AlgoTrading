# Стратегия NY ORB CP
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя диапазона открытия Нью-Йорка с подтверждением отката. Торгует пробои диапазона 9:30-9:45 по NY после ретеста и подтверждения тренда и объёма.

## Детали

- **Условия входа**:
  - Лонг: цена возвращается к максимуму диапазона после пробоя при подтверждении трендом и объёмом.
  - Шорт: цена возвращается к минимуму диапазона после пробоя вниз при подтверждении трендом и объёмом.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Тейк-профит на 0.33 диапазона * `RiskReward`.
  - Стоп-лосс на 0.33 диапазона.
- **Стопы**: Да, динамические.
- **Значения по умолчанию**:
  - `MinRangePoints` = 60
  - `RiskReward` = 3
  - `MaxTradesPerSession` = 3
  - `MaxDailyLoss` = -1000
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: EMA, VWAP, SMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
