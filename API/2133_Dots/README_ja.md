# ドット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MQL5 "Exp_Dots" から変換されました。Dots インジケーターが色を変えたときにリバーサルを取引する戦略です。
インジケーターが青から赤に変わったときにロング、赤から青に変わったときにショートになります。

## 詳細

- **エントリー条件**:
  - ロング: インジケーターの色が青から赤に変わる。
  - ショート: インジケーターの色が赤から青に変わる。
- **ロング/ショート**: 両方
- **エグジット条件**: 反対シグナル
- **ストップ**: なし
- **デフォルト値**:
  - `Length` = 10
  - `Filter` = 0m
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンドリバーサル
  - 方向: 両方
  - インジケーター: Dots (NonLag Moving Average)
  - ストップ: なし
  - 複雑さ: 中級
  - 時間軸: 4H
  - 季節性: なし
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
