# ScalpWiz Bollinger戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

**ScalpWiz Bollinger戦略**は、ボリンジャーバンドを使用して価格の行き過ぎを検出する逆張りシステムです。終値が上限バンドまたは下限バンドから大きく離れたとき、戦略は反対方向にポジションを建てて反転を期待します。

4つの距離レベルがチェックされます。各レベルは異なるシグナル強度に対応し、取引量を乗算します。ポジションサイズも現在のポートフォリオ価値に対するリスクパーセンテージによってスケーリングされます。

## パラメーター

- `BandsPeriod` – ボリンジャーバンドの計算に使用するローソク足の数。
- `BandsDeviation` – バンドの標準偏差乗数。
- `Level1Pips` … `Level4Pips` – レベル1〜4のシグナルを起動するバンドからのpips距離。
- `StrengthLevel1Multiplier` … `StrengthLevel4Multiplier` – 各レベルのボリューム乗数。
- `RiskPercent` – シグナルごとのポートフォリオ価値のリスクパーセンテージ。
- `CandleType` – 計算に使用するローソク足の時間軸。

## 取引ロジック

1. 選択した時間軸のローソク足を購読し、ボリンジャーバンドを計算します。
2. 各確定ローソク足で：
   - 終値が設定されたレベル距離分だけ上限バンドを上回っている場合、ショートポジションを建てます。
   - 終値が設定されたレベル距離分だけ下限バンドを下回っている場合、ロングポジションを建てます。
3. ボリュームはリスクパーセンテージとシグナル強度乗数から計算されます。

この戦略はオリジナルのMQLスクリプト`mcb.scalpwiz.9001.mq4`にインスパイアされています。
