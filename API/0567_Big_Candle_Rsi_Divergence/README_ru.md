# Большая свеча и RSI дивергенция
[English](README.md) | [中文](README_cn.md)

Стратегия ищет свечи с телом больше предыдущих пяти и сравнивает быстрый и медленный RSI. Сделки открываются в сторону свечи. Защита реализована отложенным трейлинг-стопом, который активируется только после движения цены на заданное количество тиков в прибыль.

Трейлинг-стоп начинает сопровождать цену после достижения порога прибыли, а фиксированный стоп устанавливается сразу после входа.

## Детали

- **Условия входа**:
  - **Long**: Текущее тело больше пяти предыдущих и свеча закрывается вверх.
  - **Short**: Текущее тело больше пяти предыдущих и свеча закрывается вниз.
- **Long/Short**: Оба направления.
- **Условия выхода**: Срабатывание начального стопа или трейлинг-стопа.
- **Стопы**: Да, отложенный трейлинг-стоп.
- **Значения по умолчанию**:
  - `TrailStartTicks` = 200
  - `TrailDistanceTicks` = 150
  - `InitialStopLossTicks` = 200
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Риск: Средний
