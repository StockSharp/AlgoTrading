# 大陽線・大陰線RSIダイバージェンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

直近5本のバーと比較して異常に大きな陽線・陰線を識別し、高速・低速RSIの値を比較します。取引はローソク足の方向に従い、価格が設定したティック数だけ利益方向に動いた後にのみ発動する遅延トレーリングストップを使用します。

トレーリングストップは利益しきい値に達した時点で開始し、その後一定の距離で価格を追跡します。一方、初期固定ストップはエントリーから取引を保護します。

## 詳細

- **エントリー条件**:
  - **ロング**: 現在のローソク足の実体が直近5本より大きく、上昇して終値を形成。
  - **ショート**: 現在のローソク足の実体が直近5本より大きく、下落して終値を形成。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 初期ストップまたはトレーリングストップに到達。
- **ストップ**: はい、遅延トレーリングストップ。
- **デフォルト値**:
  - `TrailStartTicks` = 200
  - `TrailDistanceTicks` = 150
  - `InitialStopLossTicks` = 200
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
