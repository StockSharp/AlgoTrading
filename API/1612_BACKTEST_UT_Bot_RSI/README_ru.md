# Стратегия Backtest UT Bot + RSI
[English](README.md) | [中文](README_cn.md)

Сочетает трендовый индикатор UT Bot с уровнями RSI. Открывает лонг при развороте вверх и RSI ниже порога, шорт — при развороте вниз и RSI выше порога.

## Детали

- **Условия входа**:
  - **Long**: UT Bot меняет направление вверх и RSI < `RSI Oversold`.
  - **Short**: UT Bot меняет направление вниз и RSI > `RSI Overbought`.
- **Long/Short**: Оба направления.
- **Условия выхода**:
  - Тейк-профит или стоп-лосс в процентах.
- **Стопы**: Тейк-профит и стоп-лосс.
- **Значения по умолчанию**:
  - `RSI Length` = 14
  - `RSI Overbought` = 60
  - `RSI Oversold` = 40
  - `ATR Length` = 10
  - `UT Bot Factor` = 1.0
  - `Take Profit %` = 3.0
  - `Stop Loss %` = 1.5
- **Фильтры**:
  - Категория: Trend Following
  - Направление: Оба
  - Индикаторы: Несколько
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
