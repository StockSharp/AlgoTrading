# Strategie HTF Candles Lib
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

HTF Candles Lib ist eine Hilfsstrategie, die Kerzen höherer Zeitrahmen aus Daten niedrigerer Zeitrahmen rekonstruiert und deren detaillierte Struktur protokolliert.

## Details
- **Daten**: Preiskerzen.
- **Einstiegskriterien**: Keine.
- **Ausstiegskriterien**: Keine.
- **Stops**: Keine.
- **Standardwerte**:
  - `CandleType` = 1 Minute
  - `HigherCandleType` = 5 Minuten
- **Filter**:
  - Kategorie: Dienstprogramm
  - Richtung: Keine
  - Indikatoren: Keine
  - Komplexität: Niedrig
  - Risikolevel: Niedrig
