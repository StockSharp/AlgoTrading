# Estratégia HTF Candles Lib
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

HTF Candles Lib é uma estratégia utilitária que reconstrói velas de períodos de tempo superiores a partir de dados de períodos de tempo inferiores e registra sua estrutura detalhada.

## Detalhes
- **Dados**: Velas de preço.
- **Critérios de entrada**: Nenhum.
- **Critérios de saída**: Nenhum.
- **Stops**: Nenhum.
- **Valores padrão**:
  - `CandleType` = 1 minuto
  - `HigherCandleType` = 5 minutos
- **Filtros**:
  - Categoria: Utilitário
  - Direção: Nenhum
  - Indicadores: Nenhum
  - Complexidade: Baixo
  - Nível de risco: Baixo
