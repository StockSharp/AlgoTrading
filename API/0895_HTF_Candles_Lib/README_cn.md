# HTF Candles Lib
[English](README.md) | [Русский](README_ru.md)

HTF Candles Lib 是一个实用策略，从较低时间框架数据重构较高时间框架K线，并记录其详细结构。

## 细节
- **数据**：价格K线。
- **入场条件**：无。
- **出场条件**：无。
- **止损**：无。
- **默认值**：
  - `CandleType` = 1 分钟
  - `HigherCandleType` = 5 分钟
- **过滤器**：
  - 类别：工具
  - 方向：无
  - 指标：无
  - 复杂度：低
  - 风险等级：低
