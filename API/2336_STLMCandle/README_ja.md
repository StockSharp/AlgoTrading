# STLMCandle戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、最後に完成したローソク足の方向に基づいて取引します。
終値が始値を上回る場合、ロングポジションをオープンし、ショートポジションをクローズします。
終値が始値を下回る場合、ショートポジションをオープンし、ロングポジションをクローズします。
ストップロスとテイクプロフィットのレベルをサポートし、設定可能なローソク足の時間軸で動作します。

## パラメーター
- `CandleType` – 分析に使用するローソク足の時間軸。
- `StopLoss` – 価格単位での絶対ストップロス値。
- `TakeProfit` – 価格単位での絶対テイクプロフィット値。

## 注意事項
この戦略は、オリジナルのMQL `STLMCandle` エキスパートアドバイザーの簡略化した適応版です。
標準のローソク足の始値と終値を使用してインジケーターを近似します。
