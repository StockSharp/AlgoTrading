# Стратегия прорыва VWAP

Эта стратегия прорыва отслеживает, насколько далеко цена отклоняется от объёмно-взвешенной средней цены (VWAP). Измеряя расстояние в единицах Average True Range, она пытается определить моменты ускорения импульса.

Покупка выполняется, когда рынок закрывается выше VWAP более чем на `K` ATR, что сигнализирует о сильном восходящем давлении. Аналогично открывается шорт, если цена падает ниже VWAP на `K` ATR. Сделки закрываются, когда цена возвращается к линии VWAP, предполагая, что импульс иссяк.

Подход рассчитан на краткосрочных трейдеров, которые любят торговать резкие расширения волатильности. Фиксированные защитные стопы и четкий уровень повторного входа помогают управлять риском ложных прорывов.

## Подробности
- **Условия входа**:
  - **Лонг**: Price > VWAP + K*ATR (прорыв выше верхней границы)
  - **Шорт**: Price < VWAP - K*ATR (прорыв ниже нижней границы)
- **Длинные/короткие**: обе стороны.
- **Условия выхода**:
  - **Лонг**: выход из лонга при падении цены ниже VWAP
  - **Шорт**: выход из шорта при росте цены выше VWAP
- **Стопы**: да.
- **Значения по умолчанию**:
  - `K` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `AtrPeriod` = 14
- **Фильтры**:
  - Категория: Breakout
  - Направление: оба
  - Индикаторы: VWAP Breakout
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
