# マスターマインド 2 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
MasterMind 2 は、「TheMasterMind2」MQL4 エキスパート アドバイザーの変換版です。この戦略は、Stochastic オシレーターと Williams %R インジケーターの極値を待って枯渇点を検出します。両方のインジケーターが極端な売られ過ぎの状態を示している場合はロングポジションをオープンし、両方のインジケーターが極端な買われ過ぎの状態を示している場合はショートポジションをオープンします。このロジックは完全に閉じたローソク足でのみ動作し、元の Expert Advisor の動作を模倣します。

## インジケーター
- **Stochastic オシレーター** – 買われ過ぎと売られ過ぎのレベルを測定するために長いルックバックを使用して構成されています。 %D 信号ラインがしきい値と比較されます。
- **Williams %R** – ロングの場合は -100 に近く、ショートの場合は 0 に近い読み取り値を要求することで、極端な強度を確認します。

## エントリールール
1. キャンドルが閉じるのを待ちます。
2. Stochastic オシレーターを計算し、その %D 信号値を取得します。
3. 構成されたルックバックに対して Williams %R を計算します。
4. **ロングエントリー**: `%D < 3` と `Williams %R < -99.9` の場合、既存のショートエクスポージャーを決済して購入します。
5. **簡単なエントリー**: `%D > 97` と `Williams %R > -0.1` の場合、既存のロングエクスポージャーをクローズして売ります。

## 終了ルール
- ストップロスとテイクプロフィットのレベルは、設定可能なポイント距離を使用してエントリー価格に対して適用されます。
- トレーリングストップは、価格が指定されたステップで有利に推移した場合に保護ストップを強化することができます。
- 損益分岐点オプションは、取引で必要な利益距離が蓄積された後、ストップロスをエントリーレベルに移動します。
- 反対のシグナルは、新しいポジションを開く前に、現在のポジションを直ちに閉じます。

## パラメーター
- `Trade Volume` – 各成行注文で送信された契約量。
- `Stochastic Period`、`Stochastic %K`、`Stochastic %D` – Stochastic オシレーターのパラメーター。
- `Williams %R Period` – Williams %R 計算のルックバック期間。
- `Stop Loss`、`Take Profit` – 価格帯での保護距離。
- `Trailing Stop`、`Trailing Step` – 動的停止管理を制御します。
- `Break Even` – エントリー価格を固定するために必要なポイント単位の距離。
- `Candle Type` – 計算に使用されるタイムフレームまたはカスタムのローソク足タイプ。

## 注意事項
- この戦略は、元の MQL4 実装と一致する完成したキャンドルのみに依存します。
- すべての注文は、`Trade Volume` で定義された数量で市場で発行されます。
- 距離パラメータをゼロに設定して、保護機能を有効または無効にします。
