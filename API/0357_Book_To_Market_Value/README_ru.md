# Book to Market Value
[English](README.md) | [中文](README_zh.md)

Стратегия **Book-to-Market Value** демонстрирует настройку списка инструментов и подписку на дневные свечи для фактора book-to-market.
Текущая версия служит примером и не содержит торговой логики.

## Подробности
- **Условия входа**: логика фактора не реализована.
- **Направление**: обе стороны.
- **Условия выхода**: нет.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `MinTradeUsd = 200`
  - `CandleType = TimeSpan.FromDays(1).TimeFrame()`
- **Фильтры**:
  - Категория: Фундаментальная
  - Направление: Оба
  - Индикаторы: Fundamentals
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
