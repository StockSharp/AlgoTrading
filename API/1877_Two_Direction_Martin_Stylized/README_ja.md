# 双方向マーチンゲール戦略（スタイライズド）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、簡略化された双方向マーチンゲールアプローチを実装しています。開始時にロングとショート両方のポジションを同時に開き、設定可能な距離に指値注文を置いて利益を確定します。

## 仕組み
1. スプレッドを計算し、現在のアスク価格のパーセンテージとしてテイクプロフィット距離を設定します。
2. ビッド価格の下に買い指値ターゲットを持つ初期売り成行注文と、アスク価格の上に売り指値ターゲットを持つ買い成行注文を送信します。
3. 指値注文の一方が欠けているか、価格が事前定義された範囲外に移動した場合、アルゴリズムは `Same Side %` を使用してボリュームを再計算し、保留注文を置き換えます。必要に応じてポジションのバランスを取るために追加の成行注文が送信されます。
4. すべての注文は `Volume Limit` パラメータを超えない部分に分割されます。

## パラメータ
- **Take Profit %** – 現在の価格からの利益ターゲットまでの距離。
- **Base Volume** – 各初期注文の最小ボリューム。
- **Volume Limit** – 単一注文ピースの最大ボリューム。
- **Same Side %** – 優勢側に割り当てられる総ボリュームのパーセンテージ。
- **Candle Type** – 時間ドライバーとして使用されるローソク足のサブスクリプション。
