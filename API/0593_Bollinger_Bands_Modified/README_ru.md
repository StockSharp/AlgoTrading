# Стратегия Bollinger Bands Modified
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробои полос Боллинджера с опциональным фильтром тренда по EMA. Входит в лонг при пересечении верхней полосы и в шорт при пересечении нижней полосы.

Стоп-лосс ставится на ближайший максимум или минимум, а тейк-профит равен риску, умноженному на коэффициент.

## Детали

- **Условия входа**:
  - Long: цена пересекает верхнюю полосу Боллинджера
  - Short: цена пересекает нижнюю полосу Боллинджера
- **Направление**: Оба
- **Условия выхода**:
  - Long: стоп по ближайшему минимуму, цель = риск * коэффициент
  - Short: стоп по ближайшему максимуму, цель = риск * коэффициент
- **Стопы**: максимум/минимум последних N свечей
- **Значения по умолчанию**:
  - `BollingerLength` = 20
  - `BollingerDeviation` = 0.38m
  - `EmaLength` = 80
  - `HighestLength` = 7
  - `LowestLength` = 7
  - `TargetFactor` = 1.6m
  - `EmaTrend` = true
  - `CrossoverCheck` = false
  - `CrossunderCheck` = false
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Bollinger Bands, EMA, Highest, Lowest
  - Стопы: Да
  - Сложность: Новичок
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
