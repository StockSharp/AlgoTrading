# Стратегия BBTrend SuperTrend Decision
[English](README.md) | [中文](README_cn.md)

Стратегия вычисляет значение **BBTrend** на основе двух полос Боллинджера с различными периодами
и передаёт его в расчёт SuperTrend. Полученное направление SuperTrend определяет открытие длинных
или коротких позиций. Дополнительно можно включить защиту в виде тейк‑профита и стоп‑лосса в
процентах.

## Детали

- **Критерии входа**:
  - Лонг: направление SuperTrend вверх.
  - Шорт: направление SuperTrend вниз.
- **Длинные/короткие**: обе стороны, настраивается.
- **Критерии выхода**:
  - Противоположное направление SuperTrend.
- **Стопы**: опциональные тейк‑профит и стоп‑лосс в процентах.
- **Значения по умолчанию**:
  - Длина коротких BB = 20, длинных BB = 50, StdDev = 2.
  - Длина SuperTrend = 10, множитель = 7.
  - Тейк‑профит = 30%, стоп‑лосс = 20%.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Bollinger Bands, SuperTrend
  - Стопы: Опциональные TP/SL
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
