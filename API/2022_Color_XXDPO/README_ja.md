# Color XXDPO戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

二重平滑化されたデトレンデッド・プライス・オシレーターを使用して傾きの反転を捉える戦略。

## 詳細
- **エントリー条件**: 上向きの傾きで現在値が上昇中はロングを開く；下向きの傾きで現在値が下落中はショートを開く。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対方向の傾き変化でポジションを閉じる。
- **ストップ**: なし。
- **デフォルト値**: 第1 MA長さ 21、第2 MA長さ 5、ローソク足時間軸 6時間。
- **フィルター**: なし。
