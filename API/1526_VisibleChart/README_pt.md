# Estratégia de Gráfico Visível
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Este exemplo mostra como trabalhar com um intervalo visível de velas. A estratégia rastreia a máxima mais alta e a mínima mais baixa ao longo de um número configurável de barras recentes. Um rompimento acima da máxima visível aciona uma entrada comprada, enquanto uma quebra abaixo da mínima visível aciona uma entrada vendida.

## Detalhes
- **Entrada**: Rompimento da máxima/mínima visível.
- **Saída**: Rompimento oposto.
- **Indicadores**: Highest, Lowest.
