# Estrategia de Gráfico Visible
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Este ejemplo muestra cómo trabajar con un rango visible de velas. La estrategia rastrea el máximo más alto y el mínimo más bajo durante un número configurable de barras recientes. Una ruptura por encima del máximo visible activa una entrada larga, mientras que una ruptura por debajo del mínimo visible activa una entrada corta.

## Detalles
- **Entrada**: Ruptura del máximo/mínimo visible.
- **Salida**: Ruptura opuesta.
- **Indicadores**: Highest, Lowest.
