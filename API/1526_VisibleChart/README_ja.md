# 可視チャート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このサンプルは、ローソク足の可視範囲の操作方法を示します。戦略は設定可能な数の直近バーにわたって最高値と最安値を追跡します。可視の高値を上回るブレイクアウトでロングエントリーが発動し、可視の安値を下回るブレイクでショートエントリーが発動します。

## 詳細
- **エントリー**: 可視の高値/安値のブレイクアウト。
- **エグジット**: 反対方向のブレイクアウト。
- **インジケーター**: Highest, Lowest.
