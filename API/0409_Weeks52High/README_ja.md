# 52週高値ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このモメンタム戦略は、新たな52週高値を付けている株式に注目します。こうした株式はその後も市場をアウトパフォームし続けることが多いです。システムはブレイクアウトで買い、モメンタムが衰えるまでポジションを保持します。

対象株式は流動性とファンダメンタルズの強さでフィルタリングされます。

## 詳細

- **データ**: 過去1年の日次高値と出来高。
- **エントリー**: 価格が直近52週高値を上抜けたときに買い。
- **エグジット**: 10%の押し目または新安値で売り。
- **銘柄**: 株式。
- **リスク**: ボラティリティによるポジションサイジング。
