# 52-Wochen-Hoch-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Momentum-Strategie konzentriert sich auf Aktien, die neue 52-Wochen-Hochs erreichen – diese übertreffen den Markt häufig weiter. Das System kauft Ausbrüche und hält die Position, bis der Momentum nachlässt.

Geeignete Aktien werden nach Liquidität und fundamentaler Stärke gefiltert.

## Details

- **Daten**: Tägliche Hochs über das vergangene Jahr und Volumen.
- **Einstieg**: Kauf, wenn der Kurs das vorherige 52-Wochen-Hoch übersteigt.
- **Ausstieg**: Verkauf bei 10%-Rücksetzer oder neuem Tief.
- **Instrumente**: Aktien.
- **Risiko**: Positionsgrößenbestimmung nach Volatilität.
