# Estrategia de Máximo de 52 Semanas
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia de momentum se centra en acciones que alcanzan nuevos máximos de 52 semanas, las cuales a menudo continúan superando al mercado. El sistema compra las rupturas y mantiene hasta que el momentum se desvanece.

Las acciones calificadas se filtran por liquidez y solidez fundamental.

## Detalles

- **Datos**: Máximos diarios del último año y volumen.
- **Entrada**: Comprar cuando el precio supera el máximo previo de 52 semanas.
- **Salida**: Vender ante una corrección del 10% o un nuevo mínimo.
- **Instrumentos**: Acciones.
- **Riesgo**: Dimensionamiento de posición según volatilidad.
