# ハイイールドスプレッドSMAフィルター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はハイイールド信用スプレッドまたはVIX指数のレベルに基づいて取引します。選択したスプレッドが閾値を超え、価格が単純移動平均の適切な側にある場合、選択した方向にポジションを開きます。

ポジションは固定本数のローソク足の間保持され、その後決済されます。この戦略は日足ローソク足のみで動作します。

## 詳細

- **エントリー条件**:
  - **ロング**: スプレッド > 閾値 かつ（フィルター有効時 価格 > SMA）
  - **ショート**: スプレッド < 閾値 かつ（フィルター有効時 価格 < SMA）
- **ロング/ショート**: 一度に一方向（パラメーター）
- **エグジット条件**: 保有期間後にポジション決済
- **ストップ**: なし
- **デフォルト値**:
  - `Basis` = HighYieldSpread
  - `Threshold` = 5
  - `IsLong` = true
  - `HoldingPeriod` = 5
  - `UseSmaFilter` = true
  - `SmaLength` = 50
  - `CandleType` = 1 day
- **フィルター**:
  - カテゴリ: スプレッド
  - 方向: 設定可能
  - インジケーター: SMA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
