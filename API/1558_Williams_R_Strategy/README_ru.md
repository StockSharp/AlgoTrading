# Стратегия Williams %R
[English](README.md) | [中文](README_cn.md)

Стратегия покупает, когда Williams %R падает в глубокую перепроданность, и выходит при пробое вверх или переходе в перекупленность.

## Детали

- **Критерий входа**: %R < -90
- **Лонг/Шорт**: только лонг
- **Критерий выхода**: закрытие выше предыдущего максимума или %R > -30
- **Стопы**: нет
- **Значения по умолчанию**:
  - `LookbackPeriod` = 2
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Лонг
  - Индикаторы: WilliamsR
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
