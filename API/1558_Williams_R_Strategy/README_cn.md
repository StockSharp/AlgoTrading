# Williams %R 策略
[English](README.md) | [Русский](README_ru.md)

该策略在 Williams %R 深度超卖时买入，在价格突破前高或指标进入超买时退出。

## 细节

- **入场条件**: %R < -90
- **多空方向**: 仅多头
- **出场条件**: 收盘价高于前高或 %R > -30
- **止损**: 无
- **默认值**:
  - `LookbackPeriod` = 2
- **过滤器**:
  - 分类: 振荡器
  - 方向: 多头
  - 指标: WilliamsR
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
