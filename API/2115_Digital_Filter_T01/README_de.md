# Digitaler Filter T01-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

## Zusammenfassung
DigitalF-T01-Filter im Vergleich mit einer verschobenen Triggerlinie. Die Strategie eröffnet eine Long-Position, wenn der digitale Filter die Triggerlinie von oben nach unten kreuzt, und eine Short-Position, wenn er von unten nach oben kreuzt. Stop-Loss- und Take-Profit-Schutz werden in Punkten angewendet.

## Parameter
- **Half Channel** - halber Kanalabstand, der zur Triggerlinie addiert wird.
- **Stop Loss** - Stop-Loss-Wert in Punkten.
- **Take Profit** - Take-Profit-Wert in Punkten.
- **Candle Type** - Kerzentyp, der für die Berechnungen verwendet wird.
