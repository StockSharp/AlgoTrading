# エリオット波動オシレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はローソク足の終値にエリオット波動オシレーター（EWO）を適用します。EWOは高速と低速の単純移動平均の差として計算されます（デフォルトは5期間と35期間）。取引ロジックはオシレーターの転換点を探し、潜在的なトレンド転換を捉えます。

オシレーターが局所的な谷を形成して上昇し始めるとロングポジションが建てられます。オシレーターが局所的な山を形成して下降し始めるとショートポジションが建てられます。既存のポジションはそれに応じて反転します。`StartProtection` を通じたオプションのパーセンテージベースのテイク‑プロフィットとストップ‑ロスがサポートされています。

## 詳細

- **インジケーター**: エリオット波動オシレーター = SMA(高速) − SMA(低速)。
- **エントリー条件**:
  - **ロング**: オシレーター値が下降後に上向きに転換。
  - **ショート**: オシレーター値が上昇後に下向きに転換。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対シグナルでポジションが反転するか、ストップまたはテイク‑プロフィットで退場。
- **ストップ**: パーセンテージベースのストップ‑ロスとテイク‑プロフィット。
- **フィルター**: なし。
