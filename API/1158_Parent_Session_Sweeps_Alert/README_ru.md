# Слипы родительской сессии с алертом
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает дневные сессии и фиксирует, когда текущая сессия выбивает максимум или минимум предыдущей. При возврате свечи внутрь диапазона открывается сделка в противоположном направлении с настраиваемым соотношением риск-прибыль.

## Подробности

- **Критерий входа**: Слип предыдущего максимума/минимума и опциональный фильтр закрытия свечи.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: стоп по экстремуму сессии или цель по риск-прибыли.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `MinRiskReward` = 1
  - `UseCandleFilter` = true
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Price Action
  - Направление: Обе стороны
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
