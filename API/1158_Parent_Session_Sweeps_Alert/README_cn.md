# 父级会话清扫警报
[English](README.md) | [Русский](README_ru.md)

该策略监控每日会话，当当前会话突破前一会话的高点或低点并收回到范围内时，反向开仓，风险回报比可配置。

## 细节

- **入场条件**：扫过前一会话的高/低并可选地收回。
- **多空方向**：双向。
- **出场条件**：会话极值止损或按风险回报比设定的目标。
- **止损**：是。
- **默认值**：
  - `MinRiskReward` = 1
  - `UseCandleFilter` = true
  - `CandleType` = TimeSpan.FromMinutes(15)
- **过滤器**：
  - 类别：Price Action
  - 方向：双向
  - 指标：无
  - 止损：是
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
