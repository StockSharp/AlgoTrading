# 親セッション・スイープアラート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は日次セッションを監視し、現在のセッションが前セッションの高値または安値をスイープするときを検出します。スイープが発生し、ローソク足が前のレンジ内に戻って引けたとき、設定可能なリスクリワード比で逆方向にトレードを建てます。

## 詳細

- **エントリー条件**: 前セッションの高値/安値のスイープ（オプションのローソク足クローズフィルター付き）。
- **ロング/ショート**: 両方向。
- **エグジット条件**: セッション極値でのストップ、またはリスクリワードに基づく目標。
- **ストップ**: あり。
- **デフォルト値**:
  - `MinRiskReward` = 1
  - `UseCandleFilter` = true
  - `CandleType` = TimeSpan.FromMinutes(15)
- **フィルター**:
  - カテゴリ: Price Action
  - 方向: 両方
  - インジケーター: なし
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: なし
  - ニューラルネットワーク: なし
  - ダイバージェンス: なし
  - リスクレベル: 中
