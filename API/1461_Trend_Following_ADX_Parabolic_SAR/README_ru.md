# Следование тренду ADX Parabolic SAR
[English](README.md) | [中文](README_cn.md)

Использует ADX и Parabolic SAR для торговли по тренду. Лонг открывается, когда ADX выше порога, +DI превышает -DI и цена находится выше линии SAR. Шорт — при обратных условиях.

## Детали

- **Критерии входа**: ADX > порога, +DI > -DI и цена > SAR.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `AdxPeriod` = 14
  - `AdxThreshold` = 25
  - `SarStep` = 0.02
  - `SarMax` = 0.2
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ADX, Parabolic SAR
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
