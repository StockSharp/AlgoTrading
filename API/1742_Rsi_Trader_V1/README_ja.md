# RSI Trader V1 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は相対力指数 (RSI) を使用して、短期的な極値後の反転を識別します。RSIが2本連続のローソク足で売られすぎ閾値を下回った後、その閾値を上抜けしたときに買いシグナルが発生します。RSIが2本連続のローソク足で買われすぎ閾値を上回った後、その閾値を下抜けしたときに売りシグナルが発生します。この戦略は既存の反対ポジションをオプションで決済し、設定可能な時間窓内でのみ取引します。

## 詳細

- **エントリー条件**:
  - **ロング**: `RSI > BuyPoint` かつ前の2本のローソク足のRSIが `< BuyPoint`。
  - **ショート**: `RSI < SellPoint` かつ前の2本のローソク足のRSIが `> SellPoint`。
- **エグジット条件**: 逆シグナルまたは保護ストップ/テイクプロフィット。
- **時間フィルター**: ローソク足の始値時刻が`StartHour`と`EndHour`の間にあるときのみ取引。
- **ストップ**: 価格単位で表示された固定テイクプロフィットとストップロス。
- **パラメーター**:
  - `RsiPeriod` – RSI計算期間。
  - `BuyPoint` – ロングエントリーの売られすぎレベル。
  - `SellPoint` – ショートエントリーの買われすぎレベル。
  - `CloseOnOpposite` – 逆シグナルが出たときに現在のポジションを決済。
  - `StartHour` / `EndHour` – 取引時間。
  - `TakeProfit` / `StopLoss` – 価格での保護レベル。

この例はStockSharpのハイレベルAPIで構築されたミニマリスト的なRSIクロスオーバーシステムを実演しています。さらなる実験のためのテンプレートとして使用できます。
