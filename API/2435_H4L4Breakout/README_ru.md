# Прорыв H4/L4
[English](README.md) | [中文](README_cn.md)

Дневная стратегия прорыва рассчитывает уровни H4 и L4 на основе максимума, минимума и закрытия предыдущего дня.
В начале каждого дня размещаются отложенные заявки: Sell Limit на уровне H4 и Buy Limit на уровне L4.
Перед установкой новых заявок стратегия отменяет все активные заявки и закрывает позиции.
Защитные стоп-лосс и тейк-профит задаются в тиках.

## Детали

- **Критерии входа**: Sell Limit на H4 и Buy Limit на L4, рассчитанных по свече предыдущего дня.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: стоп-лосс или тейк-профит.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `TakeProfit` = 57
  - `StopLoss` = 7
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
