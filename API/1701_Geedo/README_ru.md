# Стратегия Geedo
[English](README.md) | [中文](README_cn.md)

Стратегия на основе времени: в заданный час сравниваются цены открытия двух прошлых свечей. Если более ранняя свеча выше недавней на пороговое значение — открывается короткая позиция. Если недавняя свеча выше ранней — открывается длинная позиция. Каждая сделка имеет фиксированные стоп‑лосс и тейк‑профит и закрывается по истечении максимального времени удержания.

## Детали

- **Критерии входа**: В `TradeTime` сравниваются цены открытия `T1` и `T2` свечей назад. Если `Open[T1] - Open[T2]` больше `DeltaShort` — продажа, если `Open[T2] - Open[T1]` больше `DeltaLong` — покупка.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: стоп‑лосс, тейк‑профит или `MaxOpenTime` часов после входа.
- **Стопы**: фиксированные.
- **Значения по умолчанию**:
  - `TakeProfitLong` = 39
  - `StopLossLong` = 147
  - `TakeProfitShort` = 15
  - `StopLossShort` = 6000
  - `TradeTime` = 18
  - `T1` = 6
  - `T2` = 2
  - `DeltaLong` = 6
  - `DeltaShort` = 21
  - `Volume` = 0.01
  - `MaxOpenTime` = 504
- **Фильтры**:
  - Категория: Временная
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Фиксированные
  - Сложность: Начальная
  - Таймфрейм: Внутридневной (1ч)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
