# MasterMind 3戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、異なる期間を持つ4つの**Williams %R**インジケーターを使用して極端なリバーサルを取引します。すべてのインジケーターが深い売られすぎの値まで下落したとき、戦略はロングポジションに入ります。すべてのインジケーターが強い買われすぎの値まで上昇したとき、ショートポジションに入ります。

## 取引ロジック

1. 選択した時間軸のローソク足を購読する。
2. 期間26、27、29、30の4つのWilliams %Rインジケーターを計算する。
3. すべてのインジケーターが`-99.99`を下回ったとき**買い**。
4. すべてのインジケーターが`-0.01`を上回ったとき**売り**。
5. シグナルは完成したローソク足でのみ処理される。

注文量は戦略の`Volume`プロパティから取得されます。既存の反対ポジションは必要なサイズの成行注文を送ることで自動的に決済されます。

## パラメーター

| 名前 | 説明 | デフォルト |
|------|------|-----------|
| `WprPeriod1` | 最初のWilliams %Rインジケーターの期間 | 26 |
| `WprPeriod2` | 2番目のWilliams %Rインジケーターの期間 | 27 |
| `WprPeriod3` | 3番目のWilliams %Rインジケーターの期間 | 29 |
| `WprPeriod4` | 4番目のWilliams %Rインジケーターの期間 | 30 |
| `CandleType` | ローソク足のタイプと時間軸 | 1分足 |

## 注記

* 戦略はインジケーター処理に`Bind`を使用した高レベルAPIを使います。
* ストップロスやテイクプロフィットレベルは含まれていません。反対のシグナルでポジションを反転します。
