# Estratégia de Pesquisa Donchian Quest
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia usa dois canais Donchian com comprimentos diferentes. Um rompimento acima da banda superior do canal mais longo abre uma posição comprada, enquanto um rompimento abaixo da banda inferior abre uma posição vendida. As posições são fechadas quando o preço toca a banda oposta do canal mais curto.

## Parâmetros

- `OpenPeriod` = 50
- `ClosePeriod` = 50

## Indicadores

- Canal Donchian
