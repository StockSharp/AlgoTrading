# Donchian Quest リサーチ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、異なる長さの2つのDonchianチャネルを使用します。長い方のチャネルの上限バンドを上回るブレイクアウトでロングポジションを開き、下限バンドを下回るブレイクアウトでショートポジションを開きます。短い方のチャネルの反対側のバンドに価格が触れたときにポジションを決済します。

## パラメーター

- `OpenPeriod` = 50
- `ClosePeriod` = 50

## インジケーター

- Donchianチャネル
