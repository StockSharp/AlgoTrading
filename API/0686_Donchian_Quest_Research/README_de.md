# Donchian Quest Forschungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet zwei Donchian-Kanäle mit unterschiedlichen Längen. Ein Ausbruch über das obere Band des längeren Kanals eröffnet eine Long-Position, während ein Ausbruch unter das untere Band eine Short-Position eröffnet. Positionen werden geschlossen, wenn der Preis das gegenüberliegende Band des kürzeren Kanals berührt.

## Parameter

- `OpenPeriod` = 50
- `ClosePeriod` = 50

## Indikatoren

- Donchian-Kanal
