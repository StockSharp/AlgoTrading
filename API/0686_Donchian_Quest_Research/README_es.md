# Estrategia de Investigación Donchian Quest
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza dos canales Donchian con diferentes longitudes. Una ruptura por encima de la banda superior del canal más largo abre una posición larga, mientras que una ruptura por debajo de la banda inferior abre una posición corta. Las posiciones se cierran cuando el precio toca la banda opuesta del canal más corto.

## Parámetros

- `OpenPeriod` = 50
- `ClosePeriod` = 50

## Indicadores

- Canal Donchian
