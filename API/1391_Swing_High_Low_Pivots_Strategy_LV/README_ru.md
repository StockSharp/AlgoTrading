# Стратегия свинговых пивотов [LV]
[English](README.md) | [中文](README_cn.md)

Торговля строится вокруг подтверждённых свинговых максимумов и минимумов. При появлении пивота-минимума выставляется лимитная заявка на покупку по цене бара пивота и фиксируются уровни стопа и профита. Пивот-максимум даёт сигнал к продаже. Опциональный фильтр на основе скользящей средней позволяет торговать только по тренду.

## Детали

- **Входные параметры**:
  - Длина пивота.
  - Дистанция стоп-лосса в тиках.
  - Дистанция тейк-профита в тиках.
  - Второй тейк-профит и удвоенный вход.
  - Тип и период скользящей средней.
- **Лонг/Шорт**: оба.
- **Выход**: фиксированный стоп и до двух целей.
- **Фильтры**:
  - Категория: распознавание паттернов
  - Направление: оба
  - Индикаторы: скользящая средняя
  - Стопы: фиксированные
  - Сложность: высокая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
