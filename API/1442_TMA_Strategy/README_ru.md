# Стратегия TMA
[English](README.md) | [中文](README_cn.md)

Стратегия TMA использует несколько сглаженных скользящих средних и свечные паттерны для торговли по направлению тренда SMA(200). Сигналы формируются по паттернам «3 пробоя» и «поглощение» с учётом фильтра сессии.

## Детали

- **Критерий входа**: бычье поглощение или 3 медвежьих свечи с разворотом при EMA(2) выше SMA(200) / медвежье поглощение или 3 бычьи свечи при EMA(2) ниже SMA(200) и опциональный фильтр сессии
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: пересечение EMA(2) и SMA(200)
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = свечи 5 минут
  - `FastLength` = 21
  - `MidLength` = 50
  - `Mid2Length` = 100
  - `SlowLength` = 200
  - `UseSession` = false
  - `SessionStart` = 08:30
  - `SessionEnd` = 12:00
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: SMA, EMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
