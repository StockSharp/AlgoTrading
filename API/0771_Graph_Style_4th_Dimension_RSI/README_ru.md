# Graph Style 4th Dimension RSI
[English](README.md) | [中文](README_cn.md)

Стратегия, сочетающая изменение цены с уровнями RSI.

Тестирование показывает среднегодовую доходность около 80%. Хорошо работает на волатильных рынках.

Стратегия проверяет направление последнего изменения цены вместе с экстремумами RSI. Открытие позиции происходит при выходе RSI из зон перекупленности/перепроданности, подтвержденное движением цены. Закрытие выполняется при возврате RSI к середине или появлении противоположного сигнала.

## Детали

- **Условия входа**: Направление цены с экстремумами RSI.
- **Длинные/короткие**: В обе стороны.
- **Условия выхода**: Противоположный сигнал или возврат RSI к середине.
- **Стопы**: Процентный стоп-лосс.
- **Значения по умолчанию**:
  - `RsiPeriod` = 14
  - `OverboughtLevel` = 70m
  - `OversoldLevel` = 30m
  - `StopLossPercent` = 1m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Процент
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
