# Fibonacci専用戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Fibonacci専用戦略は、直近100本のローソク足から導出した19%と82.56%のカスタムリトレースメントレベルを使用します。価格がこれらのレベルに触れるか突破し、ローソク足の方向で確認されたときにエントリーします。オプションのブレイクアウトエントリー、ATRベースのストップロス、トレーリングストップ、7段階のテイクプロフィットをサポートします。

## 詳細

- **エントリー条件**: 確認を伴うFibonacciレベルへの接触またはブレイクアウト
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロス、トレーリングストップ、またはテイクプロフィット目標
- **ストップ**: ATRまたはパーセント
- **デフォルト値**:
  - `CandleType` = 15分
- **フィルター**:
  - カテゴリ: Fibonacci
  - 方向: 両方
  - インジケーター: Highest, Lowest, ATR
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
