# Стратегия Fibonacci-Only
[English](README.md) | [中文](README_cn.md)

Стратегия Fibonacci-Only использует кастомные уровни 19% и 82.56% от диапазона последних 100 свечей. Вход происходит при касании или пробое этих уровней с подтверждением направления свечи. Поддерживаются опциональные пробойные входы, ATR-стоп, трейлинг-стоп и семь ступенчатых тейк-профитов.

## Детали

- **Критерий входа**: касание или пробой уровней Фибоначчи с подтверждением
- **Длинная/короткая сторона**: обе
- **Критерий выхода**: стоп-лосс, трейлинг-стоп или уровни тейк-профита
- **Стопы**: ATR или процент
- **Значения по умолчанию:**
  - `CandleType` = 15 минут
- **Фильтры:**
  - Категория: Фибоначчи
  - Направление: обе
  - Индикаторы: Highest, Lowest, ATR
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
