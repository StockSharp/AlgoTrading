# Стратегия Unicode Font Function V2 JD
[English](README.md) | [中文](README_cn.md)

Эта утилитарная стратегия демонстрирует преобразование строк в разные стили Unicode. Торговые операции не выполняются.

## Детали

- **Условия входа**: Н/Д
- **Длин/Корот**: Нет
- **Условия выхода**: Н/Д
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Input1` = "This function brought to you by"
  - `Input2` = "Please input your custom text, then try to change the font type also."
  - `FontType1` = "Sans Bold Italic"
  - `FontType2` = "Regional Indicator"
- **Фильтры**:
  - Категория: Утилита
  - Направление: Нет
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Неприменим
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
