# US Index First 30m Candle Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробой диапазона первой 30-минутной свечи американской сессии, одна сделка в день.

## Детали

- **Условия входа**: после фиксации диапазона первой 30-минутной свечи цена пробивает максимум или минимум
- **Длинные/Короткие**: Оба направления
- **Условия выхода**: стоп на противоположной границе диапазона, цель = размер диапазона * коэффициент риск/прибыль
- **Стопы**: Да
- **Значения по умолчанию**:
  - `RiskReward` = 1
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
