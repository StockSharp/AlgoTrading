# US Index First 30m Candle Strategy
[English](README.md) | [Русский](README_ru.md)

本策略在美股开盘后首个30分钟区间形成后，突破该区间进行交易，每日仅一笔。

## 详情

- **入场条件**：锁定首个30分钟区间后，价格突破其高点或低点
- **多空方向**：双向
- **出场条件**：止损设在区间另一侧，目标=区间大小 * 风险回报系数
- **止损**：有
- **默认值**：
  - `RiskReward` = 1
- **过滤器**：
  - 类别：突破
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
