# USインデックス最初の30分足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

US セッションの最初の30分レンジのブレイクアウトを1日1トレードで取引する。

## 詳細

- **エントリー条件**: 最初の30分レンジ確定後、価格が高値を上抜けまたは安値を下抜け
- **ロング/ショート**: 両方
- **エグジット条件**: 逆側のレンジ水準にストップ、レンジ幅 * リスクリワードを目標
- **ストップ**: はい
- **デフォルト値**:
  - `RiskReward` = 1
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
