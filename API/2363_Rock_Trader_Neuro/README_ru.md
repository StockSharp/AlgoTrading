# Rock Trader Neuro Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия торгует с помощью полос Боллинджера и простого нейрона.
Ширина полос за последние семь свечей нормализуется в диапазон [-1,1]
и умножается на фиксированные веса. Взвешенная сумма проходит через
активацию гиперболическим тангенсом. Отрицательный выход открывает
длинную позицию, положительный — короткую. Позиции закрываются по
стоп-лоссу или тейк-профиту.

## Детали

- **Условия входа**:
  - Лонг: выход нейрона < 0
  - Шорт: выход нейрона > 0
- **Длинные/Короткие**: обе стороны
- **Условия выхода**:
  - Достижение стоп-лосса или тейк-профита
- **Стопы**: абсолютное расстояние по цене
- **Значения по умолчанию**:
  - `StopLoss` = 30
  - `TakeProfit` = 100
  - `Lot` = 1
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Neural
  - Направление: обе стороны
  - Индикаторы: Bollinger Bands
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: краткосрочный
  - Сезонность: нет
  - Нейросети: да
  - Дивергенция: нет
  - Уровень риска: средний
