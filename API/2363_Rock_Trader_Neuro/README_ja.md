# Rock Trader Neuro戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ボリンジャーバンドと単純なニューロンを使用して取引する戦略です。
直近7本のボリンジャーバンド幅を[-1,1]の範囲に正規化し、
固定の重みと組み合わせます。加重和は双曲線正接活性化関数を通過します。
負の出力はロングポジションを建て、正の出力はショートポジションを建てます。
ポジションはストップロスまたはテイクプロフィットで決済されます。

## 詳細

- **エントリー条件**:
  - ロング: ニューロン出力 < 0
  - ショート: ニューロン出力 > 0
- **ロング/ショート**: 両方
- **エグジット条件**:
  - ストップロスまたはテイクプロフィット到達
- **ストップ**: 絶対価格距離
- **デフォルト値**:
  - `StopLoss` = 30
  - `TakeProfit` = 100
  - `Lot` = 1
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: Neural
  - 方向: 両方
  - インジケーター: Bollinger Bands
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: はい
  - ダイバージェンス: いいえ
  - リスクレベル: 中
