# IU 4-Bar-Aufwärts-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die IU 4-Bar-Aufwärts-Strategie ist ein Nur-Long-Ansatz, der nach vier aufeinanderfolgenden bullischen Kerzen kauft, wenn der Kurs über dem SuperTrend-Indikator liegt.

## Details
- **Daten**: Kurskerzen.
- **Einstiegskriterien**:
  - **Long**: Vier aufeinanderfolgende bullische Kerzen und Schluss über SuperTrend.
- **Ausstiegskriterien**: Schluss unter SuperTrend.
- **Stops**: Keine.
- **Standardwerte**:
  - `SupertrendLength` = 14
  - `SupertrendMultiplier` = 1
- **Filter**:
  - Kategorie: Trendfolge
  - Richtung: Long
  - Indikatoren: SuperTrend
  - Komplexität: Niedrig
  - Risikolevel: Mittel
