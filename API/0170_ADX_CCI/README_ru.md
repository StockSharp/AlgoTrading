# Стратегия Adx Cci
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторах ADX и CCI. Вход в покупки при ADX > 25 и CCI ниже -100 (перепроданность). Вход в продажи при ADX > 25 и CCI выше 100 (перекупленность).

ADX оценивает силу тренда, а CCI определяет момент входа после откатов. Сделки открываются по направлению ADX.

Подходит трейдерам импульсных стратегий, заходящим на коррекциях. Стопы управляются через множитель ATR.

## Подробности

- **Условия входа**:
  - Длинная: `ADX > 25 && CCI < -100`
  - Короткая: `ADX > 25 && CCI > 100`
- **Long/Short**: Оба
- **Условия выхода**: ослабление тренда или пересечение CCI нуля
- **Стопы**: процентный уровень через `StopLossPercent`
- **Параметры по умолчанию**:
  - `AdxPeriod` = 14
  - `CciPeriod` = 20
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: Оба
  - Индикаторы: ADX, CCI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
