# Break Even Master
[English](README.md) | [中文](README_cn.md)

Стратегия **Break Even Master** автоматически переводит стоп-лосс в точку входа после того как прибыль по позиции достигает заданного количества тиков. Дополнительно можно фильтровать заявки по комментарию и магическому номеру, что повторяет логику оригинального советника MetaTrader.

## Подробности
- **Критерии входа**: внешние, стратегия лишь управляет уже открытыми позициями.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: цена достигает стопа в безубытке.
- **Стопы**: только перевод в безубыток.
- **Значения по умолчанию**:
  - `BreakEvenTicks = 20`
  - `UseComment = false`
  - `Comment = ""`
  - `UseMagicNumber = false`
  - `MagicNumber = 12345`
  - `CandleType = TimeSpan.FromMinutes(1).TimeFrame()`
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Безубыток
  - Сложность: Новичок
  - Таймфрейм: Внутридневной (1m)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
