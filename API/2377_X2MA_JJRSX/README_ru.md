# Стратегия X2MA JJRSX
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает фильтр тренда на основе двух скользящих средних и входы по RSI.
Тренд определяется на старшем таймфрейме сравнением быстрой и медленной средней.
Сделки совершаются на младшем таймфрейме, когда RSI выходит из зон перепроданности или перекупленности по направлению тренда.

## Подробности

- **Условия входа**:
  - Лонг: тренд вверх и RSI пересекает уровень `Oversold` снизу вверх
  - Шорт: тренд вниз и RSI пересекает уровень `Overbought` сверху вниз
- **Long/Short**: Оба
- **Условия выхода**: Обратный порог RSI или разворот тренда
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `TrendCandleType` = свечи 4h
  - `SignalCandleType` = свечи 30m
  - `FastMaPeriod` = 12
  - `SlowMaPeriod` = 5
  - `RsiPeriod` = 8
  - `Overbought` = 70
  - `Oversold` = 30
