# 自己学習エキスパート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格の歴史的な二値パターンから学習し、将来の上昇または下降の確率を推定します。確率がユーザー定義の閾値を超えると、戦略はその方向に成行ポジションを開きます。収集された統計は、最近の動向により重みを付けるために、忘却係数によって時間とともに減衰します。システムは、新しいシグナルが現れたときにストップレベルをオプションで移動でき、価格ステップに基づくトレーリングストップをサポートします。

## 詳細

- **エントリー条件**:
  - **ロング**: 上昇移動の確率 ≥ `ProbabilityThreshold`。
  - **ショート**: 下降移動の確率 ≥ `ProbabilityThreshold`。
- **ストップ**: 対称的なストップロスとテイクプロフィットを持つオプションのトレーリングストップ。
- **デフォルト値**:
  - `PatternSize` = 10
  - `ProbabilityThreshold` = 0.8
  - `ForgetRate` = 1.05
  - `Trailing` = 0 (無効)
- **フィルター**:
  - カテゴリ: パターン認識
  - 方向: 両方
  - インジケーター: なし
  - ストップ: オプション
  - 複雑さ: 高
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
