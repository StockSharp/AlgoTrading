# Стратегия Liquidex
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя, входящая при выходе цены за границы каналов Келтнера и управляющая риском через стоп‑лосс, тейк‑профит, перевод в безубыток и трейлинг‑стоп.

## Подробности

- **Условия входа**:
  - Лонг: закрытие выше верхней полосы Келтнера.
  - Шорт: закрытие ниже нижней полосы Келтнера.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - Срабатывание стоп‑лосса или тейк‑профита.
  - Перенос стопа в безубыток после достижения цели.
  - Активированный трейлинг‑стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `KcPeriod` = 10
  - `UseKcFilter` = true
  - `StopLoss` = 30
  - `TakeProfit` = 0
  - `MoveToBe` = 15
  - `MoveToBeOffset` = 2
  - `TrailingDistance` = 5
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Channel
  - Направление: обе
  - Индикаторы: Keltner
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
