# ストップロスをブレイクイーブンに移動する戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ポジションが指定したpipの利益に達したときに、保護ストップロスをエントリー価格に移動させます。注文を手動で調整せずに利益を確保するのに役立ちます。

## 仕組み

- 選択したローソク足タイプを使用して価格を監視する。
- 現在のポジションの利益が設定されたpip数を超えたとき、エントリー価格にストップ注文を出す。
- ロングとショートの両ポジションに対応し、インスツルメントの価格刻みを使用してpipサイズを自動計算する。

## パラメーター

| 名前 | 説明 |
| ---- | ----------- |
| **BreakEvenPips** | ストップロスをエントリー価格に移動させるのに必要なpip単位の利益。 |
| **CandleType** | 価格動向の監視に使用するローソク足のタイプ。 |

## 注意事項

この戦略はエントリーシグナルを生成しません。ポジションは他の戦略または手動で開かれる必要があります。ポジションが閉じられると、内部状態がリセットされ次の取引を待ちます。
