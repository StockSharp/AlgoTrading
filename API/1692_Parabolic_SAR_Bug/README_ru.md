# Parabolic SAR Bug
[English](README.md) | [中文](README_cn.md)

Стратегия **Parabolic SAR Bug** торгует на разворотах тренда с использованием индикатора Parabolic SAR. Когда SAR опускается ниже цены, стратегия открывает длинную позицию, а когда поднимается выше — короткую. Опциональный режим reverse инвертирует сигналы. Защита позиции реализована через стоп‑лосс, тейк‑профит и при необходимости трейлинг‑стоп.

## Детали

- **Условия входа**: смена направления Parabolic SAR.
- **Длинные/короткие**: оба направления.
- **Условия выхода**: противоположный сигнал SAR или защитный стоп.
- **Стопы**: стоп‑лосс, тейк‑профит, опциональный трейлинг‑стоп.
- **Значения по умолчанию**:
  - `Step` = 0.02
  - `MaxStep` = 0.2
  - `StopLossPercent` = 2
  - `TakeProfitPercent` = 1
  - `UseTrailingStop` = false
  - `Reverse` = false
  - `CloseOnSar` = true
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Parabolic SAR
  - Стопы: Стоп‑лосс, тейк‑профит
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5 м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
