# Stochastic RSI SuperTrend戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このシステムはStochastic RSIの高速な振動をトレンドフィルターと簡略化された
SuperTrendモデルと組み合わせます。オシレーターは短期モメンタムの極値を強調し、
移動平均とATRバンドが支配的なトレンドを定義します。%KラインがZ関連ゾーン内で
%Dをクロスし、より広いトレンドが整合している場合にのみ取引が開かれ、横ばい状況
でのだましを減らします。

デフォルト設定はロングトレードに焦点を当てていますが、オプションでショートエント
リーを有効にすることができます。この戦略はStochastic RSIシグナルが頻繁に現れ、
ATRベースのバンドがボラティリティ適応バイアスを提供するイントラデイからスイング
の時間軸向けに設計されています。エグジットは反対のクロスオーバーで発生し、モメン
タムが衰えるまでマーケットを走らせます。

## 詳細

- **エントリー条件**:
  - **ロング**: トレンドMAを上回る終値、%K < 20、%KがDを上抜け、SuperTrendが上昇トレンドを示す。
  - **ショート**: トレンドMAを下回る終値、%K > 80、%KがDを下抜け、SuperTrendが下降トレンドを示す。
- **ロング/ショート**: デフォルトはロング、ショートはオプション。
- **エグジット条件**:
  - **ロング**: %K > 80 かつ %Dを下抜け。
  - **ショート**: %K < 20 かつ %Dを上抜け。
- **ストップ**: デフォルトはなし。外部で追加可能。
- **デフォルト値**:
  - RSI期間 = 14、Stochasticの長さ = 14。
  - MAタイプ = EMA、MAの長さ = 100。
  - ATR期間 = 10、ATRファクター = 3.0。
- **フィルター**:
  - カテゴリ: モメンタム + トレンド
  - 方向: 主にロング
  - インジケーター: RSI、ATR、移動平均
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 短期/中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
