# Strategy Connectable Strategy
[English](README.md) | [中文](README_zh.md)

Шаблонная стратегия, которую можно подключить к внешним источникам сигналов.
Поддерживает длинные и короткие позиции и использует простой стоп‑лосс и тейк‑профит в процентах.

## Детали

- **Условия входа**: внешний сигнал
- **Лонг/Шорт**: оба направления
- **Условия выхода**: внешний сигнал или стоп‑лосс/тейк‑профит
- **Стопы**: да, в процентах
- **Значения по умолчанию**:
  - `CandleType` = 1 минута
  - `StopLossPercent` = 2%
  - `TakeProfitPercent` = 4%
- **Фильтры**:
  - Категория: Другое
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Начальная
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
