# 資産クラス別トレンドフォロー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は複数の資産クラスにわたってトレンドを追います。ユニバース内の各銘柄に単純移動平均フィルターを適用し、毎月の最初の取引日にポートフォリオをリバランスします。移動平均を上回っているときのみポジションを取ります。

テストでは年平均リターン約15%を示しています。分散されたフューチャーズポートフォリオで最も優れたパフォーマンスを発揮します。

毎月の初めに、SMAを上回って取引されている銘柄に均等なキャピタル配分が行われます。価格がSMAを下回るか、次回のリバランス時に資本が再配分される際にポジションが閉じられます。

## 詳細

- **エントリー条件**: `Close > SMA`
- **ロング/ショート**: ロングのみ
- **エグジット条件**: `Close <= SMA` またはリバランス時に除外
- **ストップ**: なし; 資本は毎月再配分される
- **デフォルト値**:
  - `SmaLength` = 210
  - `MinTradeUsd` = 50
  - `CandleType` = daily
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロングのみ
  - インジケーター: SMA
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 長期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
