# RSIブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSIブレイクアウト戦略は、Relative Strength Index（RSI）が典型的な範囲を超えたときのモメンタムの爆発を探します。RSIの移動平均からの乖離を測定することで、このシステムは新しいトレンドが始まった瞬間を捉えることを目指します。

テストでは平均年間リターンが約88%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

RSIが平均値に`Multiplier`倍の標準偏差を足した値を超えてクローズしたときにロングポジションを開きます。RSIが平均値からその乗数を引いた値を下回ったときにショートポジションを取ります。RSIが平均値に向かって戻ったらポジションを決済します。

モメンタムトレーダーは、定義されたエグジットレベルを維持しながら初期のブレイクアウトを識別するためにこのアプローチが有益だと感じるかもしれません。ストップロスのパーセンテージが突然の反転から保護します。

## 詳細
- **エントリー条件**:
  - **ロング**: RSI > Avg + Multiplier * StdDev
  - **ショート**: RSI < Avg - Multiplier * StdDev
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: RSI < Avg のときに決済
  - **ショート**: RSI > Avg のときに決済
- **ストップ**: あり、パーセントストップロス。
- **デフォルト値**:
  - `RsiPeriod` = 14
  - `AveragePeriod` = 20
  - `Multiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
