# Стратегия BabyShark VWAP
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает полосы вокруг объёмно-взвешенной средней цены (VWAP) и индикатор RSI, рассчитанный по OBV. Длинные сделки открываются при уходе цены ниже нижней полосы и сигнале перепроданности RSI. Короткие позиции появляются, когда цена поднимается выше верхней полосы и RSI указывает на перекупленность.

Стопы основаны на небольшом процентном убытке, а перед повторным входом выдерживается пауза.

## Детали

- **Условия входа**: Пересечение ценой полос с подтверждением RSI.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Возврат к VWAP или срабатывание стопа.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `Length` = 60
  - `RsiLength` = 5
  - `HigherLevel` = 70
  - `LowerLevel` = 30
  - `Cooldown` = 10
  - `StopLossPercent` = 0.6m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Средневозврат
  - Направление: Оба
  - Индикаторы: VWAP, RSI, OBV
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
