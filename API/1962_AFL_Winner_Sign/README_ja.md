# AFL Winner Sign戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

AFL WinnerSignインジケーターに基づく戦略です。出来高加重価格シリーズに二重平滑化ストキャスティクスオシレーターを適用します。高速ストキャスティクスラインが低速ラインを上方クロスするとロングポジションが開かれ、高速ラインが低速ラインを下方クロスするとショートポジションが開かれます。

## 詳細

- **エントリー条件**:
  - ロング: 高速%Kが低速%Dを上方クロス
  - ショート: 高速%Kが低速%Dを下方クロス
- **ロング/ショート**: 両方
- **エグジット条件**: 反対シグナルでポジションを決済または反転
- **ストップ**: `StartProtection`を使用したパーセンテージベース
- **デフォルト値**:
  - `Period` = 10
  - `KPeriod` = 5
  - `DPeriod` = 5
  - `CandleType` = `TimeSpan.FromMinutes(5).TimeFrame()`
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: ストキャスティクスオシレーター
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
