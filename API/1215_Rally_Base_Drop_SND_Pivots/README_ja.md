# Rally Base Drop SND ピボット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Rally Base Drop SND ピボット戦略は、需給ピボットレベルのブレイクアウトを取引します。ピボットは、強気と弱気のローソク足の連続がrally-base-dropまたはdrop-base-rallyパターンを形成するときに検出されます。価格がこれらのピボットレベルをクロスすると、ポジションが建てられます。エグジットにはATRベースのストップとリスクリワードのターゲットを使用します。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格がピボット高値を上抜け（または反転時はピボット安値を上抜け）。
  - **ショート**: 価格がピボット安値を下抜け（または反転時はピボット高値を下抜け）。
- **ロング/ショート**: 設定可能（ロングのみ、ショートのみ、または両方）。
- **エグジット条件**:
  - 価格がATRストップまたはリスクリワードターゲットに到達。
- **ストップ**: ATR倍数とリスクリワードターゲット。
- **デフォルト値**:
  - `Length` = 3
  - `Mult` = 1.0
  - `RiskReward` = 6.0
  - `ReverseConditions` = false
- **フィルター**:
  - カテゴリ: サポート/レジスタンスブレイクアウト
  - 方向: 両方
  - インジケーター: ATR
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
