# Стратегия Rally Base Drop SND Pivots
[English](README.md) | [中文](README_cn.md)

Стратегия Rally Base Drop SND Pivots торгует пробои уровней спроса и предложения. Пивоты определяются по последовательности бычьих и медвежьих свечей, формирующих паттерны rally-base-drop или drop-base-rally. При пересечении ценой таких уровней открывается позиция. Выход выполняется по стопу на основе ATR и цели по соотношению риск/прибыль.

## Детали

- **Условия входа**:
  - **Лонг**: цена пересекает сверху уровень пивота (или снизу при реверсе).
  - **Шорт**: цена пересекает снизу уровень пивота (или сверху при реверсе).
- **Лонг/Шорт**: настраивается (только лонг, только шорт или оба).
- **Условия выхода**:
  - Цена достигает ATR-стопа или цели риск/прибыль.
- **Стопы**: множитель ATR и цель риск/прибыль.
- **Значения по умолчанию**:
  - `Length` = 3
  - `Mult` = 1.0
  - `RiskReward` = 6.0
  - `ReverseConditions` = false
- **Фильтры**:
  - Категория: Пробой уровней поддержки/сопротивления
  - Направление: Оба
  - Индикаторы: ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
