# Kauf WMA Cross-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt, wenn der Kaufman Adaptive Moving Average (KAMA) einen Weighted Moving Average (WMA) kreuzt. Ein Aufwärtskreuz schließt Short-Positionen und eröffnet optional eine Long-Position, wenn erlaubt. Ein Abwärtskreuz schließt Long-Positionen und eröffnet optional eine Short-Position. Das Öffnen und Schließen für jede Seite kann über Parameter aktiviert oder deaktiviert werden.

## Parameter
- Kerzentyp
- AMA-Periode
- Schneller EMA
- Langsamer EMA
- WMA-Periode
- Long öffnen
- Short öffnen
- Long schließen
- Short schließen
