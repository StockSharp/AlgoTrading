# Estratégia Kauf WMA Cross
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera quando a Média Móvel Adaptativa de Kaufman (KAMA) cruza uma Média Móvel Ponderada (WMA). Um cruzamento ascendente fecha posições vendidas e opcionalmente abre uma posição comprada se permitido. Um cruzamento descendente fecha posições compradas e opcionalmente abre uma posição vendida. A abertura e o fechamento de cada lado podem ser habilitados ou desabilitados por parâmetros.

## Parâmetros
- Tipo de vela
- Período AMA
- EMA rápido
- EMA lento
- Período WMA
- Abrir comprado
- Abrir vendido
- Fechar comprado
- Fechar vendido
