# Kauf WMA クロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Kaufman 適応型移動平均（KAMA）が加重移動平均（WMA）とクロスしたときに取引を行います。上向きのクロスはショートポジションを決済し、許可されている場合はロングポジションを開きます。下向きのクロスはロングポジションを決済し、許可されている場合はショートポジションを開きます。各サイドの開始と終了はパラメーターで有効または無効にできます。

## パラメーター
- ローソク足タイプ
- AMA 期間
- 短期 EMA
- 長期 EMA
- WMA 期間
- ロング開始
- ショート開始
- ロング終了
- ショート終了
