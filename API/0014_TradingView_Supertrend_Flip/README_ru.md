# Переключение Supertrend из TradingView
[English](README.md) | [中文](README_cn.md)

Стратегия основана на смене цвета индикатора Supertrend с подтверждением объёма.

Тестирование показывает среднегодичную доходность около 79%\. Стратегию лучше запускать на фондовом рынке.

TradingView Supertrend Flip имитирует изменение цвета популярного индикатора. Смена с красного на зелёный служит сигналом к покупке, а с зелёного на красный — к продаже. Стратегия закрывает позицию при следующем переключении.

Подтверждение объёмом помогает избежать ложных сигналов в периоды низкой ликвидности. Реагируя только на смены, поддержанные объёмом, метод стремится захватывать более надёжные развороты.

## Подробности

- **Критерии входа**: сигналы на основе ATR и Supertrend.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `SupertrendPeriod` = 10
  - `SupertrendMultiplier` = 3.0м
  - `VolumeAvgPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ATR, Supertrend
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
