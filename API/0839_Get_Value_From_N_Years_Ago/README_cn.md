# 获取 N 年前的值
[English](README.md) | [Русский](README_ru.md)

演示如何获取 N 年前K线的时间组成，不包含交易逻辑。

## 细节

- **入场条件**：无
- **方向**：无
- **出场条件**：无
- **止损**：无
- **默认参数**：
  - `YearsBack` = 1
  - `K线类型` = 1 天
- **过滤器**：
  - 类别：时间工具
  - 方向：无
  - 指标：无
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：是
  - 神经网络：无
  - 背离：无
  - 风险水平：低
