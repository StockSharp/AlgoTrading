# 流動性スウィープ・フィルター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このトレンドフォロー戦略は、ボリンジャーバンドを使って市場の方向性を検出し、潜在的な流動性スウィープのために出来高を監視します。選択されたトレードモードに応じてトレンドが強気または弱気に転換したときにポジションをオープンします。

## 詳細

- **エントリー条件**:
  - **ロング**: トレンドが強気に転換し、モードがロングトレードを許可している。
  - **ショート**: トレンドが弱気に転換し、モードがショートトレードを許可している。
- **ロング/ショート**: トレードモードで設定可能。
- **エグジット条件**:
  - **ロング**: トレンドが弱気に転換するか、モードがロングを禁止する。
  - **ショート**: トレンドが強気に転換するか、モードがショートを禁止する。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 12.
  - `Multiplier` = 2.0.
  - `Major Sweep Threshold` = 50.
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
