# Стратегия Liquidity Sweep Filter
[English](README.md) | [中文](README_cn.md)

Эта трендовая стратегия использует полосы Боллинджера для определения направления рынка и контролирует объём для возможных ликвидационных свечей. Позиции открываются при смене тренда на бычий или медвежий в зависимости от режима торговли.

## Детали

- **Условия входа**:
  - **Long**: Тренд становится бычьим и режим позволяет длинные сделки.
  - **Short**: Тренд становится медвежьим и режим позволяет короткие сделки.
- **Направление торговли**: Настраивается.
- **Условия выхода**:
  - **Long**: Тренд становится медвежьим или режим запрещает длинные.
  - **Short**: Тренд становится бычьим или режим запрещает короткие.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `Length` = 12.
  - `Multiplier` = 2.0.
  - `Major Sweep Threshold` = 50.
- **Фильтры**:
  - Категория: Трендовая
  - Направление: Оба
  - Индикаторы: Несколько
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
