# Power Hour Money 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は選択されたニューヨークセッション中に取引し、主要な時間軸すべてが一致したときにポジションを建てます。
月足、週足、日足、時間足のローソク足が始値より高く引けたときにロングポジションを建てます。
すべてが低く引けたときにショートポジションを建てます。
オプションのトレーリングストップが利益を保護し、16:45にポジションを決済することができます。

## 詳細
- **エントリー**: すべての時間軸が緑のときロング、すべて赤のときショート。
- **セッションフィルター**: NY 9:30-11:30、拡張 8:00-16:00 またはすべてのセッション。
- **トレーリングストップ**: ロングとショート両側のパーセントベース。
- **デイエンド**: 16:45にすべてのポジションをオプションで決済。
