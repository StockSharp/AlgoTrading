# Стратегия Combo 123 Reversal и Fractal Chaos Bands
[English](README.md) | [中文](README_cn.md)

Стратегия комбинирует разворот 123 на основе стохастика и пробой полос Fractal Chaos Bands.
Длинная позиция открывается, когда формируется бычий разворот 123 и цена закрывается выше верхней фрактальной полосы.
Короткая позиция открывается, когда медвежий разворот 123 совпадает с закрытием ниже нижней фрактальной полосы.

## Детали

- **Критерии входа**:
  - Long: сигнал Reversal123 на покупку и закрытие выше верхней полосы.
  - Short: сигнал Reversal123 на продажу и закрытие ниже нижней полосы.
- **Long/Short**: Оба
- **Критерии выхода**: Противоположный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Length` = 15
  - `KSmoothing` = 1
  - `DLength` = 3
  - `Level` = 50m
  - `Pattern` = 1
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Паттерн и пробой
  - Направление: Оба
  - Индикаторы: Stochastic Oscillator, Fractal Chaos Bands
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
