# Стратегия Stochastic Z-Score Oscillator
[English](README.md) | [中文](README_cn.md)

Комбинирует масштабированный стохастический осциллятор и Z-Score цены. Сделка открывается при пересечении их среднего порога и закрывается, когда Z-Score возвращается к нулю. Период охлаждения предотвращает частые сигналы.

## Детали

- **Критерий входа**: среднее между стохастиком и Z-Score пересекает порог после периода охлаждения
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: пересечение Z-Score уровня 0
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RollingWindow` = 80
  - `ZThreshold` = 2.8
  - `CoolDown` = 5
  - `StochLength` = 14
  - `StochSmooth` = 7
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Обе стороны
  - Индикаторы: Stochastic, SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
