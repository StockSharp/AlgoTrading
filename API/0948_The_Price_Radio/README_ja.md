# Price Radio戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はJohn EhlersのPrice Radioインジケーターを実装しています。価格の導関数が振幅と周波数の両方のしきい値を超えるとロングエントリーし、その負の値を下回るとショートエントリーします。

## 詳細

- **エントリー条件**:
  - **ロング**: 導関数が振幅と周波数より大きい。
  - **ショート**: 導関数が負の振幅と負の周波数より小さい。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対のシグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 14.
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: Custom
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
