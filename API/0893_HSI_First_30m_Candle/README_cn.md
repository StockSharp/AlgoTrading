# HSI首个30分钟蜡烛策略
[English](README.md) | [Русский](README_ru.md)

该策略在香港市场开盘后的前30分钟记录最高价和最低价，并在5分钟图上交易其突破。每天仅允许一笔交易。

## 细节

- **入场条件**：
  - **多头**：价格在交易时段内突破前30分钟高点。
  - **空头**：价格在交易时段内跌破前30分钟低点。
- **多空方向**：双向。
- **出场条件**：
  - 止损位于区间另一侧。
  - 止盈为区间大小乘以`RiskReward`。
- **止损**：有。
- **默认值**：
  - `RiskReward` = 1。
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame().
- **筛选**：
  - 类型: 突破
  - 方向: 双向
  - 指标: 价格行为
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
