# Стратегия первой 30-минутной свечи HSI
[English](README.md) | [中文](README_cn.md)

Стратегия фиксирует максимум и минимум первых 30 минут после открытия сессии Гонконга и торгует их пробой на 5-минутном графике. Разрешена только одна сделка в день.

## Детали

- **Условия входа**:
  - **Лонг**: цена пробивает максимум первых 30 минут в пределах сессии.
  - **Шорт**: цена опускается ниже минимума первых 30 минут в пределах сессии.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - Стоп-лосс на противоположной границе диапазона.
  - Тейк-профит на расстоянии размера диапазона, умноженного на `RiskReward` от точки входа.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `RiskReward` = 1.
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame().
- **Фильтры**:
  - Категория: Пробой
  - Направление: обе стороны
  - Индикаторы: ценовое действие
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
