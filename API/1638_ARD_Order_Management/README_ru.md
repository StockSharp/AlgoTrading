# Стратегия ARD Order Management
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор DeMarker и открывает позиции при пересечении уровня 0.5.

Когда DeMarker опускается ниже порога после того, как находился выше, стратегия покупает. Когда DeMarker поднимается выше порога после того, как находился ниже, стратегия продаёт. Выход выполняется по противоположному сигналу. Стопов нет.

## Детали

- **Условия входа**:
  - Long: `DeMarker пересекает Threshold сверху вниз`
  - Short: `DeMarker пересекает Threshold снизу вверх`
- **Long/Short**: Оба направления
- **Условия выхода**: Противоположный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `DeMarkerPeriod` = 2
  - `Threshold` = 0.5
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Индикаторы
  - Направление: Оба
  - Индикаторы: DeMarker
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
