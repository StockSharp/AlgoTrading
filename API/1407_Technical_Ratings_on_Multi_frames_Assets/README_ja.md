# マルチタイムフレーム資産のテクニカルレーティング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は複数の時間軸からテクニカルレーティングを集計します。
1h、4h、日足ロウソク足で価格を移動平均とRSIの閾値と比較します。
統合レーティングがプラスのときロングポジションを、マイナスのときショートポジションを開きます。

## 詳細

- **エントリー**: 平均レーティング > 0 で買い；平均レーティング < 0 で売り。
- **インジケーター**: SMA, RSI。
- **時間軸**: 1h, 4h, 1d。
- **タイプ**: トレンドフォロー。
- **ストップ**: なし。
- **方向**: ロングとショート。
- **リスク**: 中。
- **複雑さ**: 中。
