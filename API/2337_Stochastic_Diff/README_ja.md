# Stochastic 差分戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はStochasticオシレーターの%Kラインと%Dラインの差に基づいてトレードします。差はノイズを減らすために指数移動平均で平滑化されます。平滑化された差が局所的な谷を形成して上昇に転じたとき、ロングポジションを建てます。平滑化された差が局所的な山を形成して下落に転じたとき、ショートポジションを建てます。

## 仕組み

1. ユーザー定義の期間でStochasticの%Kと%Dを計算します。
2. 差 `%K - %D` を計算し、EMAで平滑化します。
3. 平滑化された差の転換点を検出します：
   - 値が下降していて上昇に転じた場合、ロングポジションを建てます。
   - 値が上昇していて下落に転じた場合、ショートポジションを建てます。
4. オプションのパーセント単位のストップロスとテイクプロフィットを適用します。

## パラメーター

| 名前 | 説明 |
| --- | --- |
| Candle Type | 計算に使用するローソク足の種類 |
| %K Period | %Kラインの期間 |
| %D Period | %Dラインの期間 |
| Slowing | %Kの追加平滑化 |
| Smoothing Length | 差のEMA長さ |
| Stop Loss % | ストップロスのサイズ（パーセント） |
| Take Profit % | テイクプロフィットのサイズ（パーセント） |

## 注意事項

- データフィードがサポートする任意の銘柄と時間軸で動作します。
