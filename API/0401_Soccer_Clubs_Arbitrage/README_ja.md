# サッカークラブ裁定取引戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、複数の取引所で取引されるサッカークラブのファントークン間の裁定取引機会を追求します。価格スプレッドとファンディングレートの不均衡を監視することで、価格の歪みを捉えるためにロングとショートのオフセットポジションを建てます。

取引所間のスプレッドが閾値を超えると取引がトリガーされます。価格が収束するか、保護的なストップに達した場合にポジションがヘッジされ、決済されます。

## 詳細

- **データ**: ファントークンの価格とファンディングレート。
- **エントリー**: スプレッド > X% のときに逆方向ポジションを建てる。
- **エグジット**: スプレッド < Y% またはタイムストップ時にクローズ。
- **銘柄**: 取引所上場のファントークン。
- **リスク**: スリッページ対策のための固定パーセントストップ。
