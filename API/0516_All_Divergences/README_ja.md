# 全ダイバージェンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

全ダイバージェンス戦略は、移動平均トレンドでフィルタリングされた RSI の強気・弱気ダイバージェンスを探します。価格が安値を更新しながら RSI が移動平均の上で高い安値を形成するときにロングポジションを開きます。価格が高値を更新しながら RSI が移動平均の下で低い高値を形成するときにショートポジションを開きます。オプションのストップロスとテイクプロフィット保護がポジションを自動的に決済し、移動平均リスクコントロールがトレンドに逆らった複数の終値の後に決済します。

## 詳細

- **エントリー条件**:
  - 移動平均に対する価格の位置がトレンドを定義します。
  - **ロング**: 価格が安値を更新、RSI が高い安値、価格が MA の上。
  - **ショート**: 価格が高値を更新、RSI が低い高値、価格が MA の下。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆シグナルまたは MA リスク退出。
- **ストップ**: オプションのストップロスとテイクプロフィット。
- **デフォルト値**:
  - `MaLength` = 50
  - `RsiLength` = 14
  - `MaRiskCandles` = 3
  - `UseProtection` = False
- **フィルター**:
  - カテゴリ: ダイバージェンス
  - 方向: 両方
  - インジケーター: RSI, Moving Average
  - ストップ: オプション
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
