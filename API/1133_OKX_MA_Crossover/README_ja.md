# OKX MA クロスオーバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は単純移動平均線のクロスオーバーを取引します。ローソク足の安値が前の MA 値を下回ったときにロングポジションを建てます。ローソク足の高値が前の MA 値を上回ったときにショートポジションを建てます。テイクプロフィットとストップロスはエントリー価格からの割合として適用されます。取引は指定された開始日以降にのみ開始されます。

## パラメーター
- ローソク足の種類
- MA の長さ
- テイクプロフィット %
- ストップロス %
- 開始日
