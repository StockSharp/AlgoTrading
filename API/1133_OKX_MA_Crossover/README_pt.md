# Estratégia OKX MA Crossover
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera um cruzamento de média móvel simples. Uma posição comprada é aberta quando a mínima do candle cai abaixo do valor anterior da MA. Uma posição vendida é aberta quando a máxima do candle sobe acima do valor anterior da MA. Take profit e stop loss são aplicados como percentuais do preço de entrada. A negociação começa apenas após a data de início especificada.

## Parâmetros
- Tipo de candle
- Comprimento da MA
- Take Profit %
- Stop Loss %
- Data de início
