# OKX MA Crossover-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt einen einfachen gleitenden Durchschnitt-Crossover. Eine Long-Position wird eröffnet, wenn das Tief der Kerze unter den vorherigen MA-Wert fällt. Eine Short-Position wird eröffnet, wenn das Hoch der Kerze über den vorherigen MA-Wert steigt. Take-Profit und Stop-Loss werden als Prozentsätze des Einstiegspreises angewendet. Der Handel beginnt erst nach dem angegebenen Startdatum.

## Parameter
- Kerzentyp
- MA-Länge
- Take Profit %
- Stop Loss %
- Startdatum
