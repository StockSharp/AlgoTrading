# ZapTeam Pro v6 — EMA 策略
[English](README.md) | [Русский](README_ru.md)

简化版策略，使用 EMA21/EMA50 交叉并结合 EMA200 趋势过滤。当出现看涨交叉时买入，看跌交叉时卖出（可选做空）。

## 细节

- **入场条件**: EMA21 与 EMA50 交叉并通过趋势过滤
- **多空**: 双向（可选做空）
- **出场条件**: 反向交叉
- **止损**: 无
- **默认值**:
  - `Ema21Length` = 21
  - `Ema50Length` = 50
  - `Ema200Length` = 200
- **筛选**:
  - 类别: 趋势
  - 方向: 双向
  - 指标: EMA
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中等
