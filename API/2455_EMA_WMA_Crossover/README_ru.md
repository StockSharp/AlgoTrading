# Стратегия EMA WMA Crossover
[English](README.md) | [中文](README_cn.md)

Стратегия основана на пересечении экспоненциальной скользящей средней (EMA) и взвешенной скользящей средней (WMA), рассчитанных по ценам открытия свечей.
Вход в длинную позицию происходит, когда EMA пересекает WMA сверху вниз.
Вход в короткую позицию происходит, когда EMA пересекает WMA снизу вверх.
Размер позиции определяется процентом капитала, находящегося под риском.
Стратегия использует фиксированные уровни тейк-профита и стоп-лосса, заданные в тиках.

## Подробности

- **Условия входа**:
  - Длинная: `EMA crosses below WMA`
  - Короткая: `EMA crosses above WMA`
- **Long/Short**: Оба
- **Условия выхода**: стоп-лосс или тейк-профит
- **Стопы**: Да
- **Параметры по умолчанию**:
  - `EmaPeriod` = 28
  - `WmaPeriod` = 8
  - `StopLossTicks` = 50
  - `TakeProfitTicks` = 50
  - `RiskPercent` = 10
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: пересечение скользящих средних
  - Направление: Оба
  - Индикаторы: EMA, WMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
