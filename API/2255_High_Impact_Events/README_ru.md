# Стратегия отслеживания важных новостей
[English](README.md) | [中文](README_cn.md)

Стратегия загружает экономический календарь с сайта ForexFactory и предупреждает трейдера перед выходом новостей высокого влияния. Это помогает избегать открытия новых позиций непосредственно перед событиями, способными вызвать резкие движения цены.

После запуска стратегия скачивает календарь на текущий день и сохраняет только записи с высоким уровнем важности. Таймер каждую минуту проверяет список событий и сравнивает текущее время с временем новости. Если до события остается меньше указанного интервала, в лог выводится информационное сообщение.

Стратегия не совершает сделок и предназначена как вспомогательный элемент управления рисками. Её можно использовать совместно с другими алгоритмами для остановки торговли или корректировки позиций перед выходом значимых новостей.

## Параметры

- `AlertBeforeMinutes` – за сколько минут до события показывать предупреждение. Значение по умолчанию `5`.

## Последовательность работы

1. Загрузка календаря на текущий день с ForexFactory.
2. Отбор строк, содержащих признак `calendar__impact--high`.
3. Сохранение времени, валюты и названия события.
4. Проверка списка каждую минуту.
5. Вывод сообщения вида `GDP (USD) через 5 минут.` при достижении порога.

Для корректной работы необходимо разрешить сетевой доступ. В случае ошибки загрузки календаря сообщение об ошибке будет записано в журнал.
