# Donchian Volume 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
Donchian Volume は、出来高の増加で確認されたドンチャンチャネルのブレイクアウトを使ってトレードを開始します。
強い出来高を伴うチャネル外への動きは、新たなトレンドの始まりを示唆します。

テストでは平均年間リターンは約160%を示しています。外国為替市場で最も良いパフォーマンスを発揮します。

この戦略はブレイクアウトの方向にエントリーし、価格がチャネル内に戻ってクローズするか出来高が衰えたときにイグジットします。

ダマシの動きから保護するため、ストップはチャネル内のすぐ内側に設定されます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Donchian Channel, Volume
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
