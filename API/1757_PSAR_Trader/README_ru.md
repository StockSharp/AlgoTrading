# PSAR Trader
[English](README.md) | [中文](README_cn.md)

Стратегия на основе индикатора Parabolic SAR. PSAR Trader следует точкам индикатора и реагирует, когда цена пересекает индикатор. Открывает длинную позицию, когда цена поднимается выше SAR, и короткую, когда опускается ниже. Торговля может быть ограничена временным диапазоном, а при появлении противоположного сигнала позиция может закрываться. Стратегия также использует тейк-профит и стоп-лосс, измеряемые в тиках.

## Детали

- **Критерии входа**: Пересечение цены и Parabolic SAR.
- **Длинные/короткие**: Оба направления.
- **Критерии выхода**: Противоположный сигнал (опционально), стоп-лосс или тейк-профит.
- **Стопы**: Тейк-профит и стоп-лосс в тиках.
- **Значения по умолчанию**:
  - `Step` = 0.001m
  - `Maximum` = 0.2m
  - `TakeProfitTicks` = 50
  - `StopLossTicks` = 50
  - `StartHour` = 0
  - `EndHour` = 23
  - `CloseOnOpposite` = true
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Parabolic SAR
  - Стопы: Тейк-профит, Стоп-лосс
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
