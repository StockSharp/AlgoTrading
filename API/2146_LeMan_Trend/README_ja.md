# LeManトレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

LeManトレンド戦略は、直近の高値と安値から強気・弱気の圧力を導き出します。現在のローソク足と3つの異なるルックバック期間における最高値・最安値との距離を計算します。これらの距離は指数移動平均（EMA）で平滑化され、2本のライン（強気線と弱気線）を形成します。これらのラインのクロスオーバーは潜在的なトレンド転換を示します。

強気線が弱気線を上回ってクロスすると、戦略はロングポジションを開くか、既存のショートポジションを決済します。逆に、弱気線が強気線を上回ると、ショートポジションを開くかロングポジションを終了します。この手法は追加フィルターを使用せず、直近の高値と安値の相対的な強さのみに焦点を当てています。

## 詳細

- **エントリー条件**
  - **ロング**: 強気線が弱気線を上抜け。
  - **ショート**: 弱気線が強気線を上抜け。
- **ロング/ショート**: 両方向対応。
- **エグジット条件**
  - 反対方向のクロスオーバーで現在のポジションを決済。
- **ストップ**: デフォルトなし。
- **デフォルト値**
  - `Min` = 13
  - `Midle` = 21
  - `Max` = 34
  - `EMA period` = 3
  - `Time frame` = 4 hours
- **フィルター**
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Highest, Lowest, EMA
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
