# Полосы Боллинджера с трейлинг-стопом
[English](README.md) | [中文](README_cn.md)

Вход в длинную позицию происходит, когда цена закрытия выше верхней полосы Боллинджера.
Выход – при закрытии ниже нижней полосы или срабатывании трейлинг-стопа на основе ATR.

## Подробности

- **Условия входа**: закрытие выше верхней полосы.
- **Длинные/короткие позиции**: только лонг.
- **Условия выхода**: закрытие ниже нижней полосы или срабатывание трейлинг-стопа.
- **Стопы**: трейлинг-стоп.
- **Значения по умолчанию**:
  - `BbLength` = 20
  - `BbDeviation` = 2m
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `MaType` = MovingAverageTypeEnum.Simple
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Trend
  - Направление: Long
  - Индикаторы: Bollinger Bands, ATR
  - Стопы: Да
  - Сложность: Beginner
  - Таймфрейм: Intraday
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
