# MACD 直方图反转策略
[English](README.md) | [Русский](README_ru.md)

MACD 直方图表示 MACD 线与信号线的差值。当直方图穿越零轴时常意味着动能转向。本策略在这些穿越发生时入场，并以百分比止损控制风险。

每根K线计算 MACD 直方图。当其从负值转为正值时做多；从正值转为负值时做空。由于仅关注零轴交叉，交易规则简单且多为短线。

若动能未按预期延续，止损用于限制亏损。

## 细节

- **入场条件**：MACD 直方图穿越零轴。
- **多/空**：双向。
- **退出条件**：止损。
- **止损**：有，按百分比。
- **默认值**：
  - `FastPeriod` = 12
  - `SlowPeriod` = 26
  - `SignalPeriod` = 9
  - `StopLoss` = 2%
  - `CandleType` = 15 分钟
- **过滤条件**：
  - 类别: 动量
  - 方向: 双向
  - 指标: MACD
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
