# Стратегия разворота гистограммы MACD

Гистограмма MACD отображает разницу между линией MACD и сигнальной линией. Пересечения нулевой отметки сверху или снизу часто указывают на смену импульса. Эта стратегия торгует такие пересечения и управляет риском с помощью процентного стопа.

На каждой свече рассчитывается гистограмма MACD. Когда значение переходит из отрицательной области в положительную, открывается длинная позиция. Пересечение из плюса в минус запускает короткую продажу. Поскольку стратегия отслеживает только переход через ноль, сделки просты и обычно краткосрочны.

Стопы используются для ограничения убытков, если импульс не продолжает движение в ожидаемом направлении.

## Детали

- **Условия входа**: гистограмма MACD пересекает ноль.
- **Лонг/шорт**: оба направления.
- **Условия выхода**: стоп-лосс.
- **Стопы**: да, на процентной основе.
- **Значения по умолчанию**:
  - `FastPeriod` = 12
  - `SlowPeriod` = 26
  - `SignalPeriod` = 9
  - `StopLoss` = 2%
  - `CandleType` = 15 минут
- **Фильтры**:
  - Категория: Импульс
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
