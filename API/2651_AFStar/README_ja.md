# AFStar戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

AFStar戦略は、広範な高速/低速EMAクロスオーバーとWilliams %Rチャネルブレイクアウト
フィルターを組み合わせることで、短期的なモメンタムの転換を検出します。両方のコンポーネントが
一致した場合にのみ、実行可能なシグナルが生成されます。

設定区間内の少なくとも1つの高速EMAが互換性のある低速EMAを上抜けし、かつWilliams %R
ベースのオシレーターが中立ゾーン内に留まった後に下限バンドから抜け出した場合に買いシグナル
が生成されます。売りシグナルは、弱気クロスオーバーと上限バンドからの離脱という対称的な条件
で生成されます。シグナルは**Signal Bar**パラメーターで定義されたバー数の後に実行され、元の
MetaTraderエキスパートと同様の動作をします。

ポジションがオープンされると、価格ステップで表現された保護的なストップロスとテイクプロフィット
レベルをオプションで設定できます。これらの保護は毎回の確定足で確認されます。すべての取引は
一定の**Order Volume**パラメーターを使用するため、MQL5バージョンの複雑な資金管理ルールは
よりシンプルな固定サイズのアプローチに置き換えられています。

## エントリーロジック

- **ロング:**
  - `[Start Fast, End Fast]`内の少なくとも1つの高速EMAが、`Step Period`の増分を使用して
    `[Start Slow, End Slow]`内の低速EMAを上抜けする。
  - `[Start Risk, End Risk]`範囲のリスク値と`Risk Step`で評価されるWilliams %Rチャネルが、
    中立バンド内に留まった後に上限を上方ブレイクアウトするのを検出する。
  - **Enable Sell Exits**がオンの場合、オプションのショートポジションが事前にクローズされる。
- **ショート:**
  - 反対方向の対称的なクロスオーバーとWilliams %Rブレイクアウト。
  - **Enable Buy Exits**が有効な場合、オプションのロングエグジットが最初に発生する。

## エグジットロジック

- 対応するエグジットフラグが有効な場合、反対方向のシグナルがポジションをクローズする
  （買いシグナルがショートをクローズ、売りシグナルがロングをクローズ）。
- 価格ステップで測定されたオプションのストップロスとテイクプロフィットレベルにより、
  価格がそれらの閾値に達した場合、ポジションが早期にクローズされることがある。

## パラメーター

- **Order Volume** – 成行注文に使用される取引サイズ。
- **Candle Type** – 市場データの時間軸（デフォルト: 4時間足）。
- **Start Fast / End Fast / Step Period** – クロスオーバースキャンの高速EMAレンジ。
- **Start Slow / End Slow** – 高速EMA値とペアになる低速EMAレンジ。
- **Start Risk / End Risk / Risk Step** – Williams %Rリスクスキャンの境界。
- **Signal Bar** – シグナルを実行する前に待機する完了バーの数。
- **Stop Loss (pips)** – オプションのストップロス距離（価格ステップ）。
- **Take Profit (pips)** – オプションのテイクプロフィット距離（価格ステップ）。
- **Enable Buy Entries / Enable Sell Entries** – ロングまたはショートエントリーを許可する。
- **Enable Buy Exits / Enable Sell Exits** – 反対方向でのクローズを有効にする。

## 注意事項

- 戦略はAFStarロジックを評価するために最新の512本の足を保持します。
- 有価証券の価格ステップが利用できない場合、ストップロスとテイクプロフィットの距離を
  計算する際に値1が使用されます。
- シグナルはキューに入れられるため、**Signal Bar = 0**は即座に実行され、より大きな値は
  その数の完了バーだけ実行を遅延させます。
