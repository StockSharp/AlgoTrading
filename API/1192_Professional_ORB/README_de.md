# Professionelle ORB-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Implementiert eine Opening Range Breakout-Strategie. Das Hoch und Tief zwischen 09:15 und einer konfigurierbaren Dauer bilden die Range. Nachdem die Range abgeschlossen und breit genug ist, lösen Ausbrüche nach oben oder unten Long- bzw. Short-Einstiege aus. Positionen verwenden einen ATR-basierten Stop-Loss, ein festes Gewinnziel in Punkten und werden am Sitzungsende geschlossen. Die Anzahl der Trades pro Tag ist begrenzt.
