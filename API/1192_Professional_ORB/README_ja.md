# プロフェッショナルORB戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

オープニングレンジブレイクアウト戦略を実装します。09:15から設定可能な時間までの高値と安値がレンジを形成します。レンジが完成し十分な幅があれば、上方または下方へのブレイクアウトによりロングまたはショートエントリーが発動します。ポジションはATRベースのストップロス、固定のポイント利確目標を使用し、セッション終了時にクローズします。1日あたりの取引回数は制限されています。
