# Стратегия Parabolic SAR CCI

Эта стратегия задействует индикаторы Parabolic SAR и CCI. Длинная позиция открывается, когда цена находится выше уровня SAR и CCI опускается ниже -100, что показывает восходящий тренд при перепроданности. Короткая позиция открывается, когда цена ниже SAR и CCI выше 100, то есть нисходящий тренд и перекупленность. Стратегия пригодна для торговли на смешанных рынках.

## Подробности
- **Условия входа**:
  - **Long**: Price > SAR && CCI < -100
  - **Short**: Price < SAR && CCI > 100
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход при падении цены ниже SAR
  - **Short**: выход при росте цены выше SAR
- **Стопы**: нет
- **Значения по умолчанию**:
  - `SarAccelerationFactor` = 0.02m
  - `SarMaxAccelerationFactor` = 0.2m
  - `CciPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: Parabolic SAR CCI
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
