# Прорыв по OBV

Индикатор On-Balance Volume (OBV) отслеживает давление покупателей и продавцов, накапливая объём. Стратегия ищет выход OBV выше максимума или ниже минимума за период, при подтверждении ценой.

Прорыв OBV говорит о сильном интересе. Система покупает, если OBV превышает прежний максимум, или продаёт, если пробивает минимум. Пересечение скользящей средней OBV служит сигналом выхода.

Метод сочетает импульс объёма с ценовым движением.

## Подробности

- **Критерий входа**: OBV выходит за максимум или минимум в периоде отслеживания.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: OBV пересекает свою MA или стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `LookbackPeriod` = 20
  - `OBVMAPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе стороны
  - Индикаторы: OBV, MA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
