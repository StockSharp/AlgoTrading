# Стратегия Zahorchak Measure
[English](README.md) | [中文](README_cn.md)

Вычисляет взвешенный показатель на основе нескольких скользящих средних. Покупает при переходе показателя выше нуля и продаёт при переходе ниже.

## Детали

- **Условие входа**: Показатель переходит выше нуля
- **Длин/Корот**: Оба направления
- **Условие выхода**: Обратный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Points` = 1
  - `EmaLength` = 10
- **Фильтры**:
  - Категория: Рыночная ширина
  - Направление: Оба
  - Индикаторы: SMA, EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
