# MACD Паттерн Трейдер
[English](README.md) | [中文](README_cn.md)

Стратегия открывает сделки на резких разворотах MACD. Она ищет два крупных импульса вокруг малого промежуточного значения линии MACD. Продажа совершается, когда прошлое значение MACD положительное, а текущее глубоко уходит в отрицательную область. Покупка выполняется при обратном условии. Стоп-лосс и тейк-профит рассчитываются по недавним максимумам и минимумам.

Алгоритм подходит для волатильных рынков, где импульс быстро меняет направление. Используются только рыночные ордера, риск рассчитывается на основе истории свечей.

## Детали

- **Критерии входа**: отношение импульсов MACD на основе `RatioThreshold`.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: стоп по ближайшему экстремуму с отступом или противоположный импульс.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `FastEmaPeriod` = 24
  - `SlowEmaPeriod` = 13
  - `StopLossBars` = 22
  - `TakeProfitBars` = 32
  - `OffsetPoints` = 40
  - `RatioThreshold` = 5m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
