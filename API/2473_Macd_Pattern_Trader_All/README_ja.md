# MACDパターントレーダー戦略 (All)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MACD の急激なリバーサル時にポジションを建てる戦略です。MACD ラインの小さな中間値の前後にある 2 つの大きなスパイクを探します。前回の MACD 値が正で現在の値が深くマイナス圏に落ち込んだ場合に売りを建て、逆の条件で買いを建てます。ストップロスとテイクプロフィットは直近の高値・安値から算出されます。

このアルゴリズムはモメンタムが素早く方向を変えるボラティリティの高い市場に適しています。成行注文のみを使用し、ローソク足の履歴からリスクレベルを計算します。

## 詳細

- **エントリー条件**: `RatioThreshold` に基づく MACD スパイク比率。
- **ロング/ショート**: 両方。
- **エグジット条件**: 直近の極値プラスオフセットでのストップ、または反対方向のスパイク。
- **ストップ**: あり。
- **デフォルト値**:
  - `FastEmaPeriod` = 24
  - `SlowEmaPeriod` = 13
  - `StopLossBars` = 22
  - `TakeProfitBars` = 32
  - `OffsetPoints` = 40
  - `RatioThreshold` = 5m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: MACD
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
