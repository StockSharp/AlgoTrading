# Exp X2MA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Exp X2MA戦略は、二重平滑化移動平均の転換点を取引します。
価格はまず単純移動平均で平滑化され、次にJurik移動平均で平滑化されます。
平滑化ラインが局所的な最小値を形成すると、戦略は買いを行い、ショートポジションをクローズします。
局所的な最大値を形成すると、戦略は売りを行い、ロングポジションをクローズします。
オプションの固定ストップロスとテイクプロフィットがオープンポジションを保護します。

## 詳細
- **データ**: 価格ローソク足（デフォルト4時間）。
- **エントリー条件**:
  - **ロング**: 直前のX2MA値が古い値より低く、現在の値が上向きに転換する。
  - **ショート**: 直前のX2MA値が古い値より高く、現在の値が下向きに転換する。
- **エグジット条件**: 反対の極値、ストップロスまたはテイクプロフィット。
- **ストップ**: 固定ストップロスとテイクプロフィット（ポイント単位）。
- **デフォルト値**:
  - `FirstMaLength` = 12
  - `SecondMaLength` = 5
  - `StopLossPoints` = 1000
  - `TakeProfitPoints` = 2000
- **フィルター**:
  - カテゴリ: トレンドリバーサル
  - 方向: ロングおよびショート
  - インジケーター: SMA, JurikMovingAverage
  - ストップ: はい
  - 複雑さ: 低
  - リスクレベル: 中
