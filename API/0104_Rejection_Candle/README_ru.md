# Стратегия Rejection Candle

Свеча-отказ возникает, когда цена тестирует уровень, но не может закрепиться за ним, оставляя длинную тень и небольшое тело.
Такие свечи показывают, что попытка движения была решительно отвергнута рынком.

Стратегия входит в противоположном направлении тени после закрытия свечи, ожидая разворота цены обратно в диапазон.

Стоп ставится за пределами отвергнутого максимума или минимума, ограничивая риск; сделка закрывается, если импульс не развивается.

## Детали

- **Критерий входа**: совпадение паттерна
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
