# リジェクション・キャンドル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
リジェクション・キャンドルは、価格があるレベルを探っても、そこを超えて維持できなかったときに形成され、長いヒゲと小さな実体を残します。
このようなローソク足は、ある方向へ動こうとした試みが市場に断固として拒否されたことを示します。

テストでは年平均リターンが約49%であることが示されています。暗号資産市場で最も良いパフォーマンスを発揮します。

この戦略は、ローソク足が引けた後、ヒゲの反対方向にエントリーし、価格がレンジ内に戻るリバーサルを期待します。

ストップは拒絶された高値または安値の外側に置かれてリスクを抑制し、モメンタムが現れなければトレードを終了します。

## 詳細

- **エントリー条件**: パターン一致
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
