# レンジ拡張インデックス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はTom DeMarkの**レンジ拡張インデックス（REI）**を使用して、価格の強弱を評価します。インジケーターは現在の高値・安値を過去の価格と比較し、正と負の値の間で振動します。

## 仕組み

- REIが**下限レベル**（デフォルト `-60`）を下回った後に上抜けすると、戦略は**ロング**ポジションをオープンします。
- REIが**上限レベル**（デフォルト `60`）を上回った後に下抜けすると、戦略は**ショート**ポジションをオープンします。
- 逆のシグナルが発生すると、反対側のポジションは自動的にクローズされます。

## パラメーター

- `REI Period` – REI計算に使用するバー数（デフォルト `8`）。
- `Up Level` – 下方向にクロスすると価格の弱さを示す上限しきい値（デフォルト `60`）。
- `Down Level` – 上方向にクロスすると価格の強さを示す下限しきい値（デフォルト `-60`）。
- `Candle Type` – インジケーター計算用のローソク足の時間軸（デフォルト `8時間`）。

## 使用方法

戦略を銘柄に紐付けて開始します。戦略は指定されたローソク足シリーズを購読し、REIシグナルに基づいて成行注文でポジションの出入りを行います。
