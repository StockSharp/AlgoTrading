# Directional Index K xDMI-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet die Differenz zwischen +DI und -DI des Average Directional Index, um die Trendrichtung zu beurteilen. Wenn der Richtungsindex über das Schlüsselniveau steigt, kauft die Strategie; wenn er unter das negative Schlüsselniveau fällt, verkauft sie. Positionen werden geschlossen, wenn der Absolutwert des Index wieder unter den Schwellenwert fällt.
