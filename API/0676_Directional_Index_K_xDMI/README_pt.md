# Estratégia de Índice Direcional K xDMI
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia utiliza a diferença entre +DI e -DI do Índice Direcional Médio para avaliar a direção da tendência. Quando o índice direcional sobe acima do nível-chave, a estratégia compra; quando cai abaixo do nível negativo chave, vende. As posições são fechadas quando o valor absoluto do índice volta a cair abaixo do limiar.
