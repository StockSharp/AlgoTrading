# Estrategia de Índice Direccional K xDMI
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza la diferencia entre +DI y -DI del Índice Direccional Medio para medir la dirección de la tendencia. Cuando el índice direccional sube por encima del nivel clave, la estrategia compra; cuando cae por debajo del nivel negativo clave, vende. Las posiciones se cierran cuando el valor absoluto del índice vuelve a caer por debajo del umbral.
