# Full Candle戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Full Candleの設定は、ローソク足がEMAを超えて引け、ブレイクアウト側に小さなヒゲのみを残したときにエントリーします。目的は、大きな拒絶なく決定的な動きを示すモメンタムキャンドルを取引することです。オプションのパーセンテージベースのテイクプロフィットとストップロスが、ポジションオープン後のトレードを管理します。

このシステムは、強いローソク足がしばしば素早いフォロースルーをもたらす短期ブレイクアウトに最も適しています。

## 詳細

- **エントリー条件**:
  - **ロング**: 陽線がEMAの上で引け、ヒゲ ≤ 閾値
  - **ショート**: 陰線がEMAの下で引け、ヒゲ ≤ 閾値
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 有効な場合、テイクプロフィットまたはストップロスのパーセンテージ
- **ストップ**: オプション
- **デフォルト値**:
  - `EmaLength` = 10
  - `ShadowPercent` = 5
  - `TPPercent` = 1.2
  - `SLPercent` = 1.8
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: EMA, price action
  - ストップ: オプション
  - 複雑さ: 低
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
