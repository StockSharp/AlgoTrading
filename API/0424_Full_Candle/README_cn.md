# Full Candle 策略
[English](README.md) | [Русский](README_ru.md)

当蜡烛收盘突破 EMA 且突破侧仅留下很小的影线时触发入场，旨在捕捉带有明确动力的K线。可选的百分比止盈与止损用于在持仓期间管理风险。

该系统适用于短期突破，强势K线往往会带来快速跟随。

## 详情

- **入场条件**:
  - **多头**: 看涨K线收在 EMA 之上且影线 ≤ 阈值
  - **空头**: 看跌K线收在 EMA 之下且影线 ≤ 阈值
- **多空方向**: 双向
- **退出条件**:
  - 若启用，则按百分比止盈或止损
- **止损**: 可选
- **默认值**:
  - `EmaLength` = 10
  - `ShadowPercent` = 5
  - `TPPercent` = 1.2
  - `SLPercent` = 1.8
- **过滤器**:
  - 类型: 突破
  - 方向: 双向
  - 指标: EMA, 价格行为
  - 止损: 可选
  - 复杂度: 低
  - 时间框架: 短期
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
