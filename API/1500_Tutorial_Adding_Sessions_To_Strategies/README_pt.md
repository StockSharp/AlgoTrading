# Tutorial - Adicionando sessões às estratégias
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia baseada em RSI que opera apenas durante uma sessão configurável. Entra comprado quando o RSI cruza acima do nível inferior, entra vendido quando cruza abaixo do nível superior. As posições são encerradas quando o mercado sai da sessão.
