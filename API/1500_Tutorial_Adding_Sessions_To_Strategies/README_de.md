# Tutorial - Sitzungen zu Strategien hinzufügen
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

RSI-basierte Strategie, die nur während einer konfigurierbaren Sitzung handelt. Long-Einstieg, wenn der RSI den unteren Level nach oben kreuzt; Short-Einstieg, wenn er den oberen Level nach unten kreuzt. Positionen werden geschlossen, wenn der Markt die Sitzung verlässt.
