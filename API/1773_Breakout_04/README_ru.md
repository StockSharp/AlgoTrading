# Стратегия Breakout 04
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробой диапазона предыдущего дня.
Покупает при выходе цены выше максимума прошлого дня и продаёт при падении ниже минимума.
Использует трейлинг-стоп и фиксированный тейк-профит, а также может рассчитывать объём позиции от капитала.
Торговля отключается до заданного часа понедельника и после указанного часа пятницы.

## Подробности

- **Условия входа**:
  - Длинная: `Цена > Предыдущий максимум`
  - Короткая: `Цена < Предыдущий минимум`
- **Long/Short**: Оба
- **Условия выхода**: трейлинг-стоп или тейк-профит
- **Стопы**: трейлинг и фиксированный стоп-лосс
- **Параметры по умолчанию**:
  - `MondayHour` = 18
  - `FridayHour` = 14
  - `TrailingStop` = 21
  - `TakeProfit` = 550
  - `StopLoss` = 124
  - `UseMoneyManagement` = false
  - `PercentMM` = 8m
  - `Volume` = 0.1m
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
