# Breakout 04 策略 (中文)
[English](README.md) | [Русский](README_ru.md)

该策略交易前一日区间的突破。
当价格突破前一天最高价时做多，跌破前一天最低价时做空。
使用追踪止损和固定止盈，并可根据账户资金调整下单量。
在周一设定时间之前和周五设定时间之后停止交易。

## 细节

- **入场条件**:
  - 多头: `价格 > 前高`
  - 空头: `价格 < 前低`
- **多空**: 双向
- **出场条件**: 追踪止损或止盈
- **止损**: 追踪止损与固定止损
- **默认参数**:
  - `MondayHour` = 18
  - `FridayHour` = 14
  - `TrailingStop` = 21
  - `TakeProfit` = 550
  - `StopLoss` = 124
  - `UseMoneyManagement` = false
  - `PercentMM` = 8m
  - `Volume` = 0.1m
- **过滤器**:
  - 分类: Breakout
  - 方向: 双向
  - 指标: 无
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中等
