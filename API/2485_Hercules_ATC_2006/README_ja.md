# Hercules A.T.C. 2006戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

Hercules A.T.C. 2006は、2006年に公開されたMetaTraderエキスパートアドバイザーを再現する高時間軸のトレンドフォロー戦略です。
StockSharpバージョンは、プライマリ時間軸で完了したローソク足を監視し、高速EMA(1)と低速SMA(72)の強気/弱気クロスオーバーを
検出し、追加フィルターがブレイクアウトを確認した場合にのみトレードを開始します。戦略は独立したテイクプロフィットレベルを
持つ2つのトランシェにポジションを分割し、価格が進むとストップをトレーリングします。

## インジケーターとデータ

- **プライマリローソク足:** 設定可能（デフォルト: 1時間ローソク足）。
- **高速MA:** 長さ `FastMaPeriod` のEMA（デフォルト: 1）。
- **低速MA:** 長さ `SlowMaPeriod` のSMA（デフォルト: 72）。
- **RSIフィルター:** `RsiTimeFrame`（デフォルト: 1時間）上の長さ `RsiLength` のRSI。
- **日次エンベロープ:** `DailyEnvelopeTimeFrame` 上の長さ `DailyEnvelopePeriod` のSMA、
  ±`DailyEnvelopeDeviation` パーセントのオフセット付き。
- **H4エンベロープ:** `H4EnvelopeTimeFrame` 上の長さ `H4EnvelopePeriod` のSMA、
  ±`H4EnvelopeDeviation` パーセントのオフセット付き。
- **ローリング高値/安値:** プライマリ時間軸での過去 `HighLowHours` 時間の最高値と最安値。

## 取引ルール

1. **クロスオーバー検出**
   - 最後の3つの完了したバーのEMA(1)とSMA(72)の値を監視します。
   - EMAが前2バーのいずれかでSMAを上回るクロスをした場合、強気シグナルを検出。
   - EMAが前2バーのいずれかでSMAを下回るクロスをした場合、弱気シグナルを検出。
   - クロスオーバー価格（高速と低速の値の平均）を保存し、2バーのトリガーウィンドウを開始。

2. **トリガー条件**
   - `TriggerPrice = CrossPrice ± TriggerPips`（価格単位に変換）を計算。
   - トリガーはクロスオーバー時刻後の2プライマリローソク足の間有効。
   - ロングはローソク足の高値が強気トリガー価格に達するか超える必要があります。
   - ショートはローソク足の安値が弱気トリガー価格に達するか割り込む必要があります。

3. **エントリーフィルター**
   - 既存のポジションなし、アクティブなクールダウン（`BlackoutHours`）なし。
   - RSIフィルター: ロングは `RSI > RsiUpper`、ショートは `RSI < RsiLower`。
   - ブレイクアウトフィルター: 現在の終値がロングはローリング高値を超えるか、ショートはローリング安値を下回る必要があります。
   - エンベロープ確認: 現在の終値がロングは両方の上側エンベロープバンドの上にあるか、ショートは両方の下側バンドの下にある必要があります。

4. **注文実行**
   - 戦略ボリューム（デフォルト: 2ユニット、つまり2つの等しいサブポジション）を使用して成行注文を送信。
   - ストップロス: `StopLossLookback`番目のローソク足の安値（ロング）または高値（ショート）。
   - テイクプロフィットレベル: 前半分に `TakeProfit1Pips`、残りに `TakeProfit2Pips`。
   - `BlackoutHours` 時間の間新しいエントリーをブロックするためのブラックアウトタイマーを開始。

5. **ポジション管理**
   - `TrailingStopPips` > 0 の場合、トレーリングストップは即座に有効になり、トレードに有利な方向にのみ動きます。
   - 最初のテイクプロフィットレベルでポジションの半分をスケールアウト。
   - 最終テイクプロフィットがトリガーされたとき、ストップロスに達したとき、またはトレーリングストップを価格が超えたときに残りのポジションをクローズ。

## リスク管理

- ストップは常に完了したローソク足から導出され、バー内のノイズを低減します。
- 2つのテイクプロフィット目標により、トレードを続ける前に部分的な利益を確保します。
- トレーリングストップは、市場が望む方向に動いた後、利益が保護されることを保証します。
- 長いブラックアウト期間（デフォルト: 144時間）はブレイクアウト後の急速な再エントリーを防ぎ、元のEAの動作を反映します。

## 備考

- StockSharpのポートは、戦略ボリュームをデフォルトで2ユニットに設定することで元のマネーマネジメントのアイデアを保持しており、部分エグジットでポジションの半分が引き続き実行されます。
- MetaTraderのエンベロープシフト値は、高レベルAPIが前方シフトをサポートしていないため、最新のエンベロープ値を使用して近似されます。
- 戦略はpip距離を正しく変換するために価格ステップ情報を必要とします。銘柄のメタデータが入力されていることを確認してください。
