# CP Strat ORB
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует пробой диапазона открытия Нью-Йоркской сессии (9:30-9:45) с ретестом. Лонг открывается после пробоя максимума диапазона и закрытия выше него, шорт — после пробоя минимума и закрытия ниже. Выход осуществляется по фиксированным стоп-лоссу и тейк-профиту.

## Детали

- **Условия входа**: Пробой диапазона открытия NY и ретест с закрытием за пределами диапазона.
- **Длинные/короткие позиции**: Обе стороны.
- **Условия выхода**: Фиксированные тейк-профит и стоп-лосс.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `MinRangePoints` = 60m
  - `StopPoints` = 20m
  - `TakePoints` = 60m
  - `MaxTradesPerSession` = 3
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Both
  - Индикаторы: None
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
