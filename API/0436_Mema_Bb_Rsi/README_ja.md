# マルチタイムフレーム EMA + BB + RSI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

2本の指数移動平均、Bollinger BandsとRSIを組み合わせてバウンスを取引します。価格が下部バンドに触れた後、速いEMAの上で終値が付くとロング取引が発生します。価格が上部バンドを突き抜けた後、速いEMAの下で終値が付きRSIが50を上回るとショート取引が発動します。

オプションの利益確定機能は、価格が有利に動いた場合にユーザー定義のバー数後にポジションをクローズします。システムはスイングまたはイントラデイ取引に対して十分な柔軟性があり、ロングとショートのサイドを独立して有効または無効にできます。

## 詳細

- **エントリー条件**:
  - **ロング**: 安値がBollinger Bandsの下部バンドを突き抜けた後、速いEMAの上で終値が付く。
  - **ショート**: 高値が上部バンドを突き抜け、速いEMAの下で終値が付き、RSI > 50。
- **エグジット条件**:
  - ロング: RSIが売られすぎレベルを上回る。
  - ショート: 価格が下部バンドの下で終値が付く。
- **インジケーター**:
  - 2本のEMA (期間10と55)
  - Bollinger Bands (長さ20、乗数2)
  - RSI (長さ14、売られすぎ71)
- **ストップ**: Xバー後のオプションの利益目標；固定ストップロスなし。
- **デフォルト値**:
  - `Ma1Period` = 10
  - `Ma2Period` = 55
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
  - `RSILength` = 14
  - `RSIOversold` = 71
  - `XBars` = 12
- **フィルター**:
  - トレンドフィルター付き平均回帰
  - 時間軸: 設定可能
  - インジケーター: EMA、Bollinger Bands、RSI
  - ストップ: オプション
  - 複雑さ: 中程度
