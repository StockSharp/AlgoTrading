# Стратегия Traling Line
[English](README.md) | [中文](README_cn.md)

Стратегия управляет открытыми позициями с помощью динамической трейлинг-линии, рассчитанной по предыдущей четырехчасовой свече.
Для длинных позиций линия размещается ниже минимума свечи, для коротких – выше максимума.
Когда цена закрытия пересекает эту линию, позиция закрывается.
Система не открывает сделки сама и предназначена для защиты вручную открытых позиций.

## Детали
- **Условия входа**: нет – позиции должны быть открыты отдельно.
- **Защита Long**: `Стоп = Low(H4) - StopLevel * PriceStep`.
- **Защита Short**: `Стоп = High(H4) + StopLevel * PriceStep`.
- **Направление**: оба.
- **Критерий выхода**: цена закрытия пересекает трейлинг-линию.
- **Стопы**: только трейлинг стоп.
- **Значения по умолчанию**:
  - `StopLevel` = 70 шагов цены.
  - `CandleType` = свечи 4 часа.
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Простая
  - Таймфрейм: Долгосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
