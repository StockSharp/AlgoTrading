# Стратегия XRVI Crossover
[English](README.md) | [中文](README_cn.md)

Стратегия XRVI Crossover основана на расширенном индикаторе относительной силы XRVI.
XRVI рассчитывается сглаживанием индикатора RVI и последующим применением второго скользящего среднего для построения сигнальной линии.
Стратегия открывает длинную позицию при пересечении XRVI сигнальной линии снизу вверх и короткую при пересечении сверху вниз.
Существующие позиции переворачиваются при противоположных сигналах.

## Детали

- **Критерий входа**: пересечение XRVI и сигнальной линии
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположное пересечение
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RviLength` = 10
  - `SignalLength` = 5
  - `CandleType` = таймфрейм H4
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Обе стороны
  - Индикаторы: Relative Vigor Index, Simple Moving Average
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
