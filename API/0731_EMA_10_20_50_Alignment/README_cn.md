# EMA 10/20/50 Alignment
[English](README.md) | [Русский](README_ru.md)

该策略仅做多，当 EMA(10) > EMA(20) > EMA(50) 时入场，EMA 按降序排列时平仓。交易仅在设定的时间范围内进行。

## 详情

- **入场条件**: 在指定时间范围内，EMA(10) 高于 EMA(20)，EMA(20) 高于 EMA(50)。
- **多空方向**: 仅多头。
- **出场条件**: EMA 按下降顺序排列 (EMA(10) < EMA(20) < EMA(50))。
- **止损**: 无。
- **默认值**:
  - `StartTime` = 2023-05-17 00:00:00 +00:00
  - `EndTime` = 2025-05-17 00:00:00 +00:00
  - `CandleType` = TimeSpan.FromMinutes(1)
- **过滤器**:
  - 类别: Trend
  - 方向: Long
  - 指标: EMA
  - 止损: 无
  - 复杂度: 初级
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 低
