# EMA 10/20/50 Alignment
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда EMA(10) > EMA(20) > EMA(50), и закрывает её при обратном расположении EMA. Торговля ведётся только в заданном временном диапазоне.

## Детали

- **Условия входа**: EMA(10) выше EMA(20), а EMA(20) выше EMA(50) в пределах указанного периода.
- **Длинные/короткие позиции**: Только лонг.
- **Условия выхода**: EMA располагаются вниз по порядку (EMA(10) < EMA(20) < EMA(50)).
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `StartTime` = 17.05.2023 00:00
  - `EndTime` = 17.05.2025 00:00
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Trend
  - Направление: Long
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Начальная
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
