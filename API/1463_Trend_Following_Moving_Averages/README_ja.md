# 移動平均トレンドフォロー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

移動平均を計算し、動的な価格チャンネル内でそのトレンドを測定します。
トレンドスコアが正のときロングポジション、負のときショートポジションを取ります。

## 詳細

- **エントリー**:
  - **ロング**: トレンドスコア > 0
  - **ショート**: トレンドスコア < 0
- **エグジット**: 反転シグナル
- **インジケーター**: SMA, Highest, Lowest
- **時間軸**: 設定可能
- **タイプ**: トレンドフォロー
