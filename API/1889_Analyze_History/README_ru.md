# Стратегия Analyze History
[English](README.md) | [中文](README_cn.md)

Стратегия сканирует исторические данные свечей и выводит в лог временные разрывы больше указанного числа баров. Разрывы на выходных игнорируются, а разрывы из‑за праздников фиксируются.

## Детали

- **Условия входа**: нет
- **Направление**: нет
- **Условия выхода**: нет
- **Стопы**: нет
- **Значения по умолчанию**:
  - `MinGapInBars` = 10
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Анализ
  - Направление: Нет
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
