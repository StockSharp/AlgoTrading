# Analyze History 策略
[English](README.md) | [Русский](README_ru.md)

该策略扫描历史K线数据并在日志中记录超过指定数量的时间间隔。周末缺口会被忽略，而节假日缺口会被报告。

## 细节

- **入场条件**：无
- **方向**：无
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `MinGapInBars` = 10
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **筛选**：
  - 类型：分析
  - 方向：无
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：低
