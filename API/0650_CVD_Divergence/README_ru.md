# CVD Дивергенция
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает дивергенцию кумулятивной дельты объёма (CVD) с скользящими средними Халла, RSI, MACD и фильтром по объёму. Сделки открываются, когда тренд, моментум и объём согласованы, а CVD показывает дивергенцию или продолжение. Позиции закрываются при противоположном сигнале или пересечении индикаторов.

## Детали

- **Условия входа**: Совпадение тренда по HMA, подтверждение RSI и MACD, высокий объём и дивергенция/продолжение CVD.
- **Длинные/Короткие**: Обе стороны.
- **Условия выхода**: Противоположный сигнал или пересечение индикаторов.
- **Стопы**: Нет явных стопов.
- **Значения по умолчанию**:
  - `HmaFastLength` = 20
  - `HmaSlowLength` = 50
  - `RsiLength` = 14
  - `RsiOverbought` = 70
  - `RsiOversold` = 30
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `VolumeMaLength` = 20
  - `VolumeMultiplier` = 1.5m
  - `CvdLength` = 14
  - `DivergenceLookback` = 5
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Дивергенция
  - Направление: Обе
  - Индикаторы: HMA, RSI, MACD, Объём
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
