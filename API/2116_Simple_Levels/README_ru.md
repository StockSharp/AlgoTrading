# Стратегия Simple Levels
[English](README.md) | [中文](README_cn.md)

Открывает сделки при пересечении цены с заданными трендовыми линиями. Каждая линия может активировать покупку, продажу или обе стороны. Стоп-лосс и тейк-профит задаются в шагах цены.

## Детали

- **Условия входа**: Пересечение цены с заданной линией
- **Длинная/Короткая**: Определяется направлением линии (Buy/Sell/Both)
- **Условия выхода**: Стоп-лосс или тейк-профит
- **Стопы**: Да
- **Значения по умолчанию**:
  - `StopLoss` = 300 шагов
  - `TakeProfit` = 900 шагов
  - `Volume` = 1
  - `CandleType` = 1 минута
- **Фильтры**:
  - Категория: Уровни
  - Направление: Обе стороны
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

## Использование

1. Создайте и настройте трендовые линии через `AddLine`.
2. Запустите стратегию для обработки свечей.
3. При пересечении цены с активной линией в заданном направлении стратегия отправляет рыночный ордер.
4. Позиция закрывается при достижении стоп-лосса или тейк-профита.
