# ADX 出来高乗数戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ADX 出来高乗数戦略は、Average Directional Index によるトレンド強度と出来高急増フィルターを組み合わせます。ADX がしきい値を超え、優勢な方向線がトレンド方向を示し、現在の出来高がユーザー定義の係数で乗算した移動平均を上回る場合にのみ取引に参入します。

## 詳細

- **エントリー条件**:
  - ADX がしきい値を超え DI+ > DI- で出来高が SMA(出来高) * 乗数より大きい → ロング。
  - ADX がしきい値を超え DI- > DI+ で出来高が SMA(出来高) * 乗数より大きい → ショート。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆シグナルがポジションの反転をトリガーします。
- **ストップ**: なし。
- **デフォルト値**:
  - `AdxPeriod` = 21
  - `AdxThreshold` = 26
  - `VolumeMultiplier` = 1.8
  - `VolumePeriod` = 20
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: ADX, Volume SMA
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
