# MNQ EMA-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Strategie basierend auf mehreren exponentiellen gleitenden Durchschnitten für NASDAQ-Mikro-Futures. Sie eröffnet Long-Positionen, wenn der Preis oberhalb der schnellen und langsamen EMAs liegt und eine bullische Kerze vorliegt. Der Ausstieg erfolgt über dynamische EMA-Niveaus, Gewinnmitnahmen und Stop-Loss.
