# Bollinger Aroon戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Bollinger Aroon戦略は、強い上昇トレンド内の押し目を探します。
価格がBollinger Bandsの下限バンドを下回っても、Aroon Upの値が
高い水準を維持している場合、システムはトレンドが継続していると判断し、
平均への回帰を狙います。ロングのみ取引し、一時的な下落後の
反発を捉えることを目指します。

確定した足が下限バンドを下回って終値を付け、かつ *Aroon Up* が
確認レベルを超えた後にセットアップが発動します。Aroon の値が
ストップ閾値を下回るか、価格が上限バンドまで上昇するまで、
ポジションは保持されます。バンド幅はボラティリティに応じて適応するため、
この戦略は静かな市場と活発な市場の両方で取引できます。

主要な暗号通貨ペアでのバックテストでは、このアプローチが
時折のシェイクアウトを伴う強いトレンド時に優れた成績を示しています。
エントリーにはボラティリティ拡大と持続的なAroon Up の読みの両方が
必要なため、単純なBollinger リバーサルと比べてダマシのシグナルが減少します。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 終値が下限バンドを下回り、かつ `Aroon Up` > 確認レベル。
  - **ショート**: 使用しない。
- **エグジット条件**:
  - 終値が上限バンドに触れる、または `Aroon Up` < ストップレベル。
- **ストップ**: インジケーターベース；デフォルトでは固定ストップなし。
- **デフォルト値**:
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
  - `AroonLength` = 288
  - `AroonConfirmation` = 90
  - `AroonStop` = 70
- **フィルター**:
  - カテゴリ: トレンド内の平均回帰
  - 方向: ロングのみ
  - インジケーター: Bollinger Bands, Aroon
  - 複雑さ: 中程度
  - リスクレベル: 中
