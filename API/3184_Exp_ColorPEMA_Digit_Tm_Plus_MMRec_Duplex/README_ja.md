# Exp Color PEMA Digit TM Plus MMRec Duplex (C#) 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、StockSharpの高レベルAPIを使用して「Exp_ColorPEMA_Digit_Tm_Plus_MMRec_Duplex」エキスパートアドバイザーを再現します。異なるタイムフレームと価格ソースを使用できる2つの独立したPentuple Exponential Moving Average（PEMA）ストリームで動作します。ロングモジュールはPEMAの傾きが強気に切り替わったときにトレードを開き、ショートモジュールは弱気の転換に反応します。各サイドはインジケーターベースの出口と、設定可能な分数後に強制的にポジションを閉じるセーフティタイマーをサポートします。

## インジケーター
* **Pentuple EMA** – 同じ長さの8つの指数移動平均をカスケードし、古典的な係数（8, -28, 56, -70, 56, -28, 8, -1）を使用してそれらを組み合わせるカスタムインジケーター。このインジケーターは現在の値と前のサンプルの両方を公開し、戦略が傾きの方向（上、下、フラット）を分類できるようにします。
* **色ロジック** – 傾きは3つの離散状態にマッピングされます：上（緑）、下（マゼンタ）、ニュートラル（グレー）で、元のColorPEMAインジケーターの動作を再現します。

## シグナル生成
### ロングモジュール
1. 選択したロングタイムフレームで完了したローソク足を待ちます。
2. 設定された価格モードと丸め桁数を使用してPEMAの値を要求します。
3. `SignalBar`本前のローソク足のカラー状態を評価し、前のバーと比較します。
4. **エントリー**：色が`Up`に切り替わり、エントリーが許可されている場合、戦略は共有の`TradeVolume`を使用して買い、エントリー時刻を記録します。
5. **エグジット**：色が`Down`に切り替わったとき、インジケーターベースの出口が有効であれば、戦略はロングポジションを閉じます。
6. **タイムガード**：オープンなロングポジションが`LongTimeExitMinutes`より長く生き残った場合、インジケーターの状態に関係なく閉じられます。

### ショートモジュール
ショートサイドは同じワークフローを独立して繰り返します：
1. ショートタイムフレームのローソク足を監視します。
2. ショートPEMAシリーズを計算します。
3. `ShortSignalBar`本前を見て`Down`色への切り替えを検出します。
4. **エントリー**：色が弱気になりショートが有効な場合、戦略は売ります。
5. **エグジット**：色が`Up`になったとき、出口が許可されていればショートはカバーされます。
6. **タイムガード**：`ShortTimeExitMinutes`が超過した場合、ショートポジションは閉じられます。

## リスク管理
* デフォルトの注文サイズを設定するために`TradeVolume`パラメーターを使用します。
* オプションのストップロスとテイクプロフィットを価格ステップで設定できます。いずれかが正の場合、戦略はマーケット出口注文で`StartProtection`を有効にし、MQLバージョンに存在するマネーマネジメント保護を反映します。
* ロングとショートモジュールの独立したタイムベースの出口タイマーにより、トレードが無期限に継続されることを防ぎます。

## パラメーター
| パラメーター | 説明 |
|-----------|-------------|
| `LongCandleType` | ロングインジケーターストリームに使用するタイムフレーム。 |
| `ShortCandleType` | ショートインジケーターストリームのタイムフレーム。 |
| `LongEmaLength`, `ShortEmaLength` | Pentuple EMAのスムージング長（小数値をサポート）。 |
| `LongPriceMode`, `ShortPriceMode` | 各ストリームに適用する価格モード（終値、始値、高値、安値、中間、典型、加重、シンプル、クォーター、トレンドフォロー、Demark）。 |
| `LongDigits`, `ShortDigits` | 計算されたPEMA値に適用する小数点以下の丸め。 |
| `LongSignalBar`, `ShortSignalBar` | 色の変化を評価するために使用する完了バー数。 |
| `LongAllowOpen`, `ShortAllowOpen` | 各サイドの新規エントリーを有効/無効にする。 |
| `LongAllowClose`, `ShortAllowClose` | インジケーターベースの出口を有効/無効にする。 |
| `LongAllowTimeExit`, `ShortAllowTimeExit` | タイムベースの出口ウォッチドッグをオン/オフにする。 |
| `LongTimeExitMinutes`, `ShortTimeExitMinutes` | ロングおよびショートトレードの最大保有時間（分）。 |
| `TradeVolume` | マーケット注文のデフォルトボリューム。 |
| `StopLossSteps`, `TakeProfitSteps` | 証券の価格ステップで表されるオプションの保護距離。 |

## 注記
* 戦略はロングとショートの両方のローソク足シリーズを購読します；両方のパラメーターが同じタイムフレームを指している場合、StockSharpは自動的にデータフィードを再利用します。
* 両モジュールは同じ証券とボリューム設定を共有し、対称的な動作を保証します。
* インジケーターの計算は、再描画を避けるために完了したローソク足のみで実行されます。
