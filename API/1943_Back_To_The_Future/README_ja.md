# Back to the Future戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このモメンタム戦略は、現在の終値を指定された分数前の価格と比較します。価格が過去の価格に対して定義されたしきい値を超えて上昇した場合、システムはロングポジションをオープンします。逆に、価格がマイナスのしきい値を下回った場合、ショートポジションをオープンします。このアプローチは、過去の価格から大きく離れた強い動きが新興トレンドを示すという前提に基づいています。

この戦略は完成したローソク足で動作し、StockSharpがサポートする任意の銘柄と時間軸に対応しています。組み込みのテイクプロフィットとストップロスのレベルがポジションオープン後のリスクを管理します。過去の価格のキューが価格差を評価するためのローリング履歴を維持します。

## 詳細

- **エントリー条件**:
  - **ロング**: `Close(t) - Close(t-Δ) > BarSize`.
  - **ショート**: `Close(t) - Close(t-Δ) < -BarSize`.
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: `Close >= Entry + TakeProfit` または `Close <= Entry - StopLoss`.
  - **ショート**: `Close <= Entry - TakeProfit` または `Close >= Entry + StopLoss`.
- **ストップ**: あり、価格単位での固定テイクプロフィットとストップロス。
- **デフォルト値**:
  - `BarSize = 0.25`
  - `HistoryMinutes = 60`
  - `TakeProfit = 10`
  - `StopLoss = 5000`
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: なし
  - ストップ: あり
  - 複雑さ: シンプル
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
