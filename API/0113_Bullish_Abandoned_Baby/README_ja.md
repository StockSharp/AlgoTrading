# 強気の捨て子戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
強気の捨て子は、ギャップダウンしたドージに続いてギャップアップする珍しい三本足パターンです。
この形成は真ん中のローソク足を「捨てられた」状態にし、多くの場合、急激な上方への反転に先行します。

テストでは年平均リターン約 76% を示しています。外国為替市場で最も良いパフォーマンスを発揮します。

戦略は、第三のローソク足がドージを上回ってギャップアップした時点でその始値で買い、ショートカバーを伴う強い追随を期待します。

ストップはドージの安値のすぐ下に置かれ、反転が維持されない場合の損失を小さく抑えます。

## 詳細

- **エントリー条件**: パターン一致
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
