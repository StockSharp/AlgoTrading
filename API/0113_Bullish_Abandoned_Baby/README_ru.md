# Стратегия Bullish Abandoned Baby

Bullish Abandoned Baby — редкая трёхсвечная формация, где после гэпа вниз появляется доджи, а затем происходит гэп вверх.
Эта конфигурация оставляет среднюю свечу "брошенной" и нередко предшествует резкому развороту вверх.

Стратегия покупает на открытии третьей свечи, если она открывается выше доджи, ожидая сильного продолжения благодаря закрытию шортов.

Стоп размещается сразу под минимумом доджи, позволяя держать потери небольшими, если разворот не удержится.

## Детали

- **Критерий входа**: совпадение паттерна
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
