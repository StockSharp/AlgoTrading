# Стратегия Entry Fragger
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает последовательности красных и зелёных свечей относительно EMA с периодом 50. После серии красных свечей ниже EMA зелёная свеча, закрывающаяся выше облака волатильности, даёт сигнал на покупку. Аналогичная логика для зелёных свечей формирует условия для продажи. Опция обратной торговли позволяет переворачивать позиции.

## Детали

- **Условия входа**:
  - **Лонг**: `redCount >= Buy Signal Accuracy` && последняя красная ниже EMA50 && зелёная свеча закрывается выше `EMA50 + stdev/4`.
  - **Шорт**: `greenCount >= Sell Signal Accuracy` && предыдущая свеча зелёная && текущая красная закрывается выше `EMA50 + stdev/4`.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**: Обратный сигнал.
- **Индикаторы**: EMA, StandardDeviation.
- **Значения по умолчанию**:
  - `Buy Signal Accuracy` = 2
  - `Sell Signal Accuracy` = 2
- **Фильтры**:
  - Категория: Моментум
  - Направление: Обе стороны
  - Индикаторы: Несколько
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
