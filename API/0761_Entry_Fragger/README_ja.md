# エントリー・フラッガー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、50期間EMAに対する赤と緑のローソク足のシーケンスを追跡します。EMAの下で赤いローソク足が連続した後、ボラティリティクラウドの上で終値をつける緑のローソク足がロングエントリーをトリガーします。緑のローソク足の連続による類似のセットアップがショートエントリーに先行します。オプションの逆張り取引でポジションを反転できます。

## 詳細

- **エントリー条件**:
  - **ロング**: `redCount >= Buy Signal Accuracy` && 直近の赤がEMA50の下 && 緑のローソク足が `EMA50 + stdev/4` を上回って終値。
  - **ショート**: `greenCount >= Sell Signal Accuracy` && 前のローソク足が緑 && 赤のローソク足が `EMA50 + stdev/4` を上回って終値。
- **ロング/ショート**: 両サイド。
- **エグジット条件**: 逆シグナル。
- **インジケーター**: EMA, StandardDeviation。
- **デフォルト値**:
  - `Buy Signal Accuracy` = 2
  - `Sell Signal Accuracy` = 2
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: なし
  - 複雑さ: 中程度
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
