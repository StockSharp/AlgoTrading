# 自動FibonacciおよびGannファン・リトレースメント組み合わせ
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は新しいスイング高値と安値を自動的に検出します。極値が見つかると、最後のスイングポイント間でFibonacciおよびGannリトレースメントレベルを計算します。レベルはログに書き込まれ、取引ロジックは実行されません。
