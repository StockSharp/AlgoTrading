# Combinación de Retrocesos Automáticos Fibonacci y Gann Fan
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia detecta automáticamente nuevos máximos y mínimos de oscilación. Cuando se encuentra un extremo, calcula los niveles de retroceso de Fibonacci y Gann entre los últimos puntos de oscilación. Los niveles se escriben en el registro y no se ejecuta ninguna lógica de trading.
