# Automatische Fibonacci- und Gann-Fan-Retracement-Kombination
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie erkennt automatisch neue Swing-Hochs und -Tiefs. Wird ein Extremum gefunden, berechnet sie Fibonacci- und Gann-Retracement-Niveaus zwischen den letzten Swing-Punkten. Die Niveaus werden ins Log geschrieben und es wird keine Handelslogik ausgeführt.
