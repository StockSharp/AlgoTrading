# Combinação de Retrações Automáticas Fibonacci e Gann Fan
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia detecta automaticamente novos máximos e mínimos de swing. Quando um extremo é encontrado, calcula os níveis de retração de Fibonacci e Gann entre os últimos pontos de swing. Os níveis são gravados no log e nenhuma lógica de negociação é executada.
