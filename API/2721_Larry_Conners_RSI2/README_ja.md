# Larry Connors RSI-2戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

クラシックなLarry Connors RSI-2システムのStockSharp忠実なポートです。この戦略は、2期間の高速RSIオシレーターと時間足のMoving Averageフィルターを組み合わせ、より上位の時間軸のトレンドに沿いながら短期的な平均回帰のセットアップを捉えます。pipsで表されるオプションのストップロスとテイクプロフィットレベルは、オリジナルのMetaTraderの資金管理ルールを再現します。

## 概念の概要

- **タイプ**: トレンドフィルター付き平均回帰。
- **市場**: H1チャートで取引されるFX通貨ペア向け。
- **方向**: ロングとショートの両方を取引しますが、低速SMAフィルターの方向のみ。
- **コアインジケーター**: 5期間SMA（エグジットタイミング）、200期間SMA（トレンドフィルター）、2期間RSI（シグナルトリガー）。

## 取引ルール

### ロングエントリー
- RSI値が`RSI Long Entry`（デフォルト6）を下回る。
- 完成したローソク足の終値が`Slow SMA`（デフォルト200期間）を上回っている。
- オープンポジションがない。

### ショートエントリー
- RSI値が`RSI Short Entry`（デフォルト95）を上回る。
- 終値が`Slow SMA`を下回っている。
- オープンポジションがない。

### エグジット条件
- **ロングポジション**は終値が`Fast SMA`（デフォルト5）を上回ったときに決済。有効化されている場合、pipsで計測されるオプションのストップロスとテイクプロフィットレベルも取引を決済できます。
- **ショートポジション**は終値が`Fast SMA`を下回ったときに決済。オプションのストップロスとテイクプロフィットレベルをpipsで対称的に適用します。

### リスク管理
- `Use Stop Loss`はエントリー価格から相対的なpipsでの固定ストップ距離を切り替えます。
- `Use Take Profit`はpipsでの対称的な利益目標を有効にします。
- pip距離は、4/5桁の相場に対するMT5ロジックを反映して、インストゥルメントの`PriceStep`と小数精度を通じて絶対価格に変換されます。

## デフォルト値

| パラメーター | デフォルト | 説明 |
|-------------|-----------|------|
| `Trade Volume` | 1 | 各エントリーのベース注文量。 |
| `Fast SMA Period` | 5 | エグジットタイミング平均。 |
| `Slow SMA Period` | 200 | トレンド方向フィルター。 |
| `RSI Period` | 2 | RSIオシレーターのルックバック。 |
| `RSI Long Entry` | 6 | ロング取引の売られすぎ閾値。 |
| `RSI Short Entry` | 95 | ショート取引の買われすぎ閾値。 |
| `Use Stop Loss` | true | 保護ストップを有効/無効にする。 |
| `Stop Loss (pips)` | 30 | pipsでのストップロス距離。 |
| `Use Take Profit` | true | 固定利益目標を有効/無効にする。 |
| `Take Profit (pips)` | 60 | pipsでの利益目標距離。 |
| `Candle Type` | 1時間 | 作業ローソク足の時間軸。 |

すべての調整可能なパラメーターは`.SetCanOptimize(true)`を公開し、Designer/Tester内でのバッチ最適化を可能にします。

## 実行ノート

- シグナルはオリジナルのMetaTrader実装に合わせて、クローズしたローソク足で評価されます。
- 保護レベルは内部で追跡され、違反時に成行注文でポジション全体を決済します。
- 戦略は再現可能なバックテストを保証するために、各再起動時に内部状態（`pipSize`、エントリーアンカー）をリセットします。
- 公開されているパフォーマンス結果を再現するために、信頼性の高いFXデータとともに戦略をプロジェクトに追加してください。

## 推奨される使用方法

1. 1時間ローソク足を供給するFXデータフィードを接続する。
2. 戦略をDesignerに追加するか、StockSharp APIを通じてプログラム的に実行する。
3. 必要に応じてブローカーの契約仕様に合わせてpipベースのリスクパラメーターを調整する。
4. オプションでRSI閾値や移動平均の長さを最適化して、モデルを他のシンボルに適応させる。

正確なRSIと移動平均ロジックを維持することで、このポートによりMT5ユーザーはStockSharpエコシステム内でLarry Connors RSI-2の方法論を評価することができます。
