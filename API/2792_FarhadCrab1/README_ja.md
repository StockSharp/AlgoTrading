# FarhadCrab1 戦略 (C#)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
FarhadCrab1 戦略は、指数移動平均（EMA）へのプルバックで取引を行うトレンドフォロー・システムであり、固定ストップ、テイクプロフィット、Parabolic SAR にインスパイアされたトレーリングストップ、および上位時間軸フィルターを使用して決済を管理します。オリジナルの MetaTrader 5 エキスパートアドバイザーは、実行にはロウソク足ごとに時間足を使用し、オープンポジションをいつ閉じるかを決定するために日足データを参照します。この C# ポートは、作業時間軸の EMA フィルターと日足 EMA クロスオーバー決済ルールを組み合わせることで同じコアロジックを維持します。

## 主要コンセプト
- **トレンドフィルター：** 作業時間軸で計算された EMA（デフォルト：1時間足での15期間 EMA）。前のロウソク足の安値が EMA より上にある場合のみロングシグナルが許可され、前のロウソク足の高値が EMA より下にある場合のみショートシグナルが許可されます。
- **日足フィルター：** 日足ロウソク足で計算された別の EMA。日足 EMA が日足の終値を上回ると、すべてのロングポジションが閉じられます。下回ると、すべてのショートポジションが閉じられます。これは MQL5 コードのオリジナルの `ClosePositions` ロジックを模倣しています。
- **リスク管理：** 固定のストップロスとテイクプロフィットレベルは pip 距離から導き出されます。トレーリングストップは、ポジションが十分な利益を得ると保護ストップを移動させ、`TrailingStop` と `TrailingStep` 設定を組み合わせた MT5 トレーリング機能をエミュレートします。
- **単一ポジション管理：** 戦略は単一のネットポジションで取引します。ショートを保持しながらロングポジションに入る（またはその逆）場合、新しい取引を開く前にまず反対側のエクスポージャーを閉じます。

## 取引ルール
1. **シグナル検出（作業時間軸）：**
   - 前のロウソク足の安値が EMA 値より大きい場合（設定されたシフトを適用後）にロングエントリー。
   - 前のロウソク足の高値が EMA 値より小さい場合にショートエントリー。
2. **ポジションサイジング：** `Volume` パラメーターが基本注文サイズを設定します。ショートからロングに反転する場合（またはその逆）、エンジンはネットポジションを転換するために必要な追加量を自動的に送信します。
3. **ストップロスとテイクプロフィット：**
   - 距離は pips で定義されます。pip サイズは銘柄のティックサイズに自動的に適応し、5桁および3桁の外国為替シンボルは MT5 の動作に合わせるために10倍の乗数を使用します。
   - ストップロスまたはテイクプロフィットは、それぞれの pip 距離をゼロに設定することで無効にできます。
4. **トレーリングストップ：**
   - `TrailingStopPips` がゼロより大きい場合のみ有効になります。
   - ポジションの利益が `TrailingStopPips + TrailingStepPips` を超えると、ストップは `現在価格 - TrailingStopPips`（ロング用）または `現在価格 + TrailingStopPips`（ショート用）に移動します。
   - 追加のトレーリングステップにより頻繁な修正が防止されます。
5. **日足決済フィルター：**
   - 最後の2つの完成した日足ロウソク足を使用します。
   - 2日前に日足 EMA が日足終値を下回り、最新の日に日足終値を上回った場合（弱気クロスオーバー）、ロングポジションが閉じられます。
   - 逆のクロスオーバーが発生した場合、ショートポジションが閉じられます。

## パラメーター
| 名前 | 型 | デフォルト | 説明 |
| --- | --- | --- | --- |
| `CandleType` | `DataType` | 1時間時間軸 | 実行 EMA とエントリーロジックに使用される作業時間軸。 |
| `MaLength` | `int` | 15 | 作業時間軸の EMA の期間。 |
| `MaShift` | `int` | 0 | EMA を後方にシフトするために使用する完成ロウソク足の数。 |
| `DailyMaLength` | `int` | 15 | クロスオーバー決済フィルターを提供する日足 EMA の期間。 |
| `StopLossPips` | `decimal` | 50 | ストップロス距離（pips）。無効にするには `0` に設定。 |
| `TakeProfitPips` | `decimal` | 50 | テイクプロフィット距離（pips）。無効にするには `0` に設定。 |
| `TrailingStopPips` | `decimal` | 10 | トレーリングストップ距離（pips）。トレーリングを無効にするには `0` に設定。 |
| `TrailingStepPips` | `decimal` | 5 | トレーリングストップが再び更新される前の最小追加利益（pips）。 |
| `Volume` | `decimal` | 0.1 | ロット/コントラクト単位の基本取引サイズ。 |

## MQL バージョンとの注意点と相違点
- このポートは常に指数移動平均を使用し、オリジナルのデフォルト（`MODE_EMA`）を反映しています。他の MT5 スムージングモードはサポートされていません。
- MT5 エキスパートアドバイザーはすべてのティックで売値/買値で動作します。この変換は完成したロウソク足で動作するため、ストップロスとテイクプロフィットの確認はロウソク足の高値/安値で評価されます。
- オリジナルファイルに存在した Parabolic SAR インジケーターは取引の決定に影響を与えなかったため、C# 実装から省略されています。
- トレーリングロジックは保存されたストップレベルを調整しますが、ブローカーのストップ注文を送信しません。ロウソク足の範囲が計算されたストップまたはテイクプロフィットレベルに達したとき決済が発生します。

## 使用のヒント
- 希望する取引の時間軸に合ったロウソク足タイプを選択してください。デフォルトの1時間ロウソク足はソーススクリプトの動作を再現します。
- イントラデイエントリーと上位時間軸トレンドフィルター間の反応性を調整するために、`MaLength` と `DailyMaLength` を一緒に調整してください。
- 5桁で表示される外国為替シンボル（例：EURUSD）の場合、1 pip が 0.0001 に等しくなるように pip 距離が自動的にスケーリングされます。
- バックテストを実行する際は、決済フィルターが正しく機能できるように日足データストリームが利用可能であることを確認してください。
