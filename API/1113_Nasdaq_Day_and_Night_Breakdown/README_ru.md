# Разбор дня и ночи Nasdaq
[English](README.md) | [中文](README_cn.md)

Сравнивает движения фьючерса Nasdaq и индекса Nasdaq до и после открытия рынка.
Вычисляет разницу до открытия и после открытия, а также внутридневные колебания.

## Детали

- **Условия входа**: нет
- **Длинная/короткая**: нет
- **Условия выхода**: нет
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = 5 minute
- **Фильтры**:
  - Категория: Анализ
  - Направление: нет
  - Индикаторы: разница цен
  - Стопы: нет
  - Сложность: начальная
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
