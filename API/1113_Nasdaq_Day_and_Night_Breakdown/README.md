# Nasdaq Day and Night Breakdown
[Русский](README_ru.md) | [中文](README_cn.md)

Compares pre-market and regular session movements of Nasdaq futures and the Nasdaq index.
Calculates differences before and after market open and shows intraday swings.

## Details

- **Entry Criteria**: none
- **Long/Short**: None
- **Exit Criteria**: none
- **Stops**: none
- **Default Values**:
  - `CandleType` = 5 minute
- **Filters**:
  - Category: Analysis
  - Direction: None
  - Indicators: Price difference
  - Stops: No
  - Complexity: Beginner
  - Timeframe: Intraday
  - Seasonality: No
  - Neural networks: No
  - Divergence: No
  - Risk level: Low
