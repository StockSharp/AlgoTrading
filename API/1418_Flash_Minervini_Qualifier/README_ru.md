# Стратегия Flash Strategy Minervini Qualifier
[English](README.md) | [中文](README_cn.md)

Сочетает пересечение EMA, направление SuperTrend и импульсный RSI с фильтром стадий Минервини.

## Детали

- **Условия входа**: EMA выше трейловой линии, тренд SuperTrend и RSI импульса выше порога с фильтром Минервини
- **Лонг/Шорт**: Оба
- **Условия выхода**: противоположный сигнал трейла или разворот SuperTrend
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `MomRsiLength` = 10
  - `MomRsiThreshold` = 60
  - `EmaLength` = 12
  - `EmaPercent` = 0.01
  - `SuperTrendPeriod` = 10
  - `SuperTrendMultiplier` = 3
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA, SuperTrend, RSI
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
