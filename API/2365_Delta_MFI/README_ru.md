# Стратегия Delta MFI
[English](README.md) | [中文](README_cn.md)

Стратегия сравнивает значения быстрого и медленного индикатора Money Flow Index. Покупка осуществляется, когда быстрый MFI выше медленного и медленный MFI превышает сигнальный уровень. Продажа осуществляется, когда быстрый MFI ниже медленного и медленный MFI опускается ниже значения `100 - уровень`.

## Детали

- **Вход в позицию**:
  - Покупка при `slow MFI > Level` и `fast MFI > slow MFI`
  - Продажа при `slow MFI < 100 - Level` и `fast MFI < slow MFI`
- **Длинные/короткие позиции**: Оба направления
- **Выход из позиции**: Обратный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `FastPeriod` = 14
  - `SlowPeriod` = 50
  - `Level` = 50
  - `CandleType` = свечи H4
- **Фильтры**:
  - Категория: Индикатор
  - Направление: Оба
  - Индикаторы: Money Flow Index
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: H4
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
