# KDJ エキスパートアドバイザー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略はsenlin geによるMetaTrader 5「KDJ Expert Advisor」を再現します。%Kラインが2回平滑化されるストキャスティクスオシレーターの進化形であるKDJオシレーターからのシグナルを使用して単一のシンボルで取引します。戦略は%Kと%Dライン（しばしばJラインと呼ばれる）の差を観察してモメンタム反転を識別し、一度に1つのポジションのみをオープンします。取引管理は元のエキスパートアドバイザーを反映します：各取引はpipsで表現され、インスツルメントの設定を使用して価格距離に変換された固定ストップロスとテイクプロフィットを即座に受け取ります。

実装はローソク足サブスクリプションとMQL5バージョンのKDJパラメーターに一致するよう設定された組み込みの`Stochastic`インジケーターを使用してStockSharpの高レベルAPIを使用します。コードは3桁または5桁のFORXシンボルを自動的に検出し、それに応じてpip値を調整します。

## インジケーターロジック
基礎となるインジケーターは3段階で動作します：

1. **RSV計算** – 各完成したローソク足に対して、`KDJ Length`ローソク足のRaw Stochastic Valueを計算します：
   \[
   RSV = \frac{Close - LowestLow}{HighestHigh - LowestLow} \times 100
   \]
2. **%K平滑化** – %Kラインを得るために最後の`Smooth %K`個のRSV値を平均します。
3. **%D平滑化** – %Dラインを得るために最後の`Smooth %D`個の%K値を平均します。

戦略は次に`K - D`（元のソースで*KDC*と呼ばれる）と%Kの傾きを分析して反転を検出します。

## エントリー条件
シンボルに対して既存のポジションがない場合のみ市場ポジションがオープンされます。シグナルは完成したローソク足で評価されます：

- **買い**は以下のいずれかの条件が真の場合：
  - `K - D`がゼロを上向きにクロスする（ネガティブからポジティブへ）；または
  - `K - D`がゼロより上で、%Kラインが上昇している（`K_current > K_previous`）。
- **売り**は以下のいずれかの条件が真の場合：
  - `K - D`がゼロを下向きにクロスする（ポジティブからネガティブへ）；または
  - `K - D`がゼロより下で、%Kラインが下落している（`K_current < K_previous`）。

これは元のMQL5エキスパートアドバイザーのブール構造に一致し、同一の取引タイミングを保証します。

## リスク管理
- 各約定した注文は保護ストップロスとテイクプロフィットを受け取り、pipsで測定され、インスツルメントのティックサイズを通じて価格距離に変換されます。ゼロ値は対応する保護レグを無効にします。
- 戦略はピラミッドや平均コストを行いません。現在のポジションが保護注文または手動介入によって閉じられるまでフラットを維持します。

## パラメーター
| パラメーター | 説明 | デフォルト値 |
|-----------|------|------------|
| **Candle Type** | 入力ローソク足のデータタイプ/時間軸。 | 15分時間軸 |
| **KDJ Length** | RSV計算のローソク足数。 | 30 |
| **Smooth %K** | %Kラインを平滑化するために使用されるRSV値の数。 | 3 |
| **Smooth %D** | %Dラインを平滑化するために使用される%K値の数。 | 6 |
| **Stop Loss (pips)** | 保護ストップロスの距離。無効にするには0に設定。 | 25 |
| **Take Profit (pips)** | 保護テイクプロフィットの距離。無効にするには0に設定。 | 45 |
| **Order Volume** | 市場注文で送信される数量。 | 1 |

すべてのパラメーターは元のエキスパートアドバイザーの入力と同一の最適化範囲をサポートします。

## 使用上の注意
1. テスターまたはライブ環境で希望のインスツルメントとコネクターを設定します。
2. MetaTraderからエミュレートしたいチャートの時間軸に合わせてローソク足タイプを調整します。
3. オプションでKDJパラメーター、ストップロス、テイクプロフィット、または注文ボリュームを最適化します。
4. 戦略を開始します。注文は完全に形成されたローソク足でのみ生成されます。
5. チャートには視覚的な確認のためにローソク足、KDJインジケーター、および実行された取引が自動的に表示されます。

## 元のEAとの違い
- MQL5 KDJバッファーを複製するために平滑化期間付きのStockSharpの`Stochastic`インジケーターを使用します；外部インジケーターファイルは不要です。
- 保護注文は`StartProtection`を通じて管理され、トリガーされると市場決済を送信します。
- ボリュームはMQL5の`MoneyFixedMargin`リスクモデルの代わりに固定パラメーターであり、実装を簡潔にシグナルロジックに集中させます。
