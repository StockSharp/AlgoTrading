# Стратегия PPO Cloud
[English](README.md) | [中文](README_cn.md)

Стратегия моментума торгует пересечения между осциллятором процентной цены (PPO) и его сигнальной линией. Длинная позиция открывается, когда PPO пересекает сигнальную линию снизу вверх, короткая – при обратном пересечении. Существующие позиции могут закрываться по противоположному сигналу. Стратегия работает на одном таймфрейме.

## Детали

- **Условия входа**:
  - **Long**: `PPO пересекает сигнал снизу вверх`.
  - **Short**: `PPO пересекает сигнал сверху вниз`.
- **Направление торговли**: Long и Short.
- **Условия выхода**:
  - **Long**: `PPO пересекает сигнал сверху вниз` (опционально).
  - **Short**: `PPO пересекает сигнал снизу вверх` (опционально).
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `Fast Period` = 12.
  - `Slow Period` = 26.
  - `Signal Period` = 9.
- **Фильтры**:
  - Категория: Моментум
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
