# PPO クラウド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このモメンタム戦略はPercentage Price Oscillator (PPO)とそのシグナルラインのクロスオーバーを取引します。PPOがシグナルラインを上回ったときにロングポジションを開き、反対のクロスオーバーでショートポジションを開きます。既存ポジションは反対シグナルでオプションでクローズできます。戦略は単一の時間軸で動作します。

## 詳細

- **エントリー条件**:
  - **ロング**: `PPOがシグナルを上回る`.
  - **ショート**: `PPOがシグナルを下回る`.
- **ロング/ショート**: 両方.
- **エグジット条件**:
  - **ロング**: `PPOがシグナルを下回る`（オプション）.
  - **ショート**: `PPOがシグナルを上回る`（オプション）.
- **ストップ**: なし.
- **デフォルト値**:
  - `Fast Period` = 12.
  - `Slow Period` = 26.
  - `Signal Period` = 9.
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
