# PPO Cloud 策略
[English](README.md) | [Русский](README_ru.md)

该动量策略交易百分比价格振荡器（PPO）与其信号线的交叉。当 PPO 上穿信号线时开多单，当下穿时开空单。现有头寸可在相反信号上选择平仓。策略使用单一时间框架运行。

## 细节

- **入场条件**：
  - **多头**：`PPO 上穿信号线`。
  - **空头**：`PPO 下穿信号线`。
- **多空方向**：双向。
- **出场条件**：
  - **多头**：`PPO 下穿信号线`（可选）。
  - **空头**：`PPO 上穿信号线`（可选）。
- **止损**：无。
- **默认值**：
  - `Fast Period` = 12。
  - `Slow Period` = 26。
  - `Signal Period` = 9。
- **过滤器**：
  - 类别: 动量
  - 方向: 双向
  - 指标: 单一
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 低
