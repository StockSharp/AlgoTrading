# Стратегия Dkoderweb Repainting Issue Fix
[English](README.md) | [中文](README_cn.md)

Эта стратегия обнаруживает гармонические паттерны с помощью простого зигзага и открывает сделки при возврате цены к уровню Фибоначчи. При формировании бычьего паттерна и откате цены в зону входа открывается длинная позиция с фиксированными уровнями тейк‑профита и стоп‑лосса. Медвежий паттерн инициирует зеркальные действия для короткой позиции.

## Детали

- **Критерии входа**:
  - **Длинная позиция**: паттерн ABCD и цена закрытия на уровне или ниже входного уровня Фибоначчи.
  - **Короткая позиция**: паттерн ABCD и цена закрытия на уровне или выше входного уровня Фибоначчи.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - Достижение ценой уровней тейк‑профита или стоп‑лосса.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `TradeSize` = 1
  - `EntryRate` = 0.382
  - `TakeProfitRate` = 0.618
  - `StopLossRate` = -0.618
- **Фильтры**:
  - Категория: Распознавание паттернов
  - Направление: Оба
  - Индикаторы: ZigZag
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
