# Dkoderweb Repainting Issue Fix 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、シンプルなジグザグアプローチを使ってハーモニックパターンを検出し、価格がフィボナッチリトレースメントレベルに戻ったときにトレードを行います。強気パターンが形成され、価格がエントリー窓まで引き戻されると、戦略は事前に定義されたテイクプロフィットとストップロスのレベルでロングポジションを開きます。弱気パターンは逆方向に同じロジックを発動させます。

## 詳細

- **エントリー条件**:
  - **ロング**: ABCDハーモニックパターンかつ終値がエントリーFibonacciレベル以下。
  - **ショート**: ABCDハーモニックパターンかつ終値がエントリーFibonacciレベル以上。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 価格がテイクプロフィットまたはストップロスのFibonacciレベルに到達。
- **ストップ**: はい。
- **デフォルト値**:
  - `TradeSize` = 1
  - `EntryRate` = 0.382
  - `TakeProfitRate` = 0.618
  - `StopLossRate` = -0.618
- **フィルター**:
  - カテゴリ: パターン認識
  - 方向: 両方
  - インジケーター: ZigZag
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
