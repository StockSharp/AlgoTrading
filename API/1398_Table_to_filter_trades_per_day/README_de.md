Einfache gleitender-Durchschnitt-Crossover-Strategie mit SMA50 und SMA200 und festen Gewinn- und Verlustzielen.

## Details

- **Einstieg**
  - Long: SMA50 kreuzt über SMA200.
  - Short: SMA50 kreuzt unter SMA200.
- **Ausstieg**: Position schließen, wenn Ziel oder Stop erreicht wird.
