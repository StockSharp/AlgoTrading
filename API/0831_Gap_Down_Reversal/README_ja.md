# ギャップダウン・リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ギャップダウン・リバーサル戦略は、ギャップダウン始値後の強気リバーサルを探します。
新しいセッションが前回安値を下回って始まりながら始値を上回って終了した場合、売り手を罠にかけてリバウンドを示すことが多いです。

このパターンが現れたときにロングでエントリーし、価格が前回高値を上回って終了したときにエグジットします。

## 詳細

- **エントリー条件**: ギャップダウン・リバーサルパターン
- **ロング/ショート**: ロングのみ
- **エグジット条件**: 前回高値を上回るクローズ
- **ストップ**: いいえ
- **デフォルト値**:
  - `CandleType` = 1 day
  - `StartTime` = 2014-01-01
  - `EndTime` = 2099-01-01
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ロングのみ
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
