# コインフリップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

**コインフリップ戦略**は、ポジションが開いていない場合、新しいローソク足ごとにランダムにロングまたはショートを選択します。ポジションを閉じた後、取引が損失で終わった場合、次の取引サイズはマーチンゲール乗数を使って増加されます。戦略は価格ステップで定義された固定のテイクプロフィットとストップロスレベルを使ってポジションを閉じ、オプションで指定距離後に利益をトレールすることもできます。

## パラメーター

- `Volume` – 最初の試みに使用するベース注文サイズ。
- `Martingale` – 損失取引後にボリュームに適用される乗数。
- `MaxVolume` – マーチンゲール増加後のポジションサイズの上限。
- `TakeProfit` – 価格ステップでの利益目標。
- `StopLoss` – 価格ステップでの損失制限。
- `TrailingStart` – トレーリングが有効になる価格ステップでの距離。
- `TrailingStop` – 価格ステップでのトレーリングストップ距離。
- `CandleType` – 意思決定に使用するローソク足の時間軸。

## 動作方法

1. 完了したローソク足ごとに、戦略はオープンポジションを確認します。
2. ポジションが存在する場合、現在の終値を使って利益または損失を監視します。テイクプロフィット、ストップロス、またはトレーリングストップの条件が満たされると、ポジションが閉じられます。
3. ポジションが開いていない場合、仮想コインを投げます：
   - 表はロングポジションを開きます。
   - 裏はショートポジションを開きます。
4. 前回の取引が損失だった場合、ボリュームは `Martingale` で乗算されますが、`MaxVolume` でキャップされます。
5. 価格が有利な方向に `TrailingStart` 動いた後、トレーリングストップが有効になります。
