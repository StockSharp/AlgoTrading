# BykovTrend ReOpen戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
BykovTrend ReOpen戦略は、Williams %RとAverage True Rangeインジケーターに基づくBykovTrendロジックを使用します。トレンドが強気に転換したときに買いシグナルが発生し、弱気に転換したときに売りシグナルが発生します。ポジションに入った後、戦略はトレンドが続く間、事前に定義された価格ステップごとに追加ポジションを再度開くことができます。ストップロスとテイクプロフィットは最後のエントリー価格から適用されます。

## インジケーター
この戦略は別のインジケーターファイルを必要としません。以下を使用してシグナルを計算します：
- **Williams %R** 期間 `SSP`。
- **ATR** 固定期間15。
Williams %Rが閾値 `-100 + K` と `-K`（`K = 33 - Risk`）をクロスしたときにトレンドが切り替わります。

## 取引ルール
1. 強気シグナルで、ショートポジションを閉じ（許可されている場合）、ロングポジションを開きます。
2. 弱気シグナルで、ロングポジションを閉じ（許可されている場合）、ショートポジションを開きます。
3. ポジションが開いている間、`Max Positions`に達するまで `Price Step` 単位ごとに同方向への新規ポジションが追加されます。
4. 各ポジションには最後のエントリー価格から計測されたストップロスとテイクプロフィット距離があります。

## パラメーター
- `Risk` – インジケーターの閾値を定義するリスクファクター。
- `SSP` – Williams %Rの期間。
- `Price Step` – 新規ポジションを追加するための価格距離。
- `Max Positions` – 片側の最大オープンポジション数。
- `Stop Loss` – 価格単位でのストップロス距離。
- `Take Profit` – 価格単位でのテイクプロフィット距離。
- `Enable Long Open` – ロングポジションの開設を許可。
- `Enable Short Open` – ショートポジションの開設を許可。
- `Enable Long Close` – 逆シグナルでのロングポジションの決済を許可。
- `Enable Short Close` – 逆シグナルでのショートポジションの決済を許可。
- `Candle Type` – 計算に使用する時間軸。
