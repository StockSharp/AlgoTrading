# Donky MA TP SL
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пересечения скользящих средних с двумя уровнями фиксации прибыли и стоп-лоссом. Покупка происходит при пересечении быстрой SMA выше медленной, продажа — при обратном пересечении. Половина позиции закрывается на первом уровне, остаток — на втором уровне или по стоп-лоссу.

## Детали

- **Условия входа**:
  - **Лонг**: Быстрая SMA пересекает медленную снизу вверх.
  - **Шорт**: Быстрая SMA пересекает медленную сверху вниз.
- **Длинные/короткие позиции**: Обе стороны.
- **Условия выхода**: Два фиксированных тейк-профита или фиксированный стоп-лосс.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `FastLength` = 10
  - `SlowLength` = 30
  - `TakeProfit1Pct` = 0.03m
  - `TakeProfit2Pct` = 0.06m
  - `StopLossPct` = 0.01m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Trend
  - Направление: Both
  - Индикаторы: SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
