# JSatl Digit System
[English](README.md) | [中文](README_cn.md)

JSatl Digit System использует сглаженную среднюю Jurik (JMA) для определения направления тренда.
Стратегия измеряет наклон JMA и открывает позицию, когда цена подтверждает направление наклона.

Длинная позиция открывается, когда JMA растет и цена закрытия выше средней.
Короткая позиция открывается, когда JMA падает и цена закрытия ниже средней.
Противоположный сигнал закрывает позицию.

## Детали

- **Критерии входа**: наклон JMA с подтверждением цены.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `JmaLength` = 14
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: JMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Свинг (4h)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
