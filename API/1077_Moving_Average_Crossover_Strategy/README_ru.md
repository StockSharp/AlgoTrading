# Стратегия Moving Average Crossover
[English](README.md) | [中文](README_cn.md)

Покупает, когда короткая SMA пересекает длинную SMA снизу вверх, и продаёт при обратном пересечении. Позиции переворачиваются при противоположных сигналах.

## Детали

- **Условия входа**:
  - Лонг при пересечении короткой SMA выше длинной.
  - Шорт при пересечении короткой SMA ниже длинной.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: переворот при противоположном пересечении.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `ShortLength` = 9
  - `LongLength` = 21
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Crossover
  - Направление: Обе стороны
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
