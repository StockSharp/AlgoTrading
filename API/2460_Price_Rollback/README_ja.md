# 価格リバウンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は日次の価格ギャップを取引します。
選択した曜日の開始時に、直近の終値と24時間前の始値を比較します。
ギャップが**Corridor**パラメーターを超えた場合、リバウンドの方向にポジションを開きます：

- 上方ギャップ → 売り。
- 下方ギャップ → 買い。

取引では価格単位で固定のストップロスとテイクプロフィットを使用します。
ポジションが利益方向に動いた後、ステップ付きのトレーリングストップが適用されます。
すべてのポジションは1日の終わり近く（22:45）に決済されます。

## パラメーター
- `Corridor` – ギャップの閾値。
- `StopLoss` – 固定損失距離。
- `TakeProfit` – 固定利益目標。
- `TrailingStop` – トレーリング距離。
- `TrailingStep` – トレーリングを更新するために必要な動き。
- `TradingDay` – 取引を開く曜日（0=日曜日）。
- `CandleType` – 計算に使用する時間軸。
