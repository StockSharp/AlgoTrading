# 价格回撤策略

该策略用于交易日内价格缺口。
在选定的交易日开始时，比较上一根收盘价与24小时前的开盘价。
当差值超过 **Corridor** 参数时，在期望回撤的方向开仓：

- 向上跳空 → 做空。
- 向下跳空 → 做多。

交易使用固定的止损和止盈距离。
当持仓盈利时启用带步长的跟踪止损。
所有持仓在接近收盘时段（22:45）被关闭。

## 参数
- `Corridor` – 缺口阈值。
- `StopLoss` – 固定止损距离。
- `TakeProfit` – 固定止盈目标。
- `TrailingStop` – 跟踪止损距离。
- `TrailingStep` – 更新跟踪止损所需的移动。
- `TradingDay` – 开仓的星期几（0=周日）。
- `CandleType` – 计算所用的时间框架。
