# Sunil BB Blast Heikin Ashi Strategy
[English](README.md) | [中文](README_cn.md)

Комбинирует пробой полос Боллинджера с подтверждением свечами Хейкина Аши.

Стратегия ждёт пробоя полосы Боллинджера в направлении предыдущих свечей Хейкина Аши и обычной свечи. Позиции используют противоположную полосу как стоп и цель на основе соотношения риск/прибыль.

## Детали

- **Условия входа**: цена пробивает полосы Боллинджера при совпадении направления предыдущих свечей.
- **Длинные/короткие**: задаётся параметром `Direction`.
- **Условия выхода**: тейк-профит или стоп-лосс по полосам.
- **Стопы**: полоса Боллинджера и соотношение риск/прибыль.
- **Значения по умолчанию**:
  - `BollingerPeriod` = 19
  - `BollingerMultiplier` = 2m
  - `RiskRewardRatio` = 1m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `Direction` = TradeDirection.Both
  - `SessionBegin` = 09:20:00
  - `SessionEnd` = 15:00:00
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Bollinger, HeikinAshi
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
