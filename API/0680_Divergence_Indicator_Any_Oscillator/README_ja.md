# ダイバージェンスインジケーター（任意のオシレーター）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

価格とRSIの間の通常のダイバージェンスと隠れたダイバージェンスを検出します。戦略は強気ダイバージェンスで買い、弱気ダイバージェンスで売ります。

## パラメーター
- **Pivot Left** – ピボット左側のバー数
- **Pivot Right** – ピボット右側のバー数
- **Min Range** – ピボット間の最小バー数
- **Max Range** – ピボット間の最大バー数
- **RSI Length** – RSIの期間
- **Candle Type** – ローソク足シリーズのタイプ

## インジケーター
- RSI

## ルール
- **エントリー**:
  - 価格が安値を切り下げているのにRSIが安値を切り上げているとき（強気ダイバージェンス）に買い
  - 価格が高値を切り上げているのにRSIが高値を切り下げているとき（弱気ダイバージェンス）に売り
  - 隠れたダイバージェンスは反対方向で取引
