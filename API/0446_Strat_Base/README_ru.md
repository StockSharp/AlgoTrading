# Базовый шаблон стратегии
[English](README.md) | [中文](README_cn.md)

Эта папка содержит минимальный каркас для создания собственных торговых идей.
Стратегия рассчитывает одну экспоненциальную скользящую среднюю и предоставляет
набор типовых параметров: включение лонгов и шортов, опциональный тейк‑профит и
стоп‑лосс, диапазоны оптимизации. Разработчик может вставить свою логику входа и
выхода в соответствующие заглушки и быстро протестировать новые подходы.

Пример также показывает запуск встроенного модуля защиты с целями в процентах,
что упрощает экспериментирование с управлением риском. Поскольку реальные
сигналы отсутствуют, скрипт не предназначен для немедленной торговли, а служит
отправной точкой для дальнейших исследований.

## Детали

- **Условия входа**: не реализованы – замените собственными правилами.
- **Лонг/Шорт**: настраивается параметрами.
- **Условия выхода**: не реализованы – замените собственными правилами.
- **Стопы**: опциональные тейк‑профит и стоп‑лосс в процентах через модуль защиты.
- **Параметры по умолчанию**:
  - Длина EMA = 10.
  - Тейк‑профит = 1.2%, стоп‑лосс = 1.8% (по умолчанию отключены).
- **Фильтры**:
  - Категория: шаблон
  - Направление: настраиваемое
  - Индикаторы: EMA
  - Стопы: опционально
  - Сложность: низкая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: задаётся пользователем
