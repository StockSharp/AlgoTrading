# Heikin Ashi ユニバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このユニバーサルテンプレートは、標準ローソク足をHeikin Ashiローソク足に変換し、その実体の方向に取引します。この手法は価格ノイズを平滑化し、トレンドをより明確に表示します。軽量でカスタムフィルターやエグジットのベースとして使用できます。

Heikin Ashiの終値が始値を上回るときにロングでエントリーし、終値が始値を下回ったときにショートに切り替えます。

## 詳細

- **エントリー条件**:
  - **ロング**: `HA_Close > HA_Open`
  - **ショート**: `HA_Close < HA_Open`
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 逆シグナル
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = 1 minute
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Heikin Ashi
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
