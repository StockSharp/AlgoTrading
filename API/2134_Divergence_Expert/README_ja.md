# ダイバージェンス・エキスパート
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSI価格ダイバージェンスを取引する戦略。価格が安値を更新しながらRSIが高い安値を形成する場合は強気ダイバージェンスを検出し、価格が高値を更新しながらRSIが低い高値を形成する場合は弱気ダイバージェンスを検出します。それに応じてロングまたはショートポジションを取り、パーセントベースのストップロスを使用します。

## 詳細

- **エントリー条件:**
  - ロング: 価格が新安値を形成し、RSIがより高い安値を形成する（強気ダイバージェンス）
  - ショート: 価格が新高値を形成し、RSIがより低い高値を形成する（弱気ダイバージェンス）
- **ロング/ショート:** 両方
- **エグジット条件:**
  - ロング: 価格がストップロスに達するか、弱気ダイバージェンスが現れる
  - ショート: 価格がストップロスに達するか、強気ダイバージェンスが現れる
- **ストップ:** エントリー価格からのパーセント
- **デフォルト値:**
  - `RsiPeriod` = 14
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター:**
  - カテゴリ: ダイバージェンス
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: なし
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
