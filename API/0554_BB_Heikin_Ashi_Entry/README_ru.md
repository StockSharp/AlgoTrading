# Стратегия Bollinger Heikin Ashi Entry
[English](README.md) | [中文](README_cn.md)

Стратегия использует полосы Боллинджера на свечах Хейкин-Аши. Покупает после двух подряд медвежьих свечей Хейкин-Аши, коснувшихся нижней полосы, и последующей бычьей свечи выше неё. Продаёт наоборот.

После входа фиксируется первая цель размером с риск и стоп подтягивается по минимумам/максимумам предыдущих свечей.

## Детали

- **Условия входа**:
  - Long: две медвежьи HA-свечи, касающиеся нижней полосы, затем бычья выше неё
  - Short: две бычьи HA-свечи, касающиеся верхней полосы, затем медвежья ниже неё
- **Направление**: Оба
- **Условия выхода**:
  - Long: первая цель = 1R, затем трейлинг по минимумам
  - Short: первая цель = 1R, затем трейлинг по максимумам
- **Стопы**: минимум/максимум предыдущей свечи
- **Значения по умолчанию**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: Bollinger Bands, Heikin Ashi
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
