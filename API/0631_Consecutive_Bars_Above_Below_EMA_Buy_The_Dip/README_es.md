# Estrategia de Compra en Caída con Barras Consecutivas Por Encima/Debajo de EMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia compra después de varios cierres consecutivos por debajo de una media móvil y sale cuando el precio cierra por encima del máximo de la vela anterior.

## Detalles

- **Entrada**: Comprar cuando N barras consecutivas cierran por debajo de SMA o EMA.
- **Salida**: Cerrar la posición cuando el precio de cierre supera el máximo de la vela anterior.
- **Indicadores**: SMA o EMA.
