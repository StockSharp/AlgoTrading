# Aufeinanderfolgende Balken Ober-/Unterhalb EMA Dip-Kauf-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie kauft nach mehreren aufeinanderfolgenden Schlusskursen unterhalb eines gleitenden Durchschnitts und steigt aus, wenn der Kurs über das Hoch der vorherigen Kerze schließt.

## Details

- **Einstieg**: Kaufen, wenn N aufeinanderfolgende Balken unterhalb von SMA oder EMA schließen.
- **Ausstieg**: Position schließen, wenn der Schlusskurs das Hoch der vorherigen Kerze übersteigt.
- **Indikatoren**: SMA oder EMA.
