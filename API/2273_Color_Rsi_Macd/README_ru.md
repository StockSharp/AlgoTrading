# Стратегия Color RSI MACD
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор MACD и поддерживает четыре режима анализа:

- **Breakdown** – вход при пересечении гистограммой MACD нулевой линии.
- **MACD Twist** – вход при изменении направления линии MACD.
- **Signal Twist** – вход при изменении направления сигнальной линии.
- **MACD Disposition** – вход при пересечении линии MACD и сигнальной линии.

Каждый режим позволяет независимо открывать и закрывать длинные и короткие позиции.

По умолчанию стоп-лосс и тейк-профит не используются.

## Детали

- **Критерий входа**: сигнал индикатора
- **Лонг/Шорт**: оба направления
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = 4 часа
  - `FastPeriod` = 12
  - `SlowPeriod` = 26
  - `SignalPeriod` = 9
  - `Mode` = MACD Disposition
- **Фильтры**:
  - Категория: следование тренду
  - Направление: оба
  - Индикаторы: MACD
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
