# Cnagda Fixed Swing Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Eine Strategie, die Heikin-Ashi-Kerzen mit zwei Modi verwendet:
- **RSI**: Einstiege, wenn der kurze EMA des RSI den langen EMA bei hohem Volumen kreuzt.
- **Scalp**: Einstiege basierend auf EMA- und WMA-Kreuzungen des Heikin-Ashi-Schlusskurses.

Der Stop Loss wird beim letzten Swing-Hoch oder -Tief gesetzt und der Take Profit verwendet ein festes Risiko/Rendite-Vielfaches.

## Parameter
- Kerzentyp
- Handelslogik
- Swing-Rückblick
- Risiko/Rendite
