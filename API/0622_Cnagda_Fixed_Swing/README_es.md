# Estrategia Cnagda Fixed Swing
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Una estrategia que utiliza velas Heikin Ashi con dos modos:
- **RSI**: entradas cuando la EMA corta del RSI cruza la EMA larga con volumen alto.
- **Scalp**: entradas basadas en cruces de EMA y WMA del cierre de Heikin Ashi.

El stop loss se coloca en el swing alto o bajo reciente y el take profit utiliza un múltiplo fijo de riesgo/recompensa.

## Parámetros
- Tipo de vela
- Lógica de operación
- Retroceso de swing
- Riesgo/recompensa
