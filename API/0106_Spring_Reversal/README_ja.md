# スプリング・リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
スプリング・リバーサルはワイコフのコンセプトで、価格がサポートを一時的に割り込んだ後、その上に跳ね返る動きです。
このシェイクアウトは遅い売り手を罠にかけ、上昇トレンドの始まりを示すことが多いです。

テストでは年平均リターンが約55%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

この戦略は、価格が割り込んだレベルを回復した時点で買いに入り、急速な空売り解消と新たな需要を期待します。

スプリング安値のすぐ下のストップが下値リスクを制限し、継続が見られない場合はポジションを閉じます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: Wyckoff
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
