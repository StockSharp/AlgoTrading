# Стратегия Spring Reversal

Spring Reversal — это концепция Вайкоффа, при которой цена кратковременно пробивает поддержку и затем быстро возвращается выше.
Такой "выброс" ставит в тупик поздних продавцов и зачастую отмечает начало восходящего тренда.

Стратегия покупает, как только цена восстанавливает пробитый уровень, ожидая резкого закрытия шортов и притока спроса.

Стоп размещается чуть ниже минимума spring, ограничивая убыток; позиция закрывается, если развития не происходит.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Разворот
  - Направление: Обе
  - Индикаторы: Wyckoff
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
