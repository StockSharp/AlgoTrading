# Estratégia de Rompimento SMC BB
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina o rompimento das Bandas de Bollinger com mudanças de swing de Smart Money Concepts. Uma posição é aberta quando o preço cruza a Banda de Bollinger na direção de uma mudança de estrutura de mercado. Opcionalmente, a entrada requer uma vela de momentum com um corpo grande. As posições são fechadas em um movimento de retorno à banda média ou um rompimento além dos níveis de blocos de ordens recentes.
