# SMC BB Ausbruch-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert den Bollinger Bands-Ausbruch mit Smart Money Concepts-Swing-Wechseln. Eine Position wird eröffnet, wenn der Preis das Bollinger Band in Richtung eines Marktstrukturwechsels kreuzt. Optional erfordert der Einstieg eine Momentum-Kerze mit einem großen Körper. Positionen werden bei einer Rückkehr zum mittleren Band oder einem Ausbruch über aktuelle Order-Block-Niveaus geschlossen.
