# Прорыв наклона объёма
[English](README.md) | [中文](README_zh.md)

Стратегия Volume Slope Breakout отслеживает скорость изменения объёма. Чрезмерно резкий наклон может сигнализировать о начале нового тренда.

Позиция открывается, когда наклон превышает типичный уровень на несколько стандартных отклонений. Торговля ведётся в направлении ускорения и сопровождается стопом.

Стратегия рассчитана на активных трейдеров, стремящихся рано войти в тренд. Позиции закрываются, когда наклон возвращается к нормальным значениям. Значение по умолчанию `VolumeSMAPeriod` = 20.

## Подробности

- **Условие входа**: Индикатор превышает среднее значение на величину коэффициента отклонения.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `VolumeSMAPeriod` = 20
  - `SlopePeriod` = 20
  - `BreakoutMultiplier` = 2.0m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Volume
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 151%\. Стратегию лучше запускать на фондовом рынке.
