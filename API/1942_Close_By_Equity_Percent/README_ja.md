# 資産パーセントによるクローズ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このリスク管理戦略はポートフォリオの資産を監視し、資産が現在の残高にユーザー定義の乗数を掛けた値を上回った場合に、オープンポジションをクローズします。口座価値が基準値に対して希望するパーセンテージに達した時点で利益を確定するよう設計されています。

この戦略はローソク足を使用して定期的にチェックを行い、自らはトレードエントリーを生成しません。既存のポジションのみを管理します。クローズ後は基準残高が更新され、後続のトレードに対してプロセスが繰り返されます。

## 詳細

- **エントリー条件**: なし（既存ポジションを管理）。
- **ロング/ショート**: 両方。
- **エグジット条件**: 資産が `balance * EquityPercentFromBalance` を上回る。
- **ストップ**: なし。
- **デフォルト値**:
  - `EquityPercentFromBalance` = 1.2m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: リスク管理
  - 方向: 両方
  - インジケーター: なし
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ (1m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
