# Стратегия The Z-Score
[English](README.md) | [中文](README_cn.md)

Стратегия вычисляет z-score EMA на базе свечей Heikin-Ashi и торгует по пересечениям динамических уровней, рассчитанных из последнего диапазона значений.

## Детали

- **Критерий входа**: пересечение score выше недавнего минимума или EMA score выше средней линии
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: EMA score пересекает вниз недавний максимум или минимум
- **Стопы**: нет
- **Значения по умолчанию**:
  - `HaEmaLength` = 10
  - `ScoreLength` = 25
  - `ScoreEmaLength` = 20
  - `RangeWindow` = 100
- **Фильтры**:
  - Категория: Средняя реверсия
  - Направление: Обе стороны
  - Индикаторы: EMA, SMA, StdDev, Highest, Lowest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
