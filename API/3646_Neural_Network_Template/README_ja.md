# ニューラル ネットワーク テンプレート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、RSI および MACD の機能をニューラル ネットワークにフィードする MQL5 エキスパート アドバイザー テンプレートの動作を複製します。 StockSharp には元のプロジェクトのカスタム ネットワーク ローダーが同梱されていないため、この戦略では、同じ市場構造とリスク管理を維持しながら、ブラックボックス ネットワークを決定論的なスコアリング モデルに置き換えます。目標は、RSI と MACD の両方が方向性に同意し、予想される動きが取引を正当化するのに十分な大きさであるときに勢いを捉えることです。

## 指標とデータ
- **相対強度指数 (RSI、12 期間)** はローソク足の終値で計算され、元の典型的な価格入力を反映しています。
- **Moving Average Convergence Divergence (MACD 12/48/12)** は、運動量ヒストグラムおよび信頼度の代用として使用されます。
- **時間枠** 構成可能。デフォルトはソースの専門家と一致する 5 分足のローソク足です。

## 取引ロジック
1. 終了したローソクごとに、戦略は `BarsToPattern` によって制御されるウィンドウを使用して、RSI と MACD のヒストグラム値のローリング キューを更新します。
2. 50 からの RSI の偏差と、ローリング平均からの MACD ヒストグラムの偏差は、双曲線正接を使用して信頼スコアに結合され、ネットワークのスカッシング関数をエミュレートします。
3. 絶対信頼度が `TradeLevel` を超え、ポイントに変換された予測移動が `MinTargetPoints` を超えている場合、ストラテジーはスコアが示唆する方向に成行注文を発行します。
4. 予想される動きに `ProfitMultiply` を乗算し、`MaxTakeProfitPoints` を上限とする動的なテイクプロフィットが、手動の手仕舞い処理のために保存されます。ポイント単位の対称ストップロスは、元の動作を反映します。
5. ポジションがオープンしている間、ストラテジーは終了したローソク足をすべてチェックします。価格が保存されているストップまたはターゲットに達すると、市場でポジションをクローズし、内部状態をリセットします。

## パラメーター
| パラメータ | 説明 |
| --- | --- |
| `BarsToPattern` | RSI および MACD 統計の計算に使用されるローリング ウィンドウに保存されているローソク足の数。 |
| `TradeLevel` | ポジションをオープンするために必要な最小の信頼度 (0 ～ 1)。 |
| `ProfitMultiply` | `MaxTakeProfitPoints` でキャップする前に、予想される動きに適用される乗数。 |
| `MinTargetPoints` | 取引に参加するために予測から必要な価格ポイントの最小数。 |
| `MaxTakeProfitPoints` | テイクプロフィットに許可される最大距離 (ポイント単位)。 |
| `StopLossPoints` | エントリー価格に対する保護ストップまでの距離 (ポイント単位)。 |
| `TradeVolume` | すべての成行注文で送信されるボリューム。 |
| `CandleType` | サブスクライブするローソク足のデータ タイプまたはタイムフレーム。 |

## 注意事項
- 信頼モデルは、元のニューラル ネットワーク アプローチの構造を維持しながら動作の透明性を保つために、意図的に決定論的になっています。
- MQL5 バージョンがネットワーク出力を使用する方法と同様に、各取引が独自の動的ターゲットを維持できるように、テイクプロフィットとストップロスのレベルは手動で管理されます。
- この戦略は、オープンなポジションがない場合にのみ新しいエントリーを評価し、ソースエキスパートアドバイザーの単一ポジション制約を複製します。
