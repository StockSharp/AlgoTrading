# Color XCCX キャンドル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MQL コード `MQL/14260` から変換されました。

この戦略は、ローソク足の始値と終値から構築された2本の単純移動平均（SMA）を比較します。終値から計算されたSMAが始値に基づくSMAを上抜けすると、ロングポジションが開かれます。終値ベースのSMAが始値ベースのSMAを下抜けすると、ショートポジションが開かれます。新しいポジションを開く前に、既存の反対ポジションは閉じられます。

パラメーター:

- `SMA Length` – 両方のSMAを計算するために使用するローソク足の本数。
- `Candle Type` – 受信するローソク足の時間軸。
- `Stop Loss %` – エントリー価格のパーセントとして表したストップロスのサイズ。
- `Take Profit %` – エントリー価格のパーセントとして表したテイクプロフィットのサイズ。

この戦略はStockSharpの高レベルAPIを使用してローソク足をサブスクライブし、インジケーターをバインドします。また、ビジュアライゼーションが利用可能な場合、チャートに両方のSMAと約定済みトレードをプロットします。
