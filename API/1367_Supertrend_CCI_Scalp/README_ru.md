# Стратегия Supertrend & CCI Strategy Scalp
[English](README.md) | [中文](README_cn.md)

Стратегия использует две линии Supertrend и сглаженный CCI для поиска краткосрочных разворотов.
Покупка выполняется, когда первая линия Supertrend выше цены, вторая ниже цены, а сглаженный CCI опускается ниже -100. Логика шорта зеркальна.

## Детали

- **Критерий входа**: Supertrend1 выше цены, Supertrend2 ниже цены, сглаженный CCI < -100 (лонг); противоположное условие для шорта
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: обратное расположение Supertrend или пересечение CCI уровня ±100
- **Стопы**: нет
- **Значения по умолчанию**:
  - `AtrLength1` = 14
  - `Factor1` = 3
  - `AtrLength2` = 14
  - `Factor2` = 6
  - `CciLength` = 20
  - `SmoothingLength` = 5
  - `MaType` = MovingAverageTypeEnum.Simple
  - `CciLevel` = 100
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: Supertrend, CCI, Moving Average
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
