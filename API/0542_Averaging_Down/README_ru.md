# Стратегия Averaging Down
[English](README.md) | [中文](README_cn.md)

Стратегия Averaging Down открывает длинные позиции, когда индикатор RSI опускается ниже заданного уровня. Каждое срабатывание добавляет к уже открытой позиции, усредняя цену входа. Выход происходит, когда цена закрытия превышает максимум предыдущей свечи.

## Подробности

- **Условия входа**:
  - RSI ниже `RsiBuyThreshold`.
- **Длинные/Короткие**: Только длинные.
- **Условия выхода**:
  - Цена закрытия выше максимума предыдущей свечи.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `RsiLength` = 10
  - `RsiBuyThreshold` = 33
- **Фильтры**:
  - Категория: Средний возврат
  - Направление: Long
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
