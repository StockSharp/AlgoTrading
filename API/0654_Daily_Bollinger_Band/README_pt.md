# Estratégia Diária de Bollinger Band
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia que opera rompimentos das Bollinger Bands diárias usando um filtro de tendência e dimensionamento de posição baseado em ATR.

## Detalhes

- **Critérios de entrada**: O preço cruza a banda superior com inclinações positivas para comprado ou a banda inferior com inclinações negativas para vendido.
- **Critérios de saída**: Fechar a posição quando o preço cruza a banda do meio.
