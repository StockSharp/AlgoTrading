# Tägliche Bollinger Band-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Strategie, die tägliche Bollinger Band-Ausbrüche mit einem Trendfilter und ATR-basierter Positionsgröße handelt.

## Details

- **Einstiegskriterien**: Preis kreuzt das obere Band bei positiver Neigung für Long oder das untere Band bei negativer Neigung für Short.
- **Ausstiegskriterien**: Position schließen, wenn der Preis das mittlere Band kreuzt.
