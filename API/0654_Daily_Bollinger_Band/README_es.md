# Estrategia Diaria de Bollinger Band
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Estrategia que opera rompimientos de las Bollinger Bands diarias usando un filtro de tendencia y dimensionamiento de posición basado en ATR.

## Detalles

- **Criterios de entrada**: El precio cruza la banda superior con pendientes positivas para largo o la banda inferior con pendientes negativas para corto.
- **Criterios de salida**: Cerrar la posición cuando el precio cruza la banda media.
