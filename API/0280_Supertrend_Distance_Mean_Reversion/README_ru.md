# Средняя коррекция расстояния Supertrend
[English](README.md) | [中文](README_zh.md)

Стратегия Supertrend Distance Mean Reversion ориентирована на экстремальные значения индикатора Supertrend для игры на возврат к среднему. Значительные отклонения от обычного уровня редко держатся долго.

Сделки открываются, когда индикатор сильно отклоняется от своей средней и начинает обратное движение. Возможны длинные и короткие позиции с защитным стопом.

Подходит свинг-трейдерам, ожидающим колебаний. Закрытие происходит, когда Supertrend возвращается к балансу. Начальный параметр `AtrPeriod` = 10.

## Подробности

- **Условие входа**: Индикатор возвращается в сторону средней.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `AtrPeriod` = 10
  - `Multiplier` = 3.0m
  - `LookbackPeriod` = 20
  - `DeviationMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Оба
  - Индикаторы: Supertrend
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
