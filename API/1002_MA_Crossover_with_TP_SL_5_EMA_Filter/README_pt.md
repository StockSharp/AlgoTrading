# Cruzamento de MA com TP/SL e Filtro de 5 EMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia entra comprada quando uma SMA rápida cruza acima de uma SMA lenta enquanto o preço de fechamento está acima de uma EMA de 5 períodos. Entra vendida na condição oposta. A estratégia gerencia posições usando níveis de take-profit e stop-loss baseados em percentual.

## Parâmetros
- Fast MA Length
- Slow MA Length
- EMA Length
- Target %
- Stop %
- Candle Type
