# MA-Crossover-Strategie mit TP/SL und 5-EMA-Filter
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie geht Long, wenn eine schnelle SMA eine langsame SMA von unten kreuzt und der Schlusskurs über einer 5-Perioden-EMA liegt. Bei umgekehrter Bedingung geht sie Short. Die Strategie verwaltet Positionen mit prozentualen Take-Profit- und Stop-Loss-Niveaus.

## Parameter
- Fast MA Length
- Slow MA Length
- EMA Length
- Target %
- Stop %
- Candle Type
