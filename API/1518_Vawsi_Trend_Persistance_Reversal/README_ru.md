# Стратегия VAWSI и разворот по устойчивости тренда
[English](README.md) | [中文](README_cn.md)

Разворотная стратегия, комбинирующая VAWSI, устойчивость тренда и ATR для построения динамического порога на свечах Хейкин-Аши.

## Детали

- **Критерий входа**: закрытие Хейкин-Аши пересекает динамический порог
- **Лонг/шорт**: оба направления
- **Критерий выхода**: обратное пересечение или защитные стопы
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `SlTp` = 5
  - `RsiWeight` = 100
  - `TrendWeight` = 79
  - `AtrWeight` = 20
  - `CombinationMult` = 1
  - `Smoothing` = 3
  - `CycleLength` = 20
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: RSI, ATR
  - Стопы: Да
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
