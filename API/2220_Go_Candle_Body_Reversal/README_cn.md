# Go Candle Body Reversal 策略
[English](README.md) | [Русский](README_ru.md)

该策略基于 Go 指标，对蜡烛实体大小进行平滑。当实体 SMA 从正值转为负值时做多，反向交叉时做空。当前持仓在出现反向信号时平仓。

## 细节

- **入场条件**: 实体 SMA 符号变化（正→负做多，负→正做空）
- **多空方向**: 双向
- **出场条件**: 实体 SMA 的相反符号变化
- **止损**: 无
- **默认值**:
  - `Period` = 174
  - `CandleType` = 1 小时
- **过滤器**:
  - 分类: Reversal
  - 方向: 多空
  - 指标: SMA
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
