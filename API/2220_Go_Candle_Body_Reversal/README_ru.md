# Стратегия Go Candle Body Reversal
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторе Go, который усредняет размер тела свечи. Открывает длинную позицию, когда сглаженное тело свечи переходит из положительной области в отрицательную, и короткую позицию при обратном переходе. Текущие позиции закрываются по противоположным сигналам.

## Детали

- **Критерий входа**: изменение знака SMA тела свечи (из + в − для покупки, из − в + для продажи)
- **Длин/Шорт**: оба направления
- **Критерий выхода**: противоположное изменение знака SMA тела свечи
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Period` = 174
  - `CandleType` = 1 час
- **Фильтры**:
  - Категория: Reversal
  - Направление: Long & Short
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
