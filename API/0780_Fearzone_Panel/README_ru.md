# Fearzone Panel
[English](README.md) | [中文](README_cn.md)

Стратегия, основанная на панели FearZone из книги «Framgångsrik Aktiehandel». Ищет экстремальные распродажи, когда рынок движим страхом.

Стратегия ждёт, когда оба индикатора Fearzone активны и сработает хотя бы один из триггеров паники, при этом цена остаётся выше скользящей средней 200.

## Детали

- **Условия входа**: FZ1 и FZ2 активны плюс отрицательный импульс, зона рикошета или стохастик ниже порога, при закрытии выше MA200.
- **Длинные/короткие**: только длинные.
- **Условия выхода**: цена падает ниже MA200.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `LookbackPeriod` = 22
  - `BollingerPeriod` = 200
  - `ImpulsePeriod` = 10
  - `ImpulsePercent` = 0.1m
  - `MaPeriod` = 200
  - `StochThreshold` = 30m
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Long
  - Индикаторы: BollingerBands, RateOfChange, StochasticOscillator, SimpleMovingAverage, Highest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
