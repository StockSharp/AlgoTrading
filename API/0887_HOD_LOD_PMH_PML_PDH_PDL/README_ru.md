# Стратегия HOD/LOD/PMH/PML/PDH/PDL
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробои уровней пре-маркета и прошлого дня.
Длинная позиция открывается при пересечении ценой пре-маркет или максимума прошлого дня.
Короткая позиция открывается при пересечении ценой пре-маркет или минимума прошлого дня.
Позиции закрываются при достижении ценой максимума или минимума текущего дня.

## Детали

- **Критерий входа**: пересечение уровней пре-маркета или прошлого дня
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: достижение максимума или минимума текущего дня
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Уровни
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
