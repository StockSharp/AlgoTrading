# HOD/LOD/PMH/PML/PDH/PDL 策略
[English](README.md) | [Русский](README_ru.md)

该策略利用盘前和前一天的关键水平突破进行交易。
当价格向上突破盘前高点或前一天高点时做多。
当价格向下跌破盘前低点或前一天低点时做空。
当价格达到当日高点或低点时平仓。

## 细节

- **入场条件**: 突破盘前或前一天的水平
- **多头/空头**: 双向
- **出场条件**: 触及当日高点或低点
- **止损**: 无
- **默认值**:
  - `CandleType` = 5 分钟
- **过滤器**:
  - 分类: 水平
  - 方向: 双向
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
