# HOD/LOD/PMH/PML/PDH/PDL 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、プレマーケットおよび前日の水準のブレイクアウトを取引します。
ロングエントリーは、価格がプレマーケットまたは前日高値を上抜けたときに発生します。
ショートエントリーは、価格がプレマーケットまたは前日安値を下抜けたときに発生します。
価格が当日高値または安値に達したときにポジションをクローズします。

## 詳細

- **エントリー条件**: 価格がプレマーケットまたは前日の水準をクロス
- **ロング/ショート**: 両方
- **エグジット条件**: 当日高値または安値に到達
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = 5分
- **フィルター**:
  - カテゴリ: レベル
  - 方向: 両方
  - インジケーター: なし
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
