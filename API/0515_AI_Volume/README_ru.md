# Стратегия AI Volume
[English](README.md) | [中文](README_cn.md)

Стратегия AI Volume отслеживает внезапные всплески объёма. Вход выполняется, когда текущий объём превышает свою EMA во столько раз, сколько задано множителем, цена находится выше/ниже 50 EMA, а цвет свечи подтверждает движение. Каждая сделка закрывается через фиксированное число баров.

## Подробности

- **Условия входа**: Объём > VolumeEMA × VolumeMultiplier и цена выше/ниже 50 EMA с подтверждением цвета свечи.
- **Направление**: Обе стороны.
- **Условия выхода**: Закрытие позиции через `ExitBars` свечей.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `VolumeEmaLength` = 20
  - `VolumeMultiplier` = 2.0
  - `ExitBars` = 5
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Объёмный пробой
  - Направление: Обе
  - Индикаторы: EMA, Volume EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
