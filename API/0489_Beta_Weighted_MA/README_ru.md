# Стратегия Beta Weighted MA
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия Beta Weighted MA (BWMA) использует распределение Бета для взвешивания последних цен, создавая скользящую среднюю с регулируемой задержкой и сглаживанием с помощью параметров alpha и beta. Стратегия открывает длинную позицию при пересечении цены выше BWMA и короткую при пересечении ниже.

## Подробности

- **Условия входа**:
  - Цена пересекает BWMA снизу вверх — вход в лонг.
  - Цена пересекает BWMA сверху вниз — вход в шорт.
- **Направление**: Лонг/Шорт.
- **Условия выхода**:
  - Обратное пересечение закрывает текущую позицию и открывает противоположную.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `Length` = 50
  - `Alpha` = 3
  - `Beta` = 3
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг/Шорт
  - Индикаторы: Beta Weighted Moving Average
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
