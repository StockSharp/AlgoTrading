# Стратегия Stop Loss Mover
[English](README.md) | [中文](README_cn.md)

Эта утилитарная стратегия отслеживает открытую позицию и переносит стоп-лосс на цену входа, когда рынок достигает заданного уровня. Она подписывается на поток свечей и анализирует каждую завершённую свечу. Для длинной позиции после того, как максимум свечи превышает `MoveSlPrice`, по цене входа выставляется стоп-заявка. Для короткой позиции стоп переносится, когда минимум свечи опускается ниже уровня.

Стратегия не генерирует торговых сигналов. В примере она открывает одну длинную позицию при запуске и затем защищает её, перенося стоп в безубыток при выполнении условия. Это позволяет фиксировать прибыль и продолжать удерживать сделку.

## Детали

- **Критерий входа**: При запуске открывается длинная позиция. Дополнительные сигналы не используются.
- **Длинные/короткие**: Поддерживаются оба направления, но пример открывает длинную позицию.
- **Критерий выхода**: Позиция закрывается при срабатывании стоп-заявки на цене входа.
- **Стопы**: Стоп-лосс переносится на цену входа при достижении уровня `MoveSlPrice`.
- **Значения по умолчанию**:
  - `MoveSlPrice` = 0 (необходимо задать перед запуском).
  - `CandleType` = таймфрейм 1 минута.
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Простая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
