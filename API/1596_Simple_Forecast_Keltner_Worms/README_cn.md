# Simple Forecast - Keltner Worms 策略
[English](README.md) | [Русский](README_ru.md)

该策略构建动态凯尔特纳通道，当价格突破通道时进行交易。

## 细节

- **入场条件**：
  - 收盘价高于上轨时做多。
  - 收盘价低于下轨时做空。
- **方向**：多空皆可。
- **出场条件**：
  - 反向信号平仓。
- **止损**：无。
- **默认参数**：
  - `Length` = 10
- **过滤器**：
  - 类型：通道
  - 方向：双向
  - 指标：EMA, ATR
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
