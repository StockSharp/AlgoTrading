# Стратегия Simple Forecast - Keltner Worms
[English](README.md) | [中文](README_cn.md)

Стратегия строит динамический канал Келтнера и входит в сделки, когда цена выходит за его пределы.

## Подробности

- **Условия входа**:
  - Цена закрытия выше верхнего канала — открывается лонг.
  - Цена закрытия ниже нижнего канала — открывается шорт.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Противоположный сигнал закрывает позицию.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `Length` = 10
- **Фильтры**:
  - Категория: Канал
  - Направление: Оба
  - Индикаторы: EMA, ATR
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
