# NRTR ATR Stop戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

**NRTR ATR Stop戦略**は、MetaTraderエキスパートアドバイザー`Exp_NRTR_ATR_STOP_Tm`の直接変換です。このシステムは、非再描画トレンドリバーサル（NRTR）ストップと平均真の値幅（ATR）フィルターを組み合わせて、支配的なトレンドを判断し、保護レベルをトレーリングします。取引決定は選択した時間軸のクローズ時に生成され、元のシグナルシフトを模倣するために完全に形成されたバーの設定可能な数だけ遅延できます。

この戦略はStockSharpの高レベルAPIの上に実装されています。すべての取引ロジックはキャンドルサブスクリプション、インジケーターバインディング、および管理された注文ヘルパーによって駆動され、Designer、Shell、Runner、APIの各製品との互換性を確保しています。

## 取引ロジック

1. **インジケーター計算**
   - ATRは提供された期間で選択された時間軸で計算されます。
   - ATR値に係数を掛けてNRTRの上限と下限レベルを構築します。
   - 前のキャンドルが反対のNRTRレベルを突破すると、トレンド方向が変化します；これらのイベントはエントリーをトリガーできる矢印シグナルも作成します。
2. **シグナル遅延**
   - `SignalBarDelay`パラメーターはMetaTraderの`SignalBar`入力を再現します。選択した数の完成したキャンドルだけ実行を遅延させ、戦略がソースエキスパートと同様に過去のシグナルを評価できるようにします。
3. **エントリー**
   - **ロング**ポジションは、強気のNRTRリバーサルが発生し、ロングエントリーが有効な場合に開かれます。
   - **ショート**ポジションは、弱気のNRTRリバーサルが発生し、ショートエントリーが有効な場合に開かれます。
4. **エグジット**
   - 方向リバーサルは、そのサイドのクローズが許可されている場合、反対のポジションを閉じます。
   - オプションのセッションフィルターは、許可された取引ウィンドウ外のすべてのポジションを強制的に閉じることができます。
   - 追加のリスク管理は価格ステップで表されたストップロスとテイクプロフィット距離によって処理されます。NRTRレベルはトレンドの方向に保護ストップを締めることで、アクティブなポジションもトレーリングします。

## リスク管理

- **ボリューム**: 取引は設定可能な`OrderVolume`パラメーターで開かれます。ボリュームはMetaTraderバージョンと同様に最適化できます。
- **ストップロス / テイクプロフィット**: 距離は証券価格ステップの倍数で指定され、元のポイントベースの設定と一致します。手動ストップとNRTRレベルの両方が利用可能な場合、リスクの拡大を避けるために保護価格は保守的に（市場に最も近い）選択されます。
- **セッション制御**: `UseTradingWindow`が有効な場合、戦略は定義された`[StartHour:StartMinute, EndHour:EndMinute]`間隔内でのみポジションを開き、市場がそのウィンドウを離れるとすぐにオープンポジションを閉じます。

## パラメーター

| 名前 | デフォルト | 説明 |
| --- | --- | --- |
| `OrderVolume` | 1 | 成行注文を送信する際に使用するボリューム。 |
| `StopLossPoints` | 1000 | 価格ステップでのストップ距離。無効にするには`0`に設定。 |
| `TakeProfitPoints` | 2000 | 価格ステップでのテイクプロフィット距離。無効にするには`0`に設定。 |
| `BuyPosOpen` / `SellPosOpen` | `true` | NRTRリバーサルでのロングまたはショートポジションの開設を許可。 |
| `BuyPosClose` / `SellPosClose` | `true` | 反対シグナルが現れたときのロングまたはショートポジションのクローズを許可。 |
| `UseTradingWindow` | `true` | 元のエキスパートアドバイザーを模倣する時間フィルターを有効化。 |
| `StartHour` / `StartMinute` | 0 / 0 | 許可された取引セッションの開始。 |
| `EndHour` / `EndMinute` | 23 / 59 | 許可された取引セッションの終了。オーバーナイト範囲をサポート。 |
| `CandleType` | 1時間時間軸 | ATRとNRTR計算に使用するキャンドルタイプ。 |
| `AtrPeriod` | 20 | ATR計算に使用するバーの数。 |
| `AtrMultiplier` | 2 | NRTRレベル構築時にATRに適用する係数。 |
| `SignalBarDelay` | 1 | シグナル実行を遅延させる完成したバーの数。 |

## 注意事項

- 戦略はキャンドルレベルの処理のみを使用します；元のEAのティック単位の複製は、StockSharpの高レベルアーキテクチャと一致するよう意図的に避けています。
- コード内のコメントはプロジェクトガイドラインに従って英語で提供されています。
- Python版はユーザーリクエストに合わせて意図的に省略されています。
