# Стратегия Levels With Revolve
[English](README.md) | [中文](README_cn.md)

Эта стратегия открывает сделки, когда рыночная цена пересекает заданный уровень. При пробое вверх выполняется покупка, при пробое вниз — продажа. При включённом режиме разворота текущая позиция закрывается и открывается противоположная. Также поддерживаются стоп‑лосс и тейк‑профит в ценовых единицах.

Стратегия подписывается на свечи и реагирует только на полностью сформированные свечи. Все расчёты выполняются по цене закрытия завершившейся свечи. При активном развороте открытая позиция закрывается и переворачивается при появлении противоположного сигнала.

## Подробности

- **Условия входа**:
  - Лонг: цена закрытия пересекает уровень `LevelPrice` снизу вверх.
  - Шорт: цена закрытия пересекает уровень `LevelPrice` сверху вниз.
- **Направление**: обе стороны.
- **Разворот**: опционально через параметр `EnableReversal`.
- **Стопы**: опциональные стоп‑лосс и тейк‑профит.
- **Значения по умолчанию**:
  - `LevelPrice` = 100.
  - `StopLoss` = 0 (отключён).
  - `TakeProfit` = 0 (отключён).
  - `EnableReversal` = false.
  - `CandleType` = таймфрейм 1 минута.
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе
  - Индикаторы: Нет
  - Стопы: Опционально
  - Сложность: Простая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
