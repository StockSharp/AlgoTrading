# NRTR ATR Stop 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
NRTR ATR Stop戦略は、StockSharpの高レベルAPIを使用してMetaTraderのエキスパート `Exp_NRTR_ATR_STOP` の動作を再現します。Average True Range（ATR）から構築されたNRTR（Non-Repainting Trailing Reverse）レベルを追跡します。価格が反対のトレイリングストップを越えると、トレンドが反転し、新しいマーケットエントリーを生成しながら以前の方向のオープンポジションも閉じます。

## インジケーターロジック
* 単一の **Average True Range** (`AtrPeriod`) がサブスクライブされたローソク足シリーズから計算されます。ATR値に `Coefficient` を掛けて、価格と現在のストップレベルの間の距離を生成します。
* 2つの動的なストップラインが維持されます：
  * `upper stop` はロングポジションを保護します。トレンドが強気の間、価格より下でトレイルします。
  * `lower stop` はショートポジションを保護します。トレンドが弱気の間、価格より上でトレイルします。
* 価格が反対のストップを越えてクローズすると、トレンドが即座に反転します。新しい側のストップは、前のローソク足の極値にATR距離を加えた/引いた値で初期化されます。
* オリジナルのエキスパートはインジケーターバッファーの `SignalBar` 本前の値を読み取ることで実行を遅延させます。戦略は内部キューを通じてこの動作を反映します。完成した各ローソク足がシグナルをキューにプッシュし、キューの長さが `SignalBar` を超えた場合にのみエンジンが動作します。

## トレードルール
1. **買いシグナル** – 計算されたトレンドが中立/弱気から強気に変化します。戦略はオプションでショートのエクスポージャーを閉じて、単一の成行注文を使用して新しいロングポジションを開きます。そのボリュームは必要な出口サイズに新しいロングエントリー用の設定済み `Volume` を加えたものに等しいです。
2. **売りシグナル** – トレンドが中立/強気から弱気に変化します。戦略はオプションでロングのエクスポージャーを閉じて、同様に新しいショートポジションを開きます。
3. `EnableLongEntry`、`EnableShortEntry`、`EnableLongExit`、`EnableShortExit` プロパティにより、シグナルが現れたときに実行されるアクションを正確に制御できます。
4. シグナルは完成したローソク足でのみ処理され、戦略がオンラインで取引許可を受けている間のみ処理されます。

## パラメーター
| 名前 | 説明 |
| --- | --- |
| `AtrPeriod` | ATR計算に使用するローソク足の数。 |
| `Coefficient` | トレイリングストップの構築時にATR値に適用する乗数。 |
| `SignalBar` | 保存されたシグナルに対して行動する前に待つ完全に閉じたローソク足の数。現在のローソク足で即座に取引するには `0` に設定します。 |
| `CandleType` | 受信するローソク足の時間軸。 |
| `EnableLongEntry` | 買いシグナル時にロングポジションを開くことを許可します。 |
| `EnableShortEntry` | 売りシグナル時にショートポジションを開くことを許可します。 |
| `EnableLongExit` | 売りシグナルが発生したときにロングポジションを閉じることを許可します。 |
| `EnableShortExit` | 買いシグナルが発生したときにショートポジションを閉じることを許可します。 |

## 注意事項
* 戦略は完成したローソク足のみに依存します。イントラバーのティックは無視されます。
* 注文は `BuyMarket`/`SellMarket` で送信され、ポジションのクローズと新しいエントリーを単純化のために単一の成行注文に組み合わせます。
* ライブ取引やバックテストを開始する前に、`Volume` プロパティが正の値に設定されていることを確認してください。
