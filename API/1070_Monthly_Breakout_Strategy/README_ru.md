# Стратегия Monthly Breakout
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробои текущего месячного максимума или минимума только в выбранные месяцы календаря. Направление входа задаётся параметром `EntryOption`, позиции закрываются через заданное число баров.

## Детали

- **Условия входа**: согласно `EntryOption` и выбранным месяцам.
- **Длинные/Короткие**: настраивается.
- **Условия выхода**: закрытие через `HoldingPeriod` баров.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `EntryOption` = LongAtHigh
  - `HoldingPeriod` = 5
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Настраивается
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
