# Стратегия индекса NSE с маркерами входа и выхода
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда цена выше трендовой SMA и RSI пересекает вверх уровень перепроданности. ATR используется для стоп-лосса и тейк-профита.

## Детали

- **Условия входа**:
  - **Лонг**: цена выше SMA и RSI пересекает снизу вверх уровень перепроданности.
- **Лонг/Шорт**: только лонг.
- **Условия выхода**:
  - закрытие длинной позиции при достижении ATR-стопа или тейк-профита.
- **Стопы**: стоп-лосс и тейк-профит на основе ATR.
- **Значения по умолчанию**:
  - `SmaPeriod` = 200.
  - `RsiPeriod` = 14.
  - `RsiOversold` = 40.
  - `AtrPeriod` = 14.
  - `AtrMultiplier` = 1.5.
  - `CandleType` = TimeSpan.FromDays(1).TimeFrame().
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: SMA, RSI, ATR
  - Стопы: на основе ATR
  - Сложность: базовая
  - Таймфрейм: дневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
