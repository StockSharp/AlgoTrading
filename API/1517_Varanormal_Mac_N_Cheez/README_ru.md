# Стратегия Varanormal Mac N Cheez
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия на пересечении SMA с трейлинг-стопом и дневной целью прибыли.

## Детали

- **Критерии входа**:
  - **Лонг**: Быстрая SMA пересекает сверху медленную.
  - **Шорт**: Быстрая SMA пересекает снизу медленную.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**:
  - Трейлинг-стоп или фиксированный стоп-лосс.
  - Дневная цель прибыли закрывает все позиции.
- **Стопы**: да, фиксированный и трейлинг.
- **Значения по умолчанию**:
  - `FastLength` = 9
  - `SlowLength` = 21
  - `DailyTarget` = 200
  - `StopLossAmount` = 100
  - `TrailOffset` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
