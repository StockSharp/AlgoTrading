# Стратегия Safa Bot Alert
[English](README.md) | [中文](README_cn.md)

Стратегия Safa Bot Alert использует короткую SMA и фильтр ADX для торговли по пересечениям цены. Покупка выполняется при пересечении цены выше SMA при сильном тренде, продажа — при пересечении ниже. Позиции управляются фиксированным тейк-профитом, стоп-лоссом и трейлинг-стопом, а также закрываются в указанное время сессии.

## Детали

- **Условия входа**: Цена пересекает SMA и ADX > `AdxThreshold`.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: Тейк-профит, стоп-лосс, трейлинг-стоп или закрытие сессии.
- **Стопы**: Фиксированные и трейлинг.
- **Значения по умолчанию**:
  - `SmaLength` = 3
  - `TakeProfitPoints` = 80m
  - `StopLossPoints` = 35m
  - `TrailPoints` = 15m
  - `AdxLength` = 15
  - `AdxThreshold` = 15m
  - `SessionCloseHour` = 16
  - `SessionCloseMinute` = 0
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA, ADX
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
