# チャート・オシレーター
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は選択可能なオシレーターを使用して取引します。ストキャスティクス、RSI、MFIから選択できます。オシレーターが売られ過ぎ状態を示すと買い、買われ過ぎ状態を示すと売ります。ストキャスティクスオプションでは、%Kと%Dのクロスを使ってシグナルを生成します。

テストでは暗号通貨などのボラティリティの高い市場で良好なパフォーマンスが示されています。

逆方向の条件が現れるか、ストップロスが発動するとポジションが反転します。

## 詳細

- **エントリー条件**: オシレーターの売られ過ぎ/買われ過ぎのレベルと%K/%Dのクロス。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆シグナルまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `Choice` = OscillatorChoice.Stochastic
  - `Length` = 14
  - `KPeriod` = 14
  - `DPeriod` = 3
  - `SmoothK` = 3
  - `Overbought` = 80
  - `Oversold` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `StopLossPercent` = 2.0m
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: Stochastic/RSI/MFI
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
