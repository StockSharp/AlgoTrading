# Осциллятор графика
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует, используя выбранный осциллятор: стохастик, RSI или MFI. Она покупает при сигнале перепроданности и продаёт при перекупленности. Для стохастика сигналы строятся на пересечении %K и %D.

Тесты показывают хорошую работу на волатильных рынках, например криптовалютах.

Позиции переворачиваются при появлении противоположных условий или срабатывании стоп-лосса.

## Детали

- **Условия входа**: Перепроданность/перекупленность осциллятора и пересечения %K/%D.
- **Длинные/короткие позиции**: Оба направления.
- **Условия выхода**: Противоположный сигнал или стоп.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `Choice` = OscillatorChoice.Stochastic
  - `Length` = 14
  - `KPeriod` = 14
  - `DPeriod` = 3
  - `SmoothK` = 3
  - `Overbought` = 80
  - `Oversold` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `StopLossPercent` = 2.0m
- **Фильтры**:
  - Категория: Oscillator
  - Направление: Оба
  - Индикаторы: Stochastic/RSI/MFI
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
