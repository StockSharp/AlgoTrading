# Стратегия Sigma Spike по времени суток и дню недели
[English](README.md) | [中文](README_cn.md)

Использует z-оценку доходности для выявления сильных движений по часам с фильтром по дням недели.
Покупает при всплесках и выходит, когда волатильность нормализуется.

## Детали

- **Критерий входа**: абсолютная z-оценка >= `Threshold`
- **Длин/Шорт**: только лонг
- **Критерий выхода**: z-оценка падает ниже `Threshold`
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Threshold` = 2.5
  - `AllDays` = false
  - `DayOfWeekFilter` = Monday
  - `StdevLength` = 20
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Лонг
  - Индикаторы: StandardDeviation
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: да
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
