# 祝前日強気戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
祝前日強気とは、主要な市場の祝日直前に出来高が薄く、センチメントが楽観的なため、強気の傾向が見られることを指します。
トレーダーは休暇前にポジションを取ることが多く、最後の1〜2セッションで価格を押し上げます。

テストでは年平均リターンが約109%であることが示されています。仮想通貨市場で最も効果的です。

この戦略は祝日前日にロングで入り、翌セッションまたは引け時に決済して、その短期的な偏りを捉えます。

期待される上昇が現れなかった場合に備え、タイトなストップを使用します。

## 詳細

- **エントリー条件**: カレンダー効果のトリガー
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: 季節性
  - 方向: 両方
  - インジケーター: 季節性
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
