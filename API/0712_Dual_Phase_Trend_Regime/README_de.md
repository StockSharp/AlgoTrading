# Duale-Phasen-Trend-Regime-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Duale-Phasen-Trend-Regime-Strategie wechselt je nach aktueller Volatilität zwischen langsamem und schnellem Trendoszillator. Die Volatilität wird aus der Standardabweichung der Renditen abgeleitet und in niedrige oder hohe Regime eingeteilt. Lineare Regressionssteigungen bestimmen die Trendrichtung. Einstiege können bei Regimewechseln oder Oszillatorkreuzungen erfolgen.

## Parameter
- Kerzentyp
- Handelsrichtung
- Signalquelle
- Länge des langsamen Oszillators
- Länge des schnellen Oszillators
- Volatilitäts-Neuberechnungsintervall
- Aktueller Volatilitätszeitraum
- Volatilitäts-Glättungslänge
