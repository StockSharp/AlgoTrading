# デュアルフェーズ・トレンドレジーム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

デュアルフェーズ・トレンドレジーム戦略は、現在のボラティリティに基づいて低速と高速のトレンドオシレーターを切り替えます。ボラティリティはリターンの標準偏差から算出され、低または高のレジームに分類されます。線形回帰の傾きがトレンドの方向を決定します。エントリーはレジームの切り替えまたはオシレーターのクロスオーバーで行えます。

## パラメーター
- ローソク足タイプ
- 取引方向
- シグナルソース
- 低速オシレーターの長さ
- 高速オシレーターの長さ
- ボラティリティ再計算間隔
- 現在のボラティリティ期間
- ボラティリティ平滑化の長さ
