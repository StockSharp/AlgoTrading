# Color Schaff RSIトレンドサイクル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Color Schaff RSI Trend Cycle（STC）オシレーターに基づくトレンドフォローシステム。この戦略はSTCインジケーターのカラー推移に反応して、ロングおよびショートポジションの出入りを行います。

## 詳細

- **エントリー条件**:
  - **ロング**: 2本前のインジケーター色 > 5 かつ最後のバー < 6。
  - **ショート**: 2本前のインジケーター色 < 2 かつ最後のバー > 1。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 2本前のインジケーター色 < 2 のときロングポジションを閉じる。
  - 2本前のインジケーター色 > 5 のときショートポジションを閉じる。
- **インジケーター**: Color Schaff RSI Trend Cycle。
- **デフォルト値**:
  - `Fast RSI` = 23
  - `Slow RSI` = 50
  - `Cycle` = 10
  - `High Level` = 60
  - `Low Level` = -60
- **時間軸**: デフォルトで4時間ローソク足。
- **ストップ**: なし。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
