# Стратегия Color Schaff RSI Trend Cycle
[English](README.md) | [中文](README_cn.md)

Трендовая система, основанная на осцилляторе Color Schaff RSI Trend Cycle (STC). Стратегия реагирует на изменения цвета индикатора для открытия и закрытия длинных и коротких позиций.

## Подробности

- **Условия входа**:
  - **Лонг**: Цвет индикатора два бара назад > 5 и последний бар < 6.
  - **Шорт**: Цвет индикатора два бара назад < 2 и последний бар > 1.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Лонг закрывается, когда цвет индикатора два бара назад < 2.
  - Шорт закрывается, когда цвет индикатора два бара назад > 5.
- **Индикаторы**: Color Schaff RSI Trend Cycle.
- **Параметры по умолчанию**:
  - `Fast RSI` = 23
  - `Slow RSI` = 50
  - `Cycle` = 10
  - `High Level` = 60
  - `Low Level` = -60
- **Таймфрейм**: 4-часовые свечи по умолчанию.
- **Стопы**: Нет.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
