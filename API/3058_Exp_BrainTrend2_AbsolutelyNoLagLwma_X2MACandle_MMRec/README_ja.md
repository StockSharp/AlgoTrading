# Exp BrainTrend2 AbsolutelyNoLagLwma X2MACandle MMRec戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、StockSharpの高レベルAPIで3つのフィルターを組み合わせてマルチモジュールのMetaTraderエキスパートアドバイザーを再現します：

1. **BrainTrend2インスピレーション** – Average True Range（ATR）チャンネルがボラティリティの収縮と拡張フェーズを検出します。
2. **AbsolutelyNoLagLwma近似** – 線形加重移動平均（LWMA）が最小限のラグで支配的な方向を追跡します。
3. **X2MACandle レプリカ** – 高速と低速の指数移動平均（EMA）ペアがローソク足の色を評価してモメンタムを検証します。

すべてのフィルターが同じ方向を指している場合にのみポジションが開かれます。ATRドリブンのストップロスとテイクプロフィットターゲットがエグジットプロセスを管理し、元のMMRecマネー管理コンセプトをエミュレートします。

## トレードロジック
- **強気設定**：高速EMAが低速EMAより高い間にローソク足がLWMAより上でクローズします。新しいロングエントリーは以前の強気バイアスが消えた後にのみ許可され、同一のシグナルで複数の注文を防ぎます。
- **弱気設定**：高速EMAが低速EMAより低い間にローソク足がLWMAより下でクローズします。ショートポジションはロングサイドと同じ確認とクールダウンルールに従います。
- **リスク管理**：ATRが動的な出口レベルを定義します。ストップロスとテイクプロフィットの両方が現在のボラティリティとともにスケールし、すべてのローソク足で再評価されます。市場がいずれかのレベルに触れると、戦略は成行注文でポジション全体をクローズします。

## パラメーター
| 名前 | 説明 |
| --- | --- |
| `CandleType` | 作業ローソク足シリーズのタイムフレーム。元のEAのデフォルトを反映するためデフォルトは6時間ローソク足。 |
| `AtrPeriod` | ATRボラティリティフィルターが使用するルックバック期間。 |
| `LwmaLength` | トレンドフィルターの線形加重移動平均の期間。 |
| `FastMaLength` | ローソク足の色付けに使用する高速EMAの期間。 |
| `SlowMaLength` | ローソク足の色付けに使用する低速EMAの期間。 |
| `StopLossAtrMultiplier` | 保護ストップ距離を計算するためにATRに適用される乗数。 |
| `TakeProfitAtrMultiplier` | テイクプロフィット距離を決定するためにATRに適用される乗数。 |

すべてのパラメーターは`StrategyParam<T>`を通じて公開されており、StockSharp内で最適化できます。

## 注意事項
- 元のエキスパートアドバイザーは独自のインジケーターバッファに依存しています。このポートは外部スクリプトに依存せずに同じ方向性の手がかりを再現する標準のStockSharpインジケーターを使用します。
- StockSharpの戦略は通常ポートフォリオサイズの注文で動作するため、マネー管理は完全ポジションエグジットに簡略化されます。ATRドリブンの距離はMMRecモジュールに期待される適応動作を提供します。
- コード内のコメントは変換ガイドラインの要件に従って英語です。
