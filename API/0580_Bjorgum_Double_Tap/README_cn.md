# Bjorgum Double Tap 策略
[English](README.md) | [Русский](README_ru.md)

该策略寻找双顶和双底形态。当价格跌破双顶颈线时做空，当价格突破双底颈线时做多。目标和止损以形态高度的斐波那契延伸计算。

## 详情

- **入场条件**：
  - **多头**：双底突破颈线上方。
  - **空头**：双顶跌破颈线下方。
- **多/空**：双向。
- **出场条件**：止损或目标价位。
- **止损**：通过 `StopLossFib` 的斐波那契百分比。
- **默认值**：
  - 枢轴长度 50。
  - 枢轴容差 15%。
  - 目标 fib 100%。
  - 止损 fib 0%。
- **过滤器**：
  - 类别：形态
  - 方向：双向
  - 指标：Highest/Lowest
  - 止损：有
  - 复杂度：中等
  - 时间框架：中期
