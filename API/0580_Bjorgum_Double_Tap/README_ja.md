# Bjorgum Double Tap戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はダブルトップとダブルボトムのパターンを探します。価格がダブルトップのネックラインを下回るとショートポジションを建て、ダブルボトムのネックラインを上抜けるとロングポジションを建てます。目標値とストップレベルはパターンの高さのフィボナッチ拡張として計算されます。

## 詳細

- **エントリー条件**:
  - **ロング**: ダブルボトムのネックライン上抜けブレイクアウト。
  - **ショート**: ダブルトップのネックライン下抜けブレイクアウト。
- **ロング/ショート**: 両方。
- **エグジット条件**: ストップまたは目標レベル。
- **ストップ**: `StopLossFib` によるフィボナッチ割合。
- **デフォルト値**:
  - ピボット長さ 50。
  - ピボット許容誤差 15%。
  - 目標 fib 100%。
  - ストップ fib 0%。
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Highest/Lowest
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 中期
