# Omni トレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

Omni トレンド戦略は、MetaTrader エキスパート「Exp_Omni_Trend」の直接移植版です。移動平均と ATR ベースのチャネルを組み合わせて支配的なトレンドを検出し、ロングとショートのエクスポージャーを切り替えます。StockSharp バージョンは、シグナル検出と注文実行の間の遅延や、個別のエントリーまたはエグジットレッグを無効にする機能など、元のふるまいを維持します。

戦略は設定されたローソク足シリーズにサブスクライブし、完成した各バーを Omni トレンドロジックに入力します。移動平均は中心傾向の推定として機能し、ATR 倍率がボラティリティエンベロープを構築します。エンベロープはトレーリングストップのように動作します：前のエンベロープ境界を超えて価格がクローズすると、トレンドが反転し、新しい方向に新しいエントリーシグナルが生成され、反対のエクスポージャーが即座にクローズされます。

オプションのストップロスとテイクプロフィットの閾値が有効な場合、それらはブローカー側で価格ステップで機能し、インジケーターベースの出口を補完します。ポジションサイズは戦略の組み込み `Volume` プロパティで制御されます（デフォルト `1`）。

## 取引ロジック

1. ローソク足ストリームで選択した移動平均（`MaType`、`MaLength`、`AppliedPrice`）を計算します。
2. ATR（`AtrLength`）を計算し、`VolatilityFactor` と `MoneyRisk` を使用して 2 つの適応バンドを導出します。上バンドはショートポジションを保護し、下バンドはロングポジションを保護します。
3. 価格が前のバーの保護バンドを超えると、トレンドが変化します：
   - 強気のブレイクアウト（前の上バンドを超える `HighPrice`）はトレンドを「上昇」に変え、許可されていればショートポジションをクローズし、`SignalBar` 本の完成したローソク足後にロングポジションをオープンします。
   - 弱気のブレイクアウト（前の下バンドを下回る `LowPrice`）はトレンドを「下降」に変え、許可されていればロングポジションをクローズし、設定された遅延後にショートポジションをオープンします。
4. トレンドが強気の間、戦略はショートの出口を要求し続けます；弱気のトレンドとロングの出口には対称のルールが適用されます。これは MetaTrader エキスパートのふるまいを反映しており、そこでは反対のバンドが支配的な方向に対して常にフラットなエクスポージャーを強制します。
5. オプションのリスク管理は完成した各ローソク足を監視します。現在のバーがストップまたはターゲット価格（価格ステップで表現）に達した場合、ポジションは即座にクローズされ、保存されたエントリー価格がリセットされます。

シグナルは FIFO キューを通じてスケジュールされます。`SignalBar` がゼロの場合、同じローソク足のクローズで実行されます。それ以外の場合、遅延を完成させるローソク足のオープンで引き起こされ、ソースエキスパートの「前のバー」実行スタイルを複製します。

## パラメーター

| 名前 | 説明 | デフォルト |
|------|-------------|---------|
| `CandleType` | 計算に使用するローソク足タイプ（時間軸）。 | 4 時間時間軸 |
| `MaLength` | 移動平均の期間。 | 13 |
| `MaType` | 移動平均の方法：単純、指数、平滑、または線形加重。 | 指数 |
| `AppliedPrice` | 移動平均に渡される価格フィールド（終値、始値、高値、安値、中間、典型、加重）。 | 終値 |
| `AtrLength` | ボラティリティチャネルで使用される ATR 期間。 | 11 |
| `VolatilityFactor` | 生チャネルを構築する際に ATR に適用される乗数。 | 1.3 |
| `MoneyRisk` | チャネルを移動平均から離れた方向にシフトするオフセット係数。MQL 入力と同一。 | 0.15 |
| `SignalBar` | シグナルに対して行動する前に待機する完成したローソク足の数。 | 1 |
| `EnableBuyOpen` | ロングポジションのオープンを許可する。 | true |
| `EnableSellOpen` | ショートポジションのオープンを許可する。 | true |
| `EnableBuyClose` | 弱気トレンド検出時にロングポジションのクローズを許可する。 | true |
| `EnableSellClose` | 強気トレンド検出時にショートポジションのクローズを許可する。 | true |
| `StopLossPoints` | 価格ステップでのオプション保護ストップ距離。`0` で無効化。 | 1000 |
| `TakeProfitPoints` | 価格ステップでのオプション利益目標距離。`0` で無効化。 | 2000 |
| `Volume` | トレードサイズを制御する戦略プロパティ。 | 1 |

## 注意事項と推奨事項

- StockSharp 実装はオリジナルと同じインジケーター値を入力し、そのトレンド転換を再現します。ただし、正確な約定はデータソースと実行レイテンシーに依存します。
- `SignalBar = 1` に設定して、シグナルが利用可能になった後の次のローソク足のオープンで注文が実行されるエキスパートアドバイザーのデフォルトを模倣します。大きな値はさらに実行を遅らせます；`0` に設定すると現在のクローズで実行されます。
- ストップロスとテイクプロフィットの閾値はポイント（価格ステップ）で表現されます。接続された有価証券が有効な `PriceStep` を公開していることを確認してください。
- 組み込みチャートはローソク足シリーズ、選択した移動平均、および戦略自身のトレードを描画して、素早い視覚的検証を可能にします。
- 特定のエントリーまたはエグジットトグルを無効にして、戦略を一方向のみの操作に制限したり、出口を手動で処理したりします。
- 戦略はペンディング注文を作成しません；ソースエキスパートの直接注文配置と同様に `BuyMarket` と `SellMarket` を使用して成行注文を発行します。
