# Fourier Smoothed Volume Zone Oscillator WFSVZ0
[English](README.md) | [中文](README_cn.md)

Стратегия, использующая сглаженный по Фурье осциллятор объёмных зон. Открывает длинную позицию при росте осциллятора выше порога и короткую при падении ниже отрицательного порога. При желании закрывает позицию при отсутствии сигнала.

## Детали

- **Условия входа**: Осциллятор растёт выше порога / падает ниже отрицательного порога.
- **Длинные/короткие**: В обе стороны.
- **Условия выхода**: Противоположный сигнал или опция закрыть всё.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `VzoLength` = 2
  - `SmoothLength` = 2
  - `Threshold` = 0m
  - `CloseAllPositions` = true
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Объём
  - Направление: Оба
  - Индикаторы: Volume Zone Oscillator
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
