# 収益性の高いプルバック戦略 Mark804
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

指数移動平均のリボンを使ったトレンドフォロー型プルバック戦略です。確認されたトレンド内でシグナルEMAへの価格の押し目を探します。プルバック後に価格がトレンド方向に再びクローズすると、ポジションを開き、パーセンテージベースの利確・損切りレベルで保護します。

## 詳細

- **エントリー条件**:
  - **ロング**: 速いEMA > シグナルEMA > 中EMA、オプションで中EMA > 遅いEMA、前回終値がシグナルEMA以下で現在終値が以上。
  - **ショート**: 速いEMA < シグナルEMA < 中EMA、オプションで中EMA < 遅いEMA、前回終値がシグナルEMA以上で現在終値が以下。
- **ロング/ショート**: 両方。
- **エグジット条件**: 利確または損切りに到達。
- **ストップ**: あり、固定パーセンテージの利確・損切り。
- **デフォルト値**:
  - Fast EMA Length = 8
  - Signal EMA Length = 21
  - Medium EMA Length = 50
  - Slow EMA Length = 200
  - Take Profit % = 2
  - Stop Loss % = 1
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA
  - ストップ: あり
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
