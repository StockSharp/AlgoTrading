# Bollinger Squeeze
[English](README.md) | [中文](README_cn.md)

Стратегия на основе сжатия полос Боллинджера

Bollinger Squeeze ожидает сужения полос, указывающего на низкую волатильность. Прорыв за пределы полос открывает сделку в направлении пробоя, закрытие происходит при ослаблении импульса или противоположном пробое.

Условие сжатия намекает на скорое расширение волатильности. После срабатывания сигналов позиция сопровождает движение и закрывается по стопу ATR или обратному пересечению полос.

## Детали

- **Условия входа**: сигналы на основе полос Боллинджера.
- **Длинные/короткие**: в обе стороны.
- **Условия выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 2m
  - `SqueezeThreshold` = 0.1m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Bollinger
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 100%\. Стратегию лучше запускать на рынке Форекс.
