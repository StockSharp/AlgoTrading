# コモディティ・モメンタム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Commodity Momentum** 戦略は、12ヶ月モメンタムが最も強いコモディティ（直近1ヶ月を除く）をロングします。
ポジションは毎月の最初の取引日にリバランスされます。

テストでは年平均リターン約10%を示しています。分散されたコモディティ市場で最も優れたパフォーマンスを発揮します。

ポジションは毎月調整されます。イントラデイシグナルは使用されません。

## 詳細
- **エントリー条件**: 直近1ヶ月を除く12ヶ月モメンタムで上位`TopN`コモディティを購入。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 次回予定日にリバランス。
- **ストップ**: 明示的なストップロジックなし。
- **デフォルト値**:
  - `TopN = 5`
  - `MinTradeUsd = 200`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: ロングのみ
  - インジケーター: Price
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 日足
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
