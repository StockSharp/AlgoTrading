# Triple Top Triple Bottom戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Triple Top Triple Bottom戦略**は、同名のMetaTrader Expert Advisorを移植したものです。元のシステムは、市場へ入る前に複数の確認層（トレンド方向、モメンタム強度、MACDフィルター）を組み合わせます。このStockSharp実装は同じ中核アイデアを保ち、重要なしきい値を戦略パラメーターとして公開します。

## コアロジック

1. **トレンドフィルター:** 典型価格（H+L+C）/3で計算される2本の線形加重移動平均（LWMA）が取引方向を定義します。ロング取引を許可するには高速LWMAが低速LWMAを上回り、ショート取引を許可するには下回る必要があります。
2. **モメンタム確認:** 設定可能なルックバック長を持つ組み込みモメンタム指標は、直近3本の確定ローソク足内で中立レベル100からユーザー定義しきい値以上離れている必要があります。EAは過去のモメンタム値を分析して同じ動作を要求しており、横ばい市場でのエントリーを避けるためこの検証を再現しています。
3. **MACDフィルター:** 古典的な12/26/9 MACDシグナルラインフィルターが、強いトレンドに逆らう取引を防ぎます。戦略はMACDラインがシグナルラインを上回るときだけ買い、下回るときだけ売ります。
4. **リスク管理:** 成行注文は、絶対価格単位で測定されるストップロスとテイクプロフィット目標で保護されます。パラメーターは任意で、ゼロに設定すると該当注文を無効化します。コードはまた、ローソク足処理中に反対側のリスクしきい値へ到達した場合、ポジションを閉じます。

## パラメーター

- **Entry Candle:** 作業ローソク足の時間軸を定義する`DataType`。
- **Fast LWMA / Slow LWMA:** 高速および低速トレンドフィルターの長さ。
- **Momentum Period / Momentum Threshold:** モメンタム指標のルックバックと、取引アイデアを確認する100からの最小偏差。
- **Stop Loss / Take Profit:** 絶対価格単位での保護距離。`SetStopLoss`と`SetTakeProfit`を通じてネイティブ保護注文としても送信され、戦略セッションが停止してもリスク制御が適用されます。

## MQL版との差異

- すべての資金管理追加機能（ロット乗数、エクイティ保護、ローソク足トレーリング、ブレイクイーブン、手動トレンドライン確認）は省略されています。StockSharp高レベルAPIが既にポジションサイズ調整ユーティリティを提供し、元のEAで使われたグラフィカルオブジェクトはMetaTrader固有であるためです。
- リスクしきい値はpipsではなく絶対価格単位で表されます。これにより実装はブローカー中立となり、ユーザーは希望するpip距離にブローカーのpipサイズを掛けるだけで簡単に変換できます。
- チャート出力は、価格ローソク足、移動平均、モメンタム、MACD指標にStockSharpのエリアを使用します。

## 使用上の注意

1. 戦略を銘柄へ接続し、開始前に希望するローソク足タイプを設定してください。
2. 銘柄のボラティリティに応じて、モメンタムしきい値とストップ距離を調整してください。
3. 戦略は単一のネットポジションを取引します。反対シグナルが現れた場合、現在のエクスポージャーを先に閉じるため、重複取引を防ぎます。

コードはすべて英語でコメントされており、リポジトリにあるStockSharp高レベルAPIガイドラインに従っています。
