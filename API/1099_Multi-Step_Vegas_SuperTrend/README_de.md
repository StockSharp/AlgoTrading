# Multi-Step Vegas SuperTrend-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Kombiniert einen volatilitätsangepassten SuperTrend mit einem Vegas-Kanal.
Eröffnet Long-Positionen, wenn der Kurs über dem Trend liegt, und Short-Positionen, wenn er darunter liegt.
Optionale mehrstufige Take-Profits schließen Teile der Position bei steigenden Zielen.
