# 时间序列滞后减少滤波
[Русский](README_ru.md) | [English](README.md)

该策略使用带滞后减少滤波的EMA，当价格与滤波线交叉时开仓。

## 细节

- **入场条件**：价格穿越滞后滤波EMA。
- **多/空**：双向。
- **出场条件**：反向交叉。
- **止损**：无。
- **默认参数**:
  - `LagReduction` = 20m
  - `EmaLength` = 100
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类别: 趋势
  - 方向: 双向
  - 指标: EMA
  - 止损: 无
  - 复杂度: 基础
  - 周期: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
