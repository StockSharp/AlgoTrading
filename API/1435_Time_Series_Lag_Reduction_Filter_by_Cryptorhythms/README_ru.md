# Фильтр снижения лага временного ряда
[English](README.md) | [中文](README_cn.md)

Стратегия использует EMA с фильтром снижения лага и совершает сделки при пересечении цены и фильтра.

## Детали

- **Условия входа**: Пересечение цены с лаг-сглаженной EMA.
- **Лонг/Шорт**: В обе стороны.
- **Условия выхода**: Обратное пересечение.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `LagReduction` = 20m
  - `EmaLength` = 100
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
