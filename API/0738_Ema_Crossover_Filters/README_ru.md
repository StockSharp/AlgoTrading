# Стратегия пересечения EMA с фильтрами
[English](README.md) | [中文](README_cn.md)

Стратегия использует несколько экспоненциальных скользящих средних для торговли на их пересечениях с дополнительными фильтрами тренда.

Покупка выполняется, когда EMA100 пересекает EMA200 снизу вверх и EMA9 находится выше EMA50. Короткая позиция открывается при пересечении EMA100 ниже EMA200 и EMA9 ниже EMA50. Длинные позиции закрываются при пересечении EMA100 ниже EMA50, короткие — при пересечении EMA100 выше EMA50.

## Параметры
- Тип свечей
- Длина EMA 9
- Длина EMA 50
- Длина EMA 100
- Длина EMA 200
