# Statistical Arbitrage Pairs Trading - Long-Side Only
[English](README.md) | [中文](README_cn.md)

Стратегия реализует простую парную торговлю на основе z-score спреда между двумя инструментами. Открывает длинную позицию, когда спред опускается ниже заданного порога, и закрывает её, когда спред пересекает ноль снизу вверх.

## Детали

- **Условия входа**: Z-скор спреда ниже порога.
- **Long/Short**: Только покупки.
- **Условия выхода**: Z-скор спреда пересекает ноль.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `ZScoreLength` = 20
  - `ExtremeLevel` = -1
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Long
  - Индикаторы: SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Начальная
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
