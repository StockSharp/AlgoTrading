# Стратегия AI SuperTrend
[English](README.md) | [中文](README_cn.md)

Стратегия AI SuperTrend комбинирует индикатор SuperTrend с взвешенными скользящими средними цены и линии SuperTrend. Длинная позиция открывается, когда SuperTrend разворачивается вверх и WMA цены выше WMA SuperTrend. Короткая позиция открывается при обратных условиях. Позиции защищаются плавающим стопом на основе ATR.

## Подробности

- **Условия входа**:
  - **Long**: направление SuperTrend меняется вверх и WMA цены выше WMA SuperTrend.
  - **Short**: направление SuperTrend меняется вниз и WMA цены ниже WMA SuperTrend.
- **Условия выхода**:
  - Разворот тренда или ATR-стоп.
- **Стопы**: динамический ATR-стоп.
- **Значения по умолчанию**:
  - `AtrPeriod` = 10
  - `AtrFactor` = 3
  - `PriceWmaLength` = 20
  - `SuperWmaLength` = 100
  - `EnableLong` = true
  - `EnableShort` = true
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SuperTrend, WMA, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
