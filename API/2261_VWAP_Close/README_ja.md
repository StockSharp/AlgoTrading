# VWAP終値戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は終値の出来高加重移動平均（VWMA）を計算します。VWMAが方向を変えると、潜在的なエントリーまたはエグジットのシグナルとなります：

- VWMAが下落していて上向きに転じる場合（谷を形成）、戦略はショートポジションをクローズし、ロングポジションを建てることができます。
- VWMAが上昇していて下向きに転じる場合（山を形成）、戦略はロングポジションをクローズし、ショートポジションを建てることができます。

## パラメーター
- **Period** – VWMA計算に使用するローソク足の数。
- **Candle Type** – 処理するローソク足の時間軸。
- **Buy Open** – ロングポジションの建て入れを有効にする。
- **Sell Open** – ショートポジションの建て入れを有効にする。
- **Buy Close** – VWMAが下向きに転じたときのロングポジションのクローズを許可する。
- **Sell Close** – VWMAが上向きに転じたときのショートポジションのクローズを許可する。

## 注意事項
この戦略はStockSharpの `VolumeWeightedMovingAverage` インジケーターを使用し、完成したローソク足のみを処理します。取引ボリュームは戦略の `Volume` プロパティから取得されます。新しいポジションを建てる際、反対のポジションは自動的にクローズされます。
