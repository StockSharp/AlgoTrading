# バーバランス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、各ローソク足内の上昇と下降の動きのバランスを測定します。プラスのバランスは買い手がバーを支配していることを示し、マイナスのバランスは売り圧力を示します。

このバランスは移動平均でスムージングされます。現在のバランスとその平均の両方がゼロを超えている場合、戦略はロングポジションに入ります。両方がゼロを下回ると、ショートに入ります。

## 詳細

- **エントリー条件**: バランス > 0 かつ平均 > 0 でロング；バランス < 0 かつ平均 < 0 でショート。
- **エグジット条件**: 逆シグナルがポジションの反転を引き起こす。
- **インジケーター**: カスタムバーバランス、SMA。
- **ロング/ショート**: 両方。
- **ストップロス**: なし。
