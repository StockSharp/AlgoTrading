# Скалпер ChopFlow ATR
[English](README.md) | [中文](README_cn.md)

Стратегия ChopFlow ATR входит, когда рынок выходит из фаз боковика и OBV пересекает свою EMA. Выход осуществляется по симметричным уровням стоп-лосса и тейк-профита, рассчитанным по ATR.

Цель — поймать ранние импульсы, зарождающиеся после периода хаотичности.

## Подробности

- **Критерий входа**: `Choppiness < ChopThreshold` и OBV выше/ниже EMA.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: стоп или тейк-профит по ATR.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `AtrLength` = 14
  - `AtrMultiplier` = 1.5
  - `ChopLength` = 14
  - `ChopThreshold` = 60
  - `ObvEmaLength` = 10
  - `SessionInput` = "1700-1600"
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Скальпинг
  - Направление: Обе стороны
  - Индикаторы: ATR, Choppiness, OBV
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
