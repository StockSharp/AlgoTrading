# Стратегия Trend Impulse Tester
[English](README.md) | [中文](README_cn.md)

Trend Impulse Tester входит в сделки, когда сильный тренд подтверждён EMA и ADX, и появляется импульс RSI.
Покупает при бычьем импульсе в восходящем тренде и продаёт при медвежьем импульсе в нисходящем тренде.

## Детали

- **Критерий входа**: тренд по EMA + подтверждение ADX и пересечение порога RSI
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `FastEmaLength` = 50
  - `SlowEmaLength` = 200
  - `AdxLength` = 14
  - `AdxMin` = 18
  - `RsiLength` = 14
  - `RsiUp` = 55
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: EMA, ADX, RSI
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
