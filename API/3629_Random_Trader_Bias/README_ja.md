# ランダムバイアストレーダー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ランダム バイアス トレーダー戦略は、StockSharp の高レベルの API を使用して、MetaTrader の「ランダム トレーダー」エキスパート アドバイザーをエミュレートします。
キャンドルが終了するたびに、この戦略は仮想コインを反転し、取引がアクティブでないときにその方向にポジションをオープンします。
ストップロスとテイクプロフィットのレベルは、ATR(10) または固定ピップ距離から導出され、報酬とリスクの比率によってサイズが決定されます。
ポジションサイズは設定されたリスクパーセンテージから計算され、商品の取引高制限によって自動的に上限が設定されます。
オプションの損益分岐点トリガーにより、指定されたピップゲインに達するとストップロスをエントリー価格に移動できます。

## 詳細
- **データ**: `CandleType` によって定義された 1 つのキャンドル サブスクリプション。
- **エントリー基準**:
  - ロング: オープン ポジションなし、コイン トスはロングで戻ります。エントリー価格は直近の終値と同じです。
  - ショート: オープンポジションなし、コイントスはショートに戻ります。エントリー価格は直近の終値と同じです。
- **終了基準**:
  - ストップロス: 距離は、`LossType` に応じて、`LossPipDistance` × ピップ サイズまたは `LossAtrMultiplier` × ATR(10) に等しくなります。
  - テイクプロフィット: 停止距離に `RewardRiskRatio` を掛けます。
  - 損益分岐点: 有効にすると、`BreakevenDistancePips` ゲイン後にストップをエントリーに移動します。
- **ストップ**: 取引ごとに動的なストップロスとテイクプロフィット、ブレークイーブンストップはオプションです。
- **デフォルト値**:
  - `CandleType` = 1 分の時間枠
  - `RewardRiskRatio` = 2.0
  - `LossType` = ピップ
  - `LossAtrMultiplier` = 5.0
  - `LossPipDistance` = 20 ピップス
  - `RiskPercentPerTrade` = 1%
  - `UseBreakeven` = 有効
  - `BreakevenDistancePips` = 10 ピップス
  - `UseMaxMargin` = 有効
- **フィルター**:
  - カテゴリ: ランダム化されたトレンド中立
  - 方向: 両方、フリップごとに決定
  - インジケーター: ATR(10) (オプション)
  - 複雑さ: 初心者
  - リスクレベル: 中、ストップサイズに依存

## 注意事項
- リスクベースのボリュームが小さすぎる場合、戦略はオプションで最大取引可能ボリュームに戻ります。
- ストップおよびターゲットレベルは、注文が発注される前に商品価格ステップに丸められます。
- 損益分岐点ロジックは、元の MetaTrader ロジックを反映して、常に 1 つのポジションのみをオープン状態に保ちます。
