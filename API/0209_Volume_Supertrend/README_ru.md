# Стратегия Volume Supertrend
[English](README.md) | [中文](README_cn.md)

Стратегия опирается на индикаторы объёма и Supertrend для определения сигналов. Длинная позиция открывается, когда объём выше среднего и цена находится выше линии Supertrend (скачок объёма на восходящем тренде). Короткая позиция появляется при объёме выше среднего и цене ниже Supertrend — скачок объёма на нисходящем тренде.

Стратегия подходит трейдерам, торгующим по тренду.

## Детали
- **Условия входа**:
  - **Лонг**: Объём > среднего значения и цена > Supertrend (скачок объёма при росте)
  - **Шорт**: Объём > среднего значения и цена < Supertrend (скачок объёма при падении)
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - **Лонг**: Закрыть позицию, когда Supertrend разворачивается вниз
  - **Шорт**: Закрыть позицию, когда Supertrend разворачивается вверх
- **Стопы**: да.
- **Значения по умолчанию**:
  - `VolumeAvgPeriod` = 20
  - `SupertrendPeriod` = 10
  - `SupertrendMultiplier` = 3m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `StopLossPercent` = 2.0m
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Volume Supertrend
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 64%\. Стратегию лучше запускать на рынке Форекс.
