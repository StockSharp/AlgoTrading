# WPRヒストグラム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はWilliams %Rインジケーターの動作に基づいて取引します。インジケーターが買われすぎまたは売られすぎのゾーンを離れたときを監視し、反対方向にポジションを開きます。

## ロジック

- Williams %Rが高いレベルを超えてから下落したとき、市場が買われすぎゾーンを離れたと判断されます。戦略はロングポジションを開きます。
- Williams %Rが低いレベルを下回ってから上昇したとき、市場が売られすぎゾーンを離れます。戦略はショートポジションを開きます。
- 新しいポジションを開く前に、既存の反対ポジションが閉じられます。

## パラメーター

- **WPR Period** – Williams %Rの計算期間。
- **High Level** – 買われすぎゾーンの閾値。
- **Low Level** – 売られすぎゾーンの閾値。
- **Candle Type** – 計算に使用するローソク足のタイプと時間軸。

## 注意事項

この戦略は成行注文のみを使用し、ストップロスやテイクプロフィットのレベルを設定しません。ポジションサイズはユーザーが定義する `Volume` プロパティによって決まります。
