# Стратегия Improvisando
[English](README.md) | [中文](README_cn.md)

Improvisando сочетает простой трендовый фильтр на основе EMA с колебаниями RSI. Цель — следовать за направлением, задаваемым EMA, входя только тогда, когда RSI пересекает нейтральный уровень 50. Изначально в стратегии использовались элементы MACD, но в упрощённой версии акцент сделан на ясности и лёгкости настройки.

Пользователь может отдельно включать сделки в длинную и/или короткую сторону.

## Детали

- **Критерии входа**:
  - **Длинная позиция**: `Close > EMA` и `RSI > 50`
  - **Короткая позиция**: `Close < EMA` и `RSI < 50`
- **Длинные/короткие**: настраивается
- **Критерии выхода**:
  - Противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `EmaLength` = 10
  - `RsiLength` = 14
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Настраиваемое
  - Индикаторы: EMA, RSI
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
