# Improvisando戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ImpravisandoはEMAの基本的なトレンドフィルターとRSIのスイングを組み合わせています。目標は、EMAが示す優勢な方向に従いながら、RSIが中立の50ラインをクロスしたときにのみエントリーすることです。元の設計ではMACDスタイルのモメンタムも実験しましたが、このシンプルなバージョンは明確さと調整のしやすさに焦点を当てています。

ユーザーはロングおよび/またはショートの取引を個別に有効にできます。

## 詳細

- **エントリー条件**:
  - **ロング**: `Close > EMA` かつ `RSI > 50`
  - **ショート**: `Close < EMA` かつ `RSI < 50`
- **ロング/ショート**: 設定可能
- **エグジット条件**:
  - 逆シグナル
- **ストップ**: なし
- **デフォルト値**:
  - `EmaLength` = 10
  - `RsiLength` = 14
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 設定可能
  - インジケーター: EMA, RSI
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
