# Estratégia de Inclinação SMA com TP/SL Dinâmico
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera quando o preço cruza a SMA de máximas ou mínimas com uma inclinação significativa. As posições usam um take profit fixo e um stop dinâmico de trailing.
