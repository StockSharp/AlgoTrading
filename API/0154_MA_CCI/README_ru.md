# Стратегия MA CCI
[English](README.md) | [中文](README_cn.md)

Стратегия объединяет скользящую среднюю и индикатор CCI. Покупка совершается, когда цена выше MA и CCI в зоне перепроданности. Продажа — когда цена ниже MA и CCI в зоне перекупленности.

Скользящая средняя задаёт тренд, а CCI ищет отклонения от этого среднего значения. Входы происходят на экстремальных значениях CCI по направлению MA.

Подходит свинг-трейдерам, входящим на откатах. Стопы по ATR защищают от резких разворотов.

## Подробности

- **Условия входа**:
  - Лонг: `Close > MA && CCI < OversoldLevel`
  - Шорт: `Close < MA && CCI > OverboughtLevel`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - CCI возвращается к нулевой линии
- **Стопы**: процентные через `StopLossPercent`
- **Значения по умолчанию**:
  - `MaPeriod` = 20
  - `CciPeriod` = 20
  - `OverboughtLevel` = 100m
  - `OversoldLevel` = -100m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Средняя обратная
  - Направление: Оба
  - Индикаторы: Скользящая средняя, CCI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 49%\. Стратегию лучше запускать на крипторынке.
