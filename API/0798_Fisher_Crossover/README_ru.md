# Стратегия Fisher Crossover
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор Fisher Transform. Покупка выполняется, когда Fisher пересекает вверх своё предыдущее значение и находится ниже 1. Закрытие позиции происходит, когда Fisher пересекает вниз своё предыдущее значение и находится выше 1.

## Детали

- **Условия входа**:
  - **Long**: `Fisher пересекает вверх предыдущий Fisher` && `Fisher < 1`
- **Длинные/Короткие**: Только long
- **Условия выхода**:
  - `Fisher пересекает вниз предыдущий Fisher` && `Fisher > 1`
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Fisher Length` = 9
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Только long
  - Индикаторы: Fisher Transform
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
