# Estrategia de Continuación Lineal
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Este ejemplo convierte el estudio "Linear Continuation" de TradingView a StockSharp. La estrategia calcula tres medias móviles y registra su proyección de continuación lineal. Está pensada como demostración de indicadores y no incluye lógica de trading.

## Detalles

- **MA Type** – tipo de media móvil (SMA o EMA)
- **MA1 Period** – período para la primera media móvil
- **MA2 Period** – período para la segunda media móvil
- **MA3 Period** – período para la tercera media móvil
- **Aggressive Mode** – alterna el cálculo de la distancia de proyección

## Notas

- No se envían órdenes.
- Diseñada con fines de demostración.
