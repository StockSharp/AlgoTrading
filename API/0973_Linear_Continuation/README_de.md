# Strategie der Linearen Fortsetzung
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Dieses Beispiel konvertiert die TradingView-Studie "Linear Continuation" nach StockSharp. Die Strategie berechnet drei gleitende Durchschnitte und protokolliert deren lineare Fortsetzungsprojektion. Sie dient als Indikator-Showcase und enthält keine Handelslogik.

## Details

- **MA Type** – Typ des gleitenden Durchschnitts (SMA oder EMA)
- **MA1 Period** – Periode für den ersten gleitenden Durchschnitt
- **MA2 Period** – Periode für den zweiten gleitenden Durchschnitt
- **MA3 Period** – Periode für den dritten gleitenden Durchschnitt
- **Aggressive Mode** – schaltet die Berechnungsdistanz der Projektion um

## Hinweise

- Es werden keine Aufträge gesendet.
- Für Demonstrationszwecke konzipiert.
