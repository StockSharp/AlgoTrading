# Estratégia de Continuação Linear
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Este exemplo converte o estudo "Linear Continuation" do TradingView para o StockSharp. A estratégia calcula três médias móveis e registra sua projeção de continuação linear. Destina-se como demonstração de indicadores e não inclui lógica de negociação.

## Detalhes

- **MA Type** – tipo de média móvel (SMA ou EMA)
- **MA1 Period** – período para a primeira média móvel
- **MA2 Period** – período para a segunda média móvel
- **MA3 Period** – período para a terceira média móvel
- **Aggressive Mode** – alterna o cálculo de distância de projeção

## Notas

- Nenhuma ordem é submetida.
- Projetada para fins de demonstração.
