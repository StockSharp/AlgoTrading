# Bezier ReOpen 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Bezier ReOpen 戦略**はカスタムのベジエ曲線インジケーターを適用してトレンド方向を追従します。
インジケーターが上向きに転換し、最新の値が前の値を上回ると、戦略はロングポジションを開くことができます。
インジケーターが下向きに転換すると、ショートポジションを開くことができます。インジケーターが方向を変えると既存のポジションが閉じられます。
エントリー後、ユーザー定義のステップだけ価格が進むたびに追加ポジションが再開され、トレンドへのスケーリングが可能になります。

この実装は MetaTrader Expert Advisor `Exp_Bezier_ReOpen.mq5`（ID 16883）に基づいています。

## 詳細

- **インジケーター**: 最後の `BPeriod` 価格とカーブテンションを定義するパラメーター `T` から構築されたベジエ曲線。
- **エントリー**:
  - **ロング**: インジケーターの傾きが上向きに転換し、現在の値が前の値を上回る。
  - **ショート**: インジケーターの傾きが下向きに転換し、現在の値が前の値を下回る。
- **エグジット**:
  - **ロング**: インジケーターの傾きが下向きに転換。
  - **ショート**: インジケーターの傾きが上向きに転換。
- **再エントリー**: 最初のエントリー後、価格が最後のエントリー価格から `PriceStep` 離れるたびに追加注文が送られ、最大 `PosTotal` 注文まで。
- **ストップ**: 絶対価格単位で定義されたオプションのストップロスとテイクプロフィット。

## パラメーター

- `CandleType` – 計算に使用するローソク足の時間軸。デフォルト: 4時間。
- `BPeriod` – ベジエ計算のバー数。デフォルト: 8。
- `T` – ベジエ曲線のテンション (0..1)。デフォルト: 0.5。
- `PriceType` – インジケーターの価格ソース (close, open, high, low, median, typical, weighted)。デフォルト: weighted。
- `PriceStep` – 追加注文を送る価格距離。デフォルト: 300。
- `PosTotal` – スケーリングシーケンスの最大ポジション数。デフォルト: 10。
- `BuyPosOpen` – ロングポジションのオープンを許可。デフォルト: true。
- `SellPosOpen` – ショートポジションのオープンを許可。デフォルト: true。
- `BuyPosClose` – 反対シグナルでロングをクローズ。デフォルト: true。
- `SellPosClose` – 反対シグナルでショートをクローズ。デフォルト: true。
- `StopLoss` – 価格単位でのストップロス。デフォルト: 1000。
- `TakeProfit` – 価格単位でのテイクプロフィット。デフォルト: 2000。
