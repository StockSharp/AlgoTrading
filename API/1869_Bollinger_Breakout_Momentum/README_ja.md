# Bollinger ブレイクアウト・モメンタム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

オリジナルの MQL 戦略から変換されました。EMA、MACD、RSI によって確認されたボリンジャーバンドのブレイクアウトを取引します。戦略はボラティリティ拡大ごとに一度だけエントリーし、中間バンドに沿ってストップを移動させながら、固定のピップ数でテイクプロフィットを使用します。

## 詳細

- **エントリー条件**:
  - ロング: バンド幅が `BreakoutFactor` を上回り、MACD > 0、RSI > 50、EMA が中間バンドの上方、前回の終値が前回の上限バンドを上回る
  - ショート: バンド幅が `BreakoutFactor` を上回り、MACD < 0、RSI < 50、EMA が中間バンドの下方、前回の終値が前回の下限バンドを下回る
- **ロング/ショート**: 両方
- **エグジット条件**:
  - ロング: 価格がトレーリング中間バンドストップに触れるか、テイクプロフィットに到達する
  - ショート: 価格がトレーリング中間バンドストップに触れるか、テイクプロフィットに到達する
- **ストップ**: ストップレベルは現在のボリンジャーバンド中間バンドで、ローソク足ごとに更新される
- **テイクプロフィット**: ピップ数で指定された固定距離
- **デフォルト値**:
  - `BollingerLength` = 18
  - `BollingerDeviation` = 2m
  - `BreakoutFactor` = 0.0015m
  - `TakeProfitPips` = 100
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Bollinger Bands, EMA, MACD, RSI
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
