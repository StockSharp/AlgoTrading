# Color Trend CF戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQLエキスパート**Exp_ColorTrend_CF**を変換したものです。2本の指数移動平均線を使用してトレンドの変化を検出します。速いEMAは価格変動に素早く反応し、遅いEMAはトレンドフィルターとして機能します。速いEMAが遅いEMAを上抜けするとロングポジションが開かれます。速いEMAが遅いEMAを下抜けするとショートポジションが開かれます。

## パラメーター

- `Period` – 速いEMAの基本期間。遅いEMAはこの値の2倍を使用します。
- `StopLoss` – 価格単位でのストップロス距離。
- `TakeProfit` – 価格単位でのテイクプロフィット距離。
- `AllowBuyOpen` – ロングポジションを開く許可。
- `AllowSellOpen` – ショートポジションを開く許可。
- `AllowBuyClose` – 売りシグナルでロングポジションを閉じる許可。
- `AllowSellClose` – 買いシグナルでショートポジションを閉じる許可。
- `CandleType` – インジケーター計算に使用する時間軸。

## トレードロジック

1. 選択した時間軸のローソク足を購読する。
2. 速いEMAと遅いEMAを計算する。
3. 速いEMAが遅いEMAを上抜けした場合：
   - 許可されている場合、ショートポジションを決済する。
   - 許可されている場合、ロングポジションを開く。
4. 速いEMAが遅いEMAを下抜けした場合：
   - 許可されている場合、ロングポジションを決済する。
   - 許可されている場合、ショートポジションを開く。
5. 保有ポジションにストップロスおよびテイクプロフィットレベルを適用する。

この実装はインジケーターバインディングを使用したStockSharpの高レベルAPIを使用します。
