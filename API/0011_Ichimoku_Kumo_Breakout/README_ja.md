# 一目均衡表・雲ブレイクアウト
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

一目均衡表の雲（Kumo）のブレイクアウトに基づく戦略。

テスト結果では、年間平均リターンが約70%であることが示されています。株式市場で最も効果を発揮します。

このアプローチはIchimokuの雲のシグナルに依存しています。価格が雲の上をブレイクし、転換線（Tenkan-sen）が基準線（Kijun-sen）をクロスアップすると買いが発動し、逆に雲の下へのブレイクはショートを開始します。価格が雲を通って戻るまでポジションが保持されます。

雲は主要なサポートとレジスタンスレベルを示しているため、システムはそれを超える決定的な終値を待ちます。複数の一目均衡表コンポーネントを組み合わせることで、横ばい市場での低確率の取引を避けます。


## 詳細

- **エントリー条件**: Ichimokuに基づくシグナル。
- **ロング/ショート**: 両方の方向。
- **エグジット条件**: 逆シグナル。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanPeriod` = 52
  - `CandleType` = TimeSpan.FromMinutes(15)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Ichimoku
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ (15m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
