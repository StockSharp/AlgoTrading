# 布林带多头策略
[English](README.md) | [Русский](README_ru.md)

该策略在价格收于布林带下轨且 RSI 进入超卖区时买入，当价格回到中轨时平仓。

## 细节

- **入场条件**：
  - 价格收于布林带下轨。
  - RSI 低于超卖水平。
- **多空方向**：仅做多。
- **出场条件**：
  - 价格收于或高于布林带中轨。
- **止损**：无。
- **默认值**：
  - `BbLength` = 10
  - `BbDeviation` = 2
  - `RsiLength` = 14
  - `RsiOversold` = 30
- **筛选**：
  - 类型：均值回归
  - 方向：多头
  - 指标：布林带、RSI
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
