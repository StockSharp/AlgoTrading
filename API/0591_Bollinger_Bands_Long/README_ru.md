# Стратегия Bollinger Bands Long
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия покупает, когда цена закрывается ниже нижней границы полос Боллинджера и RSI находится в зоне перепроданности. Позиция закрывается при возвращении цены к средней полосе.

## Детали

- **Условия входа**:
  - Цена закрывается ниже нижней полосы Боллинджера.
  - RSI ниже уровня перепроданности.
- **Направление**: Только лонг.
- **Условия выхода**:
  - Цена закрывается на уровне или выше средней полосы.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `BbLength` = 10
  - `BbDeviation` = 2
  - `RsiLength` = 14
  - `RsiOversold` = 30
- **Фильтры**:
  - Категория: Возврат к среднему
  - Направление: Лонг
  - Индикаторы: Bollinger Bands, RSI
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
