# RCI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ランク相関指数とその移動平均を使用してクロスオーバーを取引します。RCIが移動平均を上回ったときにロングポジションを開きます。下回ったときにショートポジションを開きます。取引方向はロングのみまたはショートのみに制限できます。

## 詳細
- **エントリー条件**: RCIがその移動平均を交差する。
- **ロング/ショート**: 設定可能（両方、ロングのみ、ショートのみ）。
- **エグジット条件**: 反対方向のクロスオーバー。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `RciLength` = 10
  - `MaType` = SMA
  - `MaLength` = 14
  - `Direction` = Long & Short
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 設定可能
  - インジケーター: RCI, MA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
