# Стратегия RCI
[English](README.md) | [中文](README_cn.md)

Стратегия использует индекс ранговой корреляции (RCI) и его скользящую среднюю для торговли на пересечениях. Длинная позиция открывается, когда RCI пересекает свою среднюю снизу вверх. Короткая позиция открывается при обратном пересечении. Направление торговли можно ограничить только длинными или только короткими сделками.

## Детали
- **Критерии входа**: пересечение RCI и его средней.
- **Длинные/короткие**: настраиваемо (оба, только лонг, только шорт).
- **Критерии выхода**: противоположное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `RciLength` = 10
  - `MaType` = SMA
  - `MaLength` = 14
  - `Direction` = Long & Short
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Настраиваемое
  - Индикаторы: RCI, MA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
