# プロジェクション戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は直近の日次始値の平均変化率を計算し、当日の始値を中心にブレイクアウトレベルを投影します。価格が上方プロジェクションを上回るとロングポジションを開き、下方プロジェクションを下回るとショートポジションを開きます。プロテクティブストップはプロジェクションの反対側付近に置かれます。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格が `open + threshold` を上抜く。
  - **ショート**: 価格が `open - threshold` を下抜く。
- **エグジット条件**:
  - **ロング**: 価格がロングストップを下回る。
  - **ショート**: 価格がショートストップを上回る。
- **ストップ**: あり、平均変化に基づく。
- **パラメーター**:
  - `TargetMultiple` – 平均変化の乗数（デフォルト 0.2）。
  - `Threshold` – ブレイクアウトレベル形成に使用する平均変化のパーセンテージ（デフォルト 1.0）。
  - `CalculationPeriod` – 平均計算に使用する日数（デフォルト 5）。
