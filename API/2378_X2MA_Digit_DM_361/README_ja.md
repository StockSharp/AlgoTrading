# X2MA Digit DM 361戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、2本の移動平均と平均方向指数（ADX）を組み合わせます。
高速移動平均が低速移動平均より上にあり、かつ正方向指数（+DI）が負方向指数（-DI）より大きい場合にロングポジションをオープンします。
高速移動平均が低速移動平均より下にあり、かつ-DIが+DIより大きい場合にショートポジションをオープンします。

この戦略はパーセンテージベースのストップロスとテイクプロフィットの保護を使用します。計算用の足は指定された時間軸から取得されます。

## パラメーター
- **Fast MA Length** – 高速移動平均の期間。
- **Slow MA Length** – 低速移動平均の期間。
- **ADX Length** – 平均方向指数の計算期間。
- **Stop Loss %** – エントリー価格のパーセントで表したストップロスのサイズ。
- **Take Profit %** – エントリー価格のパーセントで表したテイクプロフィットのサイズ。
- **Candle Type** – 処理に使用する足の時間軸。
