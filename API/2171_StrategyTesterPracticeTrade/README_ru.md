# Стратегия Strategy Tester Practice Trade
[English](README.md) | [中文](README_cn.md)

Утилита для отработки ручной торговли в тестере. Стратегия отслеживает появление файлов команд и исполняет рыночные приказы. Создайте пустой файл `buy.txt`, `sell.txt` или `close.txt` в выбранной папке, чтобы на следующей завершённой свече исполнить покупку, продажу или закрытие позиции.

## Детали

- **Критерий входа**: наличие `buy.txt` или `sell.txt` в папке команд
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: `close.txt` закрывает любую открытую позицию
- **Стопы**: нет
- **Значения по умолчанию**:
  - `LotSize` = 1
  - `CommandDir` = системная временная папка
  - `CandleType` = свечи 1 минута
- **Фильтры**:
  - Категория: Утилиты
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
