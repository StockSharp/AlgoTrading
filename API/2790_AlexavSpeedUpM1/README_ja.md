# Alexav SpeedUp M1 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
- MetaTrader 5 の「Alexav SpeedUp M1」エキスパートアドバイザーを StockSharp 高レベル API に変換したもの。
- デフォルトで1分足を使用し、異常に大きなロウソク足本体に反応します。
- 強いロウソク足本体の方向に単一のネットポジションを建て、固定ストップロス、テイクプロフィット、段階的トレーリングストップで管理します。
- pip ベースの入力を使用し、銘柄のティックサイズと小数点精度に従って自動的に価格距離に変換されます。

## オリジナルのアイデアと StockSharp 実装の違い
- オリジナル EA はヘッジ口座でロングとショートの取引を同時に開いていました。StockSharp 戦略はネッティング環境で動作するため、このポートは一度に1つのポジションのみを保持し、大きなロウソク足の方向に入ります。
- トレーリングストップのロジックは MT5 バージョンに従います：`TrailingStop + TrailingStep` 分価格が動くのを待ってからストップをトレーリング距離だけ近づけ、前のストップより少なくとも1つのトレーリングステップ分価格が進んだときのみ更新します。
- pip 距離は最小ティックサイズを掛けることで価格単位に変換されます。3桁または5桁小数の外国為替シンボルの場合、コードは MT5 の pip 処理を模倣するためにティックを10倍します。

## エントリールール
1. 選択した時間軸の完成したロウソク足で作業します（デフォルト：1分）。
2. ロウソク足本体を測定：`abs(Close - Open)`。
3. 本体が `MinimumBodySizePips * pipSize` を超え、アクティブなポジションがない場合、ロウソク足本体の方向にエントリーします：
   - 陽線 → ロングポジションを開く。
   - 陰線 → ショートポジションを開く。

## エグジットルール
- **ストップロス** – エントリー価格から `StopLossPips * pipSize` の位置に設定。パラメーターがゼロの場合は無効。
- **テイクプロフィット** – エントリーから `TakeProfitPips * pipSize` の位置に設定。パラメーターがゼロの場合は無効。
- **トレーリングストップ** – `TrailingStopPips > 0` かつ `TrailingStepPips > 0` のとき有効。
  - 取引が少なくとも `TrailingStopPips + TrailingStepPips` pips 利益を上げた後に起動します。
  - ロング取引の場合、条件が満たされ価格が前のストップより少なくとも1つのトレーリングステップ分進んだとき、ストップは `Close - TrailingStopPips * pipSize` に移動します。
  - ショート取引の場合、同じステップ条件を使用してストップは `Close + TrailingStopPips * pipSize` に移動します。

## パラメーター
- `OrderVolume` – 取引サイズ（ロット）（デフォルト `0.1`）。
- `StopLossPips` – ストップロス距離（pips）（デフォルト `30`）。
- `TakeProfitPips` – テイクプロフィット距離（pips）（デフォルト `90`）。
- `TrailingStopPips` – トレーリングストップ距離（pips）（デフォルト `10`）。
- `TrailingStepPips` – トレーリングストップが更新される前の最小有利な動き（デフォルト `5`）。トレーリングストップが有効の場合、ゼロより大きくなければなりません。
- `MinimumBodySizePips` – 取引をトリガーするのに必要な最小本体サイズ（pips）（デフォルト `100`）。
- `CandleType` – ロウソク足の時間軸（デフォルト `1 Minute`）。

## 可視化
- 戦略は、利用可能なチャートエリアがあれば選択したロウソク足シリーズと自身の取引を自動的に描画し、テスト中のシグナル検査を簡素化します。

## 使用上の注意
- デフォルト設定は MT5 設定を反映しています。取引する銘柄のボラティリティに合わせて pip 距離を調整してください。
- ネットポジションのみがサポートされているため、同時ロングとショートポジションを期待するヘッジ口座では戦略を実行しないでください。
- ティックサイズが大きい市場では、比較可能な価格距離を維持するために pip ベースの入力を相応に減らしてください。
