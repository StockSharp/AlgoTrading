# Стратегия RSI Donchian

Стратегия ищет экстремальные значения RSI, совпадающие с пробоями канала Дончиана. RSI оценивает перекупленность и перепроданность, а канал задаёт ключевые уровни цены.

Покупка совершается, когда RSI опускается ниже 30 и цена пробивает верхнюю границу канала. Продажа — при RSI выше 70 и пробое нижней границы. Выход из позиции происходит после возврата цены к средней линии канала.

Метод подойдёт активным трейдерам, предпочитающим торговать на истощении импульса при наличии чётких уровней пробоя. Процентный стоп‑лосс ограничивает риск.

## Подробности
- **Условия входа**:
  - **Long**: RSI < 30 && Close > Donchian High
  - **Short**: RSI > 70 && Close < Donchian Low
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход при закрытии ниже Donchian Middle
  - **Short**: выход при закрытии выше Donchian Middle
- **Стопы**: да, процентный стоп‑лосс
- **Значения по умолчанию**:
  - `RsiPeriod` = 14
  - `DonchianPeriod` = 20
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: RSI, Donchian Channel
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
