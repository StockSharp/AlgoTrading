# FmOneスキャルピング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
FmOneスキャルピング戦略は、FMOneEA MetaTrader 4エキスパートアドバイザーを簡略化して移植したものです。この戦略は、高速・低速の指数移動平均とMACDインジケーターを組み合わせて、あらゆる時間軸で短期モメンタムを捉えます。

## 仕組み
1. 高速EMAと低速EMAが現在のトレンド方向を定義します。
2. MACDヒストグラムがトレンド方向のモメンタムを確認します。
3. 高速EMAが低速EMAを上回り、MACDヒストグラムがプラスの場合に買い注文を開きます。
4. 高速EMAが低速EMAを下回り、MACDヒストグラムがマイナスの場合に売り注文を開きます。
5. 各ポジションは設定可能なストップロスとテイクプロフィットで保護されます。トレーリングストップを有効にして利益方向の動きに追随できます。

## パラメーター
- **FastMaPeriod** – 高速EMAの長さ。
- **SlowMaPeriod** – 低速EMAの長さ。
- **MacdSignalPeriod** – MACDインジケーターのシグナルライン期間。
- **StopLossPercent** – エントリー価格に対するパーセントでのストップロスサイズ。
- **TakeProfitPercent** – エントリー価格に対するパーセントでのテイクプロフィットサイズ。
- **EnableTrailingStop** – トレーリングストップ管理を有効にします。
- **CandleType** – 受信ローソク足の時間軸。

## 注記
このポートはオリジナルEAのコアロジックに焦点を当てています。MQLバージョンの換金サイクルやブレークイーブン自動化などの高度な機能は、例を読みやすくするために意図的に省略されています。
