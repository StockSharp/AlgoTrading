# RSI 钩形反转策略
[English](README.md) | [Русский](README_ru.md)

当 RSI 从极端水平回钩时，RSI 钩形反转试图捕捉短期转折。指标在超买或超卖后常先向中线回钩，随后价格才会反应。

测试表明年均收益约为 163%，该策略在股票市场表现最佳。

策略等待这种回钩出现，而价格仍在原方向上走。RSI 在超卖区抬头而价格创新低时做多；在超买区下弯且价格创新高时做空。

交易使用固定百分比止损控制风险，通常在 RSI 再次向相反方向钩回时平仓。

## 细节

- **入场条件**：指标信号。
- **多/空**：双向。
- **退出条件**：止损或相反信号。
- **止损**：是，按百分比。
- **默认值**：
  - `CandleType` = 15 分钟
  - `StopLoss` = 2%
- **过滤条件**：
  - 类别: 反转
  - 方向: 双向
  - 指标: RSI
  - 止损: 有
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
