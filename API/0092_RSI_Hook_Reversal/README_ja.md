# RSI Hook Reversal Strategy
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSI Hook Reversal戦略は、RSIが極端な水準から脱するときの短期的な転換点を捉えることを目的としています。買われすぎや売られすぎの後、インジケーターは価格が反応する前に中央ラインに向けて「フック」することがよくあります。

テストでは平均年間リターンが約163%であることが示されています。株式市場で最もパフォーマンスが優れています。

この戦略は、価格が前の方向に押し続ける間にそのフックを待ちます。RSIが売られすぎから上向きにカールし、価格が新安値をつけると同時にロングエントリーがトリガーされます。一方、RSIが買われすぎから下向きに転じ、新高値が形成されるときにショートがオープンします。

取引はリスクをコントロールするためにシンプルなパーセントストップを使用し、RSIが反対方向にフックすると通常クローズします。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15分
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
