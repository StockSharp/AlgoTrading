# 3つのMAクロスチャネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
**3 MA クロスチャネル戦略** は、MetaTrader エキスパート アドバイザー `3MaCross_EA` を StockSharp の高レベル API に変換します。設定可能な 3 つの移動平均を監視し、速い平均が遅い平均を通過したときに取引を開始します。 Donchian 価格チャネルはオプションでエグジットの管理に使用され、「価格チャネル」インジケーターを参照した元の EA を厳密に模倣します。

## 取引ロジック
- **ロングエントリー**: 現在の足で、高速移動平均と中移動平均の両方が低速移動平均を上回って終了し、2 つの高速移動平均のいずれかが低速移動平均を上回ったときに生成されます。
- **ショートエントリー**: 高速移動平均と中移動平均の両方が低速移動平均を下回って終了し、2 つの高速移動平均のいずれかが低速移動平均を下回ったときにトリガーされます。
- **ポジション出口**:
  - 逆クロスオーバー信号。
  - オプションの Donchian チャネルストップ: 価格が下限バンドを下回った場合、ロングポジションは閉じられます。価格が上部バンドを超えて上昇すると、ショートポジションはクローズされます。
  - 絶対価格単位で測定されるオプションの固定テイクプロフィットまたはストップロス距離。

この戦略は、元のスクリプトの `TradeAtCloseBar` の動作と一致して、常にキャンドルが完了するのを待ちます。一度に維持されるのは 1 方向の位置のみです。既存のポジションに対してシグナルが表示されると、新しい取引が開始される前に現在の取引が終了します。

## パラメーター
| 名前 | 種類 | デフォルト | 説明 |
|------|------|---------|-------------|
| `FastLength` | `int` | `2` | 高速移動平均を振り返ります。 |
| `MediumLength` | `int` | `4` | 中移動平均を振り返ります。 |
| `SlowLength` | `int` | `30` | 遅い移動平均を振り返ってみましょう。 |
| `ChannelLength` | `int` | `15` | Donchian チャネル ウィンドウはチャネルベースの終了に使用されます。 |
| `FastType` | `MovingAverageTypeEnum` | `EMA` | 高速平均に適用される移動平均アルゴリズム (SMA、EMA、SMMA、WMA)。 |
| `MediumType` | `MovingAverageTypeEnum` | `EMA` | 移動平均アルゴリズムが中間平均に適用されます。 |
| `SlowType` | `MovingAverageTypeEnum` | `EMA` | 移動平均アルゴリズムが低速平均に適用されます。 |
| `TakeProfit` | `decimal` | `0` | 絶対価格単位での利益目標。無効にするには、`0` に設定します。 |
| `StopLoss` | `decimal` | `0` | 絶対価格単位での損失制限。無効にするには、`0` に設定します。 |
| `UseChannelStop` | `bool` | `true` | Donchian チャネル出口を有効にします。 |
| `CandleType` | `DataType` | `TimeSpan.FromMinutes(1).TimeFrame()` | 計算に使用されるローソクの種類。 |

## 注意事項
- すべての移動平均は終値を使用し、元の EA の `FasterMode`、`MediumMode`、および `SlowerMode` のオプションと一致するように個別に設定できます。
- `TakeProfit` と `StopLoss` は絶対価格距離を使用します (例: `0.0010` は 5 桁の外国為替記号の 10 ピップに対応します)。これらはローソク足の終値で評価され、EA の足終値管理を再現します。
- `UseChannelStop` が有効になっている場合、戦略は `Price Channel` カスタム インジケーターに依存した自動ストップロス動作を再現します。
- この戦略では、視覚的に確認できるように、チャート上に 3 つの移動平均、Donchian チャネル、取引マーカーを描画します。
