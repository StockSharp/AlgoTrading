# ADX スロープ・ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ADX スロープ・ブレイクアウト戦略は、ADXの変化率を追跡します。異常に急な傾きは、新しいトレンドが形成されつつあることを示唆します。

テストによると、年平均リターンは約145%です。暗号資産市場で最もよく機能します。

傾きが標準偏差の倍数だけ典型的なレベルを超えたときにエントリーし、加速の方向にトレードを行い、保護的なストップを設定します。

早期のトレンド参入を求めるアクティブなトレーダーに向いています。傾きが通常の水準に戻ったときにポジションを手仕舞います。デフォルト `AdxPeriod` = 14。

## 詳細

- **エントリー条件**: インジケーターが偏差乗数分だけ平均を超える。
- **ロング/ショート**: 両方向。
- **エグジット条件**: インジケーターが平均に戻る。
- **ストップ**: はい。
- **デフォルト値**:
  - `AdxPeriod` = 14
  - `SlopePeriod` = 20
  - `BreakoutMultiplier` = 2.0m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: ADX
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
