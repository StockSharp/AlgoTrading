# Стратегия Max Pain
[English](README.md) | [中文](README_cn.md)

Эта стратегия открывает длинные позиции, когда объём и ценовое движение превышают заданные пороги, а индекс VIX остаётся ниже указанного уровня. При входе устанавливается стоп-лосс на основе волатильности, позиция закрывается после фиксированного количества периодов.

## Детали

- **Условия входа**:
  - **Лонг**: объём больше среднего × `VolumeMultiplier` и изменение цены больше предыдущего закрытия × `PriceChangeMultiplier` при VIX ниже `VixThreshold`.
- **Лонг/Шорт**: только лонг.
- **Условия выхода**:
  - Стоп-лосс на расстоянии `StopLossMultiplier` × волатильность ниже цены входа.
  - Закрытие позиции после `HoldPeriods` баров.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `LookbackPeriod` = 70.
  - `VolumeMultiplier` = 1.
  - `PriceChangeMultiplier` = 0.029.
  - `StopLossMultiplier` = 2.4.
  - `VixThreshold` = 44.
  - `HoldPeriods` = 8.
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
  - `VixCandleType` = TimeSpan.FromDays(1).TimeFrame().
- **Фильтры**:
  - Категория: Пробой
  - Направление: Только лонг
  - Индикаторы: Объём, ценовое действие, волатильность
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
