Opera mudanças de direção do SuperTrend com take profit e stop loss fixos opcionais.

- **Comprado**: SuperTrend muda para tendência de alta.
- **Vendido**: SuperTrend muda para tendência de baixa.
- **Stop Loss**: distância fixa da entrada (opcional).
- **Take Profit**: distância fixa da entrada (opcional).
- **Indicadores**: SuperTrend.
