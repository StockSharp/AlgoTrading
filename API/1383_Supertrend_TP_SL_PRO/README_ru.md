# Стратегия Supertrend TP SL PRO
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Торгует смену направления SuperTrend с фиксированными тейк-профитом и стоп-лоссом.

- **Лонг**: SuperTrend переключается вверх.
- **Шорт**: SuperTrend переключается вниз.
- **Стоп-лосс**: фиксированное расстояние от входа (опционально).
- **Тейк-профит**: фиксированное расстояние от входа (опционально).
- **Индикаторы**: SuperTrend.
