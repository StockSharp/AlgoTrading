# Supertrend TP SL PRO戦略

オプションの固定テイクプロフィットとストップロスを使ってSuperTrendの方向転換でトレードします。

- **ロング**: SuperTrendが上昇トレンドに転換。
- **ショート**: SuperTrendが下降トレンドに転換。
- **ストップロス**: エントリーからの固定距離（オプション）。
- **テイクプロフィット**: エントリーからの固定距離（オプション）。
- **インジケーター**: SuperTrend。
