# 注文エキスパート戦略 (1916)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、銘柄の価格が事前定義されたレベルに達したときに市場ポジションを開きます。チャートラインで注文を管理した元のMQLエキスパートの動作を模倣しています。

## 動作原理
- 設定可能な時間軸のローソク足を購読します。
- 終値が`BuyLevel`または`SellLevel`のしきい値を超えると、ロングまたはショートの成行ポジションを開きます。
- ストップロスとテイクプロフィットの値は、`StopLossPip`と`TakeProfitPip`を使用してエントリー価格から計算されます。
- オプションのトレーリングストップは、価格が有利な方向に動くにつれてストップロスを現在価格に向けて移動させます。

## パラメーター
- **TakeProfitPip** – エントリー価格からテイクプロフィットまでのpips距離。
- **StopLossPip** – エントリー価格からストップロスまでのpips距離。
- **EnableTrailingStop** – トレーリングストップロジックの有効/無効。
- **CandleType** – 計算に使用するローソク足タイプ。
- **BuyLevel** – ロングエントリーを起動する価格レベル（0で無効）。
- **SellLevel** – ショートエントリーを起動する価格レベル（0で無効）。

## 注意事項
- 戦略は高レベルAPIを使用し、完成したローソク足のみを処理します。
- 偶発的な大きなポジションを避けるため、起動時に保護サブシステムが有効化されます。
