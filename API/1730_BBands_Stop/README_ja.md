# BBands Stop戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はボリンジャーバンドから派生したBBands Stopインジケーターを使用して市場のトレンドを追従します。ストップラインが上向きに転換すると、ショートポジションを決済してロングポジションを建てます。下向きの転換はロングポジションを決済してショートを建てます。パラメーターはボリンジャーの期間、偏差、リスクオフセット、ロングとショートへの参入・退出の許可を制御します。

## 詳細

- **エントリー条件**:
  - **ロング**: 上昇トレンドのストップラインが有効。
  - **ショート**: 下降トレンドのストップラインが有効。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 反対のストップシグナル。
- **ストップ**: ボリンジャーバンドからのトレーリングストップ。
- **フィルター**: なし。
