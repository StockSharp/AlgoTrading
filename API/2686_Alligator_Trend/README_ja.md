# Alligatorトレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MetaTraderのオリジナルスクリプト（`Alligator.mq5`）からクラシックなビル・ウィリアムズのAlligatorシステムを再現しています。中値を基に構築され、市場のフェーズを可視化するために前方にシフトされた3本の平滑化移動平均を使用します。高速のLipsラインがTeethの上にあり、TeethがJawの上にある場合にロングポジションを建てます。逆の並び順の場合にショートポジションを建てます。同時に1つのポジションしかアクティブにできません。

取引中、戦略はpipsで表されたストップロスとテイクプロフィットでポジションを保護します。市場が設定可能なゼロレベル距離だけ取引に有利な方向に動いた場合、ストップはブレイクイーブンに移動されます。トレーリングストップは最高値（ロング）または最安値（ショート）を追跡し、頻繁なストップ更新を避けるために最小ステップが設定されています。ストップロス、トレーリングストップ、またはテイクプロフィットレベルに達した場合にポジションが決済されます。

デフォルト設定は30分足とFX形式のpip値を対象としていますが、他の市場向けにパラメーターを最適化できます。元のMQLバージョンはブローカー固有のpip処理を使用するため、変換は`PriceStep`に依存してpip距離を絶対価格に換算します。

## 取引ルール

### エントリー
- **ロング**: オープンポジションなし、かつ最後に完成したローソク足で`Lips > Teeth > Jaw`。
- **ショート**: オープンポジションなし、かつ最後に完成したローソク足で`Lips < Teeth < Jaw`。

### 決済とリスク管理
- **初期ストップ**: 約定価格から`StopLossPips`下（ロング）または上（ショート）に設置。
- **テイクプロフィット**: 約定価格から`TakeProfitPips`の位置に設置。
- **ゼロレベル**: 価格が`ZeroLevelPips`進んだ場合、ストップをエントリー価格に移動。
- **トレーリングストップ**: ゼロレベル発動後、ストップは`TrailingStopPips`の距離で極値を追跡し、改善が`TrailingStepPips`を超えた場合のみ更新されます。
- ローソク足データで任意のストップまたはテイクプロフィットレベルに触れた場合、ポジションは即座に決済されます。

## パラメーター

| パラメーター | デフォルト | 説明 |
|-------------|-----------|------|
| `CandleType` | 30分の時間軸 | インジケーター計算とシグナル評価に使用するローソク足シリーズ。 |
| `JawLength` | 13 | 青いJawラインの平滑化移動平均期間。 |
| `TeethLength` | 8 | 赤いTeethラインの平滑化移動平均期間。 |
| `LipsLength` | 5 | 緑のLipsラインの平滑化移動平均期間。 |
| `JawShift` | 8 | Jawラインの前方シフト（バー数）。 |
| `TeethShift` | 5 | Teethラインの前方シフト（バー数）。 |
| `LipsShift` | 3 | Lipsラインの前方シフト（バー数）。 |
| `EnableLong` | `true` | ロングエントリーを許可または禁止します。 |
| `EnableShort` | `true` | ショートエントリーを許可または禁止します。 |
| `StopLossPips` | 45 | 約定価格からのストップロス距離（pips）。 |
| `TakeProfitPips` | 145 | 約定価格からのテイクプロフィット距離（pips）。 |
| `ZeroLevelPips` | 30 | ストップをブレイクイーブンに移動するために必要な距離（pips）。 |
| `TrailingStopPips` | 50 | 現在の極値とトレーリングストップ間の距離。 |
| `TrailingStepPips` | 10 | トレーリングストップを更新する前に必要な最小pip改善。 |

## 注記

- AlligatorインジケーターはMetaTraderの実装に合わせて中値`(High + Low) / 2`で計算されます。
- シフトされたラインの値は内部バッファでエミュレートされ、オリジナルスクリプトと同じシフトデータを使用して比較が行われます。
- 戦略は、同じバーで新しいシグナルが処理される前に1つの取引が実行されることを想定しており、ソースEAのバー単位実行を反映しています。
- 取引する銘柄のティックサイズとボラティリティに合わせてpip距離を最適化してください。
