# 時間軸戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

時間軸を意識したリスク管理を備えたEMAクロスオーバー戦略。

テストでは年平均リターン約31%を示しています。暗号資産市場で最も優れたパフォーマンスを発揮します。

この戦略は、速いEMAが遅いEMAを上抜け、長期トレンドが強気のときに買いを入れます。ショートエントリーは逆のクロスで発生します。取引時間帯とシンプルなADXフィルターが低モメンタム期間を回避するのに役立ちます。リスクはパーセンテージベースのテイクプロフィットとストップロスで管理されます。

## 詳細

- **エントリー条件**:
  - **ロング**: EMA50がEMA200より上にある間に、EMA9がEMA20を上抜けるとき。
  - **ショート**: EMA50がEMA200より下にある間に、EMA9がEMA20を下抜けるとき。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: ストップロスまたはテイクプロフィット。
  - **ショート**: ストップロスまたはテイクプロフィット。
- **ストップ**: あり、オプションのトレーリングストップ。
- **デフォルト値**:
  - `TakeProfitPercent` = 1.5
  - `StopLossPercent` = 1.0
  - `TrailingPercent` = 0.5
  - `StartHour` = 15
  - `EndHour` = 20
  - `CooldownBars` = 5
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA, RSI, ADX
  - ストップ: あり
  - 複雑さ: 中程度
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
