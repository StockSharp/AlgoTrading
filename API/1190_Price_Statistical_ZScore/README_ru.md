# Price Statistical Z-Score
[English](README.md) | [中文](README_cn.md)

Стратегия использует сглаженный Z-Score и фильтр по свечному импульсу.

Покупка происходит, когда краткосрочный Z-Score поднимается выше долгосрочного, а выход — когда он опускается ниже. Стратегия пропускает сигналы после нескольких одинаковых и избегает входа после трёх бычьих свечей подряд.

## Детали

- **Критерии входа**: краткосрочный Z-Score выше долгосрочного, отсутствие бычьей серии из 3 свечей, зазор между сигналами.
- **Длинные/короткие**: только лонг.
- **Критерии выхода**: краткосрочный Z-Score ниже долгосрочного, отсутствие медвежьей серии из 3 свечей, зазор между сигналами.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `ZScoreBasePeriod` = 3
  - `ShortSmoothPeriod` = 3
  - `LongSmoothPeriod` = 5
  - `GapBars` = 5
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
