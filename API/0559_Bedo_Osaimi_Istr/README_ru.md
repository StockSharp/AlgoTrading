# Стратегия Bedo Osaimi Istr
[English](README.md) | [中文](README_cn.md)

Простая трендовая стратегия, сравнивающая скользящие средние цен закрытия и открытия. Длинная позиция открывается, когда скользящая средняя закрытия пересекает сверху скользящую среднюю открытия. При обратном пересечении позиция разворачивается.

## Подробности

- **Условия входа**:
  - MA закрытия пересекает MA открытия снизу вверх.
- **Направление**: в обе стороны.
- **Условия выхода**:
  - MA закрытия пересекает MA открытия сверху вниз (для выхода из лонга или входа в шорт).
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `MaLength` = 20
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA по цене закрытия и открытия
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
