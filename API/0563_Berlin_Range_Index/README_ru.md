# Стратегия Berlin Range Index
[English](README.md) | [中文](README_cn.md)

Стратегия Berlin Range Index использует индекс хаотичности, отфильтрованный фактором на основе ATR, чтобы определить фазы тренда и бокового движения. Когда фильтрованный индекс опускается ниже минимального порога, стратегия открывает позицию в направлении текущей свечи. Позиции закрываются, когда индекс указывает на флэт или ослабевающий тренд.

## Детали

- **Условия входа**:
  - Фильтрованный индекс ниже `ChopMin`, направление свечи определяет покупку или продажу.
- **Условия выхода**:
  - Индекс выше `ChopMax` или признаки ослабления тренда.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `Length` = 9
  - `ChopMax` = 40
  - `ChopMin` = 10
  - `AtrLength` = 14
  - `LowLookback` = 14
  - `UseNormalized` = true
  - `StdDevLength` = 14
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Оба
  - Индикаторы: Choppiness Index, ATR, Standard Deviation
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
