# Стратегия на финансовых коэффициентах
[English](README.md) | [中文](README_cn.md)

Стратегия анализирует квартальные финансовые коэффициенты компании. Сравниваются текущий коэффициент ликвидности, покрытие процентов, оборачиваемость кредиторской задолженности и валовая маржа. Покупка производится, когда любой из показателей улучшился относительно предыдущего периода.

## Детали

- **Условия входа**:
  - **Лонг**: `currentRatio > previousCurrent` ИЛИ `interestCoverage < previousInterest` ИЛИ `payableTurnover > previousPayable` ИЛИ `grossMargin > previousGross`.
- **Лонг/Шорт**: Только лонг.
- **Условия выхода**:
  - **Лонг**: `currentRatio < previousCurrent` ИЛИ `interestCoverage > previousInterest` ИЛИ `payableTurnover < previousPayable` ИЛИ `grossMargin < previousGross`.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Тип свечей` = дневные свечи.
- **Фильтры**:
  - Категория: Фундаментальная
  - Направление: Лонг
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Долгосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
