# ADX MACD 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
ADX MACD は、Average Directional Index のトレンド強度と MACD のモメンタム変化を組み合わせます。
ADX が上昇しているとき、特に MACD が同じ方向にクロスすれば、ブレイクアウトが継続する可能性が高まります。

テストでは平均年間リターンは約139%を示しています。株式市場で最も良いパフォーマンスを発揮します。

この戦略はそれらの一致したシグナルでトレードし、ADX が弱まり始めるか MACD がポジションに逆らって転換したときにイグジットします。

適度なパーセントストップが、乱高下する相場での損失を抑えます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: ADX, MACD
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
