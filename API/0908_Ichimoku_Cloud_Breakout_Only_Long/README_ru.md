# Ichimoku Cloud Breakout Only Long
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинные позиции при пробое цены выше облака Ишимоку и закрывает их, когда цена опускается ниже облака. Торгуются только лонги.

## Детали

- **Условия входа**:
  - Лонг: цена пересекает сверху `max(SenkouA, SenkouB)`
- **Длинные/короткие**: только длинные
- **Условия выхода**:
  - Цена пересекает снизу `min(SenkouA, SenkouB)`
- **Стопы**: нет
- **Значения по умолчанию**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanBPeriod` = 52
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: Ichimoku
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
