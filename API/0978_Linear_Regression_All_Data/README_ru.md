# Стратегия Linear Regression (All Data)
[English](README.md) | [中文](README_cn.md)

Стратегия строит линию линейной регрессии по всем доступным барам и отображает её на графике.
Также выводятся значения наклона, интерсепта и коэффициентов корреляции.

## Детали

- **Условия входа**: нет.
- **Лонг/Шорт**: нет.
- **Условия выхода**: нет.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `MaxBarsBack` = 5000.
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **Фильтры**:
  - Категория: Служебная
  - Направление: нет
  - Индикаторы: Linear Regression
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
