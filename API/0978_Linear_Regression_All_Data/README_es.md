# Estrategia de Regresión Lineal con Todos los Datos
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia calcula una línea de regresión lineal utilizando todas las barras disponibles y la traza en el gráfico.
También registra los coeficientes de pendiente, intercepto y correlación.

## Detalles

- **Criterios de entrada**: Ninguno.
- **Largo/Corto**: Ninguno.
- **Criterios de salida**: Ninguno.
- **Stops**: No.
- **Valores predeterminados**:
  - `MaxBarsBack` = 5000.
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **Filtros**:
  - Categoría: Utilidad
  - Dirección: Ninguno
  - Indicadores: Linear Regression
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Cualquiera
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Bajo
