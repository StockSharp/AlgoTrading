# Plateau戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Plateau戦略は、元のMetaTrader 5エキスパートアドバイザーを変換したものです。価格が下部バンド付近で取引されている場合の潜在的なリバーサルを検出するために、線形加重移動平均のペアとBollinger Bandsを組み合わせています。

## トレーディングアイデア

* 選択された平滑化方法と価格ソースを使用して高速および低速の移動平均を計算します。
* 同じ価格シリーズの周りにBollinger Bandsを構築します。
* 前のローソク足が下部バンドを下回ってクローズしながら高速平均が低速平均を上回ると、ロングポジションを開きます。
* 前のローソク足が下部バンドを上回ってクローズしながら高速平均が低速平均を下回ると、ショートポジションを開きます。
* `Reverse`スイッチが有効な場合、オプションでシグナルを反転します。

## 注文管理

* ポジションは固定ロットまたは取引ごとにポートフォリオ価値のパーセンテージをリスクにさらすことでサイジングできます。
* ストップロスとテイクプロフィットレベルはpipで表され、マーケット注文が約定した直後に設定されます。
* トレーリング距離とステップの両方が正の場合、トレーリングストップを有効にできます。
* `Close Opposite`が有効な場合、戦略は新しい取引に入る前に自動的に反対のポジションを閉じます。

## パラメーター

| パラメーター | 説明 |
| --- | --- |
| Stop Loss | pipでのストップロス距離。 |
| Take Profit | pipでのテイクプロフィット距離。 |
| Trailing Stop | pipでのトレーリングストップ距離。 |
| Trailing Step | トレーリングストップを動かすために必要な最小増分（pip単位）。 |
| Money Mode | 固定ロットとリスクパーセンテージのサイジングのどちらかを選択します。 |
| Lot / Risk | 選択されたマネーモードに応じた固定ロットサイズまたはリスクパーセンテージ。 |
| Fast MA / Slow MA | 移動平均ペアの期間。 |
| MA Shift | 両方の移動平均に適用される水平シフト。 |
| MA Method | 移動平均の平滑化アルゴリズム。 |
| MA Price | 移動平均計算に使用する価格ソース。 |
| Bands Period | Bollinger Bandsの平均化期間。 |
| Bands Shift | Bollinger Bands値に適用される水平シフト。 |
| Bands Deviation | Bollinger Bandsの標準偏差乗数。 |
| Bands Price | Bollinger Bands計算に使用する価格ソース。 |
| Reverse | ロングとショートのシグナルロジックを反転します。 |
| Close Opposite | 新しいポジションを開く前に反対方向の既存ポジションを閉じます。 |
| Verbose Log | 詳細な実行情報をログに印刷します。 |
| Candle Type | インジケーター計算に使用するローソク足データシリーズ。 |

## 注意事項

* pip サイズは元のエキスパートの動作に合わせるために、3桁または5桁の小数点を持つインストゥルメントに自動的に調整されます。
* トレーリングストップが有効な場合、トレーリングステップは厳密に正の値でなければなりません。そうでなければ、戦略は起動時にエラーをスローします。
* リスクベースのポジションサイジングには、有効なストップロス距離とポートフォリオ評価データの両方が必要です。利用できない場合、戦略はデフォルトボリュームにフォールバックします。
