# ADX CCI MA
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает ADX, CCI и настраиваемую скользящую среднюю для торговли сильных трендов.

Система покупает, когда +DI пересекает -DI снизу, CCI > 100 и ADX выше порога (при включенном MA-фильтре закрытие выше MA). Продает, когда -DI пересекает +DI снизу, CCI < -100 и ADX выше порога (закрытие ниже MA).

Стоп-лосс и тейк-профит задаются в процентах. Необязательное управление риском по MA закрывает позицию после нескольких свечей против скользящей средней.

## Детали

- **Условия входа**: пересечение +DI/-DI с экстремальными значениями CCI и ADX > `AdxThreshold`, опционально проверка закрытия относительно MA.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Срабатывание стоп-лосса или тейк-профита, необязательное управление риском по MA.
- **Стопы**: Да, тейк-профит и стоп-лосс.
- **Значения по умолчанию**:
  - `EnableLong` = true
  - `EnableShort` = true
  - `TakeProfitPercent` = 2m
  - `StopLossPercent` = 1m
  - `CciPeriod` = 15
  - `AdxLength` = 10
  - `AdxThreshold` = 20m
  - `UseMaTrend` = true
  - `MaType` = MovingAverageTypeEnum.Simple
  - `MaLength` = 200
  - `UseMaRiskManagement` = false
  - `MaRiskExitCandles` = 2
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ADX, CCI, MA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
