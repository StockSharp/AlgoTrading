# Стратегия оповещения Parabolic SAR
[English](README.md) | [中文](README_cn.md)

Эта стратегия отслеживает индикатор Parabolic SAR (Stop and Reverse) для выявления возможных разворотов тренда. Когда значение SAR переходит из области выше цены в область ниже, стратегия рассматривает это как бычий сигнал и открывает длинную позицию. Если SAR движется из-под цены наверх, открывается короткая позиция.

Параметры ускорения 0.02 и максимального ускорения 0.2 соответствуют классическим настройкам индикатора. Они определяют, насколько быстро точки SAR приближаются к цене: большие значения делают индикатор более чувствительным, но увеличивают число ложных сигналов. Стратегия обрабатывает только завершённые свечи и сохраняет предыдущие значения SAR и цены, чтобы определить пересечения без обращения к истории индикатора.

Система не задаёт конкретных стопов и закрывает позицию при появлении противоположного сигнала. При необходимости можно подключить встроенную защиту капитала.

## Детали

- **Условия входа**: Parabolic SAR пересекает цену закрытия.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Не заданы.
- **Значения по умолчанию**:
  - `InitialAcceleration` = 0.02
  - `MaxAcceleration` = 0.2
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Parabolic SAR
  - Стопы: Опционально
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
