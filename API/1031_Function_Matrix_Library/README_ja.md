# 関数行列ライブラリ
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

基本的な行列代数を使った重回帰を実演します。この戦略は2つの説明変数の係数を計算しますが、トレードは生成しません。

## 詳細

- **エントリー条件**: なし
- **ロング/ショート**: なし
- **エグジット条件**: なし
- **ストップ**: いいえ
- **デフォルト値**:
  - `Lookback` = 20
  - `Length1` = 10
  - `Length2` = 20
  - `Candle Type` = 5分
- **フィルター**:
  - カテゴリ: インジケーターデモ
  - 方向: なし
  - インジケーター: SMA
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
