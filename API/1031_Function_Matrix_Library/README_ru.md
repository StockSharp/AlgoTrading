# Функция Matrix Library
[English](README.md) | [中文](README_cn.md)

Демонстрирует множественную линейную регрессию на основе простой матричной алгебры. Стратегия вычисляет коэффициенты для двух объясняющих переменных и не генерирует сделок.

## Детали

- **Условия входа**: Нет
- **Длинные/Короткие**: Нет
- **Условия выхода**: Нет
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Lookback` = 20
  - `Length1` = 10
  - `Length2` = 20
  - `Candle Type` = 5-минутные
- **Фильтры**:
  - Категория: Демонстрация индикатора
  - Направление: Нет
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
