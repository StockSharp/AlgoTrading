# 函数 Matrix Library
[English](README.md) | [Русский](README_ru.md)

演示使用基础矩阵代数的多元线性回归。策略计算两个自变量的系数，不生成任何交易。

## 细节

- **入场条件**：无
- **多/空**：无
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `Lookback` = 20
  - `Length1` = 10
  - `Length2` = 20
  - `Candle Type` = 5分钟
- **筛选器**：
  - 类别：指标演示
  - 方向：无
  - 指标：SMA
  - 止损：无
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险级别：低
