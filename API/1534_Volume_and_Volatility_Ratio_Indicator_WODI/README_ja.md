# 出来高・ボラティリティ比率インジケーター WODI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradingViewスクリプト **"Volume and Volatility Ratio Indicator - WODI"** を元にした簡略化された戦略です。出来高と価格ボラティリティの積を監視し、潜在的なリバーサルを特定します。ボラティリティ指数が動的な閾値を超え、直近のローソク足が方向転換を示した場合、フィボナッチベースのリスク管理でポジションを建てます。

## 詳細

- **エントリー**: 高出来高と高ボラティリティ、およびローソク足のリバーサルパターン。
- **エグジット**: ローソク足のレンジとフィボナッチ乗数から計算されたストップロスとテイクプロフィット。
- **ロング/ショート**: 両方。
- **時間軸**: 任意。
- **インジケーター**: SMA。

これは教育目的の簡略版です。元のTradingViewロジックは削減されています。
