# Bill Williams Wise Man 2戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はBill Williamsのトレードシステムにおける第2の「賢者」パターンを実装します。
Awesome Oscillator（AO）のヒストグラムを分析してモメンタムの転換を検出します：

- **買い**：AOがゼロより上にあり、天井を形成した後に3本連続して低くなるバーが続く場合。
- **売り**：AOがゼロより下にあり、底を形成した後に3本連続して高くなるバーが続く場合。

シグナルが現れるたびに、戦略は反対ポジションをクローズし、シグナルの方向に新しいポジションを
オープンします。デフォルトでは4時間足が使用されますが、パラメーターで時間軸を変更できます。

ストップロスやテイクプロフィットのロジックは含まれていません。ポジションは逆のパターンが出現した
場合にのみ反転されます。戦略はまた、視覚的な分析のためにチャート上にローソク足、AOインジケーター、
および執行済みトレードをプロットします。
