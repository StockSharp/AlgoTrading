# Ultimate Scalping Strategie v2
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Scalping-System, das schnelle und langsame EMAs mit VWAP kombiniert. Optionale Engulfing-Kerzen- und Volumens-Spike-Filter verfeinern die Einstiege. Positionen verwenden ATR-basierte Stops und können bei entgegengesetzten Signalen geschlossen werden.

## Details

- **Long**: schnelle EMA kreuzt langsame EMA nach oben und Kurs über VWAP.
- **Short**: schnelle EMA kreuzt langsame EMA nach unten und Kurs unter VWAP.
- **Indikatoren**: EMA, VWAP, ATR, SMA.
- **Stops**: ATR-Vielfache.
