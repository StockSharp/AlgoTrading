# ExpBuySellSide 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は MetaTrader Expert Advisor の **ExpBuySellSide** を StockSharp API に変換したものです。ATR ベースのストップシステムとシンプルな Step Up/Down トレンドフィルターを組み合わせています。

ATR モジュールは各ローソク足の周囲に動的なストップレベルを計算します。価格が上部バンドを上抜けるとマーケットは強気フェーズとみなされ、下部バンドを下抜けると弱気フェーズを示します。

Step Up/Down モジュールは非常に速い SMA と遅い SMA を比較し、それらの差が拡大しているかどうかを確認します。クロスオーバーの方向への差の拡大がトレンドを確認します。

**両方**のモジュールが同じ方向を指している場合にのみトレードが開かれます。反対のシグナルが現れたとき、既存のポジションをオプションで決済することができます。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格が ATR 上部バンドを上回って終値を付け**かつ**高速 SMA が低速 SMA から上方に離れる。
  - **ショート**: 価格が ATR 下部バンドを下回って終値を付け**かつ**高速 SMA が低速 SMA から下方に離れる。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 反対のシグナルが現れ、*Close Opposite* オプションが有効。
  - ポジション保護による手動ストップ。
- **ストップ**: `ATR * Multiplier` バンドに基づく。
- **デフォルト値**:
  - `ATR Period` = 5.
  - `ATR Multiplier` = 2.5.
  - `Fast SMA` = 2.
  - `Slow SMA` = 30.
  - `Candle Type` = 1 時間足。
  - `Close Opposite` = true.
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: あり
  - 複雑さ: 中程度
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
