# グリッドトレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は基本的なグリッドトレーディングシステムを実装します。`GridStep` で定義された固定の価格間隔でバイストップとセルストップ注文を配置します。各執行済み注文は固定のテイクプロフィット距離を使用します。グローバルな利益目標がすべてのポジションをクローズしてグリッドをリセットします。オプションで、新しい注文のボリュームがマーチンゲール方式で増加します。

## 詳細

- **エントリー条件:**
  - 最終価格の1ステップ上にバイストップ。
  - 最終価格の1ステップ下にセルストップ。
- **ロング/ショート:** 両方。
- **エグジット条件:**
  - 各ポジションは固定のテイクプロフィットでクローズ。
  - 総利益が `ProfitTarget` を超えると、すべての注文とポジションがクローズされます。
- **ストップ:** テイクプロフィットのみ。
- **フィルター:** なし。
