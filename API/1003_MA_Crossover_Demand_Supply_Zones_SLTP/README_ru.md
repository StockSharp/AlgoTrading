# Стратегия MA Crossover Demand Supply Zones SLTP
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает пересечение короткой и длинной простых скользящих средних с определением зон спроса и предложения. Сигналы появляются, когда пересечение происходит рядом с подтверждённой зоной, после чего позиция управляется по процентным стоп-лоссу и тейк-профиту.

## Детали

- **Условия входа**:
  - Лонг: короткая SMA пересекает длинную сверху возле зоны спроса.
  - Шорт: короткая SMA пересекает длинную снизу возле зоны предложения.
- **Направление**: Обе стороны.
- **Условия выхода**:
  - Срабатывание стоп-лосса или тейк-профита.
- **Стопы**: Процентные стоп-лосс и тейк-профит.
- **Значения по умолчанию**:
  - `ShortMaLength` = 9
  - `LongMaLength` = 21
  - `ZoneLookback` = 50
  - `ZoneStrength` = 2
  - `StopLossPercent` = 1
  - `TakeProfitPercent` = 2
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: SMA, Highest, Lowest
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
