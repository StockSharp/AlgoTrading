# 需要・供給ゾーン付きMAクロスオーバーSLTP戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は短期・長期の単純移動平均クロスオーバーと需要・供給ゾーンの検出を組み合わせます。システムは最近確認された需要または供給ゾーン近くで発生するクロスオーバーを探し、クロスオーバーの方向にエントリーして、固定パーセントのストップロスと利確でポジションを管理します。

## 詳細

- **エントリー条件**:
  - ロング: 需要ゾーン付近で短期SMAが長期SMAを上抜け。
  - ショート: 供給ゾーン付近で短期SMAが長期SMAを下抜け。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 価格が利確またはストップロスレベルに到達。
- **ストップ**: パーセンテージベースのストップロスと利確。
- **デフォルト値**:
  - `ShortMaLength` = 9
  - `LongMaLength` = 21
  - `ZoneLookback` = 50
  - `ZoneStrength` = 2
  - `StopLossPercent` = 1
  - `TakeProfitPercent` = 2
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: SMA, Highest, Lowest
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
