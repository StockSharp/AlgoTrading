# 金曜日債券ショート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この時間ベースの戦略は、東部時間の指定した曜日と時刻にショートポジションを建て、別の曜日と時刻に決済します。デフォルトでは金曜日13:00 ETに売り、月曜日13:00 ETに買い戻します。

## 詳細

- **エントリー条件:** 設定した曜日と時刻（ET）にショート。
- **エグジット条件:** 設定した曜日と時刻（ET）にショートを決済。
- **ロング/ショート:** ショートのみ。
- **インジケーター:** なし。
- **ストップ:** なし。
- **時間軸:** デフォルトは1時間ローソク足。
- **カテゴリ:** 時間ベース。
- **複雑さ:** 低。
