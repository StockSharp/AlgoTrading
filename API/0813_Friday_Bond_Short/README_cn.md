# Friday Bond Short Strategy
[English](README.md) | [Русский](README_ru.md)

该时间策略在美国东部时间的指定星期与小时开空仓，并在另一天同一时间平仓。默认在星期五13:00 ET 做空，并在星期一13:00 ET 回补。

## 详情

- **入场条件:** 在设定的星期和小时 (ET) 建立空头。
- **出场条件:** 在设定的星期和小时 (ET) 平仓。
- **多空方向:** 仅做空。
- **指标:** 无。
- **止损:** 无。
- **时间框架:** 默认 1 小时蜡烛。
- **类别:** 时间型。
- **复杂度:** 低。
