# ポジション決済戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

利益、損失、または時間のルールに基づいてオープンポジションを閉じます。この戦略では新しい注文は開かれません。

## 詳細

- **エントリー条件**: なし。ポジションは外部で開かれているものとして扱います。
- **エグジット条件**:
  - pips単位で測定された利益または損失の上限に達した場合。
  - ポジションの保有時間が時間制限（分）を超えた場合。
  - 現在時刻が設定されたクローズ時刻を過ぎている場合。
- **ストップ**: 暗黙の利益と損失のしきい値。
- **フィルター**: 時刻と保有時間。
