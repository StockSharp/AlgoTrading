# EMA 5 Alert Candle Short
[English](README.md) | [中文](README_zh.md)

Стратегия **EMA 5 Alert Candle Short** ожидает три свечи, касающиеся EMA(5), после чего формируется свеча, не касающаяся индикатора. Когда следующая свеча пробивает минимум такой "свечи-алерта", открывается короткая позиция с тейк-профитом на расстоянии, равном стоп-лоссу.

## Подробности
- **Условия входа**: после трёх свечей с касанием EMA шорт при пробое минимума свечи без касания.
- **Длинные/короткие позиции**: только короткие.
- **Условия выхода**: стоп на максимуме свечи-алерта, тейк на равном расстоянии.
- **Стопы**: да, основаны на диапазоне свечи-алерта.
- **Значения по умолчанию**:
  - `EmaPeriod = 5`
  - `RiskPerTrade = 2m`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **Фильтры**:
  - Категория: Пробой
  - Направление: Шорт
  - Индикаторы: EMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
