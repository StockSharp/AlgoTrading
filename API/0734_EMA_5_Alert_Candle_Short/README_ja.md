# EMA 5 アラートキャンドル・ショート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**EMA 5 Alert Candle Short** 戦略は、5期間EMAに触れる3本のローソク足を待ち、その後EMAの上に留まるローソク足を特定します。次のローソク足がアラートキャンドルの安値を下回ったときにショートポジションを開き、テイクプロフィットはストップロスと等距離に設定されます。

## 詳細
- **エントリー条件**: EMAに触れる3本のローソク足の後、EMAに触れないローソク足の安値を下回ったときにショート。
- **ロング/ショート**: ショートのみ。
- **エグジット条件**: アラートキャンドルの高値でのストップロス、同距離でのテイクプロフィット。
- **ストップ**: はい、アラートキャンドルの範囲に基づく。
- **デフォルト値**:
  - `EmaPeriod = 5`
  - `RiskPerTrade = 2m`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ショート
  - インジケーター: EMA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
