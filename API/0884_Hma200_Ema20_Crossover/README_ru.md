# Стратегия HMA 200 + EMA 20 Crossover
[English](README.md) | [中文](README_cn.md)

Стратегия открывает лонг, когда цена находится выше 200-периодной Hull Moving
Average и пересекает сверху 20-периодную Exponential Moving Average. Шорт
открывается, когда цена ниже HMA и пересекает снизу EMA. Позиции переворачиваются
при противоположном сигнале.

## Детали

- **Условия входа**:
  - **Лонг**: `Close > HMA` и `Close` пересекает `EMA` снизу вверх.
  - **Шорт**: `Close < HMA` и `Close` пересекает `EMA` сверху вниз.
- **Лонг/Шорт**: оба направления.
- **Условия выхода**: разворот при обратном пересечении.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `HMA Length` = 200
  - `EMA Length` = 20
- **Фильтры**:
  - Категория: следование за трендом
  - Направление: оба
  - Индикаторы: HMA, EMA
  - Стопы: нет
  - Сложность: простая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
