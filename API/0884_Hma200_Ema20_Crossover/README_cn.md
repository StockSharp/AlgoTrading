# HMA 200 + EMA 20 交叉策略
[English](README.md) | [Русский](README_ru.md)

当价格高于200周期Hull移动平均线并上穿20周期指数移动平均线时进场做多；
当价格低于HMA并下穿EMA时进场做空。出现相反信号时仓位反转。

## 细节

- **入场条件**：
  - **多头**：`Close > HMA` 且 `Close` 向上穿越 `EMA`。
  - **空头**：`Close < HMA` 且 `Close` 向下穿越 `EMA`。
- **多/空**：双向。
- **出场条件**：出现相反交叉信号时反转。
- **止损**：无。
- **默认参数**：
  - `HMA Length` = 200
  - `EMA Length` = 20
- **筛选**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：HMA, EMA
  - 止损：无
  - 复杂度：简单
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
