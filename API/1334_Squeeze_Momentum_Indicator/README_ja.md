# スクイーズ・モメンタム・インジケーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

スクイーズ・モメンタム・インジケーター戦略は、ボリンジャーバンドがケルトナーチャンネル内に入ったときのボラティリティ収縮を検出します。スクイーズが上方向に解放され、モメンタムが上昇し、価格が100期EMAを上回ったときにロングポジションを開きます。下方向への解放時、モメンタムが下落し、価格がEMAを下回ったときにショートを取ります。モメンタムが反転するとポジションを終了します。

## 詳細

- **エントリー条件**:
  - ボリンジャーバンドがケルトナーチャンネル外に移動（スクイーズ解放）。
  - **ロング**: モメンタムが増加し、価格が前回終値とEMA100を上回り、スクイーズの色が黒からグレーに変わる。
  - **ショート**: モメンタムが減少し、価格が前回終値とEMA100を下回り、色がグレーから黒に変わる。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - モメンタムが反転する。
- **ストップ**: なし。
- **デフォルト値**:
  - `BbLength` = 20
  - `BbMultiplier` = 2
  - `KcLength` = 20
  - `KcMultiplier` = 1.5
  - `EmaLength` = 100
- **フィルター**:
  - カテゴリ: ボラティリティ・ブレイクアウト
  - 方向: 両方
  - インジケーター: Bollinger Bands, Keltner Channels, Linear Regression, EMA
  - ストップ: なし
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
