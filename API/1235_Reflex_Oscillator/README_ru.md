# Стратегия Reflex Oscillator
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует осциллятор Reflex Джона Элерса. Открывает длинную позицию при пересечении осциллятором верхнего порога и короткую позицию при пересечении нижнего порога. Позиции закрываются, когда осциллятор возвращается к нулевой линии.

## Детали

- **Критерии входа**:
  - **Лонг**: осциллятор пересекает сверху уровень `UpperLevel`.
  - **Шорт**: осциллятор пересекает снизу уровень `LowerLevel`.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - Длинная позиция: осциллятор пересекает нулевой уровень вниз.
  - Короткая позиция: осциллятор пересекает нулевой уровень вверх.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `ReflexPeriod` = 20.
  - `SuperSmootherPeriod` = 8.
  - `PostSmoothPeriod` = 33.
  - `UpperLevel` = 1.
  - `LowerLevel` = -1.
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
