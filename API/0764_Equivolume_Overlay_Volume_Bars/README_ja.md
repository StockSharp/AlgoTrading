# 等量オーバーレイ出来高バー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格ローソク足上に等量バーをエミュレートするために出来高加重ボックスを描画します。出来高の累積合計を計算し、最近のアクティビティに対して各ボックスの幅を拡縮します。出来高の移動平均を設定することができます。

この戦略は取引を行いません。StockSharp高レベルAPIを使用したカスタムチャート描画の視覚的なサンプルとして提供されています。
