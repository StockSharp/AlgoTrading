# Estratégia de Barras de Volume Sobrepostas Equivolume
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia renderiza caixas ponderadas por volume para emular barras equivolume sobre velas de preço. Calcula uma soma acumulativa de volume e dimensiona a largura de cada caixa em relação à atividade recente. Uma média móvel de volume pode ser configurada.

A estratégia não realiza operações; destina-se como exemplo visual do uso da API de alto nível do StockSharp para desenho personalizado em gráficos.
