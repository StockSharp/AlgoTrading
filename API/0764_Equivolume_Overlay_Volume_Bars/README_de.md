# Equivolumen-Überlagerungs-Volumen-Balken-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie rendert volumengewichtete Boxen, um Equivolumen-Balken über Preiskerzen nachzuahmen. Sie berechnet eine laufende Volumensumme und skaliert die Breite jeder Box relativ zur aktuellen Aktivität. Ein gleitender Durchschnitt des Volumens kann konfiguriert werden.

Die Strategie platziert keine Trades; sie dient als visuelles Beispiel für die Verwendung der StockSharp High-Level-API für benutzerdefiniertes Chart-Drawing.
