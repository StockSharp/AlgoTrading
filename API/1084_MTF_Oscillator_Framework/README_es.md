# Estrategia MTF Oscillator Framework
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza el indicador RSI en dos marcos temporales. Se produce una compra cuando ambos valores de RSI están por debajo del nivel de sobreventa, y una venta cuando ambos están por encima del nivel de sobrecompra.

## Parámetros
- Candle Type
- Higher Candle Type
- RSI Length
- Overbought
- Oversold
