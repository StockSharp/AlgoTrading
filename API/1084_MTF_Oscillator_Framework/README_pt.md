# Estratégia MTF Oscillator Framework
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia utiliza o indicador RSI em dois períodos de tempo. Uma compra ocorre quando ambos os valores de RSI estão abaixo do nível de sobrevenda, e uma venda ocorre quando ambos estão acima do nível de sobrecompra.

## Parâmetros
- Candle Type
- Higher Candle Type
- RSI Length
- Overbought
- Oversold
