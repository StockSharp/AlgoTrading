# MTF Oscillator Framework Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet den RSI-Indikator auf zwei Zeitrahmen. Ein Kauf erfolgt, wenn beide RSI-Werte unter dem Überverkauft-Level liegen, und ein Verkauf, wenn beide über dem Überkauft-Level liegen.

## Parameter
- Candle Type
- Higher Candle Type
- RSI Length
- Overbought
- Oversold
