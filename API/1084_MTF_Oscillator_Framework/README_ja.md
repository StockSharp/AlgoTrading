# MTF Oscillator Framework 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は2つの時間軸でRSIインジケーターを使用します。両方のRSI値が売られすぎレベルを下回ったときに買い、両方が買われすぎレベルを上回ったときに売りを行います。

## パラメーター
- Candle Type
- Higher Candle Type
- RSI Length
- Overbought
- Oversold
