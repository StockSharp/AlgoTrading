# Стратегия RSI Threshold
[English](README.md) | [中文](README_cn.md)

Конвертация советника *Exp_RSI* из MetaTrader в StockSharp. Стратегия открывает и закрывает позиции при пересечении RSI заданных уровней перекупленности и перепроданности.

## Детали

- **Условия входа**:
  - **Лонг**: RSI пересекает снизу уровень `RSI Low Level`.
  - **Шорт**: RSI пересекает сверху уровень `RSI High Level`.
- **Лонг/Шорт**: Обе стороны.
- **Условия выхода**:
  - Обратный сигнал или параметры стопов.
- **Стопы**: Take Profit и Stop Loss в абсолютных ценовых единицах.
- **Значения по умолчанию**:
  - `RSI Period` = 14
  - `RSI High Level` = 60
  - `RSI Low Level` = 40
  - `Stop Loss` = 1000
  - `Take Profit` = 2000
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Да
  - Сложность: Начальный
  - Таймфрейм: H4
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
