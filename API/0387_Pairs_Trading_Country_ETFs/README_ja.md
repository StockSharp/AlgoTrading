# 国別ETFペアトレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この平均回帰戦略は、価格比率のzスコアに基づいて2つの国別ETFのペアを取引します。比率が閾値を超えて乖離した場合、スプレッドが平均に回帰することを期待してロング/ショートポジションを取ります。

価格比率はローリングウィンドウで追跡され、zスコアがエグジットレベルを超えるとポジションはクローズされます。

## 詳細

- **ユニバース**: 正確に2つの国別ETF。
- **シグナル**: ローリング価格比率のzスコアが `EntryZ` を超える。
- **エグジット**: zスコアが `ExitZ` に戻ったらクローズ。
- **データ**: 日足ローソク足、デフォルトは60日ウィンドウ。
- **リスク管理**: 取引価値が `MinTradeUsd` を下回る場合は注文をスキップ。
