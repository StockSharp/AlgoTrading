# Стратегия Donchian RSI
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает каналы Дончиана и индикатор RSI. Покупка происходит на пробоях Дончиана, если RSI подтверждает, что тренд не перекуплен. Продажа — на пробое вниз, когда RSI показывает, что тренд не перепродан.

Каналы Дончиана определяют уровни пробоя, а RSI проверяет, поддерживает ли импульс движение. Позиции открываются, когда пробой совпадает с направлением RSI.

Стратегия лучше всего подходит трейдерам, ожидающим устойчивого пробоя, а не ложного. Риск ограничивается стопом по ATR.

## Подробности

- **Условия входа**:
  - Лонг: `Close > DonchianHigh && RSI < RsiOversoldLevel`
  - Шорт: `Close < DonchianLow && RSI > RsiOverboughtLevel`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Неудачный пробой или противоположный сигнал
- **Стопы**: процентные через `StopLossPercent`
- **Значения по умолчанию**:
  - `DonchianPeriod` = 20
  - `RsiPeriod` = 14
  - `RsiOverboughtLevel` = 70m
  - `RsiOversoldLevel` = 30m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Канал Дончиана, RSI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
