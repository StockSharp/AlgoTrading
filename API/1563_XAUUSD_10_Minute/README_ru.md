# Стратегия XAUUSD 10 минут
[English](README.md) | [中文](README_cn.md)

Стратегия торгует XAUUSD на 10‑минутных свечах, используя сигналы MACD, RSI и полос Боллинджера. Открывает длинные позиции при появлении бычьих условий и короткие при медвежьих сигналах. Применяет стоп-лосс и тейк-профит на основе ATR с учётом фиксированного спреда.

## Детали

- **Условия входа**:
  - **Покупка**: пересечение MACD сверху, RSI ниже зоны перепроданности или цена ниже нижней полосы Боллинджера.
  - **Продажа**: пересечение MACD снизу, RSI выше зоны перекупленности или цена выше верхней полосы Боллинджера.
- **Лонг/Шорт**: оба направления.
- **Условия выхода**:
  - Закрытие позиции по противоположному сигналу, стоп-лоссу или тейк-профиту.
- **Стопы**: стоп-лосс `3 * ATR`, тейк-профит `5 * ATR`.
- **Значения по умолчанию**:
  - MACD fast/slow/signal: `12/26/9`.
  - RSI период: `14`, перекупленность `65`, перепроданность `35`.
  - Полосы Боллинджера: период `20`, ширина `2`.
  - ATR период `14`.
  - Спред `38` тиков.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Несколько
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
