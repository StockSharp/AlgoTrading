# RSI Cyclic Smoothed
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор циклически сглаженного RSI. Вычисляет динамические процентильные границы и торгует развороты при их пересечении.

## Подробности

- **Условия входа**: Пересечение CRSI выше нижней границы или ниже верхней.
- **Длинная/короткая**: Оба направления.
- **Условия выхода**: Пересечение противоположной границы.
- **Стопы**: Да.
- **Параметры по умолчанию**:
  - `DominantCycleLength` = 20
  - `Vibration` = 10
  - `Leveling` = 10
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Да
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
