# RSI サイクリック・スムーズド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

サイクリック・スムーズドRSIインジケーターに基づく戦略。動的なパーセンタイルバンドを計算し、オシレーターがバンドを越えた際にリバーサルを取引します。

## 詳細

- **エントリー条件**: CRSIが下限バンドを上抜け、または上限バンドを下抜けする。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対側のバンドのクロス。
- **ストップ**: はい。
- **デフォルト値**:
  - `DominantCycleLength` = 20
  - `Vibration` = 10
  - `Leveling` = 10
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: はい
  - 複雑さ: 上級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
