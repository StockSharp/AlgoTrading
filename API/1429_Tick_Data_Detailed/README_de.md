# Detaillierte-Tick-Daten-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Sammelt und aggregiert Tick-Volumina in mehrere vordefinierte Bereiche für Kauf- und Verkaufsrichtungen. Nützlich für detailliertes Tape Reading ohne Generierung von Handelssignalen.

## Details

- **Einstiegskriterien**: Keine
- **Long/Short**: Keine
- **Ausstiegskriterien**: Keine
- **Stops**: Nein
- **Standardwerte**:
  - `VolumeLessThan` = 10000
  - `Volume2From` = 10000
  - `Volume2To` = 20000
  - `Volume3From` = 20000
  - `Volume3To` = 50000
  - `Volume4From` = 50000
  - `Volume4To` = 100000
  - `Volume5From` = 100000
  - `Volume5To` = 200000
  - `Volume6From` = 200000
  - `Volume6To` = 400000
  - `VolumeGreaterThan` = 400000
- **Filter**:
  - Kategorie: Volumen
  - Richtung: Keine
  - Indikatoren: Keine
  - Stops: Nein
  - Komplexität: Grundlegend
  - Zeitrahmen: Tick
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Niedrig
