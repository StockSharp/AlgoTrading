# Estrategia de Datos de Tick Detallados
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Recopila y agrega volúmenes de tick en múltiples rangos predefinidos para las direcciones de compra y venta. Útil para una lectura detallada de la cinta sin generar señales de trading.

## Detalles

- **Criterios de entrada**: Ninguno
- **Largo/Corto**: Ninguno
- **Criterios de salida**: Ninguno
- **Stops**: No
- **Valores predeterminados**:
  - `VolumeLessThan` = 10000
  - `Volume2From` = 10000
  - `Volume2To` = 20000
  - `Volume3From` = 20000
  - `Volume3To` = 50000
  - `Volume4From` = 50000
  - `Volume4To` = 100000
  - `Volume5From` = 100000
  - `Volume5To` = 200000
  - `Volume6From` = 200000
  - `Volume6To` = 400000
  - `VolumeGreaterThan` = 400000
- **Filtros**:
  - Categoría: Volumen
  - Dirección: Ninguno
  - Indicadores: Ninguno
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Tick
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Bajo
