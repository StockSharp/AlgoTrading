# Estratégia SuperTrade com Nuvem Ichimoku
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera com a Nuvem Ichimoku. Compra quando a nuvem é de alta e o preço está acima de ambas as linhas Senkou, e fecha o comprado quando a nuvem se torna de baixa e o preço cai abaixo da nuvem.
