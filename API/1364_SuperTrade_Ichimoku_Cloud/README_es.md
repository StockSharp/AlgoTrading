# Estrategia SuperTrade con Nube Ichimoku
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia opera con la Nube Ichimoku. Compra cuando la nube es alcista y el precio está por encima de ambas líneas Senkou, y cierra el largo cuando la nube se vuelve bajista y el precio cae por debajo de la nube.
