# Стратегия Adaptive CG Oscillator X2
[English](README.md) | [中文](README_cn.md)

Использует адаптивный CG осциллятор на двух таймфреймах.
Старший таймфрейм определяет направление тренда, младший генерирует точки входа и выхода по пересечению линий осциллятора.

## Детали

- **Условия входа**:
  - Long: основная линия пересекает сигнальную снизу при восходящем тренде
  - Short: основная линия пересекает сигнальную сверху при нисходящем тренде
- **Long/Short**: Оба направления
- **Выход**: Обратный сигнал или флаги принудительного закрытия
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `TrendAlpha` = 0.07m
  - `SignalAlpha` = 0.07m
  - `TrendCandleType` = TimeSpan.FromHours(6).TimeFrame()
  - `SignalCandleType` = TimeSpan.FromMinutes(30).TimeFrame()
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: Adaptive CG Oscillator
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Несколько таймфреймов
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
