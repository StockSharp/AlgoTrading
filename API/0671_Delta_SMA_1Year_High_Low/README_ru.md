# Delta SMA 1-Year High Low
[English](README.md) | [中文](README_cn.md)

Стратегия **Delta SMA 1-Year High Low** вычисляет дельту объёма (покупки минус продажи) и её простое скользящее среднее. Длинная позиция открывается, когда SMA дельты была очень низкой и пересекает ноль снизу вверх. Позиция закрывается, когда SMA дельты падает ниже 60% своего годового максимума после предыдущего пересечения 70% этого максимума.

## Детали
- **Условия входа**: SMA дельты была ниже 70% годового минимума и пересекает ноль вверх.
- **Длинные/короткие**: только long.
- **Условия выхода**: SMA дельты падает ниже 60% годового максимума после пересечения 70%.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `DeltaSmaLength = 14`
  - `CandleType = TimeSpan.FromDays(1).TimeFrame()`
- **Фильтры**:
  - Категория: Volume
  - Направление: Лонг
  - Индикаторы: SMA, Highest, Lowest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
