# Стратегия SOXL Trend Surge Profit-Only Runner
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинные позиции, когда цена находится выше EMA 200 и индикатор SuperTrend показывает восходящий тренд. Требуется растущий ATR, объём выше среднего, фильтр по времени и условие выхода цены за небольшой буфер вокруг EMA. Половина позиции закрывается по ATR-целевой, остаток сопровождается трейлинг-стопом.

## Детали

- **Критерий входа**: цена выше EMA, SuperTrend вверх, объём выше среднего, ATR растёт, цена вне буфера EMA, время между 14–19 часами, пауза после выхода
- **Длин/Шорт**: только лонг
- **Критерий выхода**: частичная фиксация 50% по ATR-цели и трейлинг-стоп остального
- **Стопы**: трейлинг-стоп
- **Значения по умолчанию**:
  - `EmaLength` = 200
  - `AtrLength` = 14
  - `AtrMultTarget` = 2.0
  - `CooldownBars` = 15
  - `SupertrendFactor` = 3.0
  - `SupertrendAtrPeriod` = 10
  - `MinBarsHeld` = 2
  - `VolFilterLen` = 20
  - `EmaBuffer` = 0.005
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: EMA, ATR, SuperTrend, объём
  - Стопы: трейлинг
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
