# Стратегия Bayesian BBSMA Oscillator
[English](README.md) | [中文](README_cn.md)

Стратегия оценивает вероятность пробоя следующей свечи вверх или вниз при помощи байесовской модели, построенной на базе полос Боллинджера и простой скользящей средней. Дополнительное подтверждение индикаторами Билла Уильямса (Accelerator и Alligator) может фильтровать сигналы. Когда вероятность пробоя вверх превышает порог, открывается длинная позиция. Высокая вероятность пробоя вниз приводит к открытию короткой позиции.

## Подробности

- **Условия входа**:
  - Лонг при превышении `LowerThreshold` вероятностью роста или сигнала `probPrime` и, при включенном подтверждении, бычьими индикаторами Билла Уильямса.
  - Шорт при превышении порога вероятностью падения или сигнала `probPrime` и, при включенном подтверждении, медвежьими индикаторами Билла Уильямса.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Обратный сигнал.
- **Стопы**: Отсутствуют.
- **Значения по умолчанию**:
  - `BbSmaPeriod` = 20
  - `BbStdDevMult` = 2.5
  - `AoFast` = 5
  - `AoSlow` = 34
  - `AcFast` = 5
  - `SmaPeriod` = 20
  - `BayesPeriod` = 20
  - `LowerThreshold` = 15
  - `UseBwConfirmation` = false
  - `JawLength` = 13
- **Фильтры**:
  - Категория: Байесовский тренд
  - Направление: Оба
  - Индикаторы: Bollinger Bands, SMA, Awesome Oscillator, Accelerator Oscillator, Alligator
  - Стопы: Нет
  - Сложность: Высокая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
