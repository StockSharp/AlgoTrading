# サンタクロース・ラリー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
サンタクロース・ラリーとは、12月最終週から1月最初の2取引日にかけて株式が上昇する傾向のことです。
年末年始の楽観的な雰囲気やポジション調整がこの短期的な強さを生み出す可能性があります。

テストでは年平均リターン約 100% を示しています。外国為替市場で最も良いパフォーマンスを発揮します。

戦略は期間の開始時に買いを入れ、新年の2取引日目の後に決済して、季節的な上昇を捉えます。

ストップは小さく保ち、その時期に市場が上昇しない場合の大きな損失を避けます。

## 詳細

- **エントリー条件**: カレンダー効果トリガー
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: 季節性
  - 方向: 両方
  - インジケーター: 季節性
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
