# PChanneルシステム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**PChannelシステム**は、遅延確認付きの価格チャネルブレイクアウトを使用します。設定可能な期間にわたって最高値と最安値を追跡します。価格がチャネルを突き抜けて再びチャネル内に戻ったとき、戦略はブレイクアウトの方向にエントリーしながら逆方向のポジションを閉じます。オプションのストップロスとテイクプロフィットレベルがリスクを管理します。

## パラメーター
- `Period` – チャネルのルックバック期間。
- `Shift` – チャネル値を遅延させるバー数。
- `StopLoss` – 保護ストップのための絶対価格距離。
- `TakeProfit` – 利益目標のための絶対価格距離。
- `CandleType` – 計算に使用するローソク足シリーズ。

## トレードロジック
1. 最後の`Period`本のローソク足からオプションの`Shift`を使ってチャネル境界を計算する。
2. 前のローソク足がチャネル外で閉じて現在のローソク足がチャネル内に戻った場合、ブレイクアウト方向にポジションを開く。
3. 新しいポジションを開く前に逆方向のポジションがあれば閉じる。
4. アクティブなトレードを監視し、`StopLoss`または`TakeProfit`に達したら決済する。
