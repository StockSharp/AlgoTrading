# Система PChannel
[English](README.md) | [中文](README_cn.md)

**PChannel System** использует прорыв ценового канала с задержанным подтверждением. Индикатор строит верхнюю и нижнюю границы по максимумам и минимумам за заданный период. Когда цена пробивает канал и затем возвращается внутрь, стратегия открывает позицию в сторону пробоя и закрывает противоположную. Для управления риском применяются стоп‑лосс и тейк‑профит.

## Параметры
- `Period` – длина истории для расчёта канала.
- `Shift` – задержка значений канала в барах.
- `StopLoss` – расстояние защитного стопа в ценовых единицах.
- `TakeProfit` – расстояние цели прибыли в ценовых единицах.
- `CandleType` – тип свечей для вычислений.

## Логика торговли
1. Рассчитываются границы канала из последних `Period` свечей с учётом `Shift`.
2. Если предыдущая свеча закрылась вне канала, а текущая вернулась внутрь, открывается позиция в направлении пробоя.
3. Противоположная позиция закрывается перед открытием новой.
4. Активная сделка закрывается при достижении `StopLoss` или `TakeProfit`.

Python‑версия стратегии пока отсутствует.
