# Стратегия MA L World
[English](README.md) | [中文](README_cn.md)

Стратегия пересечения взвешенных скользящих средних с трейлингом на основе EMA.

Открывает длинную позицию, когда быстрая WMA пересекает медленную снизу вверх. Открывает короткую позицию, когда быстрая WMA пересекает медленную сверху вниз. Для выхода используется EMA с периодом 92, а также фиксированные уровни стоп-лосса и тейк-профита.

## Подробности

- **Условия входа**:
  - Длинная: `Быстрая WMA` пересекает `Медленную WMA` снизу вверх
  - Короткая: `Быстрая WMA` пересекает `Медленную WMA` сверху вниз
- **Long/Short**: Оба
- **Условия выхода**: Обратное пересечение или цена пересекает трейлинговую EMA
- **Стопы**: Стоп-лосс и тейк-профит через `StartProtection`
- **Параметры по умолчанию**:
  - `FastMaLength` = 12
  - `SlowMaLength` = 25
  - `TrailingMaPeriod` = 92
  - `StopLoss` = 95m
  - `TakeProfit` = 670m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Trend following
  - Направление: Оба
  - Индикаторы: WMA, EMA
  - Стопы: Стоп-лосс, тейк-профит, трейлинг EMA
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
