# Стратегия Crunchsters Normalised Trend
[English](README.md) | [中文](README_cn.md)

Стратегия нормализует доходности и применяет Hull Moving Average к накопленной нормализованной цене.
Длинная позиция открывается при пересечении `nPrice` выше HMA, короткая — при пересечении ниже.

Тестирование показывает среднегодичную доходность около 105%. Лучше всего работает на крипторынке.

Нормализованные доходности масштабируют цену относительно волатильности. Стоп по ATR управляет риском.

## Подробности

- **Условия входа**:
  - Длинная: `nPrice` пересекает `HMA` снизу вверх
  - Короткая: `nPrice` пересекает `HMA` сверху вниз
- **Long/Short**: Оба
- **Условия выхода**: Обратное пересечение или стоп по ATR
- **Стопы**: На основе ATR через `StopMultiple`
- **Параметры по умолчанию**:
  - `NormPeriod` = 14
  - `HmaPeriod` = 100
  - `HmaOffset` = 0
  - `StopMultiple` = 1m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Trend following
  - Направление: Оба
  - Индикаторы: Hull Moving Average, Standard Deviation, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
