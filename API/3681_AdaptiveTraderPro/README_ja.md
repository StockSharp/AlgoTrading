# AdaptiveTrader Pro 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
AdaptiveTrader Pro は、MetaTrader 5 エキスパート アドバイザー *AdaptiveTrader_Pro_Final_EA.mq5* から変換されたマルチタイムフレームのトレンドフォロー戦略です。 RSI、ATR、移動平均を組み合わせて、資金管理コントロールを適用しながら、支配的なトレンドの方向に取引します。

この戦略は、構成可能な主な時間枠 (デフォルトは 5 分) で機能し、より高い時間枠の移動平均 (デフォルトは 1 時間) を使用してトレンドの方向を確認します。エントリーは、両方の移動平均と一致する売られすぎ/買われすぎの RSI シグナルに依存します。

## 取引ルール
- **ロングエントリー**: RSI が 30 を下回り、ローソク足の終値がメインタイムフレーム SMA と上位タイムフレーム SMA を上回っている場合。
- **ショートエントリー**: RSI が 70 を超え、ローソク足の終値が両方の SMA を下回った場合。
- **単一位置**: 一度に 1 方向の位置のみが保持されます。反対のポジションは反転する前にクローズされます。

## リスクと貿易の管理
- **ポジション サイズ**: ポジション サイズは、ポートフォリオの資産、リスクの割合、および ATR ベースのストップ ディスタンスから計算されます。
- **ストップ処理**: ATR ベースのトレーリング ストップは価格に従い、設定可能な ATR 倍数で取引が有利に進んだ後、損益分岐点まで引き締められます。
- **部分利益**: ポジションの設定可能な部分が最初のターゲット (ATR 倍) でクローズされます。残量はトレーリングストップで管理されます。

## パラメーター
| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| `MaxRiskPercent` | 取引ごとにアカウントに適用されるリスクの割合。 | `0.2` |
| `RsiPeriod` | メインタイムフレームの長さは RSI です。 | `14` |
| `AtrPeriod` | メインタイムフレームの長さは ATR です。 | `14` |
| `AtrMultiplier` | 初期停止距離の乗数は ATR です。 | `1.5` |
| `TrailingStopMultiplier` | ATR 乗数はストップのトレーリング中に使用されました。 | `1.0` |
| `TrailingTakeProfitMultiplier` | 部分的なテイクプロフィットターゲットの乗数は ATR です。 | `2.0` |
| `TrendPeriod` | メインタイムフレームの長さは SMA です。 | `20` |
| `HigherTrendPeriod` | より高いタイムフレームでは SMA の長さです。 | `50` |
| `BreakEvenMultiplier` | ストップを損益分岐点に移動させるトリガーとなる ATR 乗数。 | `1.5` |
| `PartialCloseFraction` | 最初のターゲットで閉じられた初期ポジションの割合。 | `0.5` |
| `MaxSpreadPoints` | 取引を開始する前の価格ステップにおける最大許容スプレッド。 | `20` |
| `CandleType` | 分析に使用される主なローソク足のタイプ (時間枠)。 | `5 minute candles` |
| `HigherCandleType` | 確認のために使用されるより高いタイムフレームのローソク足タイプ。 | `1 hour candles` |

## 注意事項
- この戦略では、ローソク足サブスクリプションとインジケーター バインディングを備えた StockSharp の高レベルの API を使用します。
- スプレッドは最良の買値/売値を通じて監視されます。スプレッドが設定された制限内に収まるまで、取引は一時停止されます。
- Python の実装は命令ごとに意図的に省略されています。
