# Стратегия Custom Buy BID
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор Supertrend для поиска разворота вверх. Когда цена пересекает линию Supertrend снизу вверх, открывается длинная позиция. Управление риском осуществляется через настраиваемые уровни тейк-профита и стоп-лосса.

## Детали

- **Условия входа**: цена пересекает линию Supertrend снизу вверх.
- **Направление**: Long.
- **Условия выхода**: тейк-профит или стоп-лосс.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `SupertrendPeriod` = 10
  - `SupertrendMultiplier` = 3m
  - `TakeProfitPercent` = 5m
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `StartDate` = 2018-09-01
  - `EndDate` = 9999-01-01
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Long
  - Индикаторы: Supertrend
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
