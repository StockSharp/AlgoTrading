# リサンプリング逆エンジニアリングバンド
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

リサンプリング逆エンジニアリングバンドは、設定可能なリサンプリングレートでRSIの価格レベルを逆算します。価格が下限バンドを下抜けると買い、価格が上限バンドを上抜けると売ります。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 終値がRRSIの下限バンドを下抜ける。
  - **ショート**: 終値がRRSIの上限バンドを上抜ける。
- **エグジット条件**: 逆シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `RsiPeriod` = 14
  - `HighThreshold` = 70
  - `LowThreshold` = 30
  - `SampleLength` = 1
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: ロング & ショート
  - インジケーター: RSI
  - 複雑さ: 中程度
  - リスクレベル: 中
