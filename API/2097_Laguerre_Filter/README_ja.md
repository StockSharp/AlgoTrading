# Laguerreフィルター
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はLaguerreフィルターと、最近の中間価格の加重移動平均として構築された短期FIRフィルターとのクロスオーバーを取引します。

- Laguerreフィルターはガンマパラメーターを使用して価格を平滑化し、ノイズを低減します。
- FIRラインは対称的な重みを持つ4期間の加重移動平均です。
- FIRラインがLaguerreラインの上にあり、その下をクロスした場合、戦略はロングポジションを開きます。
- FIRラインが下にあり、Laguerreラインの上をクロスした場合、ショートポジションが開かれます。
- ラインの関係が逆転すると、反対側のポジションがクローズされます。
- エントリー価格のパーセントで設定されたストップロスが各取引を保護します。

この平均回帰アプローチは、価格が平滑化されたLaguerre曲線から乖離した際に押し目を捕捉しようとします。
