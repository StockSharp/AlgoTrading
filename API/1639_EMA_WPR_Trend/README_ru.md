# Стратегия EMA WPR Trend
[English](README.md) | [中文](README_cn.md)

Стратегия комбинирует фильтр тренда по EMA и сигналы индикатора Williams %R. Покупает на перепроданности и продаёт на перекупленности. Порог отката препятствует последовательным входам. Дополнительные выходы закрывают позицию при противоположном экстремуме Williams %R или после нескольких убыточных баров.

## Детали

- **Критерий входа**:
  - Long: Williams %R <= -100 и тренд EMA вверх
  - Short: Williams %R >= 0 и тренд EMA вниз
- **Длин/Шорт**: оба направления
- **Критерий выхода**:
  - Пересечение Williams %R противоположного экстремума при включённом `UseWprExit`
  - Позиция остаётся убыточной `MaxUnprofitBars` баров при включённом `UseUnprofitExit`
- **Стопы**: нет
- **Значения по умолчанию**:
  - `WprPeriod` = 46
  - `WprRetracement` = 30
  - `EmaPeriod` = 144
  - `BarsInTrend` = 1
  - `MaxUnprofitBars` = 5
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: оба
  - Индикаторы: EMA, Williams %R
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
