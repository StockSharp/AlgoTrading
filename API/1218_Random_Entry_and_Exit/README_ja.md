# ランダムエントリー・エグジット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は乱数を使ってポジションのエントリーとエグジットを行います。各ローソク足の完成時に0から1の間の乱数値が生成されます。値がエントリー閾値を下回ると取引が開かれます。別の乱数値がエグジットを制御します。ロングとショートの取引は個別に有効化できます。

## 詳細

- **エントリー条件**: 乱数値 < エントリー閾値。
- **エグジット条件**: 乱数値 < エグジット閾値。
- **ロング/ショート**: 両方、個別に設定可能。
