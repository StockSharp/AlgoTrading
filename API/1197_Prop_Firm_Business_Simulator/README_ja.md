# Prop Firm ビジネスシミュレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Keltnerチャネルのブレイクアウトを使用し、トレードごとのリスクに基づいたポジションサイジングでプロップファームのリスク管理をシミュレートする戦略。

メソッドはチャネル境界にストップ注文を配置します。数量は、バンド間の距離が口座資本の選択したパーセンテージを表すよう計算されます。

## 詳細

- **エントリー条件**: 価格がKeltnerチャネルのバンドを突破する。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対側のバンドのブレイクアウト。
- **ストップ**: あり。
- **デフォルト値**:
  - `MaPeriod` = 20
  - `AtrPeriod` = 10
  - `Multiplier` = 2m
  - `RiskPerTrade` = 1m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Keltner, ATR
  - ストップ: あり
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
