# Prop Firm Business Simulator
[English](README.md) | [中文](README_cn.md)

Стратегия имитирует управление риском проп-фирмы, используя прорыв канала Келтнера и вычисление объёма по проценту риска.

Заявки-стопы выставляются на границах канала. Количество рассчитывается так, чтобы расстояние между полосами соответствовало заданному проценту капитала.

## Детали

- **Критерии входа**: цена пробивает границы канала Келтнера.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: пробой противоположной полосы.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `MaPeriod` = 20
  - `AtrPeriod` = 10
  - `Multiplier` = 2m
  - `RiskPerTrade` = 1m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Keltner, ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
