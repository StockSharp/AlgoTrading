# Ergebnis-Handels-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Führt Trades rund um bestimmte Earnings-Daten aus. Eröffnet eine Long- oder Short-Position an den Eintrittsdaten und schließt sie an den Austrittsdaten.

## Details

- **Daten**: Tägliche Preiskerzen.
- **Einstieg**: Öffnen entsprechend der Richtung an Eintrittsdatum 1 oder 2.
- **Ausstieg**: Position schließen an Austrittsdatum 1 oder 2.
- **Instrumente**: Beliebige Instrumente.
- **Risiko**: Nur feste Menge.
