# ADX 减弱策略
[English](README.md) | [Русский](README_ru.md)

平均方向指数（ADX）衡量趋势强度。当 ADX 开始下降时，通常说明当前走势动能减弱。本策略在价格位于简单移动平均线另一侧时，逆势交易这种减弱的趋势。

每根K线计算 ADX 和 MA。如果 ADX 相比上一值下降且价格在 MA 上方，则入场做多；若 ADX 下跌而价格在 MA 下方，则做空。风险通过固定百分比止损控制。

由于该方法预期的是动能减弱而非完全反转，持仓通常直到 ADX 再次上升或止损触发。

## 细节

- **入场条件**：ADX 低于前一值，且价格相对于 MA 的位置。
- **多/空**：双向。
- **退出条件**：止损。
- **止损**：是，按百分比。
- **默认值**：
  - `AdxPeriod` = 14
  - `MaPeriod` = 20
  - `StopLoss` = 2%
  - `CandleType` = 15 分钟
- **过滤条件**：
  - 类别: 趋势跟随
  - 方向: 双向
  - 指标: ADX, MA
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等

测试表明年均收益约为 136%，该策略在股票市场表现最佳。
