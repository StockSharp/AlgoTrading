# ADX 弱体化ストラテジー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Average Directional Indexはトレンドの強さを測定します。ADXが低下し始めると、現在の動きがモメンタムを失いつつあることを示すことが多いです。このシステムは、価格が単純移動平均の反対側にある場合に、その弱体化するトレンドに対して取引します。

テストでは年平均リターン約136%を示しています。株式市場で最も優れたパフォーマンスを発揮します。

各バーに対して、戦略はADXとMAを計算します。ADXが前の値と比較して低下し、価格がMAを上回っている場合、ロングエントリーが設定されます。価格がMAを下回っている間にADXが落ちる場合はショートに入ります。固定ストップロスがポジションを保護します。

このアプローチは完全な反転ではなく減速を想定しているため、トレードは通常ADXが再び上昇し始めるかストップが当たるまで保持されます。

## 詳細

- **エントリー条件**: ADXが前の値より低く、MAに対する価格の位置。
- **ロング/ショート**: 両方。
- **エグジット条件**: ストップロス。
- **ストップ**: はい、パーセンテージベース。
- **デフォルト値**:
  - `AdxPeriod` = 14
  - `MaPeriod` = 20
  - `StopLoss` = 2%
  - `CandleType` = 15 minute
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: ADX, MA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
