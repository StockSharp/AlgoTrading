# Стратегия Personal Assistant
[English](README.md) | [中文](README_cn.md)

Стратегия Personal Assistant — это помощник для ручной торговли на базе StockSharp. Она повторяет оригинальный скрипт MetaTrader, показывая ключевые параметры счёта и предоставляя простые методы для открытия или закрытия позиций. В стратегии отсутствуют автоматические условия входа, основное внимание уделено быстрым ручным действиям и оперативной информации для трейдера.

Стратегия подписывается на выбранные свечи только для периодического вывода информации. После завершения каждой свечи в журнал записываются текущая позиция, прибыль, объём, стоимость тика и спред. Это позволяет контролировать торговую среду без дополнительного кода.

Ручные операции доступны через публичные методы:

- `ManualBuy()` — открыть длинную позицию с текущим объёмом.
- `ManualSell()` — открыть короткую позицию с текущим объёмом.
- `CloseAllPositions()` — закрыть существующую позицию по рынку.
- `IncreaseLot()` — увеличить торговый объём на 0.01.
- `DecreaseLot()` — уменьшить торговый объём на 0.01.

Стратегия полезна как шаблон для изучения StockSharp или для трейдеров, предпочитающих полуавтоматическую торговлю.

## Детали

- **Условия входа**: Только вручную.
- **Длинные/Короткие**: Обе стороны.
- **Условия выхода**: Ручное закрытие через `CloseAllPositions`.
- **Стопы**: По умолчанию отсутствуют.
- **Значения по умолчанию**:
  - `Id` = 3900
  - `LotVolume` = 0.01
  - `Slippage` = 2
  - `CandleType` = таймфрейм 1 минута
- **Фильтры**:
  - Категория: Утилиты
  - Направление: Обе
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Простая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Ручной
