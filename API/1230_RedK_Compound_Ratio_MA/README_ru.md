# Стратегия Redk Compound Ratio MA
[English](README.md) | [中文](README_cn.md)

Открывает длинные позиции, когда скользящая средняя CoRa Wave растёт, и короткие, когда она падает.

## Детали

- **Условия входа**:
  - Long: значение CoRa Wave выше предыдущего
  - Short: значение CoRa Wave ниже предыдущего
- **Направление**: Оба
- **Условия выхода**:
  - Противоположный сигнал
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `Length` = 20
  - `RatioMultiplier` = 2m
  - `AutoSmoothing` = true
  - `ManualSmoothing` = 1
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Compound Ratio MA, Weighted Moving Average
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
