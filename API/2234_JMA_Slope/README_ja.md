# JMA 傾き戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はJurik移動平均（JMA）の傾きを監視します。選択したモードに応じて、傾きがゼロをクロスするか方向が変化したときにポジションが建てられます。

## 詳細

- **エントリー条件**:
  - **ロング**: 傾きがゼロを下抜けするか上向きに転換（モードによる）。
  - **ショート**: 傾きがゼロを上抜けするか下向きに転換。
- **ロング/ショート**: ロングとショート。
- **エグジット条件**:
  - 反対シグナルがポジションを反転させる。
- **ストップ**: なし。
- **デフォルト値**:
  - `JMA Length` = 14
  - `JMA Phase` = 0
  - `Mode` = Breakdown
  - `Candle Type` = 時間軸4h
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング & ショート
  - インジケーター: JMA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 4h
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
