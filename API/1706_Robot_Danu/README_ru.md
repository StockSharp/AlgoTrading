# Robot Danu
[English](README.md) | [中文](README_cn.md)

Стратегия сравнивает быстрый и медленный уровни ZigZag,
рассчитанные по максимумам и минимумам свечей.
Короткая позиция открывается, когда быстрый уровень выше медленного.
Длинная позиция открывается, когда быстрый уровень ниже медленного.

## Детали
- **Условия входа**: Сравнение быстрых и медленных пивотов ZigZag.
- **Длинные/короткие**: Обе стороны.
- **Условия выхода**: Противоположное соотношение ZigZag.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `FastLength` = 28
  - `SlowLength` = 56
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе
  - Индикаторы: Highest, Lowest
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
