# T3MA Alarm トレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は T3MA-ALARM インジケーターのアイデアを再現します。二重に平滑化された指数移動平均を適用してトレンドの方向変化を検出します。

平滑化された移動平均が上向きになるとロングポジションを開きます。下向きになるとショートポジションを開きます。オプションとして、反対のシグナルが現在のポジションを決済することができます。ストップロスとテイクプロフィットのレベルはエントリー価格からの絶対価格距離として設定されます。

## パラメーター

| パラメーター | 説明 |
|-----------|-------------|
| `MaPeriod` | 指数移動平均の期間。 |
| `MaShift` | 方向変化の検出に使用するバーの数。 |
| `StopLoss` | 保護ストップロスの価格距離。`0` で無効化。 |
| `TakeProfit` | テイクプロフィットの価格距離。`0` で無効化。 |
| `ReverseOnSignal` | 新しいシグナルが現れたときに反対のポジションを決済する。 |
| `CandleType` | 計算に使用するローソク足タイプ。 |

## シグナル

* **買い** – 平滑化された MA の方向が下向きから上向きに変わる。
* **売り** – 平滑化された MA の方向が上向きから下向きに変わる。

ポジションは反対のシグナル（有効な場合）またはストップロス / テイクプロフィットのレベルに達したときに決済されます。
