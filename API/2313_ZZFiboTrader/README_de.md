# ZZFibo Trader-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Verwendet ZigZag-Swings zur Ableitung von Fibonacci-Retracement-Levels. Die Strategie berechnet die Levels neu, sobald ein neues Pivot-Hoch oder -Tief erkannt wird, und tritt ein, wenn der Preis das 50%-Retracement in Trendrichtung um die angegebene Ausbruchsdistanz überschreitet. Ein optionaler Stop-Loss-Schutz ist über eine absolute Preisdistanz verfügbar.

## Parameter
- Candle Type
- Breakout
- Stop Loss
- ZigZag Depth
