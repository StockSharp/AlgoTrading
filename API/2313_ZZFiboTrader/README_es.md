# Estrategia ZZFibo Trader
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Utiliza los swings del ZigZag para derivar niveles de retroceso de Fibonacci. La estrategia recalcula los niveles cada vez que se detecta un nuevo máximo o mínimo pivote y entra cuando el precio rompe el retroceso del 50% en la dirección de la tendencia por la distancia de ruptura especificada. La protección opcional de stop-loss está disponible mediante una distancia de precio absoluta.

## Parámetros
- Candle Type
- Breakout
- Stop Loss
- ZigZag Depth
