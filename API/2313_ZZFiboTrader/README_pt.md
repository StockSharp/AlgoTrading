# Estratégia ZZFibo Trader
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Utiliza os swings do ZigZag para derivar níveis de retração de Fibonacci. A estratégia recalcula os níveis sempre que um novo pivô de máximo ou mínimo é detectado e entra quando o preço rompe a retração de 50% na direção da tendência pela distância de rompimento especificada. A proteção opcional de stop-loss está disponível por meio de uma distância de preço absoluta.

## Parâmetros
- Candle Type
- Breakout
- Stop Loss
- ZigZag Depth
