# ZZFibo Trader戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ZigZagのスイングを使用してフィボナッチ・リトレースメントレベルを導出します。新しいピボット高値または安値が検出されるたびにレベルを再計算し、価格がトレンド方向に50%リトレースメントを指定したブレイクアウト距離分突破した際にエントリーします。絶対価格距離によるオプションのストップロス保護も利用できます。

## パラメーター
- Candle Type
- Breakout
- Stop Loss
- ZigZag Depth
