# バンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、MetaTrader 5 エキスパート アドバイザ **Bands.mq5** を StockSharp の高レベル API に移植します。キャンドルの完成を待ちます
これは、Bollinger バンドを外側からチャネルに突き刺し、Donchian チャネル設定が確立された場合にのみポジションをオープンします。
設定可能なバー数の間、バンドの傾きが安定していることがわかります。平均 True Range (ATR) の倍数は ORI を再現します。
オプションの回帰トラッカーが資本曲線決定係数を出力する一方で、最終的なストップロスとテイクプロフィットの距離を表示します。
(R 二乗) 100 取引ごとに、MQL バージョンの診断出力をミラーリングします。

## 取引ロジック
1. 単一のローソク足ストリームをサブスクライブし、MetaT と同じ期間で Bollinger バンド、Donchian チャネル、および ATR を計算します
レーダーロボット。
2. 開いているポジションがない場合は、**前**の完了したローソク足を検査します。
   - ローソク足が Bollinger バンドの下側で始まり、その上で閉じており、Donchian の下側バンドが下降していない場合は、ロングとなります。
`ConfirmationPeriod` バー以上続いています。
   - ローソク足が上部 Bollinger バンドの上で開き、その下で閉じ、かつ Donchian 上部バンドが上昇していない場合は、ショートと入力します。
`ConfirmationPeriod` バーを超える場合は en。
3. ポジションが存在する場合、末尾の Donchian 境界を越えた場合 (前の終値を使用)、または ATR ベースの場合に終了します。
d 保護レベルがバー内で違反されている。
4. 実行されたすべての取引は現在のポートフォリオ資産を保存し、各ブロックの後に線形回帰 R 二乗メトリクスを出力します。
100取引。負の傾きは、元の Expert Advisor と同様に、負の R2 乗を生成します。

## リスク管理
- エントリー注文は常にユーザー定義の `TradeVolume` を使用して市場に送信されます。
- プロテクティブ レベルは、ローソク足の高値と安値を ATR ミューと比較することによって (未決注文を使用する代わりに) コード内で再作成されます。
ティプル。
- ストップロスまたはテイクプロフィットがトリガーされると、この戦略は成行注文でポジション全体をクローズし、プロテクトをリセットします。
レベルで。

## パラメーター
| パラメータ | 説明 |
|-----------|-------------|
| `TradeVolume` | 各成行注文の正味数量 (ロット単位)。 |
| `CandleType` | すべてのインジケーターに使用されるローソク足のデータ型/タイムフレーム。 |
| `BollingerPeriod` | Bollinger バンドによって使用されるキャンドルの数。 |
| `BollingerDeviation` | Bollinger バンドに適用される標準偏差乗数。 |
| `DonchianPeriod` | トレンドフィルターとして使用される Donchian チャネルの長さ。 |
| `ConfirmationPeriod` | Donchian の傾きを非減少 (ロング) または非増加 (ショート) に保つ必要がある連続バーの最小数。 |
| `AtrPeriod` | リスク管理に使用される平均トゥルー レンジの期間。 |
| `StopAtrMultiplier` | ATR の倍数でストップロス距離を定義します。 |
| `TakeAtrMultiplier` | ATR の倍数でテイクプロフィットディスタンスを定義します。 |

## 注意事項
- Donchian のスロープ チェックは、インジケーター バッファーをコピーするのではなく、ローリング カウンタとして実装されており、StockSharp を維持します。
元の EA の動作と一致しながら効率的なバージョンになります。
- すべてのコメントと診断は、プロジェクト ガイドラインの要求に従って英語で提供されます。
- MetaTrader コードの資金管理ヘルパーは再現されません。 StockSharp の実装は `TradeVolume` に依存しています
ポジションサイジングのパラメータ。
