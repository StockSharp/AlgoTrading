# アダプティブ・フラクタル・グリッド・スキャルピング
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

アダプティブ・フラクタル・グリッド・スキャルピングは、ATR を距離の基準として、最近のフラクタルピボット周辺に指値注文を配置します。トレンドは単純移動平均によって定義されます。ボラティリティがしきい値を超えると、上昇トレンドではフラクタル安値の下に買い指値が設定され、下降トレンドではフラクタル高値の上に売り指値が設定されます。エグジットは反対側のグリッドレベルか、ATR ベースのトレーリングストップで行われます。

## 詳細

- **エントリー条件**: ATR がしきい値を超え、SMA に対する価格の位置が適切; フラクタル安値 - ATR 乗数の買い指値、またはフラクタル高値 + ATR 乗数の売り指値。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対側のグリッドレベルまたはフラクタルベースのストップ。
- **ストップ**: あり。
- **デフォルト値**:
  - `AtrLength` = 14
  - `SmaLength` = 50
  - `GridMultiplierHigh` = 2.0m
  - `GridMultiplierLow` = 0.5m
  - `TrailStopMultiplier` = 0.5m
  - `VolatilityThreshold` = 1.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: スキャルピング
  - 方向: 両方
  - インジケーター: Fractal, ATR, SMA
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
