# SlopeDirectionLine戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は*Slope Direction Line*エキスパートアドバイザーの動作を再現します。終値で構築された線形回帰ラインの傾きを分析します。回帰の傾きが負から正に転じるとロングポジションがオープンされ、正から負に転じるとショートポジションがオープンされます。方向が変わるたびに反対ポジションがクローズされます。オプションのストップロスとテイクプロフィットのパーセンテージが、組み込みの`StartProtection`メカニズムを通じてポジションを保護します。

## 詳細
- **インジケーター** – StockSharpの`LinearRegression`。戦略はシグナルとして`LinearRegSlope`コンポーネントを使用します。
- **シグナル** – 傾きのゼロ交差。正の傾きは上昇トレンドを示し、負の傾きは下降トレンドを示します。
- **エントリー/エグジット** – 傾きが符号を変えると現在のポジションがクローズされ、許可されている場合は傾きの方向に新しいポジションがオープンされます。
- **リスク管理** – `StartProtection`はユーザー定義のテイクプロフィットとストップロスのパーセンテージで設定されます。

## パラメーター
| 名前 | 説明 |
|------|------|
| `CandleType` | ローソク足構築に使用する時間軸。 |
| `Length` | 線形回帰計算に使用するバー数。 |
| `TakeProfitPercent` | エントリー価格からのテイクプロフィットまでのパーセント距離。 |
| `StopLossPercent` | エントリー価格からのストップロスまでのパーセント距離。 |
| `AllowLong` | ロングポジションのオープンを許可。 |
| `AllowShort` | ショートポジションのオープンを許可。 |

## 使用方法
1. StockSharpアプリケーションに戦略を追加します。
2. 希望する時間軸とリスクに合わせてパラメーターを設定します。
3. 戦略を起動し、チャートでトレードを監視します。
