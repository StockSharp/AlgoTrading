# 予測オシレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はクラシックな予測オシレーターインジケーターをStockSharpに適応させます。線形回帰ベースラインとTillson T3スムージングを組み合わせてトレンド反転を強調します。スムージングラインがゼロを下回っているときにオシレーターがスムージングラインを上抜けると買いシグナルが現れます。反対の条件では売りシグナルが生成されます。

アルゴリズムはオリジナルのMQL実装に従い、ポジションの開設と決済を個別に有効/無効にすることをサポートします。

## 詳細

- **エントリー条件**:
  - **ロング**: オシレーターがT3を上抜けし、T3が負の値。
  - **ショート**: オシレーターがT3を下抜けし、T3が正の値。
- **ロング/ショート**: 両方向がサポートされます。
- **エグジット条件**:
  - 対応するクローズオプションが有効な場合、反対シグナルで決済。
- **ストップ**: なし。
- **フィルター**: なし。
