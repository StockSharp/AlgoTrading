# Vela Superada戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Vela Superada戦略は2本ローソク足のリバーサルパターンを取引します。弱気のローソク足の
直後に、前のオープンを上回って終値を付ける強気のローソク足が来た時に強気のセットアップ
が発生します。取引は短期EMA、RSI、MACDトレンドでフィルタリングされ、逆張りシグナルを
避けます。ロングとショートの両方を有効にすることができます。

この戦略はパーセンテージベースのテイクプロフィットとストップロスのレベルを採用し、
価格が有利に動くとトレーリングストップを動的に引き締めます。これにより、リバーサルから
保護しながら延長された動きを捉えることができます。

## 詳細

- **エントリー条件**:
  - **ロング**: 前のローソク足が弱気、現在が強気、終値と前の終値がEMAを上回り、RSI < 65、MACDが上昇。
  - **ショート**: 前のローソク足が強気、現在が弱気、終値と前の終値がEMAを下回り、RSI > 35、MACDが下降。
- **ロング/ショート**: 設定可能（デフォルトはロング）。
- **エグジット条件**:
  - トレーリングストップまたは反対のシグナル。
- **ストップ**: パーセンテージベースのストップロスとテイクプロフィット。
- **デフォルト値**:
  - `EmaLength` = 10
  - `RsiLength` = 14
  - `ShowLong` = True
  - `ShowShort` = False
  - `TpPercent` = 1.2
  - `SlPercent` = 1.8
- **フィルター**:
  - カテゴリ: パターン + インジケーター
  - 方向: 両方
  - インジケーター: EMA、RSI、MACD
  - ストップ: はい
  - 複雑さ: 中
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
