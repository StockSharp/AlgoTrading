# Estratégia de Fusão de Grade Tian Di
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Sistema de negociação em grade que coloca ordens em níveis de preço entre limites superior e inferior. Suporta espaçamento aritmético ou geométrico e pode fechar posições quando o lucro excede um passo da grade.
