# Estrategia de Fusión de Cuadrícula Tian Di
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Sistema de trading en cuadrícula que coloca órdenes en niveles de precio entre límites superior e inferior. Admite espaciado aritmético o geométrico y puede cerrar posiciones cuando el beneficio supera un paso de la cuadrícula.
