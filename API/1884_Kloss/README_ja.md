# Kloss戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Kloss戦略は、加重移動平均（WMA）、商品チャネル指数（CCI）、ストキャスティクスオシレーターを組み合わせています。すべてのインジケーターはシフトされた履歴値で評価され、過去の市場コンテキストに基づいてシグナルを生成できます。CCIが負のしきい値を下回り、ストキャスティクスのメインラインが中立レベル50からの偏差を下回り、シフトされた価格がシフトされたWMAを上回ったときにロングポジションが開かれます。ショートポジションは逆の条件で開かれます。オプションのリバースクローズは、反対のシグナルが現れたときに既存のポジションを閉じます。ストップロスとテイクプロフィットはエントリー価格からのポイントで設定されます。

## 詳細

- **エントリー条件**:
  - **ロング**: シフトされたCCIが`-CciDiffer`を下回り、シフトされたストキャスティクスが`50 - StochDiffer`を下回り、シフトされた価格がシフトされたWMAを上回る。
  - **ショート**: シフトされたCCIが`CciDiffer`を上回り、シフトされたストキャスティクスが`50 + StochDiffer`を上回り、シフトされた価格がシフトされたWMAを下回る。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - `RevClose`が有効な場合のリバースシグナル、またはストップロス / テイクプロフィットレベル。
- **ストップ**: ポイント単位の絶対ストップロスとテイクプロフィット。
- **フィルター**:
  - `CommonShift`によるインジケーターと価格のシフトにより、過去のバーからシグナルを生成できます。
