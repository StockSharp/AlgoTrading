# Parabolic SAR Bug5 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

Parabolic SAR Bug5 戦略は Parabolic SAR インジケーターによって検出された価格リバーサルを取引します。価格がSARを上方向にクロスするとロングポジションを開き、下方向にクロスするとショートポジションを開きます。戦略はオプションで取引方向を逆転させ、SARの反転時に開いたポジションをクローズし、トレーリングストップ、テイクプロフィット、ストップロスルールをサポートします。

## エントリールール

- ロングポジションが開いていない状態で価格がSARを上方向にクロスしたとき**買い**。
- ショートポジションが開いていない状態で価格がSARを下方向にクロスしたとき**売り**。
- `Reverse` が有効な場合、シグナルは反転されます。

## エグジットルール

- `SarClose` が有効な場合、逆のSARシグナルが現れたときにポジションをクローズ。
- 固定のストップロスとテイクプロフィットのターゲットを適用。
- `Trailing` が有効な場合、ストップロスはエントリー以来の最高値（ロングの場合）または最安値（ショートの場合）に追随します。

## パラメーター

| パラメーター | 説明 |
|------------|------|
| `Step` | Parabolic SAR の初期加速係数。 |
| `Maximum` | Parabolic SAR の最大加速係数。 |
| `StopLossPoints` | ポイント単位のストップロス距離。 |
| `TakeProfitPoints` | ポイント単位のテイクプロフィット距離。 |
| `Trailing` | トレーリングストップ管理を有効にする。 |
| `TrailPoints` | ポイント単位のトレーリングストップ距離。 |
| `Reverse` | 取引方向を逆転する。 |
| `SarClose` | SARの切り替え時にポジションをクローズ。 |
| `CandleType` | 処理するローソク足の時間軸。 |
