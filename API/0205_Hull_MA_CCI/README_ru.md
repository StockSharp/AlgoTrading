# Стратегия Hull MA CCI
[English](README.md) | [中文](README_cn.md)

Данная стратегия использует индикаторы Hull Moving Average и CCI. Длинная позиция открывается, когда HMA(t) > HMA(t-1), а CCI опускается ниже -100, что указывает на рост HMA при перепроданности. Короткая позиция возникает, когда HMA(t) < HMA(t-1) и CCI выше 100 — снижение HMA при перекупленности.

Она предназначена для трейдеров, работающих в смешанном рынке.

## Детали
- **Условия входа**:
  - **Лонг**: HMA(t) > HMA(t-1) и CCI < -100 (рост HMA при перепроданности)
  - **Шорт**: HMA(t) < HMA(t-1) и CCI > 100 (падение HMA при перекупленности)
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - **Лонг**: Закрыть позицию, когда HMA начинает снижаться
  - **Шорт**: Закрыть позицию, когда HMA начинает расти
- **Стопы**: да.
- **Значения по умолчанию**:
  - `HullPeriod` = 9
  - `CciPeriod` = 20
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Смешанная
  - Направление: Оба
  - Индикаторы: Hull MA CCI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 52%\. Стратегию лучше запускать на крипторынке.
