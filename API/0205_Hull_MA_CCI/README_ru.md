# Стратегия Hull MA CCI

В стратегии применяются индикаторы Hull Moving Average и CCI. Длинная позиция берётся, когда текущая HMA выше предыдущей и CCI опускается ниже -100, что говорит о росте при перепроданности. Короткая позиция берётся, когда HMA снижается и CCI превышает 100, т.е. на перекупленности. Подходит для смешанных рынков.

## Подробности
- **Условия входа**:
  - **Long**: HMA(t) > HMA(t-1) && CCI < -100
  - **Short**: HMA(t) < HMA(t-1) && CCI > 100
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход, когда HMA начинает снижаться
  - **Short**: выход, когда HMA начинает расти
- **Стопы**: да
- **Значения по умолчанию**:
  - `HullPeriod` = 9
  - `CciPeriod` = 20
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: Hull MA CCI
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
