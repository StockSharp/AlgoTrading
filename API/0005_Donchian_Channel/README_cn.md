# Donchian通道
[English](README.md) | [Русский](README_ru.md)

Donchian通道突破策略在价格突破通道高点或低点时入场。收盘价突破上轨视为强势信号，跌破下轨则做空，价格回到中线时离场。通道基于指定周期内的最高价和最低价，突破通常意味着波动扩大。

## 详情
- **入场条件**: 基于价格行为的信号
- **多空方向**: 双向
- **退出条件**: 反向信号
- **止损**: 无
- **默认值**:
  - `ChannelPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类型: 趋势
  - 方向: 双向
  - 指标: Price Action
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
