# Канал Дончиана

Стратегия основана на канале Дончиана.

Прорыв канала Дончиана торгует новые максимумы и минимумы, определяемые шириной канала. Закрытие выше верхней границы говорит о силе, а падение ниже нижней открывает короткую позицию. Выход происходит, когда цена возвращается к среднему уровню.

Канал строится по наивысшему максимуму и наинизшему минимуму за период. Когда цена пробивает эти границы, ожидается расширение волатильности, и позиция открывается соответственно.

## Детали

- **Критерии входа**: сигналы на основе ценового действия.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `ChannelPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Price Action
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
