# Стратегия Parabolic RSI
[English](README.md) | [中文](README_cn.md)

Стратегия применяет Parabolic SAR к RSI для определения смены тренда. Вход выполняется при смене положения SAR относительно линии RSI, при желании сделки фильтруются по значениям RSI.

## Детали

- **Условия входа**:
  - Long: SAR переходит под RSI и (опционально) `RSI ≥ LongRsiMin`
  - Short: SAR переходит над RSI и (опционально) `RSI ≤ ShortRsiMax`
- **Long/Short**: Настраивается
- **Условия выхода**: Обратный переворот SAR
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `SarStart` = 0.02
  - `SarIncrement` = 0.02
  - `SarMax` = 0.2
  - `LongRsiMin` = 50
  - `ShortRsiMax` = 50
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Настраиваемое
  - Индикаторы: Parabolic SAR, RSI
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
