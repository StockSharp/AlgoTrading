# 2pbIdeal XOSMA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQL5エキスパートアドバイザー **Exp_2pbIdealXOSMA** のC#翻訳版です。MACDヒストグラムの傾きを分析して市場のモメンタムを判断します。ヒストグラムが2本連続のバーで上昇すると、システムはロングポジションに入り、オープンしているショートを閉じます。ヒストグラムが2本連続のバーで下落すると、戦略はショートポジションに入り、オープンしているロングを閉じます。

デフォルトでは4時間足のローソク足で動作しますが、時間軸は設定可能です。すべての取引は成行価格で実行され、反対のシグナルが現れるとポジションが反転します。サンプル内にストップロスやテイクプロフィットは実装されていません。必要に応じて外部でリスク管理を追加できます。

## 詳細

- **エントリー条件**:
  - **ロング**: バー `t-1` のヒストグラムが `t-2` を下回り、現在のヒストグラムが `t-1` を超える。
  - **ショート**: バー `t-1` のヒストグラムが `t-2` を上回り、現在のヒストグラムが `t-1` を下回る。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対のシグナルが現在のポジションを閉じます。
- **ストップ**: なし。
- **デフォルト値**:
  - `FastPeriod` = 10
  - `SlowPeriod` = 26
  - `SignalPeriod` = 9
  - `SignalBar` = 1
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一 (MACD)
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 4時間（設定可能）
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
