# Стратегия EMA RSI Trend Reversal
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию при пересечении быстрой EMA выше медленной с подтверждением RSI и закрывает её при обратном пересечении, когда RSI опускается ниже уровня. Использует процентный тейк-профит и стоп-лосс.

## Детали

- **Условия входа**:
  - Лонг: `FastEMA пересекает SlowEMA снизу вверх && RSI > RsiLevel`
- **Лонг/Шорт**: Только лонг
- **Стопы**: Процентный тейк-профит и стоп-лосс
- **Значения по умолчанию**:
  - `FastLength` = 9
  - `SlowLength` = 21
  - `RsiLength` = 14
  - `RsiLevel` = 50m
  - `TakeProfitPercent` = 2m
  - `StopLossPercent` = 1m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: EMA, RSI
  - Стопы: Да
  - Сложность: Начальный
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
