# Стратегия Order Example
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя, основанная на примере MQL5 `OrderExample.mq5`.
Она входит в позицию, когда цена пробивает недавние максимумы или минимумы.

Стратегия использует индикаторы `Highest` и `Lowest` для отслеживания уровней пробоя на заданном окне.

## Подробности

- **Условия входа**:
  - Длинная: `Close` пробивает максимум из `Lookback` свечей
  - Короткая: `Close` пробивает минимум из `Lookback` свечей
- **Long/Short**: Оба
- **Условия выхода**: Противоположный пробой
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `Lookback` = 26
  - `CandleType` = `TimeSpan.FromMinutes(5).TimeFrame()`
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: Highest, Lowest
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
