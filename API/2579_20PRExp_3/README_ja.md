# 20PRExp-3 ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

20PRExp-3戦略は、現在の取引セッションを前日の価格極値と比較するブレイクアウトシステムです。5分足が完成するたびに日次チャネルを再構築し、30分ティックボリューム拡大でモメンタムを確認し、価格が更新されたセッションの高値または安値を超えてブレイクアウトしたときのみエントリーします。マーケットに参入すると、Parabolic SARによる終了、動的トレーリングストップ、保護ストップまでの距離に基づく固定リスクサイジングを使用して、元のMetaTrader 5エキスパートを反映します。

## コンセプト
- **日次チャネル**：現在の取引日の実行中の高値、安値、中間点を追跡します。
- **ブレイクアウト確認**：最小日次レンジフィルター（`GapPoints`）でチャネル境界を超えて価格が終わることを要求します。
- **ボリューム拡大**：最後の2つの完成した30分足を比較し、薄いブレイクアウトを避けるためにティックボリュームの少なくとも1.5倍の増加を要求します。
- **時間フィルター**：低流動性の夜間レンジを避けるために、設定されたセッション開始時刻（`SessionStartHour`）以降のみ新しいポジションを許可します。
- **リスク対称性**：ロング取引はストップロスとして日次安値を使用し、ショート取引は日次高値を使用します。テイクプロフィットとトレーリングオフセットは価格ポイントで測定されます。

## 市場データ
- 主要シグナルとParabolic SAR計算のための5分足。
- ティックボリューム比率フィルターのための30分足。
- 日次高値/安値統計は5分足データから随時導出されるため、別の日次購読は不要です。

## エントリーロジック
1. 設定された開始時間後に完成した5分足を待ちます。
2. 現在の日次高値/安値/中間点とチャネル幅を計算します。
3. チャネル幅が`GapPoints * PriceStep`を超えることを確認します。
4. ティックボリューム比率を計算 = （最後の完成した30分ボリューム）/ （前の30分ボリューム）し、1.5より大きいことを確認します。
5. **ロング設定**：ローソク足が現在の日次高値以上で終了 → 買い。
6. **ショート設定**：ローソク足が現在の日次安値以下で終了 → 売り。
7. ポジションがアクティブな間は新しいエントリーをスキップします（最大1つのオープン取引）。

## 終了管理
- **初期ストップ**：ロング取引はエントリー時に捕捉された日次安値を使用し、ショート取引は日次高値を使用します。
- **テイクプロフィット**：オプション；マーケットの両側でエントリーから`TakeProfitPoints * PriceStep`離れた位置に配置します。
- **Parabolic SARリバーサル**：SARの値が前のローソク足の終値を超えるとポジションを閉じます（元のEA動作）。
- **トレーリングストップ**：利益が`TrailingStopPoints * PriceStep`を超えると有効化し、少なくとも`TrailingStepPoints * PriceStep`移動します。
- **ミラートレーリングテイク**：トレーリングストップが更新されるたびに、テイクプロフィットレベルは現在の終値を中心に対称的に再配置されます。

## ポジションサイジング
- ポジションボリュームは`RiskPercent`から導出されます：戦略はエントリーとストップの間の距離に基づいてポートフォリオの現在価値の一定割合をリスクにさらします。
- ポートフォリオ評価が利用できない場合、アルゴリズムは`Volume + |Position|`にフォールバックし、最終手段として単一コントラクトを取引します。

## パラメーター
| パラメーター | デフォルト値 | 説明 |
|-------------|------------|------|
| `CandleType` | 5分足 | シグナルとParabolic SARの主要時間軸。 |
| `VolumeCandleType` | 30分足 | ティックボリューム拡大を評価する時間軸。 |
| `TakeProfitPoints` | 20 | 価格ポイントで表現した利益目標距離。0に設定して無効化。 |
| `TrailingStopPoints` | 10 | トレーリングストップ有効化のポイント単位距離。 |
| `TrailingStepPoints` | 10 | トレーリングストップが再移動する前の最小追加進行（ポイント）。 |
| `RiskPercent` | 5 | 取引ごとにリスクにさらすポートフォリオエクイティの割合。 |
| `GapPoints` | 50 | ブレイクアウトを有効にするために必要な最小日次チャネル幅（ポイント）。 |
| `SessionStartHour` | 7 | 戦略が新しいポジションを開くことが許可される時間（0〜23）。 |

## メモ
- Parabolic SARパラメーター（ステップ0.005、最大0.01）は元のMQL戦略と一致します。
- 日次中点値は完全性のために計算され、必要に応じて視覚的参照のためにプロットできます。
- ボリューム拡大は完成した30分足で評価されるため、ブレイクアウト確認は利用可能な最新の終値対終値情報を使用し、ヒストリカルテストとライブ取引の両方に対して堅牢です。
