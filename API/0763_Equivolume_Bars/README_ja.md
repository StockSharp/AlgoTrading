# 等量バー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ルックバック期間の出来高合計に対する出来高スパイクに基づく戦略です。

## ロジック
- 現在の出来高と過去の出来高合計の比率を計算する。
- 比率が閾値を超え、ローソク足が強気の場合にロングポジションを取る。
- 比率が閾値を超え、ローソク足が弱気の場合にショートポジションを取る。
- 比率が閾値を下回るか、ローソク足が反転した場合にポジションを閉じる。

## パラメーター
- `Lookback` – 出来高合計のバー数。
- `Volume Threshold` – 高出来高の比率閾値。
- `Candle Type` – 使用するローソク足の種類。
