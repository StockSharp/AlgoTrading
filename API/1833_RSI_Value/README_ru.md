# RSI Value
[English](README.md) | [中文](README_cn.md)

Стратегия торгует по сигналам индикатора RSI при пересечении средней линии.

Когда RSI поднимается выше заданного уровня (по умолчанию 50) – открывается длинная позиция. При обратном пересечении вниз открывается короткая позиция. Текущие позиции закрываются при противоположном пересечении. Для защиты используются необязательные стоп-лосс, тейк-профит и трейлинг-стоп.

## Детали

- **Условия входа**: Покупка при пересечении RSI вверх уровня, продажа при пересечении вниз.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Обратное пересечение или трейлинг-стоп.
- **Стопы**: Фиксированный стоп-лосс, тейк-профит и трейлинг-стоп (опционально).
- **Значения по умолчанию**:
  - `RsiPeriod` = 14
  - `RsiLevel` = 50
  - `StopLoss` = 100
  - `TakeProfit` = 200
  - `TrailingStop` = 0
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
