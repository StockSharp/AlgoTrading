# Drag SL/TP マネージャー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、約定価格から固定距離にストップロスとテイクプロフィットの注文を自動的に配置します。手動でポジションを開いた後、直ちに保護する必要がある場合に役立ちます。

## パラメーター

- **Auto Set SL** (`bool`): ストップロスの自動配置を有効にする。
- **SL Points** (`decimal`): 価格ステップ単位のストップロス距離。
- **Auto Set TP** (`bool`): テイクプロフィットの自動配置を有効にする。
- **TP Points** (`decimal`): 価格ステップ単位のテイクプロフィット距離。

## 動作

戦略が開始されると、選択した距離で `StartProtection` を呼び出します。戦略の実行中に開かれたポジションは、対応する保護注文を直ちに受け取ります。距離は価格ステップ（`Security.PriceStep`）で測定されます。

戦略自体はトレードシグナルを生成しません。手動または他の戦略によって開かれたポジションの保護注文を管理するだけです。

## 注意事項

- 高レベル API の使用を前提として設計されています。
- 拡張バージョンでは、完成したローソク足の状態のみがトレードアクションをトリガーする必要があります。
- 元の MQL スクリプトのグラフィカルなドラッグ機能は実装されていません。
