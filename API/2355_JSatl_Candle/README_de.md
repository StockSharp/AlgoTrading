# JSatl Kerzen-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet den Jurik Moving Average (JMA), um einen geglätteten Trend zu erstellen.
Eine Long-Position wird eröffnet, wenn der JMA nach oben dreht, nachdem er flach oder abwärts war.
Eine Short-Position wird eröffnet, wenn der JMA nach unten dreht, nachdem er flach oder aufwärts war.
Optionaler Stop-Loss und Take-Profit werden als Prozentsätze des Einstiegspreises angewendet.
