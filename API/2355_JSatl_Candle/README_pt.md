# Estratégia JSatl Candle
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia usa a Média Móvel Jurik (JMA) para construir uma tendência suavizada.
Uma posição comprada é aberta quando o JMA gira para cima após estar plano ou para baixo.
Uma posição vendida é aberta quando o JMA gira para baixo após estar plano ou para cima.
Stop loss e take profit opcionais são aplicados como percentagens do preço de entrada.
