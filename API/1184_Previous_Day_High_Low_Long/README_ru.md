# Стратегия пробоя максимумов-минимумов предыдущего дня (лонг)
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию при пробое максимума или минимума предыдущего дня в заданную сессию при подтверждении растущим ADX.

## Детали

- **Условия входа**:
  - **Лонг**: закрытие пересекает выше максимума или минимума предыдущего дня, ADX растёт, время в пределах сессии.
- **Лонг/Шорт**: только лонг.
- **Условия выхода**:
  - динамический стоп и тейк-профит или окончание сессии.
- **Стопы**: трейлинг-стоп.
- **Значения по умолчанию**:
  - `MaxProfit` = 150.
  - `MaxStopLoss` = 15.
  - `AdxLength` = 11.
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame().
- **Фильтры**:
  - Категория: Пробой
  - Направление: Лонг
  - Индикаторы: ADX
  - Стопы: Да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
