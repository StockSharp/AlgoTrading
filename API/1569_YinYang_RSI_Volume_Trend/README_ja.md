# YinYang RSI 出来高トレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

YinYang RSI 出来高トレンド戦略は、出来高加重価格ゾーンと RSI フィルターを使用してトレンド転換を検出します。価格が下部ゾーンを抜けたときに買い、上部ゾーンを抜けたときに売ります。オプションのストップロスとテイクプロフィットのレベルは動的ゾーンに基づきます。

## 詳細

- **エントリー条件**: 価格が計算された購入ゾーンを抜け、利用可能リセットオプションを確認。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 価格が反対ゾーンに達するか、オプションのストップロス/テイクプロフィットが発動する。
- **ストップ**: オプション。
- **デフォルト値**:
  - `TrendLength` = 80
  - `UseTakeProfit` = true
  - `UseStopLoss` = true
  - `StopLossMultiplier` = 0.1
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: VWMA, EMA, RSI
  - ストップ: オプション
  - 複雑さ: 中級
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
