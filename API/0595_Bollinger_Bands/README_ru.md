# Стратегия Bollinger Bands
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробои полос Боллинджера. Покупает, когда цена закрывается выше верхней полосы, и продаёт, когда закрывается ниже нижней полосы. Выход осуществляется при пересечении простой скользящей средней или при срабатывании стоп-лосса.

## Детали

- **Условия входа**:
  - Long: закрытие выше верхней полосы Боллинджера
  - Short: закрытие ниже нижней полосы Боллинджера
- **Направление**: Оба
- **Условия выхода**:
  - Long: закрытие ниже SMA или достижение стоп-лосса
  - Short: закрытие выше SMA или достижение стоп-лосса
- **Стопы**: процент от цены входа
- **Значения по умолчанию**:
  - `BbLength` = 120
  - `BbDeviation` = 2m
  - `SmaLength` = 110
  - `StopLossPercent` = 6m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Bollinger Bands, SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
