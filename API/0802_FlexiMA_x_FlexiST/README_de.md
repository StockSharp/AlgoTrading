# FlexiMA x FlexiST-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Kombiniert einen einfachen gleitenden Durchschnitt mit einem Supertrend-Filter.

- **Long** wenn der Preis über beiden Indikatoren liegt.
- **Short** wenn der Preis unter beiden liegt.
- **Schließen** von Positionen, wenn die entgegengesetzte Bedingung eintritt.

**Indikatoren**: SMA, SuperTrend
**Stops**: keine
