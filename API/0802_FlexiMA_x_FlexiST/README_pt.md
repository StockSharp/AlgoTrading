# Estratégia FlexiMA x FlexiST
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Combina uma média móvel simples com um filtro Supertrend.

- **Comprado** quando o preço está acima de ambos os indicadores.
- **Vendido** quando o preço está abaixo de ambos.
- **Fechar** posições quando a condição oposta aparece.

**Indicadores**: SMA, SuperTrend
**Stops**: nenhum
