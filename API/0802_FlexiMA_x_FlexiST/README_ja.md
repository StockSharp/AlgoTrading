# FlexiMA x FlexiST戦略

単純移動平均とSupertrendフィルターを組み合わせた戦略です。

- 価格が両インジケーターを上回るとき**ロング**。
- 価格が両インジケーターを下回るとき**ショート**。
- 反対の条件が現れたときポジションを**クローズ**。

**インジケーター**: SMA, SuperTrend
**ストップ**: なし
