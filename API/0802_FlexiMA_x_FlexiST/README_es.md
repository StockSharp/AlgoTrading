# Estrategia FlexiMA x FlexiST
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Combina una media móvil simple con un filtro Supertrend.

- **Largo** cuando el precio está por encima de ambos indicadores.
- **Corto** cuando el precio está por debajo de ambos.
- **Cierre** de posiciones cuando aparece la condición opuesta.

**Indicadores**: SMA, SuperTrend
**Stops**: ninguno
