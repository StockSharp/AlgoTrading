# Стратегия Quadratic Regression
[English](README.md) | [中文](README_cn.md)

Стратегия строит линию квадратичной регрессии по последним `Length` барам и торгует на пересечениях цены с этой линией.

## Детали

- **Условия входа**: Цена пересекает линию квадратичной регрессии.
- **Лонг/Шорт**: оба направления.
- **Условия выхода**: Обратное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Length` = 54.
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **Фильтры**:
  - Категория: Трендовая
  - Направление: оба
  - Индикаторы: Quadratic Regression
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
