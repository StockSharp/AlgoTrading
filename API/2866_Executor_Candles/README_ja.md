# エグゼキューター・ローソク足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTrader「Executor Candles」エキスパートの直接変換です。豊富な強気と弱気のローソク足リバーサルパターンのセットに反応し、オプションで上位タイムフレームのトレンドローソク足でトレードを確認できます。すべてのトレード管理ロジック – ストップ、テイクプロフィット、トレーリングストップ – はpips（価格ステップ）で測定されたオリジナルエキスパートの動作を反映します。

## 動作方法

- **トレンドフィルター**: `UseTrendFilter`が有効な場合、戦略は`TrendCandleType`の最後の完成したローソク足を観察します。ロングセットアップはそのローソク足が強気で閉じた場合のみ許可され、ショートセットアップは弱気のクローズが必要です。フィルターが無効（デフォルト）の場合、パターンロジックのみが使用されます。
- **ロングパターン**: ハンマー、強気のエンゴルフ、ピアシングライン、モーニングスター、最後の3つの完成した取引ローソク足から取ったモーニングドージスター構造。
- **ショートパターン**: ハンギングマン、弱気のエンゴルフ、ダーククラウドカバー、イブニングスター、イブニングドージスターの確認。
- **トレード管理**:
  - pipsで表されたロングとショートポジションの個別のストップロスとテイクプロフィット距離（`StopLossBuyPips`、`TakeProfitBuyPips`、`StopLossSellPips`、`TakeProfitSellPips`）。
  - `TrailingStopBuyPips`、`TrailingStopSellPips`、最小シフト`TrailingStepPips`によって制御される両方向のオプショナルトレーリングストップ。トレーリング更新はMetaTraderロジックを複製して、価格がストップ距離プラストレーリングステップ分進んだ後にのみ行われます。
  - 注文は`OrderVolume`ロットで配置され、エグジット条件がトリガーされたときに現在のポジションは成行注文で完全に反転されます。

戦略はトレードシグナルのために設定された`CandleType`にサブスクライブし、必要に応じて確認ローソク足のために`TrendCandleType`にもサブスクライブします。長い履歴を保存せずに複数バーパターンを評価するために最後の3つの完成した取引ローソク足の内部バッファーを保持します。

## パラメーター

- `CandleType` – ローソク足パターンを検出するために使用するタイムフレーム。
- `TrendCandleType` – トレンドフィルターがアクティブなときに使用する上位タイムフレームローソク足。
- `OrderVolume` – 成行エントリーとエグジットの注文サイズ。
- `StopLossBuyPips`、`TakeProfitBuyPips`、`TrailingStopBuyPips` – ロングポジションのリスクコントロール。
- `StopLossSellPips`、`TakeProfitSellPips`、`TrailingStopSellPips` – ショートポジションのリスクコントロール。
- `TrailingStepPips` – トレーリングストップが引き締められる前の最小有利移動。
- `UseTrendFilter` – 上位タイムフレームの確認を有効または無効にします。

## 注記

- すべてのpipsベースの距離はインストゥルメントの`PriceStep`で乗算されます。正確なリスクレベルのために正しく設定されていることを確認してください。
- エントリーチェックはすべての完成したローソク足で実行されます；ライブのティックは意思決定フローを変えずに最新のバーを単純に更新します。
- 戦略は成行注文のみを発行し、MetaTraderバージョンのように実行が即座に行われることを期待します。
