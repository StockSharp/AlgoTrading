# Стратегия Pullback Pro Dow
[English](README.md) | [中文](README_cn.md)

Стратегия использует пивоты по теории Доу для определения направления тренда и входит на откатах к EMA, когда сила тренда подтверждается ADX. Выход осуществляется по двум уровням тейк‑профита по отношению риск/прибыль.

Тесты показывают стабильное поведение на фьючерсах индексов, таких как US30.

## Детали

- **Условия входа**:
  - Лонг: восходящий тренд (выше максимумы и минимумы), цена пересекает EMA снизу вверх, ADX выше порога
  - Шорт: нисходящий тренд (ниже максимумы и минимумы), цена пересекает EMA сверху вниз, ADX выше порога
- **Лонг/Шорт**: Оба направления
- **Условия выхода**: Стоп по последнему пивоту, два тейк‑профита по R:R
- **Стопы**: По пивоту
- **Значения по умолчанию**:
  - `PivotLookback` = 10
  - `EmaLength` = 21
  - `RiskReward1` = 1.5m
  - `Tp1Percent` = 50
  - `RiskReward2` = 3m
  - `UseAdxFilter` = true
  - `AdxLength` = 14
  - `AdxThreshold` = 25m
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: EMA, Average Directional Index
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
