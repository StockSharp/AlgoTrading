# Стратегия Turn of the Month on Steroids
[English](README.md) | [中文](README_cn.md)

Сезонная стратегия, которая покупает в конце месяца после двух подряд отрицательных закрытий и выходит, когда краткосрочный RSI показывает перекупленность.

## Детали

- **Условия входа**: день месяца выше порога и два дня снижения подряд
- **Лонг/Шорт**: Лонг
- **Условия выхода**: RSI выше порогового значения
- **Стопы**: нет
- **Значения по умолчанию**:
  - `DayOfMonth` = 25
  - `RsiLength` = 2
  - `RsiThreshold` = 65
- **Фильтры**:
  - Категория: Сезонность
  - Направление: Лонг
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Новичок
  - Таймфрейм: Дневной
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
