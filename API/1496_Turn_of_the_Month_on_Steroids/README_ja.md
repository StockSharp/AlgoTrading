# 月末転換 on Steroids戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

各月末付近に2日連続で終値が下落した後に買い、短期RSIが買われすぎのシグナルを出したときに決済する季節性戦略。

## 詳細

- **エントリー条件**: 月内の日付が閾値を超えており、2日間の下落がある
- **ロング/ショート**: ロング
- **エグジット条件**: RSIが閾値を上回る
- **ストップ**: なし
- **デフォルト値**:
  - `DayOfMonth` = 25
  - `RsiLength` = 2
  - `RsiThreshold` = 65
- **フィルター**:
  - カテゴリ: 季節性
  - 方向: ロングのみ
  - インジケーター: RSI
  - ストップ: いいえ
  - 複雑さ: 初心者
  - 時間軸: 日足
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
