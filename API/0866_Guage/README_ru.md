# Стратегия Gauge
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия повторяет библиотеку TradingView Gauge, измеряя положение цены между
заданными минимальным и максимальным уровнями. При пересечении верхнего или
нижнего порога открывается сделка в соответствующем направлении.

## Детали

- **Условия входа**:
  - **Long**: доля выше верхнего порога.
  - **Short**: доля ниже нижнего порога.
- **Long/Short**: Оба направления.
- **Условия выхода**:
  - Противоположный сигнал закрывает позицию.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - Min value = 0, Max value = 100.
  - Upper threshold = 75%, Lower threshold = 25%.
- **Фильтры**:
  - Категория: Диапазон / Утилита
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
