# Gauge戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はTradingViewのGaugeライブラリを模倣し、ユーザー定義の最小値と最大値の間の価格位置を測定します。パーセンテージが上限または下限の閾値を超えると、対応する方向でトレードに入ります。

## 詳細

- **エントリー条件**:
  - **ロング**: Gauge比率が上限閾値を上回る。
  - **ショート**: Gauge比率が下限閾値を下回る。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 反対のシグナルがエグジットを生成する。
- **ストップ**: なし。
- **デフォルト値**:
  - Min value = 0, Max value = 100.
  - Upper threshold = 75%, Lower threshold = 25%.
- **フィルター**:
  - カテゴリ: レンジ / ユーティリティ
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
