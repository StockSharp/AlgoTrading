# 仪表策略
[English](README.md) | [Русский](README_ru.md)

该策略模仿 TradingView 的 Gauge 库，通过测量价格在预设最小值和最
大值之间的位置来决策。当百分比突破上限或下限时，在相应方向开仓。

## 细节

- **入场条件**:
  - **多头**: 比例高于上限阈值。
  - **空头**: 比例低于下限阈值。
- **多空方向**: 双向。
- **出场条件**:
  - 相反信号触发平仓。
- **止损**: 无。
- **默认值**:
  - Min value = 0, Max value = 100。
  - Upper threshold = 75%，Lower threshold = 25%。
- **过滤器**:
  - 类别: 区间 / 工具
  - 方向: 双向
  - 指标: 无
  - 止损: 无
  - 复杂度: 低
  - 时间框架: 任意
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中
