# Стратегия REIT ETF Trading System
[English](README.md) | [中文](README_cn.md)

Стратегия для недельных графиков REIT ETF. Сочетает пробой нижней полосы Боллинджера и пробой канала Дончиана с фильтрами по доходности казначейских облигаций и корреляции с индексами SPY и TNX.

## Подробности

- **Условия входа**:
  - Пробой полос Боллинджера с фильтрами по доходности и корреляции.
  - Пробой канала Дончиана с дополнительными фильтрами.
- **Длинные/короткие**: Только длинные.
- **Условия выхода**:
  - Трейлинг по TNX.
  - Сигнал перекупленности и стоп-условия.
- **Стопы**: ATR и процент потери.
- **Параметры по умолчанию**:
  - `BollingerLength` = 15
  - `BollingerMultiplier` = 2
  - `TnxLookbackPeriod` = 25
  - `TnxMinChangePercent` = 15
  - `DonchianChannelLength` = 30
  - `MaxCorrelationForBuy` = 0.3
  - `MinYield` = 2
  - `AtrStopMultiplier` = 1.5
  - `StopLossPercent` = 8
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Длинные
  - Индикаторы: Bollinger Bands, Donchian Channel, ATR, Stochastic, Correlation
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Weekly
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
