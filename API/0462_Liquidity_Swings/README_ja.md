# 流動性スイング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

流動性スイング戦略は、最近のピボット高値と安値を追跡して抵抗線とサポートラインを定義します。安値がサポートを上抜けながら終値が抵抗を下回っているときにロング取引が発生します。高値が抵抗を下抜けながら終値がサポートを上回っているときにショート取引が発動します。リスク管理には、バッファー付きのレベルでのストップロスと、そのリスク距離の2倍のテイクプロフィットを使用し、1:2のリスクリワード比を実現します。

## 詳細

- **エントリー条件**:
  - **ロング**: 安値がサポートを上抜けかつ終値 < 抵抗。
  - **ショート**: 高値が抵抗を下抜けかつ終値 > サポート。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - レベルまたはバッファー値でのストップロス。
  - リスク距離の2倍でのテイクプロフィット。
- **ストップ**: ストップロスとテイクプロフィット。
- **デフォルト値**:
  - `Lookback` = 5
  - `StopLossBuffer` = 0.5
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Pivot highs/lows
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: 1h（デフォルト）
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
