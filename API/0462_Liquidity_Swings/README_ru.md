# Стратегия Liquidity Swings
[English](README.md) | [中文](README_cn.md)

Стратегия Liquidity Swings использует недавние пивот-хай и пивот-лоу для определения зон сопротивления и поддержки. Лонг открывается, когда минимум пересекает поддержку снизу, а закрытие остаётся ниже сопротивления. Шорт открывается, когда максимум пересекает сопротивление сверху, а закрытие остаётся выше поддержки. Управление рисками строится на стоп‑лоссе от уровня с буфером и тейк‑профите вдвое больше этого расстояния (соотношение 1:2).

## Подробности

- **Условия входа**:
  - **Лонг**: минимум пересекает поддержку снизу и закрытие < сопротивления.
  - **Шорт**: максимум пересекает сопротивление сверху и закрытие > поддержки.
- **Направление**: обе стороны.
- **Условия выхода**:
  - Стоп‑лосс на уровне или с учётом буфера.
  - Тейк‑профит на расстоянии 2× риска.
- **Стопы**: стоп‑лосс и тейк‑профит.
- **Параметры по умолчанию**:
  - `Lookback` = 5
  - `StopLossBuffer` = 0.5
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе
  - Индикаторы: Пивот‑хай/лоу
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: 1ч (по умолчанию)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
