# サイダス v1 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
Sidus v1 は、2 セットの指数移動平均 (EMA) と相対強度指数 (RSI) フィルターを組み合わせたトレンド追跡戦略です。元の MetaTrader 4 エキスパート アドバイザーは、速い EMA が遅い EMA から分岐し、RSI が売られすぎまたは買われすぎの状態を確認したときにポジションをオープンします。この StockSharp ポートはコア ロジックを維持し、取引を出来高の少ないローソク足に制限し、ロング ポジションとショート ポジションに非対称の保護注文を付けます。

## 使用されるインジケーター
- **高速 EMA (買いレッグ)** – ロングエントリーの短期的な勢いを測定します。
- **スロー EMA (買いレッグ)** – ロングエントリーの長期トレンドフィルターを表します。
- **速い EMA (売りレッグ)** – ショートエントリーの短期的な勢いを測定します。
- **スロー EMA (売りレッグ)** – 短期エントリーの長期トレンド フィルターを表します。
- **RSI (買いレッグ)** – ロング取引の売られ過ぎ条件を検証します。
- **RSI (売りレッグ)** – ショートトレードの買われ過ぎの状態を検証します。

## 取引ロジック
1. 設定されたローソク足シリーズ (デフォルトの 15 分の時間枠) を購読します。
2. 完成した各ローソク足のすべての EMA および RSI インジケーターを計算します。
3. ろうそくの量が設定された制限 (デフォルトは 10) を超えた場合、信号の評価をスキップします。
4. **購入条件**:
   - 高速 EMA から低速 EMA を引いた値は購入基準を下回っています。
   - RSI の値は購入基準 RSI を下回っています。
   - 既存の長時間エクスポージャーはありません (ネットポジションは非プラスである必要があります)。
5. **販売条件**:
   - 高速 EMA (売りレッグ) から低速 EMA (売りレッグ) を引いた値が売り閾値を上回っています。
   - RSI (売りレッグ) は売り閾値 RSI を上回っています。
   - 既存のショートエクスポージャーはありません (ネットポジションはマイナスでない必要があります)。
6. シグナルがトリガーされると、未決の保護注文をキャンセルし、ネットポジションを希望する側に反転するサイズの成行注文を実行し、ポジションの方向に合わせたテイクプロフィット注文とストップロス注文を即座に発注します。

## リスク管理
- ロング取引では、テイクプロフィットは `entry + BuyTakeProfitPips * priceStep`、ストップロスは `entry - BuyStopLossPips * priceStep` に設定されます。
- ショートトレードでは、テイクプロフィットは `entry - SellTakeProfitPips * priceStep`、ストップロスは `entry + SellStopLossPips * priceStep` に設定されます。
- 保護注文では、現在の証券価格ステップが再利用されます。異なるティックサイズのインストゥルメントに適応するために pip パラメータを変更します。

## パラメーター
| パラメータ | 説明 | デフォルト |
|-----------|-------------|---------|
| `FastEmaLength` | 買いシグナルの高速 EMA の長さ。 | 23 |
| `SlowEmaLength` | 買いシグナルの遅い EMA の長さ。 | 62 |
| `FastEma2Length` | 売りシグナルの高速 EMA の長さ。 | 18 |
| `SlowEma2Length` | 売りシグナルの低速 EMA の長さ。 | 54 |
| `RsiPeriod` | 購入確認の期間は RSI です。 | 67 |
| `RsiPeriod2` | 販売確認の期間は RSI です。 | 97 |
| `BuyDifferenceThreshold` | 購入できる最大の高速と低速の差は EMA です。 | 63 |
| `BuyRsiThreshold` | 購入できる最大 RSI レベル。 | 59 |
| `SellDifferenceThreshold` | 売りを許可する最小のファストとスローの差は EMA です。 | -57 |
| `SellRsiThreshold` | 販売を許可する最小の RSI レベル。 | 60 |
| `BuyTakeProfitPips` | ロングトレードの利食い距離 (pips)。 | 95 |
| `BuyStopLossPips` | ロングトレードのストップロス距離 (pips)。 | 100 |
| `SellTakeProfitPips` | 短期取引の利食い距離 (pips)。 | 17 |
| `SellStopLossPips` | 短期取引のストップロス距離 (pips)。 | 69 |
| `OrderVolume` | 新しくオープンしたポジションのボリューム。 | 0.5 |
| `MaxCandleVolume` | 取引できるキャンドルの最大量。 | 10 |
| `CandleType` | 計算に使用される時間枠。 | 15分キャンドル |

## 使用上の注意
- 適切なリスク管理のために、接続された証券が同時成行注文、逆指値注文、指値注文をサポートしていることを確認します。
- 元のエキスパートが想定した MT4 ポイント値と異なる場合は、商品のティック サイズを反映するようにピップ設定を調整します。
- この戦略はネットポジションに基づいて機能します。反対方向の新しい取引を確立する前に、反対方向のエクスポージャーを平坦化します。
