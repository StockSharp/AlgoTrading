# CCIヒストグラム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は商品チャネル指数（CCI）を使用して、インジケーターが極値ゾーンを離れたときのリバーサルを検出します。CCIが上方レベルを超えた後に下回るとロングポジションを開きます。CCIが下方レベルを下回った後に上回るとショートポジションを開きます。オプションのストップロスとテイクプロフィットレベル（ポイント単位）でオープンポジションを保護できます。

## 詳細

- **エントリー条件**:
  - **ロング**: 前回のCCI > `UpperLevel` かつ現在のCCI ≤ `UpperLevel`。
  - **ショート**: 前回のCCI < `LowerLevel` かつ現在のCCI ≥ `LowerLevel`。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対シグナルが既存ポジションを決済し、新しいポジションを開きます。
- **ストップ**: オプションの固定ストップロスとテイクプロフィット（ポイント単位）。
- **デフォルト値**:
  - `CCI Period` = 14
  - `Upper Level` = 100
  - `Lower Level` = -100
  - `Stop Loss` = 100ポイント
  - `Take Profit` = 200ポイント
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: CCI
  - ストップ: オプション
  - 複雑さ: シンプル
  - 時間軸: 任意（デフォルト4H）
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
