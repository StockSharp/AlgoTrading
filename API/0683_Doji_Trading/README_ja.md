# Doji トレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、指数移動平均線の上方に現れるDoji ローソク足を探します。そのようなパターンが発生すると、ロングポジションに入ります。ストップロスは直近のバーの最安値に設定され、価格が十分に有利な方向へ動いた後、トレーリングストップで利益を守ります。

## 詳細

- **エントリー条件**: EMAより上で終値があるDojiローソク足。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 最安値ストップとトレーリングストップ。
- **ストップ**: はい、固定とトレーリング。
- **デフォルト値**:
  - `CandleType` = 5分
  - `EmaLength` = 60
  - `Tolerance` = 0.05
  - `StopBars` = 450
  - `TrailTriggerPercent` = 1
  - `TrailOffsetPercent` = 0.5
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ロング
  - インジケーター: EMA、ローソク足
  - ストップ: はい
  - 複雑さ: 初心者
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
