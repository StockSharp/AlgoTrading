# Octopus Nest 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ボリンジャーバンドとケルトナーチャネルを使用してスクイーズのブレイクアウトを狙う戦略です。方向はEMAとParabolic SARで確認されます。ストップは直近の高値/安値に設定し、リスク/リワード比は設定可能です。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格がEMAとPSARを上回り、スクイーズの外側にある。
  - **ショート**: 価格がEMAとPSARを下回り、スクイーズの外側にある。
- **ロング/ショート**: 両方。
- **エグジット条件**: 直近の極値でのストップロスとリスク/リワード比に基づくテイクプロフィット。
- **ストップ**: はい、直近の高値/安値で固定。
- **フィルター**: ボリンジャー/ケルトナースクイーズ、EMAトレンド、PSAR方向。
