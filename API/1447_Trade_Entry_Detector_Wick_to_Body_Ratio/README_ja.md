# トレードエントリー検出器、ヒゲ対実体比率戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はボリンジャーバンドと日足ローソク足のヒゲ対実体比率を使用してエントリーを検出します。ローソク足の下ヒゲが十分に大きく下限バンドを突き抜けた場合にロングポジションを開設します。反対の条件ではショートポジションを開設します。ポジションは反対側のバンドに達したとき、またはスウィングレベルが破られたときにクローズされます。
