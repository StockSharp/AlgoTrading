# Detector de Entrada de Operaciones, Estrategia de Ratio Mecha a Cuerpo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza las Bandas de Bollinger y el ratio mecha-cuerpo de las velas diarias para detectar entradas. Se abre una posición larga cuando la mecha inferior de una vela es lo suficientemente grande y perfora la banda inferior. Se abre una posición corta en la condición opuesta. Las posiciones se cierran en la banda opuesta o cuando se rompe el nivel de swing.
