# Handelseinstiegs-Detektor, Strategie Docht-zu-Körper-Verhältnis
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet Bollinger Bänder und das Docht-zu-Körper-Verhältnis von Tageskerzen zur Erkennung von Einstiegspunkten. Eine Long-Position wird eröffnet, wenn der untere Docht einer Kerze groß genug ist und das untere Band durchbricht. Eine Short-Position wird bei der entgegengesetzten Bedingung eröffnet. Positionen werden am gegenüberliegenden Band oder beim Bruch des Swing-Niveaus geschlossen.
