# Detector de Entrada de Negociação, Estratégia de Razão Pavio para Corpo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia usa as Bandas de Bollinger e a razão pavio-corpo dos candles diários para detectar entradas. Uma posição comprada é aberta quando o pavio inferior de um candle é grande o suficiente e perfura a banda inferior. Uma posição vendida é aberta na condição oposta. As posições são fechadas na banda oposta ou quando o nível de swing é rompido.
