Laguerre RSIに基づく戦略

テストでは年平均リターン約109%が示されています。暗号通貨市場で最もパフォーマンスが高くなります。

Laguerre RSIは標準RSIを平滑化してノイズを減らします。この戦略は、Laguerreの値が売られすぎゾーンから上にクロスしたときに買い、買われすぎゾーンから下にクロスしたときに売り、中間レベルに戻ったときにエグジットします。

Laguerreフィルタリングは、通常のRSIシグナルを悩ませる荒れた相場環境を避けるのに役立ちます。この手法は、マイナーな変動を無視しながらイントラデイチャートでの値動みを捉えるのに人気があります。


## 詳細

- **エントリー条件**: RSIに基づくシグナル。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆シグナルまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `Gamma` = 0.7m
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
