# Ivan CCI平均建て戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MetaTraderエキスパートアドバイザー「Ivan」のポートで、CCI極値を平均建てエントリーと平滑化移動平均ストップでトレードします。戦略はCCI(100)長期を監視してグローバルな買いまたは売りレジームを確立し、CCI(13)が戻る際に追加ポジションを重ねることができ、平滑化移動平均を中心としたブレークイーブンとトレーリングロジックでリスクを管理します。ポジションサイジングは元のパーセントリスクモデルを反映し、利益保護係数はエクイティが倍増した時にブックを閉じます。

## 詳細

- **エントリー条件**:
  - **グローバルロングシグナル**: 買いレジームがアクティブでない状態でCCI(100)が`GlobalSignalLevel`を上回る。ストップが少なくとも`MinStopDistance`価格より下にある場合、平滑化MA値を初期ストップとしてロング成行注文が送信されます。
  - **ロング平均建て**: `UseAveraging`が有効でグローバル買いフラグが設定されている場合、CCI(13)が`-GlobalSignalLevel`を下回るたびに同じストップテンプレートを使用してもう一つのロングが追加されます。
  - **グローバルショートシグナル**: 売りレジームがアクティブでない状態でCCI(100)が`-GlobalSignalLevel`を下回り、MAストップが少なくとも`MinStopDistance`価格より上にある場合にショートエントリーをトリガーします。
  - **ショート平均建て**: `UseAveraging`が有効な状態で、売りレジーム内でCCI(13)が`GlobalSignalLevel`を上回ると、ショートエクスポージャーに追加します。
- **ロング/ショート**: 両方向で取引し、アクティブなバイアス内でポジションをピラミッドにすることができます。
- **エグジット条件**:
  - CCI(100)での`±ReverseLevel`内への戻りクロスは両方のレジームをキャンセルし、フラットエクスポージャーを強制します。
  - ポートフォリオエクイティが開始残高の`ProfitProtectionFactor`倍を超えると、すべてのポジションの強制清算が発動されます。
  - 追跡されたストップ価格（ブレークイーブンまたはトレールされたMA）に到達するとポジションサイドが閉じられます。
- **ストップ**:
  - 初期ストップは`StopLossMaPeriod`の平滑化移動平均（SMMA）から取得されます。
  - ブレークイーブンはプライスが`BreakEvenDistance`進んだらストップをエントリー価格に移動します（ゼロに設定すると無効）。
  - トレーリングはMAが現在のストップを超えて少なくとも`TrailingStep`進んだ場合にのみストップを締めます。
- **フィルター**:
  - `UseZeroBar`はシグナル比較のために新たに開かれたバーまたは最後に閉じたバーを読み取るMT5オプションを複製します。
  - `MinStopDistance`はMAストップが価格に近すぎる場合のトレードを防止します。
- **ポジションサイジング**:
  - 各新規注文は現在のポートフォリオ価値の`RiskPercent`を価格とストップの距離で割ったものをリスクにし、`MinimumVolume`を安全フロアとします。

## パラメーター

- **Use Averaging** *(bool, デフォルト: true)* — アクティブなグローバルレジーム中に追加の平均建て注文を有効にする。
- **Stop MA Period** *(int, デフォルト: 36)* — ストップレベルの導出に使用される平滑化MAの期間。
- **Risk %** *(decimal, デフォルト: 10)* — 各新規トレードでリスクにさらすアカウントエクイティのパーセンテージ。
- **Use Zero Bar** *(bool, デフォルト: true)* — 真の場合、最新のローソク足値を使用する。そうでない場合、シグナルは前の閉じたバーに依存する。
- **Reverse Level** *(decimal, デフォルト: 100)* — 両方のレジームをキャンセルしてすべてのポジションを閉じる絶対CCI閾値。
- **Global Level** *(decimal, デフォルト: 100)* — 新しいグローバル買いまたは売りシグナルをアクティブにする絶対CCI閾値。
- **Min Stop Distance** *(decimal, デフォルト: 0.005)* — エントリーとMAストップ間の最小価格ギャップ（0.005 ≈ 5桁FXペアで50ピップ）。
- **Trailing Step** *(decimal, デフォルト: 0.001)* — MAトレーリングストップが進む前に必要な最小改善幅。
- **BreakEven Distance** *(decimal, デフォルト: 0.0005)* — ストップをエントリー価格にシフトするために必要な価格の動き。ブレークイーブンを無効にするには0に設定。
- **Profit Protection** *(decimal, デフォルト: 1.5)* — 利益を確定するために完全清算をトリガーするエクイティ倍数。
- **Minimum Volume** *(decimal, デフォルト: 1)* — リスクベースのサイジングが小さいまたはゼロのボリュームを生成した時のフォールバック取引サイズ。
- **Candle Type** *(DataType)* — インジケーターに使用されるローソク足シリーズ（デフォルト: 15分時間軸）。

## 注意事項

- `MinStopDistance`、`TrailingStep`、`BreakEvenDistance`などの距離は価格単位で表現され、インストゥルメントのティックサイズに合わせて調整する必要があります。
- 戦略は`BuyMarket`/`SellMarket`注文からの同期的な約定を前提としています。スリッページや部分約定が予想される場合は実行設定を調整してください。
- ポートフォリオベースのサイジングには接続されたポートフォリオアダプターが必要です。そうでない場合、`MinimumVolume`がすべての注文に使用されます。
