# Стратегия Color Step Xccx
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторе Color Step XCCX. Индикатор оценивает отклонение цены от сглаженного среднего и строит две ступенчатые линии. Лонг открывается, когда быстрая линия опускается ниже медленной. Шорт открывается, когда быстрая линия поднимается выше медленной.

## Детали

- **Условия входа**:
  - Лонг: быстрая линия пересекает медленную сверху вниз
  - Шорт: быстрая линия пересекает медленную снизу вверх
- **Направление**: оба
- **Условия выхода**:
  - Лонг: быстрая линия пересекает медленную снизу вверх
  - Шорт: быстрая линия пересекает медленную сверху вниз
- **Стопы**: отсутствуют
- **Значения по умолчанию**:
  - `DPeriod` = 30
  - `MPeriod` = 7
  - `StepSizeFast` = 5
  - `StepSizeSlow` = 30
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **Фильтры**:
  - Категория: трендовая
  - Направление: оба
  - Индикаторы: собственный, EMA
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
