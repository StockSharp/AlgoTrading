# Стратегия Hybrid EA
[English](README.md) | [中文](README_cn.md)

Стратегия Hybrid EA использует индекс относительной силы импульса (RVI) и его сигнальную линию.
Покупка выполняется, когда RVI превышает сигнал на заданную величину, а продажа — когда опускается ниже на ту же величину. Позиции защищаются фиксированными уровнями тейк-профита и стоп-лосса в ценовых пунктах.

## Детали

- **Критерий входа**: RVI минус сигнал превышает порог
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположное пересечение порога или тейк-профит/стоп-лосс
- **Стопы**: да, фиксированное расстояние в пунктах
- **Значения по умолчанию**:
  - `Volume` = 1
  - `RviLength` = 10
  - `SignalLength` = 4
  - `DifferenceThreshold` = 0.05
  - `TakeProfit` = 18
  - `StopLoss` = 9
  - `CandleType` = 5‑минутные свечи
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: обе стороны
  - Индикаторы: RVI, SMA
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
