# Стратегия Crunchster's Turtle and Trend System
[English](README.md) | [中文](README_cn.md)

Сочетает трендовый фильтр из быстрых и медленных EMA с входами по пробою канала Дончиана и стопами на основе ATR. Выйти из позиции помогает трейлинг по каналу Дончиана.

## Детали

- **Условия входа**: Пересечение дифференциала EMA или пробой канала Дончиана.
- **Направление**: Лонг и шорт.
- **Условия выхода**: Трейлинг-канал или стоп по ATR.
- **Стопы**: Да, на основе ATR.
- **Значения по умолчанию**:
  - `CandleType` = 1 час
  - `FastEmaPeriod` = 10
  - `BreakoutPeriod` = 20
  - `TrailPeriod` = 1000
  - `StopAtrMultiple` = 20
  - `OrderPercent` = 10
  - `TrendEnabled` = true
  - `BreakoutEnabled` = false
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг и шорт
  - Индикаторы: EMA, Дончиан, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
