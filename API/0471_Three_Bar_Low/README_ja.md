# 3バー安値戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

3バー安値戦略は、終値が直近 3 本のバーの最安値終値を下回ったときに買いを入れ、価格が直近 7 本のバーの最高値終値を上回って終値をつけたときに決済します。オプションの EMA フィルターを使うと、エントリー前に価格が長期平均を上回っていることを条件にできます。

## 詳細

- **エントリー条件**:
  - 終値が直近 3 本のバーの最安値終値を下回っている。
  - オプション: フィルター有効時に終値が EMA を上回っている。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - 終値が直近 7 本のバーの最高値終値を上回っている。
- **ストップ**: なし。
- **デフォルト値**:
  - `MaPeriod` = 200
  - `LowestLength` = 3
  - `HighestLength` = 7
  - `UseEmaFilter` = false
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: Long
  - インジケーター: EMA, Highest/Lowest
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
