# 3-Bar Low 策略
[English](README.md) | [Русский](README_ru.md)

3-Bar Low 策略在收盘价跌破前一根K线的三根最低收盘价时做多，并在收盘价突破前一根K线的七根最高收盘价时平仓。可选的 EMA 过滤器要求价格在进场前位于长期均线上方。

## 细节

- **入场条件**：
  - 收盘价低于前三根K线最低收盘价的前值。
  - 可选：开启过滤器时，收盘价需高于 EMA。
- **方向**：仅做多。
- **出场条件**：
  - 收盘价高于前七根K线最高收盘价的前值。
- **止损**：无。
- **默认参数**：
  - `MaPeriod` = 200
  - `LowestLength` = 3
  - `HighestLength` = 7
  - `UseEmaFilter` = false
- **过滤器**：
  - 类型：突破
  - 方向：多头
  - 指标：EMA, Highest/Lowest
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：低
