# Тренд по всплеску объёма
[English](README.md) | [中文](README_cn.md)

Данная стратегия отслеживает резкие скачки торгового объёма. Когда текущий объём превышает недавнее среднее в указанное количество раз, это говорит о сильном интересе участников.

Если объём резко растёт и цена выше скользящей средней, стратегия покупает; если объём растёт при цене ниже средней, открывается короткая позиция. Выход происходит, когда объём возвращается ниже среднего или срабатывает стоп-лосс.

Метод позволяет ловить движения, поддержанные всплеском активности.

## Подробности

- **Критерий входа**: изменение объёма больше `VolumeSpikeMultiplier` раз среднего.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: объём падает ниже среднего или стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `MAPeriod` = 20
  - `VolAvgPeriod` = 20
  - `VolumeSpikeMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе стороны
  - Индикаторы: Объём, MA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
