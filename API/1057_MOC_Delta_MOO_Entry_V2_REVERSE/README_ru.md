# Стратегия MOC Delta MOO Entry v2 Reverse
[English](README.md) | [中文](README_cn.md)

Эта стратегия является обратной версией MOC Delta MOO Entry. В период 14:50–14:55 она измеряет дельту объёма покупок и продаж и сохраняет её в процентах от дневного объёма. На следующий день в 08:30 открывается позиция в противоположном направлении, если дельта превышает порог, с фильтрацией двумя скользящими средними. Позиции закрываются по тейк-профиту, стоп-лоссу или в 14:50.

## Детали

- **Условия входа**:
  - **Лонг**: в 08:30 при сохранённой дельте ниже `-DeltaThreshold` и цене открытия выше SMA15 и SMA30, при этом SMA15 выше SMA30.
  - **Шорт**: в 08:30 при сохранённой дельте выше `DeltaThreshold` и цене открытия ниже SMA15 и SMA30, при этом SMA15 ниже SMA30.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - Тейк-профит и стоп-лосс в тиках.
  - Закрытие всех позиций в 14:50.
- **Стопы**:
  - `TpTicks` = 20 тиков тейк-профит.
  - `SlTicks` = 10 тиков стоп-лосс.
- **Значения по умолчанию**:
  - `DeltaThreshold` = 2
  - `TpTicks` = 20
  - `SlTicks` = 10
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **Фильтры**:
  - Категория: Объём
  - Направление: обе стороны
  - Индикаторы: SMA
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
