# Nifty 50 5mint Strategy
[English](README.md) | [中文](README_cn.md)

**Nifty 50 5mint Strategy** — стратегия пробоя для индекса Nifty 50, использующая подтверждение через DEMA, VWAP и полосы Боллинджера.

## Детали
- **Условия входа**:
  - **Long**: закрытие выше предыдущего максимума, выше верхней полосы Боллинджера и DEMA выше VWAP.
  - **Short**: закрытие ниже предыдущего минимума, ниже нижней полосы Боллинджера и DEMA ниже VWAP.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**: стоп-лосс.
- **Стопы**: да, фиксированные пункты.
- **Значения по умолчанию**:
  - `DemaPeriod = 6`
  - `BollingerLength = 20`
  - `BollingerStdDev = 2`
  - `LookbackPeriod = 5`
  - `StopLossPoints = 25`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе
  - Индикаторы: DEMA, VWAP, Bollinger Bands
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
