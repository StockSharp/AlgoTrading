# SmartAssTrade V2 Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия SmartAssTrade V2 использует гистограмму MACD и скользящие средние с периодом 20 на нескольких таймфреймах (1м, 5м, 15м, 30м, 60м) в сочетании с фильтрами Williams %R и RSI для фиксации тренда. Опциональный трейлинг-стоп защищает прибыль.

## Детали

- **Условие входа**: большинство таймфреймов показывают рост гистограммы MACD и MA при подтверждении WPR/RSI
- **Длинные/Короткие**: Оба
- **Условие выхода**: достижение тейк-профита или стоп-лосса; опциональный трейлинг-стоп
- **Стопы**: абсолютный стоп-лосс и тейк-профит с опцией трейлинга
- **Значения по умолчанию**:
  - `Volume` = 1
  - `TakeProfit` = 35
  - `StopLoss` = 62
  - `UseTrailingStop` = false
  - `TrailingStop` = 30
  - `TrailingStopStep` = 1
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: MACD, SMA, Williams %R, RSI
  - Стопы: Опционально
  - Сложность: Средняя
  - Таймфрейм: Мультитаймфрейм (1м,5м,15м,30м,60м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
