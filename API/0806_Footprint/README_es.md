# Estrategia Footprint
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia analiza el desequilibrio entre los volúmenes de compra y venta dentro de cada barra (footprint). Se abre una posición larga cuando el volumen de compra supera al de venta en un porcentaje configurable y la vela cierra por debajo de su apertura. Hay disponible un filtro de tendencia diaria opcional. El objetivo de ganancia y el stop loss se gestionan mediante protección integrada.
