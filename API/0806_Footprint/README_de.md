# Footprint-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie analysiert das Ungleichgewicht zwischen Kauf- und Verkaufsvolumen innerhalb jeder Kerze (Footprint). Eine Long-Position wird eröffnet, wenn das Kaufvolumen das Verkaufsvolumen um einen konfigurierbaren Prozentsatz übersteigt und die Kerze unterhalb ihrer Eröffnung schließt. Ein optionaler täglicher Trendfilter ist verfügbar. Kursziel und Stop-Loss werden über den integrierten Schutz verwaltet.
