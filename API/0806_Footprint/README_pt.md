# Estratégia Footprint
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia analisa o desequilíbrio entre os volumes de compra e venda dentro de cada barra (footprint). Uma posição comprada é aberta quando o volume de compra supera o de venda em uma porcentagem configurável e a vela fecha abaixo da abertura. Um filtro de tendência diária opcional está disponível. O alvo de lucro e o stop loss são gerenciados por meio de proteção integrada.
