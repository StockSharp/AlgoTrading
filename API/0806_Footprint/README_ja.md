# Footprint戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は各バー（フットプリント）内の買い出来高と売り出来高の不均衡を分析します。買い出来高が設定可能なパーセンテージだけ売り出来高を上回り、ローソク足が始値より低く終値を付けた場合にロングポジションを建てます。オプションの日足トレンドフィルターが利用可能です。利益目標とストップロスは組み込み保護機能で管理されます。
