# Стратегия DoubleUp2 CCI MACD
[English](README.md) | [中文](README_cn.md)

DoubleUp2 — мартингейловая стратегия, сочетающая индикаторы Commodity Channel Index (CCI) и MACD.
Шорты открываются при экстремально высоких значениях обоих индикаторов, лонги — при экстремально низких.
После убыточной сделки размер позиции удваивается для компенсации потерь.
Прибыльные сделки закрываются после движения цены на фиксированное количество пунктов.

## Детали

- **Условия входа**:
  - **Лонг**: `CCI < -Threshold` и `MACD < -Threshold`.
  - **Шорт**: `CCI > Threshold` и `MACD > Threshold`.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Противоположный сигнал или движение цены на `ExitDistance` пунктов в прибыль.
- **Стопы**: Нет явного стоп-лосса.
- **Значения по умолчанию**:
  - `CCI Period` = 8
  - `MACD Fast` = 13
  - `MACD Slow` = 33
  - `MACD Signal` = 2
  - `Threshold` = 230
  - `Base Volume` = 0.1
  - `ExitDistance` = `120 * price step`
- **Фильтры**:
  - Категория: Среднесрочный разворот
  - Направление: Оба
  - Индикаторы: CCI, MACD
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
