# TrendGuard-Flag-Finder-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

TrendGuard Flag Finder erkennt bullische und bärische Flaggenmuster, die durch SuperTrend bestätigt werden.
Kauft, wenn der Preis über eine bullische Flagge ausbricht, und verkauft, wenn er unter eine bärische Flagge fällt.

## Details

- **Einstiegskriterien**: Flaggenausbruch mit SuperTrend-Bestätigung
- **Long/Short**: Konfigurierbar
- **Ausstiegskriterien**: Entgegengesetzter Flaggenausbruch
- **Stops**: Nein
- **Standardwerte**:
  - `TradingDirection` = Both
  - `SuperTrend Length` = 10
  - `SuperTrend Factor` = 4
  - `MaxFlagDepth` = 5
  - `MinFlagLength` = 3
  - `MaxFlagLength` = 7
  - `MaxFlagRally` = 5
  - `MinBearFlagLength` = 3
  - `MaxBearFlagLength` = 7
  - `PoleMin` = 3
  - `PoleLength` = 7
  - `PoleMinBear` = 3
  - `PoleLengthBear` = 7
- **Filter**:
  - Kategorie: Muster
  - Richtung: Konfigurierbar
  - Indikatoren: SuperTrend, Lowest, Highest
  - Stops: Nein
  - Komplexität: Fortgeschritten
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
