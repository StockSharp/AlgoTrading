# Trailing Take Profit - Baseado no Fechamento
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia somente comprada entra no cruzamento da SMA rápida acima da SMA lenta. Um alvo de lucro fixo é definido a partir do preço de entrada. Quando o alvo é atingido, o stop pode seguir abaixo do fechamento mais alto para garantir os ganhos.

## Detalhes

- **Entrada**: SMA rápida cruza acima da SMA lenta.
- **Saída**: Cruzamento inverso ou take profit.
- **Indicadores**: SMA.
- **Período**: Qualquer.
- **Risco**: Trailing stop opcional.
