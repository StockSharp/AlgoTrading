# Trailing Take Profit - Schlusskurs-basiert
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese reine Long-Strategie steigt bei einem Kreuzungssignal der schnellen/langsamen SMA ein. Ab dem Einstiegspreis wird ein festes Gewinnziel gesetzt. Sobald das Ziel erreicht ist, kann der Stop unterhalb des höchsten Schlusskurses nachgezogen werden, um Gewinne zu sichern.

## Details

- **Einstieg**: Schnelle SMA kreuzt die langsame SMA von unten.
- **Ausstieg**: Umgekehrte Kreuzung oder Take-Profit.
- **Indikatoren**: SMA.
- **Zeitrahmen**: Beliebig.
- **Risiko**: Optionaler Trailing-Stop.
