# トレーリングテイクプロフィット - 終値ベース
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このロングのみ戦略は、速いSMAが遅いSMAを上抜けしたときにエントリーします。エントリー価格から固定の利益目標が設定されます。目標に達すると、ストップは最高終値の下を追従して利益を確定させます。

## 詳細

- **エントリー**: 速いSMAが遅いSMAを上抜け。
- **エグジット**: 逆クロスまたはテイクプロフィット。
- **インジケーター**: SMA。
- **時間軸**: 任意。
- **リスク**: オプションのトレーリングストップ。
