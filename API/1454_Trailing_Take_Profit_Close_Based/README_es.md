# Trailing Take Profit - Basado en Cierre
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia solo larga entra cuando la SMA rápida cruza por encima de la SMA lenta. Se establece un objetivo de ganancia fijo desde el precio de entrada. Cuando se alcanza el objetivo, el stop puede seguir por debajo del cierre más alto para asegurar las ganancias.

## Detalles

- **Entrada**: La SMA rápida cruza por encima de la SMA lenta.
- **Salida**: Cruce inverso o take profit.
- **Indicadores**: SMA.
- **Marco temporal**: Cualquiera.
- **Riesgo**: Trailing stop opcional.
