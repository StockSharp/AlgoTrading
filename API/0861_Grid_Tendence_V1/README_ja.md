# Grid Tendence V1戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

利益率のステップに基づいてポジションを再オープンまたは反転するグリッドトレーディング戦略です。

ロングで開始し、利益が指定されたパーセントに達すると決済して同じ方向で再オープンします。損失がそのパーセントに達すると決済して反対方向でオープンします。

## 詳細

- **エントリー条件**: 常にマーケットに参加し、ロングから開始。利益または損失が `Percent` に達したときに再オープンまたは反転。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 利益または損失のしきい値。
- **ストップ**: なし。
- **デフォルト値**:
  - `Percent` = 1.0
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: グリッド
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ (1m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
