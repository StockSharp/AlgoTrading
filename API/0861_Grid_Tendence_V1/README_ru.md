# Стратегия Grid Tendence V1
[English](README.md) | [中文](README_cn.md)

Стратегия сеточной торговли, которая переоткрывает или переворачивает позиции при достижении заданного процента прибыли или убытка.

Начинает с длинной позиции; когда прибыль достигает указанного процента, закрывает и снова открывает в том же направлении. При убытке в тот же процент закрывает и открывает в противоположном направлении.

## Детали

- **Критерии входа**: всегда в рынке, старт с лонга. Переоткрытие или разворот при достижении `Percent` по прибыли или убытку.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: порог прибыли или убытка.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Percent` = 1.0
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Grid
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
