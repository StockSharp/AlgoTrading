# Grid Tendence V1 策略
[English](README.md) | [Русский](README_ru.md)

基于固定利润百分比的网格交易策略。

策略从多头开始，当收益达到设定百分比时，平仓并在同方向重新开仓；当亏损达到该百分比时，平仓并在反方向开仓。

## 详情

- **入场条件**: 始终在市，初始为多头。利润或亏损达到 `Percent` 时重新开仓或反向开仓。
- **多空方向**: 双向。
- **退出条件**: 利润或亏损阈值。
- **止损**: 否。
- **默认值**:
  - `Percent` = 1.0
  - `CandleType` = TimeSpan.FromMinutes(1)
- **过滤器**:
  - 类别: Grid
  - 方向: 双向
  - 指标: 无
  - 止损: 否
  - 复杂度: 基础
  - 时间框架: 日内 (1m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
