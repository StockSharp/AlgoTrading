# イブニングスター・リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略はエキスパートアドバイザー **EveningStar.mq5**（MQL5 id 18507）の直接ポートです。クラシックなイブニングスターローソク足フォーメーションを監視し、次のバーが取引を開始するとすぐにポジションを建てます。ロジックはStockSharpの高レベルAPIをベースに書き直されており、オリジナルのリスク管理とパターンフィルターを維持しています。

## トレードロジック
1. 戦略は`CandleType`パラメーターで選択された時間軸にサブスクライブします。すべての処理は完成したローソク足のみで行われます。
2. 新しいローソク足が閉じるたびに最新のスナップショットがキャッシュされ、`Shift`で定義された3本ローソク足のウィンドウを評価できるようになります。
3. イブニングスターパターンは以下の場合に有効とみなされます：
   - ローソク足 *N-2*（最も古い）が強気（`open < close`）。
   - ローソク足 *N-1*（中央）が`Candle2Bullish`の設定を満たす（デフォルトは強気）。
   - ローソク足 *N*（最新）が弱気（`open > close`）。
   - `CheckCandleSizes`が有効な場合、中央のローソク足が3本の中で最も小さいボディを持つ必要がある。
   - `ConsiderGap`が有効な場合、元のロボットと同様にローソク足ボディ間にギャップが存在する必要がある（ギャップサイズは銘柄の価格刻みから計算した1pipsに等しい）。
4. パターンが確認されたら、戦略は`Direction`で選択された方向を確認します：
   - `Short`（デフォルト）は売り注文を開き、元のイブニングスターの動作に一致します。
   - `Long`はまったく逆のエクスポージャーを実行することを可能にします（MQLバージョンとの機能パリティのために維持）。
5. ポジションを開く前に、`CloseOppositePositions`が`true`に設定されている場合、アルゴリズムはオプションで反対のエクスポージャーを閉じます。
6. ストップロスとテイクプロフィットの価格は、pip距離（`StopLossPips`、`TakeProfitPips`）からMetaTraderにあったのと同じ3/5桁調整を使用して計算されます。
7. ポジションサイズは現在のポートフォリオ価値と`RiskPercent`から導出されます。計算されたボリュームが最小取引可能サイズより小さい場合、シグナルは無視されます。

## ポジション管理
- ロングポジションがアクティブな場合、戦略は各新しいローソク足を監視します。安値がストップレベルを下回るか、高値がテイクプロフィットレベルに達した場合、ポジション全体が成行で閉じられます。
- ショートポジションがアクティブな場合、逆の比較で同じロジックが適用されます。
- ポートフォリオ価値またはストップ距離がゼロの場合、注文サイズを計算できないため、エントリーはスキップされます。

## パラメーター
| 名前 | デフォルト値 | 説明 |
| ---- | ------- | ----------- |
| `Direction` | `Short` | パターンがロングかショートのポジションを建てるかを選択。 |
| `TakeProfitPips` | `150` | pipsで表したプロフィットターゲットまでの距離。無効にするにはゼロに設定。 |
| `StopLossPips` | `50` | pipsでの保護ストップまでの距離。非正の値はトレードを無効にする。 |
| `RiskPercent` | `5` | 1トレードあたりに許容するポートフォリオ資産の割合。注文ボリュームの計算に使用。 |
| `Shift` | `1` | パターン評価の前に最新のローソク足からスキップするバーの数。 |
| `ConsiderGap` | `true` | 元のエキスパートアドバイザーと同様にローソク足ボディ間のギャップを要求。 |
| `Candle2Bullish` | `true` | 中央のローソク足を強気にするよう強制。無効にすると弱気の中央ローソク足を要求。 |
| `CheckCandleSizes` | `true` | 中央のローソク足が最小の絶対ボディを持つことを確認。 |
| `CloseOppositePositions` | `true` | 新しい注文を送信する前に反対のエクスポージャーを閉じる。 |
| `CandleType` | `1H`時間軸 | 分析に使用するローソク足シリーズ。 |

## 注意事項
- pipサイズは銘柄の価格刻みから導出されます。3桁および5桁の外国為替シンボルでは、1pipは10価格刻みに相当し、元のEAの動作を再現します。
- `StopLossPips`がゼロの場合、ポジションサイズを計算できず、無制限のリスクを防ぐためにシグナルは無視されます。
- 戦略はキャッシュされた履歴を自動的に整理するため、長いセッションでもメモリ使用量は一定に保たれます。
