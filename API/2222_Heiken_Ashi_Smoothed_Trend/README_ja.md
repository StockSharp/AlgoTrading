# Heiken Ashi スムーズドトレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、EMAで平滑化されたHeiken-Ashiローソク足を使用してトレンドリバーサルを検出します。赤から緑に変わる強気ローソク足はロングポジションをオープンし、ショートポジションをクローズします。逆に、緑から赤に変わる弱気ローソク足はショートポジションをオープンし、ロングポジションをクローズします。

- **インジケーター**: Heikin-Ashi（EMA平滑化付き）
- **エントリーロール**:
  - 平滑化されたHeikin-Ashiローソク足が強気になったときにロングエントリー。
  - 平滑化されたローソク足が弱気になったときにショートエントリー。
- **エグジットロール**:
  - 逆シグナルでポジションをリバース。
- **パラメーター**:
  - `EmaLength` – EMAの平滑化期間。
  - `CandleType` – ローソク足の時間軸。

アルゴリズムは各完成したローソク足の平滑化された始値と終値を再計算し、それに応じてポジションを切り替えます。
