# Omega Galsky
[English](README.md) | [中文](README_cn.md)

Стратегия на пересечении EMA с переводом стопа в безубыток.

## Детали

- **Условия входа**: EMA8 пересекает EMA21 при подтверждении ценой относительно EMA89.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Стоп-лосс, тейк-профит или противоположный сигнал.
- **Стопы**: Да.
- **Параметры по умолчанию**:
  - `Ema8Period` = 8
  - `Ema21Period` = 21
  - `Ema89Period` = 89
  - `FixedRiskReward` = 1.0m
  - `SlPercentage` = 0.001m
  - `TpPercentage` = 0.0025m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
