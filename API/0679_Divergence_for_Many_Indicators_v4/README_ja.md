# 複数インジケーターのダイバージェンス戦略 v4
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は価格と複数のモメンタムインジケーター（MACD、RSI、Stochastic、CCI、Momentum、OBV、MFI）の間のダイバージェンスを検出します。
少なくとも指定数のインジケーターが同方向のダイバージェンスを示したときにポジションを建てます。

## 詳細
- **エントリー条件**: 価格が下落しているのに多数のインジケーターが上昇しているとき（ポジティブダイバージェンス）にロングエントリー。価格が上昇しているのに多数のインジケーターが下落しているとき（ネガティブダイバージェンス）にショートエントリー。
- **ロング/ショート**: 両方
- **エグジット条件**: 逆方向のダイバージェンスまたはポジション保護
- **ストップ**: 設定可能なテイクプロフィットおよびストップロスのパーセンテージ
- **デフォルト値**: 5分足ローソク、2つの確認、4%テイクプロフィット、2%ストップロス
- **フィルター**: 確認のために複数のモメンタムインジケーターを使用
