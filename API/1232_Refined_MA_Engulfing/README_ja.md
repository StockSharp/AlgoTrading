# 洗練されたMA + エンゴルフィング戦略 (M5 + 確認済み構造ブレイク)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

洗練されたMA + エンゴルフィングは、2本の単純移動平均、エンゴルフィングローソク足、および構造ブレイク確認を組み合わせています。少なくとも2つのコンフルエンス要因が一致し、クールダウン期間が過ぎたときにトレードが実行されます。

## 詳細

- **エントリー条件**: 確認済みの強気または弱気の構造ブレイクの後、価格が両方のSMAの上または下にあり、4つのコンフルエンス（エンゴルフィング、構造ブレイク、MAフィルター、フィブプレースホルダー）のうち少なくとも2つが満たされ、クールダウンが完了していること。
- **ロング/ショート**: 両方。
- **エグジット条件**: なし。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `Ma1Length` = 66
  - `Ma2Length` = 85
  - `CooldownBars` = 5
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: SMA, Engulfing, Structure Break
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 5-minute
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
