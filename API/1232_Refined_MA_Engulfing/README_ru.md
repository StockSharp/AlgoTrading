# Уточнённые MA + Поглощение (M5 + Подтверждённый пробой структуры)
[English](README.md) | [中文](README_cn.md)

Уточнённые MA + Поглощение объединяет две простые скользящие средние, свечи поглощения и подтверждение пробоя структуры. Сделка совершается, когда совпадают как минимум два фактора конфлюэнции и выдержана задержка.

## Детали

- **Условия входа**: После подтверждённого бычьего или медвежьего пробоя структуры, цена выше или ниже обеих SMA и выполнены как минимум два из четырёх условий (поглощение, пробой структуры, фильтр SMA, заглушка Fib) с соблюдением паузы.
- **Направление**: Лонг и шорт.
- **Условия выхода**: Нет.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Ma1Length` = 66
  - `Ma2Length` = 85
  - `CooldownBars` = 5
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Паттерны
  - Направление: Оба
  - Индикаторы: SMA, Поглощение, Пробой структуры
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: 5 минут
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
