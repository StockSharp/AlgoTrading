# SAR自動化戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この例は **Parabolic SAR** インジケーターに基づくシンプルなトレードアプローチを示しています。
戦略は現在価格がSAR値を上回るとロングポジションを建て、価格がSARを下回るとショートポジションを建てます。追加のリスク管理機能には固定ストップロス、テイクプロフィット、およびオプションのトレーリングストップが含まれます。

## パラメーター
- `SarStep` – SAR計算の加速係数。
- `SarMax` – SARの最大加速係数。
- `StopLoss` – 価格単位でのストップロス距離。
- `TakeProfit` – 価格単位でのテイクプロフィット距離。
- `TrailingStop` – 価格単位でのトレーリングストップ距離。
- `CandleType` – インジケーター計算に使用するローソク足の種類。

## トレードロジック
1. ローソク足を購読し、Parabolic SAR値を計算します。
2. **エントリー**:
   - SARが終値を下回り、ポジションがない場合にロングエントリー。
   - SARが終値を上回り、ポジションがない場合にショートエントリー。
3. **エグジット**:
   - 価格が反対のSARレベルに達したときにポジションをクローズ。
   - ストップロス、テイクプロフィット、トレーリングストップのルールを適用。

この戦略は教育目的のものであり、StockSharpの高水準APIでインジケーターとリスクコントロールを使用する方法を示しています。
