# 機械的トレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

毎日指定された時刻に1回の取引を実行する時間ベースの機械的戦略です。ポジション方向はロングまたはショートに設定できます。取引はパーセントベースのテイクプロフィットとストップロスで自動的に保護されます。

## 詳細

- **エントリー条件**:
  - **ロング**: `Short Mode` が無効のときに `TradeHour` で。
  - **ショート**: `Short Mode` が有効のときに `TradeHour` で。
- **ロング/ショート**: `Short Mode` に応じて両方。
- **エグジット条件**:
  - エントリーから `Profit Target (%)` 上/下。
  - エントリーから `Stop Loss (%)` 下/上。
- **ストップ**: ストップロスとテイクプロフィットの両方。
- **デフォルト値**:
  - `Profit Target (%)` = 0.4.
  - `Stop Loss (%)` = 0.2.
  - `Trade Hour` = 16.
- **フィルター**:
  - カテゴリ: 時間
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
