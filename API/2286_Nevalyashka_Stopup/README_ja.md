# Nevalyashka Stopup戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は各取引の後にポジション方向を交互に切り替え、横に倒れてもすぐ起き上がる「ネバリャシュカ」玩具を模倣しています。マルチンゲールアプローチを使用します：取引が損失で決済された場合、次の取引のストップロスとテイクプロフィットの距離はコeficienteで乗算されます。利益取引の後、距離はベース値にリセットされ、戦略はオプションで取引を停止できます。

初期方向はショートです。ポジションが決済されるたびに、新しいポジションは事前設定されたボリュームで反対方向に開かれます。

## 詳細

- **エントリー条件**：
  - 最初の取引は成行で売ります。
  - 続く取引は常に直前の決済取引とは反対方向にエントリーします。
- **ロング/ショート**：両方。
- **エグジット条件**：
  - 価格がエントリーからテイクプロフィットまたはストップロスの距離に達するとポジションは決済されます。
- **ストップ**：はい、ポイントでの固定ストップロスとテイクプロフィット。損失後は距離がマルチンゲール係数で増加します。
- **デフォルト値**：
  - `StopLossPoints` = 150
  - `TakeProfitPoints` = 50
  - `OrderVolume` = 0.1
  - `MartingaleCoeff` = 1.5
  - `StopAfterProfit` = false
- **フィルター**：
  - カテゴリ：リバーサル / マルチンゲール
  - 方向：両方
  - インジケーター：なし
  - ストップ：はい
  - 複雑さ：シンプル
  - 時間軸：任意
  - 季節性：いいえ
  - ニューラルネットワーク：いいえ
  - ダイバージェンス：いいえ
  - リスクレベル：高
