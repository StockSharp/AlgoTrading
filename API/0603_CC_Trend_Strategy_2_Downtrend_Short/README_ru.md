# Стратегия CC Trend 2 Downtrend Short
[English](README.md) | [中文](README_cn.md)

Стратегия только для коротких позиций. Продаёт, когда предыдущая цена закрытия ниже динамического уровня Фибоначчи и EMA21 ниже EMA55. Выходит при пересечении ценой EMA200 при неотрицательной прибыли или когда предыдущая цена закрытия поднимается выше уровня Фибоначчи 0.236 и нет нового сигнала на вход.

## Детали

- **Условия входа**:
  - Short: предыдущая цена закрытия ниже максимума Фибоначчи и EMA21 ниже EMA55
- **Направление**: Short
- **Условия выхода**:
  - Цена пересекает EMA200 при прибыли
  - Предыдущая цена закрытия выше уровня Фибоначчи 0.236 без нового сигнала
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `FibLength` = 100
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Short
  - Индикаторы: EMA, Фибоначчи
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
