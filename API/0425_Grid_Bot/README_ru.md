# Стратегия Grid Bot
[English](README.md) | [中文](README_cn.md)

Бот делит заранее заданный ценовой диапазон на равные уровни и торгует колебания между ними. Когда цена смещается к нижней половине сетки, стратегия накапливает длинные позиции и распродаёт их по возвращении к верхней части. Подход эффективен на боковых рынках с чёткими границами.

Направленного уклона нет; бот просто реагирует на близость к уровням.

## Детали

- **Критерии входа**:
  - **Длинная позиция**: цена касается уровня в нижней половине, позиции нет
  - **Короткая позиция**: цена касается уровня в верхней половине, позиции нет
- **Длинные/короткие**: обе стороны
- **Критерии выхода**:
  - Противоположный сигнал закрывает текущую позицию
- **Стопы**: нет
- **Значения по умолчанию**:
  - `UpperLimit` = 48000
  - `LowerLimit` = 45000
  - `GridCount` = 10
- **Фильтры**:
  - Категория: Рейндж-трейдинг
  - Направление: Оба
  - Индикаторы: Ценовые уровни
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
