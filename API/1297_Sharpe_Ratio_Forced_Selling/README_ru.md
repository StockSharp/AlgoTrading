# Стратегия принудительных продаж по коэффициенту Шарпа
[English](README.md) | [中文](README_zh.md)

Стратегия Sharpe Ratio Forced Selling открывает длинную позицию, когда скользящий коэффициент Шарпа падает ниже отрицательного порога, и закрывает её, когда он растёт выше положительного значения или срок удержания превышает лимит. Доходность может рассчитываться логарифмически или в процентах с учётом безрисковой ставки.

## Детали

- **Условия входа**: Коэффициент Шарпа ниже `EntrySharpeThreshold`.
- **Длинные/Короткие**: Только длинные.
- **Условия выхода**: Коэффициент Шарпа выше `ExitSharpeThreshold` или превышен `MaxHoldingDays`.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Length` = 8
  - `EntrySharpeThreshold` = -5
  - `ExitSharpeThreshold` = 13
  - `MaxHoldingDays` = 80
  - `UseLogReturns` = true
  - `RiskFreeRateAnnual` = 0
  - `PeriodsPerYear` = 252
- **Фильтры**:
  - Категория: Средневозвратная
  - Направление: Long
  - Индикаторы: Sharpe Ratio
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
