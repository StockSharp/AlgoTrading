# QQQ戦略 v2 ESL easy-peasy-x
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はメイン移動平均のクロスオーバーとトレンドフィルターを使ってQQQを取引します。終値がメインMAを上抜け、MAが上昇中で価格が長期トレンドMAより上にある場合に買いエントリーします。終値がメインMAを下抜け、MAが下落中で価格が短期トレンドMAより下にある場合にショートエントリーします。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値がメインMAを上抜け、MAの傾きが上昇、価格が長期トレンドMAより上。
  - **ショート**: 終値がメインMAを下抜け、MAの傾きが下降、価格が短期トレンドMAより下。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対のシグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `Main MA Length` = 200
  - `Trend Long Length` = 100
  - `Trend Short Length` = 50
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 移動平均線
  - ストップ: なし
  - 複雑さ: 中程度
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
