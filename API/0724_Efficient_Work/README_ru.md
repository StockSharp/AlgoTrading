# Стратегия Efficient Work
[English](README.md) | [中文](README_cn.md)

Стратегия использует скользящие средние на коротком, среднем и длинном горизонтах. Длинная позиция открывается, когда быстрая средняя выше обеих длинных, короткая — когда она ниже их.

## Детали

- **Условия входа**:
  - **Long**: `fast MA > medium MA` и `fast MA > high MA`.
  - **Short**: `fast MA < medium MA` и `fast MA < high MA`.
- **Long/Short**: обе стороны.
- **Условия выхода**:
  - Противоположный сигнал приводит к развороту.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `MA Period` = 20
  - `Medium TF Multiplier` = 5
  - `High TF Multiplier` = 10
- **Фильтры**:
  - Категория: следование тренду
  - Направление: обе
  - Индикаторы: множественные
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
