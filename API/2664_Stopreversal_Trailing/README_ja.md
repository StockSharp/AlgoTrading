# Stopreversal トレーリング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Stopreversal トレーリング戦略は、MT5エキスパート`Exp_Stopreversal.mq5`を再現したものです。Stopreversal カスタムインジケーターを使用して、選択したローソク足価格の周辺にダイナミックなトレーリングストップラインを構築します。価格がこのトレーリングラインを上方向に突き抜けると、戦略はこれを強気のリバーサルとして扱い、オプションでショートのエクスポージャーを閉じ、新しいロングポジションを建てます。下方向への突き抜けは対称的な弱気アクションを生成します。シグナルは、元のエキスパートアドバイザーの動作に合わせるために、設定可能な数の閉じたバーだけ遅延させることができます。

## 詳細

- **エントリーロジック**: 価格がアダプティブトレーリングストップを越えたときに生成されるStoprevesalインジケーターの矢印に反応します。
- **ロング/ショート**: 両方向がサポートされており、ロングまたはショートエントリーを有効にする独立したトグルがあります。
- **エグジットロジック**: 逆方向のStoprevesalシグナルが既存のポジションを閉じることができます; 保護的なストップロスとテイクプロフィットレベルも利用可能です。
- **ストップ**: 価格ステップでの静的ストップロスとテイクプロフィット、プラスインジケーター駆動のリバーサル。
- **データソース**: 任意の時間軸; デフォルトは4時間時間軸ローソク足を使用し、元のエキスパートのマルチタイムフレームコールを反映します。
- **シグナル遅延**: `SignalBar`パラメーターは、指定した数の完成したバーだけ注文実行を遅延させます（デフォルト1バー）。
- **リスク管理**: オプションのハードストップは証券価格ステップで表現されます; ポジション保護サービスは開始時に有効化されます。
- **インジケーターパラメーター**: トレーリングオフセット`Npips`は価格とストップの間の距離を制御します; `PriceMode`はトレーリングストップが使用するローソク足価格を選択します。
- **デフォルト値**:
  - `Volume` = 1
  - `StopLossSteps` = 1000
  - `TakeProfitSteps` = 2000
  - `BuyPositionOpen` = true
  - `SellPositionOpen` = true
  - `BuyPositionClose` = true
  - `SellPositionClose` = true
  - `Npips` = 0.004
  - `PriceMode` = Close
  - `SignalBar` = 1

## パラメーター

| パラメーター | 説明 |
|------------|------|
| `CandleType` | Stopreversal計算と取引の両方に使用するローソク足サブスクリプション。デフォルトは4時間時間軸。 |
| `Volume` | 新しいポジションに入る際に送信する基本注文サイズ。 |
| `StopLossSteps` | エントリーからストップロスまでの距離（価格ステップ単位）; 0に設定して無効化。 |
| `TakeProfitSteps` | エントリーからテイクプロフィットまでの距離（価格ステップ単位）; 0に設定して無効化。 |
| `BuyPositionOpen` | 強気シグナル発生時のロングポジション開設を有効化します。 |
| `SellPositionOpen` | 弱気シグナル発生時のショートポジション開設を有効化します。 |
| `BuyPositionClose` | 弱気シグナルを受信したとき、既存のロングポジションを閉じます。 |
| `SellPositionClose` | 強気シグナルを受信したとき、既存のショートポジションを閉じます。 |
| `Npips` | リバーサル距離を広げたり狭めたりするためにトレーリングストップに適用する比率乗数。 |
| `PriceMode` | 適用される価格バリアント（終値、始値、高値、安値、中央値、典型値、加重値、単純平均、四価平均、トレンドフォロー、またはDemark）。 |
| `SignalBar` | シグナルに反応する前に待機する完全に閉じたローソク足の数（MT5パラメーターに対応）。 |

## フィルター

- **カテゴリ**: トレンドフォローのリバーサル
- **方向**: 双方向
- **インジケーター**: Stopreversal（ATRバックのトレーリングストップ）
- **ストップ**: 静的ストップロスとテイクプロフィット、オプション
- **時間軸**: 設定可能（デフォルトH4）
- **季節性**: なし
- **ニューラルネットワーク**: いいえ
- **ダイバージェンス**: いいえ
- **複雑さ**: カスタムトレーリングロジックにより中程度
- **リスクレベル**: ストップ距離とトレーリングオフセットで調整可能
