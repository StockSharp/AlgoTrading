# Three White Soldiers 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

三白兵パターンは、3本連続した強い陽線で構成されるクラシックな強気反転です。下落トレンドの後、この連続は買い圧力が売り手を圧倒することで持続的な上昇の始まりを示すことが多いです。

テストでは平均年間リターンが約175%であることが示されています。株式市場で最もパフォーマンスが優れています。

戦略は3本目のソルジャーが形成されたらロングエントリーし、モメンタムの上昇からの継続を期待します。このセットアップは純粋に強気なのでショート取引は行いませんが、他の方法で開始されたショートポジションをクローズすることは許可されています。

偽シグナルから守るためにパターンのすぐ下にストップを置き、価格がそのレベルを下回って引けた場合にポジションを終了します。

## 詳細

- **エントリー条件**: パターン一致
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15分
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
