# Стратегия «Три белых солдата»
[English](README.md) | [中文](README_zh.md)

Паттерн «Три белых солдата» — классический бычий разворот, состоящий из трёх последовательных сильных свеч роста. После нисходящего тренда такая последовательность часто знаменует начало устойчивого движения вверх, поскольку покупатели подавляют продавцов.

Стратегия открывает длинную позицию после формирования третьего солдата, ожидая продолжения импульса. Короткие сделки не открываются, потому что сетап чисто бычий, однако система позволяет закрывать короткие позиции, открытые другими методами.

Стопы размещаются немного ниже паттерна, чтобы защититься от ложных сигналов, а позиции закрываются, если цена закрывается ниже этого уровня.

## Детали

- **Условия входа**: совпадение паттерна
- **Лонг/шорт**: оба направления
- **Условия выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, на процентной основе
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
