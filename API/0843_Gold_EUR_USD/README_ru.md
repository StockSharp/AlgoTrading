# Стратегия захвата ликвидности для золота и EUR/USD
[English](README.md) | [中文](README_cn.md)

Стратегия ищет выбросы ликвидности у зон спроса и предложения по золоту и EUR/USD, используя RSI, SMA, стохастический осциллятор и разрывы справедливой стоимости по ATR.

## Детали

- **Условия входа**:
  - **Long**: Цена делает хвост ниже ближайшего минимума, структура рынка смещается вверх, присутствует FVG, RSI в перепроданности, цена выше SMA, стохастик в перепроданности.
  - **Short**: Цена делает хвост выше ближайшего максимума, структура рынка смещается вниз, присутствует FVG, RSI в перекупленности, цена ниже SMA, стохастик в перекупленности.
- **Long/Short**: Обе стороны.
- **Выход**: Обратный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `MaLength` = 50
  - `StochLength` = 14
  - `Overbought` = 70
  - `Oversold` = 30
  - `StochOverbought` = 80
  - `StochOversold` = 20
- **Фильтры**:
  - Категория: Прайс-экшен
  - Направление: Оба
  - Индикаторы: RSI, SMA, Stochastic, ATR, Highest, Lowest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
