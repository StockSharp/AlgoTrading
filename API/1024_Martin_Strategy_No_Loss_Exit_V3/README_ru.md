# Стратегия Martin - No Loss Exit V3
[English](README.md) | [中文](README_cn.md)

Эта мартингейловая стратегия усреднения добавляет к длинной позиции при падении цены на заданный процент от первой покупки. Каждое новое добавление увеличивает вложенную сумму по коэффициенту и пересчитывает среднюю цену. Позиция закрывается, когда максимум свечи достигает средней цены, увеличенной на процент тейк-профита, что позволяет выходить только с прибылью.

## Детали

- **Условия входа**:
  - **Long**: `Нет позиции` → покупка на сумму `Initial Cash`
  - **Добор**: `Price <= EntryPrice * (1 - PriceStep% * orderCount)` && `orderCount < MaxOrders`
- **Длинные/Короткие**: Только длинные
- **Условия выхода**:
  - `High >= AvgPrice * (1 + TakeProfit%)`
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Initial Cash` = 100
  - `Max Orders` = 20
  - `Price Step %` = 1.5
  - `Take Profit %` = 1
  - `Increase Factor` = 1.05
- **Фильтры**:
  - Категория: Усреднение вниз
  - Направление: Long
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
