# TDS Global戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Alexander ElderのTriple Screenコンセプトに基づくMetaTraderオリジナルの「TDSGlobal」エキスパートを再現します。日足ローソク足を評価し、MACD（12、23、9）の傾きと24期間のWilliams %Rフィルターを組み合わせます。システムはトレンドが上向きに転じ、%Rが売られすぎの状態を示しているときに買いを求め、トレンドが下向きに転じ、%Rが買われすぎを示しているときに売りを求めます。

有効なセットアップが検出されるたびに、戦略は前のセッションの高値または安値を超えてストップ注文を置きます。エントリーは、元の「16ポイント」オフセットロジックを反映して、価格に近すぎるエントリーを避けるために設定可能なバッファによって現在の市場から遠ざけられます。ポジションを取ると、戦略は保護ストップ、オプションのテイクプロフィット、および価格ステップでのトレーリングストップを管理します。

## トレードロジック

- **データ**: デフォルトで日足ローソク足で動作します（設定可能）。
- **トレンドフィルター**: 最近の2つのMACD主線値を比較します。MACDの上昇はロングバイアスを意味し、MACDの下降はショートバイアスを意味します。
- **オシレーターフィルター**: 前のWilliams %R値を使用します。`WilliamsBuyLevel`（デフォルト-75）以下ではロングセットアップを許可し、`WilliamsSellLevel`（デフォルト-25）以上ではショートセットアップを許可します。
- **エントリー**:
  - ロング: 前の高値プラス1価格ステップを超えてbuy-stopを置きます。エントリーは市場との最小距離を保つために最後の終値より少なくとも`EntryBufferSteps`価格ステップ上に引き上げられます。
  - ショート: 前の安値マイナス1価格ステップ未満にsell-stopを置きます。注文は最大で最後の終値マイナス`EntryBufferSteps`ステップまで引き下げられます。
- **リスク管理**:
  - 初期ストップは前のローソク足の反対側の極値に固定されます（ショートには高値、ロングには安値）。
  - テイクプロフィット距離は`TakeProfitSteps`価格ステップに等しくなります。デフォルト値（999）は非常に広いターゲットを使用したMQLバージョンに近い動作を保持します。
  - トレーリングストップは`TrailingStopSteps` > 0のときに有効になります。それだけのステップ数で終値に追随し、取引の方向にのみ締まります。
- **注文処理**:
  - エントリー価格または保護レベルを更新する必要がある場合、既存のストップ注文はキャンセルされ更新されます。
  - 反対方向のトレンドシグナルは、MACDの方向と一致しなくなったペンディング注文を削除します。
  - ポジションが開かれると、保存されたペンディングレベルはライブのストップ/テイク価格を初期化するために再利用されます。
- **オプションの段階的配置**: 元のEAは、同時ペンディング注文を避けるためにFXペア間で注文配置を段階的にしていました。`UseSymbolStagger`を`true`に設定すると、EURUSD、GBPUSD、USDCHF、USDJPYに同じ分ウィンドウが適用されます。

## パラメーター

- `MacdFastLength`、`MacdSlowLength`、`MacdSignalLength` – トレンド傾斜チェックに使用されるMACDの期間。
- `WilliamsLength` – Williams %Rのルックバック。
- `WilliamsBuyLevel`、`WilliamsSellLevel` – 売られすぎ/買われすぎのしきい値（負の値、それぞれ-100/-0に近い）。
- `EntryBufferSteps` – ストップエントリーを置く際の現在の市場からの最小オフセット（価格ステップ数）。
- `TakeProfitSteps` – 価格ステップでのターゲット距離（ハードターゲットを有効にするには小さい数を設定）。
- `TrailingStopSteps` – ステップ単位のトレーリングストップ距離; トレーリングを無効にするにはゼロに設定。
- `UseSymbolStagger` – シンボル固有の分ウィンドウを有効にします。
- `CandleType` – ローソク足の時間軸（デフォルトは日足）。

## 注意事項

- ロットサイズを制御するために戦略ボリュームを使用します。ボリュームが指定されていない場合はデフォルトで1になります。
- ペンディング注文とトレーリングエグジットは完成したローソク足で動作するため、ローソク足のクローズ間の約定は保存されたエントリー価格で近似されます。
- テイクプロフィットのデフォルト値は元のEAの動作に合わせるために大きくなっています。有限のターゲットが必要な場合は調整してください。
