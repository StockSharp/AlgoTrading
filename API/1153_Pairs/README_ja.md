# ペア取引戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このペア取引戦略は、参照資産が始値より高く引けたときに、現在のシンボルが下降ローソク足を形成している場合に買いを入れます。前のローソク足の高値を上回って引けたらポジションを決済します。

## 詳細

- **エントリー条件**: 参照資産が上昇し、現在のシンボルが下降ローソク足を形成。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 前回の高値を上回って引ける。
- **ストップ**: なし。
- **デフォルト値**:
  - `CandleType` = 1分
- **フィルター**:
  - カテゴリ: ペア取引
  - 方向: ロングのみ
  - インジケーター: Price action
  - ストップ: なし
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
  - 季節性: なし
  - ニューラルネットワーク: なし
  - ダイバージェンス: なし
  - リスクレベル: 中
