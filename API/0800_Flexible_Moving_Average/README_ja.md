# フレキシブル移動平均戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

前の期間の終値と設定可能な移動平均のクロスオーバーに基づいてポジションを調整します。下向きのクロスではユーザー定義の割合でポジションを縮小し、上向きのクロスではフルポジションに戻します。

## 詳細

- **エントリー条件**:
  - **初回**: 最初のバーでオプションのフルロング。
  - **増加**: 前の終値が移動平均を上抜け → ポジションを100%へ。
- **エグジット条件**:
  - **縮小**: 前の終値が移動平均を下抜け → `SellPercentage` だけ縮小。
- **インジケーター**:
  - 単純、指数、加重、Hull、または平滑移動平均。
- **ストップ**: なし。
- **デフォルト値**:
  - `MaLength` = 200
  - `SellPercentage` = 100
  - `MaMethod` = SMA
  - `AllowInitialBuy` = true
- **フィルター**:
  - トレンドフォロー
  - 単一時間軸
  - インジケーター: 移動平均
  - ストップ: なし
  - 複雑さ: 基本
