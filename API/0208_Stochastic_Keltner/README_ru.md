# Стратегия Stochastic Keltner

Стратегия использует стохастический осциллятор и канал Келтнера. Длинная позиция берётся, когда %K ниже 20 и цена у нижней полосы, что говорит о перепроданности. Короткая позиция берётся, когда %K выше 80 и цена у верхней полосы, т.е. перекупленность. Подходит для смешанных рынков.

## Подробности
- **Условия входа**:
  - **Long**: Stoch %K < 20 && Price < Keltner lower band
  - **Short**: Stoch %K > 80 && Price > Keltner upper band
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход при возврате цены к средней полосе
  - **Short**: выход при возврате цены к средней полосе
- **Стопы**: да
- **Значения по умолчанию**:
  - `StochPeriod` = 14
  - `StochK` = 3
  - `StochD` = 3
  - `EmaPeriod` = 20
  - `KeltnerMultiplier` = 2m
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: Stochastic Keltner
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
