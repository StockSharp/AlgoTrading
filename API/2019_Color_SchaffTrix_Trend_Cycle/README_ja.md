# Color Schaff TRIX トレンドサイクル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、TRIXベースのMACDから計算される**Schaff Trend Cycle**オシレーターを実装しています。オシレーターは周期的なトレンド転換を識別し、サイクルが事前定義されたレベルを超えた際にトレーディングシグナルを生成します。

## 仕組み

1. 異なる長さの2つのTRIXオシレーターを計算してMACDシリーズを構築します。
2. MACD値は二重確率変換によって処理され、Schaff Trend Cycle (STC) を生成します。
3. STCが高レベルを上抜けするとロングポジションを開き、低レベルを下抜けするとショートポジションを開きます。
4. 反対方向のクロスが発生すると、既存のポジションを閉じます。

## パラメーター

- **Fast TRIX** – 高速TRIXオシレーターの長さ。
- **Slow TRIX** – 低速TRIXオシレーターの長さ。
- **Cycle** – 確率計算に使用する期間。
- **High Level / Low Level** – STCの上限・下限しきい値。
- **Stop Loss % / Take Profit %** – パーセンテージで表されるリスク管理パラメーター。
- **Buy/Sell Open/Close** – 対応する操作を有効または無効にします。

## 注意事項

この戦略は選択した時間軸（デフォルト4時間）のローソク足データを使用し、成行注文を執行します。ストップロスとテイクプロフィットの両方の値で保護が有効になっています。すべてのインジケーター処理は、自動バインディングを備えた高レベルAPIを使用して実行されます。
