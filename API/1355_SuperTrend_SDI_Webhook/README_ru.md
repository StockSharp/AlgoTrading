# Стратегия SuperTrend SDI Webhook
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает индикатор SuperTrend и сглаженный направленный индекс (SDI). Открывает длинные позиции, когда +DI выше -DI и SuperTrend показывает восходящий тренд. Открывает короткие позиции, когда -DI выше +DI и SuperTrend указывает на нисходящий тренд. Использует процентные уровни тейк‑профита, стоп‑лосса и трейлинг‑стопа.

## Подробности

- **Условия входа**:
  - Лонг: `+DI > -DI && SuperTrend вверх`
  - Шорт: `-DI > +DI && SuperTrend вниз`
- **Тип**: Лонг и шорт
- **Условия выхода**: Тейк‑профит, стоп‑лосс или трейлинг‑стоп
- **Индикаторы**: SuperTrend, AverageDirectionalIndex
- **Стопы**: Процентные
- **Значения по умолчанию**:
  - `AtrPeriod` = 10
  - `AtrMultiplier` = 1.8
  - `DiLength` = 3
  - `DiSmooth` = 7
  - `TakeProfitPercent` = 25
  - `StopLossPercent` = 4.8
  - `TrailingPercent` = 1.9
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: SuperTrend, SDI
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
