# TradePad 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradePad 戦略は、元のMQL TradePadエキスパートから移植された手動トレーディングパネルです。この戦略はトレードを対話的に管理するパネルをセットアップします。自動的なエントリーやエグジットのルールなしに、ティックデータ、トレード通知、タイマーイベント、チャートメッセージを処理します。

このサンプルは、StockSharp上で裁量トレードインターフェースを構築する方法を示しています。

## 詳細

- **エントリー条件**: パネルを通じた手動注文発注。
- **ロング/ショート**: 両方、ユーザーの操作による。
- **エグジット条件**: 手動によるポジションクローズ。
- **ストップ**: なし。ユーザーはカスタムロジックを実装できる。
- **フィルター**: 自動フィルターなし。
