# TradePad 策略
[English](README.md) | [Русский](README_ru.md)

TradePad 策略是从原始 MQL TradePad 专家顾问移植的手动交易面板。该策略设置面板以交互方式管理交易，处理 tick 数据、交易通知、定时器事件和图表消息，没有任何自动进出场规则。

此示例展示了如何在 StockSharp 框架上构建一个自主决策的交易界面。

## 细节

- **入场条件**：通过面板手动下单。
- **方向**：双向，取决于用户操作。
- **出场条件**：通过面板手动平仓。
- **止损**：无，用户可自行扩展逻辑。
- **过滤器**：无自动过滤。
