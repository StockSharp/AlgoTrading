# Стратегия Spread By
[English](README.md) | [中文](README_cn.md)

Spread By использует скользящую среднюю и полосы стандартного отклонения для торговли на ценовых экстремумах.
Покупает, когда цена падает ниже нижней полосы, и продаёт, когда цена поднимается выше верхней полосы.

## Детали

- **Критерий входа**: цена выходит за пределы ±1 стандартного отклонения от скользящей средней
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: цена возвращается к скользящей средней
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Length` = 100
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: обе стороны
  - Индикаторы: SMA, StandardDeviation
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
