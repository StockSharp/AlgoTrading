# Spread By 策略
[English](README.md) | [Русский](README_ru.md)

Spread By 使用移动平均线和标准差带在极端价格位置进行交易。
当价格跌破下轨时买入，价格升破上轨时卖出。

## 细节

- **入场条件**: 价格超出移动平均线±1个标准差
- **多空方向**: 双向
- **出场条件**: 价格回到移动平均线
- **止损**: 无
- **默认值**:
  - `Length` = 100
- **过滤器**:
  - 分类: Mean Reversion
  - 方向: 双向
  - 指标: SMA, StandardDeviation
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
