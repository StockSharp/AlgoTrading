# Uptrick X PineIndicators: Z-Score Flow Strategy
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия, использующая z-score, EMA и RSI для фильтрации сигналов.

## Детали

- **Условия входа**: z-score выходит за уровни покупки/продажи при подтверждении тренда и RSI
- **Длинные/Короткие**: Оба направления
- **Условия выхода**: Противоположный сигнал в выбранном режиме
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `ZScorePeriod` = 100
  - `EmaTrendLen` = 50
  - `RsiLen` = 14
  - `RsiEmaLen` = 8
  - `ZBuyLevel` = -2
  - `ZSellLevel` = 2
  - `CooldownBars` = 10
  - `SlopeIndex` = 30
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: SMA, EMA, RSI, StandardDeviation
  - Стопы: Нет
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
