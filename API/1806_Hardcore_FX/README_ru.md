# Hardcore FX Breakout
[English](README.md) | [中文](README_cn.md)

Стратегия адаптирована из эксперта MetaTrader "HardcoreFX". Она отслеживает максимумы и минимумы по индикатору ZigZag и открывает позиции при выходе цены за эти уровни. Для управления рисками используется фиксированный стоп-лосс, тейк-профит и трейлинг-стоп, защищающий накопленную прибыль.

## Детали
- **Условия входа**: Закрытие выше последнего ZigZag максимума для покупки; закрытие ниже последнего ZigZag минимума для продажи.
- **Направление**: Покупка и продажа.
- **Условия выхода**: Срабатывание стоп-лосса, тейк-профита или трейлинг-стопа.
- **Стопы**: Фиксированный стоп-лосс, тейк-профит и трейлинг-стоп.
- **Значения по умолчанию**:
  - `ZigzagLength` = 17
  - `StopLoss` = 1400
  - `TakeProfit` = 5400
  - `TrailingStop` = 500
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Highest, Lowest
  - Стопы: Стоп-лосс, тейк-профит, трейлинг-стоп
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
