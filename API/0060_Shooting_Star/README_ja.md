# 流れ星パターン (Shooting Star Pattern)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

流れ星（Shooting Star）の足は上昇後によく現れ、リバーサルを警告します。この戦略は実体に対して長い上ヒゲと、ほとんど下ヒゲのないパターンを探します。

テストでは年平均リターンが約67%となっています。株式市場での運用に最も適しています。

確認が必要な場合、次の足が安値で引けてからショートエントリーします。そうでなければ即座に取引できます。ストップはパターンの高値の上に置かれます。

## 詳細

- **エントリー条件**: 流れ星が検出され、有効な場合は確認も取れている。
- **ロング/ショート**: ショートのみ。
- **エグジット条件**: ストップロスまたは裁量的なエグジット。
- **ストップ**: はい。
- **デフォルト値**:
  - `ShadowToBodyRatio` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(15)
  - `StopLossPercent` = 1.0m
  - `ConfirmationRequired` = true
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ショートのみ
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
