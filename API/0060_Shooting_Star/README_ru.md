# Фигура "Падающая звезда"

Свеча "падающая звезда" часто появляется после роста и предупреждает о возможном развороте. Стратегия ищет длинную верхнюю тень относительно тела и почти отсутствие нижней тени.

Если требуется подтверждение, следующая свеча должна закрыться ниже, прежде чем открывать короткую позицию. Иначе сделку можно совершить сразу. Стоп размещается выше максимума паттерна.

## Детали

- **Условия входа**: Обнаружена "падающая звезда" и подтверждение, если включено.
- **Длинные/короткие позиции**: Только короткие.
- **Условия выхода**: Стоп-лосс или произвольный выход.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `ShadowToBodyRatio` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(15)
  - `StopLossPercent` = 1.0m
  - `ConfirmationRequired` = true
- **Фильтры**:
  - Категория: Pattern
  - Направление: Short
  - Индикаторы: Candlestick
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
