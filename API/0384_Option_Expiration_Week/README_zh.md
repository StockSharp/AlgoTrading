# 期权到期周策略

[English](README.md) | [Русский](README_ru.md)

该 Python 策略仅在期权到期周买入并持有某只 ETF。每月第三个星期五前的星期一开盘买入，星期五收盘平仓，利用到期周常见的短期强势。

其余时间组合保持现金。使用日线数据，每日通过市价单执行交易。

## 细节

- **标的**：单一股票型 ETF。
- **信号**：结束于第三个星期五的日历规则。
- **持有期**：到期周星期一开盘至星期五收盘。
- **仓位**：窗口期全仓，其他时间空仓。
- **风险控制**：如果交易金额低于 `MinTradeUsd` 则不下单。
