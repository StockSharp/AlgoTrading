# Многоуровневая регрессионная стратегия EUR/USD
[English](README.md) | [中文](README_cn.md)

Стратегия использует несколько слоёв линейной регрессии для оценки направления тренда по паре EUR/USD. Короткие, средние и длинные регрессии проверяются по порогу R² и наклона, после чего торгуется взвешенное решение.

## Детали

- **Условия входа**:
  - Long: взвешенный наклон > 0 и надёжность > 0.5
  - Short: взвешенный наклон < 0 и надёжность > 0.5
- **Направление**: Оба
- **Условия выхода**: Реверс при противоположном сигнале
- **Стопы**: Защита по дневным потерям
- **Значения по умолчанию**:
  - `ShortLength` = 20
  - `MediumLength` = 50
  - `LongLength` = 100
  - `MinRSquared` = 0.45m
  - `SlopeThreshold` = 0.00005m
  - `WeightShort` = 0.4m
  - `WeightMedium` = 0.35m
  - `WeightLong` = 0.25m
  - `PositionSizePct` = 50m
  - `MaxDailyLossPct` = 12m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Linear Regression
  - Стопы: Да
  - Сложность: Продвинутая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Уровень риска: Средний
