# Стратегия Reversal Finder
[English](README.md) | [中文](README_cn.md)

Reversal Finder ищет свечи с большим диапазоном, которые обновляют экстремумы и закрываются в противоположной части бара.
Покупает, когда цена делает новый минимум, но закрывается возле максимума, и продаёт, когда цена делает новый максимум, но закрывается возле минимума.

## Детали

- **Критерий входа**: расширение диапазона с закрытием возле противоположного экстремума после нового максимума/минимума
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Lookback` = 20
  - `SmaLength` = 20
  - `RangeMultiple` = 1.5
  - `RangeThreshold` = 0.5
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе стороны
  - Индикаторы: SMA, Highest, Lowest
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
