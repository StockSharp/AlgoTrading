# Reversal Finder 策略
[English](README.md) | [Русский](README_ru.md)

Reversal Finder 寻找具有大区间的蜡烛，这些蜡烛创出新高或新低后又在相反端附近收盘。
当价格创新低但收盘接近最高点时买入；当价格创新高但收盘接近最低点时卖出。

## 细节

- **入场条件**: 大区间后价格在相反端收盘并创出新高/新低
- **多空方向**: 双向
- **出场条件**: 相反信号
- **止损**: 无
- **默认值**:
  - `Lookback` = 20
  - `SmaLength` = 20
  - `RangeMultiple` = 1.5
  - `RangeThreshold` = 0.5
- **过滤器**:
  - 分类: 形态
  - 方向: 双向
  - 指标: SMA, Highest, Lowest
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
