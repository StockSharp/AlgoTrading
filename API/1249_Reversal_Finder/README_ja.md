# リバーサル・ファインダー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

リバーサル・ファインダーは、新たな極値を付けてからバーの反対側に戻って引ける大きな値幅のローソク足を探す。
価格が新安値に突入しても高値付近で引けるときに買い、価格が新高値に急騰しても安値付近で引けるときに売る。

## 詳細

- **エントリー条件**: 新高値/新安値後に反対の極値付近で引けるレンジ拡大
- **ロング/ショート**: 両方
- **エグジット条件**: 逆方向のシグナル
- **ストップ**: なし
- **デフォルト値**:
  - `Lookback` = 20
  - `SmaLength` = 20
  - `RangeMultiple` = 1.5
  - `RangeThreshold` = 0.5
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: SMA, Highest, Lowest
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
