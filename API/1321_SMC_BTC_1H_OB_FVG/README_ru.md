# SMC Strategy BTC 1H OB FVG
[English](README.md) | [中文](README_cn.md)

Стратегия Smart Money Concepts для Биткоина на часовом таймфрейме. После бычьего пробоя структуры система покупает при возврате цены в область ордерного блока или fair value gap. Стоп‑лосс рассчитывается по ATR, тейк‑профит по соотношению риск/прибыль.

## Детали

- **Условия входа**: после бычьего BOS покупка при касании OB или FVG в течение `ZoneTimeout` баров.
- **Длинные/короткие**: только лонг.
- **Условия выхода**: фиксированный тейк‑профит и стоп‑лосс.
- **Стопы**: фиксированные на основе ATR.
- **Значения по умолчанию**:
  - `UseOrderBlock` = true
  - `UseFvg` = true
  - `AtrFactor` = 6
  - `RiskRewardRatio` = 2.5
  - `ZoneTimeout` = 10
  - `CandleType` = TimeSpan.FromHours(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: ATR
  - Стопы: Фиксированные
  - Сложность: Простая
  - Таймфрейм: Внутридневной (1H)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
