# Стратегия "Медвежья харами"
[English](README.md) | [中文](README_zh.md)

"Медвежья харами" — обратный вариант бычьей модели и возникает после восходящего движения. Небольшая свеча полностью находится внутри предыдущей бычьей, что указывает на ослабление роста.

Тестирование показывает среднегодичную доходность около 43%. Стратегию лучше запускать на фондовом рынке.

Стратегия открывает короткую позицию при закрытии внутренней свечи, рассчитывая на разворот по мере утраты покупательского импульса.

Процентный стоп над максимумом паттерна ограничивает риск, а сделка закрывается при пробое новых вершин.

## Детали

- **Условия входа**: совпадение с паттерном
- **Длинная/короткая**: обе
- **Условия выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: обе
  - Индикаторы: свечные
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
