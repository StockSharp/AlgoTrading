# Стратегия Bearish Harami

Bearish Harami — противоположный бычьей версии паттерн, который появляется после подъёма.
Маленькая свеча полностью располагается внутри предыдущей бычьей, что намекает на затухание восходящего импульса.

Стратегия открывает короткую позицию после закрытия этой внутренней свечи, рассчитывая на разворот, когда покупатели теряют силу.

Процентный стоп выше максимума фигуры ограничивает риск, и сделка закрывается, если цена обновит максимум.

## Детали

- **Критерий входа**: совпадение паттерна
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
