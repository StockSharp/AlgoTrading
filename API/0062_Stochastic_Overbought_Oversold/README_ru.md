# Разворот по стохастику в зонах перекупленности/перепроданности

Стратегия реагирует на экстремальные значения осциллятора Stochastic. Когда линия %K уходит в зону перепроданности, система ждёт отскок. Перекупленность, напротив, может предвещать снижение. Метод работает на коротких внутридневных свечах, поэтому сигналы появляются быстро.

После подписки на выбранный таймфрейм стратегия отслеживает линии %K и %D. Бычий сигнал формируется, когда %K опускается ниже 20 и начинает разворачиваться вверх. Медвежий сигнал возникает при росте %K выше 80 и последующем развороте вниз. Фиксированный процентный стоп ограничивает риск с обеих сторон.

Выход из позиции происходит при обратном пересечении уровня 50 линией %K, что указывает на смену импульса. Поскольку величина стопа зависит от текущего ATR, размер позиции адаптируется к волатильности.

## Детали

- **Условия входа**:
  - **Лонг**: `%K < 20` и бычий разворот.
  - **Шорт**: `%K > 80` и медвежий разворот.
- **Направление**: Оба.
- **Условия выхода**: Пересечение уровня 50 линией %K или стоп‑лосс.
- **Стопы**: Да, на расстоянии `2%`.
- **Значения по умолчанию**:
  - `StochPeriod` = 14
  - `KPeriod` = 3
  - `DPeriod` = 3
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: Стохастик
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
