# 偏極フラクタル効率戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は**Polarized Fractal Efficiency (PFE)**インジケーターに基づいて取引します。PFEは価格動作の効率を測定し、モメンタムが変化するとサインを変えます。

## 取引ロジック

1. 選択した時間軸のローソク足をサブスクライブしてPFEを計算する。
2. 前のバーのPFEが2バー前より低く、現在の値が前の値より高い場合、ロングポジションを開く。
3. 前のバーのPFEが2バー前より高く、現在の値が前の値より低い場合、ショートポジションを開く。
4. 新しいポジションを開く前に反対のポジションを閉じる。
5. オプションのストップロスとテイクプロフィット保護を有効にする。

## パラメーター

| 名前 | 説明 |
|------|------|
| `CandleType` | 分析に使用するローソク足シリーズ。 |
| `PfePeriod` | PFEインジケーターの計算期間。 |
| `SignalBar` | シグナル生成に使用するバーのオフセット。 |
| `TakeProfit` | 価格ステップでのテイクプロフィット。 |
| `StopLoss` | 価格ステップでのストップロス。 |
