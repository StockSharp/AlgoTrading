# Estrategia de Entrelazamiento de Medias Móviles
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia compara una media móvil rápida y una lenta. Se abre una posición cuando la diferencia absoluta entre las medias supera una zona muerta basada en ATR.

## Parámetros
- Fast MA
- Slow MA
- ATR Length
- Volatility Period
- Dead Zone %
- Deviation Multiplier
- MA Type
- Candle Type
