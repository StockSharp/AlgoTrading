# Moving-Average-Verflechtungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie vergleicht einen schnellen und einen langsamen gleitenden Durchschnitt. Eine Position wird eröffnet, wenn die absolute Differenz zwischen den Durchschnitten eine ATR-basierte Totzone überschreitet.

## Parameter
- Fast MA
- Slow MA
- ATR Length
- Volatility Period
- Dead Zone %
- Deviation Multiplier
- MA Type
- Candle Type
