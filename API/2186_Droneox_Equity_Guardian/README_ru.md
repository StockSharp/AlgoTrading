# Droneox Equity Guardian
[English](README.md) | [中文](README_cn.md)

Стратегия защиты капитала, контролирующая собственный капитал и закрывающая все позиции при достижении целевого уровня или падении ниже уровня стопа. По желанию останавливает дальнейшую торговлю после закрытия позиций.

## Детали

- **Условия входа**: Нет; работает как защитный слой.
- **Длинные/короткие**: Закрывает позиции в обе стороны.
- **Условия выхода**: Достижение целевого уровня капитала или уровня стопа.
- **Стопы**: Стоп и цель по капиталу.
- **Значения по умолчанию**:
  - `EquityTarget` = 999999m
  - `EquityStop` = 0m
  - `ClosePositions` = true
  - `DisableTrading` = true
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Стоп/цель по капиталу
  - Сложность: Начальный уровень
  - Таймфрейм: Нет
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
