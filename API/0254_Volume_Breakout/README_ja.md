# 出来高ブレイクアウト
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

出来高ブレイクアウト戦略は出来高の急速な拡大を観察します。読み取り値が平均的な範囲を超えてジャンプすると、価格は新しい動きを始めることが多いです。

テストでは平均年間リターンが約103%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

最近のデータと偏差乗数から派生したバンドをインジケーターが突破したときにポジションが開かれます。ストップ付きのロングとショートの取引が可能です。

このシステムは初期のブレイクアウトを求めるモメンタムトレーダーに適しています。出来高が平均値に戻ったら取引を決済します。デフォルト値は`AvgPeriod` = 20から始まります。

## 詳細

- **エントリー条件**: インジケーターが偏差乗数分だけ平均を超える。
- **ロング/ショート**: 両方向。
- **エグジット条件**: インジケーターが平均に回帰する。
- **ストップ**: あり。
- **デフォルト値**:
  - `AvgPeriod` = 20
  - `Multiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `StopLoss` = 2.0m
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Volume
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
