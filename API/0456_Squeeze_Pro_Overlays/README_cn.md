# Squeeze Pro Overlays 策略
[English](README.md) | [Русский](README_ru.md)

Squeeze Pro Overlays 策略在布林带完全处于多个肯特纳通道内部时识别波动收缩。挤压解除后，通过收盘价线性回归的斜率来确定交易方向。

## 细节

- **入场条件**：
  - 挤压结束（布林带突破最宽的肯特纳通道）。
  - **多头**：动量斜率 > 0。
  - **空头**：动量斜率 < 0。
- **方向**：双向。
- **出场条件**：
  - 反向信号。
- **止损**：无。
- **默认参数**：
  - `SqueezeLength` = 20
- **过滤器**：
  - 类型：波动性突破
  - 方向：双向
  - 指标：布林带、肯特纳通道、线性回归
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
