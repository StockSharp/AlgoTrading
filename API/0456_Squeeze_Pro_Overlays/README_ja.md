# Squeeze Pro Overlays戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Squeeze Pro Overlays戦略は、ボリンジャーバンドが複数のケルトナーチャネルの内側に完全に収まっているときにボラティリティの収縮を検出します。スクイーズが解除されると、終値の線形回帰の傾きが取引方向を決定します。

## 詳細

- **エントリー条件**:
  - スクイーズが終了する（ボリンジャーバンドが最も広いケルトナーチャネルの外側に移動する）。
  - **ロング**: モメンタムの傾き > 0。
  - **ショート**: モメンタムの傾き < 0。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 反対のシグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `SqueezeLength` = 20
- **フィルター**:
  - カテゴリ: ボラティリティブレイクアウト
  - 方向: 両方
  - インジケーター: Bollinger Bands, Keltner Channels, Linear Regression
  - ストップ: なし
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
