# Фигура "Двойная вершина"

Фигура "Двойная вершина" характеризуется двумя пиками, разделёнными несколькими барами с близкими ценами. После формирования второго пика медвежья свеча подтверждает разворот.

Стратегия продаёт после подтверждения, ставя стоп выше максимумов фигуры, пытаясь извлечь прибыль из смены тренда после истощения покупателей.

Позиции закрываются по стоп-лоссу или по усмотрению трейдера.

## Детали

- **Условия входа**: Две вершины в пределах `SimilarityPercent` после `Distance` баров.
- **Длинные/короткие позиции**: Только короткие.
- **Условия выхода**: Цена снова растёт или стоп-лосс.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `Distance` = 5
  - `SimilarityPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(15)
  - `StopLossPercent` = 1.0m
- **Фильтры**:
  - Категория: Pattern
  - Направление: Short
  - Индикаторы: Price Action
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
