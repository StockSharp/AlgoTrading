# ダブルトップ・パターン (Double Top Pattern)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ダブルトップは、類似した価格で複数のバーを挟んで隔てられた2つの高値を識別します。2つ目の高値が形成された後、陰線がリバーサルを確認します。

テストでは年平均リターンが約58%となっています。株式市場での運用に最も適しています。

戦略は確認後にパターンの高値より上にストップを置いてショートエントリーし、買い手が枯渇した後のトレンド転換から利益を得ることを目指します。

ポジションはストップロスまたは裁量的な目標によって決済されます。

## 詳細

- **エントリー条件**: `Distance` バー後、`SimilarityPercent` 以内に2つの高値が形成される。
- **ロング/ショート**: ショートのみ。
- **エグジット条件**: 価格が反発するかストップロス。
- **ストップ**: はい。
- **デフォルト値**:
  - `Distance` = 5
  - `SimilarityPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(15)
  - `StopLossPercent` = 1.0m
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ショートのみ
  - インジケーター: Price Action
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
