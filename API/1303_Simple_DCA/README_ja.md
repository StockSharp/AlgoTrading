# シンプル DCA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はベース注文を出し、価格が指定したパーセンテージ分乖離したときにセーフティ注文を追加します。平均エントリー価格から計算したテイクプロフィットに価格が達するとイグジットします。各セーフティ注文のサイズは係数で乗算されます。

## パラメーター
- ローソク足の種類
- ベース注文サイズ（クォート通貨）
- セーフティ注文の価格乖離（%）
- 最大セーフティ注文数
- テイクプロフィット（%）
- 注文サイズ乗数
