# Estratégia LANZ 2.0 [Backtest]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia baseada em swings que registra a estrutura do mercado e a tendência usando máximas e mínimas recentes.
As posições são abertas às 02:00 horário de Nova York após uma quebra de estrutura na direção da tendência.
O stop-loss é definido a partir de pontos de swing ou cobertura total e o take-profit é calculado por um multiplicador de risco/retorno.
Qualquer posição aberta é fechada manualmente às 11:45 horário de Nova York.
