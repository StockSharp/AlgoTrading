# LANZ-Strategie 2.0 [Backtest]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Swing-basierte Strategie, die Marktstruktur und Trend anhand jüngster Swing-Hochs und -Tiefs aufzeichnet.
Positionen werden um 02:00 Uhr New Yorker Zeit nach einem Strukturbruch in Richtung des Trends eröffnet.
Der Stop-Loss wird von Swing-Punkten oder einer vollständigen Abdeckung gesetzt, und der Take-Profit wird über einen Risiko-Rendite-Multiplikator berechnet.
Alle offenen Positionen werden um 11:45 Uhr New Yorker Zeit manuell geschlossen.
