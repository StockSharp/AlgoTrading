# LANZ戦略 2.0 [バックテスト]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

直近のスウィング高値と安値を使って市場構造とトレンドを記録するスウィングベースの戦略。
トレンド方向への構造ブレイク後、ニューヨーク時間02:00にポジションを開きます。
ストップロスはスウィングポイントまたは完全カバレッジから設定し、テイクプロフィットはリスクリワード乗数で計算されます。
開いているポジションはニューヨーク時間11:45に手動で決済されます。
