# Brake Exp戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は**BrakeExp**インジケーターに基づいて取引します。インジケーターは指数曲線から構築された適応型のサポートとレジスタンスチャンネルを描画します。チャンネルが下線から上線に切り替わると売りシグナルが生成され、上線から下線に切り替わると買いシグナルが生成されます。

## 仕組み

- インジケーターが**上昇シグナル**を報告すると、戦略はショートポジションを閉じて新しいロングポジションを開きます。
- **下降シグナル**が現れると、既存のロングポジションが閉じられてショートポジションが開かれます。
- **上昇トレンド**のみが検出された場合、戦略はショートポジションを終了します。
- **下降トレンド**のみが検出された場合、戦略はロングポジションを終了します。

シグナルは完成したローソク足のみで処理されます。

## パラメーター

- `A` – BrakeExpインジケーターの曲線加速係数。
- `B` – チャンネル幅に使用される価格ステップ。
- `CandleType` – インジケーター計算用のローソク足シリーズ。
- `Volume` – 市場参入時に使用する注文量。

## 注意事項

この戦略はStockSharpの高レベルAPIを使用しており、Designer、Shell、またはその他のStockSharp製品で実行できます。
