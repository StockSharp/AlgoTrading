# 时间相对成交量
[English](README.md) | [Русский](README_ru.md)

该策略比较一天中特定时间的成交量与近期平均成交量。

## 详情

- **入场条件**: 在指定时间相对成交量高于阈值。
- **多空方向**: 双向。
- **退出条件**: 相对成交量回落至 1 以下。
- **止损**: 无。
- **默认值**:
  - `Period` = 5
  - `Threshold` = 1.5m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `TargetHour` = 9
  - `TargetMinute` = 30
- **过滤器**:
  - 类型: 成交量
  - 方向: 双向
  - 指标: SMA, Volume
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
