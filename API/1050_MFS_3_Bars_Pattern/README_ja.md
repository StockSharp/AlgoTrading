# MFS 3バーパターン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は下降トレンド内で3本足の強気リバーサルシーケンスを検出します。大きな緑の「点火」バー、小さな赤い戻り、そして戻りの高値を上回って引けた強気確認バーを探します。トレンドフィルターは長期SMA > 中期SMA > 短期SMAを要求し、点火バーの終値が短期SMAを下回っていることが条件です。

パターンが現れるとロングポジションを建て、ストップロスを点火バーの安値に、テイクプロフィットを設定したリスク・リワード倍率に置きます。

## 詳細

- **エントリー条件**: 下降トレンド内での点火バー、戻りバー、確認バー。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 点火安値でのストップロスまたはリスク・リワード倍率でのテイクプロフィット。
- **ストップ**: はい、ストップ注文とターゲット注文。
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `SmaShortLength` = 20
  - `SmaMedLength` = 50
  - `SmaLongLength` = 200
  - `IgniteMultiplier` = 3
  - `MaxPullbackSize` = 0.33
  - `MinConfirmationSize` = 0.33
  - `RiskReward` = 2
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ロング
  - インジケーター: Candlestick、Moving Average
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
