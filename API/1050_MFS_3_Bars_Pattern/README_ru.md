# Стратегия паттерна из трёх свечей MFS
[English](README.md) | [中文](README_cn.md)

Стратегия ищет бычий разворот из трёх свечей на фоне нисходящего тренда. Паттерн состоит из мощной зелёной свечи "разжигания", небольшой красной коррекции и подтверждающей бычьей свечи, закрывающейся выше максимума коррекции. Фильтр тренда требует, чтобы длинная SMA была выше средней, а средняя выше короткой, при этом закрытие свечи разжигания находится ниже короткой SMA.

После появления паттерна открывается длинная позиция. Стоп‑лосс размещается на минимуме свечи разжигания, а тейк‑профит рассчитывается как указанное отношение риск‑прибыль.

## Детали

- **Условия входа**: Свечи разжигания, отката и подтверждения в нисходящем тренде.
- **Направление**: Только лонг.
- **Условия выхода**: Стоп‑лосс по минимуму разжигания или тейк‑профит по отношению риск‑прибыль.
- **Стопы**: Да, стоп и цель.
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `SmaShortLength` = 20
  - `SmaMedLength` = 50
  - `SmaLongLength` = 200
  - `IgniteMultiplier` = 3
  - `MaxPullbackSize` = 0.33
  - `MinConfirmationSize` = 0.33
  - `RiskReward` = 2
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Лонг
  - Индикаторы: Свечной анализ, Скользящая средняя
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
