# Color HMA StDev
[English](README.md) | [中文](README_cn.md)

Стратегия на основе Hull Moving Average с динамическим фильтром стандартного отклонения.

Система отслеживает, насколько цена отклоняется от HMA. Когда закрытие пробивает выше среднего
на выбранное число стандартных отклонений, открывается длинная позиция, и наоборот для короткой.
Более широкий множитель определяет зону выхода, поэтому позиции закрываются только после значимого
возврата цены внутрь диапазона.

Такой подход позволяет ловить резкие импульсы, фильтруя рыночный шум. Hull Moving Average быстро
реагирует на изменения тренда, а стандартное отклонение подстраивается под волатильность,
расширяя порог в период турбулентности. Стратегия работает в обе стороны и не использует фиксированные
стопы, полагаясь на возврат цены к HMA.

## Подробности

- **Критерии входа**: Закрытие выше/ниже HMA ± K1 * StdDev.
- **Длинные/короткие**: Обе стороны.
- **Критерии выхода**: Закрытие выше/ниже HMA ± K2 * StdDev в противоположном направлении.
- **Стопы**: Нет фиксированных стоп-лоссов и тейк-профитов.
- **Значения по умолчанию**:
  - `HmaPeriod` = 13
  - `StdPeriod` = 9
  - `K1` = 1.5m
  - `K2` = 2.5m
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Тренд, Волатильность
  - Направление: Обе
  - Индикаторы: HMA, Стандартное отклонение
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: 4 часа
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
