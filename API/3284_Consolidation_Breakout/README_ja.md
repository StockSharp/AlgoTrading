# 保ち合いブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MetaTrader 向けの元 **Consolidation Breakout** エキスパートアドバイザーの中核動作を再現します。momentum と MACD フィルターで確認された狭い保ち合いを探し、その後ブレイクアウト方向にポジションを開きます。リスクは価格ステップ (pips) で測定される固定の take-profit と stop-loss 距離で管理されます。

## 仕組み

1. 主要時間枠は `CandleType` パラメーターで定義されます。すべてのトレンド確認と保ち合い確認は、このローソク足で評価されます。
2. 典型価格で計算した 2 本の線形加重移動平均 (LWMA) が方向フィルターを提供します。ロングセットアップでは高速 LWMA が低速 LWMA より上に留まる必要があり、ショートセットアップでは逆の整列が必要です。
3. 保ち合いは、ロングケースでは 2 バー前のローソク足の安値が前のローソク足の高値より下に残る場合、ショートケースでは前の安値が 2 バー前の高値より下にある場合に検出されます。これは MQL 版の重なりバーのロジックを再現します。
4. momentum は動きを確認する必要があります。momentum の絶対値 (ゼロ基準) が、対応する買いまたは売りしきい値を超える必要があります。これは元のエキスパートの 100 レベル周辺の momentum フィルターを近似します。
5. `MacdCandleType` 時間枠で計算された別の MACD が取引方向と一致する必要があります。戦略は、軸の正側と負側の両方で MACD ラインがシグナルラインに先行しているかを確認し、ソースコードのマルチタイムフレーム確認を再現します。
6. すべてのフィルターがそろい、口座がフラットまたは反対方向にポジションを持っている場合、戦略は `TradeVolume` による成行注文を送信します。保護水準は価格ステップで即座に再計算され、バー内の極値がエグジットを発動できるようにします。
7. 確定した各ローソク足はアクティブポジションも監視します。ローソク足レンジが stop-loss または take-profit 水準に触れると、戦略は成行でポジションを閉じ、保護目標をリセットします。

## インジケーター

- 線形加重移動平均 (高速と低速、典型価格)
- Momentum
- MACD (上位時間枠で 12/26/9 期間)

## パラメーター

- `CandleType` - ブレイクアウト検出に使用する主要時間枠。
- `MacdCandleType` - 確認用 MACD フィルターに使用する時間枠。
- `FastMaPeriod` - 高速 LWMA の長さ。
- `SlowMaPeriod` - 低速 LWMA の長さ。
- `MomentumLength` - momentum フィルターのルックバック。
- `MomentumBuyThreshold` - ロング取引に必要な最小正の momentum。
- `MomentumSellThreshold` - ショート取引に必要な最小負の momentum (絶対値で表現)。
- `StopLossPips` - 価格ステップ単位の保護ストップ距離。
- `TakeProfitPips` - 価格ステップ単位の利益目標距離。
- `TradeVolume` - 各成行注文で送信される数量。

デフォルトは公開されたエキスパートアドバイザーを再現します。LWMA 期間は 6 と 85、momentum length は 14、買い/売りしきい値は 0.3、stop-loss は 20 pips、take-profit は 50 pips です。価格ステップが異なる商品を取引する場合は、pip ベースの距離を調整してください。

## 注意事項

- MQL スクリプトの trailing stop、break-even 移動、資金管理モジュールは、StockSharp 版を中核的なブレイクアウトロジックに集中させるため意図的に省略しています。
- 選択した時間枠がデータフィードでサポートされていることを必ず確認してください。上位時間枠のデータがまばらな場合、MACD フィルターの反応性を保つため、より低い `MacdCandleType` への切り替えを検討してください。
