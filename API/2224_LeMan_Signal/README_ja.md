# LeMan シグナル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

LeManシグナル戦略は、MetaTraderのオリジナルLeManSignalエキスパートアドバイザーを移植したものです。このアプローチは、2つの連続した期間にわたる直近の高値と安値を分析して、潜在的なトレンドリバーサルを検出します。特定のパターンが見つかった場合、次のローソク足でロングまたはショートポジションがオープンされます。

## 仕組み

1. 戦略は選択した時間軸の完成したローソク足を観察します。
2. 前のバーについて、2つの連続したレンジで最高の高値と最低の安値を比較します:
   - `H1` と `H2` は2つの隣接するレンジの最大値です。
   - `H3` と `H4` は次のペアのレンジの最大値です。
   - `L1` と `L2` は2つの隣接するレンジの最小値です。
   - `L3` と `L4` は次のペアのレンジの最小値です。
3. `H3 <= H4` かつ `H1 > H2` の場合、**買い**シグナルが発動します。
4. `L3 >= L4` かつ `L1 < L2` の場合、**売り**シグナルが発動します。
5. 注文は市場価格で実行されます。オープン中の逆方向ポジションは自動的にクローズされます。
6. `StartProtection` によるオプションのリスク管理が、デフォルトのストップロス1%とテイクプロフィット2%でそれぞれ適用されます。

## パラメーター

- **Period** – インジケーターのルックバック期間の長さ。
- **Signal Bar** – シグナルを確認するために使用するオフセット（デフォルト1）。
- **Candle Type** – 分析するローソク足の時間軸。

## 注意事項

- 戦略は完成したローソク足にのみ反応します。
- 追加のコレクションは維持されません。内部バッファは計算に必要な最小限に制限されています。
- 戦略を使用するには、StockSharpターミナルに追加し、希望のインストゥルメントとパラメーターを設定して戦略を開始してください。
