# RGT RSIボリンジャー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は相対力指数（RSI）とボリンジャーバンドを組み合わせて平均回帰の機会を見つけます。RSIが売られすぎの市場を示し、価格がボリンジャーバンドの下部バンドを下回って取引されているときにロングポジションを建てます。RSIが買われすぎの市場を示し、価格が上部バンドを上回って上昇しているときにショートポジションを建てます。戦略は最初のストップロスを適用し、その後、最小利益に達したらストップをトレーリングします。

トレーリングストップはトレードが有利に動くと固定距離で価格を追いかけて利益を確保します。トレーリングストップが発動するとポジションが決済されます。

## 詳細

- **エントリー条件**: ロングはRSIが`RsiLow`を下回り価格が下部バンドを下回る場合；ショートはRSIが`RsiHigh`を上回り価格が上部バンドを上回る場合。
- **ロング/ショート**: 両方向。
- **エグジット条件**: トレーリングストップの発動。
- **ストップ**: 初期ストップロスとトレーリングストップ。
- **デフォルト値**:
  - `RsiPeriod` = 8
  - `RsiHigh` = 90
  - `RsiLow` = 10
  - `StopLossPips` = 70
  - `TrailingStopPips` = 35
  - `MinProfitPips` = 30
  - `Volume` = 1
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: RSI, ボリンジャーバンド
  - ストップ: はい
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
