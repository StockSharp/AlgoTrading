# Up3x1 Investor 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Up3x1 Investor 戦略は、強い拡張ローソク足に反応するクラシックなMetaTraderエキスパートアドバイザーを移植したものです。設定された時間軸で最後に完成したバーを監視し、前のローソク足のレンジとボディが終値の方向に十分広ければ、次のバーで新しいポジションを建てます。

この戦略はH1チャートでのFXメジャー通貨ペアなど裁量型市場向けに設計されていますが、しきい値は他の銘柄向けに調整できます。一度に保持するポジションは1つだけで、すべての注文は戦略の`Volume`プロパティを取引サイズとして使用します。

## 取引ロジック

- **シグナルソース** – `CandleType`の完成した時間軸ローソク足（デフォルト：1時間）。
- **エントリー条件**
  - 前のバーの高値–安値レンジとローソク足の絶対ボディを計算します。
  - ローソク足が始値より高く終値し、レンジとボディの両方がそれぞれのpipしきい値を超えていれば、ロングでエントリーします。
  - ローソク足が始値より低く終値し、レンジとボディの両方がしきい値を超えていれば、ショートでエントリーします。
  - ポジションが開いている間は新規エントリーを無視します。
- **ポジション管理**
  - オプションのストップロスとテイクプロフィットのレベルは、`Security.PriceStep`を使用してpipsから価格単位に変換されます。
  - エントリーから価格が`TrailingStopPips + TrailingStepPips`進むと、トレーリングストップが有効になります。
  - トレーリングストップは、新しいレベルが前のトレーリングレベルよりも少なくとも`TrailingStepPips`分価格に近い場合にのみ移動します。
  - 価格がストップロス、テイクプロフィット、またはトレーリングストップのレベルに触れると、戦略はポジションを決済します。

## パラメーター

| パラメーター | 説明 |
|------------|------|
| `CandleType` | シグナルに使用するローソク足のデータタイプ（デフォルト：1時間時間軸）。 |
| `RangeThresholdPips` | 前のローソク足の最小高値–安値距離（pips単位）。 |
| `BodyThresholdPips` | 前のローソク足の最小始値–終値距離（pips単位）。 |
| `StopLossPips` | ストップロス距離（pips）。無効にするには0に設定。 |
| `TakeProfitPips` | テイクプロフィット距離（pips）。無効にするには0に設定。 |
| `TrailingStopPips` | トレーリング時に価格の後ろに維持する距離。トレーリングを無効にするには0に設定。 |
| `TrailingStepPips` | トレーリングストップが狭められる前に必要な追加のpips移動量。 |

> **注意：** pipしきい値は`Security.PriceStep`で乗算されます。pip変換が正確に反映されるよう、銘柄に有効な`PriceStep`が設定されていることを確認してください。

## 使用上の注意

1. 戦略を開始する前に、対象の`Security`と取引コネクターを割り当ててください。
2. 市場のボラティリティを反映するようにpipしきい値を調整してください。5桁の見積もりを持つFXペアは通常10 pips = 0.0010を使用します。
3. 望ましい注文サイズに戦略の`Volume`を設定してください。元のEAのポジションサイジングロジックは、StockSharpバージョンを透明に保つために意図的に簡略化されています。
4. シグナルはクローズしたローソク足で評価されるため、拡張ローソク足の確認直後にエントリーが送信されます。
