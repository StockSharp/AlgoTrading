# ラインオーダー単一エントリー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Line Order戦略はMQL4スクリプト「LineOrder」(10715)の移植版です。市場価格が事前定義されたエントリーラインに達するとポジションを建て、ストップロス、テイクプロフィット、オプションのトレーリングストップでポジションを管理します。

## パラメーター

- `Entry Price` – ポジションをトリガーする価格レベル。
- `Stop Loss (pips)` – エントリーから初期ストップロスまでの距離。
- `Take Profit (pips)` – エントリーからテイクプロフィットまでの距離。
- `Trailing Stop (pips)` – オプションのトレーリングストップ距離。ゼロに設定するとトレーリングが無効になります。
- `Candle Type` – 処理に使用するローソク足の種類。

## 取引ロジック

1. 戦略は選択されたローソク足シリーズを購読します。
2. 完成したローソク足がエントリー価格より上で終値を付けるとロングポジションが建てられます。エントリー価格より下で終値を付けるとショートポジションが建てられます。
3. エントリー後、ストップロスとテイクプロフィットのレベルは銘柄の価格ステップを使用して計算されます。
4. トレーリングストップが有効な場合、ストップレベルはトレードの方向に移動します。
5. 価格がストップロスまたはテイクプロフィットレベルに達するとポジションが決済されます。

これはユーザーが定義したラインでの自動注文執行に焦点を当てた、元のMQLスクリプトの簡略化された移植版です。
