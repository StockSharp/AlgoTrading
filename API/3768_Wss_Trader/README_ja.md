# WSSトレーダー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

forex-instruments.info で公開された「Wss_trader」MetaTrader 4 エキスパートアドバイザーのポート。オリジナルの EA は、カマリラ スタイルの反転レベルと古典的なピボット距離を組み合わせており、ロンドン セッション中に価格が設定されたバンドを突破するたびにバーごとに 1 つの取引を開始します。

## 戦略ロジック

1. 新しい取引日が始まるたびに、戦略は前日の高値、安値、終値を読み取り、ピボット ラダーを構築します。
   - `Pivot = (High + Low + Close) / 3`
   - `Long entry = Pivot + Metric × point`
   - `Short entry = Pivot − Metric × point`
   - `Long stop = Short entry`
   - `Short stop = Long entry`
   - ターゲットは、元のコードと同じ安全クランプを備えた MetaTrader 式 `Close ± (High − Low) × 1.1 / 2` をミラーリングします。
2. 取引は `Start Hour` と `End Hour` (両端を含む) の間でのみ許可されます。ウィンドウの外では、すべてのオープンポジションは即座にクローズされます。
3. 完了したローソク足がロングエントリーレベル（終値 >= レベルおよび前の終値 < レベル）を超えると、ストラテジーは設定されたボリュームで 1 回購入し、事前に計算されたストップとターゲットを付加し、そのバーのそれ以上のエントリーをブロックします。対称ルールがショートパンツに適用されます。
4. ポジションが少なくとも `Trailing Points` 価格ステップ分有利に動いた場合、ストップは終値まで同じ距離を保つように追跡されます。ストップは決して後退しません。

## パラメーター

| 名前 | 説明 | デフォルト |
| ---- | ----------- | ------- |
| `Working Candle` | 日中の計算に使用される主なローソク足のタイプ。 | `15 Minute` |
| `Daily Candle` | 前日のピボットレベルを読み取るために使用されるローソク足のタイプ。 | `1 Day` |
| `Start Hour` | 取引が有効な場合の時間 (0 ～ 23)。 | `8` |
| `End Hour` | 取引が新規エントリーの受け付けを停止する時間 (0 ～ 23)。 | `16` |
| `Metric Points` | 価格ステップで測定されたピボットからブレイクアウトレベルまでの距離。 | `20` |
| `Trailing Points` | 価格ステップでのトレーリングストップ距離。末尾を無効にするには、`0` に設定します。 | `20` |
| `Order Volume` | 元の `lots` パラメータを反映した注文サイズ。 | `0.1` |

## 注意事項

- この戦略は、取引ウィンドウが終了するとすぐに現在のポジションをクローズし、元の EA の動作と一致します。
- 完成したキャンドルにはトレーリングが加工されます。 StockSharp はこのポートのローソク足クロージャで動作するため、バー内トレーリングは再現されません。
- キャンドルごとに 1 つの取引のみが許可され、MQL バージョンの `tenb` フラグが複製されます。
