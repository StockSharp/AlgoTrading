# Стратегия Fractal Force Index
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует на основе сглаженного индикатора Force Index, который сравнивается с заданными уровнями. Когда значение индикатора поднимается выше верхнего уровня или опускается ниже нижнего, стратегия открывает или закрывает позиции в зависимости от выбранного режима. Force Index рассчитывается как произведение изменения цены и объёма и сглаживается EMA.

## Детали

- **Критерии входа**
  - *Режим по тренду*:
    - **Лонг**: индикатор пересекает `HighLevel` снизу вверх.
    - **Шорт**: индикатор пересекает `LowLevel` сверху вниз.
  - *Режим против тренда*:
    - **Лонг**: индикатор пересекает `LowLevel` сверху вниз.
    - **Шорт**: индикатор пересекает `HighLevel` снизу вверх.
- **Критерии выхода**
  - *Режим по тренду*:
    - **Лонг**: пересечение ниже `LowLevel`.
    - **Шорт**: пересечение выше `HighLevel`.
  - *Режим против тренда*:
    - **Лонг**: пересечение выше `HighLevel`.
    - **Шорт**: пересечение ниже `LowLevel`.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Period` = 30
  - `HighLevel` = 0
  - `LowLevel` = 0
  - `Candle Type` = 4 часа
- **Фильтры**:
  - Категория: Импульсная
  - Направление: Оба
  - Индикаторы: Force Index
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
