# フラクタル強度指数戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、平滑化されたForce Indexがユーザー定義のレベルをクロスすることに基づいて取引します。インジケーターが高レベルを上回るか低レベルを下回ると、選択した取引モードに応じてポジションを開閉します。Force Indexは価格変化と出来高から計算され、EMAで平滑化されます。

## 詳細

- **エントリー条件**
  - *ダイレクトモード*:
    - **ロング**: インジケーターが`HighLevel`を上抜け。
    - **ショート**: インジケーターが`LowLevel`を下抜け。
  - *逆張りモード*:
    - **ロング**: インジケーターが`LowLevel`を下抜け。
    - **ショート**: インジケーターが`HighLevel`を上抜け。
- **エグジット条件**
  - *ダイレクトモード*:
    - **ロング**: `LowLevel`を下抜け。
    - **ショート**: `HighLevel`を上抜け。
  - *逆張りモード*:
    - **ロング**: `HighLevel`を上抜け。
    - **ショート**: `LowLevel`を下抜け。
- **ストップ**: なし。
- **デフォルト値**:
  - `Period` = 30
  - `HighLevel` = 0
  - `LowLevel` = 0
  - `Candle Type` = 4-hour
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: Force Index
  - ストップ: なし
  - 複雑さ: 中
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
