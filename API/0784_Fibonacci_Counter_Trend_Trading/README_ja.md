# Fibonacci 逆張り取引戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、出来高加重移動平均（VWMA）と標準偏差を使用してFibonacciバンドを構築します。価格が選択した下限バンドを下回ったときにロングエントリーし、上限バンドを上回ったときにショートエントリーします。オプションで、価格がVWMAベースをクロスしたときにポジションを決済します。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が選択した下限バンドを下にクロスする。
  - **ショート**: 終値が選択した上限バンドを上にクロスする。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ベース**: 価格がVWMAをクロスしたときのオプション決済。
  - **反転**: 反対側のバンドシグナルがポジションを反転させる。
- **ストップ**: なし。
- **インジケーター**: VolumeWeightedMovingAverage, StandardDeviation.
