# Стратегия торговли в диапазоне IU
[English](README.md) | [中文](README_cn.md)

Стратегия фиксирует периоды узкой волатильности, когда диапазон между максимумом и минимумом за заданный период не превышает множитель ATR. При выходе цены за границы диапазона открываются сделки в направлении пробоя. Позиции сопровождаются трейлинг-стопом на основе ATR.

## Детали

- **Критерии входа**: пробой диапазона, определённого ATR.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: трейлинг-стоп по ATR.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `RangeLength` = 10
  - `AtrLength` = 14
  - `AtrTargetFactor` = 2.0m
  - `AtrRangeFactor` = 1.75m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: ATR, Highest, Lowest
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
