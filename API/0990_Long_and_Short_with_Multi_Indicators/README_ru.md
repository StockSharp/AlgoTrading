# Стратегия Long and Short с несколькими индикаторами
[English](README.md) | [中文](README_cn.md)

Стратегия использует RSI, Rate of Change и выбранную скользящую среднюю для формирования сигналов в обе стороны. Выход выполняется по трейлинг-стопу на основе ATR.

## Подробности

- **Условия входа**:
  - Лонг: RSI между уровнем перепроданности и перекупленности, ROC > 0 и цена выше MA.
  - Шорт: Подтверждённый медвежий тренд, ROC < 0 и цена ниже MA.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Трейлинг-стоп по ATR или индикаторные условия.
- **Стопы**: Трейлинг-стоп на основе ATR.
- **Значения по умолчанию**:
  - `RsiLength` = 5
  - `RsiOverbought` = 70
  - `RsiOversold` = 44
  - `RocLength` = 4
  - `MaLength` = 24
  - `MaTypeParam` = TEMA
  - `AtrLength` = 14
  - `AtrMultiplier` = 2
  - `BearishMaLength` = 200
  - `BearishTrendDuration` = 5
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг и Шорт
  - Индикаторы: RSI, ROC, MA, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
