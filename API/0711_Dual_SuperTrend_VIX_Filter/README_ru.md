# Стратегия Dual SuperTrend с фильтром VIX
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает два индикатора SuperTrend с волатильностным фильтром VIX. Лонг открывается, когда оба SuperTrend указывают на рост, а индекс VIX находится выше своей средней. Шорт открывается, когда оба SuperTrend указывают на падение и VIX растёт выше среднего с учётом стандартного отклонения. Позиция закрывается при смене направления любого из SuperTrend.

## Детали

- **Условия входа**:
  - **Лонг**: оба SuperTrend в росте и VIX выше средней.
  - **Шорт**: оба SuperTrend в падении, а VIX выше средней и растёт.
- **Условия выхода**:
  - Обратный сигнал SuperTrend.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `StLength1` = 13
  - `StMultiplier1` = 3.5
  - `StLength2` = 8
  - `StMultiplier2` = 5
  - `UseVixFilter` = true
  - `VixLookback` = 252
  - `VixTrendPeriod` = 10
  - `StdDevMultiplier` = 1
  - `EnableLong` = true
  - `EnableShort` = true
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SuperTrend, SMA, StandardDeviation, EMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
