# Стратегия Profitable Pullback Strategy Mark804
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия откатов, основанная на ленте экспоненциальных скользящих средних. Система ищет откаты цены к сигнальной EMA в рамках подтверждённого тренда. Когда после отката цена закрывается в направлении тренда, стратегия открывает позицию и защищает её фиксированными процентами тейк‑профита и стоп‑лосса.

## Детали

- **Условия входа**:
  - **Лонг**: Быстрая EMA > Сигнальная EMA > Средняя EMA, при включённом фильтре Средняя EMA > Медленная EMA, предыдущая цена закрытия ниже Сигнальной EMA и текущая выше.
  - **Шорт**: Быстрая EMA < Сигнальная EMA < Средняя EMA, при включённом фильтре Средняя EMA < Медленная EMA, предыдущая цена закрытия выше Сигнальной EMA и текущая ниже.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: Срабатывание тейк‑профита или стоп‑лосса.
- **Стопы**: Да, фиксированные проценты тейк‑профита и стоп‑лосса.
- **Значения по умолчанию**:
  - Период быстрой EMA = 8
  - Период сигнальной EMA = 21
  - Период средней EMA = 50
  - Период медленной EMA = 200
  - Тейк‑профит % = 2
  - Стоп‑лосс % = 1
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: EMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Средний
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
