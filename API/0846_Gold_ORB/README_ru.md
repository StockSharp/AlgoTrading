# Стратегия Gold ORB
[English](README.md) | [中文](README_cn.md)

Стратегия фиксирует максимум и минимум азиатской сессии и торгует прорывы в следующие часы. Стопы и цели рассчитываются на основе размера диапазона и коэффициента прибыли к риску.

## Детали

- **Условия входа**:
  - В торговое окно покупка при закрытии выше зафиксированного максимума Азии.
  - Продажа при закрытии ниже зафиксированного минимума Азии.
- **Длинные/короткие**: Оба направления
- **Условия выхода**:
  - Стоп и цель на основе размера диапазона и множителя прибыли.
- **Стопы**: Да
- **Значения по умолчанию**:
  - `AsiaStart` = 00:00
  - `AsiaEnd` = 06:00
  - `TradeStart` = 06:00
  - `TradeEnd` = 10:00
  - `RewardMultiplier` = 2.0
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: 5м
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
