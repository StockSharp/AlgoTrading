# ゴールド ORB 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はアジアセッションの高値と安値を記録し、その後の時間帯にブレイクアウトを取引します。ストップと目標は、レンジサイズと報酬乗数から算出されます。

## 詳細

- **エントリー条件**:
  - トレードウィンドウ中、価格が記録されたアジアの高値を上回って終値をつけたときにロング。
  - 価格が記録されたアジアの安値を下回って終値をつけたときにショート。
- **ロング/ショート**: 両方
- **エグジット条件**:
  - レンジサイズと報酬乗数に基づくストップと目標。
- **ストップ**: はい
- **デフォルト値**:
  - `AsiaStart` = 00:00
  - `AsiaEnd` = 06:00
  - `TradeStart` = 06:00
  - `TradeEnd` = 10:00
  - `RewardMultiplier` = 2.0
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: 5m
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
