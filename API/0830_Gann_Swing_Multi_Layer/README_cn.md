# Gann Swing Multi Layer
[English](README.md) | [Русский](README_ru.md)

该策略使用简化的多层甘氏摆动分析。
当三个连续摆动方向一致时开仓。

该方法遵循甘氏理论的摆动转换概念。
只有在连续三次确认后才交易。

## 细节

- **入场条件**：三个连续摆动同向。
- **多空**：双向。
- **出场条件**：相反摆动。
- **止损**：无。
- **默认值**:
  - `CandleType` = TimeSpan.FromMinutes(1)
- **筛选**:
  - 分类：摆动
  - 方向：双向
  - 指标：Gann
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内 (1m)
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
