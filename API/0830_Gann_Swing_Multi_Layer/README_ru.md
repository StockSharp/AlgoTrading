# Gann Swing Multi Layer
[English](README.md) | [中文](README_cn.md)

Стратегия использует упрощённый многослойный анализ свингов Ганна.
Сделка открывается, когда три последовательных свинга совпадают по направлению.

Подход реализует идею Ганна о смене направления свинга.
Торгует только после трёх последовательных подтверждений.

## Подробности

- **Критерии входа**: три последовательных свинга в одном направлении.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный свинг.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Свинг
  - Направление: Оба
  - Индикаторы: Gann
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
