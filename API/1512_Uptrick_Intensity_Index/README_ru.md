# Стратегия Uptrick Intensity Index
[English](README.md) | [中文](README_cn.md)

Эта стратегия рассчитывает индекс интенсивности тренда на основе трёх скользящих средних и торгует по пересечению индекса и его средней.

## Детали

- **Условия входа**: пересечение TII выше своей SMA — покупка, ниже — продажа
- **Длин/Корот**: Обе стороны
- **Условия выхода**: противоположный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `Ma1Length` = 10
  - `Ma2Length` = 20
  - `Ma3Length` = 50
  - `TiiMaLength` = 50
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: SMA, TII
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
