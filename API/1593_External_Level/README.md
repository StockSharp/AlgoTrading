# External Level Strategy
[Русский](README_ru.md) | [中文](README_cn.md)

Draws support and resistance levels when an external source value equals 1 or -1. A line is drawn at the previous candle's high for value 1 and at the previous candle's low for value -1.

## Details

- **Entry Criteria**: None
- **Long/Short**: None
- **Exit Criteria**: None
- **Stops**: No
- **Default Values**:
  - `CandleType` = 1 minute
- **Filters**:

  - Indicators: None
  - Stops: No
  - Complexity: Basic
  - Timeframe: Intraday
  - Seasonality: No
  - Neural networks: No
  - Divergence: No
  - Risk level: Low
