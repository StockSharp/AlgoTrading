# Estratégia de Nível Externo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Desenha níveis de suporte e resistência quando o valor de uma fonte externa é igual a 1 ou -1. Uma linha é traçada na máxima da vela anterior para o valor 1 e na mínima para o valor -1.
