# Estrategia de Nivel Externo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Dibuja niveles de soporte y resistencia cuando el valor de una fuente externa es igual a 1 o -1. Se traza una línea en el máximo de la vela anterior para el valor 1 y en el mínimo para el valor -1.

## Detalles

- **Criterios de entrada**: Ninguno
- **Largo/Corto**: Ninguno
- **Criterios de salida**: Ninguno
- **Stops**: No
- **Valores predeterminados**:
  - `CandleType` = 1 minute
- **Filtros**:
  - Categoría: Dibujo
  - Dirección: Ninguno
  - Indicadores: Ninguno
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Intradía
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Bajo
