Zeichnet Unterstützungs- und Widerstandslevels, wenn ein externer Quellwert gleich 1 oder -1 ist. Eine Linie wird beim Hoch der vorherigen Kerze für den Wert 1 und beim Tief für den Wert -1 gezeichnet.
