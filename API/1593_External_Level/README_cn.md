# External Level 策略
[Русский](README_ru.md) | [English](README.md)

当外部信号值为 1 或 -1 时绘制支撑和阻力线。值为 1 时在上一根蜡烛的最高价处画线，值为 -1 时在最低价处画线。

## 细节

- **入场条件**：无
- **多空方向**：无
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `CandleType` = 1 分钟
- **过滤器**：
  - 类别：绘图
  - 方向：无
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：低
