# Стратегия всплеска VIX
[English](README.md) | [中文](README_cn.md)

Покупает, когда индекс VIX подскакивает выше средней на несколько стандартных отклонений, и закрывает позицию через заданное количество баров.

## Подробности

- **Критерий входа**: VIX > среднее + StdDevMultiplier * стандартное отклонение.
- **Длинные/короткие**: только длинные.
- **Критерий выхода**: закрытие через `ExitPeriods` баров.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `StdDevLength` = 15
  - `StdDevMultiplier` = 2
  - `ExitPeriods` = 10
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `VixSecurity` = "CBOE:VIX"
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Длинные
  - Индикаторы: SMA, StdDev
  - Стопы: Да
  - Сложность: Начальная
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
