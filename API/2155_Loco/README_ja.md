# Loco戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQL5で元々書かれた「Loco」インジケーターを実装したものです。インジケーターはローソク足の価格を分析し、色（緑またはマゼンタ）を割り当てます。色の変化はトレンド転換を示します。

## ロジック
- インジケーターは設定可能な価格（デフォルトは終値）と振り返り期間を使用して系列を計算します。
- 色がマゼンタから緑に変わると、戦略はショートポジションを閉じてロングポジションを開きます。
- 色が緑からマゼンタに変わると、戦略はロングポジションを閉じてショートポジションを開きます。

## パラメーター
- **Candle Type** – 戦略で使用するローソク足のタイプ。
- **Length** – 価格比較に使用するバー数。
- **Price Type** – インジケーター計算に使用する価格タイプ。

## 注意事項
戦略はLocoインジケーターのカスタム実装を使用します。Python版は提供されていません。
