# Ultimate Oscillator Trading
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kauft, wenn der Ultimate Oscillator unter einen Schwellenwert fällt, und steigt aus, wenn der Kurs über das vorherige Hoch schließt.

## Details

- **Einstieg**: Oszillator unterhalb des Kaufniveaus.
- **Ausstieg**: Schluss über dem vorherigen Hoch.
- **Indikatoren**: Ultimate Oscillator.
- **Stops**: keine.
