# Ultimate Oscillator トレーディング
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はUltimate Oscillatorが閾値を下回ったときに買い、価格が前の高値を上回って引けたときに決済する。

## 詳細

- **エントリー**: オシレーターが買いレベルを下回る。
- **エグジット**: 前高値を上回って終値が付く。
- **インジケーター**: Ultimate Oscillator。
- **ストップ**: なし。
