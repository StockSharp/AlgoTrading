# Trading con Ultimate Oscillator
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia compra cuando el Ultimate Oscillator cae por debajo de un umbral y sale cuando el precio cierra por encima del máximo anterior.

## Detalles

- **Entrada**: oscilador por debajo del nivel de compra.
- **Salida**: cierre por encima del máximo previo.
- **Indicadores**: Ultimate Oscillator.
- **Stops**: ninguno.
