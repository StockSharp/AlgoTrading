# Стратегия Прорыва Диапазона Открытия IU
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает максимум и минимум первой свечи каждой сессии и торгует пробои в любом направлении. Стопы берутся по экстремуму предыдущей свечи, цели рассчитываются по настраиваемому коэффициенту риск/прибыль, все позиции закрываются в указанное время.

## Детали

- **Условия входа**:
  - Покупка при закрытии выше максимума первой свечи.
  - Продажа при закрытии ниже минимума первой свечи.
- **Длинные/короткие**: Оба направления
- **Условия выхода**:
  - Стоп по минимуму/максимуму предыдущей свечи.
  - Цель по коэффициенту риск/прибыль.
  - Закрытие всех позиций в `EndTime`.
- **Стопы**: Да
- **Значения по умолчанию**:
  - `RiskReward` = 2.0
  - `MaxTrades` = 2
  - `EndTime` = 15:00
  - `CandleType` = 1 минута
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
