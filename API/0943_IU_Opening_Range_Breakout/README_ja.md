# IU オープニングレンジ・ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

IU Opening Range Breakout戦略は、各セッションの最初のバーの高値と安値を監視し、どちらの方向へのブレイクアウトも取引します。ストップは前のバーの極値を使用し、目標は設定可能なリスク・リワード比から導出されます。すべてのポジションはユーザー定義の終了時刻にクローズされます。

## 詳細

- **エントリー条件**:
  - 終値が最初のバーの高値を上抜けたらロング。
  - 終値が最初のバーの安値を下抜けたらショート。
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 前のバーの安値/高値でストップ。
  - リスク・リワード比に基づく目標。
  - `EndTime` にすべてのポジションをクローズ。
- **ストップ**: はい
- **デフォルト値**:
  - `RiskReward` = 2.0
  - `MaxTrades` = 2
  - `EndTime` = 15:00
  - `CandleType` = 1分
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
