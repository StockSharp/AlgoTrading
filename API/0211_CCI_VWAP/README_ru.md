# Стратегия CCI VWAP

Метод CCI VWAP нацелен на внутридневные развороты, когда цена и импульс отходят от объёмно‑взвешенной средней. Наблюдая CCI в связке с уровнем VWAP, система оценивает силу последних колебаний.

Покупка предполагается при значении CCI ниже -100 и цене под VWAP, что может означать ослабление продавцов. Продажа происходит, если CCI выше +100 и цена над VWAP, что указывает на перекупленность. Позиция закрывается, как только цена возвращается к VWAP в обратную сторону.

Стратегия рассчитана на дневных трейдеров, предпочитающих играть против экстремумов. Фиксированный стоп‑лосс помогает держать риск под контролем.

## Подробности
- **Условия входа**:
  - **Long**: CCI < -100 && Price < VWAP
  - **Short**: CCI > 100 && Price > VWAP
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход при движении цены выше VWAP
  - **Short**: выход при падении цены ниже VWAP
- **Стопы**: да
- **Значения по умолчанию**:
  - `CciPeriod` = 20
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: CCI VWAP
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
