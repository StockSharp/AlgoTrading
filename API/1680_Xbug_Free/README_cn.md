# Xbug Free 策略
[English](README.md) | [Русский](README_ru.md)

逆势移动平均策略：当价格向下穿越均线时买入，向上穿越时卖出。使用对称的止盈和止损距离。

## 细节

- **入场条件**: 价格与简单均线的交叉
- **多空方向**: 双向
- **出场条件**: 相反信号或保护性止损
- **止损**: 有
- **默认值**:
  - `MaPeriod` = 19
  - `MaShift` = 15
  - `StopPoints` = 270
  - `Volume` = 0.1
  - `CandleType` = 4 小时
- **过滤器**:
  - 分类: Mean Reversion
  - 方向: 双向
  - 指标: MA
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 波段
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
