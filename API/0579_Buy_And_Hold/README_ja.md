# 買い持ち戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は指定した開始日に単一のロングポジションをエントリーし、終了日まで保有することで、シンプルな買い持ちアプローチを実装します。

## 詳細

- **エントリー条件**:
  - ローソク足の時刻が開始日以降になったとき、戦略は一度だけ買います。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - ローソク足の時刻が終了日に達するか超えたとき、ポジションをクローズします。
- **ストップ**: なし。
- **デフォルト値**:
  - 開始日 = 2018-01-01。
  - 終了日 = 2069-12-31。
- **フィルター**:
  - カテゴリ: Buy and Hold。
  - 方向: ロング。
  - インジケーター: なし。
  - ストップ: いいえ。
  - 複雑さ: 低。
  - 時間軸: 任意。
  - 季節性: いいえ。
  - ニューラルネットワーク: いいえ。
  - ダイバージェンス: いいえ。
  - リスクレベル: 高。
