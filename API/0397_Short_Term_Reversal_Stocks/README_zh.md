# 股票短期反转
[English](README.md) | [Русский](README_ru.md)

**股票短期反转** 策略将均值回归应用于股票。每天买入上一周跌幅最大的股票并做空涨幅最大的股票，押注短暂的反转。

持仓仅持续几天，然后重新评估。

## 细节
- **入场条件**：根据上一周收益进行每日排序。
- **多空方向**：双向。
- **出场条件**：持有几天或排名更新时平仓。
- **止损**：可采用基于波动率的止损。
- **默认值**：
  - `CandleType = TimeSpan.FromDays(1).TimeFrame()`
- **筛选**：
  - 分类：均值回归
  - 方向：双向
  - 指标：价格
  - 止损：有
  - 复杂度：基础
  - 时间框架：短期
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险级别：中等
