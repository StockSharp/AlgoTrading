# MACD Stochastic 2戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTraderの「MACD Stochastic 2」エキスパートロジックをStockSharpの高レベルAPIで再現しています。3バーMACDスイングフィルターとストキャスティクスオシレーターを組み合わせ、売られすぎと買われすぎの領域付近でのモメンタム転換を探します。リスクは方向別のストップ、テイクプロフィット、およびpip単位で動作するオプションのトレーリングストップによって管理されます。

## 概要

- `CandleType`パラメーターで指定された任意の銘柄と時間軸で動作します。
- MACDメインラインを使用してローカルの谷/峰を確認し、MACDヒストグラムとシグナルラインは可視化に使用できます。
- ロングのストキャスティクス%K読み値が20未満、ショートの80超でエントリーを確認します。
- MetaTraderのpip処理を適応：銘柄の価格ステップからpipサイズを導出し、シンボルが3または5桁の場合は10を掛けます。

## 取引ロジック

### ロングエントリー

1. 現在と過去2本の完成したローソク足のMACDメインライン値がすべてゼロ未満。
2. 現在のMACD値が前の値より大きく、前の値が2バー前の値より小さい（ローカルの谷）。
3. ストキャスティクス%Kが20未満（売られすぎ）。
4. 既存のロングポジションが開いていない（`Position <= 0`）。新しいロングに入る前にショートポジションがあれば手仕舞い。

### ショートエントリー

1. 現在と過去2本の完成したローソク足のMACDメインライン値がすべてゼロ超。
2. 現在のMACD値が前の値より小さく、前の値が2バー前の値より大きい（ローカルの峰）。
3. ストキャスティクス%Kが80超（買われすぎ）。
4. 既存のショートポジションが開いていない（`Position >= 0`）。新しいショートに入る前にロングポジションがあれば決済。

### リスク管理と出口

- **ハードストップ / テイクプロフィット：** 各方向に独立したpipベースのストップロスとテイクプロフィット距離があります。Pipは計算されたpipサイズを使用して絶対価格オフセットに変換されます。
- **トレーリングストップ：** 有効な場合、価格がトレーリング距離を超えて進んだ後にトレーリングストップが起動します。過剰な注文の入れ替えを避けるため、動きが設定されたトレーリングステップを超えた場合のみストップが上昇/下降します。
- **逆シグナル：** 逆シグナルに入る際は、まず既存ポジションを手仕舞いし、その後設定された取引量で新しいポジションを開きます。

## パラメーター

| パラメーター | デフォルト | 説明 |
|------------|----------|------|
| `TradeVolume` | `1` | 新規取引ごとに送られる注文量。 |
| `StopLossBuyPips` | `50` | ロングストップロスのpip距離。`0`で無効化。 |
| `StopLossSellPips` | `50` | ショートストップロスのpip距離。`0`で無効化。 |
| `TakeProfitBuyPips` | `50` | ロングテイクプロフィットのpip距離。`0`で無効化。 |
| `TakeProfitSellPips` | `50` | ショートテイクプロフィットのpip距離。`0`で無効化。 |
| `TrailingStopPips` | `0` | トレーリングストップ距離（pip）。`0`でトレーリング無効。 |
| `TrailingStepPips` | `5` | トレーリングストップ更新前の最小pip獲得量。トレーリング有効時は正数必須。 |
| `MacdFastPeriod` | `12` | MACDの高速EMA長。 |
| `MacdSlowPeriod` | `26` | MACDの低速EMA長。 |
| `MacdSignalPeriod` | `9` | MACDのシグナル平滑化長。 |
| `StochasticKPeriod` | `5` | ストキャスティクス%Kのルックバック期間。 |
| `StochasticDPeriod` | `3` | ストキャスティクス%Dの平滑化期間。 |
| `StochasticSlowing` | `3` | ストキャスティクス%Kに適用される追加平滑化。 |
| `CandleType` | `1h時間軸` | インジケーター計算に使用するローソク足タイプ（時間軸）。 |

## 注意事項

- pip計算は元のMetaTraderエキスパートを反映：`pip = PriceStep`で、3または5桁で表示される銘柄の場合は10を掛けます。
- ストキャスティクスのしきい値（20/80）は元のスクリプトと同様に定数のままです。カスタムレベルが必要な場合はコード内で直接調整してください。
- 戦略は完全に完成したローソク足のみで動作し、MetaTraderのバークローズ実行との一貫性を確保します。

## 使用方法

1. 戦略を開始する前に希望の銘柄、`CandleType`、ボリュームを設定します。
2. 銘柄のボラティリティに合わせてストップ、テイクプロフィット、トレーリングパラメーターを調整します。
3. 公開されたパラメーターを使用してStockSharpのオプティマイザーでMACDとストキャスティクスの長さをオプションで最適化します。
4. チャートエリアが利用可能な場合に自動的に追加されるチャートオブジェクト（ローソク足、MACD、ストキャスティクス、自己取引）を監視します。
