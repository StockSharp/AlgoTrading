# Стратегия CCI Support Resistance
[English](README.md) | [中文](README_cn.md)

Стратегия строит динамические уровни поддержки и сопротивления на основе пивотов CCI. Перед торговлей пробоев применяется трендовый фильтр на основе пересечения или наклона EMA.

## Детали

- **Условия входа**:
  - Лонг: цена закрывается выше уровня поддержки, построенного по CCI, после касания и тренд бычий.
  - Шорт: цена закрывается ниже уровня сопротивления, построенного по CCI, после касания и тренд медвежий.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Стоп-лосс и тейк-профит на основе ATR.
- **Стопы**: Да, на основе ATR.
- **Параметры по умолчанию**:
  - `CciLength` = 50
  - `LeftPivot` = 50
  - `RightPivot` = 50
  - `Buffer` = 10
  - `TrendMatter` = true
  - `TrendType` = Cross
  - `SlowMaLength` = 100
  - `FastMaLength` = 50
  - `SlopeLength` = 5
  - `Ksl` = 1.1
  - `Ktp` = 2.2
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: CCI, EMA, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
