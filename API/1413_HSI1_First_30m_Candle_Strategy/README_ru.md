# HSI1 First 30m Candle Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробой диапазона первых 30 минут на графике 15 минут, допускается только одна сделка в день.

## Детали
- **Условия входа**: Цена пробивает максимум/минимум первых 30 минут в рабочей сессии.
- **Лонг/Шорт**: Оба направления, выборочно.
- **Условия выхода**: Тейк-профит или стоп-лосс по размеру диапазона.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `RiskReward` = 1
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Цена
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
