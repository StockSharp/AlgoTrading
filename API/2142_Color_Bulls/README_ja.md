# Color Bulls戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略はMetaTraderエキスパート`Exp_ColorBulls`を移植したものです。Color Bullsインジケーターを使用しており、このインジケーターはローソク足の高値と移動平均の差を計算します。得られた値はもう一つの移動平均で平滑化され、上昇値と下降値で異なる色のヒストグラムとして表示されます。

戦略はこのヒストグラムの色の変化に反応します：

- インジケーターが上昇（緑）から下降（マゼンタ）に切り替わると、ロングポジションを開く。
- インジケーターが下降から上昇に切り替わると、ショートポジションを開く。
- 新しいポジションを建てる前に、反対方向のポジションは自動的に決済される。

完成したローソク足のみが処理され、エントリーとエグジットには成行注文が使用されます。

## パラメーター

- **Fast MA Length** – 高値に適用する移動平均の期間。
- **Smooth Length** – bulls値を平滑化する移動平均の期間。
- **Candle Type** – 計算に使用するローソク足の時間軸。

## 備考

この例はカスタムインジケーターとStockSharpの高レベルAPIの統合を示しています。ストップロスとテイクプロフィットの管理は含まれていません。
