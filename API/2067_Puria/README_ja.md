# Puria 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Puria は、高速 EMA、安値の 2 本の低速 LWMA、および MACD フィルターを組み合わせたトレンドフォロー戦略です。5 期間 EMA が 75 期間と 85 期間の両方の LWMA を上回り、前回終値が EMA を上回り、MACD ラインが正の場合、ロングポジションが開かれます。反対の条件が満たされた場合、ショートポジションが開かれます。この戦略は固定のテイクプロフィットとストップロスレベルを使用し、反対のシグナルが現れるまで各方向に 1 つのポジションのみを許可します。

## 詳細
- **エントリー条件**: EMA(5) が LWMA(75) と LWMA(85) を上回り、前回終値が EMA を上回り、MACD(15,26) > 0 でロング；ショートは逆条件。
- **ロング/ショート**: 両方。
- **エグジット条件**: ストップロスまたはテイクプロフィット。
- **ストップ**: 価格ポイントでの固定ストップロスとテイクプロフィットの距離。
- **デフォルト値**:
  - `StopLoss` = 14
  - `TakeProfit` = 15
  - `Ma1Period` = 75
  - `Ma2Period` = 85
  - `Ma3Period` = 5
  - `CandleType` = 1 分足時間軸
- **フィルター**: MACD ゼロライン フィルター。
