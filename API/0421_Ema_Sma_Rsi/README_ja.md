# EMA/SMA + RSI クロスオーバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、3つの指数移動平均（速い、中間、遅い）とRSIフィルターを追跡し、
新興トレンドに参加します。速い平均が支配的な遅い平均の方向に向かって中間を
クロスすると、モメンタムが加速していることを示し、取引が発動します。
ダマシを避けるため、クロスオーバーの方向に終値を付けるローソク足のみが
考慮されます。

オプションの保護的エグジットにより、ポジションが利益を上げている場合、
ユーザー定義の本数の足の後にクローズすることができます。RSIはモメンタムが
伸びすぎたときに退出するための買われすぎ/売られすぎガードとして機能します。

バックテストでは、移動平均が明確な分離を提供するトレンドフェーズにおいて、
流動性の高い暗号通貨ペアで最も効果的であることが示されています。

## 詳細

- **エントリー条件**:
  - **ロング**: `EMA_fast > EMA_medium` かつ `EMA_fast(t-1) <= EMA_medium(t-1)` かつ `Close > EMA_slow` かつ `Close > Open`
  - **ショート**: `EMA_fast < EMA_medium` かつ `EMA_fast(t-1) >= EMA_medium(t-1)` かつ `Close < EMA_slow` かつ `Close < Open`
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: `RSI > 70` または `X本の足で利益かつ Close > entry`
  - **ショート**: `RSI < 30` または `X本の足で利益かつ Close < entry`
- **ストップ**: なし。
- **デフォルト値**:
  - `EMA_fast` = 10
  - `EMA_medium` = 20
  - `EMA_slow` = 100
  - `RSI_length` = 14
  - `X bars` = 24
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA, RSI
  - ストップ: オプションの時間ベース
  - 複雑さ: 中
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
