# 平均高低レンジ IBS リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格が平均高低レンジから導出された動的な閾値を下回った後に平均回帰を狙います。バーのレンジの移動平均、観測期間における最高値と最低値を計算します。買い閾値は最高値から平均レンジの2.5倍を引いた値として定義されます。価格が指定されたバー数だけこのレベルを下回り、取引ウィンドウ内でバー内強度（IBS）が所定の限界を下回ったとき、ロングポジションが建てられます。終値が前のバーの高値を超えた場合、ポジションが決済されます。

## 詳細

- **エントリー条件**:
  - 価格が `BarsBelowThreshold` バルの間、買い閾値を下回り続けた。
  - IBS < `IbsBuyThreshold`。
  - `StartTime` と `EndTime` の間の時刻。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - 終値が前のバーの高値を超える。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 20
  - `BarsBelowThreshold` = 2
  - `IbsBuyThreshold` = 0.2
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロング
  - インジケーター: SMA, Highest, Lowest
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
