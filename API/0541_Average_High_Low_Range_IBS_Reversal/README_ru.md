# Стратегия Average High-Low Range IBS Reversal
[English](README.md) | [中文](README_cn.md)

Эта стратегия ищет возврат к среднему после того, как цена удерживается ниже динамического порога, основанного на среднем диапазоне между максимумом и минимумом. Рассчитывается скользящая средняя диапазона бара, а также наивысший максимум и наименьший минимум за период. Порог покупки определяется как наивысший максимум минус 2.5 среднего диапазона. Когда цена остаётся ниже этого уровня заданное количество баров и показатель IBS падает ниже заданного значения в пределах торгового окна, открывается длинная позиция. Позиция закрывается, если цена закрытия превышает максимум предыдущего бара.

## Подробности

- **Условия входа**:
  - Цена находится ниже порога покупки в течение `BarsBelowThreshold` баров.
  - IBS < `IbsBuyThreshold`.
  - Время между `StartTime` и `EndTime`.
- **Направление**: только лонг.
- **Условия выхода**:
  - Цена закрытия выше максимума предыдущего бара.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `Length` = 20
  - `BarsBelowThreshold` = 2
  - `IbsBuyThreshold` = 0.2
- **Фильтры**:
  - Категория: Возврат к среднему
  - Направление: Лонг
  - Индикаторы: SMA, Highest, Lowest
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
