# スキャルピングEA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

常に2つの指値注文を維持するシンプルなスキャルピングシステムです。市場より上のバイストップと下のセルストップを保ちます。市場価格が注文に近づきすぎるか離れすぎると、現在価格から一定距離を保つよう注文が置き換えられます。約定した注文には固定のテイクプロフィットとストップロスが設定されています。

この戦略はインジケーターに依存せず、ティック価格の変化にのみ反応します。

## 詳細

- **エントリー条件**:
  - 価格の100ポイント上にバイストップ、100ポイント下にセルストップを設置。
  - 価格との距離が狭くなりすぎるか広くなりすぎると注文を置き換える。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 各注文には固定のテイクプロフィットとストップロスが付く。
- **ストップ**: はい、固定距離。
- **フィルター**: なし。
