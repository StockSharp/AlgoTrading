# 剥头皮EA策略
[English](README.md) | [Русский](README_ru.md)

该策略模拟一个简单的剥头皮EA。系统始终保持两个挂单：在价格上方的买入止损和在下方的卖出止损。当市场价格与挂单的距离过近或过远时，订单会被替换以保持固定间距。每个成交订单都带有固定的止盈和止损。

策略不使用任何指标，仅对逐笔价格变化作出反应。

## 细节

- **入场条件**：
  - 在价格上方100点挂买入止损，下方100点挂卖出止损。
  - 当与市场的间距过小或过大时重新挂单。
- **方向**：双向。
- **出场条件**：
  - 每个挂单都带有固定止盈和止损。
- **止损**：固定距离。
- **过滤器**：无。
