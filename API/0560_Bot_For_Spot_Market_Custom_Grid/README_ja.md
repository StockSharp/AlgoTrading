# スポット市場向けボット - カスタムグリッド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

スポット市場向けボット - カスタムグリッド戦略は、初期ポジションを買い建てた後、価格が最後のエントリーから指定のパーセント分下落するたびに新規注文を追加します。価格が平均エントリー価格を利益目標分上回ったときに全ポジションをクローズします。

## 詳細

- **エントリー条件**:
  - 開始時に買い。
  - 価格が最後のエントリーより`NextEntryPercent`%下落したときに追加買い。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - 価格が平均エントリー価格を`ProfitPercent`%上回り、かつオープンポジションが利益を出しているときに全ポジションをクローズ。
- **ストップ**: なし。
- **デフォルト値**:
  - `OrderValue` = 10
  - `MinAmountMovement` = 0.00001
  - `Rounding` = 5
  - `NextEntryPercent` = 0.5
  - `ProfitPercent` = 2
- **フィルター**:
  - カテゴリ: グリッドトレーディング
  - 方向: ロングのみ
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
