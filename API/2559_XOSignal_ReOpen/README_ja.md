# XOSignal 再オープン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTraderのエキスパート *Exp_XOSignal_ReOpen* を高レベルAPIを使用してStockSharp内で再現します。ATR(13)上に構築されたXOスタイルブレイクアウト検出器で選択された銘柄と時間軸のローソク足データを取引します。上矢印が現れると、アルゴリズムはショートをクローズし、オプションでロングを開き、価格が固定ティック数だけ進むたびにポジションに追加します。下矢印はショートに対して対称的に動作します。ハードストップとターゲット（ティック単位）がピラミッドの各層に適用されます。

## コアロジック

- 戦略はバンドが `Range * PriceStep` だけ拡張するXO範囲チャネルを計算します。ブレイクアウトはバンドをリセットし、現在のトレンド方向を確立します。
- ATR(13)は仮想エントリーレベル（矢印）がローソク足からどれだけ下/上に描かれるかを制御します：ロング矢印は `Low - ATR * 3/8` に、ショート矢印は `High + ATR * 3/8` に現れます。
- 完成したローソク足のみが処理されます。元のバッファリングロジックを模倣するためにシグナルを `SignalBar` バー遅延させることができます。

## エントリールール

- **ロングエントリー**: 上矢印が発生し、ロングエントリーが許可されている（`EnableBuyEntries = true`）、ショートポジションが開いていない、シグナルがまだ実行されていない場合。取引数量は `Volume` です。
- **ロング再エントリー**: ロングポジション中、ローソク足のクローズに基づいてトレードに有利な方向に追加の `PriceStepTicks` ティックごとに、`MaxPyramidingPositions` 層が開かれるまで別の買い注文がトリガーされます。各再エントリーは保護的なストップ/ターゲットレベルを更新します。
- **ショートエントリー/再エントリー**: 下矢印を使用したロングサイドのミラーロジック。

## エグジットルール

- **シグナルベースのエグジット**: 上矢印は `EnableSellExits = true` のときにすべてのアクティブなショートをクローズします；下矢印は `EnableBuyExits = true` のときにロングをクローズします。
- **リスクエグジット**: 各オープン層はティックで定義された同じストップロスとテイクプロフィット距離を持ちます（`StopLossTicks`、`TakeProfitTicks`）。価格が現在のローソク足内でレベルを突き抜けると、ポジション全体がフラット化されます。
- **手動フラット化**: 反対方向のエントリーシグナルも新しいポジションを開く前に前の方向を中立化します。

## ポジション管理

- ポジションサイズは各注文に対して `Volume` で固定されています。
- ストップロスとテイクプロフィットは証券ティックで測定されます。ゼロに設定すると対応する保護が無効になります。
- ピラミッドカウンターは完全なエグジットの後にゼロにリセットされ、次のシグナルが新鮮なベースポジションから始まるようにします。

## パラメーター

| パラメーター | 説明 | デフォルト値 |
|------------|------|------------|
| `Volume` | 各エントリーの注文サイズ | `1` |
| `StopLossTicks` | ティックでのストップ距離、0で無効化 | `1000` |
| `TakeProfitTicks` | ティックでのテイクプロフィット距離、0で無効化 | `2000` |
| `PriceStepTicks` | ポジションに追加する前の最小有利な動き | `300` |
| `MaxPyramidingPositions` | 積み上げエントリーの最大数（最初を含む） | `10` |
| `EnableBuyEntries` / `EnableSellEntries` | ロング/ショートポジションの開設を許可 | `true` |
| `EnableBuyExits` / `EnableSellExits` | 反対の矢印でロング/ショートポジションのクローズを許可 | `true` |
| `CandleType` | シグナルに使用する時間軸 | `H4` |
| `Range` | ティックでのXOボックスの高さ | `10` |
| `AppliedPrice` | XO検出器で使用する価格ソース | `Close` |
| `SignalBar` | シグナルを遅延させる終値バーの数 | `1` |

戦略は信頼性の高い価格ステップを提供する銘柄でのバックテストまたはライブ取引向けに設計されています。選択した市場のボラティリティに合わせてティックベースの距離を調整します。
