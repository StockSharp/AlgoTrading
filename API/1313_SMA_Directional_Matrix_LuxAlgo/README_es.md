# Estrategia SMA Directional Matrix LuxAlgo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Recrea el indicador SMA Directional Matrix de LuxAlgo. Para cada barra compara el cierre actual con cierres pasados en un rango configurable y registra flechas arriba o abajo en un diseño de matriz.

## Parámetros
- Tipo de vela
- Retroceso mínimo
- Retroceso máximo
- Columnas
