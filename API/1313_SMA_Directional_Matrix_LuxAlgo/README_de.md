# SMA Directional Matrix LuxAlgo-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Bildet den SMA Directional Matrix-Indikator von LuxAlgo nach. Für jeden Bar vergleicht er den aktuellen Schlusskurs mit vergangenen Schlusskursen über einen konfigurierbaren Bereich und gibt Aufwärts- oder Abwärtspfeile in einem Matrixlayout aus.

## Parameter
- Kerzentyp
- Minimaler Rückblick
- Maximaler Rückblick
- Spalten
