# SMA Directional Matrix LuxAlgo戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

LuxAlgoのSMA Directional Matrixインジケーターを再現します。各バーについて、設定可能な範囲で現在の終値と過去の終値を比較し、マトリックスレイアウトで上矢印または下矢印を記録します。

## パラメーター
- ローソク足タイプ
- 最小ルックバック
- 最大ルックバック
- 列数
