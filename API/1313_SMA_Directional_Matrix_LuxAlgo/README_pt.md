# Estratégia SMA Directional Matrix LuxAlgo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Recria o indicador SMA Directional Matrix do LuxAlgo. Para cada barra compara o fechamento atual com fechamentos passados em um intervalo configurável e registra setas para cima ou para baixo em um layout de matriz.

## Parâmetros
- Tipo de candle
- Lookback mínimo
- Lookback máximo
- Colunas
