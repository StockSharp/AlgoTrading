# LANZ-Strategie 3.0
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die LANZ-Strategie 3.0 handelt Ausbrüche aus der asiatischen Range. Die Richtung wird nach dem Entscheidungsfenster von 01:15–02:15 New Yorker Zeit gewählt und eine Limitorder wird am Range-Hoch oder -Tief mit Fibonacci-basierten Zielen und Stops platziert. Wenn die Order bis 02:15 nicht ausgeführt wird, kann sie die Richtung wechseln. Nicht ausgeführte Orders werden um 08:00 storniert und offene Positionen um 15:45 geschlossen.

## Details

- **Einstiegskriterien**:
  - Ausbruch aus dem Hoch oder Tief der asiatischen Range nach dem Entscheidungsfenster.
- **Long/Short**: Beide.
- **Ausstiegskriterien**:
  - Fibonacci-basierter Take-Profit oder Stop-Loss.
  - Alle Positionen werden um 15:45 NY geschlossen.
- **Stops**: Fibonacci-Multiplikatoren.
- **Standardwerte**:
  - `UseOptimizedFibo` = true
- **Filter**:
  - Kategorie: Ausbruch
  - Richtung: Beide
  - Indikatoren: Keine
  - Stops: Ja
  - Komplexität: Fortgeschritten
  - Zeitrahmen: Beliebig
  - Saisonalität: Ja
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
