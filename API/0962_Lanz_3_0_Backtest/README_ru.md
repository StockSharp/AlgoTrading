# Стратегия LANZ 3.0
[English](README.md) | [中文](README_cn.md)

Стратегия LANZ 3.0 торгует пробои азиатского диапазона. Направление выбирается в окне принятия решения 01:15–02:15 по Нью-Йорку, затем лимитная заявка размещается на границе диапазона с целями и стопами по уровням Фибоначчи. Если до 02:15 заявка не исполнилась, стратегия может изменить направление. Неисполненные заявки отменяются в 08:00, а все позиции закрываются в 15:45.

## Подробности

- **Условия входа**:
  - Пробой максимума или минимума азиатского диапазона после окна принятия решения.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Тейк-профит или стоп-лосс по уровням Фибоначчи.
  - Все позиции закрываются в 15:45 NY.
- **Стопы**: Фибоначчи.
- **Параметры по умолчанию**:
  - `UseOptimizedFibo` = true
- **Фильтры**:
  - Категория: Пробой
  - Направление: Лонг и шорт
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Высокая
  - Таймфрейм: Любой
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
