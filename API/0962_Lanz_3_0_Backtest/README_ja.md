# LANZ戦略 3.0
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

LANZ戦略3.0はアジアレンジのブレイクアウトを取引します。01:15–02:15ニューヨーク時間の意思決定ウィンドウ後に方向が決まり、レンジ高値または安値でフィボナッチベースのターゲットとストップを設定した指値注文が置かれます。02:15までに注文が成立しない場合は方向が反転することがあります。未成立の注文は08:00にキャンセルされ、オープンポジションは15:45に決済されます。

## 詳細

- **エントリー条件**:
  - 意思決定ウィンドウ後のアジアレンジ高値または安値のブレイクアウト。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - フィボナッチベースのテイクプロフィットまたはストップロス。
  - すべてのポジションをNY時間15:45に決済。
- **ストップ**: フィボナッチ乗数。
- **デフォルト値**:
  - `UseOptimizedFibo` = true
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 上級
  - 時間軸: 任意
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
