# Стратегия SuperTrend AI Oscillator
[English](README.md) | [中文](README_cn.md)

SuperTrend AI Oscillator объединяет трейлинг-стоп SuperTrend с фильтром из нестандартного осциллятора.
Стратегия входит при развороте SuperTrend, подтверждённом осциллятором.
Позиции сопровождаются трейлинг-стопом и опциональной целью по соотношению риск/прибыль.

## Детали

- **Критерий входа**: разворот SuperTrend с осциллятором > 50 для лонга или < 50 для шорта
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: трейлинг-стоп или цель по соотношению риск/прибыль
- **Стопы**: трейлинг-стоп
- **Значения по умолчанию**:
  - `AtrLength` = 10
  - `Factor` = 1
  - `RiskReward` = 2
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: ATR, Stochastic
  - Стопы: Трейлинг
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
