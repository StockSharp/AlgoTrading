# Binario 31戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MetaTraderスクリプト **binario_31** から変換されたブレイクアウト戦略です。アルゴリズムはローソク足の高値と安値に対してそれぞれ144期間の指数移動平均を構築し、動的なチャネルを形成します。現在の価格がチャネル内にある間、戦略はストップエントリー注文を準備します：

- EMA高値に設定可能なオフセットを加えた位置に買いストップ注文を配置；
- EMA安値から同じオフセットを引いた位置に売りストップ注文を配置。

価格がこれらのレベルのいずれかをブレイクすると、ブレイクアウトの方向にポジションが開かれます。保護ストップはチャネルの反対側に置かれ、テイクプロフィットの目標はエントリーからの相対値で計算されます。利益を守るためにオプションのトレーリングストップを有効にすることができます。

## パラメーター

- **EMA Length** – 高値・安値両方のEMAの期間。
- **Pip Difference** – EMAレベルからブレイクアウトエントリーまでの価格ステップ単位の距離。
- **Take Profit** – エントリーからテイクプロフィットまでの価格ステップ単位の距離。
- **Trailing Stop** – 価格ステップ単位のトレーリングストップ距離。無効にするにはゼロに設定。
- **Volume** – 注文の数量。
- **Candle Type** – 戦略がサブスクライブするローソク足の種類。
