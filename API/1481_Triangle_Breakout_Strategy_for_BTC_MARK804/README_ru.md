# Стратегия Triangle Breakout для BTC
[English](README.md) | [中文](README_cn.md)

Открывает позиции при пробое треугольника на простых скользящих средних при всплеске объёма и управляет сделками через ATR-стопы.

## Детали

- **Критерий входа**: цена пересекает верхнюю или нижнюю SMA линию при объёме выше своей SMA
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: стоп-лосс или тейк-профит по ATR
- **Стопы**: да
- **Значения по умолчанию**:
  - `TriangleLength` = 50
  - `VolumeSmaLength` = 20
  - `AtrLength` = 14
  - `VolumeMultiplier` = 1.5
  - `AtrMultiplierSl` = 1.0
  - `AtrMultiplierTp` = 1.5
  - `CandleType` = 1 час
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе стороны
  - Индикаторы: SMA, ATR, Объём
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
