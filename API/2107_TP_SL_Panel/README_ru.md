# Стратегия панели TP/SL
[English](README.md) | [中文](README_cn.md)

Эта стратегия имитирует простую панель для установки тейк-профита и стоп-лосса. Заявки не выставляются как отложенные. Вместо этого стратегия следит за текущей позицией и закрывает её рыночными ордерами при достижении указанных ценовых уровней.

## Подробности
- **Условия входа**:
  - Автоматический вход отсутствует. Стратегия управляет уже открытой позицией.
- **Long/Short**: зависит от текущей позиции.
- **Условия выхода**:
  - **Long**: закрытие при цене >= `TakeProfitPrice` или цене <= `StopLossPrice`.
  - **Short**: закрытие при цене <= `TakeProfitPrice` или цене >= `StopLossPrice`.
- **Стопы**: да, по ценовым уровням.
- **Параметры по умолчанию**:
  - `TakeProfitPrice` = 0 (выключено)
  - `StopLossPrice` = 0 (выключено)
- **Фильтры**:
  - Категория: Сервисная
  - Направление: зависит от позиции
  - Индикаторы: нет
  - Стопы: да
  - Сложность: низкая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
