# TP SL 面板策略
[English](README.md) | [Русский](README_ru.md)

该策略实现了一个简易的止盈/止损面板。它不下达挂单，而是监视当前持仓，当价格触及设定的止盈或止损水平时，以市价平仓。

## 详细信息
- **入场条件**:
  - 不自动开仓，只管理已有持仓。
- **多空方向**: 取决于当前持仓。
- **退出条件**:
  - **多头**: 价格 >= `TakeProfitPrice` 或 价格 <= `StopLossPrice` 时平仓。
  - **空头**: 价格 <= `TakeProfitPrice` 或 价格 >= `StopLossPrice` 时平仓。
- **止损**: 是，基于价格水平。
- **默认值**:
  - `TakeProfitPrice` = 0（禁用）
  - `StopLossPrice` = 0（禁用）
- **筛选**:
  - 分类: 工具
  - 方向: 取决于持仓
  - 指标: 无
  - 止损: 是
  - 复杂度: 低
  - 时间框架: 任意
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
