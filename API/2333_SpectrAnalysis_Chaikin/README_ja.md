# SpectrAnalysis Chaikin戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はChaikinオシレーターを使用してモメンタムの変化を検出します。オシレーターは、2つの線形加重移動平均で平滑化された累積/分配ラインから計算されます。オシレーターの傾きが上向きに転換し、最新値が前の値を上回ると、ロングポジションがオープンされます。反対に、傾きが下向きに転換し、最新値が前の値を下回ると、ショートポジションがオープンされます。

## パラメーター

| 名前 | 説明 |
|------|------|
| `FastMaPeriod` | Chaikinオシレーターで使用する高速線形加重移動平均の期間。 |
| `SlowMaPeriod` | Chaikinオシレーターで使用する低速線形加重移動平均の期間。 |
| `BuyPosOpen` | ロングポジションのオープンを有効化。 |
| `SellPosOpen` | ショートポジションのオープンを有効化。 |
| `BuyPosClose` | 条件を満たした際にロングポジションのクローズを有効化。 |
| `SellPosClose` | 条件を満たした際にショートポジションのクローズを有効化。 |
| `CandleType` | 計算に使用するローソク足の時間軸。 |

## 注意事項

- エントリーおよびエグジットには成行注文が使用されます。
- 戦略はストップロスやテイクプロフィット注文を設定しません。
- C#バージョンのみ提供され、Pythonの実装は含まれません。
