# CashMachine 5分足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は MQL の **CashMachine 5min** Expert Advisor を StockSharp 高レベル API に直接変換したものです。5 分足ローソク足用に設計されており、DeMarker インジケーターとストキャスティクスのクロスオーバーフィルターを組み合わせています。トレード管理は隠れたストップロス / テイクプロフィットレベルと段階的なトレーリングルールを使用し、価格モメンタムが現れたら利益を確定しようとします。

## トレードロジック
### エントリー条件
- **ロング**: 前の DeMarker 値が 0.30 未満で現在値が 0.30 以上**かつ**同じローソク足でストキャスティクス %K が 20 を上抜け。ポジションがオープンでないこと。
- **ショート**: 前の DeMarker 値が 0.70 超で現在値が 0.70 以下**かつ**ストキャスティクス %K が 80 を下抜け。ポジションがオープンでないこと。

### ポジション管理
- 一度に 1 つのポジションのみ維持されます。現在のトレードが決済されるまで、反対のシグナルは無視されます。
- 隠れた決済は価格が `Entry ± HiddenStopLoss` または `Entry ± HiddenTakeProfit`（pips で解釈された値）に触れたときにポジションを決済します。
- 3 つの中間利益ターゲット（`TargetTp1/2/3`）は、ロングの場合 `現在価格 - (ターゲット - 13)` pips、ショートの場合 `現在価格 + (ターゲット + 13)` pips に隠れたトレーリングストップを移動します。追加の 13 pips は元の EA 動作を模倣し、即座に決済せずに各マイルストーン後に利益を確定します。
- トレーリングストップが有効化後に触れられた場合、ポジションは成行で決済されます。

## インジケーター
- **DeMarker** – モメンタムのリバーサルを検出します。長さパラメーターは元の平均化期間と一致します。
- **ストキャスティクスオシレーター** – 元の %K 期間（`StochasticLength`）、%K 平滑化（`StochasticK`）、%D 平滑化（`StochasticD`）を使用します。

## パラメーター
| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| `HiddenTakeProfit` | 隠れたテイクプロフィット距離（pips）。 | 60 |
| `HiddenStopLoss` | 隠れたストップロス距離（pips）。 | 30 |
| `TargetTp1` | 最初のトレーリング有効化レベル（pips）。 | 20 |
| `TargetTp2` | 2 番目のトレーリング有効化レベル（pips）。 | 35 |
| `TargetTp3` | 3 番目のトレーリング有効化レベル（pips）。 | 50 |
| `DeMarkerLength` | DeMarker 平均化期間。 | 14 |
| `StochasticLength` | ストキャスティクス %K ルックバック期間。 | 5 |
| `StochasticK` | %K 平滑化の長さ。 | 3 |
| `StochasticD` | %D 平滑化の長さ。 | 3 |
| `CandleType` | 計算に使用するローソク足シリーズ（デフォルト 5 分）。 | 5 分足 |

## 注意事項
- pip サイズは `Security.PriceStep` から算出されます。ステップが不明な場合は `0.0001` のフォールバック値が使用され、3 桁・5 桁の相場に対応する EA ロジックを再現します。
- すべてのトレード決定は完了したローソク足に基づきます。MQL バージョンはすべてのティックで実行されていたため、元の EA のバー内動作は若干異なる場合があります。
- 戦略は StockSharp の標準的な注文ボリューム処理に依存しています。トレードサイズを制御するには `Strategy.Volume` を設定してください。
