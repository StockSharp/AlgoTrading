# Стратегия BB RSI Trailing Stop
[English](README.md) | [中文](README_zh.md)

Комбинирует полосы Боллинджера с RSI и использует условный трейлинг-стоп.
Покупка происходит при пробое нижней полосы и выходе RSI из перепроданности. Продажа — при пробое верхней полосы и выходе RSI из перекупленности.

Стоп-лосс задаётся фиксированным расстоянием и превращается в трейлинг после достижения заданной прибыли.

## Детали

- **Критерии входа**: пробой полосы Боллинджера с подтверждением RSI
- **Длинные/короткие**: обе стороны
- **Критерии выхода**: начальный стоп или трейлинг-стоп
- **Стопы**: да, динамический трейлинг
- **Значения по умолчанию**:
  - `BollingerPeriod` = 25
  - `BollingerDeviation` = 2
  - `RsiPeriod` = 14
  - `RsiOverbought` = 60
  - `RsiOversold` = 33
  - `StopLossPoints` = 50
  - `TrailOffsetPoints` = 99
  - `TrailStopPoints` = 40
- **Фильтры**:
  - Категория: Контртренд
  - Направление: Оба
  - Индикаторы: Полосы Боллинджера, RSI
  - Стопы: Трейлинг
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
