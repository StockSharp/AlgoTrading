# 複数ポジションの押し目買い戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Buy Dip Multiple Positions戦略は、価格の押し目が高出来高と価格サージ条件とともに発生した際にロングポジションを追加します。各トレードは資産の2%をリスクにさらし、共通のトレーリングストップと目標レベルを共有します。前回のクローズドトレードが利益を上げた場合にのみ新規ポジションが建てられます。

## 詳細

- **エントリー条件**:
  - 終値が前回の安値を0.2%下回る。
  - 出来高が直近2本のバーの平均の120%を超える。
  - 終値がN本前の終値に `PriceSurgePercent` / 100 を掛けた値を下回る。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - エントリーバーの安値に対するパーセンテージとしての初期ストップ。
  - セットアップ後、毎バー増加するトレーリングストップ。
  - エントリーバーの安値を上回る目標価格。
- **ストップ**: はい。
- **デフォルト値**:
  - `MaxPositions` = 20
  - `TrailRatePercent` = 1
  - `InitialStopPercent` = 85
  - `TargetPricePercent` = 60
  - `PriceSurgePercent` = 89
  - `SurgeLookbackBars` = 14
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロング
  - インジケーター: 出来高、価格アクション
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
