# Стратегия Buy Dip Multiple Positions
[English](README.md) | [中文](README_cn.md)

Стратегия Buy Dip Multiple Positions добавляет длинные позиции при снижении цены вместе с повышенным объёмом и условием ценового обвала. Каждая сделка рискует 2% капитала и использует общий трейлинг-стоп и цель. Новая позиция открывается только если предыдущая сделка была прибыльной.

## Подробности

- **Условия входа**:
  - Закрытие ниже предыдущего минимума на 0.2%.
  - Объём выше 120% среднего объёма двух предыдущих баров.
  - Закрытие ниже цены закрытия N баров назад, умноженной на `PriceSurgePercent` / 100.
- **Направление**: Только лонг.
- **Условия выхода**:
  - Начальный стоп как процент от минимума входного бара.
  - Трейлинг-стоп увеличивается каждый бар после сигнала.
  - Целевая цена выше минимума входного бара.
- **Стопы**: Да.
- **Параметры по умолчанию**:
  - `MaxPositions` = 20
  - `TrailRatePercent` = 1
  - `InitialStopPercent` = 85
  - `TargetPricePercent` = 60
  - `PriceSurgePercent` = 89
  - `SurgeLookbackBars` = 14
- **Фильтры**:
  - Категория: Контртренд
  - Направление: Лонг
  - Индикаторы: Объём, Прайс-экшен
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
