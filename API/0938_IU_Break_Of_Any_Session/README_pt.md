# Estratégia IU de Rompimento de Qualquer Sessão
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia registra o máximo e mínimo de uma sessão personalizada e opera rompimentos desse intervalo durante a janela de negociação. O stop-loss é colocado no extremo da vela de entrada e o take profit utiliza uma relação risco/recompensa configurável. Todas as posições são fechadas no horário de saída especificado.
