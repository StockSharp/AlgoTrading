# Estrategia IU de Ruptura de Cualquier Sesión
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia registra el máximo y mínimo de una sesión personalizada y opera rupturas de ese rango durante la ventana de negociación. El stop-loss se coloca en el extremo de la vela de entrada y el take profit utiliza una relación riesgo/recompensa configurable. Todas las posiciones se cierran en el tiempo de salida especificado.
