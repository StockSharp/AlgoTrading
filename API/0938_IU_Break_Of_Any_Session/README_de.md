# IU Break-of-Any-Session-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie erfasst das Hoch und Tief einer benutzerdefinierten Session und handelt Ausbrüche aus dieser Range während des Handelsfensters. Der Stop-Loss wird am Extrempunkt der Einstiegskerze gesetzt und der Take Profit verwendet ein konfigurierbares Risiko-Ertrags-Verhältnis. Alle Positionen werden zur angegebenen Ausstiegszeit geschlossen.
