# 多货币交易面板策略

该策略模拟原始 MQL “多货币交易面板”EA 的行为。它监控三种货币对（EURUSD、USDJPY、GBPUSD），并使用七个简单指标比较最新K线与上一根K线（开盘价、最高价、最低价、(高+低)/2、收盘价、(高+低+收)/3、(高+低+收+收)/4）。
每次比较会增加买入或卖出计数。启用自动交易时，如果买入计数大于卖出计数或相反，策略将在相应货币对上开仓或反转仓位。

## 参数
- **EURUSD** – 第一只证券。
- **USDJPY** – 第二只证券。
- **GBPUSD** – 第三只证券。
- **Candle Type** – K线时间框架。
- **Auto Trade** – 是否允许自动下单。

此策略仅用于演示，不适合真实交易。
