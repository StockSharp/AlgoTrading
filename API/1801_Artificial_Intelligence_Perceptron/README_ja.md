# 人工知能Perceptron戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はAccelerator Oscillator (AC)の値を単純な線形パーセプトロンへの入力として使用します。7バー間隔の4つのAC値がユーザー定義の係数で重み付けされます。パーセプトロンの出力が正の場合はロングポジションが開かれ、負の場合はショートポジションが開かれます。

この戦略は常にストップロスを適用します。利益がストップロスの2倍を超えた後に反対のシグナルが現れた場合、ポジションは増加したボリュームで反転します。それ以外の場合は、ストップロスがブレイクイーブンに移動されます。

## 詳細

- **エントリー条件**:
  - **ロング**: パーセプトロン出力 > 0。
  - **ショート**: パーセプトロン出力 < 0。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 利益 > 2 * StopLoss の反対シグナル → 反転。
  - より小さな利益での反対シグナル → ストップをエントリーに移動。
  - ストップロスに到達。
- **ストップ**: 固定ストップロス（ポイント）。
- **フィルター**: なし。

## パラメーター
- `StopLoss` – ストップロス距離（ポイント、デフォルト850）。
- `Shift` – インジケーター値のバーシフト（デフォルト1）。
- `X1`, `X2`, `X3`, `X4` – パーセプトロンの重み。
- `CandleType` – ローソク足の時間軸。
