# Indikatorbasierte Aufbau-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert einen EMA-Trendfilter mit einem RSI-Auslöser. Long-Trades werden eröffnet, wenn der Preis über der EMA liegt und der RSI überverkauft ist, während Short-Trades ausgelöst werden, wenn der Preis unter der EMA liegt und der RSI überkauft ist.
