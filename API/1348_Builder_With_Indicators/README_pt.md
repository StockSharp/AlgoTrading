# Estratégia Construtora com Indicadores
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina um filtro de tendência EMA com um gatilho RSI. Operações compradas são abertas quando o preço está acima da EMA e o RSI está sobrevendido, enquanto as vendidas são acionadas quando o preço está abaixo da EMA e o RSI está sobrecomprado.
