# Estrategia Constructora con Indicadores
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia combina un filtro de tendencia EMA con un disparador RSI. Las operaciones largas se abren cuando el precio está por encima de la EMA y el RSI está en zona de sobreventa, mientras que las cortas se activan cuando el precio está por debajo de la EMA y el RSI está en zona de sobrecompra.
