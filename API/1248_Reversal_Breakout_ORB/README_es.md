# Estrategia de Reversión y Ruptura con ORB
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia combina señales de reversión de RSI y SMA con rupturas de cruce de EMA y un Opening Range Breakout (ORB). Los stops basados en ATR y dos niveles de toma de ganancias gestionan el riesgo.
