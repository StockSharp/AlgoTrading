# Стратегия Bollinger Bands SMA 20-2
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует полосы Боллинджера на основе простого скользящего среднего с периодом 20 и множителем стандартного отклонения 2. Покупка выполняется, когда цена пересекает нижнюю полосу снизу вверх, а продажа — когда цена пересекает верхнюю полосу сверху вниз. Позиции переворачиваются при появлении противоположного сигнала без явных стопов.

## Детали

- **Условия входа**:
  - **Long**: `Close` пересекает снизу вверх нижнюю полосу.
  - **Short**: `Close` пересекает сверху вниз верхнюю полосу.
- **Направление торговли**: оба направления.
- **Условия выхода**:
  - Противоположный сигнал.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `Bollinger Length` = 20
  - `Bollinger Multiplier` = 2
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
