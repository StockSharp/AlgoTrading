# Estratégia de Média Móvel Adaptativa de Kaufman
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia utiliza a Média Móvel Adaptativa de Kaufman (KAMA). Uma posição comprada é aberta quando a KAMA sobe por um número específico de barras e qualquer posição vendida é encerrada. Uma posição vendida é aberta quando a KAMA cai pelo período necessário e qualquer posição comprada é encerrada.

## Parâmetros
- Tipo de candle
- Comprimento
- Período rápido
- Período lento
- Período de subida
- Período de queda
- Direção da ordem
