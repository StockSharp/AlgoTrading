# Kaufman Adaptive Moving Average-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet den Kaufman Adaptive Moving Average (KAMA). Eine Long-Position wird eröffnet, wenn der KAMA für eine bestimmte Anzahl von Bars steigt, und eine eventuell vorhandene Short-Position wird geschlossen. Eine Short-Position wird eröffnet, wenn der KAMA für den erforderlichen Zeitraum fällt, und eine eventuell vorhandene Long-Position wird geschlossen.

## Parameter
- Kerzentyp
- Länge
- Schnelle Periode
- Langsame Periode
- Steigende Periode
- Fallende Periode
- Orderrichtung
