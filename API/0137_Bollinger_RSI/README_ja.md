# Bollinger RSI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
Bollinger RSI は、ボリンジャーバンドの過伸展と RSI のモメンタムシグナルを組み合わせます。
価格がバンドの外側でクローズしているが RSI がダイバージェンスを示している場合、反転が近いことが多いです。

テストでは平均年間リターンは約148%を示しています。外国為替市場で最も良いパフォーマンスを発揮します。

このシステムはそのダイバージェンスで逆張りトレードを行い、価格がバンド内に戻るか RSI が反対にクロスしたときにイグジットします。

ボラティリティがさらに拡大する場合に備え、タイトなパーセントストップがエクスポージャーを制限します。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: Bollinger Bands, RSI
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
