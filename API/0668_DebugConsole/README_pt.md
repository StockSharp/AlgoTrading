# Estratégia Debug Console
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Demonstra um console simples que coleta mensagens durante a execução da estratégia e as envia para o registro.

## Detalhes

- **Propósito**: demonstrar o enfileiramento de mensagens para depuração.
- **Mensagens intrabarra**: opcional, desativado por padrão.
- **Tamanho do console**: número configurável de mensagens a manter.
