# Estrategia Debug Console
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demuestra una consola simple que recopila mensajes durante la ejecución de la estrategia y los envía al registro.

## Detalles

- **Propósito**: demostrar la cola de mensajes para depuración.
- **Mensajes intrabarra**: opcional, desactivado por defecto.
- **Tamaño de la consola**: número configurable de mensajes a conservar.
