# Debug Console-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demonstriert eine einfache Konsole, die Nachrichten während der Strategieausführung sammelt und ins Protokoll ausgibt.

## Details

- **Zweck**: Veranschaulichung der Nachrichtenwarteschlange für das Debugging.
- **Intrabar-Nachrichten**: optional, standardmäßig deaktiviert.
- **Konsolengröße**: konfigurierbare Anzahl zu speichernder Nachrichten.
