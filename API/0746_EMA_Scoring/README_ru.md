# Стратегия EMA Scoring
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Эта стратегия использует три линии EMA для оценки направления рынка и торгует при пересечении порога оценки.

## Подробности
- **Условия входа**:
  - **Лонг**: Счёт пересекает порог снизу вверх.
  - **Шорт**: Счёт пересекает отрицательный порог сверху вниз.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Обратный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Short EMA Period` = 21
  - `Medium EMA Period` = 50
  - `Long EMA Period` = 100
  - `Score Threshold` = 4
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Среднесрочный
