# Стратегия "Медвежий разворот по хвосту"
[English](README.md) | [中文](README_cn.md)

Стратегия открывает покупки, когда медвежья свеча имеет длинный нижний хвост, превышающий заданный процент. Дополнительный фильтр EMA может требовать, чтобы цена находилась выше скользящей средней. Позиция закрывается при закрытии выше максимума предыдущей свечи.

## Детали

- **Критерии входа:** медвежья свеча с нижним хвостом <= порога и время в допустимом окне; опционально цена выше EMA.
- **Длинные/короткие:** только длинные.
- **Критерии выхода:** закрытие выше максимума предыдущей свечи.
- **Стопы:** нет.
- **Значения по умолчанию:**
  - Порог = -1 (%)
  - Фильтр EMA отключен, период EMA = 200
  - Временное окно: 2014-01-01 до 2099-01-01
  - Таймфрейм свечей = 1 минута
- **Фильтры:**
  - Категория: Разворот
  - Направление: Лонг
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
