# 看跌影线反转策略
[English](README.md) | [Русский](README_ru.md)

该策略在出现具有较长下影线的看跌K线时做多。可选的EMA过滤器要求收盘价高于指数均线以确认趋势。当价格收于上一根K线最高点之上时，策略平仓。

## 细节

- **入场条件：** 看跌K线的下影线百分比低于阈值并处于允许的时间窗口内；可选要求价格高于EMA。
- **多空方向：** 仅做多。
- **出场条件：** 收盘价高于前一根K线的最高价。
- **止损：** 无。
- **默认值：**
  - 阈值 = -1 (%)
  - EMA过滤关闭，EMA周期 = 200
  - 时间窗口：2014-01-01 至 2099-01-01
  - K线周期 = 1分钟
- **过滤器：**
  - 类别：反转
  - 方向：多头
  - 指标：EMA
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
