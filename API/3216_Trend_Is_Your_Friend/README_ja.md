# Trend Is Your Friend戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
Trend Is Your Friend戦略は、元のMetaTraderエキスパートアドバイザーにインスパイアされたマルチタイムフレームのトレンドフォローシステムです。イントラデイのモメンタムを上位時間軸のMACDフィルターと整合させ、ボリンジャーバンド出口、古典的なストップロスとテイクプロフィットターゲット、オプションのブレイクイーブンロック、トレーリングストップ管理でリスクを管理します。

この戦略は設定可能なベース時間軸（デフォルト：1時間）で動作し、短期モメンタムパターンのためにロウソク足構造を分析します：ロング取引では弱気のロウソク足に続くより強い強気のロウソク足、またはショート取引ではその逆。これらのパターンは、ポジションが開かれる前に移動平均トレンドフィルターと月次MACDシグナルと一致している必要があります。

## エントリーロジック
1. エントリー時間軸で速いEMAと遅いLWMAを計算します。
2. モメンタムパターンを形成するために直近2つの確定ロウソク足を追跡します：
   - **ロングセットアップ:** 2バー前のロウソク足が弱気、前のロウソク足が強気でより大きなマグニチュード。
   - **ショートセットアップ:** 2バー前のロウソク足が強気、前のロウソク足が弱気でより小さなマグニチュード。
3. 移動平均トレンドフィルターでセットアップを確認します（ロング取引では速いMAが遅いMAの上、ショートでは下）。
4. 上位時間軸（デフォルト：月次）で計算されたMACDシグナルで長期トレンドを確認します。MACD線はロング取引ではシグナル線の上、ショート取引では下にある必要があります。
5. すべてのフィルターが整合したとき、設定されたボリュームで成行ポジションを開きます。

## エグジットロジック
- **ボリンジャーバンド出口:** ロングポジションは価格が上部バンドを上回って確定したときに閉じられ、ショートポジションは価格が下部バンドを下回って確定したときに閉じられます。
- **テイクプロフィット/ストップロス:** pipで測定されたオプションの固定距離。実装は証券の価格ステップを介してpipを価格距離に変換します。
- **ブレイクイーブン:** オプション、設定可能な利益閾値に達した後に保護ストップをエントリー価格（またはそれ以上）に移動します。
- **トレーリングストップ:** オプション、利益閾値後に起動し、固定pip距離で価格を追跡します。トレーリングストップはブレイクイーブンレベルと同じストレージを共有します。

## パラメーター
| 名前 | 説明 | デフォルト値 |
| ---- | ----------- | ------- |
| Entry Candle | エントリーロジック用のロウソク足タイプ | 1時間 |
| MACD Candle | MACDフィルターに使用する上位時間軸 | 30日 |
| Fast MA | 速いEMAの長さ | 8 |
| Slow MA | 遅いLWMAの長さ | 20 |
| Bollinger Length | ボリンジャーバンド期間 | 20 |
| Bollinger Width | ボリンジャーバンドの標準偏差乗数 | 2.0 |
| Stop Loss (pips) | 保護ストップ距離 | 20 |
| Take Profit (pips) | 利益ターゲット距離 | 50 |
| Use Break-Even | ブレイクイーブン調整を有効化 | true |
| Break-Even Trigger | ストップを動かすのに必要な利益（pip） | 10 |
| Break-Even Offset | ブレイクイーブンストップに適用するオフセット | 5 |
| Use Trailing | トレーリングストップを有効化 | true |
| Trailing Activation | トレーリングを起動するのに必要な利益（pip） | 40 |
| Trailing Distance | トレーリングストップが維持する距離（pip） | 40 |

## 注意事項
- 戦略は重い過去バッファを避けるために直近2つの確定ロウソク足のみを保存します。
- MACDデータは集計を有効にして設定された上位時間軸から購読され、必要に応じて日次データから月次シグナルを構築できます。
- pip-価格変換は証券の価格ステップを使用します。非標準のpip定義を持つ楽器はパラメーターの調整が必要な場合があります。
