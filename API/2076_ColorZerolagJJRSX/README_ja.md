# Color Zerolag JJRSX 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MetaTraderの**ColorZerolagJJRSX**エキスパートのロジックを再現します。2つの平滑化されたRSIオシレーターを使用して、元のColorZerolagJJRSXインジケーターを近似します。高速線と低速線のクロスがトレードシグナルを生成します。

## 仕組み

- 高速オシレーターが低速オシレーターを**下抜け**したとき、戦略は既存のショートポジションを決済し、必要に応じて新しいロングポジションを開きます。
- 高速オシレーターが低速オシレーターを**上抜け**したとき、戦略は既存のロングポジションを決済し、必要に応じて新しいショートポジションを開きます。
- 保護的なストップロスとテイクプロフィットのレベルは、組み込みの`StartProtection`メカニズムを使って適用されます。

## パラメーター

| 名前 | 説明 |
| --- | --- |
| `FastPeriod` | 高速JJRSX線の期間。 |
| `SlowPeriod` | 低速JJRSX線の期間。 |
| `BuyOpen` | ロングポジションの開設を許可する。 |
| `SellOpen` | ショートポジションの開設を許可する。 |
| `BuyClose` | 反対シグナルで既存のロングポジションを決済する。 |
| `SellClose` | 反対シグナルで既存のショートポジションを決済する。 |
| `StopLoss` | 価格単位のストップロスレベル。 |
| `TakeProfit` | 価格単位のテイクプロフィットレベル。 |
| `CandleType` | 計算に使用する時間軸。 |

## 注記

- 実装は組み込みインジケーターと高レベルの`Bind` APIを使用します。
- ボリュームは戦略の`Volume`プロパティから取得されます。
- この戦略にはPythonバージョンはありません。
