# スナイパージョー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**スナイパー ジョー戦略**は、MetaTrader 4 エキスパート アドバイザー `SniperJawEA.mq4` を StockSharp のハイレベル戦略 API に移植します。システムは、ローソク足の中央値でビル Williams の Alligator インジケーターを分析します。取引は、3 つの平滑化移動平均 (顎、歯、唇) が厳密に強気または弱気の順序で積み上げられ、そのすべてが以前に完成したローソク足と比較して同じ方向に進んだ場合にのみ開始されます。

## 取引ロジック

1. **Alligator 再構築** – 3 つの `SmoothedMovingAverage` インスタンスは、ローソク足中央値 `(High + Low) / 2` 上の顎、歯、唇を計算します。各線は、MetaTrader のプロットを反映するために、独自のバーの数だけ前方にシフトできます。
2. **トレンドの確認** – シフトされた値が `jaw < teeth < lips` を満たし、**各線が前のローソク足よりも高い場合、ロング バイアスが生成されます。ショート バイアスでは、3 つの線すべてが前のバーと比較して低くなっている `jaw > teeth > lips` が必要です。
3. **エントリー管理** – この戦略は一度に 1 つのポジションのみをオープンします。 `UseEntryToExit` が有効になっていて、新しい反対のシグナルが発火すると、現在のエクスポージャーが最初に平坦化され、次のシグナルで新しい注文が送信されます。
4. **保護的出口** – ストップロスとテイクプロフィットの距離はピップ単位で定義され、セキュリティ `PriceStep` を使用して変換されます。ロングポジションとショートポジションの両方が、完成したローソク足ごとに監視され、いずれかのしきい値に達するとクローズされます。
5. **信号スロットリング** – 元の EA はバーのタイムスタンプをチェックすることで重複エントリを防止していました。ポートは最後のシグナルローソクタイムを保存し、同じ足の間に追加の注文をスキップします。

## パラメーター

| パラメータ | デフォルト | 説明 |
|-----------|---------|-------------|
| `OrderVolume` | `0.1` | `BuyMarket`/`SellMarket` に渡されるロットまたは契約の取引サイズ。 |
| `EnableTrading` | `true` | リスク管理をアクティブに保ちながら、新しいエントリを無効にすることができるマスター スイッチ。 |
| `UseEntryToExit` | `true` | 反対のシグナルを準備する前に、既存のポジションをクローズします。 EA の「Entry to Exit」フラグをミラーリングします。 |
| `StopLossPips` | `20` | エントリー価格からの保護ストップまでの距離。ゼロは停止を無効にします。 |
| `TakeProfitPips` | `50` | エントリー価格から利益目標までの距離。ゼロはターゲットを無効にします。 |
| `MinimumBars` | `60` | 最初のシグナルが評価される前に必要な完成したキャンドルの数。 |
| `JawPeriod` / `TeethPeriod` / `LipsPeriod` | `13 / 8 / 5` | Alligator ラインを形成する平滑化移動平均の長さ。 |
| `JawShift` / `TeethShift` / `LipsShift` | `8 / 5 / 3` | Alligator バッファを MetaTrader バージョンに合わせるために使用される前方シフト (バー単位)。 |
| `CandleType` | `1 hour time frame` | プライマリーキャンドルシリーズの定期購読。 MetaTrader で使用されているグラフと一致するように調整します。 |

## 使用上の注意

- この実装では、部分的に形成された値を避けるために、完成したローソク足 (`CandleStates.Finished`) のみを評価します。
- ストップレベルとターゲットレベルは内部的に追跡されます。この戦略は、レベルが違反されたときにポジションをフラットにするために成行注文を発行します。
- 価格ステップ変換は一般的な外国為替の規則に従います。つまり、5 桁と 3 桁の小数記号は 1 ピップを 10 の価格ステップとして扱います。
- コネクタ、ポートフォリオ、セキュリティ構成とともに戦略をスキームに追加します。戦略を開始すると、チャート パネルにローソク足シリーズと再構築された Alligator ラインが表示され、視覚的に簡単に検証できます。
