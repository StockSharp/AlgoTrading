# Стратегия RSI & ADX Long Short
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует в обе стороны, используя RSI для сигналов и ADX для подтверждения тренда.
Длинная позиция открывается при пересечении RSI выше 70 и значении ADX выше порога.
Короткая позиция открывается при пересечении RSI ниже 30 и значении ADX выше порога.
Позиции закрываются при противоположных пересечениях RSI.

## Детали

- **Критерий входа**: RSI пересекает 70 вверх для лонга или 30 вниз для шорта при ADX выше порога
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположные пересечения RSI
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RsiLength` = 8
  - `AdxLength` = 20
  - `AdxThreshold` = 14
- **Фильтры**:
  - Категория: Индикатор
  - Направление: Обе стороны
  - Индикаторы: RSI, ADX
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
