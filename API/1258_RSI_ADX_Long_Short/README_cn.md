# RSI & ADX 长短策略
[English](README.md) | [Русский](README_ru.md)

该策略利用 RSI 产生信号，并通过 ADX 确认趋势，可进行多空交易。
当 RSI 上穿 70 且 ADX 高于阈值时开多头。
当 RSI 下穿 30 且 ADX 高于阈值时开空头。
在 RSI 反向穿越时平仓。

## 细节

- **入场条件**: RSI 上穿 70 做多或下穿 30 做空，且 ADX 高于阈值
- **多空方向**: 双向
- **出场条件**: RSI 反向穿越
- **止损**: 无
- **默认值**:
  - `RsiLength` = 8
  - `AdxLength` = 20
  - `AdxThreshold` = 14
- **过滤器**:
  - 分类: 指标
  - 方向: 双向
  - 指标: RSI, ADX
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
