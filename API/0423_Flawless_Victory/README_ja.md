# Flawless Victory戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Flawless Victoryは、オシレーターとボリンジャーバンドを組み合わせたモジュール式モメンタムシステムです。選択したバージョンによって、シンプルなRSIシグナルのみで動作したり、固定のテイクプロフィットとストップロスを適用したり、Money Flow Indexによる確認を要求したりすることができます。目標は、ボラティリティバンドの端での過熱を利用し、平均回帰のスイングに乗ることです。

バージョン1は、RSIがボリンジャー極値付近の売られ過ぎまたは買われ過ぎゾーンを離れたときにエントリーします。バージョン2は、パーセンテージベースの目標による明示的なリスク管理を追加します。バージョン3は、RSIとMFIの両方が一致することを要求し、弱い反転をフィルタリングします。

この戦略は、明確なボラティリティ境界を持つイントラデイ市場で最も機能します。

## 詳細

- **エントリー条件**:
  - **ロング**: バージョンルール参照（下限バンド付近でRSI <30；バージョン3は`MFI < 20`も必要）
  - **ショート**: 上限バンド付近でRSI >70（バージョン3は`MFI > 80`も必要）
- **ロング/ショート**: 両方
- **エグジット条件**:
  - **バージョン1**: 逆のRSIシグナル
  - **バージョン2**: テイクプロフィットまたはストップロスのパーセンテージ
  - **バージョン3**: 逆のRSI/MFI組み合わせ
- **ストップ**: バージョン2でオプション
- **デフォルト値**:
  - `RSI_length` = 14
  - `MFI_length` = 14
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
  - `TakeProfitPct` = 1.5
  - `StopLossPct` = 1.0
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: RSI, MFI, Bollinger Bands
  - ストップ: オプション
  - 複雑さ: 中級
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
