# Multi Timeframe RSI Buy Sell Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия использует значения RSI с трёх разных таймфреймов. Лонг открывается, когда все включённые RSI ниже порога покупки. Шорт открывается, когда все включённые RSI выше порога продажи. Период ожидания предотвращает подряд идущие сигналы.

## Детали

- **Условия входа**: Все активные RSI ниже/выше порогов.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Rsi1Length` = 14
  - `Rsi2Length` = 14
  - `Rsi3Length` = 14
  - `Rsi1CandleType` = TimeSpan.FromMinutes(5)
  - `Rsi2CandleType` = TimeSpan.FromMinutes(15)
  - `Rsi3CandleType` = TimeSpan.FromMinutes(30)
  - `BuyThreshold` = 30m
  - `SellThreshold` = 70m
  - `CooldownPeriod` = 5
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Мульти-таймфрейм
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
