# XMA Candles戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 説明
XMA Candles戦略は、始値と終値のXMA（指数移動平均）から計算された平滑化足の方向を監視します。平滑化始値が平滑化終値より低い場合、足は**強気**とみなされ、平滑化始値が平滑化終値より高い場合は**弱気**とみなされます。この戦略は、これらの平滑化足の色変化に反応します。

- 非強気の足の後に新しい強気の足が現れると、戦略はショートポジションをクローズしてロングポジションをオープンします。
- 非弱気の足の後に新しい弱気の足が現れると、戦略はロングポジションをクローズしてショートポジションをオープンします。

## パラメーター
- `Length` – 始値と終値を平滑化する期間数。
- `CandleType` – 計算に使用する足の時間軸。
- `BuyPosOpen` – ロングポジションのオープンを許可する。
- `SellPosOpen` – ショートポジションのオープンを許可する。
- `BuyPosClose` – 弱気シグナル出現時にロングポジションのクローズを許可する。
- `SellPosClose` – 強気シグナル出現時にショートポジションのクローズを許可する。
- `StopLoss` – 保護ストップ（パーセント単位）。
- `TakeProfit` – 利益目標（パーセント単位）。

## 取引ルール
1. 選択した時間軸の各足が確定するのを待つ。
2. 始値と終値の指数移動平均を計算する。
3. 足の色を判定する：
   - 平滑化始値 < 平滑化終値の場合、緑（強気）。
   - 平滑化始値 > 平滑化終値の場合、赤（弱気）。
4. 色が強気に変わった場合、ショートをクローズし、任意でロングポジションをオープンする。
5. 色が弱気に変わった場合、ロングをクローズし、任意でショートポジションをオープンする。
6. 保護ストップと目標は組み込みのリスク管理によって管理される。

この戦略は、MQL5の元のエキスパート「Exp_XMACandles」を変換したものです。
