# RSIヒストグラム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はRSI（相対力指数）ヒストグラムを使用して、オシレーターが極端なゾーンを離れたときの反転を検出します。ヒストグラムは2つの閾値に基づいてRSIの値を色付けします。高いレベルは買われすぎゾーンを示し、低いレベルは売られすぎゾーンを示します。色が緑（買われすぎ）からグレーまたは赤に変わると、戦略はショートポジションを閉じてロングポジションを開きます。色が赤（売られすぎ）からグレーまたは緑に変わると、ロングポジションを閉じてショートポジションを開きます。

実装はStockSharpの高レベルAPIで構築されており、選択された時間軸のローソク足データを購読します。RSIインジケーターがローソク足を処理し、値が定義されたゾーンを離れるたびにシグナルを生成します。オプションのパラメーターにより、各サイドのエントリーとエグジットを個別に有効または無効にすることができます。

この戦略は教育目的であり、MQLエキスパートアドバイザーをStockSharpフレームワークに変換する方法を示しています。

## 詳細

- **エントリー条件**:
  - **ロング**: 前のバーが高いレベルの上にあり、最後のバーがその下に移動した。
  - **ショート**: 前のバーが低いレベルの下にあり、最後のバーがその上に移動した。
- **ロング/ショート**: 両サイド。
- **エグジット条件**:
  - 許可されている場合、逆のシグナルが現在のポジションを閉じる。
- **ストップ**: 組み込みのストップなし。`StartProtection` フレームワークで追加可能。
- **デフォルト値**:
  - `RSI period` = 14
  - `High level` = 60
  - `Low level` = 40
  - `Timeframe` = 4 hours
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: オプション
  - 複雑さ: シンプル
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
