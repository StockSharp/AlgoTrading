# MomentumSync PSAR RSI ADX フィルタリング 3段階エグジット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はパラボリックSARを使用してトレンド反転の初期兆候を検出し、ロングポジションに参入する前にRSIとADXの両方が設定可能な閾値を超えることを必要とします。PSARが弱気転換した後、3本目のバーで取引を決済し、3段階のエグジット・シーケンスを実装します。
