# MomentumSync PSAR RSI ADX Filtrado com Saída em 3 Níveis
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia usa o SAR Parabólico para detectar reversões precoces de tendência e exige que tanto o RSI quanto o ADX excedam limites configuráveis antes de entrar em uma posição comprada. Após uma virada baixista do PSAR, a operação é encerrada na terceira barra, implementando uma sequência de saída em três etapas.
