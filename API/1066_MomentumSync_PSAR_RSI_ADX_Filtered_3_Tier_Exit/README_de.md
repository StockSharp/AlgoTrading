# MomentumSync PSAR RSI ADX Gefiltert mit 3-Stufen-Ausstieg
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet den Parabolic SAR zur Erkennung früher Trendumkehrungen und erfordert, dass sowohl RSI als auch ADX konfigurierbare Schwellenwerte überschreiten, bevor eine Long-Position eingegangen wird. Nach einem bärischen PSAR-Wechsel wird der Trade auf der dritten Kerze geschlossen, wobei eine dreistufige Ausstiegssequenz implementiert wird.
