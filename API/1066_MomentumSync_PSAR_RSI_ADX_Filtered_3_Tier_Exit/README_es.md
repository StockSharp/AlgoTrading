# MomentumSync PSAR RSI ADX Filtrado con Salida de 3 Niveles
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza el SAR Parabólico para detectar reversiones tempranas de tendencia y requiere que tanto el RSI como el ADX superen umbrales configurables antes de entrar en una posición larga. Tras un giro bajista del PSAR, la operación se cierra en la tercera barra, implementando una secuencia de salida en tres pasos.
