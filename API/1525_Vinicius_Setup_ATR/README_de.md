# Vinicius Setup ATR-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert die SuperTrend-Richtung, RSI und Volumen, um starke Momentum-Kerzen zu identifizieren. Ein Long-Signal tritt auf, wenn der Preis über dem SuperTrend liegt, der Kerzenkörper einen ATR-basierten Schwellenwert überschreitet, das Volumen höher als der Durchschnitt ist und der RSI unter 70 bleibt. Ein Short-Signal wird unter den entgegengesetzten Bedingungen mit einem RSI über 30 ausgelöst.

## Details
- **Einstieg**: Preis in SuperTrend-Richtung mit starker Kerze und hohem Volumen.
- **Ausstieg**: Entgegengesetztes Signal.
- **Indikatoren**: SuperTrend, RSI, ATR, SMA(Volume).
