# PA オシレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQL5エキスパート**Exp_PA_Oscillator.mq5**を移植したものです。ローソク足の終値に2本の指数移動平均（EMA）を適用し、それらの差の導関数を分析します。

## ロジック

1. 高速・低速EMAを計算する。
2. それらの差を計算し、前の値からの変化を追跡する。
3. 導関数のカラーコードを決定する：
   - **0** – 導関数が正でMACDが上昇中。
   - **1** – 導関数がゼロ。
   - **2** – 導関数が負でMACDが下降中。
4. 最後の2本の確定済みローソク足の色を使ってシグナルを生成する：
   - 2本前の色が`0`で前の本が`0`から変化 → ロングポジションを開きショートポジションを決済。
   - 2本前の色が`2`で前の本が`2`から変化 → ショートポジションを開きロングポジションを決済。

## パラメーター

| 名前 | 説明 |
| ---- | ---- |
| `FastLength` | 高速EMAの長さ。 |
| `SlowLength` | 低速EMAの長さ。 |
| `BuyPosOpen` | ロングポジションの開設を有効にする。 |
| `SellPosOpen` | ショートポジションの開設を有効にする。 |
| `BuyPosClose` | ロングポジションの決済を有効にする。 |
| `SellPosClose` | ショートポジションの決済を有効にする。 |
| `CandleType` | 計算に使用するローソク足の時間軸。 |

## 備考

- 確定済みのローソク足のみが処理されます。
- エントリーとエグジットには成行注文が使用されます。
- この実装は収益性よりも明瞭さと教育的目的を重視しています。
