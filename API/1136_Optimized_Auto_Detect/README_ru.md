Стратегия пересечения скользящих средних с опциональными фильтрами тренда и RSI. Стоп и тейк-профит рассчитываются на основе ATR или фиксированного процента.

## Параметры
- Use Trend Filter
- Use RSI Filter
- RSI Period
- RSI Long Threshold
- RSI Short Threshold
- Use ATR Stop
- ATR Period
- ATR Multiplier
- Risk Reward Ratio
