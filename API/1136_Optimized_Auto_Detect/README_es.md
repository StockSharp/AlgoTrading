# Estrategia Optimizada de Detección Automática
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Estrategia de cruce de medias móviles con filtros opcionales de tendencia y RSI. Los stops y take profits pueden basarse en ATR o en un porcentaje fijo.

## Parámetros
- Usar filtro de tendencia
- Usar filtro RSI
- Período RSI
- Umbral RSI largo
- Umbral RSI corto
- Usar stop ATR
- Período ATR
- Multiplicador ATR
- Ratio riesgo/recompensa
