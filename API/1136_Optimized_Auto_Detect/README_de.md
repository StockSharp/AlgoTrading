# Optimierte Auto-Detect-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Gleitender Durchschnitt-Crossover-Strategie mit optionalen Trend- und RSI-Filtern. Stops und Take-Profits können ATR-basiert oder als fester Prozentsatz berechnet werden.

## Parameter
- Trendfilter verwenden
- RSI-Filter verwenden
- RSI-Periode
- RSI Long-Schwellenwert
- RSI Short-Schwellenwert
- ATR-Stop verwenden
- ATR-Periode
- ATR-Multiplikator
- Risiko-Ertrags-Verhältnis
