# Estratégia Otimizada de Detecção Automática
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia de cruzamento de médias móveis com filtros opcionais de tendência e RSI. Stops e take profits podem ser baseados em ATR ou percentual fixo.

## Parâmetros
- Usar filtro de tendência
- Usar filtro RSI
- Período RSI
- Limite RSI longo
- Limite RSI curto
- Usar stop ATR
- Período ATR
- Multiplicador ATR
- Relação risco/recompensa
