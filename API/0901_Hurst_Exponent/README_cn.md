# Hurst Exponent
[English](README.md) | [Русский](README_ru.md)

该策略基于平滑后的赫斯特指数进行交易。  
赫斯特值通过 EMA 平滑，并与阈值比较以判断市场状态。

## 细节
- **入场条件**:
  - **多头**: 平滑赫斯特 > 阈值
  - **空头**: 平滑赫斯特 < 阈值
- **多空**: 双向。
- **出场条件**:
  - **多头**: 平滑赫斯特 < 阈值
  - **空头**: 平滑赫斯特 > 阈值
- **止损**: 有，百分比止损。
- **默认值**:
  - `HurstPeriod = 100`
  - `SmoothLength = 10`
  - `Threshold = 0.5m`
  - `CandleType = TimeSpan.FromMinutes(5)`
- **过滤器**:
  - 类别: 趋势
  - 方向: 双向
  - 指标: Hurst 指数, EMA
  - 止损: 是
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中等
