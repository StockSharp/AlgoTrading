# Hurst Exponent
[English](README.md) | [中文](README_cn.md)

Стратегия торгует на основе сглаженной экспоненты Херста.  
Значение Херста сглаживается EMA и сравнивается с порогом для определения рыночного режима.

## Детали
- **Условия входа**:
  - **Long**: См. экспонента Херста > порога
  - **Short**: См. экспонента Херста < порога
- **Long/Short**: Оба направления.
- **Условия выхода**:
  - **Long**: См. экспонента Херста < порога
  - **Short**: См. экспонента Херста > порога
- **Стопы**: Есть, процентный стоп-лосс.
- **Значения по умолчанию**:
  - `HurstPeriod = 100`
  - `SmoothLength = 10`
  - `Threshold = 0.5m`
  - `CandleType = TimeSpan.FromMinutes(5)`
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Hurst Exponent, EMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
