# ドル・キャリートレード
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**ドル・キャリートレード**戦略は、金利差によってUSD通貨ペアをランク付けし、低キャリー通貨に対してUSDをロング、高キャリー通貨に対してショートします。最初の取引日に毎月リバランシングします。

## 詳細
- **エントリー条件**: キャリーによるランク付け；低キャリーをロング、高キャリーをショート。
- **ロング/ショート**: 両方。
- **エグジット条件**: 月次リバランス。
- **ストップ**: 明示的なストップなし。
- **デフォルト値**:
  - `K = 3`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **フィルター**:
  - カテゴリ: Fundamental
  - 方向: 両方
  - インジケーター: Rates
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 日足
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
