# ファンダメンタルズ・財務戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このサンプルは、決算日前後に現金、負債、売上、流動性などのファンダメンタル指標を取得する方法を示しています。戦略は取引シグナルを生成せず、値を記録してさらなる分析のために保存します。

- **デフォルト値**:
  - Output = Per Share
  - Period = FQ
- **フィルター**:
  - カテゴリ: Fundamentals
  - 方向: なし
  - インジケーター: なし
  - Stops: いいえ
  - 複雑さ: 基本
