# Стратегия Funamental and Financials
[English](README.md) | [中文](README_cn.md)

Пример демонстрирует получение фундаментальных показателей, таких как наличные, долг, выручка и ликвидность, вокруг дат отчётности. Стратегия лишь выводит значения для анализа и не генерирует торговых сигналов.

## Детали

- **Условия входа**: отсутствуют
- **Направление**: нет
- **Условия выхода**: отсутствуют
- **Стопы**: нет
- **Значения по умолчанию**:
  - Output = Per Share
  - Period = FQ
- **Фильтры**:
  - Категория: Фундаментальные
  - Direction: None
  - Индикаторы: Нет
  - Stops: No
  - Сложность: Базовая
  - Таймфрейм: Долгосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
