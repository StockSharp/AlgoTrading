# トリプル SMA クロスオーバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
トリプル SMA クロスオーバー戦略は、元の MQL エキスパート アドバイザー `3sma.mq4` を複製します。このシステムは、終値に基づいて計算された 3 つの単純移動平均 (SMA) を分析し、短期トレンドが中長期平均と一致するときに取引します。変換では、元の取引ルールが維持されながら、StockSharp の高レベル戦略 API に適応されます。

## 取引ロジック
1. 設定可能な期間で 3 つの SMA を計算します。
2. 高速の SMA が中速の SMA を下回ったら、既存のロングポジションを手放します。
3. 高速の SMA が中速の SMA を上回ったら、既存のショートポジションを手放します。
4. 次の場合に新しいロングポジションを入力します。
   - 高速 SMA は、少なくとも設定されたスプレッド分だけ中速 SMA を上回っています。
   - 中程度の SMA は、低速の SMA を少なくとも設定されたスプレッド分上回っています。
   - 現在オープンしているロングポジションはありません。
5. 次の場合に新しいショート ポジションを入力します。
   - 高速 SMA は、少なくとも設定されたスプレッド分だけ中速 SMA を下回ります。
   - 中程度の SMA は低速の SMA よりも、少なくとも設定されたスプレッド分下回っています。
   - 現在オープンしているショートポジションはありません。

## パラメーター
- **ローソク足タイプ** – 移動平均の計算に使用される主な時間枠。
- **高速 SMA の長さ** – 高速 SMA (MQL 入力 `SMA1`) の期間。
- **ミディアム SMA 長さ** – ミディアム SMA のピリオド (MQL 入力 `SMA2`)。
- **低速 SMA の長さ** – 低速 SMA (MQL 入力 `SMA3`) の期間。
- **SMA スプレッド ステップ** – SMA が価格ステップ数で分岐することを要求する追加フィルター (MQL 入力 `SMAspread`)。
- **取引量** – ポジションをオープンするときに使用される注文量 (MQL 入力 `lots`)。

## 注意事項
- MQL バージョンからのストップロス処理は、ソース スクリプトで無効になっているため省略されています。
- すべてのエグジットは、元の専門家の率直な行動に合わせた成行注文です。
