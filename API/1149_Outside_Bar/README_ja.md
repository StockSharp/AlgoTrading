# アウトサイドバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はアウトサイドバーのブレイクアウトを取引します。現在のローソク足の高値が前の高値を上回り、安値が前の安値を下回る場合に強気のアウトサイドバーが発生します。オプションの部分利確とブレークイーブンへのストップ移動を伴い、バー内に注文が置かれます。

## 詳細

- **エントリー条件**: 強気または弱気に分類されたアウトサイドバー。
- **ロング/ショート**: 両方。
- **エグジット条件**: バーのレンジから導出されたストップロスまたはテイクプロフィット。
- **ストップ**: はい。
- **デフォルト値**:
  - `CandleType` = 5 minute
  - `EntryPercentage` = 0.5
  - `TpPercentage` = 1
  - `PartialRR` = 1
  - `PartialExitPercent` = 0.5
  - `StopLossOffset` = 10
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
