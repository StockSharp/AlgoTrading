# Kyrieクロスオーバー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は指数移動平均（EMA）のクロスオーバーを取引します。短期EMAが長期EMAを下から上に抜けたときに買い、短期EMAが長期EMAを上から下に抜けたときに売りショートします。ストップロスはエントリー価格から設定可能なパーセンテージに置かれます。

## パラメーター
- ローソク足タイプ
- 短期EMAの期間
- 長期EMAの期間
- リスクパーセンテージ
