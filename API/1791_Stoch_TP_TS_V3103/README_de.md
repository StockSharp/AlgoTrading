# Stoch TP TS V3103
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Handelt Stoch-Oszillator-Kreuzungen in einer bestimmten Minute jeder Stunde. Wenn %K %D im Bereich 20–80 kreuzt, eröffnet die Strategie eine Position. Sie setzt einen anfänglichen Stop-Loss und einen Take-Profit mit dem Dreifachen des Aktivierungsabstands, dann wechselt sie zu einem Trailing Stop, sobald sich der Preis um den konfigurierten Offset bewegt.
