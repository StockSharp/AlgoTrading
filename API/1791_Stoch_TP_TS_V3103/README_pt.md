# Estratégia Stoch TP TS V3103
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Opera cruzamentos do oscilador Stoch em um minuto específico de cada hora. Quando %K cruza %D dentro da faixa 20-80, a estratégia abre uma posição. Aplica um stop loss inicial e um take profit de três vezes a distância de ativação, depois muda para trailing stop assim que o preço se move pelo offset configurado.
