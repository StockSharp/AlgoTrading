# Stoch TP TS V3103 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

毎時間の特定の分にStochオシレーターのクロスオーバーを取引します。%Kが20〜80の範囲で%Dをクロスしたとき、戦略はポジションを開きます。初期ストップロスと起動距離の3倍のテイクプロフィットを設定し、価格が設定オフセット分動いたらトレーリングストップに切り替えます。
