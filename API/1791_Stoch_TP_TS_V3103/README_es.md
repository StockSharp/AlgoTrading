# Estrategia Stoch TP TS V3103
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Opera cruces del oscilador Stoch en un minuto específico de cada hora. Cuando %K cruza %D dentro del rango 20-80, la estrategia abre una posición. Aplica un stop loss inicial y un take profit de tres veces la distancia de activación, luego cambia a trailing stop una vez que el precio se mueve la distancia configurada.
