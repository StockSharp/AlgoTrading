# Honest Volatility Grid
[English](README.md) | [中文](README_cn.md)

Стратегия строит сетку из уровней Keltner Channel и открывает длинные и короткие позиции на заранее заданных полосах. Выход осуществляется по противоположным уровням или аварийному стопу.

## Детали

- **Критерий входа**: Цена достигает выбранных уровней канала Keltner.
- **Лонг/Шорт**: Обе стороны.
- **Критерий выхода**: Противоположный канал или аварийный стоп.
- **Стопы**: Дополнительный аварийный стоп.
- **Значения по умолчанию**:
  - `EmaPeriod` = 200
  - `Multiplier` = 1.0
  - `LEntry1Level` = -2
  - `SEntry1Level` = 2
  - `RawStopLevel` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Grid
  - Направление: Обе
  - Индикаторы: EMA, ATR
  - Стопы: Есть
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
