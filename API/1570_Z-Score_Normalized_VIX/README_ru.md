# Z-Score Normalized VIX Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия усредняет z-значения нескольких индексов волатильности и открывает длинную позицию, когда итоговое значение опускается ниже отрицательного порога.

Алгоритм вычисляет z-score для VIX, VIX3M, VIX9D и VVIX. Выбранные z-значения усредняются, отражая общий уровень страха на рынке.

## Детали

- **Условия входа**: Совокупный z-score ниже `-Threshold`.
- **Лонг/Шорт**: Только лонг.
- **Условия выхода**: Совокупный z-score поднимается выше `-Threshold`.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `ZScoreLength` = 6
  - `Threshold` = 1
  - `UseVix` = true
  - `UseVix3m` = true
  - `UseVix9d` = true
  - `UseVvix` = true
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Лонг
  - Индикаторы: Z-Score
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
