# Estrategia de Marco de Gradiente de Color
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demuestra el cálculo de un gradiente de color de rojo a verde basado en RSI y opera cuando el RSI cruza la línea central.

## Lógica
- Calcula el Índice de Fuerza Relativa con un período configurable.
- Deriva un color de gradiente para visualización.
- Compra cuando el RSI sube por encima de 50.
- Vende cuando el RSI baja por debajo de 50.
