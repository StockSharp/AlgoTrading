# Color Gradient Framework Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demonstriert die Berechnung eines Farbverlaufs von Rot nach Grün basierend auf dem RSI und handelt, wenn der RSI die Mittellinie kreuzt.

## Logik
- Berechnet einen Relative-Stärke-Index mit konfigurierbarem Zeitraum.
- Leitet eine Verlaufsfarbe für die Visualisierung ab.
- Kauft, wenn der RSI über 50 steigt.
- Verkauft, wenn der RSI unter 50 fällt.
