# カラーグラジエントフレームワーク戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSI に基づいて赤から緑へのカラーグラジエントを計算し、RSI が中心線をクロスしたときに取引することを示します。

## ロジック
- 設定可能な期間で RSI（相対力指数）を計算する。
- 視覚化のためのグラジエントカラーを導出する。
- RSI が 50 を上回ったときに買い。
- RSI が 50 を下回ったときに売り。
