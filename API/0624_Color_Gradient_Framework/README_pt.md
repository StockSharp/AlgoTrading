# Estratégia de Estrutura de Gradiente de Cores
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Demonstra o cálculo de um gradiente de cores do vermelho ao verde baseado no RSI e opera quando o RSI cruza a linha central.

## Lógica
- Calcula o Índice de Força Relativa com período configurável.
- Deriva uma cor de gradiente para visualização.
- Compra quando o RSI sobe acima de 50.
- Vende quando o RSI cai abaixo de 50.
