# Estratégia de Avaliação de Osciladores
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina o Índice de Força Relativa (RSI) e o Oscilador Estocástico para identificar pontos de entrada e saída. Uma posição comprada é aberta quando ambos os osciladores caem abaixo do limiar de sobrevenda, enquanto uma posição vendida é aberta quando ambos sobem acima do limiar de sobrecompra. As posições são encerradas quando ambos os indicadores retornam à zona neutra. A estratégia também suporta proteção de take-profit e stop-loss configuráveis.
