# Oszillator-Auswertungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert den Relative Strength Index (RSI) und den Stochastischen Oszillator, um Ein- und Ausstiegspunkte zu identifizieren. Eine Long-Position wird eröffnet, wenn beide Oszillatoren unter die Überverkauft-Schwelle fallen, während eine Short-Position eröffnet wird, wenn beide über die Überkauft-Schwelle steigen. Positionen werden geschlossen, sobald beide Indikatoren in die neutrale Zone zurückkehren. Die Strategie unterstützt auch konfigurierbaren Take-Profit- und Stop-Loss-Schutz.
