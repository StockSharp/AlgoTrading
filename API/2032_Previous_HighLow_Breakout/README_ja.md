# 前回高値・安値ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

選択した時間軸における前回ローソク足の高値と安値を監視するブレイクアウト戦略です。新しいローソク足が前回高値を上回って終値をつけるとロングポジションが開かれ、前回安値を下回って終値をつけるとショートポジションが開かれます。トレーリングストップと固定テイクプロフィットでリスクを管理し、利益を確保します。

この手法は、保合いの後の強い方向性のある動きを捉えることを目的としています。トレーリングストップは、価格が有利な方向に動く際にリスクを抑えます。

## 詳細

- **エントリー条件**:
  - ロング: `Close > PreviousHigh`
  - ショート: `Close < PreviousLow`
- **ロング/ショート**: 両方
- **エグジット条件**:
  - ストップロスまたはテイクプロフィット
- **ストップ**: `StopLoss`と`TakeProfit`によるトレーリング付き絶対値
- **デフォルト値**:
  - `StopLoss` = 50m
  - `TakeProfit` = 1000m
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい（トレーリング）
  - 複雑さ: 初心者
  - 時間軸: 長期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
