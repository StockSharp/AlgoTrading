# Стратегия Previous High Low Breakout
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя, отслеживающая максимум и минимум предыдущей свечи на выбранном таймфрейме. Длинная позиция открывается, когда новая свеча закрывается выше предыдущего максимума, короткая — когда закрытие ниже предыдущего минимума. Для управления риском применяются трейлинг-стоп и фиксированный тейк-профит.

Метод рассчитан на захват сильных направленных движений после консолидации. Трейлинг-стоп позволяет удерживать риск под контролем, когда цена движется в прибыльную сторону.

## Подробности

- **Условия входа**:
  - Лонг: `Close > PreviousHigh`
  - Шорт: `Close < PreviousLow`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Стоп-лосс или тейк-профит
- **Стопы**: абсолютные с трейлингом через `StopLoss` и `TakeProfit`
- **Значения по умолчанию**:
  - `StopLoss` = 50m
  - `TakeProfit` = 1000m
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да (трейлинг)
  - Сложность: Начальная
  - Таймфрейм: Долгосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
