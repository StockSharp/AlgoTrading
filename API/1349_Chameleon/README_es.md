# Estrategia Chameleon

- **Tipo**: Seguimiento de tendencia con gestión de riesgo
- **Entrada**: Cruce de media móvil rápida por encima o por debajo de la media móvil lenta
- **Salidas**: Stop-loss, stop móvil, take-profit
- **Filtros**: Filtro de sesión de trading opcional

Esta estrategia demuestra cómo combinar señales de medias móviles con varias herramientas de gestión de riesgo.
