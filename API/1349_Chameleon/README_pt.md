# Estratégia Chameleon
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

- **Tipo**: Seguidor de tendência com gestão de risco
- **Entrada**: Cruzamento da média móvel rápida acima ou abaixo da média móvel lenta
- **Saídas**: Stop-loss, trailing stop, take-profit
- **Filtros**: Filtro de sessão de trading opcional

Esta estratégia demonstra como combinar sinais de médias móveis com várias ferramentas de gestão de risco.
