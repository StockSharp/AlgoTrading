# SMC Order Block Zones 策略
[English](README.md) | [Русский](README_ru.md)

该策略识别摆动高点和低点以构建溢价区和折扣区。简单移动平均线作为趋势过滤器，最近的订单块用于确认信号。当价格在订单块确认下从一个区域向均衡线移动时开仓，并通过百分比止损保护仓位。

## 细节

- **入场条件**：
  - 收盘价低于均衡线但高于折扣区且高于SMA时做多。
  - 收盘价高于均衡线但低于溢价区且低于SMA时做空。
  - 价格需触及相应的订单块水平。
- **多空方向**：可配置为多头、空头或双向。
- **出场条件**：反向信号或止损。
- **止损**：百分比止损。
- **默认值**：
  - `SwingHighLength` = 8
  - `SwingLowLength` = 8
  - `SmaLength` = 50
  - `OrderBlockLength` = 20
  - `StopLossPercent` = 2
- **过滤器**：
  - 分类：趋势与SMC
  - 方向：用户定义
  - 指标：SMA、Highest、Lowest
  - 止损：是
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
