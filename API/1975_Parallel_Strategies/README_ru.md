# Стратегия Parallel Strategies
[English](README.md) | [中文](README_cn.md)

Система пробоя на основе Heikin Ashi и MACD, торгующая в обе стороны. Вход выполняется, когда новый тренд Heikin Ashi совпадает с пробоем уровня Donchian и индикатор MACD подтверждает импульс.

Комбинация определения тренда по Heikin Ashi и поиска пробоев помогает входить в начало движения. MACD фильтрует ложные сигналы.

Лучше всего подходит для трейдеров, ищущих ранний вход в пробой после разворота тренда. Работает на внутридневных таймфреймах.

## Детали

- **Условия входа**:
  - Лонг: `Тренд меняется на бычий && Close > DonchianHigh && MACD > Signal`
  - Шорт: `Тренд меняется на медвежий && Close < DonchianLow && MACD < Signal`
- **Лонг/Шорт**: Оба
- **Условия выхода**:
  - Противоположный сигнал пробоя
- **Стопы**: Не заданы
- **Значения по умолчанию**:
  - `DonchianPeriod` = 5
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Heikin Ashi, Donchian Channel, MACD
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
