# Модифицированный OBV с поиском дивергенций
[English](README.md) | [中文](README_cn.md)

Стратегия сглаживает индикатор OBV выбранным типом скользящей средней и строит сигнальную линию. Торговые сделки совершаются при пересечении OBV-M и сигнала. Дополнительно стратегия фиксирует обычные и скрытые дивергенции между ценой и OBV через фракталы.

## Подробности

- **Критерий входа**: OBV-M пересекает сигнальную линию сверху/снизу.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: обратное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `MaType` = Exponential
  - `ObvMaLength` = 7
  - `SignalLength` = 10
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Дивергенция
  - Направление: Обе стороны
  - Индикаторы: OBV, MA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
