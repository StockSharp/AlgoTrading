# ColorMETRO WPR戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はColorMETRO Williams %Rインジケーターを使用します。このインジケーターはWilliams %Rオシレーターの周囲に高速・低速のステップラインを構築します。
高速ラインは価格変動に素早く反応し、低速ラインはノイズを平滑化します。取引の判断はこれらのラインが
互いに交差したとき、つまり勢いの潜在的な変化を示すシグナルのときに行われます。高速ラインが低速ラインを下抜けたとき、戦略は
市場が売られ過ぎと判断してロングポジションに入ります。逆に、高速ラインが低速ラインを上抜けたとき、ショートポジションに入ります。
逆の条件が検出されると既存ポジションを決済します。

リスク管理はパーセンテージベースのテイクプロフィットとストップロスレベルによって行われます。これにより戦略は
異なる銘柄の価格レベルに適応できます。デフォルトのローソク足時間軸は8時間で、イントラデイのボラティリティを
フィルタリングし、中期トレンドに集中するのに役立ちます。ロジックは市場の両サイドで機能し、ロングとショートの両方の取引を可能にします。

## 詳細

- **エントリー条件**:
  - **ロング**: `高速ライン`が`低速ライン`を**下抜ける**。
  - **ショート**: `高速ライン`が`低速ライン`を**上抜ける**。
- **ロング/ショート**: 両側。
- **エグジット条件**:
  - **ロング**: `高速ライン`が`低速ライン`を上回る。
  - **ショート**: `高速ライン`が`低速ライン`を下回る。
- **ストップ**: あり、パーセンテージベースのテイクプロフィットとストップロス。
- **デフォルト値**:
  - `WprPeriod` = 7.
  - `FastStep` = 5.
  - `SlowStep` = 15.
  - `TakeProfitPercent` = 4.
  - `StopLossPercent` = 2.
  - `CandleType` = 8時間ローソク足。
- **フィルター**:
  - カテゴリ: トレンドフォロー。
  - 方向: 両方。
  - インジケーター: 単一（Williams %Rベース）。
  - ストップ: あり。
  - 複雑さ: 中程度。
  - 時間軸: 中期。
  - 季節性: なし。
  - ニューラルネットワーク: なし。
  - ダイバージェンス: なし。
  - リスクレベル: 中。
