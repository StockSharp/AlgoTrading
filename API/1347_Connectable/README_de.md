# Verbindbare Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Vorlagenstrategie, die mit externen Signalquellen verbunden werden kann.
Sie unterstützt Long- und Short-Richtungen und wendet prozentbasierte Stop-Loss- und Take-Profit-Werte an.

## Details

- **Einstiegskriterien**: externes Signal
- **Long/Short**: Beide
- **Ausstiegskriterien**: externes Signal oder Stop-Loss/Take-Profit
- **Stops**: Ja, prozentbasiert
- **Standardwerte**:
  - `CandleType` = 1 Minute
  - `StopLossPercent` = 2%
  - `TakeProfitPercent` = 4%
- **Filter**:
  - Kategorie: Sonstige
  - Richtung: Beide
  - Indikatoren: Keine
  - Stops: Ja
  - Komplexität: Anfänger
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Niedrig
