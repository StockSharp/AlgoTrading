# 200 SMA Buffer 策略
[English](README.md) | [Русский](README_ru.md)

200 SMA Buffer 策略利用价格与长期简单移动平均线之间的距离。当收盘价上穿 SMA 一定百分比时买入，当价格跌破 SMA 指定百分比时退出。该方法试图在长期趋势中保留一个缓冲区。

## 细节

- **入场条件**:
  - 收盘价 > SMA * (1 + Entry %).
- **多空方向**: 仅做多。
- **出场条件**:
  - 收盘价 < SMA * (1 - Exit %).
- **止损**: 无。
- **默认值**:
  - `SmaLength` = 200
  - `EntryPercent` = 5
  - `ExitPercent` = 3
- **过滤器**:
  - 类别: 趋势跟随
  - 方向: 多头
  - 指标: SMA
  - 止损: 无
  - 复杂度: 低
  - 时间框架: 任意
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
