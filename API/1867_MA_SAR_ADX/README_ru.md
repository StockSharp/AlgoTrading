# Стратегия MA SAR ADX
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает скользящую среднюю, Parabolic SAR и индикатор Average Directional Index (ADX).
Покупает, когда цена находится выше скользящей средней и SAR, а +DI выше -DI.
Продаёт, когда цена ниже скользящей средней и SAR, а +DI ниже -DI.
Позиция закрывается при пересечении цены и SAR.

## Детали

- **Условия входа**:
  - Long: `Close > MA && +DI >= -DI && Close > SAR`
  - Short: `Close < MA && +DI <= -DI && Close < SAR`
- **Направление**: Оба
- **Условия выхода**: Цена пересекает Parabolic SAR
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `MaPeriod` = 100
  - `AdxPeriod` = 14
  - `SarStep` = 0.02m
  - `SarMax` = 0.1m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA, Parabolic SAR, ADX
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
