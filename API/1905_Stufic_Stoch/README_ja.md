# Stufic Stochastic戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、2本の移動平均によるトレンド検出とストキャスティクスオシレーターのモメンタムシグナルを組み合わせます。
速い移動平均が遅い移動平均を上回り、ストキャスティクスの%Kラインが売られすぎのしきい値以下で%Dラインを上抜けしたときに買います。
速い移動平均が遅い移動平均を下回り、%Kが買われすぎのしきい値以上で%Dを下抜けしたときに売ります。

## ロジック
- 速い移動平均と遅い移動平均を比較して市場トレンドを検出します。
- ストキャスティクスオシレーターを使って極値水準でのモメンタム転換を探します。
- トレンドが上昇中でオシレーターが売られすぎゾーンから強気のクロスで脱出したときにロングポジションを建てます。
- トレンドが下降中でオシレーターが買われすぎゾーンから弱気のクロスで脱出したときにショートポジションを建てます。
- 逆シグナルが出るとポジションを決済または反転させます。組み込みの保護機能でストップロスのパーセントが適用されます。

## パラメーター
- **FastMaPeriod** – 速い移動平均の期間。
- **SlowMaPeriod** – 遅い移動平均の期間。
- **StochKPeriod** – ストキャスティクスの%K期間。
- **StochDPeriod** – %Dラインの平滑化期間。
- **OverboughtLevel** – ストキャスティクスオシレーターの上限しきい値。
- **OversoldLevel** – ストキャスティクスオシレーターの下限しきい値。
- **StopLossPercent** – エントリー価格の一定割合で表されるストップロス距離。
- **CandleType** – 計算に使用するローソク足シリーズ。

## インジケーター
- 単純移動平均（速い・遅い）。
- ストキャスティクスオシレーター。

## 使用方法
戦略を銘柄に接続します。希望する時間軸とリスクレベルに合わせてパラメーターを設定します。戦略を開始して取引を始めます。アルゴリズムは説明した条件に基づいてポジションを自動的に管理します。
