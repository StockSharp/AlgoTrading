# Trend Catcherブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
Trend Catcher戦略は、MetaTrader 5エキスパートアドバイザー「Trend_Catcher_v2」の変換版です。3つの指数移動平均とParabolic SARインジケーターを組み合わせて、トレンド反転とトレンド継続の機会を識別します。システムは単一のシンボルと時間軸で動作し、ローソク足終値の計算に依存しているため、StockSharp Designerでのバックテストとともに、StockSharp APIベースのランナーを通じたライブ実行にも適しています。

## インジケーターとフィルター
- **Parabolic SAR** — 潜在的な反転を示す強気および弱気のフリップを検出します。
- **低速EMA** — 支配的な方向を定義する上位時間軸トレンドフィルター。
- **高速EMA** — 現在のスイングの方向を確認するために価格変化により素早く反応します。
- **トリガーEMA** — エントリーを価格アクションに近く保ち、平均から遠すぎる取引を避けます。
- **取引日スイッチ** — 選択した曜日のトレーディングを無効にするオプションフィルター。

## トレードロジック
### ロングエントリー
1. 終値が現在のParabolic SAR値を上回って終了します。
2. 前のローソク足が前回のParabolic SAR値を下回って終了しました（強気フリップ）。
3. 高速EMAが低速EMAを上回っており、上昇トレンドを確認しています。
4. 終値がトリガーEMAを上回っており、逆張りシグナルを避けます。
5. ポジションが開いておらず、現在のローソク足中にポジションが閉じられていません。

### ショートエントリー
上記のすべての条件が反転されます：
1. 終値が現在のParabolic SAR値を下回って終了します。
2. 前のローソク足が前回のParabolic SAR値を上回って終了しました（弱気フリップ）。
3. 高速EMAが低速EMAを下回っています。
4. 終値がトリガーEMAを下回っています。
5. ポジションが開いておらず、現在のローソク足中にポジションが閉じられていません。

**Reverse Signals**スイッチが有効になると、ロングとショートの条件が逆転し、戦略が反対方向のブレイクアウトを取引できます。

## ポジション管理
- **自動ストップロス** – 有効にすると、ストップは価格とParabolic SARの間の距離に`StopLossCoefficient`を掛けて計算されます。距離は`MinStopLoss`と`MaxStopLoss`の間にクランプされます。
- **自動テイクプロフィット** – ストップ距離に`TakeProfitCoefficient`を掛けます。自動化が無効の場合は手動距離を使用できます。
- **リスクベースのポジションサイジング** – 取引サイズはポートフォリオの資本と`RiskPercent`から導出されます。最後に閉じた取引が損失で、**Use Martingale**が有効な場合、計算されたサイズは`MartingaleMultiplier`で乗算されます。
- **ブレイクイーブンとトレーリングストップ** – `BreakevenTrigger`の利益に達した後、ストップはエントリー価格に`BreakevenOffset`を加えた位置に移動されます（ショートトレードでは引く）。ポジションが`TrailingTrigger`を獲得すると、ストップは`TrailingStep`で価格をトレーリングします。
- **反対シグナルでのクローズ** – アクティブな場合、戦略は反対のセットアップが現れるとすぐに既存のポジションを終了します。
- **ローソク足あたり一取引** – アルゴリズムは最後のエグジットのタイムスタンプを保存し、次のローソク足が開くまでエントリーをスキップします。

## パラメーター
| 名前 | 説明 | デフォルト値 |
| --- | --- | --- |
| `CandleType` | すべてのインジケーターに使用される主要時間軸。 | 15分時間軸 |
| `CloseOnOppositeSignal` | 反転セットアップが検出されたら即時終了。 | `true` |
| `ReverseSignals` | ロングとショートの条件を入れ替える。 | `false` |
| `TradeMonday` … `TradeFriday` | 特定の曜日のトレーディングを有効または無効にする。 | `true` |
| `SlowMaPeriod` | 低速EMAトレンドフィルターの期間。 | `200` |
| `FastMaPeriod` | 高速EMA確認の期間。 | `50` |
| `FastFilterPeriod` | トリガーEMAの期間。 | `25` |
| `SarStep` | Parabolic SAR加速ステップ。 | `0.004` |
| `SarMax` | Parabolic SARの最大加速。 | `0.2` |
| `AutoStopLoss` | 動的ストップロス計算を有効にする。 | `true` |
| `AutoTakeProfit` | 動的テイクプロフィット計算を有効にする。 | `true` |
| `MinStopLoss` / `MaxStopLoss` | ストップ距離の下限と上限。 | `0.001` / `0.2` |
| `StopLossCoefficient` | SAR距離に適用される乗数。 | `1` |
| `TakeProfitCoefficient` | テイクプロフィット距離に使用される乗数。 | `1` |
| `ManualStopLoss` | 自動化が無効の場合の固定ストップ距離。 | `0.002` |
| `ManualTakeProfit` | 自動化が無効の場合の固定ターゲット距離。 | `0.02` |
| `RiskPercent` | 取引ごとにリスクにさらされるポートフォリオ資本の割合。 | `2` |
| `UseMartingale` | 損失取引後にサイズを増やす。 | `true` |
| `MartingaleMultiplier` | 損失後に適用される乗数。 | `2` |
| `BreakevenTrigger` | ストップをブレイクイーブンに移動する前に必要な利益。 | `0.005` |
| `BreakevenOffset` | ストップをブレイクイーブンに移動する際に追加するバッファ。 | `0.0001` |
| `TrailingTrigger` | ストップのトレーリングを開始するために必要な利益。 | `0.005` |
| `TrailingStep` | トレーリングストップによって維持される距離。 | `0.001` |

## 使用上の注意
- 戦略はエントリーとエグジットの両方にマーケット注文を送信します。必要に応じて、スリッページ制御はブローカーアダプターレベルで追加する必要があります。
- ロジックはローソク足終値データを使用するため、バックテストの精度は戦略に提供されるローソク足シリーズの粒度に依存します。
- パラメーターは`StrategyParam`オブジェクトを通じて完全に公開されており、StockSharp Designerでの最適化に利用できます。
