# Waindrops Makit0
[English](README.md) | [中文](README_cn.md)

Упрощённая стратегия, сравнивающая VWAP двух половин заданного периода.

## Детали

- **Условия входа**: Сравнение VWAP правой и левой половины.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `PeriodMinutes` = 60
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Объём
  - Направление: Оба
  - Индикаторы: VWAP
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Низкий
