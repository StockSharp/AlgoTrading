# Стратегия FlexiSuperTrend
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает фильтр SuperTrend и сглаженный осциллятор отклонения.
Позиция открывается, когда цена совпадает с направлением SuperTrend и
осциллятор подтверждает импульс.

## Детали

- **Условия входа**:
  - Цена выше SuperTrend и отклонение (SMA от разницы цены и SuperTrend) > 0 → покупка.
  - Цена ниже SuperTrend и отклонение < 0 → продажа.
- **Лонг/Шорт**: поддерживаются оба направления.
- **Условия выхода**:
  - Разворот тренда при пересечении цены и линии SuperTrend.
- **Стопы**: логика стопов по умолчанию отсутствует.
- **Параметры по умолчанию**:
  - Период ATR = 10.
  - Множитель ATR = 3.0.
  - Длина SMA = 10.
- **Фильтры**:
  - Категория: следование за трендом
  - Направление: оба
  - Индикаторы: SuperTrend, SMA
  - Стопы: нет
  - Сложность: низкая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
