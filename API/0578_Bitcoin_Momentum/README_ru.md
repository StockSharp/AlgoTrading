# Биткоин Моментум
[English](README.md) | [中文](README_cn.md)

Стратегия для Биткоина, торгующая только при цене выше недельной EMA и избегая «предупреждений». Прибыль защищается трейлинг-стопом на основе ATR.

## Подробности

- **Критерии входа**: цена выше недельной EMA и нет состояния "caution".
- **Длинные/короткие**: только длинные.
- **Критерии выхода**: цена ниже трейлинг-стопа или недельной EMA.
- **Стопы**: трейлинг-стоп на основе ATR.
- **Параметры по умолчанию**:
  - `CandleType` = TimeSpan.FromDays(1)
  - `HigherCandleType` = TimeSpan.FromDays(7)
  - `EmaLength` = 20
  - `AtrLength` = 5
  - `TrailStopLookback` = 7
  - `TrailStopMultiplier` = 0.2m
  - `StartTime` = 2000-01-01
  - `EndTime` = 2099-01-01
- **Фильтры**:
  - Категория: Momentum
  - Направление: Длинные
  - Индикаторы: EMA, ATR, Highest
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
