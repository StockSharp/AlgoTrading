# Binary Wave 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

Binary Waveは複数のクラシックなテクニカルインジケーターを単一の「バイナリー」ウェーブに組み合わせます。各インジケーターは強気または弱気の状態に応じて+1または-1を提供します。すべてのシグナルの加重和が取引判断に使用される最終ウェーブを形成します。

## パラメーター

- **Mode** – エントリーアルゴリズム：`Breakdown`はゼロクロスに反応し、`Twist`はウェーブの方向変化に反応します。
- **Candle Type** – すべての計算に使用するローソク足の時間軸。
- **Indicator Periods** – MA、MACD（速い・遅い・シグナル）、CCI、Momentum、RSI、ADXの期間。
- **Weights** – ウェーブへの各インジケーターの貢献度。0に設定するとそのインジケーターが無効になります。
- **Trading Permissions** – ロング/ショートのエントリーとエグジットを個別に有効/無効にする。
- **Risk** – エントリー価格の割合でのストップロスとテイクプロフィット。

## 仕組み

1. 指定したローソク足シリーズをサブスクライブし、すべてのインジケーターを計算します。
2. 各完成ローソク足において、すべてのインジケーターの状態を評価し、バイナリー値（+1 / -1）に変換します。
3. 加重値を合計して現在のウェーブを取得します。
4. 取引シグナルを生成します：
   - **Breakdown**: ウェーブがゼロを上抜けしたときロングに入り、下抜けしたときショートに入る。
   - **Twist**: ウェーブが上方向に転じたときロングに入り、下方向に転じたときショートに入る。
5. オプションの保護的ストップロスとテイクプロフィットは組み込みのポジション保護によって管理されます。

このアプローチにより、複数のインジケーターを柔軟に組み合わせながら、取引ロジックをシンプルに保つことができます。
