# 超级趋势成交量策略

该策略在 Supertrend 指标的基础上加入成交量确认。
当 Supertrend 出现反转并伴随成交量上升时，新的动能更可能启动。
只有当 Supertrend 信号出现且成交量高于平均水平时，策略才顺势入场。
止损沿 Supertrend 线跟踪，若收盘穿越其另一侧则离场。

## 细节

- **入场条件**：指标信号
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：趋势跟随
  - 方向：双向
  - 指标：Supertrend, 成交量
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
