# Narrow Range 策略
[English](README.md) | [Русский](README_ru.md)

该策略在出现内部K线后交易突破，当当前K线范围比 `Length` 个周期前的参考K线更窄时触发。在参考K线的高点和低点放置止损单，止盈为参考范围，止损为该范围的百分比。

## 细节

- **入场条件**：
  - 多头：窄幅K线后价格上破参考高点
  - 空头：窄幅K线后价格下破参考低点
- **方向**：双向
- **出场条件**：
  - 在参考范围处止盈
  - 以参考范围百分比作为止损
- **止损**：是
- **默认值**：
  - `Length` = 4
  - `StopLossPercent` = 0.35m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **筛选**：
  - 类型：Breakout
  - 方向：双向
  - 指标：无
  - 止损：是
  - 复杂度：基础
  - 时间框架：短期
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
