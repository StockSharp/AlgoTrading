# Стратегия Narrow Range
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробои после внутренней свечи, когда диапазон текущей свечи уже, чем диапазон контрольной свечи `Length` периодов назад. Стоп-заявки выставляются на максимум и минимум контрольной свечи с тейк-профитом, равным этому диапазону, и стоп-лоссом в проценте от него.

## Детали

- **Условия входа**:
  - Long: цена пробивает максимум контрольной свечи после узкого диапазона
  - Short: цена пробивает минимум контрольной свечи после узкого диапазона
- **Направление**: Оба
- **Условия выхода**:
  - Тейк-профит на расстоянии диапазона контрольной свечи
  - Стоп-лосс в проценте от диапазона
- **Стопы**: Да
- **Значения по умолчанию**:
  - `Length` = 4
  - `StopLossPercent` = 0.35m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
