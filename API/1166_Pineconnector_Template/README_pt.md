# Modelo de Estratégia Pineconnector
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia demonstra como conectar qualquer indicador para gerar sinais de negociação. Utiliza duas médias móveis como exemplo e entra comprado quando a média rápida cruza acima da lenta, e entra vendido no cruzamento oposto.

## Parâmetros
- **Fast Length** – período da média móvel rápida.
- **Slow Length** – período da média móvel lenta.
- **Candle Type** – tipo de candle para o cálculo.
