# Pineconnector-Strategie-Vorlage
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie zeigt, wie man einen beliebigen Indikator zur Erzeugung von Handelssignalen verbindet. Als Beispiel werden zwei gleitende Durchschnitte verwendet: Long-Einstieg, wenn der schnelle Durchschnitt den langsamen von unten kreuzt, und Short-Einstieg beim umgekehrten Crossover.

## Parameter
- **Fast Length** – Periode des schnellen gleitenden Durchschnitts.
- **Slow Length** – Periode des langsamen gleitenden Durchschnitts.
- **Candle Type** – Kerzentyp für die Berechnung.
