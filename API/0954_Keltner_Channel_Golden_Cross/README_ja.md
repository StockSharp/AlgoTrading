# ケルトナーチャネル・ゴールデンクロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、2本の移動平均のゴールデンクロスで確認されたケルトナーチャネルのブレイクアウトを取引します。

- **ロングエントリー**: 価格が上限チャネルを上抜けし、短期EMAが長期EMAを上回っている。
- **ショートエントリー**: 価格が下限チャネルを下抜けし、短期EMAが長期EMAを下回っている。
- **エグジット**: ATRベースのテイクプロフィットまたはストップ。
- **インジケーター**: Moving Average (SMA/EMA/WMA)、Average True Range。
- **ロング/ショート**: 両方。
