# チャネル内取引戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

バンド幅が変わらない場合にDonchianチャネルの極値に逆張りする逆張りチャネル戦略。システムは最新の高値/安値を前のチャネル境界と前回の終値から計算したピボットと比較して、動きに逆張りするかどうかを判断します。保護ストップはATR距離に依存し、オプションのトレーリングストップはポジションに有利に価格が動いた後に利益を維持します。

## 詳細

- **エントリー条件**:
  - ショート: チャネル上限バンドが変わらず、最後の足の高値が上限バンドに触れた、または前回の終値がピボットと上限バンドの間にある。
  - ロング: チャネル下限バンドが変わらず、最後の足の安値が下限バンドに触れた、または前回の終値がピボットと下限バンドの間にある。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 上限バンドがフラットで価格がそれに触れた場合、またはATRストップかトレーリングストップに達した場合にロングをクローズする。
  - 下限バンドがフラットで価格がそれに触れた場合、またはATRストップかトレーリングストップに達した場合にショートをクローズする。
- **ストップ**:
  - ロングの初期ストップは`support - ATR`に、ショートは`resistance + ATR`に設定。
  - トレーリングストップは利益が`TrailingStopPips`距離（価格ステップに変換）を超えた後、最良価格の背後を移動する。
- **デフォルト値**:
  - `ChannelPeriod` = 20（Donchianルックバック）
  - `AtrPeriod` = 4（ATR平滑化）
  - `Volume` = 1 コントラクト/ロット
  - `TrailingStopPips` = 30 価格ステップ
  - `CandleType` = 1時間足
- **フィルター**:
  - カテゴリ: チャネル / 平均回帰
  - 方向: ロングとショート
  - インジケーター: Donchian Channel、ATR
  - ストップ: ATRハードストップ + トレーリングストップ
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中

## 注意事項

- ピボットは`(上限バンド + 下限バンド + 前回終値) / 3`に等しく、オリジナルのMQL実装と一致しています。
- 戦略は1つのネットポジションのみを保持し、前のトレードが完全にクローズされた後にのみ方向を転換します。
- トレーリング距離は価格ステップ（「pips」）で指定され、実際の価格オフセットを得るためにインストゥルメントの`PriceStep`を乗じます。
