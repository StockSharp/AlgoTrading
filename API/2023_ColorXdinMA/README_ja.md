# ColorXdinMA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
ColorXdinMA戦略はXdinMAインジケーターを実装しています。このインジケーターは`ma_main * 2 - ma_plus`として計算され、両コンポーネントとも異なる長さの単純移動平均です。戦略はこのラインの傾きを監視し、傾きの方向が変わったときにポジションを開きます。

## 取引ロジック
- インジケーターが下降していて上向きに転じると、ロングポジションが開かれます。既存のショートポジションは閉じられます。
- インジケーターが上昇していて下向きに転じると、ショートポジションが開かれます。既存のロングポジションは閉じられます。

確定済みローソク足のみが処理されます。注文は成行執行で配置されます。

## パラメーター
| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| `MainLength` | メイン移動平均の期間。 | 10 |
| `PlusLength` | 追加移動平均の期間。 | 20 |
| `CandleType` | 計算に使用するローソク足の時間軸。 | 6時間 |

## 注意事項
この実装はStockSharpの高レベル戦略であり、必要に応じてリスク管理や可視化機能で拡張できます。
