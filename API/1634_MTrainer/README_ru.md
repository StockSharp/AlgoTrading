# Стратегия MTrainer
[English](README.md) | [中文](README_cn.md)

Стратегия MTrainer повторяет скрипт MT4 MTrainer. Открывает позицию, когда цена достигает заданной линии входа, и управляет ею через стоп-лосс, тейк-профит и опциональное частичное закрытие. Предназначена для ручной практики в визуальном тестере.

## Детали

- **Критерий входа**: цена пересекает линию входа
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: стоп-лосс, тейк-профит или частичное закрытие
- **Стопы**: да
- **Значения по умолчанию**:
  - `EntryPrice` = 0
  - `TakeProfitPrice` = 0
  - `StopLossPrice` = 0
  - `PartialClosePercent` = 0
  - `PartialClosePrice` = 0
  - `Volume` = 1
- **Фильтры**:
  - Категория: Утилита
  - Направление: Обе стороны
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
