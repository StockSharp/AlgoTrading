# IU BBB 大陽線・大陰線戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、現在のローソク足の実体が過去20本のローソク足の平均実体サイズの数倍大きいときにエントリーします。大きな強気ローソク足がロングポジションを開き、大きな弱気ローソク足がショートポジションを開きます。ポジションはATRベースのトレーリングストップで保護されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 実体 > 平均実体 * BigBodyThreshold かつ 終値 > 始値。
  - **ショート**: 実体 > 平均実体 * BigBodyThreshold かつ 終値 < 始値。
- **ロング/ショート**: 両方。
- **エグジット条件**: ATRトレーリングストップ。
- **ストップ**: ATR * AtrFactorを使用したトレーリングストップ。
- **デフォルト値**:
  - `BigBodyThreshold` = 4
  - `AtrLength` = 14
  - `AtrFactor` = 2
  - `CandleType` = 5 minute
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: SMA, ATR
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
