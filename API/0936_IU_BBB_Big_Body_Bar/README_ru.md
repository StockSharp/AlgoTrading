# Стратегия IU Big Body Bar
[English](README.md) | [中文](README_cn.md)

Стратегия открывает позицию, когда тело текущей свечи значительно больше среднего тела за последние 20 свечей. Большая бычья свеча открывает длинную позицию, большая медвежья — короткую. Позиции защищаются трейлинг-стопом на основе ATR.

## Детали

- **Условия входа**:
  - Лонг: тело > среднее тело * BigBodyThreshold и закрытие > открытие.
  - Шорт: тело > среднее тело * BigBodyThreshold и закрытие < открытие.
- **Направление**: обе стороны.
- **Условия выхода**: трейлинг-стоп по ATR.
- **Стопы**: трейлинг-стоп ATR * AtrFactor.
- **Значения по умолчанию**:
  - `BigBodyThreshold` = 4
  - `AtrLength` = 14
  - `AtrFactor` = 2
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: импульс
  - Направление: обе стороны
  - Индикаторы: SMA, ATR
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
