# トリガーライン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

トリガーライン戦略は、加重トレンドラインと最小二乗移動平均（LSMA）を組み合わせます。加重トレンドラインがLSMAを上抜けするとロングポジションを建て、下抜けするとショートポジションを建てます。

## 仕組み
- **ロングエントリー**: 加重トレンドラインがLSMAを上抜けする。
- **ロングエグジット**: 加重トレンドラインがLSMAを下抜けする。
- **ショートエントリー**: 加重トレンドラインがLSMAを下抜けする。
- **ショートエグジット**: 加重トレンドラインがLSMAを上抜けする。
- **インジケーター**: 加重移動平均、線形回帰（LSMA）。

## パラメーター
- **WT Period** – 加重トレンドラインのルックバック期間。
- **LSMA Period** – LSMAの平滑化期間。
- **Candle Type** – 計算に使用する足の時間軸。
