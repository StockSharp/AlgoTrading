# Ichimoku クラウド ロングとショート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はIchimokuインジケーターのTenkan-senとKijun-senのクロスオーバーを使用します。クロスはTenkanの値のクラウドに対する位置によって強、中立、弱に分類されます。選択した取引モードに応じて、選んだシグナル強度が発生したときにロングまたはショートポジションを建てます。オプションのパーセントベースのテイクプロフィットとストップロスでポジションを手仕舞うか、設定に応じて反対シグナルで手仕舞います。

## 詳細

- **エントリー条件**:
  - Tenkan-senがKijun-senを上抜け、シグナル強度が選択したロングオプションと一致する。
  - Tenkan-senがKijun-senを下抜け、シグナル強度が選択したショートオプションと一致する。
- **ロング/ショート**: 設定可能、デフォルトはロング。
- **エグジット条件**:
  - エグジットオプションで定義された反対シグナル。
  - オプションのテイクプロフィットまたはストップロスのパーセンテージ。
- **ストップ**: パーセントベースのテイクプロフィットとストップロス。
- **デフォルト値**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanPeriod` = 52
  - `TakeProfitPct` = 0
  - `StopLossPct` = 0
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: Ichimoku
  - ストップ: オプション
  - 複雑さ: 中級
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
