# Strategy1
[English](README.md) | [中文](README_cn.md)

Конвертация из скрипта TradingView "strategy1". Стратегия торгует отскоки от канала Боллинджера. Открывает лонг после выхода цены ниже нижней границы и возвращения выше неё. Выход при пересечении средней линии, касании верхней границы или срабатывании стопа ниже канала.

## Детали

- **Критерии входа**: Цена была ниже нижней границы и закрылась выше неё.
- **Длинные/короткие**: Только лонг.
- **Критерии выхода**: Пересечение средней линии, касание верхней границы или стоп ниже канала.
- **Стопы**: Да, фиксированный стоп ниже канала.
- **Значения по умолчанию**:
  - `Length` = 20
  - `BufferFactor` = 0.2
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Контртренд
  - Направление: Лонг
  - Индикаторы: Bollinger Bands
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Переменный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
