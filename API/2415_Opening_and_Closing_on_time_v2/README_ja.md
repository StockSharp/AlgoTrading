# 時間指定の開始・終了戦略 v2
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

特定の時刻にトレードを開き、その日の後半に閉じる時間ベースの戦略です。取引方向は速いEMAと遅いEMAを比較することで確認されます。ストップロスとテイクプロフィットのレベルはティックで表されます。

## 詳細

- **エントリー条件**: `OpenTime` に、速いEMAが遅いEMAを上回っている場合はロング、下回っている場合はショートに入ります。方向は `TradeMode` によります。
- **ロング/ショート**: 設定可能（買い、売り、または両方）。
- **エグジット条件**: ポジションは `CloseTime` に、または防護ストップによって閉じられます。
- **ストップ**: はい、ストップロスとテイクプロフィットの両方をティックで設定。
- **デフォルト値**:
  - `OpenTime` = 05:00
  - `CloseTime` = 21:01
  - `SlowPeriod` = 200
  - `FastPeriod` = 50
  - `StopLossTicks` = 30
  - `TakeProfitTicks` = 50
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: 時間ベース
  - 方向: 設定可能
  - インジケーター: EMA
  - ストップ: 固定
  - 複雑さ: 基本
  - 時間軸: イントラデイ (1m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
