# バックテストの時間範囲の設定方法
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、特定の日付とイントラデイ時間ウィンドウへの取引を制限する方法を示しています。高速SMAが低速SMAを上抜けたときにロングエントリーし、逆のクロスオーバーでエグジットします。

## 詳細
- **データ**: 価格ロウソク足。
- **エントリー条件**:
  - **ロング**: 選択した日付とエントリー時間範囲内で高速SMAが低速SMAを上抜け。
- **エグジット条件**: 選択した日付とエグジット時間範囲内で高速SMAが低速SMAを下抜け。
- **ストップ**: なし。
- **デフォルト値**:
  - `FastLength` = 14
  - `SlowLength` = 28
  - `FromDate` = 2021-01-01
  - `ThruDate` = 2112-01-01
  - `EntryStart` = 00:00
  - `EntryEnd` = 00:00
  - `ExitStart` = 00:00
  - `ExitEnd` = 00:00
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング
  - インジケーター: SMA
  - 複雑さ: 低
  - リスクレベル: 中
