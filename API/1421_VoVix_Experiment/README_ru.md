# Стратегия The VoVix Experiment
[English](README.md) | [中文](README_cn.md)

Стратегия анализирует отношение быстрой и медленной ATR. Когда z-score этого отношения достигает пика и локального максимума, открывается позиция по направлению свечи. Позиции закрываются, когда z-score падает ниже порога выхода.

## Детали

- **Критерий входа**: z-score VoVix выше `EntryZ` и на локальном максимуме
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: z-score VoVix ниже `ExitZ`
- **Стопы**: нет
- **Значения по умолчанию**:
  - `FastAtrLength` = 13
  - `SlowAtrLength` = 26
  - `ZScoreWindow` = 81
  - `EntryZ` = 1.0
  - `ExitZ` = 1.4
  - `LocalMaxWindow` = 6
  - `SuperZ` = 2.0
  - `MinVolume` = 1
  - `MaxVolume` = 2
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Обе стороны
  - Индикаторы: ATR, Highest, SMA, StdDev
  - Стопы: Нет
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
