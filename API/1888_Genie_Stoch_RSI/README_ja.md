# Genie Stoch RSI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は相対力指数（RSI）とストキャスティクス・オシレーターの組み合わせを使用して取引します。
市場が売られすぎまたは買われすぎゾーンに達するのを待ち、その後ストキャスティクスのメインラインと
シグナルラインのクロスを探してリバーサルを確認します。リスク管理のためにトレーリングストップと
固定テイクプロフィットが適用されます。

## ロジック

1. 選択した時間軸のローソク足を購読します。
2. 設定可能な期間でRSIを計算します。
3. 設定可能な%K、%D、スローイング期間でストキャスティクス・オシレーターを計算します。
4. ロングエントリーの条件:
   - RSIが売られすぎレベルを下回っている。
   - %Kがストキャスティクスの売られすぎレベルを下回っている。
   - 前の%Kが前の%Dを下回っており、現在の%Kが現在の%Dを上抜ける。
5. ショートエントリーの条件:
   - RSIが買われすぎレベルを上回っている。
   - %Kがストキャスティクスの買われすぎレベルを上回っている。
   - 前の%Kが前の%Dを上回っており、現在の%Kが現在の%Dを下抜ける。
6. ポジションサイズは戦略の `Volume` プロパティから取得されます。反対のシグナルが現れると既存のポジションは反転されます。
7. `StartProtection` は価格ポイントで測定されたトレーリングストップとテイクプロフィットを有効にします。

## パラメーター

| 名前 | 説明 |
| ---- | ----------- |
| `RsiPeriod` | RSI計算の長さ。 |
| `KPeriod` | ストキャスティクスの%K期間。 |
| `DPeriod` | ストキャスティクスの%D期間。 |
| `Slowing` | ストキャスティクスのスローイング値。 |
| `RsiOverbought` | 買われすぎと見なされるRSIレベル。 |
| `RsiOversold` | 売られすぎと見なされるRSIレベル。 |
| `StochOverbought` | 買われすぎと見なされるストキャスティクスレベル。 |
| `StochOversold` | 売られすぎと見なされるストキャスティクスレベル。 |
| `TakeProfit` | 価格ポイントでのテイクプロフィット距離。 |
| `TrailingStop` | 価格ポイントでのトレーリングストップ距離。 |
| `CandleType` | 分析するローソク足のタイプと時間軸。 |

## 注意事項

この戦略は完成したローソク足のみを処理し、すべてのインジケーターが完全に形成されるまでシグナルを無視します。
教育用のサンプルとして設計されており、ライブ取引の前に十分にテストする必要があります。
