# 月初第一天策略

许多市场在每月首个交易日因新资金流入而表现偏强。
交易者通常在前月收盘或当天早盘提前买入以抢占先机。
本策略在月初开盘做多，并在第二个交易日开始前离场，利用资金进场带来的上涨。
若预期的强势未出现，则小幅止损以控制风险。

## 细节

- **入场条件**：日历效应触发
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：季节性
  - 方向：双向
  - 指标：季节性
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：是
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
