# Strategie-Statistiken [presentTrading]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

C#-Port der TradingView-Strategie **Strategy Stats [presentTrading]** von PresentTrading.

Die Strategie kombiniert zwei SuperTrend-Indikatoren, deren Multiplikatoren durch die Breite eines Vegas-Kanals angepasst werden. Eine Position wird nur eröffnet, wenn beide Trends übereinstimmen. Optionale Halteperioden und mehrstufige Take-Profits basierend auf festen Prozentsätzen oder ATR-Vielfachen werden unterstützt. Der Stop-Loss wird über den integrierten Schutzmechanismus verwaltet.
