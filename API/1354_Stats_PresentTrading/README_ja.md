# 戦略統計 [presentTrading]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

PresentTrading による TradingView 戦略 **Strategy Stats [presentTrading]** の C# 移植版。

この戦略は、Vegas チャネルの幅によって乗数が調整される2つの SuperTrend インジケーターを組み合わせます。両方のトレンドが一致した場合にのみポジションを建てます。オプションの保有期間と、固定パーセント水準または ATR の倍数に基づくマルチステップのテイクプロフィットをサポートします。ストップロスは組み込みの保護サービスで管理されます。
