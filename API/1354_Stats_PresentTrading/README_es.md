# Estadísticas de Estrategia [presentTrading]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Puerto en C# de la estrategia TradingView **Strategy Stats [presentTrading]** de PresentTrading.

La estrategia combina dos indicadores SuperTrend cuyos multiplicadores se ajustan por el ancho de un canal Vegas. Se abre una posición solo cuando ambas tendencias coinciden. Se admiten períodos de mantenimiento opcionales y tomas de beneficio de múltiples pasos basadas en niveles de porcentaje fijo o múltiplos de ATR. El stop-loss se gestiona a través del servicio de protección integrado.
