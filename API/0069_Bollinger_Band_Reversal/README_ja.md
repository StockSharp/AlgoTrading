# ボリンジャーバンドリバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
ボリンジャーバンドの外側での価格の極端な値は、多くの場合中央バンドに向かって戻ります。このアプローチはその拡張に逆らい、ロウソクが緑で引けると下限バンドを下回る急落で買い、赤いロウソクの後に上限バンドを上回る反発で売ります。

テストでは年平均リターン約94%を示しています。株式市場で最もよく機能します。

アルゴリズムは各バーでボリンジャーバンドを計算し、引け値が外側のバンドを突破するかどうかを確認します。強気ロウソクが下限バンドの下で引けるとロングが開かれ、弱気ロウソクが上限バンドの上で引けるとショートが取られます。ストップはATRの倍数に基づき、価格が中央バンドに戻るとエグジットが発生します。

平均回帰トレードは通常わずか数バーしか続かないため、このセットアップは短期のボラティリティ収縮に適しています。

## 詳細

- **エントリー条件**: 強気ロウソクで下限バンドを下回る引けか、弱気ロウソクで上限バンドを上回る引け。
- **ロング/ショート**: 両方。
- **エグジット条件**: 価格が中央バンドを越えるかストップロス。
- **ストップ**: はい、ATRベース。
- **デフォルト値**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 2.0
  - `AtrMultiplier` = 2.0
  - `CandleType` = 5 minute
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: Bollinger Bands, ATR
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
