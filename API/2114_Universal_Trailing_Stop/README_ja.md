# ユニバーサル・トレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MQL4のオリジナルスクリプト `cm_universal_trailing_stop.mq4` のコアアイデアを再現したものです。エントリーシグナルを生成せず、既存のポジションのストップロスを利益の方向に移動させることで管理します。

アルゴリズムは現在の価格からのオフセットを維持し、市場が設定したステップ分動くたびにストップを移動させます。最小利益閾値に達すると、トレーリングストップがアクティブになり、ロングとショートの両ポジションに対して価格を自動的に追従します。

## 詳細

- **エントリー条件**: なし。ポジションは手動または別の戦略によって開始する必要があります。
- **ロング/ショート**: 両方。
- **エグジット条件**: 価格が設定したオフセット分逆行したときにストップ注文が発動。
- **ストップ**: ポイントベースのトレーリングストップ。
- **パラメーター**:
  - `Delta` – 価格からストップまでの距離（ポイント）。
  - `Step` – ストップを移動させるための最小価格変動（ポイント）。
  - `StartProfit` – トレーリングを有効にするために必要な利益（ポイント）。
  - `CandleType` – トレーリング計算に使用する時間軸。
- **フィルター**:
  - カテゴリ: リスク管理
  - 方向: 両方
  - インジケーター: なし
  - ストップ: トレーリング
  - 複雑さ: シンプル
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
