# AFL Winner V2 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

このサンプル戦略は、StockSharpのハイレベルAPIを使ってAFL Winner V2インジケーターのロジックを再現します。インジケーターはStochasticオシレーターで近似され、シグナルはその相対的な位置と事前定義された閾値レベルから導出されます。

## 戦略ロジック

- `StochasticOscillator` を使ってAFL Winnerの動作を模倣します。
- オシレーターが強い上昇モメンタムを示すとき、ロングポジションを建てます。
- オシレーターが強い下落モメンタムを示すとき、ショートポジションを建てます。
- カラー状態がニュートラルゾーンを下回ったとき、ロングをクローズします。
- カラー状態がニュートラルゾーンを上回ったとき、ショートをクローズします。
- パラメーターによりオシレーターの期間と閾値レベルの最適化が可能です。

## パラメーター

| パラメーター | 説明                                               |
|------------|---------------------------------------------------|
| `KPeriod`  | Stochasticオシレーターの%K期間。                    |
| `DPeriod`  | Stochasticオシレーターの%D期間。                    |
| `HighLevel`| 強気シグナルの上限閾値。                            |
| `LowLevel` | 弱気シグナルの下限閾値。                            |

## ファイル

- `CS/AflWinnerV2Strategy.cs` – 戦略のコア実装。

## 注意事項

この戦略は完成したローソク足のみで動作し、意図しないエクスポージャーを避けるために自動ポジション保護を使用します。
