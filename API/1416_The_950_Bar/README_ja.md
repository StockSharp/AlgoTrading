# 9時50分バー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ニューヨーク時間9時50分の5分足ローソク足を取引します。バーが完成した後、バーの方向にエントリーし、ティックで定義された固定の利益目標とストップを設定します。

## 詳細
- **エントリー条件**: ニューヨーク9:50の5分足バーの方向。
- **ロング/ショート**: 両方。
- **エグジット条件**: 目標またはストップに到達する。
- **ストップ**: 固定ストップと目標。
- **デフォルト値**:
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
  - `TickSize` = 0.25
  - `TargetTicks` = 150
  - `StopTicks` = 200
- **フィルター**:
  - カテゴリ: 時間
  - 方向: 両方
  - インジケーター: なし
  - ストップ: 固定
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
