# Стратегия VWAP MACD

Стратегия строится на VWAP и индикаторе MACD. Длинная позиция открывается, когда цена находится выше VWAP и MACD выше сигнальной линии. Короткая позиция открывается, если цена ниже VWAP и MACD ниже сигнальной линии. Выход осуществляется при пересечении MACD своей сигнальной линии в противоположном направлении.

VWAP показывает справедливую внутридневную цену, а пересечения MACD сигнализируют об изменении импульса. Сделки открываются, когда MACD разворачивается возле уровня VWAP.

Подходит краткосрочным трейдерам импульса. Процентный стоп‑лосс не дает рискам выйти из‑под контроля.

## Подробности
- **Условия входа**:
  - Long: `Close > VWAP && MACD > Signal`
  - Short: `Close < VWAP && MACD < Signal`
- **Long/Short**: оба направления
- **Условия выхода**: обратное пересечение MACD
- **Стопы**: процентный стоп через `StopLossPercent`
- **Значения по умолчанию**:
  - `MacdFastPeriod` = 12
  - `MacdSlowPeriod` = 26
  - `MacdSignalPeriod` = 9
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: оба
  - Индикаторы: VWAP, MACD
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
