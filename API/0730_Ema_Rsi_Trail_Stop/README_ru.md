# EMA RSI Trail Stop
[English](README.md) | [中文](README_cn.md)

Стратегия использует пересечения короткой и средней EMA с фильтрацией длинной EMA. Выходы выполняются по уровням RSI, трейлинг-стопу и опциональному закрытию по времени. Фиксированный стоп превращается в трейлинг после движения цены на `TrailOffset`.

## Детали

- **Условия входа**: пересечение EMA A и EMA B с подтверждением тренда EMA C и направлением свечи.
- **Лонг/Шорт**: Оба.
- **Условия выхода**: уровни RSI, трейлинг-стоп или закрытие по времени.
- **Стопы**: фиксированный процентный стоп с переходом в трейлинг.
- **Значения по умолчанию**:
  - `EmaALength` = 10
  - `EmaBLength` = 20
  - `EmaCLength` = 100
  - `RsiLength` = 14
  - `ExitLongRsi` = 70
  - `ExitShortRsi` = 30
  - `TrailPoints` = 50
  - `TrailOffset` = 10
  - `FixStopLossPercent` = 5
  - `CloseAfterXBars` = true
  - `XBars` = 24
  - `ShowLong` = true
  - `ShowShort` = false
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA, RSI
  - Стопы: Трейлинг
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
