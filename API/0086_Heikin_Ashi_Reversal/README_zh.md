# 平滑烛线反转策略
[English](README.md) | [Русский](README_ru.md)

Heikin-Ashi 烛线能够平滑噪声并突出趋势方向。当连续的看跌 HA 烛线转为看涨，或反之，可能表示动能变化。本策略交易这种颜色翻转，并用百分比止损保护。

策略根据普通K线计算 Heikin-Ashi 值。当 HA 收盘价上穿开盘价并结束此前的看跌序列时做多；在看涨序列后下穿则做空。止损按固定百分比设置。

该方法在震荡行情下简单有效，因为普通蜡烛常带来噪声。

## 细节

- **入场条件**：Heikin-Ashi 烛线颜色翻转。
- **多/空**：双向。
- **退出条件**：止损。
- **止损**：有，按百分比。
- **默认值**：
  - `CandleType` = 15 分钟
  - `StopLoss` = 2%
- **过滤条件**：
  - 类别: 反转
  - 方向: 双向
  - 指标: Heikin-Ashi
  - 止损: 有
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等

测试表明年均收益约为 145%，该策略在加密市场表现最佳。
