# Heikin Ashi リバーサルストラテジー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Heikin Ashiローソク足はノイズを平滑化し、トレンドの方向を際立たせます。ベアリッシュなHAローソク足の連続からブリッシュなものへの転換（またはその逆）は、モメンタムの変化を示す可能性があります。この戦略はそのような色の転換を取引し、パーセンテージストップを保護に使用します。

テストでは年平均リターン約145%を示しています。暗号通貨市場で最も優れたパフォーマンスを発揮します。

ロジックは通常のローソク足からHeikin-Ashi値を計算します。ベアリッシュな連続の後にHAの終値がHAの始値を上抜けすると、ロングポジションが取られます。ブリッシュな連続の後に下抜けすると、ショートトレードが開かれます。ストップはエントリーから固定パーセンテージの位置に設定されます。

この方法は、従来のローソク足チャートがノイズの多い、不安定な相場の動きにおいてシンプルながらも効果的です。
