# Close Cross MA Strategy
[English](README.md) | [中文](README_cn.md)

Эта стратегия отслеживает простое скользящее среднее и автоматически закрывает открытую позицию, когда закрытие свечи пересекает линию среднего. Предполагается, что входы совершаются вручную или другими системами, а стратегия отвечает только за выход при смене тренда.

Логика анализирует отношение цены закрытия к среднему. Когда новая завершённая свеча переходит через линию SMA, стратегия отправляет рыночную заявку для закрытия позиции. Новые позиции не открываются.

## Подробности

- **Условия входа**: отсутствуют. Позиции должны открываться извне.
- **Условия выхода**:
  - **Лонг**: предыдущая свеча выше SMA, текущая ниже SMA — позиция закрывается продажей.
  - **Шорт**: предыдущая свеча ниже SMA, текущая выше SMA — позиция закрывается покупкой.
- **Лонг/Шорт**: поддерживаются оба направления.
- **Стопы**: не используются, сигналы выхода формируются пересечением SMA.
- **Параметры по умолчанию**:
  - `MA Period` = 50.
  - `Candle Type` = тайм‑фрейм 1 минута.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Простая
  - Тайм‑фрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
