# Стратегия Vegas SuperTrend Enhanced
[English](README.md) | [中文](README_cn.md)

Сочетает канал Vegas и модифицированный SuperTrend.
Вход при смене направления SuperTrend с учётом волатильности.

## Детали

- **Критерий входа**: смена тренда по скорректированному SuperTrend
- **Длин/Шорт**: обе стороны (настраивается)
- **Критерий выхода**: противоположная смена тренда
- **Стопы**: нет
- **Значения по умолчанию**:
  - `AtrPeriod` = 10
  - `VegasWindow` = 100
  - `SuperTrendMultiplier` = 5
  - `VolatilityAdjustment` = 5
  - `TradeDirection` = "Both"
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: ATR, SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
