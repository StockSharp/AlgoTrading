# London ブレイクアウト・クラシック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はアジアレンジを使ってLondonセッションのブレイクアウトを取引します。00:00から06:55 UTCの高値と安値がボックスを形成します。07:00 UTC以降、高値を上抜けたらロングポジション、安値を下抜けたらショートポジションをオープンします。ストップロスはボックスの中間点に設置し、テイクプロフィットは設定可能なリスクリワード比を使用します。

## 詳細

- **エントリー条件**:
  - ロング: 価格がアジアセッションの高値を上抜け。
  - ショート: 価格がアジアセッションの安値を下抜け。
- **エグジット条件**:
  - ストップロスまたはテイクプロフィット。
  - 取引ウィンドウの終了。
- **ストップ**: はい。
- **デフォルト値**:
  - アジアセッション: 00:00–06:55 UTC。
  - 取引セッション: 07:00–16:00 UTC。
  - CRV = 1。
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
