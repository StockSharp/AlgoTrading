# Стратегия London BreakOut Classic
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует пробой Лондонской сессии, используя диапазон азиатской сессии. Максимум и минимум между 00:00 и 06:55 UTC образуют диапазон. После 07:00 UTC пробой вверх открывает длинную позицию, пробой вниз — короткую. Стоп-лосс устанавливается на середине диапазона, тейк-профит рассчитывается по настраиваемому соотношению риск-прибыль.

## Детали

- **Условия входа**:
  - Long: цена пересекает максимум азиатской сессии.
  - Short: цена пересекает минимум азиатской сессии.
- **Условия выхода**:
  - Стоп-лосс или тейк-профит.
  - Конец торгового окна.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - Азиатская сессия: 00:00–06:55 UTC.
  - Торговая сессия: 07:00–16:00 UTC.
  - CRV = 1.
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
