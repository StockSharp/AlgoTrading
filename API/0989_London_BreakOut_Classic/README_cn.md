# 伦敦突破经典策略
[English](README.md) | [Русский](README_ru.md)

该策略利用亚洲时段的高低点构建区间，在伦敦时段突破时入场。00:00 至 06:55 UTC 的最高价和最低价形成箱体。07:00 UTC 之后，价格突破上沿则做多，突破下沿则做空。止损设在箱体中点，止盈按可调的风险回报比计算。

## 细节

- **入场条件**:
  - 多头：价格突破亚洲时段高点。
  - 空头：价格跌破亚洲时段低点。
- **出场条件**:
  - 止损或止盈。
  - 交易窗口结束。
- **止损**: 有。
- **默认值**:
  - 亚洲时段：00:00–06:55 UTC。
  - 交易时段：07:00–16:00 UTC。
  - CRV = 1。
- **过滤器**:
  - 类型：突破
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
