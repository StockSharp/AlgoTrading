# Стратегия Limits Bot
[English](README.md) | [中文](README_cn.md)

Размещает симметричные лимитные заявки вокруг цены открытия свечи и защищает позиции стоп-лоссом, тейк-профитом и опциональным трейлинг-стопом.

## Подробности

- **Вход**:
  - Покупка лимит по цене `Open - StopOrderDistance * PriceStep`, если разрешены покупки.
  - Продажа лимит по цене `Open + StopOrderDistance * PriceStep`, если разрешены продажи.
- **Выход**: Закрытие по рынку при срабатывании стоп-лосса, тейк-профита или трейлинг-стопа.
- **Длинные/Короткие**: Оба направления.
- **Стопы**: Фиксированный стоп-лосс с трейлингом.
- **Значения по умолчанию**:
  - `StopOrderDistance` = 5
  - `TakeProfit` = 35
  - `StopLoss` = 8
  - `TrailingStart` = 40
  - `TrailingDistance` = 30
  - `TrailingStep` = 1
  - `CandleType` = 1 минута
- **Сессия**: Торговля только между `StartTime` и `EndTime`.
- **Фильтры**:
  - Категория: Ценовое действие
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
