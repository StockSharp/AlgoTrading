# Стратегия Donchian CCI

Эта стратегия использует индикаторы Donchian и CCI для формирования сигналов. Вход в длинную позицию выполняется, когда цена превышает верхнюю границу канала Donchian, а CCI опускается ниже -100, что указывает на пробой вверх при перепроданности. Короткая позиция открывается, когда цена пробивает нижнюю границу и CCI поднимается выше 100 — пробой вниз при перекупленности.

Стратегия подходит трейдерам, ищущим возможности в смешанном рынке.

## Детали
- **Условия входа**:
  - **Лонг**: Цена > верхней границы Donchian и CCI < -100 (пробой вверх при перепроданности)
  - **Шорт**: Цена < нижней границы Donchian и CCI > 100 (пробой вниз при перекупленности)
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - **Лонг**: Закрытие длинной позиции при уходе цены ниже средней линии
  - **Шорт**: Закрытие короткой позиции при уходе цены выше средней линии
- **Стопы**: да.
- **Значения по умолчанию**:
  - `DonchianPeriod` = 20
  - `CciPeriod` = 20
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Смешанная
  - Направление: Оба
  - Индикаторы: Donchian CCI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
