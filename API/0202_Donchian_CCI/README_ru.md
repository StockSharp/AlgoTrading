# Стратегия Donchian CCI

Данная стратегия использует индикаторы канала Дончиана и CCI. Длинная позиция открывается, когда цена выше верхней границы канала и CCI опускается ниже -100, что говорит о пробое вверх на фоне перепроданности. Короткая позиция открывается при цене ниже нижней границы и значении CCI выше 100, то есть пробое вниз при перекупленности. Подходит для поиска возможностей на смешанных рынках.

## Подробности
- **Условия входа**:
  - **Long**: Price > Donchian Upper && CCI < -100
  - **Short**: Price < Donchian Lower && CCI > 100
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход при падении цены ниже средней линии
  - **Short**: выход при росте цены выше средней линии
- **Стопы**: да
- **Значения по умолчанию**:
  - `DonchianPeriod` = 20
  - `CciPeriod` = 20
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: Donchian CCI
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
