# Стратегия Combo 2/20 EMA Bandpass Filter
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает пересечение быстрой и медленной EMA с полосовым фильтром. Длинные позиции открываются, когда быстрая EMA выше медленной и значение фильтра превышает зону продаж. Короткие позиции открываются, когда быстрая EMA ниже медленной и значение фильтра опускается ниже зоны покупок. Позиции закрываются при исчезновении сигналов или до наступления стартовой даты.

Тесты показывают среднегодовую доходность около 47%. Лучшая эффективность достигается на рынке криптовалют.

## Детали
- **Условия входа**:
  - **Лонг**: Быстрая EMA > Медленная EMA и фильтр > зона продаж
  - **Шорт**: Быстрая EMA < Медленная EMA и фильтр < зона покупок
- **Лонг/Шорт**: обе стороны
- **Условия выхода**: закрыть позицию при исчезновении сигналов
- **Стопы**: нет
- **Значения по умолчанию**:
  - `FastEmaLength` = 2
  - `SlowEmaLength` = 20
  - `BpfLength` = 20
  - `BpfDelta` = 0.5m
  - `BpfSellZone` = 5m
  - `BpfBuyZone` = -5m
  - `StartDate` = new DateTimeOffset(2005, 1, 1, 0, 0, 0, TimeSpan.Zero)
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Trend
  - Направление: обе
  - Индикаторы: EMA Bandpass Filter
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
