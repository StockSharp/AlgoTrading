# MACD Long戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSIの極値とMACDのクロスオーバーを組み合わせて、トレンド内の押し目を捉えます。RSIが極値に達した後、システムはエントリー前に確認のMACDクロスオーバーを待ちます。このアプローチはノイジーなモメンタムの変化をフィルタリングし、高確率のリバーサルに焦点を当てます。

この戦略は両方向で取引し、反対のシグナルが現れると素早く転換できます。MACDはモメンタムの確認を提供し、RSIは買われすぎ・売られすぎのゾーンを示します。エンジンのリスク管理機能を通じて保護的なストップを追加できます。

## 詳細

- **エントリー条件**:
  - **ロング**: RSIが売られすぎを下回り、その後MACDラインがシグナルを上抜けする。
  - **ショート**: RSIが買われすぎを上回り、その後MACDラインがシグナルを下抜けする。
- **エグジット条件**:
  - 反対のクロスオーバーまたはストップ発動。
- **インジケーター**:
  - RSI (長さ14、売られすぎ30、買われすぎ70)
  - MACD (速い12、遅い26、シグナル9)
- **ストップ**: StartProtectionまたは外部のマネー管理で実装。
- **デフォルト値**:
  - `RsiLength` = 14
  - `Oversold` = 30
  - `Overbought` = 70
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
- **フィルター**:
  - モメンタムリバーサル
  - 様々な時間軸で機能
  - インジケーター: RSI、MACD
  - ストップ: オプション
  - 複雑さ: 基本
