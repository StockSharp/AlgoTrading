# 反張りMA取引戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

毎週の高値・安値と移動平均線を評価し、各週の終わりに取引を開始するウィークリー反張りシステムです。方向に関わらずポジションは1週間保持されます。

この手法は主要通貨ペア向けに設計されていますが、週次データのある流動性の高い任意の資産に適用できます。

## 詳細

- **エントリー条件**:
  - **買い**: 前週終値がルックバック期間の最高値を上回る、または移動平均線が週の始値を上回る。
  - **売り**: 前週終値がルックバック期間の最安値を下回る、または移動平均線が週の始値を下回る。
- **ロング/ショート**: 両方。
- **エグジット条件**: 1週間保持後にポジションをクローズ。
- **ストップ**: なし。
- **時間軸**: 週足ローソク足。
