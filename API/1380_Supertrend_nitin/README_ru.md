# Supertrend Nitin
[English](README.md) | [中文](README_cn.md)

Стратегия Supertrend от Nitin на 5-минутных свечах.

## Детали

- **Критерии входа**: Изменение направления вверх.
- **Длинные/короткие**: Только длинные.
- **Критерии выхода**: Изменение направления вниз.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `AtrPeriod` = 10
  - `Multiplier` = 3m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Длинные
  - Индикаторы: ATR, Supertrend
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
