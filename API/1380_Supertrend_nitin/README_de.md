Supertrend-Strategie von nitin auf 5-Minuten-Kerzen.

## Details

- **Einstiegskriterien**: Richtungswechsel nach oben.
- **Long/Short**: Nur Long.
- **Ausstiegskriterien**: Richtungswechsel nach unten.
- **Stops**: Nein.
- **Standardwerte**:
  - `AtrPeriod` = 10
  - `Multiplier` = 3m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Filter**:
  - Kategorie: Trend
  - Richtung: Nur Long
  - Indikatoren: ATR, Supertrend
  - Stops: Nein
  - Komplexität: Grundlegend
  - Zeitrahmen: Intraday (5m)
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
