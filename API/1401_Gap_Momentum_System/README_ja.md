# ギャップ・モメンタム・システム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Perry KaufmanのGap Momentumシステムを実装します。この戦略は上昇・下降の累積ギャップを比較し、シグナルが上昇または下降したときに取引します。

## 詳細
- **エントリー条件**: シグナル上昇 -> 買い、シグナル下降 -> 売りまたは反転。
- **ロング/ショート**: 設定可能。
- **エグジット条件**: 逆シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `Period` = 40
  - `SignalPeriod` = 20
  - `LongOnly` = true
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方またはロングのみ
  - インジケーター: Gap momentum
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
