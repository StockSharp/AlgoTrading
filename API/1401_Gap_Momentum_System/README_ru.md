# Стратегия Gap Momentum System
[English](README.md) | [中文](README_cn.md)

Реализует систему Gap Momentum от Перри Кауфмана. Стратегия сравнивает накопленные гэпы вверх и вниз и торгует при росте или падении сигнала.

## Подробности
- **Условия входа**: Рост сигнала -> покупка, падение сигнала -> продажа или разворот.
- **Направление**: Задаётся параметром.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Period` = 40
  - `SignalPeriod` = 20
  - `LongOnly` = true
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба или только лонг
  - Индикаторы: Gap momentum
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Daily
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
