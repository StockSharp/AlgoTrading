# LSMA 高速シンプル代替計算戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は `3 × WMA − 2 × SMA` として計算される最小二乗移動平均 (LSMA) の高速近似を使用します。価格がLSMAを上回るクロスをしたときにロングポジションを建て、下回るクロスをしたときにショートポジションを建てます。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値がLSMAを上回るクロス。
  - **ショート**: 終値がLSMAを下回るクロス。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対のシグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - 期間 25。
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: WMA, SMA
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 指定なし
