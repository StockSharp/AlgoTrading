# TSI SuperTrend判断戦略 presentTrading
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

相関ベースのトレンド強度指数とSuperTrendの方向を組み合わせる。SuperTrendが強気でTSIがマイナス閾値を上回っているときにロングトレードを行う。SuperTrendが弱気でTSIがプラス閾値を下回っているときにショートトレードを行う。オプションのテイクプロフィットとストップロスを有効にできる。

- インジケーター: Correlation, SuperTrend
