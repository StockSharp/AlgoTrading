# Estrategia de Decisión TSI SuperTrend presentTrading
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Combina un Índice de Fuerza de Tendencia basado en correlación con la dirección del SuperTrend. Se toman operaciones largas cuando el SuperTrend es alcista y el TSI está por encima del umbral negativo. Se toman operaciones cortas cuando el SuperTrend es bajista y el TSI está por debajo del umbral positivo. Se puede habilitar take profit y stop loss opcionales.

- Indicadores: Correlation, SuperTrend
