# 三均线策略
[English](README.md) | [Русский](README_ru.md)

当短期均线上穿或下穿中期均线且两者相对于长期均线位置一致时，该策略开仓。

## 细节

- **入场条件**：
  - **多头**：短期均线上穿中期均线，且中期均线在长期均线之上。
  - **空头**：短期均线下穿中期均线，且中期均线在长期均线之下。
- **多/空**：双向。
- **出场条件**：
  - 反向交叉。
- **止损**：无。
- **默认值**：
  - `ShortMa` = 20
  - `MediumMa` = 50
  - `LongMa` = 200
