# Стратегия «Покупка, когда на улицах кровь»
[English](README.md) | [中文](README_cn.md)

Стратегия покупает, когда текущая просадка относительно недавнего максимума опускается ниже порога стандартного отклонения. Позиция закрывается после фиксированного числа баров.

## Детали

- **Условия входа**:
  - Лонг: просадка ≤ среднее + `StdDevThreshold` × стандартное отклонение
- **Направление**: только лонг
- **Условия выхода**: позиция закрывается после `ExitBars` баров
- **Стопы**: нет
- **Значения по умолчанию**:
  - `LookbackPeriod` = 50
  - `StdDevLength` = 50
  - `StdDevThreshold` = -1m
  - `ExitBars` = 35
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Reversal
  - Направление: Long
  - Индикаторы: Highest, SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
