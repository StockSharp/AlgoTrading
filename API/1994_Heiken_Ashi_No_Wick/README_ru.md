# Стратегия Heiken Ashi без теней
[English](README.md) | [中文](README_cn.md)

Стратегия использует свечи Heiken Ashi и торгует против направления свечей без теней. Бычья свеча Heiken Ashi с телом больше предыдущего и без нижней тени открывает короткую позицию. Медвежья свеча с телом больше предыдущего и без верхней тени открывает длинную позицию. Позиции закрываются при появлении противоположной свечи без соответствующей тени.

## Детали

- **Условия входа**: бычья HA без нижней тени и с большим телом для шорта; медвежья HA без верхней тени и с большим телом для лонга
- **Лонг/Шорт**: Лонг и Шорт
- **Условия выхода**: противоположная свеча HA без тени
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `CandleType` = 15-минутные свечи
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Разворот
  - Индикаторы: Heikin-Ashi
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
