# Стратегия RSI Long-Term 15min
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует сигналы перепроданности RSI в сочетании с долгосрочными скользящими средними и подтверждением объема для входа в длинные позиции. Покупка происходит, когда RSI ниже 30, SMA(250) выше SMA(500), а объем значительно превышает средний.

## Детали

- **Критерий входа**: RSI ниже 30, SMA(250) выше SMA(500) и объем более чем в 2.5 раза выше своей 20-периодной SMA
- **Длин/Шорт**: только лонг
- **Критерий выхода**: пересечение SMA(250) ниже SMA(500) или стоп-лосс
- **Стопы**: да, фиксированный процент
- **Значения по умолчанию**:
  - `RsiLength` = 10
  - `VolumeSmaLength` = 20
  - `Sma1Length` = 250
  - `Sma2Length` = 500
  - `VolumeMultiplier` = 2.5
  - `StopLossPercent` = 5
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: RSI, SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
