# Genie戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

GenieはParabolic SARをベースにしたエキスパートアドバイザーで、トレンドの強さを確認するためにAverage Directional Index（ADX）を組み合わせています。SARが価格に対して反転し、ADXの+DIと-DIが優勢を切り替えたときにポジションを開きます。トレーリングストップと固定テイクプロフィットでリスクを管理します。

テストでは、このアプローチが中程度のボラティリティを持つトレンド相場で最もよく機能することが示されています。

## 詳細

- **エントリー条件**:
  - **ロング**: 前回のSARが前回の終値より上、現在のSARが現在の終値より下、前回の+DI < 前回の-DI、現在の+DI > 現在の-DI、かつADXが現在の+DIと-DI両方より上。
  - **ショート**: 前回のSARが前回の終値より下、現在のSARが現在の終値より上、前回の+DI > 前回の-DI、現在の+DI < 現在の-DI、かつADXが現在の+DIと-DI両方より上。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - トレーリングストップが発動するか、前のローソク足がポジションに逆らって引ける。
- **ストップ**: あり、価格単位で測定されたトレーリングストップとテイクプロフィット。
- **デフォルト値**:
  - `TakeProfit` = 500
  - `TrailingStop` = 200
  - `SarStep` = 0.02
  - `AdxPeriod` = 14
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Parabolic SAR, ADX
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい（+DIと-DIの間）
  - リスクレベル: 中
