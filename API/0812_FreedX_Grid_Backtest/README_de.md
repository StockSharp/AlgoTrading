# FreedX Grid Backtest
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Grid-Trading-Strategie, die Aufträge auf mehrere Preisniveaus verteilt.
Kauft, wenn der Kurs auf ein niedrigeres Grid-Niveau fällt, und verkauft auf dem nächst höheren Niveau.
Unterstützt Long-, Short- und neutrale Handelsmodi.
