# MACD vs Signal戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MACDラインがシグナルラインをクロスする際に取引する戦略です。

MACDラインがシグナルラインを上抜けたときにロングエントリーします。
MACDラインがシグナルラインを下抜けたときにショートエントリーします。
オプションでストップロス、テイクプロフィット、トレーリングストップを適用します。

## 詳細

- **エントリー条件**:
  - ロング: `MACDがSignalを上抜け`
  - ショート: `MACDがSignalを下抜け`
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 逆のMACDクロスオーバー
  - リスク管理ルール（ストップロス、トレーリングストップ、テイクプロフィット）
- **ストップ**: ストップロス、テイクプロフィット、トレーリングストップ（オプション）
- **デフォルト値**:
  - `FastPeriod` = 12
  - `SlowPeriod` = 26
  - `SignalPeriod` = 9
  - `StopLoss` = 50ポイント
  - `TakeProfit` = 999ポイント
  - `TrailingStop` = 0ポイント（無効）
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: MACD
  - ストップ: ストップロス / テイクプロフィット / トレーリング
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
