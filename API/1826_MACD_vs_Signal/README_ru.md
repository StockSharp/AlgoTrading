# Стратегия MACD vs Signal
[English](README.md) | [中文](README_cn.md)

Стратегия основана на пересечении линии MACD и сигнальной линии.

Открывает длинную позицию, когда линия MACD пересекает сигнальную снизу вверх.
Открывает короткую позицию, когда линия MACD пересекает сигнальную сверху вниз.
Опционально применяет стоп-лосс, тейк-профит и трейлинг-стоп.

## Подробности

- **Условия входа**:
  - Длинная: `MACD пересекает Signal снизу вверх`
  - Короткая: `MACD пересекает Signal сверху вниз`
- **Long/Short**: Оба
- **Условия выхода**:
  - Обратное пересечение MACD
  - Правила управления рисками (стоп-лосс, трейлинг-стоп, тейк-профит)
- **Стопы**: Стоп-лосс, тейк-профит, трейлинг-стоп (опционально)
- **Параметры по умолчанию**:
  - `FastPeriod` = 12
  - `SlowPeriod` = 26
  - `SignalPeriod` = 9
  - `StopLoss` = 50 пунктов
  - `TakeProfit` = 999 пунктов
  - `TrailingStop` = 0 пунктов (отключено)
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Trend following
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Stop loss / Take profit / Trailing
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
