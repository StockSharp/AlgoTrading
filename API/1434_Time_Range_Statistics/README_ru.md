# Стратегия Time Range Statistics
[English](README.md) | [中文](README_cn.md)

Собирает простую статистику между выбранными индексами баров.
Логирует среднюю цену, нормализованный диапазон, процентное изменение, средний объём и количество гэпов.
В конце периода открывает позицию по направлению изменения.

## Детали

- **Критерий входа**: направление процентного изменения в `EndIndex`
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: нет
- **Стопы**: нет
- **Значения по умолчанию**:
  - `StartIndex` = 9000
  - `EndIndex` = 10000
- **Фильтры**:
  - Категория: Статистика
  - Направление: обе стороны
  - Индикаторы: нет
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
