# 时间范围统计策略
[English](README.md) | [Русский](README_ru.md)

在选定的条目索引之间收集简单统计数据。
记录平均价格、归一化范围、百分比变化、平均成交量和跳空次数。
在区间末尾根据变化方向开仓。

## 详情

- **入场条件**：`EndIndex` 处的百分比变化方向
- **多空**：双向
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `StartIndex` = 9000
  - `EndIndex` = 10000
- **过滤器**：
  - 分类：统计
  - 方向：双向
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险级别：低
