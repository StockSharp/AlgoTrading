# Стратегия Grover Llorens Activator
[English](README.md) | [中文](README_cn.md)

Адаптивная стратегия на основе ATR, меняющая направление при пересечении цены с внутренней линией.

Покупает при пересечении разницы между ценой и линией снизу вверх. Продаёт при пересечении сверху вниз.

## Детали

- **Условия входа**: Пересечение цены с линией, рассчитанной через ATR.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Length` = 480
  - `Multiplier` = 14
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ATR
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
