# Стратегия RSI и обратновзвешенного MA
[English](README.md) | [中文](README_cn.md)

Стратегия использует RSI и обратновзвешенное скользящее среднее с фильтром скорости изменения. Покупает при RSI выше порога и отрицательном ROC MA, продаёт при RSI ниже порога и положительном ROC MA. Включает трейлинг-стоп на основе ATR и управление капиталом фиксированным соотношением.

## Детали

- **Условия входа**:
  - **Лонг**: `RSI >= RsiLongSignal` и `MA ROC <= RocMaLongSignal`
  - **Шорт**: `RSI <= RsiShortSignal` и `MA ROC >= RocMaShortSignal`
- **Длинные/короткие**: обе стороны.
- **Условия выхода**: противоположный сигнал, стоп или трейлинг-стоп.
- **Стопы**: да, ATR трейлинг и ограничение потерь.
- **Значения по умолчанию**:
  - `RsiLength` = 20
  - `MaType` = RWMA
  - `MaLength` = 19
  - `RsiLongSignal` = 60
  - `RsiShortSignal` = 40
  - `TakeProfitActivation` = 5
  - `TrailingPercent` = 3
  - `MaxLossPercent` = 10
  - `FixedRatio` = 400
  - `IncreasingOrderAmount` = 200
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: RSI, Moving Average, ATR
  - Стопы: Да
  - Сложность: Высокая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
