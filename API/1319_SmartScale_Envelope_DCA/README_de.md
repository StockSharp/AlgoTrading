# SmartScale Envelope DCA-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Verwendet einen gleitenden Durchschnitt-Envelope für Dollar-Cost-Averaging. Kauft, wenn der Preis unter das untere Band fällt, und schließt alle Positionen, wenn der Preis über das obere Band steigt, mit Take-Profit- oder Stop-Loss-Regeln.
