# Estrategia de DCA con Envelope SmartScale
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Utiliza un envelope de media móvil para realizar promedios de costo en dólares. Compra cuando el precio cae por debajo de la banda inferior y cierra todas las posiciones cuando el precio cruza por encima de la banda superior con reglas de take-profit o stop-loss.
