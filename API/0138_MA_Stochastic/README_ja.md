# MA Stochastic 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
MA Stochastic は、移動平均のトレンドフィルターとストキャスティクスオシレーターの押し目を組み合わせます。
価格が平均を上回りながら上昇トレンドにあり、ストキャスティクスが売られすぎゾーンに下落したとき、次の上昇転換での買いを準備します。

テストでは平均年間リターンは約151%を示しています。株式市場で最も良いパフォーマンスを発揮します。

ショートトレードは下降トレンドでこのロジックを反映し、ストキャスティクスが買われすぎに達したときにラリーを売ります。

固定パーセントストップが、トレンドが突然反転した場合の大きな損失を防ぎます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Moving Average, Stochastic
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
