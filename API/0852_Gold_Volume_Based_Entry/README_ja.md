# ゴールド・出来高ベースエントリー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、連続する2本の強気出来高バーが出来高移動平均を上回ったときに買います。2本目のバーは1本目よりも出来高が多い必要があります。価格が所定の金額分有利方向に動いたら、固定利益目標でポジションを決済します。

## 詳細

- **エントリー条件**:
  - 出来高移動平均を上回る2本の連続した強気出来高バーで出来高が増加していること。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - `entry price + Target Move` での固定利益目標。
- **ストップ**: なし。
- **デフォルト値**:
  - `Volume MA Period` = 20。
  - `Target Move` = 5。
- **フィルター**:
  - カテゴリ: 出来高
  - 方向: ロング
  - インジケーター: 単一
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
