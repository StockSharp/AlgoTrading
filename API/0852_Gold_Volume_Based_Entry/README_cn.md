# 基于成交量的黄金入场策略
[English](README.md) | [Русский](README_ru.md)

该策略在连续两根放量的看涨柱体超过成交量均线时买入，且第二根柱体的成交量必须高于第一根。价格按预设幅度上涨后，以固定盈利目标平仓。

## 细节

- **入场条件**：
  - 连续两根看涨成交量柱体高于成交量均线且成交量递增。
- **方向**：仅做多。
- **出场条件**：
  - 固定盈利目标 `entry price + Target Move`。
- **止损**：无。
- **默认参数**：
  - `Volume MA Period` = 20。
  - `Target Move` = 5。
- **筛选**：
  - 类型：成交量
  - 方向：做多
  - 指标：单一
  - 止损：否
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：低
