# Стратегия входа по золоту на основе объема
[English](README.md) | [中文](README_cn.md)

Эта стратегия покупает, когда два подряд зеленых объемных бара превышают скользящую среднюю объема. Второй бар должен иметь больший объем, чем первый. Фиксированная цель по прибыли закрывает позицию после движения цены на заданную величину.

## Детали

- **Условия входа**:
  - Два бычьих объемных бара выше скользящей средней объема и с ростом объема.
- **Направление торговли**: только лонг.
- **Условия выхода**:
  - Фиксированная цель `entry price + Target Move`.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `Volume MA Period` = 20.
  - `Target Move` = 5.
- **Фильтры**:
  - Категория: Объем
  - Направление: Лонг
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
