# Expert MACD EURUSD 1時間足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略はMetaTrader 5エキスパートアドバイザー**Expert MACD EURUSD 1 Hour**のC#翻訳版です。**5 / 15 / 3**の短期、長期、シグナル期間でMACDインジケーターを使用して1時間足ローソク足で取引します。MACDメインラインがゼロレベルを上下にクロスし、シグナルラインがその動きを確認する強力なモメンタムシフトを探します。トレーリングストップがオープンポジションを保護するために使用され、MACD の傾きが現在のポジションに逆らうとトレードが閉じられます。

## パラメーター

- `FastLength` – MACDの高速EMA期間（デフォルト：5）。
- `SlowLength` – MACDの低速EMA期間（デフォルト：15）。
- `SignalLength` – MACDのシグナルライン期間（デフォルト：3）。
- `TrailingPoints` – 価格ポイントでのトレーリングストップ距離（デフォルト：25）。
- `CandleType` – ローソク足の時間軸（デフォルト：1時間）。
- 戦略の`Volume`プロパティが注文サイズを制御します。

## 取引ロジック

### ロングエントリー
1. シグナルライン値：`mac8 > mac7 > mac6`かつ`mac6 < mac5`（シグナルライン上昇）。
2. メインライン値：`mac4 > mac3 < mac2 < mac1`（落ち込んだ後にメインライン上昇）。
3. `mac2 < -0.00020`、`mac4 < 0`かつ`mac1 > 0.00020` – メインラインがゼロを上方クロス。
4. すべての条件が満たされ、ロングポジションが開いていなければ、成行きで買い。

### ショートエントリー
1. シグナルライン値：`mac8 < mac7 < mac6`かつ`mac6 > mac5`（シグナルライン下降）。
2. メインライン値：`mac4 < mac3 > mac2 > mac1`（ピークの後にメインライン下降）。
3. `mac2 > 0.00020`、`mac4 > 0`かつ`mac1 < -0.00035` – メインラインがゼロを下方クロス。
4. すべての条件が満たされ、ショートポジションが開いていなければ、成行きで売り。

### エグジットルール
- 現在のメイン値が前の値より低い時にロングを閉じる。
- 現在のメイン値が前の値より高い時にショートを閉じる。
- トレーリングストップは各ローソク足で更新され、価格がストップレベルをクロスするとエグジットします。

## 注意事項

この例はインジケーター結合と手動トレーリングストップ管理を備えたStockSharpの高レベルAPIの使用を示しています。教育目的を意図しており、固定`Volume`パラメーターを超えるマネー管理は含まれていません。
