# Estratégia de Teste de Sinais
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia alterna entre uma posição comprada e posição neutra a cada vela concluída. Compra quando não há posição aberta e vende na próxima vela para fechar. Use-a para verificar a entrega de sinais sem depender de indicadores.
