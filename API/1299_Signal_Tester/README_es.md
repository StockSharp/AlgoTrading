# Estrategia de Prueba de Señales
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia alterna entre una posición larga y posición plana en cada vela completada. Compra cuando no hay posición abierta y vende en la siguiente vela para cerrar. Úsala para verificar la entrega de señales sin depender de indicadores.
