# Signal-Tester-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie wechselt bei jeder abgeschlossenen Kerze zwischen einer Long-Position und einer Flat-Position. Sie kauft, wenn keine Position offen ist, und verkauft auf der nächsten Kerze zum Schließen. Verwende sie, um die Signalzustellung zu überprüfen, ohne auf Indikatoren zu setzen.
