# Стратегия ATR Sell the Rip Mean Reversion
[English](README.md) | [中文](README_cn.md)

Шортовая стратегия, продающая при росте цены выше сглаженного порога ATR и закрывающая позицию при падении ниже предыдущего минимума. Дополнительный фильтр EMA ограничивает сделки нисходящими трендами.

## Детали

- **Критерий входа**: закрытие выше сглаженного (close + ATR * множитель)
- **Длин/Шорт**: шорт
- **Критерий выхода**: закрытие ниже предыдущего минимума
- **Стопы**: нет
- **Значения по умолчанию**:
  - `AtrPeriod` = 20
  - `AtrMultiplier` = 1.0
  - `SmoothPeriod` = 10
  - `EmaPeriod` = 200
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Шорт
  - Индикаторы: ATR, SMA, EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
