# ATR 急騰売り平均回帰戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

価格が平滑化されたATR閾値を上回ったときに売り、前の安値を下回ったときに手仕舞うショートのみの戦略。オプションのEMAフィルターで取引を下降トレンドに限定する。

## 詳細

- **エントリー条件**: 終値が平滑化された (close + ATR * 乗数) を上回る
- **ロング/ショート**: ショート
- **エグジット条件**: 終値が前の安値を下回る
- **ストップ**: いいえ
- **デフォルト値**:
  - `AtrPeriod` = 20
  - `AtrMultiplier` = 1.0
  - `SmoothPeriod` = 10
  - `EmaPeriod` = 200
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ショート
  - インジケーター: ATR, SMA, EMA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
