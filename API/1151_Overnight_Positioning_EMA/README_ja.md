# EMAを使ったオーバーナイト・ポジショニング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

選択した市場の終値直前にロングポジションを建て、市場オープン後に決済します。オプションのEMAフィルターでエントリーを確認します。米国・アジア・欧州のセッションをサポートし、週末前にすべてのオープンポジションを閉じます。

## 詳細

- **エントリー**: EMAより価格が上にある場合（有効時）、市場終値の数分前。
- **エグジット**: 指定した分数だけ市場オープン後、または金曜日の終値5分前。
- **市場**: 米国、アジア、または欧州。
- **インジケーター**: EMA。
- **方向**: ロングのみ。
- **ストップ**: なし。
