# Macd パターン トレーダー ダブルトップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

MetaTrader 4 エキスパート アドバイザー **MacdPatternTraderv04cb** のポート。この戦略は、構成可能な MACD 主回線をスキャンします。
弱気のダブルトップパターンと強気のダブルボトムパターン。 MACD の間に 2 回目のスイングが最初のスイングを超えられなかった場合
プラスまたはマイナスのトリガーレベルを超えたままの場合、戦略は市場ポジションを次の方向に開きます。
予想される逆転。プロテクト注文は、元の固定の 100 ピップのストップロスと 300 ピップのテイクプロフィット距離を再現します。

## 取引ルール

1. 選択したローソク足シリーズ (デフォルト: 30 分の時間枠) をサブスクライブし、次の値を使用して MACD メインラインを計算します。
構成された高速、低速、および信号期間 (デフォルト: 5、13、および 1)。
2. 完了した最後の 3 つの MACD 値を追跡します。 MACD が `TriggerLevel` を上回れば、弱気のセットアップは準備完了です。
局所的な高値を形成してから下落します。セットアップは、次の MACD の高値が以前に保存された高値より低いかどうかを検証します。
MACD がまだトリガーを上回っている間は High になります。その瞬間に市場の売りが送信されます。
3. 同じロジックをゼロ以下でミラーリングします。 MACD が `-TriggerLevel` を下回ったままになると、谷と次の谷が形成されます
前回よりも高い場合、この戦略は市場での買いを開始します。
4. MACD の線が `[-TriggerLevel, TriggerLevel]` 内で交差するたびに、保存されている山と谷をリセットします。
範囲。これは、勢いが弱まったときにパターン検索をキャンセルする元の EA の動作と一致します。
5. ポジションサイズは設定された `TradeVolume` から始まります。方向を切り替えるとき、この戦略は十分なボリュームを追加します。
新しい取引を確立する前に、反対のエクスポージャをフラット化します。
6. 開始時に `StartProtection` を 1 回呼び出して、100 ピップのストップロスと 300 ピップのテイクプロフィットの両方が
再起動後もプラットフォーム。

## パラメーター

| 名前 | 説明 |
| ---- | ----------- |
| `FastPeriod` | MACD によって使用される高速な EMA の長さ。 |
| `SlowPeriod` | MACD が使用する低速の EMA の長さ。 |
| `SignalPeriod` | MACD の信号線の平滑化長。 |
| `TriggerLevel` | ダブルトップ/ダブルボトム検出を有効にするには絶対 MACD レベルが必要です。 |
| `StopLossPips` | ピップ単位での保護停止の距離 (デフォルトは 100)。 |
| `TakeProfitPips` | 利益確定の距離 (pips 単位) (デフォルトは 300)。 |
| `TradeVolume` | 新しいポジションの基本注文量。 |
| `CandleType` | インジケーターの計算に使用されるローソク足シリーズ。 |

## 注意事項

- ストップロスとテイクプロフィットは、ピップスから商品ステップに変換されてから、
`StartProtection`、動作は元の MQL4 エキスパートと同一に保たれます。
