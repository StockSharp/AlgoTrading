# 80-20 策略
[English](README.md) | [Русский](README_ru.md)

该策略寻找收盘价位于蜡烛区间顶部或底部20%的形态。若收盘价接近最高价且开盘价接近最低价，则产生看多信号；若开盘价接近最高价且收盘价接近最低价，则产生看空信号。该方法旨在捕捉价格从极端位置快速反转。

## 详情

- **入场条件**：
  - 收盘位于区间上部20%，开盘位于下部20% → 做多。
  - 开盘位于上部20%，收盘位于下部20% → 做空。
- **多空方向**：双向。
- **出场条件**：
  - 相反信号反向开仓。
- **止损**：无。
- **默认值**：
  - 区间百分比 = 0.2。
- **过滤器**：
  - 类别：形态识别
  - 方向：双向
  - 指标：无
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
