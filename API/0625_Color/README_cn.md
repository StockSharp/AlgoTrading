# 色彩策略
[English](README.md) | [Русский](README_ru.md)

根据设定颜色的亮度进行交易的示例策略。
如果颜色较亮（亮度 > 0.5）则做多，否则做空。

## 细节

- **入场条件**:
  - 多头：`颜色亮度 > 0.5`
  - 空头：`颜色亮度 <= 0.5`
- **多空方向**: 双向
- **出场条件**: 相反信号
- **止损**: 无
- **默认值**:
  - `ColorHex` = "#f23645"
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **过滤器**:
  - 分类: 其他
  - 方向: 双向
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 任意
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险: 低
