# Стратегия Color
[English](README.md) | [中文](README_cn.md)

Стратегия торгует в зависимости от яркости выбранного цвета.
Если цвет светлый (яркость > 0.5), открывает long, иначе открывает short.

## Детали

- **Условия входа**:
  - Long: `яркость цвета > 0.5`
  - Short: `яркость цвета <= 0.5`
- **Long/Short**: обе стороны
- **Условия выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `ColorHex` = "#f23645"
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: другое
  - Направление: обе
  - Индикаторы: нет
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Риск: низкий
