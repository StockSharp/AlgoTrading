# VR Steals 2戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MetaTrader 5のエキスパート「VR---STEALS-2」をStockSharpに変換したものです。単一のロングポジションを開き、インジケーターなしのシンプルなポジション管理を実演します。

## 動作方法
1. 開始時に`BuyMarket`を使って買い、約定価格を記録します。
2. ローソク足データ（デフォルトで1分）を`SubscribeCandles`で購読します。
3. 完成した各ローソク足ごとに：
   - 価格がトレードに有利な方向に`Breakeven`ステップ動いたら、ストップレベルをエントリーより`BreakevenOffset`ステップ上に移動します。
   - 価格がエントリー価格＋`TakeProfit`ステップに達したらポジションを決済します。
   - 価格がストップレベル（初期の`StopLoss`エントリー以下、または移動したブレークイーブンストップ）まで下落したらポジションを決済します。
4. 決済後、戦略は新しいポジションを開きません。

## パラメーター
| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| TakeProfit | テイクプロフィットレベルまでの価格ステップ距離。 | 50 |
| StopLoss | 初期ストップ距離（価格ステップ）。 | 50 |
| Breakeven | ブレークイーブンストップを起動するために必要な利益（ステップ）。 | 20 |
| BreakevenOffset | ブレークイーブンストップ設定時のエントリー上のオフセット。 | 9 |
| CandleType | 価格処理に使用するローソク足タイプ。 | 1分時間軸 |

`StartProtection()`は組み込みのポジション保護を有効にするために使用されます。
