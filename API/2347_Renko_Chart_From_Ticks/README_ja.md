# ティックからのRenkoチャート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ティックから直接Renkoブロックを生成し、ブロックの方向が変化したときにトレードします。StockSharpのハイレベルAPIを使って非時間ベースのローソク足を構築する方法を示します。

## 詳細

- **エントリー条件**:
  - 新しく完成したブロックが方向を反転したとき、新しいブロックの方向にエントリーします。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆方向のブロックが現れたとき、ポジションを反転します。
- **ストップ**: なし。
- **デフォルト値**:
  - `BrickSize` = 10
  - `Volume` = 1
- **フィルター**:
  - カテゴリ: Renko
  - 方向: 両方
  - インジケーター: なし
  - ストップ: なし
  - 複雑さ: 初心者
  - 時間軸: ティックベース
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
