# 高 IV 積極戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

高 IV 積極戦略は EMA クロスオーバーと ATR ボラティリティフィルターを組み合わせます。ボラティリティがその平均を 1 標準偏差上回る場合にのみ取引を開始し、ATR ベースのターゲットで決済します。

テストでは高ボラティリティ市場で堅実なリターンが示されています。

戦略はボラティリティが高まった際の EMA クロスでエントリーし、事前定義されたリスクコントロールのもとで素早い利益を狙います。

ポジションは ATR ベースのストップロスおよびテイクプロフィットレベルで決済されます。

## 詳細

- **エントリー条件**: 高速 EMA が低速 EMA をクロスし、ATR がその平均プラス標準偏差を上回る。
- **ロング/ショート**: 両方。
- **エグジット条件**: ATR ベースのストップロスまたはテイクプロフィット達成。
- **ストップ**: あり。
- **デフォルト値**:
  - `FastEmaLength` = 10
  - `SlowEmaLength` = 30
  - `AtrLength` = 14
  - `AtrMeanLength` = 20
  - `AtrStdLength` = 20
  - `RiskFactor` = 0.01m
  - `CandleType` = TimeSpan.FromMinutes(15)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: EMA, ATR
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
