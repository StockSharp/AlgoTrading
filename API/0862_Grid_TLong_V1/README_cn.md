# Grid TLong V1 策略
[English](README.md) | [Русский](README_ru.md)

基于网格的策略，始终保持持仓。当盈利或亏损达到设定百分比时重新开仓。

## 详情
- **入场条件**: 始终在市场中；按网格重新开仓。
- **多空方向**: 双向。
- **退出条件**: 反向信号或达到百分比后重新开仓。
- **止损**: 否。
- **默认值**:
  - `Percent` = 1
  - `UseLimitOrders` = false
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类型: 网格
  - 方向: 双向
  - 指标: 无
  - 止损: 否
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
