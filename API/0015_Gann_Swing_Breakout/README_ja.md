Gannスイングブレイクアウト手法に基づく戦略

テストでは年平均リターンが約82%であることが示されています。株式市場で最もよく機能します。

Gann Swing Breakoutはガン分析によるスイング高値と安値を追跡します。最新のスイングを超えるブレイクアウトがその方向への取引を開始し、反対のスイングが破られるまでポジションを維持します。

この手法は過去のスイングポイントを重要なサポートとレジスタンスと見なすトレーダー向けに設計されています。ブレイクで取引することにより、トレンドの次の局面に乗ることを試みます。
