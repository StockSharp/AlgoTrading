# LUBE戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は過去のローソク足をスキャンして現在の終値周辺の「摩擦」を測定します。FIRフィルターがトレンド方向を定義します。

- **ロング** 摩擦がトリガーレベルを下回り、トレンドが上向きの場合。
- **ショート** 摩擦がトリガーレベルを下回り、トレンドが下向きの場合。
- **エグジット** 摩擦が中間レベルを上回るか、反対のシグナルが現れた場合。

## 詳細
- **インジケーター**: カスタム摩擦計算、FIRフィルター。
- **時間軸**: デフォルトで30mローソク足。
- **両方向**: はい、ショートはオプション。
