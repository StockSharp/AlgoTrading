# Стратегия RSI с настраиваемым RSI и стоп-лоссом
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Покупает, когда значение RSI опускается ниже порога, и закрывает длинную позицию, когда цена пробивает максимум предыдущей свечи. Каждую сделку защищает стоп-лосс в процентах.

## Детали

- **Условия входа**:
  - Лонг: RSI ниже `RsiThreshold`
- **Лонг/Шорт**: Лонг
- **Условия выхода**:
  - Цена закрытия выше максимума предыдущей свечи
  - Срабатывание стоп-лосса
- **Стопы**: Да
- **Значения по умолчанию**:
  - `RsiLength` = 8
  - `RsiThreshold` = 28m
  - `StopLossPercent` = 5m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Лонг
  - Индикаторы: RSI
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
