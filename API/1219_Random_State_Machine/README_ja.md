# ランダム状態機械戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ランダム状態機械戦略は、移動平均でフィルタリングされたランダムな状態変化に基づいて取引を建てます。状態変化が起き、価格が移動平均より上にあるときロングポジションを建てます。価格が平均より下のときショートポジションを建てます。オプションのテイクプロフィット/ストップロス、時間制限による出口、移動平均クロスによる出口をサポートします。
