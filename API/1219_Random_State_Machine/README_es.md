# Estrategia de Máquina de Estados Aleatoria
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia de máquina de estados aleatoria abre operaciones en cambios de estado aleatorios filtrados por una media móvil. Las posiciones largas se abren cuando se produce un cambio de estado y el precio está por encima de la media móvil. Las posiciones cortas se abren cuando el precio está por debajo de la media. La estrategia admite take-profit/stop-loss opcionales, salidas temporizadas y salidas por cruce de media móvil.
