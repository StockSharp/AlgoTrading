# Zufällige Zustandsmaschinen-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die zufällige Zustandsmaschinen-Strategie eröffnet Trades bei zufälligen Zustandswechseln, die durch einen gleitenden Durchschnitt gefiltert werden. Long-Positionen werden eröffnet, wenn ein Zustandswechsel eintritt und der Preis über dem gleitenden Durchschnitt liegt. Short-Positionen werden eröffnet, wenn der Preis unter dem Durchschnitt liegt. Die Strategie unterstützt optionale Take-Profit/Stop-Loss-Aufträge, zeitbasierte Ausstiege und Ausstiege bei Kreuzung des gleitenden Durchschnitts.
