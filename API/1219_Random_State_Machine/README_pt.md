# Estratégia de Máquina de Estados Aleatória
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia de máquina de estados aleatória abre operações em mudanças de estado aleatórias filtradas por uma média móvel. Posições compradas são abertas quando ocorre uma mudança de estado e o preço está acima da média móvel. Posições vendidas são abertas quando o preço está abaixo da média. A estratégia suporta take-profit/stop-loss opcionais, saídas temporizadas e saídas por cruzamento de média móvel.
