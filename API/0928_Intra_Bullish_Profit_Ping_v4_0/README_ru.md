# Стратегия Intra Bullish Strategy - Profit Ping v4.0
[English](README.md) | [中文](README_cn.md)

Лонговая система, входящая при пересечении EMA вверх с подтверждением по MACD и RSI.

## Детали

- **Условия входа**:
  - Короткая EMA пересекает длинную вверх
  - Гистограмма MACD > 0
  - RSI > 50
  - Цена закрытия выше цены открытия
- **Условия выхода**:
  - Короткая EMA пересекает длинную вниз
  - Гистограмма MACD < 0
  - RSI < 50
  - Цена закрытия ниже цены открытия
- **Индикаторы**:
  - Экспоненциальные скользящие средние
  - MACD
  - RSI
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `ShortEmaLength` = 7
  - `LongEmaLength` = 14
  - `RsiLength` = 14
  - `MacdFastPeriod` = 12
  - `MacdSlowPeriod` = 26
  - `MacdSignalPeriod` = 9
- **Фильтры**:
  - Трендовая
  - Один таймфрейм
  - Индикаторы: EMA, MACD, RSI
  - Стопы: нет
  - Сложность: низкая
