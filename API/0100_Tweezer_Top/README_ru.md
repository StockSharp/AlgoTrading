# Стратегия Tweezer Top

Фигура Tweezer Top является зеркальным отражением Tweezer Bottom и возникает после роста.
Две свечи имеют почти одинаковые максимумы, показывая, что покупатели не смогли преодолеть определённый уровень.

Стратегия открывает короткую позицию, когда вторая свеча подтверждает уровень сопротивления, ожидая откат при ослаблении бычьего импульса.

Короткий стоп над двойной вершиной ограничивает риск, и сделка закрывается, если цена поднимается выше этого сопротивления.

## Детали

- **Критерий входа**: совпадение паттерна
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
