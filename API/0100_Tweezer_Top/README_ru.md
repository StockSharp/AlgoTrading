# Стратегия «Пинцетный верх»
[English](README.md) | [中文](README_zh.md)

Пинцетный верх является зеркальной разновидностью нижнего варианта и появляется после подъёма. Две свечи имеют почти одинаковые максимумы, показывая, что покупатели не смогли пробиться выше определённого уровня.

Тестирование показывает среднегодичную доходность около 187%. Стратегию лучше запускать на фондовом рынке.

Стратегия открывает короткую позицию после того, как вторая свеча подтверждает потолок, ожидая отката при остановке бычьего импульса.

Плотный стоп выше двойного максимума ограничивает риск, а сделка закрывается, если цена снова поднимается выше этого сопротивления.

## Детали

- **Условия входа**: совпадение паттерна
- **Лонг/шорт**: оба направления
- **Условия выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, на процентной основе
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
