# Tweezer Top Strategy
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ツイーザートップはボトム版の鏡像ですが、上昇後に現れます。2本のローソク足がほぼ同じ高値を共有しており、買い手が一定のレベルを超えることができなかったことを示しています。

テストでは平均年間リターンが約187%であることが示されています。株式市場で最もパフォーマンスが優れています。

戦略は2本目のローソク足が天井を確認したらショートポジションをオープンし、強気モメンタムが失速するにつれて押し戻しを期待します。

ツイン高値の上に置かれたタイトなストップがリスクを管理し、価格がその抵抗線を再び上回った場合に取引が終了します。

- **エントリー条件**: パターン一致
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース

  - `CandleType` = 15分
  - `StopLoss` = 2%
