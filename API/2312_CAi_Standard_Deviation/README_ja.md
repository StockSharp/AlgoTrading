# CAi 標準偏差戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、オリジナルのMQL5エキスパート**Exp_i-CAi_StDev**のStockSharpへの移植です。移動平均と標準偏差バンドを組み合わせて、ブレイクアウトとその後の反転を検出します。

## 戦略ロジック

1. 指定した期間の単純移動平均（SMA）を計算する。
2. 同期間の終値の標準偏差を計算する。
3. SMA周囲に2組のバンドを構築する：
   - **エントリーバンド**: SMA ± `OpenMultiplier` × StdDev。
   - **エグジットバンド**: SMA ± `CloseMultiplier` × StdDev。
4. 価格が上部エントリーバンドを超えて終値を付けたとき、ロングポジションを開く。
5. 価格が下部エントリーバンドを下回って終値を付けたとき、ショートポジションを開く。
6. 価格が上部エグジットバンドを下回ったとき、既存のロングポジションを決済する。
7. 価格が下部エグジットバンドを上回ったとき、既存のショートポジションを決済する。

## パラメーター

| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| `MaLength` | 移動平均と標準偏差計算の長さ | 12 |
| `StdDevPeriod` | 標準偏差インジケーターの期間 | 9 |
| `OpenMultiplier` | エントリーバンドの乗数 | 2.5 |
| `CloseMultiplier` | エグジットバンドの乗数 | 1.5 |
| `CandleType` | 戦略で使用するローソク足の種類 | 5分足 |

## 注意事項

- 戦略はインジケーター値を受け取るために`Bind`を使用した高レベルAPIを使います。
- 早期シグナルを避けるため、完成したローソク足のみが処理されます。
- ソースコード内のすべてのコメントは英語で記述されています。
