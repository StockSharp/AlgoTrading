# Bollinger Bands + RSI-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie eröffnet Long-Positionen, wenn der Preis unterhalb des unteren Bollinger Bands schließt und der RSI unter dem überverkauften Niveau liegt. Die Position wird geschlossen, wenn der Preis über das obere Band steigt und der RSI über dem überkauften Niveau liegt, oder wenn der RSI das Ausstiegsniveau kreuzt.
