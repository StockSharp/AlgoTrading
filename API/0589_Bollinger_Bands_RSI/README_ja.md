# Bollinger Bands + RSI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格がボリンジャーバンド下限を下回って終値をつけ、RSIが売られすぎレベルを下回ったときにロングポジションを取ります。価格が上限バンドを上回り、RSIが買われすぎレベルを上回ったとき、またはRSIが出口レベルを超えたときにポジションを決済します。
