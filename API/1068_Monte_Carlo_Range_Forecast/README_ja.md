# Monte Carlo レンジ予測戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Monte Carlo レンジ予測は、ATRベースのボラティリティを用いたMonte Carloシミュレーションにより将来の価格レンジを予測します。シミュレーションの平均価格が現在の価格を上回る場合にロングを、下回る場合にショートをエントリーします。

## 詳細
- **データ**: ATR付き価格ローソク足。
- **エントリー条件**:
  - **ロング**: シミュレーションから期待される価格が現在の価格を上回る。
  - **ショート**: シミュレーションから期待される価格が現在の価格を下回る。
- **エグジット条件**: 逆シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `ForecastPeriod` = 20
  - `Simulations` = 100
- **フィルター**:
  - カテゴリ: 統計
  - 方向: ロング＆ショート
  - インジケーター: ATR
  - 複雑さ: 中程度
  - リスクレベル: 中
