# Monte Carlo Range Forecast
[English](README.md) | [中文](README_cn.md)

Monte Carlo Range Forecast использует моделирование Монте-Карло и волатильность на основе ATR для прогнозирования будущего диапазона цены. Стратегия открывает длинную позицию, когда средняя имитация выше текущей цены, и короткую, когда ниже.

## Детали
- **Данные**: Свечи цен и ATR.
- **Условия входа**:
  - **Лонг**: Ожидаемая цена из симуляций выше текущей.
  - **Шорт**: Ожидаемая цена из симуляций ниже текущей.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `ForecastPeriod` = 20
  - `Simulations` = 100
- **Фильтры**:
  - Категория: Статистическая
  - Направление: Лонг и Шорт
  - Индикаторы: ATR
  - Сложность: Средняя
  - Уровень риска: Средний
