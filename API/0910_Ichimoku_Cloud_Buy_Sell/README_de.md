# Ichimoku Cloud Kauf & Verkauf
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet die Ichimoku Cloud mit EMA- und Volumenfiltern. Sie kauft, wenn der Preis über der Wolke liegt und das Volumen stark ist, und verkauft, wenn der Preis unter der Wolke liegt. Positionen werden geschlossen, wenn der Preis den EMA in der entgegengesetzten Richtung kreuzt.
