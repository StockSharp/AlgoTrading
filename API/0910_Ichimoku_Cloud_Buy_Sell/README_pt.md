# Ichimoku Cloud Compra e Venda
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia utiliza a Nuvem Ichimoku com filtros de EMA e volume. Compra quando o preço está acima da nuvem com volume forte e vende quando o preço está abaixo da nuvem. As posições são encerradas quando o preço cruza a EMA na direção oposta.
