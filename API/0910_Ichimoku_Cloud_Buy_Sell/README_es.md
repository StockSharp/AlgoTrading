# Ichimoku Cloud Compra y Venta
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza la Nube Ichimoku con filtros de EMA y volumen. Compra cuando el precio está por encima de la nube con volumen fuerte y vende cuando el precio está por debajo de la nube. Las posiciones se cierran cuando el precio cruza la EMA en la dirección opuesta.
