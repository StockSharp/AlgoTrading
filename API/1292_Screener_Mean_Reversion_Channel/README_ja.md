# スクリーナー平均回帰チャネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は移動平均と ATR から構築した平均回帰チャネルで取引します。価格が上部バンドを上回って終値をつけた場合にショートし、下部バンドを下回って終値をつけた場合にロングします。価格が中心線に戻ったときにポジションをクローズします。

## 詳細
- エントリー: 上部バンドを上回る終値 -> ショート、下部バンドを下回る終値 -> ロング。
- エグジット: 価格が中心線まで戻るクロス。
- インジケーター: SMA と ATR。
- ストップ: なし。
