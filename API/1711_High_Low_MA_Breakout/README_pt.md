# Rompimento de MA de Máximas e Mínimas
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia de ação do preço que entra comprado quando o preço fecha acima de uma média móvel das máximas e entra vendido quando o preço fecha abaixo de uma média móvel das mínimas. As posições são gerenciadas com níveis opcionais de take profit e stop loss.
