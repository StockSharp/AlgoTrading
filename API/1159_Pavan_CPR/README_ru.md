# Стратегия Pavan CPR
[English](README.md) | [中文](README_cn.md)

Стратегия покупает, когда цена пересекает верхний уровень центрального пивота после закрытия ниже него. Стоп размещается на уровне пивота, тейк-профит — на фиксированном расстоянии.

## Детали

- **Условия входа**: предыдущее закрытие ниже верхнего CPR и текущее закрытие выше него.
- **Длинные/Короткие**: только лонг.
- **Условия выхода**: тейк-профит или стоп на пивоте.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `TakeProfitTarget` = 50
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Long
  - Индикаторы: Pivot
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
