# Стратегия Turtle Trader
[English](README.md) | [中文](README_cn.md)

Turtle Trader реализует классическую систему прорыва Turtle с каналами Дончиана и управлением капиталом на основе ATR. Покупает при пробое недавнего максимума и продаёт при пробое минимума. Пирамидинг добавляет к выигрышным позициям при движении цены в нужную сторону.

## Детали

- **Критерий входа**: пробой максимумов/минимумов по периодам `S1` или `S2`
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный пробой или ATR-стоп
- **Стопы**: по ATR
- **Значения по умолчанию**:
  - `RiskPercent` = 1
  - `AtrPeriod` = 20
  - `StopMultiplier` = 1.5
  - `PyramidProfit` = 0.5
  - `S1Long` = 20
  - `S2Long` = 55
  - `S1LongExit` = 10
  - `S2LongExit` = 20
  - `S1Short` = 15
  - `S2Short` = 55
  - `S1ShortExit` = 7
  - `S2ShortExit` = 20
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: обе стороны
  - Индикаторы: ATR, Highest, Lowest
  - Стопы: ATR
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
