# RPM5 ブルズベアーズアイズ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
**RPM5 BullsBearsEyes 戦略** は、MetaTrader 4 エキスパート *Rpm5_mt4v1* の C# ポートです。アドバイザーは、ブルズパワーとベアズパワーの測定値からカスタムのブルズベアーズアイズオシレーターを再構築し、一般的なバイアスに従う単一のポジションをオープンしました。この StockSharp バージョンは、元のリスク パラメーター、トレーリング ロジック、信号しきい値を維持しながら、高レベルの API を使用して同じ動作を再現します。

## 指標の再構築
- 2 つの古典的なオシレーター – **強気パワー** と **弱気パワー** – は、設定されたローソク足シリーズに基づいて計算されます。
- それらの合計は、MT4 インジケーターで使用される同一の 4 段階 IIR スムーザーを通過します。平滑化係数 (`Gamma`) は、オシレーターの反応速度を制御します。
- フィルター処理された出力は、**0** から **1** までの値に変換されます。中央のしきい値を超える値は強気の支配を示し、それを下回る値は弱気のコントロールを示します。どちらかの側が完全に使い果たされた場合、正確な 0 または 1 が表示され、元のインジケーターのエッジ ケースと一致します。

## 取引ルール
1. この戦略は、選択した時間枠 (デフォルトでは 5 分) をサブスクライブし、完了したローソク足のみを待ちます。
2. フラットの場合、BullsBearsEyes 比率を評価します。
   - **長いエントリ** – 現在の値は厳密に `Threshold` (デフォルトは 0.5) を上回ります。
   - **簡単なエントリ** – 現在の値は厳密に `Threshold` を下回ります。
   - アルゴリズムは最大でも 1 つのオープンポジションを保持します。反対のシグナルは、リスク管理によってアクティブなポジションが完全にクローズされるまで無視されます。
3. 取引が開始されると、ポジションはストップロス、テイクプロフィット、またはトレーリングストップイベントが発生するまで変更されません。

## リスク管理
- **ストップロス / テイクプロフィット** 距離は、元の 25 / 150 ピップ設定から再作成されます。新しいポジションがオープンされるたびに、商品 `PriceStep` (pip) を使用して再計算されます。
- **ATR トレーリング**: 終了したローソク足ごとに、平均トゥルー レンジ (期間 `AtrPeriod`、デフォルトは 5) が評価されます。トレーリング距離は 1 ピップに `AtrMultiplier × ATR` を加えたものに等しくなります。クローズがその距離を超えて進むと、ギャップを維持するために保護停止が強化されます。これは、`OrderModify` を繰り返し呼び出す MQL ロジックと同じです。
- 新しい信号を処理する前に保護レベルがチェックされ、ソース EA と同様に、出口が常に新しいエントリよりも優先されることが保証されます。

## パラメーター
| 名前 | デフォルト | 説明 |
| --- | --- | --- |
| `Bulls/Bears Period` | 13 | ブルズパワーインジケーターとベアーズパワーインジケーターの平均期間。 |
| `Gamma` | 0.5 | BullsBearsEyes オシレーターの 4 段階 IIR スムージング比。 |
| `Threshold` | 0.5 | 強気 (> しきい値) ゾーンと弱気 (< しきい値) ゾーンの境界線。 |
| `ATR Period` | 5 | ATR ベースのトレーリング ストップに使用されるルックバック。 |
| `ATR Multiplier` | 1.5 | 後続距離を求めるときに ATR に適用される乗数。 |
| `Stop Loss (pips)` | 25 | プロテクティブストップ距離。ピップから価格に変換されます。 |
| `Take Profit (pips)` | 150 | 利益目標距離。ピップから価格に変換されます。 |
| `Trade Volume` | 1 | 新しいポジションごとに使用される成行注文の量。 |
| `Candle Type` | 5分キャンドル | 戦略によって処理されるタイムフレーム。 |
