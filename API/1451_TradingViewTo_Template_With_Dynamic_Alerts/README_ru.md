# TradingViewTo Strategy Template With Dynamic Alerts
[English](README.md) | [中文](README_cn.md)

Шаблонная стратегия, открывающая позиции по уровням RSI и управляющая ими с помощью процентного стоп-лосса и тейк-профита.

## Детали
- **Критерии входа**:
  - **Лонг**: RSI > `UpperLevel`
  - **Шорт**: RSI < `LowerLevel`
- **Длинные/короткие**: обе стороны
- **Критерии выхода**:
  - Стоп-лосс или тейк-профит
- **Стопы**: процентный стоп-лосс и тейк-профит
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `UpperLevel` = 60
  - `LowerLevel` = 40
  - `StopLossPct` = 2m
  - `TakeProfitPct` = 4m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Моментум
  - Направление: обе стороны
  - Индикаторы: RSI
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
