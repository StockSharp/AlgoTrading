# Very Blonde System 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MetaTrader向けオリジナル「Very Blonde System」に着想を得たグリッドベースの逆張り戦略。現在価格と直近の極値との大きな乖離を探し、反対方向でトレードします。

## 戦略ロジック
1. 直近 *Count Bars* 本のローソク足における最高値と最安値を計算する。
2. オープンポジションがない場合：
   - 直近の高値から現在価格までの距離が *Limit* ティックを超えた場合、成行で買い。
   - 現在価格から直近の安値までの距離が *Limit* ティックを超えた場合、成行で売り。
   - ポジション取得後、*Grid* ティックごとに4つの追加指値注文を置き、各レベルでボリュームを倍増する。
3. ポジションが存在する場合：
   - 合計利益が *Amount* 通貨単位を超えた場合、ポジションをクローズしてすべての未決注文をキャンセルする。
   - *Lock Down* がゼロより大きい場合、価格が有利な方向にその分のティック動いた時点でブレークイーブン保護が有効になる。価格がエントリーレベルに戻ると、すべてのポジションがクローズされる。

## パラメーター
| 名前 | 説明 |
|------|------|
| `CountBars` | 高値・安値を探すローソク足の本数。 |
| `Limit` | トレードを開くための極値からの最小距離（ティック）。 |
| `Grid` | 追加グリッド注文間の距離（ティック）。 |
| `Amount` | すべてのポジションをクローズするための目標利益（通貨）。 |
| `LockDown` | ブレークイーブン保護を有効にする距離（ティック）。 |
| `CandleType` | 計算に使用するローソク足タイプ。 |

戦略は最初のエントリーに成行注文を使用し、グリッドレベルには指値注文を使用します。コード内のすべてのコメントは英語で書かれています。
