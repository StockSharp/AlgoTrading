# VQZL Z-Score
[English](README.md) | [中文](README_cn.md)

Стратегия использует z-оценку относительно сглаженного среднего.

Тестирование показывает среднегодовую доходность около 42%. Лучше всего работает на рынке акций.

Стратегия рассчитывает сглаженную скользящую среднюю и стандартное отклонение для получения z-оценки. Когда цена выходит за порог, позиция открывается в сторону движения.

## Детали

- **Условия входа**:
  - **Лонг**: `Z-Score > threshold`.
  - **Шорт**: `Z-Score < -threshold`.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `PriceSmoothing` = 15
  - `ZLength` = 100
  - `Threshold` = 1.64
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
