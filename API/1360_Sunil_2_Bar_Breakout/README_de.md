# Sunil 2-Balken-Ausbruch-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie platziert Stop-Orders basierend auf einem Zwei-Balken-Ausbruchmuster. Ein Long-Stop wird gesetzt, wenn der letzte Schlusskurs über dem vorherigen Schlusskurs liegt und das vorherige Hoch die Hochs der zwei Balken davor übersteigt. Ein Short-Stop wird gesetzt, wenn der letzte Schlusskurs unter dem vorherigen Schlusskurs liegt und das vorherige Tief niedriger ist als die Tiefs der zwei Balken davor. Ausstiege werden durch Stop-Loss-Niveaus am entgegengesetzten Ende der Signalkerze ausgelöst.
