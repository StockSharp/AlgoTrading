# Estrategia de Ruptura de 2 Barras de Sunil
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia coloca órdenes stop basadas en un patrón de ruptura de dos barras. Se coloca un stop largo cuando el último cierre está por encima del cierre anterior y el máximo previo supera los máximos de las dos barras anteriores. Se coloca un stop corto cuando el último cierre está por debajo del cierre anterior y el mínimo previo es inferior a los mínimos de las dos barras anteriores. Las salidas se activan por niveles de stop-loss en el extremo opuesto de la barra de señal.
