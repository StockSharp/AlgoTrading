# Estratégia de Rompimento de 2 Barras de Sunil
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia coloca ordens stop baseadas em um padrão de rompimento de duas barras. Um stop comprado é colocado quando o último fechamento está acima do fechamento anterior e a máxima anterior supera as máximas das duas barras anteriores. Um stop vendido é colocado quando o último fechamento está abaixo do fechamento anterior e a mínima anterior é inferior às mínimas das duas barras anteriores. As saídas são acionadas por níveis de stop-loss na extremidade oposta da barra de sinal.
