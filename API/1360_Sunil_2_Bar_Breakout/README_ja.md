# Sunil 2バー・ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は2バーのブレイクアウトパターンに基づいてストップ注文を発注します。直近の終値が前の終値を上回り、直前の高値がその2本前の高値を超えたとき、ロングのストップ注文が発注されます。直近の終値が前の終値を下回り、直前の安値がその2本前の安値を下回ったとき、ショートのストップ注文が発注されます。決済はシグナルバーの反対側のストップロス水準によって行われます。
