# Forex通貨ペア利回りモメンタム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、選択したFX通貨ペアの2年利回りスプレッドのモメンタムを使って取引します。モメンタムはスプレッドとその移動平均の差として測定されます。モメンタムに適用したボリンジャーバンドが買われ過ぎ・売られ過ぎゾーンを定義します。ポジションは固定本数のバー後に決済されます。

## 主な特徴

- 2年利回りスプレッドのモメンタムをシグナルに使用。
- モメンタムのボリンジャーバンドが極端な状態を識別。
- エントリーロジックのオプション反転。
- 指定したバー数後にポジションを決済。

## パラメーター

| 名前 | 説明 |
|------|------|
| `YieldASecurity` | 最初の利回り証券。 |
| `YieldBSecurity` | 2番目の利回り証券。 |
| `CandleType` | 分析用ローソク足の時間軸。 |
| `MomentumLength` | 利回りスプレッド平均の期間。 |
| `BollingerLength` | ボリンジャーバンドの期間。 |
| `BollingerStdDev` | バンドの標準偏差乗数。 |
| `HoldPeriods` | ポジションを保持するバー数。 |
| `ReverseLogic` | ロングとショートの条件を反転。 |
