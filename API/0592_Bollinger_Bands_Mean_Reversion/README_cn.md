# Bollinger Bands Mean Reversion 策略
[English](README.md) | [Русский](README_ru.md)

该策略在价格收盘低于布林带下轨时买入，在收盘高于上轨时退出多头。

## 详情

- **入场条件**：
  - **多头**：收盘价低于下轨。
- **多/空**：仅多头。
- **出场条件**：收盘价高于上轨。
- **止损**：无。
- **默认值**：
  - 布林带长度 20。
  - 系数 2。
- **过滤**：
  - 类别：均值回归
  - 方向：多头
  - 指标：布林带
  - 止损：无
  - 复杂度：简单
  - 时间框架：短期
