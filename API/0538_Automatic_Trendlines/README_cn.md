# Automatic Trendlines 策略
[English](README.md) | [Русский](README_ru.md)

通过连接最近的枢轴高点和枢轴低点来构建动态的支撑和阻力趋势线。当价格收于阻力线之上时发出做多信号，当价格跌破支撑线时发出做空信号。

## 细节

- **入场条件**:
  - **多头**: 收盘价上穿阻力趋势线。
  - **空头**: 收盘价下破支撑趋势线。
- **出场条件**:
  - 反向信号或仓位反转。
- **指标**:
  - 基于枢轴点的趋势线。
- **止损**: 无。
- **默认值**:
  - `LeftBars` = 100
  - `RightBars` = 15
- **过滤器**:
  - 趋势跟随
  - 单一时间框架
  - 指标: 枢轴趋势线
  - 止损: 无
  - 复杂度: 低
