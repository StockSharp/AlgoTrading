# 自動トレンドライン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

直近のピボット高値と安値を結ぶことで動的なサポートとレジスタンスのトレンドラインを構築します。価格がレジスタンスラインを上回って終値をつけたときにロングシグナルが発生し、価格がサポートラインを下回ったときにショートシグナルが発火します。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値がレジスタンストレンドラインを上抜け。
  - **ショート**: 終値がサポートトレンドラインを下抜け。
- **エグジット条件**:
  - 反対シグナルまたはポジション転換。
- **インジケーター**:
  - ピボットベースのトレンドライン。
- **ストップ**: なし。
- **デフォルト値**:
  - `LeftBars` = 100
  - `RightBars` = 15
- **フィルター**:
  - トレンドフォロー
  - 単一時間軸
  - インジケーター: ピボットトレンドライン
  - ストップ: なし
  - 複雑さ: 低
