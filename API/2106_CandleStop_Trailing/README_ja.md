# CandleStopトレーリング戦略

## 概要
この戦略はCandleStopアプローチに基づくトレーリングストップ管理を実装します。完成したローソク足を分析し、ストップレベルを取引の方向にのみ移動させます。アルゴリズムはロングとショートポジションに対して別々のルックバック期間を持つドンチャンチャネルに依存しており、手動取引や他のエントリー戦略への接続に適しています。

## パラメーター
- **Up Trail Periods** – ショートポジションのトレーリング用に最高高値を計算するために使用するローソク足の数。
- **Down Trail Periods** – ロングポジションのトレーリング用に最低安値を計算するために使用するローソク足の数。
- **Candle Type** – 分析に使用するローソク足の時間軸。

## 戦略ロジック
1. 既存のポジションを待ちます。この戦略は自らは取引を開きません。
2. ロングポジションの場合:
   - *Down Trail Periods*にわたって最低安値を計算します。
   - この値が前のストップより高い場合、ストップをこの値に移動します。
   - 価格がストップに触れるか下回った場合、成行注文でポジションを終了します。
3. ショートポジションの場合:
   - *Up Trail Periods*にわたって最高高値を計算します。
   - この値が前のストップより低い場合、ストップをこの値に移動します。
   - 価格がストップに触れるか上回った場合、成行注文でポジションを買い戻します。

## 使用上の注意
- StockSharpの高レベルAPIとローソク足購読で使用するように設計されています。
- 手動または他の戦略によって開かれたポジションの保護に適しています。
- チャート出力には視覚化のためのローソク足、チャンネルライン、実行済み取引が含まれます。
