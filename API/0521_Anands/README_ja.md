# Anands戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このブレイクアウトシステムは、前日のローソク足を使って取引方向を設定します。
前日の終値がその日の高値を上回っていればロングを探し、安値を下回っていれば弱気になります。
15分の時間軸では、直近の2本の完成ローソク足を監視します。
前のローソク足が2本前の高値を上回って終値を付けたときにロングポジションを開きます。
前の終値が2本前の安値を下回ったときにショートポジションを開きます。

## 詳細

- **エントリー条件**:
  - 前日の終値がその日のレンジの上/下にあることで強気/弱気バイアスを設定。
  - **ロング**: 直前の15m終値 > 2本前の高値。
  - **ショート**: 直前の15m終値 < 2本前の安値。
- **ロング/ショート**: 両サイド。
- **エグジット条件**: 定義なし、反対シグナルで決済。
- **ストップ**: ブレイクアウトバーの反対側に推奨。
- **デフォルト値**:
  - `CandleType` = 15分
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: ローソク足
  - ストップ: オプション
  - 複雑さ: 低
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
