# Стратегия Anands
[English](README.md) | [中文](README_cn.md)

Стратегия определяет направление сделки по свечам дневного таймфрейма.
Если закрытие прошлого дня выше его максимума — рассматриваются покупки,
если ниже минимума — продажи.
На таймфрейме 15 минут анализируются две завершённые свечи.
Лонг открывается, когда предыдущая свеча закрывается выше максимума двух свечей назад.
Шорт открывается, когда предыдущее закрытие ниже минимума двух свечей назад.

## Подробности

- **Условия входа**:
  - Закрытие предыдущего дня вне его диапазона задаёт бычий/медвежий уклон.
  - **Лонг**: закрытие 15‑мин свечи > максимуму двух свечей назад.
  - **Шорт**: закрытие 15‑мин свечи < минимуму двух свечей назад.
- **Направление**: обе стороны.
- **Условия выхода**: нет, противоположный сигнал закрывает позицию.
- **Стопы**: рекомендуется по противоположной стороне пробойной свечи.
- **Параметры по умолчанию**:
  - `CandleType` = 15 минут
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе
  - Индикаторы: Свечи
  - Стопы: Опционально
  - Сложность: Низкая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
