# Bitcoin レバレッジセンチメント戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はBitcoinのロングとショートポジションの比率のZ-Scoreを分析します。Z-Scoreが設定可能な閾値を上抜けたときにロングトレードを開始し、ロング終了レベルを下抜けたときにクローズします。ショートトレードは逆の閾値を使用します。取引方向はロング、ショート、または両方に制限できます。

## 詳細

- **エントリー条件**:
  - Z-Scoreがロングエントリー閾値を上抜け → ロング。
  - Z-Scoreがショートエントリー閾値を下抜け → ショート。
- **ロング/ショート**: 設定可能
- **エグジット条件**:
  - Z-Scoreがロング終了閾値を下抜け。
  - Z-Scoreがショート終了閾値を上抜け。
- **ストップ**: なし
- **デフォルト値**:
  - Z-Score長 = 252
  - ロングエントリー = 1.0
  - ロングエグジット = -1.618
  - ショートエントリー = -1.618
  - ショートエグジット = 1.0
  - ローソク足タイプ = 1日
- **フィルター**:
  - カテゴリ: Sentiment
  - 方向: 両方
  - インジケーター: SMA, StdDev
  - ストップ: なし
  - 複雑さ: 低
  - 時間軸: 長期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
