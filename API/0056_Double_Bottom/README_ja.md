# ダブルボトム・パターン (Double Bottom Pattern)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このパターンベースの戦略は、設定された間隔で隔たれた概ね同じ価格の2つの連続した安値を探します。2つ目の底が形成された後、陽線がリバーサルを確認します。

テストでは年平均リターンが約55%となっています。株式市場での運用に最も適しています。

確認が取れると、システムはパターンの安値より下にストップを置いて買いエントリーします。このセットアップは売り圧力が枯渇した後の急激な反発を捉えることを目的としています。

エグジットは事前に定義されたストップロスまたは手動の利益目標に依存します。

## 詳細

- **エントリー条件**: `Distance` バー後、`SimilarityPercent` 以内に2つの底が形成される。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 価格が失敗するかストップロス。
- **ストップ**: はい。
- **デフォルト値**:
  - `Distance` = 5
  - `SimilarityPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(15)
  - `StopLossPercent` = 1.0m
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ロングのみ
  - インジケーター: Price Action
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
