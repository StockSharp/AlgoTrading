# X-Trail戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、中間価格で計算された速い単純移動平均と遅い単純移動平均が交差したときにトレードを生成します。ロジックは、このようなクロスでアラートを使用したオリジナルのMQLスクリプト **X_trail.mq4** を反映しています。

現在と1本前のバーで速いMAが遅いMAを上回り、2本前のバーでは下回っていた場合にロングポジションが建てられます。逆のパターンがショートポジションをトリガーします。新しいシグナルごとにポジションが反転されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 直近2本の確定ローソク足で速いMA > 遅いMA、かつ2本前は速いMAが遅いMAを下回っていた。
  - **ショート**: 直近2本の確定ローソク足で速いMA < 遅いMA、かつ2本前は速いMAが遅いMAを上回っていた。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆方向のクロス（ポジション反転）。
- **ストップ**: なし。
- **インジケーター**:
  - 中間価格で計算された2本の単純移動平均。
