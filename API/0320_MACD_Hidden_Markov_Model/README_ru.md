# MACD и скрытая модель Маркова

Стратегия **MACD Hidden Markov Model** построена на индикаторе MACD с использованием скрытой модели Маркова.

Сигналы формируются, когда модель Маркова подтверждает изменение тренда на внутридневных данных (5м). Такой метод подходит активным трейдерам.

Стопы рассчитываются на основе кратных ATR и параметров MacdFast, MacdSlow. Настройте эти значения для баланса риска и прибыли.

## Подробности
- **Критерии входа**: см. реализацию условий индикаторов.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: противоположный сигнал или логика стопов.
- **Стопы**: да, расчёт на основе индикаторов.
- **Значения по умолчанию**:
  - `MacdFast = 12`
  - `MacdSlow = 26`
  - `MacdSignal = 9`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
  - `HmmHistoryLength = 100`
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Оба
  - Индикаторы: Markov
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейронные сети: Да
  - Дивергенция: Нет
  - Уровень риска: Средний
