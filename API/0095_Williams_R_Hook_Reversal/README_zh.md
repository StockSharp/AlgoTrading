# 威廉%R 钩形反转策略
[English](README.md) | [Русский](README_ru.md)

威廉%R 钩形反转策略关注该指标从极端值快速回钩的行为。当读数高于 -20 或低于 -80 后向中线回钩时，之前的推升往往已衰竭。

价格刷新低点而 %R 从超卖区向上回钩时做多；在价格创新高而 %R 从超买区向下钩时做空。

采用紧凑的百分比止损控制风险，当 %R 朝相反方向钩回或止损触发时退出。

## 细节

- **入场条件**：指标信号。
- **多/空**：双向。
- **退出条件**：止损或相反信号。
- **止损**：是，按百分比。
- **默认值**：
  - `CandleType` = 15 分钟
  - `StopLoss` = 2%
- **过滤条件**：
  - 类别: 反转
  - 方向: 双向
  - 指标: 威廉%R
  - 止损: 有
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等

测试表明年均收益约为 172%，该策略在外汇市场表现最佳。
