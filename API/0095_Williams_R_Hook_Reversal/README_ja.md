# Williams %R Hook Reversal 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Williams %R Hook Reversal戦略は、Williams %Rインジケーターが極端な水準から素早く戻るときの動きを追います。値が-20を上回るか-80を下回り、その後中央に向けてフックすると、前の推進力はおそらく枯渇しています。

テストでは平均年間リターンが約172%であることが示されています。外国為替市場で最もパフォーマンスが優れています。

%Rが売られすぎから上向きに反転し、価格が新安値を押しているときにロングします。%Rが買われすぎから下向きにフックし、新高値が形成されているときにショートします。

タイトなパーセントストップがリスクをコントロールし、%Rが反対方向にフックするかストップが発動したときに取引が終了します。
