# 供給・需要・オーダーブロックによるブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Donchianのサポート・レジスタンスレベルを使ったブレイクアウト戦略で、EMAトレンドフィルターと出来高急増の確認を組み合わせます。ポジションはストップロスとトレーリングストップで保護されます。

## 詳細

- **エントリー条件**: トレンド・出来高フィルターを伴うDonchianチャネルのブレイクアウト。
- **ロング/ショート**: 両方向。
- **エグジット条件**: ストップロスまたはトレーリングストップ。
- **ストップ**: はい、固定とトレーリング。
- **デフォルト値**:
  - `Length` = 20
  - `StopLossTicks` = 1000
  - `TrailingStartTicks` = 2000
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Donchian, EMA, SMA
  - ストップ: 固定 & トレーリング
  - 複雑さ: 中級
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
