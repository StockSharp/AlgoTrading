# Стратегия Color Zerolag JCCX
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторе ColorZerolagJCCX из MetaTrader. В реализации используется приближение двумя простыми скользящими средними.
Длинная позиция открывается при пересечении быстрой средней ниже медленной, короткая — при пересечении быстрой средней выше медленной.

## Подробности

- **Условия входа**:
  - Long: `быстрая SMA пересекает медленную сверху вниз`
  - Short: `быстрая SMA пересекает медленную снизу вверх`
- **Long/Short**: обе стороны
- **Условия выхода**: противоположный сигнал
- **Стопы**: `StartProtection()`
- **Значения по умолчанию**:
  - `FastPeriod` = 8
  - `SlowPeriod` = 21
  - `CandleType` = 4-часовые свечи
- **Фильтры**:
  - Категория: Trend following
  - Направление: Оба
  - Индикаторы: Скользящая средняя
  - Стопы: Опционально
  - Сложность: Базовая
  - Таймфрейм: Свинг
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
