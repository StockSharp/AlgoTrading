# Стратегия Adaptive Oscillator Threshold
[English](README.md) | [中文](README_cn.md)

Adaptive Oscillator Threshold использует RSI с динамическим порогом на основе Bufi's Adaptive Threshold (BAT). Покупает, когда RSI опускается ниже фиксированного уровня или адаптивного порога.

## Детали

- **Критерий входа**: RSI падает ниже фиксированного или адаптивного порога
- **Длин/Шорт**: Длинные
- **Критерий выхода**: выход по количеству баров или долларовый стоп-лосс
- **Стопы**: долларовый стоп-лосс
- **Значения по умолчанию**:
  - `UseAdaptiveThreshold` = true
  - `RsiLength` = 2
  - `BuyLevel` = 14
  - `AdaptiveLength` = 8
  - `AdaptiveCoefficient` = 6
  - `ExitBars` = 28
  - `DollarStopLoss` = 1600
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Длинные
  - Индикаторы: RSI, StandardDeviation, LinearRegression
  - Стопы: Долларовый
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
