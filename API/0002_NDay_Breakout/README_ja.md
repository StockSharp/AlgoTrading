# Nデイ・ブレイクアウト
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

NデイのN日高値/安値ブレイクアウト戦略。Nデイ・ブレイクアウトは、指定された期間における新しい高値または安値を探します。モメンタムを予測しながら、価格が直近のN日高値または安値を突破した時にエントリーが発生します。移動平均フィルターとパーセンテージストップがエグジットを管理します。

テスト結果では、年間平均リターンが約43%であることが示されています。株式市場で最も効果を発揮します。

前回の極値のブレイクを待つことで、システムは方向性のある動きの開始を捉えようとします。トレンドフォロー型の移動平均でフィルタリングすることで、保ち合い中に発生するダマシシグナルを回避するのに役立ちます。


## 詳細

- **エントリー条件**: MAに基づくシグナル。
- **ロング/ショート**: 両方の方向。
- **エグジット条件**: 逆シグナルまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `LookbackPeriod` = 20
  - `MaPeriod` = 20
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromDays(1)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: MA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
