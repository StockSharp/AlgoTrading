# N日突破
[English](README.md) | [Русский](README_ru.md)

该策略在价格突破最近N日的最高或最低点时入场，并通过均线过滤和百分比止损控制风险。等待极值被突破有助于捕捉方向性行情，趋势均线过滤可避免盘整期间的假信号。

## 详情
- **入场条件**: 基于均线的信号
- **多空方向**: 双向
- **退出条件**: 反向信号或止损
- **止损**: 是
- **默认值**:
  - `LookbackPeriod` = 20
  - `MaPeriod` = 20
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromDays(1)
- **过滤器**:
  - 类型: 突破
  - 方向: 双向
  - 指标: MA
  - 止损: 是
  - 复杂度: 基础
  - 时间框架: 日线
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
