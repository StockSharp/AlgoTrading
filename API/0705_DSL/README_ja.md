# DSL 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Discontinued Signal Lines（DSL）と ATR バンドおよび Beluga オシレーターを組み合わせます。価格が 3 本のバーにわたって DSL ラインを上回り、オシレーターが下方 DSL ラインを上抜けたときにロングポジションを建てます。逆の条件でショートポジションを建てます。各取引では対応する DSL バンドをストップとして使用し、リスク・リワード倍率に基づく利益目標を設定します。

## 詳細

- **エントリー条件**:
  - ロングでは DSL 上部バンドが下部ラインより上、ショートでは逆。
  - 3 本連続のローソク足の始値と終値が DSL ライン上方（または下方）にある。
  - DSL-Beluga オシレーターのクロスオーバーシグナル。
- **ロング/ショート**: ロングとショート。
- **エグジット条件**:
  - DSL バンドでのストップロス。
  - リスク・リワード倍率での利益確定。
- **ストップ**: あり、ATR ベース。
- **デフォルト値**:
  - `Length` = 34
  - `Offset` = 30
  - `BandsWidth` = 1
  - `RiskReward` = 1.5
  - `BelugaLength` = 10
  - `DslFastMode` = true
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: DSL, ATR, RSI
  - ストップ: あり
  - 複雑さ: 高
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
