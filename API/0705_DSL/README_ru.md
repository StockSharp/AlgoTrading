# Стратегия DSL
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает линии DSL с полосами ATR и осциллятором Beluga. Лонг открывается, когда цена три бара подряд находится выше линии DSL и осциллятор пересекает снизу свою нижнюю DSL-линию. Шорт открывается при противоположных условиях. Для каждой сделки используется соответствующая полоса DSL в качестве стоп-лосса и цель по принципу риск-к прибыли.

## Детали

- **Условия входа**:
  - Для лонга верхняя полоса DSL выше нижней линии, для шорта нижняя полоса ниже верхней линии.
  - Открытие и закрытие свечи три бара подряд выше (или ниже) линии DSL.
  - Сигнал пересечения осциллятора DSL-Beluga.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Стоп по полосе DSL.
  - Тейк-профит по коэффициенту риск/прибыль.
- **Стопы**: Да, на базе ATR.
- **Значения по умолчанию**:
  - `Length` = 34
  - `Offset` = 30
  - `BandsWidth` = 1
  - `RiskReward` = 1.5
  - `BelugaLength` = 10
  - `DslFastMode` = true
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: DSL, ATR, RSI
  - Стопы: Да
  - Сложность: Высокая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
