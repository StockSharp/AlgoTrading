# Стратегия Arb Synthetic
[English](README.md) | [中文](README_cn.md)

Эта стратегия выполняет треугольный арбитраж между EURUSD, GBPUSD и EURGBP. Она строит синтетические цены для выявления несоответствия между кросс-парой и её базовыми ногами. Если расхождение превышает настраиваемый порог спреда, стратегия открывает рыночные позиции для извлечения прибыли из конвергенции.

Лонг открывается, когда рыночная цена ниже синтетической. Шорт открывается, когда рыночная цена выше синтетической. Позиции переворачиваются при закрытии спреда.

## Детали
- **Условия входа**:
  - **Покупка**: Разница между синтетической и рыночной ценой больше `Spread` в положительном направлении.
  - **Продажа**: Разница между синтетической и рыночной ценой меньше `Spread` в отрицательном направлении.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Противоположный сигнал переворачивает позицию.
- **Стопы**: Используется встроенная защита стратегии.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(1)
  - `Spread` = 35
- **Фильтры**:
  - Категория: Арбитраж
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
