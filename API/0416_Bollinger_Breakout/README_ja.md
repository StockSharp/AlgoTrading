# Bollinger ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Bollinger ブレイクアウトは、Bollinger Bandsを超えて動き、さらにその方向へ
進む値動きを捉えようとします。価格が上限バンドを上回って終値を付けるか、
下限バンドを下回って終値を付けると、オプションの確認がトレードを支持する
場合に、その戦略はブレイクアウトの方向にエントリーします。

RSI、Aroon、移動平均フィルターを有効にして、モメンタムとトレンドを
検証することができます。オプションのストップロスがリスク管理を助けます。
価格が反対側のバンドに達するか、ストップが発動したときにポジションがクローズ
されます。

このアプローチは、バンドのブレイクが平均回帰ではなく継続につながる、
強いトレンドが生じやすい市場に適しています。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 終値が上限バンドを上回り、有効なすべてのフィルターが確認する。
  - **ショート**: 終値が下限バンドを下回り、有効なすべてのフィルターが確認する。
- **エグジット条件**: 反対側バンドへのタッチ、または `UseSL` の場合はストップロス。
- **ストップ**: オプションのストップロス（`UseSL`）。
- **デフォルト値**:
  - `UseRSI` = True
  - `UseAroon` = False
  - `UseMA` = True
  - `UseSL` = True
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ロング/ショート
  - インジケーター: Bollinger Bands, RSI, Aroon, Moving Average
  - 複雑さ: 中程度
  - リスクレベル: 高
