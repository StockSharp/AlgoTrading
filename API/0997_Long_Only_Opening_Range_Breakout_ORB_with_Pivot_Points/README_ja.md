# ロングのみ 始値レンジ ブレイクアウト (ORB) とピボットポイント戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

価格が始値レンジの高値を上回るブレイクアウトをし、日足ピボットポイントの第一抵抗線 (R1) がその高値を上回っている場合に買いを入れる戦略です。トレーリングストップがピボットレベルに従います。

## 詳細

- **エントリー条件**:
  - 始値レンジ終了後、R1 が高値より上にある場合、セッション高値を上回るブレイクアウトでロングエントリー。
- **ロング/ショート**: ロングのみ
- **エグジット条件**:
  - ピボットレベルと日次終値に合わせたトレーリングストップ。
- **ストップ**: はい
- **デフォルト値**:
  - `RangeMinutes` = 15
  - `SessionStart` = 09:30
  - `MaxTradesPerDay` = 1
  - `StopLossPercent` = 3
  - `InitialSlType` = Percentage
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ロング
  - インジケーター: Pivot Points
  - ストップ: はい
  - 複雑さ: 中
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
