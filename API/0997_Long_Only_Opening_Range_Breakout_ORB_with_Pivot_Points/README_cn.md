# 多头开盘区间突破（ORB）结合枢轴点
[English](README.md) | [Русский](README_ru.md)

当价格突破开盘区间高点且日枢轴点的第一阻力位 R1 高于该高点时，策略做多。止损沿枢轴点跟踪。

## 细节

- **入场条件**：
  - 开盘区间结束后，如果 R1 在高点之上，突破该高点时做多。
- **多空方向**：仅多头
- **出场条件**：
  - 止损沿枢轴点移动，并在新交易日开始时平仓。
- **止损**：是
- **默认值**：
  - `RangeMinutes` = 15
  - `SessionStart` = 09:30
  - `MaxTradesPerDay` = 1
  - `StopLossPercent` = 3
  - `InitialSlType` = Percentage
- **过滤器**：
  - 类别：突破
  - 方向：多头
  - 指标：Pivot Points
  - 止损：是
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
