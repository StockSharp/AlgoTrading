# Strategie zum Kaufen von Kursrücksetzern im Trend
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

EMA-Crossover-Trendstrategie mit festem Take-Profit und Stop-Loss.

## Details

- **Einstieg**: Kaufen wenn die schnelle EMA die langsame EMA nach oben kreuzt; verkaufen wenn die schnelle EMA nach unten kreuzt.
- **Ausstieg**: Position durch Schutz oder gegenteiliges Signal geschlossen.
- **Indikatoren**: EMA(50), EMA(200).
- **Stops**: Take-Profit 0.7%, Stop-Loss 2%.
