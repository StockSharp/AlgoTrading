# 押し目買いトレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

固定のテイクプロフィットとストップロスを持つEMAクロスオーバートレンド戦略。

## 詳細

- **エントリー**: 速いEMAが遅いEMAを上抜けたときに買い、下抜けたときに売り。
- **エグジット**: 保護または逆シグナルによってポジション決済。
- **インジケーター**: EMA(50)、EMA(200)。
- **ストップ**: テイクプロフィット 0.7%、ストップロス 2%。
