# Supertrend-Strategie mit Money Ocean Trade

Verwendet den SuperTrend-Indikator zur Bestätigung von Trendumkehrungen.

- **Long**: SuperTrend dreht nach oben und der Preis kreuzt seine Linie von unten.
- **Short**: SuperTrend dreht nach unten und der Preis kreuzt seine Linie von oben.
- **Indikatoren**: SuperTrend.
