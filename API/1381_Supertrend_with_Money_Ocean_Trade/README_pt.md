Usa o indicador SuperTrend para confirmar reversões de tendência.

- **Comprado**: SuperTrend vira para cima e o preço cruza acima de sua linha.
- **Vendido**: SuperTrend vira para baixo e o preço cruza abaixo de sua linha.
- **Indicadores**: SuperTrend.
