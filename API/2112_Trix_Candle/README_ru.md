# Стратегия Trix Candle
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует развороты на основе индикатора Trix Candle, который применяет тройное экспоненциальное сглаживание к ценам открытия и закрытия свечей и окрашивает каждую свечу в зависимости от того, выше или ниже сглаженная цена закрытия относительно сглаженной цены открытия.

## Детали

- **Условия входа**:
  - **Long**: предыдущая свеча бычья (цвет 2) и текущий цвет свечи < 2
  - **Short**: предыдущая свеча медвежья (цвет 0) и текущий цвет свечи > 0
- **Длинные/Короткие**: Длинные и короткие
- **Условия выхода**:
  - Long: предыдущая свеча медвежья (цвет 0)
  - Short: предыдущая свеча бычья (цвет 2)
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `TRIX Period` = 14
  - `Candle Type` = 4h
  - `Allow Buy Open` = true
  - `Allow Sell Open` = true
  - `Allow Buy Close` = true
  - `Allow Sell Close` = true
- **Фильтры**:
  - Категория: Разворотная
  - Направление: Оба
  - Индикаторы: Triple Exponential Moving Average
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
