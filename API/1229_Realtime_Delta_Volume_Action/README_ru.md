# Стратегия Realtime Delta Volume Action
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает разницу между объёмами покупок и продаж в каждой свече. Позиция открывается, когда дельта объёма превышает заданный порог.

## Детали

- **Условия входа**: Дельта объёма выше/ниже порога.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: Обратный сигнал или стоп.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `DeltaThreshold` = 100
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Delta объёма
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
