# Стратегия Arsi Vwap Atr
[English](README.md) | [中文](README_cn.md)

Адаптивная стратегия RSI, где уровни перекупленности и перепроданности расширяются или сужаются на основе ATR или отклонения от VWAP. Позиции открываются при пересечении RSI адаптивных уровней и закрываются при возвращении RSI в среднюю зону.

## Подробности

- **Критерии входа**:
  - Лонг: `RSI` пересекает снизу адаптивный уровень перепроданности
  - Шорт: `RSI` пересекает сверху адаптивный уровень перекупленности
- **Лонг/Шорт**: Оба направления
- **Критерии выхода**:
  - RSI пересекает уровень 50 или противоположную адаптивную линию
- **Стопы**: Процентные через `StopLossPercent` и `RiskReward`
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `BaseK` = 1m
  - `RiskPercent` = 2m
  - `StopLossPercent` = 2.5m
  - `RiskReward` = 2m
  - `SourceOb` = ATR
  - `SourceOs` = ATR
  - `AtrLengthOb` = 14
  - `AtrLengthOs` = 14
  - `ObMultiplier` = 10m
  - `OsMultiplier` = 10m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: RSI, ATR, VWAP
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
