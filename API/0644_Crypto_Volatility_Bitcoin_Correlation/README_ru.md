# Волатильность криптовалют и корреляция с Bitcoin
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда индекс VIXFix и индекс волатильности BVOL7D растут, а цена выше EMA. Выход из позиции происходит при пересечении ценой EMA сверху вниз.

## Подробности

- **Условия входа**: VIXFix выше предыдущего значения, BVOL7D выше предыдущего значения, закрытие выше EMA.
- **Лонг/Шорт**: Только лонг.
- **Условия выхода**: Закрытие ниже EMA.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `VixFixLength` = 22
  - `EmaLength` = 50
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Лонг
  - Индикаторы: Highest, EMA
  - Стопы: Нет
  - Сложность: Начальный уровень
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
