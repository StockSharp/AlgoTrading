# Forex ハンマーと首吊り線戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は古典的なローソク足のリバーサルパターン、強気のハンマーと弱気の首吊り線を取引します。ハンマーの後にロング、首吊り線の後にショートでエントリーし、固定本数のバーの間ポジションを保持します。

保有期間が終了するか保護ストップに触れるとポジションを決済します。

## 詳細

- **エントリー条件**: ロングにハンマー、ショートに首吊り線。
- **ロング/ショート**: 両方。
- **エグジット条件**: 保有期間またはストップロス/テイクプロフィット。
- **ストップ**: あり。
- **デフォルト値**:
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `BodyLengthMultiplier` = 5
  - `ShadowRatio` = 1
  - `HoldPeriods` = 26
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
