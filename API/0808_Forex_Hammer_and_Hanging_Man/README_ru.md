# Форекс: молот и повешенный
[English](README.md) | [中文](README_cn.md)

Стратегия торгует классические свечные разворотные модели: бычий молот и медвежий повешенный. Покупка происходит после молота, короткая позиция открывается после повешенного, удержание длится заданное количество баров.

Позиция закрывается по истечении периода удержания или при срабатывании защитных стопов.

## Детали

- **Условия входа**: Молот для лонга, повешенный для шорта.
- **Длинные/короткие позиции**: Оба направления.
- **Условия выхода**: Период удержания или стоп-лосс/тейк-профит.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `BodyLengthMultiplier` = 5
  - `ShadowRatio` = 1
  - `HoldPeriods` = 26
- **Фильтры**:
  - Категория: Pattern
  - Направление: Both
  - Индикаторы: Candlestick
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
