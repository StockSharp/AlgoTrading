# Yesterday Today戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Yesterday Today戦略は、今日の価格を昨日の高値・安値と比較するクラシックなMetaTraderのブレイクアウトを再現します。この戦略は最後に完了した日足ローソクを追跡し、価格が昨日のレンジから抜け出した時に素早く反応するためにイントラデイのローソクを監視します。反転する前には必ず逆方向のポジションを決済するため、クリーンな1ポジションのワークフローを実現します。

## 概要

- 前日の日足レンジを追跡し、イントラデイのローソクの終値がそれを突破するのを待ちます。
- 終値が昨日の高値を超えた場合はロングポジションを建て、昨日の安値を下回った場合はショートポジションを建てます。
- ピップで表された固定距離のストップロスとテイクプロフィットレベルを適用します。ピップサイズはオリジナルのMQL実装と同様に、3桁または5桁のFX相場に適応します。
- リスクレベルはイントラデイのローソクが完了するたびに、その高値・安値を使ってストップロスやテイクプロフィットのヒットを検出します。
- 予期しないマージン問題から保護するために組み込みの保護フレームワークを使用します。

## ワークフロー

1. 日足ローソクを購読し、最後に完了したセッションの高値・安値を保存します。
2. シグナル評価のためにイントラデイのローソク（デフォルトは15分）を購読します。
3. イントラデイのローソクが完了するたびに：
   - ローソクがアクティブなストップロスまたはテイクプロフィットに違反する場合は即座に決済します。
   - 終値が昨日の高値を上回り、ロングポジションが開いていない場合はロングで参入します。
   - 終値が昨日の安値を下回り、ショートポジションが開いていない場合はショートで参入します。
   - 逆方向のポジションは市場注文の数量を増やすことで最初に決済されます。
4. 新しい日足ローソクが完了するたびに、翌取引日のために保存されたレンジを更新します。

## パラメーター

- `TradeVolume` — 新規ポジションのロットサイズ。反転時は、戦略が最初にフラットにするために逆方向の露出を自動的に追加します。
- `StopLossPips` — エントリー価格からの保護的なストップまでの距離（ピップ単位）。`0` はストップを無効にします。
- `TakeProfitPips` — エントリー価格からの利益目標までの距離（ピップ単位）。`0` は目標を無効にします。
- `SignalCandleType` — ブレイクアウト検出に使用されるイントラデイのローソクタイプ（デフォルトは15分ローソク）。

## 詳細

- **エントリー条件**: イントラデイのローソクが昨日の高値を上回って終値を付ける（ロング）か、昨日の安値を下回って終値を付ける（ショート）。
- **ロング/ショート**: 両方向対応。
- **エグジット条件**: イントラデイのローソクの高値・安値がストップロスまたはテイクプロフィットレベルに触れる。
- **ストップ**: はい、固定ピップ距離。
- **デフォルト値**:
  - `TradeVolume` = 1
  - `StopLossPips` = 50
  - `TakeProfitPips` = 50
  - `SignalCandleType` = `TimeSpan.FromMinutes(15).TimeFrame()`
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Price Action
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 日足コンテキストを使ったイントラデイエントリー
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中

## 注記

- この戦略は単一の銘柄向けに設計されています。開始前に `Security` と `Portfolio` を設定してください。
- ピップサイズは `Security.PriceStep` から計算され、3桁または5桁のFXシンボルに対して自動的にスケーリングされ、オリジナルのEAロジックを反映します。
- 保護は `OnStarted` で有効化されるため、戦略が取引中もグローバルな口座保護機能が有効なまま維持されます。
