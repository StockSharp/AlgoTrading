# Pivot Heiken戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

日次ピボットポイントとHeikin-Ashiローソク足を組み合わせ、オプションのトレーリングストップを使用する戦略です。日次ピボットは前日の高値、安値、終値から計算されます。Heikin-Ashiの平滑化により価格のノイズをフィルタリングしてトレンドの方向を強調します。

## ロジック
- **ロングエントリー**: Heikin-Ashiローソク足が強気で、終値が日次ピボットを上回っている。
- **ショートエントリー**: Heikin-Ashiローソク足が弱気で、終値が日次ピボットを下回っている。
- **エグジット**: ストップロス、テイクプロフィット、またはトレーリングストップのレベルでポジションをクローズ。

## パラメーター
- `CandleType` – 使用するローソク足シリーズ。
- `StopLossPips` – ストップロスの距離（pips）。
- `TakeProfitPips` – テイクプロフィットの距離（pips）。
- `TrailingStopPips` – トレーリングストップの距離（pips）（0でトレーリング無効）。

## インジケーター
- Heikin-Ashi（内部計算）。
- 日次ピボットポイント。

## 注記
- ローソク足サブスクリプションとインジケーターバインディングを使用した高レベルAPIを使用。
- ロングおよびショート取引の両方に適しています。
