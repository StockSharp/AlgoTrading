# Стратегия Improved EMA & CDC Trailing Stop
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает фильтр тренда на основе EMA, подтверждение MACD и трейлинг-стоп CDC на базе ATR.

## Детали

- **Условия входа**:
  - **Лонг**: цена > EMA60, EMA60 > EMA90, линия MACD > сигнальной.
  - **Шорт**: цена < EMA60, EMA60 < EMA90, линия MACD < сигнальной.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - Трейлинг-стоп или тейк-профит на основе ATR.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `Ema60Period` = 60
  - `Ema90Period` = 90
  - `AtrPeriod` = 24
  - `Multiplier` = 4
  - `ProfitTargetMultiplier` = 2
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: обе стороны
  - Индикаторы: EMA, MACD, ATR
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
