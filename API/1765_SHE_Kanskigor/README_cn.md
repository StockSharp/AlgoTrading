# SHE Kanskigor 策略
[English](README.md) | [Русский](README_ru.md)

该日内策略每天只交易一次，依据前一日蜡烛的方向。在设定时间，若前一日收盘价低于开盘价则买入；若收盘价高于开盘价则卖出。使用固定点数的止盈和止损来控制风险，每天仅允许一笔交易。

## 细节

- **入场条件**：在 `StartTime` 比较前一日的开盘与收盘；`open > close` 时买入，`open < close` 时卖出
- **多/空**：双向
- **离场条件**：止盈或止损
- **止损**：是
- **默认值**：
  - `Volume` = 0.1
  - `StartTime` = 00:05
  - `TakeProfit` = 350
  - `StopLoss` = 550
- **过滤器**：
  - 类别：反转
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：基础
  - 时间框架：日线
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
