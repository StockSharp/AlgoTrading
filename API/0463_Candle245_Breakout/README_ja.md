# 2時45分ローソク足ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このイントラデイ戦略は、午前2時45分のローソク足を監視し、次の数本のバー内でその高値または安値のブレイクアウトを取引します。価格がローソク足の高値を超えるとロングポジションに入り、価格がローソク足の安値を下回るとショートポジションを開きます。観察ウィンドウの終わりまでに反対方向のブレイクアウトが発生しない場合、ポジションは決済されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 次の`LookForwardBars`本のローソク足内で価格が2時45分ローソク足の高値を上抜け。
  - **ショート**: 次の`LookForwardBars`本のローソク足内で価格が2時45分ローソク足の安値を下抜け。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 観察ウィンドウの終了または反対方向のブレイクアウト。
- **ストップ**: なし。
- **デフォルト値**:
  - `TargetHour` = 2
  - `TargetMinute` = 45
  - `LookForwardBars` = 2
  - `CandleType` = 45分足
- **フィルター**:
  - カテゴリ: 時間ベースのブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: イントラデイ
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
