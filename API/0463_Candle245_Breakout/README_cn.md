# 2:45 AM蜡烛突破策略
[English](README.md) | [Русский](README_ru.md)

本策略监控每天2:45 AM的K线，并在接下来数根K线中交易其高点或低点的突破。当价格突破该K线高点时做多，跌破低点时做空。如果在观察窗口结束前未出现相反突破，持仓将被平仓。

## 细节

- **入场条件**：
  - **多头**：价格在接下来的`LookForwardBars`根K线内向上突破2:45 AM K线的最高价。
  - **空头**：价格在接下来的`LookForwardBars`根K线内向下跌破2:45 AM K线的最低价。
- **方向**：双向。
- **出场条件**：
  - 观察窗口结束或出现相反突破。
- **止损**：无。
- **默认参数**：
  - `TargetHour` = 2
  - `TargetMinute` = 45
  - `LookForwardBars` = 2
  - `CandleType` = 45分钟K线
- **过滤器**：
  - 类型：基于时间的突破
  - 方向：双向
  - 指标：无
  - 止损：无
  - 复杂度：低
  - 时间框架：日内
  - 季节性：是
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
