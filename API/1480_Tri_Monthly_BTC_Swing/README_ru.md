# Стратегия Tri-Monthly BTC Swing
[English](README.md) | [中文](README_cn.md)

Tri-Monthly BTC Swing использует EMA200, пересечение MACD и фильтр RSI.
Стратегия допускает только одну сделку каждые 90 дней.

## Детали

- **Критерий входа**: закрытие выше EMA200, линия MACD выше сигнальной, RSI выше порога и прошло минимум 90 дней с последней сделки
- **Длин/Шорт**: только лонг
- **Критерий выхода**: линия MACD ниже сигнальной или RSI ниже порога
- **Стопы**: нет
- **Значения по умолчанию**:
  - `EmaLength` = 200
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `RsiLength` = 14
  - `RsiThreshold` = 50
  - `TradeInterval` = 90 дней
  - `CandleType` = 1 день
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: EMA, MACD, RSI
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
