# Candle Trader 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

**Candle Trader 戦略**は、直近4本の完成したローソク足の方向（強気または弱気）を分析して、短期的な反転の機会を特定します。単一の銘柄で動作し、事前に定義されたテイクプロフィットとストップロスのレベルで成行注文を送信します。

## 戦略ロジック

1. **ロングエントリー（直接）** – 最新のローソク足が強気、その前の2本が弱気。
2. **ロングエントリー（継続）** – 最新のローソク足が強気、1本前が弱気、さらにその前の2本が強気。このルールは*Continuation*が`true`の場合のみ有効。
3. **ショートエントリー（直接）** – 最新のローソク足が弱気、その前の2本が強気。
4. **ショートエントリー（継続）** – 最新のローソク足が弱気、1本前が強気、さらにその前の2本が弱気。*Continuation*が`true`の場合のみ有効。
5. *Reverse Close*が有効で現在のポジションと反対の新しいシグナルが出た場合、戦略は新しいポジションを開く前に既存のポジションをクローズする。
6. すべての注文は価格ステップ単位で固定されたテイクプロフィットとストップロスの値で保護される。

## パラメーター

| 名前 | 説明 |
|------|-------------|
| `Volume` | 各取引の注文ボリューム。 |
| `TakeProfitTicks` | 価格ステップ単位のテイクプロフィット距離。 |
| `StopLossTicks` | 価格ステップ単位のストップロス距離。 |
| `Continuation` | 追加エントリーのための継続パターンを有効にする。 |
| `ReverseClose` | 反対方向にエントリーする前にオープンポジションをクローズする。 |
| `CandleType` | 分析に使用するローソク足の時間軸。 |

## 注記

- 戦略は完成したローソク足のみを評価する。
- 成行注文を使用し、新しい注文を送信する前にアクティブな注文をキャンセルする。
- ストップロスとテイクプロフィットのレベルは`StartProtection`で適用される。
- ポジションサイズは`Volume`パラメーターを通じて最適化できる。
