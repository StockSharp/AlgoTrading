# Стратегия аномального контртренда
[English](README.md) | [中文](README_cn.md)

Алгоритм ищет резкие процентные движения за короткий интервал и торгует против них. При резком росте выше порога открывает шорт, при падении ниже порога – лонг. Стоп-лосс и тейк-профит задаются в тиках.

## Детали

- **Условия входа**: Процентное изменение за период превышает порог.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Стоп-лосс или тейк-профит.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `PercentageThreshold` = 1
  - `LookbackMinutes` = 30
  - `StopLossTicks` = 100
  - `TakeProfitTicks` = 200
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Контртренд
  - Направление: Оба
  - Индикаторы: Цена
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
