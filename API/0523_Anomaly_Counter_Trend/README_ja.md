# 異常逆張り戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このアルゴリズムは短い期間内の急激なパーセンテージ変動を検出し、それに逆らって取引します。価格が閾値を上回ると売り、価格が閾値を下回ると買います。ストップロスとテイクプロフィットはティックで設定されます。

## 詳細

- **エントリー条件**: ルックバック期間内のパーセンテージ変化が閾値を超える。
- **ロング/ショート**: 両方向。
- **エグジット条件**: ストップロスまたはテイクプロフィット。
- **ストップ**: はい。
- **デフォルト値**:
  - `PercentageThreshold` = 1
  - `LookbackMinutes` = 30
  - `StopLossTicks` = 100
  - `TakeProfitTicks` = 200
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: 逆張り
  - 方向: 両方
  - インジケーター: 価格
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ (1m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
