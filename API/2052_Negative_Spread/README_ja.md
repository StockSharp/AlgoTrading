# ネガティブスプレッド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Negative Spread戦略は、最良売り値が最良買い値を下回り、ネガティブなスプレッドが生じる稀な瞬間を活用します。
この価格の歪みが現れると、戦略は成行売りを行い、異常なスプレッドを捉えようとします。
ショートポジションが開かれると、市場が通常の状態に戻った次の板更新で決済されます。

システムは板情報のイベントのみを監視し、ローソク足や指標には依存しません。
オプションのストップロスとテイクプロフィットのパラメーターが安全対策として提供され、銘柄のティックサイズを使用してpipsで計算されます。

## 詳細
- **エントリー条件**: `BestAsk < BestBid` かつアクティブなポジションなし。
- **ロング/ショート**: ショートのみ。
- **エグジット条件**: ポジションは開かれた直後に決済される。
- **ストップ**: オプションのpipsでのストップロスとテイクプロフィット。
- **デフォルト値**:
  - `Volume` = 1
  - `TakeProfitPips` = 5000
  - `StopLossPips` = 5000
- **フィルター**:
  - カテゴリ: 裁定取引
  - 方向: ショート
  - インジケーター: なし
  - ストップ: オプション
  - 複雑さ: 基本
  - 時間軸: ティック
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
