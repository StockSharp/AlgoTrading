# Стратегия Test Bot: Bearish Buy / Bullish Sell
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия покупает при первом медвежьем баре и закрывает позицию при первом бычьем баре.

## Детали

- **Условия входа**: Первый медвежий бар при отсутствии позиции.
- **Лонг/Шорт**: Только лонг.
- **Условия выхода**: Первый бычий бар.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Разворот
  - Направление: Лонг
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
