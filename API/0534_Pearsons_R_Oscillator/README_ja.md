# Pearson's Rオシレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Pearson's Rオシレーター戦略は、ピアソン相関係数を使って価格が線形回帰チャネルに最もよく適合する期間を動的に探索します。相関が指定された正または負の閾値に達すると、戦略は回帰チャネルを形成しブレイクアウトを取引します。

価格がチャネル境界を越えた際にポジションを建て、中心線のクロスで決済できます。このアプローチは最も強い相関にルックバックウィンドウを自動調整することで市場状況に適応します。

## 詳細

- **エントリー条件**:
  - 価格が上部回帰ラインを上抜け → **ロング**。
  - 価格が下部回帰ラインを下抜け → **ショート**。
- **ロング/ショート**: 両サイド。
- **エグジット条件**:
  - 反対方向への中心線クロス。
- **ストップ**: なし。
- **デフォルト値**:
  - `MinPeriod` = 48
  - `MaxPeriod` = 360
  - `Step` = 12
  - `IdealPositive` = 0.85
  - `IdealNegative` = -0.85
  - `Deviations` = 2
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Pearson's R, 線形回帰
  - ストップ: なし
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
