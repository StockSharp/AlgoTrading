# Pearson's R振荡策略
[English](README.md) | [Русский](README_ru.md)

Pearson's R振荡策略根据皮尔逊相关系数自动寻找价格与线性回归通道最匹配的周期。当相关系数达到设定的正或负阈值时，形成回归通道并在突破时交易。

当价格穿越通道边界时开仓，穿越中线时可平仓。该方法通过自动调整观察窗口以匹配最强相关性，从而适应市场变化。

## 详情

- **入场条件**：
  - 价格突破上轨 → **做多**。
  - 价格跌破下轨 → **做空**。
- **方向**：双向。
- **出场条件**：
  - 价格反向穿越中线。
- **止损**：无。
- **默认参数**：
  - `MinPeriod` = 48
  - `MaxPeriod` = 360
  - `Step` = 12
  - `IdealPositive` = 0.85
  - `IdealNegative` = -0.85
  - `Deviations` = 2
- **过滤器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：皮尔逊相关、线性回归
  - 止损：无
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
