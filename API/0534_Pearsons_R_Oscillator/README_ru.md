# Стратегия Pearson's R Oscillator
[English](README.md) | [中文](README_cn.md)

Стратегия Pearson's R Oscillator динамически ищет период, при котором цена наилучшим образом укладывается в канал линейной регрессии по коэффициенту корреляции Пирсона. Когда корреляция достигает заданного положительного или отрицательного порога, формируется канал и выполняются сделки на пробой.

Сделки открываются при пересечении цены с границами канала и могут закрываться при пересечении средней линии. Подход адаптируется к рынку, автоматически подстраивая окно анализа под наиболее сильную корреляцию.

## Подробности

- **Условия входа**:
  - Пересечение цены выше верхней линии канала → **Лонг**.
  - Пересечение ниже нижней линии канала → **Шорт**.
- **Направление**: обе стороны.
- **Условия выхода**:
  - Пересечение средней линии в противоположную сторону.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `MinPeriod` = 48
  - `MaxPeriod` = 360
  - `Step` = 12
  - `IdealPositive` = 0.85
  - `IdealNegative` = -0.85
  - `Deviations` = 2
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: Коэффициент Пирсона, линейная регрессия
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
