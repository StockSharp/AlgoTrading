# Стратегия Internal Bar Strength IBS Mean Reversion
[English](README.md) | [中文](README_cn.md)

Шортовая стратегия возврата к среднему, использующая показатель Internal Bar Strength (IBS). Продаёт, когда IBS высок и цена пробивает предыдущий максимум, и выходит, когда IBS опускается ниже нижнего порога.

## Детали

- **Критерий входа**: IBS >= верхний порог и закрытие > предыдущего максимума
- **Длин/Шорт**: шорт
- **Критерий выхода**: IBS <= нижний порог
- **Стопы**: нет
- **Значения по умолчанию**:
  - `UpperThreshold` = 0.9
  - `LowerThreshold` = 0.3
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Шорт
  - Индикаторы: IBS
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
