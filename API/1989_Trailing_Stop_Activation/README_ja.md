# トレーリングストップ・アクティベーション戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

**トレーリングストップ・アクティベーション戦略**は、既存のポジションの保護ストップレベルを管理します。エントリーシグナルは生成せず、代わりにポジションを開いた後にストップを調整して利益を確保します。

## パラメーター

- `TrailingStop` – トレーリングストップが有効になる前に、市場がポジションに有利な方向に動かなければならない価格単位の距離。
- `StopLoss` – 価格単位での初期ストップロス距離（オプション）。無効にするには `0` に設定。
- `CandleType` – 価格追跡に使用するローソク足の種類。

## 取引ルール

1. ポジションが開かれたとき、`StopLoss` がゼロより大きければ初期ストップロスが設定されます。
2. 利益が `TrailingStop` を超えると、ストップレベルが指定された距離を維持しながら価格を追跡します。
3. 価格がトレーリングストップのレベルに触れると、ポジションが決済されます。
4. この戦略はロングおよびショートポジションの両方に機能します。

## 注意事項

この戦略はエントリーシグナルを提供する別の戦略と組み合わせて使用するよう設計されています。トレーリングストップによるエグジット管理のみに特化しています。
