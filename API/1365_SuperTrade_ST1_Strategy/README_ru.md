# SuperTrade ST1 Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия только на покупку, объединяющая индикатор Supertrend с фильтром EMA и управлением риском на основе ATR.

Тестирование показывает среднюю годовую доходность около 45%. Лучшие результаты на рынке криптовалют.

Система ждёт снижения направления Supertrend при условии, что цена находится выше линии Supertrend и EMA. Риск контролируется стоп‑лоссом и тейк‑профитом по ATR с соотношением 1:4.

## Детали

- **Условия входа**:
  - Предыдущее направление Supertrend > текущего
  - Цена > Supertrend
  - Цена > EMA
- **Лонг/Шорт**: Только лонг
- **Условия выхода**: `Close <= entry - StopAtrMultiplier * ATR` или `Close >= entry + TakeAtrMultiplier * ATR`
- **Стопы**: Стоп‑лосс и тейк‑профит по ATR
- **Параметры по умолчанию**:
  - `AtrPeriod` = 10
  - `Factor` = 3.0
  - `EmaPeriod` = 200
  - `StopAtrMultiplier` = 1.0
  - `TakeAtrMultiplier` = 4.0
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: Supertrend, EMA, ATR
  - Стопы: Да
  - Сложность: Простая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
