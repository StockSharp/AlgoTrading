# TSIクラウドクロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TSIクラウドクロス戦略は、True Strength Index (TSI)をその遅延コピーと比較してクラウドを形成します。TSIがシフトされたラインを上抜けするとロングポジションを開き、強気のモメンタムを示します。TSIがシフトされたラインを下抜けするとショートポジションを開きます。シグナルを反転させることができ、逆のポジションをオプションでクローズすることもできます。

## 詳細

- **エントリー条件**:
  - TSIがシフト値を上抜け (ロング)。
  - TSIがシフト値を下抜け (ショート)。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆シグナルでのオプションクローズ。
- **ストップ**: なし。
- **デフォルト値**:
  - `LongLength` = 25
  - `ShortLength` = 13
  - `TriggerShift` = 1
  - `Invert` = false
- **フィルター**:
  - カテゴリ: モメンタムオシレーター
  - 方向: ロング/ショート
  - インジケーター: True Strength Index
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
