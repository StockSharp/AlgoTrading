# Estrategia JMA Quantum Edge
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza la Media Móvil Jurik (JMA) para detectar puntos de giro.
Se abre una posición larga cuando la JMA gira hacia arriba y está por encima de la JMA del marco temporal superior.
Se abre una posición corta cuando la JMA gira hacia abajo y está por debajo de la JMA del marco temporal superior.
Se aplican opciones de stop-loss y take profit como porcentajes del precio de entrada.
