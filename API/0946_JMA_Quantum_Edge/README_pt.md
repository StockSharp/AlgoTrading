# Estratégia JMA Quantum Edge
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia utiliza a Média Móvel Jurik (JMA) para detectar pontos de reversão.
Uma posição comprada é aberta quando a JMA vira para cima e está acima da JMA do período superior.
Uma posição vendida é aberta quando a JMA vira para baixo e está abaixo da JMA do período superior.
Stop-loss e take profit opcionais são aplicados como percentagens do preço de entrada.
