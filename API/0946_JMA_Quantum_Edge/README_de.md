# JMA Quantum Edge Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet den Jurik Moving Average (JMA) zur Erkennung von Wendepunkten.
Eine Long-Position wird eröffnet, wenn der JMA nach oben dreht und über dem JMA des höheren Zeitrahmens liegt.
Eine Short-Position wird eröffnet, wenn der JMA nach unten dreht und unter dem JMA des höheren Zeitrahmens liegt.
Optionaler Stop-Loss und Take Profit werden als Prozentsatz vom Einstiegspreis angewendet.
