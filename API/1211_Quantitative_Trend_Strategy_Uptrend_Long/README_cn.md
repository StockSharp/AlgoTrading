# 定量趋势策略——多头上升
[English](README.md) | [Русский](README_ru.md)

当收盘价突破最近的枢轴高点时，本策略做多。支撑和阻力水平基于枢轴高点和低点计算，并使用百分比止盈和止损保护持仓。

## 细节

- **入场条件**：
  - **多头**：收盘价突破最后的枢轴高点。
- **多空方向**：仅做多。
- **出场条件**：
  - 收盘价跌破最后的枢轴低点；
  - 最后的枢轴高点低于最后的枢轴低点。
- **止损/止盈**：是，百分比止盈和止损。
- **默认值**：
  - `PivotLeft` = 46
  - `PivotRight` = 32
  - `StopLossPercent` = 1.75
  - `TakeProfitPercent` = 2
