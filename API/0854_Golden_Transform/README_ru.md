# Стратегия Golden Transform
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает индикатор Rate of Change с тройным Hull-TRIX, фильтром на основе Hull MA и сглаженным Fisher Transform. Длинные позиции открываются, когда ROC пересекает TRIX снизу вверх, TRIX находится ниже нуля, а цена открытия выше Hull MA. Короткие позиции открываются при обратном сигнале. Позиции закрываются при обратном пересечении или когда сглаженный Fisher выходит за пороги и разворачивается.

## Детали

- **Условия входа**:
  - **Long**: `ROC пересекает выше TRIX` && `TRIX < 0` && `Open > Hull MA`
  - **Short**: `ROC пересекает ниже TRIX` && `TRIX > 0` && `Open < Hull MA`
- **Длинные/Короткие**: Длинные и короткие
- **Условия выхода**:
  - Long: `ROC пересекает ниже TRIX` ИЛИ (`Fisher HMA > 1.5` && `Fisher HMA пересекает ниже предыдущего Fisher`)
  - Short: `ROC пересекает выше TRIX` ИЛИ (`Fisher HMA < -1.5` && `Fisher HMA пересекает выше предыдущего Fisher`)
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `ROC Length` = 50
  - `Hull TRIX Length` = 90
  - `Hull Entry Length` = 65
  - `Fisher Length` = 50
  - `Fisher Smooth Length` = 5
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: ROC, Hull MA, Fisher Transform
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
