# Стратегия RSI-CCI Fusion
[English](README.md) | [中文](README_cn.md)

Стратегия объединяет стандартизированные значения RSI и CCI в единый осциллятор с динамическими полосами.
Покупает, когда объединённое значение пересекает нижнюю полосу снизу вверх, и продаёт или открывает шорт при пересечении верхней полосы сверху вниз.

## Детали

- **Критерий входа**: пересечение нижней полосы снизу вверх для лонга; пересечение верхней полосы сверху вниз для шорта
- **Длин/Шорт**: обе стороны (шорт опционален)
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Length` = 14
  - `RsiWeight` = 0.5
  - `EnableShort` = false
- **Фильтры**:
  - Категория: Моментум
  - Направление: Обе стороны
  - Индикаторы: RSI, CCI, SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
