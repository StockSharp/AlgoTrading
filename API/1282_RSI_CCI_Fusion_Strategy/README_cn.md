# RSI-CCI Fusion 策略
[English](README.md) | [Русский](README_ru.md)

将标准化后的 RSI 与 CCI 融合为单一振荡器，并为其构建动态带。
当融合值向上穿越下轨时买入，向下穿越上轨时卖出或做空。

## 细节

- **入场条件**: 融合值向上穿越下轨做多；向下穿越上轨做空
- **多空方向**: 双向（做空可选）
- **出场条件**: 相反信号
- **止损**: 无
- **默认值**:
  - `Length` = 14
  - `RsiWeight` = 0.5
  - `EnableShort` = false
- **过滤器**:
  - 分类: 动量
  - 方向: 双向
  - 指标: RSI, CCI, SMA, StandardDeviation
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
