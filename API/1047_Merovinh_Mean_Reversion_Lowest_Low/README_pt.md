# Merovinh - Reversão à Média pela Mínima Mais Baixa
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia compra quando a mínima mais baixa atual de um período de retrospectiva rompe mínimas anteriores sucessivas um número configurável de vezes. A posição é encerrada quando uma nova máxima mais alta aparece dentro do mesmo período.

## Parâmetros
- Bars — comprimento de retrospectiva para máximas/mínimas.
- Number Of Lows — número de mínimas consecutivas rompidas necessárias para entrar.
- Start Date / End Date — intervalo de datas de negociação.
- Candle Type — tipo de velas.
