# Стратегия RSI Hull MA

Стратегия основана на индикаторах RSI и Hull Moving Average. Длинная позиция открывается, если RSI опускается ниже 30 и HMA растёт (текущее значение выше предыдущего). Короткая позиция открывается при RSI выше 70 и падающей HMA. Подходит для смешанных рынков.

## Подробности
- **Условия входа**:
  - **Long**: RSI < 30 && HMA(t) > HMA(t-1)
  - **Short**: RSI > 70 && HMA(t) < HMA(t-1)
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход, когда RSI возвращается в нейтральную зону
  - **Short**: выход, когда RSI возвращается в нейтральную зону
- **Стопы**: да
- **Значения по умолчанию**:
  - `RsiPeriod` = 14
  - `HullPeriod` = 9
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: RSI Hull MA
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
