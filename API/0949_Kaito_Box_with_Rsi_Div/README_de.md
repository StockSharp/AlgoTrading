Die Strategie kauft, wenn der Kurs in einem Abwärtstrend unter die jüngste Box-Range durchbricht und eine bullische RSI-Divergenz vorliegt, und schließt die Position, wenn in einem Aufwärtstrend eine bärische Divergenz erscheint.
