# Стратегия Live RSI
[English](README.md) | [中文](README_cn.md)

Использует несколько расчётов RSI (close, weighted, typical, median, open) и Parabolic SAR для обнаружения смены тренда. Входит в лонг при бычьем порядке RSI и цене выше SAR, открывает шорт при медвежьем порядке и цене ниже SAR. Значение SAR выступает в роли трейлинг-стопа.

## Детали

- **Условия входа**:
  - Лонг при бычьем порядке RSI и цене выше SAR.
  - Шорт при медвежьем порядке RSI и цене ниже SAR.
- **Long/Short**: Оба направления.
- **Условия выхода**:
  - Противоположный сигнал или трейлинг-стоп по SAR.
- **Стопы**: Опциональный фиксированный стоп-лосс и трейлинг по SAR.
- **Значения по умолчанию**:
  - `RSI Period` = 30
  - `SAR Step` = 0.08
  - `Stop Loss` = 40
  - `Check Hour` = false
  - `Start Hour` = 17
  - `End Hour` = 1
  - `Candle Type` = 1 hour
- **Фильтры**:
  - Категория: Trend Following
  - Направление: Long & Short
  - Индикаторы: RSI, Parabolic SAR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Опционально (фильтр по времени)
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
