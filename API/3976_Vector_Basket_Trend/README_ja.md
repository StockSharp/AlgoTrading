# ベクターバスケットトレンド戦略（MT4ポート）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このフォルダには、MetaTrader 4 Expert Advisor **Vector** の StockSharp の高レベル API ポートが含まれています (元のスクリプト: `MQL/8305/Vector.mq4`)。この戦略は、最大 4 つの主要外国為替ペア (EURUSD (プライマリー)、GBPUSD、USDCHF、および USDJPY) を調整し、共有の平滑移動平均の調整が表示されると、それらを同じ方向に取引します。この変換では、中心となる Vector のアイデアを維持しながら、慣用的な StockSharp パターンに適応させます。

## 取引ロジック

1. **平滑移動平均 (SMMA)** – 各商品は、設定可能な取引時間枠 (デフォルトでは 15 分) の中央値に基づいて計算された高速 (3 期間) および低速 (7 期間) の SMMA を追跡します。
2. **ベクトル トレンド フィルター** – すべての高速/低速ペアの差が合計されます。プラスの合計はバスケット全体で同期した強気の勢いを示し、マイナスの合計は全体的な弱気の圧力を示します。
3. **エントリールール** – この戦略は、次の場合にのみ成行注文でポジションをオープンまたは反転します。
   - バスケットのトレンドはポジティブであり、商品の高速 SMMA は低速 SMMA (ロングエントリー) を上回っています。
   - バスケットのトレンドはネガティブで、ファースト SMMA はスロー SMMA (ショート エントリー) を下回っています。
4. **H4 レンジからのピップターゲット** – 各商品について、個別の 4 時間のローソク足サブスクリプションで以前のレンジを測定します。その範囲の 5 分の 1 (13 ピップに上限) がポジションごとの利益目標となり、MT4 コードからの固定ピップの出口を反映しています。
5. **グローバル株式ガード** – パーセンテージベースの利益とドローダウンのしきい値（元の `PrcProfit` および `PrcLose` 入力から取得）は、トリガーされるとすべてのオープンポジションをクローズします。

## オリジナルの EA との主な違い

- StockSharp の **高レベルのローソク足サブスクリプションとインジケーター バインディング**は、MT4 (`SubscribeCandles().Bind(...)`) にある低レベルのポーリングを置き換えます。
- このポートは **オプションのセカンダリ商品**をサポートしています。メイン証券のみを取引するには、GBPUSD / USDCHF / USDJPY スロットを空のままにしておきます。
- MT4 アカウントの証拠金に関連付けられた動的なロットサイジングは、各証券の `VolumeStep`、`MinVolume`、および `MaxVolume` に正規化されたクリーンな `BaseVolume` パラメータに置き換えられました。
- 取引管理は、`OnNewMyTrade` コールバックを通じてエントリー価格を保存し、許可されていないインジケーター値の直接検索を回避します。

## パラメーター

| 名前 | デフォルト | 説明 |
| ---- | ------- | ----------- |
| `CandleType` | `TimeSpan.FromMinutes(15)` | SMMA の計算とエントリのチェックに使用される時間枠。 |
| `RangeCandleType` | `TimeSpan.FromHours(4)` | 適応型 pip ターゲットを導出するために使用されるより高いタイムフレーム。 |
| `SecondSecurity` | `null` | オプションの GBPUSD スロット (開始前に `Security` を設定)。 |
| `ThirdSecurity` | `null` | オプションのUSDCHFスロット。 |
| `FourthSecurity` | `null` | オプションのUSDJPYスロット。 |
| `BaseVolume` | `1` | 取引限度に対して正規化された、注文ごとの要求された取引量。 |
| `TakeProfitPercent` | `0.5` | ポートフォリオ全体のエグジットを引き起こすグローバル株式利益（%）。 |
| `MaxDrawdownPercent` | `30` | すべてのポジションがクローズされるまでの最大許容資本ドローダウン (%)。 |

## 使用上の注意

- 戦略を開始する前に、パラメータによって参照されるすべての証券に同じコネクタとポートフォリオを割り当てます。
- データ ソースがすべての商品の取引時間枠とレンジ時間枠の両方を提供していることを確認してください。
- オプションの証券が提供されていない場合、ベクトル計算は利用可能な商品に自動的に適応します。
- 元の MT4 の動作と一致するように、成行注文では常に終了が発生します。
