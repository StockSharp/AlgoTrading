# Exp HLRSign戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はStockSharpでHLRSignインジケーターロジックを実装しています。
高値-安値比率（HLR）が事前定義されたレベルをクロスするときにポジションを開閉します。

## 動作方法

- 設定可能な範囲でドンチャンチャネルの値を計算します。
- チャネル内での中間価格の割合位置としてHLR値を計算します。
- 選択したモードに応じてHLRが上限または下限の閾値をクロスすると買いまたは売りシグナルを生成します：
  - **ModeIn** – 上限レベルを上方クロスで買い、下限レベルを下方クロスで売り。
  - **ModeOut** – 上限レベルを下方クロスで売り、下限レベルを上方クロスで買い。
- ロングおよびショートポジションの開閉を個別に有効/無効にできます。

## パラメーター

| 名前 | 説明 |
| --- | --- |
| `Mode` | インジケーターの動作モード（ModeIn または ModeOut）。 |
| `Range` | 最高値と最安値のルックバック期間。 |
| `UpLevel` | HLRの上限閾値（パーセント）。 |
| `DnLevel` | HLRの下限閾値（パーセント）。 |
| `CandleType` | 計算に使用するローソク足の時間軸。 |
| `BuyOpen` | ロングポジションの開設を許可する。 |
| `SellOpen` | ショートポジションの開設を許可する。 |
| `BuyClose` | ロングポジションの決済を許可する。 |
| `SellClose` | ショートポジションの決済を許可する。 |

## 注意事項

- この戦略は`DonchianChannels`インジケーターを使用した高レベルAPIを使います。
- 確定済みローソク足のみを処理し、取引前にポジション権限を確認します。
- ストップロスやテイクプロフィットのレベルは定義されていません。ポジション保護は手動で追加できます。
