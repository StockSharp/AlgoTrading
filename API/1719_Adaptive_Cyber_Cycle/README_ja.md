# アダプティブ Cyber Cycle 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はジョン・エーラーズによるAdaptive Cyber Cycleオシレーターを利用します。平滑化された価格サイクルを計算し、前の値をトリガーラインとして使用します。サイクルがトリガーラインを上抜けするとロングポジションを開き、下抜けするとショートポジションを開きます。

## 詳細

- **エントリー条件**:
  - **ロング**: サイクル > 前のサイクル。
  - **ショート**: サイクル < 前のサイクル。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 反対のシグナルがポジションをクローズし、反転させる。
- **ストップ**: デフォルトではなし。保護は別途有効化可能。
- **デフォルト値**:
  - `Alpha` = 0.07
  - `Candle Type` = 1分足の時間軸
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Adaptive Cyber Cycle
  - ストップ: オプション
  - 複雑さ: 中程度
  - 時間軸: イントラデイ
