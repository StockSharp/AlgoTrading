# 三本足下方リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
強気版の鏡像で、このセットアップは素早い弱気リバーサルを狙います。新高値を押し上げる2本の強い上昇ロウソクの後、決定的な弱気ロウソクが直前のバーの安値を下回って引けます。直前の短い上昇トレンドが買い手の枯渇を確認するのに役立ちます。

テストでは年平均リターン約88%を示しています。株式市場で最もよく機能します。

アルゴリズムは3本のロウソクのローリングウィンドウを追跡します。パターンが現れ、上昇トレンドの要件が満たされると、パターンの高値の上にストップを置いてショートポジションが取られます。ルールは簡単なので、シグナルはロウソクの引けと同時に発生します。

トレードは保護的なストップで、または別のパターンが形成されたときに終了します。潜在的な下降スイング内での短期的な押しを狙うため、ボラティリティの高い市場で最もよく機能します。

## 詳細

- **エントリー条件**: より高い高値を伴う2本の強気ロウソク、その後中央バーの安値を下回って引ける弱気ロウソク。
- **ロング/ショート**: ショートのみ。
- **エグジット条件**: ストップロスまたは次のパターン。
- **ストップ**: はい、パターンの高値の上。
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLossPercent` = 1
  - `RequireUptrend` = true
  - `UptrendLength` = 5
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ショート
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
