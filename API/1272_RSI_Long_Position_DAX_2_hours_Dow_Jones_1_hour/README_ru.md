# Стратегия RSI Long Position
[English](README.md) | [中文](README_cn.md)

RSI Long Position покупает, когда RSI пересекает снизу уровень перепроданности, и закрывает позицию, когда RSI превышает уровень фиксации прибыли или опускается ниже уровня выхода.

## Детали

- **Критерий входа**: RSI пересекает сверху `Oversold`
- **Длин/Шорт**: только длинная
- **Критерий выхода**: RSI выше `TakeProfit` или RSI пересекает снизу `StopLoss`
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `Oversold` = 35
  - `TakeProfit` = 55
  - `StopLoss` = 30
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Длинная
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
