# IU ギャップフィル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

IU Gap Fill戦略は、価格が前のセッションの終値からギャップを形成し、そのギャップを埋めるときに取引に入ります。ギャップアップ後に価格が前の終値を下回り、再びその上で終値をつけた場合にロングポジションを開きます。ギャップダウン後に価格が前の終値を上回り、再びその下で終値をつけた場合にショートポジションを開きます。ATRベースのトレーリングストップが出口を管理します。

## 詳細
- **データ**: ユーザー定義の時間軸のローソク足。
- **エントリー条件**:
  - **ロング**: 少なくとも `GapPercent` のギャップアップで価格が前のセッション終値を上抜け。
  - **ショート**: 少なくとも `GapPercent` のギャップダウンで価格が前のセッション終値を下抜け。
- **エグジット条件**: ATRトレーリングストップ。
- **ストップ**: ATR `AtrLength` * `AtrFactor` トレーリングレベル。
- **デフォルト値**:
  - `CandleType` = 1m
  - `GapPercent` = 0.2
  - `AtrLength` = 14
  - `AtrFactor` = 2
- **フィルター**:
  - カテゴリ: ギャップ
  - 方向: ロング & ショート
  - インジケーター: ATR
  - 複雑さ: 低
  - リスクレベル: 中
