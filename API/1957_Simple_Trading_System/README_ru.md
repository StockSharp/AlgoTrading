# Стратегия Simple Trading System
[English](README.md) | [中文](README_cn.md)

Эта стратегия воспроизводит Simple Trading System из MetaTrader. Она использует скользящую среднюю, сдвинутую на несколько баров, и сравнивает текущую цену закрытия с прошлыми закрытиями, чтобы обнаружить краткосрочные развороты тренда. Сигнал на покупку появляется, когда скользящая средняя ниже своего значения `MaShift` баров назад и цена закрытия находится между закрытиями `MaShift` и `MaPeriod + MaShift` баров назад при медвежьей свече. Сигнал на продажу является зеркальным. В зависимости от параметров стратегия может открывать и/или закрывать длинные и короткие позиции при появлении сигналов. Также можно настроить уровни стоп‑лосса и тейк‑профита.

## Детали

- **Условия входа**:
  - **Лонг**: `MA(t) <= MA(t+MaShift)` && `Close(t) >= Close(t+MaShift)` && `Close(t) <= Close(t+MaPeriod+MaShift)` && `Close(t) < Open(t)`
  - **Шорт**: `MA(t) >= MA(t+MaShift)` && `Close(t) <= Close(t+MaShift)` && `Close(t) >= Close(t+MaPeriod+MaShift)` && `Close(t) > Open(t)`
- **Лонг/Шорт**: обе стороны в зависимости от `BuyPositionOpen` и `SellPositionOpen`.
- **Условия выхода**: противоположный сигнал закрывает позицию, если включены `BuyPositionClose` или `SellPositionClose`.
- **Стопы**: по желанию. `StopLoss` и `TakeProfit` в абсолютных ценовых единицах через `StartProtection`.
- **Значения по умолчанию**:
  - `MaType` = EMA
  - `MaPeriod` = 2
  - `MaShift` = 4
  - `PriceType` = Close
  - `CandleType` = 6‑часовые свечи
  - `TakeProfit` = 2000
  - `StopLoss` = 1000
  - `Volume` = 1
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Обе стороны
  - Индикаторы: Скользящая средняя
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
