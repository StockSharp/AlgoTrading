# Range Filter + ATR (Baixo Drawdown)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina um filtro de intervalo com gestão de risco baseada em ATR. As operações são abertas quando o preço cruza o filtro adaptativo e fechadas usando níveis de stop-loss e take-profit derivados de múltiplos de ATR.
