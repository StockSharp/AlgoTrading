# Range Filter + ATR (Bajo Drawdown)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia combina un filtro de rango con gestión del riesgo basada en ATR. Las operaciones se abren cuando el precio cruza el filtro adaptativo y se cierran utilizando niveles de stop-loss y take-profit derivados de múltiplos de ATR.
