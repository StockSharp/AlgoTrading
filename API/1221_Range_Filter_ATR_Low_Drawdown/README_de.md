# Range Filter + ATR (Niedriger Drawdown)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert einen Bereichsfilter mit ATR-basiertem Risikomanagement. Trades werden eröffnet, wenn der Preis den adaptiven Filter kreuzt, und mit Stop-Loss- und Take-Profit-Niveaus geschlossen, die aus ATR-Vielfachen abgeleitet werden.
