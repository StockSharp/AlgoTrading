# Стратегия скальпинга золота с точными входами
[English](README.md) | [中文](README_cn.md)

Скальпинговая стратегия для золота, использующая фильтр тренда EMA, диапазон RSI и паттерны поглощения.

## Детали

- **Критерии входа**: трендовый фильтр EMA, RSI между 45 и 55 и бычье/медвежье поглощение около EMA50.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: тейк‑профит или стоп‑лосс.
- **Стопы**: стоп‑лосс по ATR и фиксированный пипсовый тейк‑профит.
- **Значения по умолчанию**:
  - `EmaFastPeriod` = 50
  - `EmaSlowPeriod` = 200
  - `RsiPeriod` = 14
  - `AtrPeriod` = 14
  - `RsiLower` = 45
  - `RsiUpper` = 55
  - `PipTarget` = 2
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Скальпинг
  - Направление: Оба
  - Индикаторы: EMA, RSI, ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
