# Funktion ADF
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Das Modul **Function Adf** implementiert den Augmented Dickey-Fuller-Test (ADF) zur Überprüfung der Stationarität von Zeitreihen.

## Details
- **Zweck**: Hilfsmittel für Stationaritätsprüfungen.
- **Indikatoren**: keine.
- **Long/Short**: nicht anwendbar.
- **Stops**: keine.
