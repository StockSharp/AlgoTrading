# Função ADF
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

O módulo **Function Adf** implementa o teste de Dickey-Fuller Aumentado (ADF) para verificar a estacionariedade de séries temporais.

## Detalhes
- **Propósito**: utilitários de teste de estacionariedade.
- **Indicadores**: nenhum.
- **Comprado/Vendido**: não aplicável.
- **Stops**: nenhum.
