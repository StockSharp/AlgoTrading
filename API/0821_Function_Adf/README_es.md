# Función ADF
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

El módulo **Function Adf** implementa el test de Dickey-Fuller Aumentado (ADF) para verificar la estacionariedad de series temporales.

## Detalles
- **Propósito**: utilidades para prueba de estacionariedad.
- **Indicadores**: ninguno.
- **Largo/Corto**: no aplica.
- **Stops**: ninguno.
