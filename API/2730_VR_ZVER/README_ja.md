# VR ZVER戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
VR ZVER戦略は、高速/低速/超低速EMAスタック、ストキャスティックオシレーター、および相対力指数（RSI）の3つの確認レイヤーを組み合わせたトレンドフォロー型システムです。すべてのアクティブなフィルターが一致してから初めてポジションが開かれ、荒れたまたは矛盾したマーケット相場での取引を避けることができます。変換はStockSharpの高レベルAPIを使用しながら、オリジナルのブレイクイーブンと保護ロジックを維持しています。

## マーケットレジーム検出
1. **EMA構造** – デフォルト設定は期間3、5、7の指数移動平均を使用します。ロングバイアスには高速EMAが低速EMAを上回り、低速EMAが超低速EMAを上回り続けることが必要です。ショートバイアスはこの関係を逆にします。
2. **ストキャスティックオシレーター** – %K/%Dペアは方向とレベルの両方について検査されます。ロング取引は%Kが下部バンドを下回り%Dを上回ることを必要とし、売られすぎからのバウンスを示します。ショート取引は%Kが上部バンドを上回り%Dを下回ることを必要とし、買われすぎからの反転を示します。
3. **RSIフィルター** – RSIはロングエントリーを許可するために下部閾値を下回るか、ショート取引を有効にするために上部閾値を上回る必要があります。

有効な各フィルターが整列したときのみ、戦略は設定されたボリュームを使用した成行注文を送信します。

## リスク管理
- **ストップロス** – 各エントリーは`StopLossPips`設定にインストゥルメントのpipサイズを乗算したプライスベースのストップを投影します。ロングポジションはローソク足の安値がストップを突き破るときに出口し、ショートポジションはローソク足の高値がそのストップに達するとクローズします。
- **テイクプロフィット** – 対称的なテイクプロフィットレベルが適用されます。現在のローソク足が取引に有利な方向でターゲットに達すると、ポジションは即座にクローズされます。
- **ブレイクイーブン保護** – 価格が`BreakevenPips`距離を進むと、ブレイクイーブンモードが武装されます。エントリー価格への反落は、資本を保全するためにポジションをフラット化します。
- **注文クリーンアップ** – 意図しないスタッキングを避けるために、新しい取引を開く前にすべてのアクティブな注文がキャンセルされます。

## パラメーター
| パラメーター | 説明 |
|-------------|------|
| `CandleType` | 計算に使用されるローソク足シリーズ。 |
| `UseMovingAverage` | EMAトレンドフィルターを有効または無効にする。 |
| `FastMaPeriod`, `SlowMaPeriod`, `VerySlowMaPeriod` | 高速、低速、超低速EMAsの期間。 |
| `UseStochastic` | ストキャスティック確認レイヤーを切り替える。 |
| `StochasticKPeriod`, `StochasticDPeriod`, `StochasticSlowing` | ストキャスティックオシレーターの期間設定。 |
| `StochasticUpperLevel`, `StochasticLowerLevel` | %Kの買われすぎと売られすぎ閾値。 |
| `UseRsi` | RSI確認レイヤーを有効または無効にする。 |
| `RsiPeriod` | RSI平均化期間。 |
| `RsiUpperLevel`, `RsiLowerLevel` | 買われすぎ/売られすぎ領域を定義するRSI閾値。 |
| `StopLossPips`, `TakeProfitPips` | ストップロスとテイクプロフィット配置の距離（pips単位）。 |
| `BreakevenPips` | ブレイクイーブン保護を有効にする前に必要な価格の進行。 |
| `Volume` | 各成行注文の取引数量。 |

## 実装ノート
- pip サイズはインストゥルメントの価格ステップと小数点以下の桁数から導出されます。3または5桁の小数点以下の桁数を持つインストゥルメントは、オリジナルのMQLバージョンで使用される標準の10倍調整を自動的に適用します。
- すべてのインジケーターデータは`BindEx`を通じてアクセスされ、戦略が最終化されたインジケーター値を持つ完了したローソク足にのみ反応することを保証します。
- 戦略はデフォルトでフラットです；既存のエクスポージャーを最初にクローズせずにポジションを反転させることは決してありません。
