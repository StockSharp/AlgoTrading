# CCI Breakout
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
CCI（商品チャネル指数）ブレイクアウトに基づく戦略

テストでは年平均リターンが約94%であることが示されています。株式市場で最もよく機能します。

CCI Breakoutは商品チャネル指数を使用して強力な動きを見つけます。CCIの正または負の閾値を超える急増がエントリーを生成します。CCIがゼロに向かって後退するか、逆のシグナルが形成されると退場します。

CCIは移動平均からの乖離を測定するため、極端な読みは持続不可能な価格を意味します。このシステムはそれらの極値を待ち、その後の動きから利益を得ようとします。


## 詳細

- **エントリー条件**: CCI、Momentumに基づくシグナル。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 逆シグナルまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `CciPeriod` = 20
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: CCI、Momentum
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - Neural Networks: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
