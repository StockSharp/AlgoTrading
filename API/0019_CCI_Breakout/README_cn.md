# CCI 突破
[English](README.md) | [Русский](README_ru.md)

该策略利用CCI（商品通道指数）捕捉强劲走势。当CCI突破正负阈值时入场，CCI回落至零附近或出现反向信号时离场。CCI衡量价格相对均线的偏离，极端读数意味着价格可能不可持续，本系统等待这些极端后跟随行情。

## 详情
- **入场条件**: 基于 CCI、动量 的信号
- **多空方向**: 双向
- **退出条件**: 反向信号或止损
- **止损**: 是
- **默认值**:
  - `CciPeriod` = 20
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类型: 突破
  - 方向: 双向
  - 指标: CCI, Momentum
  - 止损: 是
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中

测试表明年均收益约为 94%，该策略在股票市场表现最佳。
