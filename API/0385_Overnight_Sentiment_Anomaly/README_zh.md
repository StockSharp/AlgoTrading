# 隔夜情绪异常策略

[English](README.md) | [Русский](README_ru.md)

该策略仅在外部情绪指标显示极度乐观时隔夜持有股票 ETF。如果指标值高于阈值，则在收盘买入，次日开盘卖出，利用正向情绪下的隔夜偏离。

策略不使用监監数据，只根据收盘时的情绪值在收盘和次日开盘下市价单。

## 细节

- **标的**：股票 ETF 及情绪数据序列。
- **信号**：情绪值高于 `Threshold`。
- **持有期**：收盘到次日开盘。
- **仓位**：情绪达标时做多，否则空仓。
- **风险控制**：如果交易金额低于 `MinTradeUsd` 则不下单。
