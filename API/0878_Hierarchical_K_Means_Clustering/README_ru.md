# Стратегия иерархической и K-Means кластеризации
[English](README.md) | [中文](README_cn.md)

Стратегия использует кластеризацию волатильности в системе SuperTrend. Значения ATR делятся на три кластера для определения режима рынка, а разворот SuperTrend служит сигналом входа. Дополнительные фильтры на основе скользящей средней и ADX подтверждают силу тренда. Позиции могут закрываться досрочно, когда отношение бычьего и медвежьего объёма стремится к балансу.

## Детали

- **Условия входа**:
  - **Лонг**: SuperTrend становится бычьим && тренд кластера > 0 && фильтры выполнены.
  - **Шорт**: SuperTrend становится медвежьим && тренд кластера < 0 && фильтры выполнены.
- **Стороны**: Лонг и шорт.
- **Условия выхода**:
  - Баланс объёмов или противоположный сигнал.
- **Стопы**: Только на основе объёмов.
- **Значения по умолчанию**:
  - `ATR Length` = 11.
  - `SuperTrend Factor` = 3.
  - `Training Data Length` = 200.
  - `Moving Average Length` = 50.
  - `Trend Strength Period` = 14.
  - `Trend Strength Threshold` = 20.
  - `Volume Ratio Threshold` = 0.9.
  - `Delay Bars` = 4.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: Несколько
  - Стопы: Да
  - Сложность: Сложная
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
