# 威廉%R 背离策略

威廉%R 振荡指标衡量超买和超卖状态。当价格创出新低而 %R 形成更高的低点，或价格创新高而 %R 转弱时，动能可能出现反转。本策略在指标极端区间寻找这样的背离。

系统在每个K线记录最新收盘价和 %R 值，与之前读数比较。当 %R 低于 -80 且出现看涨背离时做多；当 %R 高于 -20 并出现看跌背离时做空。止损按价格的百分比设置。

当振荡指标回到相反的极端区域时平仓，捕捉背离信号后的快速回撤。

## 细节

- **入场条件**：价格与 %R 背离，且做多时 %R 低于 -80，做空时高于 -20。
- **多/空**：双向。
- **退出条件**：威廉%R 达到相反极端或止损。
- **止损**：是，按百分比。
- **默认值**：
  - `WilliamsRPeriod` = 14
  - `DivergencePeriod` = 5
  - `CandleType` = 5 分钟
  - `StopLossPercent` = 2
- **过滤条件**：
  - 类别: 背离
  - 方向: 双向
  - 指标: 威廉%R
  - 止损: 有
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 是
  - 风险级别: 中等
