# RSI付きトレーリングストップ - モメンタムベース戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はモメンタムのブレイクアウトでエントリーし、RSIフィルターを使って買われすぎ・売られすぎレベルでポジションを反転させます。設定可能な利益に達するとトレーリングストップが有効になり、価格に追従して利益を保護します。

## 詳細

- **エントリー**: モメンタムとその変化が同じ方向に揃う。
- **エグジット**: RSIレベルまたはトレーリングストップ。
- **インジケーター**: Momentum, RSI。
- **時間軸**: 任意。
- **リスク**: トレーリングストップを使用。
