# Trailing Stop con RSI - Estrategia Basada en Momentum
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia entra en rupturas de momentum y utiliza un filtro RSI para revertir posiciones en niveles de sobrecompra o sobreventa. Un trailing stop se activa después de un beneficio configurable y sigue al precio para proteger las ganancias.

## Detalles

- **Entrada**: El momentum y su cambio se alinean en la misma dirección.
- **Salida**: Niveles de RSI o trailing stop.
- **Indicadores**: Momentum, RSI.
- **Marco temporal**: Cualquiera.
- **Riesgo**: Utiliza trailing stop.
