# Trailing Stop com RSI - Estratégia Baseada em Momentum
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia entra em rompimentos de momentum e usa um filtro RSI para reverter posições em níveis de sobrecompra ou sobrevenda. Um trailing stop é ativado após um lucro configurável e acompanha o preço para proteger os ganhos.

## Detalhes

- **Entrada**: O momentum e sua variação se alinham na mesma direção.
- **Saída**: Níveis de RSI ou trailing stop.
- **Indicadores**: Momentum, RSI.
- **Período**: Qualquer.
- **Risco**: Utiliza trailing stop.
