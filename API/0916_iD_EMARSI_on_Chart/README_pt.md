# Estratégia iD EMARSI no Gráfico
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera cruzamentos entre o RSI traçado no gráfico de preços e sua média móvel exponencial. Uma posição comprada é aberta quando o RSI cruza acima da EMA, e uma posição vendida no cruzamento oposto. Os sinais podem ser filtrados pela duração mínima de tendência e as posições são protegidas com take profit opcional e trailing stop em percentual.

## Parâmetros
- Tipo de vela
- Comprimento do RSI
- Comprimento da EMA
- Barras mínimas de tendência
- Usar sinais filtrados
- Take Profit %
- Trailing Stop %
