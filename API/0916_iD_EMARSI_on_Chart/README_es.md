# Estrategia iD EMARSI en Gráfico
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia opera cruces entre el RSI trazado en el gráfico de precios y su media móvil exponencial. Se abre una posición larga cuando el RSI cruza por encima de la EMA, y una posición corta en el cruce contrario. Las señales pueden filtrarse por duración mínima de tendencia y las posiciones se protegen con take profit opcional y stop trailing en porcentaje.

## Parámetros
- Tipo de vela
- Longitud del RSI
- Longitud de la EMA
- Barras mínimas de tendencia
- Usar señales filtradas
- Take Profit %
- Trailing Stop %
