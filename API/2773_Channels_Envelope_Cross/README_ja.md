# チャネル・エンベロープ・クロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

チャネル・エンベロープ・クロス戦略は、MetaTrader のエキスパートアドバイザー「Channels」の直接移植です。システムは時間足のローソク足を取引し、220 期間の遅い EMA から計算されたEMAベースの 3 つのエンベロープ（0.3%、0.7%、1.0% の偏差）に対する 2 期間の速い指数移動平均（EMA）を監視します。速い EMA がこれらのエンベロープを突破すると方向性のあるエントリーが生成され、オプションの時間フィルターで特定の時間帯に取引を制限できます。

## 取引ロジック

1. **インジケータースタック**
   - ローソク足の終値で計算した速い EMA（長さ 2）。
   - ローソク足の始値で計算した速い EMA（長さ 2）。
   - ローソク足の終値で計算した遅い EMA（長さ 220）。
   - 0.3%、0.7%、1.0% の偏差で遅い EMA から導出した 3 つのエンベロープレベル。
2. **ロングセットアップ**
   - 速いクローズ EMA が 1.0% または 0.7% の下側エンベロープを上方に突破したとき、2 本連続のバーで 0.3% の下側エンベロープを下回ったとき、遅い EMA を上方に突破したとき、または 0.3% もしくは 0.7% の上側エンベロープを突破したとき。これらの条件のいずれかが、ポジションが開いていないときにロングエントリーを発動できます。
3. **ショートセットアップ**
   - 速いオープン EMA が上側エンベロープのいずれかを下方に突破したとき、遅い EMA を下回ったとき、または上から下側エンベロープを突き抜けたとき。これらの条件のいずれかが、ポジションが開いていないときにショートエントリーを発動できます。
4. **リスク管理**
   - 固定のストップロスとテイクプロフィットのレベル（側ごと）は pips で表され、インストゥルメントのティックサイズを使用して価格距離に変換されます。入力がゼロに設定されると、それぞれのレベルは適用されません。
   - ロングおよびショートポジションの独立したトレーリングストップは、利益がトレーリング距離と設定可能なステップ増分を超えると、保護ストップを市場価格に近づけます。
5. **時間フィルター**
   - 有効にすると、戦略は設定された包括的な時間範囲内のエントリーのみを処理します。フィルターがアクティブなときでもポジションは管理されます。

## パラメーター

| パラメーター | 説明 |
|------------|------|
| `OrderVolume` | 成行エントリーに使用される注文サイズ（証券によってロットまたは契約数）。 |
| `UseTradeHours` | エントリーの時間フィルターを有効にします。 |
| `FromHour` / `ToHour` | トレーディングウィンドウの開始時間と終了時間（夜間の範囲をサポート）。 |
| `StopLossBuyPips` / `StopLossSellPips` | ロング/ショートトレードの pips で表されたストップロス距離。 |
| `TakeProfitBuyPips` / `TakeProfitSellPips` | ロング/ショートトレードの pips で表されたテイクプロフィット距離。 |
| `TrailingStopBuyPips` / `TrailingStopSellPips` | ロング/ショートトレードの pips 単位のトレーリングストップ距離。 |
| `TrailingStepPips` | トレーリングストップを移動するために必要な最小増分（pips）。 |
| `CandleType` | 計算に使用されるローソク足シリーズ（デフォルトは 1 時間足の時間軸）。 |

## ポジション管理

- エントリー時に戦略は約定価格を記録し、絶対価格単位でストップロスとテイクプロフィットターゲットを計算し、トレーリングレベルをリセットします。
- ロングポジションが開いている間、利益が `TrailingStopBuyPips + TrailingStepPips` を超えるたびにストップロスが上方に追従します。戦略はストップロスまたはテイクプロフィットのいずれか先に当たった方で出口します。
- ショートポジションが開いている間、ストップロスはショート側のトレーリングパラメーターを使用して下方に追従し、出口は対称的に実行されます。

## 注意事項

- pip サイズはインストゥルメントのティックサイズから導出されます。小数点以下 3 桁または 5 桁の商品の場合、MetaTrader のロジックをエミュレートするために pip が 10 倍されます。
- 戦略は一度に 1 つのポジションで動作します。既存のポジションが閉じられた後にのみ新しいエントリーが配置されます。
- 予期しない未決済ポジションから保護するために、基底クラスの `StartProtection` を有効にしてください（実装内ですでに呼び出されています）。
