# R&D支出戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このクロスセクション戦略は、研究開発（R&D）費用と時価総額の比率で銘柄をランク付けします。毎月初め、R&D集約度が最も高い上位20%の企業を買い、最下位20%を空売りし、積極的なR&D支出が将来のアウトパフォームを予測するという仮説に賭けます。

各サイドで等ウェイトが割り当てられ、日次価格データを使用して毎月リバランスされます。

## 詳細

- **ユニバース**: R&Dデータを持つ銘柄のリスト。
- **シグナル**: R&D支出を時価総額で割った値。
- **ポートフォリオ**: 最上位クインタイルをロング、最下位クインタイルをショート。
- **リバランス**: 毎月。
- **リスク管理**: 注文価値が `MinTradeUsd` を下回る場合は取引をスキップ。
