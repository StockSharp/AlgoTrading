# Hawaiian Tsunami Surfer
[English](README.md) | [中文](README_cn.md)

Стратегия ищет внезапные всплески момента и торгует против них. Она вычисляет процентное изменение цены закрытия за один бар с помощью индикатора Momentum. Когда изменение превышает небольшой порог, движение считается «цунами». Стратегия продаёт после сильного роста и покупает после сильного падения. Защита в виде стоп-лосса и тейк-профита задаётся в шагах цены через StartProtection.

## Подробности

- **Условия входа**:
  - Продажа, если процентное изменение момента > `TsunamiStrength`.
  - Покупка, если процентное изменение момента < `-TsunamiStrength`.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Стоп-лосс или тейк-профит.
- **Стопы**: Да, через StartProtection.
- **Значения по умолчанию**:
  - `MomentumPeriod` = 1
  - `TsunamiStrength` = 0.24
  - `TakeProfitPoints` = 500
  - `StopLossPoints` = 700
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: Оба
  - Индикаторы: Momentum
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
