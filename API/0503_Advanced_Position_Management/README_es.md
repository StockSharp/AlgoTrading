# Estrategia Avanzada de Gestión de Posiciones
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia entra en el cruce de EMA y gestiona las posiciones mediante tres niveles de take-profit y un trailing stop opcional.

## Detalles

- **Criterios de entrada**: Cruce de la EMA rápida sobre la EMA lenta.
- **Largo/Corto**: Ambos.
- **Criterios de salida**: Stop-loss, niveles de take-profit o trailing stop.
- **Stops**: Inicial y trailing.
