# Estratégia Avançada de Gerenciamento de Posições
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia entra no cruzamento de EMA e gerencia posições usando três níveis de take-profit e um trailing stop opcional.

## Detalhes

- **Critérios de entrada**: EMA rápida cruzando a EMA lenta.
- **Comprado/Vendido**: Ambos.
- **Critérios de saída**: Stop-loss, níveis de take-profit ou trailing stop.
- **Stops**: Inicial e trailing.
