# Fortgeschrittene Positionsverwaltungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie steigt beim EMA-Kreuzungspunkt ein und verwaltet Positionen mit drei Take-Profit-Niveaus und einem optionalen Trailing-Stop.

## Details

- **Einstiegskriterien**: Schneller EMA kreuzt langsamen EMA.
- **Long/Short**: Beide.
- **Ausstiegskriterien**: Stop-Loss, Take-Profit-Niveaus oder Trailing-Stop.
- **Stops**: Initial und Trailing.
