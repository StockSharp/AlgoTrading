# ブレイクアウト・バー・トレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はParabolic SARインジケーターを使用してトレンドの反転を検出します。新しいトレンド方向へエントリーする前に、設定可能な数の負のリバーサルを待ちます。ストップロスとテイクプロフィットの距離は、pipsまたはエントリー価格の割合で測定されます。

## パラメーター

- **Reversal Mode** – pip基準またはパーセント基準の距離計算を選択します。
- **Delta** – リバーサル間に必要な最小価格変動。
- **Negative Signals** – 取引を開く前に発生しなければならない失敗したリバーサルの数。
- **Stop Loss** – エントリー価格からの損失保護距離。
- **Take Profit** – エントリー価格からの利益目標距離。
- **Candle Type** – インジケーター計算に使用するローソク足シリーズ。

## ロジック

1. ローソク足データを購読し、Parabolic SARを計算します。
2. Parabolic SARが方向を反転し、価格が少なくとも*Delta*動いた場合、リバーサル価格を記録します。
3. 価格が前のトレンドに逆らって動いた負のリバーサルをカウントします。
4. カウンターが**Negative Signals**の値に達したら、新しいトレンド方向にポジションを開きます。
5. 各ローソク足は選択した**Reversal Mode**を使用してストップロスとテイクプロフィットのレベルを確認します。
6. 逆のトレンド変化またはリスク限界に達した場合にポジションをクローズします。

この戦略はトレンドフォローのブレイクアウトシステムに適しており、デルタ、ストップロス、テイクプロフィットの距離を調整することで最適化できます。
