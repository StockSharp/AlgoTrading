# Стратегия Balance of Power
[English](README.md) | [中文](README_cn.md)

Стратегия Balance of Power измеряет силу покупателей и продавцов внутри каждой свечи, сравнивая цену закрытия с диапазоном. Когда значение пересекает положительный порог, это указывает на сильное давление покупателей.

Стратегия открывает длинную позицию, когда Balance of Power пересекает значение `Threshold` сверху вниз и закрывает её, когда индикатор опускается ниже отрицательного порога.

## Детали

- **Критерии входа**:
  - Balance of Power пересекает уровень `Threshold` снизу вверх.
- **Длинные/Короткие**: Только длинные.
- **Критерии выхода**:
  - Balance of Power опускается ниже `-Threshold`.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Threshold` = 0.8
- **Фильтры**:
  - Категория: Моментум
  - Направление: Лонг
  - Индикаторы: Balance of Power
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
