# MACDアグレッシブスキャルプシンプル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

50期間EMAフィルターを使用したMACDヒストグラムによるスキャルピング戦略を実装します。

- MACDヒストグラムがゼロを上抜けし、価格がEMAより上にあるときにロングエントリー。
- ヒストグラムがゼロを下抜けし、価格がEMAより下にあるときにショートエントリー。
- ヒストグラムのモメンタムが反転したときにポジションを決済。
