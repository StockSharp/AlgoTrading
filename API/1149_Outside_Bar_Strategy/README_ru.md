# Стратегия внешнего бара
[English](README.md) | [中文](README_zh.md)

Стратегия торгует пробои внешних баров. Бычий внешний бар определяется как свеча, чей максимум выше и минимум ниже предыдущей. Заявки выставляются внутри бара. Поддерживаются частичное фиксирование прибыли и перевод стопа в безубыток.

## Детали

- **Критерий входа**: Внешний бар с классификацией на бычий или медвежий.
- **Лонг/Шорт**: Оба направления.
- **Критерий выхода**: Стоп-лосс или тейк-профит, рассчитанные от диапазона бара.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `CandleType` = 5 минут
  - `EntryPercentage` = 0.5
  - `TpPercentage` = 1
  - `PartialRR` = 1
  - `PartialExitPercent` = 0.5
  - `StopLossOffset` = 10
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Свечи
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
