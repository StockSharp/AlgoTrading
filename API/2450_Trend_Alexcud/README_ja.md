# トレンド Alexcud
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Trend Alexcud戦略は、3つの時間軸にわたって複数の単純移動平均とAccelerator Oscillatorを整合させることで、強い方向性のある動きを探します。オリジナルのMQL5エキスパート「TREND_alexcud v_2」から変換されました。

システムは3つの時間軸（デフォルト15分、1時間、4時間）を監視します。各時間軸で5本の単純移動平均（期間5、8、13、21、34）とAccelerator Oscillatorを計算します。終値がすべての移動平均を上回り、Acceleratorが正の場合、その時間軸は強気とみなされます。終値がすべての移動平均を下回り、Acceleratorが負の場合、その時間軸は弱気とみなされます。

3つの時間軸が一致した場合にのみトレードが開かれます。同時に強気であれば戦略は買い、共通の弱気読みは売りをトリガーします。逆シグナルが現れるたびにポジションが反転されます。保護注文はStockSharpの組み込みリスクシステムを通じて管理されます。

## 詳細

- **エントリー条件**
  - **ロング**: 各時間軸でのすべてのMAを上回る価格かつAccelerator > 0。
  - **ショート**: 各時間軸でのすべてのMAを下回る価格かつAccelerator < 0。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 逆シグナルが形成されるとポジションが反転する。
- **ストップ**: 組み込み保護を使用（デフォルト値なし）。
- **デフォルト値**:
  - Timeframe1 = 15m, Timeframe2 = 1h, Timeframe3 = 4h
  - MA期間 = 5, 8, 13, 21, 34
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: マルチタイムフレーム
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
