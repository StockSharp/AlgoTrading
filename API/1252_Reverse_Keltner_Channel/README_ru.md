# Обратный канал Келтнера
[English](README.md) | [中文](README_cn.md)

Стратегия входит при возвращении цены в канал Келтнера снаружи и стремится к противоположной границе, с необязательным фильтром ADX.

Длинная позиция открывается, когда цена пересекает нижнюю границу снизу и закрывается на верхней границе или по стопу на половине ширины канала. Короткие сделки симметричны. Фильтр ADX позволяет торговать только при слабом или сильном тренде.

## Детали

- **Критерии входа**: пересечение внешней границы Keltner внутрь, опциональный фильтр ADX.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположная граница или стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `EmaPeriod` = 20
  - `AtrPeriod` = 10
  - `AtrMultiplier` = 2m
  - `StopLossFactor` = 0.5m
  - `AdxLength` = 14
  - `AdxThreshold` = 25m
  - `UseAdxFilter` = true
  - `WeakTrendOnly` = true
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: Keltner, ADX
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
