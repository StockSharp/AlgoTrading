# JMAローソク足シグナル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、各ローソク足の始値と終値に基づいて計算された2本のJurik移動平均（JMA）を使用します。始値のJMAが終値のJMAを下から上に突き抜けたときに強気シグナルが発生し、ロングエントリーが行われます。始値のJMAが終値のJMAを上から下に突き抜けたときに弱気シグナルが発生し、ショートエントリーが行われます。

デフォルトの時間軸は4時間足で、JMAの期間は7です。ストップロスとテイクプロフィットのレベルはポイントで定義され、組み込みのリスク管理を通じて適用されます。戦略は確定済みのローソク足のみに作用し、最大1つのオープンポジションを維持します。

## パラメーター
- **JMA Length** – 両JMAの期間。
- **Candle Type** – 処理するローソク足の時間軸。
- **Take Profit** – 利益目標（ポイント）。
- **Stop Loss** – 最大損失（ポイント）。
