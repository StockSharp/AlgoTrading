# ボラティリティ・パルス・ダイナミック・エグジット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ボラティリティ拡大を検知するモメンタムベースの戦略です。ATRが平均を上回ったときにモメンタムの方向にエントリーし、保有期間後にATRベースのストップとテイクプロフィットを使ってエグジットします。

## 詳細

- **エントリー条件**: ATRボラティリティ拡大とモメンタム確認
- **ロング/ショート**: 両方
- **エグジット条件**: 保有期間後のストップロスとテイクプロフィット
- **ストップ**: ATRベースのストップ、リスク・リワード比によるテイクプロフィット
- **デフォルト値**:
  - `AtrLength` = 14
  - `MomentumLength` = 20
  - `VolThreshold` = 0.5
  - `MinVolatility` = 1.0
  - `ExitBars` = 42
  - `RiskReward` = 2
- **フィルター**:
  - カテゴリ: ボラティリティ
  - 方向: 両方
  - インジケーター: ATR, SMA, Momentum
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
