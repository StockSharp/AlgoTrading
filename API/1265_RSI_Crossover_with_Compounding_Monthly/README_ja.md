# RSI クロスオーバー複利戦略（月次）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

月次 RSI がその SMA を上回って終値を迎えたときに全資金を投資し、RSI が SMA を下回ったときにエグジットします。利益は複利のために資金に加算されます。

バックテストでは年間平均リターン約 20% が示されています。株式に対して最もよく機能します。

## 詳細

- **エントリー条件**: RSI が SMA を上回る
- **ロング/ショート**: ロング
- **エグジット条件**: RSI が SMA を下回る
- **ストップ**: いいえ
- **デフォルト値**:
  - `CandleType` = 1ヶ月
  - `RsiPeriod` = 14
  - `InitialCapital` = 100000
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロングのみ
  - インジケーター: RSI, SMA
  - ストップ: いいえ
  - 複雑さ: 初心者
  - 時間軸: 月次
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
