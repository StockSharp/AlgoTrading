# Estrategia Donchian Zig-Zag LuxAlgo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Recrea el indicador Donchian Zig-Zag de LuxAlgo. La estrategia invierte la dirección cuando el valor anterior del zigzag cruza la banda superior o inferior de Donchian. Se abre una operación larga cuando el oscilador se vuelve positivo tras alcanzar la banda inferior. Se abre una operación corta cuando el oscilador se vuelve negativo tras tocar la banda superior.

## Parámetros
- Tipo de vela
- Longitud de Donchian
- Velocidad de rebote
