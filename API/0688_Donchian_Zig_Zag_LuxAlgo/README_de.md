# Donchian Zig-Zag LuxAlgo-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Nachbildung des Donchian Zig-Zag-Indikators von LuxAlgo. Die Strategie kehrt die Richtung um, wenn der vorherige Zickzack-Wert das obere oder untere Band des Donchian-Kanals kreuzt. Ein Long-Trade wird eröffnet, wenn der Oszillator positiv wird, nachdem der Preis das untere Band erreicht hat. Ein Short-Trade wird eröffnet, wenn der Oszillator negativ wird, nachdem der Preis das obere Band berührt hat.

## Parameter
- Kerzentyp
- Donchian-Länge
- Rückprallgeschwindigkeit
