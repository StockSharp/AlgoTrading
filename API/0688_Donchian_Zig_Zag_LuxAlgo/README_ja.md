# Donchian Zig-Zag LuxAlgo 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

LuxAlgoのDonchian Zig-Zagインジケーターを再現します。前のジグザグ値がDonchianの上限または下限バンドをクロスしたときに方向を反転させます。価格が下限バンドに到達した後にオシレーターがプラスに転じたときにロングトレードを開きます。価格が上限バンドに触れた後にオシレーターがマイナスに転じたときにショートトレードを開きます。

## パラメーター
- ローソク足タイプ
- Donchian長さ
- バウンス速度
