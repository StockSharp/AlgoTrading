# Estratégia Donchian Zig-Zag LuxAlgo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Recria o indicador Donchian Zig-Zag da LuxAlgo. A estratégia inverte a direção quando o valor anterior do zigzag cruza a banda superior ou inferior do Donchian. Uma operação comprada é aberta quando o oscilador se torna positivo após o preço atingir a banda inferior. Uma operação vendida é aberta quando o oscilador se torna negativo após o preço tocar a banda superior.

## Parâmetros
- Tipo de vela
- Comprimento do Donchian
- Velocidade de rebote
