# UltraFATL閾値戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はUltraFATLオシレーターを使用してトレンド強度の変化を検出します。インジケーターは0から8の離散レベルを出力します。前の値がレベル4を超え、現在の値が5を下回りながら正の値を維持している場合にロングポジションが開かれます。前の値が5を下回るがゼロより大きく、現在の値が4を超えて上昇した場合にショートポジションが開かれます。アルゴリズムはデフォルトで4時間ローソク足で動作しますが、時間軸は調整可能です。

このアプローチは、UltraFATLの極端な読み値からの反落後のトレンド継続を期待します。反対の条件が現れるとポジションは反転されます。

## 詳細

- **エントリー条件**:
  - **ロング**: `UltraFATL(prev) > 4` かつ `UltraFATL(curr) < 5` かつ `UltraFATL(curr) != 0`。
  - **ショート**: `UltraFATL(prev) < 5` かつ `UltraFATL(prev) != 0` かつ `UltraFATL(curr) > 4`。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対シグナルでポジションを反転。
- **ストップ**: デフォルトでは使用しない。
- **デフォルト値**:
  - `Candle Type` = 4時間ローソク足。
  - `Length` = 3。
  - `Signal Bar` = 1（シグナルに前のバーを使用）。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一 (UltraFATL)
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
