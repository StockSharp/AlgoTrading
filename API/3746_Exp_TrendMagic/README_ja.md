# 経験値トレンドマジック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
Exp TrendMagic 戦略は、MetaTrader 5 エキスパート アドバイザー「Exp_TrendMagic」を直接移植したものです。これは、コモディティ チャネル インデックス (CCI) と平均トゥルー レンジ (ATR) チャネルを組み合わせた TrendMagic インジケーターの色の変化を監視します。インジケーターの色が切り替わると、戦略は反対側からポジションを閉じ、オプションで新しいトレンドの方向に新しい取引を開始します。

この変換では、元の資金管理オプション、設定可能なシグナル オフセット (`Signal Bar`)、およびロング取引とショート取引の開始または終了のための同じ許可の切り替えが維持されます。

## 取引ロジック
1. **インジケータ入力**
   - `CCI` (コモディティ チャネル インデックス) を構成可能な期間と適用価格で指定します。
   - `ATR` (平均 True Range)、期間を設定可能。
   - TrendMagic の値は次のように計算されます。
     - CCI ≥ 0 の場合: `TrendMagic = Low - ATR`、サポート ラインの減少を避けるためにクランプされます。
     - CCI < 0 の場合: `TrendMagic = High + ATR`、抵抗線の増加を避けるためにクランプされます。
   - 結果として得られる線の色は、強気 (価格を下回るサポート) の場合は **0**、弱気 (価格を上回る抵抗) の場合は **1** になります。

2. **信号評価**
   - この戦略は、インジケーターの色を時系列で保存して MetaTrader バッファーをエミュレートし、`Signal Bar` オフセットを使用して最新の完成したバーを読み取ります。
   - 以前の色 (`Signal Bar + 1`) が **0** で、現在の色 (`Signal Bar`) が **1** の場合、アルゴリズムはこれを強気スイッチとして扱います。ショート ポジションをすべて閉じ、許可されている場合はロング取引を開始します。
   - 以前の色が **1** で、現在の色が **0** の場合、アルゴリズムはオープンしているロング ポジションをすべて閉じ、許可されている場合はショート取引を開始します。
   - 取引許可フラグ (`Allow Buy Entry`、`Allow Sell Entry`、`Allow Buy Exit`、`Allow Sell Exit`) は、MT5 バージョンのセマンティクスに正確に従います。

3. **お金の管理**
   - `Money Management` は、取引ごとに割り当てられる資金の量を決定します。負の値は固定ロットサイズとして解釈されます。
   - `Margin Mode` は、資金管理値の解釈を選択します。
     - `FreeMargin` / `Balance`: 口座資本のシェアを価格で割ったものを投資します。
     - `LossFreeMargin` / `LossBalance`: ストップロス距離に応じた資本シェアのリスク。
     - `Lot`: 値を固定量として扱います。
   - ボリュームは、選択した証券の `VolumeStep`、`MinVolume`、および `MaxVolume` と調整されます。

4. **リスク管理**
   - 新しい取引が行われると、ストラテジーはエントリー価格を記録し、元のストップロスとテイクプロフィットの距離（ポイント、つまり `PriceStep` の倍数で表現）を適用します。
   - ストップロスまたはテイクプロフィットに達すると、即座にポジションが決済され、保存されたエントリー価格がクリアされます。
   - スロットルは、次のローソク足が開く前に同じ方向の位置を再度開くことを防ぎ、MQL の「時間レベル」ガードを再現します。

## パラメーター
| パラメータ | 説明 |
|-----------|-------------|
| `Money Management` | サイジングに使用される資本の割合 (負の値は固定量になります)。 |
| `Margin Mode` | 資金管理をボリュームに変換するモード。 |
| `Stop Loss` | 価格帯における保護停止距離。 |
| `Take Profit` | 価格帯での利益目標。 |
| `Deviation` | MT5 入力との互換性のために予約されています (直接使用されません)。 |
| `Allow Buy Entry` / `Allow Sell Entry` | 長いエントリと短いエントリを切り替えます。 |
| `Allow Buy Exit` / `Allow Sell Exit` | ショート/ロング取引の終了を切り替えます。 |
| `Candle Type` | インジケーターとシグナル評価に使用される主な時間枠。 |
| `CCI Period` / `CCI Price` | CCI の長さと適用される価格ソース。 |
| `ATR Period` | ATR の長さ。 |
| `Signal Bar` | シグナルに使用される完成したバーのインデックス (0 = 現在、1 = 前など)。 |

## 注意事項
- この戦略は、MT5 ティックベースの実装を模倣するために、完成したローソク足 (`CandleStates.Finished`) のみに依存します。
- 戦略が再開されるとすべてのインジケーター値と状態変数がリセットされ、決定的な最適化が確実に実行されます。
- StockSharp 成行注文では明示的なスリッページ パラメーターが使用されていない場合でも、`Deviation` パラメーターは完全な互換性を確保するために提供されています。
