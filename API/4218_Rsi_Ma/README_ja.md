# 4218 RSI MA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、`MQL/9925` にある元の MetaTrader Expert Advisor の C# ポートです。従来の RSI と加重価格 `(High + Low + 2 * Close) / 4` に基づいて構築された指数移動平均の傾きを組み合わせることにより、RSI_MA のモメンタム オシレーターを再作成します。シグナルは完成したキャンドルに対してのみ生成され、動作はソース実装と同一に保たれます。

このスクリプトは日次 EURUSD ローソク足 (D1 タイムフレーム) 用に設計されており、一度に 1 つのポジションをオープンします。それにもかかわらず、ローソク足タイプがそれに応じて設定されている限り、意味のある価格ステップを持つあらゆる商品を使用できます。

## 戦略ロジック
1. **指標の計算**
   - 設定可能な長さの相対強度指数は、終値に基づいて計算されます。
   - 同じ長さの指数移動平均が加重価格に基づいて計算されます。
   - インジケーター値は `RSI * (EMA(current) - EMA(previous)) / pipSize` に等しく、`[1, 99]` の範囲にクリップされます。
2. **ロングエントリー**
   - 売られ過ぎの極値 (デフォルト 5) を下回る以前のインジケーターの値。
   - 売られすぎアクティベーションしきい値 (デフォルトは 20) を超える最新のインジケーター値。
   - オープン ポジションまたは既存のショート ポジションがない (新しいロングを開く前にショートがクローズされている)。
3. **短いエントリー**
   - 買われすぎの極値 (デフォルト 95) を超える以前のインジケーターの値。
   - 買われすぎアクティベーションしきい値 (デフォルトは 80) を下回る最新のインジケーター値。
   - オープン ポジションまたは既存のロング ポジションがない (新しいショートを開く前にロングがクローズされている)。
4. **インジケーターベースの出口**
   - ロングポジションは、インジケーターが買われすぎの極値を超えてアクティブ化レベル (デフォルトでは 95 → 80) を下回るとクローズされます。
   - ショートポジションは、インジケーターが売られすぎの極値以下からアクティブレベル（デフォルトでは 5 → 20）以上に上昇するとクローズされます。
5. **保護出口**
   - オプションのストップロス、テイクプロフィット、トレーリングストップの距離はpipsで表されます。距離は、セキュリティ `PriceStep` (フォールバック 0.0001) を使用して自動的に価格に変換されます。
   - トレーリング ストップの引き締めは、元の EA の動作に従います。設定された距離を超えて価格が有利な方向に変動した場合にのみ有効になります。

## パラメーター
| パラメータ | 説明 |
|-----------|-------------|
| `RsiPeriod` | RSI と EMA の長さ。|
| `OversoldActivationLevel` | 極端に売られすぎた後の長期セットアップを確認するしきい値。 |
| `OversoldExtremeLevel` | 長く許される前に到達しなければならない極限。 |
| `OverboughtActivationLevel` | 買われすぎの後のショートセットアップを確認するしきい値。 |
| `OverboughtExtremeLevel` | ショートパンツが許可される前に到達する必要がある極限。 |
| `StopLossPips` | 保護ストップロスの距離。 `UseStopLoss` 経由で有効/無効にします。 |
| `TakeProfitPips` | 利益目標までの距離。 `UseTakeProfit` 経由で有効/無効にします。 |
| `TrailingStopPips` | トレーリングストップまでの距離。 `UseTrailingStop` 経由で有効/無効にします。 |
| `UseStopLoss` | ストップロス管理を有効にします。 |
| `UseTakeProfit` | テイクプロフィット経営を活性化します。 |
| `UseTrailingStop` | トレーリングストップの更新をアクティブにします。 |
| `UseMoneyManagement` | `RiskPercent` に基づいてポジションのサイジングを有効にします。 |
| `RiskPercent` | 資金管理がアクティブな場合の取引ごとのポートフォリオのリスクの割合。 |
| `TradeVolume` | 資金管理が無効な場合に使用される固定量。 |
| `CandleType` | ストラテジーによって処理されるローソク足のデータ型 (デフォルトは日次)。 |

## 使用上の注意
- この戦略を EURUSD の日足ローソク足にアタッチして、元の EA の動作を再現します。他の金融商品/時間枠は、`CandleType` としきい値を調整した後にサポートされます。
- 常にオープンなポジションは 1 つだけです。新しい取引を開始すると、最初に反対方向が自動的にクローズされます。
- ポートフォリオ情報が利用できない場合、または計算されたボリュームがプラスでなくなると、資金管理は固定の `TradeVolume` に戻ります。
- セキュリティ `PriceStep` がピップ (ほとんどの FX ペアでは 0.0001) を反映していることを確認します。それ以外の場合は、それに応じてパラメータを調整します。

## リスク管理
- ストップロスとテイクプロフィットのレベルは、ローソクの高値/安値の範囲を使用して、完了したローソクごとに評価されます。
- トレーリングストップは、取引が設定された距離を超えて利益を上げており、不利な方向に動いていない場合にのみ更新されます。
- リスク制御が無効になっている場合でもインジケーターベースの exit は機能し、MQL バージョンと同様の正常な機能低下が保証されます。
