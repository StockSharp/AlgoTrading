# Sniper Trade Pro戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ニューヨーク午前セッション中にE-mini S&P（ES）の15分足ローソク足を取引します。EMA 9/21によるトレンド確認、VWAP、マネーフロー・ダイバージェンス、ADXを強気・弱気の包み足パターンと組み合わせています。ポジションサイズはATRベースの1取引あたりリスクから導き出され、ストップ0.8 ATR、ターゲット2 ATRを使用し、1 ATRの利益後にストップをブレークイーブンへ移動します。$1,000の損失後はその日の取引を停止します。
