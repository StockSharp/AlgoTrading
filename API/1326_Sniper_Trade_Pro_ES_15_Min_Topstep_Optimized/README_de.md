# Sniper Trade Pro-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt den E-mini S&P (ES) auf 15-Minuten-Kerzen während der New Yorker Morgensession. Sie kombiniert Trendbestätigung durch EMA 9/21, VWAP, Geldfluss-Divergenz und ADX mit bullischen und bärischen Engulfing-Mustern. Die Positionsgröße wird aus dem ATR-basierten Risiko pro Trade abgeleitet, mit einem Stop von 0.8 ATR und einem Ziel von 2 ATR; nach 1 ATR Gewinn wird der Stop auf Break-even verschoben. Der Handel wird für den Tag nach einem Verlust von $1.000 beendet.
