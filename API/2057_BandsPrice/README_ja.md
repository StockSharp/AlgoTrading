# BandsPrice戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTraderの**i-BandsPrice**エキスパートの移植版です。ボリンジャーバンドを使用してチャンネル内の価格の相対的な位置を測定し、値が極端なゾーンを離れたときに反応します。

## ロジック

1. 設定可能な期間と偏差でボリンジャーバンドを構築。
2. バンド内の価格位置を-50から+50の値として計算。
3. 単純移動平均で値を平滑化。
4. カラーコードを生成:
   - 平滑化値が上部レベルを上回るとき `4`。
   - 平滑化値が下部レベルを下回るとき `0`。
   - その他の数字は中間ゾーンを表す。
5. インジケーターが上部ゾーンを離れたとき（`4` → `4`以外）ロングポジションを開設。
6. インジケーターが下部ゾーンを離れたとき（`0` → 正の値）ショートポジションを開設。
7. 値が非正になるとロングポジションを決済。
8. 値が非負になるとショートポジションを決済。

## パラメーター

| 名前 | 説明 |
|------|------|
| **BuyOpen** | ロングエントリーを有効化。 |
| **SellOpen** | ショートエントリーを有効化。 |
| **BuyClose** | ロングポジションの決済を有効化。 |
| **SellClose** | ショートポジションの決済を有効化。 |
| **BandsPeriod** | ボリンジャーバンドの期間。 |
| **BandsDeviation** | バンドの偏差。 |
| **Smooth** | 内部値の平滑化長。 |
| **UpLevel** | 上部閾値、デフォルト `25`。 |
| **DnLevel** | 下部閾値、デフォルト `-25`。 |
| **CandleType** | 計算に使用するローソク足の時間軸。 |

## 備考

この戦略は、`SubscribeCandles`と`Bind`を使用した高レベルAPIによって、MetaTraderのインジケーターベースのロジックをStockSharpに移植する方法を示しています。
