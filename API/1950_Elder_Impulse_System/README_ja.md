# Elderインパルスシステム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、指数移動平均線（EMA）の方向とMACDヒストグラムのモメンタムを組み合わせたElderインパルスシステムを再現します。上位時間軸のローソク足で強気または弱気のインパルスが衰えたときにトレードをオープンします。

EMAの傾きとMACDヒストグラムのダイナミクスから導出された色分けされたインパルスを観察します：
- **緑 (2)** — EMAが上昇し、MACDヒストグラムが上昇かつプラス。
- **赤 (1)** — EMAが下降し、MACDヒストグラムが下降かつマイナス。
- **青 (0)** — その他の状態。

直前の強気インパルス（緑）が弱まるとロングポジションをオープンし、弱気インパルス（赤）が弱まるとショートポジションをオープンします。対応するインパルスが検出されると、逆方向のポジションがクローズされます。

## 詳細

- **エントリー条件**: 完成したローソク足でのElderインパルスの色変化。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆インパルスまたはポジション保護。
- **ストップ**: デフォルトで2%のストップとテイクプロフィットを設定した`StartProtection`を使用。
- **デフォルト値**:
  - `EmaPeriod` = 13
  - `MacdFastPeriod` = 12
  - `MacdSlowPeriod` = 26
  - `MacdSignalPeriod` = 9
  - `CandleType` = TimeSpan.FromHours(4)
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: EMA, MACD
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: 4H
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
