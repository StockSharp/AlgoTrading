# MAチャネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MAチャネル戦略は、高値と安値の移動平均で構築された移動平均チャネルのブレイクアウトを取引します。価格が対応する方向にチャネルを抜けるとポジションを開き、トレンドが反転するとポジションを反転させます。チャネルの境界は固定オフセットを持つ指数移動平均から計算されます。

このシステムはロングとショート両方の取引向けに設計されており、完成したローソク足にのみ反応します。チャネル内のノイズを回避しながらトレンド転換を早期に捉えることを目的としています。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格が上部チャネルを上抜け。
  - **ショート**: 価格が下部チャネルを下抜け。
- **エグジット条件**:
  - 逆のブレイクアウトによりポジションの反転が発生。
- **インジケーター**: 設定可能な長さと価格オフセットを持つ高値・安値の指数移動平均。
- **ストップ**: デフォルトでは使用しない。逆シグナルのみでポジションをクローズ。
- **デフォルト値**:
  - `Length` = 8
  - `Offset` = 10
  - `CandleType` = 1時間足
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
