# 調整可能なMAと交互極値戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はボリンジャーバンドを使用して、交互の極値を持つ調整可能な移動平均を模倣します。価格が上部バンドを上抜けたときにロングポジションを開き、価格が下部バンドを下抜けたときにショートポジションを開きます。行き過ぎ状態が交互に切り替わり、同方向での連続取引を防ぎます。

## 詳細

- **エントリー条件**:
  - ローソク足の高値が上部バンドを上抜けたときにロング。
  - ローソク足の安値が下部バンドを下抜けたときにショート。
- **エグジット条件**:
  - 反対側バンドのブレイクアウト。
- **インジケーター**: ボリンジャーバンド（SMA + 標準偏差）。
- **デフォルト値**:
  - Length = 50
  - Multiplier = 2
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - 時間軸: 短期/中期
  - リスクレベル: 中
