# Стратегия Autostop
[English](README.md) | [中文](README_cn.md)

Утилитарная стратегия, которая автоматически выставляет тейк-профит и стоп-лосс для открытых позиций.
Сама стратегия не генерирует торговых сигналов. Любые сделки, открытые вне её логики, защищаются фиксированными расстояниями.

## Детали

- **Условия входа**: отсутствуют, сделки открываются извне.
- **Направление**: длинные и короткие позиции.
- **Условия выхода**: только защитные приказы.
- **Стопы**: фиксированные тейк-профит и стоп-лосс через StartProtection.
- **Значения по умолчанию**:
  - `MonitorTakeProfit` = true
  - `MonitorStopLoss` = true
  - `TakeProfitTicks` = 30
  - `StopLossTicks` = 30
- **Фильтры**:
  - Категория: управление рисками
  - Направление: обоих направлений
  - Индикаторы: нет
  - Стопы: фиксированные
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенции: нет
  - Уровень риска: низкий
