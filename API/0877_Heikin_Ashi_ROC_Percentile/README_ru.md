# Стратегия Heikin Ashi ROC Percentile
[English](README.md) | [中文](README_cn.md)

Стратегия преобразует свечи в Heikin Ashi, сглаживает закрытие SMA и измеряет его Rate of Change. Перцентильные полосы недавних максимумов и минимумов ROC образуют уровни пробоя. Пересечение ROC выше нижней полосы открывает или переворачивает позицию в лонг, пересечение ниже верхней полосы переворачивает в шорт.

## Детали

- **Условия входа**:
  - Лонг: ROC пересекает снизу вверх нижнюю перцентильную линию.
  - Шорт: ROC пересекает сверху вниз верхнюю перцентильную линию.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: Противоположный сигнал или стоп.
- **Стопы**: Процентный стоп.
- **Значения по умолчанию**:
  - `RocLength` = 100
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
  - `StartDate` = new DateTimeOffset(2015, 3, 3, 0, 0, 0, TimeSpan.Zero)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Heikin Ashi, RateOfChange, Highest, Lowest
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
