# Стратегия Kalman Filter Signal
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор Калмана для определения смены направления. В зависимости от выбранного режима сигнал формируется по сравнению фильтра с ценой либо по наклону фильтра. При появлении бычьего сигнала открывается длинная позиция, при медвежьем — короткая. Позиции разворачиваются при противоположном сигнале. Стоп-лосс и тейк-профит задаются абсолютными значениями.

## Подробности

- **Условия входа**:
  - Длинная: сигнал сменился на бычий
  - Короткая: сигнал сменился на медвежий
- **Long/Short**: Оба
- **Условия выхода**: противоположный сигнал
- **Стопы**: абсолютный стоп-лосс и тейк-профит
- **Параметры по умолчанию**:
  - `ProcessNoise` = 1.0
  - `MeasurementNoise` = 1.0
  - `CandleType` = TimeSpan.FromHours(3).TimeFrame()
  - `Mode` = SignalMode.Kalman
  - `StopLoss` = 1000m
  - `TakeProfit` = 2000m
- **Фильтры**:
  - Категория: Trend following
  - Направление: Оба
  - Индикаторы: Kalman Filter
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
