# トレンドフォロー KNN 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Trend Following KNN は、ウィンドウ内の平均価格変化を測定し、価格を移動平均と比較する簡略化された戦略です。
平均変化が正で価格が移動平均より高い場合に買い、平均変化が負で価格が移動平均より低い場合に売ります。

## 詳細

- **エントリー条件**: 正/負の平均変化と移動平均より上/下の価格
- **ロング/ショート**: 両方
- **エグジット条件**: 反対のシグナル
- **ストップ**: なし
- **デフォルト値**:
  - `WindowSize` = 20
  - `MaLength` = 50
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: SMA
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
