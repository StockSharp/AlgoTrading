# Trend Following KNN 策略
[English](README.md) | [Русский](README_ru.md)

Trend Following KNN 是一种简化策略，通过窗口内的平均价格变化并与移动平均比较。
当平均变化为正且价格在均线上方时买入，平均变化为负且价格在均线下方时卖出。

## 细节

- **入场条件**: 平均变化正/负且价格高于/低于均线
- **多空方向**: 双向
- **出场条件**: 反向信号
- **止损**: 无
- **默认值**:
  - `WindowSize` = 20
  - `MaLength` = 50
- **过滤器**:
  - 分类: 趋势
  - 方向: 双向
  - 指标: SMA
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
