# Стратегия DecEMA
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор DecEMA для отслеживания направления тренда. DecEMA строится с помощью десяти последовательных экспоненциальных сглаживаний, комбинируемых специальными коэффициентами, что позволяет получить линию с минимальным запаздыванием. Сравниваются три последних значения DecEMA: когда линия поворачивает вверх и текущее значение выше предыдущего, совершается покупка и закрывается короткая позиция; когда линия поворачивает вниз и текущее значение ниже предыдущего, совершается продажа и закрывается длинная позиция.

## Подробности

- **Условия входа**:
  - Лонг: наклон DecEMA вверх и текущее значение > предыдущего
  - Шорт: наклон DecEMA вниз и текущее значение < предыдущего
- **Лонг/Шорт**: Оба направления
- **Условия выхода**:
  - Лонг: наклон меняется на вниз
  - Шорт: наклон меняется на вверх
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `EmaPeriod` = 3
  - `Length` = 15
  - `BuyPosOpen` = true
  - `SellPosOpen` = true
  - `BuyPosClose` = true
  - `SellPosClose` = true
  - `CandleType` = TimeSpan.FromHours(8).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: DecEMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
