# Стратегия Non-Repainting Renko Emulation
[English](README.md) | [中文](README_cn.md)

Эмулирует кирпичи Renko по ценам закрытия и торгует на смене направлений без перерисовки.

## Детали

- **Условия входа**:
  - После формирования нового кирпича открывает лонг при признаках продолжения вверх.
  - Открывает шорт при обратной последовательности.
- **Длинные/Короткие**: Да.
- **Условия выхода**: Закрытие позиции при смене направления кирпича.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `BrickSize` = 3
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Pattern
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
