# Стратегия MACD с подтверждением 1D Stochastic
[English](README.md) | [中文](README_cn.md)

Стратегия покупает при пересечении линии MACD выше сигнальной с подтверждением от дневного стохастика. Сделка закрывается при достижении стоп-лосса на основе ATR либо при падении цены ниже скользящей EMA, используемой как трейлинг-тейкпрофит.

## Детали

- **Условия входа**:
  - Лонг: `MACD пересекает Signal снизу && DailyK > DailyD && DailyK < 80`
- **Лонг/Шорт**: Только лонг
- **Стопы**: Стоп-лосс по ATR и трейлинг-тейкпрофит по EMA
- **Значения по умолчанию**:
  - `MacdFastLength` = 12
  - `MacdSlowLength` = 26
  - `MacdSignalLength` = 9
  - `TrailingEmaLength` = 20
  - `StopLossAtrMultiplier` = 3.25m
  - `TrailingActivationAtrMultiplier` = 4.25m
  - `CandleType` = TimeSpan.FromHours(1).TimeFrame()
- **Фильтры**:
  - Категория: Разворот
  - Направление: Лонг
  - Индикаторы: MACD, Stochastic, ATR, EMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
