# Estrategia BB + RSI
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia compra cuando el precio cierra por debajo de la banda inferior de Bollinger Bands mientras el RSI está por debajo del nivel de compra. La posición se cierra cuando el RSI sube por encima del nivel de salida o cuando el precio cae desde el máximo el porcentaje de paso trailing especificado.

## Parámetros
- Tipo de vela
- Período de Bollinger Bands
- Desviación de Bollinger Bands
- Período del RSI
- Nivel de compra del RSI
- Nivel de salida del RSI
- Porcentaje de paso trailing
