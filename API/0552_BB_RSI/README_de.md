# BB + RSI-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kauft, wenn der Kurs unterhalb der unteren Bollinger Band schließt und der RSI unter dem Kaufniveau liegt. Die Position wird geschlossen, wenn der RSI über das Ausstiegsniveau steigt oder wenn der Kurs vom Höchststand um den angegebenen Trailing-Schritt-Prozentsatz fällt.

## Parameter
- Kerzentyp
- Bollinger-Bands-Periode
- Bollinger-Bands-Abweichung
- RSI-Periode
- RSI-Kaufniveau
- RSI-Ausstiegsniveau
- Trailing-Schritt in Prozent
