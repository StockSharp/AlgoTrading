# Gandalf PRO戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQLのGandalf PROエキスパートアドバイザーを複製します。ローソク足の終値で2つの移動平均（LWMAとSMA）を計算し、
それらを元の再帰的スムージングロジックに入力して将来の目標価格を予測します。予測目標が現在の終値から十分に離れると
（15価格ステップ以上）、戦略は成行注文を出し、予測から導出されたストップロスとテイクプロフィットのレベルを保存します。

ロングトレードでは、スムージングされた目標が現在の終値を少なくとも15ステップ上回る必要があります。ショートトレードでは、
目標が終値を同じ幅だけ下回る必要があります。ストップロスは価格ステップで定義され、銘柄の価格ステップを使用して変換されます。
テイクプロフィットレベルは予測目標と等しく、完了したすべてのローソク足で監視されます。リスク乗数は戦略のベースボリュームを
再スケールし、シンプルなマネーマネジメントルールを可能にします。

## パラメーター
- ローソク足タイプ
- 買いを有効化
- 買い長さ
- 買い価格係数
- 買いトレンド係数
- 買いストップロス
- 買いリスク乗数
- 売りを有効化
- 売り長さ
- 売り価格係数
- 売りトレンド係数
- 売りストップロス
- 売りリスク乗数
