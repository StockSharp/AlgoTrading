# 长短方向风险管理策略
[English](README.md) | [Русский](README_ru.md)

此示例策略展示如何使用百分比风控来管理多空头寸。策略依赖价格等于预设值的触发条件，并可选用时间退出。

## 详情

- **入场条件**：收盘价等于设定的多头或空头数值。
- **多/空**：双向。
- **出场条件**：止损、止盈或在 N 根K线后时间退出。
- **止损**：百分比止损和止盈，可选跟踪止损。
- **默认值**：
  - `StopLossPercent` = 2
  - `TakeProfitPercent` = 3
  - `ExitBars` = 10
  - `BarsToWait` = 10
  - `MaxTradesPerDay` = 3
  - `CandleType` = TimeSpan.FromMinutes(1)
- **过滤器**：
  - 类别：风险管理
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
