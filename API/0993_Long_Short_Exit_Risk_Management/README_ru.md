# Стратегия Long Short Exit Risk Management
[English](README.md) | [中文](README_cn.md)

Шаблонная стратегия, демонстрирующая управление длинными и короткими позициями с процентными параметрами риска. Используются простые ценовые триггеры и опциональный выход по времени.

## Детали

- **Условия входа**: Цена закрытия равна заданному значению для лонга или шорта.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Стоп‑лосс, тейк‑профит или выход после N баров.
- **Стопы**: Процентные стоп‑лосс и тейк‑профит с опциональным трейлингом.
- **Значения по умолчанию**:
  - `StopLossPercent` = 2
  - `TakeProfitPercent` = 3
  - `ExitBars` = 10
  - `BarsToWait` = 10
  - `MaxTradesPerDay` = 3
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Управление риском
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
