# 線形回帰傾きトリガー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、線形回帰傾きインジケーターとその派生トリガーラインを使用してトレンド転換を識別します。トリガーラインが傾きラインを上抜けした場合にロングポジションを開き、トリガーラインが傾きラインを下抜けした場合にショートポジションを開きます。反対のシグナルが現れると既存のポジションを決済します。このアプローチは、元のMQL5戦略「Exp_LinearRegSlopeV2」にインスピレーションを受けています。

## インジケーターロジック
1. **線形回帰傾き**は、設定可能な期間にわたってローソク足の終値で計算されます。
2. **トリガーライン**は `2 * slope - slope[Shift]` として計算されます。ここで `slope[Shift]` は数本前のバーの傾き値です。
3. トリガーラインと傾きラインのクロスオーバーが取引シグナルとして機能します。

## 取引ルール
- **ロングエントリー:** トリガーが傾きを上抜けし、ショート取引が許可されている場合。
- **ショートエントリー:** トリガーが傾きを下抜けし、ロング取引が許可されている場合。
- **ロング決済:** 傾きがトリガーを上回る。
- **ショート決済:** トリガーが傾きを上回る。

## パラメーター
- `SlopeLength` – 線形回帰傾きを計算する期間。
- `TriggerShift` – トリガーラインの計算に使用するバーの数。
- `EnableLong` – ロングエントリーを許可する。
- `EnableShort` – ショートエントリーを許可する。
- `TakeProfitPercent` – エントリー価格に対するテイクプロフィットのパーセント。
- `StopLossPercent` – エントリー価格に対するストップロスのパーセント。
- `CandleType` – 戦略が使用するローソク足の時間軸。

## 注意事項
- この戦略は完成したローソク足のみで動作します。
- `StartProtection` による保護は固定パーセントベースのテイクプロフィットとストップロスレベルを適用します。
- インジケーターが値を形成できるよう、十分な過去データを確保してください。
