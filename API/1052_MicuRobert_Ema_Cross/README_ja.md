# MicuRobert EMAクロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は2本のゼロラグ指数移動平均（ZLEMA）を使ってクロスオーバーを取引します。取引を特定のセッションに制限でき、オプションでトレーリングストップを使用できます。

## 詳細

- **エントリー条件:**
  - **ロング:** 高速ZLEMAが低速ZLEMAを上抜け、または高速が低速を上回っている状態で価格が高速ZLEMAを上抜け。
  - **ショート:** 高速ZLEMAが低速ZLEMAを下抜け、または高速が低速を下回っている状態で価格が高速ZLEMAを下抜け。
- **エグジット条件:** トレーリングストップまたは固定のストップロス・テイクプロフィットでポジション決済。
- **ストップ:** 固定テイクプロフィット・ストップロスを伴うオプションのトレーリングストップ。
- **フィルター:** セッション時間フィルター。
