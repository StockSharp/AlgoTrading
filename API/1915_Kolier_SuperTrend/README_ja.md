# Kolier SuperTrend
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ATRバンドを使用してトレンドリバーサルを検出するKolier SuperTrendインジケーターに基づく戦略です。

インジケーターはATRから導出された動的なサポートとレジスタンスレベルを描きます。価格が下のバンドを上回って終値をつけ、ラインが価格の下に反転すると強気のリバーサルが発生します。価格が上のバンドを下回って終値をつけると弱気のリバーサルが発生します。

この適応型トレイルに従うことで、戦略は強いトレンドを捉えながら、モメンタムが弱まったときに保護を維持しようとします。

## 詳細

- **エントリー条件**: 価格がSuperTrendラインを交差する。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対方向のクロスオーバー。
- **ストップ**: なし。
- **デフォルト値**:
  - `Period` = 10
  - `Multiplier` = 3.0m
  - `CandleType` = TimeSpan.FromHours(4)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: ATR, SuperTrend
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: スイング (4h)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
