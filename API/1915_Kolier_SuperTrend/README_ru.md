# Kolier SuperTrend
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторе Kolier SuperTrend, который использует полосы ATR для определения разворотов тренда.

Индикатор строит динамические уровни поддержки и сопротивления на основе ATR. Бычий разворот появляется, когда цена закрывается выше нижней линии и линия перемещается под цену. Медвежий — когда цена закрывается ниже верхней линии.

Следуя этому адаптивному следу, стратегия стремится захватывать сильные тренды, сохраняя защиту при ослаблении импульса.

## Детали

- **Критерии входа**: пересечение цены с линией SuperTrend.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Period` = 10
  - `Multiplier` = 3.0m
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ATR, SuperTrend
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Свинг (4ч)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
