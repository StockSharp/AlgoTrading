# Fedモデル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このマクロ・タイミング・システムは、株式市場の益回りと10年物米国債利回りを比較します。株式の方が高い利回りを提供している場合は株式ETFを保有し、債券の利回りが高い場合は現金に移行します。毎月の利回り格差に対する回帰分析が翌月の値を予測し、ノイズによる売買切替を減らします。

毎月末に、アルゴリズムは直近1年のデータを使って翌月の利回りスプレッドを予測します。予測が正であれば株式を買い、そうでなければ現金プロキシを保有します。予測がゼロをまたいだ時のみポジションが変わり、売買回転を最小化します。

## 詳細

- **エントリー条件**:
  - 月末に、`(EarningsYield - BondYield)` の直近`RegressionMonths`個の観測値で回帰を行い、次の値を予測する。
  - 予測がゼロを上回り、かつ注文が`MinTradeUsd`を満たす場合に株式ETFを買う。
- **ロング/ショート**: 株式または現金のロングのみ。
- **エグジット条件**: 予測された利回りスプレッドがマイナスに転じたら株式ポジションを決済する。
- **ストップ**: なし。
- **デフォルト値**:
  - `Universe` – [株式ETF、オプションで現金ETF]。
  - `BondYieldSym` – 10年物国債利回り系列。
  - `EarningsYieldSym` – 株式市場の益回り。
  - `RegressionMonths` = 12。
  - `CandleType` = 1日。
  - `MinTradeUsd` – 最低取引金額。
- **フィルター**:
  - カテゴリ: マクロ。
  - 方向: ロングのみ。
  - 時間軸: 月次。
  - リバランス: 毎月。
