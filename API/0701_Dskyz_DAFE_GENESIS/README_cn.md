# Dskyz (DAFE) GENESIS 策略
[English](README.md) | [Русский](README_ru.md)

Dskyz (DAFE) GENESIS 策略的简化版本。当短期动量与趋势过滤器和 RSI 一致时，该系统进行交易。

## 细节

- **入场条件**：
  - **做多**：`SMA(9) > SMA(30)` 且 `RSI > 55` 且 `EMA(8) > EMA(21)`。
  - **做空**：`SMA(9) < SMA(30)` 且 `RSI < 45` 且 `EMA(8) < EMA(21)`。
- **多/空**：双向。
- **出场条件**：
  - **多头**：`EMA(8) < EMA(21)`。
  - **空头**：`EMA(8) > EMA(21)`。
- **止损**：无。
- **默认值**：
  - `RSI Length` = 9。
- **过滤器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：RSI、EMA、SMA
  - 止损：无
  - 复杂度：简单
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险水平：中等
