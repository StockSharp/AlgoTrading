# Стратегия Dskyz (DAFE) GENESIS
[English](README.md) | [中文](README_cn.md)

Упрощённая версия стратегии Dskyz (DAFE) GENESIS. Система торгует, когда краткосрочный импульс согласуется с трендом и RSI.

## Детали

- **Критерии входа**:
  - **Длинная позиция**: `SMA(9) > SMA(30)` и `RSI > 55` и `EMA(8) > EMA(21)`.
  - **Короткая позиция**: `SMA(9) < SMA(30)` и `RSI < 45` и `EMA(8) < EMA(21)`.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - **Длинная позиция**: `EMA(8) < EMA(21)`.
  - **Короткая позиция**: `EMA(8) > EMA(21)`.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `RSI Length` = 9.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: RSI, EMA, SMA
  - Стопы: Нет
  - Сложность: Простая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
