# Custom Signal Oscillator 自定义信号振荡器
[English](README.md) | [Русский](README_ru.md)

该策略使用两个价格信号的差值。当振荡器上穿零线时做多，下穿零线时做空。启用仅做多模式时，振荡器下穿零线会平仓。

## 详情
- **入场条件**: 振荡器穿越零线。
- **多空方向**: 双向或仅做多。
- **退出条件**: 反向信号或仅做多模式下的零线穿越。
- **止损**: 无。
- **默认值**:
  - `LongOnly` = false
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**:
  - 类型: 振荡器
  - 方向: 双向
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
