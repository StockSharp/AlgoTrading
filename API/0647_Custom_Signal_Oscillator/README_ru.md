# Custom Signal Oscillator
[English](README.md) | [中文](README_cn.md)

Стратегия использует разницу между двумя ценовыми сигналами. Входит в лонг, когда осциллятор пересекает ноль снизу вверх, и в шорт, когда пересекает сверху вниз. В режиме только лонг отрицательное пересечение закрывает позицию.

## Детали

- **Критерии входа**: пересечение осциллятора с нулевой линией.
- **Длинные/короткие**: оба направления или только лонг.
- **Критерии выхода**: противоположный сигнал или пересечение нуля в режиме только лонг.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `LongOnly` = false
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
