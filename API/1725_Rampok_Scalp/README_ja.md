# Rampok スキャルピング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

移動平均エンベロープをブレイクしたときに取引するスキャルピングシステムです。
価格が下限バンドを上抜けするとロングでエントリーし、
上限バンドを下抜けするとショートでエントリーします。ポジションはオプションのテイクプロフィット、ストップロス、トレーリングストップのパラメーターで保護されます。

## 詳細

- **エントリー条件**:
  - **買い**: 前のクローズが下限バンドを下回り、現在のクローズが上回る。
  - **売り**: 前のクローズが上限バンドを上回り、現在のクローズが下回る。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - テイクプロフィット、ストップロスまたはトレーリングストップ。
- **ストップ**: 設定可能なSL/TPおよびトレーリング。
- **フィルター**: なし。
