# Rampok Scalp 策略
[English](README.md) | [Русский](README_ru.md)

基于移动平均包络线的剥头皮系统。
当价格向上突破下轨时做多，向下跌破上轨时做空。
仓位可通过可选的止盈、止损和跟踪止损参数进行保护。

## 细节

- **入场条件**:
  - **买入**：前一根蜡烛收于下轨之下，当前收于下轨之上。
  - **卖出**：前一根蜡烛收于上轨之上，当前收于上轨之下。
- **多空方向**：双向。
- **出场条件**：
  - 止盈、止损或跟踪止损。
- **止损**：可配置的 SL/TP 与跟踪。
- **过滤器**：无。
