# NRatio Sign 策略
[Русский](README_ru.md) | [English](README.md)

该策略使用 NRatio 指标，这是一种基于 NRTR 的振荡器，用于衡量价格与动态跟踪水平之间的归一化距离。当 NRatio 穿越预设阈值时产生交易信号。根据所选模式，系统要么在突破上下界时入场，要么在价格返回区间内时入场。

策略可同时在多头和空头方向运行，并通过百分比的止盈和止损进行风险管理。归一化距离采用指数移动平均进行平滑，以快速响应并过滤噪音。

## 详细信息

- **入场条件**：
  - **模式 In**：
    - **做多**：`NRatio` 上穿 `UpLevel`。
    - **做空**：`NRatio` 下破 `DownLevel`。
  - **模式 Out**：
    - **做多**：`NRatio` 上穿 `DownLevel`。
    - **做空**：`NRatio` 下破 `UpLevel`。
- **方向**：可做多亦可做空。
- **出场条件**：反向信号或保护性止损。
- **止损/止盈**：是，使用百分比设置。
- **默认值**：
  - `CandleType` = 4 小时K线
  - `Kf` = 1
  - `Length` = 3
  - `Fast` = 2
  - `Sharp` = 2
  - `UpLevel` = 80
  - `DownLevel` = 20
  - `TakeProfitPercent` = 2
  - `StopLossPercent` = 2
- **过滤条件**：
  - 类型：趋势跟随
  - 方向：双向
  - 指标：NRTR、EMA
  - 止损：是
  - 复杂度：中等
  - 时间框架：中期
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
