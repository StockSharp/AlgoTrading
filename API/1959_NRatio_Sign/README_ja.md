# NRatio Sign戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はNRatioインジケーターを使用します。これはNRTRベースのオシレーターで、価格と動的なトレーリングレベルの間の正規化された距離を測定します。NRatioが事前定義された閾値をクロスするとトレードシグナルが発生します。選択したモードに応じて、システムは上限と下限を超えたブレイクアウトに反応するか、それとも内側への反転に反応します。

このアプローチは市場の両側で動作でき、エグジットにパーセンテージベースのリスク管理を使用します。距離メトリクスの平滑化は指数移動平均で行われ、ノイズをフィルタリングしながら素早く反応できます。

## 詳細

- **エントリー条件**:
  - **モード In**:
    - **ロング**: `NRatio`が`UpLevel`を上方クロス。
    - **ショート**: `NRatio`が`DownLevel`を下方クロス。
  - **モード Out**:
    - **ロング**: `NRatio`が`DownLevel`を上方クロス。
    - **ショート**: `NRatio`が`UpLevel`を下方クロス。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対シグナルまたは保護ストップ。
- **ストップ**: あり、テイクプロフィットとストップロスをパーセントで設定。
- **デフォルト値**:
  - `CandleType` = 4時間ローソク足
  - `Kf` = 1
  - `Length` = 3
  - `Fast` = 2
  - `Sharp` = 2
  - `UpLevel` = 80
  - `DownLevel` = 20
  - `TakeProfitPercent` = 2
  - `StopLossPercent` = 2
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: NRTR, EMA
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
