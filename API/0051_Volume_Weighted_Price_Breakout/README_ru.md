# Прорыв цены, взвешенный по объему
[English](README.md) | [中文](README_cn.md)

Эта стратегия сочетает скользящую среднюю с объемной скользящей средней (VWMA). Когда цена находится выше VWMA, это подразумевает доминирование покупателей. Прорыв происходит, когда цена пересекает VWMA с противоположной стороны.

Сделки открываются по направлению VWMA, а простая скользящая средняя служит фильтром более высокого уровня. Выход осуществляется при развороте цены относительно скользящей средней.

Цель стратегии — ловить прорывы, подтверждённые объёмом.

## Детали

- **Условия входа**: Цена выше или ниже VWMA при подтверждении от MA.
- **Длинные/короткие позиции**: Оба направления.
- **Условия выхода**: Цена пересекает MA в обратную сторону или срабатывает стоп.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `MAPeriod` = 20
  - `VWAPPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Обе стороны
  - Индикаторы: VWMA, MA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
