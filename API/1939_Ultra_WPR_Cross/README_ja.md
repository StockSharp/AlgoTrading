# Ultra WPRクロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、2つの移動平均でスムージングされたWilliams %Rオシレーターを適用します。速いスムージングラインと遅いスムージングラインのクロスが取引シグナルを生成します。速いラインが遅いラインを上抜けするとロングポジションが開かれ、速いラインが遅いラインを下抜けするとショートポジションが開かれます。

このアプローチは、設定可能なテイクプロフィットとストップロスレベルでリスクを抑えながら、新たに生まれるモメンタムに追随することを目指します。

## 詳細
- **エントリー条件**:
  - **ロング**: 速いラインが遅いラインを上抜け
  - **ショート**: 速いラインが遅いラインを下抜け
- **ロング/ショート**: 両方
- **エグジット条件**:
  - **ロング**: 速いラインが遅いラインを下抜けしたとき
  - **ショート**: 速いラインが遅いラインを上抜けしたとき
- **ストップ**: はい、価格ベースのテイクプロフィットとストップロス
- **デフォルト値**:
  - `CandleType` = TimeSpan.FromHours(4)
  - `WprPeriod` = 13
  - `FastLength` = 3
  - `SlowLength` = 53
  - `TakeProfit` = 0.2m
  - `StopLoss` = 0.1m
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Williams %R, Moving Average
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: H4
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
