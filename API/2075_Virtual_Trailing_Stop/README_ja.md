# バーチャル・トレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ロングおよびショートポジションのバーチャル・トレーリングストップをエミュレートします。エントリーシグナルは生成しません。注文は外部またはマニュアルで開く必要があります。ポジションが存在すると、戦略は価格が有利な方向に動くにつれてトレーリングストップを追従させます。価格がストップレベルに達した場合、ポジションは成行で決済されます。

## パラメーター

- `StopLoss` – 価格ステップ単位の固定ストップロス距離。
- `TakeProfit` – 価格ステップ単位の固定テイクプロフィット距離。
- `TrailingStop` – 現在価格からトレーリングストップまでの距離。
- `TrailingStart` – トレーリング開始前の価格ステップ単位の最小利益。
- `TrailingStep` – トレーリングレベルを移動させるために必要な最小追加利益。
- `CandleType` – 価格データ処理に使用するローソク足シリーズ。

## 注記

この戦略は指定されたタイプのローソク足を購読し、閉じたローソク足のみでトレーリングロジックを評価します。
