# Стратегия Daily Breakout Daily Shadow
[English](README.md) | [中文](README_cn.md)

Стратегия торгует дневные пробои, анализируя две последние завершённые дневные свечи. Любая открытая позиция закрывается в начале нового дня.

## Детали

- **Условия входа**:
  - Лонг: предыдущий день закрывается выше верхней границы тела свечи позавчера и открывается ниже этого уровня.
  - Шорт: предыдущий день закрывается ниже нижней границы тела свечи позавчера и открывается выше этого уровня.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Позиция закрывается в начале нового дня.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `CandleType` = 1 день
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
