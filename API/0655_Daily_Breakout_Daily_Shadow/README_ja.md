# 日足ブレイクアウト・日足シャドウ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は直近2本の完成した日足ローソク足を使用して日足ブレイクアウトを取引します。新しい日の開始時に全ての未決済ポジションをクローズします。

## 詳細

- **エントリー条件**:
  - ロング: 前日が前々日のローソク足の実体高値を上回って引け、その水準を下回って始値をつけた場合。
  - ショート: 前日が前々日のローソク足の実体安値を下回って引け、その水準を上回って始値をつけた場合。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 新しい日の開始時にポジションをクローズ。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `CandleType` = 1 Day
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
