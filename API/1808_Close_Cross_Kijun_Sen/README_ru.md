# Стратегия закрытия по пересечению Киджун-сен
[English](README.md) | [中文](README_cn.md)

Данная стратегия выступает в роли инструмента управления позицией. Она закрывает открытые позиции, когда цена закрытия пересекает линию Киджун-сен индикатора Ишимоку.

Во время работы стратегия подписывается на свечи и вычисляет значение Киджун-сен. При наличии длинной позиции и падении цены ниже линии на заданный отступ позиция закрывается. При наличии короткой позиции и росте цены выше линии позиция также закрывается. Новые сделки стратегия не открывает.

## Детали

- **Условия входа**: Нет, стратегия только управляет уже открытыми позициями.
- **Длинные/короткие**: Обе (закрытие).
- **Условия выхода**: Пересечение цены закрытия линии Киджун-сен с учётом заданного отступа.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `KijunPeriod` = 50
  - `PointsToCross` = 0
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Управление позицией
  - Направление: Оба
  - Индикаторы: Ишимоку
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
