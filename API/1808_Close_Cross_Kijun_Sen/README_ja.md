# Kijun-Senクロスによる決済戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はトレード管理ツールとして機能します。一目均衡表のKijun-senラインを終値がクロスしたとき、既存のポジションを決済します。

実行中、戦略はローソク足を購読してKijun-senの値を計算します。ロングポジションが存在し、価格が設定可能なオフセットだけKijunラインを下回ると、ポジションが決済されます。ショートポジションが開かれており、価格がラインを上回ると、ポジションも決済されます。この戦略は新しいトレードを開きません。

## 詳細

- **エントリー条件**: この戦略は新しいトレードを開きません。既存のポジションのみを管理します。
- **ロング/ショート**: 両方（決済）。
- **エグジット条件**: 指定されたオフセットによりKijun-senラインを終値がクロスする。
- **ストップ**: なし。
- **デフォルト値**:
  - `KijunPeriod` = 50
  - `PointsToCross` = 0
  - `CandleType` = 5分
- **フィルター**:
  - カテゴリ: トレード管理
  - 方向: 両方
  - インジケーター: Ichimoku
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
