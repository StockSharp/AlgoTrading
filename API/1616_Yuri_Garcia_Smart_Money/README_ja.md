# Yuri Garcia スマートマネー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このスマートマネーコンセプト戦略は、高ボリュームゾーンと4時間足のサポート/レジスタンスエリア内の価格反応を探します。累積デルタとヒゲのプルバックでエントリーを確認し、機関投資家のオーダーフローに追従することを目指します。

テストでは年間平均リターン約42%を示しています。BTCと主要指数で最も効果を発揮します。

システムはATRベースのストップロスとテイクプロフィットを、設定可能なリスク/リワード比で計算します。ロング・ショート・両方向のトレードが可能で、価格がゾーン内にあり、ヒゲのプルバックが発生し、デルタが動きを支持する場合にのみポジションが開かれます。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格がバッファー付き高値/安値ゾーン内、強気のヒゲのプルバック、累積デルタ上昇中。
  - **ショート**: 価格がゾーン内、弱気のヒゲのプルバック、累積デルタ下降中。
- **ロング/ショート**: 設定可能（両方向、買いのみ、または売りのみ）。
- **エグジット条件**:
  - ATRベースのストップロスまたはテイクプロフィット。
- **ストップ**: あり、ATRベース。
- **フィルター**:
  - HTFゾーン、累積デルタ確認、ヒゲのプルバック。
