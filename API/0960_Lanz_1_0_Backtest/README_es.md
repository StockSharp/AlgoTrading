# Estrategia LANZ 1.0 [Backtest]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia LANZ 1.0 coloca una única orden limitada cada día basándose en la dirección del precio entre las 08:00 y las 18:00 hora de Nueva York. La orden se coloca entre las 18:00 y las 08:00 con distancias fijas de stop-loss y take-profit en pips. Cualquier posición abierta se cierra a la hora de Nueva York configurada.

## Parámetros
- Tipo de vela
- Tamaño de cuenta (USD)
- Porcentaje de riesgo
- Fracción de desplazamiento EP
- Stop Loss (pips)
- Take Profit (pips)
- Hora de cierre manual (NY)
- Habilitar compra
- Habilitar venta
