# LANZ戦略 1.0 [バックテスト]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

LANZ戦略1.0は、ニューヨーク時間の08:00から18:00の間の価格方向に基づき、毎日1つの指値注文を出します。注文は18:00から08:00の間に、固定のストップロスとテイクプロフィットのpips距離で配置されます。開いているポジションは設定されたニューヨーク時間に決済されます。

## パラメーター
- ローソク足タイプ
- 口座サイズ (USD)
- リスクパーセント
- EPオフセット比率
- ストップロス (pips)
- テイクプロフィット (pips)
- 手動決済時刻 (NY)
- 買いを有効にする
- 売りを有効にする
