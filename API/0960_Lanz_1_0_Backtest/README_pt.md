# Estratégia LANZ 1.0 [Backtest]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia LANZ 1.0 coloca uma única ordem limitada por dia com base na direção do preço entre 08:00 e 18:00 horário de Nova York. A ordem é colocada entre 18:00 e 08:00 com distâncias fixas de stop-loss e take-profit em pips. Qualquer posição aberta é fechada na hora de Nova York configurada.

## Parâmetros
- Tipo de vela
- Tamanho da conta (USD)
- Percentual de risco
- Fração de deslocamento EP
- Stop Loss (pips)
- Take Profit (pips)
- Hora de fechamento manual (NY)
- Habilitar compra
- Habilitar venda
