# LANZ-Strategie 1.0 [Backtest]
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die LANZ-Strategie 1.0 platziert täglich eine einzelne Limitorder basierend auf der Preisrichtung zwischen 08:00 und 18:00 Uhr New Yorker Zeit. Die Order wird zwischen 18:00 und 08:00 Uhr mit festen Stop-Loss- und Take-Profit-Abständen in Pips platziert. Alle offenen Positionen werden zur konfigurierten New Yorker Stunde geschlossen.

## Parameter
- Kerzentyp
- Kontogröße (USD)
- Risikoprozentsatz
- EP-Versatz-Fraktion
- Stop Loss (Pips)
- Take Profit (Pips)
- Manuelle Schlussstunde (NY)
- Kauf aktivieren
- Verkauf aktivieren
