# Стратегия ADX Donchian

Стратегия базируется на индикаторе ADX и канале Дончиана. Длинная позиция открывается при значении ADX выше порога и пробое верхней границы канала. Короткая позиция берётся, когда ADX превышает порог, а цена пробивает нижнюю границу. Метод подходит для смешанных рынков.

## Подробности
- **Условия входа**:
  - **Long**: ADX > AdxThreshold && Price >= upperBorder
  - **Short**: ADX > AdxThreshold && Price <= lowerBorder
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход, когда ADX опускается ниже (порог − 5)
  - **Short**: выход, когда ADX опускается ниже (порог − 5)
- **Стопы**: да
- **Значения по умолчанию**:
  - `AdxPeriod` = 14
  - `DonchianPeriod` = 5
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `AdxThreshold` = 10
  - `Multiplier` = 0.1m
- **Фильтры**:
  - Категория: Mixed
  - Направление: оба
  - Индикаторы: ADX Donchian
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
