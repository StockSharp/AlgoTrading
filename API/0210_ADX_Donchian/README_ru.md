# Стратегия ADX Donchian
[English](README.md) | [中文](README_cn.md)

В данной стратегии сочетаются индикаторы ADX и канал Donchian. Длинная позиция открывается, когда ADX выше порогового значения и цена пробивает верхнюю границу канала — признак сильного восходящего тренда. Короткая позиция формируется при ADX выше порога и пробое нижней границы канала, что говорит о сильном нисходящем движении.

Стратегия подходит трейдерам, работающим в смешанном рынке.

## Детали
- **Условия входа**:
  - **Лонг**: ADX > порога и цена >= верхней границы (сильный тренд и пробой вверх)
  - **Шорт**: ADX > порога и цена <= нижней границы (сильный тренд и пробой вниз)
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - **Лонг**: Выход при снижении ADX ниже (порог - 5)
  - **Шорт**: Выход при снижении ADX ниже (порог - 5)
- **Стопы**: да.
- **Значения по умолчанию**:
  - `AdxPeriod` = 14
  - `DonchianPeriod` = 5
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `AdxThreshold` = 10
  - `Multiplier` = 0.1m
- **Фильтры**:
  - Категория: Смешанная
  - Направление: Оба
  - Индикаторы: ADX Donchian
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
