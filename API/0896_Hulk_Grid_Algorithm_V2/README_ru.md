# Стратегия Hulk Grid Algorithm V2
[English](README.md) | [中文](README_cn.md)

Грид-стратегия выставляет десять лимитных заявок на покупку вокруг заданной средней цены. Объём заявок увеличивается по мере приближения к средней. Все позиции закрываются, а оставшиеся заявки снимаются при достижении стоп-лосса ниже нижней сетки или тейк-профита выше верхней сетки.

## Подробности

- **Критерии входа**: Сетка из десяти лимитных покупок от нижнего к верхнему уровню.
- **Длинные/Короткие**: Только long.
- **Критерии выхода**: Стоп-лосс под нижней сеткой или тейк-профит над верхней.
- **Стопы**: Процентные стоп-лосс и тейк-профит.
- **Значения по умолчанию**:
  - `MidPrice` = 0
  - `StopLossPercent` = 2.0
  - `TakeProfitPercent` = 2.0
  - `GridStep` = 200
  - `Lot` = 50
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Grid
  - Направление: Long
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
