# Hulk Grid Algorithm V2戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ユーザー定義の中間価格を中心に10段階の指値買い注文を並べるグリッド取引戦略です。注文サイズは中間レベルに近づくほど大きくなります。価格が最下位グリッド以下のストップロスに達するか、上位グリッド以上のテイクプロフィットに達した場合、全ポジションを決済し残りの注文をキャンセルします。

## 詳細

- **エントリー条件**: 最低レベルから最高レベルまで10本の指値買い注文のグリッド。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 最下位グリッド以下のストップロスまたは上位グリッド以上のテイクプロフィット。
- **ストップ**: パーセンテージベースのストップロスとテイクプロフィット。
- **デフォルト値**:
  - `MidPrice` = 0
  - `StopLossPercent` = 2.0
  - `TakeProfitPercent` = 2.0
  - `GridStep` = 200
  - `Lot` = 50
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: Grid
  - 方向: Long
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
