# リサンプリングフィルターパック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はNバーごとに価格をサンプリングし、移動平均で平滑化します。フィルター値が上昇して価格がその上で取引される場合にロングに入り、フィルター値が下落して価格がその下にある場合にショートに入ります。

## 詳細
- **エントリー条件**:
  - **ロング**: フィルターの傾きが上向きで終値がフィルターの上にある。
  - **ショート**: フィルターの傾きが下向きで終値がフィルターの下にある。
- **エグジット条件**: 逆シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `BarsPerSample` = 5
  - `MovingAverageType` = EMA
  - `MaPeriod` = 9
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング & ショート
  - インジケーター: 移動平均
  - 複雑さ: シンプル
  - リスクレベル: 中
