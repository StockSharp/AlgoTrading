# Стратегия Resampling Filter Pack
[English](README.md) | [中文](README_cn.md)

Стратегия пересэмплирует цену каждые N баров и сглаживает её скользящей средней. Открывает длинную позицию, когда фильтр растёт и цена выше него, и короткую, когда фильтр падает и цена ниже.

## Детали
- **Условия входа**:
  - **Лонг**: Фильтр направлен вверх и закрытие выше фильтра.
  - **Шорт**: Фильтр направлен вниз и закрытие ниже фильтра.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `BarsPerSample` = 5
  - `MovingAverageType` = EMA
  - `MaPeriod` = 9
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Лонг и Шорт
  - Индикаторы: Скользящая средняя
  - Сложность: Простая
  - Уровень риска: Средний
