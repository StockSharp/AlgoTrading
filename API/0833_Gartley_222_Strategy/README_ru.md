# Стратегия Gartley 222
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию при формировании бычьего паттерна Гартли 222.
Паттерн определяется по пивотам и проверяется по уровням Фибоначчи.

Вход выполняется через `PivotLength` баров после подтверждения, когда цена закрывается выше точки C.
Выход происходит по тейк‑профиту на расширении Фибоначчи или по стоп‑лоссу в процентах.

## Детали

- **Условия входа**:
  - Подтверждённый бычий паттерн Гартли 222
  - Вход отложен на `PivotLength` баров
- **Long/Short**: Только long
- **Условия выхода**:
  - Стоп‑лосс или тейк‑профит
- **Стопы**:
  - `Stop Loss %` ниже входа
  - `TP Fib Extension` выше входа
- **Значения по умолчанию**:
  - `Pivot Length` = 5
  - `Fib Tolerance` = 0.05
  - `TP Fib Extension` = 1.27
  - `Stop Loss %` = 2

- **Фильтры**:
  - Категория: Паттерны
  - Направление: Long
  - Индикаторы: Pivot, Фибоначчи
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Средний
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
