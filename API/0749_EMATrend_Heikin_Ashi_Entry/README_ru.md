# Стратегия EMA Trend Heikin Ashi Entry
[English](README.md) | [中文](README_cn.md)

Стратегия использует полосы Боллинджера на свечах Хейкин-Аши с фильтром тренда по EMA на старшем таймфрейме. Покупает после нескольких подряд медвежьих свечей Хейкин-Аши, касающихся нижней полосы, и последующей бычьей свечи выше неё при условии, что быстрая EMA на старшем таймфрейме выше медленной. Продаёт наоборот.

После входа фиксируется первая цель размером с риск и стоп подтягивается по минимумам/максимумам предыдущих свечей.

## Детали

- **Условия входа**:
  - Long: минимум две медвежьи HA-свечи, касающиеся нижней полосы, затем бычья выше неё при быстр. EMA > медл. EMA на старшем ТФ
  - Short: минимум две бычьи HA-свечи, касающиеся верхней полосы, затем медвежья ниже неё при быстр. EMA < медл. EMA на старшем ТФ
- **Направление**: Оба
- **Условия выхода**:
  - Long: первая цель = 1R, затем трейлинг по минимумам
  - Short: первая цель = 1R, затем трейлинг по максимумам
- **Стопы**: минимум/максимум предыдущей свечи
- **Значения по умолчанию**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 2m
  - `FastEmaPeriod` = 9
  - `SlowEmaPeriod` = 21
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
  - `HigherTimeframe` = TimeSpan.FromMinutes(180).TimeFrame()
- **Фильтры**:
  - Категория: Откат
  - Направление: Оба
  - Индикаторы: Bollinger Bands, Heikin Ashi, EMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
