# Стратегия "Накопление по Вайкоффу"
[English](README.md) | [中文](README_zh.md)

Накопление по Вайкоффу описывает фазу основания, когда крупные участники тайно набирают позиции после снижения. Объём и движение цены формируют серию тестов поддержки и повышающихся минимумов, что свидетельствует о растущем спросе.

Тестирование показывает среднегодичную доходность около 61%. Стратегию лучше запускать на крипторынке.

Стратегия входит в лонг при выходе цены из диапазона накопления, ожидая начало нового восходящего тренда на основе этих покупок.

Защитный стоп размещается непосредственно под базой, ограничивая убытки в случае неудачного пробоя.

## Детали

- **Условия входа**: сигнал индикатора
- **Длинная/короткая**: обе
- **Условия выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: обе
  - Индикаторы: Объём, Цена
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
