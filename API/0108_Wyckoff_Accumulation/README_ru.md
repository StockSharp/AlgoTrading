# Стратегия Wyckoff Accumulation

Wyckoff Accumulation описывает фазу накопления, когда крупные участники тихо строят позиции после снижения.
Объём и динамика цены формируют серию тестов поддержки и более высокие минимумы, что говорит о растущем спросе.

Стратегия входит в длинную позицию при выходе цены из диапазона накопления, ожидая новый рост, подкреплённый этими покупками.

Защитный стоп размещается чуть ниже основания, чтобы ограничить убытки в случае неудачного пробоя.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Обе
  - Индикаторы: Объём, цена
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
