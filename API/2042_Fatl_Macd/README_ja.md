# FATL MACDトレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は**FATL MACD**インジケーターに基づくトレンドフォローシステムを実装します。FATL（Fast Adaptive Trend Line）を価格から引くことでMACDに似たオシレーターを生成し、適応型移動平均でスムージングします。正の値は強気のモメンタムを示し、負の値は弱気のモメンタムを示します。

アルゴリズムは完成した各ローソク足でこのオシレーターの傾きを分析します。

- 前の値がその前の値より低い場合、オシレーターは上向きに転換しています。現在の値がさらに上昇すると、戦略はロングポジションを開き、ショートポジションを閉じます。
- 前の値がその前の値より高い場合、オシレーターは下向きに転換しています。現在の値がさらに下落すると、戦略はショートポジションを開き、ロングポジションを閉じます。

すべての主要パラメーターは設定可能です。

- **Fast EMA** – MACDの速い移動平均の期間（デフォルト12）。
- **Slow EMA** – MACDの遅い移動平均の期間（デフォルト26）。
- **Signal EMA** – MACDシグナルラインの期間（デフォルト9）。
- **Candle Type** – インジケーター計算に使用するローソク足シリーズ。

ポジションは成行注文で建てられ、反対シグナルが現れると決済されます。
