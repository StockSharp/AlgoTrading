# Стратегия "Назад на N баров"
[English](README.md) | [中文](README_cn.md)

Эта стратегия держит длинную позицию только в течение последних баров, отсчитываемых от текущего времени. Она демонстрирует, как ограничить торговлю движущимся историческим окном.

## Детали

- **Условия входа**: время свечи попадает в последние *N* баров от момента запуска.
- **Условия выхода**: время свечи выходит за пределы окна.
- **Длинная/короткая**: только длинная.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `Количество баров` = 50
  - `Тип свечи` = 1‑минутные свечи
- **Фильтры**:
  - Категория: по времени
  - Направление: длинное
  - Индикаторы: нет
  - Стопы: нет
  - Сложность: простая
  - Таймфрейм: краткосрочный
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: низкий
