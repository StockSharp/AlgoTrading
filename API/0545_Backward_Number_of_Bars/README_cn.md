# 向后N根K线策略
[English](README.md) | [Русский](README_ru.md)

该策略仅在距当前时间最近的若干根K线期间保持多头仓位，示范如何将交易限制在移动的历史窗口内。

## 细节

- **入场条件**：K线时间位于启动时间起向后数的 *N* 根K线之内。
- **出场条件**：K线时间超出该窗口。
- **多空方向**：仅做多。
- **止损**：无。
- **默认值**：
  - `K线数量` = 50
  - `K线类型` = 1 分钟K线
- **过滤器**：
  - 类别：基于时间
  - 方向：多头
  - 指标：无
  - 止损：无
  - 复杂度：简单
  - 时间框架：短期
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险级别：低
