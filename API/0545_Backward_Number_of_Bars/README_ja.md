# 過去バー数遡り戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、現在時刻から遡って数えた最新のバーのみロングポジションを保持します。移動する履歴ウィンドウに取引を制限する方法を示します。

## 詳細

- **エントリー条件**: ローソク足の時刻が開始時刻から直近の *N* バル以内にある。
- **エグジット条件**: ローソク足の時刻がこのウィンドウ外になる。
- **ロング/ショート**: ロングのみ。
- **ストップ**: なし。
- **デフォルト値**:
  - `Bar count` = 50
  - `Candle type` = 1分ローソク足
- **フィルター**:
  - カテゴリ: 時間ベース
  - 方向: ロング
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
