# 自動操縦付きトレードパネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MQL5の**Trade panel with autopilot**サンプルをStockSharpフレームワークに移植したものです。
複数の時間軸にわたって強気と弱気の圧力を計算します。対応するパーセンテージが*Open %*の閾値を超えるとポジションが開かれ、*Close %*レベルを下回ると閉じられます。オプションで、10分足ローソク足を使用したフラクタルベースのストップロスを適用できます。

## パラメーター

- **Autopilot** – 自動取引を有効または無効にする。
- **Open %** – ポジションを開くために必要な票数の閾値。
- **Close %** – 既存ポジションを閉じるための閾値。
- **Use Fixed Volume** – 真の場合、*Fixed Volume*の値を使用する。
- **Fixed Volume** – 絶対的な注文量。
- **Volume %** – ボリュームが動的な場合に使用するポートフォリオのパーセンテージ。
- **Use Stop Loss** – 直近のフラクタルに基づくストップロスを有効にする。

## ロジック

1分から1ヶ月までの各時間軸に対して、戦略は最新のローソク足と前のものを比較します。始値、高値、安値と派生平均の各比較が買いまたは売りの票を追加します。買いと売りの票のパーセンテージが注文の配置を決定します。有効にすると、10分足ローソク足の最後のフラクタルがトレーリングストップとして機能します。

この例は教育目的であり、投資アドバイスを表すものではありません。
