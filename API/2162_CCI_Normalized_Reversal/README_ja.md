# CCI 正規化リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は商品チャネル指数（CCI）を使用して、指標が極端なゾーンを離れた後のリバーサルを検出します。

## 概要

指標は設定可能な期間で8時間ローソク足に対して計算されます。2つのしきい値レベルが買われ過ぎと売られ過ぎのエリアを定義します。CCIが極値に達した後にこれらの境界内に戻ったとき、戦略は平均回帰を期待して反対方向にポジションを取ります。

## 取引ルール

- **ロングエントリー**: 2本前のバーでCCIが高レベルを上回り、直前のバーでその下に移動した。
- **ショートエントリー**: 2本前のバーでCCIが低レベルを下回り、直前のバーでその上に移動した。
- **ロング決済**: 直前のバーのCCIが中間レベルを下回った。
- **ショート決済**: 直前のバーのCCIが中間レベルを上回った。

## パラメーター

- `CciPeriod` – CCIのルックバック期間。
- `HighLevel` – 買われ過ぎとみなされるCCIの上限しきい値。
- `MiddleLevel` – ポジション決済に使用される中間しきい値。
- `LowLevel` – 売られ過ぎとみなされるCCIの下限しきい値。
- `CandleType` – 計算に使用するローソク足シリーズ（デフォルト8時間）。

## 注意事項

この戦略は同時に最大1つのポジションを保有し、成行注文を使用します。デフォルトのリスク管理は`StartProtection`によって有効化されます。
