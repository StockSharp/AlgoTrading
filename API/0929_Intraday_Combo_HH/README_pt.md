# Estratégia Intraday Combo HH
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia combina múltiplos indicadores (Stoch RSI, MACD, Supertrend, Bollinger Bands e ADX).
Uma operação é aberta quando um número configurável de condições é atendido. Cooldown opcional e
stop-loss/take-profit são suportados.
