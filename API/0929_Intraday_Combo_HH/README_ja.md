# イントラデイ コンボ戦略 HH
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は複数のインジケーター（Stoch RSI、MACD、Supertrend、Bollinger Bands、ADX）を組み合わせます。
設定可能な数の条件が満たされたときにトレードが開かれます。オプションのクールダウンと
ストップロス/テイクプロフィットがサポートされています。
