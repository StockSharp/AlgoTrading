# Estrategia Intraday Combo HH
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia combina múltiples indicadores (Stoch RSI, MACD, Supertrend, Bollinger Bands y ADX).
Se abre una operación cuando se cumple un número configurable de condiciones. Se admiten un
tiempo de espera opcional y stop-loss/take-profit.
