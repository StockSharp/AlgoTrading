# Intraday Combo HH-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie kombiniert mehrere Indikatoren (Stoch RSI, MACD, Supertrend, Bollinger Bands und ADX).
Ein Trade wird eröffnet, wenn eine konfigurierbare Anzahl von Bedingungen erfüllt ist. Optionale
Abklingzeit und Stop-Loss/Take-Profit werden unterstützt.
