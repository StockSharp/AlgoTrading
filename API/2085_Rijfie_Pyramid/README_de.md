# Rijfie Pyramid-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie eröffnet eine erste Long-Position, wenn der Stochastic-Oszillator über ein konfigurierbares unteres Niveau steigt. Anschließend werden neue Positionen hinzugefügt, jedes Mal wenn der Kurs um einen festen Prozentsatz fällt, solange er über einem EMA-Filter und einem Mindestpreis bleibt. Ein optionaler Timer kann alle Positionen zu einem angegebenen Zeitpunkt schließen.

## Parameter
- Kerzentyp
- Stochastic-Untergrenze
- Höchstpreis für den ersten Einstieg
- Minimal zulässiger Preis
- EMA-Periode
- Stufenhöhe in Prozent
- Positionen zu Zeitpunkt schließen
- Schlussstunde
- Schlussminute
