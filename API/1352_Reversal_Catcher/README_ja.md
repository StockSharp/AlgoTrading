# リバーサル・キャッチャー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Reversal Catcher は、価格がボリンジャーバンドを突破してから戻り、モメンタムが転換するときにエントリーします。短期・長期 EMA でトレンド方向を定義し、RSI が買われすぎ・売られすぎレベルをクロスするタイミングでエントリーします。ターゲットとストップはボリンジャーバンドのレベルと前のローソク足の極値から導出されます。ポジションは指定した日中終了時刻にオプションで決済できます。

## 詳細

- **エントリー条件**: 価格がボリンジャーバンドに戻り、高値・安値が切り上がる構造を形成し、RSI が極値をクロスする。
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロス、ターゲット、またはデイエンドでフラット
- **ストップ**: 前のローソク足の極値
- **デフォルト値**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 1.5
  - `FastEmaPeriod` = 21
  - `SlowEmaPeriod` = 50
  - `RsiPeriod` = 14
  - `Overbought` = 70
  - `Oversold` = 30
  - `EndOfDay` = 1500
  - `CandleType` = 5分
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: Bollinger Bands, EMA, RSI
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
