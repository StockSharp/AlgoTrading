# Стратегия Reversal Catcher
[English](README.md) | [中文](README_cn.md)

Reversal Catcher входит в позицию, когда цена выходит за пределы полос Боллинджера и затем возвращается внутрь на фоне изменения импульса. Быстрые и медленные EMA определяют направление тренда, а пересечения RSI уровней перекупленности или перепроданности дают точку входа. Цели и стопы берутся из уровней полос Боллинджера и экстремума предыдущей свечи. Позиции могут закрываться в заданное время конца дня.

## Детали

- **Условия входа**: возврат цены в полосы Боллинджера с формированием более высоких максимумов/минимумов и пересечением RSI крайних уровней.
- **Длинные/короткие**: обе стороны
- **Условия выхода**: стоп, цель или закрытие в конце дня
- **Стопы**: экстремум предыдущей свечи
- **Значения по умолчанию**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 1.5
  - `FastEmaPeriod` = 21
  - `SlowEmaPeriod` = 50
  - `RsiPeriod` = 14
  - `Overbought` = 70
  - `Oversold` = 30
  - `EndOfDay` = 1500
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Разворот
  - Направление: обе стороны
  - Индикаторы: Bollinger Bands, EMA, RSI
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
