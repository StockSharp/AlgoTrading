# PowerHouse SwiftEdge AI v2.10 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ピボットベースのブレイクアウト戦略で、モメンタムフィルターとシンプルなリスク管理を備えています。
価格が直近のピボット高値を十分なモメンタムと前回シグナルからの最小距離で突破したときにロングエントリーが発生します。
ショートエントリーは、負のモメンタムでピボット安値を割り込んだときに発動します。
ポジションはポイント単位で指定されたテイクプロフィットとストップロスを使用します。
