# TradingConToto によるスキャルピング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradingConToto によるスキャルピング戦略は、EMA のトレンドに応じて連続するピボット高値またはピボット安値の間にラインを引きます。上昇トレンド中に価格が下降するピボット高値ラインを上抜けるとロングエントリーします。下降トレンド中に価格が上昇するピボット安値ラインを下抜けるとショートエントリーします。取引は指定されたセッション中のみ許可されます。

## 詳細

- **エントリー条件**: 上昇トレンドで価格が下降ピボット高値ラインをブレイクアウトした場合にロング；下降トレンドで価格が上昇ピボット安値ラインをブレイクアウトした場合にショート。
- **ロング/ショート**: 両方向。
- **エグジット条件**: テイクプロフィットとストップロス。
- **ストップ**: はい。
- **デフォルト値**:
  - `Pivot` = 16
  - `Pips` = 64
  - `Spread` = 0
  - `Session` = "0830-0930"
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: EMA, pivot
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
