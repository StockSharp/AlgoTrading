# Bbsr Extreme
[English](README.md) | [中文](README_cn.md)

**Bbsr Extreme** сочетает пробои полос Боллинджера с фильтром тренда на основе скользящей средней.
Лонг открывается при отбое от нижней полосы и росте средней, шорт — при откате от верхней полосы и снижении средней.
Выход осуществляется по стопу и тейку, рассчитанным через ATR.

## Детали
- **Условия входа**: пересечение полос с подтверждением тренда.
- **Длинная/короткая**: обе стороны.
- **Условия выхода**: стоп или тейк по ATR.
- **Стопы**: да, на базе ATR.
- **Параметры по умолчанию**:
  - `BollingerPeriod = 20`
  - `BollingerMultiplier = 2`
  - `MaLength = 7`
  - `AtrLength = 14`
  - `AtrStopMultiplier = 2`
  - `AtrProfitMultiplier = 3`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Bollinger Bands, EMA, ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
