# Стратегия парной торговли по соотношению цен
[English](README.md) | [中文](README_cn.md)

Стратегия работает с отношением цен двух коррелированных инструментов
(Asset1 / Asset2). К этому соотношению применяются полосы Боллинджера, которые
дают скользящее среднее и канал стандартных отклонений. По расстоянию текущего
отношения до среднего рассчитывается z-score, определяющий сигналы входа и
выхода.

Когда z-score превышает пороговое значение, отношение считается завышенным по
отношению к равновесному уровню: стратегия продаёт Asset1 и покупает Asset2.
Если z-score опускается ниже отрицательного порога, открывается
противоположная пара. Позиции закрываются, как только z-score возвращается
внутрь полосы выхода, то есть отношение подходит к среднему значению.

Параметр `HedgeRatio` задаёт объём второй ноги, чтобы пара оставалась
приблизительно долларово- или бета-нейтральной. Процентный стоп-лосс
устанавливается на каждую ногу и защищает на случай временного нарушения
взаимосвязи.

## Детали
- **Условия входа**:
  - **Длинная пара (Лонг Asset1 / Шорт Asset2)**: Z-Score отношения <= -`EntryZScore`
  - **Короткая пара (Шорт Asset1 / Лонг Asset2)**: Z-Score отношения >= `EntryZScore`
- **Лонг/Шорт**: Обе стороны.
- **Условия выхода**:
  - Модуль Z-Score опустился до `ExitZScore` или ниже.
- **Стопы**: Да, процентный стоп-лосс на каждую ногу.
- **Значения по умолчанию**:
  - `LookbackPeriod` = 20
  - `EntryZScore` = 2.0m
  - `ExitZScore` = 0.5m
  - `HedgeRatio` = 1.0m
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Арбитраж
  - Направление: Оба
  - Индикаторы: Полосы Боллинджера на отношении, Z-Score
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
