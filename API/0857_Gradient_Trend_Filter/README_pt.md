# Estratégia de Filtro de Tendência por Gradiente
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia aplica suavização exponencial tripla para detectar a direção da tendência. As operações são abertas quando a diferença do filtro cruza o zero e fechadas com sinais opostos.
