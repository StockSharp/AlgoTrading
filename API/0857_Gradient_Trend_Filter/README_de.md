# Gradienten-Trendfilter-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie wendet dreifache exponentielle Glättung an, um die Trendrichtung zu erkennen. Trades werden eröffnet, wenn die Filterdifferenz die Nulllinie kreuzt, und bei entgegengesetzten Signalen geschlossen.
