# Стратегия «Сходимость цены»
[English](README.md) | [中文](README_cn.md)

Стратегия оценивает вероятность роста или падения цены, сравнивая суммы OHLC4 для бычьих и медвежьих свечей. Когда вероятность роста превышает 50%, открывается длинная позиция, а когда вероятность падения выше 50% — короткая.

Тестирование показывает среднегодовую доходность около 37%. Наилучшие результаты достигаются на рынке криптовалют.

Стратегия может работать по всей истории или в скользящем окне, задаваемом параметром `Range`. Значение OHLC4 каждой свечи используется для взвешивания вклада роста и падения.

## Детали

- **Условия входа**: вероятность роста > 50% — покупка, вероятность падения > 50% — продажа.
- **Длинная/короткая**: обе стороны.
- **Условия выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `FullHistory` = true
  - `Range` = 200
  - `CandleType` = 1 minute
- **Фильтры**:
  - Категория: статистическая
  - Направление: обе
  - Индикаторы: пользовательские
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
