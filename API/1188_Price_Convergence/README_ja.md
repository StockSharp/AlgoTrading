# 価格収束戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、上昇ローソク足と下降ローソク足のOHLC4値の合計を比較することで、価格が上昇または下落する確率を推定します。上昇確率が50%を超えるとロングポジションを開き、下落確率が50%を超えるとショートポジションを開きます。

テストでは平均年間リターン約37%を示しています。暗号資産市場で最もよいパフォーマンスを発揮します。

この戦略は全履歴で動作するか、`Range`パラメータで定義されるローリングウィンドウで動作します。各ローソク足のOHLC4値は、上昇・下降の寄与度の重み付けに使用されます。

## 詳細

- **エントリー条件**: 上昇確率が50%を超えるとロングエントリー；下落確率が50%を超えるとショートエントリー。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `FullHistory` = true
  - `Range` = 200
  - `CandleType` = 1 minute
- **フィルター**:
  - カテゴリ: 統計的
  - 方向: 両方
  - インジケーター: カスタム
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
