# 价格收敛策略
[Русский](README_ru.md) | [English](README.md)

该策略通过比较上涨和下跌蜡烛的 OHLC4 值之和来估计价格上涨或下跌的概率。当上涨概率超过 50% 时开多仓，当下跌概率超过 50% 时开空仓。

测试显示年化收益约为 37%，在加密货币市场表现最佳。

策略可以使用全部历史数据，也可以使用由 `Range` 参数指定的滚动窗口。每根蜡烛的 OHLC4 值用于加权上涨和下跌的贡献。

## 详情

- **入场条件**：上涨概率大于 50% 开多仓，下跌概率大于 50% 开空仓。
- **多空方向**：双向。
- **离场条件**：出现反向信号。
- **止损**：无。
- **默认参数**：
  - `FullHistory` = true
  - `Range` = 200
  - `CandleType` = 1 minute
- **筛选条件**：
  - 分类：统计
  - 方向：双向
  - 指标：自定义
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
