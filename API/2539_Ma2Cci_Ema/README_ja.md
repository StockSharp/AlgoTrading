# Ma2Cci EMA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

2本の指数移動平均線のクロスオーバー戦略で、コモディティ・チャンネル・インデックス（CCI）のゼロライン突破によって確認されます。ポジションサイズとストップの配置は、Average True Range（ATR）のボラティリティと設定可能なリスクパーセンテージから導出されます。

## 詳細

- **データ**: 選択した`Candle Type`パラメーターによって供給される時間ベースのローソク足（デフォルト1時間）。
- **エントリー**: 速い方のEMAが遅い方のEMAを上回り、同じバーでCCIがゼロを上回ったときにロング；逆のクロスオーバーでCCIがゼロを下回ったときにショート。
- **エグジット**: 速い方のEMAが遅い方のEMAを下回るか、価格が固定ストップに触れたときにロングをクローズ；速い方のEMAが遅い方のEMAを上回るか、価格がショートストップに達したときにショートをクローズ。
- **リスク**: ストップ距離はATR（長さ`AtrPeriod`）と`MinStopPoints`に銘柄の価格ステップを掛けたものの、いずれか大きい方。取引サイズはポートフォリオ価値に`RiskPercent`を掛けて、そのストップ距離で割ったもの。
- **銘柄**: 元のMetaTraderバージョンでヘッジをサポートするトレンドフォローのforexまたは指数シンボル；明確なモメンタムのスイングがある他のアセットにも適用可能。
- **環境**: EMA/CCIシグナルがATRベースのリスク管理と整合するように連続セッション市場向けに設計されています。

## パラメーター

- `CandleType` – 計算と注文フローに使用する時間軸とデータタイプ。
- `FastMaPeriod` – 速い方のEMAの期間（デフォルト10）。
- `SlowMaPeriod` – 遅い方のEMAの期間（デフォルト37）。
- `CciPeriod` – モメンタムを確認するCCIオシレーターのルックバック（デフォルト39）。
- `AtrPeriod` – ストップ配置のための現在のボラティリティを推定するATRの長さ（デフォルト3）。
- `RiskPercent` – 1回の取引でリスクにさらされる現在のポートフォリオ資産の割合（デフォルト2%）。
- `MinStopPoints` – MetaTraderのpipフィルターをエミュレートするために価格ステップで表した最小ストップ距離（デフォルト15）。

## 注意事項

- EMA/CCIのクロスが信頼できる流動性の高いペアやインデックスで実行すると最もよく機能します；薄い市場では早期のエグジットを引き起こす可能性があります。
- ストップはエントリー時にのみ再計算されるため、戦略はリスクプロファイルを安定に保ち、元のMQLエキスパートの固定ストップロスロジックを反映します。
- ポジションサイジングが機能するには、ポートフォリオ評価を接続されたアカウントから提供する必要があります；そうでない場合、エンジンは戦略の`Volume`または銘柄の最小ボリュームにフォールバックします。
