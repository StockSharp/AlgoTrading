# 人工知能アクセラレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ビル・ウィリアムズの**加速/減速オシレーター（AC）**を基にした単純なパーセプトロンモデルを実装しています。0、7、14、21バーの遅延でオシレーターの4つの値をサンプリングし、調整可能な重みを掛けます。加重和が意思決定シグナルとして機能し、正の値は強気モメンタム、負の値は弱気モメンタムを示します。シグナルが符号を変えるたびにポジションを反転させ、エントリー価格から固定のストップロスを設定します。

AC自体はAwesome Oscillator（AO）からAOの5期間移動平均を引くことで導出されます。これにより、この戦略は市場の加速度変化に敏感になります。

## 詳細

- **エントリー条件**:
  - **ロング**: パーセプトロンシグナル > 0。
  - **ショート**: パーセプトロンシグナル < 0。
- **ロング/ショート**: 両方向。シグナルが反転すると逆転。
- **エグジット条件**:
  - エントリー価格からストップロスが発動。
  - シグナルがゼロを越えたときに反転。
- **ストップ**: あり、価格単位での固定ストップロス。
- **デフォルト値**:
  - `X1` = 76
  - `X2` = 47
  - `X3` = 153
  - `X4` = 135
  - `StopLoss` = 8355
  - `CandleType` = 1分足ローソク
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: AC（AOより導出）
  - ストップ: あり
  - 複雑さ: 中程度
  - 時間軸: 短期
  - ニューラルネットワーク: パーセプトロン
  - リスクレベル: 高
