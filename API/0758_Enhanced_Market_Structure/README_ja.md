# 強化版マーケット・ストラクチャー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

強化版マーケット・ストラクチャーは、スイング高値・安値分析とATR、RSI、出来高、MACD、EMAフィルターを組み合わせます。複数のフィルターがモメンタムを確認したとき、ブレイクアウトまたはスイープリバーサルでエントリーします。

## 詳細

- **エントリー条件**: フィルター付きの直近スイングのブレイクアウトまたはスイープ
- **ロング/ショート**: 両方
- **エグジット条件**: 逆シグナル
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = 1時間
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: ATR, RSI, MACD, EMA, 出来高
  - ストップ: なし
  - 複雑さ: 上級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
