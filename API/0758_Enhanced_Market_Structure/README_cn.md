# 增强型市场结构策略
[English](README.md) | [Русский](README_ru.md)

该策略结合摆动高低点与 ATR、RSI、成交量、MACD 和 EMA 滤波器。当多个条件确认动量时，在突破或扫单反转处入场。

## 详情

- **入场条件**：结合滤波器的摆动突破或扫单
- **多/空**：双向
- **出场条件**：反向信号
- **止损**：无
- **默认值**：
  - `CandleType` = 1 小时
- **筛选器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：ATR、RSI、MACD、EMA、成交量
  - 止损：无
  - 复杂度：高级
  - 周期：日内
  - 季节性：无
  - 神经网络：无
  - 背离：是
  - 风险等级：中等
