# Exp MAラウンディングチャネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は移動平均を固定のティックステップに丸め、その周囲にATRベースのチャネルを構築します。前のローソク足が上限バンドを上回って終値を付けた場合、ロングポジションを開きます。前のローソク足が下限バンドを下回って終値を付けた場合、ショートポジションを開きます。逆方向のシグナルで既存のポジションを決済します。ストップロスとテイクプロフィットはティックで定義され、自動的に管理されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 前のローソク足の終値が上限バンドを上回っている。
  - **ショート**: 前のローソク足の終値が下限バンドを下回っている。
- **エグジット条件**:
  - **ロング**: 前のローソク足の終値が下限バンドを下回っている。
  - **ショート**: 前のローソク足の終値が上限バンドを上回っている。
- **インジケーター**:
  - 指数移動平均。
  - チャネル幅のためのAverage True Range。
- **ストップ**: あり、ティック単位の固定ストップロスとテイクプロフィット。
- **デフォルト値**:
  - `MA period` = 12.
  - `ATR period` = 12.
  - `ATR factor` = 1.
  - `MA round` = 500 ティック.
  - `Stop loss` = 1000 ティック.
  - `Take profit` = 2000 ティック.
  - `Timeframe` = 4時間.
