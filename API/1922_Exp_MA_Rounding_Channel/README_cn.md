# 指数MA舍入通道策略
[English](README.md) | [Русский](README_ru.md)

该策略将移动平均线按固定的跳动步长进行舍入，并在其周围构建基于ATR的通道。当上一根K线收于上轨上方时开多仓；当上一根K线收于下轨下方时开空仓。相反信号会平掉当前仓位。止损和止盈以跳动点表示并自动管理。

## 细节

- **入场条件**：
  - **多头**：上一根K线收盘价高于上轨。
  - **空头**：上一根K线收盘价低于下轨。
- **出场条件**：
  - **多头**：上一根K线收盘价低于下轨。
  - **空头**：上一根K线收盘价高于上轨。
- **指标**：
  - 指数移动平均线。
  - 平均真实波幅用于通道宽度。
- **止损/止盈**：是，固定跳动点止损和止盈。
- **默认参数**：
  - `MA 周期` = 12。
  - `ATR 周期` = 12。
  - `ATR 系数` = 1。
  - `MA 舍入` = 500 跳动点。
  - `止损` = 1000 跳动点。
  - `止盈` = 2000 跳动点。
  - `时间框架` = 4 小时。

## 过滤器

- 类别：趋势跟随
- 方向：双向
- 指标：多个
- 止损：有
- 复杂度：中等
- 时间框架：中期
- 季节性：无
- 神经网络：无
- 背离：无
- 风险等级：中等
