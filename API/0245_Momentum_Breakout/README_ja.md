# モメンタムブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このブレイクアウトシステムは歴史的平均と比較してモメンタムの突然の急増を探します。モメンタムの読み取り値が平均を大きく超えると、価格が速い方向性のある動きを始める可能性があります。

テストでは平均年間リターンが約82%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

モメンタムが平均値に`Multiplier`倍の標準偏差を足した値を上回ったときに買いエントリーします。モメンタムが平均値から同じ乗数を引いた値を下回ったときにショートを開始します。モメンタムが平均値に向かって戻ったらポジションを決済します。

速い動きを好むトレーダーは、力強いバーストを捉えるための明確なルールを評価するかもしれません。価格のパーセンテージに基づいたストップロスが失敗したブレイクアウトから保護します。

## 詳細
- **エントリー条件**:
  - **ロング**: Momentum > Avg + Multiplier * StdDev
  - **ショート**: Momentum < Avg - Multiplier * StdDev
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: Momentum < Avg のときに決済
  - **ショート**: Momentum > Avg のときに決済
- **ストップ**: あり、パーセントストップロス。
- **デフォルト値**:
  - `MomentumPeriod` = 14
  - `AveragePeriod` = 20
  - `Multiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Momentum
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
