# Color Bears戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は二重平滑化されたBears Powerオシレーターを構築し、その傾きの変化に基づいて取引します。

## アイデア
1. 終値の指数移動平均（MA1）を計算する。
2. Bears Powerをローソク足の安値とMA1の差として計算する。
3. Bears PowerをもうひとつのEMA（MA2）でさらに平滑化する。
4. 平滑化された値の上昇・下降を追跡し、傾きの反転に反応する。

## 取引ルール
- インジケーターが上昇から下降に転換した場合（色 0 → 2）、ショートポジションを決済してロングポジションを開く。
- インジケーターが下降から上昇に転換した場合（色 2 → 0）、ロングポジションを決済してショートポジションを開く。
- 各ポジションの注文サイズには戦略の`Volume`プロパティを使用する。

## パラメーター
| 名前 | 説明 |
|------|------|
| `Ma1Period` | Bears Power構築に使用する最初のEMAの期間。 |
| `Ma2Period` | 平滑化EMAの期間。 |
| `CandleType` | 計算に使用するローソク足の時間軸。 |

## 備考
このC#実装はMQLエキスパート「ColorBears」（フォルダ `MQL/14314`）から移植されています。
アルゴリズムはStockSharpの標準インジケーターと高レベルAPIバインディングを使用しています。
