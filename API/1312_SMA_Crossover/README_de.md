# SMA-Kreuzungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet zwei Einfache Gleitende Durchschnitte (SMA): einen kurzperiodigen und einen langperiodigen.

- Kaufen, wenn der kurze SMA den langen SMA von unten kreuzt.
- Verkaufen, wenn der kurze SMA den langen SMA von oben kreuzt.

## Parameter
- **Kurze Länge** – Periode des kurzen SMA.
- **Lange Länge** – Periode des langen SMA.
- **Kerzentyp** – Zeitrahmen für die Berechnung.
