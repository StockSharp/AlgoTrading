# バイアス比率戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

バイアス比率戦略は、長期移動平均線からの価格乖離に基づくブレイクアウトを取引します。終値を指数移動平均線（EMA）と単純移動平均線（SMA）の両方と比較します。価格がEMAを指定した比率だけ上回るとロングポジションを建て、価格がSMAを同じ比率だけ下回るとショートポジションを建てます。

## 詳細

- **エントリー条件**:
  - `close / EMA >= 1 + BiasThreshold` → ロングエントリー。
  - `close / SMA <= 1 - BiasThreshold` → ショートエントリー。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 逆シグナルでポジションをクローズして反転。
- **ストップ**: なし。
- **デフォルト値**:
  - `MaPeriod` = 200
  - `BiasThreshold` = 0.025
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング & ショート
  - インジケーター: EMA, SMA
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
