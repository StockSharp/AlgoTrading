# Стратегия Bias Ratio
[English](README.md) | [中文](README_cn.md)

Стратегия Bias Ratio торгует пробои, основанные на отклонении цены от долгосрочных скользящих средних. Она сравнивает цену закрытия с экспоненциальной (EMA) и простой (SMA) средними. Длинная позиция открывается, когда цена превышает EMA на заданное отношение, короткая – когда цена опускается ниже SMA на то же отношение.

## Подробности

- **Условия входа**:
  - `close / EMA >= 1 + BiasThreshold` → вход в лонг.
  - `close / SMA <= 1 - BiasThreshold` → вход в шорт.
- **Направление**: оба направления.
- **Условия выхода**:
  - Противоположный сигнал закрывает и разворачивает позицию.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `MaPeriod` = 200
  - `BiasThreshold` = 0.025
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг и Шорт
  - Индикаторы: EMA, SMA
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
