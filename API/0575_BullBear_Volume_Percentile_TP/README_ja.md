# BullBear 出来高パーセンタイル TP 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はZ-Scoreで正規化されたBull/Bear Powerを使用します。
Z-Scoreが閾値を上抜けたときにロングポジションをオープンし、
負の閾値を下抜けたときにショートポジションをオープンします。
テイクプロフィットレベルは、出来高と価格パーセンタイルで調整されたATR乗数に基づいています。

## 詳細

- **エントリー条件:**
  - **ロング**: Z-Scoreが`ZThreshold`を上抜け。
  - **ショート**: Z-Scoreが`-ZThreshold`を下抜け。
- **ロング/ショート**: 両方。
- **エグジット条件**: Z-Scoreがゼロを逆方向に突破、またはテイクプロフィットレベルに到達。
- **ストップ**: ATR乗数によるテイクプロフィット。
- **デフォルト値:**
  - EMA長 21、Z-Score長 252、閾値 1.618。
  - ATR期間 20、乗数 1.618 / 2.382 / 3.618。
  - 出来高MA期間 100、パーセンタイル期間 100。
- **フィルター:**
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: EMA、ATR
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 中期
