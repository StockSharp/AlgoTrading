# Стратегия AcceleratorBot USDJPY H4
[English](README.md) | [中文](README_cn.md)

Стратегия AcceleratorBot представляет собой конвертацию оригинального эксперта MQL4 для пары USDJPY на таймфрейме H4. Она объединяет силу тренда через индекс направленного движения (ADX), импульс по стохастическому осциллятору и значения ускорения/замедления (AC) с нескольких таймфреймов. Свечные модели используются как фильтры направления.

## Подробности

- **Условия входа**: Сигналы тренда или импульса, подтверждённые свечными фильтрами.
- **Позиции**: Длинные и короткие.
- **Условия выхода**: Противоположный сигнал, стоп-лосс, тейк-профит или трейлинг-стоп.
- **Стопы**: Фиксированные и трейлинг.
- **Значения по умолчанию**:
  - `StopLossPoints` = 750
  - `TakeProfitPoints` = 9999
  - `TrailPoints` = 0
  - `AdxPeriod` = 14
  - `AdxThreshold` = 20m
  - `X1` = 0
  - `X2` = 150
  - `X3` = 500
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Тренд и импульс
  - Направление: Оба
  - Индикаторы: ADX, Stochastic, AC
  - Стопы: Да
  - Сложность: Продвинутая
  - Таймфрейм: H4
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
