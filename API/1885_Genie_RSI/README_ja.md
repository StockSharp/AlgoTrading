# Genie RSI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、相対強度指数（RSI）を使用して買われすぎと売られすぎのリバーサルを取引します。RSIが80を上回ると戦略はショートポジションを開き、RSIが20を下回るとロングポジションを開きます。任意のテイクプロフィットとトレーリングストップレベルがエントリー後のリスクを管理します。

この戦略は、価格がサポートとレジスタンスの間を頻繁に動くオシレート市場向けに設計されています。`CandleType`パラメータで定義された任意の時間軸で機能します。

## 詳細

- **エントリー条件**  
  - **ロング**: RSI値が完成したローソク足で20を下回り、ポジションが開いていない場合。  
  - **ショート**: RSI値が完成したローソク足で80を上回り、ポジションが開いていない場合。
- **エグジット条件**  
  - **ロング**: RSIが80を上回る、価格がテイクプロフィット距離に達する、または価格がトレーリングストップレベルに触れる。  
  - **ショート**: RSIが20を下回る、価格がテイクプロフィット距離に達する、または価格がトレーリングストップレベルに触れる。
- **インジケーター**: RSI。
- **パラメータ**:  
  - `RSI Period` – RSIインジケーターの長さ。  
  - `Take Profit` – 利益目標のための価格単位の距離。  
  - `Trailing Stop` – トレーリングストップのための価格単位の距離。  
  - `Candle Type` – 処理されるローソク足の時間軸。
- **ポジション管理**: エントリーとエグジットに成行注文を使用します。トレーリングストップは各完成ローソク足で再計算されます。
