# Стратегия Gold Breakout RR4
[English](README.md) | [中文](README_cn.md)

Стратегия Gold Breakout RR4 торгует пробои канала Дончиана на золоте, используя фильтр объёма и трендовый LWTI. В день выполняется не более одной сделки в заданном временном окне с фиксированным соотношением риск/прибыль 4:1.

## Детали

- **Условия входа**: пробой канала Дончиана с объёмом выше среднего и подтверждением LWTI в пределах сессии
- **Лонг/Шорт**: Оба направления
- **Условия выхода**: фиксированные стоп и цель по соотношению риск/прибыль
- **Стопы**: Да
- **Значения по умолчанию**:
  - `DonchianLength` = 96
  - `MaVolumeLength` = 30
  - `LwtiLength` = 25
  - `LwtiSmooth` = 5
  - `StartHour` = 20
  - `EndHour` = 8
  - `RiskReward` = 4
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Donchian Channel, SMA, WMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
