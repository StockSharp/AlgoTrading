# JFATL デジタルシステム
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

JurikMoving Average（JFATL）の傾きを中心に構築された戦略です。移動平均が上向きに転じるとロングポジションを開き、下向きに転じるとショートポジションを開きます。このアイデアはオリジナルMQLバージョンのカラーコードデジタルシステムを模倣しています。

## 詳細
- **エントリー条件**: Jurik移動平均の傾きが符号を変える。上向きの傾きでロングポジションを開き、下向きの傾きでショートポジションを開く。
- **ロング/ショート**: 両方向を取引する。
- **エグジット条件**: 逆方向の傾きでポジションが転換されるか、リスク管理によって閉じられる。
- **ストップ**: `StartProtection`で設定されるパーセンテージベースのテイクプロフィットとオプションのストップロス。
- **デフォルト値**: Length = 5, Phase = -100, Timeframe = 4 hours.
- **フィルター**: なし。この戦略はJMAの傾きのみに依存する。
