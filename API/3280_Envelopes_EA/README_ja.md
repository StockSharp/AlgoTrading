# Envelopes EA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略は、MetaTrader 4 のエキスパートアドバイザー "EnvelopesEA" を再現します。主要ローソク足ストリームに指数移動平均のエンベロープを適用し、Mean Reversion を取引します。市場がエンベロープの大きく外側へ押し出されると、逆張りの成行注文を送信します。価格が反対方向からエンベロープへ戻ると、ポジションは直ちに決済されます。元のエキスパートは 2019 年に EUR/USD でテストされました。StockSharp 版は同じロジックを維持し、主要な入力をすべて最適化可能なパラメーターとして公開します。

## 取引ロジック
1. 選択したローソク足で、長さ `EnvelopePeriod` の指数移動平均 (EMA) を計算します。
2. EMA をそれぞれ `UpperDeviationPercent` と `LowerDeviationPercent` で拡張し、上側と下側のエンベロープを作ります。
3. 早すぎる取引を避けるため、`EntryOffsetPoints` で定義される追加エントリーバッファー (商品の価格ステップを掛けたもの) を適用します。
4. オープンポジションがない場合:
   - 終値が下側エンベロープからエントリーバッファーを引いた水準を下回るとロングに入ります。
   - 終値が上側エンベロープにエントリーバッファーを加えた水準を上回るとショートに入ります。
5. ポジションがある場合:
   - 終値が上側エンベロープを上回って戻ると、ロングポジションを閉じます。
   - 終値が下側エンベロープを下回って戻ると、ショートポジションを閉じます。

戦略は常に最大 1 つのオープンポジションだけを保持し、エントリーとエグジットの両方に成行注文を使います。

## 資金管理
注文数量は `Volume` パラメーター (ロット) で直接指定します。自動 martingale やピラミッディングのルールはなく、スケーリング機能がデフォルトで無効だった最新 MQ4 実装と同じ動作を保ちます。

## パラメーター
| パラメーター | 説明 | デフォルト |
|-----------|------|------------|
| `Volume` | ロット単位の注文数量。 | 0.2 |
| `EnvelopePeriod` | エンベロープの基準を形成する EMA の長さ。 | 50 |
| `UpperDeviationPercent` | 上側バンドに適用する割合偏差。 | 0.5 |
| `LowerDeviationPercent` | 下側バンドに適用する割合偏差。 | 0.5 |
| `EntryOffsetPoints` | エントリー前に価格がバンドを超えて進む必要がある追加距離 (価格ステップ単位)。 | 100 |
| `CandleType` | ローソク足とインジケーター計算に使う時間枠。 | 30 分足 |

すべての数値パラメーター (`CandleType` を除く) は、元の最適化ワークフローを再現しやすいように最適化可能としてマークされています。

## 注意事項
- エンベロープは、以前のバージョンの SMA ではなく EMA を使います。これは MQ4 スクリプトの最新反復が指数ベースへ進化したためです。これにより価格変動への反応が速くなり、Mean Reversion のタイミングが改善されます。
- エントリーバッファーは商品の `PriceStep` を掛けます。銘柄メタデータに有効なステップサイズが含まれていることを確認してください。ない場合、戦略は保守的なデフォルト `0.0001` にフォールバックします。
- チャート表示には価格ローソク足、EMA エンベロープ、戦略の取引が含まれるため、元の Expert Advisor と比較してシグナル動作を簡単に検証できます。
