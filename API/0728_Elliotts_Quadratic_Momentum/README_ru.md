# Elliott's Quadratic Momentum
[English](README.md) | [中文](README_zh.md)

Стратегия **Elliott's Quadratic Momentum** использует несколько индикаторов SuperTrend для отслеживания волнового импульса.

Вход в лонг осуществляется, когда все четыре линии SuperTrend указывают на восходящий тренд, а вход в шорт — когда все линии указывают на нисходящий тренд. Позиция закрывается при изменении направления любого SuperTrend.

## Детали
- **Условия входа**: все индикаторы SuperTrend направлены в одну сторону.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**: любой SuperTrend разворачивается против позиции.
- **Стопы**: нет явных стопов.
- **Значения по умолчанию**:
  - `AtrLength1 = 7`
  - `Multiplier1 = 4.0m`
  - `AtrLength2 = 14`
  - `Multiplier2 = 3.618m`
  - `AtrLength3 = 21`
  - `Multiplier3 = 3.5m`
  - `AtrLength4 = 28`
  - `Multiplier4 = 3.382m`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе стороны
  - Индикаторы: SuperTrend
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
