# 月份效应策略
[English](README.md) | [Русский](README_ru.md)

该策略利用不同月份表现的统计差异。例如，股票在十一、十二月往往上涨，而九月相对疲弱。
根据历史平均值，在每月初选择做多或做空，到月末平仓。
如季节性走向未显现，则通过止损保护资金。

测试表明年均收益约为 88%，该策略在股票市场表现最佳。

## 细节

- **入场条件**：日历效应触发
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：季节性
  - 方向：双向
  - 指标：季节性
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：是
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
