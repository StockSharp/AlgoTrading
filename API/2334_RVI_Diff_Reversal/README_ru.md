# Стратегия RVI Diff Reversal
[English](README.md) | [中文](README_cn.md)

Стратегия торгует по сглаженной разнице между Relative Vigor Index (RVI) и его сигнальной линией.
Вход в длинную позицию происходит, когда эта разница перестаёт снижаться и начинает расти, и наоборот для короткой позиции.

## Детали

- **Критерий входа**: изменение наклона сглаженной разницы RVI
- **Длинные/Короткие**: обе стороны
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RviLength` = 12
  - `SmoothingLength` = 13
  - `CandleType` = 6-часовые свечи
- **Фильтры**:
  - Категория: осциллятор
  - Направление: обе
  - Индикаторы: RVI, SMA, EMA
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: 6H
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
