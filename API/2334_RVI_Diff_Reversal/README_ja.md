# RVI差分リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Relative Vigor Index (RVI)とそのシグナルラインの平滑化された差分に基づいて取引します。
この差分が下落を止めて上昇し始めるポイントを検出してロングに入り、ショートポジションはその逆です。

## 詳細

- **エントリー条件**: 平滑化RVI差分の傾きのリバーサル
- **ロング/ショート**: 両方
- **エグジット条件**: 反対のシグナル
- **ストップ**: いいえ
- **デフォルト値**:
  - `RviLength` = 12
  - `SmoothingLength` = 13
  - `CandleType` = 6時間足ローソク足
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: RVI、SMA、EMA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 6H
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
