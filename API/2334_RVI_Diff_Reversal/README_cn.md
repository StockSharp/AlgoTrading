# RVI Diff Reversal 策略
[English](README.md) | [Русский](README_ru.md)

该策略基于相对活力指数（RVI）与其信号线之间的平滑差值进行交易。
当该差值停止下降并开始上升时做多，反之做空。

## 详情

- **入场条件**：平滑 RVI 差值的斜率反转
- **多空方向**：双向
- **退出条件**：相反信号
- **止损**：无
- **默认值**：
  - `RviLength` = 12
  - `SmoothingLength` = 13
  - `CandleType` = 6 小时K线
- **过滤器**：
  - 类别：振荡器
  - 方向：双向
  - 指标：RVI、SMA、EMA
  - 止损：无
  - 复杂度：基础
  - 时间框架：6小时
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
