# Стратегия Ichimoku RSI MACD
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия, комбинирующая облако Ишимоку, RSI и сигналы пересечения MACD.

## Детали

- **Условия входа**: Цена выше/ниже облака Ишимоку с фильтром RSI и пересечением линий MACD.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Обратное пересечение MACD.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanBPeriod` = 52
  - `RsiLength` = 14
  - `RsiOverbought` = 70
  - `RsiOversold` = 30
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `CandleType` = TimeSpan.FromHours(1)
- **Фильтры**:
  - Категория: Trend Following
  - Направление: Оба
  - Индикаторы: Ichimoku, RSI, MACD
  - Стопы: Нет
  - Сложность: Начальная
  - Таймфрейм: Внутридневной (1ч)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
