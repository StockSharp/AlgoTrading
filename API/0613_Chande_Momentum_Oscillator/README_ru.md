# Стратегия Chande Momentum Oscillator
[English](README.md) | [中文](README_cn.md)

Стратегия покупает, когда индикатор Chande Momentum Oscillator опускается ниже заданного порога, и закрывает позицию, когда он поднимается выше верхнего порога или по истечении фиксированного числа баров.

Тестирование показывает среднегодовую доходность около 40%. Наилучшие результаты стратегия демонстрирует на трендовых рынках.

Индикатор сравнивает недавние прибыли и потери, чтобы оценить импульс. Сильные отрицательные значения сигнализируют о перепроданности, что используется для входа в длинную позицию. Позиция закрывается при восстановлении импульса или по истечении периода удержания.

## Детали

- **Условия входа**: `CMO < LowerThreshold`.
- **Длинные/короткие**: Только длинные.
- **Условия выхода**: `CMO > UpperThreshold` или `MaxBarsInPosition` баров прошло.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `CmoPeriod` = 9
  - `LowerThreshold` = -50
  - `UpperThreshold` = 50
  - `MaxBarsInPosition` = 5
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: Long
  - Индикаторы: CMO
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
