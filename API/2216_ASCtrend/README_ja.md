# ASCtrend戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はWilliams %Rインジケーターを使用してASCtrendアプローチに似た急速なリバーサルを検出します。インジケーターが売られすぎレベルから買われすぎレベルに上昇すると売り、その逆の場合は買います。

## 詳細

- **エントリー条件**:
  - Williams %Rが売られすぎ（`x2`以下）から買われすぎ（`x1`以上）にクロスすると売り。
  - Williams %Rが買われすぎ（`x1`以上）から売られすぎ（`x2`以下）にクロスすると買い。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆シグナルがポジションを決済して反転します。
- **ストップ**: なし。
- **デフォルト値**:
  - `Risk` = 4
  - `CandleType` = 1時間
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: Williams %R
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
