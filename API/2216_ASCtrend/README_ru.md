# Стратегия ASCtrend
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор Williams %R для выявления быстрых разворотов в духе ASCtrend. Продажа выполняется, когда индикатор выходит из зоны перепроданности в зону перекупленности, покупка — при обратном переходе.

## Детали

- **Условия входа**:
  - Продажа при переходе Williams %R из перепроданности (ниже `x2`) в перекупленность (выше `x1`).
  - Покупка при переходе Williams %R из перекупленности (выше `x1`) в перепроданность (ниже `x2`).
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Противоположный сигнал закрывает позицию и открывает обратную.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `Risk` = 4
  - `CandleType` = 1 час
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: Williams %R
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
