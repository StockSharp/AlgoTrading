# Autostop Cyriac 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このユーティリティ戦略は、アクティブな間、各トレードにテイクプロフィットとストップロスを自動的に付加します。自身ではエントリーやエグジットを作成せず、手動取引や他の戦略と組み合わせることができます。

## 詳細

- **エントリー条件**: なし。ポジションは手動または外部ロジックによって開かれます。
- **ロング/ショート**: ロングとショートの両方のポジションがサポートされています。
- **エグジット条件**: ポジションは付加されたテイクプロフィットまたはストップロスによってクローズされます。
- **ストップ**: あり。`StartProtection` を通じたテイクプロフィットとストップロスの絶対価格オフセット。
- **フィルター**: なし。

この戦略は2つのパラメーターを公開します:

- `TakeProfit` – 価格単位でのテイクプロフィットまでの距離。
- `StopLoss` – 価格単位でのストップロスまでの距離。
