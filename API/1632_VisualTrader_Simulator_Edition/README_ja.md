# VisualTrader シミュレーター・エディション
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTraderのVisualTraderスクリプトを簡略化して移植したものです。

選択した方向で単一の成行ポジションを建て、保護用のストップロスとテイクプロフィット注文を付加します。パラメーターにより、方向・テイクプロフィット・ストップロスを絶対価格値で設定できます。この戦略は、手動トレード管理スクリプトをStockSharpの高レベルAPIで再現する方法を示します。

## パラメーター

- **Trade Direction** – 初回注文の方向としてBuyまたはSellを選択。
- **Take Profit** – 絶対価格でのオプションのテイクプロフィット値。無効にするには0に設定。
- **Stop Loss** – 絶対価格でのオプションのストップロス値。無効にするには0に設定。
- **Volume** – 成行注文に使用する基本戦略ボリューム。

## トレーディングロジック

開始時に戦略は以下を行います:

1. `StartProtection`を使用して保護注文を作成。
2. 選択したトレード方向に基づいて成行注文を送信。

このサンプルはインジケーターや市場データに依存せず、デモンストレーション目的で設計されています。
