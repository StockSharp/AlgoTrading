# Estrategia de Funciones de Núcleo de Estadística Matemática
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demuestra varias funciones de núcleo estadístico y opera cuando la salida del núcleo seleccionado cruza el nivel 0.5.

## Parámetros
- **Kernel** – nombre de la función de núcleo (`uniform`, `triangle`, `epanechnikov`, `quartic`, `triweight`, `tricubic`, `gaussian`, `cosine`, `logistic`, `sigmoid`).
- **Bandwidth** – ancho de banda del núcleo.
- **Candle Type** – marco temporal de las velas.
