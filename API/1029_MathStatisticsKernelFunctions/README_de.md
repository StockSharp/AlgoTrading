# Strategie mit Mathematisch-Statistischen Kernelfunktionen
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Demonstriert mehrere statistische Kernelfunktionen und handelt, wenn der Ausgang des gewählten Kernels den Wert 0.5 kreuzt.

## Parameter
- **Kernel** – Name der Kernelfunktion (`uniform`, `triangle`, `epanechnikov`, `quartic`, `triweight`, `tricubic`, `gaussian`, `cosine`, `logistic`, `sigmoid`).
- **Bandwidth** – Kernelbandbreite.
- **Candle Type** – Zeitrahmen der Kerzen.
