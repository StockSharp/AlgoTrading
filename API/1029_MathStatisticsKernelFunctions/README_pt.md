# Estratégia de Funções de Kernel de Estatística Matemática
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Demonstra várias funções de kernel estatístico e negocia quando a saída do kernel selecionado cruza 0.5.

## Parâmetros
- **Kernel** – nome da função de kernel (`uniform`, `triangle`, `epanechnikov`, `quartic`, `triweight`, `tricubic`, `gaussian`, `cosine`, `logistic`, `sigmoid`).
- **Bandwidth** – largura de banda do kernel.
- **Candle Type** – período de tempo das velas.
