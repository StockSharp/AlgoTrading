# D-BoT Alpha ショート SMA と RSI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSIが閾値を上回り、価格が単純移動平均線を下回っている場合に売りエントリーするショート戦略です。トレーリングストップが新しい安値に追従し、RSIがストップまたはテイクプロフィットレベルに達した場合にポジションをクローズします。

## 詳細

- **エントリー条件**: RSIがエントリーレベルを上抜けし、価格がSMAを下回っている。
- **エグジット条件**: 価格がトレーリングストップを上抜けするか、RSIがストップまたはテイクプロフィットレベルに達した時。
