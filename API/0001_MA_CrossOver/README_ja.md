# HMA 季節性ダイバージェンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
この戦略は、ハル移動平均線（HMA）と季節性の建玉クラスタリングを組み合わせ、価格とマーケットポジショニングのダイバージェンスを見つけます。価格が上昇する建玉の方向に一時的に逆行する場合、トレンド継続の可能性が高いと想定しています。このシステムはロングとショートの両方を取引するよう設計されており、HMAの傾きでモメンタムを測り、季節性の建玉データで参加レベルを測定します。

テストでは年平均リターンが約40%を示しています。暗号資産市場で最もよく機能します。

トレードセットアップは、HMAが直前バーと比較して変化し、季節性の建玉がその動きを確認する一方で、価格が逆方向に動いたときに発生します。この価格とポジショニングの強気または弱気のダイバージェンスは、より大きなトレンドの中での短期的な押し目・戻りの終わりを示すことが多いです。この条件が整うのを待ってからエントリーし、リスク管理のためにボラティリティベースのストップを置きます。

HMAの傾きが反転したとき、モメンタムが転換したことを示すためポジションをクローズします。ストップレベルはAverage True Range（ATR）の倍数を使用するため、リスクは市場のボラティリティに適応します。これにより、拡大局面での早期退出を防ぎ、ボラティリティが収縮したときの損失を抑えます。

## 詳細

- **エントリー条件**:
  - **ロング**: `HMA(t) > HMA(t-1)` && `OI_Cluster_Seasonal(t) > OI_Cluster_Seasonal(t-1)` && `Price(t) < Price(t-1)` （強気ダイバージェンス）。
  - **ショート**: `HMA(t) < HMA(t-1)` && `OI_Cluster_Seasonal(t) < OI_Cluster_Seasonal(t-1)` && `Price(t) > Price(t-1)` （弱気ダイバージェンス）。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: `HMA(t) < HMA(t-1)` （HMAが下落し始める）。
  - **ショート**: `HMA(t) > HMA(t-1)` （HMAが上昇し始める）。
- **ストップ**: はい、エントリーから `N * ATR` の位置にストップロスを置く。
- **デフォルト値**:
  - `HMA period` = 9。
  - `OI_Cluster_Seasonal` = 5年間のクラスターレベルにおける季節性の建玉。
  - `N` = 2 （ストップロス = `2 * ATR`）。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: はい
  - 複雑さ: 複雑
  - 時間軸: 中期
  - 季節性: はい
  - ニューラルネットワーク: はい
  - ダイバージェンス: はい
  - リスクレベル: 高
