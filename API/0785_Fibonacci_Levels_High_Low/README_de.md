# Fibonacci-Hoch-Tief-Niveau-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Verwendet Fibonacci-Niveaus einer übergeordneten Zeitrahmen-Kerze zusammen mit Kriterien für Höchst- und Tiefstkurse.
Kauft, wenn der Schlusskurs über dem vorherigen tiefsten Tief und dem Fibonacci-Niveau liegt. Verkauft, wenn der Schlusskurs unter dem vorherigen höchsten Hoch und demselben Niveau liegt.
