# Estrategia Fibonacci de Niveles Máximo/Mínimo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Utiliza el nivel Fibonacci de una vela de marco temporal superior junto con criterios de precio máximo y mínimo.
Compra cuando el cierre está por encima del mínimo previo más bajo y del nivel Fibonacci. Vende cuando el cierre está por debajo del máximo previo más alto y del mismo nivel.
