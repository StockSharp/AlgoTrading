# Стратегия F2a AO
[English](README.md) | [中文](README_cn.md)

Эта стратегия воспроизводит оригинального советника MetaTrader «F2a_AO». Осциллятор Awesome фильтруется короткой SMA и сделки открываются только в направлении свечи на старшем таймфрейме.

Индикатор рассчитывается на отдельном таймфрейме. Если контрольная свеча закрывается выше открытия, положительный фильтрованный AO открывает длинную позицию и закрывает короткие. Если свеча закрывается ниже открытия, отрицательный фильтрованный AO открывает короткую позицию и закрывает длинные.

## Подробности

- **Условия входа**:
  - **Лонг**: контрольная свеча бычья и фильтрованный AO > 0.
  - **Шорт**: контрольная свеча медвежья и фильтрованный AO < 0.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - Фильтрованный AO < 0 закрывает лонг.
  - Фильтрованный AO > 0 закрывает шорт.
- **Стопы**: явный стоп-лосс и тейк-профит отсутствуют, модуль защиты включён.
- **Значения по умолчанию**:
  - `IndicatorTimeFrame` = 12 часов.
  - `TrendTimeFrame` = 1 день.
  - `FastPeriod` = 13.
  - `SlowPeriod` = 144.
  - `FilterLength` = 3.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе стороны
  - Индикаторы: Awesome Oscillator, SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
