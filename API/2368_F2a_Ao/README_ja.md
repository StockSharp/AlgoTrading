# F2a AO戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はオリジナルのMetaTraderエキスパートアドバイザー「F2a_AO」を再現しています。Awesome Oscillatorを短期SMAでフィルタリングし、上位時間軸の参照ローソク足の方向にのみ取引を建てます。

オシレーターは独自の時間軸で計算されます。参照ローソク足が始値を上回って終値をつけた場合、正のフィルタリングされたAOがロングエントリーをトリガーしてショートポジションを決済します。参照ローソク足が始値を下回って終値をつけた場合、負のフィルタリングされたAOがショートエントリーをトリガーしてロングポジションを決済します。

## 詳細

- **エントリー条件**:
  - **ロング**: 参照ローソク足が強気でフィルタリングされたAO > 0。
  - **ショート**: 参照ローソク足が弱気でフィルタリングされたAO < 0。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - フィルタリングされたAO < 0でロングポジションを決済。
  - フィルタリングされたAO > 0でショートポジションを決済。
- **ストップ**: 明示的なストップロスやテイクプロフィットなし、保護モジュールが有効。
- **デフォルト値**:
  - `IndicatorTimeFrame` = 12時間。
  - `TrendTimeFrame` = 1日。
  - `FastPeriod` = 13。
  - `SlowPeriod` = 144。
  - `FilterLength` = 3。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Awesome Oscillator, SMA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
