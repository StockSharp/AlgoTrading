# Стратегия XDerivative
[English](README.md) | [中文](README_cn.md)

Стратегия XDerivative отслеживает изменения импульса цены через сглаженную производную. Оригинальный эксперт MQL сочетает расчёт Rate of Change с сглаживанием Jurik для определения точек разворота. В версии StockSharp используются встроенные индикаторы для реализации той же идеи.

Индикатор вычисляет Rate of Change за `RocPeriod` баров и сглаживает его Jurik Moving Average длиной `MaLength`. Когда сглаженная производная формирует впадину (предыдущее значение ниже позапрошлого, а текущее поднимается выше предыдущего), стратегия открывает или переворачивает позицию в покупку. При формировании вершины (предыдущее значение выше позапрошлого, а текущее опускается ниже), стратегия открывает или переворачивает позицию в продажу. Выход контролируется защитными стопами.

## Подробности

- **Критерии входа**:
  - Лонг: производная разворачивается вверх после локального минимума.
  - Шорт: производная разворачивается вниз после локального максимума.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: обратный разворот производной или защитный стоп.
- **Стопы**: да, процентные take-profit и stop-loss.
- **Значения по умолчанию**:
  - `RocPeriod` = 34
  - `MaLength` = 7
  - `TakeProfitPercent` = 2
  - `StopLossPercent` = 1
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Трендовая
  - Направление: Обе стороны
  - Индикаторы: RateOfChange, JurikMovingAverage
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: 4H
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
