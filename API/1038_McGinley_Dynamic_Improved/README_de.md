# McGinley Dynamic (Verbessert)-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Implementiert den Indikator "McGinley Dynamic (Improved)" von John R. McGinley, Jr. und handelt, wenn der Schlusskurs die dynamische Linie kreuzt. Die Strategie unterstützt moderne, originale und benutzerdefinierte Koeffizientenformeln und kann optional die unkontrollierte Variante zum Vergleich anzeigen.

## Details

- **Einstieg Long**: Schlusskurs kreuzt McGinley Dynamic nach oben.
- **Einstieg Short**: Schlusskurs kreuzt McGinley Dynamic nach unten.
- **Indikatoren**: McGinley Dynamic, optionales Unconstrained McGinley Dynamic, EMA als Referenz.
- **Standardwerte**: Period = 14, Formula = Modern, Custom k = 0.5, Exponent = 4.
- **Richtung**: Beide.
