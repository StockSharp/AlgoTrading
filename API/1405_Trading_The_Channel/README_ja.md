# チャネル・トレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は線形回帰チャネルを使用してシグナルを生成します。

3つの取引ルールをサポートしています：

1. トレンドに乗る取引 – 傾きの変化でエントリー。
2. チャネルブレイクアウトの取引 – チャネル上下のブレイクでエントリー。
3. チャネル内での取引 – オプションのトレンドフィルターを使用してチャネル内の動きに逆張りする。

パラメーターには期間、ゾーンのパーセント、レンジソース、取引ルール、トレンドフィルター、ロングのみオプションが含まれます。
