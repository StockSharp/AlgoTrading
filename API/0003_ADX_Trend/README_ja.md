# ADXトレンド
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

平均方向指数（ADX）トレンドに基づく戦略。ADXトレンド戦略は、ADXインジケーターを使用して市場の強さを測定します。ADXが閾値を上回り、価格が移動平均の正しい側にある場合、システムはその方向に取引します。ADXが弱まるか、逆の条件が現れると、ポジションが閉じられます。

テスト結果では、年間平均リターンが約46%であることが示されています。株式市場で最も効果を発揮します。

しっかりしたADX値を待つことで、このアプローチはモメンタムが確固として確立されている時のみ取引します。ストップは通常ATRの倍数を使用し、リスクがボラティリティに応じて調整されます。


## 詳細

- **エントリー条件**: MA、ADX、ATRに基づくシグナル。
- **ロング/ショート**: 両方の方向。
- **エグジット条件**: 逆シグナルまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `AdxPeriod` = 14
  - `MaPeriod` = 50
  - `AtrMultiplier` = 2m
  - `AdxExitThreshold` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: MA, ADX, ATR
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
