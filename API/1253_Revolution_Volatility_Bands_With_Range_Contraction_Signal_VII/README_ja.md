# Revolution ボラティリティバンドとレンジ収縮シグナルVII戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は指数移動平均を使用して価格の周りにエンベロープを構築し、バンド間の距離が収縮するタイミングを検出する。収縮が観察され、価格が平滑化されたバンドを上方または下方にブレイクした際、ブレイクアウトの方向にポジションを建てる。

## 詳細

- **エントリー条件**:
  - **ロング**: レンジが収縮しており、終値が上側の平滑化バンドを上抜けする。
  - **ショート**: レンジが収縮しており、終値が下側の平滑化バンドを下抜けする。
- **エグジット条件**: 逆方向のブレイクアウト。
- **インジケーター**: EMAベースのエンベロープ。
- **時間軸**: 任意。
