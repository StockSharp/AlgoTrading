# Delta-RSI Oscillator
[English](README.md) | [中文](README_cn.md)

Стратегия использует осциллятор Delta-RSI, представляющий собой изменение RSI, сглаженное EMA. Сигналы формируются при пересечении нуля, пересечении сигнальной линии или смене направления. Выход осуществляется по выбранному условию.

## Детали

- **Критерии входа**: Зависит от `BuyCondition` (пересечение нуля, сигнальной линии или смена направления) по Delta-RSI.
- **Длинные/короткие**: Оба варианта, управляется `UseLong` и `UseShort`.
- **Критерии выхода**: Основаны на `ExitCondition` по Delta-RSI.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `RsiLength` = 21
  - `SignalLength` = 9
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: RSI, EMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
