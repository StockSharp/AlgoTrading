# Dskyz (DAFE) MAtrix com Precisão Potenciada por ATR
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia entra comprada quando uma média móvel rápida está acima de uma média móvel lenta com filtros de ATR, volume e tempo. Posições vendidas são tomadas nas condições opostas.

## Parâmetros
- Tipo de vela
- Tipo de MA rápida
- Comprimento da MA rápida
- Tipo de MA lenta
- Comprimento da MA lenta
- Período ATR
- Multiplicador ATR
- Usar filtro de tendência
- Volume mínimo
- Limiar de volatilidade
- Hora de início das operações
- Hora de fim das operações
- Multiplicador ATR para stop trailing
- Multiplicador ATR para meta de lucro
- Multiplicador ATR para stop fixo
- Quantidade de operações
