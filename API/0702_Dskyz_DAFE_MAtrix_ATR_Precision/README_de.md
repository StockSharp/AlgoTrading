# Dskyz (DAFE) MAtrix mit ATR-gestützter Präzision
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie eröffnet Long-Positionen, wenn ein schneller gleitender Durchschnitt über einem langsamen gleitenden Durchschnitt liegt, mit ATR-, Volumen- und Zeitfiltern. Short-Positionen werden bei entgegengesetzten Bedingungen eingegangen.

## Parameter
- Kerzentyp
- Typ des schnellen MA
- Länge des schnellen MA
- Typ des langsamen MA
- Länge des langsamen MA
- ATR-Periode
- ATR-Multiplikator
- Trendfilter verwenden
- Mindestvolumen
- Volatilitätsschwelle
- Handelsstartzeit
- Handelsendzeit
- ATR-Multiplikator für Trailing-Stop
- ATR-Multiplikator für Gewinnziel
- ATR-Multiplikator für festen Stop
- Handelsmenge
