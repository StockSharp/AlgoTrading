# Dskyz (DAFE) MAtrix con Precisión Potenciada por ATR
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia entra en largo cuando una media móvil rápida está por encima de una media móvil lenta con filtros de ATR, volumen y tiempo. Las posiciones cortas se toman en las condiciones opuestas.

## Parámetros
- Tipo de vela
- Tipo de MA rápida
- Longitud de MA rápida
- Tipo de MA lenta
- Longitud de MA lenta
- Período ATR
- Multiplicador ATR
- Usar filtro de tendencia
- Volumen mínimo
- Umbral de volatilidad
- Hora de inicio de operaciones
- Hora de fin de operaciones
- Multiplicador ATR para stop trailing
- Multiplicador ATR para objetivo de beneficio
- Multiplicador ATR para stop fijo
- Cantidad de operaciones
