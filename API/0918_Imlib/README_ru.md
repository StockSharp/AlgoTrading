# Библиотека ImLib
[English](README.md) | [中文](README_cn.md)

Эта стратегия-утилита предоставляет функции для загрузки и отображения небольших изображений, закодированных в строке. Реализована RLE-декомпрессия, парсер метаданных изображения и отрисовка через коллбэк для каждого пикселя.

## Подробности

- **Критерии входа**: Нет, утилита.
- **Длинные/короткие**: Не применимо.
- **Критерии выхода**: Не применимо.
- **Стопы**: Нет.
- **Значения по умолчанию**: Нет.
- **Фильтры**:
  - Категория: Утилита
  - Направление: Нет
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Не применимо
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
