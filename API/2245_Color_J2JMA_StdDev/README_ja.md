# Color J2JMA StdDev 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はJurik移動平均（JMA）の傾きを計算し、最近の傾きの標準偏差と比較します。傾きが最近のボラティリティの倍数を超えたときに、強い方向性のある動きを捉えることを目的としています。

JMAの傾きが高い閾値（K2 × 標準偏差）を上回ると、新規ロングポジションが開設されます。傾きがマイナスの高い閾値を下回ると、新規ショートポジションが開設されます。傾きが反対側の低い閾値（K1 × 標準偏差）を越えると、既存ポジションが決済されます。ストップロスとテイクプロフィットのレベルはエントリー価格からのポイントで適用されます。

パラメーター:
- **JMA Length** – Jurik移動平均の期間。
- **StdDev Period** – 標準偏差に使用する直近の傾きの数。
- **K1** – ポジション決済に使用する低い閾値の乗数。
- **K2** – ポジション開設に使用する高い閾値の乗数。
- **Candle Type** – 計算に使用するローソク足の時間軸。
- **Stop Loss** – ポイント単位の防護ストップ。
- **Take Profit** – ポイント単位の利益目標。
