# ダブルMAクロスオーバーブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略はStockSharpフレームワーク内でMetaTraderエキスパートアドバイザー「DoubleMA Crossover」を再現します。ロジックは高速と低速のシンプル移動平均を監視し、方向性クロスオーバーを待ってから、市場に入る前にブレイクアウト確認を必要とします。アルゴリズムは一度に1つのポジションのみを管理し、元の3つのトレーリングモードを模倣するオプションのトレーリングストップ動作を含みます。

## 機能の仕組み

1. **シグナル検出** – 2つのシンプル移動平均（デフォルト：2と5）が選択されたローソク足シリーズで計算されます。高速平均が低速平均を上回りクロスすると強気クロスオーバーが発生し、その逆が弱気クロスオーバーです。
2. **ブレイクアウト確認** – クロスオーバー後、戦略は価格ステップ（`BreakoutPips`）で定義されたブレイクアウトレベルを保存します。ポジションは価格が後続のローソク足でそのレベルに到達した場合にのみ開かれ、MQLバージョンのストップ注文動作を複製します。
3. **ポジション管理** – 1つのポジションのみが許可されます。取引がアクティブな間、戦略はストップロス、テイクプロフィット、設定されたトレーリングストップタイプを監視します。内部トラッカーはバックテストでの動作を確定的に保つためにブローカー側の実行をエミュレートします。
4. **セッションフィルター** – 取引は特定の時間ウィンドウ（`StartHour`..`StopHour`）に制限できます。戦略はウィンドウ外でもオープンな取引を管理しますが、フィルターが取引をブロックする場合は新しいブレイクアウトレベルを作成しません。
5. **トレーリングストップ** – 3つのトレーリングモードがサポートされています：初期ストップ距離での即時トレーリング、カスタム距離後のトレーリング、元のEAと同様のブレークイーブンシフトを伴う3レベルロジック。

## パラメーター

| パラメーター | 説明 |
|-----------|-------------|
| `FastMaPeriod`, `SlowMaPeriod` | 高速と低速のシンプル移動平均の期間。 |
| `BreakoutPips` | ブレイクアウトトリガーを定義するためにシグナルローソク足のクローズに追加される価格ステップの距離。 |
| `StopLossPips`, `TakeProfitPips` | 価格ステップでの保護ストップとオプションのテイクプロフィット。テイクプロフィットをゼロに設定して無効にする。 |
| `UseTrailingStop` | トレーリングストップ管理を有効にする。 |
| `TrailingMode` | トレーリングタイプ：Type1は元のストップ距離を使用、Type2はカスタム距離（`TrailingStopPips`）を待つ、Type3は3つのMQLレベルを使用。 |
| `TrailingStopPips` | Type2トレーリングの距離。 |
| `Level1TriggerPips`, `Level1OffsetPips` | Type3トレーリングの最初のトリガーレベルとオフセット（デフォルトでストップをブレークイーブンに移動）。 |
| `Level2TriggerPips`, `Level2OffsetPips` | Type3トレーリングの2番目のトリガーレベルとオフセット。 |
| `Level3TriggerPips`, `Level3OffsetPips` | Type3トレーリングの3番目のトリガーレベルとオフセット（クラシックなトレーリングストップに変換）。 |
| `UseTimeLimit`, `StartHour`, `StopHour` | 取引セッションフィルターを有効にし、包括的な時間範囲を定義する。 |
| `CandleType` | シグナル計算に使用されるローソク足シリーズ。 |
| `TradeVolume` | ロットで表現された注文ボリューム。 |

## トレーリングストップモード

- **Type1** – 価格がその量だけ進んだら元のストップロス距離を使用してストップを移動します。
- **Type2** – トレーリングを始める前に価格が`TrailingStopPips`移動するまで待ち、その後その距離で利益を固定します。
- **Type3** – 3つのレベルを使用：最初の2つは定義されたオフセットでストップをシフトし、3番目は現在のクローズと`Level3OffsetPips`を使用して継続的なトレーリングストップに変換します。

## 使用上のヒント

- MetaTraderエキスパートアドバイザーと同じ動作を維持するために`BreakoutPips`をインストゥルメントのティックサイズに合わせます。
- 取引時間に合わせるためにセッションフィルターを確認してください。デフォルトでは現地時間の11:00から16:00の間のエントリーが許可されます。
- 24/7インストゥルメントの場合はタイムフィルターを無効にします（`UseTimeLimit = false`）。
- トレーリングタイプ3をテストする場合、オフセット値が対応するトリガーレベルより大きくないことを確認してください。そうでないと、ストップがエントリー価格の後ろに残る可能性があります。
