# Стратегия пробоя цены и объёма
[English](README.md) | [中文](README_cn.md)

Стратегия входит в позицию, когда цена и объём одновременно пробивают свои максимумы за период, при этом цена находится выше скользящей средней. Короткие позиции открываются при пробое минимума на повышенном объёме и цене ниже SMA. Выход из позиции осуществляется после пяти последовательных закрытий по другую сторону SMA.

## Подробности
- **Условия входа**:
  - **Long**: закрытие > предыдущего максимума && объём > предыдущего максимального объёма && закрытие > SMA
  - **Short**: закрытие < предыдущего минимума && объём > предыдущего максимального объёма && закрытие < SMA
- **Направление**: настраиваемое
- **Условия выхода**:
  - **Тренд**: пять закрытий по другую сторону SMA
- **Стопы**: нет
- **Значения по умолчанию**:
  - `PriceBreakoutPeriod` = 60
  - `VolumeBreakoutPeriod` = 60
  - `TrendlineLength` = 200
  - `OrderDirection` = "Long"
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Настраиваемое
  - Индикаторы: Highest, SMA, Volume
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
