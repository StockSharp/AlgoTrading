# Estrategia de Cuadrícula Basada en el Canal Keltner
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia ajusta el tamaño de la posición en función de la distancia relativa del precio respecto a una media móvil dentro de un Canal Keltner. Cuando el precio se desvía de la media, la estrategia reequilibra utilizando órdenes de mercado en una cuadrícula.
