# Estratégia de Grade Baseada no Canal Keltner
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia ajusta o tamanho da posição com base na distância relativa do preço em relação a uma média móvel dentro de um Canal Keltner. Quando o preço se desvia da média, a estratégia reequilibra usando ordens de mercado em uma grade.
