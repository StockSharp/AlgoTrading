# FatlSatlOsma戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この例はStockSharpの高レベルAPIを使ってMetaTraderエキスパート**Exp_FatlSatlOsma**のロジックを再現しています。  
元のシステムはFatl/SatlオシレーターというMACDに類似したカスタムインジケーターで動作します。  
戦略はオシレーターの方向変化を探します：

- オシレーターが2バー上昇し、最後の値が前の値より高い場合、ロングポジションが開かれ、ショートポジションが決済されます。
- オシレーターが2バー下落し、最後の値が前の値より低い場合、ショートポジションが開かれ、ロングポジションが決済されます。

オシレーターは設定可能な短期と長期の期間を持つ組み込みの`MovingAverageConvergenceDivergenceSignal`インジケーターを通して実装されています。  
デフォルト値は元のFATL/SATLパラメーターに対応しています。

## 詳細

- **エントリー条件**: オシレーターの加速。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対方向の加速。
- **ストップ**: なし。
- **デフォルト値**:
  - `Fast` = 39
  - `Slow` = 65
  - `CandleType` = 12時間の時間軸
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: MACD
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
