# Стратегия FatlSatlOsma
[English](README.md) | [中文](README_cn.md)

Пример реализует логику советника MetaTrader **Exp_FatlSatlOsma** на высокоуровневом API StockSharp.  
Исходная система использует осциллятор Fatl/Satl (аналог MACD).

- Когда осциллятор растет две свечи подряд и последнее значение выше предыдущего, открывается длинная позиция и закрываются короткие.
- Когда осциллятор падает две свечи подряд и последнее значение ниже предыдущего, открывается короткая позиция и закрываются длинные.

Осциллятор построен на встроенном индикаторе `MovingAverageConvergenceDivergenceSignal`.  
Параметры по умолчанию соответствуют оригинальным настройкам FATL/SATL.

## Детали

- **Условия входа**: ускорение осциллятора.
- **Длинные/короткие**: обе стороны.
- **Условия выхода**: противоположное ускорение.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `Fast` = 39
  - `Slow` = 65
  - `CandleType` = таймфрейм 12 часов
- **Фильтры**:
  - Категория: Моментум
  - Направление: Обе
  - Индикаторы: MACD
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
