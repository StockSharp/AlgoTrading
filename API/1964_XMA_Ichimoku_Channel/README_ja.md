# XMA Ichimoku チャネル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略は、XMA Ichimoku のコンセプトに基づいたチャネルブレイクアウトシステムを実装しています。直近の高値と安値の平滑化平均を中心に動的チャネルを構築し、価格行動がプルバックを伴うブレイクアウトを確認したときにトレードを生成します。

## 仕組み

1. **最高値と最低値**: 各完成済みローソク足について、設定可能なルックバック期間にわたる最高値と最安値を計算します。
2. **平滑化中心線**: 最高値と最低値の中点を単純移動平均で平滑化します。
3. **チャネルの構築**: 上位バンドと下位バンドは、平滑化中心線にパーセンテージオフセットを適用して導出されます。
4. **トレードロジック**:
   - 前回の終値が前回の上位バンドを上回っており、現在の終値が現在の上位バンドを下回って戻った場合、戦略はロングポジションを開き、既存のショートポジションをクローズします。
   - 前回の終値が前回の下位バンドを下回っており、現在の終値が現在の下位バンドを上回って戻った場合、戦略はショートポジションを開き、既存のロングポジションをクローズします。

## パラメーター

- **Up Period** – 高値のルックバック期間。
- **Down Period** – 安値のルックバック期間。
- **MA Length** – 平滑化移動平均の長さ。
- **Up Percent** – 上位バンドを形成するために中心線に加算するパーセンテージ。
- **Down Percent** – 下位バンドを形成するために中心線から減算するパーセンテージ。
- **Candle Type** – 計算に使用するローソク足の時間軸。

## 使用上の注意

- トレードは成行注文で実行されます。
- 偽シグナルを避けるため、完成済みのローソク足のみが処理されます。
- 戦略は新しいポジションを開く前に反対ポジションをクローズします。
