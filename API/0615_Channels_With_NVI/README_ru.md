# Стратегия Channels with NVI
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует полосы Боллинджера или каналы Кельтнера в сочетании с индикатором отрицательного объёма (NVI). Длинная позиция открывается, когда цена закрытия ниже нижней полосы и NVI выше своей EMA. Позиция закрывается, когда NVI опускается ниже своей EMA. Также доступны необязательные проценты стоп-лосса и тейк-профита.

## Детали

- **Условия входа**:
  - **Лонг**: закрытие < нижней полосы и NVI > EMA NVI.
- **Лонг/Шорт**: только лонг.
- **Условия выхода**:
  - **Лонг**: NVI < EMA NVI.
- **Стопы**: опционально, процент от цены входа.
- **Значения по умолчанию**:
  - `ChannelType` = "BB"
  - `ChannelLength` = 20
  - `ChannelMultiplier` = 2
  - `NviEmaLength` = 200
  - `EnableStopLoss` = false
  - `StopLossPercent` = 0
  - `EnableTakeProfit` = false
  - `TakeProfitPercent` = 0
- **Фильтры**:
  - Категория: каналы
  - Направление: лонг
  - Индикаторы: полосы Боллинджера или каналы Кельтнера, EMA, NVI
  - Стопы: опционально
  - Сложность: низкая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенции: нет
  - Уровень риска: средний
