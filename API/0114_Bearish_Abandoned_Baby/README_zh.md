# 看跌弃婴策略
[English](README.md) | [Русский](README_ru.md)

看跌弃婴与多头版本相反，常预示顶部的形成。
该形态先出现向上跳空的十字星，随后向下跳空，使中间蜡烛孤立在上方。
当第三根蜡烛向下跳空时做空，捕捉情绪突变后的下行。
止损放在十字星高点上方，以防价格回升。

## 细节

- **入场条件**：形态匹配
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：形态
  - 方向：双向
  - 指标：蜡烛图
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等

测试表明年均收益约为 79%，该策略在股票市场表现最佳。
