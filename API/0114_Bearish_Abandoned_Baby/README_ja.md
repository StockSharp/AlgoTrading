# 弱気の捨て子戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
弱気の捨て子は強気バージョンを反映していますが、潜在的な天井を示します。
ギャップアップしたドージに続いてギャップダウンし、真ん中のローソク足を前のレンジの上に孤立させます。

テストでは年平均リターン約 79% を示しています。株式市場で最も良いパフォーマンスを発揮します。

戦略は、第三のローソク足がドージの下にギャップダウンした時点でショートに入り、感情の急激な変化から利益を得ることを目指します。

価格が回復した場合に備えて、ドージの高値のすぐ上にストップを置いてリスクを制限します。

## 詳細

- **エントリー条件**: パターン一致
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
