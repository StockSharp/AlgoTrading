# Стратегия Bearish Abandoned Baby

Bearish Abandoned Baby — медвежья версия фигуры, указывающая на возможное формирование вершины.
Она включает гэп вверх и доджи, за которым следует гэп вниз, оставляя среднюю свечу изолированной над предыдущим диапазоном.

Стратегия открывает короткую позицию, когда третья свеча открывается ниже доджи, стремясь извлечь выгоду из резкой смены настроений.

Риск ограничивается стопом сразу над максимумом доджи на случай возврата цены.

## Детали

- **Критерий входа**: совпадение паттерна
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
