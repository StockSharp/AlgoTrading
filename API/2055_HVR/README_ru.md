# Стратегия Historical Volatility Ratio
[English](README.md) | [中文](README_cn.md)

Стратегия основывается на коэффициенте исторической волатильности (HVR). Он сравнивает краткосрочную волатильность за 6 баров с долгосрочной за 100 баров, используя логарифмические доходности. Когда коэффициент поднимается выше порога, система открывает длинную позицию, ожидая расширения волатильности. Когда он опускается ниже порога, открывается короткая позиция.

## Подробности

- **Условия входа**:
  - Лонг: `HVR > RatioThreshold`
  - Шорт: `HVR < RatioThreshold`
- **Лонг/Шорт**: Оба направления
- **Условия выхода**: Противоположный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `ShortPeriod` = 6
  - `LongPeriod` = 100
  - `RatioThreshold` = 1.0
  - `CandleType` = `TimeSpan.FromMinutes(15).TimeFrame()`
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Оба
  - Индикаторы: Историческая волатильность (краткосрочная и долгосрочная)
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
