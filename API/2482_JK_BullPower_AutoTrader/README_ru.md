# JK BullP AutoTrader
[English](README.md) | [中文](README_cn.md)

JK BullP AutoTrader — это порт оригинального советника MetaTrader, использующего осциллятор Bulls Power. Стратегия сравнивает значения индикатора на двух предыдущих барах: если два бара назад Bulls Power выше, чем на предыдущем баре, а сам предыдущий бар выше нуля, считается, что бычья сила ослабевает, и открывается продажа; если предыдущий бар индикатора ниже нуля, открывается покупка. Для защиты позиций применяются фиксированные стоп-лосс и тейк-профит, а также пошаговый трейлинг-стоп, который подтягивается по мере роста прибыли.

## Подробности

- **Условия входа**: Продажа, когда Bulls Power два бара назад выше предыдущего значения и предыдущий бар выше нуля. Покупка, когда предыдущий бар индикатора ниже нуля.
- **Направление торговли**: В обе стороны.
- **Условия выхода**: Срабатывание тейк-профита, стоп-лосса или трейлинг-стопа. Противоположный сигнал переворачивает позицию.
- **Стопы**: Фиксированный тейк-профит, фиксированный стоп-лосс, трейлинг-стоп.
- **Параметры по умолчанию**:
  - `BullsPeriod` = 13
  - `TakeProfitPoints` = 350
  - `StopLossPoints` = 100
  - `TrailingStopPoints` = 100
  - `TrailingStepPoints` = 40
  - `CandleType` = TimeSpan.FromHours(1)
- **Фильтры**:
  - Категория: Осцилляторы
  - Направление: Оба
  - Индикаторы: Bulls Power
  - Стопы: Фиксированные + трейлинг
  - Сложность: Базовая
  - Таймфрейм: Внутридневной / свинг (1 час)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
