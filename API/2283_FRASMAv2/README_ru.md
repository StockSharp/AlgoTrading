# FRASMAv2
[English](README.md) | [中文](README_cn.md)

Стратегия на основе Fractal Adaptive Simple Moving Average (FRASMAv2).

Стратегия вычисляет фрактально-адаптивное простое скользящее среднее с использованием индикатора Fractal Dimension. Цвет индикатора меняется в зависимости от наклона: зелёный при росте, серый при плоском движении, пурпурный при падении. Стратегия отслеживает смену цвета на последней завершённой свече:

- Если индикатор был зелёным на предыдущем баре и стал не зелёным (серым или пурпурным) на последнем баре, стратегия закрывает короткие позиции и открывает новую длинную позицию.
- Если индикатор был пурпурным и стал не пурпурным, стратегия закрывает длинные позиции и открывает новую короткую позицию.

Управление рисками осуществляется через параметры стоп-лосса и тейк-профита, задаваемые в пунктах.

## Детали

- **Условия входа**: Смена цвета FRASMAv2.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Противоположная смена цвета.
- **Стопы**: Тейк-профит и стоп-лосс через модуль защиты.
- **Значения по умолчанию**:
  - `Period` = 30
  - `TakeProfit` = 2000 пунктов
  - `StopLoss` = 1000 пунктов
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Разворот тренда
  - Направление: Оба
  - Индикаторы: FractalDimension, FRASMAv2
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: 4 часа
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
