# SILVER Mitternachtskerzen-Farb-Strategie mit 1-Stunden-Verzögerung und SL/TP
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Handelt um 1:00 Uhr New Yorker Zeit basierend auf der Farbe der Mitternachtskerze des Vortages. Wenn die Mitternachtskerze höher als ihr Eröffnungskurs schloss, geht die Strategie long; andernfalls geht sie short. Long-Trades zielen auf 57 Ticks mit einem Stop von 200 Ticks ab, während Short-Trades 48 Ticks mit demselben Stop von 200 Ticks anpeilen.
