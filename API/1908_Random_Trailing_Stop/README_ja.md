# ランダムトレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ランダムトレーリングストップ戦略は、単純移動平均でバイアスをかけたランダムなトレードを建て、トレーリングストップで管理します。

## 詳細

- **エントリー条件**: SMAのバイアスによるランダム方向
- **ロング/ショート**: 両方
- **エグジット条件**: トレーリングストップ
- **ストップ**: はい
- **デフォルト値**:
  - `MinStopLevel` = 0.00036
  - `TrailingStep` = 0.00001
  - `SleepMinutes` = 5
  - `SmaPeriod` = 100
  - `Volume` = 0.1
- **フィルター**:
  - カテゴリ: 実験的
  - 方向: 両方
  - インジケーター: SMA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 1m
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
