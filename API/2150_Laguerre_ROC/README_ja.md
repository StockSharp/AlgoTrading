# Laguerre ROC戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はLaguerre変化率オシレーターを使用してトレンドの反転を捉えます。

Laguerre ROCオシレーターは4段階のラゲールフィルターを通じて変化率を平滑化します。
値は0と1の間に正規化されます。2つの閾値が買われすぎと売られすぎのゾーンを定義します：

- **Up Level** – この水準より高い値は強い上昇モメンタムを示します。
- **Down Level** – この水準より低い値は強い下降モメンタムを示します。

取引ロジック：

1. オシレーターが買われすぎゾーンから下落した場合（前の値がUp Levelより上で
   現在の値がその下）、戦略はロングポジションに入ります。
2. オシレーターが売られすぎゾーンから上昇した場合（前の値がDown Levelより下で
   現在の値がその上）、戦略はショートポジションに入ります。
3. ロングポジションが開いていてオシレーターが弱気に転じた場合（前の値が中立
   水準0.5より下）ポジションが決済されます。
4. ショートポジションが開いていてオシレーターが強気に転じた場合（前の値が0.5より上）
   ポジションが決済されます。

パラメーター：

- **Period** – 変化率計算のルックバック期間の長さ。
- **Gamma** – ラゲールフィルターの平滑化係数。
- **Up Level** – 買われすぎ閾値。
- **Down Level** – 売られすぎ閾値。
- **Candle Type** – ローソク足データに使用する時間軸。

この例は、内蔵の変化率とマニュアルのラゲールフィルタリングを使用して、高レベルの
StockSharp戦略内でカスタムインジケーターロジックを再現する方法を示しています。
