# モメンタム ロング + ショート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このモメンタム戦略は、3時間の時間軸でロングとショートの両ポジションを取引します。ロングのセットアップは、価格が100期間と500期間の移動平均線を上回って推移することを必要とし、RSI、ADX、ATR、トレンドの整合性でフィルタリングできます。ショートのエントリーは、価格が両方の移動平均線を下回りながらボリンジャーバンドの下限を割り込む場面を探し、オプションのATR確認と強い上昇トレンド中のショートをブロックする機能があります。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格がMA100とMA500を上回り、オプションのトレンド整合性、RSIが平滑値を上回り、ADXが平滑値を上回り、ATRが平滑値を上回る。
  - **ショート**: 価格がMA100とMA500を下回り、ボリンジャーバンド下限を割り込み、RSIが閾値を下回り、ATRが平滑値を上回り、オプションの上昇トレンドブロックあり。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: エントリー価格の`slPercentLong`%下にストップロス；価格がMA500を下回った場合の早期決済。
  - **ショート**: `slPercentShort`と`tpPercentShort`のパーセンテージに基づくストップロスとテイクプロフィット。
- **ストップ**: はい。
- **デフォルト値**:
  - `slPercentLong = 3`
  - `slPercentShort = 3`
  - `tpPercentShort = 4`
  - `rsiLengthLong = 14`
  - `rsiLengthShort = 14`
  - `adxLength = 14`
  - `atrLength = 14`
  - `bbLength = 20`
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 中期
