# Estrategia de Tres Supertrend con EMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Utiliza tres líneas Supertrend con diferentes multiplicadores y un filtro EMA. Abre largos cuando todos los Supertrends son alcistas y el precio está por encima de la EMA; abre cortos en la condición opuesta. Sale cuando el Supertrend más rápido cambia de dirección.
