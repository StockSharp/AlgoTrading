# 総合勢力インジケーターX戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略はStockSharpの高レベルAPIを使用してMetaTraderエキスパート「Exp_TotalPowerIndicatorX」の動作を再現します。内部EMAベースラインより上または下で閉じるロウソク足をローリングウィンドウ内でカウントすることで強気・弱気の優位性を測定する、総合勢力インジケーターのカスタム実装に依拠しています。強気と弱気の勢力線が互いにクロスしたときにトレードが決定されます。

インジケーターはあらゆるシンボルと時間軸で機能します。デフォルトでは戦略は元のエキスパートアドバイザーの設定に合わせて4時間ロウソク足を購読しますが、パラメーターで時間軸を調整できます。

## トレードロジック
1. 完成した各ロウソク足で、戦略はロウソク足データを総合勢力インジケーターに供給します。インジケーターは：
   - **Power Period**の期間でEMAを計算します。
   - **Lookback Period**内で`High > EMA`（強気）と`Low < EMA`（弱気）のロウソク足数をカウントします。
   - カウントを0–100の範囲のパーセンタイルスタイルの勢力値に変換します。
2. **強気クロスオーバー**（強気勢力が弱気勢力を上回る）は、ロングトレードが有効でオープンポジションがない場合にロングエントリーをトリガーします。
3. **弱気クロスオーバー**（弱気勢力が強気勢力を上回る）は、ショートトレードが有効でオープンポジションがない場合にショートエントリーをトリガーします。
4. 逆のクロスオーバーは、関連するエグジットスイッチが有効な場合に既存のポジションをクローズします。
5. オプションのトレードセッションフィルターは設定した時間ウィンドウ外でのすべてのポジションのクローズを強制し、その期間中は新規エントリーを無効にします。
6. オプションのストップロスとテイクプロフィットレベルは証券の価格ステップの倍数で表されます。各エントリー後に再計算され、ロウソク足の高値または安値がレベルを超えたらすぐに決済されます。

## パラメーター
- **Candle Type** – インジケーター計算に使用する時間軸。デフォルト：4時間ロウソク足。
- **Power Period** – インジケーター内のEMA長；MQL入力を反映。デフォルト：10。
- **Lookback** – 強気・弱気の優位性をカウントするために使用するロウソク足数。デフォルト：45。
- **Volume** – 取引所またはシミュレーターに送信される注文サイズ。デフォルト：1。
- **Enable Long Entry / Enable Short Entry** – 対応する方向への新規ポジションを許可または禁止。
- **Enable Long Exit / Enable Short Exit** – 反対シグナルでポジションをクローズ。無効にするとポジションは手動クローズまたはストップアウトまでオープンのままになります。
- **Use Trading Hours** – タイムフィルターを有効化。アクティブな場合、戦略は**Start Hour/Minute**と**End Hour/Minute**の間のみ取引し、その間隔外のオープンポジションをクローズします。夜間ウィンドウ（開始が終了より後）をサポートします。
- **Stop Loss Points / Take Profit Points** – 価格ステップで測定したエントリー価格からの距離。無効にするにはゼロに設定。計算は`Security.PriceStep`を使用するため、証券メタデータが利用可能であることを確認してください。

## 注記
- 戦略は証券にアクティブなポジションがない場合のみ新規ポジションをオープンし、元のエキスパートの動作を模倣します。
- ストップロスとテイクプロフィットの計算はインストゥルメントの価格ステップに依拠するため、そのメタデータなしで戦略を実行すると保護レベルが自動的に無効のままになります。
- UIが利用可能な場合、インジケーター値はチャートエリアにプロットされ、強気と弱気の勢力のクロスを視覚化するのに役立ちます。
