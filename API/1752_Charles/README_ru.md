# Стратегия Charles
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя, основанная на максимумах и минимумах предыдущего дня. Цена должна пробить диапазон предыдущего дня, при этом тренд подтверждается фильтром из RSI и EMA. Стратегия рассчитывает дневные максимум и минимум, смещает их на настраиваемую величину и открывает длинные позиции выше верхнего уровня или короткие ниже нижнего уровня при выполнении условий тренда.

## Подробности

- **Условия входа**:
  - Длинная: `Close > DailyHigh + Delta` и `RSI > 55` и `FastEMA > SlowEMA`
  - Короткая: `Close < DailyLow - Delta` и `RSI < 45` и `FastEMA < SlowEMA`
- **Long/Short**: Оба
- **Условия выхода**: Противоположный сигнал или защита
- **Стопы**: Настраиваемые take profit и stop loss в процентах
- **Параметры по умолчанию**:
  - `Delta` = 0.0002m
  - `FastPeriod` = 18
  - `SlowPeriod` = 60
  - `RsiPeriod` = 14
  - `TakeProfit` = 1m
  - `StopLoss` = 0.5m
  - `CandleType` = TimeSpan.FromHours(1).TimeFrame()
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: EMA, RSI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
