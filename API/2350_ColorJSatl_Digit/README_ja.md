# Color JSatl Digit戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQL5エキスパート「Exp_ColorJSatl_Digit」をStockSharpに変換したものです。Jurik移動平均（JMA）の傾きをデジタル化し、各バーを上昇または下降として分類します。状態が0から1に変わると上昇トレンドの発生を示し、1から0への変化は下降トレンドを示します。

アルゴリズムは選択した時間軸のローソク足を購読し、JMAインジケーターをバインドします。JMAが上向きに転換すると、戦略はロングポジションを開き、ショートポジションを閉じます。JMAが下向きに転換すると、ショートポジションを開き、ロングポジションを閉じます。オプションの`DirectMode`パラメーターはシグナルを反転させ、逆張り取引を可能にします。

ポジションはパーセンテージベースのストップロスとテイクプロフィットレベルで保護されます。すべてのパラメーターは`StrategyParam`で定義され、最適化が可能です。

## 詳細

- **エントリー条件**
  - **ロング**: JMAが上向きに転換（`prev > prevPrev` && `current >= prev`）し、`DirectMode`が真の場合。逆モードでは下向きの転換でロングが開かれます。
  - **ショート**: JMAが下向きに転換（`prev < prevPrev` && `current <= prev`）し、`DirectMode`が真の場合。逆モードでは上向きの転換でショートが開かれます。
- **エグジット条件**: 反対のシグナルが逆方向への即時成行注文をトリガーします。保護注文もポジションをクローズする場合があります。
- **ストップ**: `StartProtection`によるパーセンテージストップロスとテイクプロフィット。
- **デフォルト値**
  - `JMA Length` = 30
  - `Candle Type` = 4時間足
  - `Stop Loss %` = 1
  - `Take Profit %` = 2
  - `Direct Mode` = true
- **フィルター**
  - カテゴリ: トレンドフォロー
  - 方向: 両方（反転可能）
  - インジケーター: Jurik Moving Average
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
