# Стратегия ADX Stochastic
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает индикатор ADX (Average Directional Index), определяющий силу тренда, и осциллятор Стохастик для точного входа на уровнях перепроданности или перекупленности.

ADX указывает на силу тренда, а Стохастик помогает находить откаты. Сигналы на покупку или продажу возникают, когда импульс разворачивается, оставаясь при этом в зоне высокого ADX.

Стратегия подходит трейдерам, совмещающим трендовое следование с точками входа по осциллятору. Защитные стопы по ATR помогают контролировать просадки.

## Подробности

- **Условия входа**:
  - Лонг: `ADX > AdxThreshold && StochK < StochOversold && Bullish`
  - Шорт: `ADX > AdxThreshold && StochK > StochOverbought && Bearish`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Выход при `ADX < AdxThreshold`
- **Стопы**: основаны на проценте `StopLossPercent`
- **Значения по умолчанию**:
  - `AdxPeriod` = 14
  - `AdxThreshold` = 25m
  - `StochPeriod` = 14
  - `StochK` = 3
  - `StochD` = 3
  - `StochOversold` = 20m
  - `StochOverbought` = 80m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Средняя обратная
  - Направление: Оба
  - Индикаторы: ADX, Стохастик
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 172%. Стратегию лучше запускать на рынке Форекс.
