# 四画面戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

四画面戦略は4つの時間軸（5分、15分、30分、60分）でHeikin-Ashiローソク足を使って取引します。
すべての時間軸が強気ローソク足を示す場合にロングに入り、すべてが弱気ローソク足を示す場合にショートに入ります。
ストップロスとテイクプロフィットのレベルはポイントで設定され、オプションのトレーリングストップがあります。

## 仕組み
1. 5分、15分、30分、60分のローソク足ストリームをサブスクライブします。
2. 各ローソク足のHeikin-Ashiの始値と終値を計算します。
3. 各時間軸を強気または弱気としてマークします。
4. すべてが強気の場合にロングエントリー、すべてが弱気の場合にショートエントリー。
5. `StartProtection` を使用してストップロス、テイクプロフィット、オプションのトレーリングを適用します。

## パラメーター
- `CandleType` – 5分ローソク足のベース時間軸。
- `StopLossPoints` – ストップロス距離（ポイント）。
- `TakeProfitPoints` – テイクプロフィット距離（ポイント）。
- `UseTrailing` – トレーリングストップを有効化（true/false）。

取引ボリュームは戦略の `Volume` プロパティで定義されます。

## 注意事項
- `SubscribeCandles` と `Bind` を使用した高レベルAPIで動作します。
- 完成したローソク足のみを処理します。
- コードのコメントは英語で記述されています。
