# Стратегия Fast2 Crossover
[English](README.md) | [中文](README_cn.md)

Стратегия основана на гистограмме Fast2. Гистограмма комбинирует тела последних трёх свечей с весами, зависящими от квадратного корня, и сглаживается двумя взвешенными скользящими средними. Длинная позиция открывается при пересечении быстрой средней ниже медленной, короткая — при пересечении выше.

## Детали

- **Условия входа**:
  - Long: быстрая WMA пересекает медленную сверху вниз
  - Short: быстрая WMA пересекает медленную снизу вверх
- **Направление**: Оба
- **Условия выхода**:
  - Обратное пересечение
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `FastLength` = 3
  - `SlowLength` = 9
  - `CandleType` = TimeSpan.FromHours(8).TimeFrame()
- **Фильтры**:
  - Категория: Пересечение
  - Направление: Оба
  - Индикаторы: WeightedMovingAverage
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
