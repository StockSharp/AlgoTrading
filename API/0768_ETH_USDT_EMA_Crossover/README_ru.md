# Стратегия пересечения EMA для ETH/USDT
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует парой ETH/USDT, используя пересечение EMA с дополнительными фильтрами.

Длинная позиция открывается, когда EMA(20) пересекает сверху EMA(50), цена находится выше EMA(200), RSI превышает 30, ATR выше своего среднего, а объём больше среднего. Короткая позиция открывается при обратных условиях.

Позиции разворачиваются при появлении обратного сигнала. Стоп-лосс и тейк-профит не используются.

## Детали

- **Условия входа**:
  - **Лонг**: `EMA20 пересекает сверху EMA50` && `Close > EMA200` && `RSI > 30` && `ATR > SMA(ATR,10)` && `Объём > SMA(Volume,20)`
  - **Шорт**: `EMA20 пересекает снизу EMA50` && `Close < EMA200` && `RSI < 70` && `ATR > SMA(ATR,10)` && `Объём > SMA(Volume,20)`
- **Направление**: Обе стороны
- **Условия выхода**:
  - Обратный сигнал
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `EMA200 Length` = 200
  - `EMA20 Length` = 20
  - `EMA50 Length` = 50
  - `RSI Length` = 14
  - `ATR Length` = 14

- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: EMA, RSI, ATR
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
