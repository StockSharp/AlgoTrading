# Color Zerolag HLR 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は MQL5 エキスパート `Exp_ColorZerolagHLR` の C# 変換版です。異なる長さと重みを持つ複数の Hi-Lo Range (HLR) オシレーターを組み合わせ、指数平滑化を適用して高速・低速トレンドラインを構築します。これらのライン間のクロスオーバーがトレードシグナルを生成します。

## 概要
- 指定された期間での最高値と最安値を使用して 5 つの HLR 値を構築します。
- 各 HLR を重み付けして合計し、高速トレンドラインを生成します。
- ゼロラグ平滑化を適用して低速トレンドラインを導出します。
- 高速ラインが低速ラインをクロスしたときにトレードします。

## パラメーター
- `Smoothing` – EMA 平滑化係数。
- `Factor1`..`Factor5` – 各 HLR コンポーネントの重み。
- `HlrPeriod1`..`HlrPeriod5` – HLR 計算のルックバック期間。
- `BuyPosOpen`/`SellPosOpen` – ロングまたはショートポジションのオープンを許可。
- `BuyPosClose`/`SellPosClose` – 既存ポジションのクローズを許可。
- `CandleType` – 足の時間軸。

## インジケーター
- Highest、Lowest（各 5 ペア）。

## トレードロジック
- 前回の高速ラインが低速ラインの上にあり、現在下にクロスした場合、戦略はロングポジションを開き、ショートをクローズします。
- 前回の高速ラインが低速ラインの下にあり、現在上にクロスした場合、戦略はショートポジションを開き、ロングをクローズします。

このテンプレートを出発点として使用し、ニーズに応じてパラメーターまたはリスク管理を調整してください。
