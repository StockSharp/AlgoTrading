# 人工知能パーセプトロン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

人工知能戦略は、単純なパーセプトロンを使用して、異なる時間シフトで複数のAccelerator Oscillator（AC）の値を組み合わせます。現在のAC値と3つの遅延値（7、14、21バー前）の加重和が取引の方向を決定します。パーセプトロンの出力が正の場合、戦略はロングポジションを開くか維持します；負の場合はショートポジションを開くか維持します。

エントリー後、戦略はポイント単位のストップロスで取引を保護します。価格が利益方向に動くにつれて、ストップレベルは価格を追跡します。ポジションが利益を出している間にパーセプトロンの出力が符号を反転させると、戦略は現在のポジションをクローズして逆方向のポジションを開くことでリバーサルします。

テストではこのアプローチがリスクをコントロールしながらモメンタムの変化に素早く反応できることが示されています。ローソク足データを提供するあらゆる銘柄で機能し、特定の市場レジームに依存しません。

## 詳細

- **エントリー条件**  
  - **ロング**: パーセプトロン出力 > 0 かつ既存のロングポジションなし。  
  - **ショート**: パーセプトロン出力 < 0 かつ既存のショートポジションなし。
- **エグジット / リバーサル**  
  - トレーリングストップが発動。  
  - パーセプトロン出力が符号を変更；戦略がポジションをリバーサル。
- **ストップ**: あり、`StopLoss`パラメーターに基づくトレーリングストップ。
- **デフォルト値**  
  - `X1 = 135`  
  - `X2 = 127`  
  - `X3 = 16`  
  - `X4 = 93`  
  - `StopLoss = 85`
- **フィルター**  
  - カテゴリ: モメンタム  
  - 方向: 両方  
  - インジケーター: Accelerator Oscillator  
  - ストップ: はい  
  - 複雑さ: 中  
  - 時間軸: 短期  
  - ニューラルネットワーク: パーセプトロン  
  - ダイバージェンス: いいえ  
  - リスクレベル: 中
