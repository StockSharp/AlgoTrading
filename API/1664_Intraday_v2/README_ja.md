# イントラデイ v2 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、2組のボリンジャーバンドを使用したイントラデイ平均回帰アプローチを実装します。外側のバンド（偏差2.4）がエントリーゾーンを定義し、内側のバンド（偏差1）がエグジットを管理します。オプションのストップロスとテイクプロフィットのレベルは、価格が設定可能な量だけトレードに逆行した場合にポジションをクローズします。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が外側の下限バンドを下回る。
  - **ショート**: 終値が外側の上限バンドを上回る。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - ロング: 価格が内側の下限バンドを上抜けるか、ストップロス/テイクプロフィットに達する。
  - ショート: 価格が内側の上限バンドを下抜けるか、ストップロス/テイクプロフィットに達する。
- **ストップ**: 設定可能な絶対ストップロスとテイクプロフィット。
- **フィルター**: なし。
