# Intraday v2 策略
[English](README.md) | [Русский](README_ru.md)

该策略使用两组布林带实现日内均值回归。外层带（偏差2.4）用于确定入场区域，内层带（偏差1）用于管理出场。可选的止损和止盈在价格对持仓不利移动时平仓。

## 细节

- **入场条件**：
  - **多头**：收盘价跌破外层下轨。
  - **空头**：收盘价突破外层上轨。
- **方向**：双向。
- **出场条件**：
  - 多头：价格上穿内层下轨或触发止损/止盈。
  - 空头：价格下穿内层上轨或触发止损/止盈。
- **止损**：可配置的绝对止损和止盈。
- **过滤器**：无。
