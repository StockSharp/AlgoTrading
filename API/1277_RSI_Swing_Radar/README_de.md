# RSI SwingRadar-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Einstieg in eine Long-Position, wenn der RSI seine SMA nach oben kreuzt, nachdem er sich unterhalb eines überverkauften Niveaus befand. Der Stop-Loss verwendet ATR, und der Take-Profit basiert auf einem konfigurierbaren Risiko-Ertrags-Verhältnis.
