# Стратегия Bollinger Volume
[English](README.md) | [中文](README_cn.md)

Стратегия использует пробои полос Боллинджера с подтверждением объёма. Позиции открываются, когда цена пробивает верхнюю или нижнюю полосу на повышенном объёме.

Тестирование показывает среднегодичную доходность около 178%. Стратегию лучше запускать на фондовом рынке.

Полосы Боллинджера показывают расширение волатильности, а объём подтверждает пробой. Сделки заключаются, когда цена закрывается за пределами полосы при активных торгах.

Подходит для трейдеров, играющих на продолжение движения. Стоп по ATR помогает удерживать убытки под контролем.

## Подробности

- **Условия входа**:
  - Лонг: `Close > UpperBand && Volume > AvgVolume * VolumeMultiplier`
  - Шорт: `Close < LowerBand && Volume > AvgVolume * VolumeMultiplier`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Цена возвращается к средней полосе
- **Стопы**: основаны на ATR через `StopLossAtr`
- **Значения по умолчанию**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 2.0m
  - `VolumePeriod` = 20
  - `VolumeMultiplier` = 1.5m
  - `StopLossAtr` = 2.0m
  - `AtrPeriod` = 14
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Средняя обратная
  - Направление: Оба
  - Индикаторы: Полосы Боллинджера, Объём
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
