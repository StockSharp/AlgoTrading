# Стратегия BTC Seasonality
[English](README.md) | [中文](README_cn.md)

Стратегия открывает позицию согласно выбранным дню недели и часу по времени EST. Пользователь задаёт день и час входа, день и час выхода, а также направление сделки. В указанное время стратегия входит в позицию и закрывает её в момент выхода.

## Подробности

- **Условия входа**:
  - Текущий день и час в EST совпадают с `EntryDay` и `EntryHour`.
- **Направление**: Настраивается.
- **Условия выхода**:
  - Текущий день и час в EST совпадают с `ExitDay` и `ExitHour`.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `EntryDay` = Saturday
  - `ExitDay` = Monday
  - `EntryHour` = 10
  - `ExitHour` = 10
  - `IsLong` = true
- **Фильтры**:
  - Категория: Сезонность
  - Направление: Настраиваемое
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
