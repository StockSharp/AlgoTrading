# BTC季節性戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、東部標準時（EST）の曜日と時間に基づく事前定義ルールに従ってポジションを建てます。ユーザーはエントリーの曜日・時間、エグジットの曜日・時間、ロングかショートかを選択します。ポジションは指定したエントリー時点で建てられ、指定したエグジット時点でクローズされます。

## 詳細

- **エントリー条件**:
  - 現在のEST曜日が`EntryDay`と等しく、現在の時間が`EntryHour`と等しい。
- **ロング/ショート**: 設定可能。
- **エグジット条件**:
  - 現在のEST曜日が`ExitDay`と等しく、現在の時間が`ExitHour`と等しい。
- **ストップ**: なし。
- **デフォルト値**:
  - `EntryDay` = Saturday
  - `ExitDay` = Monday
  - `EntryHour` = 10
  - `ExitHour` = 10
  - `IsLong` = true
- **フィルター**:
  - カテゴリ: 季節性
  - 方向: 設定可能
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
