# RSIトレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**RSIトレンド戦略**は、相対力指数（RSI）を使用してトレンド転換を検出し、ATRベースのトレーリングストップでポジションを管理します。RSIが買われすぎ閾値を上抜けたときにロングポジションを建て、RSIが売られすぎ閾値を下回ったときにショートポジションに入ります。リスクは平均真の値幅（ATR）から導出されたトレーリングストップで制御され、現在のボラティリティにストップレベルを適応させます。

この実装は教育目的で設計されており、インジケーターバインディングを使用して高レベルのStockSharp戦略を構築する方法を示します。戦略は完成したロウソク足のみで取引し、以前のインジケーター値を直接参照しないため、StockSharpのベストプラクティスに沿っています。

## 詳細

- **エントリー条件**:
  - **ロング**: `RSI(t) > BuyLevel` かつ `RSI(t-1) <= BuyLevel`。
  - **ショート**: `RSI(t) < SellLevel` かつ `RSI(t-1) >= SellLevel`。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - ATRの倍数に基づくトレーリングストップ。
- **ストップ**: あり、動的トレーリングストップ。
- **デフォルト値**:
  - `RSI Period` = 14。
  - `BuyLevel` = 73。
  - `SellLevel` = 27。
  - `ATR Period` = 100。
  - `ATR Multiple` = 3。
- **フィルター**:
  - カテゴリ: トレンドフォロー。
  - 方向: 両方。
  - インジケーター: RSI、ATR。
  - ストップ: あり。
  - 複雑さ: 中程度。
  - 時間軸: 任意（デフォルトは1分足）。
  - 季節性: いいえ。
  - ニューラルネットワーク: いいえ。
  - ダイバージェンス: いいえ。
  - リスクレベル: 中程度。
