# 成交量高潮反转策略
[English](README.md) | [Русский](README_ru.md)

该策略寻找在强势趋势后出现的极端放量转折点。
如此剧烈的成交量往往意味着最后的买家或卖家蜂拥而入，动能可能衰竭。
当巨量成交后价格开始回撤时，策略逆势进场。
使用紧密的百分比止损，若成交量未明显下降或价格延续原趋势则退出。

测试表明年均收益约为 82%，该策略在股票市场表现最佳。

## 细节

- **入场条件**：指标信号
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：成交量
  - 方向：双向
  - 指标：成交量
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
