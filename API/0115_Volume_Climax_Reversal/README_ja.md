# 出来高クライマックスリバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
出来高クライマックスリバーサルは、強いトレンドの後に極めて高い出来高によって特徴づけられる転換点を探します。
このような急激なスパイクは、モメンタムが衰える前に最後の買い手や売り手が殺到することによる枯渇を示唆します。

テストでは年平均リターン約 82% を示しています。株式市場で最も良いパフォーマンスを発揮します。

戦略は、大きな出来高バーがクローズして価格が反落し始めたときに、前の動きに逆らってエントリーします。

タイトなパーセントストップがポジションを保護し、出来高が減少しない場合や価格が元の方向に続く場合はトレードを終了します。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: 出来高
  - 方向: 両方
  - インジケーター: 出来高
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
