# Color BB Candles 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はボリンジャーバンドを使用してローソク足を強気・弱気・中立ゾーンに分類します。価格が上限バンドを上抜けして引けるとロングポジションを開き、価格が下限バンドを下抜けして引けるとショートポジションを開き、価格がバンド間に戻るとポジションを決済します。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が外側から上限バンドを上抜け。
  - **ショート**: 終値が外側から下限バンドを下抜け。
- **エグジット条件**: 価格が上限バンドと下限バンドの間に戻る。
- **インジケーター**: ボリンジャーバンド。
- **デフォルト値**:
  - `BollingerPeriod` = 100
  - `BollingerDeviation` = 1.0
  - `CandleType` = 4時間足
- **方向**: ロングおよびショート。
- **ストップ**: なし。
- **複雑さ**: 中程度。
- **時間軸**: 中期。
