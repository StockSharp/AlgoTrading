# EA テンプレート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTraderのEAテンプレートに由来します。前の完成したローソク足を分析し、ローソク足の実体の方向にポジションをオープンします。陽線はロング取引を引き起こし、陰線はショート取引を引き起こします。リバースモードはローソク足の解釈を反転させ、バーの色に逆らって取引します。

戦略は固定ポジションサイズまたは資本ベースの計算をサポートします。ストップロスとテイクプロフィットのレベルはエントリー価格からのポイントで設定されます。スプレッドが許容しきい値を超えると取引はスキップされます。

## 詳細

- **エントリー条件**:
  - **ロング**: 前のローソク足のクローズ > オープン かつ `ReverseTrade` が無効。
  - **ショート**: 前のローソク足のクローズ < オープン かつ `ReverseTrade` が無効。
  - `ReverseTrade` が有効な場合、シグナルは反転されます。
  - スプレッドは `SpreadLimit` ポイント未満でなければなりません。
- **エグジット条件**:
  - 反対のローソク足の色またはストップロス/テイクプロフィットの発動。
- **ポジションサイジング**:
  - 固定サイズ `Lots`、または `UseMoneyManagement` が true の場合 `RiskPercent` を使った資本ベースのサイズ。
- **ストップ**:
  - `StopLoss` と `TakeProfit` はエントリー価格からのポイントで表示。
- **ロング/ショート**: 両方向。
- **インジケーター**: なし。
- **リスクレベル**: 中。

パラメーターを使用して、ローソク足の種類、リバースモード、資金管理ルール、リスク制限を調整できます。
