# Стратегия восходящего разворота из трёх свечей

Этот паттерн ловит быстрый бычий разворот после короткого снижения. Требуется две подряд идущие падающие свечи, за которыми следует сильная растущая свеча, закрывающаяся выше максимума предыдущей. Логика при желании проверяет, что ранее цена двигалась вниз.

Стратегия запоминает последние три свечи. Как только последовательность соответствует критериям и выполнен фильтр нисходящего тренда, открывается длинная позиция. Волатильный стоп под минимумом паттерна ограничивает риск сделки.

После входа система ждёт либо срабатывания стопа, либо появления противоположного паттерна. Такой простой подход подходит рынкам, склонным к резким отскокам из перепроданности.

## Детали

- **Условия входа**: Две медвежьи свечи с понижающимися минимумами, затем бычья свеча, закрывающаяся выше максимума средней свечи.
- **Направление**: Только лонг.
- **Условия выхода**: Стоп‑лосс или следующий паттерн.
- **Стопы**: Да, под минимумом паттерна.
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLossPercent` = 1
  - `RequireDowntrend` = true
  - `DowntrendLength` = 5
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Лонг
  - Индикаторы: Свечной анализ
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
