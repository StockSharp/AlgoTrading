# Tripple MACD-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Kombiniert vier MACD-Berechnungen mit einem RSI-Filter. Eine Long-Position wird eröffnet, wenn das gemittelte MACD-Histogramm bei einer bullischen Kerze positiv wird und der RSI unter 55 liegt. Der Trade schließt bei der entgegengesetzten Bedingung.

- Indikatoren: MACD, RSI
