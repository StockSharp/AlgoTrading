# Estratégia Tripple MACD
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Combina quatro cálculos de MACD com um filtro RSI. Uma posição comprada é aberta quando o histograma MACD médio se torna positivo em uma vela de alta e o RSI está abaixo de 55. A operação fecha na condição oposta.

- Indicadores: MACD, RSI
