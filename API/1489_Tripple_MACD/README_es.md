# Estrategia Tripple MACD
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Combina cuatro cálculos de MACD con un filtro RSI. Se abre una posición larga cuando el histograma MACD promediado se vuelve positivo en una vela alcista y el RSI está por debajo de 55. La operación se cierra en la condición opuesta.

- Indicadores: MACD, RSI
