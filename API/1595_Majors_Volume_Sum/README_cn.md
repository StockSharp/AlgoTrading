# Majors Volume Sum 策略
[English](README.md) | [Русский](README_ru.md)

该策略对最近的带符号成交量求和，当短期总和超过其历史最大值的一定比例时入场。

## 细节

- **入场条件**：
  - 10周期带符号成交量之和高于 `Threshold` × 最大值且无持仓时做多。
  - 10周期带符号成交量之和低于 `-Threshold` × 最大值且无持仓时做空。
- **方向**：多空皆可。
- **出场条件**：
  - 反向信号平仓。
- **止损**：无。
- **默认参数**：
  - `Threshold` = 0.75
- **过滤器**：
  - 类型：成交量
  - 方向：双向
  - 指标：SMA
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
