# Стратегия DCA Support and Resistance with RSI and Trend Filter
[English](README.md) | [中文](README_cn.md)

Стратегия усреднения использует уровни поддержки и сопротивления, индикатор RSI и фильтр тренда EMA. Покупает на поддержке в восходящем тренде при перепроданности RSI и продаёт на сопротивлении в нисходящем тренде при перекупленности RSI.

## Детали

- **Условия входа**:
  - Long: цена у поддержки, RSI ниже уровня перепроданности, выше EMA
  - Short: цена у сопротивления, RSI выше уровня перекупленности, ниже EMA
- **Направление**: Оба
- **Условия выхода**:
  - Long: цена достигает сопротивления или RSI выше уровня перекупленности
  - Short: цена достигает поддержки или RSI ниже уровня перепроданности
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `LookbackPeriod` = 50
  - `RsiLength` = 14
  - `Overbought` = 70
  - `Oversold` = 40
  - `EmaPeriod` = 200
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: RSI, EMA, Highest, Lowest
  - Стопы: Нет
  - Сложность: Начальный
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
