# Stochastic Martingale戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はクラシックなStochasticオシレーターのエントリーとマーチンゲール方式の平均化を組み合わせます。
%K線が%D線を交差し、オシレーターが設定可能なゾーンの上/下にある時にポジションを開きます。
価格がポジションに対して定義されたステップ分逆行した場合、戦略はボリュームをマルチプライヤーで増加させます。
累積利益が定義されたポイント数に達するとポジションを決済します。

## 詳細
- **エントリー条件**
  - ロング: %K > %D かつ %D > ZoneBuy
  - ショート: %K < %D かつ %D < ZoneSell
- **平均化**
  - 追加注文は `Step` ポイントごとに発注されます（モード1では `Step * 注文数`）。
  - 各新規注文のボリュームは `Mult` で乗算されます。
- **エグジット条件**
  - ロング: 価格 ≥ 最後の買値 + `ProfitFactor * 注文数` ポイント。
  - ショート: 価格 ≤ 最後の売値 − `ProfitFactor * 注文数` ポイント。
- **パラメーター** にはステップサイズ、ステップモード、利益係数、マルチプライヤー、初期ボリューム、Stochasticの各期間が含まれます。
- **フィルター**
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Stochastic
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
