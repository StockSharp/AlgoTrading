# Estrategia RSI con TP/SL (Marco Temporal Inferior)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Estrategia basada en RSI para marcos temporales inferiores. Abre largos por debajo del nivel de compra y cortos por encima del nivel de venta. Las posiciones se cierran usando take profit y stop loss porcentuales.
