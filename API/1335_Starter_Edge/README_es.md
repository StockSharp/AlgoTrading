# Estrategia Starter Edge
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza un MACD basado en ZLEMA con un filtro de tendencia EMA 100 y confirmación RSI.

## Detalles

- **Entrada**: Precio por encima de EMA100 con la línea MACD cruzando por encima de la señal y RSI por encima de 50 para largo; condiciones inversas para corto.
- **Salida**: Cruce MACD opuesto, histograma MACD girando hacia abajo, o RSI saliendo de zonas de sobrecompra/sobreventa.
- **Indicadores**: ZLEMA, MACD, EMA 100, RSI.
- **Tipo**: Seguimiento de tendencia.
