# Estratégia Starter Edge
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia usa um MACD baseado em ZLEMA com um filtro de tendência EMA 100 e confirmação RSI.

## Detalhes

- **Entrada**: Preço acima da EMA100 com a linha MACD cruzando acima do sinal e RSI acima de 50 para comprado; condições inversas para vendido.
- **Saída**: Cruzamento MACD oposto, histograma MACD virando para baixo, ou RSI saindo das zonas de sobrecompra/sobrevenda.
- **Indicadores**: ZLEMA, MACD, EMA 100, RSI.
- **Tipo**: Seguidor de tendência.
