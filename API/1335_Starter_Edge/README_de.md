# Starter Edge-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet einen ZLEMA-basierten MACD mit einem EMA-100-Trendfilter und RSI-Bestätigung.

## Details

- **Einstieg**: Preis über EMA100 mit MACD-Linie, die über das Signal kreuzt, und RSI über 50 für Long; umgekehrte Bedingungen für Short.
- **Ausstieg**: Entgegengesetzter MACD-Crossover, MACD-Histogramm dreht nach unten, oder RSI verlässt überkaufte/überverkaufte Zonen.
- **Indikatoren**: ZLEMA, MACD, EMA 100, RSI.
- **Typ**: Trendfolge.
