# Starter Edge戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はZLEMAベースのMACDをEMA 100トレンドフィルターとRSI確認と組み合わせて使用します。

## 詳細

- **エントリー**: EMA100を上回る価格でMACDラインがシグナルを上抜け、RSIが50を上回るときにロング；逆条件でショート。
- **エグジット**: 逆のMACDクロス、MACDヒストグラムの下落転換、またはRSIが買われすぎ/売られすぎゾーンを離れる。
- **インジケーター**: ZLEMA, MACD, EMA 100, RSI。
- **タイプ**: トレンドフォロー。
