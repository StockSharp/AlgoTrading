# Стратегия Post Open Long ATR Stop Loss Take Profit
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию во время торговой сессии при пробое сопротивления, когда цена находится рядом с серединой полос Боллинджера. Используются фильтры EMA, RSI, ADX и ATR, выход осуществляется по стоп-лоссу и тейк-профиту на основе ATR.

## Детали

- **Условия входа**:
  - **Лонг**: пробой сопротивления в рабочие часы, цена около средней полосы Боллинджера, RSI выше порога, ADX выше порога, краткосрочный тренд вверх и отсутствие отката.
- **Лонг/Шорт**: только лонг.
- **Условия выхода**:
  - достижение стоп-лосса или тейк-профита по ATR.
- **Стопы**:
  - стоп-лосс и тейк-профит на основе ATR.
- **Значения по умолчанию**:
  - `BB Length` = 14
  - `BB Mult` = 1.5
  - `EMA Length` = 10
  - `EMA Long Length` = 200
  - `RSI Length` = 7
  - `RSI Threshold` = 30
  - `ADX Length` = 7
  - `ADX Threshold` = 10
  - `ATR Length` = 14
  - `ATR SL Mult` = 2.0
  - `ATR TP Mult` = 4.0
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: Bollinger Bands, EMA, RSI, ADX, ATR
  - Стопы: ATR
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
