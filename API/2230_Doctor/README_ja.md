# Doctor戦略

MQLからStockSharpに変換された戦略#15233「Doctor」の実装。

## 概要
この戦略は、トレンドの方向とモメンタムを検出するために複数のクラシック指標を組み合わせています：

- **傾き検出**: 40期間の加重移動平均を使用してトレンドの方向を評価。
- **線形位置**: 400期間の加重移動平均を最後の3本のローソク足の高値・安値と比較。
- **モメンタム確認**: 期間14および5の相対力指数を使用。
- **トレンド反転フィルター**: Parabolic SARによる。

強気条件がすべて揃ったときにロングポジションを、弱気条件がすべて揃ったときにショートポジションを建てます。反対シグナルまたは保護レベルに達したときに既存ポジションをクローズします。オプションのトレーリングストップは、ストップ距離の半分が達成されるとストップロスを前進させます。

## パラメーター
- `StopLossTicks` – ティック単位のストップロス距離。
- `TakeProfitTicks` – ティック単位のテイクプロフィット距離。
- `TrailingStop` – トレーリングストップロジックを有効化。
- `CandleType` – ローソク足に使用する時間軸（デフォルト30分）。

## 取引ルール
1. **買い**の条件：
   - WMA(40)の傾きが上昇している。
   - WMA(400)が最後の3本のローソク足の高値より上にある。
   - RSI(14)が50より上で、RSI(5)がRSI(14)より下にある。
   - オープンなロングポジションがない。
2. **売り**の条件：
   - WMA(40)の傾きが下降している。
   - WMA(400)が最後の3本のローソク足の安値より下にある。
   - RSI(14)が50より下で、RSI(5)がRSI(14)より上にある。
   - オープンなショートポジションがない。
3. **エグジット**: 反対条件が発生するか、ストップロス/テイクプロフィットレベルに達したとき。トレーリングストップは、価格がストップ距離の半分、有利な方向に動いた後にストップレベルを更新します。

## インジケーター
- 加重移動平均 (40, 400)
- 相対力指数 (14, 5)
- Parabolic SAR
