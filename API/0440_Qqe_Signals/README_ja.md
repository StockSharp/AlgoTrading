# QQE Signals戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSI上でQuantitative Qualitative Estimation技術を実装します。このインジケーターは平滑化されたRSIラインの周囲に動的な上部および下部バンドを構築し、バンドのクロスを追跡してトレンドの変化を示します。RSIがトレーリングバンドを上抜けするとロングシグナルが生成され、下抜けするとエグジットが発動します。

バンドをボラティリティに適応させることで、QQEはノイズを平滑化しながら反応性を維持しようとします。この戦略はロング取引に集中し、ポジションのクローズはエンジンのトレードリバーサルに依存します。

## 詳細

- **エントリー条件**:
  - **ロング**: 平滑化されたRSIラインがトレーリングバンドを上抜けする。
- **エグジット条件**:
  - RSIが反対側のバンドを下回るか、反対のシグナルが現れる。
- **インジケーター**:
  - RSI (期間14、スムージング5)
  - ファクター4.238のRSIのATRから導出されたQQEバンド
- **ストップ**: デフォルトでなし；反対のシグナルに依存。
- **デフォルト値**:
  - `RsiPeriod` = 14
  - `RsiSmoothing` = 5
  - `QqeFactor` = 4.238
  - `Threshold` = 10
- **フィルター**:
  - トレンドフォロー
  - 単一時間軸
  - インジケーター: RSI、QQE
  - ストップ: なし
  - 複雑さ: 中程度
