# Estrategia EMA con Desplazamiento Paralelo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia opera cuando una EMA rápida desplazada cruza una EMA lenta. Los lados largo y corto pueden habilitarse por separado. Cada lado admite stop loss opcional y trailing stop basados en niveles porcentuales.

## Parámetros
- Tipo de vela
- Habilitar largo
- EMA rápida para largo
- EMA lenta para largo
- Factor de desplazamiento descendente
- Stop loss largo %
- Activación de trailing largo %
- Offset de trailing largo %
- Cerrar largo por señal
- Habilitar corto
- EMA rápida para corto
- EMA lenta para corto
- Factor de desplazamiento ascendente
- Stop loss corto %
- Activación de trailing corto %
- Offset de trailing corto %
- Cerrar corto por señal
