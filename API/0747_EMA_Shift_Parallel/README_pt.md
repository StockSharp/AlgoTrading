# Estratégia EMA com Deslocamento Paralelo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera quando uma EMA rápida deslocada cruza uma EMA lenta. Os lados comprado e vendido podem ser habilitados separadamente. Cada lado suporta stop loss opcional e trailing stop baseados em níveis percentuais.

## Parâmetros
- Tipo de candle
- Habilitar comprado
- EMA rápida para comprado
- EMA lenta para comprado
- Fator de deslocamento descendente
- Stop loss comprado %
- Ativação de trailing comprado %
- Offset de trailing comprado %
- Fechar comprado por sinal
- Habilitar vendido
- EMA rápida para vendido
- EMA lenta para vendido
- Fator de deslocamento ascendente
- Stop loss vendido %
- Ativação de trailing vendido %
- Offset de trailing vendido %
- Fechar vendido por sinal
