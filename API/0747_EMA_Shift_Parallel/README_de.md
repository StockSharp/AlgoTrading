# EMA-Parallelverschiebungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt, wenn eine verschobene schnelle EMA eine langsame EMA kreuzt. Long- und Short-Seiten können separat aktiviert werden. Jede Seite unterstützt optionalen Stop-Loss und Trailing-Stop auf Prozentbasis.

## Parameter
- Kerzentyp
- Long aktivieren
- Schnelle EMA für Long
- Langsame EMA für Long
- Abwärts-Verschiebungsfaktor
- Long-Stop-Loss %
- Long-Trailing-Aktivierung %
- Long-Trailing-Offset %
- Long bei Signal schließen
- Short aktivieren
- Schnelle EMA für Short
- Langsame EMA für Short
- Aufwärts-Verschiebungsfaktor
- Short-Stop-Loss %
- Short-Trailing-Aktivierung %
- Short-Trailing-Offset %
- Short bei Signal schließen
