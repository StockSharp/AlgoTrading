# Прорыв наклона Stochastic
[English](README.md) | [中文](README_zh.md)

Стратегия Stochastic Slope Breakout отслеживает скорость изменения стохастика. Слишком крутой наклон свидетельствует о зарождении нового тренда.

Позиция открывается, когда наклон превышает обычный уровень на несколько стандартных отклонений и торгуется в направлении ускорения с защитным стопом.

Она ориентирована на активных трейдеров, ищущих раннее вхождение в тренд. Закрытие происходит, когда наклон возвращается к нормальным значениям. Значение по умолчанию `StochPeriod` = 14.

## Подробности

- **Условие входа**: Индикатор превышает среднее значение на величину коэффициента отклонения.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `StochPeriod` = 14
  - `KPeriod` = 3
  - `DPeriod` = 3
  - `LookbackPeriod` = 20
  - `DeviationMultiplier` = 2m
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Stochastic
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 91%\. Стратегию лучше запускать на фондовом рынке.
