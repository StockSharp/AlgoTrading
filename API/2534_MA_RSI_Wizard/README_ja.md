# MA + RSI Wizard戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略はMetaTrader 5の`MQL/17489`フォルダーにある「MQL5 Wizard MA RSI」エキスパートのStockSharpポートです。元のロボットは移動平均フィルターとRSIフィルターを組み合わせ、フィルターの加重合計が設定可能な閾値をクロスしたときにトレードを開きます。C#バージョンはStockSharpの高レベルAPIと最新のリスク管理ヘルパーでロジックを表現しながら同じ構造を維持します。

ボットはOHLCVローソク足を提供する任意のインストゥルメントで動作します。ユーザーが定義したバー数だけ遅らせることができる移動平均と、異なる価格ソースで供給できるRSIを評価します。両方のインジケーターが複合スコアに貢献します。スコアが開場閾値を超えるとポジションが開かれ、反対のスコアが終了閾値に達したときに閉じられます。オプションの距離、ストップロス、テイクプロフィットの設定は元のエキスパートアドバイザーの資金管理パラメーターを再現します。

## インジケーターとスコアリング

* **移動平均** – 設定可能な期間、メソッド（単純、指数、スムーズ、線形加重）、価格ソースおよび前方シフト。終値がシフトされた平均を上回っている場合、MAスコアは100に等しく、そうでない場合は0です。
* **相対力指数（RSI）** – 設定可能な期間と価格ソース。ロングシグナルではRSIが50から100になるにつれて0から100に線形的に増加し、ショートシグナルでも同じ動作を反映します。
* **複合スコア** – MAとRSIスコアは`MaWeight`と`RsiWeight`で重み付けされます。最終値は加重平均 `score = (maScore * MaWeight + rsiScore * RsiWeight) / (MaWeight + RsiWeight)` で、MetaTraderバージョンと同様に[0;100]区間内に収まります。
* **価格距離フィルター** – `PriceLevelPoints`はローソク足終値とシフトされた移動平均間の最小距離を定義します（インストゥルメントステップを使用して価格に変換）。閾値より近いシグナルは無視されます。

## トレードルール

1. 完了したローソク足ごとにインジケーターとスコアを更新します。
2. 反対のスコアが`ThresholdClose`を超えた場合、現在のポジションは成行で閉じられます。
3. ロングエントリー – ロングエクスポージャーがなく、ロングスコアが少なくとも`ThresholdOpen`であり、クールダウン（`ExpirationBars`）が経過し、価格距離フィルターが満たされている場合に許可されます。注文サイズは`Volume + |Position|`で、必要に応じてショートポジションを自動的に反転します。
4. ショートエントリー – ロングロジックと対称。
5. オプションの`StartProtection`は絶対的な価格ポイントを使用してストップロスとテイクプロフィットを適用します。

## リスク管理

戦略は開始時に一度`StartProtection`をアクティブにします。距離は価格ポイントで定義されます（`StopLevelPoints`、`TakeLevelPoints`）。現在の`Security.PriceStep`で変換されます。両方の値を0に設定して対応する保護を無効にすることができます。クールダウンパラメーターは同じ方向への即時再エントリーを防ぎ、元のEAの指値注文期限切れ設定をエミュレートします。

## パラメーター

| パラメーター | 説明 | デフォルト値 |
|-----------|-------------|---------|
| `CandleType` | 分析に使用するデータシリーズ。 | 15分の時間軸 |
| `ThresholdOpen` | ポジションを開くための最小加重スコア。 | 55 |
| `ThresholdClose` | ポジションを閉じるための最小反対スコア。 | 100 |
| `PriceLevelPoints` | 価格とシフトされたMAの間の必要な距離（ポイント）。 | 0 |
| `StopLevelPoints` | ストップロス距離（ポイント）。 | 50 |
| `TakeLevelPoints` | テイクプロフィット距離（ポイント）。 | 50 |
| `ExpirationBars` | 同じ方向への再エントリー前のバーでのクールダウン。 | 4 |
| `MaPeriod` | 移動平均期間。 | 20 |
| `MaShift` | MA出力に適用される前方シフト（バー）。 | 3 |
| `MaMethods` | 移動平均メソッド（Simple、Exponential、Smoothed、LinearWeighted）。 | Simple |
| `MaAppliedPrice` | MAの価格ソース。 | Close |
| `MaWeight` | MAスコアに割り当てられた重み。 | 0.8 |
| `RsiPeriod` | RSI期間。 | 3 |
| `RsiAppliedPrice` | RSIの価格ソース。 | Close |
| `RsiWeight` | RSIスコアに割り当てられた重み。 | 0.5 |

## 注意事項

* 戦略は完成したローソク足でのみ厳密に実行され、部分的な更新を無視します。
* 両方のインジケーター重みをゼロに設定すると、複合スコアが閾値に達することができなくなるため、取引が無効になります。
* クールダウン（`ExpirationBars`）がゼロに等しい場合、待機なしで同じ方向への複数エントリーが可能になります。
* StockSharpはデフォルトで成行注文を実行するため、元のEAの指値注文の期限切れは実際の注文キャンセルの代わりにクールダウンメカニズムで表されます。
