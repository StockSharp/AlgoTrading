# EMA Moving Away 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

EMA Moving Away は、価格が指数移動平均からどれだけ離れているかを追跡します。
一連のローソク足が価格をEMAから設定されたパーセンテージ分押し下げると、
この戦略は平均への回帰に賭けます。

このセットアップはロング側に焦点を当てています：価格がEMAを `MovingAwayPercent`
分下回る延長された弱気な動きの後、ポジションが開かれます。ボディサイズと
連続フィルターを追加して、その動きがノイズではなく伸びていることを確認できます。
パーセンテージストップロスが逆張りが失敗した場合に資本を保護します。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 終値がEMAを `MovingAwayPercent` 分下回り、必要な連続/サイズフィルターを満たす。
  - **ショート**: 使用しない。
- **エグジット条件**: EMAへの回帰またはストップロスの発動。
- **ストップ**: `StopLossPercent` に基づくパーセンテージストップ。
- **デフォルト値**:
  - `EmaLength` = 55
  - `MovingAwayPercent` = 2.0
  - `StopLossPercent` = 2.0
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロングのみ
  - インジケーター: EMA
  - 複雑さ: 中程度
  - リスクレベル: 中
