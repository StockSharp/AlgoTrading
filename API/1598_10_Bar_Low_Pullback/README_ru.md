# Стратегия Short Only 10 Bar Low Pullback
[English](README.md) | [中文](README_cn.md)

Стратегия открывает короткую позицию, когда цена пробивает минимум предыдущих баров и показатель IBS превышает порог. Дополнительный фильтр EMA подтверждает нисходящий тренд.

## Подробности

- **Условия входа**:
  - Цена минимального значения пробивает минимум за `LowestPeriod` баров.
  - IBS > `IbsThreshold`.
  - Опционально: цена закрытия ниже EMA при включённом фильтре.
  - Время внутри интервала `StartTime` – `EndTime`.
- **Направление**: Только шорт.
- **Условия выхода**:
  - Цена закрытия ниже предыдущего минимума закрывает позицию.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `LowestPeriod` = 10
  - `IbsThreshold` = 0.85
  - `UseEmaFilter` = true
  - `EmaPeriod` = 200
- **Фильтры**:
  - Категория: Откат
  - Направление: Шорт
  - Индикаторы: Lowest, EMA
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
