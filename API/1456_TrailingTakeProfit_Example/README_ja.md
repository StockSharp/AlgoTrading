## 詳細
- **エントリー条件**: 速いSMAが遅いSMAを上抜けまたは下抜け。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 前のバーの高値または安値からのATRベースのトレーリングテイクプロフィット。
- **ストップ**: なし。
- **デフォルト値**: 速い 14、遅い 28、ATR期間 7、ATR乗数 1.5。
- **フィルター**: なし。
