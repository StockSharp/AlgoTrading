## Details
- **Einstiegskriterien**: Schnelle SMA kreuzt die langsame SMA von oben oder unten.
- **Long/Short**: Beide Richtungen.
- **Ausstiegskriterien**: ATR-basierter Trailing-Take-Profit vom Hoch oder Tief des vorherigen Balkens.
- **Stops**: Keine.
- **Standardwerte**: Schnell 14, Langsam 28, ATR-Länge 7, ATR-Multiplikator 1.5.
- **Filter**: Keine.
