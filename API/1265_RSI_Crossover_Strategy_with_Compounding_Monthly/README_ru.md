# Стратегия "RSI Crossover Strategy with Compounding (Monthly)"
[English](README.md) | [中文](README_zh.md)

Стратегия инвестирует весь капитал, когда месячный RSI закрывается выше своей SMA, и выходит, когда RSI опускается ниже SMA. Прибыль добавляется к капиталу, обеспечивая сложный процент.

Тесты показывают среднегодовую доходность около 20%. Лучше всего работает на акциях.

## Детали

- **Условия входа**: RSI выше SMA
- **Длинная/короткая**: только длинная
- **Условия выхода**: RSI ниже SMA
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = 1 месяц
  - `RsiPeriod` = 14
  - `InitialCapital` = 100000
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Длинная
  - Индикаторы: RSI, SMA
  - Стопы: Нет
  - Сложность: Начальный уровень
  - Таймфрейм: Месячный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
