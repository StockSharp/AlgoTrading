# MFIヒストグラム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MFIヒストグラム戦略は、マネーフローインデックス（MFI）を使用して、設定可能な閾値を通じて買われすぎと売られすぎの状態を検出します。MFIは価格とボリュームを組み合わせて資金流入と流出の強度を測定します。インジケーターが下から高いレベルを上に突き抜けると、戦略はこれを買い圧力の上昇と解釈してロングポジションを開き、既存のショートポジションを閉じます。逆に、低いレベルを下に突き抜けるとショートエントリーが発動し、既存のロングポジションを閉じます。ストップロスとテイクプロフィットの値は組み込みの保護メカニズムを通じてティックで管理されます。

戦略はユーザー定義のローソク足の時間軸（デフォルト4時間）で動作し、追加フィルターなしで単一のインジケーターに依存します。パラメーターによりMFI期間、閾値レベル、リスク上限を最適化でき、様々な市場とボラティリティのレジームに適応可能です。

## 詳細

- **エントリー条件**:
  - **ロング**: `MFI` が下から `HighLevel` を上に突き抜ける。
  - **ショート**: `MFI` が上から `LowLevel` を下に突き抜ける。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆のシグナルが反転を生成する。
  - ストップロスまたはテイクプロフィットに到達する。
- **ストップ**: `StopLoss` および `TakeProfit`（ティック単位）。
- **デフォルト値**:
  - `MFI Period` = 14
  - `HighLevel` = 60
  - `LowLevel` = 40
  - `Candle Type` = 4-hour
  - `StopLoss` = 1000 ticks
  - `TakeProfit` = 2000 ticks
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
