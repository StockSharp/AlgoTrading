# Стратегия EMA Pullback Speed
[English](README.md) | [中文](README_cn.md)

Стратегия EMA Pullback Speed использует динамическую EMA, адаптирующуюся к ускорению цены. Лонг открывается, когда цена возвращается к динамической EMA во время восходящего тренда с бычьим разворотом и достаточной скоростью роста. Шорт открывается при противоположных условиях. Выходы осуществляются по стоп-лоссу на основе ATR и фиксированному тейк-профиту.

## Детали

- **Условия входа**:
  - **Лонг**: цена выше динамической EMA, бычий разворот, цена вернулась к EMA, положительная скорость, короткая EMA выше длинной EMA, скорость ≥ `LongSpeedMin`.
  - **Шорт**: цена ниже динамической EMA, медвежий разворот, цена вернулась к EMA, отрицательная скорость, короткая EMA ниже длинной EMA, скорость ≤ `ShortSpeedMax`.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**: стоп-лосс по ATR и фиксированный тейк-профит.
- **Стопы**: стоп-лосс `AtrMultiplier`×ATR, тейк-профит `FixedTpPct`%.
- **Значения по умолчанию**:
  - `MaxLength` = 50
  - `AccelMultiplier` = 3
  - `ReturnThreshold` = 5
  - `AtrLength` = 14
  - `AtrMultiplier` = 4
  - `FixedTpPct` = 1.5
  - `ShortEmaLength` = 21
  - `LongEmaLength` = 50
  - `LongSpeedMin` = 1000
  - `ShortSpeedMax` = -1000
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: EMA, ATR
  - Стопы: стоп-лосс по ATR, фиксированный тейк-профит
  - Сложность: Средняя
  - Таймфрейм: 5m
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
