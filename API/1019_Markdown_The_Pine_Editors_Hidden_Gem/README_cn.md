# Markdown The Pine Editor's Hidden Gem 策略
[English](README.md) | [Русский](README_ru.md)

该策略使用建立在简单移动平均线之上的布林带。当收盘价突破上轨时开多仓，当收盘价跌破下轨时开空仓。

## 详情

- **入场条件**：
  - **多头**：收盘价上穿上轨。
  - **空头**：收盘价下穿下轨。
- **多空方向**：双向。
- **出场条件**：
  - 相反信号。
- **止损**：无。
- **默认值**：
  - `Length` = 50
  - `Multiplier` = 2
- **过滤器**：
  - 分类：趋势跟随
  - 方向：双向
  - 指标：布林带
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
