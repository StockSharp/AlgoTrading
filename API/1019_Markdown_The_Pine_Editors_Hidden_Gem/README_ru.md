# Стратегия Markdown The Pine Editor's Hidden Gem
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует полосы Боллинджера, построенные на простой скользящей средней. Длинная позиция открывается, когда цена закрытия выше верхней полосы, короткая — когда цена закрытия ниже нижней полосы.

## Подробности

- **Критерии входа**:
  - **Long**: цена закрытия пересекает верхнюю полосу.
  - **Short**: цена закрытия пересекает нижнюю полосу.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**:
  - Противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Length` = 50
  - `Multiplier` = 2
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Полосы Боллинджера
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
