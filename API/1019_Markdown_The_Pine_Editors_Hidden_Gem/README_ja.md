# Markdown Pine エディターの隠れた名作戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は単純移動平均に基づいたボリンジャーバンドを使用します。価格が上限バンドを上回って終値をつけたときにロングポジションを開き、下限バンドを下回って終値をつけたときにショートポジションを開きます。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が上限バンドを上抜け。
  - **ショート**: 終値が下限バンドを下抜け。
- **ロング/ショート**: 両サイド。
- **エグジット条件**:
  - 反対シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 50
  - `Multiplier` = 2
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Bollinger Bands
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
