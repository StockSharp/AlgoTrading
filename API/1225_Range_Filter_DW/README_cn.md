# Range Filter DW 策略
[Русский](README_ru.md) | [English](README.md)

该策略实现基于 ATR 的范围过滤器，灵感来自 Donovan Wall 的 Range Filter。过滤器只在价格超过基于波动率的区间时移动，从而忽略细小波动。收盘价突破上轨时开多单，跌破下轨时开空单。

## 细节

- **入场条件**:
  - **做多**: 收盘价高于上轨。
  - **做空**: 收盘价低于下轨。
- **多空方向**: 双向。
- **出场条件**:
  - 突破相反轨道。
- **止损**: 无。
- **默认值**:
  - `RangePeriod` = 14
  - `RangeMultiplier` = 2.618
- **过滤器**:
  - 分类: 趋势跟随
  - 方向: 双向
  - 指标: ATR
  - 止损: 无
  - 复杂度: 中等
  - 时间框架: 任意
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
