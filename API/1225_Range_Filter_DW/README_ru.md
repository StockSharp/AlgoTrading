# Стратегия Range Filter DW
[English](README.md) | [中文](README_cn.md)

Стратегия реализует диапазонный фильтр на основе ATR по мотивам Range Filter Донована Уолла. Фильтр игнорирует небольшие колебания цены и сдвигается только при превышении диапазона, зависящего от волатильности. Лонг открывается при закрытии выше верхней границы, шорт — при закрытии ниже нижней.

## Подробности

- **Условия входа**:
  - **Лонг**: Закрытие выше верхней границы.
  - **Шорт**: Закрытие ниже нижней границы.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Пробой противоположной границы.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `RangePeriod` = 14
  - `RangeMultiplier` = 2.618
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: ATR
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
