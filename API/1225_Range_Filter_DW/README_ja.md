# レンジフィルター DW戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はDonovan WallのRange Filterに類似したATRベースのレンジフィルターを実装しています。フィルターはボラティリティベースのレンジを超えたときのみ移動することで、細かい価格変動を無視します。終値が上バンドを上回った場合にロングポジションを開き、終値が下バンドを下回った場合にショートポジションを開きます。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が上バンドを上回る。
  - **ショート**: 終値が下バンドを下回る。
- **ロング/ショート**: 両サイド。
- **エグジット条件**:
  - 反対バンドのブレイクアウト。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `RangePeriod` = 14
  - `RangeMultiplier` = 2.618
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: ATR
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
