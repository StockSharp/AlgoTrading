# Индикатор Williams %R
[English](README.md) | [中文](README_cn.md)

Стратегия основана на показателе Williams %R.

Williams %R определяет зоны перекупленности и перепроданности. Когда индикатор поднимается выше верхнего порога, это сигнал о возможной слабости и повод для коротких позиций; значения ниже нижнего порога указывают на покупки. Позиции закрываются, когда %R возвращается к нейтральной зоне.

Так как %R быстро колеблется, стратегия может давать много сигналов на волатильных рынках. Некоторые трейдеры комбинируют его с другими фильтрами для уменьшения шума.

## Подробности

- **Критерии входа**: сигналы на основе Williams.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал или стоп.
- **Стопы**: да.
- **Параметры по умолчанию**:
  - `Period` = 14
  - `StopLossPercent` = 2м
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Williams
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
