# 3Commas Turtle戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ドンチャンチャネルを使用したTurtleスタイルの簡易ブレイクアウトシステムです。価格が遅いチャネルより上にある状態で速いチャネルを上方ブレイクしたときに買い、遅いチャネルの確認を受けて速いチャネルを下方ブレイクしたときに売ります。価格が逆方向の終了チャネルを越えたときにエグジットします。

## 詳細
- **エントリー条件**: 遅いチャネルの確認を伴う速いチャネルのブレイクアウト。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 価格が終了チャネルを越える。
- **ストップ**: チャネルベース。
- **デフォルト値**:
  - `PeriodFast` = 20
  - `PeriodSlow` = 20
  - `PeriodExit` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: ドンチャンチャネル
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
