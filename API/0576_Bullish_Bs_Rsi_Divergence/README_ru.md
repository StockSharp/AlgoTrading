# Bullish B's RSI Divergence
[English](README.md) | [中文](README_cn.md)

Использует RSI для обнаружения обычных и скрытых бычьих дивергенций с помощью пивотов. Открывает длинные позиции при дивергенции и закрывает при медвежьем сигнале, достижении цели RSI или трейлинг-стопе.

## Детали

- **Критерии входа**:
  - **Лонг**: Обычная или скрытая бычья дивергенция RSI.
- **Направление**: Только лонг.
- **Критерии выхода**: Медвежья дивергенция, пересечение RSI выше цели или трейлинг-стоп.
- **Стопы**: Необязательный трейлинг-стоп на ATR или процентах.
- **Значения по умолчанию**:
  - `RsiPeriod` = 9
  - `PivotLookbackRight` = 3
  - `PivotLookbackLeft` = 1
  - `TakeProfitRsiLevel` = 80
  - `RangeUpper` = 60
  - `RangeLower` = 5
  - `StopType` = None
  - `StopLoss` = 5
  - `AtrLength` = 14
  - `AtrMultiplier` = 3.5
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Дивергенция
  - Направление: Лонг
  - Индикаторы: RSI, ATR
  - Стопы: Трейлинг-стоп (опционально)
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Риск: Средний
