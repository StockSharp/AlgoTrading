# Стратегия Trend Following Candles
[English](README.md) | [中文](README_cn.md)

Стратегия следует за трендом, используя скользящую среднюю и простые свечные сигналы.
Покупает, когда цена выше средней и бычья свеча пробивает пивотное сопротивление, продаёт при цене ниже средней и медвежьей свече, пробивающей поддержку.

## Детали

- **Критерий входа**: бычья/медвежья свеча выше/ниже средней, пробивающая уровни пивота
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `MaPeriod` = 10
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
