# Trend Following Candles 策略
[English](README.md) | [Русский](README_ru.md)

该策略使用移动平均线和简单的蜡烛信号来跟随趋势。
当价格位于均线上方且看涨蜡烛突破枢轴阻力时买入；当价格位于均线下方且看跌蜡烛跌破支撑时卖出。

## 细节

- **入场条件**: 看涨/看跌蜡烛在均线上方/下方突破枢轴水平
- **多空方向**: 双向
- **出场条件**: 反向信号
- **止损**: 无
- **默认值**:
  - `MaPeriod` = 10
- **过滤器**:
  - 分类: 趋势
  - 方向: 双向
  - 指标: SMA
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
