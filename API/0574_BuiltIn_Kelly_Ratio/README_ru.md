# Встроенное соотношение Келли
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя канала на основе скользящей средней и ATR с управлением размером позиции по критерию Келли.

## Подробности

- **Критерии входа**: цена пересекает верхнюю или нижнюю границу канала ATR.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: опциональные тейк-профит и стоп-лосс.
- **Стопы**: опционально.
- **Параметры по умолчанию**:
  - `Length` = 20
  - `Multiplier` = 1
  - `AtrLength` = 10
  - `UseEma` = true
  - `UseKelly` = true
  - `UseTakeProfit` = false
  - `UseStopLoss` = false
  - `TakeProfit` = 10
  - `StopLoss` = 1
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: MA, ATR
  - Стопы: Опционально
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
