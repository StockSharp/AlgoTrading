# Стратегия Good Mode RSI v2
[English](README.md) | [中文](README_cn.md)

Эта система торгует на экстремальных значениях индикатора RSI с заданными уровнями фиксации прибыли и трейлинг-стопом. Продажа открывается при превышении RSI заданного верхнего уровня и закрывается при падении RSI до уровня фиксации прибыли. Покупка открывается при снижении RSI ниже нижнего уровня и закрывается при подъёме RSI до целевого значения. В обоих случаях используется динамический трейлинг-стоп, следящий за максимальной выгодной ценой.

## Детали

- **Условия входа**:
  - **Long**: `RSI < buy level`.
  - **Short**: `RSI > sell level`.
- **Long/Short**: Оба направления.
- **Условия выхода**:
  - **Long**: `RSI > take profit level buy` или срабатывание трейлинг-стопа.
  - **Short**: `RSI < take profit level sell` или срабатывание трейлинг-стопа.
- **Стопы**: Трейлинг-стоп в тиках.
- **Параметры по умолчанию**:
  - `RSI Period` = 2
  - `Sell Level` = 96
  - `Buy Level` = 4
  - `Take Profit Level Sell` = 20
  - `Take Profit Level Buy` = 80
  - `Trailing Stop Offset` = 100
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: Один
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
