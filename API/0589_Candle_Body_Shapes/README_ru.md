# Стратегия Candle Body Shapes
[English](README.md) | [中文](README_cn.md)

Стратегия торгует в зависимости от того, где свеча открывается и закрывается внутри своего диапазона.
Длинная позиция открывается, когда свеча открылась рядом с минимумом и закрылась возле максимума, что показывает сильное давление быков.
Короткая позиция открывается, когда свеча открылась рядом с максимумом и закрылась возле минимума, указывая на сильное давление медведей.

Подход основан исключительно на ценовом действии и подходит для любых ликвидных рынков.

## Подробности

- **Условия входа**:
  - Длинная: `Open near Low && Close near High`
  - Короткая: `Open near High && Close near Low`
- **Long/Short**: Оба
- **Условия выхода**: противоположный сигнал
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `BodyThreshold` = 0.2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Candlestick pattern
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
