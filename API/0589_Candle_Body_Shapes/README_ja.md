# ローソク足の実体形状戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ローソク足がその値幅内のどこで始値と終値をつけるかに基づいて取引する戦略。
ローソク足が安値付近で始まり高値付近で終わるとき（強い強気圧力）にロングで参入します。
ローソク足が高値付近で始まり安値付近で終わるとき（強い弱気圧力）にショートで参入します。

このアプローチは純粋にプライスアクションに基づいており、あらゆる流動性の高い市場に適用できます。

## 詳細

- **エントリー条件**:
  - ロング: `Open near Low && Close near High`
  - ショート: `Open near High && Close near Low`
- **ロング/ショート**: 両方
- **エグジット条件**: 逆シグナル
- **ストップ**: いいえ
- **デフォルト値**:
  - `BodyThreshold` = 0.2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: ローソク足パターン
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
