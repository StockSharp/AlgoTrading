# Adam and Eve戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Heiken Ashiローソク足と単純移動平均のカスケードを組み合わせたトレンドフォロー戦略。上ヒゲのない弱気なHeiken Ashiローソク足が現れ、監視中の全ての移動平均（5、7、9、10、12、14、20）が下向きのときにショートポジションが開かれます。下ヒゲのない強気のローソク足と全ての平均が上向きのときにロングポジションが発動します。各取引はエントリーから1 ATR(14)の距離でのターゲット利益を目指し、ストップロスはありません。

## 詳細

- **エントリー条件**: 上ヒゲなし（ショート）または下ヒゲなし（ロング）の前のHeiken Ashiローソク足と整列したSMAスタック
- **ロング/ショート**: 両方
- **エグジット条件**: ATR(14)距離での利益目標
- **ストップ**: なし
- **デフォルト値**:
  - `AtrPeriod` = 14
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: SMA (5,7,9,10,12,14,20), Heiken Ashi, ATR
  - ストップ: 目標のみ
  - 複雑さ: 中級
  - 時間軸: 設定可能、デフォルト15分
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
