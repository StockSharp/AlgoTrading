# Стратегия Adam and Eve
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия, сочетающая свечи Heiken Ashi и каскад простых скользящих средних. Короткая позиция открывается, когда появляется медвежья свеча Heiken Ashi без верхней тени и все контролируемые скользящие средние (5, 7, 9, 10, 12, 14, 20) направлены вниз. Длинная позиция открывается при бычьей свече без нижней тени и всех средних, направленных вверх. Каждая сделка ориентируется на прибыль на расстоянии одного ATR(14) от входа без стоп-лосса.

## Детали

- **Условия входа**: предыдущая свеча Heiken Ashi без верхней (шорт) или нижней (лонг) тени и согласованный набор SMA
- **Лонг/Шорт**: Оба
- **Условия выхода**: цель по прибыли на расстоянии ATR(14)
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `AtrPeriod` = 14
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA (5,7,9,10,12,14,20), Heiken Ashi, ATR
  - Стопы: Только цель
  - Сложность: Средняя
  - Таймфрейм: Настраиваемый, по умолчанию 15 минут
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
