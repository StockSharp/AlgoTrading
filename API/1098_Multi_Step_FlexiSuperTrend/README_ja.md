# マルチステップ FlexiSuperTrend 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

平滑化された偏差オシレーターと組み合わせたSuperTrendフィルター。
戦略には3つの設定可能なテイクプロフィットレベルが含まれます。

## 詳細

- **エントリー条件**:
  - 価格がSuperTrendを下回り、偏差（価格からSuperTrendを引いたSMA）> 0 → 買い。
  - 価格がSuperTrendを上回り、偏差 < 0 → 売り。
- **ロング/ショート**: ロング、ショート、または両方向。
- **エグジット条件**:
  - 3つのレベルで部分的なテイクプロフィット。
  - 価格がSuperTrendを越えてトレンド転換したとき、残りのポジションを決済。
- **ストップ**: デフォルトではストップロジックなし。
- **デフォルト値**:
  - ATR期間 = 10。
  - ATRファクター = 3.0。
  - SMAの長さ = 10。
  - テイクプロフィットレベル = 2%、8%、18%。
  - テイクプロフィット割合 = 30%、20%、15%。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: SuperTrend, SMA
  - ストップ: テイクプロフィット
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
