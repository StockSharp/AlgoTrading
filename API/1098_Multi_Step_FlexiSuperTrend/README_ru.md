# Стратегия Multi-Step FlexiSuperTrend
[English](README.md) | [中文](README_cn.md)

Фильтр SuperTrend в сочетании со сглаженным осциллятором отклонения.
Стратегия содержит три настраиваемых уровня фиксации прибыли.

## Детали

- **Условия входа**:
  - Цена ниже SuperTrend и отклонение (SMA от разницы цены и SuperTrend) > 0 → покупка.
  - Цена выше SuperTrend и отклонение < 0 → продажа.
- **Лонг/Шорт**: можно выбрать лонг, шорт или оба.
- **Условия выхода**:
  - Частичная фиксация прибыли на 3 уровнях.
  - Оставшаяся позиция закрывается при развороте тренда, когда цена пересекает SuperTrend.
- **Стопы**: логика стопов по умолчанию отсутствует.
- **Параметры по умолчанию**:
  - Период ATR = 10.
  - Множитель ATR = 3.0.
  - Длина SMA = 10.
  - Уровни фиксации прибыли = 2%, 8%, 18%.
  - Процент фиксации прибыли = 30%, 20%, 15%.
- **Фильтры**:
  - Категория: следование за трендом
  - Направление: оба
  - Индикаторы: SuperTrend, SMA
  - Стопы: фиксация прибыли
  - Сложность: средняя
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
