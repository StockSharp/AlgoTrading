# Multi-Step FlexiSuperTrend 策略
[English](README.md) | [Русский](README_ru.md)

结合 SuperTrend 滤波和平滑偏差振荡器。
策略包含三个可配置的分批止盈级别。

## 细节

- **入场条件**：
  - 价格低于 SuperTrend 且偏差（价格减去 SuperTrend 的 SMA）> 0 → 做多。
  - 价格高于 SuperTrend 且偏差 < 0 → 做空。
- **多/空**：可选择做多、做空或双向。
- **出场条件**：
  - 在三个级别分批止盈。
  - 当价格穿越 SuperTrend 时关闭剩余头寸。
- **止损**：默认无止损逻辑。
- **默认参数**：
  - ATR 周期 = 10。
  - ATR 系数 = 3.0。
  - SMA 长度 = 10。
  - 止盈级别 = 2%、8%、18%。
  - 止盈比例 = 30%、20%、15%。
- **过滤器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：SuperTrend、SMA
  - 止损：止盈
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
