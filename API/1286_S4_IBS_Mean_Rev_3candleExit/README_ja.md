# S4 IBS 平均回帰戦略（3本ローソク足エグジット）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、前のローソク足の内部バー強度（IBS）が閾値を下回ったときに買いエントリーし、平均回帰を期待します。価格がエントリー価格を上回って終値をつけたとき、またはトレードが損失のまま3本のローソク足が経過したときにエグジットします。

## 詳細

- **エントリー条件**: 前回の IBS <= 閾値
- **ロング/ショート**: ロングのみ
- **エグジット条件**: エントリー価格を上回る終値、またはエントリー価格を下回ったまま3本経過後；終了時刻後に強制エグジット
- **ストップ**: なし
- **デフォルト値**:
  - `IbsThreshold` = 0.25
  - `StartTime` = 2024-01-01 05:00:00 UTC
  - `EndTime` = 2024-12-31 00:00:00 UTC
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロング
  - インジケーター: なし
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: 日足
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
