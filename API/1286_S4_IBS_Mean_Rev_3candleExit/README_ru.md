# Стратегия S4 IBS Mean Rev 3candleExit
[English](README.md) | [中文](README_cn.md)

Стратегия покупает, когда внутренняя сила бара (IBS) предыдущей свечи ниже порога, рассчитывая на возврат к среднему. Выход происходит при закрытии цены выше входа или после трёх свечей, если сделка остаётся убыточной.

## Детали

- **Критерий входа**: IBS предыдущей свечи <= порога
- **Длин/Шорт**: только лонг
- **Критерий выхода**: закрытие выше цены входа или выход через 3 свечи, если цена ниже; принудительный выход после конца периода
- **Стопы**: нет
- **Значения по умолчанию**:
  - `IbsThreshold` = 0.25
  - `StartTime` = 2024-01-01 05:00:00 UTC
  - `EndTime` = 2024-12-31 00:00:00 UTC
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Лонг
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
