# Smart Money Concept-Strategie - Uncle Sam
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Ausbruch-Strategie überwacht aktuelle Swing-Hochs und -Tiefs. Ein Long-Trade wird eröffnet, wenn der Preis über das letzte Pivot-Hoch schließt, während ein Short-Trade eröffnet wird, wenn der Preis unter das letzte Pivot-Tief schließt. Ein optionaler Filter für gleitende Durchschnitte kann aktiviert werden, um nur in Richtung des vorherrschenden Trends zu handeln.

## Details

- **Einstiegskriterien**:
  - **Long**: Schlusskurs überschreitet das jüngste Pivot-Hoch (und liegt über dem MA, wenn aktiviert).
  - **Short**: Schlusskurs unterschreitet das jüngste Pivot-Tief (und liegt unter dem MA, wenn aktiviert).
- **Long/Short**: Beide.
- **Indikatoren**: Pivot-Erkennung, Gleitender Durchschnitt (optional).
- **Zeitrahmen**: Konfigurierbar.
- **Komplexität**: Moderat.
