# Smart Money コンセプト戦略 - Uncle Sam
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このブレイクアウト戦略は直近のスイング高値・安値を監視します。価格が直近のピボット高値を上回って終値をつけたときにロングトレードを開き、直近のピボット安値を下回って終値をつけたときにショートトレードを開きます。オプションの移動平均フィルターを有効にすると、優勢なトレンド方向にのみ取引できます。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が直近のピボット高値を上抜け（MAが有効な場合はMAの上方）。
  - **ショート**: 終値が直近のピボット安値を下抜け（MAが有効な場合はMAの下方）。
- **ロング/ショート**: 両方。
- **インジケーター**: ピボット検出、移動平均（オプション）。
- **時間軸**: 設定可能。
- **複雑さ**: 中程度。
