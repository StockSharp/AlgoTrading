# Стратегия Bollinger ADX

Стратегия сочетает полосы Боллинджера и индикатор ADX. Она ищет пробои с подтверждением сильного тренда.

Движения цены за пределы полос Боллинджера фильтруются с помощью ADX для подтверждения силы. Торговля ведётся, когда пробой совпадает с высоким значением ADX.

Полезна при всплесках волатильности, сопровождаемых сильным трендом. Размер стопа определяется по ATR.

## Подробности

- **Условия входа**:
  - Лонг: `Close < LowerBand && ADX > AdxThreshold`
  - Шорт: `Close > UpperBand && ADX > AdxThreshold`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Возврат к средней полосе Боллинджера
- **Стопы**: на основе ATR через `AtrMultiplier`
- **Значения по умолчанию**:
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 2.0m
  - `AdxPeriod` = 14
  - `AdxThreshold` = 25m
  - `AtrMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Средняя обратная
  - Направление: Оба
  - Индикаторы: Полосы Боллинджера, ADX
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
