# Estratégia DCA 2
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia compra quando um candle Heikin Ashi verde segue um vermelho dentro do intervalo de anos especificado. As posições são parcialmente fechadas quando o RSI cruza abaixo de um limiar ou a data final é atingida.

## Parâmetros
- Tipo de candle
- Ano de início
- Ano de fim
- Percentual de saída
- RSI de saída
