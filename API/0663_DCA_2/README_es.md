# Estrategia DCA 2
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia compra cuando una vela Heikin Ashi verde sigue a una roja dentro del rango de años especificado. Las posiciones se cierran parcialmente cuando el RSI cruza por debajo de un umbral o se alcanza la fecha final.

## Parámetros
- Tipo de vela
- Año de inicio
- Año de fin
- Porcentaje de salida
- RSI de salida
