# DCA-Strategie 2
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kauft, wenn eine grüne Heikin Ashi-Kerze einer roten innerhalb des angegebenen Jahresbereichs folgt. Positionen werden teilweise geschlossen, wenn der RSI unter einen Schwellenwert fällt oder das Enddatum erreicht ist.

## Parameter
- Kerzentyp
- Startjahr
- Endjahr
- Ausstiegsprozent
- Ausstiegs-RSI
