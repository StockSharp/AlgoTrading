# DCA戦略 2
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、指定した年範囲内で赤いHeikin Ashiローソクに続いて緑のローソクが出現したときに購入します。RSIが閾値を下回るか、終了日に達したときにポジションを部分的に決済します。

## パラメータ
- ローソク足タイプ
- 開始年
- 終了年
- 決済パーセント
- 決済RSI
