# Demo File Write Array 策略
[English](README.md) | [Русский](README_ru.md)

该策略记录最优买价和卖价，并分批将其保存到二进制文件中。

## 细节

- **入场条件：** 无。
- **多空方向：** 无。
- **出场条件：** 无。
- **止损：** 无。
- **默认值：**
  - 文件名 = data.bin
  - 目录名 = SomeFolder
  - 缓冲区大小 = 20 条记录
- **过滤器：**
  - 类别：工具
  - 方向：无
  - 指标：无
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：无
