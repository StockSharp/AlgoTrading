# Стратегия Demo File Write Array
[English](README.md) | [中文](README_cn.md)

Стратегия записывает лучшие цены Bid и Ask и сохраняет их в бинарный файл пакетами.

## Детали

- **Критерии входа:** отсутствуют.
- **Длинные/короткие:** нет.
- **Критерии выхода:** отсутствуют.
- **Стопы:** нет.
- **Значения по умолчанию:**
  - Имя файла = data.bin
  - Имя директории = SomeFolder
  - Размер буфера = 20 записей
- **Фильтры:**
  - Категория: Утилиты
  - Направление: Нет
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Нет
