# RSI適応型T3
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

RSI適応型のTillson T3移動平均に基づくトレンドフォロー戦略です。T3が2バー前の値を上抜けたときにロングエントリーし、逆方向のクロスで決済します。

日足チャートでのバックテストでは、トレンド相場で安定したパフォーマンスを示しています。

## 詳細

- **エントリー条件**: T3が2バー前の値を上抜け。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 逆方向のクロス。
- **ストップ**: なし。
- **デフォルト値**:
  - `RsiLength` = 14
  - `MinT3Length` = 5
  - `MaxT3Length` = 50
  - `VolumeFactor` = 0.7
  - `CandleType` = TimeSpan.FromDays(1)
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロングのみ
  - インジケーター: RSI, T3
  - ストップ: なし
  - 複雑さ: 中級
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
