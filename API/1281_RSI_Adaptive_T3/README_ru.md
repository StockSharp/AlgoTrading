# RSI Adaptive T3
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия на базе адаптивной скользящей Tillson T3. Покупает при пересечении T3 выше её значения двумя барами ранее и выходит при обратном пересечении.

Тесты на дневных графиках показывают стабильную работу на трендовых рынках.

## Детали

- **Условия входа**: пересечение T3 выше её значения 2 бара назад.
- **Длинные/короткие**: только лонг.
- **Условия выхода**: обратное пересечение.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `MinT3Length` = 5
  - `MaxT3Length` = 50
  - `VolumeFactor` = 0.7
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Trend Following
  - Направление: Лонг
  - Индикаторы: RSI, T3
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
