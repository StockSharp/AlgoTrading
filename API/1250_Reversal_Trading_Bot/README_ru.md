# Стратегия Reversal Trading Bot
[English](README.md) | [中文](README_cn.md)

Reversal Trading Bot использует дивергенцию RSI с дополнительными фильтрами по объёму, ADX, полосам Боллинджера и пересечению RSI для поиска разворотов. Позиции защищены фиксированными процентными стоп-лоссом и тейк-профитом.

## Детали

- **Критерий входа**: дивергенция RSI с дополнительными фильтрами объёма, ADX, полос Боллинджера и пересечения RSI
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: стоп-лосс или тейк-профит
- **Стопы**: фиксированный процент
- **Значения по умолчанию**:
  - `RsiLength` = 8
  - `FastRsiLength` = 14
  - `SlowRsiLength` = 21
  - `BbLength` = 20
  - `AdxThreshold` = 20
  - `DivLookback` = 5
  - `StopLossPercent` = 1
  - `TakeProfitPercent` = 2
- **Фильтры**:
  - Категория: Разворот
  - Направление: обе стороны
  - Индикаторы: RSI, ADX, полосы Боллинджера, SMA
  - Стопы: фиксированные
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
