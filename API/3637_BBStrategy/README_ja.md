# BB戦略戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

BBStrategy は、MetaTrader エキスパート アドバイザー「BBStrategy」から変換された Bollinger バンド ブレイクアウト システムです。この戦略は、周期は同じだが偏差乗数が異なる 2 セットの Bollinger バンドを追跡します。価格が外側のバンドを突き抜けると、アルゴリズムは取引を準備しますが、実際のエントリーは、価格が内側のバンドに戻るまで延期されます。この行動は、ボラティリティ拡大後の継続の動きを捉えながら、過度に延長されたブレイクアウトの購入や深く売られすぎた状態での売りを回避しようとします。

## 中核ロジック

1. ローソク足を購読し、2 つの Bollinger バンドを計算します。
   - **アウターバンド**は、設定可能な偏差乗数 (デフォルトは 3.0) を使用します。
   - **内側のバンド**では、より低い偏差が使用されます (デフォルトは 2.0)。
2. 終値が外側のバンドの外側で終了するときを検出します。
   - 上部外側のバンドアームの上に長いセットアップがあります。
   - 下部外側バンドアームの下に短いセットアップがあります。
3. 次に完了したローソク足がブレイクアウトの方向にある内側のバンドの内側で閉じた場合にのみエントリーします。価格が再参入するのを待っている間、戦略は対応する方向に対して「待機」状態に留まります。
4. 条件が一致し、オープンポジションやアクティブな注文がない場合に、単一の成行注文を送信します。既存の反対ポジションは、成行注文の量を増やすことによって決済されます。
5. オプションのテイクプロフィットとストップロスの距離 (ポイントで表現) は、絶対価格オフセットに変換され、組み込みの保護ヘルパーを介して管理されます。

## パラメーター

| 名前 | 説明 |
|------|-------------|
| **注文量** | 各ポジションの取引サイズ。 |
| **Bollinger 期間** | 両方の Bollinger バンドの計算に使用されるローソク足の数。 |
| **内部偏差** | プルバックを検証する内側バンドの偏差乗数。 |
| **外側の偏差** | ブレイクアウトを検出する外側バンドの偏差乗数。 |
| **ストップロスポイント** | 保護停止距離 (ポイント単位) (0 は停止を無効にします)。 |
| **利益確定ポイント** | テイクプロフィット距離 (ポイント単位) (0 はターゲットを無効にします)。 |
| **キャンドルタイプ** | 計算用のローソク足の時間枠。 |

## 注意事項

- この戦略は一度に 1 つのポジションを取引し、注文がアクティブである間は新しいシグナルを無視します。
- リスク管理のために、ヘルパーは商品ティック サイズに基づいて MetaTrader の「ポイント」を実際の価格増分に変換します。
- チャートの描画には、視覚的なデバッグを容易にするために、ローソク足、Bollinger バンドと戦略自体のトレードの両方が含まれています。
