# 動的契約サイズによる月次購入戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

口座資産の固定パーセンテージを使用して、毎月選択した日に動的な数の契約を購入します。ドローダウンは参考情報として記録されます。

## 詳細

- **エントリー条件**: 時刻 >= StartDate かつ 月の日付 = BuyDay
- **ロング/ショート**: ロングのみ
- **エグジット条件**: なし
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = 1日
  - `StartDate` = 2010-01-01
  - `PercentOfEquity` = 0.03
  - `BuyDay` = 1
- **フィルター**:
  - カテゴリ: ドルコスト平均法
  - 方向: ロング
  - インジケーター: いいえ
  - ストップ: いいえ
  - 複雑さ: 初心者
  - 時間軸: 長期
  - 季節性: 月次
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
