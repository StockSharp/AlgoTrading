# 每月购买策略（动态合约规模）
[English](README.md) | [Русский](README_ru.md)

在每月指定日期以账户权益的固定百分比购买动态数量的合约，并跟踪回撤用于参考。

## 细节

- **入场条件**：时间 >= StartDate 且 月份日 = BuyDay
- **多空方向**：仅做多
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `CandleType` = 1 天
  - `StartDate` = 2010-01-01
  - `PercentOfEquity` = 0.03
  - `BuyDay` = 1
- **过滤器**：
  - 类别：定投
  - 方向：多头
  - 指标：无
  - 止损：无
  - 复杂度：初级
  - 时间框架：长期
  - 季节性：每月
  - 神经网络：否
  - 背离：否
  - 风险等级：低
