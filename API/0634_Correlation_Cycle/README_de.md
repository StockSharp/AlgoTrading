# Korrelationszyklus-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie implementiert John Ehlers' Konzepte des Korrelationszyklus, Korrelationswinkels und Marktzustands. Sie berechnet die Korrelation zwischen dem Kurs und Sinus-/Kosinus-Komponenten, um einen Korrelationswinkel zu ermitteln. Wenn der Winkel stabil und über null ist, eröffnet die Strategie eine Long-Position. Wenn der Winkel stabil und unter null ist, eröffnet sie eine Short-Position.

## Parameter
- Kerzentyp
- Periode
- Marktzustand-Schwellenwert
