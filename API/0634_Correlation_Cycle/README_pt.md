# Estratégia de Ciclo de Correlação
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia implementa os conceitos de Ciclo de Correlação, Ângulo de Correlação e Estado do Mercado de John Ehlers. Calcula a correlação entre o preço e os componentes seno/cosseno para obter um ângulo de correlação. Quando o ângulo é estável e está acima de zero, a estratégia entra em uma posição comprada. Quando o ângulo é estável e está abaixo de zero, entra em uma posição vendida.

## Parâmetros
- Tipo de vela
- Período
- Limiar do estado do mercado
