# 相関サイクル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、John Ehlersの相関サイクル、相関角度、マーケット状態の概念を実装しています。価格とサイン/コサイン成分の相関を計算して相関角度を求めます。角度が安定してゼロを上回っているときにロングポジションを取ります。角度が安定してゼロを下回っているときにショートポジションを取ります。

## パラメーター
- ローソク足の種類
- 期間
- マーケット状態のしきい値
