# 出来高ブロック注文アナライザー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradingViewスクリプト **"Volume Block Order Analyzer"** をベースにした簡略化された戦略です。大きな出来高スパイクが価格の方向に与える影響を測定し、この効果を時間とともに累積します。累積的な影響がユーザー定義の閾値を超えると、戦略はポジションに入り、トレーリングストップで保護します。

## 詳細

- **エントリー**: 累積影響が閾値を上回るまたは下回ること。
- **エグジット**: エントリーからのパーセンテージベースのトレーリングストップ。
- **ロング/ショート**: 両方。
- **インジケーター**: SMA。
- **時間軸**: 任意。

このポートはコアのアイデアに焦点を当てており、元のスクリプトの多くのビジュアル機能は省略されています。
