# LeMan Trend Hist戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は元のMQL5エキスパート「LeManTrendHist」を簡略化して変換したものです。EMAベースのヒストグラムを使用して取引シグナルを生成します。

## アイデア

元のアルゴリズムは価格の極値と平滑化されたレンジから導出されるカスタムヒストグラムを計算します。このサンプルでは、ヒストグラムはローソク足レンジの指数移動平均で近似されています。

## 戦略ロジック

1. 確定した各ローソク足のEMA値を計算する。
2. 直近3つのEMA値を比較する。
3. 中央値が最古の値より低く、最新値がその上に上昇した場合、ロングポジションを開きショートポジションを決済する。
4. 中央値が最古の値より高く、最新値がその下に下落した場合、ショートポジションを開きロングポジションを決済する。

## パラメーター

- **Candle Type** – 処理するローソク足の時間軸。
- **EMA Period** – プレースホルダーヒストグラムで使用するEMAの長さ。
- **Signal Bar** – インジケーター値の履歴シフト（互換性のために保持されており、簡略化されたロジックでは使用されない）。
- **Buy/Sell Open** – ロングまたはショートのエントリーを有効にする。
- **Buy/Sell Close** – 既存ポジションの決済を有効にする。

## 備考

真のLeManTrendHistインジケーターは複雑な平滑化アルゴリズムを使用しており、まだ実装されていません。現在の実装はプレースホルダーとして機能しており、本番使用前に完全なインジケーターに置き換える必要があります。
