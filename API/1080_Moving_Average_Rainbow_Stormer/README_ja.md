# 移動平均レインボー戦略 (Stormer)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は12本の移動平均でレインボーを描きます。トレンドが確認され、価格がいずれかの平均に触れたときに取引を開始します。

価格が新高値を更新し、すべての中央平均が上向きで、ローソク足がすべての平均の平均値より上で引けたときにロングポジションが開かれます。逆の条件ではショートポジションが開かれます。

ストップロスは直前に触れた移動平均に設定されます。テイクプロフィットはエントリー価格とストップロスの距離の倍数として計算されます。

## 詳細

- **インジケーター**: 設定可能なタイプの移動平均12本。
- **ロング**: 上昇トレンド、新高値、直前のタッチ価格。
- **ショート**: 下降トレンド、新安値、直前のタッチ価格。
- **エグジット**: タッチした平均でストップロス、目標 = エントリー ± 距離 * 係数。トレンドが反転シグナルを示したときのオプション反転エグジット。
- **パラメーター**: 移動平均タイプ、期間、目標係数、反転オプション。
- **時間軸**: 任意。
