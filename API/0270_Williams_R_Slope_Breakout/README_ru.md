# Прорыв наклона Williams %R

Стратегия Williams R Slope Breakout отслеживает скорость изменения индикатора Williams %R. Слишком крутой наклон указывает на появление нового тренда.

Вход осуществляется, когда наклон превышает обычный уровень на несколько стандартных отклонений. Сделка открывается в направлении ускорения и сопровождается стопом.

Подходит активным трейдерам, желающим рано войти в тренд. Позиции закрываются, когда наклон возвращается к нормальным значениям. Значение по умолчанию `WilliamsRPeriod` = 14.

## Подробности

- **Условие входа**: Индикатор превышает среднее значение на величину коэффициента отклонения.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `WilliamsRPeriod` = 14
  - `SlopePeriod` = 20
  - `BreakoutMultiplier` = 2.0m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Williams
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
