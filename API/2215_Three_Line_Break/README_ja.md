# 三線転換足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Three Line Breakインジケーターが検出したリバーサルを取引する戦略です。
インジケーターは現在の高値と安値を、前のN本の完成したローソク足の最高値と最安値と比較します。
下降トレンド中に直近高値を上抜けると新しい上昇トレンドを示しロングエントリーを発動します。上昇トレンド中に直近安値を下抜けるとショートエントリーを発動します。
各シグナルでポジションが反転されます。

## 詳細

- **エントリー条件**:
  - ロング: `Downtrend` が `Uptrend` に切り替わる
  - ショート: `Uptrend` が `Downtrend` に切り替わる
- **ロング/ショート**: 両方
- **エグジット条件**: 反対シグナル（ポジション反転）
- **ストップ**: なし
- **デフォルト値**:
  - `LinesBreak` = 3
  - `CandleType` = TimeSpan.FromHours(12).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Highest, Lowest (Three Line Breakロジック)
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: スイング
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
