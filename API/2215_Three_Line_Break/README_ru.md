# Стратегия Three Line Break
[English](README.md) | [中文](README_cn.md)

Стратегия торгует развороты, определяемые индикатором Three Line Break.
Индикатор сравнивает текущие максимум и минимум с наивысшим максимумом и наинизшим минимумом предыдущих N завершенных свечей.
Пробой недавнего максимума в нисходящем тренде сигнализирует о смене тренда вверх и открывает длинную позицию; пробой недавнего минимума в восходящем тренде открывает короткую позицию.
Позиции переворачиваются при каждом сигнале.

## Детали

- **Критерии входа**:
  - Long: переход `Downtrend` → `Uptrend`
  - Short: переход `Uptrend` → `Downtrend`
- **Long/Short**: обе стороны
- **Критерии выхода**: противоположный сигнал (переворот позиции)
- **Стопы**: нет
- **Значения по умолчанию**:
  - `LinesBreak` = 3
  - `CandleType` = TimeSpan.FromHours(12).TimeFrame()
- **Фильтры**:
  - Категория: следование тренду
  - Направление: обе
  - Индикаторы: Highest, Lowest (логика Three Line Break)
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
