# SuperTrend + EMAリバウンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このシステムはSuperTrendの方向に取引し、指数移動平均へのプルバックを探します。
SuperTrendラインが方向を反転させた時、または支配的なSuperTrendバイアスを維持しながら
価格がEMAからリバウンドした時にポジションが開かれます。この組み合わせは新しい動きの
最初の段階と、確立されたトレンド内のその後の押し目を捉えることを試みます。

パーセンテージベースのテイクプロフィットは、テイクプロフィットタイプを「%」に設定する
ことで組み込み保護モジュールを通じて有効にできます。デフォルトはロングトレードを優先
しますが、ショートエントリーも有効にできます。この戦略は方向転換に依存しているため、
SuperTrendがモメンタムの変化に素早く反応するトレンド相場で最も効果的です。

## 詳細

- **エントリー条件**:
  - SuperTrendが上昇トレンドに転換、または上昇トレンド中に価格がEMAを上回ってリバウンド。
  - SuperTrendが下降トレンドに転換、または下降トレンド中に価格がEMAを下回ってリバウンド。
- **ロング/ショート**: デフォルトはロング有効、ショートはオプション。
- **エグジット条件**:
  - 反対のSuperTrendの転換。
  - 保護モジュールで処理されるオプションのテイクプロフィット。
- **ストップ**: 保護によるパーセンテージテイクプロフィット。ストップロスは含まれない。
- **デフォルト値**:
  - ATR期間 = 10、ATRファクター = 3.0。
  - EMAの長さ = 20、TP = 1.5%。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方（デフォルトはロング）
  - インジケーター: SuperTrend、EMA
  - ストップ: オプションのTP
  - 複雑さ: 中程度
  - 時間軸: 短期/中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
