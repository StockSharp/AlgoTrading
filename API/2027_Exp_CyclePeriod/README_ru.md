# Exp CyclePeriod Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор CyclePeriod для определения разворотов рыночного цикла. Открывает покупки при росте индикатора и продажи при его падении, закрывая противоположные позиции.

## Подробности

- **Критерии входа:**
  - **Покупка**: CyclePeriod растёт и текущее значение выше предыдущего.
  - **Продажа**: CyclePeriod падает и текущее значение ниже предыдущего.
- **Направление**: Покупка и продажа.
- **Критерии выхода:**
  - Закрыть продажу при развороте CyclePeriod вверх.
  - Закрыть покупку при развороте CyclePeriod вниз.
- **Стопы**: Используются тейк-профит и стоп-лосс в ценовых единицах.
- **Параметры по умолчанию:**
  - `CandleType` = TimeSpan.FromHours(6).TimeFrame().
  - `Alpha` = 0.07.
  - `SignalBar` = 1.
  - `TakeProfit` = 2000.
  - `StopLoss` = 1000.
  - `BuyPosOpen` = true.
  - `SellPosOpen` = true.
  - `BuyPosClose` = true.
  - `SellPosClose` = true.
- **Фильтры:**
  - Категория: Следование тренду
  - Направление: Покупка и продажа
  - Индикаторы: CyclePeriod
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: 6 часов
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
