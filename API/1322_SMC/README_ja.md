# SMC戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

SMC戦略は直近のスイング高値・安値からプレミアム、均衡、ディスカウントゾーンを定義します。SMAトレンドフィルターとシンプルなオーダーブロック確認を使用して、ディスカウントゾーンまたはプレミアムゾーンで取引します。

## 詳細

- **エントリー条件**: オーダーブロックサポートを伴うSMA上方のディスカウントゾーンに価格がある場合；オーダーブロック抵抗を伴うSMA下方のプレミアムゾーンに価格がある場合
- **ロング/ショート**: 両方
- **エグジット条件**: 逆シグナル
- **ストップ**: いいえ
- **デフォルト値**:
  - `SwingHighLength` = 8
  - `SwingLowLength` = 8
  - `SmaLength` = 50
  - `OrderBlockLength` = 20
- **フィルター**:
  - カテゴリ: Zone
  - 方向: 両方
  - インジケーター: Highest, Lowest, SMA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
