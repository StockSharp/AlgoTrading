# Estrategia MTF Seconds Values JD
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia demuestra el manejo de velas personalizadas multitemporal basadas en un intervalo de segundos especificado. Calcula una media móvil simple sobre velas agregadas y opera cuando el precio cruza la media.

## Parámetros

- `SecondsTimeframe` – intervalo en segundos para la agregación de velas.
- `AverageLength` – período para la media móvil simple.
