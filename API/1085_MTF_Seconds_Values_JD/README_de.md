# MTF Seconds Values JD Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie demonstriert die Verarbeitung benutzerdefinierter Multi-Timeframe-Kerzen basierend auf einem angegebenen Sekundenintervall. Sie berechnet einen einfachen gleitenden Durchschnitt über aggregierte Kerzen und handelt, wenn der Preis die Durchschnittslinie kreuzt.

## Parameter

- `SecondsTimeframe` – Sekundenintervall für die Kerzen-Aggregation.
- `AverageLength` – Periode für den einfachen gleitenden Durchschnitt.
