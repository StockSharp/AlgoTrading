# Estratégia MTF Seconds Values JD
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia demonstra o tratamento de velas personalizadas multi-período baseadas em um intervalo de segundos especificado. Calcula uma média móvel simples sobre velas agregadas e opera quando o preço cruza a média.

## Parâmetros

- `SecondsTimeframe` – intervalo em segundos para agregação de velas.
- `AverageLength` – período para a média móvel simples.
