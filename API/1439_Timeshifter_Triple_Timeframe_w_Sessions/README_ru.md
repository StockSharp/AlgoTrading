# Стратегия Timeshifter Triple Timeframe
[English](README.md) | [中文](README_cn.md)

Стратегия, работающая на трёх таймфреймах с дополнительным подтверждением ADX и фильтром по торговым сессиям.

Тестирование показывает среднегодичную доходность около 37%. Наилучшие результаты достигаются на валютном рынке.

Система ориентируется на тренд старшего таймфрейма, вход выполняется при пробое на среднем таймфрейме, а выход — при развороте на младшем. Торговлю можно ограничить сессиями Лондона, Нью‑Йорка и Токио. Фильтр ADX позволяет убедиться в наличии достаточного импульса.

## Подробности

- **Условия входа**:
  - **Лонг**: Цена старшего таймфрейма выше своей SMA, а цена среднего таймфрейма пересекает свою SMA снизу вверх.
  - **Шорт**: Цена старшего таймфрейма ниже своей SMA, а цена среднего таймфрейма пересекает свою SMA сверху вниз.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - **Лонг**: Цена младшего таймфрейма пересекает свою SMA сверху вниз.
  - **Шорт**: Цена младшего таймфрейма пересекает свою SMA снизу вверх.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `HigherMaLength` = 50
  - `MediumMaLength` = 20
  - `LowerMaLength` = 10
  - `AdxLength` = 14
  - `AdxThreshold` = 25
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA, ADX
  - Стопы: Нет
  - Сложность: Сложная
  - Таймфрейм: Мульти‑таймфрейм
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
