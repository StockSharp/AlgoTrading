# Timeshifterトリプル時間軸戦略（セッションフィルター付き）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

オプションのADX確認とセッションフィルターを備えた3つの時間軸でトレードする戦略。

テストでは年平均リターン約37%を示しています。外国為替市場で最も優れたパフォーマンスを発揮します。

このシステムは上位時間軸のトレンドに沿い、中間時間軸のブレイクアウトでエントリーし、下位時間軸のリバーサルでエグジットします。ロンドン、ニューヨーク、東京セッションにトレードを限定することもできます。ADXフィルターで十分なモメンタムを確保できます。

## 詳細

- **エントリー条件**:
  - **ロング**: 上位時間軸の終値がそのSMAを上回り、中間時間軸の価格がそのSMAを上抜けるとき。
  - **ショート**: 上位時間軸の終値がそのSMAを下回り、中間時間軸の価格がそのSMAを下抜けるとき。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: 下位時間軸の価格がそのSMAを下抜けるとき。
  - **ショート**: 下位時間軸の価格がそのSMAを上抜けるとき。
- **ストップ**: なし。
- **デフォルト値**:
  - `HigherMaLength` = 50
  - `MediumMaLength` = 20
  - `LowerMaLength` = 10
  - `AdxLength` = 14
  - `AdxThreshold` = 25
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: SMA, ADX
  - ストップ: なし
  - 複雑さ: 複雑
  - 時間軸: 複数
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
