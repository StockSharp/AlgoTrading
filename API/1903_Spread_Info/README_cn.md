# Spread Info 策略
[English](README.md) | [Русский](README_ru.md)

该策略展示如何使用 StockSharp 的高级 API 获取市场点差信息。策略订阅订单簿更新，并记录所选证券的最佳卖价与最佳买价之间的差值。

策略不执行任何交易，仅用于监控与教育目的。

## 细节
- **入场条件**：无（仅信息）。
- **多/空**：无。
- **出场条件**：无。
- **止损**：无。
- **默认值**：无。
- **指标**：无。
- **过滤器**：无。
- **复杂度**：初级。
- **时间框架**：日内（取决于数据源）。
- **风险等级**：无（无持仓）。
