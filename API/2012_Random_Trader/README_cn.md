# 随机交易策略
[Русский](README_ru.md) | [English](README.md)

在没有持仓时随机选择做多或做空，并为每笔交易设置固定的止盈和止损（以价格单位表示）。

## 详情

- **入场条件**：无持仓且随机选择
- **多/空**：双向
- **离场条件**：价格达到止盈或止损
- **止损**：有
- **默认值**：
  - `Volume` = 1
  - `TakeProfit` = 10
  - `StopLoss` = 10
- **筛选**：
  - 类别：其他
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：基础
  - 时间框架：逐笔
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：高
