# Zufalls-Trader-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Eröffnet zufällig eine Long- oder Short-Position, wenn keine Position offen ist. Jeder Trade verwendet feste Take-Profit- und Stop-Loss-Werte in Preiseinheiten.

## Details

- **Einstiegskriterien**: keine Position und zufällige Wahl
- **Long/Short**: Beide
- **Ausstiegskriterien**: Preis erreicht Take-Profit oder Stop-Loss
- **Stops**: Ja
- **Standardwerte**:
  - `Volume` = 1
  - `TakeProfit` = 10
  - `StopLoss` = 10
- **Filter**:
  - Kategorie: Sonstige
  - Richtung: Beide
  - Indikatoren: Keine
  - Stops: Ja
  - Komplexität: Grundlegend
  - Zeitrahmen: Tick
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Hoch
