# Случайная стратегия трейдера
[English](README.md) | [中文](README_cn.md)

Стратегия случайно открывает длинную или короткую позицию при отсутствии открытых позиций. Для каждой сделки задаются фиксированные уровни тейк-профита и стоп-лосса в единицах цены.

## Детали

- **Критерий входа**: нет позиции и случайный выбор
- **Длин/Корот**: обе стороны
- **Критерий выхода**: достижение тейк-профита или стоп-лосса
- **Стопы**: да
- **Значения по умолчанию**:
  - `Volume` = 1
  - `TakeProfit` = 10
  - `StopLoss` = 10
- **Фильтры**:
  - Категория: Другое
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Тиковый
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
