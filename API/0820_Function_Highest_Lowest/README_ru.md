# Функция высоты и минимума
[English](README.md) | [中文](README_cn.md)

Этот пример демонстрирует пользовательские функции поиска максимума и минимума с чередующимися периодами. Сделки не выполняются.

## Детали

- **Условия входа**: Нет
- **Направление**: Нет
- **Условия выхода**: Нет
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `LengthA` = 10
  - `LengthB` = 20
  - `Тип свечей` = 5 минут

- **Фильтры**:
  - Категория: Демонстрация индикаторов
  - Направление: Нет
  - Индикаторы: Highest, Lowest
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
