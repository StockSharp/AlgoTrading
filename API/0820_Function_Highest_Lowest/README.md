# Function Highest Lowest
[Русский](README_ru.md) | [中文](README_cn.md)

This example demonstrates custom highest and lowest functions with alternating lookback periods. No trades are generated.

## Details

- **Entry Criteria**: None
- **Long/Short**: None
- **Exit Criteria**: None
- **Stops**: No
- **Default Values**:

  - `LengthB` = 20
  - `Candle Type` = 5-minute

- **Filters**:

  - Stops: No

  - Seasonality: No
  - Neural networks: No
  - Divergence: No
