# 最高最低函数示例
[English](README.md) | [Русский](README_ru.md)

该示例演示具有交替周期的自定义最高和最低函数，不进行交易。

## 细节

- **入场条件**：无
- **方向**：无
- **出场条件**：无
- **止损**：无
- **默认参数**：
  - `LengthA` = 10
  - `LengthB` = 20
  - `K线类型` = 5 分钟

- **过滤器**：
  - 类别：指标演示
  - 方向：无
  - 指标：Highest, Lowest
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险水平：低
