## 詳細

- **エントリー条件**: なし
- **ロング/ショート**: なし
- **エグジット条件**: なし
- **ストップ**: いいえ
- **デフォルト値**:
  - `LengthA` = 10
  - `LengthB` = 20
  - `Candle Type` = 5-minute

- **フィルター**:
  - カテゴリ: インジケーターデモ
  - 方向: なし
  - インジケーター: Highest, Lowest
  - Stops: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
