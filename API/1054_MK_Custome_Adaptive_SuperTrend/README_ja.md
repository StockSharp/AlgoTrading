# MK Custome Adaptive SuperTrend 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ATRボラティリティを3つのレベルにクラスタリングするアダプティブSuperTrend。
トレンドが上向きに反転したときにロング、下向きに反転したときにショートを取引します。
ストップはSuperTrend線を使用し、オプションのパーセントテイクプロフィットとストップロスも利用できます。

- **ロング**: 方向が上昇トレンドに転換。
- **ショート**: 方向が下降トレンドに転換。
- **エグジット**: 逆シグナル、SuperTrendブレイク、またはパーセントストップ。

- インジケーター: SuperTrend、ATR。
- ストップ: SuperTrend、パーセントストップロス。
