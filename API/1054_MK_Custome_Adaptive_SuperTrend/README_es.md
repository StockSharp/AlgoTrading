# Estrategia MK Custome Adaptive SuperTrend
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

SuperTrend adaptativo que agrupa la volatilidad ATR en tres niveles.
Las operaciones largas ocurren cuando la tendencia gira hacia arriba, las cortas cuando gira hacia abajo.
Los stops usan la línea SuperTrend con take-profit y stop-loss porcentuales opcionales.

- **Largo**: La dirección cambia a tendencia alcista.
- **Corto**: La dirección cambia a tendencia bajista.
- **Salida**: Señal opuesta, ruptura de SuperTrend o stop porcentual.

- Indicadores: SuperTrend, ATR.
- Stops: SuperTrend, stop-loss porcentual.
