# Estratégia MK Custome Adaptive SuperTrend
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

SuperTrend adaptativo que agrupa a volatilidade ATR em três níveis.
Operações compradas ocorrem quando a tendência vira para cima, vendidas quando vira para baixo.
Os stops usam a linha SuperTrend com take-profit e stop-loss percentuais opcionais.

- **Comprado**: A direção muda para tendência de alta.
- **Vendido**: A direção muda para tendência de baixa.
- **Saída**: Sinal oposto, ruptura do SuperTrend ou stop percentual.

- Indicadores: SuperTrend, ATR.
- Stops: SuperTrend, stop-loss percentual.
