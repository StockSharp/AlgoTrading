# Portfolio Tracker v2
[English](README.md) | [Русский](README_ru.md)

该策略跟踪最多十个仓位和现金余额，根据所配置品种最新 K 线收盘价计算投资组合价值及盈亏。策略仅记录信息，不会发送任何订单。

## 细节

- **入场条件**：无（仅跟踪）
- **多空方向**：无
- **出场条件**：无
- **止损**：无
- **默认值**：
  - `CandleType` = TimeSpan.FromMinutes(1)
  - `UseCash` = true
  - `Cash` = 10000
  - 十组仓位参数（启用、品种、数量、成本）
- **筛选**：
  - 分类：工具
  - 方向：无
  - 指标：无
  - 止损：否
  - 复杂度：中等
  - 时间框：盘中 (1m)
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：低
