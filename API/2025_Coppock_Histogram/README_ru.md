# Стратегия Coppock Histogram
[English](README.md) | [中文](README_cn.md)

Стратегия торгует развороты индикатора Coppock Histogram. Индикатор суммирует два значения Rate of Change и сглаживает их скользящей средней. Когда импульс разворачивается вверх, стратегия открывает длинные позиции и закрывает короткие. Разворот вниз закрывает лонги и открывает шорты. Сигналы оцениваются только по завершённым свечам.

## Подробности

- **Условия входа**: наклон Coppock Histogram вверх для покупки или вниз для продажи.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: противоположный сигнал закрывает позицию.
- **Стопы**: стоп-лосс и тейк-профит не используются.
- **Значения по умолчанию**:
  - `Roc1Period` = 14
  - `Roc2Period` = 11
  - `SmoothPeriod` = 3
  - `Volume` = 1m
  - `CandleType` = TimeSpan.FromHours(8)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: RateOfChange, SimpleMovingAverage
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: 8 часов
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
