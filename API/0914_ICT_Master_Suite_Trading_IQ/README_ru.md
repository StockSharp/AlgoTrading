# Стратегия ICT Master Suite Trading IQ
[English](README.md) | [中文](README_cn.md)

Стратегия ICT Master Suite торгует пробои дневных максимумов и минимумов. При закрытии цены выше максимума сессии открывается длинная позиция, при закрытии ниже минимума — короткая. Позиции сопровождаются трейлинг-стопом на основе ATR.

## Подробности

- **Условия входа**:
  - Цена закрывается выше текущего максимума сессии (лонг).
  - Цена закрывается ниже текущего минимума сессии (шорт).
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Трейлинг-стоп по ATR.
- **Стопы**: Трейлинг-стоп по ATR.
- **Параметры по умолчанию**:
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 1.5
  - `AllowLong` = true
  - `AllowShort` = true
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: ATR
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
