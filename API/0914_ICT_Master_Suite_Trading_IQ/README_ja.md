# ICT Master Suite Trading IQ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ICT Master Suite戦略は、日次セッションの高値と安値のブレイクアウトを取引します。価格がセッション高値を上回って終値を付けた場合にロングポジションを建て、セッション安値を下回って終値を付けた場合にショートポジションを建てます。ポジションはATRベースのトレーリングストップで管理されます。

## 詳細

- **エントリー条件**:
  - 価格が現在のセッション高値を上回って終値を付けた場合（ロング）。
  - 価格が現在のセッション安値を下回って終値を付けた場合（ショート）。
- **ロング/ショート**: ロングおよびショート。
- **エグジット条件**:
  - ATRベースのトレーリングストップ。
- **ストップ**: ATRトレーリングストップ。
- **デフォルト値**:
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 1.5
  - `AllowLong` = true
  - `AllowShort` = true
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: ATR
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
