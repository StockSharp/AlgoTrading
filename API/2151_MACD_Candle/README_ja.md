# MACD ローソク足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMetaTraderのエキスパート「Exp_MACDCandle」を再現したものです。MACDベースのローソク足インジケーターの色出力を、StockSharpの高レベルAPIを使ってトレードシグナルに変換します。

## コンセプト

MACD Candleインジケーターは、始値と終値で計算されたMACD値から合成ローソク足を構築します。終値のMACDが始値のMACDより上にある場合、そのローソク足は強気（色2）とみなされます。逆の場合は弱気のローソク足（色0）になります。両方の値が等しい場合はニュートラル（色1）となります。

戦略は、非強気のローソク足の後に強気のローソク足が現れるとロングポジションを開き、非弱気のローソク足の後に弱気のローソク足が現れるとショートポジションを開きます。既存のポジションは新しい方向に反転されます。

## パラメーター

- `FastLength` – MACDの高速EMA期間（デフォルト12）。
- `SlowLength` – MACDの低速EMA期間（デフォルト26）。
- `SignalLength` – MACDのシグナルライン期間（デフォルト9）。
- `CandleType` – 計算に使用するローソク足タイプ、デフォルトは4時間足の`TimeFrameCandle`。

すべてのパラメーターは設定可能で最適化をサポートします。

## エントリー・エグジット条件

- **ロングエントリー**: 前のローソク足が非強気であった状態で、終値のMACDが始値のMACDを上回る。
- **ショートエントリー**: 前のローソク足が非弱気であった状態で、始値のMACDが終値のMACDを上回る。
- **エグジット**: 反対のシグナルが発生すると現在のポジションを閉じる。明示的なストップロスやテイクプロフィットは適用されない。

## 注意事項

- 戦略は成行注文（`BuyMarket`および`SellMarket`）を使用します。
- ノイズを避けるため、シグナルは確定したローソク足のみで評価されます。
- この例は教育目的を意図しており、リスク管理は含まれていません。
