# Стратегия Aroon WPR Crossover
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия, объединяющая пересечения линий Aroon с фильтрами осциллятора Williams %R. Покупка открывается, когда линия Aroon Up пересекает сверху Aroon Down и Williams %R показывает зону перепроданности. Продажа выполняется при обратном пересечении и перекупленности по Williams %R. Открытые позиции закрываются при обратном сигнале Williams %R или по необязательным стоп-лоссу и тейк-профиту, задаваемым в шагах цены.

## Подробности

- **Критерии входа**:
  - Лонг: Aroon Up пересекает Aroon Down вверх и Williams %R < `-(100 - OpenWprLevel)`
  - Шорт: Aroon Down пересекает Aroon Up вверх и Williams %R > `-OpenWprLevel`
- **Направление торговли**: Лонг и шорт
- **Критерии выхода**:
  - Williams %R выходит из зон перепроданности/перекупленности, определённых параметром `CloseWprLevel`
  - Необязательные тейк-профит и стоп-лосс в шагах цены
- **Стопы**: Необязательные фиксированные уровни в шагах цены
- **Значения по умолчанию**:
  - `AroonPeriod` = 14
  - `WprPeriod` = 35
  - `OpenWprLevel` = 20
  - `CloseWprLevel` = 10
  - `TakeProfitSteps` = 0m (отключено)
  - `StopLossSteps` = 0m (отключено)
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба направления
  - Индикаторы: Aroon, Williams %R
  - Стопы: Необязательные
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
