# Стратегия Gaussian Anomaly Derivative
[English](README.md) | [中文](README_cn.md)

Использует скользящую среднюю ценовой аномалии `1 - (high + low) / (2 * close)` и её сглаженную производную.
Покупает, когда производная превышает положительный порог, и продаёт, когда опускается ниже отрицательного порога.

## Детали

- **Критерий входа**: аномалия или её производная пересекает порог
- **Длин/Шорт**: настраивается
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `UseSma` = true
  - `MaPeriod` = 3
  - `DerivativeMaPeriod` = 2
  - `ThresholdCoeff` = 1.0
  - `DerivativeThresholdCoeff` = 1.0
  - `StartBarCount` = 600
- **Фильтры**:
  - Категория: Тренд
  - Направление: обе стороны
  - Индикаторы: SMA, EMA
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
