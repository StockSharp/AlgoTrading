# Стратегия Bollinger Bands с DEMA
[English](README.md) | [中文](README_cn.md)

Эта стратегия сочетает полосы Боллинджера, рассчитанные на 30-минутных свечах, с двойной экспоненциальной скользящей средней (DEMA) по дневным данным для торговли пробоев с подтверждением тренда.

Длинная позиция открывается, когда бычья свеча пересекает нижнюю полосу снизу вверх, а DEMA растет, подтверждая восходящий импульс. Короткая позиция открывается, когда медвежья свеча пересекает верхнюю полосу сверху вниз, а DEMA падает. Позиции закрываются, когда свеча противоположного цвета пересекает внешнюю полосу против направления сделки.

## Детали

- **Условия входа**:
  - **Long**: Свеча закрывается выше нижней полосы и открывается ниже нее И DEMA растет три дня подряд.
  - **Short**: Свеча закрывается ниже верхней полосы и открывается выше нее И DEMA падает три дня подряд.
- **Long/Short**: Обе стороны.
- **Условия выхода**:
  - **Long**: Медвежья свеча закрывается ниже верхней полосы после открытия выше нее.
  - **Short**: Бычья свеча закрывается выше нижней полосы после открытия ниже нее.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `BollingerPeriod` = 20
  - `DemaPeriod` = 20
  - `Deviation` = 2
  - `CandleType` = интервал 30 минут
- **Фильтры**:
  - Категория: Средневозвратная
  - Направление: Оба
  - Индикаторы: Полосы Боллинджера, DEMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной с дневным фильтром тренда
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
