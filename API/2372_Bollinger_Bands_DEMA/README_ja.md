# Bollinger BandsとDEMAを使った戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、30分足で計算されたBollinger Bandsと日足データの二重指数移動平均（DEMA）を組み合わせて、トレンド確認付きのブレイクアウトを取引します。

ロングセットアップは、強気の足が下限バンドを上抜けし、DEMAが上昇して上方モメンタムを確認したときに発生します。ショートセットアップは、弱気の足が上限バンドを下抜けし、DEMAが下降しているときに発生します。ポジションは、反対色の足がトレードに逆らって外側のバンドを突き抜けたときにクローズされます。

## 詳細

- **エントリー条件**:
  - **ロング**: 足が下限バンドより上でクローズし、下限バンドより下でオープンする、かつ日足DEMAが3日連続で上昇している。
  - **ショート**: 足が上限バンドより下でクローズし、上限バンドより上でオープンする、かつ日足DEMAが3日連続で下降している。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: 弱気の足が上限バンドより上でオープンした後、上限バンドより下でクローズする。
  - **ショート**: 強気の足が下限バンドより下でオープンした後、下限バンドより上でクローズする。
- **ストップ**: なし。
- **デフォルト値**:
  - `BollingerPeriod` = 20
  - `DemaPeriod` = 20
  - `Deviation` = 2
  - `CandleType` = 30分時間軸
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: Bollinger Bands, DEMA
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: イントラデイ（日足トレンドフィルター付き）
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
