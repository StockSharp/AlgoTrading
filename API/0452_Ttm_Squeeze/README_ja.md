# TTM Squeeze戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TTM Squeeze戦略は、Bollinger BandsがKeltner Channels内に収縮する価格圧縮の期間を
探します。この「スクイーズ」は潜在的なボラティリティ拡大を示します。スクイーズの間、
戦略は方向を測るために線形回帰モメンタムオシレーターとRSIを監視します。スクイーズが
解放されてモメンタムが転換すると、動きの方向にポジションが取られます。

この手法は、静かなレンジからの爆発的なブレイクアウトを探します。取引はロングのセット
アップにはRSIが30を超えた状態でモメンタムがゼロ以下から上昇することを要求し、ショート
のセットアップにはRSIが70を下回った状態でモメンタムがプラス領域から下落することが必要
なようにフィルタリングされます。オプションのテイクプロフィットパラメーターは、事前定義
された利益で取引を自動的に閉じることができます。

## 詳細

- **エントリー条件**:
  - スクイーズオフ（Bollinger BandsがKeltner Channelsの外側）。
  - **ロング**: モメンタム < 0 かつ上昇中、RSI > 30。
  - **ショート**: モメンタム > 0 かつ下降中、RSI < 70。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 反対のシグナルまたは有効な場合はテイクプロフィット。
- **ストップ**: デフォルトはなし、オプションのテイクプロフィット。
- **デフォルト値**:
  - `SqueezeLength` = 20
  - `RsiLength` = 14
  - `UseTP` = False
  - `TpPercent` = 1.2
- **フィルター**:
  - カテゴリ: ボラティリティブレイクアウト
  - 方向: 両方
  - インジケーター: Bollinger Bands、Keltner Channels、RSI、線形回帰
  - ストップ: オプション
  - 複雑さ: 中
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
