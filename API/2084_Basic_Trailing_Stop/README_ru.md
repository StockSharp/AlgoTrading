# Базовый трейлинг-стоп
[English](README.md) | [中文](README_cn.md)

Стратегия Basic Trailing Stop сочетает фильтры Commodity Channel Index (CCI) и Relative Strength Index (RSI) с простым трейлинг-стопом. Когда оба индикатора показывают перепроданность или перекупленность, стратегия открывает рыночную позицию и сразу ставит трейлинг-стоп в пунктах. По мере движения цены в нужную сторону уровень стопа следует за трендом, фиксируя прибыль.

Тестирование показало среднегодовую доходность около 32%. Лучшие результаты достигаются на рынке форекс.

Поскольку уровень стопа постоянно следует за ценой, риск автоматически сокращается при развитии тренда. Выходы выполняются только при срабатывании трейлинг-стопа. Система держит одну позицию и работает в обе стороны.

## Детали

- **Условия входа**:
  - **Лонг**: `CCI` между -150 и -100 и `RSI` между 0 и 30.
  - **Шорт**: `CCI` между 100 и 250 и `RSI` между 70 и 100.
- **Направление**: Лонг и шорт.
- **Условия выхода**: Срабатывание трейлинг-стопа.
- **Стопы**: Только трейлинг-стоп.
- **Значения по умолчанию**:
  - `StopLossPips` = 20
  - `CciPeriod` = 14
  - `RsiPeriod` = 14
  - `CandleType` = `TimeSpan.FromMinutes(1)`
- **Фильтры**:
  - Категория: Моментум
  - Направление: Обе стороны
  - Индикаторы: CCI, RSI
  - Стопы: Да
  - Сложность: Новичок
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
