# Ultimate Trading Bot
[English](README.md) | [中文](README_cn.md)

Стратегия только для длинных позиций, комбинирующая пересечения RSI, скользящей средней, MACD и стохастика для определения входа и выхода.

## Подробности

- **Условия входа**: RSI пересекает уровень перепроданности вверх, цена выше MA, MACD и стохастик пересекают вверх.
- **Направление**: Только лонг.
- **Условия выхода**: Обратные условия пересечения.
- **Стопы**: Нет явных стопов.
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `RsiOverbought` = 70
  - `RsiOversold` = 30
  - `MaLength` = 50
  - `StochLength` = 14
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
- **Фильтры**:
  - Категория: Моментум
  - Направление: Лонг
  - Индикаторы: RSI, MA, MACD, Стохастик
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Средний
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
