# 暗号資産リバランシング・プレミアム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

暗号資産リバランシング・プレミアム戦略は、ビットコインとイーサリアムの均等ウェイトポートフォリオを維持します。毎週バスケットをリバランシングすることで、2つの資産間のボラティリティから生まれるプレミアムを獲得しようとします。

戦略は時間足ローソク足を監視し、毎週月曜日の最初の1時間にリバランシングを実行します。必要な調整額がユーザー定義のUSDしきい値を下回る場合、取引はスキップされます。

## 詳細

- **ユニバース**: ビットコインとイーサリアムのシンボル。
- **シグナル**: BTC と ETH を 50/50 の比率に維持する。
- **リバランス**: 毎週月曜日 00:00 UTC。
- **ポジショニング**: ロングのみ、均等ウェイト。
- **パラメーター**:
  - `BTC` – ビットコインの有価証券。
  - `ETH` – イーサリアムの有価証券。
  - `MinTradeUsd` – 最低取引価値（USD）。
  - `CandleType` – ローソク足の時間軸（デフォルト: 1時間）。
- **注意**: 実装は簡略化されており、手数料や資金調達コストは含まれていません。
