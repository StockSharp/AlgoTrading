# MACDクロス AUDUSD D1
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

日足の時間軸でMACDラインのクロスオーバーを使用してAUDUSDを取引する戦略です。

MACDメインラインがシグナルラインを上に抜けるとロングポジションを建て、下に抜けるとショートポジションを建てます。取引はサーバー時間の06:00から14:00の間のみ許可され、同時に開けるポジションは1つだけです。各トレードにはデフォルトで40pipsのストップロスと3倍のテイクプロフィットが設定されます。

## 詳細

- **エントリー条件**: MACDメインラインがシグナルラインをクロス。
- **ロング/ショート**: 両方向。
- **エグジット条件**: ストップロスまたはテイクプロフィット。
- **ストップ**: はい。
- **デフォルト値**:
  - `Volume` = 0.1
  - `StopLossPips` = 40
  - `RewardRatio` = 3
  - `CandleType` = TimeSpan.FromDays(1)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: MACD
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
