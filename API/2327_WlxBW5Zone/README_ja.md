# Wlx BW5ゾーン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ビル・ウィリアムズのAwesome Oscillator (AO)とAccelerator Oscillator (AC)を使用して強いモメンタムシーケンスを識別します。両オシレーターが5本連続で上昇（下降）すると買い（売り）シグナルが発生します。システムはそれに応じてポジションを反転またはオープンします。

## 詳細

- **エントリー条件**:
  - **ロング**: `AO` と `AC` が5本連続で上昇。
  - **ショート**: `AO` と `AC` が5本連続で下落。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対のシグナルでポジションを反転。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `Timeframe` = 4時間。
  - `Direct` = true。
  - `SignalBar` = 1。
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: いいえ
  - 複雑さ: 中程度
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
