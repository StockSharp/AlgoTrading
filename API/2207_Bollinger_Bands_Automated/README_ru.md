# Автоматизированная стратегия Bollinger Bands
[English](README.md) | [中文](README_cn.md)

Стратегия размещает отложенные ордера Buy Limit на нижней полосе Боллинджера и Sell Limit на верхней. Позиции закрываются при касании средней полосы. Все отложенные ордера обновляются в начале каждой свечи.

## Детали

- **Условия входа**:
  - Long: покупка лимитным ордером на нижней полосе
  - Short: продажа лимитным ордером на верхней полосе
- **Направление**: Оба
- **Условия выхода**:
  - Long: цена пересекает среднюю полосу сверху
  - Short: цена пересекает среднюю полосу снизу
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `BbPeriod` = 20
  - `BbDeviation` = 2m
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Возврат к среднему
  - Направление: Оба
  - Индикаторы: Bollinger Bands
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
