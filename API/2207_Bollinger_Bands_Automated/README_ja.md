# ボリンジャーバンド自動売買戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ボリンジャーバンドの下限バンドに買い指値注文を、上限バンドに売り指値注文を配置する戦略です。価格が中央バンドに触れるとポジションを決済します。未決注文は各ローソク足の開始時に更新されます。

## 詳細

- **エントリー条件**:
  - ロング: ボリンジャーバンド下限での買い指値
  - ショート: ボリンジャーバンド上限での売り指値
- **ロング/ショート**: 両方
- **エグジット条件**:
  - ロング: 価格がボリンジャーバンド中央を上抜け
  - ショート: 価格がボリンジャーバンド中央を下抜け
- **ストップ**: なし
- **デフォルト値**:
  - `BbPeriod` = 20
  - `BbDeviation` = 2m
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: Bollinger Bands
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
