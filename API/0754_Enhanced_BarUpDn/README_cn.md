# 增强型 BarUpDn 策略
[English](README.md) | [Русский](README_ru.md)

该策略结合布林带与趋势确认，寻找向上或向下跳空的 K 线。上涨趋势中出现向上跳空的看涨 K 线时做多；下跌趋势中出现向下跳空的看跌 K 线时做空。退出通过基于 ATR 的止损和止盈实现。

## 详情

- **入场条件**：
  - 多头：看涨 K 线向上跳空，收盘价高于趋势均线且高于布林带下轨。
  - 空头：看跌 K 线向下跳空，收盘价低于趋势均线且低于布林带上轨。
- **多空方向**：双向。
- **离场条件**：
  - 价格触及基于 ATR 的止损或止盈（1.5× ATR）。
- **止损**：使用 ATR 计算的止损和止盈。
- **默认参数**：
  - `BbLength` = 20
  - `BbMultiplier` = 2
  - `MaLength` = 50
  - `AtrLength` = 14
  - `AtrMultiplierSl` = 2
  - `AtrMultiplierTp` = 3
- **筛选器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：布林带、SMA、ATR
  - 止损：有
  - 复杂度：低
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
