# Улучшенная стратегия BarUpDn
[English](README.md) | [中文](README_cn.md)

Стратегия ищет бычьи и медвежьи свечи в сочетании с полосами Боллинджера и подтверждением тренда. Вход в лонг выполняется при бычьем разрыве на восходящем тренде, в шорт — при медвежьем разрыве на нисходящем тренде. Выход осуществляется по уровням стоп-лосса и тейк-профита на основе ATR.

## Подробности

- **Условия входа**:
  - Лонг: бычья свеча с разрывом вверх, закрытие выше трендовой MA и выше нижней полосы Боллинджера.
  - Шорт: медвежья свеча с разрывом вниз, закрытие ниже трендовой MA и ниже верхней полосы Боллинджера.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Цена достигает ATR-стопа или ATR-профита (1.5× ATR).
- **Стопы**: Используются уровни стоп-лосса и тейк-профита по ATR.
- **Параметры по умолчанию**:
  - `BbLength` = 20
  - `BbMultiplier` = 2
  - `MaLength` = 50
  - `AtrLength` = 14
  - `AtrMultiplierSl` = 2
  - `AtrMultiplierTp` = 3
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Полосы Боллинджера, SMA, ATR
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
