# Стратегия EMA MACD Signal Trend
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда быстрая EMA выше медленной, а линия сигнала MACD растет. Короткая позиция открывается, когда быстрая EMA ниже медленной, а линия сигнала падает. Опционально можно использовать стоп-лосс, тейк-профит и трейлинг-стоп.

## Детали

- **Условия входа**:
  - Быстрая EMA > Медленной EMA и линия сигнала MACD растет → Покупка.
  - Быстрая EMA < Медленной EMA и линия сигнала MACD падает → Продажа.
- **Условия выхода**:
  - Противоположный сигнал закрывает позицию.
- **Индикаторы**: EMA, MACD signal.
- **Тип**: Следование тренду.
- **Таймфрейм**: 5 минут (по умолчанию).
