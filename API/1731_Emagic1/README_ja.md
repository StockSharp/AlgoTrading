# EMA MACDシグナルトレンド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、速いEMAが遅いEMAより上にあり、MACDシグナルラインが上昇しているときにロングポジションを建てます。速いEMAが遅いEMAより下にあり、シグナルラインが下落しているときにショートを建てます。ストップロス、テイクプロフィット、トレーリングストップはオプションです。

## 詳細

- **エントリー条件**:
  - 速いEMA > 遅いEMA かつ MACDシグナルが上昇中 → 買い。
  - 速いEMA < 遅いEMA かつ MACDシグナルが下降中 → 売り。
- **エグジット条件**:
  - 反対のエントリーシグナルでポジションを決済。
- **インジケーター**: EMA, MACD signal。
- **タイプ**: トレンドフォロー。
- **時間軸**: 5分（デフォルト）。
