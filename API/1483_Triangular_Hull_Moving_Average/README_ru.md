# Треугольная Hull Moving Average
[English](README.md) | [中文](README_cn.md)

Стратегия основана на пересечении скользящей средней Халла с её значением двух баров назад. В зависимости от выбранного режима открываются длинные и/или короткие позиции.

## Детали
- **Критерии входа**: пересечение HMA с задержкой 2 бара.
- **Длинные/короткие**: по выбору.
- **Критерии выхода**: противоположный сигнал или фильтр направления.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Length` = 40
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `EntryMode` = EntryDirection.LongAndShort
- **Фильтры**:
  - Категория: Тренд
  - Направление: Настраиваемое
  - Индикаторы: MA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
