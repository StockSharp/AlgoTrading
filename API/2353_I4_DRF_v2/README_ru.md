# Стратегия I4 DRF v2
[English](README.md) | [中文](README_cn.md)

Стратегия I4 DRF v2 использует индикатор i4_DRF_v2, который считает количество растущих и падающих закрытий за выбранный период.
В зависимости от параметра TrendMode работает против тренда (Direct) или по тренду (NotDirect).
Открывает и закрывает позиции при смене знака индикатора и поддерживает стоп-лосс и тейк-профит в шагах цены.

## Детали

- **Критерий входа**: смена знака индикатора согласно TrendMode
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал или стоп-лосс/тейк-профит
- **Стопы**: да
- **Значения по умолчанию**:
  - `Period` = 11
  - `BuyPosOpen` = true
  - `SellPosOpen` = true
  - `BuyPosClose` = true
  - `SellPosClose` = true
  - `TrendMode` = Direct
  - `StopLoss` = 1000
  - `TakeProfit` = 2000
- **Фильтры**:
  - Категория: Тренд
  - Направление: обе стороны
  - Индикаторы: Пользовательский
  - Стопы: да
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
