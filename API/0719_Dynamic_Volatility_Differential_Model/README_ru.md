# Модель динамического дифференциала волатильности
[English](README.md) | [中文](README_zh.md)

Стратегия **Dynamic Volatility Differential Model (DVDM)** сравнивает подразумеваемую и историческую волатильности. Лонг открывается, когда подразумеваемая волатильность превышает реализованную на величину динамического порога стандартного отклонения, и шорт, когда спред опускается ниже отрицательного порога.

Сигналы работают на дневных данных и не используют стопов.

## Подробности
- **Критерии входа**: спред волатильностей выше/ниже динамических порогов стандартного отклонения.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: спред волатильностей пересекает нулевую линию.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Length = 5`
  - `StdevMultiplier = 7.1m`
  - `VolatilitySecurity = "TVC:VIX"`
  - `CandleType = TimeSpan.FromDays(1).TimeFrame()`
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Оба
  - Индикаторы: StandardDeviation
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
