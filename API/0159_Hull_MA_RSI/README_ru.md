# Стратегия Hull MA RSI
[English](README.md) | [中文](README_cn.md)

Реализация стратегии №159 — Hull Moving Average + RSI. Покупать, когда HMA растёт и RSI ниже 30 (перепроданность). Продавать, когда HMA падает и RSI выше 70 (перекупленность).

Hull MA формирует сглаженную линию тренда, а RSI выявляет дивергенции импульса. Сделки совершаются, когда RSI разворачивается на экстремумах, а цена движется по направлению Hull.

Подходит краткосрочным свинг-трейдерам, желающим ранние сигналы. Стопы на основе ATR защищают сделку.

## Подробности

- **Условия входа**:
  - Лонг: `HullMA turning up && RSI < RsiOversold`
  - Шорт: `HullMA turning down && RSI > RsiOverbought`
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Изменение направления Hull MA
- **Стопы**: на основе ATR через `StopLoss`
- **Значения по умолчанию**:
  - `HmaPeriod` = 9
  - `RsiPeriod` = 14
  - `RsiOversold` = 30m
  - `RsiOverbought` = 70m
  - `StopLoss` = new Unit(2, UnitTypes.Absolute)
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Средняя обратная
  - Направление: Оба
  - Индикаторы: Hull MA, Скользящая средняя, RSI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
