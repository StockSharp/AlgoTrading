# Стратегия MACD Bollinger
[English](README.md) | [中文](README_cn.md)

В этой стратегии используются индикаторы MACD и полосы Боллинджера. Длинная позиция открывается, когда MACD выше сигнальной линии, а цена находится ниже нижней полосы Боллинджера — рост на фоне перепроданности. Короткая позиция берётся при MACD ниже сигнала и цене выше верхней полосы, что указывает на перекупленность.

Стратегия предназначена для поиска возможностей в смешанном рынке.

## Детали
- **Условия входа**:
  - **Лонг**: MACD > Signal и цена < нижней полосы Боллинджера (рост при перепроданности)
  - **Шорт**: MACD < Signal и цена > верхней полосы Боллинджера (падение при перекупленности)
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - **Лонг**: Закрытие позиции при возвращении цены к средней полосе
  - **Шорт**: Закрытие позиции при возвращении цены к средней полосе
- **Стопы**: да.
- **Значения по умолчанию**:
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `BollingerPeriod` = 20
  - `BollingerDeviation` = 2.0m
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Смешанная
  - Направление: Оба
  - Индикаторы: MACD Bollinger
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 55%\. Стратегию лучше запускать на фондовом рынке.
