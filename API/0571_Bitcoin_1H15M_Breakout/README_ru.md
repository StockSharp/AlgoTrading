# Стратегия пробоя Bitcoin 1H-15M
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает максимум и минимум предыдущей часовой свечи и входит в позицию, когда 15‑минутная свеча закрывается за пределами этого диапазона. Риск управляется фиксированным буфером стоп‑лосса и тейк‑профитом, вычисляемым через настраиваемое отношение риск/прибыль.

## Детали

- **Условия входа**:
  - Закрытие 15‑минутной свечи выше максимума предыдущего часа → покупка.
  - Закрытие 15‑минутной свечи ниже минимума предыдущего часа → продажа.
- **Длинные/короткие**: обе стороны
- **Условия выхода**:
  - Стоп‑лосс на фиксированном расстоянии.
  - Тейк‑профит = буфер × отношение риск/прибыль.
- **Стопы**: стоп‑лосс и тейк‑профит через модуль защиты.
- **Значения по умолчанию**:
  - Низший таймфрейм = 15 минут.
  - Высший таймфрейм = 1 час.
  - Буфер стоп‑лосса = 50.
  - Отношение риск/прибыль = 2.0.
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: SL & TP
  - Сложность: Низкая
  - Таймфрейм: Короткий
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
