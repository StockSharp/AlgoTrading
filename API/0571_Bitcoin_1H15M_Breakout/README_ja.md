# Bitcoin 1H-15M ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は直前の1時間足の高値と安値を追跡し、15分足のローソク足がその範囲外で引けるとトレードに入ります。リスクは固定のストップロスバッファと、設定可能なリスクリワード比から導出されるテイクプロフィットで管理されます。

## 詳細

- **エントリー条件**:
  - 15分足終値が直前の1時間足高値を上回る → ロングエントリー。
  - 15分足終値が直前の1時間足安値を下回る → ショートエントリー。
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 固定バッファ距離でストップロス。
  - バッファ × リスクリワード比でテイクプロフィット。
- **ストップ**: プロテクションモジュールによるストップロスとテイクプロフィット。
- **デフォルト値**:
  - 下位時間軸 = 15分。
  - 上位時間軸 = 1時間。
  - ストップロスバッファ = 50。
  - リスクリワード比 = 2.0。
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: SL & TP
  - 複雑さ: 低
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
