# ICT インジケーターとペーパートレーディング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はオーダーブロックの高値と安値を記録し、終値が最新のオーダーブロック高値を上抜けたときにロングポジションを建てます。記録されたオーダーブロックの安値が価格を上抜けたときにロングポジションを手仕舞います。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が最新のオーダーブロック高値を上抜ける。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - オーダーブロックの安値が価格を上抜けたときにロングを手仕舞う。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame().
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロングのみ
  - インジケーター: Price action
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
