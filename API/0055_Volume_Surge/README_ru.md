# Всплеск объёма

Стратегия "Всплеск объёма" ищет необычно высокий объём относительно скользящей средней. Когда отношение превышает заданный множитель, это сигнализирует о сильном интересе и возможном продолжении движения цены относительно её средней.

Сделки открываются только при всплеске и закрываются, как только объём возвращается ниже среднего или срабатывает стоп-лосс.

Этот простой подход захватывает импульс, вызванный резким участием.

## Детали

- **Условия входа**: Отношение объёма выше `VolumeSurgeMultiplier`.
- **Длинные/короткие позиции**: Оба направления.
- **Условия выхода**: Объём падает ниже среднего либо стоп.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `MAPeriod` = 20
  - `VolumeAvgPeriod` = 20
  - `VolumeSurgeMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: Volume
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
