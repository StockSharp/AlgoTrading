# 出来高急増 (Volume Surge)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

出来高急増は、移動平均に対して異常に高い出来高を識別します。比率が定義した乗数を超えると、強い関心と価格が移動平均に対してその方向へ継続する可能性を示します。

テストでは年平均リターンが約52%となっています。暗号通貨市場での運用に最も適しています。

取引は急増時のみ開始され、出来高が平均を下回るかストップロスに達したときに決済されます。

このシンプルなアプローチは、突然の参加によって生み出されるモメンタムを捉えます。

## 詳細

- **エントリー条件**: 出来高比率が `VolumeSurgeMultiplier` を超える。
- **ロング/ショート**: 両方。
- **エグジット条件**: 出来高が平均を下回るかストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `MAPeriod` = 20
  - `VolumeAvgPeriod` = 20
  - `VolumeSurgeMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Volume
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
