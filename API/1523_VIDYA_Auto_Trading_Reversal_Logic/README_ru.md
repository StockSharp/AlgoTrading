# Стратегия VIDYA Auto-Trading (Reversal Logic)
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия использует скользящую VIDYA и широкие полосы ATR.
Покупка совершается при пробое верхней полосы, продажа — при пробое нижней.

## Детали

- **Критерий входа**: цена пересекает полосы ATR вокруг VIDYA
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный пробой полосы
- **Стопы**: нет
- **Значения по умолчанию**:
  - `VidyaLength` = 10
  - `VidyaMomentum` = 20
  - `BandDistance` = 2
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: VIDYA, ATR
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
