# 原油が株式を予測する戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は原油と株式リターンの関係を利用します。直近1ヶ月の原油リターンがプラスであれば、株式ETFに投資します。そうでなければ、原油が弱い時期に株式を避けるため、現金または債券ETFに資本を回転させます。

アルゴリズムは日足ローソク足を監視し、毎月最初の取引日にシグナルを確認します。注文は市場価格で提出され、小さな約定を避けるために最低取引サイズを遵守します。

## 詳細

- **ユニバース**: 株式ETF1本、原油商品1本、現金または債券ETF1本。
- **シグナル**: 原油の`Lookback`期間リターンがゼロより大きい場合は株式ETFをロング、そうでなければ現金ETFを保有。
- **リバランス**: 月次、月初。
- **ポジショニング**: 株式または現金のロング、両方同時には持たない。
- **パラメーター**:
  - `Equity` – 対象の株式ETF。
  - `Oil` – シグナル用の原油有価証券。
  - `CashEtf` – 原油リターンがマイナスの時の守備的資産。
  - `Lookback` – 原油リターン計算に使用するローソク足の本数。
  - `CandleType` – ローソク足の時間軸（デフォルト: 1日）。
- **注意**: このサンプルは構造に焦点を当てており、取引コストとスリッページは省略しています。
