# S&P 100 期权到期周策略
[English](README.md) | [Русский](README_ru.md)

该策略在期权到期周的周一买入（该周包含当月第三个周五），并在第三个周五平仓。

## 细节

- **入场条件**：
  - **多头**：在期权到期周的周一做多。
- **多空方向**：仅多头。
- **出场条件**：
  - 在当月第三个周五平多头仓位。
- **止损**：无。
- **默认值**：
  - `CandleType` = TimeSpan.FromDays(1).TimeFrame()。
- **筛选**：
  - 类型: 季节性
  - 方向: 多头
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日线
  - 季节性: 是
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
