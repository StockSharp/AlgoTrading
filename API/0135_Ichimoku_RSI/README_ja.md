# Ichimoku RSI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
Ichimoku RSI は、一目均衡表のクラウドレベルでトレンド方向を定義し、RSI で短期的な押し目を特定します。
トレードはクラウドに沿って行われ、上昇トレンドでは RSI が売られすぎから回復したとき、下降トレンドでは買われすぎから下落したときにエントリーします。

テストでは平均年間リターンは約142%を示しています。株式市場で最も良いパフォーマンスを発揮します。

広いトレンドフィルターとモメンタムオシレーターを組み合わせることで、この戦略は短い小休止の後に強い動きに乗ることを目指します。

より深い修正から保護するため、ストップはクラウドの境界線の外側に設定されます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは反対シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Ichimoku, RSI
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
