# ブレイクイーブン・トレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ストップロスをブレイクイーブンに移動し、その後価格の前進に合わせてトレールする方法を示す戦略。
戦略はロングポジションに入り、2つのフェーズで管理します：
1. 価格が `BreakevenPlus` ポイント上昇した後、ストップはエントリーより `BreakevenStep` ポイント上に移動されます。
2. 価格がストップより `TrailingPlus` ポイント以上の利益で継続する場合、ストップは価格を `TrailingStep` ポイントの距離でトレールします。

ショートポジションを手動で開いた場合、ロジックは対称的に機能します。

## 詳細

- **エントリー条件**: 最初の完了したローソク足でロングポジションを開く。
- **ロング/ショート**: 両方（例ではロングを使用）。
- **エグジット条件**: 価格がトレーリングストップを超える。
- **ストップ**: ブレイクイーブンおよびトレーリングストップ。
- **デフォルト値**:
  - `BreakevenPlus` = 5
  - `BreakevenStep` = 2
  - `TrailingPlus` = 3
  - `TrailingStep` = 1
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **フィルター**:
  - カテゴリ: ストップ管理
  - 方向: 両方
  - インジケーター: なし
  - ストップ: ブレイクイーブン、トレーリング
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: なし
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
