# 線形相関オシレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

線形相関オシレーター戦略は、ローリングウィンドウで価格と時間の相関を測定します。オシレーターがゼロを上抜けするとロング、ゼロを下抜けするとショートになります。

## 詳細

- **エントリー条件**:
  - オシレーターがゼロを上抜け → **ロング**。
  - オシレーターがゼロを下抜け → **ショート**。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 逆方向のゼロ交差。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 14
- **フィルター**:
  - カテゴリ: オシレーター
  - 方向: 両方
  - インジケーター: Linear Correlation
  - ストップ: なし
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
