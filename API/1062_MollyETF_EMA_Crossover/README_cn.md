# Molly ETF EMA Crossover 策略
[English](README.md) | [Русский](README_ru.md)

当快速 EMA 上穿慢速 EMA 时，该策略开多单；当快速 EMA 下穿慢速 EMA 时平仓。策略可以通过参数设置仅在指定日期范围内交易。

## 详情

- **入场条件**：
  - **多头**：快速 EMA 在日期范围内上穿慢速 EMA。
- **多空方向**：仅多头。
- **出场条件**：
  - 快速 EMA 下穿慢速 EMA 或日期范围结束。
- **止损**：无。
- **默认参数**：
  - `Fast EMA` = 10
  - `Slow EMA` = 21
  - `Start Date` = 2018-01-01
  - `End Date` = 2023-09-07
- **过滤器**：
  - 分类：趋势跟随
  - 方向：多头
  - 指标：EMA
  - 止损：无
  - 复杂度：基础
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：低
