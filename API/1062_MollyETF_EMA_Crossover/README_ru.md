# Стратегия Molly ETF EMA Crossover
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда быстрая EMA пересекает медленную снизу вверх, и выходит при обратном пересечении. Дополнительно можно ограничить торговлю выбранным диапазоном дат.

## Детали

- **Условия входа**:
  - **Лонг**: быстрая EMA пересекает медленную сверху в пределах указанного диапазона дат.
- **Лонг/Шорт**: только лонг.
- **Условия выхода**:
  - быстрая EMA пересекает медленную сверху вниз или диапазон дат заканчивается.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Fast EMA` = 10
  - `Slow EMA` = 21
  - `Start Date` = 2018-01-01
  - `End Date` = 2023-09-07
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
