# TMAブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、三角移動平均（TMA）に対するブレイクアウトを活用します。設定可能なローソク足シリーズを監視し、前のローソク足の終値をTMA値にユーザー定義のオフセットを加減した値と比較します。前の終値が`TMA + UpLevel`を上回るとロングポジションを建て、`TMA - DownLevel`を下回るとショートポジションを建てます。逆のシグナルはポジションを反転させます。

## パラメーター

- **TMA Length** – 三角移動平均の計算に使用する期間。
- **Upper Level** – ロングシグナルを検出するためにTMAに加算する価格オフセット。
- **Lower Level** – ショートシグナルを検出するためにTMAから減算する価格オフセット。
- **Candle Type** – 戦略が使用するローソク足の時間軸。

## 動作の仕組み

1. 選択したローソク足シリーズをサブスクライブします。
2. 三角移動平均インジケーターをローソク足にバインドします。
3. 各ローソク足の完成時に：
   - TMAと終値の前回値を保存します。
   - 前の終値が上限または下限を超えたかどうかを確認します。
   - 状況に応じてポジションを建てまたは反転させる成行注文を送信します。
4. ビジュアル分析のためにローソク足、インジケーターライン、自己取引をチャートに描画します。

## 注意事項

この戦略はストップロスやテイクプロフィットの管理なしに成行注文を使用します。教育目的を意図しており、実取引の前に適切なリスク管理を追加して拡張する必要があります。
