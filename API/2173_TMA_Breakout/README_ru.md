# Стратегия TMA Breakout

Стратегия использует прорывы относительно треугольной скользящей средней (TMA). Она анализирует выбранный поток свечей и сравнивает цену закрытия предыдущей свечи со значением TMA с учётом заданных смещений. Если предыдущая свеча закрылась выше `TMA + UpLevel`, открывается длинная позиция. Если ниже `TMA - DownLevel`, открывается короткая позиция. Противоположные сигналы переворачивают позицию.

## Параметры

- **TMA Length** – период расчёта треугольной скользящей средней.
- **Upper Level** – смещение цены вверх от TMA для поиска сигналов на покупку.
- **Lower Level** – смещение цены вниз от TMA для поиска сигналов на продажу.
- **Candle Type** – временной интервал свечей, используемых стратегией.

## Как работает

1. Подписывается на выбранный поток свечей.
2. Привязывает индикатор Triangular Moving Average к свечам.
3. На каждой завершённой свече:
   - Сохраняет предыдущие значения TMA и цены закрытия.
   - Проверяет, пробита ли верхняя или нижняя граница.
   - Открывает или разворачивает позицию рыночными заявками.
4. На графике отображаются свечи, линия индикатора и собственные сделки.

## Примечания

Стратегия предназначена для учебных целей и не содержит управления рисками. Перед реальной торговлей необходимо добавить стоп-лоссы, тейк-профиты и другие элементы защиты.
