# MACFibo戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMACFiboトレードシステムを実装しています。5期間EMAと20期間SMAのクロスオーバーを待ちます。クロス後、アルゴリズムはクロスバーの終値（点A）から直近の極値（点B）までのスウィングを測定し、フィボナッチ拡張レベルを構築します。ポジションはこれらのレベルから導出されたテイクプロフィットとストップロスで成行価格で建てられます。オプションの退出は、速いEMAが反対方向に中間SMAをクロスすると損失中のトレードを閉じます。

## 詳細

- **エントリー条件:**
  - **ロング:** 5 EMAが20 SMAを上抜け。点Bは下降動作が始まって以来の最安値。
  - **ショート:** 5 EMAが20 SMAを下抜け。点Bは上昇動作が始まって以来の最高値。
- **エグジット条件:**
  - フィボナッチ161.8%レベルまたは最小テイクプロフィット距離でテイクプロフィット。
  - フィボナッチ38.2%レベルまたは最大ストップロス距離でストップロス。
  - 5 EMAがポジションに逆らって8 SMAをクロスし、かつトレードが損失中の場合はオプションで決済。
- **フィルター:**
  - 設定された開始時間と終了時間の間のみ取引。
  - 月曜日または金曜日の取引を無効にできます。
- **パラメーター:**
  - `FastLength` – 速いEMAの長さ。
  - `MidLength` – 保護的退出のための中間SMAの長さ。
  - `SlowLength` – トレンド検出のための遅いSMAの長さ。
  - `MinTakeProfit` – 価格単位での最小テイクプロフィット。
  - `MaxStopLoss` – 価格単位での最大ストップロス。
  - `StartHour` / `EndHour` – 許可された取引時間窓。
  - `FridayTrade` / `MondayTrade` – これらの曜日の取引を有効にする。
  - `CloseAtFastMid` – 速い/中間クロスで損失トレードを決済。
  - `CandleType` – 計算用のローソク足の種類。
