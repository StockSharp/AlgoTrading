# Gemini Trend Following System
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия, покупающая откаты к 50-дневной SMA в рамках сильного восходящего тренда, подтверждённого 200-дневной SMA и годовым фильтром Rate of Change.

## Подробности

- **Критерии входа**: цена возвращается выше 50 SMA после недавнего отката в подтверждённом восходящем тренде.
- **Длинные/короткие**: только длинные.
- **Критерии выхода**: пересечение 50 ниже 200 SMA или катастрофический стоп.
- **Стопы**: опциональный катастрофический стоп.
- **Параметры по умолчанию**:
  - `Sma50Length` = 50
  - `Sma200Length` = 200
  - `RocPeriod` = 252
  - `RocMinPercent` = 15m
  - `UseCatastrophicStop` = true
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Длинные
  - Индикаторы: SMA, RateOfChange, Lowest
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
