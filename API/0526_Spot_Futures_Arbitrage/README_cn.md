# 现货期货套利策略

在现货与其期货合约之间进行价差套利。
当期货价格高于现货达到阈值时做多现货/做空期货，反之做空现货/做多期货。
阈值可根据价差的均值和标准差动态调整，仓位在价差回归或持仓时间超过上限时平仓。

## 参数
- **Spot** — 现货品种。
- **Future** — 期货品种。
- **CandleType** — K线周期。
- **MinSpreadPct** — 入场所需的最小价差百分比。
- **LookbackPeriod** — 价差统计的周期长度。
- **AdaptiveThreshold** — 启用动态阈值。
- **MaxHoldHours** — 最大持仓时间（小时）。
