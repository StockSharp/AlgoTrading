# Стратегия MA Parabolic SAR
[English](README.md) | [中文](README_cn.md)

Стратегия MA Parabolic SAR стремится захватывать продолжительные тренды, используя скользящую среднюю для определения направления и Parabolic SAR для точки входа и размещения стопа. Когда оба индикатора совпадают, предполагается, что импульс достаточно сильный, чтобы его поддержать.

Длинная позиция открывается, когда цена выше скользящей средней и точки SAR располагаются под графиком. Короткая позиция берётся, когда цена ниже средней, а точки SAR над ценой, указывая на нисходящее давление. Выход происходит при пересечении цены и SAR в обратную сторону, фиксируя прибыль или ограничивая убыток.

Такой подход подойдёт трейдерам, предпочитающим системное следование тренду с понятными механическими стопами. Parabolic SAR постоянно подстраивается к изменению волатильности, удерживая позицию в согласии с рыночными условиями, а скользящая средняя не позволяет торговать против общего тренда.

## Детали
- **Условия входа**:
  - **Лонг**: Цена > MA и цена > Parabolic SAR
  - **Шорт**: Цена < MA и цена < Parabolic SAR
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - **Лонг**: Выход при падении цены ниже Parabolic SAR
  - **Шорт**: Выход при росте цены выше Parabolic SAR
- **Стопы**: да, динамические по Parabolic SAR и при желании фиксированный стоп.
- **Значения по умолчанию**:
  - `MaPeriod` = 20
  - `SarStep` = 0.02m
  - `SarMaxStep` = 0.2m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `TakeValue` = new Unit(0, UnitTypes.Absolute)
  - `StopValue` = new Unit(2, UnitTypes.Percent)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: MA, Parabolic SAR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
