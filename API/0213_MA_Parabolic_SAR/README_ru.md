# Стратегия MA Parabolic SAR

Стратегия комбинирует простую скользящую среднюю и индикатор Parabolic SAR, чтобы удерживать позиции по тренду и использовать трейлинг‑стоп. Скользящая средняя задаёт направление, а точки SAR подсказывают вход и уровень стопа. При совпадении обоих сигналов предполагается устойчивый импульс.

Длинная позиция открывается, когда цена закрытия выше средней, а точки SAR расположены под ценой. Короткая позиция открывается, когда закрытие ниже средней и точки SAR находятся над ценой. Выход происходит при пересечении SAR в обратную сторону.

Метод подходит сторонникам системного следования за трендом. Parabolic SAR адаптируется к волатильности, а скользящая средняя не позволяет торговать против основного движения.

## Подробности
- **Условия входа**:
  - **Long**: Price > MA && Price > Parabolic SAR
  - **Short**: Price < MA && Price < Parabolic SAR
- **Long/Short**: обе стороны
- **Условия выхода**:
  - **Long**: выход при падении ниже Parabolic SAR
  - **Short**: выход при росте выше Parabolic SAR
- **Стопы**: да, динамические через SAR и при необходимости фиксированные
- **Значения по умолчанию**:
  - `MaPeriod` = 20
  - `SarStep` = 0.02m
  - `SarMaxStep` = 0.2m
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `TakeValue` = new Unit(0, UnitTypes.Absolute)
  - `StopValue` = new Unit(2, UnitTypes.Percent)
- **Фильтры**:
  - Категория: Trend
  - Направление: оба
  - Индикаторы: MA, Parabolic SAR
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: Intraday
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
