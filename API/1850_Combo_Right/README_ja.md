# Combo Right 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MQLスクリプト `combo_right.mq5` から変換された戦略の実装です。
このシステムは基本的なCCIシグナルと、価格差に基づいて動作する3つのシンプルなパーセプトロンを組み合わせています。

## ロジック

1. **基本シグナル** – Commodity Channel Index（CCI）の値。正の値はロングトレードを優先し、負の値はショートトレードを優先します。
2. **パーセプトロン** – 各パーセプトロンはシフトされた終値のセットを参照し、線形重みを適用します。モードパラメーター `Pass` でどのパーセプトロンをアクティブにするかを選択します：
   - `1`: 基本CCIシグナルのみ。
   - `2`: 売りパーセプトロンがCCIを上書きしてショートポジションを開くことができます。
   - `3`: 買いパーセプトロンがCCIを上書きしてロングポジションを開くことができます。
   - `4`: 汎用パーセプトロンが買いと売り両方のパーセプトロンを監督します。

アクティブなパーセプトロンがシグナルを発すると、基本CCI出力を置き換えます。そうでない場合はCCI値を使用します。

## パラメーター

- `TakeProfit1`, `StopLoss1` – 基本CCIシグナルの利益目標と損失目標（ティック単位）。
- `CciPeriod` – CCIインジケーターのルックバック期間。
- 各パーセプトロンの重みと期間（`x12`, `x22`, …, `p4`）。
- `Pass` – 動作モード。
- `Shift` – 価格データに使用するバーインデックス（0は現在、1は前）。
- `Volume` – 取引量。
- `CandleType` – 計算に使用するローソク足の種類。

## インジケーター

- CCI。
