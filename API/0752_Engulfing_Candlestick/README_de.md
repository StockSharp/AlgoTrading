# Engulfing-Candlestick-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt ein ausgewähltes Engulfing-Muster. Wählen Sie entweder bullisches oder bearisches Engulfing und eine Handelsseite, die beim Auftreten des Musters eröffnet wird. Die Position wird für eine festgelegte Anzahl von Bars gehalten, bevor sie geschlossen wird.

## Details

- **Einstiegskriterien**: Ausgewähltes Engulfing-Muster (bullisch oder bearisch).
- **Long/Short**: Konfigurierbarer Long oder Short.
- **Ausstiegskriterien**: Position wird nach der angegebenen Anzahl von Bars geschlossen.
- **Stops**: Nein.
- **Standardwerte**:
  - `CandleType` = 15 Minuten
  - `HoldPeriods` = 17
  - `Pattern` = Bullish
  - `Side` = Long
- **Filter**:
  - Kategorie: Muster
  - Richtung: Beide
  - Indikatoren: Candlestick
  - Stops: Nein
  - Komplexität: Anfänger
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
