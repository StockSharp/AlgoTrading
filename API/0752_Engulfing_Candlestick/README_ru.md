# Стратегия по свечному паттерну Поглощение
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует выбранный паттерн поглощения. Можно выбрать бычье или медвежье поглощение и сторону сделки. Позиция удерживается заданное количество баров, после чего закрывается.

## Детали

- **Условия входа**: Выбранный паттерн поглощения (бычий или медвежий).
- **Направление**: Настраиваемый лонг или шорт.
- **Условия выхода**: Закрытие позиции после заданного числа баров.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `HoldPeriods` = 17
  - `Pattern` = Bullish
  - `Side` = Long
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе стороны
  - Индикаторы: Свечной анализ
  - Стопы: Нет
  - Сложность: Начальная
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
