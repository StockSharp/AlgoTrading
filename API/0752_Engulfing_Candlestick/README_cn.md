# 吞没形态策略
[English](README.md) | [Русский](README_ru.md)

该策略在出现所选吞没形态时入场，可选择看涨或看跌，并可设定交易方向。持仓在固定的条数后自动平仓。

## 细节

- **入场条件**：所选吞没形态（看涨或看跌）。
- **多/空**：可配置做多或做空。
- **退出条件**：持仓达到设定的条数后平仓。
- **止损**：无。
- **默认值**：
  - `CandleType` = 15 分钟
  - `HoldPeriods` = 17
  - `Pattern` = Bullish
  - `Side` = Long
- **过滤条件**：
  - 类别: 形态
  - 方向: 双向
  - 指标: K线形态
  - 止损: 无
  - 复杂度: 入门
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
