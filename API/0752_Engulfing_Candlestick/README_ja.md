# エングルフィング・キャンドルスティック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、選択したエングルフィングパターンで取引を行います。強気または弱気のエングルフィングを選択し、パターンが出現したときにオープンする取引サイドを設定します。ポジションは指定したバー数だけ保持された後、クローズされます。

## 詳細

- **エントリー条件**: 選択したエングルフィングパターン（強気または弱気）。
- **ロング/ショート**: ロングまたはショートを設定可能。
- **エグジット条件**: 指定したバー数後にポジションをクローズ。
- **ストップ**: なし。
- **デフォルト値**:
  - `CandleType` = 15分
  - `HoldPeriods` = 17
  - `Pattern` = Bullish
  - `Side` = Long
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: なし
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
