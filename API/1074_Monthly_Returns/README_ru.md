# Стратегия Monthly Returns
[English](README.md) | [中文](README_cn.md)

Использует прорывы последних пивотных максимумов и минимумов и вычисляет сложные месячные и годовые доходности капитала стратегии.

## Детали

- **Условия входа**: покупка при пробое последнего пивотного максимума и продажа при пробое последнего пивотного минимума.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: позиции переворачиваются при противоположном сигнале.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `LeftBars` = 2
  - `RightBars` = 1
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Long & Short
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
