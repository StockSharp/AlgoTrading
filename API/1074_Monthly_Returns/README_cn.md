# Monthly Returns 策略
[English](README.md) | [Русский](README_ru.md)

该策略利用最近的枢轴高点和低点突破进行交易，并计算策略权益的复利月度和年度收益。

## 详情

- **入场条件**：价格突破最新枢轴高点时做多，突破最新枢轴低点时做空。
- **多空方向**：双向。
- **出场条件**：出现反向信号时反转持仓。
- **止损**：无。
- **默认值**：
  - `LeftBars` = 2
  - `RightBars` = 1
  - `CandleType` = TimeSpan.FromDays(1)
- **过滤器**：
  - 分类：Breakout
  - 方向：Long & Short
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：日线
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
