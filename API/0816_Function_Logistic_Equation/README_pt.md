# Estratégia de Equação Logística
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia aplica a equação logística às variações de preço ao longo de um período. Quando o valor logístico positivo cruza acima da metade do intervalo recente, compra; e quando o valor logístico negativo cai abaixo da metade inferior, vende.
