# Logistische-Gleichungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie wendet die logistische Gleichung auf Preisänderungen über einen Zeitraum an. Wenn der positive logistische Wert die Hälfte der jüngsten Spanne überschreitet, wird gekauft; wenn der negative logistische Wert unter die untere Hälfte fällt, wird verkauft.
