# Estrategia de Ecuación Logística
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia aplica la ecuación logística a los cambios de precio durante un período. Cuando el valor logístico positivo supera la mitad del rango reciente, compra; y cuando el valor logístico negativo cae por debajo de la mitad inferior, vende.
