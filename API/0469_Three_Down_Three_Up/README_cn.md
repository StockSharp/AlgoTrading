# 三跌三涨策略
[English](README.md) | [Русский](README_ru.md)

该策略在连续多根下跌收盘后买入，在连续上涨收盘后平仓。可选的 EMA 过滤器要求价格高于均线才允许入场。

## 细节

- **入场条件**：价格连续 N 根收盘低于前一根。可选条件是价格高于 EMA。
- **出场条件**：价格连续 M 根收盘高于前一根。
- **多空方向**：仅做多。
- **止损**：无。
- **默认值**：买入触发=3，卖出触发=3，EMA 周期=200。
- **过滤器**：
  - 类型：反转
  - 方向：做多
  - 指标：EMA（可选）
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
