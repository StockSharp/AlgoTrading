# Three Down Three Up 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、指定した回数の連続する下降終値の後に買いを入れ、上昇終値が連続した後にポジションを閉じます。オプションの EMA フィルターを使うと、価格が移動平均を上回っているときのみエントリーが許可されます。

## 詳細

- **エントリー条件**: 価格が前のバーより N 本連続して安値引け。オプション条件: 価格が EMA を上回っている。
- **エグジット条件**: 価格が前のバーより M 本連続して高値引け。
- **ロング/ショート**: ロングのみ。
- **ストップ**: なし。
- **デフォルト値**: 買いトリガー = 3、売りトリガー = 3、EMA 期間 = 200。
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: Long
  - インジケーター: EMA (オプション)
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
