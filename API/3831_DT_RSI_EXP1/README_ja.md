# DT RSI EXP1 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このポートは、MT4 エキスパート アドバイザ **DT-RSI-EXP1** を複製します。この戦略は 15 分間の RSI スイングをスキャンして、60/40 レベル付近のダブルトップまたはダブルボトムを検出します。最近の RSI のピークが 40 を下回る谷を表示せずに引き戻し、4 時間のトレンド フィルターが下向きになった場合、ロングトレードが行われます。ショートは、60 を超える谷と上昇トレンド フィルターを備えたロジックを反映しています。固定のストップロスとテイクプロフィットがすべてのポジションに付加され、オプションのトレーリングストップが利益を保護します。 RSI が極端な 70/30 レベルに達すると、ポジションは強制的にクローズされ、元の決済動作がコピーされます。

## 詳細

- **エントリー基準**:
  - **ロング**: 2 つの強気の RSI のピークがあり、2 番目のピークは 60 を超え、その間に 40 を下回る弱気の谷はない、4 時間足の EMA は前終値を下回っており、RSI(1) は予想ネックラインを上回っており、RSI(2) はまだ下にあり、RSI(2) < 50、およびRSI(0) < 55。
  - **ショート**: RSIの弱気の谷が2つあり、2番目の谷が40未満、間に60を超える強気の山がない、4時間足のEMAが前終値を上回っている、RSI(1)が予想ネックラインを下回っている、RSI(2) > 50およびRSI(0) > 47。
- **ロング/ショート**: 両方向。
- **終了基準**:
  - RSI の極値 (ロングの場合は RSI > 70、ショートの場合は RSI < 30)。
  - 価格ステップから計算されたストップロス/テイクプロフィットターゲット。
  - 価格が `TrailingStopPoints` だけ変動すると利益を固定するオプションのトレーリング ストップ。
- **ストップ**: 固定ストップロスとテイクプロフィット、オプションのトレーリングストップ。
- **デフォルト値**:
  - `CandleType` = 15 分のキャンドル。
  - `TrendCandleType` = 240 分足のローソク足 (トレンド フィルター EMA)。
  - `RsiPeriod` = 47。
  - `StopLossPoints` = 26。
  - `TakeProfitPoints` = 76。
  - `TrailingStopPoints` = 0 (無効)。
- **フィルター**:
  - カテゴリ: RSI 構造のトレンドフォローエントリー。
  - 方向性：両方。
  - インジケーター: RSI、EMA トレンド フィルター。
  - 停止: はい。
  - 複雑さ: 中 (複数の制約のスイング検出)。
  - 時間枠: 日中 (H4 フィルターを使用した M15)。
  - 季節性：いいえ。
  - ニューラルネットワーク: いいえ。
  - ダイバージェンス: いいえ。
  - リスクレベル：中。

## パラメーター

| 名前 | デフォルト | 説明 | 最適化可能 |
| ---- | ------- | ----------- | ----------- |
| `CandleType` | 15分 | RSI とシグナルの計算に使用される主なローソク足シリーズ。 | はい |
| `TrendCandleType` | 240分 | EMA トレンド フィルターで使用されるより高いタイムフレーム (MT4 RFTL インジケーターの代替)。 | はい |
| `RsiPeriod` | 47 | RSI の長さが主ローソク足に適用されます。 | はい |
| `StopLossPoints` | 26 | 価格ステップにおけるストップロスまでの距離。 | はい |
| `TakeProfitPoints` | 76 | 価格ステップにおける利食いまでの距離。 | はい |
| `TrailingStopPoints` | 0 | 価格ステップ内のトレーリングストップのオフセット (`0` はトレーリングを無効にします)。 | はい |

## 注意事項

- MetaTrader `RFTL` カスタム インジケーターは、240 分の時間枠で 10 期間の EMA で近似されます。元の環境に合わせて、より長い時間枠または EMA の長さを調整してください。
- ポイントベースのストップがブローカーのティック サイズと一致するように、商品の `PriceStep` と `StepPrice` が設定されていることを確認します。
- トレーリングストップは、価格がエントリー価格から `TrailingStopPoints` を超えて進んだ場合にのみ有効になり、元のストップを超えて緩むことはありません。
