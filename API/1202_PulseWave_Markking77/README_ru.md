# PulseWave Strategy
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия использует VWAP, пересечение MACD и фильтр RSI.

Вход в лонг происходит, когда цена выше VWAP, MACD пересекает сигнальную линию снизу вверх и RSI ниже уровня перекупленности. Выход выполняется, когда цена падает ниже VWAP, MACD пересекает сигнальную линию сверху вниз и RSI выше уровня перепроданности.

## Детали

- **Критерии входа**: цена выше VWAP, MACD пересекает вверх, RSI ниже перекупленности.
- **Длинные/короткие**: только длинные.
- **Критерии выхода**: цена ниже VWAP, MACD пересекает вниз, RSI выше перепроданности.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `RsiPeriod` = 14
  - `RsiOverbought` = 70
  - `RsiOversold` = 30
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: VWAP, MACD, RSI
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
