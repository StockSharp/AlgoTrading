# Библиотека Cleaner Screeners
[English](README.md) | [中文](README_cn.md)

Простая стратегия-скринер, которая оценивает RSI для нескольких инструментов и выводит сигналы покупки или продажи. Служит основой для создания пользовательских мультирыночных скринеров.

## Детали

- **Условия входа**: значения RSI проверяются относительно порогов для каждого инструмента.
- **Направление**: отсутствует (только сигналы)
- **Условия выхода**: нет
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RsiLength` = 14
  - `StrongThreshold` = 70m
  - `WeakThreshold` = 60m
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Скринер
  - Направление: Н/Д
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Н/Д
