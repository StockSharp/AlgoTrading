# Machine Learning SuperTrend TP SL
[English](README.md) | [中文](README_cn.md)

Стратегия на основе индикатора SuperTrend с плавающими тейк-профитом и стоп-лоссом.

Уровни тейк-профита и стоп-лосса следуют за линией SuperTrend, стремясь захватить тренд и фиксировать прибыль при ослаблении импульса.

## Детали
- **Критерии входа**: пересечение цены с линией SuperTrend.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: противоположный сигнал или срабатывание плавающего тейк-профита/стоп-лосса.
- **Стопы**: есть, следуют за SuperTrend.
- **Значения по умолчанию**:
  - `AtrPeriod` = 4
  - `AtrFactor` = 2.94m
  - `StopLossMultiplier` = 0.0025m
  - `TakeProfitMultiplier` = 0.022m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ATR, SuperTrend
  - Стопы: Есть
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
