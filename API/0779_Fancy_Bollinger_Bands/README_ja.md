# スタイリッシュ Bollinger Bands
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はBollinger Bandsのブレイクアウトをトレードします。終値が上バンドを上抜けるときに買い、終値が下バンドを下抜けるときに売ります。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が上バンドを上抜け
  - **ショート**: 終値が下バンドを下抜け
- **エグジット条件**: 逆クロス
- **タイプ**: ブレイクアウト
- **インジケーター**: Bollinger Bands
- **時間軸**: 1分（デフォルト）
- **ストップ**: なし
