# Прорыв наклона ATR
[English](README.md) | [中文](README_zh.md)

Стратегия ATR Slope Breakout отслеживает скорость изменения ATR. Слишком резкий наклон намекает на начало нового тренда.

Вход происходит, когда наклон превышает свой обычный уровень на несколько стандартных отклонений. Позиция открывается в направлении ускорения с защитным стопом.

Подходит активным трейдерам, ищущим ранний вход в тренд. Сделки закрываются, когда наклон возвращается к нормальным значениям. Значение по умолчанию `AtrPeriod` = 14.

## Подробности

- **Условие входа**: Индикатор превышает среднее значение на величину коэффициента отклонения.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `AtrPeriod` = 14
  - `SlopePeriod` = 20
  - `BreakoutMultiplier` = 2.0m
  - `StopLossAtrMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 148%\. Стратегию лучше запускать на рынке Форекс.
