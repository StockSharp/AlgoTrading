# Стратегия Adaptive Trend Flow
[English](README.md) | [中文](README_cn.md)

Стратегия Adaptive Trend Flow строит волатильностный канал на основе быстрых и медленных EMA типичной цены. Когда цена пересекает границы канала, внутренний тренд сменяется. Длинные позиции открываются при смене тренда вверх, если дополнительные фильтры SMA и MACD подтверждают сигнал. Позиции закрываются при смене тренда вниз.

## Подробности

- **Условия входа**:
  - Тренд меняется с нисходящего на восходящий и фильтры подтверждают сигнал.
- **Направление**: только лонг.
- **Условия выхода**:
  - Тренд меняется с восходящего на нисходящий.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `Length` = 2
  - `SmoothLength` = 2
  - `Sensitivity` = 2.0
  - `UseSmaFilter` = true
  - `SmaLength` = 4
  - `UseMacdFilter` = true
  - `MacdFastLength` = 2
  - `MacdSlowLength` = 7
  - `MacdSignalLength` = 2
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: EMA, SMA, MACD, Standard Deviation
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
