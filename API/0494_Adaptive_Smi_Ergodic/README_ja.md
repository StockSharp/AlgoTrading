# アダプティブ SMI エルゴディック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

アダプティブ SMI エルゴディック戦略は、EMA シグナルラインを持つ True Strength Index (TSI) オシレーターを使用して、買われすぎまたは売られすぎの極端な状態からのリバーサルを検出します。TSI が売られすぎしきい値を上抜けし、シグナルラインを上回っているときにロングポジションが建てられます。TSI が買われすぎしきい値を下抜けし、シグナルラインを下回っているときにショートポジションが建てられます。

## 詳細

- **エントリー条件**:
  - TSI が売られすぎを上抜けし、TSI > シグナル (ロング)。
  - TSI が買われすぎを下抜けし、TSI < シグナル (ショート)。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆シグナルが反対方向のトレードを引き起こす。
- **ストップ**: なし。
- **デフォルト値**:
  - `LongLength` = 12
  - `ShortLength` = 5
  - `SignalLength` = 5
  - `OversoldThreshold` = -0.4
  - `OverboughtThreshold` = 0.4
- **フィルター**:
  - カテゴリ: モメンタムオシレーター
  - 方向: ロング/ショート
  - インジケーター: True Strength Index, EMA
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
