# チャネル・スキャルパー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ATRベースのチャネルブレイクアウト・スキャルピングシステムです。各ローソク足の高値と安値の平均値を中心点として計算します。上部バンドと下部バンドは、Average True Rangeに係数を乗じた値を加減して構築されます。終値が前の上部バンドを上抜けするとロングポジションを開きます。下部バンドを下抜けするとショートポジションが発動されます。バンドはトレード方向に追随し、ダイナミックなストップとして機能します。反対のバンドを交差するとポジションが反転します。

## 詳細

- **エントリー条件**:
  - **買い**: 終値が前の上部バンドを上抜け。
  - **売り**: 終値が前の下部バンドを下抜け。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 価格が反対のバンドを交差したときに反転シグナル。
- **ストップ**: トレーリングチャネルバンドがストップとして機能。
- **フィルター**: なし。

## パラメーター

- **ATR Period** – ATR計算に使用するバー数。
- **ATR Multiplier** – バンド幅のためにATRに適用する係数。
- **Candle Type** – 入力ローソク足の時間軸。
