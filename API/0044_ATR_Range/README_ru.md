# Прорыв диапазона ATR
[English](README.md) | [中文](README_cn.md)

Прорыв диапазона ATR измеряет движение цены за фиксированное число баров и сравнивает его со средним истинным диапазоном. Когда движение превышает значение ATR, открывается позиция в направлении движения.

Стратегия проверяет цену каждые N баров и использует скользящую среднюю для сигналов выхода. Цель — поймать импульс при расширении волатильности выше обычного уровня.

Сделки закрываются при пересечении скользящей средней или срабатывании стопа, основанного на ATR.

## Подробности

- **Критерий входа**: движение цены больше ATR за период отслеживания.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: пересечение MA или стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `MAPeriod` = 20
  - `ATRPeriod` = 14
  - `LookbackPeriod` = 5
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе стороны
  - Индикаторы: ATR, MA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
