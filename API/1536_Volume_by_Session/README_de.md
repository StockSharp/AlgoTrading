# Volumen-nach-Sitzung-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Vereinfachte Strategie, abgeleitet vom TradingView-Indikator **"Volume by Session"**. Der Handelstag wird in vier Sitzungen unterteilt, jede mit ihrem eigenen Durchschnittsvolumen. Wenn das aktuelle Volumen innerhalb einer Sitzung von seinem Durchschnitt abweicht, eröffnet die Strategie entsprechende Trades.

## Details

- **Einstieg**: Aktuelles Sitzungsvolumen über oder unter seinem gleitenden Durchschnitt.
- **Ausstieg**: Entgegengesetztes Signal schließt die bestehende Position.
- **Long/Short**: Beide.
- **Indikatoren**: SMA.
- **Zeitrahmen**: Intraday.
