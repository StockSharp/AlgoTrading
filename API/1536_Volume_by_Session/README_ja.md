# セッション別出来高戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradingViewインジケーター **"Volume by Session"** から派生した簡略化された戦略です。取引日は4つのセッションに分割され、それぞれが独自の平均出来高を持ちます。セッション内の現在の出来高がその平均から乖離すると、戦略はそれに応じてポジションに入ります。

## 詳細

- **エントリー**: 現在のセッションの出来高がその移動平均を上回るまたは下回ること。
- **エグジット**: 反対のシグナルが既存のポジションを閉じる。
- **ロング/ショート**: 両方。
- **インジケーター**: SMA。
- **時間軸**: イントラデイ。

これは最小限の教育的翻訳であり、元のスクリプトのビジュアライゼーションと広範な設定は省略されています。
