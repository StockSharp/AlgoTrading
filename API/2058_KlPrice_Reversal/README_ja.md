# KlPrice リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はオリジナルのMQL5エキスパート**exp_i-KlPrice.mq5**のC#変換版です。正規化された価格オシレーターに基づくリバーサルシステムを実装しています。オシレーターは現在の価格を、移動平均とATR（平均真の値幅）から導出された平滑化された価格バンドと比較します。事前定義された境界を越えると取引シグナルが生成されます。

## 仕組み

1. 単純移動平均（SMA）が終値を平滑化します。
2. 平均真の値幅（ATR）が市場のボラティリティを推定します。
3. オシレーターは次のように計算されます:
   
   `jres = 100 * (Close - (SMA - ATR)) / (2 * ATR) - 50`
4. オシレーター値は5つのカラーゾーンにマッピングされます:
   - **4** – 上部レベルより上
   - **3** – ゼロと上部レベルの間
   - **2** – 上部レベルと下部レベルの間
   - **1** – 下部レベルとゼロの間
   - **0** – 下部レベルより下
5. オシレーターがゾーン4を離れるとロングポジションが開かれます。ゾーン0を離れるとショートポジションが開かれます。オシレーターがゼロを横切ると既存ポジションが決済されます。

## パラメーター

| 名前 | 説明 |
|------|------|
| `CandleType` | 価格データの時間軸。 |
| `PriceMaLength` | 価格平滑化のSMA期間。 |
| `AtrLength` | 価格バンド計算に使用するATR期間。 |
| `UpLevel` | オシレーターの上部閾値。 |
| `DownLevel` | オシレーターの下部閾値。 |
| `EnableBuy` | ロングポジションの開設を許可。 |
| `EnableSell` | ショートポジションの開設を許可。 |

## 使用方法

1. `KlPriceReversalStrategy`のインスタンスを作成。
2. 希望するパラメーターを設定。
3. 戦略をポートフォリオと銘柄に接続。
4. シグナルを受け取り注文を発注するために戦略を開始。

戦略は`BuyMarket`と`SellMarket`を通じてマーケットオーダーを使用します。ポジション保護は`StartProtection()`で有効化されます。

## 備考

- 実装はStockSharpの組み込みインジケーター（`SimpleMovingAverage`と`AverageTrueRange`）を使用して元のMLQインジケーターを近似しています。
- すべての計算は完了したローソク足のみで実行されます。
