# Стратегия Macd Momentum Reversal
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия использует гистограмму MACD для поиска разворота момента.
Продаёт, когда бычья свеча увеличивается, а гистограмма MACD снижается.
Покупает, когда медвежья свеча увеличивается, а гистограмма MACD растёт.

## Подробности

- **Критерии входа**: увеличенное тело свечи и ослабевающий импульс MACD.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `FastLength` = 12
  - `SlowLength` = 26
  - `SignalLength` = 9
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Импульс
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
