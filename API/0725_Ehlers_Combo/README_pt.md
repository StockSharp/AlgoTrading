# Estratégia Ehlers Combo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina vários filtros de Ehlers, incluindo o Elegant Oscillator, o Decycler, o Instantaneous Trend e um filtro de relação sinal/ruído. As entradas compradas e vendidas ocorrem quando todos os filtros se alinham, com saídas baseadas no Instantaneous Trend após um número especificado de barras.

## Parâmetros
- Tipo de vela
- Comprimento
- Comprimento RMS
- Limiar SNR
- Comprimento de saída
