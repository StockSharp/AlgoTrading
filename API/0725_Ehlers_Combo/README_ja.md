# Ehlers Combo戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Elegant Oscillator、Decycler、Instantaneous TrendおよびS/N比フィルターを含む複数のEhlersフィルターを組み合わせます。全フィルターが一致したときにロングおよびショートのエントリーが発生し、指定バー数後にInstantaneous Trendに基づいてエグジットします。

## パラメーター
- ローソク足タイプ
- 期間
- RMS期間
- SNR閾値
- エグジット期間
