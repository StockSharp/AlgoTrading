# Estrategia Ehlers Combo
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia combina varios filtros de Ehlers, incluyendo el Elegant Oscillator, el Decycler, el Instantaneous Trend y un filtro de relación señal/ruido. Las entradas largas y cortas se producen cuando todos los filtros se alinean, con salidas basadas en el Instantaneous Trend tras un número especificado de barras.

## Parámetros
- Tipo de vela
- Longitud
- Longitud RMS
- Umbral SNR
- Longitud de salida
