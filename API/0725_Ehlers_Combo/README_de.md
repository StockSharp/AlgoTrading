# Ehlers Combo-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert mehrere Ehlers-Filter, darunter den Elegant Oscillator, den Decycler, den Instantaneous Trend und einen Signal-Rausch-Verhältnis-Filter. Long- und Short-Einstiege erfolgen, wenn alle Filter übereinstimmen, mit Ausstiegen basierend auf dem Instantaneous Trend nach einer festgelegten Anzahl von Bars.

## Parameter
- Kerzentyp
- Länge
- RMS-Länge
- SNR-Schwellenwert
- Ausstiegslänge
