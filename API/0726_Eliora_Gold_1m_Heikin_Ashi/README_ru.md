# Стратегия Eliora Gold 1m Heikin Ashi
[English](README.md) | [中文](README_cn.md)

Стратегия использует свечи Хейкин-Аши на минутном графике. Вход осуществляется при появлении сильной свечи в направлении тренда при отсутствии консолидации. После каждой сделки задаётся пауза в несколько баров. Выход основан на трейлинговом стопе по ATR.

## Детали

- **Условия входа**: сильная свеча Хейкин-Аши, согласованная с SMA, отсутствие консолидации и фильтр волатильности.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**: трейлинг-стоп по ATR.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `AtrPeriod` = 14
  - `CooldownBars` = 5
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: Heikin Ashi, ATR, SMA, Highest/Lowest
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
