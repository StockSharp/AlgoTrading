# ATR GOD
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор Supertrend для входа и ATR для выставления стоп-лосса и тейк-профита.

## Детали

- **Критерии входа**: разворот Supertrend.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: ATR-стоп или противоположный сигнал.
- **Стопы**: ATR.
- **Значения по умолчанию**:
  - `Period` = 10
  - `Multiplier` = 3m
  - `RiskMultiplier` = 4.5m
  - `RewardRiskRatio` = 1.5m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ATR, Supertrend
  - Стопы: ATR
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
