# RSI с динамическими уровнями перекупленности/перепроданности
Стратегия **RSI Dynamic Overbought Oversold** построена на индикаторе RSI с динамическими уровнями перекупленности и перепроданности.

Сигналы возникают, когда уровни перекупленности подтверждают изменение тренда на внутридневных данных (5м). Такой метод подходит активным трейдерам.

Стопы рассчитываются на основе кратных ATR и параметров RsiPeriod, MovingAvgPeriod. Настройте эти значения для баланса риска и прибыли.

## Подробности
- **Критерии входа**: см. реализацию условий индикаторов.
- **Длинные/короткие**: обе стороны.
- **Критерии выхода**: противоположный сигнал или логика стопов.
- **Стопы**: да, расчёт на основе индикаторов.
- **Значения по умолчанию**:
  - `RsiPeriod = 14`
  - `MovingAvgPeriod = 50`
  - `StdDevMultiplier = 2.0m`
  - `StopLossPercent = 2.0m`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Оба
  - Индикаторы: перекупленность, перепроданность
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
