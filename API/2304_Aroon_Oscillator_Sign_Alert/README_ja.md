# Aroonオシレーター符号アラート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はAroonオシレーターを使用して、オシレーターが事前定義されたレベルをクロスした際にトレードシグナルを生成します。オシレーターが下限レベル（デフォルト -50）を上向きにクロスすると、ロングポジションが開かれます。上限レベル（デフォルト +50）を下向きにクロスすると、ショートポジションが開かれます。逆シグナルはポジションを決済または反転させます。

## 詳細

- **エントリー条件:**
  - **ロング**: Aroonオシレーターが下限レベルを上向きにクロスする。
  - **ショート**: Aroonオシレーターが上限レベルを下向きにクロスする。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 逆シグナルが現在のポジションを自動的に決済または反転させる。
- **ストップ**: なし。
- **フィルター**: なし。
- **時間軸**: デフォルト4時間足（設定可能）。

## パラメーター

- `AroonPeriod` – Aroonオシレーターのルックバック期間（デフォルト9）。
- `UpLevel` – 売りシグナルの上限閾値（デフォルト +50）。
- `DownLevel` – 買いシグナルの下限閾値（デフォルト -50）。
- `CandleType` – 計算に使用するローソク足の時間軸（デフォルト4時間）。
