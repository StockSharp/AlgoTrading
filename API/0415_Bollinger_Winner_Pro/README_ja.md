# Bollinger Winner Pro 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Bollinger Winner Pro は、Lite版にモジュール式フィルターとリスク管理を追加して
拡張したものです。引き続きBollinger Bandsの外側で終値を付ける価格を探しますが、
オプションの確認が一致する場合にのみ取引が実行されます。

トレーダーはRSI、Aroon、移動平均フィルターを有効にして、モメンタムとトレンドの
方向を確認できます。統合されたストップロスも有効にしてリスクを制限することが
できます。この柔軟性により、戦略はさまざまな市場やテストニーズに適応できます。

このアプローチは平均回帰を目標としています：価格がバンドに再び入るか反対側に
触れると、ポジションがクローズされるかストップが発動します。複数のフィルターを
重ねられるため、シグナルは少なくなりますが、質が高くなります。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**: ローソク足がバンドの外側で終値を付け、有効なすべてのフィルターが一致する。
- **エグジット条件**: 中央/反対側バンドへの回帰、または `UseSL` が真の場合はストップロス。
- **ストップ**: `UseSL` で制御されるオプションのストップロス。
- **デフォルト値**:
  - `UseRSI` = True
  - `UseAroon` = False
  - `UseMA` = True
  - `UseSL` = True
- **フィルター**:
  - カテゴリ: 確認付き平均回帰
  - 方向: ロング/ショート
  - インジケーター: Bollinger Bands, RSI, Aroon, Moving Average
  - 複雑さ: 上級
  - リスクレベル: 中
