# Стратегия Dark Pool Prints

Dark Pool Prints отслеживает крупные внебиржевые сделки, которые часто приводят к резким движениям после их раскрытия.
Необычные объёмы на ленте могут сигнализировать о позиционировании институционалов, которое ещё не отразилось на основном рынке.

Стратегия входит в том же направлении, что и активные покупки или продажи в тёмных пулах, ожидая продолжения, когда остальная часть рынка отреагирует.

Небольшой процентный стоп ограничивает риск, и позиция закрывается, если ожидаемый импульс не проявляется.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Объём
  - Направление: Обе
  - Индикаторы: Объём
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
