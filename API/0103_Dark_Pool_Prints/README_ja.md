# ダークプール・プリンツ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
ダークプール・プリンツは、活動が明らかになると急激な動きに先行することが多い、大規模な取引所外取引を追跡します。
テープに現れる異常な出来高は、まだ通常市場に影響を与えていない機関投資家のポジショニングを示す可能性があります。

テストでは年平均リターンが約46%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

この戦略は、大規模なダークプールの買いまたは売りと同じ方向にエントリーし、残りの市場が反応したときの継続を期待します。

小さなパーセントストップがリスクを抑制し、期待されるモメンタムが現れない場合はポジションを閉じます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: 出来高
  - 方向: 両方
  - インジケーター: Volume
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
