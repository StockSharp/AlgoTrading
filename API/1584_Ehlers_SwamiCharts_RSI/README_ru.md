# Индикатор Ehlers SwamiCharts RSI
[English](README.md) | [中文](README_cn.md)

Усредняет значения RSI с периодами 2–48 для построения цветовой карты. Зеленый цвет даёт сигнал на покупку, красный — на продажу.

## Детали

- **Условия входа**: Средний цвет зелёный (`Color1Avg` == 255 и `Color2Avg` > `LongColor`) — лонг; красный (`Color1Avg` > `ShortColor` и `Color2Avg` == 255`) — шорт.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `LongColor` = 50
  - `ShortColor` = 50
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Oscillator
  - Направление: Обе
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
