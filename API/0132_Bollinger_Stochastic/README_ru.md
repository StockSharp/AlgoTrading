# Стратегия Bollinger Stochastic

Bollinger Stochastic объединяет полосы Боллинджера со стохастическим осциллятором для выявления чрезмерно растянутых движений.
Если цена касается внешней полосы при нахождении осциллятора в экстремальной зоне, вероятен откат.

Система торгует против этих крайностей: покупает, когда цена достигает нижней полосы и стохастик в перепроданности, и продаёт у верхней полосы при стохастике в перекупленности.

Процентный стоп ограничивает риск, если возврат к среднему не происходит.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинная/короткая сторона**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию:**
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры:**
  - Категория: Среднее возвращение
  - Направление: обе
  - Индикаторы: Bollinger Bands, Stochastic
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
