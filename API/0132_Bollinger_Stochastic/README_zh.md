# 布林随机指标策略

本策略将布林带与随机振荡指标结合，用以识别过度扩张的行情。
当价格触及布林带外侧且随机指标处于极端区域时，往往预示将向均值回归。
因此价格触及下轨且随机超卖时买入，触及上轨且随机超买时卖出。
若回归未能发生，则采用百分比止损控制风险。

## 细节

- **入场条件**：指标信号
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：均值回归
  - 方向：双向
  - 指标：布林带, 随机指标
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
