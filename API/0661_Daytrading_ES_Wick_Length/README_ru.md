# Стратегия "Дэйтрейдинг ES по длине теней"
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда суммарная длина верхней и нижней тени свечи превышает её скользящую среднюю плюс смещение, и закрывает позицию после удержания фиксированного числа баров.

## Детали

- **Условия входа**: суммарная длина теней больше скользящей средней со смещением.
- **Условия выхода**: позиция закрывается после удержания `Hold periods` баров.
- **Длинная/короткая**: только длинная.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `MA length` = 20
  - `MA type` = VolumeWeighted
  - `MA offset` = 10
  - `Hold periods` = 18
  - `Candle type` = 1‑минутные свечи
- **Фильтры**:
  - Категория: волатильность
  - Направление: длинное
  - Индикаторы: скользящая средняя, длина теней
  - Стопы: нет
  - Сложность: простая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
