# The 950 Bar Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пятиминутную свечу 9:50 по Нью-Йорку. После её завершения открывается позиция в направлении свечи с фиксированными целями и стопом в тиках.

## Детали
- **Критерии входа**: направление свечи 9:50 NY.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: достижение цели или стопа.
- **Стопы**: фиксированные.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
  - `TickSize` = 0.25
  - `TargetTicks` = 150
  - `StopTicks` = 200
- **Фильтры**:
  - Категория: Время
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Фиксированные
  - Сложность: Низкая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
