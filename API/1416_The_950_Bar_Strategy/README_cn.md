# The 950 Bar Strategy
[English](README.md) | [Русский](README_ru.md)

该策略交易纽约时间9:50的五分钟K线。K线结束后，按其方向入场，并设定固定的获利目标和止损（以跳动点计）。

## 详情
- **入场条件**: 纽约9:50五分钟K线方向。
- **多空方向**: 双向。
- **退出条件**: 达到目标或止损。
- **止损**: 固定。
- **默认值**:
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
  - `TickSize` = 0.25
  - `TargetTicks` = 150
  - `StopTicks` = 200
- **过滤器**:
  - 类型: 时间
  - 方向: 双向
  - 指标: 无
  - 止损: 固定
  - 复杂度: 低
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
