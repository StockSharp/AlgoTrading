# Стратегия Sigma Spike Filtered Binned OPR
[English](README.md) | [中文](README_cn.md)

Sigma Spike Filtered Binned OPR собирает распределение показателя OPR и открывает сделки, когда значение OPR достигает экстремальных бинов после сигма-спайка доходности.

## Детали

- **Критерий входа**: OPR в экстремальных бинах (<= `OprThreshold` или >= `100 - OprThreshold`) с опциональным фильтром сигма-спайка
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `SigmaSpikeLength` = 20
  - `FilterBySigmaSpike` = true
  - `SigmaSpikeThreshold` = 2
  - `OprThreshold` = 10
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе стороны
  - Индикаторы: StandardDeviation
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
