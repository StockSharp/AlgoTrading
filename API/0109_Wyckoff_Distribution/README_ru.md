# Стратегия Wyckoff Distribution

Wyckoff Distribution описывает фазу распределения, когда на подъёмах активизируются продажи и постоянно тестируется сопротивление.
Объём обычно растёт при снижении и сокращается на отскоках, что говорит о выходе крупных участников из позиций.

Стратегия продаёт при пробое вниз из зоны распределения, рассчитывая на затяжное падение.

Стоп размещается немного выше диапазона, защищая от ложного пробоя; позиция закрывается, если цена возвращается к его верхней границе.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Обе
  - Индикаторы: Объём, цена
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
