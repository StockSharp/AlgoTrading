# Стратегия "Распределение по Вайкоффу"

Распределение по Вайкоффу описывает фазу вершины, когда на ростах наблюдаются активные продажи и тесты сопротивления. Объём часто возрастает при снижении и уменьшается на откатах, что указывает на выход крупных игроков из позиций.

Стратегия продаёт, когда цена пробивает границу диапазона распределения вниз, рассчитывая на продолжительное падение.

Стоп размещается немного выше диапазона, защищая от ложных пробоев, а позиция закрывается, если цена возвращается к верхней границе структуры.

## Детали

- **Условия входа**: сигнал индикатора
- **Длинная/короткая**: обе
- **Условия выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: обе
  - Индикаторы: Объём, Цена
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
