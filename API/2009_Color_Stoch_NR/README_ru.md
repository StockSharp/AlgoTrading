# Color Stochastic NR Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия использует стохастический осциллятор и несколько режимов сигналов. Каждый режим по-своему интерпретирует линии %K и %D для формирования сделок.

Режимы:

- **Breakdown** – покупка при пробое %K уровня 50 вверх и продажа при падении ниже.
- **OscTwist** – реагирует на изменение направления %K.
- **SignalTwist** – реагирует на изменение направления %D.
- **OscDisposition** – покупка при пересечении %K и %D снизу вверх, продажа при обратном пересечении.
- **SignalBreakdown** – сделки при пересечении %D уровня 50.

Противоположный сигнал закрывает текущую позицию и открывает новую в обратную сторону. Риски контролируются фиксированными процентными уровнями стоп-лосса и тейк-профита.

## Подробности

- **Условия входа**:
  - **Long**: зависит от выбранного режима.
  - **Short**: зависит от выбранного режима.
- **Long/Short**: оба направления.
- **Условия выхода**: противоположный сигнал или защитные уровни.
- **Стопы**: да, `StopLossPercent` и `TakeProfitPercent`.
- **Значения по умолчанию**:
  - `KPeriod` = 5
  - `DPeriod` = 3
  - `Mode` = `OscDisposition`
  - `StopLossPercent` = 2
  - `TakeProfitPercent` = 2
  - `CandleType` = 4 часа
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: Stochastic
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: 4H
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
