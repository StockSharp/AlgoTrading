# オープンドライブ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
オープンドライブとは、オーバーナイトのニュースを契機に、寄り付きから即座に強い方向性を持った動きが生じることを指します。
トレーダーは最初の数分間で大きな出来高と持続するモメンタムを確認します。

テストでは年平均リターンが約118%であることが示されています。株式市場で最も効果的です。

この戦略はそのモメンタムに乗り、寄り付きレンジ内でロングまたはショートにエントリーし、価格が伸びるにつれてトレーリングストップを引き上げます。

勢いが止まった場合は素早くポジションを閉じ、荒い寄り付きでの損失を最小限に抑えます。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15 minute
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: イントラデイ
  - 方向: 両方
  - インジケーター: Price Action
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
