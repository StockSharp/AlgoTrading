# FTMO Rules Monitor
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает правила FTMO и управляет сделками на основе ATR.

Стратегия рассчитывает размер позиции по ATR и завершает работу при выполнении условий челленджа. Она контролирует максимальный дневной убыток, общий убыток, целевую прибыль и минимальное количество торговых дней.

## Детали

- **Условия входа**: бычья свеча — покупка, медвежья — продажа.
- **Длинные/короткие**: в обе стороны.
- **Условия выхода**: завершение челленджа или противоположный сигнал.
- **Стопы**: основаны на ATR.
- **Значения по умолчанию**:
  - `AccountSize` = 10000m
  - `RiskPercent` = 1m
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: ATR
  - Стопы: ATR
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
