# オープニングレンジブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

オープニングレンジブレイクアウト戦略は、取引セッションの最初の数分間における最高値と最低値を追跡します。レンジが終了した後、設定可能なバッファーを加えたレンジの外側にブレイクアウト注文を配置します。目標はリワード・リスク比から導き出され、ストップはレンジの反対側に設定されます。

## 詳細

- **エントリー条件**:
  - オープニングレンジ後、価格が高値プラスバッファーを上回って終値を付けたときにロング。
  - 価格が安値マイナスバッファーを下回って終値を付けたときにショート。
- **ロング/ショート**: 両方
- **エグジット条件**:
  - レンジとリワード・リスク比に基づくストップとターゲット。
- **ストップ**: はい
- **デフォルト値**:
  - `RangeMinutes` = 15
  - `RewardRisk` = 2.0
  - `EntryBuffer` = 0.0001
  - `SessionStart` = 08:00
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
