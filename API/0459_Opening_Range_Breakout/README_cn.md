# 开盘区间突破策略
[English](README.md) | [Русский](README_ru.md)

该策略记录交易开始后前几分钟内的最高价和最低价。区间结束后，在区间外加上可配置缓冲的价格处布置突破订单。目标基于收益风险比，止损设置在区间的另一侧。

## 细节

- **入场条件**：
  - 区间结束后，收盘价高于区间高点加缓冲时做多。
  - 收盘价低于区间低点减缓冲时做空。
- **多空方向**：双向
- **出场条件**：
  - 根据区间和收益风险比设置止损和目标。
- **止损**：是
- **默认值**：
  - `RangeMinutes` = 15
  - `RewardRisk` = 2.0
  - `EntryBuffer` = 0.0001
  - `SessionStart` = 08:00
- **过滤器**：
  - 类别：突破
  - 方向：双向
  - 指标：无
  - 止损：是
  - 复杂度：低
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
