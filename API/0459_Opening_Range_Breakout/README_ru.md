# Стратегия Прорыва Диапазона Открытия
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает максимум и минимум первых минут торговой сессии. После завершения диапазона выставляются ордера на прорыв за пределами диапазона с заданным буфером. Цели устанавливаются по коэффициенту прибыль/риск, стопы — с противоположной стороны диапазона.

## Детали

- **Условия входа**:
  - После диапазона открытия покупка при закрытии выше максимума плюс буфер.
  - Продажа при закрытии ниже минимума минус буфер.
- **Длинные/короткие**: Оба направления
- **Условия выхода**:
  - Стоп и цель основаны на диапазоне и коэффициенте прибыль/риск.
- **Стопы**: Да
- **Значения по умолчанию**:
  - `RangeMinutes` = 15
  - `RewardRisk` = 2.0
  - `EntryBuffer` = 0.0001
  - `SessionStart` = 08:00
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
