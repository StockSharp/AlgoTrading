# Fortgeschrittene Multi-Saisonalitäts-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Handelt auf Basis vordefinierter saisonaler Zeiträume. Bis zu vier Perioden können mit individuellem Eintrittsdatum, Haltetagen und Richtung konfiguriert werden.

## Details

- **Einstiegskriterien**: Kalenderdatum stimmt mit konfigurierter Periode überein
- **Long/Short**: Beide
- **Ausstiegskriterien**: Haltetage erreicht
- **Stops**: Nein
- **Standardwerte**:
  - `CandleType` = 1 day
- **Filter**:
  - Kategorie: Saisonalität
  - Richtung: Beide
  - Indikatoren: Keine
  - Stops: Nein
  - Komplexität: Anfänger
  - Zeitrahmen: Täglich
  - Saisonalität: Ja
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
