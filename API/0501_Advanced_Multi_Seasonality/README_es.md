# Estrategia Avanzada de Multi-Estacionalidad
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Opera basándose en períodos estacionales predefinidos. Se pueden configurar hasta cuatro períodos con fecha de entrada individual, días de mantenimiento y dirección.

## Detalles

- **Criterios de entrada**: la fecha del calendario coincide con el período configurado
- **Largo/Corto**: Ambos
- **Criterios de salida**: se alcanza el número de días de mantenimiento
- **Stops**: No
- **Valores predeterminados**:
  - `CandleType` = 1 day
- **Filtros**:
  - Categoría: Estacionalidad
  - Dirección: Ambos
  - Indicadores: Ninguno
  - Stops: No
  - Complejidad: Principiante
  - Marco temporal: Diario
  - Estacionalidad: Sí
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Medio
