# Стратегия Lux Clara EMA + VWAP
[English](README.md) | [中文](README_cn.md)

Стратегия Lux Clara EMA + VWAP торгует пересечение быстрой и медленной EMA с фильтрацией по VWAP и времени. Лонг открывается, когда быстрая EMA пересекает медленную снизу вверх и медленная EMA находится выше VWAP в пределах торговой сессии. Шорт открывается при противоположных условиях. Позиции закрываются при обратном пересечении EMA.

## Подробности

- **Условия входа**:
  - Быстрая EMA пересекает медленную снизу вверх, медленная EMA выше VWAP, текущее время в пределах сессии.
  - Шорт: быстрая EMA пересекает медленную сверху вниз, медленная EMA ниже VWAP, время в пределах сессии.
- **Направление**: Лонг и шорт.
- **Условия выхода**:
  - Обратное пересечение EMA.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `FastEmaLength` = 8
  - `SlowEmaLength` = 50
  - `StartTime` = 07:30
  - `EndTime` = 14:30
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг и шорт
  - Индикаторы: EMA, VWAP
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
