# Pure Price Action Strategy
[English](README.md) | [中文](README_cn.md)

Упрощённая стратегия прайс-экшн, определяющая Break of Structure (BOS) и Market Structure Shift (MSS) по недавним максимумам и минимумам.

Стратегия открывает длинную позицию при BOS и короткую при MSS с фиксированными процентами стоп-лосса и тейк-профита.

## Детали

- **Критерии входа**: BOS для лонга, MSS для шорта.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: стоп-лосс или тейк-профит.
- **Стопы**: фиксированный процент.
- **Значения по умолчанию**:
  - `Length` = 5
  - `SlPercent` = 1m
  - `TpPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Price Action
  - Направление: Оба
  - Индикаторы: Highest, Lowest
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
