# Стратегия Crypto MVRV ZScore
[English](README.md) | [中文](README_cn.md)

Стратегия использует концепцию MVRV Z-Score для выявления экстремальных отклонений между рыночной и реализованной стоимостью.
Позиции открываются при пересечении спредового z-score заданных уровней и закрываются при обратном пересечении.

## Подробности

- **Условия входа**:
  - Лонг при пересечении спредового z-score выше `LongEntryThreshold`.
  - Шорт при пересечении спредового z-score ниже `ShortEntryThreshold`.
- **Направление**: Настраивается (`TradeDirection`).
- **Условия выхода**:
  - Пересечение противоположного уровня.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `ZScoreCalculationPeriod` = 252
  - `LongEntryThreshold` = 0.382
  - `ShortEntryThreshold` = -0.382
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Средняя реверсия
  - Направление: Оба
  - Индикаторы: SMA, StandardDeviation, Z-Score
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: День
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
