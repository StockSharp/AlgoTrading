# Стратегия G-Channel с EMA
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия сочетает логику канала G-Channel с трендовым фильтром EMA.

Покупает, когда последним было пересечение вниз и цена ниже EMA. Продаёт, когда последним было пересечение вверх и цена выше EMA.

## Детали

- **Критерии входа**: состояние G-Channel с фильтром EMA.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `ChannelLength` = 100
  - `EmaLength` = 200
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: G-Channel, EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
