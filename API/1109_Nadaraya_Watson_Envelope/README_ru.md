# Стратегия Nadaraya-Watson Envelope
[English](README.md) | [中文](README_cn.md)

Строит огибающие Надарая-Уотсона в логарифмическом масштабе. Открывает длинную позицию при пересечении ценой нижней огибающей снизу вверх и при необходимости открывает короткую при пересечении верхней огибающей сверху вниз.

## Детали

- **Условия входа**:
  - Лонг при пересечении закрытием нижней огибающей снизу вверх.
  - Шорт при пересечении закрытием верхней огибающей сверху вниз (в режиме Long/Short).
- **Длинные/Короткие**: настраивается.
- **Условия выхода**: обратное пересечение огибающих.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `LookbackWindow` = 8
  - `RelativeWeighting` = 8
  - `StartRegressionBar` = 25
  - `StrategyType` = Long Only
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Envelope
  - Направление: Настраивается
  - Индикаторы: Nadaraya-Watson
  - Стопы: Нет
  - Сложность: Продвинутая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
