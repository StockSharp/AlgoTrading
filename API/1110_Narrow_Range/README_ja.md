# ナローレンジ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

インサイドバーの後、直近のローソク足のレンジが`Length`期間前の参照バーより狭い場合にブレイクアウトを取引する戦略です。参照の高値と安値にストップ注文を置き、テイクプロフィットは参照レンジ、ストップロスはそのレンジのパーセンテージとして設定します。

## 詳細

- **エントリー条件**:
  - ロング: ナローレンジバーの後、価格が参照高値を上抜ける
  - ショート: ナローレンジバーの後、価格が参照安値を下抜ける
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 参照レンジでテイクプロフィット
  - レンジのパーセンテージでストップロス
- **ストップ**: あり
- **デフォルト値**:
  - `Length` = 4
  - `StopLossPercent` = 0.35m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: あり
  - 複雑さ: 基本
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
