# Стратегия Vwap Cci
[English](README.md) | [中文](README_cn.md)

Реализация стратегии №165 — VWAP + CCI. Покупка, когда цена ниже VWAP и CCI ниже -100 (перепроданность). Продажа, когда цена выше VWAP и CCI выше 100 (перекупленность).

VWAP служит ориентиром справедливой цены, а CCI показывает импульсные отклонения от него. Вход осуществляется при сильных значениях CCI относительно VWAP.

Стратегия рассчитана на дейтрейдеров, наблюдающих за взаимодействием с VWAP. Стопы по ATR помогают сохранять дисциплину.

## Подробности

- **Условия входа**:
  - Длинная: `Close < VWAP && CCI < CciOversold`
  - Короткая: `Close > VWAP && CCI > CciOverbought`
- **Long/Short**: Оба
- **Условия выхода**:
  - Цена пересекает VWAP в обратную сторону
- **Стопы**: процентный уровень через `StopLoss`
- **Параметры по умолчанию**:
  - `CciPeriod` = 20
  - `CciOversold` = -100m
  - `CciOverbought` = 100m
  - `StopLoss` = new Unit(2, UnitTypes.Percent)
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: Оба
  - Индикаторы: VWAP, CCI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
