# ALMA Optimized戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、Arnaud Legoux移動平均を長期EMA、ADX、RSI、ボリンジャーバンドと組み合わせたものです。ATRベースのフィルターが十分なボラティリティを確保します。ポジションはATR倍数をストップロスとテイクプロフィットに使用し、オプションの時間ベースの出口があります。

## 詳細

- **エントリー条件**:
  - **ロング**: ATRが閾値を超え、終値がEMAおよびALMAを上回り、RSI > 30、ADX > 30、終値がボリンジャーバンド上限を下回り、クールダウンが経過した場合。
  - **ショート**: 同じボラティリティフィルター下で終値が高速EMAを下回った場合。
- **エグジット条件**:
  - ATR倍数に基づくストップロスまたはテイクプロフィット。
  - バー数による任意の時間ベース出口。
- **デフォルト値**:
  - 高速EMA = 20。
  - ATR長 = 14。
  - EMA長 = 72。
  - ADX長 = 10。
  - RSI長 = 14。
  - クールダウン = 7バー。
  - ボリンジャー乗数 = 3.0。
  - ストップATR乗数 = 5.0。
  - テイクATR乗数 = 4.0。
  - 時間出口 = 0。
  - 最小ATR = 0.005。
- **フィルター**:
  - カテゴリ: トレンド + モメンタム
  - 方向: 両方
  - インジケーター: EMA、ALMA、ADX、RSI、ATR、Bollinger Bands
  - ストップ: ATRベース
  - 複雑さ: 中程度
  - 時間軸: 短期/中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
