# Стратегия MADH Moving Average Difference, Hann
[English](README.md) | [中文](README_cn.md)

Реализует индикатор MADH, описанный Джоном Эллерсом. Стратегия покупает, когда индикатор выше нуля, и продаёт, когда ниже.

## Подробности
- **Условия входа**: MADH > 0 для лонга, MADH < 0 для шорта.
- **Направление**: Оба.
- **Условия выхода**: Разворот при противоположном сигнале.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `ShortLength` = 8
  - `DominantCycle` = 27
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: MADH
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
