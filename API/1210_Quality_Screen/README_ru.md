# Quality Screen
[English](README.md) | [中文](README_cn.md)

Утилита оценивает качество компании по набору финансовых коэффициентов и выводит оценки в журнал.
Сделки не совершаются.

## Детали

- **Условия входа**: нет.
- **Длинная/короткая**: нет.
- **Условия выхода**: нет.
- **Стопы**: нет.
- **Значения по умолчанию**: нет.
