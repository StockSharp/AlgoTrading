# ORB 15m – 最初の15分ブレイクアウト戦略（ロング/ショート）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ストックホルム時間のセッション開始後の最初の15分足の終値でエントリーします。最初の足が陽線の場合はロング、陰線の場合はショートを取引します。ポジションサイズはリスク率とストップまでの距離から計算されます。

## 詳細

- **エントリー条件**: セッション開始後の最初の15分足で取引；足が始値より高く終値した場合はロング、低く終値した場合はショート。
- **エグジット条件**: 参照足の反対側の極値にストップロス；オプションで`RMultiple`倍のリスクでテイクプロフィット、またはセッション終了時にクローズ。
- **ロング/ショート**: 両方。
- **ストップ**: はい。
- **デフォルト値**:
  - `RiskPct = 1`
  - `TpTenR = true`
  - `RMultiple = 10`
  - `SessionOpenHour = 15`
  - `SessionOpenMinute = 30`
  - `SessionEndHour = 22`
  - `SessionEndMinute = 0`
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
