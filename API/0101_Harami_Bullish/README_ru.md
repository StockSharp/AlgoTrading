# Стратегия Bullish Harami

Bullish Harami — это двухсвечный паттерн, в котором небольшое тело полностью находится внутри диапазона предыдущей медвежьей свечи.
Он указывает, что импульс продаж иссяк и покупатели могут вновь вступить.

Стратегия открывает длинную позицию, когда вторая свеча закрывается внутри первой, ожидая продолжения роста на следующей свече.

Стоп на процент ниже паттерна защищает позицию, и сделка закрывается, если цена опускается ниже формации.

## Детали

- **Критерий входа**: совпадение паттерна
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Обе
  - Индикаторы: Свечные
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
