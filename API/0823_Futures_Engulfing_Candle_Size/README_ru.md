# Стратегия Futures Engulfing Candle Size
[English](README.md) | [中文](README_cn.md)

Открывает одну сделку в день, когда диапазон свечи превышает заданный порог в тиках в пределах торгового окна. Направление следует за телом свечи, выход осуществляется по тейк-профиту и стоп-лоссу.

## Детали

- **Условия входа**: диапазон свечи в тиках внутри торговой сессии.
- **Лонг/Шорт**: оба направления.
- **Условия выхода**: тейк-профит или стоп-лосс.
- **Стопы**: тейк-профит и стоп-лосс.
- **Значения по умолчанию**:
  - `CandleType` = 1 минута
  - `CandleSizeThresholdTicks` = 25
  - `TakeProfitTicks` = 50
  - `StopLossTicks` = 40
  - `StartHour` = 7
  - `StartMinute` = 0
  - `EndHour` = 9
  - `EndMinute` = 15
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Свечи
  - Стопы: Да
  - Сложность: Новичок
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
