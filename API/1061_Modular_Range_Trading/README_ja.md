# モジュール式レンジ取引戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は同時にアクティブになれない2つのモジュールを使用して、レンジ相場を対象としています。第1モジュールはMACDのモメンタム確認とRSIおよびボリンジャーバンドの平均回帰に依拠しています。第2モジュールは、RSIが売られすぎまたは買われすぎの水準にある状態で価格がボリンジャーバンド内に戻ったときに極値で買いまたは売りを入れます。ATRベースのストップとボリンジャーバンドまたはRSI反転による任意のエグジットでリスクを管理します。

## 詳細

- **エントリー条件**:
  - **ロジック1 ロング**: ADXがしきい値を下回り、MACDがシグナルを上抜け、RSIがその SMAを上回り、価格がボリンジャーバンド中央線を下回っているとき。
  - **ロジック1 ショート**: ADXがしきい値を下回り、MACDがシグナルを下抜け、RSIがその SMAを下回り、価格がボリンジャーバンド中央線を上回っているとき。
  - **ロジック2 ロング**: ADXがしきい値を下回り、価格が下側バンドを上方に再クロスし、RSIが売られすぎ水準を下回っているとき。
  - **ロジック2 ショート**: ADXがしきい値を下回り、価格が上側バンドを下方に再クロスし、RSIが買われすぎ水準を上回っているとき。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - ATRストップロス。
  - アクティブなロジックに応じた任意のボリンジャーまたはRSIシグナル。
- **ストップ**: ATRの倍数。
- **デフォルト値**: Bollinger 20/2、RSI 14、MACD 12/26/9、ATR 14、ADX 14。
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: 複数
  - ストップ: はい
  - 複雑さ: 複雑
  - 時間軸: 中期
