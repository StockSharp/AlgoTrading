# WODIsMA Dreifach-MA-Kreuzungs-Strategie zur Bullen-Bären-Trendbestätigung
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Strategie auf Basis von drei gleitenden Durchschnitten zur Trendbestätigung. Der Long-Einstieg erfolgt, wenn der schnelle Durchschnitt über dem mittleren und dem langsamen liegt; der Short-Einstieg bei umgekehrter Anordnung.
