# Стратегия ETH Signal 15m
[English](README.md) | [中文](README_cn.md)

Стратегия ETH Signal 15m использует индикатор Supertrend для обнаружения смены направления и RSI для фильтрации входов. Длинная позиция открывается, когда направление Supertrend уменьшается и RSI ниже уровня перекупленности. Короткая позиция открывается, когда направление Supertrend растёт и RSI выше уровня перепроданности. Выходы осуществляются по стоп-лоссу и тейк-профиту на основе ATR.

## Детали

- **Условия входа**:
  - **Лонг**: направление Supertrend уменьшается и RSI ниже `RsiOverbought`.
  - **Шорт**: направление Supertrend увеличивается и RSI выше `RsiOversold`.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**: стоп-лосс и тейк-профит на основе ATR.
- **Стопы**: стоп-лосс 4×ATR, тейк-профит 2×ATR для лонга и 2.237×ATR для шорта.
- **Значения по умолчанию**:
  - `AtrPeriod` = 12
  - `Factor` = 2.76
  - `RsiLength` = 12
  - `RsiOverbought` = 70
  - `RsiOversold` = 30
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Supertrend, RSI, ATR
  - Стопы: стоп-лосс и тейк-профит по ATR
  - Сложность: Низкая
  - Таймфрейм: 15m
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
