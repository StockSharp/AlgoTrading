# Стратегия Hammer + EMA с тиковой SL/TP
[English](README.md) | [中文](README_cn.md)

Сочетает свечные паттерны молот и перевёрнутый молот с трендовым фильтром EMA и тиковой защитой.

## Детали

- **Критерии входа**: молот над EMA или перевёрнутый молот под EMA.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: тик-профит или тик-стоп.
- **Стопы**: тиковые.
- **Значения по умолчанию**:
  - `EmaLength` = 50
  - `StopLossTicks` = 1
  - `TakeProfitTicks` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: EMA, молот, перевёрнутый молот
  - Стопы: Тиковые
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
