# トレンドコレクター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は元のMQLアルゴリズム`TrendCollector.mq4`の変換版です。2つの指数移動平均線を使用したトレンド検出とモメンタムおよびボラティリティフィルターを組み合わせています。

## 戦略ロジック

- **速いEMA vs 遅いEMA** – 戦略は速いEMAと遅いEMAを比較して主要なトレンドに従います。
- **ストキャスティクスオシレーター** – 買われすぎと売られすぎの状態を判断します。ストキャスティクス値が下限しきい値を下回り、速いEMAが遅いEMAを上回るとロングポジションを開きます。ストキャスティクス値が上限しきい値を上回り、速いEMAが遅いEMAを下回るとショートポジションを起動します。
- **ATRボラティリティフィルター** – 高ボラティリティの時期を避けるため、現在のATR値がボラティリティ上限を下回っている場合にのみ取引が行われます。
- **取引ウィンドウ** – 設定された開始時間と終了時間の間にのみ注文が生成されます。

## パラメーター

| 名前 | 説明 | デフォルト |
| --- | --- | --- |
| FastMaLength | 速いEMAのピリオド | 4 |
| SlowMaLength | 遅いEMAのピリオド | 204 |
| StochasticPeriod | ストキャスティクスオシレーターのピリオド | 14 |
| StochasticUpper | ストキャスティクスの上限レベル | 80 |
| StochasticLower | ストキャスティクスの下限レベル | 20 |
| AtrPeriod | ATRのピリオド | 14 |
| AtrLimit | 取引を許可する最大ATR値 | 2 |
| StartHour | 取引ウィンドウの開始時間 | 5 |
| EndHour | 取引ウィンドウの終了時間 | 24 |
| CandleTimeFrame | 処理されるローソク足の時間軸 | 5分 |

Python版は現在提供されていません。
