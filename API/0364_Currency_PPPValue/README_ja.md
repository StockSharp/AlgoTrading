# 通貨PPP価値戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

通貨PPP価値戦略は、購買力平価（PPP）に対する価格の歪みを探します。PPP価値を下回って取引されている通貨は買われ、PPP価値を上回って取引されている通貨はショートされます。ロング/ショートエクスポージャーを維持するために、ポートフォリオは毎月リバランシングされます。

PPPデータの更新頻度が低いため、必要な調整額が最低USD金額を超えた場合にのみ取引が行われます。サンプルコードは通貨をランク付けするためのフレームワークを提供していますが、実際のPPP計算はプレースホルダーとして残されています。

## 詳細

- **ユニバース**: PPP推計値が利用可能な通貨ペアのセット。
- **シグナル**: 最も過小評価されている `K` 通貨をロング、最も過大評価されている `K` 通貨をショート。
- **リバランス**: 月次。
- **ポジショニング**: ロング/ショート、均等ウェイト。
- **パラメーター**:
  - `Universe` – 取引可能な通貨。
  - `K` – ロングおよびショートする通貨の数。
  - `MinTradeUsd` – 最低取引サイズ（USD）。
  - `CandleType` – ローソク足の時間軸（デフォルト: 1日）。
- **注意**: PPP偏差の取得（`TryGetPPPDeviation`）は実装されておらず、ユーザーが提供する必要があります。
