# Стратегия XKRI Histogram
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторе Kairi Relative Index, сглаженном экспоненциальной скользящей средней. Система ищет локальные минимумы и максимумы сглаженного осциллятора и открывает длинные или короткие позиции при появлении разворотного паттерна.

## Детали

- **Условия входа**:
  - Long: `Kri[1] < Kri[2] && Kri[0] > Kri[1]`
  - Short: `Kri[1] > Kri[2] && Kri[0] < Kri[1]`
- **Long/Short**: Оба направления
- **Стопы**: Тейк‑профит и стоп‑лосс в пунктах
- **Значения по умолчанию**:
  - `KriPeriod` = 20
  - `SmoothPeriod` = 7
  - `TakeProfit` = 2000
  - `StopLoss` = 1000
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: Kairi, EMA
  - Стопы: Да
  - Сложность: Начальный уровень
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
