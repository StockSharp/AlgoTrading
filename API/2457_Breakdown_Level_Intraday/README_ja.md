# ブレイクダウンレベルイントラデイ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

指定した時刻に前日の高値と安値の周囲にブレイクアウト注文を置きます。価格が高値にデルタを加えた水準を上抜けしたときにロング、安値からデルタを引いた水準を下抜けしたときにショートに入ります。ポジション管理にはオプションのストップロス、テイクプロフィット、ブレークイーブン、トレーリングストップが含まれます。

## 詳細

- **エントリー条件**:
  - ロング: 価格が前日高値 + `Delta`を上抜け
  - ショート: 価格が前日安値 − `Delta`を下抜け
- **ロング/ショート**: 両方
- **エグジット条件**:
  - ストップロスまたはテイクプロフィット到達
  - トレーリングストップまたはブレークイーブン調整発動
- **ストップ**: エントリー価格からのポイント
- **デフォルト値**:
  - `OrderTime` = TimeSpan.Zero
  - `Delta` = 6
  - `StopLoss` = 120
  - `TakeProfit` = 90
  - `NoLoss` = 0
  - `Trailing` = 0
  - `Volume` = 1m
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: なし
  - ストップ: あり
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
