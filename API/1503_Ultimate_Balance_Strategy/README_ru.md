# Стратегия Ultimate Balance
[English](README.md) | [中文](README_cn.md)

Стратегия Ultimate Balance объединяет ROC, RSI, CCI, Williams %R и ADX в один взвешенный осциллятор. Сигналы генерируются по скользящей средней этого осциллятора: пересечение выше уровня перепроданности открывает лонг, а пересечение ниже уровня перекупленности закрывает позицию или разворачивает её.

## Детали

- **Условие входа**: MA осциллятора пересекает `OversoldLevel` снизу вверх.
- **Длинные/короткие**: Обе стороны (шорты по параметру `EnableShort`).
- **Условие выхода**: MA осциллятора пересекает `OverboughtLevel` сверху вниз.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `WeightRoc` = 2
  - `WeightRsi` = 0.5
  - `WeightCci` = 2
  - `WeightWilliams` = 0.5
  - `WeightAdx` = 0.5
  - `EnableShort` = false
  - `OverboughtLevel` = 0.75
  - `OversoldLevel` = 0.25
  - `MaType` = SMA
  - `MaLength` = 9
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Oscillator
  - Направление: Both
  - Индикаторы: ROC, RSI, CCI, WilliamsR, ADX
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
