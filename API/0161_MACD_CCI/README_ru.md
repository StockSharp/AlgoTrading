# Стратегия Macd Cci

Реализация стратегии №161 - MACD + CCI. Покупка, когда MACD выше сигнальной линии и CCI ниже -100 (зона перепроданности). Продажа, когда MACD ниже сигнальной линии и CCI выше 100 (зона перекупленности).

Колебания MACD показывают смену импульса, а CCI помогает выбрать момент входа на откатах. Возможны как длинные, так и короткие сделки.

Подойдет трейдерам, совмещающим импульсные и осцилляторные индикаторы. Для контроля риска используется стоп по ATR.

## Подробности

- **Условия входа**:
  - Длинная: `MACD > Signal && CCI < CciOversold`
  - Короткая: `MACD < Signal && CCI > CciOverbought`
- **Long/Short**: Оба
- **Условия выхода**: пересечение MACD в противоположном направлении
- **Стопы**: процентный уровень через `StopLoss`
- **Параметры по умолчанию**:
  - `FastPeriod` = 12
  - `SlowPeriod` = 26
  - `SignalPeriod` = 9
  - `CciPeriod` = 20
  - `CciOversold` = -100m
  - `CciOverbought` = 100m
  - `StopLoss` = new Unit(2, UnitTypes.Percent)
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: Оба
  - Индикаторы: MACD, CCI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
