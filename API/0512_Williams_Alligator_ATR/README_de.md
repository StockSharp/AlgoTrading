# Williams Alligator ATR-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet den Williams Alligator-Indikator in Kombination mit einem ATR-basierten Stop-Loss. Eine Long-Position wird eröffnet, wenn die Lips-Linie die Jaw-Linie nach oben kreuzt. Die Position wird geschlossen, wenn Lips Jaw nach unten kreuzt oder wenn der Preis auf ein ATR-basiertes Stop-Niveau fällt.

## Details
- **Einstiegskriterien**: Lips kreuzt Jaw nach oben.
- **Ausstiegskriterien**: Lips kreuzt Jaw nach unten oder ATR-Stop-Loss.
- **Indikatoren**: Smoothed Moving Averages, Average True Range.
- **Typ**: Nur Long.
