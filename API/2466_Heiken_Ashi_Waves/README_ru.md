# Стратегия Heiken Ashi Waves
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает свечи Heikin-Ashi с фильтром из двух скользящих средних. Пересечение быстрой SMA (2) и медленной SMA (30) указывает на изменение волны, которое подтверждается направлением текущей свечи Heikin-Ashi.

## Детали

- **Условия входа**:
  - Лонг: бычья свеча Heikin-Ashi и пересечение быстрой SMA выше медленной
  - Шорт: медвежья свеча Heikin-Ashi и пересечение быстрой SMA ниже медленной
- **Лонг/Шорт**: оба
- **Условия выхода**:
  - Обратное пересечение
  - Трейлинг-стоп
- **Стопы**: трейлинг в пунктах через параметр `StopLoss`
- **Значения по умолчанию**:
  - `FastLength` = 2
  - `SlowLength` = 30
  - `StopLoss` = new Unit(20, UnitTypes.Point)
  - `UseTrailing` = true
  - `CandleType` = TimeSpan.FromHours(1).TimeFrame()
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: оба
  - Индикаторы: Heikin Ashi, SMA
  - Стопы: да
  - Сложность: начальная
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
