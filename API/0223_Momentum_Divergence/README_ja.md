# モメンタム・ダイバージェンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

モメンタム・ダイバージェンス戦略は、モメンタムの読みを価格方向と比較して、反転の早期兆候を発見します。ダイバージェンスは、価格が新しい極値を作るのにモメンタム指標が確認できない場合に発生し、勢いの弱まりを示唆します。

テストによると、平均年間リターンは約106%です。株式市場で最も高いパフォーマンスを発揮します。

価格がより低い安値を記録しながらモメンタムオシレーターがより高い安値を示す時に強気セットアップが発生します。価格がより高い高値を押し上げるがモメンタムが追随しない時に弱気セットアップが形成されます。モメンタムがゼロに戻るか、ダイバージェンスが無効になるとポジションが決済されます。

このアプローチは、トレンドを追うのではなく転換点を予測しようとするトレーダーにアピールします。ダイバージェンスシグナルに反してマーケットが動き続ける場合のリスクを管理するためにストップを使用します。

## 詳細
- **エントリー条件**:
  - **ロング**: 価格がより低い安値を作る && Momentumがより高い安値を示す
  - **ショート**: 価格がより高い高値を作る && Momentumがより低い高値を示す
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: モメンタムがゼロを下抜けた時に決済
  - **ショート**: モメンタムがゼロを上抜けた時に決済
- **ストップ**: あり、固定ストップロス。
- **デフォルト値**:
  - `MomentumPeriod` = 14
  - `MaPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: Momentum
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
