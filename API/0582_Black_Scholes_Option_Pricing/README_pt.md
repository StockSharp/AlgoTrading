# Modelo de Precificação de Opções Black-Scholes
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Calcula os preços teóricos de opções usando o modelo Black-Scholes.
Fornece o preço de call ou put dependendo do tipo selecionado.

## Detalhes

- **Spot**: Usa o preço de fechamento como ativo subjacente.
- **Strike**: Strike da opção definido pelo usuário.
- **Volatilidade**: Parâmetro de volatilidade implícita.
- **Taxa**: Taxa de juros livre de risco.
- **Vencimento**: Dias até o vencimento da opção.
- **Tipo**: Call ou Put (europeia).
