# Modelo de Valoración de Opciones Black-Scholes
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Calcula los precios teóricos de opciones utilizando el modelo Black-Scholes.
Proporciona el precio de la opción call o put según el tipo seleccionado.

## Detalles

- **Spot**: Utiliza el precio de cierre como subyacente.
- **Strike**: Strike de la opción definido por el usuario.
- **Volatilidad**: Parámetro de volatilidad implícita.
- **Tasa**: Tasa de interés libre de riesgo.
- **Vencimiento**: Días hasta el vencimiento de la opción.
- **Tipo**: Call o Put (europea).
