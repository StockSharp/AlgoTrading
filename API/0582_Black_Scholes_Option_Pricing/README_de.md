# Black-Scholes-Optionspreismodell
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Berechnet theoretische Optionspreise mit dem Black-Scholes-Modell.
Liefert den Call- oder Put-Preis je nach gewähltem Typ.

## Details

- **Spot**: Verwendet den Schlusskurs als Basiswert.
- **Strike**: Vom Benutzer definierter Optionsstrike.
- **Volatilität**: Parameter der impliziten Volatilität.
- **Rate**: Risikofreier Zinssatz.
- **Laufzeit**: Tage bis zum Optionsverfall.
- **Typ**: Call oder Put (europäisch).
