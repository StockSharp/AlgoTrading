# Black-Scholesオプション価格算出モデル
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Black-Scholesモデルを使ってオプションの理論価格を計算します。
選択されたタイプに応じてコールまたはプットの価格を提供します。

## 詳細

- **スポット**: 原資産として終値を使用。
- **ストライク**: ユーザー定義のオプションストライク価格。
- **ボラティリティ**: インプライドボラティリティパラメーター。
- **レート**: リスクフリー金利。
- **満期**: オプション満期までの日数。
- **タイプ**: コールまたはプット（欧州型）。
