# VRS Vegas Reversal Strategy
[English](README.md) | [Русский](README_ru.md)

基于K线影线的反转策略。

测试显示年化收益约37%，在加密货币市场表现最好。

系统寻找相对于收盘价的较大影线。较长的下影线触发做多，较长的上影线触发做空。价格向有利方向移动两倍影线长度时平仓。

## 详情

- **入场条件**：
  - **做多**：下影线 ≥ Spike% * close 且没有上影线。
  - **做空**：上影线 ≥ Spike% * close 且没有下影线。
- **多空方向**：双向。
- **出场条件**：目标价 = 入场价 ± (影线 * 2)。
- **止损**：无。
- **默认值**：
  - `SpikePercent` = 0.025
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤条件**：
  - 类型：反转
  - 方向：双向
  - 指标：Price action
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：高
