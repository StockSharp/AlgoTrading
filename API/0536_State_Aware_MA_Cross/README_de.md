# Zustandsbewusste MA-Kreuzungs-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie wählt verschiedene Moving-Average-Paare basierend auf der Steigung und der relativen Position einer Basis-EMA aus. Eine Long-Position wird eröffnet, wenn der gewählte schnelle MA den entsprechenden langsamen MA nach oben kreuzt, und geschlossen, wenn er zurück nach unten kreuzt.
