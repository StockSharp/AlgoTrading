# Estrategia de Cruce MA con Conciencia de Estado
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia selecciona diferentes pares de medias móviles según la pendiente y la posición relativa de una EMA base. Se abre una posición larga cuando la MA rápida elegida cruza por encima de la MA lenta correspondiente, y se cierra cuando cruza de vuelta por debajo.
