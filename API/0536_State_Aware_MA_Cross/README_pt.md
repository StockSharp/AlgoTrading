# Estratégia de Cruzamento de MA com Consciência de Estado
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia seleciona diferentes pares de médias móveis com base na inclinação e na posição relativa de uma EMA base. Uma posição comprada é aberta quando a MA rápida escolhida cruza acima da MA lenta correspondente, e fechada quando cruza de volta abaixo.
