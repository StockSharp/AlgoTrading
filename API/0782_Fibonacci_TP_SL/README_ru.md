# Стратегия Fibonacci TP SL
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует уровни коррекции Фибоначчи для входа в позицию, устанавливая стоп-лосс на основе ATR и тейк-профит в процентах. Торговля ограничена минимальным интервалом между сделками и недельным ограничением прибыли.

## Детали

- **Условия входа**:
  - **Лонг**: `Close <= Fib 38.2%` && `Close >= Fib 78.6%` && `Минимальное количество баров с последней сделки`
  - **Шорт**: `Close <= Fib 23.6%` && `Close >= Fib 61.8%` && `Минимальное количество баров с последней сделки`
- **Направление**: Обе стороны
- **Условия выхода**:
  - `ATR стоп-лосс` или `Тейк-профит`
- **Стопы**: Да
- **Значения по умолчанию**:
  - `Take Profit %` = 4
  - `Min Bars Between Trades` = 10
  - `Lookback` = 100
  - `ATR Period` = 14
  - `ATR Multiplier` = 1.5
  - `Max Weekly Return` = 0.15

- **Фильтры**:
  - Категория: Среднее возвращение
  - Направление: Обе
  - Индикаторы: Highest, Lowest, ATR
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
