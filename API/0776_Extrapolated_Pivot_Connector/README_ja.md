# Extrapolated Pivot Connector戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

直近のピボット高値と安値を結び、サポートラインとレジスタンスラインを構築します。価格がレジスタンスラインを上回って引けるとロングシグナルが発生し、サポートラインを下回るとショートシグナルが発火します。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値がレジスタンスラインを上抜け。
  - **ショート**: 終値がサポートラインを下抜け。
- **エグジット条件**:
  - 逆シグナルまたはポジション反転。
- **インジケーター**:
  - ピボットベースのサポート/レジスタンスライン。
- **ストップ**: なし。
- **デフォルト値**:
  - `PivotLength` = 100
  - `HighStart` = 1
  - `HighEnd` = 0
  - `LowStart` = 1
  - `LowEnd` = 0
- **フィルター**:
  - トレンドフォロー
  - 単一時間軸
  - インジケーター: ピボットライン
  - ストップ: なし
  - 複雑さ: 低
