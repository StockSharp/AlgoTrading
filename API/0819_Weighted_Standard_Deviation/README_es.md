# Estrategia de Desviación Estándar Ponderada
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Implementa una estrategia basada en la media ponderada por volumen y la desviación estándar. Compra cuando el precio cae por debajo de la banda inferior y vende cuando el precio sube por encima de la banda superior. Las posiciones se cierran cuando el precio vuelve a la media.
