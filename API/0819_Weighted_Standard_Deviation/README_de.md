# Strategie mit gewichteter Standardabweichung
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Implementiert eine Strategie auf Basis des volumengewichteten Mittelwerts und der Standardabweichung. Kauft, wenn der Preis unter das untere Band fällt, und verkauft, wenn er über das obere Band steigt. Positionen werden geschlossen, wenn der Preis zum Mittelwert zurückkehrt.
