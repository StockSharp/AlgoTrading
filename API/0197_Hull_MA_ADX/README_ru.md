# Стратегия Hull MA ADX

Стратегия основана на скользящей средней Халла и индикаторе ADX. Длинная позиция открывается, когда HMA растет и ADX выше 25 (сильный тренд). Короткая позиция открывается, когда HMA падает и ADX выше 25. Выход из сделки выполняется при падении ADX ниже 20, что сигнализирует об ослаблении тренда.

HMA отображает направление, а ADX подтверждает его силу. Вход выполняется по наклону HMA при условии, что ADX демонстрирует трендовость.

Подходит трейдерам, предпочитающим плавные тренды с подтверждением. Стоп по ATR удерживает убытки под контролем.

## Подробности

- **Условия входа**:
  - Long: `HullMA turning up && ADX > 25`
  - Short: `HullMA turning down && ADX > 25`
- **Long/Short**: оба направления
- **Условия выхода**: разворот Hull MA
- **Стопы**: ATR через `AtrMultiplier`
- **Значения по умолчанию**:
  - `HmaPeriod` = 9
  - `AdxPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Trend
  - Направление: оба
  - Индикаторы: Hull MA, Moving Average, ADX
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейронные сети: нет
  - Дивергенция: нет
  - Уровень риска: средний
