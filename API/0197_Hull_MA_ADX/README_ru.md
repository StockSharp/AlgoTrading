# Hull Ma Adx Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия основана на Hull Moving Average и ADX. Вход в длинные позиции, когда HMA растет и ADX > 25 (сильный тренд). Вход в короткие, когда HMA падает и ADX > 25. Выход, когда ADX < 20 (ослабление тренда).

Hull MA показывает направление, ADX подтверждает его силу. Входы происходят по наклону HMA при сильном ADX.

Эффективна для трейдеров, предпочитающих плавные тренды с подтверждением. Стопы по ATR удерживают убытки под контролем.

## Детали

- **Критерии входа**:
  - Long: `HullMA turning up && ADX > 25`
  - Short: `HullMA turning down && ADX > 25`
- **Long/Short**: Оба направления
- **Критерии выхода**: разворот Hull MA
- **Стопы**: ATR на основе `AtrMultiplier`
- **Значения по умолчанию**:
  - `HmaPeriod` = 9
  - `AdxPeriod` = 14
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(15).TimeFrame()
- **Фильтры**:
  - Категория: Trend
  - Направление: Оба
  - Индикаторы: Hull MA, Moving Average, ADX
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний

Тестирование показывает среднегодичную доходность около 178%\. Стратегию лучше запускать на фондовом рынке.
