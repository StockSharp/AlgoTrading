# Estratégia de Rompimento Lite FVG
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia detecta brechas de valor justo (FVG) usando um padrão de três velas e abre uma posição quando o preço fecha além dos limites da brecha. Um stop-loss percentual gerencia o risco.
