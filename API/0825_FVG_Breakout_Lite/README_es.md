# Estrategia de Ruptura Lite FVG
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia detecta brechas de valor justo (FVG) mediante un patrón de tres velas y abre una posición cuando el precio cierra más allá de los límites de la brecha. Un stop-loss porcentual gestiona el riesgo.
