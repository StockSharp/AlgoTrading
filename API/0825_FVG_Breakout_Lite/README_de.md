# FVG Breakout Lite-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie erkennt Fair-Value-Gaps anhand eines Drei-Kerzen-Musters und eröffnet eine Position, wenn der Preis über die Gap-Grenzen hinaus schließt. Ein prozentualer Stop-Loss steuert das Risiko.
