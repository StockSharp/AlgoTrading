# COSTAR戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

COSTAR戦略は終値の線形回帰を構築し、残差の標準偏差を計測します。偏差にファクターを乗じた値を加減することで上限バンドと下限バンドを形成します。取引は極端な偏差に逆張りして行われ、価格が回帰線に戻ったときにイグジットします。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格が下限バンドを上抜け。
  - **ショート**: 価格が上限バンドを下抜け。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 価格が回帰線を再度クロス。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 100
  - `Multiplier` = 1
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: Linear Regression, Standard Deviation
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
