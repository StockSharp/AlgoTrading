# Стратегия Donchian WMA Crossover
[English](README.md) | [中文](README_cn.md)

Нижняя линия канала Дончиана, пересекающая сверху взвешенное скользящее среднее, открывает длинные позиции только в течение календарного 2025 года. Позиции закрываются при достижении тейк-профита, при обратном пересечении на фоне падающей WMA либо при выходе даты за пределы 2025 года.

## Подробности

- **Условия входа**:
  - Лонг: `DonchianLow` пересекает `WMA` сверху, дата внутри 2025 года
- **Длинные/короткие**: только лонг
- **Условия выхода**:
  - Тейк-профит через `TakeProfitPercent`
  - Пересечение `DonchianLow` ниже `WMA` при падающей `WMA`
  - Дата вне 2025 года
- **Стопы**: только тейк-профит
- **Значения по умолчанию**:
  - `DonchianLength` = 7
  - `WmaLength` = 62
  - `TakeProfitPercent` = 0.01m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Пробой
  - Направление: Лонг
  - Индикаторы: Канал Дончиана, взвешенное скользящее среднее
  - Стопы: Да
  - Сложность: Начальная
  - Таймфрейм: Среднесрочный
  - Сезонность: Только 2025 год
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
