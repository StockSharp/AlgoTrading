# 智能因子动量市场
[English](README.md) | [Русский](README_ru.md)

**智能因子动量市场** 策略将多个股票因子与大盘趋势过滤器结合。仅当因子组合和市场指数都显示上升趋势时才做多市场，否则持有现金。

## 细节
- **入场条件**：因子动量与市场趋势同时为正。
- **多空方向**：仅做多。
- **出场条件**：当因子动量或市场趋势转负时退出。
- **止损**：无显式止损。
- **默认值**：
  - `CandleType = TimeSpan.FromDays(1).TimeFrame()`
- **筛选**：
  - 分类：动量
  - 方向：多头
  - 指标：多重
  - 止损：无
  - 复杂度：中等
  - 时间框架：中期
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险级别：中等
