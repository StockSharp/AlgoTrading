# スマート・ファクター・モメンタム・マーケット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**スマート・ファクター・モメンタム・マーケット**戦略は、複数の株式ファクターと広範な市場トレンド・フィルターを組み合わせます。モメンタム・ファクター・バスケットと総合指数の両方が正のトレンドを示す場合にのみ市場でロングポジションを取り、そうでない場合は現金に移行します。

## 詳細
- **エントリー条件**: 複合ファクター・モメンタムと市場トレンドの確認。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: ファクター・モメンタムまたは市場トレンドがマイナスになったときに撤退。
- **ストップ**: 明示的なストップなし。
- **デフォルト値**:
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: ロング
  - インジケーター: 複数
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
