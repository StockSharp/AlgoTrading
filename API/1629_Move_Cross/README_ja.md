# Move Cross戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略は、元の `move_cross.mq4` スクリプトの簡略化された変換を示しています。トレンド方向を決定するために、2つの単純移動平均から計算されたRAVI（Range Action Verification Index）インジケーターを採用しています。

戦略は時間足と日足のRAVI値を比較します：

- 時間足のRAVIが負で、日足のRAVIが正かつ上昇しているときに**買い**。
- 時間足のRAVIが正で、日足のRAVIが負かつ下落しているときに**売り**。

ポジションはオプションの利益目標とストップロスを伴う成行注文で開かれます。

## パラメーター

| 名前       | 説明                          | デフォルト |
|------------|-------------------------------|-----------|
| TakeProfit | ポイント単位の利益目標         | 50        |
| StopLoss   | ポイント単位の損失限度         | 100       |

## 注意事項

- 戦略は時間足と日足のローソク足のRAVIを計算するために、2つのSMAペア（2と24期間）を使用します。
- 教育目的で作成されており、実際の取引には追加の調整が必要な場合があります。
