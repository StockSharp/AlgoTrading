# Стратегия Labouchere EA
[English](README.md) | [中文](README_cn.md)

Эта стратегия сочетает пересечения стохастического осциллятора с системой управления капиталом Labouchere. Индикатор Stochastic генерирует сигналы при пересечении линии %K и %D. Система Labouchere корректирует объём сделки после закрытия позиции: при убытке в последовательность добавляется сумма первого и последнего элементов, при прибыли эти элементы удаляются.

Сделки совершаются только по завершённым свечам. Последовательность может перезапускаться при удалении всех чисел. Встроенный фильтр времени позволяет торговать лишь в заданном интервале суток, а противоположные сигналы могут закрывать открытую позицию. Поддерживаются фиксированные уровни стоп-лосса и тейк-профита (в шагах цены).

## Подробности
- **Условия входа**:
  - **Покупка**: %K пересекает %D снизу вверх.
  - **Продажа**: %K пересекает %D сверху вниз.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Опционально выход по противоположному сигналу.
  - Фиксированный стоп-лосс и тейк-профит (если заданы).
- **Стопы**: Да.
- **Управление капиталом**: последовательность Labouchere.
- **Значения по умолчанию**:
  - `LotSequence` = "0.01,0.02,0.01,0.02,0.01,0.01,0.01,0.01"
  - `NewRecycle` = true
  - `StopLoss` = 40
  - `TakeProfit` = 50
  - `IsReversed` = false
  - `UseOppositeExit` = false
  - `UseWorkTime` = false
  - `StartTime` = 00:00
  - `StopTime` = 24:00
  - `KPeriod` = 10
  - `DPeriod` = 190
- **Фильтры**:
  - Категория: смешанная
  - Направление: оба
  - Индикаторы: Stochastic Oscillator
  - Стопы: есть
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенции: нет
  - Уровень риска: средний
