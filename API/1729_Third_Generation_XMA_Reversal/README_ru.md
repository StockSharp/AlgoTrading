# Стратегия 3rd Generation XMA Reversal
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Использует двойное экспоненциальное сглаживание, известное как скользящая средняя третьего поколения, для выявления локальных максимумов и минимумов. Лонг открывается, когда XMA разворачивается вверх от локального дна. Шорт открывается при развороте XMA вниз от локальной вершины. Позиция переворачивается при противоположном сигнале, без явных стопов и тейк‑профитов.

## Детали
- **Условия входа**: XMA образует локальный минимум или максимум и разворачивается.
- **Длинные/короткие**: Обе стороны.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `MaLength` = 50
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (4 часа)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
