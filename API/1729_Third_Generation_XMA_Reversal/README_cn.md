# 3rd Generation XMA Reversal
[English](README.md) | [Русский](README_ru.md)

采用双重指数平滑的第三代移动平均线来寻找局部高点和低点。当 XMA 从局部低点向上转折时开多单；当 XMA 在局部高点向下转折时开空单。出现反向信号时反向开仓，不设置固定止损或止盈。

## 细节
- **入场条件**：XMA 形成局部极值并反转。
- **多空方向**：双向。
- **出场条件**：反向信号。
- **止损**：无。
- **默认数值**：
  - `MaLength` = 50
  - `CandleType` = TimeSpan.FromHours(4)
- **过滤器**：
  - 类别：趋势
  - 方向：双向
  - 指标：EMA
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内（4 小时）
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
