# 发薪异常策略
[English](README.md) | [Русский](README_ru.md)

该策略在每月选定的日期（1日、2日、16日和31日）开多头仓位，并在下一天平仓。

## 细节

- **入场条件**：
  - **多头**：在选定的日期开多头。
- **多空方向**：仅多头。
- **出场条件**：
  - 在非选定日期平掉多头仓位。
- **止损**：无。
- **默认值**：
  - `Trade1st` = true。
  - `Trade2nd` = true。
  - `Trade16th` = true。
  - `Trade31st` = true。
  - `CandleType` = TimeSpan.FromDays(1).TimeFrame()。
- **筛选**：
  - 类型: 季节性
  - 方向: 多头
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日线
  - 季节性: 是
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
