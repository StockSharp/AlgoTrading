# Стратегия Scale In Scale Out
[English](README.md) | [中文](README_cn.md)

Стратегия постепенно наращивает позицию, покупая фиксированный процент доступных средств на каждой свече. Когда значение позиции достигает заданного уровня прибыли, часть позиции продаётся, а часть полученной прибыли может быть отложена.

## Подробности

- **Критерий входа**: Покупка при наличии свободных средств.
- **Критерий выхода**: Продажа при превышении порога прибыли.
- **Длинная/Короткая**: Только длинная.
- **Значения по умолчанию**:
  - `Buy Scaling Size %` = 2
  - `Take Profit Level %` = 50
  - `Take Profit Size %` = 1
  - `Retain Profit Portion %` = 50
  - `Minimum Position Value` = 200000
  - `Minimum Buy Value` = 100
- **Фильтры**:
  - Категория: Другая
  - Направление: Лонг
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
