# Estratégia de Retração Fibonacci para Criptomoedas
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera utilizando níveis de retração Fibonacci calculados a partir dos preços mais altos e mais baixos ao longo de um período de lookback.

- Entra comprado quando o preço cruza acima do nível 61,8%.
- Entra vendido quando o preço cruza abaixo do nível 38,2%.
- Sai do comprado quando o preço atinge o nível 23,6%.
- Sai do vendido quando o preço atinge o nível 78,6%.
