# Fibonacci-Retracement-Strategie für Krypto
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt mit Fibonacci-Retracement-Niveaus, die aus den höchsten und niedrigsten Kursen über einen Rückblickzeitraum berechnet werden.

- Geht long, wenn der Kurs über das 61,8%-Niveau steigt.
- Geht short, wenn der Kurs unter das 38,2%-Niveau fällt.
- Schließt Long-Position, wenn der Kurs das 23,6%-Niveau erreicht.
- Schließt Short-Position, wenn der Kurs das 78,6%-Niveau erreicht.
