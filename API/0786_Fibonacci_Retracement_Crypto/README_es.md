# Estrategia de Retroceso Fibonacci para Criptomonedas
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia opera utilizando niveles de retroceso Fibonacci calculados a partir de los precios más altos y más bajos durante un período de lookback.

- Entra en largo cuando el precio cruza por encima del nivel 61.8%.
- Entra en corto cuando el precio cruza por debajo del nivel 38.2%.
- Sale del largo cuando el precio alcanza el nivel 23.6%.
- Sale del corto cuando el precio alcanza el nivel 78.6%.
