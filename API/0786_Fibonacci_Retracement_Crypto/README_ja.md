# 暗号資産向けFibonacciリトレースメント戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ルックバック期間における最高値と最安値から計算したFibonacciリトレースメントレベルを使用して取引します。

- 価格が61.8%レベルを上にクロスしたときにロングエントリー。
- 価格が38.2%レベルを下にクロスしたときにショートエントリー。
- 価格が23.6%レベルに達したときにロングを決済。
- 価格が78.6%レベルに達したときにショートを決済。
