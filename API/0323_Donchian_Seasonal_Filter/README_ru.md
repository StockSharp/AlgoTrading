# Дончан с сезонным фильтром

Стратегия **Donchian Seasonal Filter** торгует по пробоям канала Дончиана с учётом сезонного фильтра.
Сигналы формируются, когда Дончиан подтверждает отфильтрованные входы на внутридневных данных (15м). Такой подход подходит активным трейдерам.
Стопы рассчитываются исходя из кратных ATR и параметров DonchianPeriod, SeasonalThreshold. Эти значения можно изменять для баланса риска и прибыли.

## Подробности
- **Условия входа**: см. реализацию для условий по индикаторам.
- **Длинные/короткие позиции**: обе стороны.
- **Условия выхода**: обратный сигнал или логика стопов.
- **Стопы**: да, вычисляются на основе индикаторов.
- **Значения по умолчанию**:
  - `DonchianPeriod = 20`
  - `SeasonalThreshold = 0.5m`
  - `CandleType = TimeSpan.FromMinutes(15).TimeFrame()`
- **Фильтры**:
  - Категория: Следование за трендом
  - Направление: Оба
  - Индикаторы: Donchian, Seasonal
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (15m)
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
