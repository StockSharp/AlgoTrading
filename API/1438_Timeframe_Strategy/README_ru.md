# Стратегия Таймфрейма
[English](README.md) | [中文](README_cn.md)

Стратегия пересечения EMA с учётом торговых часов и управления рисками по процентам.

Тестирование показывает среднегодичную доходность около 31%. Стратегия лучше всего работает на крипторынке.

Система покупает, когда быстрая EMA пересекает медленную снизу вверх и долгосрочный тренд восходящий. Короткие позиции открываются при обратном пересечении. Торговые часы и фильтр ADX помогают избегать периодов низкого импульса. Риск контролируется через процентные уровни тейк-профита и стоп-лосса.

## Подробности

- **Условия входа**:
  - **Лонг**: EMA9 пересекает EMA20 снизу, EMA50 выше EMA200.
  - **Шорт**: EMA9 пересекает EMA20 сверху, EMA50 ниже EMA200.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - **Лонг**: Стоп-лосс или тейк-профит.
  - **Шорт**: Стоп-лосс или тейк-профит.
- **Стопы**: Да, возможен трейлинг.
- **Значения по умолчанию**:
  - `TakeProfitPercent` = 1.5
  - `StopLossPercent` = 1.0
  - `TrailingPercent` = 0.5
  - `StartHour` = 15
  - `EndHour` = 20
  - `CooldownBars` = 5
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: EMA, RSI, ADX
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
