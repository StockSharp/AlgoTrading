# Bollinger ダイバージェンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Bollinger Divergence は、価格がバンドを突き抜けると同時に反対側のバンドが
収縮し始める極端な状況を探します。価格のモメンタムとボラティリティの間の
このダイバージェンスは、レンジの中央へのスナップバックに先行することが
よくあります。

ローソク足が下限バンドを下回って終値を付け、かつ上限バンドが少なくとも
設定されたパーセンテージ分狭まると、ロングシグナルが発生します。
ショートの場合、パターンは上限バンドを中心に反転します。ポジションは
Bollinger の中央ラインへの素早い戻りを目標とし、オプションで固定の
利確設定もあります。

このセットアップは、レンジ相場やボラティリティの急上昇が落ち着き始めた
後に最も効果を発揮します。`CandlePercent` パラメーターは、取引を許可する前に
反対側のバンドがどれだけ収縮しなければならないかを制御し、強いトレンド時の
ダマシを回避するのに役立ちます。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 終値が下限バンドを下回り、かつ上限バンドが `CandlePercent` 収縮。
  - **ショート**: 終値が上限バンドを上回り、かつ下限バンドが `CandlePercent` 収縮。
- **エグジット条件**:
  - 中央バンドへの回帰、または利確パーセンテージ。
- **ストップ**: ハードストップなし；利確または手動エグジットに依存。
- **デフォルト値**:
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
  - `CandlePercent` = 30
  - `TakeProfit` = 5
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロング/ショート
  - インジケーター: Bollinger Bands
  - 複雑さ: シンプル
  - リスクレベル: 中
