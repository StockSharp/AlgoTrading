直近のスイング高値と安値から算出した61.8%のFibonacci延長水準を上回りつつ、終値がBollinger Bandsの下バンドを下回るときに買いポジションを建てます。価格が上バンドまたは直前のスイング高値に到達したときにポジションをクローズします。
