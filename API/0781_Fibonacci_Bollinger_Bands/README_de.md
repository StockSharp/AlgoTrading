# Fibonacci- und Bollinger-Bands-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie kauft, wenn der Preis unter dem unteren Bollinger Band schließt und dabei über der 61,8% Fibonacci-Extension vom letzten Swing-Hoch und -Tief bleibt. Die Position wird geschlossen, wenn der Preis das obere Band oder das vorherige Swing-Hoch erreicht.
