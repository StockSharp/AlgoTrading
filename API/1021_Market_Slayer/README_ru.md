# Стратегия Market Slayer
[English](README.md) | [中文](README_cn.md)

Стратегия использует пересечение взвешенных скользящих средних с фильтром тренда SSL на старшем таймфрейме. Лонг открывается, когда короткая WMA пересекает длинную снизу вверх и тренд бычий; шорт — при обратных условиях. При желании можно включить тейк-профит и стоп-лосс в абсолютных пунктах.

## Детали

- **Условия входа**:
  - **Лонг**: короткая WMA пересекает длинную снизу вверх и тренд по SSL бычий.
  - **Шорт**: короткая WMA пересекает длинную сверху вниз и тренд по SSL медвежий.
- **Лонг/Шорт**: обе стороны.
- **Условия выхода**:
  - Трендовый фильтр меняется на противоположный.
  - Опциональные стоп-лосс или тейк-профит.
- **Стопы**: опционально.
- **Значения по умолчанию**:
  - `ShortLength` = 10.
  - `LongLength` = 20.
  - `ConfirmationTrendValue` = 2.
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame().
  - `TrendCandleType` = TimeSpan.FromMinutes(240).TimeFrame().
  - `TakeProfitEnabled` = false.
  - `TakeProfitValue` = 20.
  - `StopLossEnabled` = false.
  - `StopLossValue` = 50.
- **Фильтры**:
  - Категория: Тренд
  - Направление: обе стороны
  - Индикаторы: WMA, SSL
  - Стопы: опционально
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
