# Стратегия Rate of Change
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор Rate of Change для обнаружения пузырей и торговли пересечений нулевой линии с динамическим изменением объёма.

Тесты показывают стабильную работу на дневных данных по основным инструментам.

## Детали

- **Условия входа**: пересечение ROC нулевой линии; дополнительный шорт при выходе из пузыря.
- **Длинные/короткие**: в обе стороны.
- **Условия выхода**: противоположный сигнал или стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `RocLength` = 365
  - `BubbleThreshold` = 180m
  - `StopLossPercent` = 6m
  - `FixedRatioValue` = 400m
  - `IncreasingOrderAmount` = 200m
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Momentum
  - Направление: Оба
  - Индикаторы: RateOfChange
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
