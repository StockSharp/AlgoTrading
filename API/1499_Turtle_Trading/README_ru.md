# Стратегия Turtle Trading
[English](README.md) | [中文](README_cn.md)

Классическая система Turtle Trading с прорывом каналов Дончиана и управлением риском на основе ATR.

## Детали

- **Критерий входа**: прорыв верхней/нижней границы канала Дончиана
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: пересечение короткого канала Дончиана или трейлинг-стоп
- **Стопы**: начальный и трейлинг-стоп на основе ATR
- **Значения по умолчанию**:
  - `EntryLength` = 20
  - `ExitLength` = 10
  - `EntryLengthMode2` = 55
  - `ExitLengthMode2` = 20
  - `AtrPeriod` = 14
  - `RiskPerTrade` = 0.02
  - `InitialStopAtrMultiple` = 2
  - `PyramidAtrMultiple` = 0.5
  - `MaxUnits` = 4
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: DonchianChannels, ATR
  - Стопы: ATR
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
