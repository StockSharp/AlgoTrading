# MP ローソク足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
**MP ローソク足戦略**は、MetaTrader 5 エキスパートアドバイザー `mp candlestick.mq5` を StockSharp の高レベル戦略フレームワークに変換したものです。このシステムは完成したローソク足の方向を評価し、厳格なリスク管理を適用しながら同じ方向に取引を開始します。 MetaTrader ピップで表される固定ストップロス距離と、平均トゥルー レンジ (ATR) から導出される適応ストップロス配置の両方をサポートします。

## 取引ロジック
1. この戦略は、単一の構成可能なローソク足シリーズ (デフォルト: 1 時間のローソク足) をサブスクライブします。
2. 完成したキャンドルごとに:
   - 強気のローソク足（終値が始値を上回る） → ロングポジションを検討します。
   - 弱気のローソク足（始値を下回る終値）→ショートポジションを検討します。
   - 童子キャンドルは無視されます。
3. エントリーの前に、ストラテジーは ATR または固定ピップ距離からストップロス価格を計算します。テイクプロフィット価格は、設定されたリスク対報酬比を使用して計算されます。
4. 証拠金の使用量が許容パーセンテージ内に留まり、計算されたポジションサイズが有効な場合、取引は市場で開始されます。
5. ポジションがアクティブである間、戦略は各新しいローソク足を監視します。
   - ローソク足の極値を使用したストップロスまたはテイクプロフィットのヒット。
   - ATR ストップのときにストップを損益分岐点に向けて移動するトレーリング調整が有効になっています。
6. 位置が平らになると、次の完成したキャンドルからプロセスが再開されます。

## リスクと資金の管理
- **リスクパーセント**は、取引ごとにリスクがかかる株式の割合を定義します。ポジションサイズは、エントリーとストップロスの間の価格距離と商品価格/ステップ値から導出されます。
- **リスク/リワードレシオ**は、初期リスクに対するエントリー価格とテイクプロフィットターゲットの間の距離を決定します。
- **最大証拠金使用量** は、現在のポートフォリオの株式と比較して、新しい取引で消費できる推定証拠金の量を制限します。
- **トレーリング ストップ**は、ATR ベースのリスク管理が使用されている場合に自動的に有効になります。最新のローソク足の終値を超えずに利益目標に向かってストップを半分に移動し、為替の制約を尊重しながら利益を確定しようとします。

## パラメーター
| パラメータ | デフォルト | 説明 |
|-----------|---------|-------------|
| `RiskPercent` | 1 | 単一取引の最大損失として割り当てられたポートフォリオ資本の割合。 |
| `RiskRewardRatio` | 1.5 | テイクプロフィットターゲットを定義するために初期リスクディスタンスに適用される乗数。 |
| `MaxMarginUsage` | 30 | 証拠金消費の上限を資本に対する割合で表します。 |
| `StopLossPips` | 50 | ATR が無効な場合のストップロス サイズを MetaTrader ピップ単位で修正しました。 |
| `UseAutoSl` | 本当の | 乗数 1.5 で ATR (長さ 14) のストップロス サイズ設定を有効にします。 |
| `CandleType` | 1時間枠 | シグナルと ATR の計算に使用されるローソク足シリーズ。 |

## 実装メモ
- この戦略は、StockSharp の高レベルのサブスクリプション (`SubscribeCandles`) とインジケーター バインディング (`AverageTrueRange`) に依存しています。
- 位置のサイジングは、楽器の音量ステップ、最小および最大音量の制約に合わせて調整されます。
- マージンチェックは、利用可能な商品マージンヒント (`MarginBuy`/`MarginSell`) を再利用し、価格ベースの見積もりにフォールバックします。
- ストップロスとテイクプロフィットのレベルは、ローソク足の高値と安値を監視することによって内部的に強制され、ブローカー全体で一貫した動作が保証されます。
