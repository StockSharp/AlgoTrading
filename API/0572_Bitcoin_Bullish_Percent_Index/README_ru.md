# Стратегия Bitcoin Bullish Percent Index
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует индикатор RSI для приближенной оценки индекса Bitcoin Bullish Percent. Открывает длинную позицию, когда RSI поднимается выше уровня перепроданности, и короткую, когда RSI опускается ниже уровня перекупленности.

## Подробности

- **Условия входа**:
  - **Лонг**: RSI пересекает снизу уровень перепроданности.
  - **Шорт**: RSI пересекает сверху уровень перекупленности.
- **Лонг/Шорт**: Обе стороны.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `RSI Period` = 14
  - `Overbought` = 70
  - `Oversold` = 30
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Среднесрочный
