# ドミナンス・タグクラウド戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このビジュアライゼーション戦略は、暗号資産のドミナンス値のタグクラウドを描画します。各シンボルはそのドミナンスに応じたサイズのボックスを受け取り、重複を避けながらランダムに配置されます。このサンプルはカスタム描画ロジックと複数シンボルの処理を示しており、トレードシグナルは生成しません。

## 詳細

- **エントリー条件**: なし（ビジュアライゼーションのみ）
- **ロング/ショート**: なし
- **エグジット条件**: なし
- **インジケーター**: 主要暗号資産のドミナンス値
- **ストップ**: いいえ
- **カテゴリ**: ビジュアライゼーション
