# Estrategia de Nube de Etiquetas de Dominancia
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia de visualización renderiza una nube de etiquetas con los valores de dominancia de criptomonedas. Cada símbolo recibe un cuadro dimensionado según su dominancia y posicionado aleatoriamente evitando superposiciones. El ejemplo demuestra lógica de dibujo personalizada y procesamiento de múltiples símbolos sin producir señales de trading.

## Detalles

- **Criterios de entrada**: ninguno (solo visualización)
- **Largo/Corto**: ninguno
- **Criterios de salida**: ninguno
- **Indicadores**: valores de dominancia de los principales activos cripto
- **Stops**: no
- **Categoría**: Visualización
