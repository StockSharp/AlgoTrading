# Estratégia de Nuvem de Tags de Dominância
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia de visualização renderiza uma nuvem de tags com os valores de dominância de criptomoedas. Cada símbolo recebe uma caixa dimensionada de acordo com sua dominância e posicionada aleatoriamente, evitando sobreposições. O exemplo demonstra lógica de desenho personalizada e processamento de múltiplos símbolos sem produzir sinais de trading.

## Detalhes

- **Critérios de entrada**: nenhum (somente visualização)
- **Comprado/Vendido**: nenhum
- **Critérios de saída**: nenhum
- **Indicadores**: valores de dominância dos principais ativos cripto
- **Stops**: não
- **Categoria**: Visualização
