# Dominanz-Tag-Cloud-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Visualisierungsstrategie rendert eine Tag-Cloud der Kryptowährungs-Dominanzwerte. Jedes Symbol erhält ein nach seiner Dominanz dimensioniertes Kästchen, das zufällig platziert wird, ohne Überlappungen zu erzeugen. Das Beispiel demonstriert benutzerdefinierte Zeichenlogik und die Verarbeitung mehrerer Symbole ohne Handelssignale zu erzeugen.

## Details

- **Einstiegskriterien**: keine (nur Visualisierung)
- **Long/Short**: keine
- **Ausstiegskriterien**: keine
- **Indikatoren**: Dominanzwerte der wichtigsten Krypto-Assets
- **Stops**: nein
- **Kategorie**: Visualisierung
