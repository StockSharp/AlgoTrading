# Linearer Regressionskanal
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Handelt Rücksetzer, wenn der Preis außerhalb eines linearen Regressionskanals läuft und zum Mittelwert zurückkehrt.

## Details

- **Daten**: Preiskerzen.
- **Einstieg**: Kaufen, wenn der Schlusskurs in einem Aufwärtstrend unter das untere Band fällt; verkaufen, wenn der Schlusskurs in einem Abwärtstrend über das obere Band steigt.
- **Ausstieg**: Schließen, wenn der Preis zur Regressionslinie zurückkehrt.
- **Instrumente**: Beliebige Instrumente.
- **Risiko**: Kanalgrenzen wirken als dynamische Limits.
