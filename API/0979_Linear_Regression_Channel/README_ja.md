# 線形回帰チャネル
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

価格が線形回帰チャネルの外に出て平均に回帰するときの押し目・戻りを取引します。

## 詳細

- **データ**: 価格ローソク足。
- **エントリー**: 上昇トレンド中に終値が下限バンドを下抜けたら買い。下降トレンド中に終値が上限バンドを上抜けたら売り。
- **エグジット**: 価格が回帰ラインに戻ったら決済。
- **銘柄**: 任意の銘柄。
- **リスク**: チャネルの境界が動的な上限として機能します。
