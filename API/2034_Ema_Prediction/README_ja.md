# EMA予測戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

EMA Predictionインジケーターに基づく戦略で、速いEMAと遅いEMAが方向を確認するローソク足でクロスしたときにシグナルを生成します。

速いEMAが弱気ローソク足で遅いEMAを上抜けしたときにロングポジションを開き、ショートポジションをすべて決済します。速いEMAが弱気ローソク足で遅いEMAを下抜けしたときにショートポジションを開き、ロングポジションをすべて決済します。

## 詳細

- **エントリー条件**:
  - ロング: 速いEMAが遅いEMAを上抜けし、ローソク足が陽線である。
  - ショート: 速いEMAが遅いEMAを下抜けし、ローソク足が陰線である。
- **ロング/ショート**: 両方
- **エグジット条件**: 反対のシグナル
- **ストップ**: 固定テイクプロフィットとストップロス
- **デフォルト値**:
  - `CandleType` = 6時間ローソク足
  - `FastPeriod` = 1
  - `SlowPeriod` = 2
  - `StopLossTicks` = 1000
  - `TakeProfitTicks` = 2000
- **フィルター**:
  - カテゴリ: 移動平均クロスオーバー
  - 方向: 両方
  - インジケーター: EMA
  - ストップ: テイクプロフィットとストップロス
  - 複雑さ: 基本
  - 時間軸: 6時間
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
