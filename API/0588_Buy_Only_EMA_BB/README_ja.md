# EMAとBBを使ったロングのみ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格がEMAを上回って終値をつけるとロングポジションを開きます。
初期ストップロスはボリンジャーバンド下限に置かれ、価格が上限バンドを上回って終値をつけるとEMAに移動します。
テイクプロフィットはバンドまでの距離に基づくリワード/リスク比を使って設定されます。
テイクプロフィット達成後、戦略は新しいエントリーを許可する前に価格がEMAを下抜けるのを待ちます。

## 詳細
- **エントリー条件:** アクティブなブロックがなく、オープンポジションがない状態でcloseがEMAを上回る。
- **ロング/ショート:** ロングのみ。
- **エグジット条件:** 価格がストップレベルを下抜けるかテイクプロフィットに達する。
- **ストップ:** 初期ストップは下限バンド、強い動きの後にEMAへ移動。
- **デフォルト値:** EMA長 = 40、バンド偏差 = 0.7、リワード/リスク比 = 3。
