# トレーリングストップEA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はトレーリングストップを適用して既存のポジションを管理します。ティックトレードを監視し、価格が有利な方向に動くにつれてストップレベルを移動させます。市場が反転してトレーリングレベルに到達すると、戦略はポジションを決済します。

## 詳細

- **エントリー**: 戦略はポジションを開かず、すでにポジションが開かれていることを前提とします。
- **ロングロジック**: ロングポジションの場合、価格がトレーリング距離分上昇すると、ストップが価格に追随して上昇します。
- **ショートロジック**: ショートポジションの場合、価格が下落するとストップも下方に移動します。
- **エグジット**: 価格がトレーリングストップに達するとポジションが決済されます。
- **インジケーター**: なし。
- **時間軸**: ティックベース、各トレードに反応。
- **ストップ**: トレーリングストップのみ。

## パラメーター

- `TrailingPoints` — トレーリングストップの距離（価格ステップ単位）。デフォルト: 200。
