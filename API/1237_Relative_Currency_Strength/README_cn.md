# 相对货币强度
[English](README.md) | [Русский](README_ru.md)

Relative Currency Strength 策略将交易货币对与主要货币篮子进行比较。
当交易对强于其他货币的平均值时买入，弱于平均值时卖出。
比较基于自会话开始以来的百分比变化。

## 细节

- **入场条件**：交易对强度高于平均值一定阈值。
- **方向**：双向。
- **出场条件**：强度低于平均值一定阈值。
- **止损**：无。
- **默认值**：
  - `Threshold` = 0.01m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **筛选器**：
  - 类别：趋势
  - 方向：双向
  - 指标：价格变动
  - 止损：无
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
