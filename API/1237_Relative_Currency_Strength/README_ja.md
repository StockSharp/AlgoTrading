# 相対通貨強度
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

相対通貨強度は、通貨ペアを主要通貨のバスケットと比較します。
取引ペアが他の主要通貨の平均を上回る場合に買い、下回る場合に売ります。
比較はセッション開始からの変化率に基づいています。

## 詳細

- **エントリー条件**: メインペアの強度が平均をしきい値以上上回る。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 強度が平均をしきい値以上下回る。
- **ストップ**: なし。
- **デフォルト値**:
  - `Threshold` = 0.01m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: 価格変化
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
