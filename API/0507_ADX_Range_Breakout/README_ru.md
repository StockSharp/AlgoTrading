# ADX Range Breakout Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинные позиции, когда цена закрытия пробивает предыдущий максимум за выбранный период, а значение ADX остаётся ниже заданного порога, что указывает на спокойный рынок. Торговля ведётся только в заданной сессии и ограничена максимальным числом сделок в день. Фиксированный стоп‑лосс в ценовых единицах защищает каждую позицию.

## Подробности

- **Условия входа**: `Close >= previous highest close` и `ADX < threshold` в пределах сессии
- **Длинные/короткие**: Только длинные
- **Условия выхода**: Стоп‑лосс или конец сессии
- **Стопы**: Да
- **Значения по умолчанию**:
  - `AdxPeriod` = 14
  - `HighestPeriod` = 34
  - `AdxThreshold` = 17.5
  - `StopLoss` = 1000
  - `MaxTradesPerDay` = 3
  - `CandleType` = TimeSpan.FromMinutes(30)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Длинные
  - Индикаторы: ADX
  - Стопы: Да
  - Сложность: Начальный уровень
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
