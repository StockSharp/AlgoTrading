# Стратегия Xbug Free V4
[English](README.md) | [中文](README_cn.md)

Стратегия открывает позиции, когда скользящая средняя от медианной цены пересекает саму медиану. Симметричные тейк-профит и стоп-лосс выставляются на фиксированном расстоянии от цены входа.

## Детали

- **Условия входа**:
  - Long: скользящая средняя выше медианной цены и была ниже две свечи назад
  - Short: скользящая средняя ниже медианной цены и была выше две свечи назад
- **Направление**: Оба
- **Условия выхода**:
  - Тейк-профит на расстоянии `StopPoints` от входа
  - Стоп-лосс на расстоянии `StopPoints` в противоположную сторону
- **Стопы**: Да
- **Значения по умолчанию**:
  - `MaPeriod` = 19
  - `StopPoints` = 270
  - `Volume` = 0.1m
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **Фильтры**:
  - Категория: Crossover
  - Направление: Оба
  - Индикаторы: SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Долгосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
