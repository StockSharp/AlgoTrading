# Xbug Free V4戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、中値価格の移動平均が中値価格自体をクロスしたときにポジションを建てます。エントリー価格から固定距離に対称的なテイクプロフィットとストップロスが設置されます。

## 詳細

- **エントリー条件**:
  - ロング: 移動平均が中値価格を上回っており、2本前のローソク足では下回っていた
  - ショート: 移動平均が中値価格を下回っており、2本前のローソク足では上回っていた
- **ロング/ショート**: 両方
- **エグジット条件**:
  - エントリーから`StopPoints`距離上/下でテイクプロフィット
  - 反対側の`StopPoints`距離でストップロス
- **ストップ**: はい
- **デフォルト値**:
  - `MaPeriod` = 19
  - `StopPoints` = 270
  - `Volume` = 0.1m
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **フィルター**:
  - カテゴリ: Crossover
  - 方向: 両方
  - インジケーター: SMA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 長期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
