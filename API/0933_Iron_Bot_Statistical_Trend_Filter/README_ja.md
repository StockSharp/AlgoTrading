# Iron Bot 統計的トレンドフィルター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、フィボナッチ範囲とZ-scoreから計算された統計的トレンドレベルに基づくブレイクアウトをトレードします。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格がトレンドラインと上方トレンドレベルを上抜け、Z-scoreが非負。
  - **ショート**: 価格がトレンドラインと下方トレンドレベルを下抜け、Z-scoreが非正。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - エントリーから`SlRatio`パーセントのストップロス。
  - エントリーから4つのレベル（`Tp1Ratio`–`Tp4Ratio`）のいずれかでテイクプロフィット。
- **ストップ**: はい。
- **デフォルト値**:
  - `ZLength` = 40.
  - `AnalysisWindow` = 44.
  - `HighTrendLimit` = 0.236.
  - `LowTrendLimit` = 0.786.
  - `EmaLength` = 200.
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Z-score, EMA, プライスアクション
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
