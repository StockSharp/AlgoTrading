# 三本足強気の包み足戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、三本足の強気または弱気の包み足パターンを探します。オプションのRSIブレイクアウトエントリー、トレーリングストップ、時間ベースのエグジットをサポートします。

## 詳細

- **エントリー条件**:
  - **ロング**: 強気の足、小さなドージ、強気の包み足。
  - **ショート**: 弱気の足、小さなドージ、弱気の包み足。
- **ロング/ショート**: 両方（ロングのみモード利用可能）。
- **エグジット条件**:
  - トレーリングストップ、逆方向の足のブレイクまたはセッション終了。
- **ストップ**: はい。
- **デフォルト値**:
  - `TrailPerc` = 1.5
  - `ExitHour` = 15
  - `ExitMinute` = 15
  - `RsiLength` = 14
  - `RsiLevel` = 80
  - `StopLossPerc` = 5
