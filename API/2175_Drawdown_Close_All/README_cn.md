# 回撤全平策略
[English](README.md) | [Русский](README_ru.md)

该策略监控账户权益，当从峰值权益的回撤超过指定百分比时，立即关闭所有持仓，可作为在巨大亏损时保护资金的保险。

## 细节

- **入场条件**：无
- **多/空**：都支持
- **出场条件**：回撤 ≥ `MaxDrawdownPercent`
- **止损**：否
- **默认值**：
  - `MaxDrawdownPercent` = 100
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **过滤器**：
  - 类别：风险管理
  - 方向：双向
  - 指标：无
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险级别：取决于当前持仓
