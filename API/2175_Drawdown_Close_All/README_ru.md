# Стратегия Drawdown Close All
[English](README.md) | [中文](README_cn.md)

Стратегия контролирует собственный капитал счёта и закрывает все открытые позиции, когда просадка от максимального значения превышает заданный процент. Используется как страховка для защиты капитала при крупных потерях.

## Детали

- **Условия входа**: Нет
- **Длинные/Короткие позиции**: Обе
- **Условия выхода**: Просадка ≥ `MaxDrawdownPercent`
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `MaxDrawdownPercent` = 100
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Зависит от текущих позиций
