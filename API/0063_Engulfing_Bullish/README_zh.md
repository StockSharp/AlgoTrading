# 看涨吞没形态策略
[English](README.md) | [Русский](README_ru.md)

当一根阳线完全吞没前一根阴线时，往往预示着短期下跌结束并可能启动新的上升动能。本策略利用这一形态，并可选用下跌趋势过滤器，通过统计连续的下跌蜡烛数量来确认空头力量衰竭。

测试表明年均收益约为 76%，该策略在外汇市场表现最佳。

在实盘中，算法跟踪每根新蜡烛及其前一根。如果新蜡烛收盘价高于开盘价且实体包住前一根蜡烛，则触发做多。止损位于形态低点下方以控制风险。

持仓将持续到触发止损或出现其他信号需要手动离场。由于先前的下跌确认会增强形态，本策略避免追逐疲弱的反转。

## 细节

- **入场条件**：阳线吞没前一根阴线，可选下跌趋势确认。
- **多/空**：仅做多。
- **退出条件**：止损或人工离场。
- **止损**：有，位于形态低点下方。
- **默认值**：
  - `CandleType` = 15 分钟
  - `StopLossPercent` = 1
  - `RequireDowntrend` = true
  - `DowntrendBars` = 3
- **过滤条件**：
  - 类别: 形态
  - 方向: 多头
  - 指标: K线形态
  - 止损: 有
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
