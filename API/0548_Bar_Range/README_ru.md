# Стратегия Bar Range
[English](README.md) | [中文](README_cn.md)

Стратегия Bar Range входит в лонг, когда диапазон текущей свечи находится в верхнем процентиле последних значений и свеча закрывается ниже открытия. Позиция закрывается через заданное количество свечей.

## Подробности

- **Условия входа**:
  - Диапазон = High − Low
  - Процентиль диапазона за `LookbackPeriod` ≥ `PercentRankThreshold`
  - Close < Open
- **Условия выхода**: Закрытие позиции через `ExitBars` свечей.
- **Параметры по умолчанию**:
  - `LookbackPeriod` = 50
  - `PercentRankThreshold` = 95
  - `ExitBars` = 1
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Лонг
  - Индикаторы: Percent Rank
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
