# Inside-Candle-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie erkennt Inside-Candles und steigt bei Ausbrüchen aus dem Bereich der Mutterkerze ein. Ausstiege basieren auf einem benutzerdefinierten Risiko/Rendite-Verhältnis.

## Parameter
- **Candle Type** — Zeitrahmen für die Kerzen.
- **RR Ratio** — Risiko/Rendite-Verhältnis für die Zielberechnung.
