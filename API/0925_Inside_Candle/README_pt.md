# Estratégia de Vela Interna
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia detecta velas internas e entra em rompimentos do intervalo da vela mãe. As saídas são baseadas em uma relação risco/recompensa definida pelo usuário.

## Parâmetros
- **Candle Type** — período das velas.
- **RR Ratio** — relação risco/recompensa para o cálculo do alvo.
