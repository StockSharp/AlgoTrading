# Estrategia de Vela Interior
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia detecta velas interiores y entra en rupturas del rango de la vela madre. Las salidas se basan en una relación riesgo/recompensa definida por el usuario.

## Parámetros
- **Candle Type** — marco temporal de las velas.
- **RR Ratio** — relación riesgo/recompensa para el cálculo del objetivo.
