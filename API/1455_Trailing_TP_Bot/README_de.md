# Trailing-TP-Bot
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt SMA-Kreuzungen auf der Long- und Short-Seite. Jede Position definiert einen festen Take-Profit und Stop-Loss. Nachdem das Gewinnziel erreicht ist, kann der Stop nachgezogen werden, um Gewinne zu schützen.

## Details

- **Einstieg**: Schnelle SMA kreuzt die langsame SMA.
- **Ausstieg**: Stop-Loss, Take-Profit oder Trailing-Stop.
- **Indikatoren**: SMA.
- **Richtung**: Beide.
- **Risiko**: Fester Stop-Loss mit optionalem Trailing.
