# トレーリングTPボット
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はSMAクロスオーバーでロングとショートの両方向を取引します。各ポジションには固定のテイクプロフィットとストップロスが設定されます。利益目標に達すると、ストップが追従して利益を保護します。

## 詳細

- **エントリー**: 速いSMAが遅いSMAを交差。
- **エグジット**: ストップロス、テイクプロフィットまたはトレーリングストップ。
- **インジケーター**: SMA。
- **方向**: 両方。
- **リスク**: 固定ストップロスとオプションのトレーリング。
