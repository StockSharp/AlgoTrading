# スーパートレンド
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Supertrendインジケーターに基づく戦略。

テスト結果では、年間平均リターンが約67%であることが示されています。株式市場で最も効果を発揮します。

Super TrendはATRから動的なラインを計算し、サポートとレジスタンスの間で切り替わります。価格がその上を越えるとバイアスが強気になり、下を越えると弱気になります。ラインが反転すると取引が終了します。

この適応型ラインに従うことで、戦略はダマシを最小化しながら持続的な動きを捉えようとします。ストップレベルが価格に追従するため、モメンタムが衰えると利益を確定します。


## 詳細

- **エントリー条件**: ATR、Supertrendに基づくシグナル。
- **ロング/ショート**: 両方の方向。

- **ストップ**: いいえ。
- **デフォルト値**:

  - `Multiplier` = 3.0m
  - `CandleType` = TimeSpan.FromMinutes(5)

  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: ATR, Supertrend

  - 複雑さ: 基本
