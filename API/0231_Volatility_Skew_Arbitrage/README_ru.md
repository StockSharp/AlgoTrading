# Арбитраж волатильности по перекосу

Эта стратегия с использованием опционов отслеживает разницу в подразумеваемой волатильности между двумя страйками. Когда перекос значительно отклоняется от своей исторической средней, открывается позиция в ожидании его возврата.

При «длинном» перекосе покупается опцион с более дешёвой волатильностью и продаётся опцион с дорогой, когда перекос превышает среднее на `Threshold` стандартных отклонений. «Короткий» перекос делает противоположное, когда значение уходит ниже среднего на ту же величину. Позиции закрываются, когда перекос возвращается к среднему уровню.

Стратегия рассчитана на опытных трейдеров, знакомых с опционной оценкой. Стоп‑лосс применяется для защиты от устойчивых изменений ожиданий волатильности.

## Подробности
- **Условия входа**:
  - **Long**: перекос волатильности > среднее + Threshold * StdDev
  - **Short**: перекос волатильности < среднее − Threshold * StdDev
- **Long/Short**: обе стороны.
- **Условия выхода**:
  - **Long**: выход при возвращении перекоса к среднему
  - **Short**: выход при возвращении перекоса к среднему
- **Стопы**: да, процентный стоп‑лосс по опционной позиции.
- **Параметры по умолчанию**:
  - `LookbackPeriod` = 20
  - `Threshold` = 2m
  - `StopLossPercent` = 2m
- **Фильтры**:
  - Категория: Арбитраж
  - Направление: Обе стороны
  - Индикаторы: Перекос волатильности
  - Стопы: Да
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Высокий
