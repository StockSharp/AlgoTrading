# Ichimoku 転換線/基準線クロスストラテジー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Ichimokuインジケーターは完全なトレンドフォローシステムを提供します。このアプローチは、価格が雲（Kumo）に対してどこで取引されているかを見ながら、転換線（Tenkan-sen）が基準線（Kijun-sen）を交差する点に焦点を当てます。雲の上でのブルッシュなクロスはトレンド継続の上昇を示し、雲の下でのベアリッシュなクロスは弱さを示唆します。

テストでは年平均リターン約142%を示しています。株式市場で最も優れたパフォーマンスを発揮します。

稼働中、戦略は各バーでIchimokuのコンポーネントを計算します。転換線が基準線を上抜け、価格が雲の上にある場合、基準線近くにストップを置いたロングトレードが開始されます。雲の下での逆方向のクロスが同様のストップ配置でショートを引き起こします。

システムはストップが当たるかクロスが反転するまでトレードを保持し、雲の方向に沿った持続的な動きを捉えることを目指します。

## 詳細

- **エントリー条件**: 雲（Kumo）に対する価格の位置を伴う転換線/基準線クロス。
- **ロング/ショート**: 両方。
- **エグジット条件**: ストップロスまたは逆方向のクロス。
- **ストップ**: はい、基準線レベル。
- **デフォルト値**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanBPeriod` = 52
  - `CandleType` = 30 minute
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Ichimoku
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: スイング
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
