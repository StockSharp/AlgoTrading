# Angrybird xScalpingn
[English](README.md) | [中文](README_cn.md)

Angrybird xScalpingn — мартингейловая скальпирующая стратегия. Первый вход определяется направлением последнего закрытия и фильтром RSI. Если цена движется против позиции на динамический шаг, рассчитанный по диапазону последних свечей, стратегия добавляет сделку с объемом, умноженным на коэффициент. Все позиции закрываются при развороте CCI или достижении стоп‑лосса/тейк‑профита.

## Подробности

- **Условия входа**: Первый вход следует за направлением последних свечей при выполнении фильтра RSI. Дополнительные сделки открываются при движении цены против позиции на рассчитанный шаг.
- **Long/Short**: Обе стороны.
- **Условия выхода**: Разворот CCI или защитный стоп‑лосс/тейк‑профит.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `Volume` = 0.01
  - `LotExponent` = 2
  - `DynamicPips` = true
  - `DefaultPips` = 12
  - `Depth` = 24
  - `Del` = 3
  - `TakeProfit` = 20
  - `StopLoss` = 500
  - `Drop` = 500
  - `RsiMinimum` = 30
  - `RsiMaximum` = 70
  - `MaxTrades` = 10
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Grid
  - Направление: Обе
  - Индикаторы: RSI, CCI
  - Стопы: Да
  - Сложность: Высокая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Высокий
