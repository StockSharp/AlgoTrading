# ヘッジング付きDCA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、連続する3本のローソク足がEMAの上で終値をつけた後にロングを建て、連続する3本がEMAの下で終値をつけた後にショートを建てます。直近エントリーに対して価格が指定パーセント逆行した場合に追加ポジションを建てます。平均エントリー価格からテイクプロフィットパーセント分価格が動いたときにポジション全体を決済します。

## パラメータ
- ローソク足タイプ
- EMAの長さ
- DCAインターバル %
- テイクプロフィット %
- 初期ポジションサイズ
