# External Signals Strategy Tester
[English](README.md) | [中文](README_cn.md)

Шаблон стратегии, выполняющий сделки по внешним сигналам на покупку и продажу при их пересечении уровня ноль. Поддерживает опциональный разворот позиции, закрытие по противоположному сигналу и процентные уровни тейк-профита, стоп-лосса и перевода в безубыток.

## Детали

- **Условия входа**: Серия сигналов на покупку или продажу пересекает ноль в заданном диапазоне дат.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**: Тейк-профит, стоп-лосс или перевод в безубыток.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `StartDate` = 2024-11-01 00:00:00
  - `EndDate` = 2025-03-31 23:59:00
  - `EnableLong` = true
  - `EnableShort` = true
  - `CloseOnReverse` = true
  - `ReversePosition` = false
  - `TakeProfitPerc` = 2
  - `StopLossPerc` = 1
  - `BreakevenPerc` = 1
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Signals
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
