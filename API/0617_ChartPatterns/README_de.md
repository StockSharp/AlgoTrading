# Chart-Muster-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie erkennt einfache aufsteigende und absteigende Kanäle basierend auf aufeinanderfolgenden Kerzenhochs und -tiefs. Sie kauft, wenn eine Kerze ein höheres Hoch und ein höheres Tief bildet, und verkauft, wenn ein niedrigeres Hoch und ein niedrigeres Tief erscheinen.

## Parameter
- Kerzentyp
