# Estrategia de Patrones de Gráfico
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia detecta canales ascendentes y descendentes simples basados en máximos y mínimos consecutivos de velas. Compra cuando una vela forma un máximo más alto y un mínimo más alto, y vende cuando aparece un máximo más bajo y un mínimo más bajo.

## Parámetros
- Tipo de vela
