# US 30 Daily Breakout Strategy
[English](README.md) | [Русский](README_ru.md)

交易前一天高点或低点的突破，每个突破仅一次交易。

## 详情

- **入场条件**：收盘价高于前一天高点或低于前一天低点，每日仅一次
- **多空方向**：双向
- **出场条件**：固定止盈或止损
- **止损**：有
- **默认值**：
  - `TakeProfitPips` = 50
  - `StopLossPips` = 50
- **过滤器**：
  - 类别：突破
  - 方向：双向
  - 指标：无
  - 止损：有
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
