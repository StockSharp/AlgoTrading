# US 30 Daily Breakout Strategy
[Русский](README_ru.md) | [中文](README_cn.md)

Trades breakouts of the previous day's high or low with single trade per breakout.

## Details

- **Entry Criteria**: Close above previous day high or below previous day low, once per day
- **Long/Short**: Both
- **Exit Criteria**: Fixed take profit or stop loss
- **Stops**: Yes
- **Default Values**:
  - `TakeProfitPips` = 50
  - `StopLossPips` = 50
- **Filters**:
  - Category: Breakout
  - Direction: Both
  - Indicators: None
  - Stops: Yes
  - Complexity: Basic
  - Timeframe: Intraday
  - Seasonality: No
  - Neural networks: No
  - Divergence: No
  - Risk level: Medium
