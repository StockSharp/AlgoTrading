# US 30 Tägliche Ausbruch-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Handelt Ausbrüche aus dem Hoch oder Tief des Vortages mit einem einzigen Trade pro Ausbruch.

## Details

- **Einstiegskriterien**: Schlusskurs über Vortages-Hoch oder unter Vortages-Tief, einmal pro Tag
- **Long/Short**: Beide
- **Ausstiegskriterien**: Fester Take-Profit oder Stop-Loss
- **Stops**: Ja
- **Standardwerte**:
  - `TakeProfitPips` = 50
  - `StopLossPips` = 50
- **Filter**:
  - Kategorie: Ausbruch
  - Richtung: Beide
  - Indikatoren: Keine
  - Stops: Ja
  - Komplexität: Grundlegend
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
