# US 30 Daily Breakout Strategy
[English](README.md) | [中文](README_cn.md)

Торговля пробоями максимума или минимума предыдущего дня, по одной сделке на пробой.

## Детали

- **Условия входа**: закрытие выше максимума или ниже минимума предыдущего дня, не более одной сделки
- **Длинные/Короткие**: Оба направления
- **Условия выхода**: фиксированный тейк-профит или стоп-лосс
- **Стопы**: Да
- **Значения по умолчанию**:
  - `TakeProfitPips` = 50
  - `StopLossPips` = 50
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
