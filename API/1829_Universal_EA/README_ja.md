# Universal EA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MQL4の「Universal_EA」から変換された戦略。

このアルゴリズムはストキャスティクス・オシレーターを使用してエントリーポイントを決定します。
%Kラインが%Dラインを上抜け、両方が売られすぎの閾値を下回っている場合にロングポジションを開きます。%Kが%Dを下抜け、両方が買われすぎの閾値を上回っている場合にショートポジションを開きます。シグナルは確定したローソク足でのみ確認され、ポジションは成行注文で開かれます。

## パラメーター
- **%K Period** – %Kを計算するための基本期間。
- **%D Period** – %Dラインの平滑化期間。
- **Slowing** – %Kに適用される追加の平滑化。
- **Oversold** – これを下回るとき市場が売られすぎと見なされるレベル。
- **Overbought** – これを上回るとき市場が買われすぎと見なされるレベル。
- **Candle Type** – 分析に使用するローソク足の時間軸またはタイプ。
