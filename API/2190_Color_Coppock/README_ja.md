# Color Coppock戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**Color Coppock Strategy** は、改良されたCoppockオシレーターに基づいた取引システムを実装します。このオシレーターは2つのRate of Change（ROC）値を合計し、移動平均で結果を平滑化します。上昇するモメンタムはロングシグナルを生成し、下降するモメンタムはショートシグナルを生成します。

## 動作の仕組み

1. 異なる期間で2つのROC値を計算します。
2. 両方のROC値を合計し、平滑化のためにシンプル移動平均を適用します。
3. 現在のオシレーター値を前の2つの値と比較します：
   - 下落後にオシレーターが上向きに転換した場合、戦略はロングポジションを開くか既存のショートポジションをクローズします。
   - 上昇後にオシレーターが下向きに転換した場合、戦略はショートポジションを開くか既存のロングポジションをクローズします。
4. ポジションの数量は戦略の `Volume` プロパティから取得します。

## パラメーター

| 名前 | 説明 |
|------|------|
| `Roc1Period` | 最初のROC計算の期間。 |
| `Roc2Period` | 2番目のROC計算の期間。 |
| `SmoothingPeriod` | 両ROC値の合計に適用するSMAの期間。 |
| `CandleType` | インジケーター計算に使用するローソク足の種類。 |

## 使用方法

1. 戦略を銘柄に接続し、希望するパラメーターを設定します。
2. 戦略は指定されたローソク足を購読し、完成したローソク足のみを処理します。
3. 取引はデフォルト数量で成行注文を使用して実行されます。

## 注意事項

- 戦略は `SubscribeCandles` や成行注文ヘルパーなど、高レベルAPIコールのみを使用します。
- コード内のすべてのコメントは英語で記述されています。
