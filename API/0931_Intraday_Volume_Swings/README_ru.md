# Стратегия Intraday Volume Swings
[English](README.md) | [中文](README_cn.md)

Стратегия торгует, когда цена входит в области объёма свингов текущего или предыдущего дня.

## Детали

- **Условия входа**:
  - **Лонг**: Цена входит в область верхнего свинга.
  - **Шорт**: Цена входит в область нижнего свинга.
- **Длинные/Короткие**: Обе стороны.
- **Условия выхода**:
  - Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `RegionMustClose` = true
  - `CandleType` = 1 minute
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Объём
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Риск: Средний
