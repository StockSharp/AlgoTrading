# イントラデイ・出来高スイング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、当日または前日の出来高によって定義されたスイング領域に価格が入ったときにトレードします。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格が上方スイング領域に進入する。
  - **ショート**: 価格が下方スイング領域に進入する。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - 逆シグナル。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `RegionMustClose` = true
  - `CandleType` = 1 minute
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: 出来高
  - ストップ: いいえ
  - 複雑さ: 中
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
