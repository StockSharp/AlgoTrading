# Enhanced Doji Candle
[English](README.md) | [中文](README_cn.md)

Стратегия торгует дожи со простыми правилами подтверждения и фиксированным соотношением риск-прибыль. Вход выполняется при появлении дожи, когда текущая или предыдущая свеча подтверждает направление, закрываясь за уровнем открытия при небольших хвостах. Защитные ордера используют стоп-лосс в пунктах и тейк-профит на основе коэффициента риск-прибыль.

## Детали

- **Условия входа**: Дожи (тело <= 30% диапазона). Если свеча бычья с нижней тенью <=1% или предыдущая свеча была бычьей — вход в лонг. Если свеча медвежья с верхней тенью <=1% или предыдущая медвежья — вход в шорт.
- **Длинные/короткие позиции**: Обе.
- **Условия выхода**: Тейк-профит или стоп-лосс, либо закрытие позиции при появлении нового дожи.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `RiskRewardRatio` = 2.0m
  - `StopLossPips` = 5
  - `SmaPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Candlestick
  - Направление: Both
  - Индикаторы: SMA
  - Стопы: Да
  - Сложность: Начальный
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
