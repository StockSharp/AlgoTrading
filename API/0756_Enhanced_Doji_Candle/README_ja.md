# 強化版ドージ・キャンドル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、シンプルな確認ルールと固定リスクリワード管理でドージローソク足を取引します。Dojiが出現し、そのローソク足または前のローソク足が小さいヒゲで始値より先にクローズして方向を確認したときにエントリーします。保護注文にはピップ単位のストップロスとリスクリワード比で決まるテイクプロフィットを使用します。

## 詳細

- **エントリー条件**: Dojiローソク足（実体 <= 値幅の30%）。下ヒゲが<=1%の強気またはひとつ前のローソク足が強気なら、ロング。上ヒゲが<=1%の弱気またはひとつ前のローソク足が弱気なら、ショート。
- **ロング/ショート**: 両方。
- **エグジット条件**: テイクプロフィットもしくはストップロス、またはポジションをクローズする新しいDojiの出現。
- **ストップ**: あり。
- **デフォルト値**:
  - `RiskRewardRatio` = 2.0m
  - `StopLossPips` = 5
  - `SmaPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: Candlestick
  - 方向: 両方
  - インジケーター: SMA
  - ストップ: あり
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
