# Estrategia de Niveles Fibonacci con Criterios Máximo/Mínimo AYNET
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Utiliza un nivel de Fibonacci de una vela de marco temporal superior junto con criterios de precio máximo y mínimo.
Compra cuando el cierre está por encima del mínimo anterior más bajo y el nivel de Fibonacci. Vende cuando el cierre está por debajo del máximo anterior más alto y el mismo nivel.
