# Waddah Attar Win戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はオリジナルのWaddah Attar Winエキスパートアドバイザーを反映しています。現在のbid/askから固定のポイント数でスペースを置いて、買い指値と売り指値の対称的なグリッドを継続的に維持します。市場価格が最後に送信された注文に近づくたびに、戦略は同じ距離にオプションのボリューム増分を付けた新しい指値を積み重ねます。各注文板更新ごとに浮動利益が監視され、口座通貨で設定された利益目標に達すると、すべてのポジションと保留中の注文が決済されます。

## 詳細

- **エントリー条件**:
  - 初期の買い指値をbidより`Step Points`下に、売り指値を同じ距離ask上に配置します。
  - 価格が各サイドの最新注文の5価格ステップ以内に来ると、追加の保留注文が追加されます。
- **ロング/ショート**: 両方、ヘッジグリッド。
- **エグジット条件**:
  - 資本が保存された残高を`Min Profit`上回ったら、すべてのポジションを閉じ、注文をキャンセルします。
- **ストップ**: なし。
- **デフォルト値**:
  - `Step Points` = 20
  - `First Volume` = 0.1
  - `Increment Volume` = 0.0
  - `Min Profit` = 910
- **注意事項**:
  - ロングとショートポジションが共存できるため、ヘッジングポートフォリオで機能します。
  - bid/askの変化に即座に反応するために注文板データを使用します。
