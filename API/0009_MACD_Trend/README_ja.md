# MACDトレンド
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MACDインジケーターに基づく戦略。

テスト結果では、年間平均リターンが約64%であることが示されています。外国為替市場で最も効果を発揮します。

MACDトレンドはMACDラインとシグナルラインのクロスオーバーに反応します。強気のクロスはロングを開始し、弱気のクロスはショートを開始します。逆のクロスまたはストップが取引を終了させます。

移動平均収束拡散インジケーターはモメンタムを測定することで変化する市場にうまく適応します。このアプローチは、インジケーターが明確な強気または弱気のバイアスを維持している間、トレンドのスウィングに乗ることを目指しています。


## 詳細

- **エントリー条件**: MA、MACDに基づくシグナル。
- **ロング/ショート**: 両方の方向。
- **エグジット条件**: 逆シグナルまたはストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `FastEmaPeriod` = 12
  - `SlowEmaPeriod` = 26
  - `SignalPeriod` = 9
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: MA, MACD
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ (5m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
