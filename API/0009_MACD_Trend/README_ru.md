# Тренд по MACD
[English](README.md) | [中文](README_cn.md)

Стратегия основана на индикаторе MACD.

Тестирование показывает среднегодичную доходность около 64%\. Стратегию лучше запускать на рынке Форекс.

MACD Trend реагирует на пересечение линии MACD и сигнальной линии. Бычье пересечение открывает длинную позицию, медвежье — короткую. Противоположное пересечение или стоп закрывают сделку.

Индикатор MACD хорошо адаптируется к меняющимся рынкам, измеряя импульс. Такой подход позволяет ловить трендовые колебания, пока индикатор сохраняет явный бычий или медвежий уклон.

## Детали

- **Критерии входа**: сигналы на основе MA, MACD.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал или стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `FastEmaPeriod` = 12
  - `SlowEmaPeriod` = 26
  - `SignalPeriod` = 9
  - `StopLossPercent` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: MA, MACD
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
