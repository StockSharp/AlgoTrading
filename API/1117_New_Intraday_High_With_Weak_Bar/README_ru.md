# Стратегия New Intraday High With Weak Bar
[English](README.md) | [中文](README_cn.md)

Открывает длинную позицию, когда формируется новый максимум за `HighestLength` свечей и свеча закрывается близко к минимуму. Позиция закрывается при закрытии выше максимума предыдущей свечи.

## Детали

- **Условия входа**:
  - Нет позиции, максимум равен наибольшему максимуму за `HighestLength` свечей и `(close - low)/(high - low) < WeakRatio`.
- **Длинные/Короткие**: Только длинные позиции.
- **Условия выхода**: Закрытие выше максимума предыдущей свечи.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `HighestLength` = 10
  - `WeakRatio` = 0.15
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Long
  - Индикаторы: Highest
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
