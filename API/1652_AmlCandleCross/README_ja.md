# AMLローソク足クロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はAdaptive Market Level（AML）インジケーターに基づいて取引します。
AMLの値が現在のローソク足の実体内にあるときに取引が開始されます：
ローソク足が始値より高く終値し、AMLがその間にある場合、ロングポジションが
開かれます。陰線の場合、逆の条件でショートポジションが開かれます。オプションで、
反対のシグナルが現れたときにポジションを反転させることができます。

## 詳細

- **エントリー条件**:
  - **ロング**: 陽線かつ `open <= AML <= close`。
  - **ショート**: 陰線かつ `open >= AML >= close`。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 有効な場合、逆シグナルでポジションを反転。
- **ストップ**: なし。
- **デフォルト値**:
  - `Fractal` = 70
  - `Lag` = 18
  - `Shift` = 0
  - `UseOpposite` = true
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一（AML）
  - ストップ: いいえ
  - 複雑さ: 中
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
