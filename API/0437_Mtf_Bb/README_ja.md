# マルチタイムフレーム Bollinger Bands戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

プライマリと上位の時間軸の両方にBollinger Bandsを適用します。価格が上位時間軸のバンドを突き抜けた時に取引し、オプションで長期移動平均によりエントリーをフィルタリングします。目標は広義のトレンドに逆らって極値をフェードすることです。

この戦略はロングとショートの両ポジションをサポートします。リスク管理のためにストップロスのパーセンテージを有効にできます。複数の時間軸を使用することで、支配的なマーケット構造に逆らった取引を避けるのに役立ちます。

## 詳細

- **エントリー条件**:
  - **ロング**: 上位時間軸の下部バンドを下回り、MAフィルター（有効な場合）を上回って終値が付く。
  - **ショート**: 上位時間軸の上部バンドを上回り、MAフィルター（有効な場合）を下回って終値が付く。
- **エグジット条件**:
  - ロング: 現在の時間軸の上部バンドを上回って価格が終値を付ける。
  - ショート: 現在の時間軸の下部バンドを下回って価格が終値を付ける。
- **インジケーター**:
  - 2つの時間軸のBollinger Bands (長さ20、乗数2)
  - オプションのEMAフィルター (期間200)
- **ストップ**: StartProtectionによるオプションのストップロス (%-ベース)。
- **デフォルト値**:
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
  - `UseMaFilter` = False
  - `MaLength` = 200
  - `SLPercent` = 2
- **フィルター**:
  - MTFコンテキストとの逆張り
  - 時間軸: メイン5m、MTF 60m デフォルト
  - インジケーター: Bollinger Bands、EMA
  - ストップ: オプション
  - 複雑さ: 中程度
