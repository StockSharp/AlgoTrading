# Фильтр диапазона
[English](README.md) | [中文](README_cn.md)

Стратегия фильтра диапазона с реалистичным расчетом диапазона и фиксированными уровнями риска/прибыли.

Используется сглаженный диапазон для построения динамических полос вокруг цены. Сделки открываются при пробое цены выше или ниже этих полос. Управление риском осуществляется через фиксированный стоп-лосс и тейк-профит.

## Детали

- **Критерии входа**: цена пробивает полосы фильтра диапазона.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: стоп-лосс или тейк-профит.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `SamplingPeriod` = 100
  - `RangeMultiplier` = 3
  - `RiskPoints` = 50
  - `RewardPoints` = 100
  - `MaxTradesPerDay` = 5
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Range filter
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
