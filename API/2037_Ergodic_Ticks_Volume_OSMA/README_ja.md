# Ergodic Ticks出来高OSMA戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この戦略はMQL5エキスパート"Exp_Ergodic_Ticks_Volume_OSMA"をStockSharpに適応させたものです。オリジナルのエキスパートはティック出来高のモメンタムを評価するカスタムインジケーターを使用しています。このバージョンではカスタムインジケーターをMACDヒストグラムで近似しています。

この戦略はヒストグラムの連続的な増加または減少を探します：
- 2回連続して上昇するとロングエントリーが発動し、ショートポジションを決済します。
- 2回連続して下落するとショートエントリーが発動し、ロングポジションを決済します。

戦略開始時に既存のポジションとの競合を避けるため`StartProtection()`が使用されます。

## パラメーター
- `FastLength` – MACDの速いEMA期間。デフォルト: 12。
- `SlowLength` – MACDの遅いEMA期間。デフォルト: 26。
- `SignalLength` – MACDのシグナルEMA期間。デフォルト: 9。
- `CandleType` – ローソク足の時間軸、デフォルトは8時間。

## トレードロジック
1. 選択した`CandleType`のローソク足を購読する。
2. 完成した各ローソク足のMACDヒストグラムを計算する。
3. ヒストグラムが2本連続して増加した場合、ショートを決済して買いに入る。
4. ヒストグラムが2本連続して減少した場合、ロングを決済して売りに入る。
5. 新しいローソク足ごとに処理を継続する。
