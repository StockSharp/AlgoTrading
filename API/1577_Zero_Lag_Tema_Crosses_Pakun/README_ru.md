# Zero-Lag TEMA Crosses Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия на пересечении нулелаговых TEMA. Используются последние максимумы и минимумы для стопов и цели по соотношению риск/прибыль.

## Детали

- **Критерии входа**: пересечение быстрой и медленной TEMA.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: стоп по экстремуму или цель по коэффициенту.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `Lookback` = 20
  - `FastPeriod` = 69
  - `SlowPeriod` = 130
  - `RiskReward` = 1.5m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: TEMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
