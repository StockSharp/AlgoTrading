# 零滞后TEMA交叉策略
[English](README.md) | [Русский](README_ru.md)

该策略基于零滞后三重指数均线的交叉，使用最近的高点和低点设置止损，并按照风险回报比设置目标。

## 详情

- **入场条件**：快 TEMA 上穿/下穿慢 TEMA。
- **多空方向**：双向。
- **退出条件**：近期极值止损或按照比例止盈。
- **止损**：是。
- **默认值**：
  - `Lookback` = 20
  - `FastPeriod` = 69
  - `SlowPeriod` = 130
  - `RiskReward` = 1.5m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**：
  - 类型: 趋势
  - 方向: 双向
  - 指标: TEMA
  - 止损: 是
  - 复杂度: 中等
  - 时间框架: 日内 (5m)
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
