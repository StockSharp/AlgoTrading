# Стратегия поиска чаш
[English](README.md) | [中文](README_cn.md)

Эта стратегия ищет в ценах закругленные формации типа "чаша". После завершения чаши и пробоя кромки открывается позиция в направлении пробоя.

Тесты показывают среднегодовую доходность около 47%. Лучше всего работает на акциях.

Покупка выполняется при пробое бычьей чаши, продажа — при пробое медвежьей перевернутой чаши. Позиции защищаются стоп-лоссом.

## Детали

- **Условия входа**: Формирование чаши и пробой кромки.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Разворот цены или срабатывание стоп-лосса.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `Lookback` = 150
  - `WidthPercent` = 5m
  - `StopLossPercent` = 1m
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Long/Short
  - Индикаторы: Price Action
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
