# 杯形策略
[English](README.md) | [Русский](README_ru.md)

该策略在价格中寻找圆弧形“杯形”结构。当价格突破杯口时，根据方向开多或开空。

测试显示年化收益约为47%，在股票市场表现最佳。

策略在看涨杯形突破时买入，在倒置杯形跌破时卖出，并使用止损保护持仓。

## 细节

- **入场条件**：出现杯形并突破杯口。
- **多空**：双向。
- **出场条件**：价格反向或触发止损。
- **止损**：有。
- **默认值**：
  - `Lookback` = 150
  - `WidthPercent` = 5m
  - `StopLossPercent` = 1m
  - `CandleType` = TimeSpan.FromMinutes(15)
- **过滤器**：
  - 类别：形态
  - 方向：多/空
  - 指标：价格行为
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：有
  - 风险等级：中等
