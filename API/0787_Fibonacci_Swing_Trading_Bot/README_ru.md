# Бот на основе уровней Фибоначчи
[English](README.md) | [中文](README_cn.md)

Стратегия использует уровни коррекции Фибоначчи для поиска свинговых движений.

Бот вычисляет уровни 0.618 и 0.786 от диапазона последних 50 свечей и открывает позиции при пробое этих уровней. Управление риском осуществляется через параметры стоп-лосса и соотношения риск/прибыль.

## Детали

- **Критерии входа**: ценовое действие и уровни Фибоначчи.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: стоп-лосс или тейк-профит.
- **Стопы**: да, процентные.
- **Значения по умолчанию**:
  - `FiboLevel1` = 0.618
  - `FiboLevel2` = 0.786
  - `RiskRewardRatio` = 2
  - `StopLossPercent` = 1
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Свинг
  - Направление: Оба
  - Индикаторы: Фибоначчи, Дончиан
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: 4ч
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
