# Alpha RSI ブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

SMAとRSIを使用して、価格がSMAを上回っている時にRSIが閾値を上抜けするクロスを捉えます。RSIがテイクプロフィットレベルに達した後、トレーリングストップが起動します。RSIストップロス、テイクプロフィット到達、またはトレーリングストップで決済します。

## 詳細

- **データ**: 価格ローソク足。
- **エントリー**: RSIがエントリーレベルを上抜けし、価格がSMAを上回っている時に買い。
- **エグジット**: RSIがストップレベルを下回る、RSIがテイクプロフィットに達する、または起動後に価格がトレーリングストップを下回る。
- **銘柄**: 任意。
- **リスク**: RSIベースのストップロスと利益後のトレーリングストップ。
