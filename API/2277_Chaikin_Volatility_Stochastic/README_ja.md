# Chaikinボラティリティストキャスティック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、トレンドの反転を捉えるためにChaikinボラティリティ値にストキャスティックオシレーターを適用します。各ローソク足の高値・安値の範囲をEMAで平滑化し、次にストキャスティック計算で正規化し、最後に加重移動平均でさらに平滑化します。

平滑化されたオシレーターが上昇後に下向きに転換すると、ロングポジションを開きショートポジションをクローズします。オシレーターが下落後に上向きに転換すると、ショートポジションを開きロングポジションをクローズします。

## パラメーター
- **Candle Type**: ローソク足サブスクリプションの時間軸。
- **EMA Length**: 高値・安値の範囲の平滑化期間。
- **Stochastic Length**: ストキャスティック計算のルックバック期間。
- **WMA Length**: オシレーターを平滑化するための加重移動平均期間。
- **Enable Longs / Enable Shorts**: 許可する取引方向の切り替え。

## インジケーター
- ExponentialMovingAverage
- Highest と Lowest
- WeightedMovingAverage

## 取引ルール
- **ロングエントリー**: オシレーターが上昇していて下向きに転換。
- **ショートエントリー**: オシレーターが下落していて上向きに転換。
- シグナル変化時に逆方向のポジションをクローズ。
