# EM VOL 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はピボットベースのサポートとレジスタンスレベル周辺のブレイクアウトを取引します。
前日の高値と安値にATRバッファを加減してエントリートリガーを形成します。
ADXインジケーターが低ボラティリティ環境を示しているときのみ取引が行われます。

## ロジック

1. 前のバーのピボットを計算し、ATRを加減してレジスタンスとサポートを取得します。
2. ADXが閾値を下回り、価格がレジスタンスを上回って終値を付けた場合、ロングポジションに入ります。
3. 価格がサポートを下回って終値を付けた場合、ショートポジションに入ります。
4. エントリー後、保護ストップとテイクプロフィット注文が配置されます。
5. 利益が指定レベルに達したら、トレーリングストップが保護ストップを引き締めることができます。

## パラメーター

- `TakeProfit` — 価格ステップでのテイクプロフィット距離。
- `StopLoss` — 価格ステップでのストップロス距離。
- `AtrPeriod` — ATRインジケーター期間。
- `AdxPeriod` — ADXインジケーター期間。
- `AdxThreshold` — 取引を許可するADXの最大値。
- `TrailStart` — トレーリングストップ開始前に必要な利益。
- `TrailStep` — トレーリングストップの距離。
- `CandleType` — 計算に使用する時間軸。

## 使用インジケーター

- Average True Range
- Average Directional Index
