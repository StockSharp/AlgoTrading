# Time Trader Strategy
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинные и/или короткие позиции в заданное время суток с фиксированными уровнями стоп-лосса и тейк-профита. Подходит для тестирования входов по времени без использования индикаторов.

## Детали

- **Условия входа**: Срабатывание по времени на указанном часе и минуте.
- **Длинные/Короткие**: Обе стороны (настраивается).
- **Условия выхода**: Защитные стоп или цель.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `TradeHour` = 0
  - `TradeMinute` = 0
  - `AllowBuy` = true
  - `AllowSell` = true
  - `TakeProfitTicks` = 20
  - `StopLossTicks` = 20
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Другое
  - Направление: Обе
  - Индикаторы: Нет
  - Стопы: Фиксированные
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
