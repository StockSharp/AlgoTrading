# BTC-Trading-Roboter
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Ausbruch-Strategie auf Basis des Donchian-Kanals. Eröffnet eine Long-Position, wenn der Preis das obere Band überschreitet, und eine Short-Position, wenn der Preis unter das untere Band fällt. Ein Trailing-Stop und ein Take-Profit werden als Prozentsätze des aktuellen Preises berechnet.

## Parameter
- Candle Type
- Bars N
- TP %
- SL %
- Trail %
- Start Hour
- End Hour
