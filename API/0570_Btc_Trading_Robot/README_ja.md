# BTC トレーディングロボット
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ドンチャンチャネルを使ったブレイクアウト戦略です。価格が上限バンドを超えるとロングポジションを建て、下限バンドを下回るとショートポジションを建てます。トレーリングストップとテイクプロフィットは現在価格のパーセンテージとして計算されます。

## パラメーター
- Candle Type
- Bars N
- TP %
- SL %
- Trail %
- Start Hour
- End Hour
