# Robô de Trading BTC
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Estratégia de rompimento usando o canal Donchian. Abre uma operação comprada quando o preço ultrapassa a banda superior e uma operação vendida quando o preço cai abaixo da banda inferior. Um trailing stop e um take-profit são calculados como percentuais do preço atual.

## Parâmetros
- Candle Type
- Bars N
- TP %
- SL %
- Trail %
- Start Hour
- End Hour
