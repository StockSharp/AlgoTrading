# Robot de Trading BTC
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Estrategia de ruptura que usa el canal Donchian. Abre una operación larga cuando el precio supera la banda superior y una operación corta cuando el precio cae por debajo de la banda inferior. Un trailing stop y un take-profit se calculan como porcentajes del precio actual.

## Parámetros
- Candle Type
- Bars N
- TP %
- SL %
- Trail %
- Start Hour
- End Hour
