# ADX System
[English](README.md) | [中文](README_cn.md)

Стратегия **ADX System** торгует на основе индикатора Average Directional Index и его компонент +DI и -DI. Позиция открывается, когда ADX растёт, а соответствующая направленная линия пересекает ADX сверху. Для защиты используются фиксированные уровни тейк-профита и стоп-лосса, а также трейлинг-стоп.

## Подробности

- **Условия входа**
  - ADX растёт (предыдущее значение меньше текущего).
  - **Лонг**: предыдущий +DI ниже предыдущего ADX и текущий +DI выше текущего ADX.
  - **Шорт**: предыдущий -DI ниже предыдущего ADX и текущий -DI выше текущего ADX.
- **Условия выхода**
  - Противоположный сигнал от ADX и линий DI.
  - Срабатывание трейлинг-стопа.
  - Достижение тейк-профита или стоп-лосса.
- **Направление**: Лонг и шорт.
- **Стопы**: Фиксированный стоп-лосс, тейк-профит и трейлинг-стоп в абсолютных величинах.
- **Значения по умолчанию**:
  - `AdxPeriod` = 14
  - `TakeProfit` = 15
  - `StopLoss` = 100
  - `TrailingStop` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: ADX, +DI, -DI
  - Стопы: Да
  - Сложность: Начальная
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
