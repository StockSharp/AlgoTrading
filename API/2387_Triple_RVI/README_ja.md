# Triple RVI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は3つの異なる時間軸で**Relative Vigor Index（RVI）**を使用して取引します。より長期のRVIトレンドはフィルターとして機能し、最短の時間軸はエントリーに使用されます。短期RVIがシグナルラインを下に抜けながら両方の上位時間軸が強気のままのときにロングポジションを建てます。短期RVIがシグナルラインを上に抜けながら両方の上位時間軸が弱気のときにショートポジションを建てます。いずれかの時間軸が現在のポジションに対してトレンド変化を示すとポジションをクローズします。

## パラメーター
- **RviPeriod** – RVIの計算期間。
- **CandleType1** – 最上位RVIフィルターの時間軸。
- **CandleType2** – 中間RVIフィルターの時間軸。
- **CandleType3** – エントリーシグナルが生成される取引時間軸。
- **Volume** – 成行注文に使用する注文サイズ。

## 注記
- 完成したローソク足のみ処理されます。
- 戦略はStockSharpの高レベルAPIを使用しています。
- デフォルトの時間軸は30分、15分、5分足に対応しています。
