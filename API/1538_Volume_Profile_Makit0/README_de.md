# Volumen-Profil-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese vereinfachte Volumen-Profil-Strategie verfolgt das Sitzungshoch, -tief und den Point of Control, der durch den Preis der Kerze mit dem höchsten Volumen definiert wird. Die Strategie kauft, wenn der Preis über dem Point of Control liegt, und verkauft, wenn er darunter liegt. Positionen werden geschlossen, wenn der Preis zum mittleren Sitzungsniveau zurückkehrt.

## Parameter
- **Candle Type** – Zeitrahmen der Eingabekerzen.
