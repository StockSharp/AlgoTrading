# 出来高プロファイル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この簡略化された出来高プロファイル戦略は、セッションの高値・安値と最大出来高を持つローソク足の価格によって定義されるポイント・オブ・コントロールを追跡します。価格がポイント・オブ・コントロールより上にある場合に買い、下にある場合に売りを行います。価格がセッションの中間レベルに戻ったときにポジションを閉じます。

## パラメーター
- **Candle Type** – 入力ローソク足の時間軸。
