## 概要
ブル対メドベド戦略は、MetaTrader 4 エキスパート *Bull_vs_Medved.mq4* を StockSharp に変換したものです。システムは次のことを試みます
あらかじめ定義された5分間に6回の保留中の指値注文を発注することで、強い強気または弱気の衝動の中で引き戻しに入る
ウィンドウは取引日全体に広がります。 StockSharp バージョンでは、ウィンドウごとに 1 回だけ取引するという考えが維持され、古いものはキャンセルされます
保留中の注文を処理し、シグナルローソクの実体サイズを使用して動的なストップロスとテイクプロフィットの距離を導き出します。

## 取引ロジック

2. 最後の 2 つの完了したローソク足を維持します。つまり、現在のローソク足 (`shift1`)、前のローソク足 (`shift2`)、そしてローソク足
その前に (`shift3`) は、MetaTrader で使用される `Close[1..3]` 参照を複製します。

   - **強気**: `shift3` は `shift2` の始値を上回って終了します。`shift2` の本体は少なくとも 10 ブローカー ポイントであり、本体は
`shift1` は少なくとも `CandleSizePoints` ポイントです。 `IsBadBull` が false (ロングボディが 3 つ連続) の場合、購入制限を設定します。
   - **クールブル**: `shift2` は最低 20 ポイントのプルバックであり、`shift1` の始値を下回って終了し、その後、`shift1` を上回って終了します。
`shift2` は、しきい値の少なくとも 40% の本文で開きます。購入制限を設けます。

4. 買い指値は `ask - BuyIndentPoints * PriceStep`、売り指値は `bid + SellIndentPoints * PriceStep` に設定されます。一つだけ
未決注文またはポジションが一度に存在する可能性があるため、取引がすでにアクティブである場合、戦略は新しいシグナルをスキップします。

## パラメーター
| 名前 | 種類 | デフォルト | 説明 |
| --- | --- | --- | --- |
| `OrderVolume` | `decimal` | `0.1` | 各指値注文に使用されるボリューム。 |
| `CandleSizePoints` | `decimal` | `75` | シグナルローソク足の最小強気/弱気本体サイズ (ブローカーポイント単位)。 |
| `StopLossMultiplier` | `decimal` | `0.8` | 停止距離を構築するためにシグナルキャンドル本体に適用される乗数。 |
| `TakeProfitMultiplier` | `decimal` | `0.8` | ターゲット距離を構築するためにシグナルキャンドル本体に適用される乗数。 |
| `BuyIndentPoints` | `decimal` | `16` | 買い指値を設定するときにアスクから差し引かれるブローカー ポイントの数。 |
| `SellIndentPoints` | `decimal` | `20` | 売り指値を設定するときに入札に追加されるブローカー ポイントの数。 |
| `EntryWindowMinutes` | `int` | `5` | 各セッションの所要時間 (分単位)。 |
| `CandleType` | `DataType` | 5分キャンドル | 戦略的に加工されたキャンドルシリーズ。 |
| `StartTime0..5` | `TimeSpan` | `00:05`, `04:05`, `08:05`, `12:05`, `16:05`, `20:05` | 各取引ウィンドウの開始時間。 |
