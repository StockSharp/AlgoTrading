# Pivot Point Supertrend
[English](README.md) | [中文](README_cn.md)

Стратегия, объединяющая pivot points и основанный на ATR Supertrend для ловли разворотов тренда.

Тестирование показывает среднегодичную доходность около 65%. Лучшие результаты на фондовом рынке.

Pivot points формируют динамическую центральную линию. ATR задаёт верхние и нижние полосы, которые следуют за ценой. Когда тренд меняется, стратегия открывает позицию в новом направлении.

## Детали

- **Критерии входа**: сигналы на основе pivot points и ATR Supertrend.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `PivotPeriod` = 2
  - `AtrFactor` = 3m
  - `AtrPeriod` = 10
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Pivot Points, ATR
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
