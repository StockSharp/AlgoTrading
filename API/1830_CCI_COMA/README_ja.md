# CCI COMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

コモディティ・チャネル・インデックス（CCI）と複数の時間軸の移動平均を使用して、主要なトレンドに従います。

## 詳細

- **データ**: 複数の時間軸からの価格ローソク足。
- **エントリー**: CCIがゼロより上、RSIが50より上、ローソク足が始値より高く引け、監視しているすべての時間軸が上昇トレンドを示している場合にロング。逆の条件でショート。
- **エグジット**: 反対のシグナルが出たときにポジションを閉じる。
- **銘柄**: 任意の銘柄。
- **リスク**: 明示的なストップロスまたはテイクプロフィットなし。
