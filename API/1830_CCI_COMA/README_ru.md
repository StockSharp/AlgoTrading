# Стратегия CCI COMA
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Использует индикатор CCI и скользящие средние нескольких таймфреймов для следования текущему тренду.

## Подробности

- **Данные**: ценовые свечи с разных таймфреймов.
- **Вход**: лонг, когда CCI выше нуля, RSI выше 50, свеча закрывается выше открытия и все отслеживаемые таймфреймы показывают восходящий тренд; шорт при обратных условиях.
- **Выход**: позиция закрывается при обратном сигнале.
- **Инструменты**: любые инструменты.
- **Риск**: стоп-лосс и тейк-профит не используются.
