# Стратегия High Low Breakout Statistical Analysis
[English](README.md) | [中文](README_cn.md)

Торгует пробои уровней High или Low выбранного таймфрейма. Вход в лонг или шорт определяется параметром, позиция закрывается после фиксированного числа баров.

## Детали

- **Критерии входа**: пересечение закрытием уровня High или Low.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал или по истечении HoldingPeriod баров.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `EntryOption` = LongAtHigh
  - `TimeframeOption` = Daily
  - `HoldingPeriod` = 5
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: High, Low
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
