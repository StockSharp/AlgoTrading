# Trades To CSV 策略
[English](README.md) | [Русский](README_ru.md)

该策略展示了如何在基于指标信号进行交易时，将已平仓交易信息导出到 CSV 文件。当 CCI 指标和 MACD 直方图落入预设范围时，策略开多或开空。出现反向信号或达到盈利/亏损目标后，仓位被关闭，交易记录追加到 CSV 文件中。

CSV 文件包含交易方向、盈亏、票据编号、开仓价、平仓价、开仓时间、平仓时间、品种和数量。

## 详情

- **入场条件**：
  - **多头**：`CCI` 在 -125 与 -42 之间且 `MACD` 直方图在 -0.00114 与 0.00038 之间。
  - **空头**：`CCI` 在 125 与 208 之间且 `MACD` 直方图在 -0.00038 与 0.00190 之间。
- **出场条件**：
  - 反向信号。
  - 触发止盈或止损。
- **止损**：使用绝对值设置止盈和止损。
- **默认值**：
  - `CandleType` = 1 分钟K线
  - `TakeProfit` = 50
  - `StopLoss` = 50
  - `Volume` = 0.1
  - `CciPeriod` = 14
  - `MacdFastPeriod` = 12
  - `MacdSlowPeriod` = 26
  - `MacdSignalPeriod` = 9
  - `FileName` = `myfilename.csv`
- **过滤器**：
  - 类别：指标
  - 方向：双向
  - 指标：CCI、MACD
  - 止损：是
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
