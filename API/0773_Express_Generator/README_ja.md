# Express Generator戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、RSIとMACDシグナルによって確認された移動平均クロスオーバーでトレードします。ポジションサイズはATRベースのボラティリティ係数と固定リスク割合を使用します。ピップ単位のトレーリングストップがエグジットを管理します。

## 詳細

- **エントリー ロング**: 短期SMAが長期SMAを上抜け、RSIが買われすぎ水準以下、MACDラインがシグナルを上抜け。
- **エントリー ショート**: 短期SMAが長期SMAを下抜け、RSIが売られすぎ水準以上、MACDラインがシグナルを下抜け。
- **エグジット**: ピップ単位のトレーリングストップ。
- **ポジションサイジング**: 資産に対するリスク%をATRで調整したストップ距離で除算。
- **インジケーター**: SMA、RSI、MACD、ATR。
- **方向**: 両方向。
