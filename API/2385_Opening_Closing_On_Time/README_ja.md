# 指定時間での開始・終了戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

指定した時刻にマーケットポジションを建て、別の事前定義した時刻にクローズする、シンプルな時間ベースの戦略です。方向（買いまたは売り）と注文量は設定可能です。この例では、インジケーターや追加フィルターを使用せずにスケジュール取引実行を実証します。

## 詳細

- **エントリー条件**:
  - **ロング**: `Is Buy` が有効なときに `Open Time` で。
  - **ショート**: `Is Buy` が無効なときに `Open Time` で。
- **ロング/ショート**: `Is Buy` に応じて両方向。
- **エグジット条件**:
  - 損益に関係なく `Close Time` でポジションをクローズ。
- **ストップ**: なし。
- **デフォルト値**:
  - `Open Time` = 13:00。
  - `Close Time` = 13:01。
  - `Volume` = 1。
  - `Is Buy` = true。
  - `Candle Type` = 1分足。
- **フィルター**:
  - カテゴリ: 時間
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
