# 按时间开仓和平仓策略
[English](README.md) | [Русский](README_ru.md)

一个简单的时间型策略，在指定的时间开仓，并在另一个预设时间平仓。可以配置交易方向（买入或卖出）和下单量。该示例展示了不使用指标或其他过滤条件的定时交易执行。

## 细节

- **入场条件**：
  - **多头**：在 `Open Time` 且 `Is Buy` 为真时。
  - **空头**：在 `Open Time` 且 `Is Buy` 为假时。
- **多空方向**：根据 `Is Buy`，可做多或做空。
- **出场条件**：
  - 在 `Close Time` 平仓，与盈亏无关。
- **止损**：无。
- **默认值**：
  - `Open Time` = 13:00。
  - `Close Time` = 13:01。
  - `Volume` = 1。
  - `Is Buy` = true。
  - `Candle Type` = 1 分钟。
- **过滤器**：
  - 类别: 时间
  - 方向: 双向
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 低
