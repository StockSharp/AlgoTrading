# 非重绘 Renko 仿真策略
[English](README.md) | [Русский](README_ru.md)

使用收盘价模拟 Renko 砖块，并在方向变化时交易，不发生重绘。

## 细节

- **入场条件**：
  - 新砖形成后，前砖方向和价格序列显示上行延续时做多。
  - 反向序列时做空。
- **多/空**：同时支持。
- **出场条件**：砖块方向反转时平仓。
- **止损**：无。
- **默认值**：
  - `BrickSize` = 3
  - `CandleType` = TimeSpan.FromMinutes(1)
- **过滤器**：
  - 类别：Pattern
  - 方向：双向
  - 指标：无
  - 止损：无
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中
