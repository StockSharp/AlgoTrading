# 再描画なし Renko エミュレーション戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

終値を使ってRenkoブロックをエミュレートし、再描画なしでパターン転換に基づいてトレードする。

## 詳細

- **エントリー条件**:
  - 新しいブロックが形成された後、前ブロックの方向と価格シーケンスが上昇継続を示す場合にロング。
  - 逆のシーケンスでショート。
- **ロング/ショート**: 両方。
- **エグジット条件**: ブロックの方向が反転した際にポジションを決済。
- **ストップ**: いいえ。
- **デフォルト値**:
  - `BrickSize` = 3
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 中級
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
