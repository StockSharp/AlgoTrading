# アダプティブ RSI 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

アダプティブ RSI 戦略は、Relative Strength Index から平滑化係数を導出します。RSI がニュートラルレベルの 50 から乖離すると係数が増加し、アダプティブ RSI が価格をより密に追跡します。50 付近では係数が小さくなり、曲線が滑らかになります。アダプティブ RSI が上向きに転じたときにロングポジションが建てられ、下向きに転じたときにショートポジションが建てられます。

## 詳細

- **エントリー条件**:
  - アダプティブ RSI が前の値を上回る。
  - アダプティブ RSI が前の値を下回る。
- **ロング/ショート**: ロングとショートの両方のトレード。
- **エグジット条件**:
  - 逆シグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 14
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
