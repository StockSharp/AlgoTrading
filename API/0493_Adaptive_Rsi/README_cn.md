# 自适应RSI策略
[English](README.md) | [Русский](README_ru.md)

自适应RSI策略根据相对强弱指数计算平滑系数。当RSI偏离50的中性水平时，系数增大，使自适应RSI更加贴近价格；当RSI接近50时，系数减小，曲线更加平滑。自适应RSI向上转折时开多仓，向下转折时开空仓。

## 详情

- **入场条件**：
  - 自适应RSI上穿其前一个值。
  - 自适应RSI下破其前一个值。
- **方向**：多空双向。
- **出场条件**：
  - 反向信号。
- **止损**：无。
- **默认参数**：
  - `Length` = 14
- **过滤器**：
  - 类别：动量
  - 方向：双向
  - 指标：RSI
  - 止损：否
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：低
