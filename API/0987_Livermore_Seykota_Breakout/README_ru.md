# Пробой Ливермора–Сейкоты
[English](README.md) | [中文](README_cn.md)

Система пробоя, сочетающая pivot-точки Ливермора с трендовым фильтром Сейкоты и выходами на основе ATR.

Тестирование показывает среднегодичную доходность около 87%. Стратегия лучше всего работает на рынке акций.

Стратегия ищет пробой последней pivot-точки, подтверждая направление тренда согласованностью EMA и силой объема. Выходы выполняются по ATR-стопу или трейлингу.

## Подробности

- **Критерии входа**: Цена пробивает последний pivot при подтвержденном тренде и объеме.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: ATR-стоп или трейлинг.
- **Стопы**: ATR-стоп и трейлинг.
- **Параметры по умолчанию**:
  - `MainEmaLength` = 50
  - `FastEmaLength` = 20
  - `SlowEmaLength` = 200
  - `PivotLength` = 3
  - `AtrLength` = 14
  - `StopAtrMultiplier` = 3
  - `TrailAtrMultiplier` = 2
  - `VolumeSmaLength` = 20
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: EMA, Объем, ATR, Pivot
  - Стопы: ATR-трейлинг
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (15м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
