# Стратегия Triangle Breakout с TP SL и EMA фильтром
[English](README.md) | [中文](README_cn.md)

Определяет треугольник по пивотам и открывает длинные позиции при пробое вверх. Дополнительно можно требовать цену выше EMA20 и EMA50. Стоп и тейк рассчитываются в процентах.

## Детали

- **Критерий входа**: закрытие выше верхней линии треугольника с опциональным фильтром EMA20/EMA50
- **Длин/Шорт**: только лонг
- **Критерий выхода**: процентный тейк-профит или стоп-лосс
- **Стопы**: да
- **Значения по умолчанию**:
  - `PivotLength` = 5
  - `TakeProfitPercent` = 3
  - `StopLossPercent` = 1.5
  - `UseEmaFilter` = true
  - `EmaFast` = 20
  - `EmaSlow` = 50
  - `CandleType` = 1 час
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Лонг
  - Индикаторы: EMA, Pivot
  - Стопы: Да
  - Сложность: Продвинутая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
