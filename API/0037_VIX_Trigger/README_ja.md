# VIX Trigger
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)
 
VIX Triggerはボラティリティ指数の変化に反応します。上昇するVIXは恐怖と原資産における潜在的なリバーサルを示します。この戦略はVIXの方向を移動平均に対する価格と比較します。

テストでは年間平均リターン約148%が示されています。外国為替市場で最も優れたパフォーマンスを発揮します。

VIXが上昇し価格が移動平均を下回ると、回復を期待して買いを入れます。逆に、価格が平均を上回っている状態でVIXが上昇すると、ショートポジションを取ります。

VIXが下落するかストップロスの割合に達するとポジションをクローズします。

## 詳細

- **エントリー条件**: VIXが上昇し、MAに対する価格がロングまたはショートを引き起こす。
- **ロング/ショート**: 両方向。
- **エグジット条件**: VIXが下落するかストップ。
- **ストップ**: はい。
- **デフォルト値**:
  - `MAPeriod` = 20
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: Contrarian
  - 方向: 両方
  - インジケーター: VIX, MA
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
