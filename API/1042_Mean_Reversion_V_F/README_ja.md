# 平均回帰 V-F 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格が移動平均を事前定義されたパーセンテージ下回るたびに最大5つの層で買いポジションを建てます。ポジションは固定テイクプロフィットまたはオプションのトレーリングで決済されます。

## 詳細

- **エントリー**: 価格が選択した移動平均からの偏差レベルを下抜ける。
- **エグジット**: 目標利益達成またはトレーリングストップの発動。
- **インジケーター**: 移動平均。
- **方向**: ロングのみ。
- **ストップ**: テイクプロフィット、オプションのトレーリング。
