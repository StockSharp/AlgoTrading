# レンジブレイクアウト戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、最後の `RangePeriod` ローソク足内の最高値と最低価格を測定します。ローソク足がこの範囲外で終了し、範囲の合計幅が `MaxRangePoints` より狭い場合、戦略はブレイクアウト方向に入ります。

## エントリールール
- **ロング**: ローソク足の終値 >= ルックバック レンジの最高値 かつ ポイント単位の範囲 <= `MaxRangePoints` かつオープン ポジションなし。
- **ショート**: ローソク足の終値 <= ルックバック レンジの最安値 かつ ポイント単位の範囲 <= `MaxRangePoints` かつオープン ポジションなし。

## 終了ルール
- 保護的なストップロスとテイクプロフィットは、ポジションがオープンされた直後に適用されます。
- 追加の終了ルールは使用されません。ポジションは保護がクローズするまでオープンのままです。

## パラメーター
- `RangePeriod` – 最高/最低を計算するためのローソク足の数。
- `MaxRangePoints` – 取引を許可するポイント単位の範囲の最大幅。
- `CandleType` – 分析と取引に使用されるローソク足の時間枠。
- `Volume` – 成行注文の量。
- `StopLossPoints` – ポイント単位のストップロス距離。
- `TakeProfitPoints` – テイクプロフィット距離（ポイント単位）。

## インジケーター
- 最高
- 最低
