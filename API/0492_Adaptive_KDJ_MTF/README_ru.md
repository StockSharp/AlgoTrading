# Адаптивный KDJ (MTF)
[English](README.md) | [中文](README_cn.md)

Стратегия Adaptive KDJ комбинирует значения осциллятора KDJ с трёх таймфреймов. Каждый таймфрейм сглаживается EMA и объединяется с помощью регулируемых весов. Сила тренда измеряется SMA от комбинированного осциллятора, что позволяет динамически менять зоны перекупленности и перепроданности.

Стратегия открывает длинную позицию, когда линия J ниже адаптивного уровня покупки и линия K пересекает линию D снизу вверх. Короткая позиция открывается, когда линия J выше адаптивного уровня продажи и линия K пересекает линию D сверху вниз.

## Подробности

- **Условия входа**: пересечение K и D при J ниже/выше динамических уровней.
- **Длинные/короткие позиции**: обе стороны.
- **Условия выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `TimeFrame1` = TimeSpan.FromMinutes(1)
  - `TimeFrame2` = TimeSpan.FromMinutes(3)
  - `TimeFrame3` = TimeSpan.FromMinutes(15)
  - `KdjLength` = 9
  - `SmoothingLength` = 5
  - `TrendLength` = 40
  - `WeightOption` = 1
- **Фильтры**:
  - Категория: Oscillator
  - Направление: обе стороны
  - Индикаторы: Stochastic, EMA, SMA
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
