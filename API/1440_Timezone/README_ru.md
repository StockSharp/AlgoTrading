# Стратегия Timezone
[English](README.md) | [中文](README_cn.md)

Демонстрационная стратегия, показывающая форматирование времени свечей в разных часовых поясах.

Тестирование показывает среднегодичную доходность около 0%. Она подходит для любого рынка.

Стратегия подписывается на свечи и выводит их время закрытия в выбранном пользователем часовом поясе. Может служить шаблоном для исследований, завязанных на время.

## Подробности

- **Условия входа**: Нет
- **Лонг/Шорт**: Нет
- **Условия выхода**: Нет
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `CandleType` = 5m
  - `Timezone` = Utc
- **Фильтры**:
  - Категория: Утилита
  - Направление: Нет
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Простая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
