# 时区策略
[English](README.md) | [Русский](README_ru.md)

演示如何将蜡烛时间转换到不同的时区的策略。

测试表明年均收益约为 0%，适用于任何市场。

策略订阅蜡烛，并将其收盘时间转换到用户选择的时区并记录，可作为时间研究或调度的模板。

## 详情

- **入场条件**: 无
- **多空方向**: 无
- **退出条件**: 无
- **止损**: 无
- **默认值**:
  - `CandleType` = 5m
  - `Timezone` = Utc
- **过滤器**:
  - 类别: 工具
  - 方向: 无
  - 指标: 无
  - 止损: 否
  - 复杂度: 简单
  - 时间框架: 任意
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 低
