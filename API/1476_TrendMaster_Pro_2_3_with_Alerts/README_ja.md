# TrendMaster Pro 2.3 アラート付き戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は TrendMaster Pro のコンパクト版です。短い移動平均が長い移動平均を越え、RSI がモメンタムを確認したときにトレードします。オプションの上位時間軸の移動平均でトレードを主要トレンドに限定できます。リスクはパーセンテージベースのテイクプロフィットとストップロスで管理されます。

## 詳細
- **エントリー**: 速い MA が遅い MA を越え、RSI フィルターとオプションのトレンド確認を満たす。
- **エグジット**: 逆クロスまたは保護注文。
- **インジケーター**: SMA または EMA、RSI。
- **フィルター**: オプションの上位時間軸トレンド。
