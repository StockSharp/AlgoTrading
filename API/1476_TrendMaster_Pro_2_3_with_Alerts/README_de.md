# TrendMaster Pro 2.3 mit Alerts-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie ist eine kompakte Version von TrendMaster Pro. Sie handelt, wenn ein kurzer gleitender Durchschnitt einen langen gleitenden Durchschnitt kreuzt und der RSI den Momentum bestätigt. Ein optionaler gleitender Durchschnitt auf einem höheren Zeitrahmen kann Trades auf den Haupttrend beschränken. Das Risiko wird mit prozentualen Take-Profit- und Stop-Loss-Niveaus gesteuert.

## Details
- **Einstieg**: schneller MA kreuzt langsamen MA mit RSI-Filter und optionaler Trendbestätigung.
- **Ausstieg**: entgegengesetzter Kreuzungspunkt oder Schutzorders.
- **Indikatoren**: SMA oder EMA, RSI.
- **Filter**: optionaler Trend auf höherem Zeitrahmen.
