# Estrategia TrendMaster Pro 2.3 con Alertas
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia es una versión compacta de TrendMaster Pro. Opera cuando una media móvil corta cruza una media móvil larga y el RSI confirma el momentum. Una media móvil opcional de mayor marco temporal puede restringir las operaciones a la tendencia principal. El riesgo se gestiona con niveles de take-profit y stop-loss basados en porcentajes.

## Detalles
- **Entrada**: la MA rápida cruza la MA lenta con filtro RSI y confirmación de tendencia opcional.
- **Salida**: cruce opuesto u órdenes de protección.
- **Indicadores**: SMA o EMA, RSI.
- **Filtros**: tendencia opcional de mayor marco temporal.
