# InwCoin マーチンゲール戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はビットコインのロングポジションにシンプルなマーチンゲール手法を実装します。
3つのオプションエントリーシグナルをサポートします：MACDヒストグラムがゼロを上抜け、
Stochastic RSIの%Dラインが20を上抜け、またはATRベースのチャネルを価格がブレイクアウト。
各買いの後、価格が設定したパーセンテージ下落するとポジションサイズが倍増する場合があります。
利益が平均エントリー価格を指定したパーセント上回ったときにポジション全体をクローズします。

## 詳細

- **エントリーシグナル**
  - **MACD Line > 0**: ヒストグラムがゼロを上抜け。
  - **STO RSI cross up**: %Kが売られすぎゾーンにいる間に%DラインがI20を上抜け。
  - **ATR Channel**: 終値がEMAにATRの乗数を加えた値を上抜け。
- **テイクプロフィット**: 価格が平均価格を設定パーセント上回ったときにポジションをクローズ。
- **マーチンゲール**: 価格が平均価格から設定パーセント下落したときに追加買いが発生。
- **方向**: ロングのみ。
