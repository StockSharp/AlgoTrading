# Стратегия Ichimoku by FarmerBTC
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинные позиции, когда цена выше облака Ichimoku, облако бычье, SMA старшего таймфрейма подтверждает тренд, а объём превышает свою скользящую среднюю, умноженную на коэффициент. Выход происходит при падении цены ниже облака.

## Детали

- **Условия входа**: сигнал индикаторов
- **Длинные/короткие**: только лонг
- **Условия выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `TenkanPeriod` = 10
  - `KijunPeriod` = 30
  - `SenkouSpanBPeriod` = 53
  - `SmaLength` = 13
  - `VolumeLength` = 20
  - `VolumeMultiplier` = 1.5
  - `CandleType` = 1 час
  - `HtfCandleType` = 1 день
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: Ichimoku, SMA, объём
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
