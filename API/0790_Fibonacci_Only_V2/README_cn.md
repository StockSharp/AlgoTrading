# 斐波那契专用策略 V2
[English](README.md) | [Русский](README_ru.md)

该策略基于 93 根 K 线计算的 19% 和 82.56% 斐波那契回撤位。当价格触及或突破这些水平并得到 K 线确认时入场。风险通过 ATR 止损和可选的跟踪止损进行管理。

## 详情

- **入场条件**：触及或突破 19% / 82.56% 斐波那契水平并得到 K 线确认
- **多/空**：双向
- **出场条件**：止损或跟踪止损
- **止损**：是
- **默认值**：
  - `CandleType` = 15 分钟
- **筛选器**：
  - 类别：斐波那契突破
  - 方向：双向
  - 指标：ATR、Highest、Lowest
  - 止损：是
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
