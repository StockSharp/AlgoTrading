93本のローソク足から計算した19%と82.56%のFibonacciリトレースメントレベルを中心に取引します。価格がこれらのレベルに触れるか突破し、ローソク足の確認が取れたときにエントリーします。リスク管理はオプションのATRベースのストップロスとトレーリングストップで行います。

- **エントリー条件**: ローソク足確認を伴う19% / 82.56% Fibonacciレベルへの接触またはブレイクアウト
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたはトレーリングストップ
- **ストップ**: はい
- **デフォルト値**:
  - `CandleType` = 15分
- **フィルター**:
  - カテゴリ: Fibonacciブレイクアウト
  - 方向: 両方
  - インジケーター: ATR, Highest, Lowest
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
