# ゴールド・プルバック戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ゴールド・プルバック戦略は、EMAのトレンド方向とMACD・TDIフィルターを組み合わせます。上昇トレンド中に価格が21期間EMAに触れ、MACDとTDIの両方が強気のときにロングエントリーします。下降トレンド中にEMA21へのプルバックが起きてMACDとTDIが弱気のときにショートエントリーします。各ポジションはシグナルローソク足にオフセットを加えた1:1のテイクプロフィットとストップロスを使用します。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: EMA14がEMA60を上回り、ローソク足がEMA21に触れ、MACDラインがシグナルラインを上回り、TDI MAがTDIシグナルを上回り、RSIが50を上回る。
  - **ショート**: EMA14がEMA60を下回り、ローソク足がEMA21に触れ、MACDラインがシグナルラインを下回り、TDI MAがTDIシグナルを下回り、RSIが50を下回る。
- **エグジット条件**: エントリーから等距離のストップロスまたはテイクプロフィットに加えてオフセットで決済。
- **ストップ**: `Offset` = 0.1 をローソク足の安値/高値に適用。
- **デフォルト値**:
  - `EmaFastLength` = 14
  - `EmaSlowLength` = 60
  - `EmaPullbackLength` = 21
  - `SlOffset` = 0.1
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング・ショート
  - インジケーター: EMA, MACD, RSI, SMA
  - 複雑さ: 中
  - リスクレベル: 中
