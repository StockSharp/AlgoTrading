# Простая RSI стратегия для акций 1D
[English](README.md) | [中文](README_cn.md)

Система открывает лонг, когда RSI падает ниже уровня перепроданности, а цена находится выше SMA(200) дневного таймфрейма. Позиция защищена стопом на основе ATR и тремя уровнями фиксации прибыли.

## Детали

- **Условия входа**: RSI ниже `OversoldLevel` и закрытие выше SMA фильтра.
- **Длинные/короткие позиции**: Только лонг.
- **Условия выхода**: Стоп по ATR или достижение одного из уровней прибыли.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `RsiPeriod` = 5
  - `OversoldLevel` = 30
  - `SmaLength` = 200
  - `AtrLength` = 20
  - `AtrMultiplier` = 1.5
  - `TakeProfit1` = 5
  - `TakeProfit2` = 10
  - `TakeProfit3` = 15
  - `StopLossPercent` = 25
  - `CandleType` = TimeSpan.FromDays(1)
- **Фильтры**:
  - Категория: Oscillator
  - Направление: Long
  - Индикаторы: RSI, SMA, ATR
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Daily
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
