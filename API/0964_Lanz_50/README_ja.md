# LANZ戦略 5.0
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

LANZ戦略5.0は200期間EMAの方向に取引し、同じ色の3本連続ローソク足を必要とします。1日の取引回数、ニューヨーク時間帯、エントリー間の最小距離で取引を制限します。

## 詳細

- **エントリー条件**:
  - ロングエントリー: EMA上方に価格があり、3本の強気ローソク足。
  - ショートエントリー: EMA下方に価格があり、3本の弱気ローソク足（オプション）。
- **ロング/ショート**: デフォルトはロング。
- **エグジット条件**:
  - 固定ストップロスまたはテイクプロフィット。
  - 設定時刻に手動決済。
- **ストップ**:
  - ストップロス = 40 pips。
  - テイクプロフィット = 120 pips。
- **デフォルト値**:
  - `EmaPeriod` = 200
  - `MaxTrades` = 99
  - `MinDistancePips` = 25
  - `StopLossPips` = 40
  - `TakeProfitPips` = 120
  - `StartHour` = 19
  - `EndHour` = 15
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: ロング
  - インジケーター: EMA
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
