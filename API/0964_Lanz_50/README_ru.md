# Стратегия LANZ 5.0
[English](README.md) | [中文](README_cn.md)

Стратегия LANZ 5.0 торгует по направлению EMA 200 и требует три последовательные свечи одного цвета. Ограничивает число сделок в день, использует торговый интервал по времени Нью‑Йорка и проверяет минимальное расстояние между входами.

## Детали

- **Критерии входа**:
  - Цена выше EMA и три бычьи свечи для входа в лонг.
  - Цена ниже EMA и три медвежьи свечи для входа в шорт (опционально).
- **Лонг/Шорт**: По умолчанию лонг.
- **Критерии выхода**:
  - Фиксированный стоп‑лосс или тейк‑профит.
  - Ручное закрытие в заданное время.
- **Стопы**:
  - Стоп‑лосс = 40 пунктов.
  - Тейк‑профит = 120 пунктов.
- **Значения по умолчанию**:
  - `EmaPeriod` = 200
  - `MaxTrades` = 99
  - `MinDistancePips` = 25
  - `StopLossPips` = 40
  - `TakeProfitPips` = 120
  - `StartHour` = 19
  - `EndHour` = 15
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: EMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
