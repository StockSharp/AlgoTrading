# Timer Trade
[English](README.md) | [中文](README_cn.md)

Стратегия Timer Trade чередует длинные и короткие позиции через фиксированные интервалы времени. Таймер инициирует рыночные заявки, каждая позиция автоматически защищена стоп‑лоссом и тейк‑профитом.

## Детали

- **Условия входа**: Срабатывание таймера.
- **Длинная/Короткая**: Оба направления.
- **Условия выхода**: Стоп‑лосс или тейк‑профит.
- **Стопы**: Да, через StartProtection.
- **Значения по умолчанию**:
  - `TimerInterval` = TimeSpan.FromSeconds(30)
  - `Volume` = 1
  - `StopLossLevel` = 10 пунктов
  - `TakeProfitLevel` = 50 пунктов
- **Фильтры**:
  - Категория: Таймер
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Начальная
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
