# Vicious Mortgage Rates V1戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

4つのボラティリティ指標から構築された合成指数をトレードします。
積の速いEMAが遅いEMAを上回るとロングポジションを建て、逆のクロスでショートポジションを建てます。

## 詳細

- **エントリー条件**: 合成指数の速いEMAが遅いEMAを上回るクロス
- **ロング/ショート**: 両方
- **エグジット条件**: 反対方向のクロス
- **ストップ**: なし
- **デフォルト値**:
  - `FastLength` = 8
  - `SlowLength` = 21
- **フィルター**:
  - カテゴリ: ボラティリティ
  - 方向: 両方
  - インジケーター: EMA
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
