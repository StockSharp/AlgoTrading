# Стратегия Vicious Mortgage Rates V1
[English](README.md) | [中文](README_cn.md)

Стратегия торгует синтетический индекс из четырёх волатильностных показателей.
Длинная позиция открывается, когда быстрая EMA произведения выше медленной, короткая – при обратном пересечении.

## Детали

- **Критерий входа**: быстрая EMA комбинированного индекса пересекает сверху медленную
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: обратное пересечение
- **Стопы**: нет
- **Значения по умолчанию**:
  - `FastLength` = 8
  - `SlowLength` = 21
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Обе стороны
  - Индикаторы: EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
