# Стратегия Parabolic SAR с ранним входом и выходом по MA
[English](README.md) | [中文](README_cn.md)

Стратегия использует индикатор Parabolic SAR для входа в сделки при смене расположения индикатора относительно цены. Дополнительное правило выхода — закрытие длинных позиций, когда цена опускается ниже скользящей средней, а SAR находится выше цены.

## Детали

- **Условия входа**: SAR меняет сторону относительно цены.
- **Длинные/короткие**: обе стороны.
- **Условия выхода**: для длинных позиций, SAR > цена и цена < MA.
- **Стопы**: не заданы.
- **Значения по умолчанию**:
  - `Acceleration` = 0.02
  - `AccelerationStep` = 0.02
  - `MaxAcceleration` = 0.2
  - `MaPeriod` = 11
  - `CandleType` = 5 минут
- **Фильтры**:
  - Категория: следование за трендом
  - Направление: обе
  - Индикаторы: Parabolic SAR, SMA
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: внутридневной
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
