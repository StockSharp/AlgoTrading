# 抛物线 SAR 早入场并基于均线退出策略
[English](README.md) | [Русский](README_ru.md)

该策略利用抛物线 SAR 指标在指标相对价格翻转时入场。额外的退出规则是在 SAR 位于价格之上且价格跌破简单移动平均线时平掉多头。

## 细节

- **入场条件**：SAR 相对于价格翻转。
- **多/空**：双向。
- **退出条件**：对于多头，当 SAR > 价格且价格 < MA。
- **止损**：未定义。
- **默认值**：
  - `Acceleration` = 0.02
  - `AccelerationStep` = 0.02
  - `MaxAcceleration` = 0.2
  - `MaPeriod` = 11
  - `CandleType` = 5 分钟
- **过滤条件**：
  - 类别: 趋势跟随
  - 方向: 双向
  - 指标: Parabolic SAR, SMA
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险级别: 中等
