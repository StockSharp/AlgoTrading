# 株式トレンドフォロー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、シンプルなトレンドフィルターを使って個別株を取引します。移動平均線を上回って取引されている株はロングし、下回っている株は避けるかショートします。

ポートフォリオは毎週等サイズポジションで更新され、資本を守るためにトレーリングストップを使用します。

## 詳細

- **データ**: 株式の日次終値。
- **エントリー**: 価格 > 移動平均線のときロング；下回るときショート。
- **エグジット**: 価格が再度平均線を越えるか、ストップが発動したとき。
- **銘柄**: 流動性の高い株式。
- **リスク**: トレーリングストップとポジション上限。
