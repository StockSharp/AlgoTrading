# MACD: средневозврат по наклону

Стратегия MACD Slope Mean Reversion сосредоточена на экстремальных показаниях индикатора MACD, чтобы использовать возврат к среднему. Широкие отклонения от нормального уровня редко продолжаются долго.

Сделки открываются, когда индикатор значительно отклоняется от своего среднего значения и начинает разворачиваться. Вход в длинные и короткие позиции сопровождается защитным стопом.

Подходит свинг‑трейдерам, ожидающим колебаний; стратегия закрывает позицию, когда MACD возвращается к равновесию. Начальное значение `FastMacdPeriod` = 12.

## Детали

- **Условия входа**: Индикатор разворачивается в сторону среднего значения.
- **Длинные/Короткие**: Оба направления.
- **Условия выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `FastMacdPeriod` = 12
  - `SlowMacdPeriod` = 26
  - `SignalMacdPeriod` = 9
  - `LookbackPeriod` = 20
  - `DeviationMultiplier` = 2.0m
  - `StopLossPercent` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Средневозвратная
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
