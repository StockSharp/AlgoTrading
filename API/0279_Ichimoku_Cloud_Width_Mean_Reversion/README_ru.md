# Средняя коррекция ширины облака Ичимоку

Стратегия Ichimoku Cloud Width Mean Reversion концентрируется на экстремальных значениях облака Ичимоку для игры на возврат к среднему. Широкие отклонения от нормального уровня редко сохраняются долго.

Сделки открываются, когда индикатор сильно удаляется от своей средней и затем начинает обратное движение. Возможны длинные и короткие позиции с защитным стопом.

Подходит свинг-трейдерам, ожидающим колебаний. Закрытие производится, когда облако Ичимоку возвращается к балансу. Начальный параметр `TenkanPeriod` = 9.

## Подробности

- **Условие входа**: Индикатор возвращается в сторону средней.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Индикатор возвращается к среднему.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `TenkanPeriod` = 9
  - `KijunPeriod` = 26
  - `SenkouSpanBPeriod` = 52
  - `LookbackPeriod` = 20
  - `DeviationMultiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Mean Reversion
  - Направление: Оба
  - Индикаторы: Ichimoku
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
