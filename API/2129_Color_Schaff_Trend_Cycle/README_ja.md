# カラーSchaffトレンドサイクル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は**Schaff Trend Cycle (STC)**インジケーターに基づいて取引します。STCはMACDシリーズにダブルストキャスティクス計算を適用し、-100から100の間で振動します。高いレベルを超えた値は強気圧力を示し、低いレベルを下回った値は弱気圧力を示します。

## 取引ロジック

- 選択した時間軸のローソク足をサブスクライブする。
- 高速・低速の指数移動平均を使ってMACDを計算する。
- 2回連続してストキャスティクス計算を適用してSTCを導出する。
- STCが高いレベルを上抜けしてさらに上昇を続けるとき:
  - ショートポジションを決済する。
  - ロングポジションに入る。
- STCが低いレベルを下抜けしてさらに下落を続けるとき:
  - ロングポジションを決済する。
  - ショートポジションに入る。

戦略は常に完全に形成されたローソク足に基づいて動作します。

## パラメーター

| 名前 | 説明 | デフォルト |
|------|------|-----------|
| `FastPeriod` | MACDで使用する高速EMAの期間 | `23` |
| `SlowPeriod` | MACDで使用する低速EMAの期間 | `50` |
| `Cycle` | ストキャスティクスのサイクル長 | `10` |
| `HighLevel` | STCの買われすぎ閾値 | `60` |
| `LowLevel` | STCの売られすぎ閾値 | `-60` |
| `CandleType` | 処理するローソク足の時間軸 | `4h` |

## 注意事項

- STC値はデフォルトレベルとの比較を容易にするために -100…100 の範囲に再スケーリングされます。
- 注文は `BuyMarket()` と `SellMarket()` の呼び出しで送信され、逆シグナルが現れるとポジションは自動的に反転します。
- この戦略はインジケーターシグナルのみに集中し、ストップロスやテイクプロフィット注文は使用しません。
