# Hull MA リバーサルストラテジー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Hull Moving Averageは滑らかさを保ちながら価格変化に素早く反応します。その方向の変化は短期的なリバーサルを予兆することがあります。この戦略は連続するHull MA値を監視し、傾きが転換したときに取引します。

テストでは年平均リターン約154%を示しています。株式市場で最も優れたパフォーマンスを発揮します。

移動平均が下降から上昇に転じると、ロングポジションが開かれます。上昇から下降への変化がショートを開始します。リスクは直近のローソク足を超えたATRベースのストップで管理されます。

決済はその保護ストップに依存し、Hull MAが示すモメンタムの転換に続く動きの一部を捉えます。

## 詳細

- **エントリー条件**: Hull MAの傾きが方向を変える。
- **ロング/ショート**: 両方。
- **エグジット条件**: ストップロス。
- **ストップ**: はい、ATRベース。
- **デフォルト値**:
  - `HmaPeriod` = 9
  - `AtrMultiplier` = 2 ATR
  - `CandleType` = 15 minute
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Hull MA, ATR
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
