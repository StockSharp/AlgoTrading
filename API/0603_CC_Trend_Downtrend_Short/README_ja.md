# CC トレンド 2 下降トレンド・ショート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ショートのみの戦略で、前の終値が動的フィボナッチ高値を下回り、EMA21がEMA55を下回った場合に売ります。価格が利益を維持したままEMA200を上抜けたとき、または前の終値が0.236フィボナッチレベルを上抜けて新しいショートシグナルが出ない場合にエグジットします。

## 詳細

- **エントリー条件**:
  - ショート: 前の終値がフィボナッチ高値を下回り、EMA21がEMA55を下回っている
- **ロング/ショート**: ショート
- **エグジット条件**:
  - 価格が利益を伴ってEMA200を上抜け
  - 新しいショートシグナルなしに前の終値が0.236フィボナッチレベルを上回る
- **ストップ**: なし
- **デフォルト値**:
  - `FibLength` = 100
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: ショート
  - インジケーター: EMA, Fibonacci
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
