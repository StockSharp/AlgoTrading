# Exp Moving Average FN Strategy
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует на разворотах наклона экспоненциальной скользящей средней (EMA). Покупает, когда EMA разворачивается вверх после снижения, и продаёт, когда EMA разворачивается вниз после роста. Стоп-лосс и тейк-профит задаются в абсолютных ценовых единицах.

## Детали

- **Условия входа**:
  - **Лонг**: наклон EMA меняется с нисходящего на восходящий.
  - **Шорт**: наклон EMA меняется с восходящего на нисходящий.
- **Направление**: обе стороны.
- **Условия выхода**:
  - Разворот наклона в противоположную сторону.
  - Срабатывание стоп-лосса или тейк-профита.
- **Стопы**: да, используются абсолютные ценовые расстояния.
- **Значения по умолчанию**:
  - `EMA Length` = 12
  - `Stop Loss` = 1000
  - `Take Profit` = 2000
  - `Candle Type` = таймфрейм 4 часа
- **Фильтры**:
  - Категория: следование тренду
  - Направление: обе стороны
  - Индикаторы: один (EMA)
  - Стопы: да
  - Сложность: средняя
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
