# COSTAR策略
[English](README.md) | [Русский](README_ru.md)

COSTAR策略基于收盘价的线性回归，并计算残差的标准差。通过在回归线上加减标准差乘以系数形成上下轨。当价格偏离过大并回到回归线时进行交易，以寻求均值回归。

## 细节

- **入场条件**：
  - **多头**：价格上穿下轨。
  - **空头**：价格下穿上轨。
- **方向**：双向。
- **出场条件**：
  - 价格穿回回归线。
- **止损**：无。
- **默认参数**：
  - `Length` = 100
  - `Multiplier` = 1
- **过滤器**：
  - 类型：均值回归
  - 方向：双向
  - 指标：线性回归、标准差
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
