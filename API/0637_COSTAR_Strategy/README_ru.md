# Стратегия COSTAR
[English](README.md) | [中文](README_cn.md)

Стратегия COSTAR строит линейную регрессию по ценам закрытия и измеряет стандартное отклонение остатка. Верхняя и нижняя границы формируются добавлением и вычитанием отклонения, умноженного на коэффициент. Сделки открываются при сильном отклонении и закрываются, когда цена возвращается к линии регрессии.

## Подробности

- **Условия входа**:
  - **Лонг**: цена пересекает нижнюю границу снизу вверх.
  - **Шорт**: цена пересекает верхнюю границу сверху вниз.
- **Направление**: обе стороны.
- **Условия выхода**:
  - цена пересекает линию регрессии.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `Length` = 100
  - `Multiplier` = 1
- **Фильтры**:
  - Категория: возврат к среднему
  - Направление: обе
  - Индикаторы: линейная регрессия, стандартное отклонение
  - Стопы: нет
  - Сложность: низкая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
