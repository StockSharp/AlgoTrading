# Hamster Bot MRS 2
[Русский](README_ru.md) | [中文](README_cn.md)

Simplified strategy based on a single moving average level.

## Details

- **Entry Criteria**: Price crosses MA level adjusted by shift
- **Long/Short**: Both
- **Exit Criteria**: Reverse crossing of the same level
- **Stops**: No
- **Default Values**:
  - `MA Length` = 3
  - `Shift` = 1
- **Filters**:
  - Category: Trend following
  - Direction: Both
  - Indicators: Simple Moving Average
  - Stops: No
  - Complexity: Low
  - Timeframe: Any
  - Seasonality: No
  - Neural networks: No
  - Divergence: No
  - Risk level: Medium
