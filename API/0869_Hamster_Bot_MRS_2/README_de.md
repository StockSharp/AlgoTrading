Vereinfachte Strategie basierend auf einem einzigen gleitenden Durchschnittsniveau.

## Details

- **Einstiegskriterien**: Preis kreuzt das durch Verschiebung angepasste MA-Niveau
- **Long/Short**: Beide
- **Ausstiegskriterien**: Umgekehrte Kreuzung desselben Niveaus
- **Stops**: Nein
- **Standardwerte**:
  - `MA Length` = 3
  - `Shift` = 1
- **Filter**:
  - Kategorie: Trendfolge
  - Richtung: Beide
  - Indikatoren: Einfacher Gleitender Durchschnitt
  - Stops: Nein
  - Komplexität: Niedrig
  - Zeitrahmen: Beliebig
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
