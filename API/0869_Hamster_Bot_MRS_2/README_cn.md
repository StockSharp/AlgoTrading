# Hamster Bot MRS 2
[English](README.md) | [Русский](README_ru.md)

基于单一移动平均线水平的简化策略。

## 细节

- **入场条件**：价格穿越带有偏移的MA水平
- **方向**：双向
- **出场条件**：价格反向穿越同一水平
- **止损**：无
- **默认参数**：
  - `MA Length` = 3
  - `Shift` = 1
- **过滤器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：简单移动平均
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险水平：中等
