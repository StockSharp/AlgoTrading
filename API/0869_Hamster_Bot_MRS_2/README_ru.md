# Hamster Bot MRS 2
[English](README.md) | [中文](README_cn.md)

Упрощённая стратегия на основе одного уровня скользящей средней.

## Детали

- **Условия входа**: Цена пересекает уровень МА с учётом смещения
- **Направление**: Оба
- **Условия выхода**: Обратное пересечение того же уровня
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `MA Length` = 3
  - `Shift` = 1
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: Простая скользящая средняя
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
