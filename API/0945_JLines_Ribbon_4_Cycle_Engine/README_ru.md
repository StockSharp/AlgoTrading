# Стратегия J-Lines Ribbon 4-Cycle Engine
[English](README.md) | [中文](README_cn.md)

Стратегия J-Lines Ribbon 4-Cycle Engine классифицирует рынок на циклы CHOP, LONG и SHORT с помощью ленты EMA и индикатора Average Directional Index. Входы выполняются при смене цикла и отскоках от ключевых EMA, выходы — при противоположных пересечениях или пробоях свингов.

## Подробности

- **Условия входа**:
  - **Лонг**: Новый цикл LONG или отскок выше EMA72/EMA126 при EMA72 > EMA89.
  - **Шорт**: Новый цикл SHORT или отскок ниже EMA72/EMA126 при EMA72 < EMA89.
- **Стопы**: Последний локальный максимум/минимум.
- **Параметры по умолчанию**:
  - `DmiLength` = 8
  - `AdxFloor` = 12
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA, ADX
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
