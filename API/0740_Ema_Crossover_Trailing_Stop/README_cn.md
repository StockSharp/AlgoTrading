# EMA 均线交叉追踪止损策略
[English](README.md) | [Русский](README_ru.md)

该策略使用两条指数移动平均线。当短期 EMA 向上穿越长期 EMA 时做多；当短期 EMA 向下穿越长期 EMA 时做空。进场后根据最高价或最低价设置的追踪止损在价格逆转达到设定百分比时平仓。

## 细节
- **入场条件**：短期 EMA 与长期 EMA 的交叉。
- **多/空**：双向。
- **离场条件**：反向交叉或追踪止损。
- **止损**：基于进场后最高/最低价的百分比追踪止损。
- **默认值**：
  - `ShortLength = 9`
  - `LongLength = 21`
  - `TrailStopPercent = 1`
- **过滤器**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：EMA
  - 止损：追踪止损
  - 复杂度：基础
  - 时间框架：短期
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险水平：中等
