# Стратегия EMA Crossover с трейлинг-стопом
[English](README.md) | [中文](README_cn.md)

Стратегия открывает лонг, когда короткая EMA пересекает длинную снизу вверх, и шорт, когда она пересекает сверху вниз. Трейлинг-стоп по максимальной или минимальной цене после входа закрывает позицию при движении цены против сделки на заданный процент.

## Детали
- **Критерии входа**: пересечение короткой и длинной EMA.
- **Лонг/Шорт**: оба направления.
- **Критерии выхода**: обратное пересечение или трейлинг-стоп.
- **Стопы**: трейлинг-стоп по максимуму/минимуму после входа.
- **Значения по умолчанию**:
  - `ShortLength = 9`
  - `LongLength = 21`
  - `TrailStopPercent = 1`
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: EMA
  - Стопы: Трейлинг-стоп
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
