# EMAクロスオーバー・トレーリングストップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

**EMAクロスオーバー・トレーリングストップ戦略**は、短期EMAが長期EMAを上抜けしたときにロングポジションを開き、下抜けしたときにショートポジションを開きます。エントリー後の最高値または最安値に基づくトレーリングストップは、価格が設定したパーセンテージ分反転したときにポジションを決済します。

## 詳細
- **エントリー条件**: 短期EMAが長期EMAをクロスする。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 逆クロスまたはトレーリングストップ。
- **ストップ**: エントリー後の最高値/最安値を使用したトレーリングストップ。
- **デフォルト値**:
  - `ShortLength = 9`
  - `LongLength = 21`
  - `TrailStopPercent = 1`
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA
  - ストップ: トレーリングストップ
  - 複雑さ: 基本
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
