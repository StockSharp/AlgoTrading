# IBS インターナルバーストレングス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

IBS Internal Bar Strength は、前のバーのレンジ内における終値の位置を利用して売られすぎや買われすぎの状態を検出する平均回帰戦略です。オプションのEMAフィルターはトレードをトレンドに合わせ、最後のエントリーから価格が最小限のパーセンテージ動いた場合にのみエントリーが許可されます。IBSが反対側の閾値を超えるか、最大保有時間に達するとポジションを手仕舞います。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: IBSがエントリー閾値を下回り、EMA条件を満たし、方向が許可されている。
  - **ショート**: IBSがエグジット閾値を上回り、EMA条件を満たし、方向が許可されている。
- **エグジット条件**: IBSが反対側の閾値を超えるか、取引期間の制限に達する。
- **ストップ**: 時間ベースのエグジット。
- **デフォルト値**:
  - `IbsEntryThreshold` = 0.09
  - `IbsExitThreshold` = 0.985
  - `EmaPeriod` = 220
  - `MinEntryPct` = 0
  - `MaxTradeDuration` = 14
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロング & ショート
  - インジケーター: IBS, EMA
  - 複雑さ: 低
  - リスクレベル: 中
