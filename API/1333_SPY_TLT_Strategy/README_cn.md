# SPY TLT 策略
[English](README.md) | [Русский](README_ru.md)

当 TLT 价格上穿其 SMA 时，策略买入主交易品种；当 TLT 收盘价下穿 SMA 时平仓。仅在指定的时间范围内允许交易。

## 详情

- **入场条件**：
  - **多头**：在时间窗口内，TLT 收盘价上穿其 SMA。
- **多空方向**：仅多头。
- **出场条件**：
  - TLT 收盘价下穿其 SMA。
- **止损**：无。
- **默认参数**：
  - `Start Time` = 2014-01-01
  - `End Time` = 2099-01-01
  - `TLT Symbol` = TLT
  - `SMA Length` = 20
- **过滤器**：
  - 分类：趋势跟随
  - 方向：多头
  - 指标：SMA
  - 止损：无
  - 复杂度：基础
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：低
