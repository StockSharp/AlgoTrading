# MA微調整戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は単純移動平均の傾きを監視します。一方向に2本連続したバーの後、移動平均の方向転換がエントリーのトリガーになります。下落後の上昇転換はロングポジションを開き、上昇後の下落転換はショートポジションを開きます。反対シグナルで既存の取引を決済します。

このシステムはMQLの「Exp_FineTuningMA」エキスパートから変換され、オリジナルのカスタムインジケーターを明確さのために標準的な単純移動平均に置き換えています。

## 詳細

- **エントリー条件**: MAが2本のバーの後に方向を変える。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 反対シグナルまたはストップ。
- **ストップ**: あり、パーセントベース。
- **デフォルト値**:
  - `MaLength` = 10
  - `TakeProfitPercent` = 1
  - `StopLossPercent` = 1
  - `CandleType` = TimeSpan.FromHours(4)
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: SMA
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: スイング / H4
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
