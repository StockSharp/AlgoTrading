# Стратегия Fine Tuning MA
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает наклон простой скользящей средней. После двух последовательных баров в одном направлении разворот средней формирует сигнал. Подъём после снижения открывает лонг, а снижение после подъёма открывает шорт. Противоположные сигналы закрывают позиции.

Система конвертирована из MQL‑советника "Exp_FineTuningMA". Вместо оригинального индикатора используется стандартная простая скользящая средняя.

## Детали

- **Условия входа**: Изменение направления MA после двух баров.
- **Длинные/короткие позиции**: Оба направления.
- **Условия выхода**: Противоположный сигнал или стоп.
- **Стопы**: Да, в процентах.
- **Значения по умолчанию**:
  - `MaLength` = 10
  - `TakeProfitPercent` = 1
  - `StopLossPercent` = 1
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Trend following
  - Направление: Оба
  - Индикаторы: SMA
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Свинг / H4
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
