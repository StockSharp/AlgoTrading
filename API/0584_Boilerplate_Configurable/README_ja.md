# 設定可能なボイラープレート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ボイラープレート設定可能戦略は、単純移動平均クロスオーバーまたはボリンジャースクイーズブレイクアウトの2つのモードを切り替えることができます。取引日とセッションフィルター、日付範囲、ニュースウィンドウ、ATRまたは静的リスク/リターンによるリスク管理機能を備えています。

## 詳細

- **エントリー条件**:
  - `SmaCross`モードでは、高速SMAが低速SMAを上抜けするとロング、逆のクロスでショート。
  - `Squeeze`モードでは、価格が狭いバンドの内側にある状態で外側のボリンジャーバンドをブレイクするとエントリー。
- **ロング/ショート**: ロング、ショート、または両方を任意の反転設定で設定可能。
- **エグジット条件**:
  - ATRまたは静的パーセンテージに基づくストップロスとテイクプロフィット。
  - 日次エグジット期間とニュースウィンドウがすべてのポジションをクローズ。
- **ストップ**: ドローダウン保護付きのトレードごとのストップロスとテイクプロフィット。
- **デフォルト値**:
  - `Length` = 20
  - `WideMultiplier` = 1.5
  - `NarrowMultiplier` = 2
  - `MaxLossPerc` = 0.02
  - `AtrMultiplier` = 1.5
  - `StaticRr` = 2
  - `NewsWindow` = 5
  - `MaxDrawdown` = 0.1
- **フィルター**:
  - カテゴリ: モジュラー
  - 方向: ロングとショート
  - インジケーター: SMA, Bollinger Bands, ATR
  - ストップ: はい
  - 複雑さ: 上級
  - 時間軸: 任意
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 高
