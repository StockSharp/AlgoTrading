# Стратегия Z-Strike Recovery
[English](README.md) | [中文](README_cn.md)

Открывает длинную позицию, когда z-score изменения цены превышает порог, и закрывает её через заданное число баров.

## Детали

- **Условие входа**: z-score изменения цены > порога
- **Длин/Корот**: Только длинные
- **Условие выхода**: Выход по времени
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `ZLength` = 16
  - `ZThreshold` = 1.3
  - `ExitPeriods` = 10
- **Фильтры**:
  - Категория: Статистическая
  - Направление: Длинные
  - Индикаторы: SMA, StandardDeviation
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
