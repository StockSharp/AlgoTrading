# シミュレーター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

オプションのストップロスとテイクプロフィットを備えたEMAクロスオーバーで取引する戦略です。高速EMAが低速EMAを上抜けたときに買い、高速EMAが低速EMAを下抜けたときに売ります。逆のシグナルまたは価格目標でポジションを決済します。

## 詳細

- **エントリー条件**:
  - ロング: 高速EMAが低速EMAを上抜け
  - ショート: 高速EMAが低速EMAを下抜け
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 逆のEMAクロスオーバー
  - ロング: 価格がテイクプロフィットまたはストップロスに到達
  - ショート: 価格がテイクプロフィットまたはストップロスに到達
- **ストップ**: 固定価格オフセット
- **デフォルト値**:
  - `FastPeriod` = 13
  - `SlowPeriod` = 50
  - `StopLoss` = 0.005m
  - `TakeProfit` = 0.005m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
