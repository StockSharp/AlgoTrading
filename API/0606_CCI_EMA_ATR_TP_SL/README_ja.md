# CCI + EMA戦略（パーセントまたはATR TP/SL）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はCommodity Channel Index（CCI）をオプションのEMAトレンドフィルターおよびRSI確認と組み合わせます。
CCIが極端なゾーンを離れ、オプションのフィルターが取引を許可した場合にポジションが建てられます。
テイクプロフィットとストップロスは、エントリー価格のパーセンテージとして、またはリスクリワード比を用いたATRベースのレベルとして計算できます。

## 詳細

- **エントリー条件:**
  - **ロング:** CCIが売られ過ぎレベルを上抜け、価格がEMAを上回る（有効な場合）、RSIが売られ過ぎを下回る（有効な場合）。
  - **ショート:** CCIが買われ過ぎレベルを下抜け、価格がEMAを下回る（有効な場合）、RSIが買われ過ぎを上回る（有効な場合）。
- **エグジット条件:**
  - テイクプロフィットまたはストップロスレベルへの到達。
  - CCIが買われ過ぎレベルを上抜けた場合にロングポジションをクローズ。
  - CCIが売られ過ぎレベルを下抜けた場合にショートポジションをクローズ。

デフォルトパラメーターは元のスクリプトに従います。
