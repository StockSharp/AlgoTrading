# TripleStochasticMTF戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は異なる時間軸で3つのStochastic Oscillatorを実行し、最小の時間軸が上位時間軸で確認された方向にシグナルラインをクロスしたときに取引します。より大きなトレンドの文脈の中で短期的なリバーサルを捉えるように設計されています。

プライマリ時間軸（デフォルト30分）とセカンダリ時間軸（デフォルト15分）が市場バイアスを決定します。エントリー時間軸（デフォルト5分）は前のバーと逆の%Kと%Dのクロスオーバーを待ち、プルバックをシグナルします。監視中の時間軸のいずれかがアクティブな取引に対してトレンド変化をシグナルするとポジションがクローズされます。

## 詳細

- **エントリー条件**:
  - **ロング**: 5分チャートで前の%K > %D、現在の%K ≤ %D、かつ両方の上位時間軸が%K > %Dを示している。
  - **ショート**: 5分チャートで前の%K < %D、現在の%K ≥ %D、かつ両方の上位時間軸が%K < %Dを示している。
- **ロング/ショート**: 両方向。
- **エグジット条件**:
  - **ロング**: いずれかの時間軸が下降トレンドに転換（%K < %D）。
  - **ショート**: いずれかの時間軸が上昇トレンドに転換（%K > %D）。
- **ストップ**: デフォルトでは実装なし。
- **デフォルト値**:
  - `Timeframe 1` = 30分。
  - `Timeframe 2` = 15分。
  - `Timeframe 3` = 5分。
  - `%K Period` = 5。
  - `%D Period` = 3。
  - `Slowing` = 3。
- **フィルター**:
  - カテゴリ: トレンドフォロー / プルバック
  - 方向: 両方
  - インジケーター: Stochastic Oscillator
  - ストップ: いいえ
  - 複雑さ: 中
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中程度
