# Фильтр торговой сессии - пример MACD
[English](README.md) | [中文](README_cn.md)

Стратегия показывает использование временного фильтра с MACD и трендовой EMA. Торги ведутся только в заданные часы.

## Детали

- **Условия входа**: Пересечение MACD и сигнальной линии в активной сессии и положение цены относительно трендовой EMA.
- **Лонг/Шорт**: В обе стороны.
- **Условия выхода**: Обратное пересечение или окончание сессии при включенной опции.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `SessionStart` = 11:00
  - `SessionEnd` = 15:00
  - `CloseAtSessionEnd` = false
  - `FastEmaPeriod` = 11
  - `SlowEmaPeriod` = 26
  - `SignalPeriod` = 9
  - `TrendMaLength` = 55
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: MACD, EMA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
