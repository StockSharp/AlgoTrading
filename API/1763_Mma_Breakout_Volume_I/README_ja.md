# MMAブレイクアウト出来高I戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、終値が長期の平滑移動平均（SMMA）を突破したときにブレイクアウトを取引します。
価格がSMMAを上抜けたときにロングポジションを建て、下抜けたときにショートポジションを建てます。
価格が取引に逆行して指数移動平均（EMA）を突破したときにポジションを決済します。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値がSMMA(200)を上抜け。
  - **ショート**: 終値がSMMA(200)を下抜け。
- **エグジット条件**:
  - **ロング**: 終値がEMA(5)を下回る。
  - **ショート**: 終値がEMA(5)を上回る。
- **ロング/ショート**: 両方。
- **ストップ**: 固定ストップロスなし、決済はEMAシグナルによって決定。
- **デフォルト値**:
  - `SMMA period` = 200
  - `EMA period` = 5
  - `Candle type` = 5分足
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 移動平均線
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
