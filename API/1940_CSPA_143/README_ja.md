# CSPA 1.43戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MQLエキスパートアドバイザー **CSPA-1_43** の移植版です。RSI（相対力指数）を使用して通貨ペアの強さを測定します。ペアが十分に強くなるか弱くなると、戦略は優勢なモメンタムの方向にポジションを開き、モメンタムが薄れると決済します。

## ロジック

- 選択した銘柄のローソク足を購読する。
- 完成した各ローソク足のRSI値を計算する。
- RSIが上限閾値を上回ったときにロングポジションを開く。
- RSIが下限閾値を下回ったときにショートポジションを開く。
- RSIが中立ゾーンに戻ったときに現在のポジションを決済する。

## パラメーター

| 名前 | 説明 | デフォルト |
|------|------|-----------|
| `StrengthPeriod` | RSIインジケーターに使用する期間。 | `14` |
| `Threshold` | シグナル生成に使用するRSIの中立レベル50からの距離。 | `10` |
| `CandleType` | ローソク足の時間軸。 | `1時間` |

## 注意事項

- この戦略は自動インジケーターバインディングを持つ高レベルAPIを使用します。
- 注文は成行注文（`BuyMarket` および `SellMarket`）を使用して実行されます。
- 完成したローソク足のみが処理されます。
