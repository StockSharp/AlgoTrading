# Williams %R-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie kauft, wenn Williams %R in tief überverkauftes Terrain fällt, und steigt beim Ausbruch nach oben oder bei überkauften Niveaus aus.

## Details

- **Einstiegskriterien**: %R < -90
- **Long/Short**: Nur Long
- **Ausstiegskriterien**: Schlusskurs > vorheriges Hoch oder %R > -30
- **Stops**: Nein
- **Standardwerte**:
  - `LookbackPeriod` = 2
- **Filter**:
  - Kategorie: Oszillator
  - Richtung: Long
  - Indikatoren: WilliamsR
  - Stops: Nein
  - Komplexität: Grundlegend
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
