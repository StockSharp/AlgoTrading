# Estrategia Williams %R
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia compra cuando Williams %R cae en territorio de sobreventa profunda y sale en ruptura alcista o en niveles de sobrecompra.

## Detalles

- **Criterios de entrada**: %R < -90
- **Largo/Corto**: Solo largos
- **Criterios de salida**: cierre > máximo anterior o %R > -30
- **Stops**: No
- **Valores predeterminados**:
  - `LookbackPeriod` = 2
- **Filtros**:
  - Categoría: Oscilador
  - Dirección: Largo
  - Indicadores: WilliamsR
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Intradía
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Medio
