# 経済政策不確実性戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

経済政策不確実性（EPU）戦略は、EPUインデックスの2期間SMAがユーザー定義の閾値を上抜けたときにロングポジションを取ります。ポジション取得後、戦略は固定数のバーが経過するまで待機してからクローズします。

このアプローチは、政策の不確実性が通常レベルを超えるタイミングを捉えることを目的としています。

## 詳細

- **エントリー条件**: SMAが閾値を上抜け。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 指定バー数後に決済。
- **ストップ**: なし。
- **デフォルト値**:
  - `Threshold` = 187
  - `SmaLength` = 2
  - `ExitPeriods` = 10
  - `CandleType` = TimeSpan.FromDays(1)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: ロング
  - インジケーター: SMA
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: 日足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
