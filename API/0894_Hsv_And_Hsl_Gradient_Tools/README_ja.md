# HSV・HSLグラデーションツール
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradingViewスクリプトから変換されたカラーユーティリティのライブラリ。HSVおよびHSL変換、色の反転、グラデーション生成のためのヘルパー関数を提供します。

## 詳細

- 機能:
  - RGBとHSV/HSLの相互変換。
  - 制御値に基づくグラデーションの生成。
  - 滑らかなトランジションのためのイージング関数。
- 目的: 視覚的なカラー計算。
- 取引ロジック: なし。
