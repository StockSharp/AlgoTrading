# Herramientas de Gradiente HSV y HSL
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Biblioteca de utilidades de color convertida desde un script de TradingView. Proporciona funciones auxiliares para conversiones HSV y HSL, inversión de color y generación de gradientes.

## Detalles

- Funciones:
  - Convertir entre RGB y HSV/HSL.
  - Generar gradientes basados en valores de control.
  - Funciones de suavizado para transiciones fluidas.
- Propósito: Cálculos visuales de color.
- Lógica de trading: Ninguno.
