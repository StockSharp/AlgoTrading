# HSV- und HSL-Gradientenwerkzeuge
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Bibliothek mit Farbhilfsmitteln, konvertiert aus einem TradingView-Skript. Bietet Hilfsfunktionen für HSV- und HSL-Konvertierungen, Farbinversion und Gradientengenerierung.

## Details

- Funktionen:
  - Konvertierung zwischen RGB und HSV/HSL.
  - Generierung von Gradienten basierend auf Steuerwerten.
  - Easing-Funktionen für weiche Übergänge.
- Zweck: Visuelle Farbberechnungen.
- Handelslogik: Keine.
