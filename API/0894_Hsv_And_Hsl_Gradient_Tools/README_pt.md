# Ferramentas de Gradiente HSV e HSL
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Biblioteca de utilitários de cor convertida de um script do TradingView. Fornece funções auxiliares para conversões HSV e HSL, inversão de cor e geração de gradientes.

## Detalhes

- Funções:
  - Converter entre RGB e HSV/HSL.
  - Gerar gradientes com base em valores de controle.
  - Funções de suavização para transições fluidas.
- Propósito: Cálculos visuais de cor.
- Lógica de trading: Nenhum.
