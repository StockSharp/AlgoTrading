# Baseline2-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie geht Long, wenn der Kurs einen aktuellen Support berührt, die schnelle EMA über der langsamen EMA liegt und RSI > 50 ist. Ein Short-Einstieg erfolgt bei einer Widerstandsberührung mit schneller EMA unterhalb der langsamen EMA und RSI < 45. Das Volumen muss über seinem Durchschnitt liegen und der ADX über 15. Der Ausstieg erfolgt über ATR-basierte Stop-Loss und Take-Profit.
