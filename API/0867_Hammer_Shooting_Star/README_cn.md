# Hammer & Shooting Star 策略
[English](README.md) | [Русский](README_ru.md)

该策略识别锤子线和射击之星形态。
当前一根K线为锤子线时开多，
为射击之星时开空。
止损和止盈分别设置在信号K线的最低点和最高点。

## 详情

- **入场条件**：
  - 多头：前一根K线为锤子线
  - 空头：前一根K线为射击之星
- **多/空**：双向
- **出场条件**：止损和止盈在信号K线的低/高点
- **止损**：是，固定
- **默认值**：
  - `WickFactor` = 0.9
  - `MaxOppositeWickFactor` = 0.45
  - `MinBodyRangePct` = 0.2
  - `CandleType` = 1 分钟
- **过滤器**：
  - 类别：形态
  - 方向：双向
  - 指标：K线
  - 止损：是
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
