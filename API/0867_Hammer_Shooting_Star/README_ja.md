# ハンマーと流れ星戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はHammerとShooting Starのローソク足パターンを取引します。
前のローソク足がHammerの場合にロングポジションを建て、
Shooting Starの場合にショートポジションに入ります。
エグジットはシグナルローソク足の高値と安値をテイクプロフィットとストップロスに使用します。

## 詳細

- **エントリー条件**:
  - ロング: 前のローソク足がHammer
  - ショート: 前のローソク足がShooting Star
- **ロング/ショート**: 両方
- **エグジット条件**: 前のローソク足の安値/高値でストップロスとテイクプロフィット
- **ストップ**: あり、シグナルローソク足の極値で固定
- **デフォルト値**:
  - `WickFactor` = 0.9
  - `MaxOppositeWickFactor` = 0.45
  - `MinBodyRangePct` = 0.2
  - `CandleType` = 1分
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: ローソク足
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
