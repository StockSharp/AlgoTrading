# Стратегия Hammer & Shooting Star
[English](README.md) | [中文](README_cn.md)

Стратегия торгует свечные модели Hammer и Shooting Star.
Длинная позиция открывается после появления Hammer на предыдущей свече,
короткая — после Shooting Star.
Выход осуществляется по максимуму и минимуму сигнальной свечи.

## Детали

- **Критерии входа**:
  - Лонг: предыдущая свеча — Hammer
  - Шорт: предыдущая свеча — Shooting Star
- **Лонг/Шорт**: оба направления
- **Критерии выхода**: стоп и тейк на минимум/максимум сигнальной свечи
- **Стопы**: да, фиксированные
- **Значения по умолчанию**:
  - `WickFactor` = 0.9
  - `MaxOppositeWickFactor` = 0.45
  - `MinBodyRangePct` = 0.2
  - `CandleType` = 1 минута
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Свечи
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
