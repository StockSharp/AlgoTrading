# Стратегия Long Term Profitable Swing
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда быстрая EMA пересекает сверху медленную EMA и RSI выше заданного порога. Выход осуществляется при достижении ценой уровней стоп-лосса или тейк-профита, основанных на множителях ATR.

## Подробности

- **Условия входа**:
  - Лонг: быстрая EMA пересекает медленную сверху и RSI > порога.
- **Направление**: Только лонг.
- **Условия выхода**:
  - Цена достигает ATR-стопа или ATR-профита.
- **Стопы**: Множители ATR для стоп-лосса и тейк-профита.
- **Параметры по умолчанию**:
  - `FastEmaLength` = 16
  - `SlowEmaLength` = 30
  - `RsiLength` = 9
  - `AtrLength` = 21
  - `RsiThreshold` = 50
  - `AtrStopMult` = 8
  - `AtrTpMult` = 11
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: EMA, RSI, ATR
  - Стопы: Да
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
