# Mawreez RSI-Divergenz-Detektor-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie prüft auf Kurs- und RSI-Divergenz über einen Bereich von Rückschau-Längen. Sie kauft, wenn der Kurs ein tieferes Tief bildet, während der RSI ein höheres Tief formt, und verkauft beim entgegengesetzten Signal.

## Parameter
- Kerzentyp
- RSI-Länge
- Minimale Divergenzlänge
- Maximale Divergenzlänge
