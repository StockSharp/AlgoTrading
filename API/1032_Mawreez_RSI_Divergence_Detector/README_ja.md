# MawreezのRSIダイバージェンス検出戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、さまざまな振り返り期間にわたって価格とRSIのダイバージェンスを確認します。価格がより低い安値を付ける一方でRSIがより高い安値を形成したときに買い、逆のシグナルで売ります。

## パラメーター
- ローソク足タイプ
- RSI期間
- 最小ダイバージェンス期間
- 最大ダイバージェンス期間
