# Estrategia Detectora de Divergencia RSI de Mawreez
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia verifica la divergencia entre el precio y el RSI en un rango de longitudes de retroceso. Compra cuando el precio forma un mínimo más bajo mientras el RSI forma un mínimo más alto, y vende con la señal opuesta.

## Parámetros
- Tipo de vela
- Longitud RSI
- Longitud mínima de divergencia
- Longitud máxima de divergencia
