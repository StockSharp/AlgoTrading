# Estratégia Detectora de Divergência RSI de Mawreez
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia verifica a divergência entre o preço e o RSI em um intervalo de comprimentos de retrocesso. Compra quando o preço forma uma mínima mais baixa enquanto o RSI forma uma mínima mais alta, e vende no sinal oposto.

## Parâmetros
- Tipo de vela
- Comprimento RSI
- Comprimento mínimo de divergência
- Comprimento máximo de divergência
