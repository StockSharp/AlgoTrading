# ZPF体积过滤 (ZPF Volume Filter)
[English](README.md) | [Русский](README_ru.md)

ZPF体积过滤将两条移动平均线与成交量平均结合。指标值是快速与慢速移动平均之差乘以平滑后的成交量。当该值上穿零线时表示多头压力，下穿零线则表示空头压力。

策略可做多做空。当ZPF指标穿越零线时入场，反向穿越时平仓。

## 详情

- **入场条件**: ZPF穿越零线。
- **多空方向**: 双向。
- **出场条件**: 反向穿越零线。
- **止损**: 否。
- **默认值**:
  - `Length` = 12
  - `CandleType` = TimeSpan.FromHours(4)
- **过滤器**:
  - 类别: 趋势
  - 方向: 双向
  - 指标: 移动平均, 成交量
  - 止损: 否
  - 复杂度: 基础
  - 时间框架: 摆动
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
