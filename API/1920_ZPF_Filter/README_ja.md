# ZPFボリュームフィルター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ZPFボリュームフィルターは、2本の移動平均とボリューム平均を組み合わせます。インジケーターの値は、高速と低速の移動平均の差をボリュームで平滑化したものです。この値がゼロを上回ると強気の圧力と判断され、ゼロを下回ると弱気の圧力を示します。

この戦略は両方向で取引します。ZPFインジケーターがゼロラインをクロスするとエントリーします。逆方向のクロスが発生するとポジションを決済します。

## 詳細

- **エントリー条件**: ZPFがゼロを上回るまたは下回るクロス。
- **ロング/ショート**: 両方。
- **エグジット条件**: 逆方向のゼロラインクロス。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 12
  - `CandleType` = TimeSpan.FromHours(4)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: Moving Average, Volume
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: スイング
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
