# Фильтр ZPF по объёму
[English](README.md) | [中文](README_cn.md)

Фильтр ZPF сочетает две скользящие средние и среднее по объёму. Значение индикатора представляет собой разницу между быстрой и медленной средними, умноженную на сглаженный объём. Пересечение выше нуля указывает на бычье давление, ниже нуля — на медвежье.

Стратегия торгует в обе стороны. Входы выполняются при пересечении индикатора через ноль. Выходы — при обратном пересечении.

## Подробности

- **Критерий входа**: ZPF пересекает уровень 0.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: обратное пересечение нуля.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Length` = 12
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: Скользящая средняя, Объём
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Свинг
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
