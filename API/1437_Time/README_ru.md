# Time
[English](README.md) | [中文](README_cn.md)

Стратегия демонстрирует работу с таймингом. Покупка происходит, когда максимум превышает цену открытия на заданное число тиков в течение указанного времени.

## Детали

- **Условия входа**: Максимум минус открытие держится выше порога заданное число секунд.
- **Лонг/Шорт**: Только лонг.
- **Условия выхода**: Условие перестает выполняться.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `TicksFromOpen` = 0
  - `SecondsCondition` = 20
  - `ResetOnNewBar` = true
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Прорыв
  - Направление: Лонг
  - Индикаторы: Цена
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (1m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
