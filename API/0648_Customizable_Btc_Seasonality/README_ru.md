# Настраиваемая сезонность BTC
[English](README.md) | [中文](README_cn.md)

Стратегия использует внутридневную сезонность биткойна и открывает длинную позицию в заданный час UTC, закрывая её в другой заданный час.

## Детали

- **Условия входа**: время совпадает с заданным часом входа
- **Длинная/короткая**: только длинная
- **Условия выхода**: время совпадает с заданным часом выхода
- **Стопы**: нет
- **Значения по умолчанию**:
  - `CandleType` = 1 минута
  - `EntryHour` = 21
  - `ExitHour` = 23
- **Фильтры**:
  - Категория: Сезонность
  - Направление: Длинная
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Новичок
  - Таймфрейм: Внутридневной
  - Сезонность: Да
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
