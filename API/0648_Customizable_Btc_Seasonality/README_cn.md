# 可自定义的 BTC 季节性策略
[English](README.md) | [Русский](README_ru.md)

该策略利用比特币在一天中的季节性规律，在用户指定的 UTC 小时进出场。
在进入时间开多头，在退出时间平仓。

## 细节

- **入场条件**：当前时间等于设定的入场小时
- **多/空**：仅做多
- **退出条件**：当前时间等于设定的退出小时
- **止损**：无
- **默认值**:
  - `CandleType` = 1 分钟
  - `EntryHour` = 21
  - `ExitHour` = 23
- **过滤器**:
  - 类别：季节性
  - 方向：做多
  - 指标：无
  - 止损：否
  - 复杂度：初级
  - 时间框架：日内
  - 季节性：是
  - 神经网络：否
  - 背离：否
  - 风险等级：低
