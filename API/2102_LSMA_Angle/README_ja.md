# LSMAアングル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は最小二乗移動平均線（LSMA）の角度を使用してトレンド方向を検出します。角度は設定可能な数のバーで区切られた2つのLSMA値の差で近似されます。

- **ロングエントリー**: LSMAの角度が正のしきい値を上回る。
- **ロングエグジット**: 角度が正のしきい値を下回る。
- **ショートエントリー**: LSMAの角度が負のしきい値を下回る。
- **ショートエグジット**: 角度が負のしきい値を上回る。

## パラメーター
- `LSMA Period`: LSMA計算の長さ。
- `Angle Threshold`: ゼロ周辺のニュートラルゾーンを定義する絶対値。
- `Start Shift`: 角度計算に使用する古いバー。
- `End Shift`: 角度計算に使用する直近のバー。
- `Candle Type`: 計算用のローソク足データタイプ。

## 注意事項
- 角度の値は銘柄に応じてポイントにスケーリングされます（JPYペアは1000、それ以外は100000）。
- 完成したローソク足のみで機能します。
