# Polynomial Regression Bands Channel 策略
[English](README.md) | [Русский](README_ru.md)

该策略对最近价格进行多项式回归，并根据残差的标准差构建上下通道。当价格跌破下轨时开多头仓位，当价格突破上轨时开空头仓位。

## 细节

- **入场条件**:
  - **多头**: `Close < LowerBand`。
  - **空头**: `Close > UpperBand`。
- **多空方向**: 双向。
- **出场条件**:
  - 相反信号。
- **止损**: 否。
- **默认值**:
  - `Length` = 100。
  - `Degree` = 2。
  - `Std Dev Multiplier` = 2。
- **筛选器**:
  - 类别: 均值回归
  - 方向: 双向
  - 指标: 多项式回归
  - 止损: 否
  - 复杂度: 中等
  - 时间框架: 中期
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
