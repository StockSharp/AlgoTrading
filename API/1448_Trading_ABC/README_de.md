# Trading ABC-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie approximiert das Trading-ABC-Konzept mithilfe von Trendfiltern und einem Stochastik-Oszillator. Ein Long-Trade wird eröffnet, wenn der Kurs über mehreren gleitenden Durchschnitten liegt und der Stochastik-%K den %D von unten kreuzt. Ein Short-Trade wird bei den entgegengesetzten Bedingungen eröffnet. Die Extremwerte des vorherigen Balkens dienen als Schutzstopps.
