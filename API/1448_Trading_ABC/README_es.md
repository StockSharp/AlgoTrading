# Estrategia Trading ABC
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia aproxima el concepto Trading ABC utilizando filtros de tendencia y un oscilador estocástico. Se abre una operación larga cuando el precio se mantiene por encima de varias medias móviles y el estocástico %K cruza por encima de %D. Se abre una operación corta en las condiciones opuestas. Los extremos de la barra anterior actúan como stops de protección.
