# Estratégia Trading ABC
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia aproxima o conceito Trading ABC usando filtros de tendência e um oscilador estocástico. Uma operação comprada é aberta quando o preço permanece acima de várias médias móveis e o estocástico %K cruza acima do %D. Uma operação vendida é aberta nas condições opostas. Os extremos da barra anterior atuam como stops de proteção.
