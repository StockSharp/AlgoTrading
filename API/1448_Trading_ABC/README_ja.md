# Trading ABC戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はトレンドフィルターとストキャスティクスオシレーターを使用してTrading ABCコンセプトを近似します。価格が複数の移動平均線を上回り、ストキャスティクスの%Kが%Dを上抜けした場合にロングトレードが開始されます。反対の条件ではショートトレードが開始されます。前のバーの高値・安値が保護ストップとして機能します。
