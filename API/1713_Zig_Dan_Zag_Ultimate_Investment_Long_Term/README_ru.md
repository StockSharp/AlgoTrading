# Zig Dan Zag Ultimate Investment Long Term
[English](README.md) | [中文](README_cn.md)

Долгосрочная инвестиционная стратегия, совмещающая пивоты ZigZag с медленным SMA фильтром тренда. Позиция открывается при формировании нового минимума ZigZag выше SMA, а выход происходит при противоположном пивоте ниже SMA.

## Детали
- **Условия входа**: Новый минимум ZigZag выше SMA.
- **Длинные/короткие**: Только лонг.
- **Условия выхода**: Максимум ZigZag ниже SMA.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `ZigzagDepth` = 12
  - `SmaLength` = 200
  - `CandleType` = TimeSpan.FromHours(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: Highest, Lowest, SimpleMovingAverage
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Долгосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
