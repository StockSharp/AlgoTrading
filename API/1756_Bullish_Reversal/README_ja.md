# 強気リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

古典的な強気リバーサルのローソク足パターンを検索する戦略です。これらのパターンのいずれかが50期間の単純移動平均線より下に現れたとき、戦略はロングポジションを開きます。オプションのトレーリングストップが未決利益を保護します。

## パターン
- **Abandoned Baby** – 2本の連続した弱気ローソク足に続き、2本目がギャップダウンしながら、1本目の始値を上回って終値となる強気ローソク足。
- **Morning Doji Star** – 弱気ローソク足、次に十字線または小さいボディのローソク足、その後1本目のローソク足のボディに戻って終値となる強気ローソク足。
- **Three Inside Up** – 弱気ローソク足、その範囲内に収まるより小さい強気ローソク足、次に1本目の始値を上回って終値となる強気ローソク足。
- **Three Outside Up** – 弱気ローソク足に続き、それを包み込むより大きな強気ローソク足、その後リバーサルを確認する3本目の強気ローソク足。
- **Three White Soldiers** – 終値が上昇し続ける3本の連続した強気ローソク足。

## 詳細
- **エントリー条件**: リストされたパターンのいずれかと、最後のローソク足が移動平均線より下で始値を付けること
- **ロング/ショート**: ロング
- **エグジット条件**: オプションのトレーリングストップ
- **ストップ**: トレーリング
- **デフォルト値**:
  - `MaPeriod` = 50
  - `TrailingStop` = 50
  - `UseTrailingStop` = true
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ロングのみ
  - インジケーター: SMA
  - ストップ: トレーリング
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
