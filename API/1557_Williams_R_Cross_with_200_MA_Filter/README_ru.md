# Стратегия Williams %R Cross с фильтром 200 MA
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия торгует пересечения Williams %R около уровня -50 с трендовым фильтром SMA(200).
Позиции закрываются по фиксированному тейк-профиту и стоп-лоссу.

## Детали

- **Критерий входа**: пересечение %R с порогами при проверке относительно SMA 200
- **Лонг/Шорт**: обе стороны
- **Критерий выхода**: тейк-профит или стоп-лосс
- **Стопы**: да
- **Значения по умолчанию**:
  - `WrLength` = 14
  - `CrossThreshold` = 10
  - `TakeProfit` = 30
  - `StopLoss` = 20
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: WilliamsR, SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
