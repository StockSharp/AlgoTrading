# Williams %R クロス戦略（200 MAフィルター付き）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

200期間SMAのトレンドフィルターを用い、Williams %Rが-50レベル付近でクロスするタイミングを取引します。
ポジションは固定の利益目標とストップロス距離でクローズします。

## 詳細

- **エントリー条件**: 200 SMAに対する価格の位置とともに%Rが閾値をクロス
- **ロング/ショート**: 両方
- **エグジット条件**: 利益目標またはストップロス
- **ストップ**: はい
- **デフォルト値**:
  - `WrLength` = 14
  - `CrossThreshold` = 10
  - `TakeProfit` = 30
  - `StopLoss` = 20
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: WilliamsR, SMA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
