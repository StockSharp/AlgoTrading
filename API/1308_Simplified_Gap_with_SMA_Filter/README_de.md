# Vereinfachte Gap-Strategie mit SMA-Filter
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt Auf- oder Ab-Gaps entsprechend der gewählten Richtung. Ein optionaler SMA-Filter bestätigt den Trend, und Positionen werden nach einer festgelegten Anzahl von Bars geschlossen.
