# Estrategia de Gap Simplificada con Filtro SMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia opera gaps al alza o a la baja según la dirección seleccionada. Un filtro SMA opcional confirma la tendencia, y las posiciones se cierran tras un número de barras especificado.
