# Estratégia de Gap Simplificada com Filtro SMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia opera gaps para cima ou para baixo de acordo com a direção selecionada. Um filtro SMA opcional confirma a tendência, e as posições são fechadas após um número especificado de barras.
