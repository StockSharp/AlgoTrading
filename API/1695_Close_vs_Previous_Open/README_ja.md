# 終値と前回始値の比較戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、最後に完成したローソク足の終値を、その前のローソク足の始値と比較します。
最新の終値が前回の始値を上回る場合にロングポジション、下回る場合にショートポジションを建てます。

## エントリー条件
- **Long**: 直近の完成したローソク足の終値が、その前のローソク足の始値より高い。
- **Short**: 直近の完成したローソク足の終値が、その前のローソク足の始値より低い。

## リスク管理
- オプションのストップロスとテイクプロフィット（ポイント単位）。
- ストップロスのオプションのトレーリング。

## パラメーター
- `Volume` – 注文数量。
- `UseStopLoss` – ストップロスを有効にする。
- `StopLoss` – ストップロスの距離（ポイント）。
- `UseTakeProfit` – テイクプロフィットを有効にする。
- `TakeProfit` – テイクプロフィットの距離（ポイント）。
- `UseTrailingStop` – 価格の動きに合わせてストップロスをトレーリングする。
- `CandleType` – 計算に使用するローソク足シリーズ。

## 注意事項
- 完全に形成されたローソク足のみで取引します。
- 反対のシグナルが現れた場合にポジションを反転します。
