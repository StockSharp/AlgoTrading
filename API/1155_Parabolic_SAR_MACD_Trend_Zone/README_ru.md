# Parabolic SAR с подтверждением MACD
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает индикатор Parabolic SAR с подтверждением MACD. Сделка открывается, когда цена пересекает SAR в направлении, подтверждённом MACD, что позволяет ловить развороты тренда.

## Подробности

- **Критерий входа**: цена пересекает SAR при соответствующем положении линии MACD относительно сигнальной.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: обратное пересечение цены/SAR или MACD.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `SarStart` = 0.02m
  - `SarIncrement` = 0.02m
  - `SarMax` = 0.2m
  - `MacdFast` = 12
  - `MacdSlow` = 26
  - `MacdSignal` = 9
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе
  - Индикаторы: Parabolic SAR, MACD
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
