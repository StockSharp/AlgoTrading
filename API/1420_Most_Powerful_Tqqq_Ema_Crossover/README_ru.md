# Стратегия Most Powerful TQQQ EMA Crossover
[English](README.md) | [中文](README_cn.md)

Стратегия входит в лонг при пересечении быстрой EMA выше медленной. Тейк-профит и стоп-лосс задаются как множители от цены входа.

## Детали

- **Критерий входа**: пересечение быстрой EMA выше медленной
- **Длин/Шорт**: только лонг
- **Критерий выхода**: достижение тейк-профита или стоп-лосса
- **Стопы**: да (фиксированный множитель)
- **Значения по умолчанию**:
  - `FastLength` = 20
  - `SlowLength` = 50
  - `TakeProfitMultiplier` = 1.3
  - `StopLossMultiplier` = 0.95
- **Фильтры**:
  - Категория: Тренд
  - Направление: Лонг
  - Индикаторы: EMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
