# Стратегия Bezier Standard Deviation
[English](README.md) | [中文](README_cn.md)

Эта стратегия торгует разворотами волатильности, используя индикатор стандартного отклонения. Локальные минимумы и максимумы индикатора рассматриваются как потенциальные точки разворота цены. Когда стандартное отклонение образует впадину, система ожидает роста волатильности и открывает длинную позицию. При появлении пика происходит продажа в расчёте на снижение волатильности.

По умолчанию стратегия работает на четырёхчасовом таймфрейме и может открывать как длинные, так и короткие позиции. Стоп‑лоссы не используются, выход осуществляется по обратному сигналу.

## Подробности

- **Условия входа**:
  - **Покупка**: значение стандартного отклонения на предыдущем баре меньше соседних значений (локальный минимум).
  - **Продажа**: значение стандартного отклонения на предыдущем баре больше соседних значений (локальный максимум).
- **Покупка/Продажа**: обе стороны.
- **Условия выхода**:
  - Обратный сигнал приводит к развороту позиции.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `StdDev Period` = 9.
  - `Candle Type` = 4‑часовые свечи.
- **Фильтры**:
  - Категория: Mean reversion
  - Направление: обе стороны
  - Индикаторы: Standard deviation
  - Стопы: нет
  - Сложность: простая
  - Таймфрейм: среднесрочный
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
