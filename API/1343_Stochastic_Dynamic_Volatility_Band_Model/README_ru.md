# Стратегия Stochastic-Dynamic Volatility Band Model
[English](README.md) | [中文](README_cn.md)

Использует волатильностные полосы по типу Боллинджера для торговли на пересечениях и закрывает позицию через фиксированное количество свечей.

## Детали

- **Критерий входа**: покупка при пересечении ценой нижней полосы вверх; продажа при пересечении верхней полосы вниз
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: позиция закрывается через `ExitBars` свечей
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Length` = 5
  - `Multiplier` = 1.67
  - `ExitBars` = 7
- **Фильтры**:
  - Категория: Волатильность
  - Направление: Обе стороны
  - Индикаторы: BollingerBands
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
