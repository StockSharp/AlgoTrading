# Stochastic動的ボラティリティバンドモデル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ボリンジャースタイルのボラティリティバンドを使用してクロスオーバーを取引し、固定本数のローソク足後に決済します。

## 詳細

- **エントリー条件**: 価格が下限バンドを上抜けたときにロング；価格が上限バンドを下抜けたときにショート
- **ロング/ショート**: 両方
- **エグジット条件**: `ExitBars` 本のローソク足後にポジションを決済
- **ストップ**: なし
- **デフォルト値**:
  - `Length` = 5
  - `Multiplier` = 1.67
  - `ExitBars` = 7
- **フィルター**:
  - カテゴリ: ボラティリティ
  - 方向: 両方
  - インジケーター: BollingerBands
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
