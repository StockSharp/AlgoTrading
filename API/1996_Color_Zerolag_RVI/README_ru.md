# Стратегия Color Zerolag RVI
[English](README.md) | [中文](README_cn.md)

Стратегия использует индекс относительной силы (RVI) и его сигнальную линию.
Покупает, когда основная линия RVI пересекает сигнал снизу, и продаёт, когда основная линия пересекает сигнал сверху.

## Детали

- **Критерий входа**: пересечение RVI и сигнальной линии
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал
- **Стопы**: нет
- **Значения по умолчанию**:
  - `RviLength` = 14
  - `SignalLength` = 9
  - `BuyOpen` = true
  - `SellOpen` = true
  - `BuyClose` = true
  - `SellClose` = true
  - `CandleType` = 4 часа
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Обе стороны
  - Индикаторы: RVI, SMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (H4)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
