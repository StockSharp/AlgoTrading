# Стратегия Enhanced Bollinger Bands SL TP
[English](README.md) | [中文](README_cn.md)

Стратегия торгует отскоки от полос Боллинджера, размещая лимитные заявки и фиксированные стоп-лосс и тейк-профит в пунктах.

## Детали

- **Условия входа**:
  - Long: предыдущее закрытие <= предыдущей нижней полосы и текущее закрытие > нижней полосы
  - Short: предыдущее закрытие >= предыдущей верхней полосы и текущее закрытие < верхней полосы
- **Направление**: Оба
- **Стопы**: Абсолютные стоп-лосс и тейк-профит в пунктах
- **Значения по умолчанию**:
  - `BollingerLength` = 20
  - `BollingerMultiplier` = 2m
  - `EnableLong` = true
  - `EnableShort` = true
  - `PipValue` = 0.0001m
  - `StopLossPips` = 10m
  - `TakeProfitPips` = 20m
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Разворот
  - Направление: Оба
  - Индикаторы: Bollinger Bands
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
