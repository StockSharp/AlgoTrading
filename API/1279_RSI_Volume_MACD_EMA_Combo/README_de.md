# RSI, Volumen, MACD, EMA Kombinations-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Kombiniert MACD- und RSI-Signale mit EMA-Trend und Volumenbestätigung. Long-Einstiege erfordern, dass der MACD über seiner Signallinie liegt, der RSI über dem überkauften Niveau, der Preis über dem EMA und das Volumen größer als seine SMA ist. Shorts verwenden die umgekehrte Logik. Positionen werden geschlossen, wenn der RSI die 50er-Linie kreuzt.
