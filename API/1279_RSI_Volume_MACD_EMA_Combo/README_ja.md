# RSI、出来高、MACD、EMA 組み合わせ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MACDとRSIのシグナルをEMAトレンドと出来高の確認と組み合わせます。ロングエントリーにはMACDがシグナルラインを上回り、RSIが買われすぎレベルを超え、価格がEMAを上回り、出来高がSMAより大きいことが必要です。ショートは逆のロジックを使用します。RSIが50ラインを越えた時点でポジションをクローズします。
