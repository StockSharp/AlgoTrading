# GRIM309 CallPut戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

GRIM309 CallPut戦略は、警告システムを伴う複数のEMAトレンドアラインメントに基づいて取引します。短期EMAが上昇トレンドを確認し、EMA5がEMA10を上回って上昇するときにロングポジションを取ります。ショートポジションは反対の条件で入ります。クールダウン期間により決済後の即時再エントリーを防ぎます。EMA5-EMA10スプレッドが急速に収縮したときに、追加の警告が早期撤退を引き起こします。

## 詳細
- **データ**: 価格ローソク足.
- **エントリー条件**:
  - **ロング**: EMA10がEMA20を上回り、価格がEMA50を上回り、EMA5がEMA10を上回って上昇、ポジションなしでクールダウン満了.
  - **ショート**: EMA10がEMA20を下回り、価格がEMA50を下回り、EMA5がEMA10を下回って下落、ポジションなしでクールダウン満了.
- **エグジット条件**: 価格がEMA15を超えるか警告シグナル.
- **ストップ**: なし.
- **デフォルト値**:
  - `Ema5Length` = 5
  - `Ema10Length` = 10
  - `Ema15Length` = 15
  - `Ema20Length` = 20
  - `Ema50Length` = 50
  - `Ema200Length` = 200
  - `CooldownBars` = 2
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング＆ショート
  - インジケーター: EMA
  - 複雑さ: 中程度
  - リスクレベル: 中
