# Индикатор типа тренда
[English](README.md) | [中文](README_cn.md)

Индикатор типа тренда определяет режим рынка с помощью ATR и ADX.
Покупает в фазе роста, продаёт в фазе падения и выходит при боковом движении.

## Детали

- **Критерий входа**: +DI больше -DI и нет боковика
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный тренд или боковик
- **Стопы**: нет
- **Значения по умолчанию**:
  - `UseAtr` = true
  - `AtrLength` = 14
  - `AtrMaLength` = 20
  - `UseAdx` = true
  - `AdxLength` = 14
  - `AdxLimit` = 25
  - `SmoothFactor` = 3
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: ATR, ADX
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
