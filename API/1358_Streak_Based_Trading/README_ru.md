# Стратегия на основе серии свечей
[English](README.md) | [中文](README_cn.md)

Стратегия отслеживает серии выигрышных и убыточных свечей. После достижения заданного количества подряд идущих свечей открывается позиция в противоположном направлении и удерживается фиксированное число свечей. Свечи-дожи игнорируются на основе размера тела.

## Детали

- **Критерии входа**: обратное направление после серии выигрышей/проигрышей.
- **Длинные/Короткие**: настраивается (`TradeDirection`).
- **Критерии выхода**: по истечении `HoldDuration` свечей.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `TradeDirection` = Long
  - `StreakThreshold` = 8
  - `HoldDuration` = 7
  - `DojiThreshold` = 0.01
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Разворот
  - Направление: Настраиваемое
  - Индикаторы: Price Action
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
