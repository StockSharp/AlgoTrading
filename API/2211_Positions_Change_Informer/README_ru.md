# Информатор об изменении позиции
[English](README.md) | [中文](README_cn.md)

Информатор об изменении позиции отслеживает совершенные сделки и сообщает об открытии, закрытии и развороте позиции. Стратегия не генерирует собственные ордера и используется для наблюдения за торговой активностью. Уведомления выводятся в журнал, сопровождаются звуком или отправляются по электронной почте в зависимости от выбранных параметров.

## Подробности

- **Критерии входа**: отсутствуют (реагирует на совершенные сделки)
- **Лонг/Шорт**: оба направления
- **Критерии выхода**: отсутствуют; только уведомления
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Alert` = Alert
  - `SoundName` = "alert.wav"
  - `Language` = Russian
- **Фильтры**:
  - Категория: Utility
  - Направление: оба
  - Индикаторы: нет
  - Стопы: нет
  - Сложность: базовая
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: отсутствует
