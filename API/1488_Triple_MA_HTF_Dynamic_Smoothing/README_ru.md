# Стратегия Triple MA HTF - Dynamic Smoothing
[English](README.md) | [中文](README_cn.md)

Стратегия сравнивает три скользящие средние, рассчитанные на старших таймфреймах.
Каждая средняя сглаживается пропорционально отношению её таймфрейма к рабочему.
Сигнал возникает при пересечении первой и второй средней при подтверждении тренда третьей.

## Детали

- **Критерии входа**: пересечение MA1 и MA2 с подтверждением тренда MA3.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: противоположный сигнал.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `HigherTimeFrame1` = TimeSpan.FromMinutes(15)
  - `HigherTimeFrame2` = TimeSpan.FromMinutes(60)
  - `HigherTimeFrame3` = TimeSpan.FromMinutes(240)
  - `Length1` = 21
  - `Length2` = 21
  - `Length3` = 50
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: MA
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (база 5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
