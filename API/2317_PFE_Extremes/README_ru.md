# PFE Экстремумы
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробои уровней индикатора Polarized Fractal Efficiency (PFE). Когда PFE пересекает верхний уровень, стратегия закрывает короткие позиции и открывает длинную. При пересечении нижнего уровня закрываются длинные позиции и открывается короткая.

Индикатор PFE оценивает эффективность движения цены относительно её пути. Значения около +1 указывают на сильное восходящее движение, около -1 — на сильное нисходящее. Пробой порогов может сигнализировать о начале нового тренда.

## Подробности

- **Вход**: PFE выше `UpLevel` для лонга или ниже `DownLevel` для шорта.
- **Направление**: В обе стороны.
- **Выход**: Пробой противоположного уровня или разворотный сигнал.
- **Стопы**: По умолчанию нет, можно добавить через защиту позиции.
- **Значения по умолчанию**:
  - `PfePeriod` = 5
  - `UpLevel` = 0.5
  - `DownLevel` = -0.5
  - `CandleType` = таймфрейм 4 часа
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: PFE
  - Стопы: Опционально
  - Сложность: Базовая
  - Таймфрейм: Свинг
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
