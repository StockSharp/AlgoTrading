# PFE 極値
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はPolarized Fractal Efficiency (PFE)インジケーターのブレイクアウトを取引します。PFEが上限レベルを上回ったとき、戦略はショートポジションを閉じてロングを開きます。PFEが下限レベルを下回ったとき、ロングポジションを閉じてショートを開きます。

PFEインジケーターは価格がそのパスに対してどれほど効率的に動いているかを評価します。+1に近い値は強い上昇動作を示し、-1に近い値は強い下降動作を示します。閾値のクロスは新しいトレンドの始まりを示す可能性があります。

## 詳細

- **エントリー条件**: PFEがロングの場合`UpLevel`を上回るか、ショートの場合`DownLevel`を下回る。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対レベルのブレイクまたはリバーサルシグナル。
- **ストップ**: デフォルトでは使用しない；ポジション保護で追加可能。
- **デフォルト値**:
  - `PfePeriod` = 5
  - `UpLevel` = 0.5
  - `DownLevel` = -0.5
  - `CandleType` = 4時間時間軸
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: PFE
  - ストップ: オプション
  - 複雑さ: 基本
  - 時間軸: スイング
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
