# TSI Long/Short-Strategie für BTC 2H
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Verwendet TSI-Ausbrüche auf einem Zwei-Stunden-Chart. Long-Trades beginnen, wenn der TSI über sein 100-Bar-Hoch kreuzt, während Short-Trades beginnen, wenn er unter sein 100-Bar-Tief fällt.

- Indikatoren: True Strength Index, Highest, Lowest
