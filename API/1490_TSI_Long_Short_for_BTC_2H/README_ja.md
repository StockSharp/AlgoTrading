# BTC 2H TSIロング/ショート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

2時間足チャートでのTSIブレイクアウトを利用する。TSIが過去100本の高値を上抜けするとロングトレードを開始し、過去100本の安値を下抜けするとショートトレードを開始する。

- インジケーター: True Strength Index, Highest, Lowest
