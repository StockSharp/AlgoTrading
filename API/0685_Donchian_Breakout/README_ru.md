# Прорыв Дончиана
[English](README.md) | [中文](README_cn.md)

Стратегия на прорывах канала Дончиана с фильтрами волатильности и объёма.

Покупка выполняется, когда цена закрывается выше верхней границы канала, при подтверждении тренда EMA и RSI выше 50. Короткие позиции открываются при пробое нижней границы. Выход из позиции происходит по обратному сигналу Дончиана или по стопу на основе ATR.

## Детали

- **Критерии входа**: Пробой канала Дончиана с фильтрами EMA, RSI, волатильности и объёма.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: обратный пробой или ATR‑стоп.
- **Стопы**: на основе ATR.
- **Значения по умолчанию**:
  - `EntryLength` = 20
  - `ExitLength` = 10
  - `AtrLength` = 14
  - `AtrMultiplier` = 1.5
  - `EmaLength` = 50
  - `VolumeSmaLength` = 20
  - `AtrSmaLength` = 20
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Donchian, ATR, EMA, RSI, Объём
  - Стопы: ATR‑стоп
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (1м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
