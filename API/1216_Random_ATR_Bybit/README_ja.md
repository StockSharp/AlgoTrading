# ランダム ATR 戦略 - Bybit
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、直近の価格レンジと現在の日付に基づいて決定論的なランダムシグナルを生成します。シグナルが1のときロングエントリー、0のときショートエントリーします。リスク管理にはATRベースのストップロスとテイクプロフィットレベルを使用します。

## 詳細

- **エントリー条件**:
  - **ロング**: ランダムシグナルが1に等しい。
  - **ショート**: ランダムシグナルが0に等しい。
- **ロング/ショート**: 両方向。
- **エグジット条件**: ストップロスまたはテイクプロフィット。
- **ストップ**: ストップロスは`SlAtrRatio * ATR`、テイクプロフィットは`SlAtrRatio * TpSlRatio * ATR`。
- **デフォルト値**:
  - `AtrLength` = 14
  - `SlAtrRatio` = 3
  - `TpSlRatio` = 1
- **フィルター**: なし。
- **複雑さ**: シンプル。
- **時間軸**: 設定可能。
