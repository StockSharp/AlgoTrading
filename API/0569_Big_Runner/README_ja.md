# ビッグランナー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ビッグランナー戦略は、終値と高速SMAが両方とも低速SMAの方向にクロスしたとき、すなわち強いモメンタムを示したときに取引します。ポジションサイズはポートフォリオ価値のパーセンテージにレバレッジを掛けた値から算出されます。オプションのストップロスとテイクプロフィットのレベルでリスクを管理します。

## 詳細

- **エントリー条件**:
  - 終値が高速SMAを上抜け、かつ高速SMAが低速SMAを上抜けたときに買い。
  - 終値が高速SMAを下抜け、かつ高速SMAが低速SMAを下抜けたときに売り。
- **ロング/ショート**: ロングとショート。
- **エグジット条件**:
  - エントリー価格に基づくオプションのストップロスとテイクプロフィット。
  - 逆シグナルで既存ポジションをクローズ。
- **ストップ**: ストップロスとテイクプロフィットのパーセンテージは設定可能。
- **デフォルト値**:
  - `FastLength` = 5
  - `SlowLength` = 20
  - `TakeProfitLongPercent` = 4
  - `TakeProfitShortPercent` = 7
  - `StopLossLongPercent` = 2
  - `StopLossShortPercent` = 2
  - `PercentOfPortfolio` = 10
  - `Leverage` = 1
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロング & ショート
  - インジケーター: SMA
  - ストップ: はい
  - 複雑さ: 低
  - 時間軸: 任意
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
