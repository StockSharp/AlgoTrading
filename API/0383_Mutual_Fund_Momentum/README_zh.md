# 共同基金动量策略

[English](README.md) | [Русский](README_ru.md)

该策略按季度在一组共同基金之间轮换。每个季度结束时，根据过去六个月的收益率对基金排序，下一季度将全部资金投入排名首位的基金，让长线投资者可以跟随主动管理产品中的持续动量。

任何时刻只持有一只基金。使用日价数据，并在每年 1 月、 4 月、 7 月和 10 月的前三个交易日进行调仓。

## 细节

- **标的范围**：共同基金列表。
- **信号**：126 天（六个月）收益率排序。
- **调仓**：每个新季度的前三个交易日。
- **仓位**：全部资金投入排名第一的基金。
- **风险控制**：若交易金额低于 `MinTradeUsd` 则不下单。
