# 投資信託モメンタム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、投資信託のセットの間で四半期ごとにローテーションを行います。各四半期末に、ファンドは過去6ヶ月のパフォーマンスでランク付けされます。翌四半期は首位ファンドに資本を配分し、長期投資家がアクティブ運用商品の持続的なモメンタムを追うことができます。

常に1つのファンドのみを保有します。日次価格データを使用し、リバランスは1月、4月、7月、10月の最初の3営業日に行われます。

## 詳細

- **ユニバース**: 投資信託のリスト。
- **シグナル**: 126日間（6ヶ月）トータルリターンのランキング。
- **リバランス**: 新四半期の最初の営業日に四半期ごと。
- **ポジショニング**: 最上位ランクのファンドに完全ロング。
- **リスク管理**: 注文価値が `MinTradeUsd` を下回る場合は取引をスキップ。
