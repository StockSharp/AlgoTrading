# ATR止损双SMA策略
[English](README.md) | [Русский](README_ru.md)

当快速简单移动平均线(SMA)向上穿越慢速SMA时开多单，向下穿越时开空单。
可选的止损根据平均真实波幅(ATR)乘以系数计算，用于确定退出价格。

## 详情

- **入场条件**：
  - **多头**：快速SMA上穿慢速SMA。
  - **空头**：快速SMA下穿慢速SMA。
- **方向**：双向。
- **出场条件**：
  - 启用时使用基于ATR的止损。
- **止损**：距离 = ATR × 系数。
- **默认参数**：
  - `FastLength` = 15
  - `SlowLength` = 45
  - `AtrLength` = 14
  - `AtrMultiplier` = 2
- **过滤条件**：
  - 类别：趋势跟随
  - 方向：双向
  - 指标：SMA, ATR
  - 止损：是
  - 复杂度：低
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险水平：中等
