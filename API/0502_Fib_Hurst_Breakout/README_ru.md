# Fib Hurst Breakout
[English](README.md) | [中文](README_cn.md)

**Fib Hurst Breakout** сочетает уровни Фибоначчи с дневного таймфрейма и фильтр экспоненты Херста. Пробитие ключевых уровней в сторону тренда даёт сигнал, а стоп 2% и соотношение риск/прибыль 1:2 управляют риском.

## Детали

- **Условия входа**:
  - Long: закрытие выше уровня 61.8% и дневной Hurst > 0.5
  - Short: закрытие ниже уровня 38.2% и дневной Hurst < 0.5
- **Long/Short**: Обе стороны
- **Условия выхода**: Stop-loss или take-profit
- **Стопы**: Да
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(15)
  - `HurstPeriod` = 50
  - `MaxTradesPerDay` = 5
  - `MaxTotalTrades` = 510
  - `RiskPercent` = 2m
  - `RiskReward` = 2m
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Hurst, Fibonacci
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
