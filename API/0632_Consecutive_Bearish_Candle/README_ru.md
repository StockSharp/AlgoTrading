# Последовательные медвежьи свечи
[English](README.md) | [中文](README_cn.md)

Стратегия входит в лонг после серии подряд идущих медвежьих свечей и выходит при пробое максимума предыдущей свечи.

Это контртрендовый подход, который покупает после сильного снижения, рассчитывая на отскок при ослаблении продавцов.

## Подробности

- **Критерии входа**: `N` подряд медвежьих свечей в заданном окне.
- **Длинные/короткие**: только лонг.
- **Критерии выхода**: закрытие выше максимума предыдущей свечи.
- **Стопы**: нет.
- **Параметры по умолчанию**:
  - `Lookback` = 3
  - `CandleType` = TimeSpan.FromDays(1)
  - `StartTime` = 2014-01-01
  - `EndTime` = 2099-01-01
- **Фильтры**:
  - Категория: Контртренд
  - Направление: Лонг
  - Индикаторы: Прайс экшен
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
