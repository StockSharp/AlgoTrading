# 连续看跌蜡烛
[English](README.md) | [Русский](README_ru.md)

在出现连续的看跌蜡烛后进场做多，当价格突破前一根K线的高点时平仓。

该策略属于均值回归类型，试图在过度下跌后买入，期待卖压减弱时出现反弹。

## 详情

- **入场条件**: 时间窗口内出现 `N` 根连续看跌蜡烛。
- **多空方向**: 仅做多。
- **出场条件**: 收盘价突破上一根K线的最高价。
- **止损**: 无。
- **默认值**:
  - `Lookback` = 3
  - `CandleType` = TimeSpan.FromDays(1)
  - `StartTime` = 2014-01-01
  - `EndTime` = 2099-01-01
- **过滤器**:
  - 类型: 均值回归
  - 方向: 多头
  - 指标: 价格行为
  - 止损: 否
  - 复杂度: 基础
  - 时间框架: 日线
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
