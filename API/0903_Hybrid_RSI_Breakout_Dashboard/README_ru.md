# Hybrid RSI Breakout Dashboard
[English](README.md) | [中文](README_cn.md)

Стратегия сочетает среднее значение RSI и пробои, фильтруемые ADX и 200 EMA.

Система покупает при флэте, когда RSI опускается ниже `RsiBuy` и цена выше EMA. Продаёт при RSI выше `RsiSell` и цене ниже EMA. В тренде входит по пробою максимумов/минимумов закрытия и сопровождает позицию по ATR.

Включает фильтр даты начала и переменные панели для последнего типа сделки и направления.

## Детали

- **Условия входа**: сигналы RSI во флэте с фильтром EMA или пробой прошлых `BreakoutLength` закрытий при ADX > `AdxThreshold`.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Для RSI — пересечение `RsiExit`. Для пробоя — трейлинг по ATR.
- **Стопы**: Трейлинг по ATR для сделок пробоя.
- **Значения по умолчанию**:
  - `AdxLength` = 14
  - `AdxThreshold` = 20m
  - `EmaLength` = 200
  - `RsiLength` = 14
  - `RsiBuy` = 40m
  - `RsiSell` = 60m
  - `RsiExit` = 50m
  - `BreakoutLength` = 20
  - `AtrLength` = 14
  - `AtrMultiplier` = 2m
  - `StartDate` = 2017-01-01
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд, Средняя реверсия
  - Направление: Оба
  - Индикаторы: ADX, EMA, RSI, ATR, Highest/Lowest
  - Стопы: Трейлинг
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
