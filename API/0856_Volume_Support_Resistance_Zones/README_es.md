# Estrategia de Zonas de Soporte y Resistencia Basada en Volumen
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia detecta zonas de soporte y resistencia utilizando patrones fractales filtrados por volumen en velas horarias. Compra cuando el precio entra en una zona de soporte y cierra la posición cuando el precio alcanza una zona de resistencia.
