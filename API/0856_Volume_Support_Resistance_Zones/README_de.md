# Strategie für Volumenbasierte Unterstützungs- und Widerstandszonen
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Strategie erkennt Unterstützungs- und Widerstandsbereiche anhand volumengefilterter Fraktalmuster auf Stundencharts. Sie kauft, wenn der Preis in eine Unterstützungszone eintritt, und schließt die Position, wenn der Preis eine Widerstandszone erreicht.
