# Стратегия Price Flip
[English](README.md) | [中文](README_cn.md)

Стратегия Price Flip зеркалирует цену относительно недавних максимумов и минимумов и торгует пересечения скользящих средних, когда предыдущая цена закрытия находится по другую сторону от инвертированной цены. Можно включить фильтр тренда на основе медленной скользящей средней.

## Подробности

- **Условия входа**:
  - Предыдущее закрытие выше инвертированной цены.
  - Быстрая скользящая средняя пересекает медленную снизу вверх.
  - Опционально: цена выше медленной скользящей средней при включённом фильтре тренда.
- **Длинные/Короткие**: Обе стороны.
- **Условия выхода**:
  - Противоположный сигнал разворачивает позицию.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `TickerMaxLookback` = 100
  - `TickerMinLookback` = 100
  - `FastMaLength` = 12
  - `SlowMaLength` = 14
  - `UseTrendFilter` = true
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA, Highest/Lowest
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
