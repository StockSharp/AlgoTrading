# Динамическая стратегия поддержки и сопротивления по пивотам
[English](README.md) | [中文](README_cn.md)

Эта стратегия строит динамические уровни поддержки и сопротивления на основе последних пивотных максимумов и минимумов. Вход в лонг выполняется при пересечении цены уровня поддержки снизу, а вход в шорт — при пересечении сопротивления сверху. Управление рисками использует фиксированные проценты стоп-лосса и тейк-профита.

## Детали

- **Условия входа**: Цена рядом с поддержкой/сопротивлением в пределах `SupportResistanceDistance` процентов и пересечение уровня.
- **Лонг/Шорт**: Оба.
- **Условия выхода**: Фиксированные тейк-профит и стоп-лосс.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `PivotLength` = 2
  - `SupportResistanceDistance` = 0.4m
  - `StopLossPercent` = 10.0m
  - `TakeProfitPercent` = 26.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Пивоты
  - Стопы: Да
  - Сложность: Начальная
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
