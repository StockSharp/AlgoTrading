# ダイナミック・サポート＆レジスタンス・ピボット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、直近のピボット高値・安値からダイナミックなサポートとレジスタンスレベルを導出します。価格がレベル付近でサポートを上抜けたときにロングエントリーし、レジスタンスを下抜けたときにショートエントリーします。リスク管理は固定パーセントのストップロスと利益確定レベルを使用します。

## 詳細

- **エントリー条件**: 価格がサポート/レジスタンスから `SupportResistanceDistance` パーセント以内にあり、サポートを上抜けまたはレジスタンスを下抜け。
- **ロング/ショート**: 両方。
- **エグジット条件**: 固定の利益確定とストップロス。
- **ストップ**: はい。
- **デフォルト値**:
  - `PivotLength` = 2
  - `SupportResistanceDistance` = 0.4m
  - `StopLossPercent` = 10.0m
  - `TakeProfitPercent` = 26.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: Pivot
  - ストップ: はい
  - 複雑さ: 初心者
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
