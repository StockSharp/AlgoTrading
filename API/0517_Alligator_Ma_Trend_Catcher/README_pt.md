# Estratégia Alligator + MA Trend Catcher
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia combina o Alligator de Bill Williams com um filtro de tendência EMA de 200 períodos. Uma posição comprada é aberta quando o preço está acima da linha de tendência e as linhas do Alligator (lábios, dentes, mandíbula) estão alinhadas para cima. Uma posição vendida é aberta quando o preço está abaixo da linha de tendência e as linhas do Alligator estão alinhadas para baixo. As posições são fechadas quando ocorre o alinhamento oposto.
