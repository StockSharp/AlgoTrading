# Alligator + MA Trend Catcher-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie kombiniert den Bill Williams Alligator mit einem 200-Perioden-EMA-Trendfilter. Eine Long-Position wird eröffnet, wenn der Preis über der Trendlinie liegt und die Alligator-Linien (Lippen, Zähne, Kiefer) nach oben ausgerichtet sind. Eine Short-Position wird eröffnet, wenn der Preis unter der Trendlinie liegt und die Alligator-Linien nach unten ausgerichtet sind. Positionen werden geschlossen, wenn die entgegengesetzte Ausrichtung auftritt.
