# Alligator + MA トレンドキャッチャー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ビル・ウィリアムズのAlligatorと200期間EMAトレンドフィルターを組み合わせたものです。価格がトレンドラインの上にあり、Alligatorのライン（リップス、ティース、ジョー）が上向きに揃っているときにロングポジションを開きます。価格がトレンドラインの下にあり、Alligatorのラインが下向きに揃っているときにショートポジションを開きます。逆の配列が発生したときにポジションを決済します。
