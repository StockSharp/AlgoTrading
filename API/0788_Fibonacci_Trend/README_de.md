# Fibonacci-Trend
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Konvertiert aus dem TradingView-Skript „Fibonacci Trend - Aynet". Die Strategie erkennt Swing-Hochs und -Tiefs, erstellt Fibonacci-Retracement-Niveaus und handelt, wenn der Kurs das 50%-Niveau kreuzt.

## Parameter
- **Lookback Period** – Bars für die Swing-Erkennung.
- **Fib Level 1-5** – Fibonacci-Retracement-Verhältnisse.
- **Candle Type** – Kerzen-Zeitrahmen.
