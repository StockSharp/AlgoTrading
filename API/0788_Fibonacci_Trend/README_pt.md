# Tendência Fibonacci
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Convertido do script TradingView "Fibonacci Trend - Aynet". A estratégia detecta topos e fundos de swing, constrói níveis de retração Fibonacci e opera quando o preço cruza o nível de 50%.

## Parâmetros
- **Lookback Period** – barras utilizadas para detecção de swings.
- **Fib Level 1-5** – razões de retração Fibonacci.
- **Candle Type** – período temporal dos candles.
