# Tendencia Fibonacci
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Convertido del script de TradingView "Fibonacci Trend - Aynet". La estrategia detecta máximos y mínimos de swing, construye niveles de retroceso Fibonacci y opera cuando el precio cruza el nivel del 50%.

## Parámetros
- **Lookback Period** – barras utilizadas para la detección de swings.
- **Fib Level 1-5** – ratios de retroceso Fibonacci.
- **Candle Type** – marco temporal de las velas.
