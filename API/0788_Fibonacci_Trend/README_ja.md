# Fibonacci トレンド
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TradingViewスクリプト「Fibonacci Trend - Aynet」から変換。この戦略はスイング高値と安値を検出し、Fibonacciリトレースメントレベルを構築し、価格が50%レベルをクロスしたときに取引します。

## パラメータ
- **Lookback Period** – スイング検出に使用するバー数。
- **Fib Level 1-5** – Fibonacciリトレースメント比率。
- **Candle Type** – ローソク足の時間軸。
