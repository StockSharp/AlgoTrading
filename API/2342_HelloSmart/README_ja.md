# HelloSmart 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、一方向のみにポジションを開くシンプルなグリッドトレードアプローチを実装しています。市場が最後のエントリーに対して設定されたティック数だけ動くたびに、新しい注文が出されます。累積ポジション量が閾値に達すると、次の注文サイズが乗算されます。総利益または損失が事前定義された限界に達すると、すべてのポジションが決済されます。

## パラメーター
- **Trade Direction** – 1を選択するとロングポジションのみ開く、2を選択するとショートポジションのみ開く。
- **Step** – 別のポジションを追加する前に市場が動く必要がある価格ティック数。
- **Initial Lot** – 最初の注文の基本ボリューム。
- **Threshold Volume** – ロット乗算を発動する累積ポジションサイズ。
- **Maximum Lot** – 個別注文ボリュームの上限。
- **Profit Target** – すべてのポジションが決済される通貨単位の利益額。
- **Loss Limit** – すべてのポジションが決済される通貨単位の損失額。
- **Lot Multiplier** – 閾値ボリュームを超えた場合に次の注文に適用される係数。
- **Candle Type** – 価格変動を測定するために使用するローソク足シリーズ。
