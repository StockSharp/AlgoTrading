# Стратегия Break the Range Bound
[English](README.md) | [中文](README_cn.md)

Эта стратегия ищет периоды затишья, когда три скользящие средние сходятся в узком диапазоне. Пробой верхней или нижней границы такого диапазона служит сигналом входа в позицию в направлении пробоя.

Система отслеживает максимальный разброс между быстрыми, средними и медленными SMA. Если этот разброс остаётся ниже заданного порога на протяжении указанного количества баров, рынок считается "боковым". Максимальный максимум и минимальный минимум этого промежутка определяют уровни пробоя.

Сделки открываются при закрытии цены за пределами этих уровней. Позиции закрываются, если цена возвращается в диапазон или прибыль превысила кратное ширины диапазона.

## Детали

- **Условия входа**:
  - **Лонг**: После `RangeLength` баров с разбросом SMA ниже `ShakeThreshold` вход при закрытии цены выше максимума диапазона.
  - **Шорт**: При тех же условиях вход при закрытии цены ниже минимума диапазона.
- **Направление**: Оба.
- **Условия выхода**:
  - **Лонг**: Закрытие, если цена опускается ниже минимума диапазона или прибыль > `4 * (max - min)`.
  - **Шорт**: Закрытие, если цена поднимается выше максимума диапазона или прибыль > `4 * (max - min)`.
- **Стопы**: Псевдостопы по границам диапазона и величине прибыли.
- **Значения по умолчанию**:
  - `FastSma` = 38
  - `MidSma` = 140
  - `SlowSma` = 210
  - `ShakeThreshold` = 250
  - `RangeLength` = 200
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: SMA, Highest, Lowest
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
