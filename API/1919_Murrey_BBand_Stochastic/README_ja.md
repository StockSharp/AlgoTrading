# Murrey Math BBand & Stochastic戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はボリンジャーバンドとストキャスティクスオシレーターを確認として使用し、Murrey Mathの極値ラインでのリバーサルを取引します。

この手法は設定可能な期間の最高値と最安値からMurreyレベルを計算します。価格が売られすぎ状態の時に0/8ラインに近づくと戦略は買います。価格が買われすぎ状態の時に8/8ラインに近づくと売ります。ボリンジャーバンドの最小幅フィルターが横ばい市場での取引を防ぎます。

## 詳細

- **エントリー条件**
  - **ロング**: 終値が0/8ラインの*Entry Margin*以内の上方にあり、ストキャスティクス <= 21 かつボリンジャーバンド幅 >= しきい値。
  - **ショート**: 終値が8/8ラインの*Entry Margin*以内の下方にあり、ストキャスティクス >= 79 かつボリンジャーバンド幅 >= しきい値。
- **ロング/ショート**: 両方。
- **エグジット条件**
  - ロングポジションは1/8ラインでクローズ、または価格が-2/8ラインを下回った場合。
  - ショートポジションは7/8ラインでクローズ、または価格が+2/8ラインを上回った場合。
- **ストップ**: Murreyラインの（-2/8または+2/8）が保護ストップとして機能します。
- **フィルター**
  - ボリンジャーバンド幅フィルター。
  - ストキャスティクスオシレーターフィルター。
