# サポート・レジスタンス MTF 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

上位時間軸でサポートとレジスタンスゾーンを計算するユーティリティ戦略。設定可能な期間における最高値と最安値がレジスタンスおよびサポートレベルとして記録されます。

## 詳細

- **インジケーター**: Highest, Lowest
- **パラメーター**:
  - `Period` – 上位時間軸のルックバックバー数。
  - `Higher Candle Type` – レベル計算に使用する時間軸。
  - `Use High/Low` – high/lowまたはclose/open価格を使用するか。
- **ストップ**: なし
- **ターゲット**: なし
- **ロング/ショート**: なし（インジケーターのみ）
