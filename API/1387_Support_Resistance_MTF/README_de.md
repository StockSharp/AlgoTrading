# Unterstützung-Widerstand-MTF-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Hilfsstrategie, die Unterstützungs- und Widerstandszonen auf einem höheren Zeitrahmen berechnet. Die höchsten und niedrigsten Preise über einen konfigurierbaren Zeitraum werden als Widerstands- und Unterstützungsniveaus protokolliert.

## Details

- **Indikatoren**: Highest, Lowest
- **Parameter**:
  - `Period` – Rückblickbalken auf dem höheren Zeitrahmen.
  - `Higher Candle Type` – Zeitrahmen für die Niveauberechnung.
  - `Use High/Low` – High/Low- oder Close/Open-Preise verwenden.
- **Stops**: keine
- **Ziele**: keine
- **Long/Short**: keine (nur Indikator)
