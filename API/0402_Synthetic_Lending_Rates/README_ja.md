# 合成レンディングレート戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、デリバティブ市場から導出される合成レンディングレートとオンチェーンレンディング利回りの差異を活用します。レートが低い場所で借り入れ、レートが高い場所で貸し出すことで、両者のスプレッドを獲得します。

中立性を維持するためにポジションは定期的にリバランスされ、リスクはレート変動の閾値と流動性フィルターによって管理されます。

## 詳細

- **データ**: 無期限スワップのファンディングとDeFiレンディングレート。
- **エントリー**: スプレッド > 閾値のとき、低レートのvenueで借り入れ、高レートのvenueで貸し出す。
- **エグジット**: スプレッドが平均回帰するか、流動性が悪化したときにクローズ。
- **銘柄**: 無期限スワップとDeFiプラットフォーム。
- **リスク**: スプレッド上限と流動性ストップ。
