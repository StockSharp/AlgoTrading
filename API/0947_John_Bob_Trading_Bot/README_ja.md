# John Bob Trading Bot 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

50バーの高値/安値レベルとシンプルなフェアバリューギャップ検出を組み合わせたブレイクアウト戦略。ATRベースのストップロスと複数のテイクプロフィットレベルを持つ5つのスケール注文を開きます。

## 詳細

- **エントリー条件**:
  - ロング: 価格が50バーの安値を上抜けるか、強気のフェアバリューギャップが出現
  - ショート: 価格が50バーの高値を下抜けるか、弱気のフェアバリューギャップが出現
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 価格が5つのテイクプロフィットレベルのいずれかに到達
  - 価格がATRベースのストップロスに到達
- **ストップ**: ATR乗数
- **デフォルト値**:
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **フィルター**:
  - カテゴリ: ブレイクアウト
  - 方向: 両方
  - インジケーター: ATR, Highest, Lowest
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
