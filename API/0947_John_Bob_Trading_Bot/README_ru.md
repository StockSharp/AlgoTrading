# Стратегия John Bob Trading Bot
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя, сочетающая уровни максимума и минимума за 50 баров с простым обнаружением FVG. Открывает пять позиций с ATR-стопом и несколькими уровнями тейк‑профита.

## Детали

- **Условия входа**:
  - Лонг: цена пересекает вверх минимум за 50 баров или появляется бычий FVG
  - Шорт: цена пересекает вниз максимум за 50 баров или появляется медвежий FVG
- **Лонг/Шорт**: Оба
- **Условия выхода**:
  - Цена достигает одного из пяти уровней тейк‑профита
  - Цена достигает стоп‑лосса на основе ATR
- **Стопы**: Множитель ATR
- **Значения по умолчанию**:
  - `AtrMultiplier` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: ATR, Highest, Lowest
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
