# 遅延付き上方ギャップ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

セッションが閾値を超える上方ギャップで始まり、前回エントリーから指定した本数のバーが経過した場合にロングエントリーする戦略。ポジションは固定バー数保持される。

## 詳細

- **エントリー条件**: 上方ギャップが閾値より大きく、遅延期間を満たしている
- **ロング/ショート**: ロング
- **エグジット条件**: 保持期間が終了した後
- **ストップ**: いいえ
- **デフォルト値**:
  - `GapThreshold` = 1
  - `DelayPeriods` = 0
  - `HoldingPeriods` = 7
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ロング
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
