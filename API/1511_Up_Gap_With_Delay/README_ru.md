# Стратегия Up Gap с задержкой
[English](README.md) | [中文](README_cn.md)

Эта стратегия открывает длинную позицию, когда сессия открывается с ценовым разрывом вверх выше порога и прошло заданное количество баров после предыдущего входа. Позиция удерживается фиксированное число баров.

## Детали

- **Условия входа**: разрыв вверх больше порога и выполнена задержка
- **Длин/Корот**: Длинные
- **Условия выхода**: после истечения периода удержания
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `GapThreshold` = 1
  - `DelayPeriods` = 0
  - `HoldingPeriods` = 7
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Длинные
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
