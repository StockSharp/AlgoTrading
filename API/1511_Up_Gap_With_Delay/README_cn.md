# Up Gap 策略（带延迟）
[English](README.md) | [Русский](README_ru.md)

该策略在出现向上跳空并满足延迟期后做多，持有固定数量的K线。

## 细节

- **入场条件**: 向上跳空大于阈值且满足延迟
- **多空方向**: 多头
- **出场条件**: 持仓期结束
- **止损**: 无
- **默认值**:
  - `GapThreshold` = 1
  - `DelayPeriods` = 0
  - `HoldingPeriods` = 7
- **过滤器**:
  - 分类: 形态
  - 方向: 多头
  - 指标: 无
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
