# Consecutive Close High1 平均回帰戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

前回高値を上回る連続終値の回数をカウントし、その回数が閾値に達したら売りを入れるショート専用戦略です。価格が前回安値を下回ったときにポジションを決済します。オプションの200 EMAフィルターで下降トレンドを確認します。

## 詳細

- **エントリー条件**: 前回高値を上回る連続終値の回数が閾値に達する
- **ロング/ショート**: ショート
- **エグジット条件**: 前回安値を下回る終値
- **ストップ**: なし
- **デフォルト値**:
  - `Threshold` = 3
  - `EmaPeriod` = 200
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ショート
  - インジケーター: EMA
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
