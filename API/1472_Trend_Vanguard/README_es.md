# Estrategia Trend Vanguard
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Trend Vanguard usa un ZigZag simple sobre máximos y mínimos para seguir los giros de tendencia.
Cambia de dirección cuando el ZigZag cambia de orientación.

## Detalles

- **Criterios de entrada**: Reversión del ZigZag
- **Largo/Corto**: Ambos
- **Criterios de salida**: Señal opuesta del ZigZag
- **Stops**: No
- **Valores predeterminados**:
  - `Depth` = 21
- **Filtros**:
  - Categoría: Tendencia
  - Dirección: Ambos
  - Indicadores: Highest, Lowest
  - Stops: No
  - Complejidad: Básico
  - Marco temporal: Intradía
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Medio
