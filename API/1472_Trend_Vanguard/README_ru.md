# Стратегия Trend Vanguard
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Trend Vanguard использует простой ZigZag по максимумам и минимумам для отслеживания разворотов тренда.
Направление меняется при смене ориентации ZigZag.

## Детали

- **Критерий входа**: разворот ZigZag
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: противоположный сигнал ZigZag
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Depth` = 21
- **Фильтры**:
  - Категория: Тренд
  - Направление: Обе стороны
  - Индикаторы: Highest, Lowest
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
