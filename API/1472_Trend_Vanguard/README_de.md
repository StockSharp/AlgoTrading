# Trend-Vanguard-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Trend Vanguard verwendet einen einfachen ZigZag auf Hochs und Tiefs, um Trendumkehrungen zu folgen.
Die Richtung wechselt, wenn der ZigZag seine Orientierung ändert.

## Details

- **Einstiegskriterien**: ZigZag-Umkehr
- **Long/Short**: Beide
- **Ausstiegskriterien**: Entgegengesetztes ZigZag-Signal
- **Stops**: Nein
- **Standardwerte**:
  - `Depth` = 21
- **Filter**:
  - Kategorie: Trend
  - Richtung: Beide
  - Indikatoren: Highest, Lowest
  - Stops: Nein
  - Komplexität: Grundlegend
  - Zeitrahmen: Intraday
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
