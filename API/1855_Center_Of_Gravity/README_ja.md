# 重心戦略 (Center of Gravity)
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

SMAとWMAを乗算して結果を平滑化するCenter of Gravityインジケーターに基づく戦略。センターラインがスムーズ化された平均線を上抜けるとロングポジションを開き、逆のクロスオーバーでショートポジションを開きます。シグナルが現在の方向に対して反転するとポジションを閉じます。

## 詳細

- **エントリー条件**: センターラインがスムーズ化された平均線を越える
- **ロング/ショート**: 両方
- **エグジット条件**: シグナルがサイドを変える
- **ストップ**: なし
- **デフォルト値**:
  - `CandleType` = H4
  - `Period` = 10
  - `SmoothPeriod` = 3
- **フィルター**:
  - カテゴリ: インジケーター
  - 方向: 両方
  - インジケーター: SMA, WMA
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
