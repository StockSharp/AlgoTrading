# 十字線リバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Doji（十字線）ローソク足は、買い手と売り手の一時的な均衡を反映します。強い方向性のある動きの後にDojiが現れると、モメンタムが薄れるにつれてリバーサルの前兆となることがあります。この戦略は、真のDojiが形成されたかどうかを判断するために、ローソク足の実体をそのレンジと比較して測定します。

テストでは年平均リターンが約103%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

Dojiが検出されると、前のローソク足が上昇トレンドまたは下降トレンドを確認するためにチェックされます。下落後のDojiはロングエントリーを、上昇後のDojiはショートポジションを開くことができます。ストップはエントリーから一定のパーセント距離に設定され、価格がDojiの高値または安値を突き抜けると決済されます。

この手法はDojiからの最初の反応を捉えることを目的としており、素早いリバーサルが頻繁に展開されるイントラデイチャートに最も適しています。

## 詳細

- **エントリー条件**: 方向性のある動きの後のDojiローソク足。
- **ロング/ショート**: 両方。
- **エグジット条件**: 価格がDojiの高値/安値を超えるか、ストップロス。
- **ストップ**: はい、パーセントベース。
- **デフォルト値**:
  - `CandleType` = 5 minute
  - `DojiThreshold` = 0.1
  - `StopLossPercent` = 1
- **フィルター**:
  - カテゴリ: パターン
  - 方向: 両方
  - インジケーター: Candlestick
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
