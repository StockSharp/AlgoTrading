# Momentumローソク足シグナル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、ローソク足の始値と終値から計算されたモメンタム値のクロスオーバーに基づいて取引します。始値のモメンタムが終値のモメンタムを下回ると、強気圧力の高まりを示し、ロングポジションを建てます。逆のクロスオーバーは弱気圧力を示し、ショートポジションを発動します。

デフォルトでは、モメンタム期間12の12時間足で動作します。

## 詳細

- **エントリー条件**:
  - **ロング**: 始値のモメンタムが終値のモメンタムを下から上に交差。
  - **ショート**: 始値のモメンタムが終値のモメンタムを上から下に交差。
- **ロング/ショート**: 両方向。
- **エグジット条件**: 逆のクロスオーバー。
- **ストップ**: なし。
- **フィルター**: なし。
