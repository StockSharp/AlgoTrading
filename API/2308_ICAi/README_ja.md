# ICAi 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ICA適応移動平均インジケーターに基づいた戦略です。インジケーターは価格を平滑化し、標準偏差を使用して傾きを適応させます。インジケーターが上向きに転じるとロングポジションが開かれ、下向きに転じるとショートポジションが開かれます。

アルゴリズムはローソク足データが利用可能なすべての市場で動作します。デフォルト設定では4時間足と平滑化長12を使用します。

## 詳細

- **エントリー条件**:
  - ロング: `Prev < PrevPrev && Current >= Prev`
  - ショート: `Prev > PrevPrev && Current <= Prev`
- **ロング/ショート**: 両方
- **エグジット条件**: 逆シグナル
- **ストップ**: オプションの固定ストップロスとテイクプロフィット
- **デフォルト値**:
  - `Length` = 12
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
  - `TakeProfit` = 2000
  - `StopLoss` = 1000
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: ICAi
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
