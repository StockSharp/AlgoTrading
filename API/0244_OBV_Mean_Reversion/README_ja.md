# OBV平均回帰戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

On Balance Volume（OBV）は累積出来高フローを追跡し、買い手と売り手のどちらが優勢かを判断します。この戦略はOBVが平均から大きく乖離するのを待ち、典型的な水準への回帰を予測して取引します。

テストでは平均年間リターンが約79%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

OBVがその平均値から`Multiplier`倍の標準偏差を引いた値を下回り、価格が移動平均を下回るときに買いシグナルが発生します。OBVが上部バンドを超え、価格が平均を上回るときに売りシグナルが生成されます。OBVが平均線を再び交差したらポジションを決済します。

このアプローチは価格行動に加えて出来高フローを考慮するトレーダーに有益です。出来高が加速し続ける状況に対処するため、設定されたパーセンテージにストップを置きます。

## 詳細
- **エントリー条件**:
  - **ロング**: OBV < Avg - Multiplier * StdDev && Close < MA
  - **ショート**: OBV > Avg + Multiplier * StdDev && Close > MA
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - **ロング**: OBV > Avg のときに決済
  - **ショート**: OBV < Avg のときに決済
- **ストップ**: あり、パーセントストップロス。
- **デフォルト値**:
  - `AveragePeriod` = 20
  - `Multiplier` = 2.0m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: OBV
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
