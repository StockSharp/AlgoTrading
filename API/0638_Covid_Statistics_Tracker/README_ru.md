# Стратегия Covid Statistics Tracker
[English](README.md) | [中文](README_cn.md)

Стратегия торгует на основании коэффициента роста подтверждённых случаев COVID-19.
Продаёт при ускорении роста и покупает при замедлении.

## Подробности

- **Условия входа**:
  - Длинная: `growth < 1`
  - Короткая: `growth > 1`
- **Long/Short**: Оба
- **Условия выхода**: Противоположный сигнал
- **Стопы**: Нет
- **Параметры по умолчанию**:
  - `Region` = "US"
  - `Lookback` = 2
  - `CandleType` = TimeSpan.FromDays(1).TimeFrame()
- **Фильтры**:
  - Категория: Other
  - Направление: Оба
  - Индикаторы: Custom
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Дневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
