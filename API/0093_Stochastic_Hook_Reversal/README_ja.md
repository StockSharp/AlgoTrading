# Stochastic Hook Reversal Strategy
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Stochastic Hook Reversal戦略は、%Kラインが買われすぎまたは売られすぎの領域からフックする動きを監視します。極端な水準に達した後、オシレーターはしばしば戻るように曲がり、モメンタムが弱まっていることを示します。

テストでは平均年間リターンが約166%であることが示されています。株式市場で最もパフォーマンスが優れています。

%Kが20以下から上向きに転じ、価格が新安値を押す場合にロングエントリーします。価格が最後の上昇を見せている間に%Kが80以上から下向きにフックする場合にショートエントリーします。

ポジションは小さなパーセントストップを使用し、ストキャスティクスが反対方向にフックするかストップに達したときにクローズします。

## 詳細

- **エントリー条件**: インジケーターシグナル
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたは逆シグナル
- **ストップ**: はい、パーセントベース
- **デフォルト値**:
  - `CandleType` = 15分
  - `StopLoss` = 2%
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: Stochastic
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
