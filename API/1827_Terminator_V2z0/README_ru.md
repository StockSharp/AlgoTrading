# Стратегия Terminator V2
[English](README.md) | [中文](README_cn.md)

Данная стратегия использует осциллятор MACD для торговли в обе стороны. Длинная позиция открывается, когда линия MACD пересекает сигнальную линию снизу вверх. Короткая позиция открывается при пересечении сверху вниз. Позиции защищаются фиксированными уровнями стоп-лосса и тейк-профита. Дополнительный трейлинг-стоп помогает фиксировать прибыль при сильном движении цены.

## Детали

- **Условия входа**:
  - **Длинная позиция**: `MACD` пересекает сигнальную линию снизу вверх.
  - **Короткая позиция**: `MACD` пересекает сигнальную линию сверху вниз.
- **Направление**: обе стороны.
- **Условия выхода**:
  - Срабатывание стоп-лосса или тейк-профита.
  - Срабатывание трейлинг-стопа.
- **Стопы**: да, поддерживаются стоп-лосс, тейк-профит и опциональный трейлинг-стоп.
- **Значения по умолчанию**:
  - `FastPeriod` = 14
  - `SlowPeriod` = 26
  - `SignalPeriod` = 1
  - `TakeProfit` = 500 пунктов
  - `StopLoss` = 2500 пунктов
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: MACD
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Краткосрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
