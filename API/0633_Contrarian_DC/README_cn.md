# 逆势DC策略
[English](README.md) | [Русский](README_ru.md)

逆势DC策略利用Donchian通道在价格触及通道边界时反向交易。价格触及下轨时买入，触及上轨时卖出。若止损被触发，则在相同方向的交易中暂停指定根数的K线。风险控制基于百分比止损和按风险回报比设定的止盈。

## 细节
- **入场条件**：
  - **多头**：最低价 <= Donchian下轨 && 满足暂停条件
  - **空头**：最高价 >= Donchian上轨 && 满足暂停条件
- **方向**：双向
- **出场条件**：
  - **止损**：百分比止损
  - **目标**：按风险回报比设定的止盈
  - **通道**：触及相反通道时平仓
- **止损**：是，百分比
- **默认值**：
  - `DonchianPeriod` = 20
  - `RiskRewardRatio` = 1.7m
  - `StopLossPercent` = 0.3m
  - `PauseCandles` = 3
  - `CandleType` = TimeSpan.FromMinutes(15)
- **筛选**：
  - 分类：均值回归
  - 方向：双向
  - 指标：Donchian Channel
  - 止损：是
  - 复杂度：基础
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
