# Контртрендовая стратегия DC
[English](README.md) | [中文](README_cn.md)

Контртрендовая стратегия Donchian Channel торгует от границ канала. Покупки совершаются при касании нижней полосы, продажи при касании верхней. После срабатывания стоп-лосса входы в том же направлении приостанавливаются на заданное число свечей. Управление рисками основано на процентном стопе и тейк-профите по отношению риск/прибыль.

## Подробности
- **Условия входа**:
  - **Long**: минимум цены <= нижняя полоса Donchian && выполнена пауза
  - **Short**: максимум цены >= верхняя полоса Donchian && выполнена пауза
- **Направление**: обе стороны.
- **Условия выхода**:
  - **Стоп**: процентный стоп-лосс
  - **Цель**: тейк-профит по соотношению риск/прибыль
  - **Полоса**: закрытие при достижении противоположной полосы
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `DonchianPeriod` = 20
  - `RiskRewardRatio` = 1.7m
  - `StopLossPercent` = 0.3m
  - `PauseCandles` = 3
  - `CandleType` = TimeSpan.FromMinutes(15)
- **Фильтры**:
  - Категория: Средняя реверсия
  - Направление: Обе
  - Индикаторы: Donchian Channel
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
