# Стратегия Revelations
[English](README.md) | [中文](README_cn.md)

Стратегия прорыва волатильности, которая входит в рынок при сильных всплесках ATR, подтверждённых локальными экстремумами и индекcом режима. Размер позиции адаптируется к силе всплеска.

## Детали

- **Условия входа**:
  - **Лонг**: всплеск ATR вверх на локальном минимуме при подтверждённом режиме.
  - **Шорт**: всплеск ATR вниз на локальном максимуме при подтверждённом режиме.
- **Лонг/Шорт**: оба направления.
- **Условия выхода**:
  - срабатывание тейк-профита или стоп-лосса.
- **Стопы**: фиксированные проценты.
- **Значения по умолчанию**:
  - `ATR Fast` = 14
  - `ATR Slow` = 21
  - `ATR StdDev` = 12
  - `Spike Threshold` = 0.5
  - `Super Spike Mult` = 1.5
  - `Regime Window` = 8
  - `Regime Events` = 3
  - `Local Window` = 3
  - `Max Quantity` = 2
  - `Min Quantity` = 1
  - `Stop %` = 0.9
  - `Take Profit %` = 1.8
- **Фильтры**:
  - Категория: Прорыв волатильности
  - Направление: Лонг/Шорт
  - Индикаторы: ATR, SMA, Highest/Lowest
  - Стопы: Да
  - Сложность: Продвинутая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
