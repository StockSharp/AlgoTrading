# ゴールドRSIダイバージェンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ゴールドRSIダイバージェンス戦略は、価格と相対力指数（RSI）の間の強気・弱気ダイバージェンスを識別することで、ゴールドをスキャルピングします。
価格が新しい安値を付けながらRSIがより高い安値を示すとき、戦略は買いを探します。
逆に、価格が新しい高値を付けながらRSIがより低い高値を示すとき、戦略は売ります。
両方のセットアップは、設定可能なバー範囲内に2つのピボットが発生した場合にのみ確認されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格が低い安値、RSIが高い安値、RSI < 40。
  - **ショート**: 価格が高い高値、RSIが低い高値、RSI > 60。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - ストップロスとテイクプロフィットを使用。
- **ストップ**: pipsでの固定ストップロスとテイクプロフィット。
- **デフォルト値**:
  - `RsiLength` = 60
  - `StopLossPips` = 11
  - `TakeProfitPips` = 33
- **フィルター**:
  - カテゴリ: ダイバージェンス
  - 方向: 両方
  - インジケーター: RSI
  - ストップ: はい
  - 複雑さ: 中程度
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
