# Gold RSI Divergence 策略
[English](README.md) | [Русский](README_ru.md)

Gold RSI Divergence 策略通过识别价格与相对强弱指数 (RSI) 之间的背离来对黄金进行短线交易。
当价格创出新低而 RSI 创出更高低点时，策略做多；
当价格创出新高而 RSI 创出更低高点时，策略做空。
两次枢轴之间的间隔必须落在设定的范围内才被视为有效信号。

## 细节

- **入场条件**：
  - **多头**：价格更低低点、RSI 更高低点、RSI < 40。
  - **空头**：价格更高高点、RSI 更低高点、RSI > 60。
- **方向**：双向。
- **出场条件**：
  - 使用止损和止盈。
- **止损**：以 pips 定义的固定止损和止盈。
- **默认值**：
  - `RsiLength` = 60
  - `StopLossPips` = 11
  - `TakeProfitPips` = 33
- **过滤器**：
  - 分类：背离
  - 方向：双向
  - 指标：RSI
  - 止损：是
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：是
  - 风险等级：中等
