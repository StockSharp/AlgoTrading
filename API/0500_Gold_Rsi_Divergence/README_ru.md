# Стратегия Gold RSI Divergence
[English](README.md) | [中文](README_cn.md)

Стратегия Gold RSI Divergence предназначена для скальпинга золота, используя дивергенцию между ценой и индексом относительной силы (RSI).
Когда цена обновляет минимум, а RSI формирует более высокий минимум, стратегия рассматривает покупку.
Когда цена обновляет максимум, а RSI формирует более низкий максимум, стратегия продаёт.
Оба сигнала учитываются только если две точки экстремума появляются в заданном диапазоне баров.

## Подробности

- **Условия входа**:
  - **Лонг**: цена делает более низкий минимум, RSI выше предыдущего минимума, RSI < 40.
  - **Шорт**: цена делает более высокий максимум, RSI ниже предыдущего максимума, RSI > 60.
- **Направление**: обе стороны.
- **Условия выхода**:
  - Используются стоп‑лосс и тейк‑профит.
- **Стопы**: фиксированные стоп‑лосс и тейк‑профит в пипсах.
- **Параметры по умолчанию**:
  - `RsiLength` = 60
  - `StopLossPips` = 11
  - `TakeProfitPips` = 33
- **Фильтры**:
  - Категория: Дивергенция
  - Направление: Обе
  - Индикаторы: RSI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
