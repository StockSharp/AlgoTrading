# Gaussian Detrended 平均回帰戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Gaussian Detrended Reversionは、Arnaud Legoux移動平均（ALMA）で平滑化されたデトレンド価格オシレーターを使用する平均回帰戦略です。平滑化されたオシレーターがゼロを下回った状態でその遅延バージョンを上抜けするとロングポジションを建て、ゼロを上回った状態で下抜けするとショートポジションを建てます。逆方向のクロスまたはオシレーターがゼロラインを越えた際にポジションを決済します。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: ALMAで平滑化されたDPOが遅延ラインを上抜けし、ゼロを下回っている。
  - **ショート**: ALMAで平滑化されたDPOが遅延ラインを下抜けし、ゼロを上回っている。
- **エグジット条件**: 逆方向の遅延クロスまたはゼロラインクロス。
- **ストップ**: なし。
- **デフォルト値**:
  - `PriceLength` = 52
  - `SmoothingLength` = 52
  - `LagLength` = 26
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロング/ショート
  - インジケーター: EMA, ALMA
  - 複雑さ: 低
  - リスクレベル: 中
