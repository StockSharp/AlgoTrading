# InstantaneousTrendFilter 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は John Ehlers の瞬間トレンドラインとトリガーラインを使用して、任意の時間軸でシグナルを生成します。トリガーは `2 * ITrend - ITrend[2]` として計算され、より遅いトレンドラインをクロスする高速ラインを形成します。下向きクロスオーバーはショートポジションをクローズしてロングを開き、上向きクロスオーバーはロングをクローズしてショートを開きます。平滑化係数 `Alpha` は応答性を制御します：低い値はよりスムーズなラインを生成し、高い値はより速く反応します。

## 詳細

- **エントリー条件**:
  - **ロング**: トリガーが前のバーでトレンドラインを上回っており、現在のバーで下をクロスします。
  - **ショート**: トリガーが前のバーでトレンドラインを下回っており、現在のバーで上をクロスします。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - ショートシグナルが現れたとき、ロングポジションがクローズされます。
  - ロングシグナルが現れたとき、ショートポジションがクローズされます。
- **ストップ**: デフォルトではなし。
- **デフォルト値**:
  - `Alpha` = 0.07。
  - `Candle Type` = 4 時間足時間軸。
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
