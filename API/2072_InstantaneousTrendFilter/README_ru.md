# Стратегия Instantaneous Trend Filter
[English](README.md) | [中文](README_cn.md)

Эта стратегия использует фильтр мгновенного тренда Джона Элерса и триггерную линию для генерации сигналов на любом таймфрейме. Триггер вычисляется как `2 * ITrend - ITrend[2]`, образуя быструю линию, пересекающую более медленную трендовую. Пересечение сверху вниз закрывает короткие позиции и открывает длинную, а пересечение снизу вверх закрывает длинные и открывает короткую. Параметр сглаживания `Alpha` определяет чувствительность индикатора: чем он меньше, тем плавнее линии, чем больше — тем быстрее реакция.

## Детали

- **Условия входа**:
  - **Длинная позиция**: триггер был выше трендовой на предыдущем баре и пересекает её снизу на текущем.
  - **Короткая позиция**: триггер был ниже трендовой на предыдущем баре и пересекает её сверху на текущем.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**:
  - Длинные позиции закрываются при появлении сигнала на продажу.
  - Короткие позиции закрываются при появлении сигнала на покупку.
- **Стопы**: по умолчанию отсутствуют.
- **Значения по умолчанию**:
  - `Alpha` = 0.07.
  - `Candle Type` = таймфрейм 4 часа.
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Обе
  - Индикаторы: Один
  - Стопы: Нет
  - Сложность: Простая
  - Таймфрейм: Среднесрочный
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенции: Нет
  - Уровень риска: Средний
