# Стратегия Upthrust Reversal

Upthrust Reversal — медвежий аналог spring: цена коротко пробивает сопротивление и затем быстро падает назад.
Такой ход выбивает запоздалых покупателей перед разворотом вниз.

Стратегия открывает короткую позицию, когда цена опускается ниже уровня пробоя, предполагая, что предложение превысит спрос.

Стоп размещается чуть выше максимума upthrust и ограничивает риск; позиция закрывается, если цена восстановится выше этого уровня.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Разворот
  - Направление: Обе
  - Индикаторы: Wyckoff
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
