# Keltner Channelリバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ボラティリティベースのチャネルは、過度に伸びた動きを際立たせることができます。この手法は、価格がKeltner Channelの外に押し出されたときに逆張りで入り、中央ラインへの戻りを予測します。チャネルの幅を算出するために指数移動平均とATRを使用します。

テストでは年平均リターンが約106%であることが示されています。株式市場で最も良いパフォーマンスを発揮します。

各ローソク足が完成するたびに、戦略は終値が上限バンドまたは下限バンドを超えているか、またローソク足の方向が一致しているかを確認します。下限バンドの外側で終値を付けた強気のローソク足はロングエントリーを促し、上限バンドの外側で終値を付けた弱気のローソク足はショートポジションを促します。価格が中央バンドを横断するか、ATRベースのストップに達した時点でポジションを決済します。

短期的な極値の反対方向で取引することにより、このシステムはより広いレンジ内での素早い平均回帰の動きを求めます。

## 詳細

- **エントリー条件**: ローソク足の方向にKeltner Channelの外側での終値。
- **ロング/ショート**: 両方。
- **エグジット条件**: 価格が中央バンドを越えるか、ストップロス。
- **ストップ**: はい、ATRベース。
- **デフォルト値**:
  - `EmaPeriod` = 20
  - `AtrPeriod` = 14
  - `AtrMultiplier` = 2.0
  - `StopLossAtrMultiplier` = 2.0
  - `CandleType` = 5 minute
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 両方
  - インジケーター: Keltner Channel
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
