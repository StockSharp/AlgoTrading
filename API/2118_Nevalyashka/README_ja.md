# Nevalyashka戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、マルチンゲールのポジションサイジングによるシンプルなロング/ショート交互システムを実装しています。

## 戦略ロジック

1. 開始時にショートポジションを建てます。
2. ポジションに固定のテイクプロフィットとストップロスを設定します。
3. ポジションが決済される（ストップまたはターゲットによって）たびに:
   - 次のトレードは逆方向に建てられます。
   - 前のトレードが損失で終わった場合、注文のボリュームは`LotMultiplier`で乗算されます。
   - 前のトレードが利益で終わった場合、ボリュームはベースの`Volume`にリセットされます。
4. ステップ2‑3が無期限に繰り返されます。

## パラメーター

- `Volume` – 最初のトレードと勝ちトレード後に使用されるベース注文ボリューム。
- `LotMultiplier` – 負けトレード後にボリュームに適用される乗数。
- `TakeProfit` – 価格ポイントでの利益目標距離。
- `StopLoss` – 価格ポイントでのストップロス距離。

## 備考

- 保護注文は`StartProtection`を通じて管理されます。
- 戦略は市場データに依存しません。ポジションの変化にのみ反応します。
