# Estratégia Cycle Biologique
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

A estratégia Cycle Biologique opera um ciclo baseado em seno. Entra comprado quando o ciclo cruza acima de zero e sai quando o ciclo cruza abaixo de zero.

## Detalhes

- **Critérios de entrada**: O ciclo cruza acima de zero.
- **Critérios de saída**: O ciclo cruza abaixo de zero.
- **Valores padrão**:
  - `CycleLength` = 30
  - `Amplitude` = 1.0
  - `Offset` = 0
- **Filtros**:
  - Categoria: Cycle
  - Direção: Comprado
  - Indicadores: Sine Wave
  - Stops: Não
  - Complexidade: Baixo
  - Período: Qualquer
  - Sazonalidade: Não
  - Redes neurais: Não
  - Divergência: Não
  - Nível de risco: Médio
