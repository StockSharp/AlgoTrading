# Cycle Biologique Strategy
[Русский](README_ru.md) | [中文](README_cn.md)

The Cycle Biologique strategy trades a sine-based cycle. It goes long when the cycle crosses above zero and exits when the cycle crosses below zero.

## Details

- **Entry Criteria**: Cycle crosses above zero.
- **Exit Criteria**: Cycle crosses below zero.
- **Default Values**:
  - `CycleLength` = 30
  - `Amplitude` = 1.0
  - `Offset` = 0
- **Filters**:
  - Category: Cycle
  - Direction: Long
  - Indicators: Sine Wave
  - Stops: No
  - Complexity: Low
  - Timeframe: Any
  - Seasonality: No
  - Neural networks: No
  - Divergence: No
  - Risk level: Medium
