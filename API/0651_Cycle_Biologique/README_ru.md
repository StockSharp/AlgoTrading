# Стратегия Cycle Biologique
[English](README.md) | [中文](README_cn.md)

Стратегия Cycle Biologique торгует на основе синусоидального цикла. Вход в лонг выполняется при пересечении циклом нуля снизу вверх, выход — при пересечении сверху вниз.

## Подробности

- **Условия входа**: Цикл пересекает ноль снизу вверх.
- **Условия выхода**: Цикл пересекает ноль сверху вниз.
- **Параметры по умолчанию**:
  - `CycleLength` = 30
  - `Amplitude` = 1.0
  - `Offset` = 0
- **Фильтры**:
  - Категория: Cycle
  - Направление: Лонг
  - Индикаторы: Синусоида
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
