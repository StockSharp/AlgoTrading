# Estrategia Cycle Biologique
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

La estrategia Cycle Biologique opera un ciclo basado en seno. Entra en largo cuando el ciclo cruza por encima de cero y sale cuando el ciclo cruza por debajo de cero.

## Detalles

- **Criterios de entrada**: El ciclo cruza por encima de cero.
- **Criterios de salida**: El ciclo cruza por debajo de cero.
- **Valores predeterminados**:
  - `CycleLength` = 30
  - `Amplitude` = 1.0
  - `Offset` = 0
- **Filtros**:
  - Categoría: Cycle
  - Dirección: Largo
  - Indicadores: Sine Wave
  - Stops: No
  - Complejidad: Bajo
  - Marco temporal: Cualquiera
  - Estacionalidad: No
  - Redes neuronales: No
  - Divergencia: No
  - Nivel de riesgo: Medio
