# Cycle Biologique-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Die Cycle Biologique-Strategie handelt einen sinusbasierten Zyklus. Sie geht long, wenn der Zyklus die Null von unten kreuzt, und schließt die Position, wenn der Zyklus die Null von oben kreuzt.

## Details

- **Einstiegskriterien**: Der Zyklus kreuzt die Null von unten nach oben.
- **Ausstiegskriterien**: Der Zyklus kreuzt die Null von oben nach unten.
- **Standardwerte**:
  - `CycleLength` = 30
  - `Amplitude` = 1.0
  - `Offset` = 0
- **Filter**:
  - Kategorie: Cycle
  - Richtung: Long
  - Indikatoren: Sine Wave
  - Stops: Nein
  - Komplexität: Niedrig
  - Zeitrahmen: Beliebig
  - Saisonalität: Nein
  - Neuronale Netze: Nein
  - Divergenz: Nein
  - Risikolevel: Mittel
