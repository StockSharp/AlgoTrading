# マルチ時間軸トレンドフォロー（200 EMA フィルター付き）- ロングのみ
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

5 分、15 分、30 分チャートで速い EMA が遅い EMA を上回り、かつ 5 分チャートで価格が 200 EMA を上回っているときにロングを建てます。いずれかの時間軸が弱気に転じるか、価格が 200 EMA を下回った場合にポジションを決済します。

## 詳細

- **エントリー条件**:
  - **ロング**: 5 分、15 分、30 分の時間軸で速い EMA > 遅い EMA、かつ終値 > 200 EMA (5m)。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**:
  - いずれかの時間軸のトレンドが弱気に転じるか、終値 < 200 EMA (5m)。
- **ストップ**:
  - ストップロス: パーセント。
  - 利食い: パーセント。
- **デフォルト値**:
  - `Fast EMA Length` = 9
  - `Slow EMA Length` = 21
  - `200 EMA Length` = 200
  - `Stop Loss %` = 1
  - `Take Profit %` = 3
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: ロングのみ
  - インジケーター: EMA
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: ベース 5m、15m および 30m で確認
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
