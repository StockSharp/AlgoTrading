# Стратегия Multi-Timeframe Trend Following with 200 EMA Filter - Longs Only
[English](README.md) | [中文](README_cn.md)

Стратегия открывает длинную позицию, когда быстрая EMA выше медленной на графиках 5, 15 и 30 минут и цена находится выше 200 EMA на 5-минутном графике. Позиция закрывается, если любой таймфрейм становится медвежьим или цена опускается ниже 200 EMA.

## Детали

- **Условия входа**:
  - **Лонг**: быстрая EMA > медленной на 5, 15 и 30 мин и цена > 200 EMA (5м).
- **Лонг/Шорт**: только лонг.
- **Условия выхода**:
  - любой таймфрейм показывает нисходящий тренд или цена < 200 EMA (5м).
- **Стопы**:
  - Стоп-лосс: процент.
  - Тейк-профит: процент.
- **Значения по умолчанию**:
  - `Fast EMA Length` = 9
  - `Slow EMA Length` = 21
  - `200 EMA Length` = 200
  - `Stop Loss %` = 1
  - `Take Profit %` = 3
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Лонг
  - Индикаторы: EMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: 5м база с подтверждением 15м и 30м
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
