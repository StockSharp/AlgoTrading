# Delta WPR
[English](README.md) | [中文](README_cn.md)

Стратегия Delta WPR сравнивает быстрый и медленный осцилляторы Williams %R для поиска изменений импульса. Когда быстрый показатель превышает медленный, а медленный находится выше заданного уровня, открывается длинная позиция и закрываются любые короткие. Обратная ситуация — быстрый ниже медленного и медленный ниже уровня — ведёт к открытию короткой позиции. Обработка происходит только после формирования очередной свечи, что уменьшает шум.

Тестирование на данных четырёхчасового таймфрейма показывает наилучшие результаты на боковом рынке, где Williams %R колеблется между зонами перепроданности и перекупленности.

## Подробности

- **Условия входа**:
  - Длинная: `WPR медленный > Level && WPR быстрый > WPR медленный`
  - Короткая: `WPR медленный < Level && WPR быстрый < WPR медленный`
- **Длинная/Короткая**: Оба направления.
- **Условия выхода**: Противоположный сигнал.
- **Стопы**: Нет.
- **Значения по умолчанию**:
  - `FastPeriod` = 14
  - `SlowPeriod` = 30
  - `Level` = -50m
  - `CandleType` = TimeSpan.FromHours(4)
- **Фильтры**:
  - Категория: Осциллятор
  - Направление: Оба
  - Индикаторы: WilliamsR
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: 4h
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
