# デュアルモメンタム戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

デュアルモメンタムを使用してリスク資産と安全資産の間でローテーションします。
リスク資産のモメンタムが正でかつ安全資産のモメンタムを上回る場合にのみ、リスク資産に投資します。

## 詳細

- **エントリー条件**: リスク資産のモメンタム > 0 かつ安全資産のモメンタムより大きい
- **ロング/ショート**: ロングのみ
- **エグジット条件**: 条件が満たされなくなった場合に安全資産へ切り替え
- **ストップ**: なし
- **デフォルト値**:
  - `Period` = 12
  - `CandleType` = TimeSpan.FromDays(30).TimeFrame()
- **フィルター**:
  - カテゴリ: モメンタム
  - 方向: ロングのみ
  - インジケーター: RateOfChange
  - ストップ: なし
  - 複雑さ: 基本
  - 時間軸: 月足
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
