# Ultimate Strategy Template
[English](README.md) | [中文](README_cn.md)

Базовый шаблон стратегии на пересечении быстрых и медленных средних. Открывает длинные или короткие позиции при пересечении и поддерживает процентные стоп-лосс и тейк-профит.

## Подробности

- **Условия входа**: Пересечение быстрой и медленной SMA.
- **Направление**: Длинные и короткие позиции.
- **Условия выхода**: Обратное пересечение или срабатывание защит.
- **Стопы**: Процентные стоп-лосс и тейк-профит.
- **Значения по умолчанию**:
  - `FastLength` = 9
  - `SlowLength` = 21
  - `StopLossPercent` = 1
  - `TakeProfitPercent` = 3
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: SMA
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Средний
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
