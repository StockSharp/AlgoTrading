# MACD Waterline Cross Expectator 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、MACDシグナルラインがゼロレベルを上抜けたときにロング、下抜けたときにショートになります。リスク管理にはストップロスと設定可能なリスク・リワード乗数を使用してテイクプロフィット距離を設定します。

## ロジック
- 設定可能な速いEMA、遅いEMA、シグナル期間でMACDインジケーターを計算する。
- 完了した各ローソク足でシグナルライン値を追跡する。
- シグナルラインがマイナスからプラスに転換し、戦略が買いの準備ができている場合、ロング成行注文が出される。
- シグナルラインがプラスからマイナスに転換し、戦略が売りの準備ができている場合、ショート成行注文が出される。
- 新しいポジションごとにストップロスとテイクプロフィットレベルが自動的に設定される。

## パラメーター
- **FastEmaPeriod** – MACDで使用する速いEMAの長さ。
- **SlowEmaPeriod** – MACDで使用する遅いEMAの長さ。
- **SignalPeriod** – シグナルラインEMAの長さ。
- **StopLoss** – 絶対価格単位でのストップロスまでの距離。
- **Volume** – 新しいポジションに使用する注文サイズ。
- **RiskBenefitRatios** – テイクプロフィット距離を定義する1:5から1:1の事前設定された比率。
- **CandleType** – 戦略が使用するローソク足の時間軸。

## 備考
- 戦略は内部フラグを使用してロングとショートの取引を交互に行います。
- 取引は成行価格で執行され、常に現在のポジションを閉じて反転させます。
