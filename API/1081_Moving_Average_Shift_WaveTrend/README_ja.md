# 移動平均シフト WaveTrend 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は設定可能な移動平均とWaveTrendスタイルのオシレーターを組み合わせます。価格が移動平均より上でオシレーターが上昇しているとき、長期EMAとボラティリティフィルターでトレンドを確認してロング取引を行います。逆の条件でショートが発動します。ポジションはパーセンテージのストップロス、テイクプロフィット、トレーリングストップで保護されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 価格がMAより上、オシレーター > 0 かつ上昇中、長期トレンドが上向き、ATRが平均を超えている、取引時間内、すでにウェーブ中でない。
  - **ショート**: 価格がMAより下、オシレーター < 0 かつ下降中、長期トレンドが下向き、ATRが平均を超えている、取引時間内、すでにウェーブ中でない。
- **ロング/ショート**: 両方。
- **エグジット条件**:
  - 価格がMAをクロスしてオシレーターが反転、またはトレーリングストップ、または保護ストップ。
- **ストップ**: あり。
- **デフォルト値**:
  - `MaType` = SMA
  - `MaLength` = 40
  - `OscLength` = 15
  - `TakeProfitPercent` = 1.5
  - `StopLossPercent` = 1
  - `TrailPercent` = 1
  - `LongMaLength` = 200
  - `AtrLength` = 14
  - `StartHour` = 9
  - `EndHour` = 17
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: MA, Hull MA, ATR
  - ストップ: あり
  - 複雑さ: 中級
  - 時間軸: 中期
