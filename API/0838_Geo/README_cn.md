# Geo 策略
[English](README.md) | [Русский](README_ru.md)

当蜡烛高低比接近黄金分割时买入的策略。

## 细节

- **入场条件**：高低比在 φ 容差范围内。
- **多/空**：双向。
- **出场条件**：相反条件。
- **止损**：无。
- **默认值**：
  - `Tolerance` = 1
  - `CandleType` = TimeSpan.FromMinutes(5)
- **过滤器**：
  - 类别：形态
  - 方向：双向
  - 指标：蜡烛比率
  - 止损：无
  - 复杂度：基础
  - 时间框架：日内 (5m)
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：中等
