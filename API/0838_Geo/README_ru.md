# Стратегия Geo
[English](README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Стратегия покупает, когда отношение high/low свечи близко к золотому сечению.

## Детали

- **Критерии входа**: Отношение high/low в пределах допуска от φ.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: Противоположное условие.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `Tolerance` = 1
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Паттерн
  - Направление: Оба
  - Индикаторы: Отношение свечи
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
