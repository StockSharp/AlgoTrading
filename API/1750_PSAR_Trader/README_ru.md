# Стратегия PSAR Trader
[English](README.md) | [中文](README_cn.md)

Стратегия PSAR Trader торгует по разворотам индикатора Parabolic SAR. Когда точка SAR перемещается ниже цены, открывается длинная позиция; при переходе SAR выше цены открывается короткая. Параметр "Close On Opposite" позволяет закрыть и развернуть позицию при противоположном сигнале. Торговля ведётся только в указанные часы сессии. Стоп‑лосс и тейк‑профит выставляются модулем защиты.

## Детали

- **Условие входа**: Пересечение цены и Parabolic SAR.
- **Лонг/Шорт**: Оба направления.
- **Условие выхода**: Обратное пересечение SAR или разворот позиции.
- **Стопы**: Да, фиксированные.
- **Значения по умолчанию**:
  - `SarStep` = 0.001m
  - `SarMaxStep` = 0.2m
  - `StartHour` = 0
  - `EndHour` = 23
  - `CloseOnOpposite` = true
  - `TakeValue` = 50 (абсолютное)
  - `StopValue` = 50 (абсолютное)
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Trend
  - Направление: Оба
  - Индикаторы: Parabolic SAR
  - Стопы: Фиксированные
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
