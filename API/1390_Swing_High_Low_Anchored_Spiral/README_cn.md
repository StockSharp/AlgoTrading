# 摆动高低锚定螺旋
[English](README.md) | [Русский](README_ru.md)

该指标在最近的摆动高点或低点绘制一条几何螺旋线，螺旋覆盖从锚定点到当前价格的范围。

## 详情

- **输入**：
  - 摆动长度。
  - 螺旋圈数。
  - 锚定模式（高点或低点）。
- **信号**：无，仅用于可视化。
- **用途**：帮助突出摆动结构及其附近潜在区域。
- **过滤器**：
  - 类别：形态识别
  - 方向：无
  - 指标：Pivot
  - 止损：无
  - 复杂度：中等
  - 时间框架：任意
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：低
