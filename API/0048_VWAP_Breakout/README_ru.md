# Прорыв VWAP
[English](README.md) | [中文](README_cn.md)

Стратегия ищет пересечение цены с объёмно-взвешенной средней ценой (VWAP) с противоположной стороны. Прорыв выше VWAP говорит о бычьих настроениях, а пробой ниже — о медвежьих.

Тестирование показывает среднегодичную доходность около 181%\. Стратегию лучше запускать на крипторынке.

Система ждёт закрытия свечи по другую сторону VWAP и затем торгует в этом направлении. Выход происходит при обратном пересечении VWAP.

Так как VWAP отражает среднюю цену сделок, его прорывы часто приводят к импульсным движениям.

## Подробности

- **Критерий входа**: цена закрывается по другую сторону VWAP.
- **Длинные/короткие**: обе стороны.
- **Критерий выхода**: обратное пересечение VWAP или стоп.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе стороны
  - Индикаторы: VWAP
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
