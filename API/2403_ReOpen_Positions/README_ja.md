# ポジション再オープン戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMQL5サンプル`Exp_ReOpenPositions`のStockSharpポートです。現在の取引が利益を生み始めたときにポジションを再オープンする方法を示しています。

## ロジック

1. 戦略は開始時に最初のロングポジションをオープンします。
2. 最後のエントリー価格から`ProfitThreshold`ポイント価格が上昇すると、別のロングポジションをオープンします。
3. 各新しいエントリーは、その価格に対してストップロスとテイクプロフィットレベルを更新します。
4. 価格がストップロスまたはテイクプロフィットに達すると、すべてのポジションがクローズされてサイクルがリセットされます。

最初のポジションがショートの場合、同じルールがショート取引にも適用されます。

## パラメーター

- `ProfitThreshold` – 新しいポジションを追加するために必要な価格の動き（ポイント）。
- `MaxPositions` – オープンするポジションの最大数。
- `StopLossPoints` – エントリーから保護ストップまでの距離。
- `TakeProfitPoints` – エントリーから目標利益までの距離。
- `CandleType` – 処理に使用するローソク足データタイプ。
