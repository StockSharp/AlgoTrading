# Стратегия Last Price
[English](README.md) | [中文](README_cn.md)

Эта стратегия выставляет лимитные заявки на лучшем бид/аске, когда последняя цена сделки отклоняется на заданный интервал. Стратегия анализирует обновления Level1 и поток сделок для определения входов.

## Детали

- **Условия входа**:
  - **Long**: Последняя цена ≥ лучший ask + интервал.
  - **Short**: Последняя цена ≤ лучший bid - интервал.
- **Направление торговли**: оба направления.
- **Условия выхода**:
  - Противоположный сигнал или выход за пределы торговых сессий.
- **Стопы**: только стоп-лосс.
- **Значения по умолчанию**:
  - `Interval` = 400
  - `Min Volume` = 1
  - `Max Volume` = 900000
  - `Spread` = 200
  - `Volume` = 1
  - `Stop Loss` = 400
- **Торговые сессии**:
  - 10:05:40 – 13:54:30
  - 14:08:30 – 15:44:30
  - 16:05:30 – 18:39:30
  - 19:15:10 – 23:44:30
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
