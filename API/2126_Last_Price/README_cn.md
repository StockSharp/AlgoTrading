# Last Price 策略
[English](README.md) | [Русский](README_ru.md)

该策略当最新成交价偏离最佳买价或卖价指定区间时，在相应的买一或卖一处挂出限价单。策略订阅Level1报价和成交数据来确定入场。

## 细节

- **入场条件**：
  - **多头**：最新价 ≥ 卖一价 + 间隔。
  - **空头**：最新价 ≤ 买一价 - 间隔。
- **多空方向**：双向。
- **出场条件**：
  - 相反信号或超出允许的交易时段。
- **止损**：仅止损。
- **默认值**：
  - `Interval` = 400
  - `Min Volume` = 1
  - `Max Volume` = 900000
  - `Spread` = 200
  - `Volume` = 1
  - `Stop Loss` = 400
- **交易时段**：
  - 10:05:40 – 13:54:30
  - 14:08:30 – 15:44:30
  - 16:05:30 – 18:39:30
  - 19:15:10 – 23:44:30
- **过滤器**：
  - 类别: Breakout
  - 方向: 双向
  - 指标: 无
  - 止损: 有
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中等
