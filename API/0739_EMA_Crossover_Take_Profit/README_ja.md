# EMAクロスオーバー・テイクプロフィット戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、20期間と50期間の指数移動平均線（EMA）のクロスオーバーに基づいて取引します。速いEMAが遅いEMAを上抜けしたときにロングポジションを開き、逆のクロスでショートポジションを開きます。

エントリー後、シグナルキャンドルの値幅からテイクプロフィットの4つのレベルが計算されます。価格がいずれかのレベルに達するか、ストップロスが発動されるとポジションが決済されます。速いEMAが遅いEMAを上回っているときはローソク足が緑に、下回っているときは赤にハイライトされます。

## 詳細

- **エントリー条件**:
  - **ロング**: EMA20がEMA50を上抜けする。
  - **ショート**: EMA20がEMA50を下抜けする。
- **テイクプロフィット**: 直前の値幅の乗数に基づく4つの目標値。
- **ストップ**: エントリー価格から3%のストップロス。
- **インジケーター**: EMA20、EMA50、EMA200。
- **時間軸**: パラメーターで設定可能。
- **方向**: ロングとショート。
