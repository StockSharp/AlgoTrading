# Стратегия CCI Failure Swing

CCI Failure Swing опирается на индекс товарного канала (CCI), когда образуется более низкий максимум выше +100 либо более высокий минимум ниже -100.
Неспособность индикатора установить новый экстремум часто означает завершение прежнего тренда.

Стратегия покупает, когда CCI держится выше -100 и поворачивает вверх, либо продаёт, если он не проходит уровень +100 и разворачивается вниз.

Процентный стоп удерживает риск небольшим, а сделка закрывается, если CCI возвращается через предыдущий уровень свинга.

## Детали

- **Критерий входа**: сигнал индикатора
- **Длинные/короткие позиции**: обе
- **Критерий выхода**: стоп-лосс или противоположный сигнал
- **Стопы**: да, процентные
- **Значения по умолчанию**:
  - `CandleType` = 15 минут
  - `StopLoss` = 2%
- **Фильтры**:
  - Категория: Разворот
  - Направление: Обе
  - Индикаторы: CCI
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
