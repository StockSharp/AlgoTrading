# Estrategia Chandelier Exit con Filtro de 200 EMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia utiliza los niveles Chandelier Exit basados en ATR para determinar la dirección del mercado. Las operaciones se realizan solo cuando el precio cruza el nivel de dirección y se alinea con el filtro de tendencia de la EMA de 200 períodos.

## Parámetros
- Tipo de vela
- Período ATR
- Multiplicador ATR
- Usar precio de cierre para extremos
- Longitud de la EMA
