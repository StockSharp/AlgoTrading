# Chandelier Exit mit 200 EMA Filter-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie verwendet ATR-basierte Chandelier Exit-Niveaus zur Bestimmung der Marktrichtung. Trades werden nur eröffnet, wenn der Preis das Richtungsniveau kreuzt und mit dem 200-Perioden-EMA-Trendfilter übereinstimmt.

## Parameter
- Kerzentyp
- ATR-Periode
- ATR-Multiplikator
- Schlusskurs für Extremwerte verwenden
- EMA-Länge
