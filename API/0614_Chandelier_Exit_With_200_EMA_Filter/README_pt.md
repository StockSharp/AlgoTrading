# Estratégia Chandelier Exit com Filtro de 200 EMA
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia utiliza os níveis Chandelier Exit baseados em ATR para determinar a direção do mercado. As operações são realizadas apenas quando o preço cruza o nível de direção e se alinha com o filtro de tendência da EMA de 200 períodos.

## Parâmetros
- Tipo de candle
- Período ATR
- Multiplicador ATR
- Usar preço de fechamento para extremos
- Comprimento da EMA
