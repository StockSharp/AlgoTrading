# Стратегия дивергенций
[English](README.md) | [中文](README_cn.md)

Стратегия ищет дивергенции цены и RSI на простых пивотах. Бычья дивергенция ведёт к покупке, медвежья — к продаже.

## Подробности

- **Критерии входа**: дивергенции цены и RSI.
- **Длинные/короткие**: оба направления (настраивается).
- **Критерии выхода**: обратный сигнал RSI или защитные ордера.
- **Стопы**: да (стоп и тейк-профит).
- **Параметры по умолчанию**:
  - `TradeDirection` = Both
  - `RsiPeriod` = 14
  - `StopLossPercent` = 2м
  - `RiskReward` = 2м
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: RSI
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной (5м)
  - Сезонность: Нет
  - Нейронные сети: Нет
  - Дивергенция: Да
  - Уровень риска: Средний
