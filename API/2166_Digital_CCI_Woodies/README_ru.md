# Стратегия Digital CCI Woodies
[English](README.md) | [中文](README_cn.md)

Стратегия использует пересечение двух индикаторов Commodity Channel Index. Быстрый CCI быстро реагирует на изменение цены, а медленный CCI фильтрует шум. Торговые сигналы появляются, когда быстрый индикатор пересекает медленный.

## Детали

- **Условия входа**:
  - Лонг: быстрый CCI пересекает медленный снизу вверх.
  - Шорт: быстрый CCI пересекает медленный сверху вниз.
- **Лонг/Шорт**: Оба направления.
- **Условия выхода**:
  - Лонг закрывается, когда быстрый CCI пересекает медленный сверху вниз.
  - Шорт закрывается, когда быстрый CCI пересекает медленный снизу вверх.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `CandleType` = 6-часовые свечи
  - `FastLength` = 14
  - `SlowLength` = 6
  - `BuyOpen` = true
  - `SellOpen` = true
  - `BuyClose` = true
  - `SellClose` = true
- **Фильтры**:
  - Категория: Следование тренду
  - Направление: Оба
  - Индикаторы: CCI
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
