# Exceeded Candle戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このパターンベースのアプローチは、価格がボリンジャーバンドの中間線を下回っている間に、前のバーを超える強気の包み足を探します。押し目の中での強い反転が、価格を上限バンドに向かって押し上げる可能性があるという考え方です。この戦略はロングのみ取引し、3本連続の陰線が出現した場合はエントリーをキャンセルします。

価格がボリンジャーバンドの上限バンドに触れると、ポジションが決済され、素早い反発を捉えます。この手法は、ボラティリティバンドが平均回帰のスイングを捉える短い時間軸に適しています。

## 詳細

- **エントリー条件**:
  - **ロング**: 前の足が陰線、現在の足が陽線で前の足の始値より高く終値、`Close < MiddleBand`、3本連続陰線なし
- **ロング/ショート**: ロングのみ
- **エグジット条件**:
  - **ロング**: `Close > UpperBand`
- **ストップ**: なし
- **デフォルト値**:
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: ロングのみ
  - インジケーター: Bollinger Bands, price action
  - ストップ: いいえ
  - 複雑さ: 低
  - 時間軸: 短期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
