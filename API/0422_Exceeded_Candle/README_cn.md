# Exceeded Candle 策略
[English](README.md) | [Русский](README_ru.md)

该策略寻找形成在布林带中轨以下的看涨吞没形态。当绿色蜡烛完全包住前一根红色蜡烛时，预示回调可能结束，价格倾向于回到上轨。若连续出现三根红色蜡烛则取消新的入场信号，系统只做多。

当价格触及上轨即平仓，捕捉这波快速反弹。适合波动性高的短周期市场，利用回归均值的特性。

## 详情

- **入场条件**:
  - **多头**: 前一根为红色，当前为绿色且收盘高于前一根开盘，`Close < MiddleBand`，最近三根不是全部红色
- **多空方向**: 仅多头
- **退出条件**:
  - **多头**: `Close > UpperBand`
- **止损**: 无
- **默认值**:
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
- **过滤器**:
  - 类型: 均值回归
  - 方向: 仅多头
  - 指标: 布林带、K线形态
  - 止损: 无
  - 复杂度: 低
  - 时间框架: 短期
  - 季节性: 否
  - 神经网络: 否
  - 背离: 是
  - 风险等级: 中等
