# Strategie für externe Signale Tester
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Öffnet Positionen, wenn die Differenz zwischen schneller und langsamer EMA die Null kreuzt oder wenn der Preis optionale EMA-Linien kreuzt. Unterstützt optionalen prozentualen Stop-Loss, Take-Profit und Breakeven.

## Details

- **Einstiegskriterien**: EMA(10) - EMA(30) kreuzt Null oder Preis kreuzt konfigurierbare EMA.
- **Long/Short**: Long und Short.
