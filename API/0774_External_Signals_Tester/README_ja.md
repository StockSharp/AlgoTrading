# 外部シグナルテスター戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

短期EMAと長期EMAの差がゼロを越えるとき、またはオプションのEMAラインを価格が越えるときにポジションを建てます。オプションのパーセントベースのストップロス、テイクプロフィット、ブレークイーブンをサポートします。

## 詳細

- **エントリー条件**: EMA(10) - EMA(30) がゼロをクロス、または価格が設定可能なEMAをクロス。
- **ロング/ショート**: ロングとショート。
