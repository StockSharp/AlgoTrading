# Estrategia de Prueba de Señales Externas
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Abre posiciones cuando la diferencia entre EMA rápida y lenta cruza cero, o cuando el precio cruza líneas EMA opcionales. Admite stop loss, take profit y breakeven opcionales basados en porcentaje.

## Detalles

- **Criterios de entrada**: EMA(10) - EMA(30) cruzando cero o el precio cruzando la EMA configurable.
- **Largo/Corto**: Largo y corto.
