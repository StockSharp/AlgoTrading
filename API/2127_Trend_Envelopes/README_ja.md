# トレンドエンベロープ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

TrendEnvelopesインジケーターを基盤としたトレンドフォロー戦略です。EMAとATRベースのバンドを組み合わせてブレイクアウトを検出します。
価格が上部バンドを上抜けして買いシグナルが出たときにロングポジションを建てます。価格が下部バンドを下抜けして売りシグナルが出たときにショートポジションを建てます。逆側のバンドでポジションを決済します。

## 詳細

- **エントリー条件**:
  - ロング: 価格が上部エンベロープを上抜けして買いシグナルが発生
  - ショート: 価格が下部エンベロープを下抜けして売りシグナルが発生
- **ロング/ショート**: 両方
- **エグジット条件**: 逆のトレンドシグナル
- **ストップ**: あり（テイクプロフィットとストップロス）
- **デフォルト値**:
  - `MaPeriod` = 14
  - `Deviation` = 0.2m
  - `AtrPeriod` = 15
  - `AtrSensitivity` = 0.5m
  - `TakeProfit` = 2000 ポイント
  - `StopLoss` = 1000 ポイント
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: EMA, ATR
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: 4h
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
