# I4 DRF戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

カスタムI4 DRFインジケーターに基づく戦略。直近のローソク足の高値と安値の方向を比較し、-100から+100の間の値を生成します。取引アクションはこのインジケーターの色の遷移と選択されたモードによって決まります。

## 詳細

- **エントリー条件**:
  - `Direct`モード: インジケーターが正から負に変わるとロングを開く；負から正に変わるとショートを開く。
  - `NotDirect`モード: 負から正への変化でロングを開く；正から負への変化でショートを開く。
- **ロング/ショート**: 両方
- **エグジット条件**:
  - 反対のシグナルが現れるとポジションが閉じられます。
- **ストップ**: なし
- **デフォルト値**:
  - `Period` = 11
  - `SignalBar` = 1
  - `TrendMode` = Direct
  - `CandleType` = TimeSpan.FromHours(4).TimeFrame()
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: I4 DRF
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: 中期
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
