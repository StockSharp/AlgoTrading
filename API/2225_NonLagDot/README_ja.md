# NonLagDot 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

NonLagDotインジケーターに着想を得た戦略です。インジケーターは滑らかな移動平均と色分けされたドットを使用して価格トレンドを近似します。
インジケーターが上向きに転じたときにロングポジションをオープンし、下向きに転じたときにショートポジションをオープンします。
新しいポジションをオープンする前に、逆方向の既存ポジションをクローズします。

## 詳細

- **エントリー条件**:
  - ロング: インジケーターが下降から上昇に転換（移動平均の傾きがプラスになる）
  - ショート: インジケーターが上昇から下降に転換（傾きがマイナスになる）
- **ロング/ショート**: 両方
- **エグジット条件**: 逆シグナル
- **ストップ**: オプションのストップロスパーセント
- **デフォルト値**:
  - `Length` = 10
  - `CandleType` = TimeSpan.FromHours(1).TimeFrame()
  - `StopLossPercent` = 1m
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: NonLagDotのSMA傾き近似
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
