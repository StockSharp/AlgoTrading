# Стратегия AutoTraderRus
[English](README.md) | [中文](README_cn.md)

Утилитарная стратегия для автоматического управления торговой сессией. Она подписывается на минутные свечи, отслеживает текущее время и разрешает торговлю только между заданными часами начала и окончания. Когда время выходит за пределы этого диапазона, стратегия закрывает все открытые позиции и не размещает новые заявки.

## Подробности

- **Условия входа**:
  - Отсутствуют. Стратегия не открывает позиции самостоятельно.
- **Направление**: Оба направления, управление имеющимися позициями.
- **Условия выхода**:
  - Все позиции закрываются, если текущее время вне сессии.
- **Стопы**: Нет.
- **Параметры по умолчанию**:
  - `StartHour` = 9
  - `StartMinute` = 30
  - `StopHour` = 23
  - `StopMinute` = 30
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Управление сессией
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
