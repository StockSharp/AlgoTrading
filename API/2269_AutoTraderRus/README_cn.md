# AutoTraderRus 策略
[English](README.md) | [Русский](README_ru.md)

该工具型策略用于自动管理交易时段。它订阅1分钟K线以追踪市场时间，只允许在设定的开始和结束时间之间进行交易。当当前时间离开该区间时，策略会平掉所有持仓并且不再发送新订单。

## 详情

- **入场条件**：
  - 无。策略本身不会开仓。
- **多空方向**：双向，仅管理已有仓位。
- **出场条件**：
  - 当前时间不在交易时段内时，关闭所有持仓。
- **止损**：无。
- **默认参数**：
  - `StartHour` = 9
  - `StartMinute` = 30
  - `StopHour` = 23
  - `StopMinute` = 30
  - `CandleType` = TimeSpan.FromMinutes(1)
- **过滤器**：
  - 分类：时段管理
  - 方向：双向
  - 指标：无
  - 止损：无
  - 复杂度：低
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：低
