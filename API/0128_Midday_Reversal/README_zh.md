# 午间反转策略
[English](README.md) | [Русский](README_ru.md)

午间反转旨在捕捉午餐时间附近的趋势转折，早盘的动能常在此时衰竭。
中盘流动性降低，交易者结清仓位可能导致价格反向。
系统在中午监测动能转变，逆早盘走势入场。
若午后反转未能形成，则按百分比止损离场。

测试表明年均收益约为 121%，该策略在加密市场表现最佳。

## 细节

- **入场条件**：指标信号
- **多/空**：均可
- **退出条件**：止损或反向信号
- **止损**：是，按百分比
- **默认值**:
  - `CandleType` = 15分钟
  - `StopLoss` = 2%
- **过滤器**:
  - 类别：日内
  - 方向：双向
  - 指标：价格行为
  - 止损：有
  - 复杂度：中等
  - 时间框架：日内
  - 季节性：否
  - 神经网络：否
  - 背离：否
  - 风险等级：中等
