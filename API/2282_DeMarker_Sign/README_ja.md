# DeMarker Sign戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はDeMarkerオシレーターを使用して潜在的なトレンド転換を検出します。各確定足（デフォルトでは4時間足）でDeMarker値が設定可能な上限・下限閾値と比較されます。オシレーターが下限閾値（デフォルト0.3）を上抜けると、戦略はロングポジションに入り、既存のショートポジションを決済します。オシレーターが上限閾値（デフォルト0.7）を下抜けると、ショートポジションに入り、既存のロングポジションを決済します。ポジションは反対のシグナルが現れるまで保持されます。

## 詳細

- **エントリー条件**：
  - **ロング**：DeMarkerが下限レベルを上抜け。
  - **ショート**：DeMarkerが上限レベルを下抜け。
- **ロング/ショート**：両方。
- **エグジット条件**：反対のシグナル。
- **ストップ**：デフォルトなし。
- **フィルター**：なし。
