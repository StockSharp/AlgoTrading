# 段階的トレーリング戦略のサンプル
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

オプションのトレーリングストップを使った3段階のトレード管理を実演するサンプル戦略です。

14期間SMAが28期間SMAを上抜けたときにロングエントリーします。リスクはストップロスと3つの利益目標で管理されます：
- 最初の目標達成後、ストップはブレークイーブンに移動します。
- 2番目の目標達成後、ストップは最初の目標に移動します。
- 3番目のステップでは、ポジションは目標3で決済するかトレーリングストップを開始します。

このサンプルは、ポジションが進む中で利益を段階的に確保し、ポジションを保護する方法を示しています。
