# Стратегия спреда высокодоходных облигаций с фильтром SMA
[English](README.md) | [中文](README_cn.md)

Стратегия торгует по спреду между высокодоходными облигациями и государственными или по индексу VIX. Сделка открывается, когда выбранный спред превышает порог и при этом выполняется необязательный фильтр цены. Для длинных позиций требуется закрытие выше SMA, для коротких – ниже SMA. Позиция закрывается после фиксированного числа свечей.

## Детали

- **Условия входа**:
  - **Long**: спред > порог и закрытие > SMA (если фильтр включён).
  - **Short**: спред < порог и закрытие < SMA (если фильтр включён).
- **Лонг/Шорт**: обе стороны, выбирается параметром.
- **Условия выхода**:
  - Закрытие позиции после указанного числа свечей.
- **Стопы**: отсутствуют.
- **Значения по умолчанию**:
  - `Threshold` = 5
  - `HoldingPeriod` = 5
  - `SmaLength` = 50
- **Фильтры**:
  - Категория: Макро
  - Направление: Обе
  - Индикаторы: High Yield Spread/VIX, SMA
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: 1d (по умолчанию)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
