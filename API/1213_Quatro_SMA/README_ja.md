# Quatro SMA 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、3本の高速単純移動平均線（SMA）と長期SMA、そしてボリュームフィルターを組み合わせます。最速のSMAが中間SMAを上回り、中間SMAが低速SMAを上回り、価格が長期SMAを上回り、ボリュームが設定可能な倍数でその平均を超えたときにロングポジションを建てます。ショートポジションには逆のアライメントが必要です。

ポジションは複数の段階でクローズされます：最大3つのテイクプロフィットレベルと1つのストップロスがトレードの一部をクローズできます。SMAのアライメントが逆転した場合もポジションがクローズされます。

## 詳細

- **インジケーター**: SMA、出来高
- **時間軸**: 4h
- **タイプ**: 出来高確認付きトレンドフォロー
- **ストップ**: 3つのテイクプロフィットレベルと1つのストップロス
