# VWAP EMA ATR Pullback
[English](README.md) | [中文](README_cn.md)

Трендовая стратегия, использующая EMA, VWAP и ATR.

Тестирование показывает среднегодовую доходность около 55%. Лучше всего работает на рынке фьючерсов.

Подход определяет сильный тренд через расстояние между быстрой и медленной EMA, превышающее ATR. Вход осуществляется при откате цены к VWAP. Тейк-профит устанавливается на уровне VWAP ± ATR * множитель.

## Детали

- **Условия входа**:
  - **Лонг**: восходящий тренд и закрытие < VWAP.
  - **Шорт**: нисходящий тренд и закрытие > VWAP.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**: цель на VWAP ± ATR * множитель.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `FastEmaLength` = 30
  - `SlowEmaLength` = 200
  - `AtrLength` = 14
  - `AtrMultiplier` = 1.5
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: EMA, ATR, VWAP
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
