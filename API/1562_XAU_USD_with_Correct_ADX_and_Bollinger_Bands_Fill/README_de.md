# XAU/USD ADX Bollinger-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie handelt Gold mit Bollinger Bands und dem ADX-Indikator. Eine Position wird eröffnet, wenn der Kurs das Band verlässt und die Trendstärke den konfigurierten Schwellenwert überschreitet.
