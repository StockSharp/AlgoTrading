# ゾーン付きフェイズクロス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

このサンプル戦略は、正のオフセットを持つ平滑化SMAが負のオフセットを持つEMAを上抜けたときにロングエントリーします。逆のクロスオーバーが発生するとポジションを決済します。

## 詳細

- **エントリー条件**: SMA + オフセットがEMA - オフセットを上抜ける。
- **ロング/ショート**: ロングのみ。
- **エグジット条件**: 逆のクロスオーバー。
- **ストップ**: なし。
- **デフォルト値**:
  - `Length` = 20.
  - `Offset` = 0.5.
- **フィルター**: なし。
- **複雑さ**: 低。
- **時間軸**: 設定可能。
