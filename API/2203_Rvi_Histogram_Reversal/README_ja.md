# RVIヒストグラムリバーサル戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要

この戦略はRVIの極端な値に逆らって取引します。相対活力指数（RVI）で動作し、インジケーターが買われすぎまたは売られすぎのゾーンを離れたとき、またはRVIがシグナルラインを交差したときにポジションを開きます。2つのシグナルモードがサポートされています:

- **Levels** – RVIが定義済みの上限または下限の閾値を交差するときに反応します。
- **Cross** – RVIがシグナルラインを交差するときに反応します。

ロジックは逆張りです。RVIが高いレベルを上回ってから下落した場合、ロングポジションが開かれます。RVIが低いレベルを下回ってから上昇した場合、ショートポジションが開かれます。

## パラメーター

| 名前 | 説明 |
| --- | --- |
| `RviPeriod` | RVIの計算期間。 |
| `HighLevel` | RVIの上限閾値。 |
| `LowLevel` | RVIの下限閾値。 |
| `Mode` | シグナル生成モード（`Levels` または `Cross`）。 |
| `EnableBuyOpen` | ロングポジションの開始を許可する。 |
| `EnableSellOpen` | ショートポジションの開始を許可する。 |
| `EnableBuyClose` | ロングポジションの決済を許可する。 |
| `EnableSellClose` | ショートポジションの決済を許可する。 |
| `CandleType` | ローソク足の時間軸。 |

## 動作方法

1. RVIとその単純移動平均は、完成した各ローソク足で計算されます。
2. 選択したモードに応じて、戦略は以下を確認します:
   - RVIが極端なレベルを離れるか、または
   - RVIがシグナルラインを交差するか。
3. ロングシグナルで戦略はショートポジションを閉じてロングポジションを開きます。ショートシグナルでロングポジションを閉じてショートポジションを開きます。

デフォルトの時間軸は4時間です。
