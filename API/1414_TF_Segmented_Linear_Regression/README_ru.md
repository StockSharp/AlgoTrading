# TF Segmented Linear Regression
[English](README.md) | [中文](README_cn.md)

Стратегия строит канал линейной регрессии в каждом временном сегменте. Длинная позиция открывается при пробое верхней границы, короткая — при пробое нижней.

## Детали
- **Критерии входа**: пробой границ канала регрессии.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: пересечение противоположной границы.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5).TimeFrame()
  - `Segment` = TimeSpan.FromDays(1)
  - `Multiplier` = 2
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: Linear Regression
  - Стопы: Нет
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
