# Двойные каналы Кельтнера
[English](README.md) | [中文](README_cn.md)

Стратегия **Двойные каналы Кельтнера** использует два канала Кельтнера с разными множителями для определения пробоев.
Сделка открывается, когда цена пробивает внешний канал и затем возвращается через внутренний.
Стопы и цели управляются фиксированными процентами.

## Детали
- **Условия входа**: Цена пересекает внешний канал и возвращается через внутренний в том же направлении.
- **Длинные/короткие**: Оба направления.
- **Условия выхода**: Стоп, тейк-профит или противоположный сигнал.
- **Стопы**: Да, на основе процентов.
- **Значения по умолчанию**:
  - `EmaPeriod = 50`
  - `InnerMultiplier = 2.75m`
  - `OuterMultiplier = 3.75m`
  - `MaxStopPercent = 10m`
  - `SlTpRatio = 1m`
  - `CandleType = TimeSpan.FromMinutes(5).TimeFrame()`
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба
  - Индикаторы: Keltner
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
