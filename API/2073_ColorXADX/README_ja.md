# ColorXADX 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ADX の強度で確認された +DI と -DI ラインのクロスに基づく戦略。

システムは方向性移動インジケーターを監視します。+DI が -DI を上回るクロスをし、Average Directional Index が設定した閾値を超えると、ロングポジションに入り、既存のショートを終了します。逆に、弱気クロス（-DI が +DI を上回る）と強い ADX があると、ショートポジションを開き、ロングをクローズします。リスク管理のためにストップロスとテイクプロフィットレベルが適用されます。

## 詳細

- **エントリー条件**: ADX が閾値を超えた状態での +DI/-DI クロス。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対シグナルまたはストップレベル。
- **ストップ**: はい、固定ストップロスとテイクプロフィット。
- **デフォルト値**:
  - `AdxPeriod` = 14
  - `AdxThreshold` = 30m
  - `StopLoss` = 1000
  - `TakeProfit` = 2000
  - `CandleType` = TimeSpan.FromHours(4)
- **フィルター**:
  - カテゴリ: トレンド
  - 方向: 両方
  - インジケーター: ADX, DMI
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: スイング (4h)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
