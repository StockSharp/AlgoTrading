# Bollinger Winner Lite 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Bollinger Winner Lite は、価格がBollinger Bandsを超えて伸びたときに反応する
シンプルな逆張りシステムです。バンドの外側で終値を付ける大きなローソク足を
観察し、素早い内側へのスナップバックを期待します。

`CandlePercent` パラメーターは、直近の動きと比べてブレイクアウトのローソク足が
どれくらいの大きさである必要があるかを定義します。このしきい値を超えるローソク足
のみが取引を発動し、小さな変動をフィルタリングします。デフォルトでは、この戦略は
ロングのみ取引しますが、`ShowShort` を有効にすると反転したショートセットアップも
使用できます。

価格が反対側のバンドに触れるか中央ラインに戻ると、ポジションはクローズされます。
ハードストップは使用しません；システムは平均回帰に依存します。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 終値が下限バンドを下回り、かつローソク足サイズ > `CandlePercent`。
  - **ショート**: 終値が上限バンドを上回り、かつローソク足サイズ > `CandlePercent`（`ShowShort` が必要）。
- **エグジット条件**: 中央バンドまたは反対側バンドへのタッチ。
- **ストップ**: デフォルトではなし。
- **デフォルト値**:
  - `BBLength` = 20
  - `BBMultiplier` = 2.0
  - `CandlePercent` = 30
  - `ShowShort` = false
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: デフォルトではロングのみ
  - インジケーター: Bollinger Bands
  - 複雑さ: シンプル
  - リスクレベル: 中
