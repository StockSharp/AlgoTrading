# XDPOヒストグラム戦略

## 概要

XDPOヒストグラム戦略は、オリジナルのMQL5エキスパート *Exp_XDPO_Histogram* を移植したものです。終値から二重平滑化されたデトレンド価格オシレーター（XDPO）を構築します。オシレーターは、移動平均を価格から差し引き、その差を第2の移動平均で平滑化することで算出されます。ヒストグラムの動きがポジションの開閉シグナルを提供します。

## トレードロジック

- オシレーターが上向きに転換すると、すべてのショートポジションが決済されます。現在のオシレーター値が前の値を上回る場合、新しいロングポジションが開設されます。
- オシレーターが下向きに転換すると、すべてのロングポジションが決済されます。現在のオシレーター値が前の値を下回る場合、新しいショートポジションが開設されます。
- 計算は完成したローソク足のみで実行されます。

## パラメーター

- `FirstMaLength` – 価格に適用する最初の移動平均の長さ。
- `SecondMaLength` – 価格と最初のMAの差に適用する移動平均の長さ。
- `CandleType` – すべての計算に使用するローソク足の種類。

## 備考

- 移動平均は `SimpleMovingAverage` インジケーターで実装されます。
- 戦略は成行注文（`BuyMarket` および `SellMarket`）を使用し、新規ポジションを開設する前に反対側のポジションを決済します。
