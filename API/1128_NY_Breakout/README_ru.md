# Стратегия NY Breakout
[English](README.md) | [中文](README_cn.md)

Стратегия торгует пробой диапазона, сформированного между 13:00 и 13:30 UTC. После закрытия окна вход выполняется при пробое максимума или минимума диапазона. Цель — двойной размер диапазона, стоп — за противоположной границей.

## Детали

- **Условия входа**:
  - Первая свеча после 13:30 UTC закрывается выше максимума диапазона — покупка.
  - Первая свеча после 13:30 UTC закрывается ниже минимума диапазона — продажа.
- **Длинные/Короткие**: обе стороны.
- **Условия выхода**:
  - Тейк-профит на уровне `RewardRisk` * диапазон.
  - Стоп за противоположной границей диапазона.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `RewardRisk` = 2
  - `CandleType` = TimeSpan.FromMinutes(1)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Обе
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Риск: Средний
