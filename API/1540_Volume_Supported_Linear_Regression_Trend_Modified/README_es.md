# Estrategia de Tendencia de Regresión Lineal Modificada con Soporte de Volumen
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Esta estrategia abre una posición larga cuando el RSI de 14 períodos supera el nivel de entrada y cierra la posición cuando el RSI excede el nivel de salida.

## Parámetros
- **RSI Period** – período de cálculo del RSI.
- **Entry Level** – valor del RSI que activa la entrada larga.
- **Exit Level** – valor del RSI que cierra la posición.
- **Candle Type** – marco temporal de las velas procesadas.
