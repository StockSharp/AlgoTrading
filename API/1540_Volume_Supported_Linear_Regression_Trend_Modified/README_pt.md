# Estratégia de Tendência de Regressão Linear Modificada com Suporte de Volume
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Esta estratégia abre uma posição comprada quando o RSI de 14 períodos sobe acima do nível de entrada e fecha a posição quando o RSI excede o nível de saída.

## Parâmetros
- **RSI Period** – período de cálculo do RSI.
- **Entry Level** – valor do RSI que aciona uma entrada comprada.
- **Exit Level** – valor do RSI que fecha a posição.
- **Candle Type** – período dos candles processados.
