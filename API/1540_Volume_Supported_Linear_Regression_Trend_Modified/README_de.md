# Volumengestützte Modifizierte Lineare Regressions-Trend-Strategie
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Diese Strategie eröffnet eine Long-Position, wenn der 14-Perioden-RSI über das Einstiegsniveau steigt, und schließt die Position, wenn der RSI das Ausstiegsniveau überschreitet.

## Parameter
- **RSI Period** – Berechnungszeitraum des RSI.
- **Entry Level** – RSI-Wert, der einen Long-Einstieg auslöst.
- **Exit Level** – RSI-Wert, der die Position schließt.
- **Candle Type** – Zeitrahmen der verarbeiteten Kerzen.
