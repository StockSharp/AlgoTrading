# 大相場キャッチャー戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、価格がボリンジャーバンドの上限を上回って終値を形成し、有効なすべてのフィルターが動きを確認した場合にロングエントリーします。価格が下限バンドを下回って終値を形成した場合にショートポジションも取ることができます。フィルターにはRSI、ADX、ATR、EMAトレンド方向、MACDが含まれます。固定パーセンテージのストップロスが適用され、価格が中間バンドに戻った時にポジションがクローズされます。また、オプションの強制利食いにより、異常に大きなローソク足でポジションを終了します。

## 詳細
- **エントリー条件:**
  - **ロング:** 終値 > ボリンジャーバンド上限、かつすべての有効フィルターがパス。
  - **ショート:** 終値 < ボリンジャーバンド下限、かつすべての有効フィルターがパス。
- **ロング/ショート:** 両方（設定可能）。
- **エグジット条件:**
  - 価格がボリンジャーバンドの中間バンドを越える。
  - 大きなローソク足での強制利食い（オプション）。
- **ストップ:** 固定パーセンテージのストップロス。
- **デフォルト値:** ボリンジャー長さ = 40、ストップロス = 2%、強制TP閾値 = 5%。
- **フィルター:** RSI (14)、ADX (28)、ATR (14)、EMA (350)、MACD (12,26,9)。
