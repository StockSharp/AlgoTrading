# OBVダイバージェンス戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

On-Balance Volume（OBV）は、出来高が価格に先行するという考えのもと、累積取引量を追跡します。価格が新しい高値を形成してもOBVが確認しない場合、またはその逆の場合、リバーサルが起きている可能性があります。この戦略はその乖離（ダイバージェンス）を使って持続不可能な動きに逆張りします。

テストでは年平均リターンが約112%であることが示されています。外国為替市場で最も良いパフォーマンスを発揮します。

各ローソク足ごとにOBVが更新され、前回の読みと比較されます。価格がより低い安値を付ける一方でOBVがより高い安値を記録する場合に強気シグナルが発生します。価格がより高い高値まで上昇するがOBVが遅れる場合に弱気シグナルが発生します。移動平均が決済ポイントを提供し、パーセントストップが損失を制御します。

このアプローチは出来高の枯渇後の平均回帰を捉えることを試み、通常は価格が平均線を再び越えるまでポジションを保持します。

## 詳細

- **エントリー条件**: 価格/OBVのダイバージェンス。
- **ロング/ショート**: 両方。
- **エグジット条件**: 価格が移動平均を越えるか、ストップロス。
- **ストップ**: はい、パーセントベース。
- **デフォルト値**:
  - `DivergencePeriod` = 5
  - `MAPeriod` = 20
  - `CandleType` = 5 minute
  - `StopLossPercent` = 2
- **フィルター**:
  - カテゴリ: ダイバージェンス
  - 方向: 両方
  - インジケーター: OBV, MA
  - ストップ: はい
  - 複雑さ: 中級
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: はい
  - リスクレベル: 中
