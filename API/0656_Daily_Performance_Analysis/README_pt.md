# Estratégia de Análise de Desempenho Diário
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md)

Rastreia o desempenho por dia da semana e dia do mês enquanto opera mudanças de direção do SuperTrend.

## Detalhes

- **Critérios de entrada**: A direção do SuperTrend muda para baixo (mudança < 0) → Comprar. A direção do SuperTrend muda para cima (mudança > 0) → Vender.
- **Comprado/Vendido**: Ambos.
- **Critérios de saída**: Direção oposta do SuperTrend.
- **Indicadores**: SuperTrend.
