# Strategie zur täglichen Leistungsanalyse
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Verfolgt die Performance nach Wochentag und Monatstag beim Handel von SuperTrend-Richtungswechseln.

## Details

- **Einstiegskriterien**: SuperTrend-Richtung wechselt nach unten (Änderung < 0) → Kaufen. SuperTrend-Richtung wechselt nach oben (Änderung > 0) → Verkaufen.
- **Long/Short**: Beide.
- **Ausstiegskriterien**: Entgegengesetzte SuperTrend-Richtung.
- **Indikatoren**: SuperTrend.
