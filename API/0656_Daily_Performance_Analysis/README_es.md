# Estrategia de Análisis de Rendimiento Diario
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Deutsch](README_de.md) | [Português](README_pt.md) | [日本語](README_ja.md)

Rastrea el rendimiento por día de la semana y día del mes mientras opera cambios de dirección del SuperTrend.

## Detalles

- **Criterios de entrada**: El SuperTrend cambia de dirección hacia abajo (cambio < 0) → Comprar. El SuperTrend cambia de dirección hacia arriba (cambio > 0) → Vender.
- **Largo/Corto**: Ambos.
- **Criterios de salida**: Dirección opuesta del SuperTrend.
- **Indicadores**: SuperTrend.
