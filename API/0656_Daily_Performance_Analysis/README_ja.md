# 日次パフォーマンス分析戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

SuperTrendの方向転換を取引しながら、曜日別および月の日付別のパフォーマンスを追跡します。

## 詳細

- **エントリー条件**: SuperTrendの方向が下向きに転換（変化 < 0）→ 買い。SuperTrendの方向が上向きに転換（変化 > 0）→ 売り。
- **ロング/ショート**: 両方。
- **エグジット条件**: SuperTrendの逆方向転換。
- **インジケーター**: SuperTrend。
