# ギャップフィリング戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

ギャップフィリング戦略は、新しいセッション開始時の夜間価格ギャップを探します。ギャップが現れると、前日の価格への戻りを期待してギャップに逆らってポジションを建てます。反転モード時はギャップ方向に取引し、ギャップレベルでストップを置きます。

## 詳細
- **データ**: 価格ローソク足。
- **エントリー条件**:
  - **ロング**: 新セッションかつ下方ギャップ（反転時は上方ギャップ）。
  - **ショート**: 新セッションかつ上方ギャップ（反転時は下方ギャップ）。
- **エグジット条件**:
  - ギャップフィル価格到達（利益目標）、または反転時はギャップレベルのストップ。
- **ストップ**: 前のセッションの価格を目標/ストップとして使用。
- **デフォルト値**:
  - `CandleType` = 1 minute
  - `Invert` = false
  - `CloseWhen` = NewSession
- **フィルター**:
  - カテゴリ: ギャップ取引
  - 方向: ロング & ショート
  - インジケーター: なし
  - 複雑さ: シンプル
  - リスクレベル: 中
