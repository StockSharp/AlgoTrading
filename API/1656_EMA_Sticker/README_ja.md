# EMA Sticker戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は指数移動平均線（EMA）を使用して短期トレンドを追います。終値がEMAを上抜けするとロングポジションを建て、下抜けするとショートポジションを建てます。オプションの固定ストップロスとテイクプロフィットレベルがリスク管理に役立ちます。

## 詳細

- **エントリー条件**:
  - **ロング**: `Close > EMA`.
  - **ショート**: `Close < EMA`.
- **ロング/ショート**: 両方のサイド。
- **エグジット条件**:
  - 反対のシグナルまたは設定されたストップレベルに到達。
- **ストップ**: あり、価格単位でのオプションのストップロスとテイクプロフィット。
- **デフォルト値**:
  - `MA period` = 5.
  - `Stop loss` = 0.001.
  - `Take profit` = 0.001.
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: 単一
  - ストップ: はい
  - 複雑さ: シンプル
  - 時間軸: 短期
