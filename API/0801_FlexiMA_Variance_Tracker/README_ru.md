# Стратегия FlexiMA Variance Tracker
[English](README.md) | [中文](README_cn.md)

Отслеживает отклонение цены от скользящей средней и открывает позиции, когда отклонение превышает порог волатильности и направление SuperTrend подтверждает.

## Детали

- **Условия входа**:
  - Цена выше SuperTrend и отклонение > среднее + стандартное отклонение × множитель → покупка.
  - Цена ниже SuperTrend и отклонение < -(среднее + стандартное отклонение × множитель) → продажа.
- **Лонг/Шорт**: поддерживаются оба направления.
- **Условия выхода**:
  - Противоположное отклонение или разворот SuperTrend.
- **Стопы**: логика стопов по умолчанию отсутствует.
- **Параметры по умолчанию**:
  - Длина MA = 20.
  - Длина StdDev = 20.
  - Множитель StdDev = 1.0.
  - Период ATR = 10.
  - Множитель ATR = 3.0.
- **Фильтры**:
  - Категория: следование за трендом
  - Направление: оба
  - Индикаторы: SMA, StandardDeviation, SuperTrend
  - Стопы: нет
  - Сложность: средняя
  - Таймфрейм: любой
  - Сезонность: нет
  - Нейросети: нет
  - Дивергенция: нет
  - Уровень риска: средний
