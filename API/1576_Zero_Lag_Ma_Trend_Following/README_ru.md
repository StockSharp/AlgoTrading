# Zero-Lag MA Trend Following
[English](README.md) | [中文](README_cn.md)

Трендовая система, ожидающая пересечения нулелаговой MA и EMA, после чего входит при пробое цены из коробки размером ATR. Позиции сопровождаются тейком и стопом по соотношению риск-доходность.

## Детали

- **Критерии входа**: пересечение ZLEMA и EMA и пробой коробки.
- **Длинные/короткие**: оба направления.
- **Критерии выхода**: стоп или тейк по ATR.
- **Стопы**: да.
- **Значения по умолчанию**:
  - `Length` = 34
  - `AtrPeriod` = 14
  - `RiskReward` = 2m
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Тренд
  - Направление: Оба
  - Индикаторы: ZLEMA, EMA, ATR
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной (5m)
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
