# Silver Trend 戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

カスタムSilverTrendインジケーターに基づいたトレンドフォロー戦略です。インジケーターはルックバックウィンドウとリスクファクターを使用して最高値と最安値でダイナミックな価格チャネルを構築します。価格がチャネルをクロスしてトレンド方向が反転すると取引シグナルが発生します。

## 詳細

- **エントリー**: インジケーターが上昇トレンドに切り替わったときに買い。インジケーターが下降トレンドに切り替わったときに売り。
- **エグジット**: 逆シグナルでポジションがリバースされる。
- **インジケーター**: Highest、Lowest、SimpleMovingAverage（SilverTrendの計算内で使用）。
- **ストップ**: なし。
- **デフォルト値**:
  - `Ssp` = 9 — チャネル計算に使用するバー数。
  - `Risk` = 3 — チャネル幅を縮小するパーセンテージ。
  - `CandleType` = 1時間ローソク足。
- **方向**: ロングとショートの両方。

SilverTrendインジケーターは`Ssp + 1`バーにわたる平均的な高値-安値レンジを計算し、`Ssp`バーにわたる最高値と最安値を求めます。チャネル境界は次のとおりです:

```
smin = minLow + (maxHigh - minLow) * (33 - Risk) / 100
smax = maxHigh - (maxHigh - minLow) * (33 - Risk) / 100
```

終値が`smin`を下回ると、トレンドは弱気になります。終値が`smax`を上回ると、トレンドは強気になります。トレンドが転換するとシグナルが生成され、戦略はそれに応じてすぐにポジションをリバースします。
