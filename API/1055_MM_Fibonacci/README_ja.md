# MM Fibonacci戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はMurrey Math Fibonacciレベルを計算し、ブレイクアウトを取引します。上昇トレンドの状況で価格が100%レベルを上回ったときに買い、下降トレンドの状況で価格が0%レベルを下回ったときに売ります。ポジションは取引方向に反して価格が50%レベルを越えたときにクローズされます。

## 詳細

- **エントリー条件**:
  - **ロング**: 直近の極値が高値であり、価格が100%レベルを上回って引けたとき。
  - **ショート**: 直近の極値が安値であり、価格が0%レベルを下回って引けたとき。
- **エグジット条件**:
  - **ロング**: 価格が50%レベルを下回ったとき。
  - **ショート**: 価格が50%レベルを上回ったとき。
- **インジケーター**: Highest、Lowest。
- **ロング/ショート**: 両方。
- **ストップ**: なし。
