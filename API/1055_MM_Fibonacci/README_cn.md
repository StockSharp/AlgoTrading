# MM Fibonacci 策略
[English](README.md) | [Русский](README_ru.md)

该策略计算 Murrey Math Fibonacci 水平并交易突破。当趋势向上且价格突破 100% 水平时做多，趋势向下且价格跌破 0% 水平时做空。价格回到 50% 水平时平仓。

## 细节

- **入场条件**：
  - **多头**：最近的极值为高点且价格收于 100% 之上。
  - **空头**：最近的极值为低点且价格收于 0% 之下。
- **出场条件**：
  - **多头**：价格跌破 50% 水平。
  - **空头**：价格升破 50% 水平。
- **指标**：Highest、Lowest。
- **多空**：双向。
- **止损**：无。
