# Exp To Close Profit 策略
[English](README.md) | [Русский](README_ru.md)

该辅助策略监控持仓的当前利润，一旦达到设定的盈利目标就立即平掉所有仓位。策略本身不产生开仓信号，可与其他交易算法一起使用以锁定收益。

## 详情

- **入场条件**：无，策略不会开仓。
- **多空方向**：同时支持多头和空头平仓。
- **出场条件**：当达到 `MaxProfit` 时平掉当前仓位。
- **止损/止盈**：仅包含盈利目标。
- **默认值**：
  - `MaxProfit` = 1000
- **筛选**：
  - 分类：风险管理
  - 方向：双向
  - 指标：无
  - 止损：是
  - 复杂度：基础
  - 时间框架：任意
  - 季节性：无
  - 神经网络：无
  - 背离：无
  - 风险等级：低
