# Стратегия Exp To Close Profit
[English](README.md) | [中文](README_cn.md)

Вспомогательная стратегия, контролирующая текущую прибыль открытых позиций и закрывающая их при достижении заданного уровня прибыли. Не открывает сделки и может использоваться совместно с другими алгоритмами для фиксации прибыли.

## Подробности

- **Условия входа**: отсутствуют, стратегия не открывает позиции.
- **Длинные/короткие**: закрывает позиции в обоих направлениях.
- **Условия выхода**: при достижении `MaxProfit` текущая позиция закрывается.
- **Стопы**: только цель по прибыли.
- **Значения по умолчанию**:
  - `MaxProfit` = 1000
- **Фильтры**:
  - Категория: Управление рисками
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
