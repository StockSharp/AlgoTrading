# Supertrend リバーサルストラテジー
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

SupertrendインジケーターはATRと価格を組み合わせてトレーリングサポートまたはレジスタンスを生成します。Supertrend線が価格の上から下へ、またはその逆に転換すると、トレンド転換の可能性を示します。この戦略はそのような転換を取引します。

テストでは年平均リターン約151%を示しています。株式市場で最も優れたパフォーマンスを発揮します。

各ローソク足でATRベースの計算がSupertrendレベルを更新します。価格の上から下への転換はロングエントリーを引き起こし、下から上への動きはショートを生成します。コードサンプルには明示的なストップがないため、決済は裁量的または別のリスクモジュールで管理されます。

このインジケーターはボラティリティに素早く反応するため、トレーダーはしばしば追加フィルターと組み合わせてダマシを減らします。

## 詳細

- **エントリー条件**: Supertrendが価格に対して側面を変える。
- **ロング/ショート**: 両方。
- **エグジット条件**: 手動または外部ストップ。
- **ストップ**: 未定義。
- **デフォルト値**:
  - `Period` = 10
  - `Multiplier` = 3.0
  - `CandleType` = 15 minute
- **フィルター**:
  - カテゴリ: トレンドフォロー
  - 方向: 両方
  - インジケーター: Supertrend
  - ストップ: オプション
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
