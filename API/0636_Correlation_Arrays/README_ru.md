# Стратегия корреляции массивов
[English](README.md) | [中文](README_cn.md)

Стратегия рассчитывает скользящую матрицу корреляции для максимум шести инструментов. Она фиксирует уровни корреляции по настраиваемым порогам, помогая оценить взаимосвязь активов. Стратегия предназначена только для анализа и не совершает сделок.

## Детали
- **Условия входа**: Нет (только анализ)
- **Длинные/Короткие**: Нет
- **Условия выхода**: Нет
- **Стопы**: Нет
- **Значения по умолчанию**:
  - `CandleType` = TimeSpan.FromMinutes(5)
  - `LookbackPeriod` = 100
  - `PositiveWeak` = 0.3
  - `PositiveMedium` = 0.5
  - `PositiveStrong` = 0.7
  - `NegativeWeak` = -0.3
  - `NegativeMedium` = -0.5
  - `NegativeStrong` = -0.7
- **Фильтры**:
  - Категория: Статистический анализ
  - Направление: Нет
  - Индикаторы: Корреляция
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Любой
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
