# 平均回帰プロ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

Mean Reversion Proは主要指数向けに構築された平均回帰システムです。2本の移動平均とイントラバーのレンジレベルを使用してプルバックを検出します。指数は上昇傾向にあるため、ロング取引が優先されます。

## 詳細

- **エントリー条件**:
  - **ロング**: 終値が高速SMA以下、終値がレンジの20%水準以下、終値が低速SMA以上、ポジションなし。
  - **ショート**: 終値が高速SMA以上、終値がレンジの80%水準以上、終値が低速SMA以下、ポジションなし。
- **ロング/ショート**: 両方（ロング推奨）。
- **エグジット条件**:
  - **ロング**: 終値が高速SMAを上抜け。
  - **ショート**: 終値が高速SMAを下抜け。
- **ストップ**: なし。
- **デフォルト値**:
  - `Fast SMA` = 5
  - `Slow SMA` = 100
  - `Direction` = ロングのみ
- **フィルター**:
  - カテゴリ: 平均回帰
  - 方向: 設定可能
  - インジケーター: SMA
  - ストップ: なし
  - 複雑さ: シンプル
  - 時間軸: 日足
