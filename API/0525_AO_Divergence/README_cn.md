# AO 背离策略
[English](README.md) | [Русский](README_ru.md)

该策略寻找价格与 Awesome Oscillator (AO) 之间的多头和空头背离。当价格创出更低低点而 AO 形成更高低点时，出现多头背离；当价格创出更高高点而 AO 形成更低高点时，出现空头背离。

出现多头背离时，策略开多仓；出现空头背离时，策略开空仓。出现相反信号时仓位反转。

## 细节

- **入场条件**：AO 与价格的多头或空头背离。
- **多/空**：双向。
- **退出条件**：相反的背离信号。
- **止损**：无。
- **默认值**：
  - `CandleType` = 5 分钟
  - `FastLength` = 5
  - `SlowLength` = 34
  - `Lookback` = 5
  - `UseEma` = false
- **过滤条件**：
  - 类别: 指标
  - 方向: 双向
  - 指标: Awesome Oscillator
  - 止损: 无
  - 复杂度: 中等
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 是
  - 风险级别: 中等
