# イントラデイ・ベータ戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、平滑化された移動平均の傾きと相対力指数（RSI）を使ってイントラデイの転換点を探します。
10期間移動平均の傾きが下降後に上向きに転じ、RSIが70未満で前のローソク足が陽線の場合にロングポジションを開きます。
傾きが上昇後に下向きに転じ、RSIが30超で前のローソク足が陰線の場合にショートポジションを開きます。

Average True Range（ATR）フィルターは、ボラティリティが高すぎる場合に新規エントリーをブロックします。オープンポジションは、
取引に有利な方向に動く適応型トレーリングストップで保護され、価格がストップレベルを超えた時に決済されます。

## パラメーター
- **RSI Period** – RSIインジケーターの期間。
- **Trailing Stop** – トレーリングストップの距離（価格単位）。
- **ATR Threshold** – 取引を許可する最大ATR値。
- **Candle Type** – 分析に使用するローソク足の時間軸。
