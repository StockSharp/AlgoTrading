# Стратегия Consecutive Bars Above MA
[English](README.md) | [中文](README_cn.md)

Шортовая стратегия, считающая количество последовательных закрытий выше скользящей средней и продающая при пробое предыдущего максимума. Выход выполняется при падении ниже предыдущего минимума. Дополнительный фильтр EMA 200 ограничивает сделки нисходящим трендом.

## Детали

- **Критерий входа**: порог последовательных закрытий выше MA и закрытие > предыдущего максимума
- **Длин/Шорт**: шорт
- **Критерий выхода**: закрытие ниже предыдущего минимума
- **Стопы**: нет
- **Значения по умолчанию**:
  - `Threshold` = 3
  - `MaType` = SMA
  - `MaLength` = 5
  - `EmaPeriod` = 200
- **Фильтры**:
  - Категория: Pattern
  - Направление: Шорт
  - Индикаторы: MA, EMA
  - Стопы: Нет
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
