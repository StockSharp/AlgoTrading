# MA上方連続バー戦略（ショートのみ）
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

移動平均線を上回る連続終値をカウントし、前の高値を上回るブレイクアウト時にショートに参入するショートのみの戦略。価格が前の安値を下回ったときに手仕舞う。オプションの200 EMAフィルターで下降トレンドを強制する。

## 詳細

- **エントリー条件**: MA上方の連続終値が閾値に達し、かつ終値 > 前の高値
- **ロング/ショート**: ショート
- **エグジット条件**: 終値が前の安値を下回る
- **ストップ**: いいえ
- **デフォルト値**:
  - `Threshold` = 3
  - `MaType` = SMA
  - `MaLength` = 5
  - `EmaPeriod` = 200
- **フィルター**:
  - カテゴリ: パターン
  - 方向: ショート
  - インジケーター: MA, EMA
  - ストップ: いいえ
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
