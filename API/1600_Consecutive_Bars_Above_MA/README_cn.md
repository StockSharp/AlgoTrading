# Consecutive Bars Above MA 策略
[English](README.md) | [Русский](README_ru.md)

该策略仅做空，统计连续收盘价在移动平均线之上的次数，并在突破前高时做空。价格跌破前低时平仓。可选的 200 EMA 过滤器用于确认下行趋势。

## 细节

- **入场条件**: 连续收盘价高于 MA 达到阈值且收盘价突破前高
- **多空方向**: 做空
- **出场条件**: 收盘价低于前一个低点
- **止损**: 无
- **默认值**:
  - `Threshold` = 3
  - `MaType` = SMA
  - `MaLength` = 5
  - `EmaPeriod` = 200
- **过滤器**:
  - 分类: Pattern
  - 方向: 做空
  - 指标: MA, EMA
  - 止损: 无
  - 复杂度: 基础
  - 时间框架: 日内
  - 季节性: 否
  - 神经网络: 否
  - 背离: 否
  - 风险等级: 中等
