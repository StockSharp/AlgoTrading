# Стратегия E TurboFx
[English](README.md) | [中文](README_cn.md)

Реализация эксперта MQL5 "e-TurboFx" на платформе StockSharp. Стратегия отслеживает серию свечей, тела которых увеличиваются в одном направлении. После нескольких медвежьих свечей с растущими телами открывается покупка в расчёте на откат. После серии бычьих свечей с расширяющимися телами открывается продажа. Стоп-лосс и тейк-профит задаются в ценовых пунктах и являются необязательными.

## Детали

- **Условия входа**:
  - Лонг: `N` подряд идущих медвежьих свечей и каждое тело больше предыдущего
  - Шорт: `N` подряд идущих бычьих свечей и каждое тело больше предыдущего
- **Направление**: Оба
- **Условия выхода**: Стоп-лосс или тейк-профит
- **Стопы**: Пункты через `StartProtection`
- **Значения по умолчанию**:
  - `BarsCount` = 3
  - `StopLossPoints` = 700
  - `TakeProfitPoints` = 1200
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **Фильтры**:
  - Категория: Price Action
  - Направление: Оба
  - Индикаторы: Нет
  - Стопы: Да
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
