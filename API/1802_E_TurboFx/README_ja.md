# E TurboFx戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

MQL5エキスパート「e-TurboFx」から適応されたモメンタムリバーサル戦略。システムは同じ方向にボディが拡大する一連のローソク足を監視します。ボディが拡大する複数の陰線の後、反発を期待して買いを入れます。ボディが拡大する複数の陽線の後、売りを入れます。オプションのストップロスとテイクプロフィットは生の価格ポイントで設定されます。

## 詳細

- **エントリー条件**:
  - ロング: `N`本連続の陰線でかつ各ボディが前のボディより大きい
  - ショート: `N`本連続の陽線でかつ各ボディが前のボディより大きい
- **ロング/ショート**: 両方
- **エグジット条件**: ストップロスまたはテイクプロフィット
- **ストップ**: `StartProtection`によるポイント
- **デフォルト値**:
  - `BarsCount` = 3
  - `StopLossPoints` = 700
  - `TakeProfitPoints` = 1200
  - `CandleType` = TimeSpan.FromMinutes(1).TimeFrame()
- **フィルター**:
  - カテゴリ: Price Action
  - 方向: 両方
  - インジケーター: なし
  - ストップ: はい
  - 複雑さ: 基本
  - 時間軸: イントラデイ
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
