# Стратегия Session Breakout Scalper Trading Bot
[English](README.md) | [中文](README_cn.md)

Session Breakout Scalper торгует пробой ценового диапазона, сформированного в заданной сессии.

## Детали

- **Критерий входа**: цена пробивает максимум или минимум сессии
- **Длин/Шорт**: обе стороны
- **Критерий выхода**: тейк-профит или стоп-лосс
- **Стопы**: ATR или фиксированные
- **Значения по умолчанию**:
  - `SessionStart` = 01:00
  - `SessionEnd` = 02:00
  - `TakeProfit` = 100
  - `StopLoss` = 50
  - `UseAtrStop` = true
  - `AtrLength` = 14
  - `AtrMultiplier` = 2
  - `CandleType` = таймфрейм 1 минута
- **Фильтры**:
  - Категория: Пробой
  - Направление: Обе стороны
  - Индикаторы: ATR
  - Стопы: ATR
  - Сложность: Базовая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
