# DeMarker 保留中の戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

## 概要
この StockSharp 戦略は、MetaTrader エキスパート アドバイザー「DeMarker Pending 2.5」の動作を再現します。ボットは、設定可能な時間枠で DeMarker オシレーターを評価し、極端なレベルを超えると、ブレイクアウト方向に指値注文を出します。注文は、固定ポイント数でオフセットされたストップ注文または指値注文のいずれかになります。オプションの取引ウィンドウのフィルタリングと自動有効期限により、未決注文を元のエキスパートの行動に合わせた状態に保つことができます。

## 取引ロジック
- 選択したローソク足シリーズをサブスクライブし、期間 `DemarkerPeriod` で DeMarker インジケーターを計算します。
- 現在および以前の終了ローソク足の値を使用して、下限 (`DemarkerLowerLevel`) と上限 (`DemarkerUpperLevel`) のしきい値のクロスオーバーを検出します。
- 下のレベルが上にクロスすると、長いセットアップをキューに入れます。上のレベルが下にクロスしたら、短いセットアップをキューに入れます。
- `Mode`に応じて、ブレークアウトモードの逆指値注文またはプルバックエントリーの指値注文を使用して、セットアップを価格`Close ± PendingIndentPoints * PriceStep`の未決注文に変換します。
- エントリー価格を `StopLossPoints` ポイントと `TakeProfitPoints` ポイントオフセットすることで、ストップロスとテイクプロフィットのレベルを未決注文に付加します。
- 新しい保留中の注文を登録する前に、`ReplacePreviousPending` と `SinglePendingOnly` に従って古い保留中の注文をキャンセルまたは再利用してください。
- `PendingExpirationMinutes` の有効期間が経過すると、保留中の注文が自動的に削除されます。
- `UseTimeWindow` が有効な場合、日中ウィンドウ外のシグナルを無視します。すべてのバーは 1 回だけ処理されるため、バーおよび方向ごとに最大 1 つの新しい未決注文が作成されます。

## 注文管理
- すべてのエントリは未決注文として作成されます (`BuyStop`、`SellStop`、`BuyLimit`、`SellLimit`)。
- 各未決注文には独自のストップロス価格とテイクプロフィット価格が設定されているため、アクティブ化直後にポジションが保護されます。
- 保留中の注文は、有効期限が切れたとき、新しい設定に置き換えられたとき、または注文の状態が非アクティブなステータス (約定、キャンセル、拒否) に変化したときにキャンセルされます。

## パラメーター
| パラメータ | 説明 |
|-----------|-------------|
| `Volume` | ロット単位での注文量。 |
| `StopLossPoints` | エントリー価格とストップロスの間の距離（ポイント単位）。 |
| `TakeProfitPoints` | エントリー価格とテイクプロフィットの間の距離（ポイント単位）。 |
| `PendingIndentPoints` | 市場価格と未決注文の間のオフセット。 |
| `PendingExpirationMinutes` | 各未決注文の存続期間 (分単位) (0 は有効期限を無効にします)。 |
| `Mode` | 未決注文のタイプ (ブレイクアウトの場合はストップ、プルバックの場合はリミット)。 |
| `SinglePendingOnly` | 有効にすると、複数のアクティブな未決注文を発注できなくなります。 |
| `ReplacePreviousPending` | 新しい注文を発行する前に、アクティブな未決注文をキャンセルします。 |
| `DemarkerPeriod` | DeMarker オシレーターのルックバック期間。 |
| `DemarkerUpperLevel` | 売り設定をトリガーする DeMarker しきい値。 |
| `DemarkerLowerLevel` | 購入セットアップをトリガーする DeMarker しきい値。 |
| `CandleType` | ローソク足のサブスクリプションとインジケーターの評価に使用される時間枠。 |
| `UseTimeWindow` | 日中の時間フィルタリングを有効にします。 |
| `StartTime` | 日中取引ウィンドウの開始。 |
| `EndTime` | 日中取引ウィンドウの終了。 |

## 注意事項
- オリジナルのエキスパートには、洗練された資金管理とトレーリングストップルーチンが含まれています。このポートはシグナル生成と未決注文の処理を維持しますが、ポジションのサイジングを単一の固定 `Volume` パラメーターに簡素化します。
- StockSharp は注文登録時にストップロス価格とテイクプロフィット価格を付加します。ブローカーによっては、ストップ注文と指値注文がこれらの保護レベルをサポートしているかどうかを確認する必要がある場合があります。
- ポイントベースの距離が取引シンボルの `PriceStep` と互換性があることを常に確認してください。 `PendingIndentPoints`、`StopLossPoints`、および `TakeProfitPoints` をブローカーの最小距離要件を満たす値に設定します。
