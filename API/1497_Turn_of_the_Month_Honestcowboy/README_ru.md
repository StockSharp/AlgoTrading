# Стратегия Turn of the Month Honestcowboy
[English](README.md) | [中文](README_cn.md)

Календарная стратегия, открывающая длинные позиции в конце месяца и закрывающая их через несколько дней нового месяца с поправкой на выходные.

## Детали

- **Условия входа**: даты возле конца месяца с корректировкой на выходные
- **Лонг/Шорт**: Лонг
- **Условия выхода**: несколько дней после начала месяца
- **Стопы**: нет
- **Значения по умолчанию**:
  - `DaysBeforeEnd` = 2
  - `DaysAfterStart` = 3
- **Фильтры**:
  - Категория: Сезонность
  - Направление: Лонг
  - Индикаторы: Нет
  - Стопы: Нет
  - Сложность: Новичок
  - Таймфрейм: Дневной
  - Сезонность: Да
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Низкий
