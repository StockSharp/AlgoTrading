# ターン・オブ・ザ・マンス戦略 Honestcowboy
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

月末近くにロングポジションを建て、新月の数日後に決済するカレンダーベースの戦略。週末を考慮して日程を調整する。

## 詳細

- **エントリー条件**: 週末調整済みの月末近傍の日付
- **ロング/ショート**: ロング
- **エグジット条件**: 月初から数日後
- **ストップ**: なし
- **デフォルト値**:
  - `DaysBeforeEnd` = 2
  - `DaysAfterStart` = 3
- **フィルター**:
  - カテゴリ: 季節性
  - 方向: ロング
  - インジケーター: なし
  - ストップ: いいえ
  - 複雑さ: 初心者
  - 時間軸: 日足
  - 季節性: はい
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 低
