# Прорыв по накопленной дельте

Накопленная дельта суммирует разницу между объёмом покупок и продаж. Эта стратегия отслеживает нарастающий итог и входит в рынок, когда он пробивает свой максимум или минимум за выбранный период.

Прорыв накопленной дельты часто предшествует продолжению движения цены. Закрытие позиций происходит, когда дельта возвращается через ноль или срабатывает стоп.

## Детали

- **Условия входа**: Накопленная дельта превышает максимальное или минимальное значение за период.
- **Длинные/короткие позиции**: Оба направления.
- **Условия выхода**: Дельта пересекает ноль или стоп.
- **Стопы**: Да.
- **Значения по умолчанию**:
  - `LookbackPeriod` = 20
  - `CandleType` = TimeSpan.FromMinutes(5)
- **Фильтры**:
  - Категория: Breakout
  - Направление: Оба
  - Индикаторы: Cumulative Delta
  - Стопы: Да
  - Сложность: Средняя
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
