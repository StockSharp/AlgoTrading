# Buy Sell Renko Based 策略
[English](README.md) | [Русский](README_ru.md)

该策略使用基于 ATR 的 Renko 砖。收盘价上穿开盘价时做多，收盘价下穿开盘价时做空。

## 详情

- **入场条件**：
  - **多头**：收盘价上穿开盘价。
  - **空头**：收盘价下穿开盘价。
- **多/空**：双向。
- **出场条件**：反向信号。
- **止损**：无。
- **默认值**：
  - ATR 长度 10。
- **过滤**：
  - 类别：反转
  - 方向：双向
  - 指标：Renko
  - 止损：无
  - 复杂度：简单
  - 时间框架：无时间框架
