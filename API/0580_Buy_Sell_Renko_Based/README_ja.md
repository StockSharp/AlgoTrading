# Renkoベースの売買戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略はATRベースのサイズで作成されたRenkoレンガを取引します。Renkoのクローズがオープンを上抜けるとロングポジションを建てます。クローズがオープンを下抜けるとショートポジションを建てます。

## 詳細

- **エントリー条件**:
  - **ロング**: クローズがオープンを上抜ける。
  - **ショート**: クローズがオープンを下抜ける。
- **ロング/ショート**: 両方。
- **エグジット条件**: 反対のシグナル。
- **ストップ**: なし。
- **デフォルト値**:
  - ATR長さ 10。
- **フィルター**:
  - カテゴリ: リバーサル
  - 方向: 両方
  - インジケーター: Renko
  - ストップ: いいえ
  - 複雑さ: シンプル
  - 時間軸: 時間ベースなし
