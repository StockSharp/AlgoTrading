# Delta RSI戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は**Delta RSI**インジケーターに基づいて取引します。異なる期間を持つ2つのRSIインジケーターを比較します：

- **高速RSI**は価格変動に素早く反応します。
- **低速RSI**はトレンドフィルターとして機能します。

以下の条件を満たす**Up**シグナルの次のバーでロングポジションを建てます：

1. 低速RSIが`Level`しきい値を上回っている。
2. 高速RSIが低速RSIより高い。
3. 前のバーがUp状態を示し、現在のバーはもうUp状態でない。

以下の条件を満たす**Down**シグナルの次のバーでショートポジションを建てます：

1. 低速RSIが`100 - Level`を下回っている。
2. 高速RSIが低速RSIより低い。
3. 前のバーがDown状態を示し、現在のバーはもうDown状態でない。

オプションのフラグにより、ロングとショートのポジションの建玉と決済を個別に有効または無効にすることができます。

## パラメーター

| 名前 | 説明 |
|------|------|
| `FastPeriod` | 高速RSIの期間。 |
| `SlowPeriod` | 低速RSIの期間。 |
| `Level` | 低速RSIのしきい値レベル。 |
| `BuyPosOpen` / `SellPosOpen` | ロング/ショートポジションの建玉を許可する。 |
| `BuyPosClose` / `SellPosClose` | ロング/ショートポジションの決済を許可する。 |
| `CandleType` | 入力ローソク足の時間軸。 |

この戦略は選択した時間軸のローソク足を購読し、両方のRSI値を計算し、完成した各ローソク足でシグナルを処理します。シグナルが現れると、戦略はオプションで反対のポジションを決済し、シグナルの方向に新しいポジションを建てます。
