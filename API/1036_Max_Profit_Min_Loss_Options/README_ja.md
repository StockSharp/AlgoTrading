# Max Profit Min Loss Options戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、高速・低速の移動平均にRSI、MACD、出来高フィルターを組み合わせます。トレンドとモメンタムの条件が揃ったときにロングでエントリーし、ストップロスとトレーリングプロフィットで決済します。

## 詳細

- **エントリー条件**:
  - **ロング**: 高速MA > 低速MA、MACDがシグナルを上抜け、RSI > 売られすぎ水準で上昇パターン、出来高が平均以上。
  - **ショート**: 高速MA < 低速MA、MACDがシグナルを下抜け、RSI < 買われすぎ水準で下降パターン、出来高が平均以上。
- **エグジット**: 逆シグナルまたはストップロス/トレーリングプロフィット。
- **ストップ**: パーセントストップロスとトレーリングプロフィット。
- **デフォルト値**:
  - 高速MA長 = 9
  - 低速MA長 = 21
  - RSI長 = 14
  - 出来高SMA長 = 20
  - ストップロス = 1%
  - トレーリングプロフィット = 4%
- **インジケーター**: MA、RSI、MACD、出来高SMA
- **時間軸**: デフォルトで5分足ローソク足
