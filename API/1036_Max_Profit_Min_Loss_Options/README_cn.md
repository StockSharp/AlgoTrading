# 最大盈利最小亏损期权策略
[English](README.md) | [Русский](README_ru.md)

该策略结合快/慢均线、RSI、MACD 和成交量均线。当趋势与动量条件一致时入场，退出时使用止损和跟踪止盈。

## 细节

- **入场条件**：
  - **做多**：快线在慢线上方，MACD 上穿信号线，RSI 高于超卖且连续上升，成交量高于平均。
  - **做空**：快线在慢线下方，MACD 下穿信号线，RSI 低于超买且连续下降，成交量高于平均。
- **退出**：相反信号或止损/跟踪止盈。
- **止损**：百分比止损和跟踪止盈。
- **默认值**：
  - 快速均线周期 = 9
  - 慢速均线周期 = 21
  - RSI 周期 = 14
  - 成交量 SMA 周期 = 20
  - 止损 = 1%
  - 跟踪止盈 = 4%
- **指标**：MA、RSI、MACD、成交量 SMA
- **时间框架**：默认 5 分钟 K 线
