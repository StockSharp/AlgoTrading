# 重心平均回帰戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、最新のローソク足の多項式回帰を解くことによって、元の MQL4 エキスパートが使用した重心チャネルを再構築します。回帰中心は最小二乗近似の切片から計算され、バンド幅は同じルックバック期間にわたる終値の標準偏差から導出されます。下のバンドは回帰中心からスケーリングされた偏差を引いたものに等しく、ソース ロボットでアクセスされた `stdl` バッファーを再現します。

ライブ処理中、アルゴリズムは **Bars Back** パラメーターで定義された長さのローリング クローズ キューを維持します。完成した各キャンドルは、正規方程式系でのガウス消去による回帰係数の再計算をトリガーします。これにより、完全なローソク足履歴の保存が回避されますが、カスタムインジケーターと同じチャネルジオメトリが得られます。マトリックスの状態が悪くなった場合、更新はスキップされ、不安定な取引の決定が妨げられます。

取引ロジックは元のエキスパートを反映しています。現在のローソク足の安値が下限偏差バンド (MQL 表記の `lowerBand < Low`) を上回っている場合、戦略はこれを平均回帰バウンスとみなします。オープンなロングポジションがない場合、ショートエクスポージャーはクローズされ、ストラテジーボリュームを使用して成行買い注文が発行されます。最新の下限値、上限値、中央値は、グラフ作成または診断用の読み取り専用プロパティを介して公開されます。

リスク管理はオプションです。 **ストップロスディスタンス**と**テイクプロフィットディスタンス**は絶対価格単位で指定されます。ゼロ以外の場合、この戦略はアクティブなロングポジションのエントリー価格を記録し、ローソク足の極値をチェックしてストップまたは利益のターゲットに触れたかどうかを判断します。どちらのパラメータも指定されない場合、位置は手動または外部モジュールによって管理できます。

### パラメーター
- **キャンドル タイプ** – 回帰をフィードするキャンドル サブスクリプションの時間枠。
- **Bars Back** – 回帰チャネルの計算に使用される履歴バーの数 (デフォルトは 125)。
- **多項式次数** – チャネルの曲率を制御する多項式回帰の次数 (デフォルトは 2)。
- **Std Multiplier** – エンベロープを形成するときに標準偏差に適用される乗数 (デフォルトは 1)。
- **ストップロス距離** – 価格単位でのオプションの長いストップロスオフセット (デフォルトでは 0 が無効になります)。
- **利益確定距離** – 価格単位でのオプションの長期利益オフセット (デフォルト 0 は無効)。

この戦略は完了したローソク足のみで動作し、エントリーには成行注文を使用し、元のエキスパートの売りブランチがコメント化されているため自動空売りは実行しません。
