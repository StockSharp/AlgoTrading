# 時刻ベースのイントラデイ取引戦略
[English](README.md) | [Русский](README_ru.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | [Português](README_pt.md)

この戦略は、1日の特定の時刻に、事前に定義されたストップロスとテイクプロフィットの距離でロングおよび/またはショートポジションを建てます。インジケーターの確認なしに時刻ベースのエントリーをテストするのに役立ちます。

## 詳細

- **エントリー条件**: 設定された時と分における時刻ベースのトリガー。
- **ロング/ショート**: 両方向（設定可能）。
- **エグジット条件**: 保護ストップまたはターゲット。
- **ストップ**: あり。
- **デフォルト値**:
  - `TradeHour` = 0
  - `TradeMinute` = 0
  - `AllowBuy` = true
  - `AllowSell` = true
  - `TakeProfitTicks` = 20
  - `StopLossTicks` = 20
  - `CandleType` = TimeSpan.FromMinutes(1)
- **フィルター**:
  - カテゴリ: その他
  - 方向: 両方
  - インジケーター: なし
  - ストップ: 固定
  - 複雑さ: 基本
  - 時間軸: イントラデイ (1m)
  - 季節性: いいえ
  - ニューラルネットワーク: いいえ
  - ダイバージェンス: いいえ
  - リスクレベル: 中
