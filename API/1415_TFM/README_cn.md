# TFM
[English](README.md) | [Русский](README_ru.md)

时间框架倍数突破策略。通过将基础时间框架乘以倍数形成更高时间框架。当价格突破上一高点时做多，跌破上一低点时可做空或平多。

## 详情
- **入场条件**: 突破乘数时间框架的高/低点。
- **多空方向**: 做多，允许做空可选。
- **退出条件**: 突破相反方向或反向操作。
- **止损**: 无。
- **默认值**:
  - `CandleTime` = TimeSpan.FromMinutes(1)
  - `Multiplier` = 2
  - `AllowShort` = false
- **过滤器**:
  - 类型: 突破
  - 方向: 双向（启用做空时）
  - 指标: High/Low
  - 止损: 无
  - 复杂度: 低
  - 时间框架: 日内
  - 季节性: 无
  - 神经网络: 无
  - 背离: 无
  - 风险等级: 中
