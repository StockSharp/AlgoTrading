# TFM
[English](README.md) | [中文](README_cn.md)

Стратегия пробоя с мультипликатором таймфрейма. Использует старший таймфрейм, полученный умножением базового. Длинная позиция открывается при пробое максимума, а при пробое минимума возможен шорт или закрытие лонга.

## Детали
- **Критерии входа**: пробой уровней старшего таймфрейма.
- **Длинные/короткие**: длинная с опциональным шортом.
- **Критерии выхода**: пробой противоположного уровня или разворот.
- **Стопы**: нет.
- **Значения по умолчанию**:
  - `CandleTime` = TimeSpan.FromMinutes(1)
  - `Multiplier` = 2
  - `AllowShort` = false
- **Фильтры**:
  - Категория: Пробой
  - Направление: Оба (при включенных шортах)
  - Индикаторы: High/Low
  - Стопы: Нет
  - Сложность: Низкая
  - Таймфрейм: Внутридневной
  - Сезонность: Нет
  - Нейросети: Нет
  - Дивергенция: Нет
  - Уровень риска: Средний
